# Format

## Input

＊第一行：由空格隔_的整担N和M。

＊第二行：由空格隔_的整担Si和Sj。

＊第三到第M+2行：第i+2行有由空格隔_的整Ei和Ej，表示三角形（Ei，Ej）是出口。

## Output

＊第一行：由空格隔_的整担OUTi和OUTj。

＊第二行：一为的整担T。
