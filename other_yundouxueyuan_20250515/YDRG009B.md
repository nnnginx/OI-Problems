### 题目描述

你在研究蚂蚁🐜的行为模式。

通过分析观察到的数据，你预测到，在接下来的 $m$ 天中，蚁后🐜🐜会发布两种类型的任务共 $n$ 个，第 $i$ 个任务的类型为 $c_i\in \{0,1\}$，会在第 $s_i$ 天发布；而如果在第 $t_i$ 天之前（包括第 $t_i$ 天）这个任务还没有被完成，蚁后🐜🐜就会取消这个任务。

蚂蚁🐜的行为模式比较单一，具体来说，在每一天，蚂蚁🐜首先会选择一个类型 $p\in \{0,1\}$，然后找到类型为 $p$ 的所有**已经被发布，且仍未被取消，且目前尚未被完成**的任务中，截止日期最靠前，在此基础上编号最小的一个任务（即，设当前是第 $k$ 天，那么蚂蚁🐜会选取 $c_i=p,s_i\le k\le t_i$ 且 $t_i$ 最小的任务 $i$ 中，$i$ 最小的一个），在这一天完成这个任务。如果类型为 $p$ 的任务都已经被完成或者被取消，或者仍未被发布，那么蚂蚁🐜在这一天就会选择休息。

你的研究已经取得了一些成果，具体来说，你可以在接下来的 $m$ 天中控制蚂蚁🐜在每一天选择的任务类型 $p$ 具体是 $0$ 还是 $1$。为了获得一些更有价值的数据，你希望你选择的这个方案能够让蚂蚁🐜完成的任务数量最小化。

现在请你设计一种方案，使得蚂蚁🐜完成的任务数量最小，并求出这个最小值是多少。

### 输入格式

从文件 `designant.in` 中读入。

第一行两个正整数 $n,m$。

接下来 $n$ 行，每行三个整数 $s_i,t_i,c_i$。

### 输出格式

输出到文件 `designant.out` 中。

第一行一个非负整数表示蚂蚁🐜完成的任务数量的最小值。

第二行一个长为 $n$ 的 01 串，以空格分隔，表示你为蚂蚁🐜每一天选择的类型。

### 样例 $1$ 输入

```
6 6
4 6 1
1 2 0
2 6 1
1 5 0
2 3 0
4 5 1
```

### 样例 $1$ 输出

```
2
1 1 1 0 0 0
```

### 样例 $1$ 解释

按照你选择的方案，蚂蚁🐜每一天的行为分别是：

- 第一天，类型为 $1$ 的任务都没有被发布，因此无事可做。
- 第二天，发布了类型为 $1$ 的任务 $3$，因此在这一天做完任务 $3$。
- 第三天，类型为 $1$ 的任务要么已经被做完，要么都没有被发布，因此这一天无事可做。
- 第四天，这天需要做类型为 $0$ 的任务，此时只有任务 $4$ 仍未结束，因此这一天做完任务 $4$。
- 第五、六天，类型为 $0$ 的任务要么被做完了，要么都结束了，因此这两天都无事可做。

这样，蚂蚁🐜一共只完成了两个任务，是可能的最小值。

### 样例 $2$ 输入

```
9 10
1 1 0
2 7 1
1 1 1
4 4 0

4 9 1

```

### 样例 $2$ 输出

```

0 0 1 1 1 1 1 1 0 1
```

### 样例 $2$ 解释

按照这个方案，蚂蚁🐜分别在第 $1,3,4,6$ 天完成了第 $1,2,8,7$ 号任务。
