# Format

## Input

第一行为字符串A。
第二行为字符串B。

## Output

输出在B的所有循环同构串中，有多少个能够与A匹配。

# Samples

```input1
样例输入1
aaaa
aaaa

样例输入2
a*a
aaaaaa

样例输入3
*a*b*c*
abacabadabacaba
```

```output1
样例输出1
4

样例输出2
6

样例输出3
15
```

# Limitation

对于100% 的测试点，1<=N<=100，1≤M≤100000。
