# Description

![image](./1866/file/3Q_MSJ8jMMnQVx7Oh6SiC.jpeg) Given two integers x and y, print the sum.

# Format

## Input

![image](./1866/file/dbP3BL6NpyWEIXGPD8AqK.jpeg) Two integers x and y, satisfying $0\leq x,y\leq 32767$ .

## Output

![image](./1866/file/Ybcpf67qUPmZ5ROb6MKsz.jpeg) One integer, the sum of x and y.

# Samples

```input1
3
3 4 5
4
1 3 3 5
3

3

3
185 185 222
4
5 45 89 113
4

```

```output1
6

```
