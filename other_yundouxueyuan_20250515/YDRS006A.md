# 题目描述

云斗小课堂结课啦！yummy 老师和云浅老师来给小朋友们发礼物。

现在有 $n$ 个小朋友排成一排，每个小朋友的手里拿着一个数字编号。你知道他们的数字编号是一串连续的严格递增的正整数。

两位老师会从左往右依次给每个小朋友发礼物。当然，yummy 老师是喜欢合数的，而云浅老师是喜欢连续的。因此当遇到的某个小朋友手里拿着的数字编号不是合数时，包括这个小朋友在内的所有还没拿到礼物的小朋友都没有了礼物。

试构造一个 $k$ 表示最左边的小朋友手里拿着的数字编号，以最大化拿到礼物的小朋友的个数。

---

**【形式化题意】**

对于给定的正整数 $n$，试构造正整数 $k$，使得以 $k$ 为开头的连续 $n$ 个整数均为合数。形式化的，对于所有 $k\le i<k+n$，都有 $i$ 为合数。

# 输入格式

**本题在单个测试点中有多组数据**。

输入共 $T+1$ 行。

第 $1$ 行 $1$ 个整数，表示单个测试点中的数据组数 $T$。

对于每组数据，输入共 $1$ 行 $1$ 个整数，表示给定的 $n$。

# 输出格式

对于每组数据，输出共 $1$ 行 $1$ 个整数，表示构造的 $k$。

若存在多解，输出其中任一即可。但是你需要保证 $k$ 应在 `long long` 的表示范围内。

# 输入输出样例

```input1
2
1
3
```

```output1
6
14
```

# 提示

**【样例 1 解释】**

在第 $2$ 组数据中，$14\sim16$ 均为合数。

---

**【数据范围】**

**本题开启捆绑测试**。

对于 $100\%$ 的数据，$1\le T\le5$，$1\le n\le41$。保证数据有解。
