# Description


我们给定一个整数集合 *A* . 考虑一个非负整数集合 *A* ', 所有属于 *A* ' 的集合的数x满足当且仅当能被表示成一些属于*A* 的元素的和(数字可重复). 比如, 当 *A* = {2,5,7}, 属于 *A* ' 的数为: 0 (0个元素的和), 2, 4 (2  +  2) and 12 (5 + 7 or 7 + 5 or 2 + 2 + 2 + 2 + 2 + 2); 但是元素1和3不属于 *A* '.

# Format

## Input

第一行有一个整数 *n* : 代表集合 *A*的元素个数, 1 <= *n* <= 5000. 接下来每行一个数*a~i~*描述一个元素, 1 <= *a~i~* <= 50000. *A* = {*a* ~1~ , *a* ~2~ , ...,  *a~n~* }, *a*~1~ < *a*~2~ < ... <  *a~n~* .

接下来一个整数 *k* , 1 <= *k* <= 10000. 每行一个0 到 1000000000, 分别代表*b* ~1~ , *b* ~2~ , ...,  *b~k~* .

## Output

输出*k*行. 第*i*行打印TAK, 如果 *b~i~* 属于 *A* ', 否则打印NIE.
