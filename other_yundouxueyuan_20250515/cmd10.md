# 骰子

## 题目描述

奶龙和暴暴龙进行一场比赛，每个人同时掷出 $n$ 枚骰子，每个骰子掷出 $1,2,3,4,5,6$ 点的概率均为 $\frac{1}{6}$ ，每个人的分数为 $n$ 个骰子点数之和，特殊的，如果一个人掷出的 $n$ 个骰子点数都相同，那么这个人总分数翻倍。

奶龙先掷出 $n$ 枚骰子，第 $i$ 枚骰子点数为 $a_i$ ，奶龙想知道自己总分数严格大于暴暴龙总分数的概率。

## 输入格式

第一行输入一个整数 $n$ 表示骰子的个数。

第二行 $n$ 个整数表示每个骰子的点数。

## 输出格式

输出用最简分数表示，形如 $P/Q$ ，其中 $0 \le P \lt Q,gcd(P,Q)=1$ 。

## 样例 #1

### 样例输入 #1

```
2
2 3
```

### 样例输出 #1

```
5/36
```

## 样例 #2

### 样例输入 #2

```
3
6 6 6
```

### 样例输出 #2

```
215/216
```

## 样例 #3

### 样例输入 #3

```
6
1 2 3 4 5 6
```

### 样例输出 #3

```
2645/5832
```

## 样例 #4

### 样例输入 #4

```
6
1 1 1 1 1 2
```

### 样例输出 #4

```
0/1
```

## 提示

对于样例1，奶龙的得分为 $5$ ，在暴暴龙的 $36$ 种可能结果种当且仅当 $(1,1),(1,2),(1,3),(2,1),(3,1)$ 这 $5$ 种结果的分数比奶龙小，注意当暴暴龙掷出 $(2,2)$ 时，暴暴龙分数为 $8$ 。

对于样例2，除非暴暴龙也掷出三个 $6$ ，否则奶龙直接获胜。

对于 $10\%$ 的数据，满足 $n =1$ 。

对于另外 $10\%$ 的数据，满足 $n =2$ 。

对于另外 $10\%$ 的数据，满足 $n =3$ 。

对于另外 $30\%$ 的数据，满足 $n \le 8$ 。

对于另外 $10\%$ 的数据，满足 $1 \le i \le n :a_i = 6$ 。

对于 $100\%$ 的数据，满足 $1 \le n \le 20,1 \le a_i \le 6$ 。
