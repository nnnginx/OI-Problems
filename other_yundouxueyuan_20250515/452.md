# Background

Special for beginners, ^_^

# Description

给出N(2<=N<=50000)个长度为4的字符串，问有且仅有D(1<=D<=4)处不相同的字符串有几对。

# Format

## Input

第1行： N，D 以下N行每行一个字符串

## Output

一个数：有多少对有且仅有处不相同的字符串。

# Samples

```input1
4 2
0000
a010
0202
a0e2
```

```output1
3
```

# Limitation

1s, 1024KiB for each test case.
