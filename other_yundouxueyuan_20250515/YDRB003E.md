# 题目描述

小七要给奶龙做菜，他手头共有类型分别为 $a_i,a_i\in \{1\le x\le n|x\in\Z\}$ 的 $t$ 个食材。并且他有 $k$ 个硬币，每个硬币可以购买任意类型 $\in \{1\le x\le n|x\in\Z\}$ 的食材（他可以选择花费多少硬币，也可以不买）。

现在他要确定每道菜的食材数量。为了避免浪费并使得菜品分量平均，他需要将所有他手头和买来的食材做 $s$ 道菜，每道菜 $m$ 个食材，且这 $m$ 个食材的类型各不相同。

奶龙很能吃，他问你每道菜包含的最多可能的食材数.

# 输入输出格式

## 输入格式

第一行两个数 $n$,$k$。
第二行 $n$ 个数，第 $i$ 个数 $a_i$ 表示类型为 $i$ 的食材数量。

## 输出格式

一行一个数表示 $m$ 的最大值。

# 输入输出样例

in：
```
10 8
7 4 6 6 9 3 10 2 8 7
```
out:
```
7
```

# 数据范围与约定

对于 $20\%$ 的数据 ，$1 \le n,a_i,k \le 5$

对于 $50\%$ 的数据 ，$1 \le n,a_i,k \le 5000$

对于 $100\%$ 的数据， $1\le n \le 500000$,$1 \le a_i,k \le 1e12$
