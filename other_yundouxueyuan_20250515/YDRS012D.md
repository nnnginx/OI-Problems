# OTK

## 题目背景

众所周知，OTK是卡牌游戏里面不可不品的一环，在冰岛原神当中，OTK卡组也非常强力

## 题目描述

奶龙初始有 $n$ 张手牌，手牌上限为 $s$，牌堆中有 $m$ 张牌（保证 $m$ 为偶数），牌有如下三种：

A. 摸 $2$ 张牌；

B. 造成 $x$ 点伤害

C. 造成 $y$ 点伤害

暴暴龙的粮仓具有 $h$ 点生命值，求出在采取最优策略时, 最大有多少概率能够摧毁粮仓. (奶龙聪明绝顶, 只要摸牌结果有可能摧毁粮仓, 则奶龙一定能找到对应的出牌策略)

每一次摸牌都是等概率从牌堆当中摸牌，如果摸牌后手牌数量超过上限，则不能打出 `A` 类牌 (例如, 手中有且仅有 $s$ 张 `A` 类牌, 则不能进行任何操作)

## 输入格式

第一行输入四个整数 $ n, s, m, h$

接着输入两个整数 $x, y$

接着输入三个整数 $ a_1, b_1, c_1$， 分别表示在手牌中每种牌有多少张。

接着输入三个整数 $ a_2, b_2, c_2$， 分别表示在牌堆中每种牌有多少张。

## 输出格式

输出一个模 $10^9+7$ 意义下的有理数，表示奶龙有多大概率能摧毁暴暴龙的粮仓.

## 样例 #1

### 样例输入 #1

```
1 100 4 3
1 1
1 0 0
1 3 0
```

### 样例输出 #1

```
500000004
```

## 提示

$$
1 \leq n, s, m \leq 3000\\
1 \leq x, y \leq 100000\\
1 \leq h \leq 10^9
$$


| 数据点编号 | 限制                                 |
| ---------- | ------------------------------------ |
| 1~2        | $n, s, m \leq 10$  ;  $x, y \leq 10$ |
| 3~6        | $n, s, m \leq 200$                   |
| 7~10       | 无额外限制                           |
