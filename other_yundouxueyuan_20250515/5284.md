# Background

公历纪年法中，能被4整除的大多是闰年，但能被100整除而不能被400整除的年份不是闰年，如1900年是平年，2000年是闰年。
具体地：

* 能被4整除但不能被100整除的年份是闰年；
* 能被100整除同时能被400整除的年份也是闰年。
  除此之外都是平年。

# Description

判断某年是否是闰年。

# Format

## Input

输入只有一行，包含一个整数$y(0 < y < 3000)$ 代表年份

## Output

一行，如果公元$y$年是闰年输出$Y$，否则输出$N$。

# Samples


```input1
2024
```

```output1
Y
```

```input2
2023
```

```output2
N
```
