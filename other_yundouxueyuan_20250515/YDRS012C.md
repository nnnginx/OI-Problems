# Description

给定序列 $a$。

若正整数 $m$ 满足：

$$
a_1 \bmod m \leq a_2 \bmod m \leq ... \leq a_n \bmod m
$$

则称 $m$ 是**好的**。

我们希望你找到所有**好的** $m$ 的和。若有无穷多个 $m$ 请输出 $-1$。

多测。

# Format

## Input

第一行一个整数 $t$ 表示数据组数。

每组数据两行，第一行是 $n$，第二行 $n$ 个整数，第 $i$ 个整数表示 $a_i$。

## Output

对于每组数据输出答案。

# Samples

# Limitation
样例 $1$ 解释：

$m=1,9,10$ 满足条件。

| 子任务 | $\sum n$ | $a_i$ | 特殊性质 | 空间限制 | 分值 | 
| :----------: | :----------: | :----------: | :----------: | :----------: | :----------: |
| $1$ | $\leq 800$ | $\leq 10^4$ | 无 | 512MB | $10$ |
| $2$ | $\leq 800$ | $\leq 10^9$ | A | 512MB | $10$ |
| $3$ | $\leq 800$ | $\leq 10^9$ | B | 512MB | $20$ |
| $4$ | $\leq 40$ | $\leq 10^9$ | 无 | 512MB | $30$ |
| $5$ | $\leq 800$ | $\leq 10^9$ | 无 | 512MB | $20$ |
| $6$ | $\leq 800$ | $\leq 10^9$ | 无 | 32MB | $10$ |

特殊性质 A：$t=1$，且 $n$ 和 $a_i$ 均匀从值域里随机

特殊性质 B：$|a_i-a_{i+1}|=1$
