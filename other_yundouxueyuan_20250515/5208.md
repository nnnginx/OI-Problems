# Background

质数又称素数。一个大于1的自然数，除了1和它自身外，不能被其他自然数整除的数叫做质数；否则称为合数（规定1既不是质数也不是合数）。

比如自然数 2 只能被1和他本身整除，所以它是质数，\
自然数4 可以被1、2、4整除 所以它不是质数。


# Description
输入一个正整数$n$是否为质数，是则输出Yes，否则输出No。
# Format

## Input
一个正整数$n$
## Output
如果是质数输出Yes,否则输出No
# Samples

```input1
3
```

```output1
Yes
```

# Limitation
1s, 1024KiB for each test case.
