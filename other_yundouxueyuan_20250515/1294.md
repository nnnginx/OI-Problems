# Description

给定一个长度为k的数字串N以及三个质数p,q,r，请你将N划分为三段非空字符串，使得第一段能被p整除，第二段能被q整除，第三段能被r整除，且每一段都不含前导0。

注意：单独的0是允许的。

# Format

## Input

第一行包含四个正整数k,p,q,r(3<=k<=1000000,2015<=p,q,r<=100000)。

第二行为一个长度为k的数字串N。

## Output

输出一个整数，即合法的划分方案的总数。

# Samples

```input1
19 2017 2027 2029
2017202741127832029
```

```output1
2
```
