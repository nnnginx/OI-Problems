# Description

有一个由数字1，2，...

，9组成的数字串（长度不超过200），问如何将M(M<=20)个加号("+")插入到这个数字串中，使所形成的算术表达式的值最小。请编一个程序解决这个问题。

注意：

加号不能加在数字串的最前面或最末尾，也不应有两个或两个以上的加号相邻。

M保证小于数字串的长度。

例如：数字串79846，若需要加入两个加号，则最佳方案为79+8+46，算术表达式的值133。

# Format

## Input

从键盘读入输入文件名。数字串在输入文件的第一行行首（数字串中间无空格且不折行），Ｍ的值在输入文件的第二行行首。

## Output

在屏幕上输出所求得的最小和的精确值。

# Samples

```input1
82363983742
```

```output1
3
```
