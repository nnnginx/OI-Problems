# Description

![image](./1903/file/gHs2tUdWcefPWmbHi0ojL.jpeg)

一个*m*×*n*的棋盘，左上至右下编号为(1, 1)至 (*m*, *n*)，并给定每个格子到周围四个格子的概率　　。

一个骑士从(1, 1)开始，按照给定概率走，问到达(*m*, *n*)的期望步数。

题目保证从任一格开始到(m, n)的概率均为1 。
