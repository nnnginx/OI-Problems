# Description

这是一道模板题。

给定一个多项式方程 $\sum_{i=0}^n a_i x^i = 0$，求出它在模素数 $p$ 意义下的所有根。



## Input

第一行两个正整数 $n$，$p$，表示多项式的次数和模数。

第二行 $n+1$ 个整数 $a_0,a_1,\cdots,a_n$，表示这个多项式。


## Output

第一行一个整数 $k$，表示方程根的个数。

第二行 $k$ 个从小到大的整数，表示所有根。注意，这里我们认为重根**只算一个**。

# Samples

```input1
2 5
1 0 1
```

```output1
2
2 3
```

```input1
2 7
1 0 1
```

```output1
0
```

# Limitation

$3\le p\le 10^9,1\le n\le 100,0\le a_i< p,a_n\neq 0$。
