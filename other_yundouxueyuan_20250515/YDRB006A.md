# 买菜

## 题目描述

奶龙在使用线上买菜软件时，发现不同商品的包装规格不同，导致难以直接比较价格。

例如：肉类可能按克计价，而薯片按袋出售。为了准确比较价格，奶龙决定计算每公斤的价格。

现在，请你编写程序，根据输入的每个商品总价和重量，输出对应的每公斤价格。

## 输入格式

第一行包含一个整数 $T$，表示商品种类数。  

接下来 $T$ 行，每行包含两个整数 $P$ 和 $W$，分别表示该商品的总价（元）和重量（克）。

## 输出格式

输出 $T$ 行，每行一个保留两位小数的浮点数，表示该商品的每公斤价格。

## 样例 #1

### 样例输入 #1

```
3
10 500
15 300
20 250
```

### 样例输出 #1

```
20.00
50.00
80.00
```

## 样例解释

第一个商品总价 10 元，重量 500 克。每公斤价格为 $(10 \div 500) \times 1000 = 20.00$ 元。

## 数据范围

- $1 \leq T \leq 1000$  
- $1 \leq P \leq 1000$  
- $1 \leq W \leq 100000$（即 100 公斤）
