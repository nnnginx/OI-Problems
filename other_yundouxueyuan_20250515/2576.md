# Description

Bytel 公司开始生产一种串-并联电路，每个这样的电路包含很多相连的电子单元以及两个电源，一个串-并联电路可能有以下组成:
![](https://darkbzoj.cc/JudgeOnline/upload/201203/1(3).jpg)

这些电路都将组装在一个双面的电路板上。现在的问题就是要确定哪些电路印在正面和哪些印在反面。因为一些技术原因电路应该尽量多的印在反面，但是要求每个单元至少要有一条连接的线路印在正面。求怎样规划使得印在正面的线路最少。

# Format

## Input

电路的定义由以下的递归方式进行:

* 如果第一行有一个大写字母S 和一个整数*n* (2 <= *n* <= 10000) ，那么这个电路将由*n*个小的串-并联电路串联在一起构成，这些小网络紧接着在下面的行进行描述,
* 如果第一行有一个大写字母R 和一个整数*n* (2 <= *n* <= 10000) ，那么这个电路将由*n*个小的串-并联电路并联在一起构成，这些小网络紧接着在下面的行进行描述,
* 如果一行只有一个大写字母X 那么这个电路仅有一个电子单元.

大写字母X 的出现总数不会超过10000000, 递归的层数最大不超过500.

## Output

输出仅一个数，表示最少有多少线路印在正面.

# Samples

```input1
R 3
S 2
X
R 2
S 2
X
X
S 2
X
X
S 3
X
X
X
R 2
X
X
```

```output1
8
```
