# Format

## Input

![image](./1869/file/7MNLte6LB0EjfEX-D_BwA.jpeg) Two integers x and y, satisfying $0\leq x,y\leq 32767$ .

## Output

![image](./1869/file/Ce-44FP_lmsfI7mSIgwZd.jpeg) One integer, the sum of x and y.

# Samples

```input1
1
1 2 2
1 1 1
```

```output1
1.414
```
