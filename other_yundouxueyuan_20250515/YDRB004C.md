# 周边

## 题目描述

小朋友太喜欢奶龙了！为了表达这种喜爱，他们购买了奶龙周边，有红奶龙，橙奶龙，……，各种奶龙。具体的，他们买了 $n$ 种奶龙，第 $i$ 种奶龙的数量是 $a_i$。小朋友不想让自己不同种类的奶龙个数相同，于是他决定继续购买奶龙。为了达成小朋友的要求，小朋友最少需要购买几只奶龙呢？

## 输入格式

第一行，$n$；

第二行 $n$ 个整数，第 $i$ 个数字表示 $a_i$。

## 输出格式

一行一个整数，小朋友最少需要购买奶龙的数量。

## 样例

### 样例输入 #1

```
3
2 3 2
```

### 样例输出 #1

```
2
```

### 样例输入 #2

```
5
3 7 5 2 6
```

### 样例输出 #2

```
0
```

### 样例输入 #3

```
7
2 1 4 7 8 9 1
```

### 样例输出 #3

```
2
```

## 样例解释

对于第一组样例，可以再买1，2两只奶龙，奶龙数量变成：3，4，2，满足各不相同。

对于第二组样例，奶龙数量各不相同，无需购买新的。

## 数据范围

对于 $30\%$ 的数据，$n \leq 10$；

对于 $75\%$ 的数据，$a_i \leq 10$；

对于 $100\%$ 的数据，$1\leq n\leq 100000, 0\leq a_i\leq 100000$。
