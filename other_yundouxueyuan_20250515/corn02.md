# 集市

## 题目描述

小七带奶龙来吃东西, 集市里有 $N$ 种小吃，第 $i$ 种小吃的价格是 $P_i$。

现在有一个特殊的促销规则：如果顾客折前总消费超过 $X$ 元，就可以获得 $Y$ 元的优惠。每位顾客只能享受一次优惠。

小七没多少钱，所以他只允许奶龙选择两份小吃，奶龙不想两份小吃都一样，所以请计算：小七最少需要花多少钱，才能让奶龙恰好吃到两份不同种类的小吃。

## 输入格式

第一行包含三个整数 $N$, $X$, $Y$。
第二行包含 $N$ 个整数，表示每种小吃的价格 $P_i$。

## 输出格式

一个整数，表示小七的最少花费。

## 样例 #1

### 样例输入 #1

```
4 10 3
5 7 3 8
```

### 样例输出 #1

```
8
```

## 样例 #2

### 样例输入 #2

```
3 20 7
9 12 11
```

### 样例输出 #2

```
14
```

## 样例解释

对样例一, 可以购买价格为 $3$ 和 $5$ 的小吃，总价为 $8$ 元，最终花费 $8$ 元。如果选择购买价格 $3$ 和 $7$ 的小吃，总价为 $10$ 元；不能使用优惠券。

对样例二, 可以购买价格为 $9$ 和 $12$ 的小吃，总价为 $21$ 元；$21 > X$，可以使用优惠券减少 $7$ 元，最终花费 $14$ 元。

## 数据范围

$1 \leq N \leq 100$, $1 \leq P_i \leq 1000$, $1 \leq Y \leq X \leq 1000$
