

# Description

有一个*n*行*n*列的棋盘，每个格子上都有一个硬币，且*n*为偶数。每个硬币要么是正面朝上，要么是反面朝上。每次操作你可以选定一个格子( *x,y* )，然后将第*x*行和第*y*列的所有硬币都翻面。求将所有硬币都变成同一个面最少需要的操作数。

# Format

## Input

第一行包含一个正整数 *n* 。

接下来*n*行，每行包含一个长度为*n*的01字符串，表示棋盘上硬币的状态。

## Output

仅包含一行，为最少需要的操作数。

# Samples

```input1
4
0101
1000
0010
0101
```

```output1
2
```

# Limitation

【样例说明】

对(2,3)和(3,1)进行操作，最后全变成1。

【数据规模】

对于100%的数据，n ≤ 1,000。
