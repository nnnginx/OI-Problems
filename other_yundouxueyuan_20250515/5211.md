

# Description

“九九乘法表”是小学数学必学内容，相信各位同学都背过，现在就请你编程输出它吧。
![image](/d/wwzx/p/33/file/gXk91vQnC4pkd2pX70gWf.png?type=additional_file)

# Format

## Input

一行，包含一个整数n，表示要求输出到“乘法表”第n行（1<=n<=9）。

## Output

输出n行，即“九九乘法表”的前n行。

每个乘式之间用单个空格隔开，其他数字和运算符之间没有空格。

# Samples

```input1
5
```

```output1
1*1=1 
1*2=2 2*2=4 
1*3=3 2*3=6 3*3=9 
1*4=4 2*4=8 3*4=12 4*4=16 
1*5=5 2*5=10 3*5=15 4*5=20 5*5=25
```
