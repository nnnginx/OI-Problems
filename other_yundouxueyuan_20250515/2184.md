# Description

Jry最近做(屠)了很多数据结构题，所以想 BS你，他希望你能实现一种数据结构维护一个序列：

![](https://darkbzoj.cc/JudgeOnline/upload/201312/aa.jpg)

# Format

## Input

第一行n；
第二行n个数；
第三行q，代表询问个数；
接下来q行，每行一个op，输入格式见描述。

## Output

对于7≤op≤11的操作，一行输出一个答案。

# Limitation

n,q≤100000；

任意时刻数列中的数≤2^31-1。

0≤任意时刻数列中的数≤2^31-1。
本题共3组数据
