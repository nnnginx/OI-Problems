# Format

## Input

第一行2个正整数N和K。

## Output

一个整数ANS，表示计算出的答案。如果无法计算，请随意输出任何理由，KD都将以该测试点0分来惩罚你的失败。

# Samples

```input1
8 5
```

```output1
10920
```

# Limitation

对于100%的数据，N<=5000；0<K<N；LENGTH_ANS<=20000。

注意事项：

由于KD的宽宏大量气度不凡，它为生命受到严重威胁的你解释传说中的冒泡排序：

i从n-1到1循环

（一轮的开始）

j从1到i循环

```
如果a[j]>a[j+1]

  交换a[j]和a[j+1]
```

（一轮的结束）
