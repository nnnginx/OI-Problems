# Format

## Input

第一行存储了绝缘层N的数，1 <= *n* <= 100000。在接下来的N行的每一行中都存储了*a* ~1~ , *a* ~2~ , ..., *a~n~*系数中的一个。这些系数都是整数，1 <= *a~i~* <= 10000

## Output

第一行，唯一的一行中，你的程序应存储一个整数，相当于绝缘体系数最大值，绝缘体A是由根据一定的顺序排列的，并给出了系数的绝缘层的构成。

# Samples

```input1
4
5
4
1
7
```

```output1
24
```
