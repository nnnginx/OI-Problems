# Description

给定一个N，N<=50 000个节点的仙人掌，其是指每条边最多在一个环中的无向图，求仙人掌有多少种自同构。自同构是指得是图的顶点集合V到V的变换M，
![image](./3571/file/mDpucAIN0HlDwdpFSZNWs.jpeg)

以P1^a1*P2^a2...Pk^ak的形式输出，其中Pk是素数。
