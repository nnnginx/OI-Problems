# Description

回文词是一种对称的字符串一一也就是说,一个回文词,从左到右读和从右到左 读得到的结果是一样的。任意给定一个字符串,通过插入若干字符,都可以变成一个 回文词。你的任务是写一个程序,求出将给定字符串变成回文词所需插入的最少字符 数。

比如字符串“Ab3bd”,在插入两个字符后可以变成一个回文词(“dAb3bAd”或 “Adb3bdA”)。然而,插入两个以下的字符无法使它变成一个回文词。

# Format

## Input

输入文件的文件名是 PALIN.IN。

文件的第一行包含一个整数 N ,表示给定字符串的长度,3≤ N≤5000。

文件的第二行是一个长度为  N  的字符串。字符串仅由大写字母“A”到“Z”,小写字母

“a”到“z”和数字“0”到“9”构成。大写字母和小写字母将被认为是不同的。

## Output

输出文件名是  PALIN.OUT,文件只有一行,包含一个整数,表示需要插入的最少 字符数。

# Samples

```input1
5
Ab3bd
```

```output1
2
```
