# Format

## Input

第一行一个正整数。

之后行描述集合。第一个数表示集合中元素的个数，之后给出集合中的元素。

之后一行一个正整数。

之后行每行描述一个询问。格式与之前相同。

## Output

对于每个询问，在单独的一行内输出答案。

# Samples

```input1
7
0
1 1
1 1
1 2
2 2 3
0
2 2 6
3
2 2 3
2 3 5
2 4 5
```

```output1
3
3
4
```

# Limitation

对于100% 的数据，1 <= n, m <= 50000，1 <= 10^9，-10^9 <= a, b, x, y <= 10^9。
