

# Description

给出N个三维空间上的点. 问有多少条直线,这些直线上至少有三个点.



## Input

第一行给出数字N,N在[4,1000] 下面N行,每行三个数字,用于描述点的坐标,其值在[-10000,10000]

## Output

有多少条直线

# Samples

```input1
7
1 0 -1
3 4 5
2 2 2
3 3 3
-5 -5 -5
1 1 1
-3 4 0
```

```output1
2
```

# Limitation

The first line contains points (-5; -5; -5), (1; 1; 1), (2; 2; 2), and (3; 3; 3), and the second line contains points (1; 0; -1), (2; 2; 2), (3; 4; 5).
