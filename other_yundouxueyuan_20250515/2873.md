# Description

第二关和很出名的斐波那契数列有关，地球上的OIer都知道：F1=1, F2=2, Fi = Fi-1 + Fi-2，每一项都可以称为斐波那契数。现在给一个正整数N，它可以写成一些斐波那契数的和的形式。如果我们要求不同的方案中不能有相同的斐波那契数，那么对一个N最多可以写出多少种方案呢？

# Format

## Input

只有一个整数N。

## Output

一个方案数

# Samples

```input1
16
```

```output1
4
```

# Limitation

Hint：16=3+13=3+5+8=1+2+13=1+2+5+8

对于30%的数据，n<=256

对于100%的数据，n<=10^18
