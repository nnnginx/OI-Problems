# 题目背景


小 $S$ 和小 $T$ 是好朋友，他们两个生活在一个平面直角坐标系上。

# 题目描述


某天，小 $S$ 要去找小 $T$ 玩。

初始时小 $S$ 在 $\left(x_1,y_1\right)$，小 $T$ 在 $\left(x_2,y_2\right)$。同时，小 $S$ 手中有 $n$ 个护符，第 $i$ 个护符的魔力值为 $a_i$。

小 $S$ 在使用第 $i$ 个护符后，可以向 **任意一个角度方向（例如与 x 轴正方向成 37 度、114 度等）** 走 $a_i$。由于护符十分宝贵，小 $S$ 想问你，从前向后依次使用护符，至少需要几个护符才能到达小 $T$ 的位置呢？

若 $n$ 个护符全使用完也无法到达，请输出 $-1$。

# 输入格式

第 $1$ 行 $5$ 个整数 $n,x_1,y_1,x_2,y_2$。

第 $2$ 行 $n$ 个整数 $a_1,a_2,a_3,\dots,a_n$。

# 输出格式

一行一个整数，表示答案。

# 样例

```input1
5 0 0 0 3
4 2 1 4 3
```

```output1
2
```

```input2
5 0 0 100 100
1 1 1 1 1
```

```output2
-1
```

对于另外 $15\%$ 的数据，$n=2$。

对于所有数据，$1\le n\le2\times10^{6}$，$-10^{9}\le x_1,x_2,y_1,y_2\le10^{9}$，$0\le a_i\le10^9$
