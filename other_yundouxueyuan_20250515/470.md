# Format

## Input

第1行：10个空格分开的整数: N, a, b, c, d, e, f, g, h, M

## Output

第1行：满足总重量最轻，且用度之和最大的N头奶牛的总体重模M后的余数。

# Samples

```input1
2 0 1 5 55555555 0 1 0 55555555 55555555
```

```output1
51
```

# Limitation

样例说明：公式生成的体重和有用度分别为： 体重：5, 6, 9, 14, 21, 30 有用度：0, 1, 8, 27, 64, 125.
