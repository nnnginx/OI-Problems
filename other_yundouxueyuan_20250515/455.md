# Background

Special for beginners, ^_^

# Description

有N(2<=N<=620000)快砖，要搭一个N层的塔，要求：如果砖A在砖B上面，那么A不能比B的长度+D要长。问有几种方法,输出 答案 mod 1000000009的值

# Format

## Input

第一行： N，D 第二行： N个数，表示每块砖的长度。

## Output

方案数。输出要mod1000000009

# Samples

```input1
4 1
1 2 3 100
```

```output1
4
```

# Limitation

1s, 1024KiB for each test case.
