# Background

Special for beginners, ^_^

# Description

1tthinking随便地画了一些圆. ftiasch认为这些圆有交集（面积非零）的可能性不大。因为他实在画了太多圆，所以你被请来判断是否存在交集。

# Format

## Input

第1行，一个整数 *N* (1 ≤ *N* ≤ 10 ^5^ ), 圆的数量。

第2到 *N* 行: 三个整数  *X~i~* ,  *Y~i~* ,  *R~i~* , 圆心在 ( *X~i~* ,  *Y~i~* ), 半径为 *R~i~* 的圆。

## Output

如果存在面积非零的交集，则输出 "YES"，否则输出 "NO"。

# Samples

```input1
3
0 0 1
0 1 1
1 0 1
```

```output1
YES
```

# Limitation

1s, 1024KiB for each test case.
