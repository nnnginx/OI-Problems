# Format

## Input

第一行是一个正整数N，表示水晶数量。

接下来N行，每行四个整数xi，yi，zi，ci，空格分隔，表示一个水晶的位置和价值。

有可能有水晶的位置重合。

N≤50000，1≤ci≤1000，|xi|, |yi|, |zi|≤1000.

## Output

仅一行，一个实数，表示剩余水晶的价值总和。四舍五入保留1位小数。

1 0 0 5
