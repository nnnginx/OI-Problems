# 题目描述

给出整数 $n,a,b,c$，判断 $a^n+b^n=c^n$ 是否成立。

注意：当 $a\ne 0$ 时 $a^0=1,a^{-n}=\dfrac{1}{a^n}$。

本题认为 $0^k$ 在 $k\le 0$ 时无意义，即任意含 $0^k(k\le 0)$ 的式子都不成立。

提示：

> 费马大定理：
> 
> 当 $n\ge 3$ 时，$x^n+y^n=z^n$ 没有正整数解。

## 输入格式

**本题有多组数据。** 输入的第一行有一个正整数 $T$。

之后 $T$ 行，每组有四个整数 $n,a,b,c$，表示一个要判断的式子。

## 输出格式

对于每组数据输出一行一个字符串。若存在解则输出 `Y`，否则输出 `N`。

## 样例 #1

### 样例输入 #1

```
4
2 3 -4 5
3 1 -3 4
-1 3 6 2
-3 -1 0 -1
```

### 样例输出 #1

```
Y
N
Y
N
```
