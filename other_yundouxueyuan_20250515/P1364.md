# Description

最多30000x30000的黑白棋盘，不超过1000个连通分量。 每个连通分量的大小不超过1000。 分别统计各种大小连通分量的数量。

## Input

第一行给出棋盘大小. 下面有n行，用来描述每一行有哪些格子是黑色的.

## Output

输出连通块的大小，及有多少块

# Samples

```input1
12
2-4,7,9;
1,4,11-12;
1,4,10,12;
1,4-8,10-12;
1,8;
1,3-6,8,10-12;
1,3,5-6,8,11;
1,8,10-12;
1-8;
;
2;
2-4,7-10,12;
```

```output1
29 1
7 3
4 2
1 3
```
