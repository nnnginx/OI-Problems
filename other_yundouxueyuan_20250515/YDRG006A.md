## 题目描述

对于非负整数对 $(x,y)$，定义一次操作为将其变成 $(x,x\bmod y)$ 或 $(y,y\bmod x)$。注意，如果 $y=0$，则不能将其变成 $(x,x\bmod 0)$，同理 $x=0$ 时不能将其变成 $(y,y\bmod 0)$。当 $x,y$ 同时等于 $0$ 时则无法操作。

现在 SA 酱有一个数对 $(x,y)$，她邀请 Alice 和 Bob 在这个数对上做游戏。具体而言，Alice 先手，两个人轮流，轮到谁谁就必须对这个数对进行一次操作，不能操作的人则输掉了游戏。

SA 酱想知道，假如 Alice 和 Bob 足够聪明，那么谁将取得游戏的胜利。

## 输入格式

第一行一个整数 $T$ 表示数据组数。

后面 $T$ 行每行两个非负整数表示数对 $(x,y)$。

## 输出格式

每行一个字符串 `Alice` 或 `Bob`，表示将赢得游戏的人。

## 样例

### 样例 1 输入

```
3
1 1
0 0
1 0
```

### 样例 1 输出
```
Bob
Bob
Alice
```

### 样例 1 解释

$(x,y)=(1,1)$ 时，以下是一种可能的游戏进程：Alice 将 $(1,1)$ 变成 $(1,1\bmod 1)=(1,0)$，然后 Bob 只能将 $(1,0)$ 变成 $(0,0\bmod 1)=(0,0)$，然后 Alice 将无法操作，Bob 取得胜利。

$(x,y)=(0,0)$ 时，Alice 直接无法操作，Bob 直接取得胜利。

$(x,y)=(1,0)$ 时，以下是一种可能的游戏进程：Alice 将 $(1,0)$ 变成 $(0,0\bmod 1)=(0,0)$，然后 Bob 将无法操作，Alice 取得胜利。

## 数据范围

对于所有测试点，满足 $0\le x,y\le 10^9$，$1\le T\le 10$。

**Subtask1** $x=y$，30 pts

**Subtask2** $0\le x,y\le 100$，30pts

**Subtask3** 无特殊限制，40pts
