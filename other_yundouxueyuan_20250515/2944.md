# Description


给定一个M*N的系数矩阵A和一个M*N的初始矩阵B，我们定义矩阵序列SB={B ~k~ }，满足序列第一项B~1~等于B，对于其他项则满足下式：

![](https://darkbzoj.cc/JudgeOnline/upload/201204/111.jpg)

其中上式b的右上角的数表示上标，即其在矩阵序列中的位置。

       你需要回答Q次询问，每次询问矩阵序列中第k项的第i行第j列的数mod 1000000007。

       提示，1000000007是一个质数

**数据范围：**

       M,N<=20,Q<=1000

# Format

## Input

The input consists of a single test case and is given in the following format:

| *m*         | *n*         | *t*     |              |
| -------------- | -------------- | ---------- | -------------- |
| *a*~11~    | *a*~12~    |    ⋯    | *a*~1*n*~ |
| *a*~21~    | *a*~22~    |    ⋯    | *a*~2*n*~ |
|      ⋮      |      ⋮      |    ⋱    |      ⋮      |
| *a~m~*~1~ | *a~m~*~2~ |    ⋯    | *a~mn~*    |
| *b*~11~    | *b*~12~    |    ⋯    | *b*~1*n*~ |
| *b*~21~    | *b*~22~    |    ⋯    | *b*~2*n*~ |
|      ⋮      |      ⋮      |    ⋱    |      ⋮      |
| *b~m~*~1~ | *b~m~*~2~ |    ⋯    | *b~mn~*    |
| *i*~1~     | *j*~1~     | *k*~1~ |              |
| *i*~2~     | *j*~2~     | *k*~2~ |              |
| ⋮           | ⋮           | ⋮       |              |
| *i~t~*     | *j~t~*     | *k~t~* |              |

Bounds on the values are: 1 ≤  *m* , *n* ≤ 20; 1 ≤ *t* ≤ 1000; 0 ≤  *a~ij~* , *b~ij~* ≤ 10; 1 ≤ *i~t~* ≤  *m* ; 1 ≤ *j~t~* ≤  *n* ; 1 ≤ *k~t~* ≤ 10 ^9^ .

## Output

For each  *t* , output *b~i~t~j~t~~^k~t~^* mod 1,000,000,007.

# Samples

```input1
2 2 5
1 2
2 1
1 1
1 1

1 1 3
```

```output1
1
2
2
9
1
```
