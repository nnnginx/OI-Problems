# Description

平面坐标系xOy内，给定n个顶点V = (x , y)。对于顶点u、v，u与v之间的距离d定义为|xu C xv| + |yu C yv|
你的任务就是求出这n个顶点的最小生成树。

# Format

## Input

第一行一个正整数n，表示定点个数。
接下来n行每行两个正整数x、y，描述一个顶点。

## Output

只有一行，为最小生成树的边的距离和。

# Samples

```input1
4
1 0
0 1
0 -1
-1 0
```

```output1
6
```

# Limitation

对于100%的数据n <= 50000；

0 <= x, y <= 100000。
