## 题目背景

**以下是经典数独的规则。如果你会数独可以跳过。**

数独是一款老少皆宜的益智类游戏，分为经典数独和异形数独等，经典数独在一个 $9$ 行 $9$ 列的方格纸内。

经典数独中，我们称第 $a$ 行第 $b$ 列的格子和第 $c$ 行第 $d$ 列的格子**在同一宫**，当且仅当 $\lfloor \dfrac{a-1}{3}\rfloor=\lfloor\dfrac{c-1}{3}\rfloor$ 且 $\lfloor \dfrac{b-1}{3}\rfloor=\lfloor\dfrac{d-1}{3}\rfloor$。从直观上看，在同一宫的格子占据 $3\times 3$ 的一个小方块。

经典数独中，谜题会给定一些格子内的数字（$1\sim 9$ 的整数），玩家需要在其他格内填入数字（$1\sim 9$ 的整数），使得每一行、每一列的数字不重复，且在同一宫的格子数字也不重复。

## 题目描述

你有 $T$ 个标准九宫数独，每个数独内有两个**不同的**数字已经填好，请找出一个填法。

## 输入格式

输入数据第一行有一个正整数 $T$，表示要填的九宫数独个数。

之后 $T$ 行，每行六个一位正整数 $x_1,y_1,v_1,x_2,y_2,v_2$，表示第 $x_1$ 行 $y_1$ 列填了 $v_1$，并且第 $x_2$ 行第 $y_2$ 列填了 $v_2$。

## 输出格式

对于每个数独，输出 $9$ 行，每行 $9$ 个**不用空格隔开的**一位数，表示你填的答案。

## 样例 #1

### 样例输入 #1

```
1
1 4 2 4 1 3
```

### 样例输出 #1

```
534267891
126589347
789134526
315426789
462798153
897315462
251643978
648971235
973852614
```

## 提示

【样例解释】

要求第 $1$ 行第 $4$ 列是 $2$，第 $4$ 行第 $1$ 列是 $3$。我们看到输出的数独确实符合题意，并且每行、每列、每个宫数字都不重复。**答案不唯一。**

【数据范围】

本题共 $50$ 分。

| Subtask |  分值  | $T\le$ |         特殊性质         |
| :-----: | :----: | :------: | :----------------------: |
|  $1$  | $14$ |  $10$  |                          |
|  $2$  | $17$ | $10^5$ | 两个格子同行，同列或同宫 |
|  $3$  | $19$ | $10^5$ |                          |

对于全体数据，保证 $1\le T\le 10^5$，$(x_1,y_1)$ 和 $(x_2,y_2)$ 是不同的格子，且 $v_1\ne v_2$。
