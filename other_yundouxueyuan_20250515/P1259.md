# Description

给一个a*b矩形，由a*b个单位正方形组成。你需要沿着网格线把它分成分空的两部分，每部分所有格子连通，且至

少有一个格子在原矩形的边界上。“连通”是指任两个格子都可以通过水平或者竖直路径连在一起。 求方案总数

。例如3*2的矩形有15种方案。

![](https://darkbzoj.cc/JudgeOnline/images/1259.jpg)

## Input

输入仅一行，为两个整数a，b。

1<=a<=6, 2<=b<=7

## Output

输出仅一行，即方案总数。

# Samples

```input1
3 2
```

```output1
15
```
