# Description

已知一个无向图，现在有若干个询问请你回答：

```
1．A B G1 G2求从A到Ｂ是否能不通过G1-G2这条已经存在的边

   2．A B C求从A到B能否不通过C这个点
```

# Format

## Input

N M表示点数和边数

```
下接M行，每行两个数表示一条边

   接下来一个数Q表示问题总数

   下面Q行每行开头一个数k表示询问的种类，如果是1就是第一种询问，后面接四个数表示A，B，G1，G2；如果是2就是第二种询问，后面接三个数A,B,C。
```

## Output

对于每一个询问如果答案是是则输出一行yes否则输出一行no
