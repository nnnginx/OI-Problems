# Description

一开始森林里面有N只互不相识的小猴子，它们经常打架，但打架的双方都必须不是好朋友。每次打完架后，打架

的双方以及它们的好朋友就会互相认识，成为好朋友。经过N-1次打架之后，整个森林的小猴都会成为好朋友。 现

在的问题是，总共有多少种不同的打架过程。 比如当N=3时，就有{1-2,1-3}{1-2,2-3}{1-3,1-2}{1-3,2-3}{2-3,1

-2}{2-3,1-3}六种不同的打架过程。

## Input

一个整数N,N<=10^6

## Output

一行，方案数mod 9999991。

# Samples

```input1
4
```

```output1
96
```
