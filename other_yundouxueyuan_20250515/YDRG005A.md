**本题满分 $30$ 分，每个测试点 $5$ 分。**

## 题目描述

你有一个 $m$ 行 $n$ 列的方格纸，第 $i$ 行第 $j$ 列填有一个自然数 $a_{i,j}$。你可以从任意一格出发，每次沿着一个方向（上下左右之一）走一步**并到达一个你没走过的格子**，并在任意时刻结束移动。

你的分数为走过的所有格子填有的数字和。你需要最大化你的分数，并给出你的路径。

## 输入格式

输入的第一行有两个正整数 $m,n$，表示方格纸的行数和列数。

之后 $m$ 行，每行 $n$ 个自然数，其中第 $i$ 行第 $j$ 列的数为 $a_{i,j}$。

## 输出格式

输出的第一行有三个自然数 $sum,x_1,y_1$，表示分数和你的起点。

第二行输出一个字符串表示从起点开始你每次移动的方向，向上、向下、向左、向右分别用 `u,d,l,r` 表示。其他字母将会被忽略。

本题采用 Special Judge 评测，具体地：

- 若 $sum$ 错误，或者输出格式错误，则该测试点得 $0$ 分。
- 若 $sum$ 正确，但路径不合法（有重复经过的格子，或跑到外面，或分数不等于 $sum$），则该测试点得 $2$ 分。
- 若 $sum$ 和路径均正确，则该测试点得 $5$ 分。

如果你只知道 $sum$，也请输出 $sum$ 后输出两个数字和一个字符串，否则可能导致未知错误。

输出的路径不宜过长（所以无关字符不要太多），若字符串长度超过 $10^6$，则不对评测结果进行保证。

**特别注意：如果你的方案只经过一个格子（而不移动），也请随便输出一个 `u,d,l,r` 以外的字母而不是空行。**

## 样例 #1

### 样例输入 #1

```
4 4
1 2 3 0
2 0 2 4
1 9 1 9
8 1 0 0
```

### 样例输出 #1

```
43 3 3
lluurreedrddlll
```

| 测试点编号 | $m,n\le$ |           特殊性质           |
