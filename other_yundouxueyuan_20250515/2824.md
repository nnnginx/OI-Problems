# Format

## Input

共N+1行。

第一行包含一个正整数N，表示操作的数目。

接下来有N行，每行包含两个个正整数T,V和一个小写英文字母C，表示一次操作，数据保证所有的操作都合法。

## Output

共N行，依次表示每次操作后满足条件的三元组个数。

# Samples

```input1
5
1 1 a
2 1 a
1 2 b
2 1 b
2 3 a
```

```output1
1
3
3
4
7
```
