# Description

You are to write a program, which, given *N* positive integer values $X_1$ , $X_2$ , бн, $X_N$, compute the number of distinct values in those *N* integer values.

# Format

## Input

Your program reads input from standard input. The first line contains one integer: the number  *N* , 2 б▄  N  б▄ 1000. The following *N* lines represent the values $X_1$ , $X_2$ , бн, $X_N$ , and each of these lines contains one integer: a value $X_i$ , 1 б▄ $X_i$ б▄ 10000.

## Output

Your program writes output to standard output. The output contains one integer: the number of distinct values in $X_1$ , $X_2$ , бн, $X_N$.

# Samples

```input1
10
1
2
3
4
5
4
3
2
1
2
```

```output1
5
```

```input2
5
123

123
```

```output2
3
```

# Limitation

If the output is correct, you get full score for the test case. Otherwise the score for the test case is 0.
