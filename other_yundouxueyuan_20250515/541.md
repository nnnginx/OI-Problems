# Format

## Input

第一行四个正整数N，M，p，q

## Output

一共输出N行，第i行表示第i个馒头的最终颜色（如果最终颜色是白色就输出0）。

# Samples

```input1
4 3 2 4
```

```output1
2
2
3
0
```
