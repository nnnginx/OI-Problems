# Background

Special for beginners, ^_^

# Description

一次考试共有n个人参加，第i个人说：“有ai个人分数比我高，bi个人分数比我低。”问最少有几个人没有说真话(可能有相同的分数)

# Format

## Input

第一行一个整数n，接下来n行每行两个整数，第i+1行的两个整数分别代表ai、bi

## Output

一个整数，表示最少有几个人说谎

# Samples

```input1
3

2 0

0 2

2 2
```

```output1
1
```

# Limitation

1s, 1024KiB for each test case.
