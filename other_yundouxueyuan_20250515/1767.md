# Description

有一位售票员给乘客售票，对于每位乘客，他会卖出多张连续的票，直到已卖出的编号的所有位置上的数的和不小于给定的正数k。然后他会按照相同的规则给下一位乘客售票。初始时，售票员持有的编号是从L到R的连续整数。请你求出，售票员可以售票给多少位乘客。

# Format

## Input

三个整数L,R,k。

## Output

一个正整数，表示能够拿到票的乘客数。

# Samples

```input1
40 218 57
```

```output1
29
```

# Limitation

数据范围：

```
1<=L<=R<=10^18，1<=k<=1000。
```
