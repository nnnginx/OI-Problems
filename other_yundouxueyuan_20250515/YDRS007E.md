### 题目描述

给定正整数 $n$，求：

$$
\sum_{i=1}^n\sum_{j=1}^n\sum_{k=1}^n[i^j=j^k]
$$

的值。答案对 $998244353$ 取模。

### 输入格式

一行一个正整数 $n$。

### 输出格式

一行，表示欲求式子的值。对 $998244353$ 取模。

### 样例一输入

```
10
```

### 样例一输出

```
24
```

### 样例二输入

```
200
```

### 样例二输出

```
439
```

### 样例三输入

```
300000000000
```

### 样例三输出

```
55982986
```

### 测试点约束

**本题采用捆绑测试。**

- Subtask 1 (10pts)：$n\le 10$。
- Subtask 2 (15pts)：$n\le 10^3$。
- Subtask 3 (25pts)：$n\le 10^6$。
- Subtask 4 (30pts)：$n\le 10^{12}$。
- Subtask 5 (20pts)：无特殊限制。

对于全部数据，保证 $1\le n\le 10^{18}$。
