## 题目描述

现有一个供水系统，由 $n$ 行 $m$ 列共 $nm$ 个调配单元组成。顾名思义，调配单元可以收集到达该单元的水并控制从每个水管输出的水量。

第 $i$ 行 $j$ 列的调配单元记作 $f_{i,j}$，则 $f_{i,j}$ 可以直接通过水管到达 $f_{i+1,j}$ 和 $f_{i,j+1}$。**水管是单向的。**

调配单元相当精密，$f_{i,j}$ 在调配 $a_i+b_j$ 吨水后会迅速报废，经过这个单元的水会全部喷到旁边的草地里，其中数列 $a,b$ 是给定的参数。

由于该系统十分重要，小I想问你，最多有多少从左上角（$f_{1,1}$）进入调配系统的水能进入右下角（$f_{n,m}$）？

## 输入格式

第一行，两个整数 $n,m$ 表示供水系统的行数和列数。

第二行，$n$ 个整数 $a_{1\ldots n}$，含义同描述。

第三行，$m$ 个整数 $a_{1\ldots m}$，含义同描述。

## 输出格式

一行一个整数，表示最大通过量。单位为吨。

## 样例 #1

### 样例输入 #1

```
4 3
19 2 3 10
20 1 50
```

### 样例输出 #1

```
38
```
