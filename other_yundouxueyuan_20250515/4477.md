
### 题目描述

给定一个长为 $n$ 的序列，仅由 $0$ 和 $1$ 构成。

可以删除**一段**仅由 $1$ 或者 $0$ 构成的区间，可以不删，求剩下的 **连续的 $0$ 或 $1$ 中**，最长的区间长度是多少。

### 输入格式

第一行一个数  $n$ , 表示序列的长度。

第二行一个长为 $n$ 的 $01$ 串表示这个序列。

### 输出格式

一行一个整数，表示答案。

```input1
5
00110
```

```output1
3
```

### 数据范围

对于 $40\%$ 的数据，满足 $1 \leq n \leq 100$。

对于 $60\%$ 的数据，满足 $1\le n\le 4\times 10^4$。

对于 $100\%$ 的数据，满足 $1 \leq n \leq 6\times 10^5$。
