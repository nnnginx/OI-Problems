# 牛肉丸

## 题目描述

奶龙和火锅店老板进行了一局牛肉丸比赛，规则类似于乒乓球, 有人比分达到或大于 $11$ 分并且双方比分出现至少 $2$ 分之差，结束本局比赛。

因为奶龙的肚肚比牛肉丸弹, 所以它赢下了这局比赛，这局比赛最终比分为 $x:y$ ，其中 $x \ge 11$ 并且 $x \gt y+1$。

但是小七忘记了具体每回合谁得分，他想知道有多少种不同的情况。

两种情况不同当且仅当存在第 $i ( 1 \le i \le x+y )$ 回合得分的人不同。

## 输入格式

输入包括两个整数 $x,y$ 满足 $x \ge 11$ 并且 $x \gt y+1$

表示这局比赛最终得分是 $x:y$

## 输出格式

输出包含一个整数，表示情况数模 $998244353$ 意义下的结果。

## 样例 #1

### 样例输入 #1

```
11 1
```

### 样例输出 #1

```
11
```

## 样例 #2

### 样例输入 #2

```
11 2
```

### 样例输出 #2

```
66
```

## 样例 #3

### 样例输入 #3

```
14 12
```

### 样例输出 #3

```
739024
```

## 样例 #4

### 样例输入 #4

```
100 98
```

### 样例输出 #4

```
611686541
```

## 提示

对于第一个样例, 老板可能在第 $[1,11]$ 回合时得分，但是不可能在最后一个回合得分，因为这样在前 $11$ 个回合中奶龙就取得了 $11$ 分结束了比赛。

对于 $10\%$ 的数据，$y=0$ ，即老板没有得分

对于另外 $20\%$ 的数据， $x=11$

对于另外 $20\%$ 的数据， $x,y \le 16$

对于另外 $20\%$ 的数据， $x,y \le 100$

对于 $100\%$ 的数据, $1 \le x,y \le 10^9$ ，并且 $x:y$ 是合理的奶龙获胜的对局结束的比分。
