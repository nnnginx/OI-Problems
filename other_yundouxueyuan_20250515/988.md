# Description

一个n*n的棋盘。要对它进行染色。最初每个格子都是颜色1。m次操作，每次
可以将矩形(x0,y0)-(x1,y1)进行棋盘式染色（即将此范围内所有x+y的奇偶性与
x0+y0的奇偶性相同的格子染为某种颜色）；或者将当前的棋盘存档；或者将当
前棋盘重新载入为之前某次存档。你需要输出这m次操作后的棋盘。

# Limitation

n <= 1000, m <= 100,000.
