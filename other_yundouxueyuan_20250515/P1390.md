

# Description

在一个大小为1000*1000的区域中，有n个固定点,m棵tree 。 现在你要建一个围栏来保护tree,建它的费用为你选用的固定点的个数 *20和 你没有圈进围栏的tree*111. 现在希望这个值越小越好. 3<=N<=100. 1<=M<=100

## Input

第一行给出n,m 下面开始n行，给出固定的坐标 下面开始m行，给出tree的坐标

## Output

输出最小费用

# Samples

```input1
4 3
800 300
200 200
200 700
600 700
400 300
600 500
800 900
```

```output1
171
```
