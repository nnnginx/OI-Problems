# Background

Special for beginners, ^_^

# Description

![image](./491/file/mqnj2yMS1amDUPwTayq3f.jpeg) Given two integers x and y, print the sum.

# Format

## Input

![image](./491/file/pUM-VkfEhUCRPrdlzaQ1s.jpeg) Two integers x and y, satisfying $0\leq x,y\leq 32767$ .

## Output

![image](./491/file/BvKeCNNMoNFJQb-zBbyvx.jpeg) One integer, the sum of x and y.

# Samples

```input1
10 714066300
122043478 641033841
```

```output1

```
