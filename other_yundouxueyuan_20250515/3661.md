# Description

某Xm被虐爆太多于是去拜访神模有N个神纳活在一维世界，住在数轴上某点，每个神亩加懈龌ゲ幌嗤的坐标值Xi，神挠猩窳Γ可以在一维世界任意移动，但是Xm不行，Xm一开始在某个神拇Γ神慕枇怂一点神力用，Xm可以借助神力向左（数轴负方向）移动任意距离L次，向右（数轴正方向）移动任意距离N-L-1次，移动耗时为两点之间的距离差的绝对值。他移动完毕后到达一个神牡奈恢檬本褪幼魉拜访了这个神模当他成功拜访完N个神暮笏就可以从Xm进化成弱菜。他希望快点成为弱菜，所以希望你能告诉他他最快需要多少时间成为弱菜？由于今天是最后一天就不需要给出方案了。

# Format

## Input

第一行3个数字N,L,S表示神氖，Xm可以向左移动的次数和Xm所在位置的神谋嗪拧

第二行N个升序排列的数字，第i个数字表示第i个神乃在的坐标大小X~i~（0=X~1~＜X~2~＜…＜X~n-1~＜X~n~≤10^9^）

## Output

一个数，Xm成为弱菜的最短耗时。如果Xm无法成为弱菜是天命就输出’-1’(不带引号)

# Samples

```input1
5 2 2
0 10 11 21 22
```

```output1
33
```

# Limitation

**【样例解释】**

Xm可以向左移动2次，向右移动2次。现在在2号神拇ψ标为10，他先向左移动一次，访问1号神模移动耗时10，然后向右移动访问3号神模移动耗时11，向右移动访问5号神模移动耗时11，然后向左移动访问4号神模移动耗时1，总耗时33，总公共向左移动2次，向右移动2次。

 

**【数据范围】**

    100%的数据   1≤N≤100000
