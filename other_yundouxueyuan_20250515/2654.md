# Format

## Input

输入文件最多包含25组测试数据，每个数据仅包含一行，有一个整数 *p* （1<= *p* <=10 ^9^ ），表示战场的图形周长。 *p* =0表示输入结束，你的程序不应当处理这一行。

## Output

对于每组数据，输出仅一行，即满足条件的战场总数除以987654321的余数。

# Samples

```input1
7
8
9
10
0
```

```output1
0
2
0
9
```
