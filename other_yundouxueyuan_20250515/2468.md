# Description

你有一个n*m的棋盘。初始状态下，每个格子或者为空，或者放着一颗棋子。
每次你可以选择两个边相邻的格子，如果它们的棋子数相同，则可以把其中一格的棋子全部转移到另一格中。
求每一格中最多可能叠放的棋子数量。

# Format

## Input

第一行两个整数n,m(1<=n,m<=200)。
接下来n行，每行是一个长度为m的01串，0表示空，1表示有棋子。

## Output

输出n行，每行是空格隔开的m个整数，表示这一格内最多含有的棋子数目。

# Samples

```input1
3 4
0111
1011
1011
```

```output1
0 2 4 4
2 0 4 4
2 0 4 4
```
