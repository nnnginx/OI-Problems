# Format

## Input

第一行两个数N,M 接下来N行,每行两个实数,分别表示宽度和高度再接下M行,每行两个实数,表示Wi,Vi

## Output

第一行输出通过左边流出浪费的水的体积第二行输出通过右边流出浪费的水的体积保留三位小数 N,M< = 100000
