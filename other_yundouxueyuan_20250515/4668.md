# Description

这是一道模板题。

给定 $n$ 个字符串，试求出这些字符串的最长公共子串。

## Input

第一行一个整数 $n$。

下面第 $2$ 到 $n+1$ 行，每行一个字符串。

## Output

仅一行，包含一个正整数，表示 $n$ 个字符串的最长公共子串长度。

# Samples

```input1
2
ababc
cbaab
```

```output1
2
```

# Limitation

对于第 $i$ 个测试点，保证 $n\,=\,i+1$。

对于每一个字符串，保证 $|str|\,\le\,10^{\lceil \frac{i}{3}\rceil}$，出现字符均为小写英文字母。
