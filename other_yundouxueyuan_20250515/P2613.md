## 题目描述
给出一个有理数 $c=\frac{a}{b}$，求 $c \bmod 19260817$ 的值。

这个值被定义为 $bx\equiv a\pmod{19260817}$ 的解。

## 输入格式
一共两行。

第一行，一个整数 $a$。  
第二行，一个整数 $b$。  

## 输出格式
一个整数，代表求余后的结果。如果无解，输出 `Angry!`。

```input1
233
666

```

```output1
18595654

```

## 提示
对于所有数据，保证 $0\leq a \leq 10^{10001}$，$1 \leq b \leq 10^{10001}$，且 $a, b$ 不同时是 $19260817$ 的倍数。
