# Background

Special for beginners, ^_^

# Description

三维凸包问题是一个基础的三维计算几何问题，不过这次你只需要做一个简单版的三维凸包问题就行了。

# Format

## Input

输入数据一共有若干行，每行三个整数，表示一个点的坐标。点的个数为五个。

## Output

输出一个实数，保留两位小数，表示三维凸包的体积。

# Samples

```input1
0 0 0
2 0 0
0 2 0
2 2 0
1 1 1
```

```output1
1.33
```

# Limitation

对于所有数据，坐标范围在[0,100]*[0,100]*[0,100]。
