# Description

![](https://darkbzoj.cc/JudgeOnline/upload/201204/fa(1).jpg)

对于任意的>1的n gcd(a, b)不是n^2的倍数
也就是说gcd(a, b)没有一个因子的次数>=2

# Format

## Input

一个正整数T表示数据组数

接下来T行 每行两个正整数 表示N、M

## Output

T行 每行一个整数 表示第i组数据的结果

# Samples

```input1
4
2 4
3 3
6 5
8 3
```

```output1
24
28
233
178
```

# Limitation

HINT

T <= 10000

N, M<=4000000
