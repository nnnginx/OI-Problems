# Description

在XYZ的dzy loves math6问世后，dzy一直觉得这道题答案太大，一点都不优美，于是他随手在外面套上一个μ。同时，他又觉得输入两个数实在太麻烦，于是题目变成了![image](./2484/file/UiIrXsTmD066VOShWimP4.jpeg) ，你能解决这个问题吗？

# Format

## Input

第一行一个整数T表示询问组数，接下来T行每行一个整数n。

## Output

对于每一个询问输出一行表示答案

# Samples

```input1
1
2
```

```output1
0
```

# Limitation

T<=10^3 n<=10^7
