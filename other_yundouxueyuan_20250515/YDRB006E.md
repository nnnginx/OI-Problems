# 迈克斯

## 题目描述

奶龙和暴暴龙各有一个**排列** $a$, $b$，他们想知道，有多少对 $1 \leq l \leq r \leq n$, 使得 $[a_l,a_r]$, $[b_l,b_r]$ 的 $\operatorname {MEX}$ 相同。

这道题目中，一段区间的 $\operatorname{MEX}$ 表示 **没有在该区间中出现过的最小的正整数**。比如，$ \operatorname{MEX}([1]) = 2 $，$ \operatorname{MEX}([2]) = 1 $，$ \operatorname{MEX}([1,1,4,5,1,4]) = 2 $。

## 输入格式

第一行，一个整数 $n$；

第二行，$n$ 个整数表示 $a_i$；

第三行，$n$ 个整数表示 $b_i$；

## 输出格式

一行一个整数，输出你的答案。

## 样例 #1

### 样例输入 #1

```
3
1 3 2
2 1 3
```

### 样例输出 #1

```
2
```

## 样例 #2

### 样例输入 #2

```
7
7 3 6 2 1 5 4
6 7 2 5 3 1 4
```

### 样例输出 #2

```
16
```

## 样例 #3

### 样例输入 #3

```
6
1 2 3 4 5 6
6 5 4 3 2 1
```

### 样例输出 #3

```
11
```

## 提示

对于 $20\%$ 的数据，保证 $a_i = b_i$；

对于 $40\%$ 的数据，$n\leq 70$；

对于 $100\%$ 的数据，$1 \leq n \leq 2\times 10^5$。
