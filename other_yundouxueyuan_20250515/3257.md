# Description

o定一觯金字塔，密室的大小Y料，有让恳方格的地形高度， 一程式⒔鹱炙放置到鲋校以及⒚苁曳胖渺督鹱炙仁沟媒鹱炙 的基底高度樗有可能性中的最高。

# Format

## Input

第一行: 包含6空白隔_的整担分e是: m, n, a, b, c ,and d. 接下淼 n 行: 每一行包含m空白隔_的整担代表一列中方格的地形高度。 第一行代表最上方的一列方格(row 1)，最後一行代表最底的一列(row n)。 m整荡表每一列的方格地形高度，牡谝恍虚_始

## Output

第一行: 必包含空白隔_的整当硎窘鹱炙基底的左上角座 第一整樾(column)，第二整榱(row)。 第二行: 必包含空白隔_的整荡表密室的左上角座 第一整樾(column)，第二整榱(row)。

8 5 5 3 2 1

限制l件（CONSTRAINTS）

3 < = a < = m
3 < = b < = n

所有的地形高度都是介於1到100的整
