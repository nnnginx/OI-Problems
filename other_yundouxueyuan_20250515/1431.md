# Description

n 个沙茶，被编号 1~n。排完队之后，每个沙茶希望，自己的相邻的两

人只要无一个人的编号和自己的编号相差为 1（+1 或-1）就行；

现在想知道，存在多少方案满足沙茶们如此不苛刻的条件。

# Format

## Input

只有一行且为用空格隔开的一个正整数 N，其中 100%的数据满足 1≤N ≤ 1000;

## Output

一个非负整数，表示方案数对 7777777 取模。

# Samples

```input1
4
```

```output1
2
```

# Limitation

样例解释：有两种方案 2 4 1 3 和 3 1 4 2
