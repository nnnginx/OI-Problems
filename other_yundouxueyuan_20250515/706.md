# Format

## Input

输入一共15行，包含一个新数独的实例。第奇数行包含左右方向的符号（<和>），第偶数行包含上下方向的符号（^和v）。

## Output

输出包含9行，每行9个1~9的数字，以单个空格隔开。输入保证解惟一。
