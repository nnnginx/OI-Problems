# Format

## Input

一行，只包含一个a,b两种字符的字符串。


## Output

一行，一个非负整数表示问题的答案。

# Samples

```input1
样例一
abaaba
样例二
aaabbbaaa
样例三
ааaааaaaa
```

```output1
样例一
14
样例二
44
样例三
53
```
