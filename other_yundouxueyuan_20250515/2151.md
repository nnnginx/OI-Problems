# Format

## Input

其中第一行是一个整数n(1<=n<=18)，随后4n2行，依次为三棱锥四个面上所填的数字。

![](https://darkbzoj.cc/JudgeOnline/upload/201112/2(4).jpg)

## Output

其中仅包含一个整数，表示最大的排序二叉树所含的节点数目。

![](https://darkbzoj.cc/JudgeOnline/upload/201112/3(2).jpg)
