# Background

Special for beginners, ^_^

# Description

给出n个三角形，求它们并的面积。

# Format

## Input

第一行为n(N < = 100)， 即三角形的个数 以下n行，每行6个整数x1, y1, x2, y2, x3, y3，代表三角形的顶点坐标。坐标均为不超过10 ^ 6的实数，输入数据保留1位小数

## Output

输出并的面积u, 保留两位小数

# Samples

```input1
2
0.0 0.0 2.0 0.0 1.0 1.0
1.0 0.0 3.0 0.0 2.0 1.0
```

```output1
1.75
```

# Limitation

1s, 1024KiB for each test case.
