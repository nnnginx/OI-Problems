# Description

约翰有N(1≤N≤50000)头牛，约翰的草地可以认为是一条直线．每只牛只喜欢在某个特定的范围内吃草．第i头牛喜欢在区间(Si，Ei)吃草，1≤Si<Ei≤1,000,000,00.
奶牛们都很自私，他们不喜欢和其他奶牛共享自己喜欢吃草的领域，因此约翰要保证任意两头牛都不会共享他们喜欢吃草昀领域．如果奶牛i和奶牛J想要同时吃草，那么要满足：Si>=Ej或者Ei≤Sj．约翰想知道在同一时刻，最多可以有多少头奶牛同时吃草？


# Format

## Input

第1行：一个整数N．
第2到N+1行：第i+l行有两个整数Si，Ei．

## Output

一个整数，最多可以有多少头牛同时吃草．

# Samples

```input1
5
2 4
1 12
4 5
7 10
7 8
```

```output1
3
```

# Limitation

第1，3，4共3只奶牛可以同时吃草，第1，3，5也可以．
