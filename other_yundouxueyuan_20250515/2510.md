# Format

## Input

第一行一个整数T，表示数据组数。

以下每一行两个数L、R（保证区间合法），代表询问[L, R]。

## Output

输出T行，每行一个数，表示在这个区间内小D喜欢的数出现了多少次。

你的输出当且仅当和标准输出一样才能得该测试点满分。

# Samples

```input1
3

1 5

3 9

8 8
```

```output1
2

2

0
```

# Limitation

L,R<=10^18,T<=20
