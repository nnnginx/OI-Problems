

# Description

给定一些点的坐标，要求求能够覆盖所有点的最小面积的矩形，

输出所求矩形的面积和四个顶点坐标



## Input

第一行为一个整数n（3<=n<=50000）

从第2至第n+1行每行有两个浮点数，表示一个顶点的x和y坐标，不用科学计数法

## Output

第一行为一个浮点数，表示所求矩形的面积（精确到小数点后5位），

接下来4行每行表示一个顶点坐标，要求第一行为y坐标最小的顶点，

其后按逆时针输出顶点坐标.如果用相同y坐标，先输出最小x坐标的顶点

# Samples

```input1
6 1.0 3.00000
1 4.00000
2.0000 1
3 0.0000
3.00000 6
6.0 3.0
```

```output1
18.00000
3.00000 0.00000
6.00000 3.00000
3.00000 6.00000
0.00000 3.00000
```
