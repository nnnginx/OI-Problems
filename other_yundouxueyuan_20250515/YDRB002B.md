## 题目描述

一片土地上有一圈闭合围墙。现在用一个字符矩阵描述这片土地的情况，空地用 `.` 表示，围墙用 `#` 表示

求围墙所围出的区域面积大小 **（包括墙本身）**

**保证围墙围成的是一块凸面积**

## 输入格式

输入共 $n+1$ 行

第 $1$ 行包含两个整数 $n,m$，表示土地的行数和列数

接下来的 $n$ 行，每行是长为 $m$ 的字符串，此字符矩阵描述了土地的情况

空地用 `.` 表示，围墙用 `#` 表示。

## 输出格式

一行一个整数，表示围墙所围出的区域面积大小 **（包括墙本身）**

## 样例数据

**样例一**

*input*

```
4 6
..##..
.#..#.
.#..#.
..##..
```

*output*

```
12
```

**样例二**

*input*

```
15 15
..........#....
.........#.#...
........#..#...
.......#....#..
......#.....#..
......#.....#..
.....#......#..
.....#......#..
......#...##...
.......#..#....
........#.#....
.........#.....
...............
...............
...............
```

*output*

```
58
```

## 数据范围与约定

保证围墙连成唯一一个闭合凸面积，非图形边界处无多余围墙

对于 $20\%$ 的数据，$n,m\le 10$

对于 $100\%$ 的数据，$3\le n, m\leq 200$
