# Format

## Input

第一行两个整数N，M表示待检查的作文数量，和小强的标准作文库
的行数
接下来M行的01串，表示标准作文库
接下来N行的01串，表示N篇作文

## Output

N行，每行一个整数，表示这篇作文的Lo 值。

# Samples

```input1
1 2
10110
000001110
1011001100
```

```output1
4
```

# Limitation

输入文件不超过1100000字节
注意：题目有改动,可识别的长度不小于90%即可，而不是大于90%
