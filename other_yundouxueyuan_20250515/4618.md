# Description

这是一道模板题。

给定正整数 $ n $ 与 $ p $，求 $ 1 \sim n $ 中的所有数在模 $ p $ 意义下的乘法逆元。



## Input

一行两个正整数 $ n $ 与 $ p $

## Output

$ n $ 行，第 $ i $ 行一个正整数，表示 $ i $ 在模 $ p $ 意义下的乘法逆元。

# Samples

```input1
10 13
```

```output1
1
7
9
10
8
11
2
5
3
4
```

# Limitation

$ 1 \leq n \leq 3 \times 10 ^ 6, n < p < 20000528 $
$ p $ 为质数。
