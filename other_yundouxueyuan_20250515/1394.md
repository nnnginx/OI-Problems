# Format

## Input

第一行为整数T，表示操作次数。 节点从1开始依次编号。开始仅含1号点。 接下来T行，每行先给出一个整数x和g

，表示连接点和添加的点数。 记p为添加前点数。 若g=1，则表示添加一个点p+1，并与x相连。 若g>1，则表示添

加一个环p+i (1<= i <= g)，点p+1与x相连，点p+i和p+i+1顺次相连(1<= i < g)，点p+g和x相连。 记加入的点数

为N。N <= 100000。

## Output

每次操作后，输出答案。
