# Description

设有1g、2g、3g、5g、10g、20g的砝码各若干枚（其总重<=1000），

# Format

## Input

a1  a2  a3  a4  a5  a6（表示1g砝码有a1个，2g砝码有a2个，…，20g砝码有a6个）

## Output

Total=N

（N表示用这些砝码能称出的不同重量的个数，但不包括一个砝码也不用的情况）

# Samples

```input1
1 1 0 0 0 0
```

```output1
TOTAL=3
```

# Limitation

（注：下划线表示空格）

表示可以称出1g，2g，3g三种不同的重量。
