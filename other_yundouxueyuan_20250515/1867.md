# Description

![image](./1867/file/OvBCeP0LEVhBDC1qHD9G3.jpeg) Given two integers x and y, print the sum.

# Format

## Input

![image](./1867/file/GoGJPPyWdFJKfI4JZnjBL.jpeg) Two integers x and y, satisfying $0\leq x,y\leq 32767$ .

## Output

![image](./1867/file/Iq4ZY5PDg4vvkVhbwRZTq.jpeg) One integer, the sum of x and y.

# Samples

```input1
1
1 396
512 619 633
```

```output1
526
```
