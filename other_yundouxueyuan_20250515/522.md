# Description

B 君有两个好朋友，他们叫宁宁和冉冉。有一天，冉冉遇到了一个有趣的题目：输入 b;d;n，求
![image](./522/file/4Jx5vsXl1rsN9w2WfKsNT.png)

# Format

## Input

一行三个整数 b;d;n

## Output

一行一个数表示模 7528443412579576937 之后的结果。

# Samples

```input1
1 5 9
```

```output1
76
```

# Limitation

其中 0<b^2< = d<(b+1)2< = 10^18,n< = 10^18，并且 b mod 2=1,d mod 4=1
