## 题目描述

正整数 $n$ 在 $k$ 进制下各位数字的和定义为 $s_k$。

如 $114514$ 的 $s_{1919810}=114514$。

给定 $n, x$ ，求有多少个 $2\le k\le\sqrt{n}$ 满足 $s_k \leq x$？

## 输入格式

一行两个整数 $n,x$。

## 输出格式

一行一个整数，表示满足条件的 $k$ 的个数。

## 样例1
```
30 5
```
```
3
```

|$k$|$(n)_k$|$s_k$|
|---|---|---|
|$2$|$11110$|$4$|
|$3$|$1010$|$2$|
|$4$|$132$|$6$|
|$5$|$110$|$2$|

故答案为 $3$。

## 样例2
```
1145141919810 500
```
```
248
```

## 数据范围

对于 $20\%$ 的数据，$n\le 10^6$。

对于所有数据，$n\le 10^{14},x\le 10^7$。
