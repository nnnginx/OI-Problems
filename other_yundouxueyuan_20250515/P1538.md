

# Description

给定三个非负整数a,b,c，则你的得分为a^2+b^2+c^2+7*min(a,b,c)。

现在再给定一个非负整数d，你需要把d分成3个非负整数，分别加到a,b,c中，使得你的得分最大。



## Input

第一行包含一个正整数n(3<=n<=7)，表示测试数据的组数。

接下来n行，每行4个非负整数a,b,c,d(0<=a,b,c,d<=10^9)。

## Output

输出n行，每行一个整数，即最大得分。

# Samples

```input1
3
2 1 2 1
3 2 1 0
1 3 0 1
```

```output1
26
21
18
```
