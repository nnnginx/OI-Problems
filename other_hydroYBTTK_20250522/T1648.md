## 题目描述

给定一个多项式 $(ax+by)^k$ ，请求出多项式展开后 $x^ny^m$ 项的系数。

## 输入

输入共一行，包含 $5$ 个整数，分别为 $a,b,k,n,m$ ，每两个整数之间用一个空格隔开。

## 输出

输出共 $1$ 行，包含一个整数，表示所求的系数，这个系数可能很大，输出对 $10,007$ 取模后的结果。

## 样例

```input1
1 1 3 1 2
```

```output1
3
```

## 提示

数据范围与提示

对于30% 的数据，有 $k≤10$；

对于50% 的数据，有 $a = 1,b = 1$；

对于100% 的数据，有 $0≤n,m≤k$，且 $n + m = k$，$0≤a,b≤10^6$ 。
