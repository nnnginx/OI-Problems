# Description

有 n 根火柴，第 i 根火柴的长度是 a[i]，求用这些火柴最多能够拼成多少个长方形（火柴不能被折断，1 根火柴最多只能用在一个长方形）。正方形也被认为是长方形。不一定要全部用完所有的火柴。

# Format

## Input

第一行，一个整数 n。1 ≤ n ≤ 100。

第二行，n 个整数，第 i 个整数是 a[i]，1 ≤ a[i] ≤ 100。

## Output

一个整数。

# Samples

```input1
5 
2 4 3 2 3
```

```output1
1
```

```input2
13
2 2 4 4 4 4 6 6 6 7 7 9 9
```

```output2
3
```
