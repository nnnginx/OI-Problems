**内存限制：128 MB 时间限制：1 S**

#### 题目描述

在某个神秘的星球上有一种货币，它有一种奇怪的兑换规则，你有且仅有一枚面值为 $n$ 的金币，你可以去银行进行兑换，也可以不兑换。 如果去银行兑换，兑换的规则是这样的：用面值为 $a$ 的金币去兑换可以换到 $a/2,a/3,a/4$ 这三枚硬币（如果是小数则截尾取整），你可以不兑换，也可以兑换多次！ 请问 $n$ 元，至多可以兑换出多少钱？

#### 输入

一个数 $T$ 表示该点的测试数据组数（$1 <= T <= 20$ ） 下面跟着 $T$ 行，每行一个整数 $n$（$0 <= n <= 1000000000$ ）

#### 输出

输出 $T$ 行 (一一对应)，每行一个整数表示最后最多拥有的钱数

#### 样例输入

```plain
2
12
2
```

#### 样例输出

```plain
13
2
```
