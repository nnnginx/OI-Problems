# 问题 G: 「一本通 1.1 练习 1」数列极差

**内存限制：512 MB**     **时间限制：1 S**

#### 题目描述

**佳佳的老师在黑板上写了 **N**个正整数作成的一个数列，要求佳佳进行如下操作：每一次擦去其中的两个数 **a** 和 **b**，然后在数列中加入一个数 **a×b+1**，如此下去直至黑板上剩下一个数，在所有按这种操作方式最后得到的数中，最大的为 **max**，最小的为 **min**，则该数列的极差定义为 **M=max−min**。**

**由于佳佳忙于准备期末考试，现请你帮助他，对于给定的数列，计算出相应的极差 **M**。**

#### 输入

**第一行为一个正整数 **n** 表示正整数序列的长度；**

**在接下来的 **n**行中，每行输入一个正整数。**

**接下来的一行有一个 0，表示数据结束。**

#### 输出

**输出只有一行，为相应的极差 **M**。**

#### 样例输入

```
3
1
2
3
0
```

#### 样例输出

```
2
```

#### 提示

**对于全部数据，**0≤n≤500000**，保证所有数据计算均在 **32** 位有符号整数范围内。**
