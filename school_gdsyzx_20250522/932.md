# Description

给出字符串 S，字符串的每一个字符是“1”或“2”或“3”。你要从 S 中选取一段连续字符，不妨假设这段连续的字符构成的字符串是 T，你的目标是使得 T 的长度最短，而且字符“1”、“2”、“3”在 T 中都出现过。如果无法完成目标则输出 0，否则输出 T 的最短长度。

# Format

## Input

一个字符串 S，长度不超过 200000。

## Output

一个整数。

# Samples

```input1
112233
```

```output1
4
```

# Limitation

对于 60%的数据，S 的长度不超过 100。
