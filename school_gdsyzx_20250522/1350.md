##### 【题目表述】

小C 有 $n$ 个数字，其中有 $m$ 个是 $1$，有 $n-m$ 个是 $0$。

他想把这些数字拼凑成一个 $n$ 位的二进制数，并且希望这个数对应的十进制尽可能大。Kitten 很快告诉了他，只要把 $1$ 放前面，$0$ 放后面就好了。

小C 想进一步考考 Kitten，于是他要求 Kitten 拼出来的数中不能有连续的三个 $1$。这当然难不倒 Kitten，你能拼出来吗？

##### 【输入格式】

从文件 *one.in* 中读入数据。

两个数 $n,m$。

##### 【输出格式】

输出到文件 *one.out* 中。

输出 $m$ 个 $1$、$n-m$ 个 $0$ 能拼出来的“最大的”、“不包含连续三个 $1$”的二进制数。

```input1
2 1
```

```output1
10
```

```input2
5 4
```

```output2
11011
```

```input3
10 3
```

```output3
1101000000
```

##### 【样例 4】

见选手目录下的 *one/one4.in* 与 *one/one4.ans*。

##### 【数据范围】

对于 $100\%$ 的数据，$1\le m\le n\le 100$，题目保证必然存在合法的方案。

其中：

对于 $10\%$ 的数据，保证  $m=1$。

对于 $20\%$ 的数据，保证  $n=m$。

对于 $30\%$ 的数据，保证 $n=8$。

对于 $40\%$ 的数据，没有特殊限制。
