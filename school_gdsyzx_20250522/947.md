# Description
王老师需要几个男生帮忙完成一个项目。因为这个项目相当复杂，所以男生来的越多越好，当然还是有点要求的。王老师选了一个容纳足够多男生的房间，未被选中的男生必须马上离开房间。房间里有 10000000 个男生，编号分别为 1 至 10000000。在王老师挑选完之后，还在房间里的男生他们中任意两个都应该是朋友关系（直接或间接），否则就只剩下一个男生。给出所有男生的朋友关系情况，请你帮王老师决定留下的男生的最大数量。

# Format

## Input
第一行，一个整数 n（0 ≤ n ≤ 100000），表示有 n 对朋友关系。

下面 n 行，每行两个数字 A 和 B，表示 A 和 B 是直接认识的，（A ≠ B，1 ≤ A，B ≤ 10000000）

## Output
一行，一个整数，为王老师保留的男生的最大数量。

# Samples

```input1
3
1 3
1 5
2 5
```

```output1
4
```
```input2
4
3 2
3 4
1 6
2 6
```

```output2
5
```
