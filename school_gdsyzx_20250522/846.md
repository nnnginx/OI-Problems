# Description
请统计输入的字符串中最大连续重复字符的个数，要求统计找出给定字符串中哪些字符连续出现的次数最多，并且要统计这些字符一共连续出现了多少次。

例如：字符序列为 AAA222BBAAAaa，则在这个字符序列中，字符 A 和字符 2 连续出现的次数最多，都是 3 次。

# Format

## Input
共两行，第一行为字母序列中的字母个数 n（1 ≤ n ≤ 1000），第二行为 n 个字符组成的字符串序列。

## Output
共两行，第一行为最多连续出现次数最多的字符，若有多个用空格分隔，输出顺序为他们在字符串中的先后顺序。第二行为连续出现重复的最大次数。

# Samples

```input1
13
AAA222BBAAAaa
```

```output1
A 2
3
```

# Limitation
对于 100% 的数据，1 ≤ n ≤ 1000，33 ≤ 字符数值即 ASCII 码 ≤ 126。
