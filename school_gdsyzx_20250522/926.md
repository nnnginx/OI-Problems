# Description
已知一个长度为 l 的序列：b1，b2，b3，......，bl（1 ≤ b1 ≤ b2 ≤ b3 ≤ ... ≤ bl ≤ n）。若这个序列满足每个元素是它后续元素的因子，换句话说就是对于任意的 i（2 ≤ i ≤ l）都满足 b<sub>i</sub> % b<sub>i-1</sub> = 0（其中“%”代表求余），则称这个序列是完美的。你的任务是对于给定的 n 和 l，计算出一共有多少序列是完美序列。由于答案很大，所有输出答案对 1000000007 取余后的结果。

# Format

## Input
两个正整数 n，l（1 ≤ n，l ≤ 2000），分别代表序列中元素大小的最大值和序列的长度。

## Output
一行包含一个整数，代表答案对 1000000007 取余后的结果。

# Samples

```input1
3 2
```

```output1
5
```
```input2
6 4
```

```output2
39
```
```input3
2 1
```

```output3
2
```
