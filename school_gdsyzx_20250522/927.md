# Description
众所周知计算机代码底层计算都是 0 和 1 的计算，牛牛知道这点之后就想使用0和1创造一个新世界！牛牛现在手里有 n 个 0 和 m 个 1，给出牛牛可以创造的 x 种物品，每种物品都由一个 01 串表示。牛牛想知道当前手中的 0 和 1 可以最多创造出多少种物品。

# Format

## Input
输入数据包括 x + 1 行：

第一行包括三个整数 x（2 ≤ x ≤ 20），n（0 ≤ n ≤ 500），m（0 ≤ m ≤ 500），以空格分隔。

接下来的 x 行，每行一个 01 串 item[i]，表示第 i 个物品。每个物品的长度 length（1 ≤ length ≤ 50）。

## Output
输出一个整数，表示牛牛最多能创造多少种物品。

# Samples

```input1
3 3 1
1
00
100
```

```output1
2
```
