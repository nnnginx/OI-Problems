# Description
广东实验中学的同学们使用硬币作为交易。他们有面值为 A1，A2，A3，...，An 的硬币。一天，Hibix打开他的钱包发现还有些硬币。他决定在附近的一家商店买一块非常漂亮的手表。他知道，手表的价格不会超过 m，他想精准地支付手表的价格（即不用找零），然而他并不知道手表的确切价格。

你要写一个程序，输入 n，m，A1，A2，A3，...，An 以及 C1，C2，C3，...，Cn（即对应的面值硬币的数量），然后计算Hibix可以用这些硬币支付多少种价格。（从 1 到 m）

# Format

## Input
第一行，两个整数 n（1 ≤ n ≤ 100），m（m ≤ 100000）。

第二行，包含 2 × n 个整数，分别是 A1，A2，A3，...，An 以及 C1，C2，C3，...，Cn。（1 ≤ Ai ≤ 100000，1 ≤ Ci ≤ 1000）。

## Output
一行，一个整数，为可支付的价格总数。

# Samples

```input1
3 10
1 2 4 2 1 1
```

```output1
8
```
```input2
2 5
1 4 2 1
```

```output2
4
```
