# P4413 [COCI 2006/2007 #2] R2

## 题目描述

如果数 S 等于 $(R1 + R2) / 2$，那么 S 就被称为两个数 R1 和 R2 的平均数。Mirko 送给 Slavko 的生日礼物是两个整数 R1 和 R2。Slavko 迅速计算了它们的平均数，这个平均数恰好也是一个整数，但随后他丢失了 R2！请帮助 Slavko 恢复 R2。

## 输入格式

输入的第一行包含两个整数 R1 和 S，范围均为 -1000 到 1000。

## 输出格式

输出 R2，单独占一行。

## 输入输出样例 #1

### 输入 #1

```
11 15
```

### 输出 #1

```
19
```

## 输入输出样例 #2

### 输入 #2

```
4 3
```

### 输出 #2

```
2
```
