# B4117 [JOI2025 预选赛 R1H3] 黑桃 J

## 题目描述

给定三个正整数 $A,B,C$，如果三个数的和 $\leq 21$，输出 $1$，否则输出 $0$。

## 输入格式

第一行一个正整数 $A$。

第二行一个正整数 $B$。

第三行一个正整数 $C$。

## 输出格式

一行一个整数，表示你的答案。

## 输入输出样例 #1

### 输入 #1

```
5
10
6
```

### 输出 #1

```
1
```

## 输入输出样例 #2

### 输入 #2

```
7
8
13
```

### 输出 #2

```
0
```

## 说明/提示

### 数据范围

$1\leq A,B,C\leq 13$。
