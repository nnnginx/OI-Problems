# P10171 [DTCPC 2024] 取模

## 题目描述

给定序列 $\{a_n\}$ 和 $[L,R]$，问有多少个 $k\in [L,R]$ 满足 $a_i\bmod k$ 互不相同。

## 输入格式

第一行三个正整数 $n,L,R$（$n\leq 5\times 10^4,1\leq L\leq R\leq 10^9$）。

第二行 $n$ 个正整数，表示 $\{a_n\}$（$0 < a_i\leq 4\times10^5$）。

## 输出格式

一行一个数，表示答案。

## 输入输出样例 #1

### 输入 #1

```
3 1 3
11 7 6
```

### 输出 #1

```
1
```
