# P7662 [COCI 2014/2015 #5] ZGODAN

## 题目描述

定义 Handsome 数：这个正整数中任意两个相邻的数位的奇偶性不同。

给你一个正整数 $n$，求与 $n$ 最近的 Handsome 数。如果有多个按从小到大的顺序输出。

## 输入格式

一个正整数 $n$。

## 输出格式

与 $n$ 最近的 Handsome 数。

## 输入输出样例 #1

### 输入 #1

```
13
```

### 输出 #1

```
12 14
```

## 输入输出样例 #2

### 输入 #2

```
5801001
```

### 输出 #2

```
5810101
```

## 说明/提示

对于 $100\%$ 的数据，$1 \leq n \leq 10^{1000}$。
