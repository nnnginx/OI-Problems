# P2882 [USACO07MAR] Face The Right Way G

## 题目描述

$N$ 头牛排成一列。每头牛或者向前或者向后。为了让所有牛都面向前方，农夫每次可以将 $K$ 头连续的牛转向 $1 \le K \le N$，求最小的操作次数 $M$ 和相应的最小 $K$。

## 输入格式

第一行一个正整数 $N$。

下面 $N$ 行，每行一个字符 `F` 或 `B`，表示一头奶牛的初始朝向。（`F` 为朝前，`B` 为朝后）

## 输出格式

请在一行输出两个数字 $K$ 和 $M$，用空格分开。

## 输入输出样例 #1

### 输入 #1

```
7
B
B
F
B
F
B
B
```

### 输出 #1

```
3 3
```

## 说明/提示

样例解释：$K=3$，$M=3$，$3$ 次操作分别让奶牛 $1/2/3,\ \ 3/4/5,\ \ 5/6/7$ 转向。


---


对于 $100\%$ 的数据，$1 \le N \le 5000$。
