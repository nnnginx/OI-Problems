# B4112 [JOI2025 预选赛 R1H2] 徒步竞走

## 题目描述

比太郎以每秒 $V\textrm{ m}$ 的速度行走，问比太郎在 $T$ 秒内可以行走多少 $\rm{m}$。

## 输入格式

第一行一个正整数 $T$。

第二行一个正整数 $V$。

## 输出格式

一行一个正整数表示答案

## 输入输出样例 #1

### 输入 #1

```
5
3
```

### 输出 #1

```
15
```

## 输入输出样例 #2

### 输入 #2

```
2
4
```

### 输出 #2

```
8
```

## 说明/提示

### 样例解释

比太郎以 $3 \textrm{ m/s}$ 的速度行走 $5 \textrm{ s}$，可以行走 $15 \textrm{ m}$。

### 数据范围

$1\leq T\leq 100$，$1\leq V\leq 100$。
