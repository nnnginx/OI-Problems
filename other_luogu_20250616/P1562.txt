# P1562 还是 N 皇后

## 题目描述

正如题目所说，这题是著名的 $N$ 皇后问题。

## 输入格式

第一行有一个 $N$。接下来有 $N$ 行 $N$ 列描述一个棋盘，`*` 表示可放，`.` 表示不可放。

## 输出格式

输出方案总数。

## 输入输出样例 #1

### 输入 #1

```
4
**.*
****
****
****
```

### 输出 #1

```
1
```

## 说明/提示

$0< n\le14$
