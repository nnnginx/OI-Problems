# P7027 [NWRRC 2017] Intelligence in Perpendicularia

## 题目描述

给你一个数 $ n $ ，再给你 $ n $ 个点（$ x_i , y_i $），这 $ n $ 个点依次连成一个多边形。（保证多边形的每条边都与坐标轴平行或垂直，点不重合，点不在边上，边无相交）

求有多长的边是安全的？

（一个单位长度的边是安全的当且仅当它向外平移后能与其余边相遇，结合一下图看看）

## 输入格式

第一行一个数$ n $,表示有多少个定点。

接下来 $ n $ 行，行两个数 $ x_i \; y_i $ 表示每个点的坐标。

## 输出格式

一个数，表示安全的长度。

## 输入输出样例 #1

### 输入 #1

```
10
1 1
6 1
6 4
3 4
3 3
5 3
5 2
2 2
2 3
1 3
```

### 输出 #1

```
6
```
