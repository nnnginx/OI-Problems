# P5051 [COCI 2017/2018 #7] Timovi

## 题目描述

把$m$个人放在$n$个队伍里。
规则：
- 每次放$k$个人
- 顺序为$1^{th}$->$2^{th}$->$3^{th}$->……->$(n-1)^{th}$->$n^{th}$->$(n-1)^{th}$->$(n-2)^{th}$->……->$2^{th}$->$1^{th}$->$2^{th}$->……
- 一直重复到$\text{当前剩余人数}<k$为止。此时剩下的人全部放在当前队里

## 输入格式

无

## 输出格式

无

## 输入输出样例 #1

### 输入 #1

```
2 1 3
```

### 输出 #1

```
2 1
```

## 输入输出样例 #2

### 输入 #2

```
3 2 7
```

### 输出 #2

```
2 3 2
```

## 输入输出样例 #3

### 输入 #3

```
4 5 6
```

### 输出 #3

```
5 1 0 0
```
