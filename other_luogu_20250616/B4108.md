# B4108 [JOI2025 预选赛 R1H1] 铅笔 2

## 题目描述

JOI 君有 $A$ 日元，一根铅笔 $5$ 日元，问 JOI 君最多可以买多少根铅笔。

## 输入格式

一行一个正整数 $A$。

## 输出格式

一行一个整数，表示答案。

## 输入输出样例 #1

### 输入 #1

```
9
```

### 输出 #1

```
1
```

## 输入输出样例 #2

### 输入 #2

```
10
```

### 输出 #2

```
2
```

## 输入输出样例 #3

### 输入 #3

```
3
```

### 输出 #3

```
0
```

## 输入输出样例 #4

### 输入 #4

```
100
```

### 输出 #4

```
20
```

## 说明/提示

### 样例解释
$1$ 根铅笔 $5$ 日元，$2$ 根铅笔 $10$ 日元，$3$ 根铅笔 $15$ 日元。

### 数据范围

$1\leq A\leq 100$。
