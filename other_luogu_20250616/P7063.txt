# P7063 [NWRRC 2014] Digits

## 题目描述

小 Petya 喜欢整数。最近他了解到数字和的不同性质。例如，如果一个数字的各位数字之和可以被 $9$ 整除，那么这个数字本身也可以被 $9$ 整除。

现在小 Petya 对具有相同数字和的数字感兴趣。他请求他的哥哥 Dima 找出 $n$ 个具有相同数字和且总和最小的正整数。Dima 有其他重要的事情要做，所以他请你为他编写一个程序来解决这个问题。

## 输入格式

输入文件包含一个整数 $n (1 \le n \le 5000)$。

## 输出格式

输出 $n$ 个正整数的最小可能总和，这些整数的数字和相同。

## 输入输出样例 #1

### 输入 #1

```
2
```

### 输出 #1

```
11
```

## 输入输出样例 #2

### 输入 #2

```
3
```

### 输出 #2

```
33
```

## 说明/提示

时间限制：2 秒，内存限制：256 MB。
