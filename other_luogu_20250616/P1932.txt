# P1932 A+B  A-B  A*B  A/B A%B Problem

## 题目背景

这个题目很新颖吧！！！

## 题目描述

求 $A$ 与 $B$ 的和、差、积、商、余。

**由于数据有修改，减法运算结果可能带负号**

## 输入格式

两行，每行一个数，为参与运算的两个数。

## 输出格式

五行，每行一个数，分别为 $A$ 与 $B$ 的和、差、积、商、余。

## 输入输出样例 #1

### 输入 #1

```
1
1
```

### 输出 #1

```
2
0
1
1
0
```

## 说明/提示

$|A|,|B| \leq 10^4$，这里表示数字的位数而不是绝对值

$ A,B > 0 $
