# B2037 奇偶数判断

## 题目描述

给定一个整数，判断该数是奇数还是偶数。如果 $n$ 是奇数，输出 `odd`；如果 $n$ 是偶数，输出 `even`。

## 输入格式

输入仅一行，一个整数 $n$。

## 输出格式

输出仅一行，如果 $n$ 是奇数，输出 `odd`；如果 $n$ 是偶数，输出 `even`。

## 输入输出样例 #1

### 输入 #1

```
5
```

### 输出 #1

```
odd
```

## 输入输出样例 #2

### 输入 #2

```
10
```

### 输出 #2

```
even
```

## 说明/提示

$-100\le n\le 100$。
