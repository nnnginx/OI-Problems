# P1952 火星上的加法运算

## 题目描述

最近欢欢看到一本有关火星的书籍，其中她被一个加法运算所困惑，由于她的运算水平有限.她想向你求助，作为一位优秀的程序员，你当然不会拒绝。

## 输入格式

第一行先输入一个运算的进制 $N\left(2 \le N \le 36\right)$ 。接下来两行为需要进行运算的字符，其中每个字符串的长度不超过 $200$ 位，其为 N 进制的数。其中包括 $0$ 到 $9$ 及 a 到 z（代表 $10$ 到 $35$ ）。

## 输出格式

在 N 进制下它们的和。

## 输入输出样例 #1

### 输入 #1

```
20
1234567890
abcdefghij
```

### 输出 #1

```
bdfi02467j
```
