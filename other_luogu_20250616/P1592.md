# P1592 互质

## 题目描述

输入两个正整数 $n$ 和 $k$，求与 $n$ 互质的第 $k$ 个正整数。

## 输入格式

仅一行，为两个正整数 $n$ 和 $k$。

## 输出格式

一个正整数，表示与 $n$ 互质的第 $k$ 个正整数。

## 输入输出样例 #1

### 输入 #1

```
10 5
```

### 输出 #1

```
11
```

## 说明/提示

#### 数据规模与约定

对于所有的数据，保证 $1 \leq n \le 10^6$，$1 \leq k\le 10^8$。
