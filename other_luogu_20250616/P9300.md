# P9300 [CCC 2023 J2] Chili Peppers

## 题目描述

Ron 正在用各种辣椒烹饪辣椒酱。

辣椒的辣度以斯科维尔辣度单位（SHU）来衡量。Ron 的辣椒酱目前一点都不辣，但每次 Ron 加入一个辣椒时，辣椒酱的总辣度就会增加该辣椒的 SHU 值。Ron 可用的辣椒的 SHU 值如下表所示：

| 辣椒名称 | 斯科维尔辣度单位 |
| :----------: | :----------: |
| Poblano | $1500$ |
| Mirasol | $6000$ |
| Serrano | $15500$ |
| Cayenne | $40000$ |
| Thai | $75000$ |
| Habanero | $125000$ |

你的任务是确定 Ron 加完辣椒后，他的辣椒酱的总辣度。

## 输入格式

输入的第一行包含一个正整数 $N$，表示 Ron 加入辣椒酱中的辣椒数量。接下来的 $N$ 行中，每行包含 Ron 加入的一个辣椒的名称。每个辣椒名称将与上表中出现的名称完全匹配。注意，可以加入多个同名的辣椒。

## 输出格式

输出将由一个正整数 $T$ 组成，表示 Ron 的辣椒酱的总辣度。

## 输入输出样例 #1

### 输入 #1

```
4
Poblano
Cayenne
Thai
Poblano
```

### 输出 #1

```
118000
```
