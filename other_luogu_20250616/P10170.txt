# P10170 [DTCPC 2024] 小方和小立方

## 题目背景

小方不想变成小立方。

## 题目描述

给你一个小写字符组成的字符串，求有多少个区间 $[l,r]$，满足这个区间所表示的串是回文的，且每个字符的出现次数不超过 $2$。

## 输入格式

一行一个字符串 $S$（$\lvert S\rvert\leq 5\times 10^4$）。

## 输出格式

一行一个数，表示答案。

## 输入输出样例 #1

### 输入 #1

```
ababa
```

### 输出 #1

```
8
```
