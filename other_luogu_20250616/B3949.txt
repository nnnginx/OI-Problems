# B3949 [语言月赛 202403] 星云

## 题目背景

你看，那一抹云，衬着星星，多美啊！

## 题目描述

定义星云数为位数不大于 $n$ 且各数位之和不超过 $k$ 的正整数，给定 $n,k$，求星云数的个数。

## 输入格式

第一行输入两个整数 $n,k$。

## 输出格式

输出一行一个整数，表示答案。

## 输入输出样例 #1

### 输入 #1

```
3 2
```

### 输出 #1

```
9
```

## 说明/提示

### 数据规模与约定

对于 $30\%$ 的数据，$n=1$。  
对于 $100\%$ 的数据，$1 \leq n \leq 7$，$1 \leq k \leq 100$。
