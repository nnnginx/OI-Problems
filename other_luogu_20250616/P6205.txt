# P6205 [USACO06JAN] Dollar Dayz S

## 题目描述

FJ 到商场买工具。商场里有 $K$ 种工具（$1 \leq K \leq 100$），价格分别为 $1,2,\ldots,K$ 元。FJ 手里有 $N$ 元（$1 \leq N \leq 1000$），必须花完。他有多少种购买方案呢？

## 输入格式

一行两个整数 $N,K$。

## 输出格式

输出不同的购买方案数。

## 输入输出样例 #1

### 输入 #1

```
5 3
```

### 输出 #1

```
5
```
