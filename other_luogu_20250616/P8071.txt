# P8071 [COCI 2009/2010 #7] SPAVANAC

## 题目描述

给定 $24$ 小时制下的时间 $H:M$，输出 $24$ 小时制下比该时间早 $45$ 分钟对应的时刻。

## 输入格式

第一行，两个正整数 $H,M$。

## 输出格式

输出两个整数，表示早 $45$ 分钟后的时刻。

## 输入输出样例 #1

### 输入 #1

```
10 10
```

### 输出 #1

```
9 25
```

## 输入输出样例 #2

### 输入 #2

```
0 30
```

### 输出 #2

```
23 45
```

## 输入输出样例 #3

### 输入 #3

```
23 40
```

### 输出 #3

```
22 55
```

## 说明/提示

**【数据规模与约定】**

- 对于 $100\%$ 的数据，$0 \le H \le 23$，$0 \le M \le 59$。
