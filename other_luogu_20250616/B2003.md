# B2003 输出第二个整数

## 题目描述

输入三个整数，整数之间由一个空格分隔。把第二个输入的整数输出。

## 输入格式

只有一行，共三个整数，整数之间由一个空格分隔。

## 输出格式

只有一行，一个整数，即输入的第二个整数。

## 输入输出样例 #1

### 输入 #1

```
123 456 789
```

### 输出 #1

```
456
```

## 说明/提示

对于 $100\%$ 的数据，输入的整数在 $[1, {10}^9]$ 内。
