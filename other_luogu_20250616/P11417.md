# P11417 [Sloi 2024]D1T1 精卫

## 题目背景

![](https://cdn.luogu.com.cn/upload/image_hosting/p38htsuu.png)

## 题目描述

$f(x)$ 为积性函数，且满足 $f(p^k)=p^{2k}+k$（$p$ 为素数）。

令 $g(x)=\prod\limits_{d|x}f(d)\space \bmod\space (10^9+7)$ ，请计算 $g(i)\space (1\le i \le n)$ 的异或和。

## 输入格式

一行一个正整数 $n$。

## 输出格式

一行一个非负整数。

## 输入输出样例 #1

### 输入 #1

```
5
```

### 输出 #1

```
78
```

## 输入输出样例 #2

### 输入 #2

```
142857
```

### 输出 #2

```
67850062
```

## 输入输出样例 #3

### 输入 #3

```
10000000
```

### 输出 #3

```
505679580
```
