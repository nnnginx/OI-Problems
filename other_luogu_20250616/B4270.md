# B4270 [蓝桥杯青少年组省赛 2023] 特殊运算符

## 题目描述

假定有一个运算符 $\texttt{>>>}$，它的功能如下所示：

$$\texttt{>>>}\ 257 = 25$$

$$\texttt{>>>}\ 182 = 18$$

$$\texttt{>>>}\ 933 = 93$$

给定一个正整数 $N(100<N<1000)$，请计算 $N - (\texttt{>>>}\ N)$ 的结果。

例如：$N=257$ 时，$257 - (\texttt{>>>}\ 257) = 257 - 25 = 232$。

## 输入格式

输入一个正整数 $N(100<N<1000)$。

## 输出格式

输出一个整数，表示 $N - (\texttt{>>>}\ N)$ 的结果

## 输入输出样例 #1

### 输入 #1

```
257
```

### 输出 #1

```
232
```
