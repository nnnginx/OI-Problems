# B2028 反向输出一个三位数

## 题目描述

将一个三位数反向输出，例如输入 $358$，反向输出 $853$。

## 输入格式

一个三位数 $n$。

## 输出格式

反向输出 $n$。

## 输入输出样例 #1

### 输入 #1

```
100
```

### 输出 #1

```
001
```

## 输入输出样例 #2

### 输入 #2

```
001
```

### 输出 #2

```
100
```

## 输入输出样例 #3

### 输入 #3

```
678
```

### 输出 #3

```
876
```
