# P2399 non hates math

## 题目背景

non 习惯将分数化成小数，但在数学中要以分数形式写，不能化成小数。

因此 non 找到了会编程的你，帮助他将小数化回分数。

## 题目描述

给出一个小数，将它化成假分数的形式。

小数的类型有2种：（不考虑无限不循环小数）

1. 普通小数；
2. 循环小数（会给出循环节，循环节用 `()` 表示）。

## 输入格式

一个小数 $n$。

## 输出格式

输出这个小数 $n$ 转化成最简分数的形式。

## 输入输出样例 #1

### 输入 #1

```
1.32
```

### 输出 #1

```
33/25
```

## 输入输出样例 #2

### 输入 #2

```
1.(3)
```

### 输出 #2

```
4/3
```

## 输入输出样例 #3

### 输入 #3

```
1.0
```

### 输出 #3

```
1/1
```

## 说明/提示

输入小数的数据范围是 $0$ 至 $1000$，保证小数点后的字符数不超过 $9$。

对于 $50\%$ 的数据保证没有循环节。
