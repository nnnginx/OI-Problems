# P1748 H 数

## 题目描述

所谓 H 数，是指只含有 $2,3,5,7$ 这些质因数的数，如 $630$ 是 H 数，而 $22$ 不是。

（特别地，$1$ 是最小的 H 数）

现在要求输出第 $n$ 小的 H 数。

## 输入格式

一个正整数 $n\ (1 \le n \le 10^4)$。

## 输出格式

一个正整数表示答案。

可以证明，答案不会超过 $2^{63}-1$。

## 输入输出样例 #1

### 输入 #1

```
30
```

### 输出 #1

```
49
```

## 输入输出样例 #2

### 输入 #2

```
1
```

### 输出 #2

```
1
```
