# P7048 [NWRRC 2015] Alex Origami Squares

## 题目描述

有一张矩形纸，要求将其裁成三个全等的正方形，且每一个正方形的侧面都与纸张的侧面平行，问裁成的小正方形边长最大是多少。

## 输入格式

一行两个整数 $h$ 和 $w$ $(1\le h,w\le 10^3)$ 表示矩形纸张的长和宽。

## 输出格式

输出一行一个实数，表示正方形边长的最大可能值。

你的答案与标准答案的相对及绝对误差不应超过 $10^{-3}$。

## 输入输出样例 #1

### 输入 #1

```
210 297
```

### 输出 #1

```
105.0
```

## 输入输出样例 #2

### 输入 #2

```
250 100
```

### 输出 #2

```
83.333333
```
