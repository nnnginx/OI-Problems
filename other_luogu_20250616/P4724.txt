# P4724 【模板】三维凸包

## 题目描述

给出空间中 $n$ 个点，求凸包表面积。

## 输入格式

第一行一个整数 $n$，表示点数。

接下来 $n$ 行，每行三个实数 $x, y, z$ 描述坐标。

## 输出格式

输出凸包表面积，保留 $3$ 位小数。

## 输入输出样例 #1

### 输入 #1

```
4 
0 0 0
1 0 0
0 1 0
0 0 1
```

### 输出 #1

```
2.366
```

## 说明/提示

$4 \le n \le 2000$
