# P2543 [AHOI2004] 奇怪的字符串

## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/1654.png)

## 输入格式

输入文件中包含两个字符串X和Y。当中两字符串非0即1。序列长度均小于9999。

## 输出格式

X和Y的最长公共子序列长度。

## 输入输出样例 #1

### 输入 #1

```
01010101010 00000011111
```

### 输出 #1

```
6
```

## 输入输出样例 #2

### 输入 #2

```
01011 010010101111111111
```

### 输出 #2

```
5
```
