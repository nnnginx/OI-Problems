# P2660 zzc 种田

## 题目背景

可能以后 zzc 就去种田了。

## 题目描述

田地是一个巨大的矩形，然而 zzc 每次只能种一个正方形,而每种一个正方形时 zzc 所花的体力值是正方形的周长，种过的田不可以再种，zzc 很懒还要节约体力去泡妹子，想花最少的体力值去种完这块田地，问最小体力值。

## 输入格式

两个正整数 $x,y$，表示田地的长和宽。

## 输出格式

输出最小体力值。

## 输入输出样例 #1

### 输入 #1

```
1 10
```

### 输出 #1

```
40
```

## 输入输出样例 #2

### 输入 #2

```
2 2
```

### 输出 #2

```
8
```

## 说明/提示

$1\le x,y\le 10^{16}$。
