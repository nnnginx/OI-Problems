<h2>Description</h2><p><span lang="EN-US">Little Y finds there is a very interesting formula in mathematics:</span></p><p align="center" style="text-align:center"><span lang="EN-US"><i>X<sup>Y</sup> mod Z = K</i></span></p><p><span lang="EN-US">Given <i>X</i>, <i>Y</i>, <i>Z</i>, we all know how to figure out <i>K</i> fast. However, given <i>X</i>, <i>Z</i>, <i>K</i>, could you figure out <i>Y</i> fast?</span></p><p>
</p><h2>Input</h2><p>Input data consists of no more than 20 test cases. For each test case, there would be only one line containing 3 integers </p><i>X</i><p>, </p><i>Z</i><p>, </p><i>K</i><p> (0 ¡Ü </p><i>X</i><p>, </p><i>Z</i><p>, </p><i>K</i><p> ¡Ü 10</p><sup>9</sup><p>).
</p>Input file ends with 3 zeros separated by spaces.
<h2>Output</h2><p>For each test case output one line. Write "No Solution" (without quotes) if you cannot find a feasible </p><i>Y</i><p> (0 ¡Ü </p><i>Y</i><p> &lt; </p><i>Z</i><p>). Otherwise output the minimum </p><i>Y</i><p> you find.</p><pre><code class="language-input1">5 58 33
2 4 3
0 0 0
</code></pre><pre><code class="language-output1">9
