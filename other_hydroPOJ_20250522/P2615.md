<h2>Description</h2><p>Given two strings of lowercase letters, a and b, print the shortest string x of lowercase letters such that exactly one (but not both) of ax or bx is a palindrome; that is, equal to itself when reversed.</p><h2>Input</h2><p>Standard input contains several pairs of a and b. Each string is on a separate line and consists of between 0 and 1,000 lowercase letters.</p><h2>Output</h2><p>For each pair, output a line containing x. If several x satisfy the criteria above, choose the first one in alphabetical order.If there no such string x, output "No Solution."</p><pre><code class="language-input1">abab
ababab
abc
def
</code></pre><pre><code class="language-output1">baba
