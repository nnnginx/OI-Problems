<h2>Description</h2><p>In many applications very large integers numbers are required. Some of these applications are using keys for secure transmission of data, encryption, etc. In this problem you are given a number, you have to determine the number of digits in the factorial of the number.</p><h2>Input</h2><p>Input consists of several lines of integer numbers. The first line contains an integer n, which is the number of cases to be tested, followed by n lines, one integer 1 &lt;= m &lt;= 10^7 on each line.</p><h2>Output</h2><p>The output contains the number of digits in the factorial of the integers appearing in the input.</p><pre><code class="language-input1">2
10
20</code></pre><pre><code class="language-output1">7
19</code></pre><h2>Source</h2><a href="searchproblem?field=source&amp;key=Dhaka+2002">Dhaka 2002</a>
