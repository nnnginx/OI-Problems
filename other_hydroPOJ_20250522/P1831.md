<h2>Description</h2><p>考虑不定方程组
</p>a1 + a2 + a3 + … + an = s
1 / a1 + 1 / a2 + 1 / a3 + … + 1 / an = 1
其中n, a1, a2 …均为参数，s已知。

给出s求出不定方程组的一组正整数解。<h2>Input</h2><p>第一行是一个正整数m，表示测试数据的组数，每组测试数据只有一行，一个正整数s( 1&lt;=s&lt;65536)。</p><h2>Output</h2><p>对每组测试数据输出一行，如果方程组无解，则输出-1，否则先输出正整数n，然后输出n个数a1 a2 … an，数据之间用空格隔开。</p><pre><code class="language-input1">3
1
2
10
</code></pre><pre><code class="language-output1">1 1
-1
3 2 4 4
</code></pre><h2>Source</h2><a href="searchproblem?field=source&amp;key=qinlu%40POJ">qinlu@POJ</a>
