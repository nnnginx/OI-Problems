<h2>Description</h2><p>There are <i>m</i> classes in the Stars Kindergarten. And each class has exactly <i>k</i> children. Every day Ms. Lee is to distribute <i>n</i> different toys to the <i>m</i> classes and make sure that each class obtains no less than <i>k</i> toys. Otherwise a fight among the children may break out in the class due to the lack of toys. Ms. Lee wants to know how many ways there are for her to distribute the toys.</p><h2>Input</h2><p>There are multiple test cases.<br>The first line contains the number of test cases, <i>T</i>.<br>Each of the next <i>T</i> lines describes a test case which consists of three integers <i>n</i>, <i>m</i>, <i>k</i> (0 ¡Ü <i>n</i>, <i>m</i>, <i>k</i> ¡Ü 1000)</p><h2>Output</h2><p>For each test case output the answer modulo 9973 in a separate line. Output 0 if the distribution is impossible.</p><pre><code class="language-input1">1
4 2 2
</code></pre><pre><code class="language-output1">3
</code></pre><h2>Hint</h2><p>The classes are considered identical.
