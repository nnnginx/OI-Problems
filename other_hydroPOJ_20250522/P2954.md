<h2>Description</h2><p>A <i>lattice point</i> is an ordered pair (<i>x</i>, <i>y</i>) where <i>x</i> and <i>y</i> are both integers. Given the coordinates of the vertices of a triangle (which happen to be lattice points), you are to count the number of lattice points which lie completely inside of the triangle (points on the edges or vertices of the triangle do not count).</p><h2>Input</h2><span lang="en-us"><p>The input test file will contain multiple test cases. Each input test case consists of six integers <i>x</i><sub>1</sub>, <i>y</i><sub>1</sub>, <i>x</i><sub>2</sub>, <i>y</i><sub>2</sub>, <i>x</i><sub>3</sub>, and <i>y</i><sub>3</sub>, where (<i>x</i><sub>1</sub>, <i>y</i><sub>1</sub>), (<i>x</i><sub>2</sub>, <i>y</i><sub>2</sub>), and (<i>x</i><sub>3</sub>, <i>y</i><sub>3</sub>) are the coordinates of vertices of the triangle. All triangles in the input will be non-degenerate (will have positive area), and −15000 ≤ <i>x</i><sub>1</sub>, <i>y</i><sub>1</sub>, <i>x</i><sub>2</sub>, <i>y</i><sub>2</sub>, <i>x</i><sub>3</sub>, <i>y</i><sub>3</sub> ≤ 15000. The end-of-file is marked by a test case with <i>x</i><sub>1</sub> =&nbsp; <i>y</i><sub>1</sub> = <i>x</i><sub>2</sub> = <i>y</i><sub>2</sub> = <i>x</i><sub>3</sub> = <i>y</i><sub>3</sub> = 0 and should not be processed.</p></span><h2>Output</h2><p>For each input case, the program should print the number of internal lattice points on a single line.</p><pre><code class="language-input1">0 0 1 0 0 1
0 0 5 0 0 5
0 0 0 0 0 0</code></pre><pre><code class="language-output1">0
6</code></pre><h2>Source</h2><a href="searchproblem?field=source&amp;key=Stanford+Local+2004">Stanford Local 2004</a>
