<h2>Description</h2><p>If a matrix satisfies the following conditions, we call it a silver matrix.
</p>
1.	The dimensions of the matrix are n * n.
2.	All its elements belong to the set S = {1, 2, 3, бн, 2n - 1}. 
3.	For every integer i (1 &lt;= i &lt;= n), all elements in the i-th row and i-th column make the set {1, 2, 3, бн, 2n - 1}.
