<h2>Description</h2><p>A Benedict monk No.16 writes down the decimal representations of all natural numbers between and including <i>m</i> and <i>n</i>, <i>m</i> ¡Ü <i>n</i>. How many 0's will he write down? </p><h2>Input</h2><p>Input consists of a sequence of lines. Each line contains two unsigned 32-bit integers <i>m</i> and <i>n</i>, <i>m</i> ¡Ü <i>n</i>. The last line of input has the value of <i>m</i> negative and this line should not be processed. </p><h2>Output</h2><p>For each line of input print one line of output with one integer number giving the number of 0's written down by the monk. </p><pre><code class="language-input1">10 11
