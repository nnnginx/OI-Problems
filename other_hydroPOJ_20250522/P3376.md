For example, "dad", "eye" and "racecar" are all palindromes, but "odd", "see" and "orange" are not palindromes.</p><p>Given <i>n</i> strings, you can generate <i>n</i> ¡Á <i>n</i> pairs of them and concatenate the pairs into single words. The task is to count how many of the so generated words are palindromes.</p><h2>Input</h2><div><p>The first line of input file contains the number of strings <i>n</i>. Two strings in different line may be the same. </p></div><h2>Output</h2><p>Print out only one integer, the number of palindromes. </p><pre><code class="language-input1">3
