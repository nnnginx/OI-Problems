<h2>Description</h2><p>For each three prime numbers p1, p2 and p3, let's define Hamming sequence Hi(p1, p2, p3), i=1, ... as containing in increasing order all the natural numbers whose only prime divisors are p1, p2 or p3.
</p>
For example, H(2, 3, 5) = 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, 16, 18, 20, 24, 25, 27, ...

So H5(2, 3, 5)=6.<h2>Input</h2><p>In the single line of input file there are space-separated integers p1 p2 p3 i.</p><h2>Output</h2><p>The output file must contain the single integer - Hi(p1, p2, p3). All numbers in input and output are less than 10^18.</p><pre><code class="language-input1">7 13 19 100</code></pre><pre><code class="language-output1">26590291</code></pre><h2>Source</h2><a href="searchproblem?field=source&amp;key=Northeastern+Europe+2000">Northeastern Europe 2000</a><p>, Far-Eastern Subregion</p>
