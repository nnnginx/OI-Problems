# UVA10006 Carmichael Numbers

## 题目描述

x^n≡x(mod n) 成立的合数n称为Carmichael Numbers .先给出一些整数n,请判断是否为Carmichael Numbers 。

## 输入格式

无

## 输出格式

无

## 输入输出样例 #1

### 输入 #1

```
1729
17
561
1109
431
0
```

### 输出 #1

```
The number 1729 is a Carmichael number.
17 is normal.
The number 561 is a Carmichael number.
1109 is normal.
431 is normal.
```
