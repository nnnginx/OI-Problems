# UVA12683 Odd and Even Zeroes

## 题目描述

题意：

给出 $n$，求出 $0!,1!,2!...,n!$ 中有几个末尾有偶数个 0。

## 输入格式

一行一个数 $n$（$1\leq n \leq 10^{18}$），读至 $-1$ 停止（不必输出 $n$ 是 $-1$ 的答案）。

## 输出格式

对应的答案，一行一个。

## 输入输出样例 #1

### 输入 #1

```
2
3
10
100
1000
2000
3000
10000
100000
200000
-1
```

### 输出 #1

```
3
4
6
61
525
1050
1551
5050
50250
100126
```
