# UVA133 救济金发放

## 题目描述

$n$ 个人站成一圈，**逆时针**编号为 $1\sim n$。有两个官员，$\text{A}$从 $1$ 开始逆时针数，$\text{B}$ 从 $n$ 开始顺时针数。

在每一轮中，官员 $\text{A}$ 数 $k$个就停下来，官员 $\text{B}$ 数 $m$ 个就停下来（两个官员有可能能停在同一个人上）。接下来被官员选中的 $1$ 个或 $2$个人离开队伍。

输出离开队伍的顺序，如果有两个人，先输出 A 选中的。

## 输入格式

**本题多测**。

对于每组数据，输入共一行，输入 $n,k,m$。

数据以 $\texttt{0 0 0}$ 结尾。

## 输出格式

对于每组数据，输出一行，为离开队伍的顺序。

输出的每个数应正好占 $3$ 列。

样例中的“ ␣ ”代表一个空格。

## 输入输出样例 #1

### 输入 #1

```
10 4 3
0 0 0
```

### 输出 #1

```
␣␣4␣␣8,␣␣9␣␣5,␣␣3␣␣1,␣␣2␣␣6,␣10,␣␣7
```

## 说明/提示

### 数据范围

对于 $100\%$ 的数据，满足 $0<n<20$，$0<k,m$。
