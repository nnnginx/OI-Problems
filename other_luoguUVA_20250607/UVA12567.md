# UVA12567 《第七页》、《时空轮回》与水的故事 Never7, Ever17 and Water

## 题目描述

### 题目背景

你需要建一个非常有趣的地下排水系统。

在这个循环水系统之中（没有源头）有 $m$ 条管道连接 $n$ 个节点。没有水在循环中凭空产生或凭空消失，总水量在任何时间都是一定的，且任意的节点输入水速等于输出水速。

管道可以限制水的速度的上界和下界，通过它的实际水流速度必须在边界之内。并且，你可以看到水流过管道的速度。你的任务是求出，怎样才能使水速的极差最小，并输出这个速度的最小值。

## 输入格式

$T$ 组数据。第一行两个整数 $n,m(2\le n\le 50,1\le m\le 200)$，意义如上。以下 $m$ 行中每行都包含四个整数 $u,v,b,c(1\le u,v\le n,u\neq v,0\le b\le c\le 100)$，意味着有一个管道连接节点 $u$ 和节点 $v$，水从 $u$ 流向 $v$，并且水流速度 $f$ 满足 $b\le f\le c$。节点编号从 $1$ 到 $n$，管道编号 $1$ 到 $m$。

## 输出格式

对于每组数据，输出最小的水流速度的极差。如果这样一种系统甚至无法实现，输出 $-1$。

## 输入输出样例 #1

### 输入 #1

```
3
4 4
1 2 1 4
2 3 2 5
3 4 3 6
4 1 4 7
3 3
1 2 1 2
2 3 2 3
3 1 3 4
2 3
1 2 3 3
2 1 0 10
2 1 0 10
```

### 输出 #1

```
Case 1: 0.00000
Case 2: -1
Case 3: 1.50000
```
