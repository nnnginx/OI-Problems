# UVA13221 Elegant Pillars

## 题目描述

假设有   $N$ 根柱子，还有很多的球，编号为   $1$，  $2$，  $3$，  $4$，  $5$...

把这些球放到柱子上，使得每一个柱子上任意两个相邻球的编号和是一个平方数，问对于这   $N$ 根柱子，最多能放进多少个球

## 输入格式

有多组数据，总数据组数不会多于   $1000$ ，每组数据仅有一行，表示  $ N(0<N<10^9) $。

## 输出格式

对于每一组数据，输出一个数，表示你最多能放的球的数量

## 输入输出样例 #1

### 输入 #1

```
1
2
```

### 输出 #1

```
1
3
```
