# UVA12895 Armstrong Number

## 题目描述

定义一个 $n$ 位的阿姆斯特朗数 $x$ 满足：

- $x=\overline{a_1a_2 \dots a_n}$

- $a_1^n+a_2^n+\dots+a_n^n=x$

例如，$153$ 是一个阿姆斯特朗数，因为 $1^3+5^3+3^3=153$。

给定 $T$ 个数，问它们是不是阿姆斯特朗数。

## 输入格式

**本题单测试点内有多组数据。**

第一行，一个整数 $T$ ，表示数据的组数。

接下来的 $T$ 行，每行一个整数 $N(1 \le N \le 10^9)$。

## 输出格式

输出 $T$ 行。对于每个数，如果它是阿姆斯特朗数则输出 `Armstrong`，否则输出 `Not Armstrong`。

## 输入输出样例 #1

### 输入 #1

```
3
153
2732
54748
```

### 输出 #1

```
Armstrong
Not Armstrong
Armstrong
```
