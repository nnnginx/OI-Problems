# UVA12517 Digit Sum

## 题目描述

给出 $M,N$，求在 $M,N$ 之间的所有数字的数位和。

例子：

$M=9,N=13$，则 $ans=(9)+(1+0)+(1+1)+(1+2)+(1+3)=19$。

## 输入格式

多测，每行给出 $M,N$。

若输入为 `0,0`，则代表结束。

## 输出格式

每行一个答案。

## 输入输出样例 #1

### 输入 #1

```
3 8
5 18
1 50
0 0
```

### 输出 #1

```
33
80
330
```
