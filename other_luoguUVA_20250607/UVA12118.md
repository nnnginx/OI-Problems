# UVA12118 检查员的难题 Inspector's Dilemma

## 题目描述

有 $V$个节点，任意两个节点间被一条边权为 $T$无向边连接，求一条最短的路径，使其经过 $E$条指定的边。

## 输入格式

**输入有多组数据。** 每组数据第一行有三个整数： $V(1 \le V \le 1000)$,  $E(0 \le E \le V * (V-1)/2)$ ， $T(1 \le T \le 10)$ ，接着的 $E$行每行都包含两个整数   $a$ 和  $b$$(1 \le a,b \le V,a \ne b) $，表示指定边$<a, b> $，输入以三个  $0$ 结尾。

## 输出格式

对于一组数据，输出序号和最短路径长。

## 输入输出样例 #1

### 输入 #1

```
5 3 1
1 2
1 3
4 5
4 4 1
1 2
1 4
2 3
3 4
0 0 0
```

### 输出 #1

```
Case 1: 4
Case 2: 4
```
