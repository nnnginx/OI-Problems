# UVA12195 叮当作曲

## 题目描述

A. C. 马科斯正在朝着创作广告音乐的方向迈出第一步。他在创作过程中遇到了一些困难，但至少已经能够创作出悦耳的旋律和吸引人的节奏。  

在音乐中，一个音符具有音高（由其频率决定，表现为声音的高低）和时值（音符持续的时间）。本问题中，我们只关注音符的时值。  

一首广告音乐由若干个小节组成，而每个小节由一系列音符构成。  

音符的时值由其形状表示。在本问题中，我们将使用大写字母来表示音符的时值。下表列出了所有可用的音符：  
![](https://cdn.luogu.com.cn/upload/image_hosting/528k49be.png)

一个小节的时值是其所有音符时值的总和。在马科斯的广告音乐中，每个小节的时值相同。由于马科斯是初学者，他的著名老师约翰·塞巴斯蒂安三世教导他，每个小节的时值必须始终为 $1$。  

例如，马科斯创作了一首包含五个小节的音乐，其中前四个小节的时值正确，而最后一个错误。在下面的示例中，每个小节由斜杠包围，每个音符用上表中的大写字母表示：  
$$
\text{/HH/QQQQ/XXXXXXXXEQH/W/HW/}
$$
马科斯对计算机和音乐同样热爱。他希望你能编写一个程序，用于确定他的每首作品中有多少个小节的时值是正确的。

## 输入格式

输入包含多个测试用例。每个测试用例用一行字符串表示，字符串的长度在 $3$ 到 $200$ 个字符之间（包括 $3$ 和 $200$），代表一首作品。一首作品以斜杠 `/` 开头和结尾。作品中的小节由斜杠 `/` 分隔。每个小节中的音符用对应的大写字母表示，如上所述。你可以假设每首作品至少包含一个小节，且每个小节至少包含一个音符。输入中的所有字符均为斜杠或上述七种表示音符的大写字母之一。  

最后一个测试用例后是一行，仅包含一个星号 `*`。

## 输出格式

对于每个测试用例，你的程序需输出一行，包含一个整数，表示时值正确的小节数量。

## 输入输出样例 #1

### 输入 #1

```
/HH/QQQQ/XXXTXTEQH/W/HW/
/W/W/SQHES/
/WE/TEX/THES/
*
```

### 输出 #1

```
4
3
0
```
