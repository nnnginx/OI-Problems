# UVA10101 Bangla数字

## 题目描述

Bangla 数字是这样表示的：用 `kuti` 表示 ${10}^7$，`lakh` 表示 ${10}^5$，`hajar` 表示 ${10}^3$，`shata` 表示 ${10}^2$。其中 `kuti` 类似中文的“万”，可以向上递归，也就是可以出现 `1 lakh 14 hajar 5 shata 14 kuti 19 lakh 19 hajar 8 shata 10` 这种表述，类似中文的“一百五十三万三千八百七十一”。你需要写一个程序把给定的数字转换成 Bangla 数字。

## 输入格式

输入包含多行，每行一个阿拉伯数字 $n$，$0 \le n<{10}^{15}$。

## 输出格式

输出多行。

对于每一行，先输出该行的编号（可参见样例），再输出给定数字转换成 Bangla 数字的结果。

## 输入输出样例 #1

### 输入 #1

```
23764
45897458973958
```

### 输出 #1

```
1. 23 hajar 7 shata 64
2. 45 lakh 89 hajar 7 shata 45 kuti 89 lakh 73 hajar 9 shata 58
```
