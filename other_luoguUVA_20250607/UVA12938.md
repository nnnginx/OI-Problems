# UVA12938 Just Another Easy Problem

## 题目描述

## 题面大意
给出一个正整数$n$，你可以修改其中一位上的数，使得这个数变为某个数的平方。

问：共有多少种修改方式，满足上述要求。

例如：$7844 -> 3844 = 62^2$，$7844 -> 7744 = 88^2$

所以对于$7844$有两种修改方式满足要求。

注意：第一位（也就是千位）不能被改为$0$

## 输入格式

本题多组数据。

第一行输入一个数$t$，代表数据组数。

下面的$t$，每行输入一个四位数$n$。

## 输出格式

对于每组数据，输出为"Case i: ans"

其中$i$代表这是第$i$组数据，$ans$即为方案数。

## 输入输出样例 #1

### 输入 #1

```
2
7844
9121
```

### 输出 #1

```
Case 1: 2
Case 2: 0
```
