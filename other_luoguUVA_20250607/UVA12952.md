# UVA12952 Tri-du

## 题目描述

给定两个数 $a,b$，求 $\max(a,b)$。

## 输入格式

输入若干行，每行两个正整数 $a,b$，输入以 `EOF` 结尾。

## 输出格式

对于每一组 $a,b$，输出一行一个正整数，表示 $\max(a,b)$。

## 输入输出样例 #1

### 输入 #1

```
10 7
2 2
```

### 输出 #1

```
10
2
```

## 说明/提示

对于 $100\%$ 的数据，$1\le a,b\le13$。
