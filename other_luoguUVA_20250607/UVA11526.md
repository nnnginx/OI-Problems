# UVA11526 H(n)

## 题目描述

请求出该函数的返回值：

```cpp
long long H(int n) {
    long long res = 0;
    for (int i = 1;i <= n;i = i + 1) {
        res = (res + n / i);
    }
    return res;
}
```

## 输入格式

每个测试点有多个测试数据。

第一行输入一个正整数 $T(T \le 1000)$，接下来 $T$ 行每行输入一个 32 位整形范围内的正整数 $n$。

## 输出格式

对于每组测试数据，输出对应的答案。

## 输入输出样例 #1

### 输入 #1

```
2
5
10
```

### 输出 #1

```
10
27
```
