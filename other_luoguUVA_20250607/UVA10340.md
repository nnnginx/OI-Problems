# UVA10340 子序列

## 题目描述

给定两个字符串 $s$ 和 $t$，判断是否可以从 $t$ 中删除 $0$ 个或多个字符（其它字符顺序不变），得到字符串 $s$。

若可以输出 $\texttt{Yes}$，否则输出 $\texttt{No}$。

## 输入格式

**本题包含多组测试数据**。

测试数据个数不定，最终文件以 $\texttt{EOF}$ 结尾。

对于每一组数据，输入一行包含两个字符串，分别为 $s$ 和 $t$。

## 输出格式

对于每一组数据，一行输出一个字符串 $\texttt{Yes}$ 或 $\texttt{No}$。

## 输入输出样例 #1

### 输入 #1

```
sequence subsequence
person compression
VERDI vivaVittorioEmanueleReDiItalia
caseDoesMatter CaseDoesMatter
```

### 输出 #1

```
Yes
No
Yes
No
```
