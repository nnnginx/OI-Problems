# UVA12289 One-Two-Three

## 题目描述

你的弟弟只会用英语写一、二、三，他在一张纸上写了很多个单词。你的任务是识别他们，因为你的弟弟只是个孩子，所以他也许会犯小错误：对于每个单词最多会错一个字母，不过单词的长度一直是正确的。保证他写的每一封信都是小写字母，他写的每一个字都有唯一的解释。

## 输入格式

输入一个 $n$ ,表示有 $n$ 个单词。

接下来 $n$ 行，每行有一个单词。

## 输出格式

判断每行是什么单词，并输出单词代表的数字。

## 输入输出样例 #1

### 输入 #1

```
3
owe
too
theee
```

### 输出 #1

```
1
2
3
```
