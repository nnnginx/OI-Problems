# UVA13178 Is it multiple of 3?

## 题目描述

### 题面

把从 $1$ 到 $n$ 的数全部拼接起来，问它是不是 $3$ 的倍数。例如 $n=6$ 时，构成的数是 $123456$，它是 $3$ 的倍数。

## 输入格式

第一行输入一个整数 $t$，表示共有 $t$ 组数据；

接下来 $t$ 行，每行输入一个正整数 $n$。

## 输出格式

对于每一个 $n$，如果构成的数是 $3$ 的倍数输出 `YES`，否则输出 `NO`。

## 输入输出样例 #1

### 输入 #1

```
3
2
6
130000000
```

### 输出 #1

```
YES
YES
NO
```
