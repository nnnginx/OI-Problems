## Description

<p>输入一个正整数n，以及 n 个学生的信息，包括姓名、学号、语文成绩、数学成绩和英语成绩，再给出一个正整数m，输出学号为 m 的学生的姓名和总成绩。<br /></p>

## Input Format

<p>第一行包括一个正整数n，接下来的n行每行描述1个学生的信息，各部分之间用一个空格分隔，最后一行为一个数m。<br /></p>

## Output Format

<p>一行，包括学号为m的学生姓名和三科总成绩，两部分之间用一个空格分隔。如果所给学生信息没有学号m，输出一个数0.<br /></p>

```input1
4
xuyu 2301 56 78 64
hanzheng 2408 87 62 90
huhua 2805 43 88 62
yangzheng 4901 89 34 88
2301
```
```output1
xuyu 198
```
## Hint

<p>数据范围：n不超过100，学生的姓名用不超过20的小写字母表示，学号使用四位数字表示，三科成绩均为不超过100的自然数，m为小于10000的正整数。<br /></p>
