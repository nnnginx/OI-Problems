## Description

期末考试结束了，小华语文、数学、英语三门功课分别考了x、y、z分，小华的家长说，如果小华三门功课的平均分在90分或者90分以上，那么就去北京旅游，如果在90分以下，那么就去南京玩。

请从键盘读入小华三门功课的成绩，编程计算出小华暑期旅游的目的地，如果去北京，则输出“beijing”，如果去南京，则输出“nanjing”。（只要输出双引号中间的拼音，不要输出双引号）
 

## Input Format

三个整数，代表小华三门功课的考试成绩。

## Output Format

按题目要求输出小华旅游的目的地。 

```input1
90 95 91
```
```output1
beijing
```
