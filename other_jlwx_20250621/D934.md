## Description

有一个电子门，门上有很多字符，经研究发现，门上的字符中包含了打开电子门的密码。

已知门上有两行字符串，其中第一行字符串长度一定是偶数，如果把第2个字符串插入到第1个子串的正中间，就是这个电子门的密码。
请你编程计算出电子门的正确密码。
 
 
	 
 

## Input Format

输入数据首先给出一个整数n，表示测试数据的组数。
然后是<strong>n组数据</strong>，每组数据2行，每行一个字符串（字符串不含空格），长度大于0，小于50，并且第一个串的长度一定是偶数。
 

## Output Format

请为每组数据输出一个能够开启电子门的字符串，每组输出占一行。

```input1
2
CSJI
BI
AB
CMCLU
```
```output1
CSBIJI
ACMCLUB
```
## Source

字符串
