## Description

 
	  设有有 2 <sup>n</sup> （ n&lt;=6 ）个球队进行单循环比赛，计划在 2 <sup>n</sup> C 1 天内完成，每个队每天进行一场比赛。设计一个比赛的安排，使在 2 <sup>n</sup> C 1 天内每个队都与不同的对手比赛。   
 

## Input Format

输入共一行，输入 n 的数值。   
 

## Output Format

输出文件  共（ 2 n C 1 ）行，第 i 行输出第 i 天的比赛安排。  
格式为： &lt;i&gt; A-B   C-D ， …… 。其中 i 是天数， A ， B 分别为比赛双方的编号，每行共 2 <sup>n-1</sup> 个比赛场次。   
 

```input1
2
```
```output1
<1>1-2 3-4
<2>1-3 2-4
<3>1-4 2-3
```
