## Description

晶晶学习了字符串的概念，她打算统计一下：一个字符串包含了大小写字母、数字、空格以及其他字符，这个字符串中，大小写字母、数字、空格、其他字符分别有多少个？

## Input Format

一行字符，包含了大小写字母、数字、空格以及其他字符
 

## Output Format

输出4个整数，空格隔开，分别代表该字符串中大小写字母、数字、空格以及其他字符的个数
 

```input1
aklsjflj123 sadf918u324 asdf91u32oasdf/.';123

```
```output1
23 16 2 4
```
## Source

字符串
