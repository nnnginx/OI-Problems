## Description

输入两个整数x和y，比较它们的大小。
如果x大于y，则输出符号“>”；
如果x等于y，则输出符号“=”；
如果x小于y，则输出符号“<”。

## Input Format

输入仅一行，包含两个整数x和y，中间用一个空格隔开。

## Output Format

输出仅一行，为“>”、“=”、“<”中的一个。
注意：
1.题目保证输入都在int范围内。
2.输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
99 8
```
```output1
>
```
