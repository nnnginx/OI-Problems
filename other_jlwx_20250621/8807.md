## Description

<p>一个旅行者有一个最多能装V公斤的背包，现在有n件物品，它们的重量分别是$W\_1，W\_2，...,W\_n$，它们的价值分别为$C\_1,C\_2,...,C\_n$。这些物品被划分为若干组，每组中的物品互相冲突，最多选一件。求解将哪些物品装入背包可使这些物品的费用总和不超过背包容量，且价值总和最大。<p>

## Input Format

<p>第一行：三个整数，V(背包容量，V≤200)，N(物品数量，N≤30)和T(最大组号，T≤10)；<p><p>第2..N+1行：每行三个整数$W\_i,C\_i,P$，表示每个物品的重量，价值，所属组号。<p>

## Output Format

<p>仅一行，一个数，表示最大总价值。<p>

```input1
10 6 3
2 1 1
3 3 1
4 8 2
6 9 2
2 8 3
3 9 3


```
```output1
20

```
