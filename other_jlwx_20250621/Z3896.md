## 输入格式

<div>
 第一行一个整数T。
</div>
<div>
 接下来T个整数，表示A0，A1…A(T-1)
</div>
<p></p>

## 输出格式

<div>
 输出T+1个整数，表示A0，A1…AT。
</div>
<p></p>

```input1
3
0 0 1
```
```output1
0 166666668 500000004 333333336
```
## 数据范围与约定

<div>
 T<=2000,所有系数为小于1e+9+7的非负整数。
</div>
