## Description

A公司评选先进员工，要求参加评选的员工在本公司工作的n年内，至少有1次连续3年的年度考评分数在90分或90分以上，才能参加今年的评选。

现给定某员工到公司n年的年度考评分数数据，请问他是满足备参加今年评选先进员工的要求，如果满足，请输出Y，不满足请输出N。
 
比如：王小芳连续5年的年度考评分数为90 89 100 99 95，该员工从第3年到第5年连续3年考评分数都≥90分，因此她满足参选要求。
 
再比如：李晓明连续6年的考评分数为100 98 89 99 90 88，他没有连续3年考评分数≥90分的记录，因此不满足参选的要求。
 

## Input Format

第1行读入一个整数n，代表员工在公司工作的年限（3≤n≤100）
 
第2行有n个整数，用空格隔开代表该员工每年获得的考评分数（0≤分数≤100）
 

## Output Format

按题目要求输出字符Y或者字符N。

```input1
5
90 89 100 99 95
```
```output1
Y
```
