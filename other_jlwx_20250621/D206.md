## Description

 做操的时间到了，小明在教室还在思考刚刚老师讲的一道题目，当他想通这个题时，同学们都已经在操场上排好队了，他赶快跑到操场上找到自己的班级队伍，希望尽快找到以前排队的位置，准备做操，小明记得应该排在第x学号同学的后面。你能不能来帮帮小明呢？ 

## Input Format

输入（两行）： 
第一行3个整数：n x y（现在队伍的长度、第x同学的学号、小明的学号）
第二行n个整数：n个同学的学号 
 

## Output Format

 n+1个学号（小明加入队伍后队伍中的每个数据） 

```input1
4 32 23
1 8 32 56
```
```output1
1 8 32 23 56
```
