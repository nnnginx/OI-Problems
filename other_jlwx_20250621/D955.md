## Description

不同国家的货币之间有一个兑换比例，这个兑换比例我们称之为汇率。比如近期人民币对美元的汇率是<span id="__kindeditor_bookmark_start_1__">6.5382，这意味着，1美元可以兑换到6.5382元人民币，那么x美元可以兑换到的人民币 = x * 6.5382元人民币。当然，美元对人民币的汇率也不是一直不变的，每天这个汇率都可能发生变化。
张老师有N美元，他从某金融机构得到了未来12天，可能的汇率值。请问，如果张老师要在未来12天将美元兑换为人民币，那么，他能兑换到的最多的人民币，是多少钱？
 

## Input Format

输入的第一行是一个小数N（1.00&lt;=N&lt;=100.00），表示张老师现有的美金数量。 
接下来一行，包含12个小数ai，表示接下来十二个月的美元对人民币汇率。 

## Output Format

输出一个小数，表示最多能获得的人民币数量，结果保留两位小数。

```input1
46.91
6.31 6.32 6.61 6.65 5.55 5.63 6.82 6.42 6.40 5.62 6.78 5.60
```
```output1
319.93
```
