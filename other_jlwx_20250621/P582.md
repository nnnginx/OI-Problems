## Description

已知m、n为整数，且满足下列两个条件：
</p>
<p>
	&nbsp;&nbsp;&nbsp;&nbsp;① m、n∈{１，２，…，k}，即1≤m，n≤k
</p>
<p>
	&nbsp;&nbsp;&nbsp;&nbsp;②（n<sup>2</sup>－m*n－m<sup>2</sup>）<sup>2</sup>＝1
</p>
<p>
	&nbsp;&nbsp;&nbsp;&nbsp;你的任务是：编程输入正整数k（1≤k≤10<sup>9</sup>），求一组满足上述两个条件的m、n，并且使m<sup>2</sup>＋n<sup>2</sup>的值最大。例如，输入k=1995，则输出：m=987 &nbsp; n=1597。


## Input Format

&nbsp;&nbsp;&nbsp;&nbsp;</strong>一个正整数k（1≤k≤10<sup>9</sup>）。

## Output Format

输出m和n，参考输出样例。

```input1
1995
```
```output1
m=987
n=1597
```
