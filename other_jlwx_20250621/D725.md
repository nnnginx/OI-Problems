## Description

我军拦截敌军传递了  2  个数字串，后经侦查得知，这两个数字串中包含了一个很重要的密码，这个密码的破译方式为，用这两个数字串中的大数   C   小数，得到结果后将结果转换为  16  进制（转换时如需使用字母请使用大写字母），就是敌军想要传递的密码。   
比如，敌军传递了下列两个数字串：   
99999999999999999999999999999999999999973 
99999999999999999999999999999999999999999 
大数   C   小数的差值   = 26  ，转换为  16  进制的结果为  1A  ，也就是  1A  就是敌军想要传递的密码。   
 请编程实现破译密码的过程。   
 

## Input Format

输入有  2  行，分别是  2  个不超过  200  位的整数，且已知  2  个整数的差是一个不超过  18  位的整数。   
 

## Output Format

输出按题意计算出的  16  进制数。   
 

```input1
99999999999999999999999999999999999999973
99999999999999999999999999999999999999999

```
```output1
1A
```
```input2
999999999
3847
```
```output2
3B9ABAF8
```
