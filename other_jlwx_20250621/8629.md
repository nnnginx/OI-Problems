## Description

<p>有个人的家族很大，辈分关系很混乱，请你帮整理一下这种关系。</p><p>给出每个人的孩子的信息。</p><p>输出一个序列，使得每个人的后辈都比那个人后列出。</p>

## Input Format

<p>第1行一个整数N（1&lt;=N&lt;=100），表示家族的人数。</p><p>接下来N行，第I行描述第I个人的儿子。</p><p>每行最后是0表示描述完毕。</p>

## Output Format

<p>输出一个序列，使得每个人的后辈都比那个人后列出。</p><p>如果有多解输出任意一解。</p>

```input1
5
0
4 5 1 0
1 0
5 3 0
3 0

```
```output1
2 4 5 3 1
```
