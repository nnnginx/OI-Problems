## 题目描述

<div>
 给定平面上n个点(x1,y1),...,(xn,yn)，找出2个半径相同的圆R1和R2，覆盖给定的n个点，且半径最小。
</div>
<div>
 <img src="https://s2.loli.net/2023/08/15/HbKDCp8sEvalf5L.png" width="740" height="224" alt="">
</div>
<div></div>
<div>
 设计一个算法，计算出所求最小覆盖双圆 R1 和 R2 的半径。
</div>
<div></div>
<p></p>

## 输入格式

<div>
 输入有多个测试实例。每个实例的第1行中给出正整数n，n＜1000，表示平面上有n个点。
</div>
<div>
 <div>
  接下来的n行中每行给出2个实数(x, y)，-100000≤x≤100000，-100000≤y≤100000。
 </div>
 <div>
  最后一行有一个0表示结束。
 </div>
 <div></div>
</div>
<p></p>

## 输出格式

<div>
 对于每组数据，输出最小的符合题意的圆的半径，保留两位小数。
</div>
<div>
 <div></div>
</div>
<div>
 <p></p>
</div>

```input1
3
0.00 0.00
1.00 0.00
0.00 4.00
10
0.00 0.00
0.00 3.00
1.00 6.00
2.00 2.00
3.00 5.00
5.00 3.00
6.00 3.00
9.00 5.00
10.00 5.00
11.00 3.00
0
```
```output1
0.50
3.05
```
## 数据范围与约定

<p>对于100%的数据，n<=1000，|xi|,|yi|<=100000，（T<=10）</p>
