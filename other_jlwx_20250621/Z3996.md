## 题目描述

<p>给出一个N*N的矩阵B和一个1*N的矩阵C。求出一个1*N的01矩阵A.使得</p>
<div>
 D=（A*B-C）*A^T最大。其中A^T为A的转置。输出D
</div>
<div></div>
<div></div>
<div></div>

## 输入格式

<div>
 第一行输入一个整数N，接下来N行输入B矩阵，第i行第J个数字代表Bij.
</div>
<div>
 接下来一行输入N个整数，代表矩阵C。矩阵B和矩阵C中每个数字都是不超过1000的非负整数。
</div>
<div></div>

## 输出格式

<p>输出最大的D</p>
<div></div>

```input1
3
1 2 1
3 1 0
1 2 3
2 3 7
```
```output1
2
```
## 数据范围与约定

<p> 1<=N<=500</p>
