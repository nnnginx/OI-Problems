## Description

<p class="MsoNormal" style="background:white;">
	 闰年的判断有两个条件，只需满足一个即可 <span style="font-family:&quot;color:windowtext;"> 
 
<p class="MsoNormal" style="background:white;">
	 1.  能够被  4  整除但不能被  100  整除的为闰年； <span style="font-family:&quot;color:windowtext;"> 
 
 
	<span style="font-family:&quot;background:white;">2.  能够被 <span style="font-family:&quot;background:white;">400  整除的为闰年； <span style="font-family:&quot;color:windowtext;background:white;"> 
 
 
	 比如： <span style="font-family:&quot;background:white;">1996  年就是闰年。 <span style="font-family:&quot;color:windowtext;background:white;"> 
 
 
	 请编程计算出，从 <span style="font-family:&quot;background:white;">1900  年 <span style="font-family:&quot;background:white;">~n  年之间有多少个闰年？ <span style="font-family:&quot;color:windowtext;background:white;"> 
 

## Input Format


一个整数 n （ n 是一个 >=1900 的四位整数）

## Output Format

一个整数，代表 1900 年 ~n 年之间闰年的有多少个。

```input1
2000
```
```output1
25
```
