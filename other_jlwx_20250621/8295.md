## Description

<p>一矩形阵列由数字$0$到$9$组成,数字$1$到$9$代表细胞,细胞的定义为沿细胞数字上下左右还是细胞数字则为同一细胞,求给定矩形阵列的细胞个数。如:<p><p>阵列<p>

## Input Format

<p>第一行为矩阵的行$n$和列$m$;<p><p>下面为一个$n×m$的矩阵。<p>

## Output Format

<p>细胞个数。<p>

```input1
4 10
0234500067
1034560500
2045600671
0000000089

```
```output1
4 10
0234500067
1034560500
2045600671
0000000089

```
