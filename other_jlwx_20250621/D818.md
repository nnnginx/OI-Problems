## Description

设有1g、2g、3g、5g、10g、20g的砝码各若干枚（其总重&lt;=1000），求这些砝码能称出的不同重量的个数 。

## Input Format

a1&nbsp;&nbsp; a2&nbsp;&nbsp; a3&nbsp;&nbsp; a4&nbsp;&nbsp; a5&nbsp;&nbsp; a6(表示1g砝码有a1个，2g砝码有a2个，......20g砝码有a6个，0≤每种砝码数量≤200)

## Output Format

整数N (N表示用这些砝码能称出的不同重量的个数，但不包括一个砝码也不用的情况)

```input1
1 1 0 0 0 0
```
```output1
3
```
