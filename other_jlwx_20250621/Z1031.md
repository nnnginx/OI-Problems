## 输入格式

<p>　　输入文件包含一行，欲加密的字符串。注意字符串的内容不一定是字母、数字，也可以是符号等。</p>

## 输出格式

<p>　　输出一行，为加密后的字符串。</p>

```input1
JSOI07
```
```output1
I0O7SJ
```
## 数据范围与约定

<p>对于100%的数据字符串的长度不超过100000。</p>
