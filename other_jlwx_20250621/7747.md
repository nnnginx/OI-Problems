## Description

小蓝最近在学习各种水果的英文单词，比如苹果的英文为apple，香蕉的英文为banana。请你编写一个程序，换行输出英文单词apple和banana。

## Input Format

无
注意：本题没有输入！ 

## Output Format

输出共两行，第一行为单词apple；
第二行为单词banana。
注意：
1.本题输出全部为小写！ 
2. 输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
（无）
```
```output1
apple
banana
```
