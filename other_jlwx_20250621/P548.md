## Description

<p>
	众所周知，字符在C语言中是使用ASCII码进行表示的。
</p>
<p>
	现要求输入3个字符（可以重复）后，按各字符的ASCII码从小到大的顺序输出这3个字符。
</p>

## Input Format

<p>
	第一行输入一个数N,表示有N组测试数据。
</p>
<p>
	后面的N行输入多组数据，每组输入数据都是占一行，有三个字符组成，之间无空格。
</p>

## Output Format

对于每组输入数据，输出一行，字符中间用一个空格分开。<br />
<br />

```input1
3
qwe
asd
zxc
```
```output1
e q w
a d s
c x z
```
## Hint

注意字符的操作，以及换行符。
