## 输入格式

<div>
 输入文件第一行一个正整数N。
</div>
<div>
 接下来每行一个操作。
</div>
<div>
  
</div>

## 输出格式

<div>
 对于每个2操作，输出一个对应的答案。
</div>
<div>
  
</div>

```input1
4
1 2 3 3
2 1 1 3 3
1 2 2 2
2 2 2 3 4
3
```
```output1
3
5
```
## 数据范围与约定

<div>
 1<=N<=500000,操作数不超过200000个，内存限制20M。
</div>
<br>
<div>
 对于100%的数据，操作1中的A不超过2000。
</div>
