## Description

在一个操场上一排地摆放着N堆石子。现要将石子有次序地合并成一堆。规定每次只能选相邻的2堆石子合并成新的一堆，并将新的一堆石子数记为该次合并的得分。
设计一个程序，计算出将N堆石子合并成一堆的最小得分。
 

## Input Format

第一行为一个正整数N (2≤N≤100)；
以下N行,每行一个正整数，小于10000，分别表示第i堆石子的个数（1≤i≤N）。
 

## Output Format

为一个正整数，即最小得分。

```input1
7
13
7
8
16
21
4
18
```
```output1
239
```
## Source

动态规划 区间动归
