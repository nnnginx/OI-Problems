## Description

Bessie像她的诸多姊妹一样，因为从Farmer&nbsp;YDS的草地吃了太多美味的草而长出了太多的赘肉。所以FY将她置于一个及其严格的节食计划之中。

她每天不能吃多过H&nbsp;(5&nbsp;≤&nbsp;H&nbsp;≤&nbsp;45,000)公斤的干草。&nbsp;Bessie只能吃一整捆干草；当她开始吃一捆干草的之后就再也停不下来了。她有一个完整的N&nbsp;(1&nbsp;≤&nbsp;N&nbsp;≤&nbsp;500)捆可以给她当作晚餐的干草的清单。她自然想要尽量吃到更多的干草。很自然地，每捆干草只能被吃一次（即使在列表中相同的重量可能出现2次，但是这表示的是两捆干草，其中每捆干草最多只能被吃掉一次）。

给定一个列表表示每捆干草的重量S_i&nbsp;(1&nbsp;≤&nbsp;S_i&nbsp;≤&nbsp;H),&nbsp;求Bessie不超过节食的限制的前提下可以吃掉多少干草（注意一旦她开始吃一捆干草就会把那一捆干草全部吃完）。

## Input Format

第一行: 两个由空格隔开的整数: H 和 N * 第2到第N+1行: 第i+1行是一个单独的整数，表示第i捆干草的重量S_i。

## Output Format

第一行: 一个单独的整数表示Bessie在限制范围内最多可以吃多少公斤的干草。

```input1
56 4
15
19
20
21
```
```output1
56
```
