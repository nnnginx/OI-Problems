## 题目描述

<div>
 给出 N 个点M 条边的无向图，定向得到有向无环图，使得最长路最短。
</div> 
<div>
 N ≤ 15, M ≤ 100
</div> 
<p></p>

## 输入格式

<div>
 第一行一个数M (1≤M≤100).
</div> 
<div>
 接下来M行，每行两个大写字母（L 到 Z），最多出线15个不同的大写字母。每行的两个大写字母不会相同
</div> 
<p></p>

## 输出格式

<div>
 第一行输出最长路最短的数值-1。
</div> 
<div></div> 
<p></p>

```input1
3
P Q
Q R
R P
```
```output1
1
```
