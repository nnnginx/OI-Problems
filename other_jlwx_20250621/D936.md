## Description

 
	晶晶同学非常喜欢方形，她希望打印出来的字符串也是方形的。老师给了晶晶同学一个字符串"ACM"，晶晶同学突发奇想，如果任意给定义一个整数n，能不能打印出由这个字符串组成的正方形字符串呢？要求是每行要使用n个给定的字符串。
 
 
	请你编程实现一下。
 

## Input Format

输入的第一行是一个正整数N（N&lt;=30），表示一共有N组数据；
接下来有N行，每行包含一个正整数M（M&lt;=100），表示一行内要使用M个老师给定的字符串，构成一个正方形矩阵。
 

## Output Format

输出指定的字符串方形矩阵。

```input1
2
1
2
```
```output1
ACM
ACM
ACM
ACMACM
ACMACM
ACMACM
ACMACM
ACMACM
ACMACM
```
## Source

字符串
