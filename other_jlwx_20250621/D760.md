## Description

平面上有两个矩形，它们的边平行于直角坐标系的 X 轴或 Y 轴。对于每个矩形，我们给出它的一对相对顶点的坐标，请你编程算出两个矩形的交的面积。    
 

## Input Format

 输入仅包含两行，每行描述一个矩形。  
  在每行中，给出矩形的一对相对顶点的坐标，每个点的坐标都用两个绝对值不超过 10^7^ 的实数表示。    
 

## Output Format

输出仅包含一个实数，为交的面积，保留到小数后两位。    
 

```input1
1 1 3 3
2 2 4 4
```
```output1
1.00
```
