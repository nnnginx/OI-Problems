## Description

打印杨辉三角形的前$n$($2≤n≤20$)行。杨辉三角形如下图：

当$n=5$时
```
        1
      1   1
    1   2   1
  1   3   3   1
1   4   6   4   1
```
输出：
```
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```

## Input Format

输入行数$n$。

## Output Format

输出如题述三角形。$n$行，每行各数之间用一个空格隔开。

```input1
5
```
```output1
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```
