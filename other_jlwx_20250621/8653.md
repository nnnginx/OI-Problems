## Description

<p>像《非诚勿扰》这样的电视节目一直很受欢迎。为了满足单身男生的需要，TJUT自行举办了一场。演出在小礼堂里举行，所以吸引了很多男孩和女孩。</p><p>现在有$n$个男孩报名。开始时，n个男孩站成一排，一个接一个地走上舞台。每个男孩都有一个不满意值$D$，如果男孩是第$k$个上台的，他的不满意值就会是$(k-1)*D$，因为他必须等待$(k-1)$人。</p><p>幸运的是，在小厅里有一个暗室，导演可以把男孩暂时放在暗室里，让他身后的男孩先上台。因为暗室很窄，先进入暗室的男孩必须最后离开。导演想要改变男生在暗室的顺序，这样总的不满意值就最小了。你能帮助他吗?</p>

## Input Format

<p>第一行是$n$ ($0 &lt; n \le 100$)</p><p>接下来包含$n$个整数$ D1,D2,D3,... ,Dn$表示男生<span style="color: rgb(51, 51, 51);">不满意值</span>($0 \le Di \le 100$)</p>

## Output Format

<p>输出包含一个数表示最小的不满意值总和。<br /></p>

```input1
5 
1 2 3 4 5 
```
```output1
20
```
```input2
3
1 2 3
```
```output2
4
```
## Hint

<p>样例2解释：先让1号和2号进暗室，3号先上台 tot_D=0，按照规则2号上台 tot_D+=2*1 =2, 最后1号上台，tot_m+=1*2=4;<br /></p>
