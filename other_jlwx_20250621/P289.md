## Description

  已知矩阵的大小定义为矩阵中所有元素的和。给定一个矩阵，你的任务是找到最大的非空(大小至少是1×11×1)子矩阵。 
![最大矩阵.jpg](http://101.34.251.72/api/public/img/5f9ec49e83dd4b46a8720fff9e6a3107.jpg)

## Input Format

 输入是一个N×N的矩阵。输入的第一行给出N(0<N≤100)。
再后面的若干行中，依次(首先从左到右给出第一行的N个整数，再从左到右给出第二行的N个整数……)给出矩阵中的N2个整数，整数之间由空白字符分隔(空格或者空行)。
已知矩阵中整数的范围都在[−127,127]。    

## Output Format

 输出最大子矩阵的大小。 

```input1
  0  -2   -7   0

  9   2   -6   2 

-4    1   -4   1

-1    8    0  -2

```
```output1
  9   2

-4   1

-1   8

```
