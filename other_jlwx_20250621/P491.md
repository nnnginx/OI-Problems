## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">某次科研调查时得到了</span>n<span style="color: rgba(0, 0, 0, 0.87);">个自然数，每个数均不超过</span><span style="color: rgba(0, 0, 0, 0.87);">1500000000</span><span style="color: rgba(0, 0, 0, 0.87);">（</span><span style="color: rgba(0, 0, 0, 0.87);">1.5</span><span style="color: rgba(0, 0, 0, 0.87);">*</span><span style="color: rgba(0, 0, 0, 0.87);">109</span><span style="color: rgba(0, 0, 0, 0.87);">）。已知不相同的数不超过</span><span style="color: rgba(0, 0, 0, 0.87);">10000</span><span style="color: rgba(0, 0, 0, 0.87);">个，现在需要统计这些自然数各自出现的次数，并按照自然数从小到大的顺序输出统计结果。</span><br /></p>

## Input Format

<p>输入包含n+1行：</p><p>第1行是整数n，表示自然数的个数。</p><p>第2~n+1行每行一个自然数。</p>

## Output Format

<p>输出包含m行（m为n个自然数中不相同数的个数），按照自然数从小到大的顺序输出。每行输出两个整数，分别是自然数和该数出现的次数，其间用一个空格隔开。<br /></p>

```input1
8
2
4
2
4
5
100
2
100
```
```output1
2 3
4 2
5 1
100 2
```
## Hint

<p>【数据范围】</p><p style="margin-left: 40px;">40%的数据满足：1&lt;=n&lt;=1000</p><p style="margin-left: 40px;">80%的数据满足：1&lt;=n&lt;=50000</p><p style="margin-left: 40px;">100%的数据满足：1&lt;=n&lt;=200000，每个数均不超过1 500 000 000（1.5*109）</p><p><br /></p><p><span style="color: rgb(51, 51, 51);">
