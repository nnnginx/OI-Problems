## Description

根据输入的正整数n，用“*”画出如图所示的“T”型图案。图案的第一行有2×nC1个“*”；接下来有n行，每行都只有一个“*”，并且和第一行水平居中对齐。
例如n = 3时，图案为：
```
*****
  *
  *
  *
```


n = 4时，图案为：
```
*******
   *
   *
   *
   *
```



## Input Format

输入1个正整数n（2 <= n <= 20）。

## Output Format

根据题目描述，输出对应的图案。

```input1
3
```
```output1
*****
  *
  *
  *
```
```input2
4
```
```output2
*******
   *
   *
   *
   *
```
## Hint

1.图案的第一行前面不能有多余的空行；
2.图案的第一行左右相邻的两个星号之间没有空格;
3.图案的最左侧从第一列开始输出，左边不能有多余的空格。
