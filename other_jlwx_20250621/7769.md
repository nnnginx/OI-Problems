## Description

小蓝的班进行比赛跳绳。已知班里共有学生n名，给定学生的跳绳成绩（1分钟跳绳的个数），请将这n名学生的跳绳成绩从高到低排序后输出。

## Input Format

共2行；
第1行是一个正整数 n（1 <= n <= 100）；
第2行有 n 个正整数（小于1000），相邻两数之间用空格隔开。


## Output Format

输出1行正整数，为n位学生的跳绳成绩从高到低排序后的序列，两数之间以一个英文逗号分隔。
注意：不得输出多余的逗号。


```input1
5
112 72 212 99 157

```
```output1
212,157,112,99,72
```
