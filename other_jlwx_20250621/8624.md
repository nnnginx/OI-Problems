## Description

<p><span style="color: rgba(0, 0, 0, 0.75);">战争时期，前线有 n 个哨所，每个哨所可能会与其他若干个哨所之间有通信联系。信使 负责在哨所之间传递信息，当然，这是要花费一定时间的（以天为单位）。指挥部设在第一 个哨所。当指挥部下达一个命令后，指挥部就派出若干个信使向与指挥部相连的哨所送信。 当一个哨所接到信后，这个哨所内的信使们也以同样的方式向其他哨所送信。直至所有 n 个 哨所全部接到命令后，送信才算成功。 因为准备充足，每个哨所内都安排了足够的信使（如果一个哨所与其他 k 个哨所有通信 联系的话，这个哨所内至少会配备 k 个信使）。 现在总指挥请你编一个程序，计算出完成整个送信过程最短需要多少时间</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.75);">输入文件，第 1 行有两个整数 n 和 m，中间用 1 个空格隔开，分别表示有 n 个哨所和 m 条通信线路。1&lt;=n&lt;=100。 第 2 至 m+1 行：每行三个整数 i、j、k，中间用 1 个空格隔开，表示第 i 个和第 j 个哨 所之间存在通信线路，且这条线路要花费 k 天。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.75);">输出文件，仅一个整数，表示完成整个送信过程的最短时间。如果不是所有的哨所都能 收到信，就输出-1。</span><br /></p>

```input1
4 4 
1 2 4 
2 3 7 
2 4 1 
3 4 6
```
```output1
11
```
