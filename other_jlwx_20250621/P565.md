## Description

我们称使用CodesOnline的用户为COIer，OJ对于新加入的COIer都会发布一条关于OJ 的使用的说明，但是由于小明这名COIer是B国公民，看不明白A国文字，所以希望你能通过编程的方式告诉小明OJ的使用方法
<br>
A国文字与B国文字的区别为：
&nbsp; &nbsp;  A国文字全部为小写字母、B国文字全部为大写字母

## Input Format

输入仅一行，
输入为一行字符串（字符个数不超过100，字符串输入仅有字母和空格）

## Output Format

输出有两行，
第一行直接输出转换后的B国文字；
但是由于小明阅读的顺序为从右至左，所以你输出时在第二行中输出转换后且倒序的B国文字

```input1
if you need to use codesonline first you need to sign up a account
```
```output1
IF YOU NEED TO USE CODESONLINE FIRST YOU NEED TO SIGN UP A ACCOUNT
ACCOUNT A UP SIGN TO NEED YOU FIRST CODESONLINE USE TO NEED YOU IF
```
## Hint

单词倒序输出，并不是字母倒序输出
