## Description

有一块 n * n （ n&gt;=5 ，且 n 是奇数）的红玫瑰花圃，由 n * n 个小正方形花圃组成，现要求在花圃中最中间的一行、最中间的一列以及 4 个顶点处种植粉色玫瑰，请问粉玫瑰占地面积占整个玫瑰花圃总面积的百分比是多少？
 
如 n=5 ，花圃形状如下，粉玫瑰占地百分比 =&nbsp; 13/25 =&nbsp; 52.0%  
<img src="http://oj.czos.cn/upload/image/20180623/20180623112031_16075.jpg" alt="" />
如 n=7 ，花圃形状为如下，粉玫瑰占地百分比 = 17 / 49 = 34.7%  
<img src="http://oj.czos.cn/upload/image/20180623/20180623112021_66159.jpg" alt="" />
 

## Input Format

一个整数n （  n&gt;=5 ，且 n 是奇数） 

## Output Format

粉玫瑰占总玫瑰园的百分比，结果保留1位小数，显示为百分比的形式

```input1
5
```
```output1
52.0%
```
