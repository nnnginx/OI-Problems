<p><span style="color: rgba(0, 0, 0, 0.87);">有1×n的一个长方形，用一个1×1、1×2和1×3的骨牌铺满方格。例如当n=3时为1×3的方格。此时用1×1、1×2和1×3的骨牌铺满方格，共有四种铺法。如下图：

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一个正整数n，表示有1×n的一个长方形，</span><span style="color: rgba(0, 0, 0, 0.87);">0&lt;n&lt;=30</span><span style="color: rgba(0, 0, 0, 0.87);">。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一个数表示共有几种铺法。</span><br /></p>

```input1
4
```
```output1
7
```
