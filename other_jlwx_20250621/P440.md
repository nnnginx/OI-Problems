## Description

<p>如果二叉树的左右子树的结构是对称的，即两棵子树皆为空，或者皆不空，则称该二叉树是对称的。编程判断给定的二叉树是否对称.<p><p>例：如下图中的二叉树T1是对称的，T2是不对称的。<p><p>![](pic/1368.gif)<p><p>二叉树用顺序结构给出，若读到#则为空，二叉树T1=ABCDE，T2=ABCD#E，如果二叉树是对称的，输出“Yes”,反之输出“No”。<p>

## Input Format

<p>二叉树用顺序结构给出，若读到#则为空。<p>

## Output Format

<p>如果二叉树是对称的，输出“Yes”,反之输出“No”。<p>

```input1
ABCDE

```
```output1
Yes

```
