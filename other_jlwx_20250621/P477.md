## Description

<p>小明正在研究趣味数学。如果一个正整数，它的二进制形式是个回文数，小明就会将它称为“精妙数”。如 33，二进制形式为 100001，则 33 是一个精妙数。现在小明找到了 T 个数字，他想让你判断这些数是不是精妙数。是精妙数回答“Yes”，否则回答“No”。<br /></p>

## Input Format

<p>第一行输入一个数 T，表示数字个数； 之后 T 行，每行输入一个数，表示每个数字。<br /></p>

## Output Format

<p>输出 T 行，每行一个字符串&quot;Yes&quot;或者&quot;No&quot;，以空格隔开。<br /></p>

```input1
3
44
33
22
```
```output1
No
Yes
No
```
## Hint

<p><b>【数据范围】</b></p><p>对于 100%的数据，1≤T≤100000，0≤每个数字≤10^9；</p>
