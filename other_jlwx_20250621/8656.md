## Description

<p>输入一个长度为n的数组$a$，$a$包括$\frac{(n+1)*n}{2}$个区间。每个区间所有数的和，被称为区间和，求所有$\frac{ (n+1)*n }{2}$个区间和的和。由于数值较大，输出mod 1e9+7的结果。</p><p>例如：</p><p>3个数1 2 3，共有6个子区间，包含的数字如下：{1} {2} {3} {1 2} {2 3} {1 2 3}，这些区间求和为1 2 3 3 5 6，这6个数字再求和为20.<br /></p>

## Input Format

<p>第一行一个整数n，表示数组长度接下来n行，每行一个整数ai，表示数组的内容。$(0&lt;=ai&lt;=50000)$<br /></p>

## Output Format

<p>输出答案mod 1e9+7<br /></p>

```input1
3
1
2
3
```
```output1
20
```
## Hint

<p>对于30%的数据，1&lt;=n&lt;=100 ；</p><p>对于50%的数据， 1&lt;=n&lt;=1000；</p><p>对于100%的数据， 1&lt;=n&lt;=100000。<br /></p>
