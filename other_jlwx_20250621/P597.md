## Description

已知一个已经从小到大排序的数组，这个数组的一个平台（Plateau）就是连续的一串值相同的元素，并且这一串元素不能再延伸。例如，在 1，2，2，3，3，3，4，5，5，6中1，2-2，3-3-3，4，5-5，6都是平台。试编写一个程序，接收一个数组，把这个数组最长的平台找出 来。在上面的例子中3-3-3就是最长的平台。

## Input Format

第一行有一个整数n，为数组元素的个数。第二行有n个整数，整数之间以一个空格分开。

## Output Format

输出最长平台的长度。

```input1
10
1 2 2 3 3 3 4 5 5 6
```
```output1
3
```
