## Description

<p>给定一个整数序列a1、a2…an，求这个序列中的一个连续子序列，使得这个序列中的数的和最大。<br /></p>

## Input Format

<p>第一行，一个整数n。</p><p>第二行，n个整数，a1、a2...an。</p>

## Output Format

<p>一行，一个整数，表示你所求的最大的和。<br /></p>

```input1
8
3 -6 -8 4 7 -5 5 2
```
```output1
13
```
## Hint

<p>数据范围：N&lt;=500000，序列中所有数的绝对值均不超过1000。<br /></p>
