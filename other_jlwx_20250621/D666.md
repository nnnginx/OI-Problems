## Description

 淘淘参加了信息学奥赛初级班培训，最后需要通过考试才能决定是否有资格进入中级班。  
 &nbsp;&nbsp; 经过 3 个小时紧张激烈的考试，淘淘做对了 p 个题目，但还是不清楚到底能不能进入中级班，最后他问了老师进入中级班的合格线需要做对 q 题。  
 &nbsp;&nbsp; 你知道淘淘有没有资格进入中级班学习呢？ 

## Input Format

 两个整数 p 和 q，用一个空格隔开，表示淘淘做对了 p 个题目和合格线要求做对 q 题。 

## Output Format

 如果能够进入中级班，就输出“Yes”，否则输出“No”。 

```input1
6 5
```
```output1
Yes
```
