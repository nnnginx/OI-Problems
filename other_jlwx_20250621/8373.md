## Description

<p>小红的班级里有 n 名爱打篮球的同学。已知这 n 名同学的身高。现在体育老师要从他们当中选择 m 名同学组成一支篮球队， 老师希望蓝球队的平均身高不小于 h。</p><p>问：一共有多少种不同选择方法使得 m 名篮球队员的平均身高不小于 h？</p>

## Input Format

<p>文件的第一行是三个整数 n, m，h（ 1≤m≤n≤30, 100≤h≤200）。</p><p>第二行是 n 个用空格分开的整数 a1，……， an，依次表示 n 名同学的身高。</p><p>ai 是不超过 200 的正整数。</p>

## Output Format

<p>文件共一行，是一个整数，表示 m 名队员平均身高不小于 h 的方法数。<br /></p>

```input1
3 2 145
148 145 144
```
```output1
2
```
```input2
6 3 153
158 155 152 144 149 161
```
```output2
11
```
## Hint

<p>【样例解释】</p><p>样例 1 中，共有 3 种不同选择方法，分别为选择（ 148,144） ， （ 148,145） ，（ 145,144）， 平均身高依次为 146、 146.5、 144.5。 共有两种方法符合要求。</p><p>样例 2 中，共有 20 种不同选择方法，其中有 11 种方法符合要求。</p>
