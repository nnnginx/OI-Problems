## Description

<p>小伟暑假期间到大草原上旅游，在一块石头上发现一块有趣的碑文。碑文似乎是一个神秘古老的语言，只包含三个大写字母C、O和W。尽管小伟看不懂，但令他高兴的是，C、O、W的顺序形式构成了一句他最喜欢的奶牛单词“COW”。现在，他想知道有多少次COW出现在文本中。 如果COW内穿插了其他字符，只要COW字符出现在正确的顺序，小伟也不介意。甚至，他也不介意出现不同的COW共享一些字母。例如COWO出现了1次COW，CCOW算出现了2次COW，CCOOWW算出现了8次COW。<br /></p>

## Input Format

<p>第一行为一个整数N。第二行为N个字符的一个字符串，每个字符串是一个C、O或W。<br /></p>

## Output Format

<p>输出COW作为输入字符串的字符出现的次数（不一定是连续的）提示：答案会很大，建议用64位整数（long long）<br /></p>

```input1
6
COOWWW
```
```output1
6
```
## Hint

<p>对于50%的数据：N&lt;=60;</p><p>对于100%的数据：N&lt;=10^5<br /></p>
