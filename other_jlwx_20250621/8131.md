## Description

<p>对于1 位二进制变量定义两种运算：<br /></p><p><span style="color: rgb(51, 51, 51);"><img src="https://cdn.luogu.com.cn/upload/pic/99.png" /></span><br /></p><p>运算的优先级是：</p><p>1. 先计算括号内的，再计算括号外的。</p><p>2. “× ”运算优先于“”运算，即计算表达式时，先计算× 运算，再计算运算。例如：计算表达式AB × C时，先 计算 B × C，其结果再与 A 做运算。</p><p>现给定一个未完成的表达式，例如\_+(\_\*\_)，请你在横线处填入数字$0 $或者$1$ ，请问有多少种填法可以使得表达式的值为$0 $。</p>

## Input Format

<p>共 2 行。</p><p>第1 行为一个整数 $L$，表示给定的表达式中除去横线外的运算符和括号的个数。</p><p>第2 行为一个字符串包含 $L$ 个字符，其中只包含’（’、’）’、’+’、’\*’这$4$ 种字符，其中’（’、’）’是 左右括号，’+’、’\*’分别表示前面定义的运算符“”和“×”。这行字符按顺序给出了给定表达式中除去变量外的运算符和括号。</p>

## Output Format

<p>共1 行。包含一个整数，即所有的方案数。注意：这个数可能会很大，请输出方案数对$10007 $取模后的结果。<br /></p>

```input1
4
+(*)
```
```output1
3
```
## Hint

<p>【输入输出样例说明】</p><p>给定的表达式包括横线字符之后为：\_+(\_\*\_)</p><p>在横线位置填入(0 、0 、0) 、(0 、1 、0) 、(0 、0 、1) 时，表达式的值均为0 ，所以共有3种填法。</p><p>【数据范围】</p><p>对于$20\%$ 的数据有$ 0 ≤ L ≤ 10$。</p><p>对于$50\%$ 的数据有 $0 ≤ L ≤ 1,000$。</p><p>对于$70\%$ 的数据有$ 0 ≤ L ≤ 10,000$ 。</p><p>对于$100\%$的数据有$ 0 ≤ L ≤ 100,000$。</p><p>对于$50\%$ 的数据输入表达式中不含括号。</p>
