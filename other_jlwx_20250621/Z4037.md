<div>

  +f(1+23)+f(12+3)+f(123)。已知字符串和m后求答案对998244353（7×17×223+1，一个质数）取模后的值。

## 输入格式

<p>第一行输入一个字符串，第二行输入m</p> 
<div></div>

## 输出格式

<p>仅输出一个数表示答案</p> 
<div></div>

```input1
123
3
```
```output1
394608467
```
## 数据范围与约定

<p> 对于100%的数据，字符串长度不超过500，m<=5</p>
