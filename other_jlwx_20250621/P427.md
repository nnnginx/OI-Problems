## Description

<p style="margin-left: 0in;">小谢虽然有很多溶液，但是还是没有办法配成想要的溶液，因为万一倒错了就没有办法挽回了。因此，小谢到网上下载了一个溶液配置模拟器。模拟器在计算机中构造一种虚拟溶液，然后可以虚拟地向当前虚拟溶液中加入一定浓度、一定体积的这种溶液，模拟器会快速地算出倒入后虚拟溶液的浓度和体积。当然，如果倒错了可以撤销。</p><p style="margin-left: 0in;">模拟器的使用步骤如下：</p><p style="margin-left: 0in;">1）为模拟器设置一个初始体积和浓度$V_0、C_0\%$。</p><p style="margin-left: 0in;">2）进行一系列操作，模拟器支持两种操作：</p><p style="margin-left: 0in;">$P(v,c)$操作：表示向当前的虚拟溶液中加入体积为 $v$ 浓度为 $c$ 的溶液；</p><p>Z操作：撤销上一步的P操作。</p>

## Input Format

<p>第一行两个整数，表示$V_0$和$C_0$，$0≤C_0≤100$；</p><p>第二行一个整数$n$，表示操作数，$n≤10000$；</p><p>接下来$n$行，每行一条操作，格式为：$P, v, c$或$Z$。</p><p>当只剩初始溶液的时候，再撤销就没有用了。</p><p>任意时刻质量不会超过$2^{31}-1$。</p>

## Output Format

<p>$n$行，每行两个数$V_i$，$C_i$，其中$V_i$为整数，$C_i$为实数（保留5位小数）。</p><p>其中，第i行表示第 $i$ 次操作以后的溶液体积和浓度。</p>

```input1
100 100
2
P 100 0
Z

```
```output1
200 50.00000
100 100.00000

```
