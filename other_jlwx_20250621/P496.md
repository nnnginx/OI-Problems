## Description

<p>有 2×n的一个长方形方格，用一个1*2的骨牌铺满方格。例如n=3时铺法如下图所示：

![0062.jpg](/api/public/img/4dc7f4ea7ab049e7a9451a042e3ebcbb.jpg)

</p><p>编写一个程序，试对给出的任意一个n(n&gt;0), 输出铺法总数。</p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一个正整数n，0&lt;n&lt;=45。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">共n行，从1~n顺序输出每种情形的骨牌铺法总数，参考样例输出。</span><br /></p>

```input1
4
```
```output1
x[1]=1
x[2]=2
x[3]=3
x[4]=5
```
