## Description

根据多边形内角和定理，正多边形内角和等于：（ n  －  2 ）× 180 ° (n 大于等于 3 且 n 为整数）（如下图所示是三角形、四边形、五边形、六边形的形状）  

请根据正多边形的边数，计算该正多边形每个内角的度数。（结果保留 1 位小数）
<img src="http://oj.czos.cn/upload/image/20180622/20180622223442_10886.jpg" alt="" />
 

## Input Format

正多边形的边数n（n&gt;=3 且 n &lt;= 10）

## Output Format

该正n边形每个内角的度数。

```input1
3
```
```output1
60.0
```
