## Description

## Input Format

<span style="font-size:14px;">　　一行：一个整数，表示电池数量。</span>

## Output Format

<span style="font-size:14px;">　　一行：一个实型数据，表示购买电池需用多少钱，小数点后保留2位。</span>

```input1
15
```
```output1
30.00
```
