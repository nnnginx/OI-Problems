## Description

小宇今年的信息学竞赛拿到好成绩，妈妈为了奖励他，想带他去旅游，但前提是小宇要讲出不少于10个我国旅游景点的英文名字，并按字典排好序给妈妈听。

聪明的你，能帮小宇完成这个easy的任务，让小宇顺利PASS，跟妈妈去旅游吗?
 

## Input Format

共n行，以end结束。（最多不超过1000行）

## Output Format

排好序的旅游景点英文名字。

```input1
HongKong
TsingTao
Macau
Canton
Amoy
KiangSu
Ningpo
Foochow
chu-hai
TienTisn
end

```
```output1
Amoy
Canton
Foochow
HongKong
KiangSu
Macau
Ningpo
TienTisn
TsingTao
chu-hai

```
## Source

字符串
