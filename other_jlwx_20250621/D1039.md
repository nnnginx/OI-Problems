## Description

在5*5格的棋盘上，有一只中国象棋的马，从（1,1）点出发，按日字跳马，它可以朝8个方向跳，但不允许出界或跳到已跳过的格子上，要求其跳遍整个棋盘。<span style="line-height:1.5;">输出跳遍整个棋盘的不同方案总数。
	<span style="line-height:1.5;">   
 
 
	&nbsp; &nbsp; 例如其中的一种跳法为：
 
 
	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1&nbsp;&nbsp; &nbsp;16&nbsp;&nbsp; 21&nbsp;&nbsp; 10&nbsp;&nbsp; 25
 
 
	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;20&nbsp; 11&nbsp;&nbsp; 24&nbsp;&nbsp; 15&nbsp;&nbsp; &nbsp;22
 
 
	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;17&nbsp; 2&nbsp;&nbsp;&nbsp; &nbsp;19&nbsp;&nbsp; 6&nbsp;&nbsp;&nbsp; &nbsp;9
 
 
	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;12&nbsp; 7&nbsp;&nbsp;&nbsp; &nbsp;4&nbsp;&nbsp;&nbsp; &nbsp;23&nbsp;&nbsp; 14
 
 
	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3&nbsp;&nbsp; 18&nbsp;&nbsp; &nbsp;13&nbsp;&nbsp; 8&nbsp;&nbsp;&nbsp; &nbsp;5
 

## Input Format

无

## Output Format

跳遍整个棋盘的不同方案总数。

```input1
无
```
```output1
无
```
