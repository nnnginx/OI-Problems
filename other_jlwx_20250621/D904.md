## Description

请从键盘读入一个英文字符串（仅含小写字母，最多是长度为100的字符串），请问这个英文字符串中是否含有重复的字符？
 
如果有重复的字符请输出单词中第一个出现的重复的字符是哪个字符；如果不存在重复的字符，请输出字符串的长度。
 
比如：单词"james"就是不含重复字符的单词，此时输出5；
 
 
再比如：单词"hibirdboy"含有重复的字符，其中第一个出现的重复字符是字符'i'，而字符串"abba"中第一个出现的重复字符就是字符'a'。
 

## Input Format

输入一个长度不超过100的仅含小写字母的英文字符串。

## Output Format

按题意输出第一个出现的重复的字符，或者输出字符串的长度。

```input1
hibirdboy
```
```output1
i
```
