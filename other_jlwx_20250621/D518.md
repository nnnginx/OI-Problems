## Description

热的夏天来的可真快，小花猫和编程兔决定去买一个又大又甜的西瓜。可是小花和编程兔是两只非常奇怪的动物，都是偶数的爱好者，它们希望把西瓜切成两半后，每一部分的重量都是 2 的倍数公斤（大于 0）。当然有编程兔在，它们很快就决定了买哪个瓜。小朋友你知道要不要买这个瓜吗？    
 

## Input Format

第一行只有一个正整数，表示西瓜的重量 w（单位是公斤）。

## Output Format

如果能达到要求，就输出 YES，否则就输出 NO。(注意区分大小写)&nbsp;

```input1
8
```
```output1
YES
```
## Hint

8 可以分成 2 和 6，或者 4 和 4。
数据规模：    
对于100%的数据，1&lt;=w&lt;=100。
