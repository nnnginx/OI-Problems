## Description

国王给骑士发金币，奖励他们的忠诚和英勇。第一天国王发的金币是一个四位数，第二天发的金币是这个四位数各个位的和，请问骑士一共拿到多少金币？  

比如：假设第 1 天骑士拿到了 1263 个金币，那么第 2 天就拿到了 12 个金币（ 1+2+6+3=12 ），那么他一共拿到了 1275 个金币（ 1263+12=1275 ）。  
 

## Input Format

一个四位整数 ，代表国王第1天发给骑士的金币数量。 
 

## Output Format

一个整数，代表骑士一共获得了多少金币。  
 

```input1
1263
```
```output1
1275
```
