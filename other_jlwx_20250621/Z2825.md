<div style="margin: 0cm -3.75pt 0pt 17.9pt; line-height: 13.5pt" align="left">
 <span style="font-size: 12pt; color: black">一个正整数，在时间</span><span style="font-size: 12pt; color: black">T</span><span style="font-size: 12pt; color: black">分钟内可收集到的最多资源总数。</span><span style="font-size: 12pt; color: black">（注：假设从某个交叉点出发，沿着南北方向或东西方向行走到下一个街道的交</span><span style="font-size: 12pt; color: black">叉点需要时间固定为</span><span style="font-size: 12pt; color: black">1</span><span style="font-size: 12pt; color: black">分钟；每个队员必须从（</span><span style="font-size: 12pt; color: black">0,0</span><span style="font-size: 12pt; color: black">）点出发，结束时不要求回</span><span style="font-size: 12pt; color: black">到出发点。如果在时间结束时恰好到达某一处有资源的坐标点，则该资源计入收</span><span style="font-size: 12pt; color: black">集量。）</span>
