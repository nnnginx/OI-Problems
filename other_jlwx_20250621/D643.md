## Description

学校运动会的跳水比赛已经结束，裁判统计出了选手们的成绩，并根据大家的成绩划分好了金牌、银牌和铜牌的获奖分数线。  
 
裁判规定，总分 100 分；
获得 92.8 分及 92.8 分以上的选手获得金牌；
在 92.8 分以下但在 85.6 分及 85.6 分以上的选手获得银牌；
在 85.6 分以下，但在 82.1 分及 82.1 分以上的选手获得铜牌；  
分数在 82.1 分以下的选手不获奖。  
 
请从键盘读入一名选手的分数，判断他应该获得哪个奖项？  
 
如获得金牌，请输出“ jin ”，银牌请输出“ yin ”，铜牌请输出“ tong ”，没有获得奖项请输出“ sorry ”。  
 

## Input Format

某位选手的分数 。 
 

## Output Format

如获奖请输出该选手获得的奖项，如不获奖，请输出“ sorry ”。  
 

```input1
92.8
```
```output1
jin
```
