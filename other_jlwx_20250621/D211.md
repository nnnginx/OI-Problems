## Description

<img src="http://43.142.138.134:80/admin/../../upload/43.142.138.134/20220812//1225_1_20220812202308_30127.jpg" alt="" /> 

汉诺塔的问题大家都已经很熟悉了，有三个柱子，每个柱子上有一些大小不一的金片，要把金片从A柱移动到C柱，可以借助B柱，请问n个金片的情况下，需要最少移动多少次？

## Input Format

输入一个整数n代表金片的数量（n&lt;=20）

## Output Format

一个整数，代表n个金片的移动次数

```input1
3
```
```output1
7
```
