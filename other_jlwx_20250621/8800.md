## Description

<p>总公司拥有高效设备M台，准备分给下属的N个分公司。各分公司若获得这些设备，可以为国家提供一定的盈利。问：如何分配这M台设备才能使国家得到的盈利最大？求出最大盈利值。其中M≤15，N≤10。分配原则：每个公司有权获得任意数目的设备，但总台数不超过设备数M。</p><p>输入数据文件格式为：第一行有两个数，第一个数是分公司数N，第二个数是设备台数M。</p><p>接下来是一个N*M的矩阵，表明了第 I个公司分配 J台机器的盈利。</p>

## Input Format

<p>第一行有两个数，第一个数是分公司数N，第二个数是设备台数M</p><p>接下来是一个N*M的矩阵，表明了第i个公司分配j台机器的盈利</p>

## Output Format

<p>第一行输出最大盈利值</p><p>接下来N行，每行两个数，表示公司编号和该公司获得设备的台数</p>

```input1
3 3
30 40 50
20 30 50
20 25 30
```
```output1
70
1 1
2 1
3 1
```
