## Description

小刚去文具店买铅笔，文具店有3种不同价格的铅笔，小刚不想买最便宜的，也不想买最贵的，他想买一个中间价位的铅笔，那么请问小刚有n元钱的情况下，最多能买到多少支铅笔？老板应该找小刚多少钱？

## Input Format

四个整数，用空格隔开，第一个整数代表小刚有的n元钱，后三个整数代表三种不同价位的铅笔的单价。

## Output Format

两个整数，用空格隔开，第一个整数代表小刚最多可以买到的铅笔的总数，第二个整数代表老板应该找给小刚多少钱？

```input1
10 2 3 4
```
```output1
3 1
```
