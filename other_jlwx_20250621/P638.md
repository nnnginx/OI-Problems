## Description

<p>有1个约数的最小数为1（1），有两个约数的最小数为2（1，2）……</p><p>有n个约数的最小数为Sn</p><p>S1=1 （1）</p><p>S2=2 （1 2）</p><p>S3=4 （1 2 4）</p><p>S4=6 （1 2 3 6）</p><p>求S100（即约数的数量恰好是100的最小正整数）</p>

## Input Format

<p>无</p>

## Output Format

<p>输出约数的数量恰好是100的最小正整数。</p>

```input1
无
```
```output1
一个整数（题目答案）
```
