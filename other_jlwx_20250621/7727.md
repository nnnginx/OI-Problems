## Description

今年大学生运动会，小蓝要根据每个学校获得的奖牌数量，按奖牌总数从多到少进行排序，整理出奖牌榜；如果有相同总奖牌数，则学校编号小的排在前面。


## Input Format

输入：第1行是1个正整数，表示参加运动会的学校数目n（1≤n≤2000）。
接下来的n行，每行包含两个正整数，分别为每个学校的编号（不大于999999）和总奖牌数（不大于600）, 中间用单个空格隔开。


## Output Format

输出：n行，是将学校按总奖牌数从高到低的顺序进行排序的结果，每行包含学校编号和总奖牌数，之间用一个空格隔开。


```input1
3
1 40
2 53
3 53

```
```output1
2 53
3 53
1 40

```
