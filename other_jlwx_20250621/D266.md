## Description

输入一个不超过100000位的十六进制数，请转换成八进制数。 注：十六进制数中，字母0-9还对应表示数字0-9，字母“A”（大写）表示10，“B”表示11，…，“F”表示15。
比如：十六进制A10B表示的10进制数是：
10x16^3^ + 1x16^2^ + 0x16^1 +11x16^0^ =41227。
转换成八进制数是：120413，因为1x8^5^+2x8^4^+0x8^3^+4x8^2^+1x8^1^+3x8^0^=41227。

## Input Format

一个十六进制数，没有前导0。

## Output Format

一个八进制数，没有前导0。

```input1
123ABC
```
```output1
4435274
```
