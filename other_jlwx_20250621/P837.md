## Description

<p>有一个箱子容量为V（正整数，0≤v≤20000），同时有n个物品（0< n ≤30），每个物品有一个体积（正整数）。<p><p>要求n个物品中，任取若干个装入箱内，使箱子的剩余空间为最小。<p>

## Input Format

<p>第一行是一个整数V，表示箱子容量。<p><p>第二行是一个整数n，表示物品数。<p><p>接下来n行，每行一个正整数（不超过10000），分别表示这n个物品的各自体积。<p>

## Output Format

<p>一个整数，表示箱子剩余空间。<p>

```input1
24
6
8
3
12
7
9
7


```
```output1
0

```
