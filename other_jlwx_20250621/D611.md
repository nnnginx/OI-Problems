## Description

小明同学期末考试考了3门，分别是语文、数学、英语，学校规定需要将分数转化为等级之后再公布给学员，学校规定的等级划分如下： 
90分~100分 A等级 
80分~89分&nbsp; B等级 
60分~79分&nbsp; C等级 
60以下&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; D等级 
请从键盘读入小明考试的3科分数，计算出小明考了几个A等级的科目？

## Input Format

输入三个整数，代表小明同学三门课的考试成绩。 

## Output Format

输出小明同学有几门功课获得了A等级。

```input1
99 89 100
```
```output1
2
```
