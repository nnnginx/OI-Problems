## Description

<p>给定一个正整数k,把所有k的方幂及所有有限个互不相等的k的方幕之和构成一个递增的序列。例如,当k=3时,这个序列是:</p><p>1.3.4.9.10,12,13, ...</p><p>请求出这个序列的第n项的值(用十进制数表示)。</p>

## Input Format

<p>一行两个正整数k和n,之间用一个空格隔开,且3&lt;=k&lt;=15, 10&lt;=n&lt;=1000。<br /></p>

## Output Format

<p>一行一个正整数。<br /></p>

```input1
3 100
```
```output1
981
```
