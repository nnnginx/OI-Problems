## Description

<p>有很多人在门口排队，每个人将会被发到一个有效的通行密码作为门票。</p><p>一个有效的密码由L(3&lt;=L&lt;=15)个小写英文字母组成，至少有一个元音(a、e、i、o或u)和两个辅音(除去元音以外的音节)，并且是按字母表顺序出现的(例如abe是有效的，而bac不是)。</p><p>现在给定一个期望长度为L和C(1&lt;=C&lt;=26)个小写字母，写一个程序，输出所有的长度为L、能由所给定的C个字母组成的有效密码。</p><p>密码必须按字母表顺序打印出来，一行一个。<br /></p>

## Input Format

<p>第1行是两个由一个空格分开的正整数,L和C,3&lt;=L&lt;=15, 1&lt;=C&lt;=26。</p><p>第2行是C个由一个空格隔开的小写字母,密码是由这个字母集中的字母来构建的。</p>

## Output Format

<p>若干行,每行输出一个长度为L个字符的密码(没有空格)。输出行必须按照字母顺序排列。</p><p>程序只需要输出前25 000个有效密码,即使后面还存在有效密码。</p>

```input1
4 6
a t c i s w

```
```output1
acis
acit
aciw
acst
acsw
actw
aist
aisw
aitw
astw
cist
cisw
citw
istw

```
