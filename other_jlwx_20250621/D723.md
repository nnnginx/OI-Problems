## Description

同构数是这样一种数：它出现在它的平方数的右端。例如：5的平方是25，5就是同构数，25的平方是625，25也是同构数。
再比如：100以内的同构数有1 5 6 25 76这5个整数。
请编程计算出1~N之间（包括N）的全部同构数有多少个？
 

## Input Format

一个正整数N，N&lt;＝10000。
 

## Output Format

一个整数，代表1~N之间的全部同构数的个数。
 

```input1
100
```
```output1
5
```
```input2
50
```
```output2
4
```
