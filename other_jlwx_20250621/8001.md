## Description

<p>小凯手中有两种面值的金币，两种面值均为正整数且彼此互素。每种金币小凯都有 无数个。在不找零的情况下，仅凭这两种金币，有些物品他是无法准确支付的。现在小凯想知道在无法准确支付的物品中，最贵的价值是多少金币？注意：输入数据保证存在小凯无法准确支付的商品。<br /></p>

## Input Format

<p>输入文件名为 math.in。 输入数据仅一行，包含两个正整数 a 和 b，它们之间用一个空格隔开，表示小凯手中金币的面值。<br /></p>

## Output Format

<p>输出文件名为 math.out。 输出文件仅一行，一个正整数 N，表示不找零的情况下，小凯用手中的金币不能准 确支付的最贵的物品的价值。<br /></p>

```input1
3 7
```
```output1
11
```
## Hint

<p>对于 30%的数据： 1 ≤ a，b ≤ 50。</p><p>对于 60%的数据： 1 ≤ a，b ≤ 10,000。</p><p>对于 100%的数据：1 ≤ a，b ≤ 1,000,000,000。</p>
