## Description

<p>给定 $N$ 个正整数 $A_1,A_2,…,A_N$，从中选出若干个数，使它们的和为 $M$，求有多少种选择方案。<br /></p>

## Input Format

<p>第一行包含两个整数 $N$ 和 $M$。</p><p>第二行包含 $N$ 个整数，表示 $A_1,A_2,…,A_N$。</p>

## Output Format

<p>包含一个整数，表示可选方案数。<br /></p>

```input1
4 4
1 1 2 2
```
```output1
3
```
## Hint

<p>数据范围</p><p>$1 \le N \le 100$,</p><p>$1 \le M \le 10000$,</p><p>$1 \le A_i \le 1000$</p>
