## Description

一只蜜蜂在下图所示的数字蜂房上爬动  ,  已知它只能从标号小的蜂房爬到标号大的相邻蜂房  ,  现在问你：蜜蜂从蜂房  M  开始爬到蜂房  N  ，  1=&lt;M&lt;N  &lt;=100，有多少种爬行路线？ 
 ![](https://oj.czos.cn/uploads/20211116/a25db1cf00cff55b356c978796402111.png)

## Input Format

输入  M  ，  N  的值。（  1=&lt;m&lt;n&lt;=100  ） 
 

## Output Format

 爬行有多少种路线。    
 

```input1
1  14
```
```output1
377
```
