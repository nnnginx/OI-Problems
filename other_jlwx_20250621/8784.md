## Description

<p>有邮票 m 种（面值各不相同），每种有 n 张，能从面额 1 开始最多连续组成多少面额（组成每个面额的总数不能超过 n 张）？<br /></p>

## Input Format

<p>第一行包含两个正整数，分别为 m 和 n，第二行包含 m 个正整数，依次表示 m 种邮票的面额。各行的多个数，两两之间用空格分隔。<br /></p>

## Output Format

<p>仅一个数，表示连续面额的最大值。</p>

```input1
3 4
1 2 4
```
```output1
14
```
## Hint

<p>数据范围：1 &lt;= m &lt;= 100，1 &lt;= n &lt;= 100，1 &lt;= 邮票面额 &lt;= 255。<br /></p>
