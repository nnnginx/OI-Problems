## Description

请你编程画一个放大的’X’（大写字母）。 如3*3的’X’应如下所示： 
![](https://oj.czos.cn/uploads/20211116/48c6494d6d94e5102be8756107a51994.jpg)

5*5的’X’如下所示：  
![](https://oj.czos.cn/uploads/20211116/01a55f8a7856bed50ab03c12b0ae0de8.jpg)


## Input Format

 有一个正奇数n（3 &lt;= n &lt;= 79），表示放大的规格。 

## Output Format

 打印一个规格为n * n放大的’X’ 

```input1
5
```
```output1
X   X
 X X
  X
 X X
X   X
```
