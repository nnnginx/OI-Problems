## Description

小明是班上的体育委员，体育老师要求小明记录班上每位同学的身高。但小明的记录方式让体育老师很为难，因为他只记录了每位同学与自己身高的差值，正数表示比小明高的，负数表示比小明矮的。现在，已知了小明的身高，和每位同学与小明身高的差值，求每位同学的身高。体育老师找到学编程的你，你能帮助他吗？
 

## Input Format

第一行两个数H、N（1<=N<=100），其中H为小明的身高，N表示班上还有N位同学。
第二行有N个数，第 i个数表示第i个同学与小明身高的差值。

## Output Format

共 N+1 个数，即全班同学的身高。
请注意，输出时，先输出小明的身高，再按读入的顺序输出班级N位同学的身高（输出同学身高时，保留2位小数）。
 

```input1
1.25 10
0.37 -0.23 0.67 -0.21 0.64 0.51 -0.79 0.31 -0.28 -0.52
```
```output1
1.25 1.62 1.02 1.92 1.04 1.89 1.76 0.46 1.56 0.97 0.73 
```
