## Description

<p>有一天，一只蚱蜢像往常一样在草地上愉快地跳跃，它发现了一条写满了英文字母的纸带。<br />蚱蜢只能在元音字母(A、E、I、O、U、Y)间跳跃，一次跳跃所需的能力是两个位置的差。</p><p>纸带所需的能力值为蚱蜢从纸带开头的前一个位置根据规则跳到纸带结尾的后-个位置的过程中能力的最大值。<br />蚱蜢想知道跳跃纸带所需的能力值(最小)是多少。</p><p>如图所示的纸带所需的能力值(最小)是4。</p><p><img alt="931fe220d5e33169e0b1bc4f820ecb347277ef29.png" src="/public/upload/a17f6223c1.png" width="616" height="87" /><br /></p>

## Input Format

<p>一行一个字符串，字符串长不超过100。<br /></p>

## Output Format

<p>一行一个整数，代表(最小)能力值。<br /></p>

```input1
KMLPTGFHNBVCDRFGHNMBVXWSQFDCVBNHTJKLPMNFVCKMLPTGFHNBVCDRFGHNMBVXWSQFDCVBNHTJKLPMNFVC
```
```output1
85
```
```input2
ABABBBACFEYUKOTT
```
```output2
4
```
