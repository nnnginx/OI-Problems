## Description

<p>铁塔尼号遇险了！他发出了求救信号。距离最近的哥伦比亚号收到了讯息，时间就是生命，必须尽快赶到那里。</p><p>通过侦测，哥伦比亚号获取了一张海洋图。这张图将海洋部分分化成n*n个比较小的单位，其中用1标明的是陆地，用0标明是海洋。船只能从一个格子，移到相邻的四个格子。</p><p>为了尽快赶到出事地点，哥伦比亚号最少需要走多远的距离。</p>

## Input Format

<p>第一行为n,下面是一个n*n的0、1矩阵，表示海洋地图</p><p>最后一行为四个小于n（n&lt;=1000）的整数，分别表示哥伦比亚号和铁塔尼号的位置。</p>

## Output Format

<p>哥伦比亚号到铁塔尼号的最短距离，答案精确到整数<br /></p>

```input1
3
001
101
100
1 1 3 3
```
```output1
4
```
