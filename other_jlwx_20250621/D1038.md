## Description

设有一个N*N方格的迷宫，入口和出口分别在左上角和右上角。
迷宫格子中分别放有0和1，0表示可通，1表示不能，
迷宫走的规则如下：
即从某点开始，有八个方向可走，前进方格中数字为0时表示可通过，为1时表示不可通过，要另找路径。
找出从入口（左上角）到出口（又上角）的路径(不能重复)。输出路径总数。如果无法到达，输出0。



## Input Format


第1行有一个整数N，表示迷宫大小（2≤N≤10）
接下来N行，每行有N个数

## Output Format

输出一个整数，代表迷宫从起点到终点的路径条数。

```input1
3
0 0 0
0 1 1
1 0 0
```
```output1
2
```
