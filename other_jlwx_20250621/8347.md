## Description

<p>lxj被cxy叫去砍树，他需要砍倒m米长的木材。现在，lxj弄到了一个奇怪的伐木机。伐木机工作过程如下：设置一个高度参数h（米），伐木机升起一个巨大的锯片到高度h，并锯掉所有的树比h高的部分（当然，树木不高于h米的部分保持不变）。lxj就得到树木被锯下的部分。</p><p>例如，如果一行树的高度分别为20、15、10和17米，lxj把锯片升到15米的高度，切割后树木剩下的高度将是15、15、10和15米，而lxj将从第1棵树得到5米，从第4棵树得到2米，共得到7米木材。</p><p>lxj非常关注生态保护，所以他不会砍掉过多的木材。这正是他为什么要尽可能高地设定伐木机锯片的原因。帮助lxj找到伐木机锯片的最大的整数高度h，使得他能得到的木材至少为m米。换句话说，如果再升高1米，则他将得不到m米木材。</p>

## Input Format

<p>第1行2个整数n和m，n表示树木的数量，m表示需要的木材总长度。</p><p>第2行n个整数，表示每棵树的高度，值均不超过10^9。保证所有木材长度之和大于m，因此必然有解。</p>

## Output Format

<p>一行一个整数，表示砍树的最高高度。<br /></p>

```input1
5 20
4 42 40 26 46

```
```output1
36

```
## Hint

<p>对于30%的数据满足：1≤n≤10，1≤m≤30。</p><p>对于70%的数据满足：1≤n≤10^3，1≤m≤10^4。</p><p>对于100%的数据满足：1≤n≤10^6，1≤m≤2×10^9。</p>
