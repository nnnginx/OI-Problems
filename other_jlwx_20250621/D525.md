## Description

小明写出了一个数列，第 i 项 ai 的值为 i<sup>2</sup>。数列从第一项(i = 1)开始如下:
1&#44; 4&#44; 9&#44; 16&#44; 25&#44; ...
编程求出这个数列前 n 项的和。


## Input Format

整数n (1≤n≤1&#44;000)

## Output Format

一个整数:a1 +a2 +⋯+an的值。

```input1
6
```
```output1
91
```
## Hint

样例1:  
6 
样例2:  
987 
样例输出 
样例1:  
91 
样例2:  
320988850
