## 题目描述

<p><span style="font-size: medium">一个数是<strong>非回文数</strong>当且仅当不包含长度大于<code>1</code>的回文数。比如<code>16276</code>是无回文数，而<code>17276</code>因为含有<code>727</code>而不是。</span></p>
<p><span style="font-size: medium">求区间内有多少个<strong>非回文数</strong>。</span></p>

## 输入格式

<p><span style="font-size: medium"><code>L, R</code> （<code>0 <= L <= R <= 10^18</code>）</span></p>

## 输出格式

<p><span style="font-size: medium">非回文数数量</span></p>

```input1
输入样例1
123 321
输入样例2
123456789 987654321
```
```output1
输出样例1
153
输出样例2
167386971
```
