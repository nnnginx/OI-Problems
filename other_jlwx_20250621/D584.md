## Description

小明和小华去帮班上的同学购买课外书，小明带了 x 元（ x 是一个四位整数），小明问小华带了多少钱，小华说自己带的钱正好是小明带的前把前两位和后两位直接换位置那么多，请你编程帮助小华计算一下，他带了多少钱？   
 
比如，假设小明带了 1298 元，那么小华带的金额就是 9812 元。
再比如，假设小明带了1208元，那么小华带的金额就是812元。 
   
 

## Input Format

一个整数 x 代表小明的带的金额， x 是一个四位的整数。   
 

## Output Format

小华带的金额 。  
 

```input1
1298
```
```output1
9812
```
