## 输入格式

<div>
 第一行有四个整数 n, p, k, r，所有整数含义见问题描述。
</div>
<div>
 1 ≤ n ≤ 10^9, 0 ≤ r < k ≤ 50, 2 ≤ p ≤ 2^30 − 1
</div>

## 输出格式

<div>
 一行一个整数代表答案。
</div>

```input1
2 10007 2 0
```
```output1
8
```
