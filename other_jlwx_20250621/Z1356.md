## 题目描述

<p>给出n个点，要你从这些点中找出四个点来组成一个矩形，面积最大.</p>

## 输入格式

<p>第一行给出N.4 ≤ n ≤ 1,500. 下面N行给出这些点的坐标，其值在 [10^8,10^8]</p>

## 输出格式

<p>最大的矩形面积</p>

```input1
8
-2 3
-2 -1
0 3
0 -1
1 -1
2 1
-3 1
-2 1
```
```output1
10
```
