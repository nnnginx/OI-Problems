## Description

如下图所示，摆 1 个火柴棒三角形需要 3 根火柴，那么摆 2 个火柴棒三角形就需要 6 根火柴，请问摆 n 个火柴棒三角形需要多少根火柴？  
 <img src="http://oj.czos.cn/upload/image/20190711/20190711233837_90681.jpg" alt="" />
 
 
	 
 

## Input Format

一个整数 n ，代表想要摆放的火柴棒三角形的数量 

## Output Format

 一个整数，代表了需要多少根火柴的数量 

```input1
3
```
```output1
9
```
