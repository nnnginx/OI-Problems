## Description

根据输入的正整数n，用数字7画出如图所示的平行四边形图案。
例如n = 5时，图案为：
```
77777
 77777
  77777
   77777
    77777
```



## Input Format

输入1个正整数n（3 <= n <= 20），表示这个平行四边形的边由n个7组成。

## Output Format

根据题目描述，输出对应的图案。

```input1
3
```
```output1
777
 777
  777
```
```input2
4
```
```output2
7777
 7777
  7777
   7777
```
## Hint

图案同一行中相邻的两个7中间没有空格。
图案的最左侧从第一列开始输出，左边不能有多余的空格。
