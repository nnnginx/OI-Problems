## Description

<p>给定一棵 n 个点的带权树，求树上最长的异或和路径。<br /></p>

## Input Format

<p>第一行一个整数 n，接下来 n−1 行每行三个整数 u,v,w，表示 u,v 之间有一条长度为 w 的边。<br /></p>

## Output Format

<p>输出一行一个整数，表示答案。<br /></p>

```input1
4
1 2 3
2 3 4
2 4 6
```
```output1
7
```
## Hint

<p>样例解释</p><p>最长的异或和路径是 1→2→3 ，它的长度是 3⨁4=7。</p><p>注意：结点下标从 1 开始到 N。</p><p>注：x⨁y 表示 x 与 y 按位异或。</p>
