## Description

请问  100~n  中连续递增或者连续递减的  3  位数有总和是多少，有多少个？    
连续递递增或递减指的是三位数相邻的两位之间都是递增的或者是递减的。    
比如：  139  ，三位数就是连续递增的，因为  1&lt;3&amp;&amp;3&lt;9  ；    
 再比如：  931  ，三位数就是连续递减的，因为 9&gt;3&amp;&amp;3&gt;1  ；    
 

## Input Format

一个三位的整数 n  。    
 

## Output Format

 第  1  行输出满足条件的数的总和是多少。    
 第  2  行输出满足条件的数有多少个。    
 

```input1
600
```
```output1
32010
100
```
## Hint

n &lt;= 999
