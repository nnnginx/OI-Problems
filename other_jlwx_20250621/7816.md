## Description

有一串编号从1开始的彩色珠子（见下图），从第一颗起，每颗珠子的颜色按照蓝、橙、绿的规律重复出现。
假定总共有n颗珠子，请编程把所有橙色珠子的编号依次找出来。![图片![](https://s1.ax1x.com/2023/09/04/pPD6gc8.png)

## Input Format

输入一个正整数n（ 3 <= n <= 100），表示彩色珠子的总数。

## Output Format

输出一行，包含若干整数，依次表示每一颗橙色珠子的编号。相邻两个数用一个空格分隔。
注意：输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
10
```
```output1
2 5 8
```
