## Description

龙湖校区有一个圆形花坛，量得花坛周围的篱笆长是 x 米，请问该花坛的面积是多少平方米？（假设π =3.14 ）
 

## Input Format

一个小数x

## Output Format

花坛的面积（结果保留2位小数）

```input1
18.84
```
```output1
28.26
```
## Hint

圆的面积和周长求解公式分别如下； 
圆的面积S = π * 半径 * 半径；（π读作pài，π = 3.1415926…，在具体的题目中π的值精确到小数点后多少位，取决于具体的题目） 
圆的周长C = π * 2 * 半径；
