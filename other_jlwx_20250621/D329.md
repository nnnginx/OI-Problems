## Description

在数学上，我们发现有一类图形是对称图形。我们对于左右一样的图形叫做沿 y 轴对称，对于上下一样的图形叫做沿 x 轴对称。   
 如下图所示的图形就是沿 y 轴对称的两个三角形：   
<img src="http://oj.czos.cn/upload/image/20180625/20180625003918_79632.jpg" alt="" /> 
而下图所示的图形就是沿 x 轴对称的两个三角形：   
<img src="http://oj.czos.cn/upload/image/20180625/20180625003926_15238.jpg" alt="" /> 
请从键盘读入一个整数 n ，代表三角形的行数，再读入一个字符，该字符为 ’x’ 或者 ’y’ ，代表是沿 x 轴对称还是沿 y 轴对称，输出对应的轴对称图形。（8.1）
 

## Input Format

第一样是一个整数 n （ n &lt;= 10 ）   
第二行是一个字符（字符 ’x’ 或者字符 ’y’ ）   
 

## Output Format

按照要求输出的轴对称图形   
 

```input1
5
x
```
```output1
    *
   ***
  *****
 *******
*********
*********
 *******
  *****
   ***
    *
```
