## Description

 农夫知道一头牛的位置，想要抓住它。农夫和牛都位于数轴上，农夫起始位于点N(0≤N≤100000)，牛位于点K(0≤K≤100000)。
农夫有两种移动方式：  
1、从 **X** 移动到 **X-1**或 **X+1**，每次移动花费一分钟  
2、从 **X** 移动到 **2*X**，每次移动花费一分钟  
假设牛没有意识到农夫的行动，站在原地不动。
农夫最少要花多少时间才能抓住牛？ 

## Input Format

 两个整数，N和K。 

## Output Format

 一个整数，农夫抓到牛所要花费的最小分钟数。 

```input1
5 17

```
```output1
4

```
