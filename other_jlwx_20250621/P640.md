## Description

<p>输入一系列的整数，读到0时停止，输出0是第几个数<br /></p><p><b>如一系列数中有多个0，则只输出第一个0的位置</b><br /></p>

## Input Format

<p>一系列数（不确定个数）</p>

## Output Format

<p>0的位置（<span style="color: rgb(51, 51, 51);">如一系列数中有多个0，则只输出第一个0的位置）</span></p>

```input1
45 532 2343 2 0 234 39 0 9
```
```output1
5
```
