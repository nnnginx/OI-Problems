## Description

<p>【题目背景】</p><p>小Tu生病了，小Tu有一些药丸。</p><p>【题目描述】</p><p>小Tu生了i天的病，家中有5种药丸，已知5种药丸的直径都小于1cm（不包括1cm，且只有1位小数），直径为x的药丸可以治10x天的病，请问小Tu至少要吃多少颗药丸才能痊愈？</p><p>（要求：药效刚好到小Tu痊愈当天结束，小时分钟等忽略不计）</p>

## Input Format

<p>第一行：输入一个整数i，代表生病的天数；</p><p>第二行：输入5个小数，代表药丸直径大小，中间用一个空格隔开（要求仅保留1位小数，例：0.1、0.2等）</p>

## Output Format

<p>一个整数n，代表小Tu五种药丸最少吃的总数。</p>

```input1
7
0.1 0.2 0.4 0.8 0.9
```
```output1
3
```
```input2
18
0.2 0.5 0.6 0.8 0.9
```
```output2
2
```
## Hint

<p>【样例1解释】</p><p style="margin-left: 40px;">已知小Tu生7天的病，五种药丸的药效分别为1天、2天、4天、8天、9天。</p><p style="margin-left: 40px;">所以小Tu最少吃的药丸直径为0.1cm,0.2cm,0.4cm共3颗</p><p>【数据范围】</p><p style="margin-left: 40px;">1≤i≤3650</p><p>【提示】</p><p style="margin-left: 40px;">所有样例均确保五种药丸直径各不相同，且一定有解。<br /></p><p>【洛谷通道】</p><p style="margin-left: 40px;">（<a href="https://www.luogu.com.cn/problem/U191308" target="_blank">点击跳转</a>）</p>
