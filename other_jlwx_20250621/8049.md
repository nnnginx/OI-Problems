## Description

<p>$N$ 头牛要去参加在某农场举行的一场编号为 $X$ 的牛的派对。</p><p>有 $M$ 条有向道路，每条路长 $T_i$；每头牛参加完派对后都必须回到家，每头牛都会选择最短路径。</p><p>求这 $N$ 头牛的最短路径（一个来回）中最长的一条的长度。</p><p>特别提醒：可能有权值不同的重边。</p>

## Input Format

<p>第一行包含三个整数 $N,M,X$。</p><p>接下来 $M$ 行，每行包含三个整数 $A\_i,B\_i,T\_i$，表示有一条从 $A\_i$ 到 $B\_i$ 的路，长度为 $T\_i$。</p>

## Output Format

<p>共一行，一个数，表示最短路中最长的一条的长度。<br /></p>

```input1
4 8 2
1 2 4
1 3 2
1 4 7
2 1 1
2 3 5
3 1 2
3 4 4
4 2 3
```
```output1
10
```
## Hint

<h4>数据范围</h4><p>$1 \le N \le 1000$,<br />$1 \le X \le N$,<br />$1 \le M \le 10^5$,<br />$1 \le T_i \le 100$</p>
