成绩分化（厦门2022小学生c++竞赛第二题）
=======================

题目描述
----

期中考结束，高老师想知道班级学生成绩两极分化的情况，以便采取有效的措施来解决此问题。已知高老师班上共有 n 个学生，每个学生有一个 0～100分的整数成绩，现在高老师请会编程的你帮忙计算班里学生成绩最高分与最低分的差值。

输入格式
----

第一行，一个整数，表示班级的学生人数。 接下来 n 行，每行一个整数，表示一个学生的期中考成绩。

输出格式
----

输出一行，包括一个整数，表示最高分与最低分的差值。

样例 #1
-----

### 样例输入 #1

    5
    78 90 60 85 100
    

### 样例输出 #1

    40
    

提示
--

    对于 20%的数据，n≤5。
    对于 60%的数据，n≤30。
    对于 100%的数据，n≤50。
