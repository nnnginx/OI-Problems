## Description

<p>给定一个自然数 $N$，要求把 $N$ 拆分成若干个正整数相加的形式，参与加法运算的数可以重复。</p><p>注意:</p><p>拆分方案不考虑顺序；</p><p>至少拆分成 $2$ 个数的和。</p><p>求拆分的方案数 $\bmod 2147483648$ 的结果。</p>

## Input Format

<p>一个自然数 $N$。<br /></p>

## Output Format

<p>输入一个整数，表示结果。<br /></p>

```input1
7
```
```output1
14
```
## Hint

<p>数据范围</p><p>$1 \le N \le 4000$</p>
