<h2>说明</h2>

一个三阶 $X$ 标志为：<br />
<pre>X X
 X
X X</pre>
<br />
一个五阶 $X$ 标志为：<br />
<pre>X &nbsp; X
 X X
 &nbsp;X
 X X
X &nbsp; X</pre>
<br />
每行末尾不需要多余的空格。现在请你输出 $n$ 阶 $X$ 标志。
<h2>输入格式</h2>

一个奇数 $n$ ($3≤n≤499$)。

<h2>输出格式</h2>

输出 $n$ 阶 $X$ 标志。

<h2>样例</h2>
<pre><code class="language-input1">3</code></pre><pre><code class="language-output1">X X
 X
X X</code></pre>
