## Description

<p>程序员输入程序出现差错时，可以采取以下的补救措施：按错了一个键时，可以补按一个退格符“#”，以表示前一个字符无效；发现当前一行有错，可以按一个退行符“@”，以表示“@”与前一个换行符之间的字符全部无效。<br /></p>

## Input Format

<p>输入一行字符，个数不超过100。<br /></p>

## Output Format

<p>输出一行字符，表示实际有效字符。<br /></p>

```input1
sdfosif@for(ii#=1,#;i<.#=8;i+++#);
```
```output1
for(i=1;i<=8;i++);
```
