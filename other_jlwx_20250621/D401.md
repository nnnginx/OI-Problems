## Description

请从一组数中找出含有x的数，统计出这样的数总共有多少个，总和是多少？ 
 

## Input Format

输入包含三行： 
第一行为N，表示这组整数总共有多少个数(N &lt;= 1000)； 
第二行为N个整数（这些整数都是1~9999之间的数），整数之间以一个空格分开； 
第三行包含一个一位的整数x（x是一个1~9之间的一位数）；

## Output Format

输出一行，有2个整数用空格隔开，第一个整数代表含有x的数的总个数，第二个整数代表含有x的数的总和。

```input1
5
12 28 190 36 1255
2
```
```output1
3 1295
```
