## 【题目描述】

输入两个很大的十进制正实数（长度超过20 位），输出数值较大的数。
说明：
1）两个数都大于10000000000000000000；
2）实数包含有小数点的数，一个数里只可能有一个小数点，且不在数的最前或最后。

## 【输入格式】

输入两行，每行一个数字

## 【输出格式】

数值较大的数。
如果两数相等，则输出“equation”。
如果输入的数不符合数字规则，则输出“error”。

## 【样例输入】（测试数据不包含本样例）

```
11111111111111111111111111111111111111
2222222222222222222222222222222222222222
```

## 【样例输出】

```
2222222222222222222222222222222222222222
```
