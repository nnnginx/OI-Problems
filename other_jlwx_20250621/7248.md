## Description

有n个人排队到r个水龙头去打水，他们装满水桶的时间t1、t2………..tn为整数且各不相等，应如何安排他们的打水顺序才能使他们总共花费的时间最少？

## Input Format

第一行n，r (n<=500,r<=75)
第二行为n个人打水所用的时间Ti (Ti<=100)；

## Output Format

最少的花费时间。

```input1
3 2
1 2 3
```
```output1
7
```
```input2
7 4
234 78 54 33 123 43 87
```
```output2
782
```
## Hint

其中80%的数据保证n<=10
