## Description

 小X喜欢研究字符串。 &nbsp;   
 这天，小X随手在草稿纸上写下了一个大写字母字符串。此时，班长把成绩报告单发到了每位同学的手中。小X看到自己每门都是A （优秀），非常高兴，灵光一闪想到一个问题： &nbsp;   
 在刚刚写下的字符串中，字母A最多连续出现了多少次呢？ &nbsp;   
 小X立刻数了起来，但这个字符串实在是太长了，希望你帮帮他。 

## Input Format

 第一行包含一个整数N，表示字符串长度。 &nbsp;   
 第二行包含一个字符串。 

## Output Format

 第一行包含一个整数，表示该字符串中字母A最多连续出现的次数。 

```input1
5
BAACA
```
```output1
2
```
## Hint


数据范围
对于30%的数据，N≤100。
对于60%的数据，N≤1000。
对于 100%的数据，1≤N≤100000。
