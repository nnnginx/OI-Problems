## Description

晶晶拿到了只有0和1的字符串，字符串是偶数位的长度，她想让这个字符串中的奇数位的位置和偶数位的位置的字符互换位置。

比如：有一个字符串是011010，那么奇数位和偶数位的字符互换位置后的结果为100101。
 

## Input Format

输入包含多组测试数据。
输入的第一行是一个整数n，表示有n组测试数据。（1&lt;=n&lt;=100）
接下来n组测试数据，每组数据输入均为0，1字符串，保证串长为偶数位(字符串长度&lt;=100)。
 

## Output Format

请为每组测试数据输出奇数位和偶数位互换后的结果，每组输出占一行。

```input1
2
0110
1100
```
```output1
1001
1100
```
## Source

字符串
