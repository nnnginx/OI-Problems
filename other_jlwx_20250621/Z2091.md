## 题目描述

<p>给出N个正整数，AB两个人轮流取数，A先取。每次可以取任意多个数，直到N个数都被取走。<br>
  每次获得的得分为取的数中的最小值，A和B的策略都是尽可能使得自己的得分减去对手的得分更大。<br>
  在这样的情况下，最终A的得分减去B的得分为多少。</p>
<p></p>

## 输入格式

<p>第一行一个正整数N (N <= 1,000,000)，第二行N个正整数（不超过10^9）。</p>
<p></p>

## 输出格式

<p><br>
  一个正整数，表示最终A与B的分差。</p>
<p></p>

```input1
3
1 3 1
```
```output1
2
```
## 数据范围与约定

<p>第一次A取走3，第二次B取走两个1，最终分差为2。<br><br></p>
