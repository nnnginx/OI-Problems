## Description

工厂生产了n个圆形零件，每个零件有一个半径，判断零件是否合格的方法是，给定一个标准零件的半径值，如果生产出来零件的半径和标准零件半径值的差&lt;=5，那么该零件就是合格的！ 
 
比如：工厂生产了5个零件，半径分别是79 97 115 96 95，标准零件的半径值是95，按照判断标准，有3个零件是合格的，半径分别是：97 96 95。 
 
请你编程计算一下，工厂生产的n个零件中，有多少个零件是合格的。 
 
 

## Input Format

第一行：一个整数n代表有n个数（10&lt;=n&lt;=100） 
第二行：n个整数，代表零件的半径（10&lt;=半径的值&lt;=1000） 
第三行：一个整数x，代表标准零件的半径（10&lt;=标准零件的半径值&lt;=1000） 
 

## Output Format

一个整数，代表合格零件的个数。

```input1
5
79 97 115 96 95
95

```
```output1
3
```
