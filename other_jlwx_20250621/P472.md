## Description

<p>给出一个字符串 S，你需要从 S 中挑选一对字符进行一次交换（不可以不交换！！！！！！），并让得到的新字符字典序最小！例如：S = &quot;abacc&quot;，如果交换字符 1(a)和 4(c)，得到字符&quot;cbaac&quot;。如果交换字符 2(a)和 3(b)，得到字符&quot;aabcc&quot;。其中：&quot;aabcc&quot;的字典序小于&quot;cbaac&quot;。并且&quot;aabcc&quot;是所有交换方法中，字典序最小的。例如：S = &quot;aaab&quot;，则交换 1 和 2，交换 1 和 3，得到的字符都是 &quot;aaab&quot;，并且 &quot;aaab&quot;是所有交换方法中，字典序最小的。输出这个字典序最小的字符。<br /></p>

## Input Format

<p>一个字符串 S（S 的只包括 a 到 z 的小写字符，2&lt;=S 的长度&lt;=500000）。<br /></p>

## Output Format

<p>字典序最小的新字符串。<br /></p>

```input1
abacc
```
```output1
aabcc
```
