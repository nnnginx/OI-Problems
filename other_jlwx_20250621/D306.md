## Description

从键盘读入 n (3&lt;=n&lt;=100)个梯形的上底、下底和高，请问这 n 个梯形中，最大面积的梯形的面积是多少？（梯形面积的求解公式为  S = (a + b) * h / 2 ，也就是 ( 上底  +  下底 ) *  高  / 2 ）   
 
<img src="http://oj.czos.cn/upload/image/20180623/20180623100402_64366.jpg" alt="" /> 
 

## Input Format

第1行为1个整数n，接下来n行每行3个整数分别代表梯形的上底、下底和高。

## Output Format

最大面积梯形的面积（结果保留1位小数）

```input1
3
1 2 3
3 4 5
2 3 4
```
```output1
17.5
```
## Source

二维数组 结构体
