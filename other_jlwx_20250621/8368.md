## Description

<p>数学家高斯小时候偶然间发现一种有趣的自然数集合Blah。对于以a为基的集合Blah定义如下：</p><p>1）a是集合Blah的基，且a是Blah的第一个元素；</p><p>2）如果x在集合Blah中，则2x+1和3x+1也都在集合Blah中；</p><p>3）没有其他元素在集合Blah中了。</p><p>现在小高斯想知道如果将集合Blah中元素按照升序排列，第n个元素会是多少？注意：集合中没有重复的元素。</p>

## Input Format

<p>一行两个正整数，分别表示集合的基a以及所求元素序号n，1≤a≤50，1≤n≤1000000。<br /></p>

## Output Format

<p>一行一个正整数，表示集合Blah的第n个元素值。<br /></p>

```input1
1 100

```
```output1
418

```
```input2
28 5437

```
```output2
900585

```
