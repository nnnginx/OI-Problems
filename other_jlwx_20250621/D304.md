## Description

柱状图是非常直观的用来表示数值的一种图形描述方式，如图所示是 2011 年我国几种常用家电的产量。  
<img src="http://oj.czos.cn/upload/image/20180623/20180623095810_79901.jpg" alt="" />
 请用 编程 来模拟一下柱状图，从键盘读入 n 种物品的数量（ n &lt;= 100 ，每种物品数量在 1~10 之间），用一颗星来代表物品数量为 1 ，输出每种物品的编号及其数量。 
 
如：假设有 3 种物品，数量分别为 3 7 6 ，则输出如下图所示的柱状图  
 
1:*** 
2:******* 
3:****** 
 
	  
 
 

## Input Format

第一行是一个整数n，代表物品种类；第二行输入每种物品的数量；

## Output Format

如图所示的柱状图

```input1
3
3 7 6
```
```output1
1:***
2:*******
3:******
```
