## Description

从键盘输入一个整数 N ，输入 N(5&lt;=N&lt;=20) 个元素，输入一个整数，判断这个数组中最后一次出现该数的位置，如果没有请输出 -1。 

## Input Format

第1行读入一个整数n（ 5&lt;=n&lt;=20 ）
第2行读入n个整数用空格隔开
第3行读入1个整数，代表要找的数。
 

## Output Format

输出1个整数，代表最后一次出现整数的位置，如果数组中不存在这样的数，请输出-1。

```input1
5
12 23 33 12 6
12
```
```output1
4
```
