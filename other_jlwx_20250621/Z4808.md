## 输入格式

<div>
 一行，两个正整数N和M。
</div>
<div>
 接下来N行，每行M个数，要么为0，表示没坏，要么为1，表示坏了。
</div>
<div>

</div>
<p></p>

## 输出格式

<div>
 一行，输出最多的个数。
</div>
<p></p>

```input1
2 3
0 1 0
0 1 0
```

