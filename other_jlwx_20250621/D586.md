## Description

国王给骑士发金币，奖励他们的忠诚和英勇。第一天国王发的金币是一个四位数，第二天发的金币是这个四位数前两位的数值，第三天发的金币是这个四位数后两位的数值，请问骑士平均每天拿到多少金币？（计算结果保留 2 位小数）   

比如：假设第 1 天骑士拿到了 1562 个金币，那么第 2 天就拿到了 15 个金币（ 1562 的前两位是 15 ），第 3 天拿到 62 个金币（ 1562 的后两位是 62 ），他平均每天获得的金币  = (1562+15+62)/3.0 = 546.33 。   
 

## Input Format

一个四位整数 。  
 

## Output Format

一个小数，代表骑士平均每天获得了多少金币。   
 

```input1
1562
```
```output1
546.33
```
