## Description

电视台举办一场模仿秀节目，评委根据选手的体态、语言和形体动作的相似程度进行打分，现给出n位选手的各项得分情况，要求统计出每位选手的总分。


## Input Format

输入：第1行有1个整数n（1≤n≤100），表示选手的人数。
接下来有n行，每行包含3个正整数，分别表示体态、语言和形体动作各项的得分（每项得分均不大于100），相邻两个数中间用一个空格分隔。


## Output Format

输出：共n行（按输入顺序输出每名选手的总分）；
每行1个整数，表示一名选手的总分。


```input1
3
90 80 100
93 99 70
89 91 80

```
```output1
270
262
260

```
