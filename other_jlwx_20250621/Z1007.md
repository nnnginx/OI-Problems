## 题目描述

<p>　　在xoy直角坐标平面上有n条直线L1,L2,...Ln,若在y值为正无穷大处往下看,能见到Li的某个子线段,则称Li为<br>
  可见的,否则Li为被覆盖的.<br>
  例如,对于直线:<br>
  L1:y=x; L2:y=-x; L3:y=0<br>
  则L1和L2是可见的,L3是被覆盖的.<br>
  给出n条直线,表示成y=Ax+B的形式(|A|,|B|<=500000),且n条直线两两不重合.求出所有可见的直线.</p>

## 输入格式

<p>　　第一行为N(0 < N < 50000),接下来的N行输入Ai,Bi</p>

## 输出格式

<p>　　从小到大输出可见直线的编号，两两中间用空格隔开,最后一个数字后面也必须有个空格</p>

```input1
3
-1 0
1 0
0 0
```
```output1
1 2
```
