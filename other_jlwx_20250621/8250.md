## Description

<p>有一个M行N列的点阵，相邻两点可以相连。一条纵向的连线花费一个单位，一条横向的连线花费两个单位。某些点之间已经有连线了，试问至少还需要花费多少个单位才能使所有的点全部连通。<p>

## Input Format

<p>第一行输入两个正整数m和n。<p><p>以下若干行每行四个正整数$x\_1,y\_1,x\_2,y\_2$，表示第$x\_1$行第$y\_1$列的点和第$x\_2$行第$y\_2$列的点已经有连线。输入保证$|x\_1-x\_2|+|y\_1-y\_2|=1$。<p>

## Output Format

<p>输出使得连通所有点还需要的最小花费。<p>

```input1
2 2
1 1 2 1

```
```output1
3

```
## Hint

【数据规模】
30%数据：n\*m≤1000
100%数据：m,n≤1000
