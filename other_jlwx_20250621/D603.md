## Description

地面上有一排长度为n的格子1-n，每个格子上都有一个数xi，开始时你在位置0，每次你可以向前跳1-2格，然后取走格子上的数，直到跳到位置n+1。取走的数的和就是你的得分，现在你想知道你可能的最大得分是多少。 
 

## Input Format

一行四个整数n,A,B,C(n≤100000,0≤A,B,C≤10000)，其中n表示格子的数量。x[i]由如下方式生成：

for (int i = 1; i <= n; i++) {
int tmp = ((long long)A * i * i + B * i + C) % 20000;
x[i] = tmp - 10000;
}

## Output Format

一行一个整数ans表示可能的最大得分。

```input1
3 1 1 1
```
```output1
-9993
```
