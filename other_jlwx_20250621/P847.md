## Description

<p>下图表示城市之间的交通路网，线段上的数字表示费用，单向通行由A->E。试用动态规划的最优化原理求出A->E的最省费用。<p><p>如图：<p><p>![](https://i.loli.net/2019/02/26/5c74f1b5eb40b.gif)<p><p>求v1到v10的最短路径长度及最短路径。<p>

## Input Format

<p>第一行为城市的数量N(<=20);<p><p>后面是N*N的表示两个城市间费用组成的矩阵。<p>
