## Description

根据输入的正整数n，用数字画出如下所述的直角三角形。
第1行有一个“1”，第2行有2个“2”…… 第n行有n个“n”。每一行的数字左边都没有空格。
例如n = 5时，图案为：
1
22
333
4444
55555

## Input Format

输入1个正整数n（2 <= n <= 9），表示这个直角三角形的底和高。

## Output Format

根据题目描述，输出对应的字符图案。

```input1
5
```
```output1
1
22
333
4444
55555
```
## Hint

1. 同一行中两个数字之间没有空格；
2. 图案的第一行前面不能有多余的空行；
3. 图案的最左侧从第一列开始输出，不能有多余的空格。
