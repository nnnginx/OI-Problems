## 题目描述

<div>
 从前有张图。图里n个顶点两两之间有N^2种最小割。告诉你这N^2个最小割。还原出这张图。
</div>
<div></div>
<p></p>

## 输入格式

<div>
 第一行一个正整数n，表示图的顶点数。
</div>
<div>
 接下来n行每行N个非负整数，第i行第j列的数表示第i个点与第j个点的最小割。
</div>
<div>
 点的编号从1开始。
</div>
<div>
 Vi<=10^5,n<=100
</div>
<div>
 保证vii=0。
</div>
<div></div>
<p></p>

## 输出格式

<div>
 第一行一个整数m，表示图的边数。
</div>
<div>
 接下来每行三个整数u,v,z。
</div>
<div>
 表示从“到”存在一条权值为z的边。
</div>
<div>
 l<=u,v<=N
</div>
<div>
 0<Z<=10^9。
</div>
<div>
 m<=n*(n-l)/2。
</div>
<div>
 请注意你给出的图要求联通。
</div>
<div>
 如果无解请输出-l。
</div>
<div>
 若有多解则输出任意一组解。
</div>
<p></p>

```input1
3
0 2 2
2 0 2
2 2 0
```
```output1
2
2 3 2
1 3 2
```
