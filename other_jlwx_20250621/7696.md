## Description

普通人的打字速度是每分钟50到80字，给出一个人的打字速度，请编写程序判定他的打字等级。

## Input Format

输入一个整数n，表示每分钟打字的数量。

## Output Format

如果n小于50，输出0；
如果n在50~80之间（包含50和80），输出1；
如果n大于80，输出2。
说明/提示：
题目保证输入数据范围为1~400。
输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
40
```
```output1
0
```
```input2
120
```
```output2
2
```
