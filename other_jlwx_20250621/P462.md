## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">小明的花店新开张,为了吸引顾客,他想在花店的门口摆上一排花,共 m 盆。通过调查顾客的喜好,小明列出了顾客最喜欢的 n 种花,从 1 到 n 标号。为了在门口展出更多种花,规定第 i 种花不能超过a_</span><span style="color: rgba(0, 0, 0, 0.87);">i</span><span style="color: rgba(0, 0, 0, 0.87);">盆。摆花时同一种花放在一起，且不同种类的花需按标号的从小到大的顺序依次摆列。试编程计算一共有多少种不同的摆花方案。</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">第一行包含两个正整数 n 和 m，中间用一个空格隔开。</span></p><p>第二行有 n 个整数，每两个整数之间用一个空格隔开，依次表示 a_1、a_2、……a_n。</p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出只有一行，一个整数，表示有多少种方案。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">注意：因为方案数可能很多，请输出方案数对 1000007 取模的结果。</span></p>

```input1
2 4
3 2
```
```output1
2
```
## Hint

<p>【数据范围】</p><p style="margin-left: 40px;"><span style="color: rgba(0, 0, 0, 0.87);">对于20%数据，有0&lt;n≤8，0&lt;m≤8，0≤ai≤8；</span></p><p style="margin-left: 40px;"><span style="color: rgba(0, 0, 0, 0.87);">对于50%数据，有0&lt;n≤20，0&lt;m≤20，0≤ai≤20；</span></p><p style="margin-left: 40px;"><span style="color: rgba(0, 0, 0, 0.87);">对于100%数据，有0&lt;n≤100，0&lt;m≤100，0≤ ai≤100。</span></p><p><span style="color: rgb(51, 51, 51);"><br /></span></p><p><span style="color: rgb(51, 51, 51);">
