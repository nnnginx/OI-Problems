## Description

光头强和熊大熊二在玩随机数比大小的游戏。 具体规则如下：平板电脑会 提供三个 1~6 范围内的数字， 分别交由熊大、熊二和光头强， 数字大的人获胜。
现在熊大和熊二已经拿到了随机数， 请问光头强取得哪几个数字，才能获胜？
注意： 如果光头强的数字与熊大熊二的数字相同， 也算光头强获胜。

## Input Format

一行， 两个正整数 x,y，中间用空格隔开，表示熊大熊二的数字。

## Output Format

一行，若干个整数， 表示光头强的数字。按照从小到大的顺序输出，相邻的数之间用一个空格隔开。

```input1
4 2
```
```output1
4 5 6
```
