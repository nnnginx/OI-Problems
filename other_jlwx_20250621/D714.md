## Description

暑假 小蓝 来到北京旅游，第一次来到北京的 小蓝 非常开心，她想买一些礼物带给自己的好朋友。 小蓝 在礼物店看上了  3  件礼物，价格分别是三个不相等的整数  x  、  y  和  z  元， 小蓝 买了三件礼物中价格最高的一件和价格最低的一件。   
请问 小蓝 总共要花多少钱，平均一件礼物要花多少钱（平均值保留  1  位小数）？   
比如，假设 小蓝 看上的  3  件礼物的价格分别是  8  、  3  、  5  ，那么 小蓝 买的两件礼物的价格分别是  8  和  3  ，总共花了  8+3=11  元，平均一件礼物花了  5.5  元。   
 

## Input Format

三个不相等的整数  x  、  y  、  z  ，分别代表 小蓝 看上的  3  件礼物的价格。   
 

## Output Format

输出有  2  行，第  1  行输出 小蓝 买礼物总共花了多少钱；   
第  2  行输出 小蓝 平均一件礼物花了多少钱，平均值保留  1  位小数。   
 

```input1
8 3 5
```
```output1
11
5.5
```
```input2
30 12 10
```
```output2
33 
16.5
```
