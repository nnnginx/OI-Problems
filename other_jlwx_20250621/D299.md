## Description

梅山小学有一块长方形花圃（花圃的长宽都是整数），长 m 米，宽未知。在修建校园时，花圃的长增加了 n 米，此时发现增加出来的面积有 s 平方米。请编程计算出原来花圃的面积是多少平方米？ 
<img src="http://oj.czos.cn/upload/image/20180622/20180622230203_52538.jpg" alt="" /> 
 

## Input Format

3个整数，分别是m、n、s

## Output Format

一个整数，原来花圃的面积数

```input1
8 3 18
```
```output1
48
```
