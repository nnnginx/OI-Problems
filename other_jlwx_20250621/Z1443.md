## 输入格式

<p>输入数据首先输入两个整数N,M，表示了迷宫的边长。 接下来N行，每行M个字符，描述了迷宫。</p>

## 输出格式

<p>若小AA能够赢得游戏，则输出一行"WIN"，然后输出所有可以赢得游戏的起始位置，按行优先顺序输出 每行一个,否则输出一行"LOSE"（不包含引号）。</p>

```input1
3 3
.##
...
#.#
```
```output1
WIN
2 3
3 2
```
## 数据范围与约定

<p>对于100%的数据，有1≤n,m≤100。 对于30%的数据，有1≤n,m≤5。</p>
