## Description

给定一个N个数的数组，M次操作，每次操作为下列操作之一。求最后的数组。 
操作1：在第X个数之后插入一个数Y。 
操作2：删除第X个数。

## Input Format

第一行两个整数N，M（N，M≤100000），N表示数组中一开始有N个数，M表示M次操作。 
第二行N个整数，表示原来的数组。 
接下来M行，每行第一个数OPT，表示操作类型。 
对于操作1，接下来两个数X，Y，表示在第X数之后插入一个数Y，保证0≤X≤当前数的个数，若X=0，表示在数组开头插入。 
对于操作2，接下来一个数X，表示要删除第X个数，保证1≤X≤当前数的个数

## Output Format

输出若干个数，表示最后的数组。

```input1
5 3
1 2 3 4 5
1 1 6
2 1
2 2
```
```output1
6 3 4 5
```
