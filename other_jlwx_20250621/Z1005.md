## 题目描述

<p>　　自从明明学了树的结构,就对奇怪的树产生了兴趣......给出标号为1到N的点,以及某些点最终的度数,允许在<br>
  任意两点间连线,可产生多少棵度数满足要求的树?</p>

## 输入格式

<p>　　第一行为N(0 < N < = 1000),<br>
  接下来N行,第i+1行给出第i个节点的度数Di,如果对度数不要求,则输入-1</p>

## 输出格式

<p>　　一个整数,表示不同的满足要求的树的个数,无解输出0</p>

```input1
3
1
-1
-1
```
```output1
2
```
## 数据范围与约定

<p>　　两棵树分别为1-2-3;1-3-2</p>
