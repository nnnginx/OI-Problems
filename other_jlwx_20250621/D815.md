## Description

有N种物品和一个容量是V的背包。 
第i种物品最多有si件，每件体积是vi，价值是wi。 
求解将哪些物品装入背包，可使物品体积总和不超过背包容量，且价值总和最大。 
输出最大价值。

## Input Format

第一行两个整数，N，V，用空格隔开，分别表示物品种数和背包容积。
接下来有 N 行，每行三个整数 vi,wi,si，用空格隔开，分别表示第 i 种物品的体积、价值和数量。
 

## Output Format

输出一个整数，表示最大价值。
0&lt;N≤1000 
0&lt;V≤2000 
0&lt;vi,wi,si≤2000
 

```input1
4 5
1 2 3
2 4 1
3 4 3
4 5 2
```
```output1
10
```
