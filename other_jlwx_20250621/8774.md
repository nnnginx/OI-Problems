## Description

<p>给定两个字符串 str1 和 str2 ，如果将 str1 去掉一些字符后得到字符串str，将str2去掉一些字符后也能得到字符串 str ，我们就说 str 是 str1 和 str2 的一个公共子串。当然在得到str的时候不能改变字符的相对顺序。请你编写程序求出字符串 str 长度的最大值。<br /></p>

## Input Format

<p>两行，每行一个只含英文字母的字符串<br /></p>

## Output Format

<p>一个数，表示两个字符串的最长公共子串的长度<br /></p>

```input1
ABCEF
BMCGUAFB
```
```output1
3
```
## Hint

<p>输入的字符串长度均不超过1000<br /></p>
