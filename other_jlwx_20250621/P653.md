## Description

给出数列{$A_1,A_2,...,A_N$}，修改最少的元素，使得数列{$A_N$}成为一个公差为1的等差数列。

## Input Format

第1行，1个整数N。<br />第2行，N个整数$A_1，A_2，……，A_N。$

## Output Format

<p>一个整数，表示最少修改的数字。<br /></p>

```input1
3
2 1 4
```
```output1
1
```
## Hint

【数据范围与约定】

对于40%的数据，$n <= 1000$；

对于100%的数据，$n <= 10^5，1<=A_i<=10^6$。
