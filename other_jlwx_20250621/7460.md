## Description

小明天天沿着未名湖环湖路跑，有时候也觉得蛮累。
累的时候跑下坡就很开心。小明想知道最长的一段下坡有多长。
环湖路是个圆形，周长n米。每隔一米测一下路面高度，两个测高点之间的高度是单调变化或不变的。
问最长的一段下坡有多少米长。小明只能顺时针跑。下坡必须高度单调减少。

## Input Format

第一行是整数n，表示环湖路一共n米长(2<=n<=100)。 第二行是n个整数，每个整数范围[0,10000]，按顺时针顺序给出了n个测高点的高度。

## Output Format

最长下坡路段的长度。

```input1
5
2 1 5 6 3
```
```output1
3
```
```input2
5
2 1 5 4 3
```
```output2
4
```
```input3
4
1 1 1 1
```
```output3
0
```
