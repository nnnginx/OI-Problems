## Description

郊区呈矩形，有四条东西方向的街道和N（1<=N<=1000）条南北方向的街道。

在郊区的西北角有一个邮局。  
如N=5时，郊区如下图所示，圆点表示邮局，直线表示街道。
 ![](http://train.usaco.org/trainpix/postal1.gif) 
每天邮政卡车从邮局出发，每个十字路口（包括边界和四角）经过且只经过一次。现在邮局希望知道邮政货车行驶的路线有几种。 例如，下面两幅图表示的是满足上图的两条路线  
![](http://train.usaco.org/trainpix/postal2.gif)
另一个例子，下面四幅图表示了当N=3时的全部四种情况  。
![](http://train.usaco.org/trainpix/postal3.gif)



## Input Format

一行：一个数值N  。

## Output Format

一行: 到INPUT中给出的街道的路径总数  。

```input1
4  

```
```output1
12
```
