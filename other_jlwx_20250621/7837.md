## Description

从n个不同元素中，任取m(m ≤ n)个元素并成一组，叫做从n个不同元素中取出m个元素的一个组合；
从n个不同元素中取出m(m ≤ n)个元素的所有组合的个数，叫做从n个不同元素中取出m个元素的组合数；用符号c(n,m) 表示。计算公式为：
c(n,m) = n!/((n - m)! * m!) 
其中，n!表示n的阶乘，计算方法：n! = 1 * 2 * 3 * … * n；
0的阶乘定义为1。


## Input Format

输入有1行，包含2个正整数，分别是n，m（1 <= m <= n <= 10 ），两数之间用空格隔开。

## Output Format

输出1个数，为组合数c(n,m)的值。

```input1
5 2
```
```output1
10
```
