## Description

以下是2021年度全国各省份的GDP排行榜前5名：
第1名：广东（Guangdong）
第2名：江苏（Jiangsu）
第3名：山东（Shandong）
第4名：浙江（Zhejiang）
第5名：河南（Henan）
请你设计一个程序，根据输入的名次显示对应省份名称的拼音。

## Input Format

输入一个正整数n，表示年度省份GDP的排行名次。
题目数据保证n的范围是1~5。

## Output Format

按照题目描述，输出对应省份名称的拼音（首字母大写，其余全部是小写，且中间没有空格）。注意：输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
3
```
```output1
Shandong
```
