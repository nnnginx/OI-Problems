## Description

<p>给出两个字符串s1,s2（(只有大写字母），求s2在s1中出现多少次。</p><p>例如：<span style="color: rgb(51, 51, 51);">s1=&quot;ABAABA&quot; ,</span>s2=&quot;ABA&quot;,答案为2。</p>

## Input Format

<p>输入两行表示两个字符串s1和s2。<br /></p>

## Output Format

<p>如题述。<br /></p>

```input1
BAPC
BAPC
```
```output1
1
```
```input2
AZAAZAAZA
AZA
```
```output2
3
```
```input3
AVERDXIVYERDLAN
VEEDI
```
```output3
0
```
## Hint

<p>1≤s1的长度≤10^4​​ ，1≤s2的长度≤10^6​ 。<br /></p>
