## Description

时钟从时间： xx:xx（xx时xx分） ，走到时间： xx:xx （xx时xx分）  ，时针共旋转了多少度？（假设第一个时间 &lt;= 第二个时间， 2 个时间都是 12 小时制，且两个时间的时间差不超过 12 小时，也就是说时针旋转的度数在360度之内，也就是 1 &lt;= 时间1 &lt;= 时间2 &lt;= 12（4.2.4）   
 
<img src="http://43.142.138.134:80/admin/../../upload/43.142.138.134/20220812//1323_1_20220812202315_86018.jpg" alt="" /> 
 

## Input Format

2行，第一行为起始时间（如：01:00），第二行为结束时间（如：01:05）

## Output Format

时针旋转的度数（结果保留1位小数）

```input1
01:00
01:05
```
```output1
2.5
```
## Source

字符串
