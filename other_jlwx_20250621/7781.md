## Description

有一个数列，它的第1项为1，第2项为3（1+2），第3项为6（1+2+3）......第n项为1+2+3+...+n的累加和，请输出这个数列的前n项。
注意：输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

## Input Format

输入1个正整数n（n <= 13），表示这个数列总共有多少项。



## Output Format

输出有1行，包含n个数，表示这个累加和数列的第1到第n项，两个数之间用一个空格隔开。



```input1
4
```
```output1
1 3 6 10
```
