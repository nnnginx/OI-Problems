## Description

 输入一个n×n的黑白图像（1表示黑色，0表示白色），任务是统计其中黑色连通块的个数。如果两个黑格子有公共边或者公共顶点，就说它们属于同一个联通块。如下图所示的图形有3个联通块。
<img src="http://oj.czos.cn/admin/../upload/pimg2068_1.jpg" alt="" /> 
 

## Input Format

第1行输入一个正整数n（n≤700），此后输入n行，每行是由n个0或1组成的字符串。

## Output Format

输出连通块的个数

```input1
6
100100
001010
000000
110000
111000
010100
```
```output1
3
```
