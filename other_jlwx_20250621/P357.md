## Description

有一个集合M是这样生成的：
(1) 已知k是集合M的元素； 
(2) 如果y是M的元素，那么，2y+1和3y+1都是M的元素；
(3) 除了上述二种情况外，没有别的数能够成为M的一个元素。
问题：任意给定k和x，请判断x是否是M的元素。
这里的k是无符号整数，x 不大于 100000，如果是，则输出YES，否则，输出NO。


## Input Format

 输入整数 k 和 x, 逗号间隔。 

## Output Format

 如果是，则输出 YES，否则，输出NO。 

```input1
0,22

```
```output1
YES

```
