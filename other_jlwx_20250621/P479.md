## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">读入一串字符，判断是否是回文串。</span><span style="color: rgb(51, 51, 51);">“回文串”是一个正读和反读都一样的字符串，比如“level”或者“noon”等等就是回文串。</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一行字符串，长度不超过255。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">如果是回文串，输出“YES”，否则输出“NO”。（均为大写）</span><br /></p>

```input1
12321
```
```output1
YES
```
