## Description

小张同学今天在信息技术课堂上学习了利用Excel进行成绩统计的方法，现在他想编写一个这样的程序:输入全级某次测验的成绩，分别统计0～59、60～69、70～79、80～89和90～100各分数段的人数。假如你就是小张，你能把这个程序编出来吗？

## Input Format

输入第一行为全级人数(1000以内)，第二行至最后一行为学生的分数,分数都为0～100间的整数。

## Output Format

输出包括5个分数段的人数，每行一个数。(顺序为0～59分的人数,60～69分的人数,70～79分人数,80～89分人数,90～100分人数)

```input1
7
80
87
90
53
65
68
100
```
```output1
1
2
0
2
2
```
