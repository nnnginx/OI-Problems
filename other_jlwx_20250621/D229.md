## Description

请从键盘读入一个整数n，代表有n行，输出n行的9*9乘法表。
比如，假设n=5，则输出如下：
  
1*'1=1 
2*'1=2 2*'2=4 
3*'1=3 3*'2=6 3*'3=9 
4*'1=4 4*'2=8 4*'3=12 4*'4=16 
5*'1=5 5*'2=10 5*'3=15 5*'4=20 5*5=25
 

## Input Format

一个整数n（n&lt;=9）

## Output Format

n行的9*9乘法表

```input1
3
```
```output1
1*1=1
2*1=2 2*2=4
3*1=3 3*2=6 3*3=9
```
