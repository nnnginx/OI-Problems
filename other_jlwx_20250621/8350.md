## Description

<p style="margin-left: 0in;">N位同学站成一排，音乐老师要请其中的（N-K）位同学出列，使得剩下的K位同学排成合唱队形。</p><p style="margin-left: 0in;">合唱队形是指这样的一种队形：设K位同学从左到右依次编号为1，2，…，K，他们的身高分别为T1，T2，…，TK，则他们的身高满足T1&lt;T2&lt;…&lt;Ti，且Ti&gt;Ti+1&gt;&gt;TK（1≤i≤K）。</p><p style="margin-left: 0in;">本题的任务是，已知所有N位同学的身高，计算最少需要几位同学出列，可以使得剩下的同学排成合唱队形。</p>

## Input Format

<p style="margin-left: 0in;">第1行是1个正整数N，表示同学的总数。</p><p style="margin-left: 0in;">第2行有n个正整数，之间用一个空格分隔，第i个整数Ti（130≤Ti≤230）是第i位同学的身高（厘米）。</p>

## Output Format

<p>一行一个整数，表示最少需要几位同学出列。<br /></p>

```input1
8
186 186 150 200 160 130 197 220

```
```output1
4

```
## Hint

<p style="margin-left: 0in;">对于50%的数据满足：n≤20。</p><p style="margin-left: 0in;">对于100%的数据满足：n≤1000。</p>
