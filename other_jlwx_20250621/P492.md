## Description

<p>【问题描述】</p><p>某石油公司计划建造一条由东向西的主输油管道。该管道要穿过一个有n 口油井的油田。从每口油井都要有一条输油管道沿最短路经(或南或北)与主管道相连。如果给定n口油井的位置，即它们的x 坐标（东西向）和y 坐标（南北向），应如何确定主管道的最优位置，即使各油井到主管道之间的输油管道长度总和最小的位置？证明可在线性时间内确定主管道的最优位置。</p><p><br /></p><p>【编程任务】</p><p>给定n 口油井的位置,编程计算各油井到主管道之间的输油管道最小长度总和。</p>

## Input Format

<p style="">输入的第1 行是油井数n，1≤n≤10000。接下来n 行是油井的位置，每行2个整数x和y，-10000≤x,y≤10000。<br /></p>

## Output Format

<p>程序运行结束时，将计算结果输出。</p><p>输出的第1 行中的数是油井到主管道之间的输油管道最小长度总和。<br /></p>

```input1
5
1 2
2 2
1 3
3 -2
3 3
```
```output1
6
```
