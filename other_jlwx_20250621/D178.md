## Description

将一个长度小于10位的数字串拆成2段，使其和为最小的素数。 
例如数字串‘13304’; 
拆的方法有:; 
1 + 3304 = 3305; 
13 + 304 = 317; 
133 + 04 = 137; 
1330 + 4 = 1334; 
从上面可看出，和为素数的有：317 与137，最小的是137 
 

## Input Format

一个长度小于10的数字串  

## Output Format

最小的和为素数的数，若无素数则输出 -1 

```input1
13304
```
```output1
137
```
