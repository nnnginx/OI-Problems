## Description

<p>对于给定的一个长度为N的正整数数列A[i]，现要将其分成连续的若干段，并且每段和不超过M（可以等于M），问最少能将其分成多少段使得满足要求。<br /></p>

## Input Format

<p>第1行包含两个正整数N，M，表示了数列A[i]的长度与每段和的最大值；</p><p>第2行包含N个空格隔开的非负整数A[i]，如题目所述。</p>

## Output Format

<p>一个正整数，输出最少划分的段数。<br /></p>

```input1
5 6 
4 2 4 5 1
```
```output1
3
```
## Hint

<p>【数据范围】</p><p>对于20%的数据，有N≤10；</p><p>对于40%的数据，有N≤1000；</p><p>对于100%的数据，有N≤100000，M≤10^9，M大于所有数的最小值，A[i]之和不超过10^9。</p>
