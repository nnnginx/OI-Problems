## 题目描述

<p><span style="font-size: medium">　　有一个长度为n的数组{a1,a2,...,an}。m次询问，每次询问一个区间内最小没有出现过的自然数。<br></span></p>

## 输入格式

<p><font size="4">　　第一行n,m。<br>
   　　第二行为n个数。<br>
   　　从第三行开始，每行一个询问l,r。<br></font></p>

## 输出格式

<p><font size="4">　　一行一个数，表示每个询问的答案。<br></font></p>

```input1
5 5
2 1 0 2 1
3 3
2 3
2 4
1 2
3 5
```
```output1
1
2
3
0
3
```
## 数据范围与约定

<p><span style="font-size: medium">数据规模和约定<br><br>
   　　对于100%的数据：<br><br>
   　　1<=n,m<=200000<br><br>
   　　0<=ai<=109<br><br>
   　　1<=l<=r<=n</span></p>
<br>
<p><span style="font-size: medium">　　对于30%的数据：</span></p>
<br>
<p><span style="font-size: medium">　　1<=n,m<=1000<br><br></span></p>
