## Description

　　给定一个二元一次方程组，形如：
　　a * x + b * y = c;
　　d * x + e * y = f;
　　x,y代表未知数，a, b, c, d, e, f为参数。
　　求解x,y

## Input Format

输入包含六个整数: a, b, c, d, e, f;

## Output Format

输出为方程组的解，两个整数x, y。

```input1
3 7 41 2 1 9
```
```output1
2 5
```
## Hint

0 &lt;= a, b, c, d, e, f &lt;= 2147483647
