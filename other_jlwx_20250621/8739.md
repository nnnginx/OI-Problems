## Description

<p> 已知矩阵的大小定义为矩阵中所有元素的和。给定一个矩阵，你的任务是找到最大的非空(大小至少是1×11×1)子矩阵。<p><p>比如，如下4×4的矩阵<p>

## Input Format

<p>输入是一个N×N的矩阵。输入的第一行给出N(0<N≤100)。再后面的若干行中，依次(首先从左到右给出第一行的N个整数，再从左到右给出第二行的N个整数……)给出矩阵中的N2个整数，整数之间由空白字符分隔(空格或者空行)。已知矩阵中整数的范围都在[−127,127]。<p><p> <p>

## Output Format

<p>输出最大子矩阵的大小。<p>

```input1
  0  -2   -7   0

  9   2   -6   2 

-4    1   -4   1

-1    8    0  -2

```
```output1
  9   2

-4   1

-1   8

```
