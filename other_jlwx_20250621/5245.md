# B4037 [GESP202409 二级] 小杨的 N 字矩阵

## 题目描述

小杨想要构造一个 $m \times m$ 的 $N$ 字矩阵（$m$ 为奇数），这个矩阵的从左上角到右下角的对角线、第 $1$ 列和第 $m$ 列都是半角加号 `+` ，其余都是半角减号 `-` 。例如，一个 $5 \times 5$ 的 N 字矩阵如下：

```plain
+---+
++--+
+-+-+
+--++
+---+
```

请你帮小杨根据给定的 $m$ 打印出对应的 N 字矩阵。

## 输入格式

输入只有一行包含一个正整数 $m$。

## 输出格式

输出对应的 $N$ 字矩阵。

## 输入输出样例 #1

### 输入 #1

```
5
```

### 输出 #1

```
+---+
++--+
+-+-+
+--++
+---+
```

## 说明/提示

### 数据规模与约定

对全部的测试数据，保证 $3 \leq m \leq 49$ 且 $m$ 是奇数。
