## Description

学校新买来 m 根跳绳，每个班分 n 根，最多可以分给几个班的同学，还剩多少根？（ m&gt;=n ）（4.1.9）
 
<img src="http://oj.czos.cn/upload/image/20180622/20180622221758_66188.jpg" alt="" />
 
 
	  
 

## Input Format

两个整数，用空格隔开，分别代表总共采购的跳绳数量以及每个班级分到的跳绳数量

## Output Format

两个整数，用空格隔开，第一个整数代表可以分到的班级的数量，第二个整数代表剩余的跳绳数量

```input1
100 30
```
```output1
3 10
```
