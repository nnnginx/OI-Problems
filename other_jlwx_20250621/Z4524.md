## 题目描述

<div>
 若一个大于1的整数M的质因数分解有k项，其最大的质因子为A<sub>k</sub>，并且满足A<sub>k</sub>^K<=N,A<sub>k</sub><128，我们就称整数M为N-伪
</div> 
<div>
 光滑数。现在给出N，求所有整数中，第K大的N-伪光滑数。
</div>

## 输入格式

<div>
 只有一行，为用空格隔开的整数N和K
</div> 
<div>
 2 ≤ N ≤ 10^18， 1 ≤ K ≤ 800000，保证至少有 K 个满足要求的数
</div> 
<div></div>

## 输出格式

<div>
 只有一行，为一个整数，表示答案。
</div> 
<div></div>

```input1
12345 20
```
```output1
9167
```
