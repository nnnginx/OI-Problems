## Description

编制一个乘法运算的程序。
从键盘读入2个100以内的正整数，进行乘法运算并以竖式输出。

## Input Format

　输入只有一行，是两个用空格隔开的数字，均在1~99之间（含1和99）。

## Output Format

输出为4行或7行，符合乘法的竖式运算格式。

```input1
89 13
```
```output1
89
×13
━━━
267
89
━━━
1157
```
## Hint

　3×89=267，则第四行267右侧对准个位输出。1×89=89，则第五行89右侧对准十位输出。267+890=1157，则1157右侧对准个位输出。
