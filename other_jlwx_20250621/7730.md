## Description

学校举办了一场足球比赛，有n个班级参加。输入每个班级的进球总数和失球总数，并计算出净胜球数（净胜球数=进球总数-失球总数）。最后按输入的顺序输出每个班级的净胜球数。


## Input Format

输入：第1行有1个整数n（2≤n≤100），表示参赛的班级数量。
接下来有n行数据，每行包含2个正整数，分别表示每个班的进球总数和失球总数，两个数中间用一个空格分隔。
题目保证输入和输出数据都在整型范围之内。


## Output Format

输出：共n行；
每行1个整数，表示一个班级的净胜球数。


```input1
3
6 12
10 3
3 4

```
```output1
-6
7
-1

```
