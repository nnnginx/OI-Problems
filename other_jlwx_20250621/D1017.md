## Description

小明有一堆 1分、2分、5分的硬币。 小明想由n个硬币组成总面值为m分，求一共有多少种可能的组合方式（某种面值的硬币可以数量可以为0）。

## Input Format

输入数据第一行有一个正整数T（1&lt;=T&lt;=50），表示有T组测试数据。接下来的T行，每行有两个数n，m，n和m的含义同上(1&lt;=n,m&lt;=5000)。

## Output Format

对于每组测试数据，请输出可能的组合方式数，每组输出占一行。

```input1
2
3 5
4 8
```
```output1
1
2
```
## Source

需要找规律的循环 数组
