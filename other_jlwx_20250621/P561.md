## Description

现请你输出指定大小的“ACM”字符串。特别地，我们要求输出的字符串是正方形的（行数和列数相等）。

## Input Format

输入的第一行是一个正整数N（N&lt;=20），表示一共有N组数据，接着是N行数据，每行包含一个正整数M（M&lt;=50），表示一行内有M个“ACM”相连。

## Output Format

输出指定的正方形字符串。

```input1
2
1
2
```
```output1
ACM
ACM
ACM
ACMACM
ACMACM
ACMACM
ACMACM
ACMACM
ACMACM
```
