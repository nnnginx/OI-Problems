## Description

每门课的成绩分为A、B、C、D、F五个等级，为了计算平均绩点，规定A、B、C、D、F分别代表4分、3分、2分、1分、0分。

## Input Format

从键盘读入一个整数n（3&lt;n&lt;100）,代表有n组成绩。每一组成绩输入数据占一行，由一个或多个大写字母组成(不含空格等其他字符)

## Output Format

每组输出结果占一行。如果输入的大写字母都在集合｛A,B,C,D,F｝中，则输出对应的平均绩点，结果保留两位小数。否则，输出“Unknown”。

```input1
3
ABCDF
BFFCCA
DCEF
```
```output1
2.00
1.83
Unknown
```

数组
