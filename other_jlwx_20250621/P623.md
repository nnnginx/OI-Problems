## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">又到学期末，小明迎来了又一次的期末考试。</span><span style="color: rgba(0, 0, 0, 0.87);">考完试后，小明拿到了全班的成绩单，这张成绩单是按学号顺序排好的。小明很想知道班里到底有多少人分数比他高，现在就请你帮帮他，帮他数一下到底有多少人的分数比他高吧。</span></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输入数据的第一行是一个正整数T，表示测试数据的组数，接下来有T组测试数据。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">每组数据包括两行。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">第一行有两个正整数N，K（0&lt;N&lt;1000，0&lt;K&lt;=N）,分别表示成绩单上总共的学生数目，和小明的学号。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">第二行有N个整数Xi（0&lt;=Xi&lt;=100）分别表示各个学生的成绩，以学号递增顺序给出，第一个学生学号为1。</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">对于每组数据，请在一行里输出班里一共有多少个学生成绩高于小明。</span><br /></p>

```input1
1
3 2
81 72 63
```
```output1
1
```
