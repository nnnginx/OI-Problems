## Description

<p>输入$n$代表有个$n×n$的棋盘，输入开始位置的坐标和结束位置的坐标，问一个骑士朝棋盘的八个方向走马字步，从开始坐标到结束坐标可以经过多少步。<p><p>![](pic/1257.gif)<p>

## Input Format

<p>首先输入一个$n$,表示测试样例的个数。<p><p>每个测试样例有三行。<p><p>第一行是棋盘的大小$L(4≤L≤300)$；<p><p>第二行和第三行分别表示马的起始位置和目标位置$(0..L-1)$。<p>

## Output Format

<p>马移动的最小步数，起始位置和目标位置相同时输出$0$。<p>

```input1
3
8
0 0
7 0
100
0 0
30 50
10
1 1
1 1

```
```output1
5
28
0


```
