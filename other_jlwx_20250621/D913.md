## Description

小明喜欢8这个数，他认为这个数字很吉祥，请编程帮助小明求解1~n中，总共有多少个数字8，如果一个数字中含有多个数字8，那么都要算上，比如：88中有2个数字8。（请注意：本题不是要求有多少个数字含有8，而是要求总共出现了多少个8）
 
如果输入n=100，那么输出应该是20，因为100以内含有8的数有：8 18 28 38 48 58 68 78 80 81 82 83 84 85 86 87 88 89 98，其中有20个8出现了。
 

## Input Format

输入一个整数n（n&lt;10000）；

## Output Format

输出1~n中有多少个数字8； 

```input1
100
```
```output1
20
```
