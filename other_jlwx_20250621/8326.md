## Description

<p>某工厂生产一批棍状零件，每个零件都有一定的长度$（Li）$和重量$（Wi）$。</p><p>现在为了加工需要，要将它们分成若干组，使每一组的零件都能排成一个长度和重量都不下降（若$ i&lt;j$，则 $Li&lt;=Lj，Wi&lt;=Wj$）的序列。请问至少要分成几组?<br /></p>

## Input Format

<p>第一行为一个整数 $N（N&lt;=1000）$，表示零件的个数。</p><p>第二行有 $N $对正整数，每对正整数表示这些零件的长度和重量，长度和重量均不超过 $10000$。<br /></p>

## Output Format

<p>仅一行，即最少分成的组数。<br /></p>

```input1
5
8 4 3 8 2 3 9 7 3 5
```
```output1
2
```
