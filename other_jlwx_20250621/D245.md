## Description

 
	 求 <span style="font-size:10.5pt;font-family:&quot;">2+2*2+2*2*2+…+2*2*2*….*2的和是多少？最后一项有多少2相乘由键盘读入的n决定（1&lt;=n&lt;=100）！  
 
 
	<span style="font-size:10.5pt;font-family:&quot;">比如：n=3，那么s=2+2*2+2*2*2=14！ 
  
 

## Input Format

从键盘读入一个整数n（1&lt;=n&lt;=100）

## Output Format

输出求出的和

```input1
3
```
```output1
14
```
