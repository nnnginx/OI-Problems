## Description

小X喜欢研究进制转换。  
在了解了进制转换的一般流程后，小X突然想起了以前学过的回文数（正着读倒着读都一样的数），于是开始思考一个奇怪的问题：1到N 
中有多少个整数的平方在M进制下是回文数呢？&nbsp;  
小X随手列了几个：  
2的平方4，10进制表示为4，是回文数；  
3的平方9，2进制表示为 1001，是回文数；  
9046的平方81830116，16进制表示为4E0A0E4，是回文数。  
&nbsp;小X觉得要全列出来太难了，希望你帮帮他。
 

## Input Format

第一行包含用一个空格隔开的两个整数N,M。
 

## Output Format

第一行包含一个整数，表示满足要求的整数个数。
 

```input1
2 10
```
```output1
2
```
## Hint

数据范围 
&nbsp;&nbsp; 对于30%的数据，M=10。 
&nbsp;&nbsp; 对于另外30%的数据，M=2。 
&nbsp;&nbsp; 对于 100%的数据，1≤N≤10000，2≤M≤16。 
 
 
来源 
常州市2015“信息与未来”夏令营选拔赛
