## Description

请从键盘读入一个整数n（n&lt;=100），代表一个班级同学的人数，然后读入n个人的语文、数学、英语成绩；请求出这n个人的语文、数学、英语三科成绩的平均分分别是多少，结果保留1位小数。

## Input Format

第一行：一个整数n，代表班级的人数！ 
第2行~第n+1行，输入n个同学的语文、数学、英语成绩，每行输入一个同学的成绩，成绩用空格隔开！ 

## Output Format

输出语文、数学、英语三科的平均成绩，分别用空格隔开，平均成绩保留1位小数！

```input1
2
100 99 98
99 98 97
```
```output1
99.5 98.5 97.5
```
## Source

二维数组
