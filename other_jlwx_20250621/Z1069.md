## 题目描述

<p>　　在某块平面土地上有N个点，你可以选择其中的任意四个点，将这片土地围起来，当然，你希望这四个点围成<br>
  的多边形面积最大。</p>

## 输入格式

<p>　　第1行一个正整数N，接下来N行，每行2个数x,y，表示该点的横坐标和纵坐标。</p>

## 输出格式

<p>　　最大的多边形面积，答案精确到小数点后3位。</p>

```input1
5
0 0
1 0
1 1
0 1
0.5 0.5
```
```output1
1.000
```
## 数据范围与约定

<p><span style="font-size: medium">数据范围 n<=2000, |x|,|y|<=100000</span></p>
