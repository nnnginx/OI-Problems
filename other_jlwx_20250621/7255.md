## Description


现代诗如蚯蚓
断成好几截都不会死
字符串断成好几截
有可能完全一样
请编写程序
输入字符串
输出该字符串最多能断成多少截完全一样的子串

## Input Format

一行，一个字符串。

## Output Format

一行，一个正整数表示该字符串最多能断成的截数。

```input1
abcabcabcabc
```
```output1
4
```
## Hint

最多能断成四个”abc”，也就是abc重复四遍便是原串<br />同时也能断成两个”abcabc”<br />最坏情况是断成一个原串”abcabcabcabc”
