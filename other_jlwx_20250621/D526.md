## Description

小明认为，如果以下条件之一成立，正整数 x 就是一个幸运数字:
1. x是4的倍数;
2. x是7的倍数;
3. x 转换为字符串后包含子串“44”或“77”。
例如，105 (7 的倍数), 442 (包含“44”), 1284 (4 的倍数)都是幸运数字，474 则不是。 编程统计 1 到 n 中幸运数字的数量。


## Input Format

整数 n (1 ≤ n ≤ 1,000,000)

## Output Format

一个整数:1, 2, ..., n中幸运数字的数量

```input1
32
```
```output1
11
```
## Hint

样例1:  
32 
样例2:  
987654 
样例输出 
样例1:  
11 
样例2:  
406754
