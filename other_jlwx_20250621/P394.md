## Description

<p>给定一个大小为 N*M 的迷宫。迷宫由通道和墙壁组成，每一步可以向邻接的上下左右四格的通道移动。请求出从起点到终点所需的最小步数。测试数据保证从起点一定可以移动到终点。<br /></p>

## Input Format

<p>第一行包含两个正整数 N 和 M 分别表示迷宫的行数和列数，接下来的 N 行，每行包含 M 个字符描述迷宫的结构用 &#039;#&#039;表示墙壁， &#039;.&#039;表示通道，&#039;S&#039;表示起点， &#039;G&#039;表示终点。<br /></p>

## Output Format

<p>一个数，表示最小步数。<br /></p>

```input1
10 10
#S######.#
......#..#
.#.##.##.#
.#........
##.##.####
....#....#
########.#
....#.....
.####.###.
....#...G#
```
```output1
22
```
## Hint

<p>数据范围：0&lt;N，M≤100。<br /></p>
