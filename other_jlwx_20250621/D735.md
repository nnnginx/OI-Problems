## Description

输入 n 个整数，代表角的度数，请编程计算出这些角中直角（ 90 度的角）、锐角（ &lt;90 度的角）、钝角（ &gt;90 度的角）分别有多少个？    
 

## Input Format

第 1 行有一个整数 n ，代表有 n 个角的度数。（ n&lt;=100 ）    
第 2 行有 n 个整数，代表角的度数（角的度数是 &gt;=1 且 &lt;180 的整数）。    
 

## Output Format

输出 3 个整数，用空格隔开，分别代表直角的个数、锐角的个数、钝角的个数。    
 

```input1
6
178 45 60 90 90 12
```
```output1
2 3 1
```
