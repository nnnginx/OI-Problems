## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">定义：一个词组中每个单词的首字母的大写组合称为该词组的缩写。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">比如，C语言里常用的EOF就是end of file的缩写。</span></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输入的第一行是一个整数T，表示一共有T组测试数据。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">接下来有T行，每组测试数据占一行，每行有一个词组，每个词组由一个或多个单词组成；每组的单词个数不超过10个，每个单词有一个或多个大写或小写字母组成；</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">单词长度不超过10，由一个或多个空格分隔这些单词。</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">请为每组测试数据输出规定的缩写，每组输出占一行。</span><br /></p>

```input1
1
end of file
```
```output1
EOF
```
