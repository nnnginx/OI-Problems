## 题目描述

<p><span style="font-size: medium">给出一个由小写字母组成的R*C的矩阵。在这个矩阵中，有一个边平行于坐标<br>
   轴的V个顶点的简单多边形。多边形的顶点按顺时针依次给出。你可以将多边形上<br>
   下左右任意移动。问有多少个能够由给出的多边形移动得到的多边形，其内部仅含<br>
   有一种字母。 <br>
    <br>
    </span></p>

## 输入格式

<p><span style="font-size: medium">第一行给出R，C</span></p>
<p><span style="font-size: medium">接下来R行C列</span></p>
<p><span style="font-size: medium">接下来给出数字V</span></p>
<p><span style="font-size: medium">接下来V行，每行两个数描述顶点坐标</span></p>

## 输出格式

<p><span style="font-size: medium">如题</span></p>

```input1
3 3
aaa
aaa
aaa
4
2 0
2 2
0 2
0 0
```
```output1
4
```
## 数据范围与约定

<p><font size="4">R, C, V <= 500. </font><br><br></p>
