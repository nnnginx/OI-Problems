## Description

根据输入的正整数n，用星号画出如图所示的空心平行四边形图案。
例如n = 5时，图案为：
```
    *****
   *   *
  *   *
 *   *
*****
```



## Input Format

输入1个正整数n（3 <= n <= 20），表示这个平行四边形的边由n个星号组成。


## Output Format

根据题目描述，输出对应的图案。

```input1
3
```
```output1
  ***
 * *
***
```
```input2
4
```
```output2
   ****
  *  *
 *  *
****
```
## Hint

图案的第一行前面不能有多余的空行；
图案的第一行和最后一行中，左右相邻的两个星号之间没有空格。
图案的最左侧从第一列开始输出，左边不能有多余的空格。
