## Description

<p>在一个数轴上有n条线段，现选取其中k条线段使得这k条线段两两没有重合部分，问最大的k为多少?<br /></p>

## Input Format

<p>第一行为一个正整数n，下面n行每行2个数字ai，bi,描述每条线段。<br /></p>

## Output Format

<p>输出文件仅包括1个整数，为k的最大值。<br /></p>

```input1
3
0 2
2 4
1 3
```
```output1
2
```
## Hint

<p>【数据规模】</p><p>对于20%的数据，n≤10。</p><p>对于50%的数据，n≤1000。</p><p>对于70%的数据，n≤100000。</p><p>对于20%的数据，n≤1000000,0≤ai&lt;bi≤1000000。</p>
