## Description

市里组织了一次青少年足球大赛，阳光队和雷霆队两支队伍最终进入决赛。   决赛共分  3  场，获胜场次多的那队将获得最终的总冠军。
  
请你编程输入两支队伍  3  场比赛的得分，计算哪一支队伍将获得最终的总冠军。 
  
如果阳光队获胜，输出“  yangguang  ”，如果雷霆队获胜，输出“  leiting  ”，由于每场比赛一定会分出胜负，因此不存在最终平局的情况。   
 

比如，假设阳光队和雷霆队三场比赛的得分依次为  3 2 1 2 3 1  ；   
前两个数代表第  1  场比赛阳光和雷霆两队的比分为  3:2  ，阳光队胜出；   
中间两个数代表第  2  场比赛阳光和雷霆两队的比分为  1:2  ，雷霆队胜出；   
最后两个数代表第  3  场比赛阳光和雷霆两队的比分为  3:1  ，阳光队胜出。   
阳光队胜  2  场，雷霆队胜  1  场，因此阳光队获得总冠军。 

## Input Format

6  个整数，分别代表  3  场比赛中阳光和雷霆两队的得分。   
 

## Output Format

获胜的队名，如果阳光队获胜输出“  yangguang  ”，如果雷霆队获胜输出“  leiting  ”。   
 

```input1
3 2 1 2 3 1
```
```output1
yangguang
```
```input2
1 2 2 0 3 0
```
```output2
yangguang
```
```input3
1 2 1 2 1 2
```
```output3
leiting
```
