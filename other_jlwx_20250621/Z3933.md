## 输入格式

<p>输入文件共三行，第一行为一个正整数n，第二行为一个非负整数t，第三行为一个非负整数m。</p>
<div></div>

## 输出格式

<p>输出一行，为N的值。</p>
<div></div>
<div></div>

```input1
3
2
2
```
```output1
10536
```
## 数据范围与约定

<p> 对于100%的数据，0<n≤103000，0≤t≤10000，0≤n-m≤5</p>
