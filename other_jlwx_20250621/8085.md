## Description

<p>给定一个含N个元素的数组A，下标从1开始。请找出下面式子的最大值：</p><p><img alt="360截图20201201100007980.jpg" src="/public/upload/737b00f528.jpg" width="995" height="69" /><br /></p>

## Input Format

<p>输入数据的第一行包含一个整数N，表示数组中的元素个数。</p><p>第二行包含N个整数A1,A2,…,AN。</p>

## Output Format

<p>输出一行包含给定表达式可能的最大值。<br /></p>

```input1
5
1 2 3 1 2
```
```output1
6
```
