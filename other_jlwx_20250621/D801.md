## Description

小蓝 把合数 483 的各位数字相加 (4+8+3)=15 ，再将 483 分解成质因数相乘： 483=3*7*23 ，把这些质因数各位数字相加 (3+7+2+3) ，其和也为 15 。即某合数的各位数字之和等于它所有质因数的各数字之<span class="GramE">和 。你帮助 小蓝 把 n 以内 ( 包括 n) 具有上述特点的所有合数找出来好吗 ? 

## Input Format

一个整数 n(10 ≤ n ≤ 30000) 。  

## Output Format

若干行，每行输出 5 个满足题目条件的数。

```input1
100
```
```output1
4 22 27 58 85
94

```
