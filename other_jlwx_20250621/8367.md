## Description

<p>假设在周末舞会上，男士们和女士们进入舞厅时，各自排成一队。跳舞开始时，依次从男队和女队的队头上各出一人配成舞伴。规定每个舞曲只能有一对跳舞者。若两队初始人数不相同，则较长的那一队中未配对者等待下一轮舞曲。现要求写一个程序，模拟上述舞伴配对问题。<br /></p>

## Input Format

<p style="margin-left: 0in;">第1行两个正整数，表示男士人数m和女士人数n，1≤m，n≤1000；</p><p style="margin-left: 0in;">第2行一个正整数，表示舞曲的数目k，k≤1000。</p>

## Output Format

<p><span style="color: rgb(0, 0, 102);">共</span><span style="color: rgb(0, 0, 102);">k</span><span style="color: rgb(0, 0, 102);">行，每行两个数，之间用一个空格隔开，表示配对舞伴的序号，男士在前，女士在后。</span><br /></p>

```input1
2 4
6

```
```output1
1 1
2 2
1 3
2 4
1 1
2 2
```
