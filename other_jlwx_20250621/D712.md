## Description

小蓝 是一个刚刚学会算术的小朋友，她非常热衷于计算；这天她发现自己的体重正好是爸爸体重的各个位的和，这也太巧了！   

请从键盘读入 小蓝 爸爸的体重（已知 小蓝 爸爸的体重是一个三位数），计算出 小蓝 有多重？   
比如：假设 小蓝 爸爸的体重是  128  公斤，那么 小蓝 的体重   = 1 + 2 + 8 = 11  。 
 

## Input Format

输入一个整数，代表 小蓝 爸爸的体重（已知 小蓝 爸爸的体重是一个三位数）。   
 

## Output Format

 一个整数，代表 小蓝 的体重。   
 

```input1
128
```
```output1
11
```
```input2
150
```
```output2
6
```
```input3
200
```
```output3
2
```
