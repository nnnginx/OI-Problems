## Description

<p>棋盘上A点有一个过河卒，需要走到目标B点。卒行走的规则：可以向下、或者向右。同时在棋盘上的某一点有一个对方的马（如C点），该马所在的点和所有跳跃一步可达的点称为对方马的控制点，如图3-1中的C点和P1，……，P8，卒不能通过对方马的控制点。棋盘用坐标表示，A点(0,0)、B点(n, m) (n,m为不超过20的整数),同样马的位置坐标是需要给出的，C≠A且C≠B。现在要求你计算出卒从A点能够到达B点的路径的条数。<p><p>![](pic/1314.gif)<p>

## Input Format

<p>给出n、m和C点的坐标。<p>

## Output Format

<p>从A点能够到达B点的路径的条数。<p>

```input1
8 6 0 4

```
```output1
1617

```
