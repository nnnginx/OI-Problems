## Description

<p>给定两个不同的正整数 a 和 b ，如果 a 的因子和等于 b ，b 的因子和等于 a ，且 a ≠ b，则 a 和 b 一对亲密数。给定正整数 N ,求 2-N 中的亲密数对。<br /></p>

## Input Format

<p>第 1 行一个正整数 N 。<br /></p>

## Output Format

<p>输出若干行，每行有两个用一个空格隔开的正整数，表示一对亲密数。<br /></p>

```input1
200
```
```output1
48 75
75 48
140 195
195 140
```
## Hint

<p>数据范围：1 &lt;= N &lt;= 2000。<br /></p>
