## Description

苹果丰收了，有 n 堆苹果，小红就在苹果堆旁。小红已经知道了每堆苹果有多少个。
她要问一问从第 a 堆到第 b 堆一共有多少个苹果。

## Input Format

输入数字 n，然后输入 n 个数据。再输入问 m，然后输入 c 行数据。

## Output Format

输出 m 次 a 到 b 堆一共有多少个。

```input1
5
1 2 3 4 5
3
1 3
2 4
1 5
```
```output1
6
9
15
```
## Hint

【数据范围】
对于80% 的数据：0≤n≤10000；
对于100% 的数据：0≤n≤100000。
