## Description

根据输入的正整数n，画出如图所示的反写字母“Z”形状的字符图案。
例如n = 2时，图案为：
```
*****
 *
  *
   *
*****
```



## Input Format

输入1个正整数n（2 <= n <= 20），表示这个图案占2 * n + 1行和2 * n + 1列。

## Output Format

根据题目描述，输出对应的图案。

```input1
2
```
```output1
*****
 *
  *
   *
*****
```
```input2
3
```
```output2
*******
 *
  *
   *
    *
     *
*******
```
## Hint

图案的第一行前面不能有多余的空行；
图案的最左侧从第一列开始输出，左边不能有多余的空格；
第一行和最后一行相邻的两个星号中间没有空格。
