## 题目描述

<p>Xaviera现在遇到了一个有趣的问题。<br>
  平面上有N个点，Xaviera想找出周长最小的三角形。<br>
  由于点非常多，分布也非常乱，所以Xaviera想请你来解决这个问题。<br>
  为了减小问题的难度，这里的三角形也包括共线的三点。<br></p>

## 输入格式

<p>第一行包含一个整数N表示点的个数。<br>
  接下来N行每行有两个整数，表示这个点的坐标。<br></p>

## 输出格式

<p>输出只有一行，包含一个6位小数，为周长最短的三角形的周长（四舍五入）。<br></p>

```input1
4
1 1
2 3
3 3
3 4
```
```output1
3.414214
```
## 数据范围与约定

<p>100%的数据中N≤200000。<br><br></p>
