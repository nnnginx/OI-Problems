## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">求Sn=1!+2!+3!+4!+5!+…+n!之值，其中n是一个数字。</span><br /></p><p><span style="color: rgba(0, 0, 0, 0.87);"><br /></span></p><p><span style="color: rgba(0, 0, 0, 0.87);">提示：<span style="color: rgba(0, 0, 0, 0.87);">n的取值比较大，Sn需要定义成long long型；</span></span></p>

## Input Format

<p>输入一个整数N</p>

## Output Format

<p>输出一个数字，表示1~N的阶乘和</p><p>即：<span style="color: rgba(0, 0, 0, 0.87);">Sn=1!+2!+3!+4!+5!+…+n!</span></p><p><span style="color: rgba(0, 0, 0, 0.87);"><br /></span></p><p><span style="color: rgba(0, 0, 0, 0.87);">提示：<span style="color: rgba(0, 0, 0, 0.87);">如使用printf，输出则要使用%lld</span></span></p>

```input1
5
```
```output1
153
```
