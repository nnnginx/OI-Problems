## Description

<p>帅帅经常跟同学玩一个矩阵取数游戏：对于一个给定的n*m的矩阵，矩阵中的每个元素aij均为非负整数。游戏规则如下：</p><p>1.每次取数时须从每行各取走一个元素，共n个。m次后取完矩阵所有元素；</p><p>2.每次取走的各个元素只能是该元素所在行的行首或行尾；</p><p>3.每次取数都有一个得分值，为每行取数的得分之和，每行取数的得分=被取走的元素值*2i，其中i表示第i次取数（从1开始编号）；</p><p>4.游戏结束总得分为m次取数得分之和。</p><p>帅帅想请你帮忙写一个程序，对于任意矩阵，可以求出取数后的最大得分。</p>

## Input Format

<p>包括n+1行：</p><p>第1行为两个用空格隔开的整数n和m。</p><p>第2~n+1行为n*m矩阵，其中每行有m个用单个空格隔开的非负整数。</p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">仅包含</span><span style="color: rgba(0, 0, 0, 0.87);">1</span><span style="color: rgba(0, 0, 0, 0.87);">行，为一个整数，即输入矩阵取数后的最大得分。</span><br /></p>

```input1
2 3
1 2 3
3 4 2
```
```output1
82
```
## Hint

<p>【样例解释】</p><p>第1次：第1行取行首元素，第2行取行尾元素，本次得分为1*21+2*21=6</p><p>第2次：两行均取行首元素，本次得分为2*22+3*22=20</p><p>第3次：得分为3*23+4*23=56。总得分为6+20+56=82</p><p><br /></p><p>【数据范围】</p><p>60%的数据满足：1&lt;=n,m&lt;=30，答案不超过10^16</p><p>100%的数据满足：1&lt;=n,m&lt;=80, 0&lt;=a_ij&lt;=1000</p><p><br />
