## 题目描述

<div>
 兔子们在玩k个串的游戏。首先，它们拿出了一个长度为n的数字序列，选出其中的一
</div>
<div>
 个连续子串，然后统计其子串中所有数字之和（注意这里重复出现的数字只被统计一次）。
</div>
<div>
 兔子们想知道，在这个数字序列所有连续的子串中，按照以上方式统计其所有数字之和，第
</div>
<div>
 k大的和是多少。
</div>
<div></div>
<p></p>

## 输入格式

<div>
 第一行，两个整数n和k，分别表示长度为n的数字序列和想要统计的第k大的和
</div>
<div>
 接下里一行n个数a_i，表示这个数字序列
</div>
<div></div>
<p></p>

## 输出格式

<div>
 一行一个整数，表示第k大的和
</div>
<div></div>
<p></p>

```input1
7 5
3 -2 1 2 2 1 3 -2
```
```output1
4
```
## 数据范围与约定

<p>1 <= n <= 100000, 1 <= k <= 200000, 0 <= |a_i| <= 10^9数据保证存在第 k 大的和</p>
<br>
