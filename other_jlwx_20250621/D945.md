## Description

一个长度为len的字符串，不断的重复多次之后，会得到一个新的字符串，比如：字符串wto，重复2次就得到了一个字符串wtowto。重复4次就得到一个字符串wtowtowtowto，虽然这个字符串也可以认为是由wtowto重复2次得到，但是如果讨论使用最少的字符串重复，那么应是由wto重复4次得到。
 
给你一个字符串，请问该字符串是由最短多长的字符串经过若干次重复得到？
 
注意：也可以经过1次重复得到，比如字符串"abcd"，不能由其某个子串重复多次得到，可以认为是由"abcd"重复一次得到。
 

## Input Format

一行字符串，中间无空格。（字符串长度不超过500）

## Output Format

输出一个整数，代表结果。

```input1
wtowtowto
```
```output1
3
```
## Source

字符串
