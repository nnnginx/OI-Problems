## Description

给定一个只包含小写字母的字符串，请编写程序从左往右找到第一个只出现k次的字符。如果没有，输出no。
例如：字符串为ebacabcc，k=2时，b和a都只出现了2次，由于b先出现，所以程序输出b。


## Input Format

输入仅1行，包含一个字符串和一个正整数k；字符串长度小于200，仅包含小写英文字母。
两项输入数据之间用一个空格隔开。


## Output Format

输出1行，如果有只出现k次的字符，则输出第一个满足条件的字符；否则输出no。

```input1
ebacabcc 2
```
```output1
b
```
```input2
abcdddd 3
```
```output2
no
```
