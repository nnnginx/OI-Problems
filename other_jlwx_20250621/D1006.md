## Description

A同学的班上有n名同学，作为班长的A同学记录下班级n个同学的姓名的汉语拼音（大家名字的拼音互不相同），他想知道最长的拼音是哪个，请你编程帮他找一下（请注意：如果有多个最长的拼音，请输出字典码最大的）。

## Input Format

第1行有一个整数n，代表有n名同学（n&lt;=1000） 
接下来n行，每行有1个同学的拼音（拼音由小写字母组成，不含空格） 

## Output Format

请输出最长的拼音，如果有多个，请输出字典码最大的拼音。

```input1
3
lihua
xiaoming
xiaozhao
```
```output1
xiaozhao
```
