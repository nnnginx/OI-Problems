## Description

假设一个只有小写字母的字符串中，如果出现次数最多的字母出现的次数 - 出现次数最少的字母出现的次数是一个素数，那么这个字符串就是一个幸运的字符串。
给定一个只有小写字母的字符串，请问这个字符串是不是幸运的字符串？
 

## Input Format

输入一个字符串，其中只可能出现小写字母，并且长度小于100。

## Output Format

输出有2行，第1行表示读入的是否是幸运字符串，如果是，输出" Lucky Word "，如果不是，输出" No Answer"。  
第二行是一个整数，如果输入单词是"Lucky Word"，输出出现次数最多的字母出现的次数-出现次数最少的字母出现的次数的结果，否则输出0。
 

```input1
【样例1】
error

【样例2】
olympic
```
```output1
【样例1】
Lucky Word
2

【样例2】
No Answer
0
```
## Hint

【输入输出样例1解释】 
单词error中出现最多的字母r出现了3次，出现次数最少的字母出现了1次，3-1=2，2是质数。 
 
【输入输出样例2解释】 
单词olympic中出现最多的字母和最少的字母都是1次，1-1=0，0不是质数。
