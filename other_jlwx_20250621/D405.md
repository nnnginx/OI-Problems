## Description

某大学在学年结束时，要根据同学们各个学科的考试成绩评定奖学金，根据同学们这学期所有科目的考试成绩来进行统计。 
奖学金评级方法如下： 
只要有不及格的科目（成绩&lt;60分为不及格），就不能参评奖学金。 
如果有5个及以上的科目优秀（成绩&gt;=90分为优秀）那么能够获取一等奖学金。 
如果有3个或者4个科目优秀，那么能够获取二等奖学金。 
如果有2个科目优秀，那么能够获取三等奖学金。 
请编写一个程序从键盘读入某同学n门科目的考试成绩，输出该同学获得奖学金的情况（一等奖学金输出1，二等奖学金输出2，三等奖学金输出3，不能获得奖学金输出no）

## Input Format

第1行是一个整数n，代表有n个考试科目（5=&lt;n&lt;=15） 
第2行是n个整数，代表该同学n门考试的成绩

## Output Format

输出该同学获得奖学金的情况

```input1
5
90 98 80 92 70
```
```output1
2
```
