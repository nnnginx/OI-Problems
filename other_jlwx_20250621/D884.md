## Description

一次数学小测试，请你设计一个程序完成下列问题（从键盘上输入三个数： L 、 R  和  V ）：

1 ） **Q** 操作：计算出  L  和  R  的乘积； 
2 ） **C** 操作：将  L  加上  R 然后再  乘以  V  的结果输出 。

## Input Format

一行包括三个正数，依次表示  L ， R  和  V  的值。

## Output Format

两行，第一行为 Q 操作，第二行为 C 操作。按照样例的要求输出。 

```input1
3 4 5
```
```output1
Q:12
C:35
```
## Hint

特别说明：输入输出数据应使用 double 类型。输入的三个正数均不超过 1000 。
