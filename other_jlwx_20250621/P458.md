## Description

<p style="margin-left: 0in;"><span style="color: rgb(46, 43, 33);">给定一个由</span><span style="color: rgb(46, 43, 33);">n</span><span style="color: rgb(46, 43, 33);">行数字组成的数字三角形如下图所示。试设计一个算法，计算出从三角形的顶至底的一条路径，使该路径经过的数字总和最大。

![0031.jpg](/api/public/img/022d9eb954514b07b9336659fd3cadb8.jpg)

编程任务：</span></p><p style="margin-left: 0in;"><span style="color: rgb(46, 43, 33);">对于给定的由</span><span style="color: rgb(46, 43, 33);">n</span><span style="color: rgb(46, 43, 33);">行数字组成的数字三角形，编程计算从三角形的顶至底的路径经过的数字和的最大值。</span></p>

## Input Format

<p>第1行是数字三角形的行数n，1&lt;=n&lt;=100。接下<span style="color: rgba(0, 0, 0, 0.87);">来 n 行是数字三角形各行中的数字。所有数字在 0..99 之间。</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">将计算结果输出，</span>第 1 行中的数是计算出的最大值。</p>

```input1
5 
7 
3 8 
8 1 0
2 7 4 4 
4 5 2 6 5 
```
```output1
30
```
