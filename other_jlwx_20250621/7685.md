## Description

烤面包时，烤箱温度一般设定在180度到220度之间，温度过高或过低都会导致烤出的面包品质不高。请你编写一个程序，根据烤箱的温度来判断温度是否合适。

## Input Format

输入一个整数t（20 <= t <= 250)，表示烤箱的温度。

## Output Format

如果温度在180度到220度之间（含180度和220度），则输出 Good；
如果温度高于220度，则输出High；
如果温度低于180度，则输出Low。

```input1
185
```
```output1
Good
```
```input2
230
```
```output2
High
```
