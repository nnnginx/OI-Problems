## Description

判断一个数能否被3和7整除，区分以下三种情况：
可以同时被3和7整除；
只能被3整除或只能被7整除；
既不能被3整除也不能被7整除。


## Input Format

输入一个整数n。（-1,000,000 <= n <= 1,000,000)

## Output Format

如果整数n能够同时被3和7整除，则输出 2；
如果整数n只能被3整除或只能被7整除，则输出1；
如果整数n既不能被3整除也不能被7整除，则输出0。
注意：输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
21
```
```output1
2
```
```input2
6
```
```output2
1
```
```input3
20
```
```output3
0
```
