<h2>说明</h2>

## 问题描述

小叶子刚学会了负数，她觉得很有意思。负数就是在正数前加负号(“-”)。为了更好的看出效果，小叶子决定写个程序，不管输入是正数还是负
数，输出结果都为负数。 例如：输入 1，输出-1；输入-1.5，输出-1.5。

## 输入文件

文件中只有一个不为 0 的数.

## 输出文件

文件中只有一个负数。

## 样例输入

```
1
```

## 样例输出

```
-1
```

## 样例输入

```
-3.01
```
## 样例输出

```
-3.01
```

## 来源

2011 年北京小学生科普日活动试题
