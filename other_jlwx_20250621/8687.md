## Description

<p><span style="color: rgb(51, 51, 51);">输入一个整数$a$，求出一个最小的$n$和$k$（$n＞k≥a$），使得能在$1~n$之间能找到一个$k$，且$1~k-1$的和等于$k+1~n$的和。</span><br /></p>

## Input Format

<p><span style="color: rgb(51, 51, 51);">一行一个正整数$a$，$3≤a≤1940500$。</span><br /></p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">一行两个数，表示最小的正整数$k$和$n$，之间用一个空格隔开。</span><br /></p>

```input1
3
```
```output1
6 8
```
