## Description

从键盘读入一个分数算式，为 2 个分数做加法或者减法，请输出分数算式的结果，结果也用分数表达，且约分到最简形式。（请注意：做减法可能得到负的分数，如果是负数要输出负号 - ，如 1/15-4/15 结果为 -1/5 ）
 

## Input Format

分数表达式（分数表达式中，每个分数的分子和分母都是正整数，两个分数中的运算符，可能是加号，也可能是减号，且分数表达式不含空格）

## Output Format

分数表达式计算的结果

```input1
1/12+5/12
```
```output1
1/2
```
## Hint

注意考虑特殊情况,如:1/2+1/2=1,1/2-1/2=0,这些情况下结果不需要表现为分数形式。

## Source

字符串
