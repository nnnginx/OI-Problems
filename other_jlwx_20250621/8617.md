## Description

<p>一列火车 $n$ 节车厢，依次编号为 $1,2,3,…,n$。</p><p>每节车厢有两种运动方式，进栈与出栈，问 $n$ 节车厢出栈的可能排列方式有多少种。</p>

## Input Format

<p>输入一个整数 $n$，代表火车的车厢数。<br /></p>

## Output Format

<p>输出一个整数 $s$ 表示 $n$ 节车厢出栈的可能排列方式数量。<br /></p>

```input1
3
```
```output1
5
```
## Hint

<p>数据范围</p><p>$1 \le n \le 60000$</p>
