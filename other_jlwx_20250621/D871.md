## Description

高精度减法，求a-b a,b都不超过240位

## Input Format

两个非负整数 
每行一个

## Output Format

一个数

```input1
33333333333333333333333333333333333333333
22222222222222222222222222222222222222222
```
```output1
11111111111111111111111111111111111111111
```
## Hint

数据并不保证a&gt;b，如果差是负数要输出'-'号；
另外差的高位上的0要去掉。

高精度算法
