## Description

<p>Pell数列$a\_1,a\_2,a\_3, ...$的定义是这样的，$a\_1 = 1, a\_2 = 2, ... , a\_n = 2 a\_{n−1} + a\_{n-2}(n>2)$。<p><p>给出一个正整数 $k$，要求Pell数列的第 $k$ 项模上 $32767$ 是多少。<p>

## Input Format

<p>第1行是测试数据的组数 $n$，后面跟着 $n$ 行输入。每组测试数据占 $1$ 行，包括一个正整数$k (1≤k<1000000)$。<p>

## Output Format

<p>$n$ 行，每行输出对应一个输入。输出应是一个非负整数。<p>

```input1
2
1
8

```
```output1
1
408


```
