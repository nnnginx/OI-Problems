## Description

<p><span style="color: rgb(51, 51, 51);">给出$n$个数$a1，a2，…，an$，求两个数相加的绝对值的最小值，即求$|ai+aj|$ 的最小值，其中$i$不等于$j$。</span><br /></p>

## Input Format

<p>第1行1个正整数$n$。</p><p>接下来1行n个整数，表示$a1，a2，…，an$，每两个数之间用一个空格隔开。</p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">一行一个整数，表示答案。</span><br /></p>

```input1
5
-2 6 7 7 -8
```
```output1
1
```
## Hint

<p>【数据规模】</p><p>对于40%的数据满足：$n≤10^3，-10^6 ≤a_i≤10^6$。</p><p>对于80%的数据满足：$n≤10^5，-10^6≤a_i≤10^6$。</p><p>对于100%的数据满足：$n≤10^6，-10^6≤a_i≤10^6$。</p>
