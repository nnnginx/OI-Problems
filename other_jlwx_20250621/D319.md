## Description

王阿姨在一块蔬菜地种了 4 种不同的蔬菜，分别是：番茄、黄瓜、韭菜、萝卜，四种蔬菜共占地 80 平方米；其中已知番茄、黄瓜、韭菜的分别占总土地的 a% 、 b% 、 c% （ a 、 b 、 c 一定是整数），请计算出番茄、黄瓜、韭菜、萝卜分别占地多少平方米。  
 

## Input Format

3个百分数，分别代表番茄、黄瓜、韭菜的占地百分比，每个百分数之间用空格隔开

## Output Format

番茄、黄瓜、韭菜、萝卜的占地面积，数与数之间用空格隔开，保留1位小数

```input1
35% 20% 21%
```
```output1
28.0 16.0 16.8 19.2
```
