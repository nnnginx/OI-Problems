## Description

有N（N&lt;=1000）张扑克牌放成一堆，每次从上面取一张牌翻开，再从上面取一张牌放到这堆牌的下面。也就是从上面奇数次取到的牌翻开放成一排，偶数次取到的牌放在下面直到取完。 
输入N张扑克牌的排满数字，输出翻开的情况。比如，N=4，牌面从上到下依次是1 2 3 4，那么翻开牌面的情况为1 3 2 4。

## Input Format

第一行是一个整数N（N&lt;=1000），第二行是N个整数，代表从上到下的牌面数。

## Output Format

按照翻开牌面的顺序，输出翻开的结果，数字之间用一个空格隔开。

```input1
4
1 2 3 4
```
```output1
1 3 2 4
```
