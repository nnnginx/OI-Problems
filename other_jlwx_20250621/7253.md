## Description

在上一季里，曾提到过质数的孤独，其实从另一个角度看，无情隔膜它们的合数全是质数的后代，因为合数可以由质数相乘结合而得。<

如果一个合数由两个质数相乘而得，那么我们就叫它是质数们的直接后代。现在，给你一系列自然数，判断它们是否是质数的直接后代。

## Input Format

第一行一个正整数T，表示需要判断的自然数数量
接下来T行，每行一个要判断的自然数

## Output Format

共T行，依次对于输入中给出的自然数，判断是否为质数的直接后代，是则输出Yes，否则输出No

```input1
4
3
4
6
12
```
```output1
No
Yes
Yes
No
```
## Hint

1<=T<=20
2<=要判断的自然数<=10^5^
