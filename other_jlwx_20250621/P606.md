## Description

<p>输入学生的人数，然后再输入每位学生的分数和姓名，求获得最高分数的学生的姓名。<p>

## Input Format

<p>第一行输入一个正整数N（N ≤ 100），表示学生人数。接着输入N行，每行格式如下：<p><p>分数 姓名<p><p>分数是一个非负整数，且小于等于100；<p><p>姓名为一个连续的字符串，中间没有空格，长度不超过20。<p><p>数据保证最高分只有一位同学。<p>

## Output Format

<p>获得最高分数同学的姓名。<p>

```input1
5
87 lilei
99 hanmeimei
97 lily
96 lucy
77 jim

```
```output1
hanmeimei

```
