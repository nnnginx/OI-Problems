## 输入格式

<div>
 第一行两个正整数N,K
</div> 
<div>
 接下来一行N个非负整数，第i个表示Ai
</div> 
<div>
 接下来一行N个非负整数，第i个表示Bi
</div> 
<p></p>

## 输出格式

<div>
 一行两个整数，表示子问题1和2的答案.
</div> 
<p></p>

```input1
5 3
5 5 4 5 5
3 0 3 2 3
```
```output1
295164 295326
```
## 数据范围与约定

<div>
 N<=50,Ai<=10^18,Bi<=10^18,K<=25
