## Description

数学中有概率的概念。概率反映的是事件出现的可能性大小。设对某一随机现象进行了n次试验与观察，其中A事件出现了m次，即其出现的频率为m/n。比如：有一个字符串happy，那么该单词中字符a出现的概率 = 1 / 5 = 0.2，字符p出现的概率 = 2 / 5 = 0.4。
现给出n行，每行有一个字符和一个单词，求该字符在单词中出现的概率。
注意：求字符出现的概率时，不区分大小写，也就是如果求字符P在happy中出现的概率，结算结果 =&nbsp; 2 / 5 = 0.4。
 

## Input Format

第1行是一个整数n（n&lt;=200），表示接下来有n组数据需要你来计算；
接下来n行，每行有一个字符和一个单词（单词长度&lt;=500），字符和单词自建用空格隔开。
 

## Output Format

输出n行，求出输入的n行数据中，每个字符在单词中出现的概率，结果保留5位小数。
 

```input1
3
a apple
c Candy
a banana
```
```output1
0.20000
0.20000
0.50000
```
