## Description

<p><span style="color: rgba(0, 0, 0, 0.75);">给定一个整数，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例2）。</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.75);">一个整数N。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.75);">一个整数，表示反转后的新数。</span><br /></p>

```input1
123
```
```output1
321
```
```input2
-380
```
```output2
-83
```
## Hint

【数据范围】

$-1,000,000,000≤N≤1,000,000,000$

$−1,000,000,000≤N≤1,000,000,000$
