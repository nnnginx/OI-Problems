## Description

某城市的出租车收费标准如下：
假设打车路程为 x 公里：
（1）起步价 9.0 元（不超过 3 公里，即： 0<x<=3;）；
（2）3 公里后：白天 1.5 元/公里；夜间 1.75 元/公里； 
（3）6 公里后：白天 2.25 元/公里；夜间 2.5 元/公里。

如： 白天打车路程 x=10 公里,费用计算方法如下：
前 3 公里起步价 9.0 元； 
3 公里到 6 公里费用 3*1.5；
6 公里后的费用4*2.25，共计 22.5 元。
给出路程 x，请计算所需的车费。

## Input Format

输入共一行，两个数，中间用空格隔开；
第一个数是整数，表示路程；
第二个数是整数，1 表示白天， 0 表示夜间。

## Output Format

输出共一行， 一个数，表示所需的车费，保留两位小数。

```input1
10 1
```
```output1
22.5
```
## Hint

【数据范围】
对于 50%的数据:    x<=10000；
对于 100%的数据：  x<=1000000000。
