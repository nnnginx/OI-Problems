## Description

 找找谁的身高超过全家的平均身高。全家n口人，输入输出数据如下： (平均身高保留一位小数) 

## Input Format

 第一行有一个整数n( 1 &lt; n &lt; 11 )。第二行是n个整数，用空格隔开。 

## Output Format

 第一行为全家的平均身高（保留一位小数）；第二行有若干个数，为超过平均身高的人的身高厘米数。 

```input1
7
175 160 172 158 178 162 142
```
```output1
AVE=163.9
1:175 3:172 5:178
```
