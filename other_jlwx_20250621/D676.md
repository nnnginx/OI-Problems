## Description

懒羊羊在前一阵准备羊村的小高考复习时表现出了罕见的勤奋，现在他得知了小高考四门必修课的成绩，他想知道根据羊村的高考政策，他的高考评分等级到底是什么？  
羊村高考政策如下：
如果懒羊羊 4 门成绩全部 >=90 ，则等级为 A
如果 2 门或者 3 门成绩 >=90 ，则等级为 B
如果只有 1 门成绩 >=90 ，则等级为 C
如果没有一门成绩 >=90 ，则等级为 D

## Input Format

 4 个整数（小于 100 ），分别表示懒羊羊四门成绩  
 

## Output Format

一个字母，表示懒羊羊的评分等级

```input1
99 99 98 95
```
```output1
A
```
