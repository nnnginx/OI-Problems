## Description

小区新超市开张搞活动，凡购买总金额满200元及超过200元的顾客可以打8折（打8折的意思是总价*0.8，比如，购买了212元的商品，那么实际付款 = 212 * 0.8 = 169.6元），购物不满200元的顾客可以打9折。
 
 小芳买了3件商品，请根据超市的活动编程计算一下，小芳实际需要付多少钱？
 

## Input Format

三个小数，代表小芳购买的三件商品的价格。

## Output Format

一个小数，表示小芳按照超市的活动实际应付的价格（结算结果保留1位小数）。

```input1
89.5 98.2 36.4
```
```output1
179.3
```
