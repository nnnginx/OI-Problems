## Description

<p>给出一个整数n（n≤2000）和k个变换规则（k≤15）。规则：<p><p>① 1个数字可以变换成另1个数字；<p><p>② 规则中，右边的数字不能为零。<p><p>例如：n=234，k=2规则为<p><p>2 → 5<p><p>3 → 6<p><p>上面的整数234经过变换后可能产生出的整数为（包括原数）234，534，264，564共4种不同的产生数。<p><p>求经过任意次的变换（0次或多次），能产生出多少个不同的整数。仅要求输出不同整数个数。<p>

## Input Format

<p>$\\begin{matrix}n\\\\k\\\\x\_1&y\_1\\\\x\_2&y\_2\\\\…&…\\\\x\_n&y\_n\\end{matrix}$<p>

## Output Format

<p>格式为一个整数（满足条件的整数个数）。<p>

```input1
234
2
2 5
3 6


```
```output1
4

```
