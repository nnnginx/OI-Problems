## Description

体育课上，体育老师要求同学们按照从矮到高或者从高到矮排好队，现在从键盘读入 n 个人的身高，请问，这 n 个人按照老师的要求排好队了吗？如果排好队了，请输出“ Good ”，如果没有排好，请输出“ No ”。   
 

## Input Format

第一行有 1 个整数 n （ 5 <= n <= 10 ），代表有 n 个人。

第二行有 n 个整数，用空格隔开表示 n 个人的身高。（这 n 个数都是 120~180 之间的整数，可能存在身高一样的同学）

## Output Format

如果排好队了，请输出“ Good ”，如果没有排好，请输出“ No ”。 

```input1
5
155 156 160 163 170
```
```output1
Good
```
