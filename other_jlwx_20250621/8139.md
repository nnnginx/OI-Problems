## Description

<p>给定一个整数，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例2）。<br /></p>

## Input Format

<p>输入共1行，一个整数N<br /></p>

## Output Format

<p>输出共1行，一个整数，表示反转后的新数<br /></p>

```input1
123
```
```output1
321
```
```input2
-380
```
```output2
-83
```
## Hint

<p>【数据范围】-1,000,000,000 ≤N≤1,000,000,000。<br /></p>
