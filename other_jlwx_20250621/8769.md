## Description

<p>一个n*m的矩形阵列由数字0到9组成，数字1到9代表细胞，细胞的定义为：沿细胞数字上下左右还是细胞数字则为同一细胞，求给定矩形阵列的细胞个数。<br /></p>

## Input Format

<p>第一行输入两个数n和m。</p><p>接下来的n行m列输入矩阵，由0到9组成。</p>

## Output Format

<p>输出矩阵里的细胞个数<br /></p>

```input1
4 10
0234500067
1034560500
2045600671
0000000089
```
```output1
4
```
## Hint

<p>1&lt;=n,m&lt;=1,000<br /></p>
