## Description

给定一个序列，每次询问序列中第l个数到第r个数中第K大的数是哪个。

## Input Format

第一行包含一个数n，表示序列长度。
第二行包含n个正整数，表示给定的序列。
第三个包含一个正整数m，表示询问个数。
接下来m行，每行三个数l,r,K，表示询问序列从左往右第l个数到第r个数中，从大往小第K大的数是哪个。序列元素从1开始标号。

## Output Format

总共输出m行，每行一个数，表示询问的答案。

```input1
5
1 2 3 4 5
2
1 5 2
2 3 2
```
```output1
4
2
```
## Hint

对于30%的数据，n,m<=100；
对于100%的数据，n,m<=1000；
保证k<=(r-l+1)，序列中的数<=106。
