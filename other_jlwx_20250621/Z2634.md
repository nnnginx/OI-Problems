## 题目描述

<p><span style="font-size: medium"><br>
   XmkAc要从QD去FJ膜拜神CP和YY。地图是一个n*m的方格图。QD在右上角，FJ在右下角。<br>
   问从QD到FJ有多少条哈密尔顿路?</span></p>
<p></p>

## 输入格式

<p><span style="font-size: medium">若干行 每行两个正整数 n m 表示一组测试数据</span></p>
<p></p>

## 输出格式

<p><span style="font-size: medium">行数同输入 每行一个整数 表示每组测试数据的答案 模7777777 ,无解输出"YM CP&YY"</span></p>
<p></p>

```input1
3 2
3 3
4 10
```
```output1
YM CP&YY
2
2329
```
## 数据范围与约定

<p><br><br>
  N <= 7 M <= 10^9 组数T <= 20<br><br>
  不卡常数<br><br></p>
