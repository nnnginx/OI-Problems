## Description

<p><span style="color: rgb(51, 51, 51);">输入一个数，判断这个数是否是素数？</span><br /></p>

## Input Format

<p>第一行输入一个数T，表示有T组数据</p><p>接下来每行输入一个n，表示需要判断的数。</p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">对于每行的n，如果是素数输出&quot;YES&quot;，否则输出&quot;NO&quot;。</span><br /></p>

```input1
3
109
9898563
373733771
```
```output1
YES
NO
NO
```
## Hint

<p>对于30%的数据：$ t\le 20,1\le n\le 2^{32}-1$</p><p>对于100%的数据：$ t\le 500,1\le n\le 2^{63}-1$</p>
