## Description

小Tu一直很讨厌小数点，一看到小数点就头疼。不幸的是，小Ya给他出了一个题目，问他：
给你一个小数x，你能算出小数点后第n位是什么吗？（1&lt;=n&lt;=6）
小Tu看到此题后，瞬间头疼晕倒，你能帮帮他吗？

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">首先输入一个t，表示有t组数据，跟着t行：</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">每行输入一个小数（输入数据保证一定是a.b的形式，为了简单化问题，没有循环小数的情况），</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">然后跟一个n，表示小数点后第几位。</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出一个数表示小数点后第n位的数。</span><br /></p>

```input1
3
1.234 1
2.345 2
3.456 3
```
```output1
2
4
6
```
