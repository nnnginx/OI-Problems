## Description

输入1行句子(不多于200个单词，每个单词长度不超过100)，只包含字母、空格和逗号。单词由至少一个连续的字母构成，空格和逗号都是单词间的间隔。
试输出第1个最长的单词和第1个最短单词。 

## Input Format

 一行句子。 

## Output Format

第1行，第一个最长的单词。  
第2行，第一个最短的单词。 

```input1
I am studying Programming language C in Peking University

```
```output1
Programming
I

```
## Hint

提示：
如果所有单词长度相同，那么第一个单词既是最长单词也是最短单词。
