## Description

输入n个学生的学号、成绩、计算平均成绩。然后根据每个学生的成绩和平均成绩的差，给出学生的等级。
成绩高于平均成绩10分及以上者为A等,
成绩低于平均成绩10分及以下者为C等,
其他为B等。

## Input Format

第一行为n，后面n行每行依次为学号和成绩（n<=100，成绩<=100），中间用一个空格隔开。

## Output Format

n行，每行为学号 成绩 等级（中间用一个空格隔开)

```input1
5
1 34
2 56
3 45
4 67
5 99
```
```output1
1 34 C
2 56 B
3 45 C
4 67 B
5 99 A
```
