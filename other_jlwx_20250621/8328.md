## Description

<p>回文词是一种对称的字符串――也就是说，一个回文词，从左到右读和从右到左读得到的结果是一样的。</p><p>任意给定一个字符串，通过插入若干字符，都可以变成一个回文词。你的任务是写一个程序，求出将给定字符串变成回文词所需插入的最少字符数。</p><p>比如字符串$“Ab3bd”$，在插入两个字符后可以变成一个回文词（$“dAb3bAd”$或$“Adb3bdA”$）。然而，插入两个以下的字符无法使它变成一个回文词。</p>

## Input Format

<p>第一行包含一个整数$N$，表示给定字符串的长度，$3 \le N \le 5000$</p><p>第二行是一个长度为$N$的字符串，字符串由大小写字母和数字构成。</p>

## Output Format

<p><br /><span style="color: rgb(77, 77, 77);">一个整数，表示需要插入的最少字符数。</span><br /></p>

```input1
5
Ab3bd
```
```output1
2
```
