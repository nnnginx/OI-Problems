## Description

<p>给定一个十进制正整数n(0<n<1000000000)，每个数位上数字均不为0。n的位数为m。<p><p>现在从m位中删除k位(0<k<m)，求生成的新整数最小为多少？<p><p>例如: n=9128456,k=2,则生成的新整数最小为12456。<p>

## Input Format

<p>第一行t, 表示有t组数据；<p><p>接下来t行，每一行表示一组测试数据，每组测试数据包含两个数字n,k。<p>

## Output Format

<p>t行，每行一个数字，表示从n中删除k位后得到的最小整数。<p>

```input1
2
9128456 2
1444 3

```
```output1
12456
1


```
