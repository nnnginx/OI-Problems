## Description

James同学发现了在二维数组中有一些回文数，请编程找出这些回文数，并按照输入的顺序输出。 （回文数指的是这个数正过来读和反过来读是同一个数的数，比如1、8、99、252、1221等）。

## Input Format

第一行是两个整数n和m（n和m都是4~100之间的整数），代表接下来的二维数组有n行m列。 
接下来n行，每行有m个整数，这些整数都是1~9999之间的整数。

## Output Format

按照输入的顺序输出满足条件的回文数，每行1个。

```input1
3 3
1 22 98
34 121 110
100 210 323
```
```output1
1
22
121
323
```
## Source

二维数组
