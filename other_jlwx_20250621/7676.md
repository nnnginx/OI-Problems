## Description

7月份放暑假，小蓝报名参加了一个夏令营。夏令营将于7月15日开营，从这天开始，学员可以报到；7月22日闭营，从这一天起就不允许学员报到了。
小蓝想帮夏令营设计一个程序，当输入学员报到的日期后，程序就会输出结果，告诉学员是否可以报到。
请帮小蓝设计这个程序。

## Input Format

输入一个正整数d（小于等于31)，表示学员7月份报到的日期。

## Output Format

如果d小于15，程序输出 Too early;
如果d大于或等于22，输出程序 Too late;
其他情况，程序输出 Welcome。

```input1
16
```
```output1
Welcome
```
