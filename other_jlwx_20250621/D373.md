## Description

完美偶数指的是，如果一个数本身是偶数，且这个数是偶数位的数，且这个数的各个位也是偶数，那么这个数就可以称为完美偶数；比如：28就是完美偶数，而246就不是，因为246是一个3位数。 
请你编程求出，从键盘读入的n个数中，哪些数是完美的偶数，输出他们。

## Input Format

第一行是一个整数n（n&lt;=100） 
第二行是n个整数（这些整数都是1~9999范围内的整数）

## Output Format

按顺序输出这n个数中的完美偶数，每个数一行

```input1
5
26 4286 228 32 1280
```
```output1
26
4286
```
