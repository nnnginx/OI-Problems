## Description

<p>两个相差为2的素数称为素数对，如5和7，17和19等，本题目要求找出所有两个数均不大于n的素数对。<br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.75);">一个正整数</span><span style="color: rgba(0, 0, 0, 0.75);">n</span><span style="color: rgba(0, 0, 0, 0.75);">。</span><span style="color: rgba(0, 0, 0, 0.75);">1≤n≤10000</span><span style="color: rgba(0, 0, 0, 0.75);">。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.75);">所有小于等于</span><span style="color: rgba(0, 0, 0, 0.75);">n</span><span style="color: rgba(0, 0, 0, 0.75);">的素数对。每对素数对输出一行，中间用单个空格隔开。若没有找到任何素数对，输出empty</span><span style="color: rgba(0, 0, 0, 0.75);">。</span><br /></p>

```input1
10
```
```output1
3 5
5 7
```
```input2
100
```
```output2
3 5
5 7
11 13
17 19
29 31
41 43
59 61
71 73
```
```input3
3
```
```output3
empty
```
