## Description

求出区间 [a,b] 中所有整数的质因数分解。
  
<strong>提示</strong>    
先筛出所有素数，然后再分解。    
 
 
   
 
 
     
 
	 
 

## Input Format

 输入两个整数 a ， b 。    
 

## Output Format

每行输出一个数的分解，形如 k=a1*a2*a3...(a1&lt;=a2&lt;=a3... ， k 也是从小到大的 )( 具体可看样例 )     
 

```input1
3 10
```
```output1
3=3
4=2*2
5=5
6=2*3
7=7
8=2*2*2
9=3*3
10=2*5
```
