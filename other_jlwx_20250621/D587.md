## Description

探险家哥伦布和好友两人来到一个小岛探险，小岛有上有一个藏有宝藏的山洞，山洞的门需要一个四位数的密码，这个密码被加密后存入一个玻璃瓶藏在岛上，两人经过苦苦寻找，终于找到了密码，也找到了解密方式；解密方式为：将这个四位数千位和百位颠倒、十位和个位颠倒，就可以解密。  
 
请你编程帮助他们计算出密码是多少？  

比如：假设他们找到加密后的四位数是 1286 ，那么千位和百位颠倒、十位和个位颠倒得到的结果就是 2168 。  
 

## Input Format

一个加密后的四位整数。（这个四位数不含数字 0 ）  
 

## Output Format

解密后的四位整数。  
 

```input1
1286
```
```output1
2168
```
