## 题目描述

<div>
 有n种不同的邮票，皮皮想收集所有种类的邮票。唯一的收集方法是到同学凡凡那里购买，每次只能买一张，并且
</div>
<div>
 买到的邮票究竟是n种邮票中的哪一种是等概率的，概率均为1/n。但是由于凡凡也很喜欢邮票，所以皮皮购买第k
</div>
<div>
 张邮票需要支付k元钱。现在皮皮手中没有邮票，皮皮想知道自己得到所有种类的邮票需要花费的钱数目的期望。
</div>

## 输入格式

<p>一行,一个数字N N<=10000</p>

## 输出格式

<p>要付出多少钱. 保留二位小数</p>

```input1
3
```
```output1
21.25
```
