## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">已知：Sn= 1＋1/2＋1/3＋…＋1/n。显然对于任意一个整数K，当n足够大的时候，Sn大于K。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">现给出一个整数K（1&lt;=k&lt;=15），要求计算出一个最小的n；使得Sn＞K。</span></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.75);">一个正整数</span><span style="color: rgba(0, 0, 0, 0.75);">k</span><span style="color: rgba(0, 0, 0, 0.75);">。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.75);">一个正整数</span><span style="color: rgba(0, 0, 0, 0.75);">n</span><span style="color: rgba(0, 0, 0, 0.75);">。</span><br /></p>

```input1
1
```
```output1
2
```
## Hint

<p>【数据范围】</p><p>对于100%的数据，1≤k≤15。
