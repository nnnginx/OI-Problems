## Description

小A最近正在看一本书，据说是《算法竞赛入门经典之训练指南》，他可是已经看了不少天啦，有一次，他突然灵机一动，灵魂出窍，想到一个问题：

在他已经看过的那些书页中，每一页都有一个唯一的页码，而且是从1开始的，现在他想知道他看过的所有书页中一共出现了多少数字？

举个例子：他已经看到了第57页，则在1~57页中，一共出现了103个数字。
由于这本书太过雄厚，作为COIer的你能编程帮他解决这个问题吧。



## Input Format

第一行为正整数T(≤100)，表示数据组数；接下来每行一个正整数n(≤10^9)。

## Output Format

对于每一组数据，输出有两行，第一行格式：Case #: (#为测试组数)，

第二行仅有一个整数，表示结果。

```input1
1
12

```
```output1
Case 1:
13

```
