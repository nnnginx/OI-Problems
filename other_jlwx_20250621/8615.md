## Description

<p>你将要实现一个功能强大的整数序列编辑器。</p><p>在开始时，序列是空的。</p><p>编辑器共有五种指令，如下：</p><p>1、I x，在光标处插入数值 $x$。</p><p>2、D，将光标前面的第一个元素删除，如果前面没有元素，则忽略此操作。</p><p>3、L，将光标向左移动，跳过一个元素，如果左边没有元素，则忽略此操作。</p><p>4、R，将光标向右移动，跳过一个元素，如果右边没有元素，则忽略次操作。</p><p>5、Q k，假设此刻光标之前的序列为 $a_1,a_2,…,a_n$，输出 $max_{1 \le i \le k}S_i$，其中 $S_i=a_1+a_2+…+a_i$。</p>

## Input Format

<p>第一行包含一个整数 $Q$，表示指令的总数。</p><p>接下来 $Q$ 行，每行一个指令，具体指令格式如题目描述。</p>

## Output Format

<p>每一个Q k指令，输出一个整数作为结果，每个结果占一行。</p>

```input1
8
I 2
I -1
I 1
Q 3
L
D
R
Q 2
```
```output1
2
3
```
## Hint

<p>数据范围</p><p>$1 \le Q \le 10^6$,</p><p>$|x| \le 10^3$,</p><p>$1 \le k \le n$</p>
