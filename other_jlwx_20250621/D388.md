## Description

求x<sup>p</sup> mod m 的值。 
提示：若 p 为偶数，x<sup>p</sup>=(x<sup>2</sup>)<sup>p/2</sup>；若 p 为奇数，x<sup>p</sup>=x*(x<sup>2</sup>)<sup>(p-1)/2</sup>，该题可以采用分治法求解。

## Input Format

 三个不超过 10000 的正整数 x，p，m。 

## Output Format

 x <sup>p</sup> &nbsp;mod m 的值。 

```input1
2 10 100
```
```output1
24
```
