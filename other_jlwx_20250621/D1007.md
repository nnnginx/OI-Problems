## Description

某班级有n名同学，请问这n名同学中，成绩在x~y之间（含x和y）的同学共有多少人，请按从大到小的顺序，输出成绩在x~y之间的所有同学的分数。

## Input Format

第1行读入一个整数n，代表班级人数（5≤n ≤100 ）
第2行有n个整数，用空格隔开代表同学的分数（0 ≤ 分数 ≤100 ）
第3行有2个整数x和y，代表要求解的分数范围（0 ≤x ≤y  ≤100  ）
 

## Output Format

第1行按分数从大到小的顺序，输出分数在x~y之间的这些同学的分数，分数之间用空格隔开
第2行输出分数在x~y之间（含x和y）的同学的人数
 

```input1
6
100 99 90 98 96 95
95 100
```
```output1
100 99 98 96 95
5
```
