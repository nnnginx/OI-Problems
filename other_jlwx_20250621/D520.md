## Description

兔子又来找乌龟赛跑啦！同样的错误兔子不会犯两次，所以兔子提出赛跑的时候，乌龟就觉得这场比赛很 不公平。于是兔子进一步放宽了条件，表示他可以在比 赛开始以后先睡 t 分钟再开始追乌龟。

乌龟这下没办法确定比赛到底公平不公平了，所以请你来帮忙。假设乌龟每分钟可以跑 x 米，兔子每分钟跑 y 米 (x &lt; y)。他希望你计算最大的整数赛跑距离(米)，满足乌龟能在兔子先睡 t 分钟的前提下，比兔子更早或同时到达终点。

## Input Format

 三个整数 x, y, t 

## Output Format

 一个整数，表示要求的最长赛跑距离 

```input1
11 21 7
```
```output1
161
```
## Hint

数据规模 1 ≤ x &lt; y ≤ 100, t ≤ 1000

2017江苏省青少年信息学奥林匹克竞赛复赛
