## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">小明班里要举行一次拔河比赛，班主任决定将所有人分为两队，每个人都必须参加。两个队伍的人数之差不能超过1，并且两个队伍的体重之和要尽可能相近，当然相同是最好的了。</span><br /></p>

## Input Format

<p>输入包含多组测试数据。</p><p>每组输入的第一行是一个正整数n（2&lt;=n&lt;=100），表示共有n个人。</p><p>接下来n行，每行输入一个整数w（1&lt;=w&lt;=450），表示每个人的体重。</p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">对于每组输入，分别输出两个队伍的体重之和，按升序排序。</span><br /></p>

```input1
3
100
90
200
```
```output1
190 200
```
