## Description

输入n(n&lt;=100)个整数，按照绝对值从大到小排序后输出。题目保证对于每一个测试实例，所有的数的绝对值都不相等。<br />
&nbsp;<br />
<br />
<br />

## Input Format

<span>输入数据有多组，每组占一行，每行的第一个数字为n,接着是n个整数，n=0表示输入数据的结束，不做处理。&nbsp;</span><br />

## Output Format

<span>对于每个测试实例，输出排序后的结果，两个数之间用一个空格隔开。每个测试实例占一行。</span>

```input1
3 3 -4 2
4 0 1 2 -3
0
```
```output1
-4 3 2
-3 2 1 0
```
