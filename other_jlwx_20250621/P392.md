## Description

<p>一个学校举行拔河比赛，所有的人被分成了两组，每个人必须（且只能够）在其中的一组，要求两个组内的所有人体重加起来尽可能地接近。</p>

## Input Format

<p>输入数据的第 1 行是一个 n，表示参加拔河比赛的总人数， n&lt;=1000，</p><p>接下来的 n 行表示第 1 到第 n 个人的体重，每个人的体重都是整数（1&lt;=weight&lt;=450）</p>

## Output Format

<p>输出数据应该包含两个整数：分别是两个组的所有人的体重和，用一个空格隔开。</p><p>注意如果这两个数不相等，则请把小的放在前面输出。</p>

```input1
3
100
90
200
```
```output1
190 200
```
