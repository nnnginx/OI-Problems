## Description

已知一个圆的半径，求解该圆的面积和周长 

## Input Format

输入只有一行，只有1个整数。 

## Output Format

输出只有两行，一行面积，一行周长。（保留两位小数）。 令pi=3.1415926 

```input1
1
```
```output1
3.14
6.28
```
## Hint

圆的面积和周长求解公式分别如下； 
圆的面积S = π * 半径 * 半径；（π读作pài，π = 3.1415926…，在具体的题目中π的值精确到小数点后多少位，取决于具体的题目） 
圆的周长C = π * 2 * 半径；
