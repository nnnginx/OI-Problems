## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">辰辰是个天资聪颖的孩子，他的梦想是成为世界上最伟大的医师。为此，他想拜附近最有威望的医师为师。医师为了判断他的资质，给他出了一个难题。医师把他带到一个到处都是草药的山洞里对他说：“孩子，这个山洞里有一些不同的草药，采每一株都需要一些时间，每一株也有它自身的价值。我会给你一段时间，在这段时间里，你可以采到一些草药。如果你是一个聪明的孩子，你应该可以让采到的草药的总价值最大。”</span><span style="color: rgba(0, 0, 0, 0.87);"><br /><br /></span><span style="color: rgba(0, 0, 0, 0.87);">如果你是辰辰，你能完成这个任务吗？</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输入</span><span style="color: rgba(0, 0, 0, 0.87);">的第一行有两个整数T</span><span style="color: rgba(0, 0, 0, 0.87);">（1 &lt;= T &lt;= 1000</span><span style="color: rgba(0, 0, 0, 0.87);">）和M</span><span style="color: rgba(0, 0, 0, 0.87);">（1 &lt;= M &lt;= 100</span><span style="color: rgba(0, 0, 0, 0.87);">），用一个空格隔开，T</span><span style="color: rgba(0, 0, 0, 0.87);">代表总共能够用来采药的时间，M</span><span style="color: rgba(0, 0, 0, 0.87);">代表山洞里的草药的数目。接下来的M</span><span style="color: rgba(0, 0, 0, 0.87);">行每行包括两个在1</span><span style="color: rgba(0, 0, 0, 0.87);">到100</span><span style="color: rgba(0, 0, 0, 0.87);">之间（包括1</span><span style="color: rgba(0, 0, 0, 0.87);">和100</span><span style="color: rgba(0, 0, 0, 0.87);">）的整数，分别表示采摘某株草药的时间和这株草药的价值。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出</span><span style="color: rgba(0, 0, 0, 0.87);">包括一行，这一行只包含一个整数，表示在规定的时间内，可以采到的草药的最大总价值。</span><br /></p>

```input1
70 3
71 100
69 1
1 2
```
```output1
3
```

<p>【数据范围】</p><p style="margin-left: 40px;"><span style="color: rgba(0, 0, 0, 0.87);">对于30%</span><span style="color: rgba(0, 0, 0, 0.87);">的数据，M &lt;= 10</span><span style="color: rgba(0, 0, 0, 0.87);">；</span><span style="color: rgba(0, 0, 0, 0.87);"><br /></span><span style="color: rgba(0, 0, 0, 0.87);">对于全部的数据，M &lt;= 100。
