## Description

全班共有100人去公园划船，其中男生m人。公园里有大船、中船和小船共租用，三种船最多可坐的人数和租金价格如下表所示。现要求男生和女生要坐到不同的船上，请问最便宜的租船方案是如何坐船，请求出大船、中船、小船各需租用多少只，才能按照最便宜的方案装下100人？
船的型号&nbsp;&nbsp;&nbsp;船的载客量&nbsp;&nbsp;&nbsp;半天的出租金额
大船&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;6人&nbsp;&nbsp;&nbsp;100元
中船&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3人&nbsp;&nbsp;&nbsp;75元
小船&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2人&nbsp;&nbsp;&nbsp;60元



## Input Format

一个整数m代表男生的人数

## Output Format

三个整数，分别代表需要租用大船的数量、中船的数量以及小船的数量，用空格隔开 3 个整数  
 

```input1
64
```
```output1
16 1 1
```
