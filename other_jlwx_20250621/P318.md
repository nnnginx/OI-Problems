## Description

字符串中只含有括号 (),\[\],<>,{},判断输入的字符串中括号是否匹配。如果括号有互相包含的形式，从内到外必须是<>,(),\[\],{}，例如。输入: \[()\] 输出:YES，而输入(\[\])，(\[)\]都应该输出NO。

## Input Format

第一行为一个整数n，表示以下有多少个由括好组成的字符串。
接下来的n行，每行都是一个由括号组成的长度不超过255的字符串。

## Output Format

 在输出文件中有n行，每行都是YES或NO。 

```input1
5
{}{}<><>()()\[\]\[\]
{{}}{{}}<<>><<>>(())(())\[\[\]\]\[\[\]\]
{{}}{{}}<<>><<>>(())(())\[\[\]\]\[\[\]\]
{<>}{\[\]}<<<>><<>>>((<>))(())\[\[(<>)\]\]\[\[\]\]
><}{{\[\]}<<<>><<>>>((<>))(())\[\[(<>)\]\]\[\[\]\]

```
```output1
YES
YES
YES
YES
NO


```
