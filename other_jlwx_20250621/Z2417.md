## 题目描述

<p><span style="font-size: medium">计算N!的前K位和后L位。 </span></p>
<div>
 <span style="font-size: medium"><br></span>
</div>

## 输入格式

<div>
 <span style="font-size: medium">输入的第一行一个整数，表示数据组数。</span>
</div>
<div>
 <span style="font-size: medium">接下来每行三个整数N, K, L。</span>
</div>

## 输出格式

<div>
 <span style="font-size: medium">对每组数据输出两行，第一行为前K位，第二行为后L位。</span>
</div>
<div>
 <span style="font-size: medium"><br></span>
</div>

```input1
1
6 2 1
```
```output1
72
0
```
## 数据范围与约定

<p><br><br>
  T <= 15, N <= 10^8, K <= 50, L <= 100</p>
