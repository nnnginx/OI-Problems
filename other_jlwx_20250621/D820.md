## Description

请在一个有序不递减的数组中（数组中有相等的值），采用二分查找，找到值x 第1次出现 的位置，如果不存在x请输出-1。
 
请注意：本题要求出q个x，每个x在数组中第一次出现的位置。
 
比如有6个数，分别是：1 2 2 2 3 3，那么如果要求3个数：3 2 5，在数组中第一次出现的位置，答案是：5 2 -1。
 

## Input Format

第一行，一个整数n，代表数组元素个数（n &lt;= 10<sup>5</sup>） 
第二行，n个整数，用空格隔开，代表数组的n个元素（1&lt;=数组元素的值&lt;=10<sup>8</sup>） 
第三行，一个整数q，代表有要求出q个数首次出现的位置（q&lt;=10<sup>5</sup>） 
第四行，q个整数，用空格隔开，代表要找的数（1&lt;=要找的数&lt;=10<sup>8</sup>）

## Output Format

输出1行，含q个整数，按题意输出要找的每个数在数组中首次出现的位置，如果不存在这样的数，请输出-1。 

```input1
6
1 2 2 2 3 3
3
3 2 5
```
```output1
5 2 -1
```
