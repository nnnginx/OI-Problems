## 题目描述
给出三个整数p,k,a,其中p为质数，求出所有满足x^k=a (mod p),0<=x<=p-1的x。 

## 输入格式
三个整数p,k,a。 

## 输出格式
第一行一个整数，表示符合条件的x的个数。 第二行开始每行一个数，表示符合条件的x,按从小到大的顺序输出。 

```input1
11 3 8
```
```output1
1
2
```
## 数据范围与约定
2<=p
