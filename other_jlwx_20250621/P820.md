## Description

<p>一个数的序列$b\_i$，当$b\_1<p><p>你的任务，就是对于给定的序列，求出最大上升子序列和。注意，最长的上升子序列的和不一定是最大的，比如序列(100,1,2,3)的最大上升子序列和为100，而最长上升子序列为(1,2,3)。<p>

## Input Format

<p>输入的第一行是序列的长度N(1≤N≤1000)。第二行给出序列中的N个整数，这些整数的取值范围都在0到10000(可能重复)。<p>

## Output Format

<p>最大上升子序列和。<p>

```input1
7
1 7 3 5 9 4 8

```
```output1
18

```
