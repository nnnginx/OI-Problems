## 题目描述

<p><span style="font-size: medium; ">n个灯围成一个圈。 每个灯可以打开或者关闭。 对于第i个灯，如果第i+1个灯是打开的，则它在下一秒该变其状态。对于第 n个灯的话，则看第 1个灯的状态 给出这些灯最开始的状态，求第M秒后的状态.</span></p>

## 输入格式

<p><span style="font-size: medium; ">第一行给出 N,M。（0</span></p>

## 输出格式

<p><span style="font-size: medium; ">输出 N 个数，代表M秒后每个灯的状态</span></p>

```input1
3 1
0
0
1
```
```output1
0
1
1
```
