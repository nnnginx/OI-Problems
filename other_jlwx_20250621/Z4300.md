## 题目描述

<div>
 给定一个长度为n的数列ai，求ai的子序列bi的最长长度，满足bi&bi-1!=0(2<=i<=len)。
</div>
<div></div>
<p></p>

## 输入格式

<div>
 输入文件共2行。
</div>
<div>
 第一行包括一个整数n。
</div>
<div>
 第二行包括n个整数，第i个整数表示ai。
</div>
<div></div>
<p></p>

## 输出格式

<div>
 输出文件共一行。
</div>
<div>
 包括一个整数，表示子序列bi的最长长度。
</div>
<div></div>
<p></p>

```input1
3
1 2 3
```
```output1
2
```
## 数据范围与约定

<p><span style="font-family: Helvetica, 'Microsoft Yahei', verdana; font-size: 14px; line-height: 15.549334526062px;">n<=100000,ai<=2*10^9</span></p>
