## Description

一个正整数，正读和反读都相同的数为回文数。例如22，131，2442，37073，6，…所有1位数都是回文数。给出一个正整数n（1≤n≤10000），求出1,2,…,n之中（包括1和n）的回文数的个数。

## Input Format

任意给定一个正整数n（0 &lt; n &lt;= 10000 ）

## Output Format

一个正整数，表示[1,n]之间的回文数的个数。

```input1
325
```
```output1
41
```
