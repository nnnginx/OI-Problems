## Description

众所周知，一台服务器的价格为xxx，且CodesOnline为提升判题效率，一次性购买了n台服务器，并全部安装上了Liuux系统，但服务器上的Liuux系统比较奇怪，需要输出指定的字符才可以使用。

指定的字符由两个因素决定，分别是开机时间和是否成功开机决定
开机时间的格式为mm-dd-hh（m,d,h分别代表月份、日期、小时、mm代表两位月份）
开机方式：如果正常开机则为A、如果未成功开机则为B

指定字符格式：mm-dd-hh-A 或 mm-dd-hh-B

## Input Format

输入共两行

第一行为开机时间，三个数字m,d,h，用空格分隔；
第二行为开机方式，为A或B

## Output Format

第一行：输出特定字符

特定字符的要求为：mm-dd-hh-A/B，且字符串中不能包含0，以及“-”
最后再进行反转输出

第二行：输出字符串长度

```input1
10 5 12
B
```
```output1
B2151
5
```
