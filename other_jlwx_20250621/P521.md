## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">2*N名编号为1~2N的选手共进行R轮比赛。每轮比赛开始前，以及所有比赛结束后，都会按照总分从高到低对选手进行一次排名。选手的总分为第一轮开始前的初始分数加上已参加过的所有比赛的得分和。总分相同的，约定编号较小的选手排名靠前。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">每轮比赛的对阵安排与该轮比赛开始前的排名有关：第1名和第2名、第3名和第4名、……、第2K-1名和第2K名、……、第2N-1名和第2N名，各进行 一场比赛。每场比赛胜者得 1分，负者得0分。也就是说除了首轮以外，其它轮比赛的安排均不能事先确定，而是要取决于选手在之前比赛中的表现。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">现给定每个选手的初始分数及其实力值，试计算在R轮比赛过后，排名第Q的选手编号是多少。我们假设选手的实力值两两不同，且每场比赛中实力值较高的总能获胜。</span></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输入的第一行是三个正整数N、R、Q，每两个数之间用一个空格隔开，表示有2*N名选手、R轮比赛，以及我们关心的名次Q。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">第二行是2*N个非负整数s1，s2，…，s2N，每两个数之间用一个空格隔开，其中si表示编号为i的选手的初始分数。</span></p><p>第三行是2*N个正整数w1，w2，…，w2N，每两个数之间用一个空格隔开，其中wi表示编号为i的选手的实力值。</p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出只有一行，包含一个整数，即R轮比赛结束后，排名第Q的选手的编号。</span><br /></p>

```input1
2 4 2
7 6 6 7
10 5 20 15
```
```output1
1
```
