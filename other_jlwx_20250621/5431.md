<h2>说明</h2>
<p>
	给定一个短字符串（不含空格），再给定若干字符串，在这些字符串中删除所含有的短字符串。
</p>
<h2>输入格式</h2>
<p>
	输入只有1组数据。<br />
输入一个短字符串（不含空格），再输入若干字符串直到文件结束为止。
</p>
<h2>输出格式</h2>
<p>
	删除输入的短字符串(不区分大小写)并去掉空格,输出。
</p>
<pre><code class="language-input1">in
#include 
int main()
{

printf(" Hi ");
}</code></pre><pre><code class="language-output1">#clude
tma()
{

prtf("Hi");
}</code></pre>
<h2>提示</h2>
<p>注:将字符串中的In、IN、iN、in删除。</p>
