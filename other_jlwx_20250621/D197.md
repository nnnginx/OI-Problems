## Description

输入一个整数n（n≤12），打印出如下要求的方阵：左上到右下对角线上的数与行数相同，右上半个区域中每个元素等于左边的和下面的元素之和。每个元素场宽为5。左下半个区域为空。 

## Input Format

一个整数n（n≤12） 

## Output Format

n*n的方阵（场宽为5）。 

```input1
4
```
```output1
    1    3    8   20
         2    5   12
              3    7
                   4
```
## Source

二维数组
