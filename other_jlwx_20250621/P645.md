## Description

<p>科学家在热带森林中发现了一种特殊的昆虫，这种昆虫的繁殖能力很强。每对成虫每过x个月产y对卵，每对卵要过两个月长成成虫。假设每个成虫不死，第一个月只有一对成虫，且卵长成成虫后的第一个月不产卵(过X个月产卵)，问过Z个月以后，共有成虫多少对？<br /></p>

## Input Format

<p>输入只有一行，为三个正整数：x y z<br /></p>

## Output Format

<p>输出只有一行，为z个月后成虫的个数（注意，表示要求第z个月结束后，即第z+1个月开始时的成虫个数）<br /></p>

```input1
1 2 8
```
```output1
37
```
## Hint

<p>【数据范围】</p><p>X≥0，Y≥0，Z≥X
