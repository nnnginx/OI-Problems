## Description

输入两个仅有小写字母组成的字符串，请求出两个字符串最长的重复部份的长度。
两个字符串的重复部份指的是一个字符串的头部的部分字符和另一个字符串的尾部的部分字符是重叠的。
 比如：abcdefg和fggoodabcde这两个字符串，第一个字符串的头部和第二个字符串的尾部有5个字符"abcde"是重叠的，第一个字符串尾部和第二个字符串的头部有2个字符"fg"是重叠的，因此最长的重叠部长度为5。
 

## Input Format

输入2行，每行有一个长度不超过100的仅由小写字母组成的字符串。 
 
 

## Output Format

输出两个字符串最长的重叠长度。 
 

```input1
abcxxxxabcxabcd
abcdxabcxxxxabcx


```
```output1
11
```
