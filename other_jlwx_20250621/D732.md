## Description

小明去超市买商品，已经买了N件，每件商品的价格都是整数，小明发现今天买的东西太多了，决定如果购买商品的总金额大于100元，就把最贵的商品放回去（如果有多个商品价格一样，且都是最贵的，只要放回一件就可以）。 
请编程帮助小明计算一下，是否要放回一件商品，最终小明付了多少钱？

## Input Format

第1行有一个整数n（n&lt;=100） 
第2行有n个整数，用空格隔开。 

## Output Format

第1行输出小明是否需要放回最贵的商品，如果要放回，请输出"Yes"，否则请输出"No"。 
第2行输出小明最终应该付款的金额。
 

```input1
8
21 18 32 10 5 46 58 1
```
```output1
Yes
133
```
