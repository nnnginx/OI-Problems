## Description

有一个很大的整数 n （ n&gt;=10000 且 n&lt;=9999999999 ），请问该数的最高位是什么量级的，输出该量级的拼音？  
 
可选单位：万（ wan ）、十万（ shi wan ）、百万（ bai wan ）、千万（ qian wan ）、亿（ yi ）、十亿（ shi yi ）。  
如： n=123456789 ，则输出： yi 

## Input Format

 一个很大的整数  n  （  n&gt;=10000  且  n&lt;=9999999999  ） 

## Output Format

n量级的拼音

```input1
123456789
```
```output1
yi
```
