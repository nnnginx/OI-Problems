## Description

<p>Bessie 处于半梦半醒的状态。过了一会儿，她意识到她在数数，不能入睡。</p><p>Bessie的大脑反应灵敏，仿佛真实地看到了她数过的一个又一个数。她开始注意每一个数码（$0 \ldots 9$）：每一个数码在计数的过程中出现过多少次？</p><p>给出两个整数 $M$ 和 $N$ （$1 \leq M \leq N \leq 2 \times 10^9$ 以及 $N-M \leq 5 \times 10^5$），求每一个数码出现了多少次。<br /></p>

## Input Format

<p>第 $1$ 行: 两个用空格分开的整数 $M$ 和 $N$。<br /></p>

## Output Format

<p>第 $1$ 行: 十个用空格分开的整数，分别表示数码 $0 \ldots 9$ 在序列中出现的次数。<br /></p>

```input1
129 137
```
```output1
1 10 2 9 1 1 1 1 0 1
```
