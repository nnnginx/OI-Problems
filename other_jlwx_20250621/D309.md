## Description

暑假来了，有 n 名同学都参加了暑假兴趣班（ n&lt;10 ），不同的兴趣班都是从同一天开始上课， n 个同学的上课频率（每隔几天上一次课，如每隔 3 天上一次课，上课频率就是 3 ）不同，请问除了第一天，最少要等到第几天这 n 个同学会在同一天一起上课？   
 
 比如：有2位同学，同学甲3天上1次课，同学乙2天上一次课，他们的上课时间分布就如下表所示，2位同学除了第1天，至少要等到第7天才能一起上课。
![兴趣班排班.png](/api/public/img/21c628875d574d5b8544d3a21906cd4c.png)


## Input Format

第一行一个整数 n 代表学生的数量，第二行 n 个数代表每位同学的上课频率。   
 

## Output Format

接下来到第几天所有人会在同一天上课。（本题测试数据确保答案≤10<sup>5</sup> ）   
 

```input1
3
3 2 4
```
```output1
13
```
