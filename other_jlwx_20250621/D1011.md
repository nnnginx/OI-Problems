## Description

学校准备召开年度文艺汇演，六年级的同学们决定一起订做新校服，参加汇演。
现给出每件校服的价钱，以及六年级每个班级的人数，请编程计算出，所有同学订做消费需要的总价钱。
 

## Input Format

第一行：一个整数x，表示每件校服的价钱。(x<=100)
第二行：一个整数n，表示六年级的班别数。(n<=10)
第三行：n个整数，表示六年级各班的人数。（各班人数在50人以内）
 

## Output Format

一个整数代表全年级订制校服所需要的总价钱。

```input1
30
6
48 50 51 49 50 52
```
```output1
9000
```
