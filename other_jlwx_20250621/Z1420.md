## 题目描述

<p>已知k，a，p，求x ^ k=a (mod p)的所有根（根的范围[0,p-1]</p>

## 输入格式

<p>三个整数p,k,a。0 < = a <p < = 10^9, 2 < = k < = 100000</p>

## 输出格式

<div>
 第一行一个整数，表示符合条件的x的个数。
</div>
<div>
 第二行开始每行一个数，表示符合条件的x,按从小到大的顺序输出。
</div>

```input1
11 3 8
```
```output1
1
2
```
