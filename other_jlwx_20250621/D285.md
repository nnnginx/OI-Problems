## Description

花花所在的学校引入了电水箱为同学们烧开水。已知电水箱的容量为 n 升（ n&lt;=10L ），同学们带的杯子平均容量为 x 毫升（ x 在 100~300 之间），请问烧一箱开水，最多能倒多少杯（不足 1 杯算 1 杯）。
 

## Input Format

2 个整数 n 和 x ； n 代表电水箱的总容量（单位为升 L ）， x 代表平均 1 个杯子的容量（单位为毫升 ml ）  
 

## Output Format

一个整数，代表最多能够倒多少杯水（不足 1 杯算 1 杯）  
 

```input1
1 120
```
```output1
9
```
