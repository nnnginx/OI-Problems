## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">给你一个n种面值的货币系统，求组成面值为m的货币有多少种方案。样例：设n=3，m=10，要求输入和输出的格式如下：</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">第一行两个整数n，m（m&lt;=5000)</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">以下n行，每行一个整数，第i+1行为第i种货币的面值</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一个整数，为方案数</span><br /></p>

```input1
3 10
1
2
5
```
```output1
10
```
