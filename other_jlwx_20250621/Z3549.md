## 题目描述

<p><span style="font-size: medium">给定N个积木，编号为1..N，每个积木高度为1，宽度为w_i，你可以把若干个积木放在一层上，堆成若干层，要求满足两个条件：<br>
   （1）对于任意一层的积木，他的宽度之和要小于等于他下面那一层的积木（最底层除外）。<br>
   （2）不允许编号小的放在编号大的的积木上面。<br>
   让你求最多能够堆多少层。</span></p>
<p></p>

## 输入格式

<p><span style="font-size: medium"><br>
   第一行一个数N。<br>
   第二行N个整数w_i。</span></p>
<p></p>

## 输出格式

<p><span style="font-size: medium">一行一个整数表示答案。</span></p>
<p></p>

```input1
3
1 2 3
```
```output1
2
```
## 数据范围与约定

<p><span style="font-size: medium">【数据范围】<br><br>
   N<=10^5，w_i<=10^4。<br><br></span></p>
