## 题目描述

<div>
 给定正整数N，R。求
</div>
<div>
 <img src="https://s2.loli.net/2023/08/15/R3l8eXtFIvYapQO.png" width="270" height="129" alt="">
</div>
<div></div>
<p></p>

## 输入格式

<div>
 第一行一个数 T，表示有 T 组测试数据。
</div>
<div>
 接下来 T 行，每行两个正整数 n,r。
</div>
<div></div>
<p></p>

## 输出格式

<div>
 输出 T 行，每行一个整数表示答案。
</div>
<div></div>
<p></p>

```input1
3
3 5
3 6
3 7
```
```output1
3
1
-1
```
## 数据范围与约定

<div>
 对于 100% 的数据，满足 n≤10^9,r≤10^4,T≤10^4。
</div>
