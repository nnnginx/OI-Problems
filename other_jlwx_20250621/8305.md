## Description

<p>大数学家高斯小时候偶然间发现一种有趣的自然数集合Blah，对于以a为基的集合Ba定义如下：<p><p>(1)a是集合Ba的基，且a是Ba的第一个元素；<p><p>(2)如果x在集合Ba中，则2x+1和3x+1也都在集合Ba中；<p><p>(3)没有其他元素在集合Ba中了。<p><p>现在小高斯想知道如果将集合Ba中元素按照升序排列，第N个元素会是多少？<p>

## Input Format

<p>输入包括很多行，每行输入包括两个数字，集合的基a(1≤a≤50))以及所求元素序号n(1≤n≤1000000)。<p>

## Output Format

<p>对于每个输入，输出集合Ba的第n个元素值。<p>

```input1
1 100
28 5437


```
```output1
418
900585


```
