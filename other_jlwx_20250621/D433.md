## Description

对于一个五位数abcde，可将其拆分为三个子数： 
sub1=abc 
sub2=bcd 
sub3=cde 
例如，五位数20207可以拆分成 
sub1=202 
sub2=020(也就是20) 
sub3=207 
现在给定一个正整数K，要求你编程求出10000到30000之间所有满足下述条件的五位数，条件是这些五位数的三个子数sub1,sub2,sub3都可被K整除。 

## Input Format

一个正整数K（0&lt;K&lt;1000）

## Output Format

每一行为一个满足条件的五位数，要求从小到大输出。不得重复输出或遗漏。如果无解，则输出“No”。

```input1
15
```
```output1
22555
25555
28555
30000
```
