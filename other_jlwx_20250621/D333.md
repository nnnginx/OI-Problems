## Description

在统计描述中，方差用来计算每一个变量（观察值）与总体均数之间的差异。比如：甲乙2个厂商生产某零件，一批零件要求在尺寸合格的情况下，大小越一致越好，由于生产工艺的问题，零件生产厂商生产的零件不可能一模一样。
 
为了检测甲乙两个厂商，那个厂商生产的零件更符合标准，分别从2个厂商生产的零件中抽取5个样品尺寸如下：
甲：100 101 102 100 99
乙：98 100 105 103 96
假设零件尺寸在95~110之间都算合格，那么两批零件都是合格的。
如果按照平均数计算，两组数据的平均值都是100.4
但如果将两组数据画到数轴上：
<img src="http://oj.czos.cn/upload/image/20180625/20180625005547_34264.jpg" alt="" />
<img src="http://oj.czos.cn/upload/image/20180625/20180625005933_96030.jpg" alt="" /> 
从两个厂抽检的零件分布图可以看出，甲厂的零件大小更加一致，更加符合标准。
为了方便计算数据的离散程度，我们引入方差的概念，方差的计算公式为：
 
 <img src="http://oj.czos.cn/upload/image/20180625/20180625005645_37927.jpg" alt="" /> 
其中x<sub>1</sub>~x<sub>n</sub>代表一组数据中的每个元素，
<img src="http://oj.czos.cn/upload/image/20180625/20180625005719_41131.jpg" alt="" />
代表这组数据的平均值。
按照公式，甲厂零件的方差为：
(100-100.4)<sup>2</sup>+(101-100.4)<sup>2</sup>+(102-100.4)<sup>2</sup>+(100-100.4)<sup>2</sup>+(99-100.4)<sup>2</sup> = 5.2
乙厂零件的方差为：
(98 -100.4)<sup>2</sup>+(100-100.4)<sup>2</sup>+(105-100.4)<sup>2</sup>+(103-100.4)<sup>2</sup>+(96-100.4)<sup>2</sup> = 53.2
从方差上也可以看出，甲厂的零件更符合标准！
现从键盘读入2个厂生产的零件尺寸（假设零件的尺寸都是合格的），请计算哪个厂的零件尺寸更加一致（方差更小）？
 
	 
 

## Input Format

第一行为一个整数n，代表2个厂抽检的零件的个数！（n在5~100之间）
第二行为甲厂的n个零件的尺寸
第三行为乙厂的n个零件的尺寸
 

## Output Format

哪个厂的零件更加符合标准，甲厂请输出“jia”，乙厂请输出“yi”

```input1
5
100 101 102 100 99
98 100 105 103 96
```
```output1
jia
```
