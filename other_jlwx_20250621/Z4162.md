## 题目描述

<div>
 给定矩阵 M，请计算 M^n，并将其中每一个元素对 1000000007 取模输出。
</div> 
<div></div> 
<p></p>

## 输入格式

<div>
 第 1 行包含两个整数 n,k，其中 n 使用二进制表示，可能含有前导零;
</div> 
<div>
 余下 k 行描述了一个 k * k 的矩阵 M。
</div> 
<div></div> 
<p></p>

## 输出格式

<div>
 输出题目描述中要求的矩阵，格式同输入。
</div> 
<div></div> 
<p></p>

```input1
010 3
5 9 5
5 4 0
8 8 8
```
```output1
110 121 65
45 61 25
144 168 104
```
## 数据范围与约定

<p>对于 100% 数据，满足 n <= 2^10000;k <= 50; 0 <= Mij < 10^9 +7</p>
<br>
