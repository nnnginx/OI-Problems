## Description

设有N 个正整数（1 &lt;= N &lt;= 50），其中每一个均是大于等于1、小于等于10000的数。从这N个数中任取出若干个数（不能取相邻的数），要求得到一种取法，使得到的和为最大。 
例如：当N=5时，有5个数分别为：13，18，28，45，21 
&nbsp;此时，有许多种取法，如： 
13,28,21和为62 
13,45和为58 
18,45和为63 
……… 
和为63就是满足要求的一种取法。

## Input Format

输入数据的第一行是一个整数N 
第二行有N个符合条件的整数，数与数之间的间隔是用一个空格分开。

## Output Format

输出数据仅有一行包含一个整数，即最大和。

```input1
5
13 18 28 45 21
```
```output1
63
```
