## Description

如果一个序列满足下面的性质，我们就将它称为摆动序列：  
   1.  序列中的所有数都是不大于<i> k </i>的正整数；  
   2.  序列中至少有两个数。  
   3.  序列中的数两两不相等；  
   4.  如果第<i> i </i>  C 1 个数比第<i> i </i>  C 2 个数大，则第<i> i </i>个数比第<i> i </i>  C 2 个数小；如果第<i> i </i>  C 1 个数比第<i> i </i>  C 2 个数小，则第<i> i </i>个数比第<i> i </i>  C 2 个数大。  
  比如，当<i> k </i>  = 3 时，有下面几个这样的序列：  
   1 2 
   1 3 
   2 1 
   2 1 3 
   2 3 
   2 3 1 
   3 1 
   3 2 
  一共有 8 种，给定<i> k </i>，请求出满足上面要求的序列的个数。    
 

## Input Format

输入包含了一个整数<i> k </i>。（<i> k </i> &lt;=20 ）    
 

## Output Format

输出一个整数，表示满足要求的序列个数。

```input1
3
```
```output1
8
```
