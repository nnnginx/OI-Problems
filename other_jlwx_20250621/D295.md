## Description

四年级二班第一小组在体育课上测试了立定跳远成绩，男生和女生分别测试，请根据记录的立定跳远成绩计算出男生和女生的平均成绩分别是多少？（结果保留 1 位小数） 
 

## Input Format

两行，第一行为若干男生的跳远成绩（人数&lt;=100），以0表示结束
第二行为若干女生的跳远成绩（人数 &lt;=100 ），以 0 表示结束
	 
 

## Output Format

一行 2 个数以空格隔开，分别代表男生和女生的立定跳远的平均分（结果保留 1 位小数）。  
 

```input1
157 154 159 160 0
142 147 144 0
```
```output1
157.5 144.3
```
