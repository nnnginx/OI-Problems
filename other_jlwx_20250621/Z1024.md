## 题目描述

<p>　　windy的生日到了，为了庆祝生日，他的朋友们帮他买了一个边长分别为 X 和 Y 的矩形蛋糕。现在包括windy<br>
  ，一共有 N 个人来分这块大蛋糕，要求每个人必须获得相同面积的蛋糕。windy主刀，每一切只能平行于一块蛋糕<br>
  的一边（任意一边），并且必须把这块蛋糕切成两块。这样，要切成 N 块蛋糕，windy必须切 N-1 次。为了使得<br>
  每块蛋糕看起来漂亮，我们要求 N块蛋糕的长边与短边的比值的最大值最小。你能帮助windy求出这个比值么？</p>

## 输入格式

<p>　　包含三个整数，X Y N。1 <= X,Y <= 10000 ； 1 <= N <= 10</p>

## 输出格式

<p>　　包含一个浮点数，保留6位小数。</p>

```input1
5 5 5
```
```output1
1.800000
```
