## Description

<p><span style="color: rgb(77, 77, 77);">现有$N$条绳子，它们的长度分别为$L1，L2，\cdots Ln$，如果从它们中切割出$K$条长度相同的绳子，这$K$条绳子每条最长能有多长？</span><br /></p>

## Input Format

<p><span style="color: rgb(77, 77, 77);">共有两行，第一行包含两个正整数$N$和$K$，用一个空格分隔；</span></p><p><span style="color: rgb(77, 77, 77);">第二行包含$N$个数，依次表示$N$条绳子的长度，两数间用一个空格分隔。每条绳子长度的小数不超过两位。（$1 \le N \le 1000，1 \le K \le 1000，1 \le Li \le 10000$）</span><br /></p>

## Output Format

<p><span style="color: rgb(77, 77, 77);">仅包含一个数，表示所得$K$绳子的最大长度。</span><br /></p>

```input1
4 11
8.02 7.43 4.57 5.39
```
```output1
2.00
```
