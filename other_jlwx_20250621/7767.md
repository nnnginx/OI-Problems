## Description

根据输入的正整数n，画出如图所示的U型图案。
例如n = 3时，图案为：
```
* *
* *
***
```



## Input Format

输入1个正整数n（3 <= n <= 20），表示这个U型图案的宽和高。


## Output Format

根据题目描述，输出对应的图案。

```input1
3
```
```output1
* *
* *
***
```
```input2
5
```
```output2
*   *
*   *
*   *
*   *
*****
```
## Hint

1、图案的第一行前面不能有多余的空行；
2、图案的最左侧从第一列开始输出，左边不能有多余的空格。
