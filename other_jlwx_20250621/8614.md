## Description

<p>格格兰郡的 $N$ 名士兵随机散落在全郡各地。</p><p>格格兰郡中的位置由一对 $(x,y)$ 整数坐标表示。</p><p>士兵可以进行移动，每次移动，一名士兵可以向上，向下，向左或向右移动一个单位（因此，他的 $x$ 或 $y$ 坐标也将加 $1$ 或减 $1$）。</p><p>现在希望通过移动士兵，使得所有士兵彼此相邻的处于同一条水平线内，即所有士兵的 $y$ 坐标相同并且 $x$ 坐标相邻。</p><p>请你计算满足要求的情况下，所有士兵的总移动次数最少是多少。</p><p>需注意，两个或多个士兵不能占据同一个位置。</p>

## Input Format

<p>第一行输入整数 $N$，代表士兵的数量。</p><p>接下来的 $N$ 行，每行输入两个整数 $x$ 和 $y$，分别代表一个士兵所在位置的 $x$ 坐标和 $y$ 坐标，第 $i$ 行即为第 $i$ 个士兵的坐标 $(x[i],y[i])$。</p>

## Output Format

<p>输出一个整数，代表所有士兵的总移动次数的最小值。<br /></p>

```input1
5
1 2
2 2
1 3
3 -2
3 3
```
```output1
8
```
## Hint

<p>数据范围</p><p>$1 \le N \le 10000$,</p><p>$-10000 \le x[i],y[i] \le 10000$</p>
