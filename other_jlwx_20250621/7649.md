## Description

根据输入的正整数n，用大写英文字母A画出如图所示的等腰字符三角形。
例如n = 5时，图案为：
```
    A
   AAA
  AAAAA
 AAAAAAA
AAAAAAAAA
```



## Input Format

输入1个正整数n（n <= 13），表示这个等腰字符三角形有多少行。

## Output Format

根据题目描述，输出对应的字符图案。

```input1
3
```
```output1
    A
  AAA
AAAAA
```
```input2
5
```
```output2
        A
      AAA
    AAAAA
  AAAAAAA
AAAAAAAAA
```
