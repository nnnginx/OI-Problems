## Description

<p>有一种很奇怪的电梯。大楼的每一层楼都可以停电梯，而且第$i$层楼$(1 \le i \le N)$上有一个数字$K_i(0 \le K_i \le N)$。</p><p>电梯只有四个按钮：开，关，上，下。上下的层数等于当前楼层上的那个数字。当然，如果不能满足要求，相应的按钮就会失灵。</p><p>例如：$3, 3 ,1 ,2 ,5$代表了$K_i(K_1=3,K_2=3,…)$，从$1$楼开始。在$1$楼，按“上”可以到$4$楼，按“下”是不起作用的，因为没有$-2$楼。</p><p>那么，从$A$楼到$B$楼至少要按几次按钮呢？<br /></p>

## Input Format

<p>共二行。</p><p>第一行为$3$个用空格隔开的正整数，表示$N,A,B(1≤N≤200, 1≤A,B≤N)$。</p><p>第二行为$N$个用空格隔开的非负整数，表示$K_i$。</p>

## Output Format

<p>一行，即最少按键次数,若无法到达，则输出$-1$。<br /></p>

```input1
5 1 5
3 3 1 2 5
```
```output1
3
```
