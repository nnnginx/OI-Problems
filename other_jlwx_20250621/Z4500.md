## 题目描述

<div>
 <div>
  有一个n*m的矩阵，初始每个格子的权值都为0，可以对矩阵执行两种操作：
 </div>
 <div>
  1.选择一行，该行每个格子的权值加1或减1。
 </div>
 <div>
  2.选择一列，该列每个格子的权值加1或减1。
 </div>
 <div>
  现在有K个限制，每个限制为一个三元组(x,y,c)，代表格子(x,y)权值等于c。
 </div>
 <div>
  问是否存在一个操作序列，使得操作完后的矩阵满足所有的限制。
 </div>
 <div>
  如果存在输出”Yes”，否则输出”No”。
 </div>
</div>
<p></p>

## 输入格式

<div>
 先输入一个T(T <= 5)代表输入有T组数据，每组数据格式为：
</div>
<div>
 第一行三个整数n, m, k (1 <= n, m,k <= 1000)。
</div>
<div>
 接下来k行，每行三个整数x, y, c。
</div>
<p></p>

## 输出格式

<div>
 对于每组数据，输出Yes或者No。
</div>
<p></p>

```input1
2
2 2 4
1 1 0
1 2 0
2 1 2
2 2 2
2 2 4
1 1 0
1 2 0
2 1 2
2 2 1
```
```output1
Yes
No
```
