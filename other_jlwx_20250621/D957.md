## Description

给定n个字符串，请问该字符串是否是回文字符串（回文字符串指的是正过来读和反过来读是一样的字符串，比如"abcba"），如果不是回文字符串，请问该字符串在调整字符之间的顺序之后能否构成回文（比如："ababc"虽然不是回文字符串，但调整字符之间的顺序之后，得到"abcba"就是回文字符串）。
 

	 
 

## Input Format

第1行输入一个整数n。（n&lt;=1000）
接下来n行，每行有一个字符串，字符串为长度不超过10的小写字母，且不包含空格。
 

	 
 

## Output Format

对于每组输入，如果输入的字符串本身就是回文串，或者将这个字符串里的字符重新排列后可以构成回文串，则输出Yes，否则输出No。
 

	 
 

```input1
3
qq
abc
aab
```
```output1
Yes
No
Yes
```
## Source

字符串
