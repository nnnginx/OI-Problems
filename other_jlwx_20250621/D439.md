## Description

夏天到了，各家各户的用电量都增加了许多，相应的电费也交的更多了。小玉家今天收到了一份电费通知单。小明看到上面写：据闽价电[2006]27号规定，月用电量在150千瓦时及以下部分按每千瓦时0.4463元执行，月用电量在151~400千瓦时的部分按每千瓦时0.4663元执行，月用电量在401千瓦时及以上部分按每千瓦时0.5663元执行;小明想编写编写一个程序，根据已知用电总计以及电价规定，计算出应交的电费应该是多少。 
比如：假设用电量是267度，那么应交电费计算=150*0.4463+(267-150)*0.4663=121.5(结果保留1位小数) 

## Input Format

输入一个整数，表示用电总计（单位以千瓦时计），不超过10000。

## Output Format

输出一个数，保留到小数点后1位（单位以元计，保留到小数点后1位）。

```input1
267
```
```output1
121.5
```
