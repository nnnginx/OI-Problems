## Description

<p>小明有一个数字 x，现在他想把 x 的各个数字调换位置，得到一个最大的数和一个最小的数，你能帮帮他吗？注：小明不希望数字的最高位是 0。<br /></p>

## Input Format

<p>输入一个数 x；<br /></p>

## Output Format

<p>输出两个数，分别表示调换后的最大数和最小数，以空格隔开；<br /></p>

```input1
9037
```
```output1
9730 3079
```
## Hint

<p><b>【数据范围】</b></p><p>对于 25%的数据，1≤x&lt;10000；</p><p>对于 100%的数据，1≤x&lt;10^9；</p><p><br /></p><p><b>【样例解释】</b></p><p>如 9037，包含&quot;9&quot;&quot;0&quot;&quot;3&quot;&quot;7&quot;；可组成最大的数是 9730；</p><p>由于&quot;0&quot;不能在千位，可组成最小的数是 3079。
