## Description

有n个球队1≤n≤100，编号为1,2,…,n，共进行n场比赛。每场比赛有一个胜队。计分方法如下：
是连胜中的第一次胜利，则本次胜利得1分。
是连胜中的第二次胜利，则本次胜利得2分。
是连胜中的第三次胜利，则本次胜利得3分。
连胜超过三次以上的胜场，每场得3分。
例如n=12，比赛的胜队为1 2 1 1 3 2 1 1 1 1 4 2，计分如下：
队1 &nbsp; 1+1+2+1+2+3+3=13分
队2 &nbsp; 1+1+1=3分
队3 &nbsp; 1
队4 &nbsp; 1
其余队为0分。
 

## Input Format

n，x<sub>1</sub>两个整数，n为球队数，x<sub>1</sub>为第一次胜队号，第 i 场比赛（i≥2）胜队的编号由以下公式确定： 
x<sub>i</sub>=((x<sub>i-1</sub>×3703+1047) &nbsp;mod n)+1

## Output Format

一个整数，即得分最多队的分数。

```input1
10 5
```
```output1
3
```
