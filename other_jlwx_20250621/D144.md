## Description

 四（1）班要举行一次歌唱比赛，以选拔更好的苗子参加校的歌唱比赛。评分办法如下：设N个评委，打N个分数（0 &lt;= 每个分数 &lt;= 10），去掉一个最高分，去掉一个最低分，剩下的评委的平均分即为该选手的最后得分。但是选手太多了，靠人工计算每个选手的得分太慢太麻烦。你能不能帮帮他们，设计一个程序让计算机来算出选手的最后得分呢？

## Input Format

 第一行为一个整数N（ 5 &lt;= N &lt;= 10 ）&nbsp;  
 第二行为N个整数Ai（ 0 &lt;= Ai &lt;= 10 ） 

## Output Format

 选手的最后得分（保留两位小数） 

```input1
5
5 6 7 8 9
```
```output1
7.00
```
