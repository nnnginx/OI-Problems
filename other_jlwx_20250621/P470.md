## Description

<p>给你两个字符串，一个母串，一个子串，请你找出子串第一次在母串中出现的位置。如果子串没有在母串中出现过，则输出-1。例如子串 ab 在母串 dceab 中第一次出现的位置是 3，而子串 abc 则在 dceab 中没有出现过。<br /></p>

## Input Format

<p>第一行一个字符串（母串），保证每个字符都是小写字母。 第二行一个字符串（子串），保证每个字符都是小写字母。 保证两个字符串的长度都不超过 10000，并且大于 0。<br /></p>

## Output Format

<p>一行一个整数，表示子串第一次在母串中出现的位置。假如子串没有在母串中出现过，则输出-1。<br /></p>

```input1
decdagee
age
```
```output1
4
```
