## 题目描述

<div>
 求从0 ~ N C 1中选出K个互不相同的数，使得这些数的和mod N = 0的方案数。输出这个方案数mod (10 ^ 9 + 7)的值。
</div>
<p></p>

## 输入格式

<div>
 仅一行，两个数N,K。
</div>
<p></p>

## 输出格式

<div>
 仅一行，即答案。
</div>
<p></p>

```input1
7 4
```
```output1
5
```
## 数据范围与约定

<div>
 {0, 1, 2, 4}, {0, 3, 5, 6}, {1, 2, 5, 6}, {1, 3, 4, 6}, {2, 3, 4, 5}
</div><br>
<div></div><br>
<div>
 100 : N <= 10 ^ 9, K <= 1000
</div><br>
<div></div><br>
<p></p>
