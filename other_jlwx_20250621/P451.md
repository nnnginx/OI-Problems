## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">回文数是从前往后和从后往前得到的数是相同的。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">现给你一个正整数N，请你找到比N大的最小的那个回文数P。</span></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输入包含多组测试数据。</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">每组输入一个正整数N，N不超过10000位，并且N不包含前导0。</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">对于每组输入，输出比N大的最小的那个回文数P。</span><br /></p>

```input1
44
3
175
```
```output1
55
4
181
```
