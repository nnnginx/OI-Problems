## Description

<p>给出一张n*m的地图，在地图上有一只虫，样子却很像龙，而且嘴能快速地喷（直射）出一种毒液，瞬间杀死敌人。</p><p>现在假设虫的初始位置在（X1，Y1），另外在（X2，Y2）处有一个敌人。假设虫移动一步需要单位1的时间，而杀死敌人不需要时间，并且虫的毒液射程无穷大，但毒液不能穿透阻碍物，虫只能攻击上、下、左、右、左上、右上、左下、右下八个方向。</p><p>请求出虫最少需要用多少时间才能消灭敌人。</p>

## Input Format

<p>第1行为2个数N和M，表示矩阵的规模（N为高，M为宽）。</p><p>接下来是N*M的矩阵，O表示空地，X表示障碍物。</p><p>下面是若干行数据，每行为一对数据，分别是敌人的位置和虫的位置。显然，敌人和虫都不可能在障碍物上。</p><p>已“0 0 0 0”输入结束标志。</p>

## Output Format

<p>输出第1行为虫能消灭掉敌人的最短时间。</p><p>显然，若能直接打到敌人，则时间为0；若无法消灭，则第2行再输出<code>Impossible!</code>。</p>

```input1
3 4
OXXO
XXOO
XOOO
3 2 2 4
3 3 1 1
0 0 0 0
```
```output1
1
Impossible!
```
## Hint

<p>对于30%的数据满足：$N*M≤5000$。</p><p>对于50%的数据满足：$N*M≤10000$。</p><p>对于100%的数据满足：$N*M≤20000$。</p>
