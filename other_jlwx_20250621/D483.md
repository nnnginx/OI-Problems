## Description

小蓝 是课代表，老师判好卷子之后让她录成绩，她念了n个人的学号(学号是一个绝对值10^9以内的整数包括0，没有负数)，但是老师总觉得她念重复了。 
还好老师用声控软件记录下了他念的每一个学号，现在请你写程序来判断 小蓝 有没有念过某个或者某些学号2次以上。 
如果有请你输出 1 
否则输出0

## Input Format

只有1组测试数据 
一个正整数n(1&lt;=n&lt;100000)代表 小蓝 念了学号的个数 
之后第2行到第n+1行，每行一个10^9^以内的整数(包括0没有负数)，代表某个同学的学号。

## Output Format

输出一个整数，如果有重复的学号输出1，否则输出0

```input1
4
11222222
0
1
11222222
```
```output1
1
```
