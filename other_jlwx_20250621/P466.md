## Description

<p>已知一棵二叉树，中序查找二叉树中值为x的结点，并指出是第几个结点。<br /></p><p>例：如图二叉树的中序遍历序列为：29、12、8、15、23、5、10，中序查找15返回4，表示15在中序遍历序列的第四位。</p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">第一行</span><span style="color: rgba(0, 0, 0, 0.87);">n</span><span style="color: rgba(0, 0, 0, 0.87);">为二叉树的结点个树，</span><span style="color: rgba(0, 0, 0, 0.87);">n&lt;=100</span><span style="color: rgba(0, 0, 0, 0.87);">；第二行</span><span style="color: rgba(0, 0, 0, 0.87);">x</span><span style="color: rgba(0, 0, 0, 0.87);">表示要查找的结点的值；</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">以下第一列数据是各结点的值（均不相等），第二列数据是左儿子结点编号，第三列数据是右儿子结点编号。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出一个数即查找的结点编号。</span><br /></p>

```input1
7
15
5 2 3
12 4 5
10 0 0
29 0 0
15 6 7
8 0 0
23 0 0
```
```output1
4
```
