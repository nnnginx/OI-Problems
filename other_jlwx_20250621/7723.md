## Description

今年大学生运动会，小蓝要根据每个学校获得的金牌、银牌和铜牌数量，统计出每个学校获得的奖牌总数。


## Input Format

输入：第1行是1个正整数，表示参加运动会的学校数目n（1≤n≤2000）。
其后有n行数据，每行包括4个非负整数，分别表示学校编号（不超过999999），以及金牌、银牌和铜牌数量（不大于200），两项数据中间用空格分隔。


## Output Format

输出：共n行，每行包含2个非负整数，表示某学校的学校编号以及获得的奖牌总数，中间用单个空格分隔。


```input1
3
1 4 12 24
2 3 20 30
3 0 30 23

```
```output1
1 40
2 53
3 53

```
