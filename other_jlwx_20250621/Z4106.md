## 题目描述

<div>
 <div>
  有一个数列price_k = p * (sin(a * k + b) + cos(c * k + d) + 2).
 </div>
 <div>
  求最大的(price_i - price_j) (1 <= i <= j <= n)
 </div>
 <div></div>
</div>
<div></div>

## 输入格式

<p>多组数据.每组数据六个整数p, a, b, c, d, n.</p>
<div></div>

## 输出格式

<p>每组数据输出一行表示答案,保留六位小数.</p>

```input1
392 456 222 152 222 348222
```
```output1
1556.786855
```
## 数据范围与约定

<p>1 <= p, a, b, c, d <= 1000, 1 <= n <= 1000000.</p>
