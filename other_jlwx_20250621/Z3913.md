## 输入格式

<p>第一行6个正整数K,L,R,A,B,C</p>

## 输出格式

<p> 第一行一个非负整数，表示答案对1000000007取模后的值</p>

```input1
1 372 989277441 666671441 337273093 691532461
```
```output1
273687536
```
## 数据范围与约定

<p>K<= 50，L<= R<= 10^9，1 <= A,B,C<=10^9</p>
