## Description

<p>已知 n 个整数b1,b2,…,bn</p><p>以及一个整数 k（k＜n）。</p><p>从 n 个整数中任选 k 个整数相加，可分别得到一系列的和。</p><p>例如当 n=4，k＝3，4 个整数分别为 3，7，12，19 时，可得全部的组合与它们的和为：<br />3＋7＋12=22　　3＋7＋19＝29　　7＋12＋19＝38　　3＋12＋19＝34。<br />现在，要求你计算出和为素数共有多少种。</p><p>例如上例，只有一种的和为素数：3＋7＋19＝29。</p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">第一行两个整数：</span><span style="color: rgba(0, 0, 0, 0.87);">n , k （1&lt;=n&lt;=20，k＜n）</span></p><p><span style="color: rgba(0, 0, 0, 0.87);">第二行n个整数：x1,x2，…,xn （1&lt;=xi&lt;=5000000）</span></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一个整数（满足条件的方案数）。</span><br /></p>

```input1
4 3
3 7 12 19
```
```output1
1
```
