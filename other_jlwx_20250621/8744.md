## Description

<p>在一个非降序列中，查找与给定值最接近的元素。<p>

## Input Format

<p>第一行包含一个整数n，为非降序列长度。1 ≤ n ≤ 100000。<p><p>第二行包含n个整数，为非降序列各元素。所有元素的大小均在0-1,000,000,000之间。<p><p>第三行包含一个整数m，为要询问的给定值个数。1 ≤ m ≤ 10000。<p><p>接下来m行，每行一个整数，为要询问最接近元素的给定值。所有给定值的大小均在0-1,000,000,000之间。<p>

## Output Format

<p>m行，每行一个整数，为最接近相应给定值的元素值，保持输入顺序。若有多个值满足条件，输出最小的一个。<p>

```input1
3
2 5 8
2
10
5

```
```output1
8
5


```
