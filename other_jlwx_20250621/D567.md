## Description

花花去文具店买了 1 支笔和 1 块橡皮，已知笔 x 元 / 支，橡皮 y 元 / 块，文具店今天有活动，可以给顾客打 9 折（用购买总价  * 0.9 ），花花付给了老板 n 元，请问老板应该找给花花多少钱？  
 

## Input Format

三个整数 x 、 y 、 n ，分别代表了笔的单价、橡皮的单价和花花付给老板的钱（已知花花付给老板的钱 n&gt;=x+y ）  
 

## Output Format

一个小数（保留 1 位小数），代表老板应该找给花花的钱  
 

```input1
2 1 5
```
```output1
2.3
```
