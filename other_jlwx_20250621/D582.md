## Description

小明和小华去购买课外书，小明带了 x 元（ x 是一个三位整数），小明问小华带了多少钱，小华说自己带的钱只有小明带的金额的个位和百位的和那么多，请你编程帮助小华计算一下，他带了多少零花钱？  
 
比如，假设小明带了 689 元，那么小华带的金额  = 6 + 9 = 15 元。  
 

## Input Format

一个整数 x 代表小明的零花钱的金额， x 是一个三位的整数 。 
 

## Output Format

小华的零花钱的金额 。 
 

```input1
689
```
```output1
15
```
