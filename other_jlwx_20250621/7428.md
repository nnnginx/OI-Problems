## Description

每个物品有一定的体积（废话），不同的物品组合，装入背包会战用一定的总体积。
假如每个物品有无限件可用，那么有些体积是永远也装不出来的。为了尽量装满背包，附中的OIER想要研究一下物品不能装出的最大体积。题目保证有解，如果是有限解，保证不超过2，000，000，000
如果是无限解，则输出0

## Input Format

第1行一个整数n（n<=10），表示物品的件数
第2行到N+1行: 每件物品的体积(1<= <=500)

## Output Format

一个整数ans，表示不能用这些物品得到的最大体积。

```input1
3
3
6
10
```
```output1
17
```
