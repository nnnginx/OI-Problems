## 题目描述

<div>
 　　每一头牛的愿望就是变成一头最受欢迎的牛。现在有N头牛，给你M对整数(A,B)，表示牛A认为牛B受欢迎。 这
</div>
<div>
 种关系是具有传递性的，如果A认为B受欢迎，B认为C受欢迎，那么牛A也认为牛C受欢迎。你的任务是求出有多少头
</div>
<div>
 牛被所有的牛认为是受欢迎的。
</div>

## 输入格式

<div>
 　　第一行两个数N,M。 接下来M行，每行两个数A,B，意思是A认为B是受欢迎的（给出的信息有可能重复，即有可
</div>
<div>
 能出现多个A,B）
</div>

## 输出格式

<p>　　一个数，即有多少头牛被所有的牛认为是受欢迎的。</p>

```input1
3 3
1 2
2 1
2 3
```
```output1
1
```
## 数据范围与约定

<div>
 <span style="font-size: medium;">100%的数据N<=10000,M<=50000</span>
</div>
