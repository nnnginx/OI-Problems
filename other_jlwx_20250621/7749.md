## Description

小蓝最近刚学习英文字母表。为了加深记忆，小蓝决定通过编程输出字母表中前5个字母的小写形式：a、b、c、d、e。

## Input Format

无
注意：本题没有输入！ 

## Output Format

输出共五行，按英文字母表顺序输出前5个字母的小写形式，每输出一个字母换一行。
注意：
1. 程序区分大小写，输出的字母如果是大写则不得分。
2. 输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
无
```
```output1
a
b
c
d
e
```
