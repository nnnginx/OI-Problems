## Description

<p>运动会上学校为大家购买了一批很漂亮的$T-shirt$，明明需要把所以衣服从商店运到赛场。</p><p>地图上有 $N$ 个路口，标号为 $1$ 的路口是商店所在地，标号为 $N$ 的路口是赛场所在地。有 $M$ 条路，每条路的数据包含三个整数 $A,B,C$ ，表示路口 $A$ 和路口 $B$之间有一条路，需要 $C$ 分钟的时间走过这条路。</p>

## Input Format

<p>输入数据第一行是两个整数$N$、$M$（$N \le 100，M \le 10000$）。</p><p>接下来 $M$ 行，每行包括 $3$ 个整数 $A$，$B$，$C$（$1 \le A,B \le N,1 \le C \le 1000$）,表示在路口 $A$ 与路口 $B$ 之间有一条路，需要 $C$ 分钟的时间走过这条路。</p><p>输入保证至少存在$1$条商店到赛场的路线。</p>

## Output Format

<p>输出一行，表示从商店走到赛场的最短时间<br /></p>

```input1
2 1
1 2 3
```
```output1
3
```
```input2
3 3
1 2 5
2 3 5
3 1 2
```
```output2
2
```
