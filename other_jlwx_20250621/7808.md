## Description

根据输入的正整数n，画一个n行n列的数字填充矩形图案。图中，第1行填充的是n个“1”，第2行是n个“2”，以此类推。
例如：当n=5时，图案如下：
11111
22222
33333
44444
55555



## Input Format

输入一个正整数n（1 <= n <= 9），表示矩形的宽和高。



## Output Format

按题目描述的要求输出对应的矩形图案。

注意事项：
图案的第一行前面不能有多余的空行；
图案的最左侧从第一列开始输出，数字之间不能有多余的空格。


```input1
5
```
```output1
11111
22222
33333
44444
55555

```
