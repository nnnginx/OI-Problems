## Description

指挥部被突来的洪水淹没了，还好指挥部有在某些重要的地方起一些围墙，用*号表示，而一个封闭的*号区域洪水是进不去的……现在给出指挥部的围墙建设图，问指挥部没被淹到的重要区域（由"0"表示）有多少。 

## Input Format

第一行是两个数，x和y（x,y&lt;=500） 
第二行及以下是一个由*和0组成的x*y的图。

## Output Format

输出没被水淹没的指挥部的“0”的数量（计算被*围住的0的个数）。

```input1
样例输入1
4 5
00000
00*00
0*0*0
00*00

样例输入2
5 5
*****
*0*0*
**0**
*0*0*
*****
```
```output1
样例输出1
1

样例输出2
5

```
## Source

深搜 广搜
