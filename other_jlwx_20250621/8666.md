## Description

<p>给定两个数L和R，求L到R之间（包括L和R）有多少个数是“优雅”的。</p><p>称一个数是优雅的，当且仅当把这个数看成一个长度为n的字符串（没有前导零），n个字符中 n-1 个字符全相同，有且仅有一个字符不同。</p><p>例如，33323、110都是优雅的，而9779、55555都不是优雅的。</p>

## Input Format

<p><span style="color: rgb(51, 51, 51);">一行两个数L和R。</span><br /></p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">一行一个数，表示L到R之间有多少个数是优雅的。</span><br /></p>

```input1
110 133
```
```output1
13
```
## Hint

<p>样例解释：13个数是110、112、113、114、115、116、117、118、119、121、122、131、133.</p><p>数据范围：</p><p>对于50%数据： $100 \le L \le R \le 10^6$</p><p><span style="color: rgb(51, 51, 51);">对于100%数据： $100 \le L \le R \le 10^{16}$</span><br /></p>
