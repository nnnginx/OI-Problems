## Description

编写一个程序，判断一个矩阵是否只由 0 和 1 两种数字构成。

## Input Format

第 1 行两个用空格隔开的正整数， n 和 m ， n 和 m 均不大于 10 ，表示矩阵的大小，以下 n 行，每行 m <span class="GramE">个 字符，都在 '0' ～ '9' 之间，表示矩阵中的对应元素。

## Output Format

一行，包含一个字符串 Yes 或 No 。如果矩阵只有 0 和 1 两个 数字，输出 Yes ；否则输出 No 。

```input1
3 3
1 1 1
0 0 1
1 1 0
```
```output1
Yes
```
## Source

二维数组
