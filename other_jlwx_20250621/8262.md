## Description

<p>如果一个图存在一笔画，则一笔画的路径叫做欧拉路，如果最后又回到起点，那这个路径叫做欧拉回路。<p><p>根据一笔画的两个定理，如果寻找欧拉回路，对任意一个点执行深度优先遍历；找欧拉路，则对一个奇点执行dfs，时间复杂度为O(m+n)，m为边数，n是点数。<p>

## Input Format

<p>第一行n，m，有n个点，m条边，以下m行描述每条边连接的两点。<p>

## Output Format

<p>欧拉路或欧拉回路,输出一条路径即可。<p>

```input1
5 5
1 2
2 3
3 4
4 5
5 1

```
```output1
1 5 4 3 2 1

```
