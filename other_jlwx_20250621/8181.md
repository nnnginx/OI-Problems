## Description

<p>对于一个01字符串，如果将这个字符串0和1取反后，再将整个串反过来和原串一样，就称作“反对称”字符串。比如00001111和010101就是反对称的，1001就不是。</p><p>现在给出一个长度为N的01字符串，求它有多少个子串是反对称的。</p>

## Input Format

<p>第一行一个正整数N (N ≤ 500,000)。第二行一个长度为N的01字符串。<br /></p>

## Output Format

<p>一个正整数，表示反对称子串的个数。<br /></p>

```input1
8
11001011
```
```output1
7
```
## Hint

<p>7个反对称子串分别是：01(出现两次), 10(出现两次), 0101, 1100和001011<br /></p>
