## Description

<p>请你帮小瓜找一找某个字符在字符串中第一次出现的位置是多少。<br /></p>

## Input Format

<p>第一行一个整数 n(1 &lt;=n &lt;= 10000)，表示字符串的长度。 第二行一个长度为 n 的字符串，保证每个字符都是小写字母。 第三行一个字符，表示需要寻找的字符。<br /></p>

## Output Format

<p>一行一个整数，表示字符在字符串中第一次出现的位置（从第 0 位开始记）。如果字符没有出现过，则输出-1。<br /></p>

```input1
5
abcde
d
```
```output1
3
```
