## Description

<p>你们中的一些人可能玩过一个叫做消木块的游戏。</p><p>$n$ 个木块排成一列，每个木块都有一个颜色。</p><p>例如下图中木块的颜色分别为：金，银，银，银，银，铜，铜，铜，金。</p><p><img src="https://www.acwing.com/media/article/image/2019/02/17/19_e646835a32-1390_1.jpg" alt="1390_1.jpg" /><br /></p><p>每次，你都可以点击一个木块，这样被点击的木块以及和它相邻并且同色的木块就会消除。</p><p>如果一次性消除了 $k$ 个木块，那么就会得到 $k \times k$ 分。</p><p>例如下图所示，点击银色木块，四个木块被消去，得到 $16$ 分。</p><p><img src="https://www.acwing.com/media/article/image/2019/02/17/19_7d967bd432-1390_2.jpg" alt="1390_2.jpg" /><br /></p><p><br /></p><p>给定你一个游戏初始状态，请你求出最高得分是多少。</p>

## Input Format

<p>第一行包含整数 $t$，表示共有 $t$ 组测试数据。</p><p>每组数据第一行包含整数 $n$，表示共有 $n$ 个木块。</p><p>第二行包含 $n$ 个整数，表示 $n$ 个木块的颜色。</p><p>代表木块颜色的整数范围是 $1 \sim n$。</p>

## Output Format

<p>每组数据输出一个结果，每个结果占一行。</p><p>输出格式为Case x: y，其中 $x$ 为数据组别编号，从 $1$ 开始，$y$ 为结果。</p>

```input1
2
9
1 2 2 2 2 3 3 3 1
1
1
```
```output1
Case 1: 29
Case 2: 1
```
## Hint

<p>数据范围</p><p>$1 \le n \le 200$</p>
