## Description

<p>在n个人中，某些人的银行账号之间可以互相转账。这些人之间转账的手续费各不相同。给定这些人之间转账时需要从转账金额里扣除百分之几的手续费，请问A最少需要多少钱使得转账后B收到100元。<p>

## Input Format

<p>第一行输入两个正整数n,m，分别表示总人数和可以互相转账的人的对数。<p><p>以下m行每行输入三个正整数x,y,z，表示标号为x的人和标号为y的人之间互相转账需要扣除z%的手续费 (z<100)。<p><p>最后一行输入两个正整数A,B。数据保证A与B之间可以直接或间接地转账。<p>

## Output Format

<p>输出A使得B到账100元最少需要的总费用。精确到小数点后8位。<p>

```input1
3 3
1 2 1
2 3 2
1 3 3
1 3

```
```output1
103.07153164

```
## Hint

【数据规模】
1≤n≤2000
