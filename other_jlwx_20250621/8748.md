## Description

<p>任何一个正整数都可以用2的幂次方表示。例如：<p><p>137=27+23+20<p><p>同时约定方次用括号来表示，即ab可表示为a(b)。由此可知，137可表示为：<p><p>2(7)+2(3)+2(0)<p><p>进一步：7=22+2+20（21用2表示）<p><p>3=2+20<p><p>所以最后137可表示为：<p><p>2(2(2)+2+2(0))+2(2+2(0))+2(0)<p><p>又如：<p><p>1315=210+28+25+2+1<p><p>所以1315最后可表示为：<p><p>2(2(2+2(0))+2)+2(2(2+2(0)))+2(2(2)+2(0))+2+2(0)<p>

## Input Format

<p>一个正整数n（n≤20000）。<p>

## Output Format

<p>一行，符合约定的n的0，2表示（在表示中不能有空格）。<p>

```input1
137

```
```output1
2(2(2)+2+2(0))+2(2+2(0))+2(0)

```
