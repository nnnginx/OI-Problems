## Description

儿童火车票购票规定：
1. 身高不足120 厘米的儿童，不需要购买火车票；
2. 身高为120～150 厘米的儿童，需购买儿童票；
3. 身高超过150厘米的儿童，需购买全价票。
现在请你设计一个程序，根据儿童的身高来判断购买什么票。

## Input Format

输入一个正整数h（50 <= h <= 180)，表示儿童的身高（单位：厘米）。

## Output Format

如果儿童身高不足120厘米，则输出 Free；
如果儿童身高为120～150厘米（含120和150厘米），则输出 Child；
如果儿童身高超过150厘米，则输出 Full。

```input1
120
```
```output1
Child
```
```input2
80
```
```output2
Free
```
