## Description

在墙角堆放着一堆完全相同的正方体小木块，如下图所示：
<img src="http://oj.czos.cn/upload/image/20171203/20171203140402_71121.jpg" alt="" /> 
因为木块堆得实在是太有规律了，你只要知道它的层数就可以计算所有木块的数量了。
 

## Input Format

 只有一个整数 n ，表示这堆小木块的层数，已知1 &lt;= n &lt;= 100 。 

## Output Format

 只有一个整数，表示这堆小木块的总数量。 

```input1
5
```
```output1
35
```
