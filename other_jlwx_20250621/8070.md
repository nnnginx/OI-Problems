## Description

<p>在一个2维平面上有两条传送带，每一条传送带可以看成是一条线段。两条传送带分别为线段AB和线段CD。lxhgww在AB上的移动速度为P，在CD上的移动速度为Q，在平面上的移动速度R。现在lxhgww想从A点走到D点，他想知道最少需要走多长时间。<br /></p>

## Input Format

<p>输入数据第一行是4个整数，表示A和B的坐标，分别为Ax，Ay，Bx，By</p><p>第二行是4个整数，表示C和D的坐标，分别为Cx，Cy，Dx，Dy</p><p>第三行是3个整数，分别是P，Q，R</p>

## Output Format

<p>输出数据为一行，表示lxhgww从A点走到D点的最短时间，保留到小数点后2位<br /></p>

```input1
0 0 0 100
100 0 100 100
2 2 1
```
```output1
136.60
```
## Hint

<p>【数据范围】</p><p>对于100%的数据，1≤ Ax，Ay，Bx，By，Cx，Cy，Dx，Dy≤1000</p><p>1≤P，Q，R≤10</p>
