## 题目描述

<div>
 对于一个正整数n，定义f(n)为它十进制下每一位数字的平方的和。现在给定三个正整数k,a,b，请求出满足a<=n<=b且k*f(n)=n的n的个数。
</div>
<div></div>
<p></p>

## 输入格式

<div>
 第一行包含三个正整数k,a,b(1<=k,a,b<=10^18,a<=b)。
</div>
<div></div>
<p></p>

## 输出格式

<div>
 输出一个整数，即满足条件的n的个数。
</div>
<div></div>
<p></p>

```input1
51 5000 10000
```
```output1
3
```
## 数据范围与约定

<p>满足的3个n分别为7293，7854和7905。</p><br>
<p></p>
