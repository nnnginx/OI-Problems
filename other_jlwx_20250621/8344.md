## Description

<p>在漆黑的夜里，N（N&lt;=1000）位旅行者来到了一座狭窄而且没有护栏的桥边。如果不借助手电筒的话，大家是无论如何也不敢过桥去的。不幸的是，N个人一共只带了一只手电筒，而桥窄得只够让两个人同时过。如果各自单独过桥的话，N人所需要的时间已知；而如果两人同时过桥，所需要的时间就是走得比较慢的那个人单独行动时所需的时间。问题是，如何设计一个方案，让这N人尽快过桥。</p>

## Input Format

<p>第一行是一个整数N(1&lt;=N&lt;=1000)表示共有N个人要过河<br /></p><p><span style="color: rgb(51, 51, 51);">第二行是N个整数Si,表示此人过河所需要花时间。(0&lt;Si&lt;=100)</span></p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">输出所有人都过河需要用的最少时间</span><br /></p>

```input1
4
1
2
5
10
```
```output1
17
```
