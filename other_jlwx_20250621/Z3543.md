## 题目描述

<p><span style="font-size: medium">给N个点，问存在多少个两边与坐标轴平行的正方形，四个顶点属于这N个点中的4个。</span></p>
<p></p>

## 输入格式

<p><span style="font-size: medium">第一行一个整数N。<br>
   接下来N行每行两个数x_i，y_i表示坐标。</span></p>
<p></p>

## 输出格式

<p><span style="font-size: medium">一行一个整数表示答案。</span></p>
<p></p>

```input1
6
0 0
0 1
1 0
1 1
3 0
3 1
```
```output1
1
```
## 数据范围与约定

<p><span style="font-size: medium">【数据范围】<br><br>
   N<=10^5，|x_i|,|y_i|<=10^6<br><br></span></p>
