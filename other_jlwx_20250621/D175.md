## Description

编写程序，根据给出的一个结束于'.'的字符字串，找出其中最长的含有字母'a'的子串。 

## Input Format

一行，为一个字符字串，结束于句点'.'。字串中的子串由一个或几个空格隔开。 

## Output Format

一行。显示找出的最长的含有字母'a'的子串。如果有多个这样的子串，只显示其中的第一个；若没有含字母'a'的子串，则显示'NO'。 

```input1
Her  name is Lilan   and she  is a  happy  student.
```
```output1
Lilan
```
## Source

字符串
