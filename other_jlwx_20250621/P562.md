## Description

小明最近对概率问题很感兴趣。一天，小明和小红一起玩一个概率游戏，首先小明给出一个字母和一个单词，然后由小红计算这个字母在这个单词中出现的概率。字母不区分大小写。<br />
例如，给定的字母是a，单词是apple，那么概率是0.20000。<br />

## Input Format

输入包含多组测试数据。每组数据包含一个字母和一个单词。单词的长度不超过200。

## Output Format

对于每一个输入，输出对应的概率，结果保留5位小数。

```input1
a apple
c Candy
a banana
```
```output1
0.20000
0.20000
0.50000
```
