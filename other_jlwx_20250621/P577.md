## Description

<p>
	输入一行字符串（字符串长度小于10000，由大小写字母、阿拉伯数字和空格组成），分别统计出其中大写英文字母，小写英文字母、阿拉伯数字和空格的个数。
</p>

## Input Format

<p>
	一行字符串（字符串长度小于10000，由大小写字母、阿拉伯数字和空格组成）
</p>

## Output Format

<p>
	分别输出大写英文字母，小写英文字母、阿拉伯数字和空格的个数，用空格隔开
</p>

```input1
h0u84nfx 7
```
```output1
0 5 4 1
```
