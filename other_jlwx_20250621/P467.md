## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">小Tu在做数据结构的作业，其中一题是给你一棵二叉树的前序遍历和中序遍历结果，要求你写出这棵二叉树的后序遍历结果。</span><br /></p>

## Input Format

<p>输入包含多组测试数据。每组输入包含两个字符串，分别表示二叉树的前序遍历和中序遍历结果。每个字符串由不重复的大写字母组成。</p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">对于每组输入，输出对应的二叉树的后续遍历结果。</span><br /></p>

```input1
DBACEGF ABCDEFG
BCAD CBAD
```
```output1
ACBFGED
CDAB
```
