## Description

<p>求关于$ x$的同余方程 $ a x \equiv 1 \pmod {b}$ 的最小正整数解。<br /></p>

## Input Format

<p>一行，包含两个正整数 $a,b$，用一个空格隔开。<br /></p>

## Output Format

<p>一个正整数 $x_0$，即最小正整数解。输入数据保证一定有解。<br /></p>

```input1
3 10
```
```output1
7
```
## Hint

<p>【数据范围】</p><p>对于 40%的数据，$2 ≤b≤ 1,000$；</p><p>对于 60%的数据，$2 ≤b≤ 50,000,000$；</p><p>对于 100%的数据，$2 ≤a, b≤ 2,000,000,000$。</p>
