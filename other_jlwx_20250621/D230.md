## Description

请从键盘读入一个整数n（n是1~10的范围内的奇数），打印出如下图所示的n行的完整的蝴蝶结！
比如，n=5，则打印图形如下：
<img src="http://oj.czos.cn/upload/image/20180107/20180107110835_42122.jpg" alt="" /> 
 
 
	 
 

## Input Format

一个整数n，代表图形的行数！

## Output Format

n行的图形！

```input1
5
```
```output1
*    *
**  **
******
**  **
*    *
```
