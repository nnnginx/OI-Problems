## Description

请从小到大输出1~n中所有个位为5或者个位为8的所有的整数，每行1个。
比如，假设  n=20  ，那么满足条件的数输出如下：    
5  
8  
15  
18  
 
 
	 
 

## Input Format

一个整数  n  。    
 

## Output Format

1~n  中所有满足条件的整数。    
 

```input1
25
```
```output1
5
8
15
18
25
```
```input2
30
```
```output2
5
8
15
18
25
28
```
```input3
20
```
```output3
5
8
15
18
```
