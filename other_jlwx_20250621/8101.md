## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">相比 wildleopard 的家，他的弟弟 mildleopard 比较穷。他的房子是狭窄的而且在他的房间里面仅有一个灯泡。每天晚上，他徘徊在自己狭小的房子里，思考如何赚更多的钱。有一天，他发现他的影子的长度随着他在灯泡和墙壁之间走到时发生着变化。一个突然的想法出现在脑海里，他想知道他的影子的最大长度。</span><br /></p><p><span style="color: rgba(0, 0, 0, 0.87);"><img src="https://i.loli.net/2018/07/03/5b3ac36a2ca4a.jpg" width="500" height="385.70710510766804" /><br /></span></p>

## Input Format

<p>输入文件的第一行包含一个整数T，表示测试数据的组数。</p><p>对于每组测试数据，仅一行，包含三个实数H，h和D，H表示灯泡的高度，h表示 mildleopard 的身高，D表示灯泡和墙的水平距离。</p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出文件T共</span><span style="color: rgba(0, 0, 0, 0.87);">行，每组数据占一行表示影子的最大长度，保留三位小数。</span><br /></p>

```input1
3
2 1 0.5
2 0.5 3
4 3 4
```
```output1
1.000
0.750
4.000
```
## Hint

<p>T&lt;=100  , 0.01&lt;=H,h,D&lt;=1000,  0.01&lt;=H-h</p>
