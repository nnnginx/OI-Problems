## Description

<p>在N*N的方格棋盘放置N个皇，使得它们不相互攻击（即任意2个皇后不允许处在同一行，或同一列，也不允许处在与棋盘边框成45角的斜线上。你的任务是，对于给定的N，求出有多少种符合要求放置方法。<br /></p><p><img alt="sss.PNG" src="/public/upload/fc9bcebf30.png" width="300" height="283.5164835164835" /><br /></p>

## Input Format

<p>输入中有一个正整数N≤20，表示棋盘和皇后的数量<br /></p>

## Output Format

<p>为一个正整数，表示N个皇后的不同放置方法数。<br /></p>

```input1
5
```
```output1
10
```
