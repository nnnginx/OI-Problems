## Description

很多小学生在学习加法时，发现“进位”特别容易出错。
你的任务是计算两个正整数在相加时有多少次进位。
你编制的程序应当可以连续处理多组数据，直到读到两个 0（这是输入结束标记）。

## Input Format

输入文件中，每行上有两个正整数 x,y，中间用一个空格隔开，最后一行是 0 0。

## Output Format

输出文件对于每行数据输出一行，每行中只有一个整数，表示 x,y 相加时有多少次进位。

```input1
123 4567
555 555
123 594
0 0
```
```output1
1
3
1
```
## Hint

【数据范围】
对于 50%的数据：0<x,y<=10000；
对于 100%的数据：0<x,y<=1000000000;
对于 100%的数据：n<=10000。
