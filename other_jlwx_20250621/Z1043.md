## 题目描述

<p><span style="font-size: medium">　　有n个圆盘从天而降，后面落下的可以盖住前面的。求最后形成的封闭区域的周长。看下面这副图, 所有的红<br>
   色线条的总长度即为所求. <img border="0" alt="" src="https://s2.loli.net/2023/08/14/NehF1iBj9V54vL7.png"> </span></p>

## 输入格式

<p>　　第一行为1个整数n,N<=1000<br>
  接下来n行每行3个实数,ri,xi,yi,表示下落时第i个圆盘的半径和圆心坐标.<br><br></p>

## 输出格式

<p><span style="font-size: medium">　　最后的周长，保留三位小数 </span></p>

```input1
2
1 0 0
1 1 0
```
```output1
10.472
```
