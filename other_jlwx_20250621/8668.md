## Description

<p><span style="color: rgb(51, 51, 51);">输入 n，从 $1，2，3，…，n $ 的数列中，找出最前面的总和为 n 的连续子数列。</span><br /></p>

## Input Format

<p><span style="color: rgb(51, 51, 51);">一行一个整数 n。</span><br /></p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">一行若干个数，表示和为 n 的连续子序列，每两个数之间有一个空格。</span><br /></p>

```input1
5
```
```output1
2 3
```
```input2
4
```
```output2
4
```
## Hint

<p>数据范围：$n \le 10^9$</p>
