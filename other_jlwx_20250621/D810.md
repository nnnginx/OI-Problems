## Description

 
从键盘读入n个整数（n&lt;=100），这n个数都是1~10之间的数，请从小到大输出每个出现过的数，以及统计出每个数出现的次数？

比如：假设读入5个数，分别为1 2 3 3 5，统计结果如下：1出现1次，2出现1次，3出现2次，5出现1次。

输出格式如下：
1 1
2 1
3 2
5 1
 

## Input Format

第1行输入一个整数n。
第2行输入n个整数，用空格隔开。
 
 

## Output Format

输出若干行，每行2个数，用空格隔开，按照从小到大的顺序，输出每个数以及每个数出现的次数。
 

```input1
5
1 2 3 3 5
```
```output1
1 1
2 1
3 2
5 1
```
