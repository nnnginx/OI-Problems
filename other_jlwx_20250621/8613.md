## Description

<p><span style="color: rgba(0, 0, 0, 0.75);">在 n 个人中，某些人的银行账号之间可以互相转账。这些人之间转账的手续费各不相 同。给定这些人之间转账时需要从转账金额里扣除百分之几的手续费，请问 A 最少需要多 少钱使得转账后 B 收到 100 元。</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.75);">第一行输入两个正整数 n,m，分别表示总人数和可以互相转账的人的对数。 以下 m 行每行输入三个正整数 x,y,z，表示标号为 x 的人和标号为 y 的人之间互相 转账需要扣除 z%的手续费 (z&lt;100)。 最后一行输入两个正整数 A,B。数据保证 A 与 B 之间可以直接或间接地转账。 1&lt;=n&lt;=2000</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.75);">输出 A 使得 B 到账 100 元最少需要的总费用。精确到小数点后 8 位。</span><br /></p>

```input1
3 3     
1 2 1     
2 3 2     
1 3 3     
1 3 
```
```output1
103.07153164 
```
