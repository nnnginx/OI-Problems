## 输入格式

<p>输入仅包含一行一个整数 N</p>

## 输出格式

<p>输出仅包含一行一个整数 f(N)。 3 < =  N < = 10^18</p>

```input1
2
```
```output1
35
提示
SN={1, 2, 11}，故 f(N)=1×2+1×11+2×11=35。
```
