## Description

<p>现在共有n个同学要排成一列，每个同学有两个属性:影响力和承受能力。给一个同学造成的心理创伤指数等于所有在他前面同学的影响力之和减去他的承受能力。</p><p>请安排一下排队顺序，尽量使受到心理创伤最大的同学少受创伤。</p>

## Input Format

<p>第1行是整数n,表示同学人数。</p><p>第2~n+1行，每行两个自然数，分别是该同学的影响力和承受能力。</p>

## Output Format

<p>输出1行1个整数，为你安排的顺序中受到心理创伤最大的同学受到的创伤。<br /></p>

```input1
3
10 3
2 5
3 3
```
```output1
2
```
## Hint

<p>对于100%的数据满足:$1&lt;=n&lt;=50000,1&lt;=影响力&lt;=10000，1&lt;=承受能力&lt;=10^9$。<br /></p>
