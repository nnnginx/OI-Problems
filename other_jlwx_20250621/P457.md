## Description

<p>小Tu在上一道题目中，把一个OJ版本号为483的各位数字相加(4+8+3)=15，再将483分解成质因数相乘：483=3×7×23，把这些质因数各位数字相加(3+7+2+3)，其和也为15，我们将这种情况称为“正确的OJ版本”，即某版本号（一个合数）的各位数字之和等于它所有质因数的各数字之和。你帮助小Tu把n以内（包括n）具有上述特点的所有正确的OJ版本找出来吗？</p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">一个整数n (10≤n≤30000)。</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">若干行，每行输出5个满足题目条件的数。</span><br /></p>

```input1
100
```
```output1
4 22 27 58 85 
94
```
