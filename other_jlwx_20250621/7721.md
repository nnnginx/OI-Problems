## Description

给定 +、-、* 中任意一个运算符，以及两个整数N、M，根据运算符计算N加M、N减M或N乘M的结果。

## Input Format

输入共2行；
第1行输入一个运算符（+、-、* 中的任意一个）；
第2行输入两个整数N、M，代表参与计算的两个数。两数中间有一个空格。

## Output Format

输出一个整数，表示计算的结果。
注意：
1. 题目保证输入和输出的整数都在int范围内；
2. 输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
+ 
2 3
```
```output1
5
```
