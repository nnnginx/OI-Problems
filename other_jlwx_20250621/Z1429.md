## 题目描述

<div>
 令F(x)=Sigma(i),1<=i<=x. 
</div>
<div>
 有一个不定方程f(x)+f(y)+f(z)+f(w)=N(0<=N<=10^12,x,y,z,w为自然数) 
</div>
<div>
 请统计方程的个数
</div>

## 输入格式

<p>一个正数N</p>

## 输出格式

<p>一个数表示解的个数</p>

```input1
input 1
2
input 2
4
```
```output1
output 1
6
output 2
13
```
