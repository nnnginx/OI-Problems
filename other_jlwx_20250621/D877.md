## Description

给定一个长度为n的字符串S，还有一个数字L，统计长度大于等于L的出现次数最多的子串（不同的出现可以相交），如果有多个，输出最长的，如果仍然有多个，输出第一次出现最早的。

## Input Format

第一行一个数字L。 
第二行是字符串S。 
L大于0，且不超过S的长度。 

## Output Format

一行，题目要求的字符串。

```input1
输入样例1：
4
bbaabbaaaaa

输入样例2：
2
bbaabbaaaaa
```
```output1
输出样例1：
bbaa

输出样例2：
aa

```
## Hint

【数据规模】
L&lt;=60 
S中所有字符都是小写英文字母。 
提示：枚举所有可能的子串，统计出现次数，找出符合条件的那个

【来源】蓝桥杯算法训练
