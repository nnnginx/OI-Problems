## Description

小蓝 是一个很会利用时间的学生，她经常晚上吃过饭后一边思考问题一边散步，由于要思考问题，并且思考问题的内容越来越多，所以她在散步时也是越来越慢的，她自己总结出一个规律，她散步时第一步走N厘米，第二步走N-1厘米，以后每步比前一步少1厘米，最后一步是0时就停下来了。现在给出你 小蓝 第一步走的长度N厘米，请你算出 小蓝 这次散步一共起了多少厘米。如 小蓝 第一步走3厘米，则第二步走2厘米，第三步走1厘米，然后就停下来了，则 小蓝 这次散步共走了3+2+1=6（厘米）。

## Input Format

输入一个整数N，代表第一步走的厘米数

## Output Format

输出一个整数，代表总共走的厘米数

```input1
3
```
```output1
6
```
## Hint

【数据规模】
70%的数据，N<=250
100%的数据，N<=30000
