## Description

<br />　　银行近期推出了一款新的理财计划“重复计息储蓄”。储户只需在每个月月初存入固定金额的现金，银行就会在每个月月底根据储户账户内的金额算出该月的利息并将利息存入用户账号。现在如果某人每月存入k元，请你帮他计算一下，n月后，他可以获得多少收益。

## Input Format

　　输入数据仅一行，包括两个整数k(100<=k<=10000)、n(1<=n<=48)和一个小数p(0.001<=p<=0.01)，分别表示每月存入的金额、存款时长、存款利息。

## Output Format

输出数据仅一个数，表示可以得到的收益。

```input1
1000 6 0.01
```
```output1
213.53
```
