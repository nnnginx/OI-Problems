## Description

有一个六位数，是左右对称图形，对称图形的意思是，这个数正过来和倒过来是一样的，比如 123321 、 506605 等数都是六位的对称数。请从键盘读入一个三位数，作为这个六位数的后三位，求出该六位数是多少？  
 
比如：输入321作为这个六位数的后三位，那么前三位一定是123，这个六位数就是123321。
 

## Input Format

一个三位数（这个三位数的个位一定不为0），作为六位数的百位、十位和个位。  
 

## Output Format

输出计算出的满足条件的六位数。  
 

```input1
321
```
```output1
123321
```
