## Description

粮食是生活的平安保障，是幸福的营养，是快乐的乐章，是发展的后盾。
生活中也经常见到用诗句“谁知盘中餐，粒粒皆辛苦”来宣传节约粮食的重要性。
小蓝同学为了记住这句诗词，决定用编程来输出这句诗词中每个字汉语拼音的首字母。

## Input Format

注意：本题没有输入！ 

## Output Format

输出仅一行，输出“szpzclljxk”。
注意：
1. 程序区分大小写，本题输出全部为小写，输出中有大写字母不得分， 
2. 输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

```input1
无
```
