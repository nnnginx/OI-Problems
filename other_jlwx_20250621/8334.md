## Description

<p>小毛准备了M磅的猫咪食物去和猫咪交换他最喜欢的食物――爪哇豆。</p><p>猫咪有N间仓库，其中第i间仓库包含着S[i]磅的爪哇豆，但是要花费F[i]磅的猫咪食物去和他们交换。</p><p>小毛很聪明，经过他的各种交涉，发现自己没有必要把每一个仓库的食物全部全部买下，他可以偷偷地买下一部分。也就是说，他可以获得S[i]*a%磅的爪哇豆而只花费F[i]*a%磅的猫咪食物。</p><p>但是，他不知道该怎么买才能买到最多的爪哇豆，所以请编程来帮他计算一下。</p>

## Input Format

<p>第1行2个整数M和N，中间用一个空格隔开。</p><p>接下来的N行每行两个非负数S[i]和F[i]，中间用一个空格隔开。</p><p>其中：所有整数不超过1000。</p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">一行一个实数，精确到小数点后三位，表示小毛最多能买到多少磅的爪哇豆。</span><br /></p>

```input1
5 3
7 2
4 3
5 2
```
```output1
13.333
```
