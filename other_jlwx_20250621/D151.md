## Description

给出N(5 &lt;= N &lt;= 150)个人的语文成绩，求N个人的语文总分和平均分，并按成绩高低排序后输出。 

## Input Format

第1行：一个整数N。 第2行：空格隔开的N个整数，表示N个人的语文成绩。 

## Output Format

输出有三行。
第1行：一个整数，为N个人的总分。
第2行：N个人的语文平均分，保留两位小数。
第3行：N个空格隔开的整数，为从高到低输出的N个人的成绩。
 

```input1
5
72 98 95 81 86
```
```output1
432
86.40
98 95 86 81 72
```
