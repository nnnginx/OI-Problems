## Description

<p><span style="color: rgba(0, 0, 0, 0.87);">在所有的N位数中，有多少个数中有偶数个数字3？由于结果可能很大，你只需要输出这个答案对12345取余的值。</span><br /></p>

## Input Format

<p><span style="color: rgba(0, 0, 0, 0.87);">读入一个数N</span><br /></p>

## Output Format

<p><span style="color: rgba(0, 0, 0, 0.87);">输出有多少个数中有偶数个数字3。</span><br /></p>

```input1
2
```
```output1
73
```
## Hint

<p>【数据规模】</p><p>1&lt;=N&lt;=1000</p><p><br /></p><p>【样例说明】</p><p>在所有的2位数字，包含0个3的数有72个，包含2个3的数有1个，共73个
