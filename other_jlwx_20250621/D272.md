## Description

期末考试结束，小明的语文老师想知道，这次考试的成绩分布情况，主要计算如下几个数据：平均分、 &gt;= 平均分的总人数、 &lt; 平均分的总人数，请你写程序帮助小明的语文老师来计算一下！  
 

## Input Format

第一行，一个整数 n 代表有 n 个人的成绩（ n&lt;=100 ）  
第二行， n 个人的语文成绩  
 

## Output Format

3 个值，分别代表平均分、 &gt;= 平均分的总人数、 &lt; 平均分的总人数，请注意，平均分保留 1 位小数！  
 

```input1
5
100 98 97 99 90
```
```output1
96.8 4 1
```
