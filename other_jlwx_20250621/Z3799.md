## 题目描述

<div>
 给出一个字符串S,现希望对它进行重新组合得到
</div>
<div>
 一个字符串，其比T大且是字典序最小的。
</div>
<div></div>
<p></p>

## 输入格式

<div>
 输入第一行为S，第二行为T
</div>
<div></div>
<p></p>

## 输出格式

<div>
 输出重组后的结果，如果不存在输出-1
</div>
<div></div>
<p></p>

```input1
abad
bob
```
```output1
daab
```
## 数据范围与约定

<p>字符串长度<=5000</p>
<br>
<p></p>
