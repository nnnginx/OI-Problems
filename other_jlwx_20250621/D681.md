## Description

请从小到大输出  m~n  之间所有的  3  位的回文数，并求出这些数有多少个。   
 

## Input Format

两个整数  m  和  n  用空格隔开（  m&lt;=n  ，且  m  和  n  一定都是  3  位数）。   
 

## Output Format

先输出  m~n  之间所有的  3  位的回文数，每行  1  个；   
最后  1  行输出  3  位回文数总共有多少个；   
 

```input1
100 200
```
```output1
101
111
121
131
141
151
161
171
181
191
10
```
