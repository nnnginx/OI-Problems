## Description

李晓芳用一条长为 n 米的彩纸制作花边，每朵花的宽度为 x 厘米，花与花之间的间隔为 y 厘米。请问 n 米的彩纸最多能做多少朵花的花边。   

如，图中的案例花的宽度为 4.5cm ，花之间的间隔为 1.2cm ，那么一条长为 0.62m 的彩纸最多可以做 11 朵花的花边（因为一朵花加一个间隔是 5.7cm ， 10 朵花以及间隔共 57cm ，剩余 5cm 还可以再做一朵花，因此最多共可以做 11 朵花）   
<img src="http://oj.czos.cn/upload/image/20180623/20180623101149_54903.jpg" alt="" /> 
 

## Input Format

三个小数n、x、y，用空格隔开

## Output Format

最多能做多少朵花的花边

```input1
0.62 4.5 1.2
```
```output1
11
```
