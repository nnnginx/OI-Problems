## Description

把一个长为len的字符串围成一个圈，然后以任意一个字符作为起点，都会产生一个长为len的字符串，字符串的最小表示就是所有字符串中字典序最小的那个。
例如字符串alabala，将它围成一个圈后，根据上面的规则会形成以下新的字符串：
labalaa
abalaal
balaala
alaalab
laalaba
aalabal
在这所有7个字符串中，字典序最小的是aalabal，它的第一个字母在原字符串中的位置是6。（位置从0开始算）<br />
现在给定你一个字符串，请你找出其最小表示的第一个字母在原字符串中的位置。如果字符串最小表示有多个，那么输出第一个字母在原字符串中位置最小的。<br />
<div>
	<br />
</div>

## Input Format

输入的第一行是一个整数t，表示有t组测试数据。<br />
接下来t行，每行先输入一个整数l（5&lt;=l&lt;=100000），表示原字符串的长度，然后输入一个字符串，表示原字符串。字符串中只包含小写字母。

## Output Format

对于每组输入，输出原字符串最小表示的第一个字母在原字符串中的位置。

```input1
2
6 baabaa
7 alabala
```
```output1
1
6
```
