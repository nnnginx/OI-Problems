## Description

小明毕业一年了，并且找到了一份好工作。这一年里他赚了很多钱，现在他想知道他这一年里的平均月薪是多少，请你写一个程序帮他计算。</span><br /></p>

## Input Format

输入包括12行。第i行为第i个月的实际月薪。（i=1,2,3...）</span><br /></p>

## Output Format

输出小明的平均月薪，保留两位小数。

```input1
100.00 
489.12 
12454.12 
1234.10 
823.05 
109.20 
5.27 
1542.25 
839.18 
83.99 
1295.01 
1.75
```
```output1
1581.42
```
