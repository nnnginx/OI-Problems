## Description

<p><span style="color: rgb(51, 51, 51);">给出两个整数 a 和 b，每次操作可以 a+1 或 a*2，问至少进行多少次操作可以使得 a=b。</span><br /></p>

## Input Format

<p><span style="color: rgb(51, 51, 51);">一行两个整数 a 和 b，之间有一个空格。</span><br /></p>

## Output Format

<p><span style="color: rgb(51, 51, 51);">一行一个整数 ans，表示至少进行多少次操作可以使得 a=b。</span><br /></p>

```input1
2 10
```
```output1
3
```
## Hint

<p>对于30%的数据，$1 \le a \le b \le 100$</p><p><span style="color: rgb(51, 51, 51);">对于60%的数据，<span style="color: rgb(51, 51, 51);">$1 \le a \le b \le 10^6$</span></span><br /></p><p><span style="color: rgb(51, 51, 51);">对于100%的数据，<span style="color: rgb(51, 51, 51);">$1 \le a \le b \le 10^9$</span></span></p>
