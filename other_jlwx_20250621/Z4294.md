## 题目描述

<p>众所周知，斐波那契数列F满足：<br>
  F[0]=0,F[1]=1,F[m]=F[m-1]+F[m-2](m>=2)<br>
  现在给出一个数字串S，请找到一个k使得F[k]以S为结尾。</p>

## 输入格式

<p>包含一行一个数字串S，S的长度不超过18。</p>

## 输出格式

<p>输出满足条件的数字k，且k需要满足0<=k<10^100。若有多组解，输出任意一组，若无解，输出NIE。</p>

```input1
025
```
```output1
1525
```
## 数据范围与约定

<p>请不要提交，尚无SPJ</p>
