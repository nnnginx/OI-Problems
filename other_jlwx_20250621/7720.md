## Description

小蓝刚学完了东南西北4个方向的英文单词，为了加强记忆，想编写一个程序。当输入某个方向的英文单词的首字母，程序就输出对应的完整英文单词。例如：
输入字母 E 或者e ，则输出 East；
输入字母 S 或者 s ，则输出 South；
输入字母 W 或者 w，则输出 West；
输入字母 N 或者 n ，则输出 North；
输入其他字母，则输出 Wrong。
请你帮助小蓝设计这个程序。

## Input Format

输入一个字母，表示某个方向的英文单词首字母。

## Output Format

按照题目描述，输出对应方向的完整英文单词（注意：单词的首字母大写）。

```input1
S
```
```output1
South
```
