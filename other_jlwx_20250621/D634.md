## Description

班级选拔学习委员，老师定了一个标准是：语文、数学、英语三门功课的平均分要在90或者90以上，为了防止有同学偏科严重，老师又加了一个标准：三门功课不允许有在86分以下的成绩。
 
请你编写一个程序，读入一位同学三门功课的成绩，判断一下他能否进入学习委员选拔的待选名单，如果能，请输出"good"，如果不能请输出"sorry"。
 
比如：小丽同学三门功课分别考了98 95 88，那么她就达到了入选标准；小华同学三门功课分别考了85 99 98，那么他就达不到入选标准。
 

## Input Format

三个整数，代表某位同学三门功课的成绩。

## Output Format

能入选请输出"good"，不能入选请输出"sorry"

```input1
98 95 88
```
```output1
good
```
