# 题目描述

小柒同学学习了“进制转换”的相关知识点后，知道了各进制之间可以互相转换的，她现在很好奇的是如何把十进制的正整数数字转为十六进制数字呢？

请编写程序完成作答，要求输出十六进制数字时要加上前缀“0x”。
![image](./3124/file/3uA_SQfw_gyrj89cUSYpM.jpeg)
## 输入

第一行，输入一个正整数 n ,表示十进制数字

## 输出

输出十进制数转为十六进制数且以“0x”为前缀的结果。

# 样例

```input1
29
```

```output1
0x1D
```

# Limitation

对于 100% 的数据： 0<n<10000
