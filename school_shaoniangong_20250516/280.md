## 题目描述

有$\red {N}$个变量$\red {X_0\sim X _{N−1}~}$ ，每个变量的可能取值为$\red {0}$或$\red {1}$。

给定$\red {M}$个算式，每个算式形如 $\red {X_aopX _b =c}$，其中 $\red {a,b}$ 是变量编号，$\red {c}$ 是数字$\red {0}$或$\red {1}$，$\red {op}$ 是 $\red {and，or，xor}$ 三个位运算之一。

求是否存在对每个变量的合法赋值，使所有算式都成立。

## 输入格式

第一行包含两个整数$\red {N}$和$\red {M}$。

接下来$\red {M}$行，每行包含三个整数$\red {a ~b ~c}$，以及一个位运算（$\red {AND,OR,XOR}$中的一个）。

## 输出格式

输出结果，如果存在，输出`“YES”`，否则输出`“NO”`

## 样例

#### 输入样例

```c
4 4
0 1 1 AND
1 2 1 OR
3 2 0 AND
3 0 0 XOR
```

#### 输出样例

```c
YES
```

## 提示

$\red {1≤N≤1000}$,

$\red {1≤M≤10^6}$
