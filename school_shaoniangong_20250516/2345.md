## 题目描述

有一辆火车上面有$\red{n }$个连续的座位（某些座位上可能已经有人了），需要安排 $\red{a}$个学生和 $\red{b}$个老师，

现在要让你安排座位，满足学生和学生不相邻，老师和老师不相邻。

并输出最多能安排多少个人（不超过 $\red{a+b）}$。

## 输入格式

第一行三个整数 $\red{n,a,b，}$座位数量，学生人数，老师人数。

第二行为一个仅包含$\red{‘*’}$ 和 $\red{‘.’}$长度为$\red{n}$的字符串，

$\red{‘.’}$表示当前座位空，$\red{‘*’}$ 表示座位上已有人。

## 输出格式

一个整数，最多安排多少人。

## 样例

#### 输入样例1

```c
6 2 3 

*...*.
```

#### 输出样例1

```c
4
```

#### 输入样例2

```c
3 2 3 

***
```

#### 输出样例2

```c
0
```

## 提示

对于$\red{100\% }$数据，

满足$\red{1<n<2×10^5,   0<a，}$$\red{b<2×10^5，}$$\red{a+b>0}$
