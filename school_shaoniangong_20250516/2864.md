## 题目描述

超市里，有$\red{n}$种物品，第$\red{i}$种物品的重量为$\red{i，}$且每种物品都有无限多个。

有一天，超市举办了一个活动。活动的规则是：首先你从盒子里抽取一个数字 $\red{x，}$
你有一个空篮子，初始时篮子内物品总重量为 $\red{0}$。你每次可以将一个物品放入篮
子，但假设当前篮子内物品总重量为$\red{w，}$则只能选择重量$\red{≥}$ $\red{w}$的物品放进篮子。

你可以按照规则放多次，但不能把已经在篮子里的物品取出篮子。如果你可以按
照规则，使得篮子内物品的总重量刚好为$\red{x，}$则超市就赠送你一个小礼品。

牛牛发现了这个活动的一个必胜策略，他参加了很多次活动，导致超市面临亏损。

于是在牛牛打算再次参与活动时，超市老板对牛牛提出挑战，如果牛牛可以不使
用第$\red{y}$种物品的前提下，仍然使篮子内物品的总重量刚好为$\red{x，}$则超市老板就赠送
牛牛一份大礼品，否则牛牛将再也不许参加活动。

牛牛欣然接受，他只是觉得无聊。于是他打算考考你，在不使用第$\red{y}$种物品的前
提下，他可以有多少种合法的放物品方案，使得篮子内物品的总重量刚好为$\red{x}$？

两种方案不同当且仅当牛牛在两种方案中篮子内物品的数量不同，或者两种方案
中第$\red{k}$次放到篮子的物品种类不同。

由于方案数可能很大，他只需要你输出方案数除以$\red{998244353}$的余数就可以啦。

## 输入格式

第一行，三个正整数$\red{n, x, y ，}$以空格相隔。

## 输出格式

一行，一个整数，表示方案数除以 $\red{998244353 }$的余数。

## 样例

#### 输入样例1

```c
5 7 3
```

#### 输出样例1

```c
3
```

#### 输入样例2

```c
8 15 4
```

#### 输出样例2

```c
4
```

#### 输入样例3

```c
12345 4321 24
```

#### 输出样例3

```c
795333476
```

## 提示

样例 $\red{1 }$说明

$\red{3 }$种方案分别是：

$\red{(1) }$放入 $\red{1, }$放入 $\red{1, }$放入 $\red{5}$

$\red{(2) }$放入 $\red{1, }$放入 $\red{2, }$放入 $\red{4}$

$\red{(3) }$放入 $\red{2, }$放入 $\red{5}$。

数据范围

对于 $\red{10\%}$数据，满足$\red{1 ≤}$ $\red{n, x ≤}$ $\red{10}$。

对于 $\red{40\%}$数据，满足$\red{1 ≤}$ $\red{n, x ≤}$ $\red{1000}$。

对于 $\red{100\%}$数据，满足$\red{1 ≤}$ $\red{n, x ≤}$ $\red{100000,1 ≤}$ $\red{y ≤}$ $\red{n}$。
