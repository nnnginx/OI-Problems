## 题目描述

甚至连五年级的教育都没有，奶牛们在教科书上的一个分数问题上遇到了麻烦。请帮助他们。

问题很简单：给定一个适当缩减的分数（即分子和分母的最大公约数是$\red{1，}$因此分数不能进一步缩减）找到分子和分母在 $\red{1..32,767 }$范围内的最小适当缩减分数最接近（但不等于）给定分数。

找一个它最接近的一个名字。你要找的分数的值的范围在$\red{1..32767 }$

## 输入格式

第$\red{1 }$行：两个以空格分隔的正整数 $\red{N }$和 $\red{D (1 <= N < D <= 32,767)，}$分别是给定分数的分子和分母

## 输出格式

第$\red{1 }$行：两个以空格分隔的整数，分别是与输入分数不同的最小、最接近分数的分子和分母。

## 样例

#### 输入样例

```c
2 3
```

#### 输出样例

```c
21845 32767
```

## 提示

输出详细信息：$\red{\frac{21845}{32767} = .666676839503.... \sim }$ $\red{0.666666.... = \frac23.}$
