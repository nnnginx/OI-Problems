## 题目描述

探险队员们跟随两位护法来到了七色虹前。七色虹，就是平面直角坐标系中赤橙黄绿青
蓝紫七个半圆，第 $\red{i }$座$\red{(1<=i<=7)}$半圆形彩虹的圆心是$\red{(x_i,0)，}$半径是 $\red{r_i，}$半圆上所有点的纵坐
标均为非负数。探险队员可以看做一条竖直的、长度等于身高的线段，线段的底端纵坐标为
$\red{0，}$最高的一位探险队员的身高为 $\red{h}$。
现在探险队员们要从$\red{(0,0)}$穿越七色虹到达$\red{(x_0,0)，}$穿越七色虹的过程中，探险队员的整
个身体必须始终在至少一个半圆形彩虹的内部。由于彩虹的半径 $\red{r_i }$可能太小了，不足以满
足这个条件，因此两位护法决定帮助他们把所有彩虹的半径都增大一个非负实数 $\red{r}$。探险队
员们想知道，$\red{r }$最小是多少呢？

## 输入格式

第一行两个实数 $\red{h}$、$\red{x_0，}$表示身高和目的地横坐标。
接下来七行每行两个实数 $\red{x_i}$、$\red{r_i，}$表示七座半圆形彩虹的圆心和半径。

## 输出格式

输出最小的 $\red{r，}$四舍五入保留 $\red{2 }$位小数。

## 样例

#### 输入样例

```c
4.0 36.0
0.0 4.0
6.0 4.0
12.0 4.0
18.0 4.0
24.0 4.0
30.0 4.0
36.0 4.0
```

#### 输出样例

```c
1.00
```

## 提示

对于 $\red{100\% }$的数据，满足 $\red{0<=x_i,x_0<=10000，}$$\red{0<h<100}$。
