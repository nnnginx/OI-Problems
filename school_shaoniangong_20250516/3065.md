## 题目描述

在Temege，小Z喜欢和小朋友们一起玩游戏，今天，他有想到了一个好玩的游戏。
游戏规则：每次点击一个小朋友，他和他的周围的小朋友都会改变状态（蹲下的变成了站起来的，站起来的变成了蹲下的）
我们将这个抽象成如下图所示的$\red{1*N}$的图。对于一个单元格,黑色表示小朋友是站起来的，反之，蹲下的小朋友是白色。Source表示初始状态,Target表示目标状态。
![image](./3065/file/epfzHEfcW4GnMw81ZckFT.png)
现在小Z有点偷懒,希望玩游戏的你算出初始状态到目标状态的最少点击数。

## 输入格式

第一行为N表示小朋友的个数
第二行是初始状态,有$\red{N}$个数，每个数不是$\red{0}$就是$\red{1}$($\red{0}$表示小朋友是蹲下的,$\red{1}$表示小朋友是站起来的)
第三行的结构跟第二行类似，表示目标状态

## 输出格式

一个数$\red{X}$,表示初始状态到目标状态的最少点击数。
如果无法到达目标,则请输出$\red{"Boring"}$

## 样例

#### 输入数据

```c++
9
0 1 0 0 0 1 0 0 0
1 0 1 0 1 0 1 0 0
```

#### 输出数据

```c++
2
```

## 提示

数据范围与提示

对于 $\red{100\%}$ 的数据，$\red{N<=10}$
