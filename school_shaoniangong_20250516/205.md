## 题目描述

农夫约翰雇佣他的$\red N$头奶牛帮他进行牛棚的清理工作。

他将全天分为了很多个班次，其中第$\red M$个班次到第$\red E$个班次（包括这两个班次）之间必须都有牛进行清理。

这$\red N$头牛中，第 $\red i$ 头牛可以从第$\red {a_i}$个班次工作到第$\red {b_i}$个班次，同时，它会索取$\red {c_i}$的佣金。

请你安排一个合理的清理班次，使得$\red {[M,E]}$时间段内都有奶牛在清理，并且所需支付给奶牛的报酬最少。

## 输入格式

第$\red 1$行：包含三个整数$\red N$，$\red M$和$\red E$。

第$\red {2..N+1}$行：第$\red {i+1}$行包含三个整数$\red {a_i}$ ,$\red {b_i}$ ,$\red {c_i}$ 。

## 输出格式

输出一个整数，表示所需的最少佣金。

如果无法做到在要求时间段内都有奶牛清理，则输出$\red{-1}$。

## 样例

#### 输入样例

```c
3 0 4
0 2 3
3 4 2
0 0 1
```

#### 输出样例

```c
5
```

## 提示

$\red {1≤N≤10000,}$

$\red {0≤M,E≤86399,}$

$\red {M≤a_i ≤b_i ≤E,}$

$\red {0≤c_i≤500000}$
