## 题目描述

给定一个字符串，输出所有长度至少为$\red{2}$的回文子串。

回文子串即从左往右输出和从右往左输出结果是一样的字符串，

比如：`abba`，`cccdeedccc`都是回文字符串。

## 输入格式

一个字符串，由字母或数字组成。长度$\red{500}$以内。

## 输出格式

输出所有的回文子串，每个子串一行。

子串长度小的优先输出，若长度相等，则出现位置靠左的优先输出。

## 样例

#### 输入样例

```C
123321125775165561
```

#### 输出样例

```C
33
11
77
55
2332
2112
5775
6556
123321
165561
```
