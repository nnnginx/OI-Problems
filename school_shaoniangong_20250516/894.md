## 题目描述

==2008==年北京奥运会，$\red{A}$国的运动员参与了$\red{n}$天的决赛项目$\red{(1≤n≤100)}$。

现在要统计一下$\red{A}$国所获得的金、银、铜牌数目及总奖牌数。

输入`第1行`是$\red{A}$国参与决赛项目的天数$\red{n}$，

其后`n行`，每一行是该国某一天获得的`金、银、铜`牌数目。

输出$\red{4}$个整数，为$\red{A}$国所获得的`金、银、铜`牌总数及总奖牌数。

## 输入格式

输入`n＋1行`，

`第1行`是$\red{A}$国参与决赛项目的天数$\red{n}$，

其后`n行`，每一行是该国某一天获得的金、银、铜牌数目，以一个空格分开。

## 输出格式

输出`1行`，包括$\red{4}$个整数，为$\red{A}$国所获得的金、银、铜牌总数及总奖牌数，以一个空格分开。

## 样例

#### 输入数据

```C
3
1 0 3
3 1 0
0 3 0
```

#### 输出数据

```C
4 4 3 11
```
