## 题目描述

平面上共有$\red {2\times N}$个点，$\red {N}$个是白点，$\red {N}$个是黑点。

对于每个白点，找到一个黑点，把二者用线连起来，要求最后所有线段都不想交，求一种方案。

## 输入格式

第一行包含整数$\red {N}$。

接下来$\red {N}$行，每行两个整数，表示一个黑点的坐标。

再接下来$\red {N}$行，每行两个整数，表示一个白点的坐标。

## 输出格式

输出共$\red {N}$行，每行一个整数。

第 $\red {i}$ 行的数，表示第 $\red {i}$ 个黑点连接的白点的编号，编号从$\red {1}$开始。

注意答案可能不唯一，任意输出一种答案即可。

## 样例

#### 输入样例

```c
5
-42 58
44 86
7 28
99 34
-13 -59
-47 -44
86 74
68 -75
-68 60
99 -60
```

#### 输出样例

```c
4
2
1
5
3
```

## 提示

$\red {1≤N≤100}$,坐标绝对值不超过$\red {10000}$。
