## 题目描述

`Pell`数列$\red{a_1, a_2, a_3, ...}$的定义是这样的，$\red{a_1 = 1, a_2 = 2, ... , a_n = 2 * a_{n − 1} + a_{n - 2} (n > 2)}$。
给出一个正整数$\red{k}$，要求`Pell`数列的第k项模上$\red{32767}$是多少。

## 输入格式

第1行是测试数据的组数$\red{n}$，后面跟着`n行`输入。每组测试数据占`1行`，包括一个正整数$\red{k}$ ($\red{1 ≤ k < 1000000}$)。

## 输出格式

`n行`，每行输出对应一个输入。输出应是一个非负整数。

## 样例

#### 输入样例

```C
2
1
8
```

#### 输出样例

```C
1
408
```
