## 题目描述

一个二进制数，将其每一位取反，称之为这个数的反码。下面我们定义一个字符的反码。

如果这是一个小写字符，则它和字符"$\red{a}$"的距离与它的反码和字符"$\red{z}$" 的距离相同；

如果是一个大写字符，则它和字符"$\red{A}$"的距离与它的反码和字符"$\red{Z}$"的距离相同；

如果不是上面两种情况，它的反码就是它自身。

举几个例子，

- "$\red{a}$"的反码是"$\red{z}$"；
- "$\red{c}$"的反码是"$\red{x}$"；
- "$\red{W}$"的反码是"$\red{D}$"；
- "$\red{1}$"的反码还是"$\red{1}$"；
- "` `"的反码还是"` `"
  一个字符串的反码定义为其所有字符的反码。我们的任务就是计算出给定字符串的反码。

## 输入格式

输入每行都是一个字符串，字符串长度不超过 $\red{80}$ 个字符。如果输入只有!，表示输入结束，不需要处理。

## 输出格式

对于输入的每个字符串，输出其反码，每个数据占一行。

## 样例

#### 输入样例

```C
Hello 
JLU-CCST-2011 
!
```

#### 输出样例

```C
Svool 
QOF-XXHG-2011
```
