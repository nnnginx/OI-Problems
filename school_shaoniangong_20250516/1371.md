
## 题目描述
大面面是广州某中学的学生， 由于成绩优异办事能力强，获得了老师的青睐，成为了老师的左膀右臂。

经历过期末考试后，数学老师想要统计该班排名靠后的学生的分数，于是把这个艰巨的任务交给了大面面，面面一筹莫展，聪明的你能够帮助面面解决这个“`世纪难题`”吗？

## 输入格式
共两行
第一行为一个正整数$\red n$，表示该班的学生总数

第二行为$\red n$个正整数，$\red{a_i}$表示第$\red i$位学生本次期末考的分数，用空格间隔


## 输出格式
共一行，表示本次学生的数学成绩，从低分往高分输出，每个分数间以空格隔开。

## 输入样例
```C
5
80 90 60 70 100
```
## 输出样例
```C
60 70 80 90 100
```
## 提示
对于所有的数据
$\red{10<=n<=100}$,

学生成绩$\red {0<=a_i<=100}$
