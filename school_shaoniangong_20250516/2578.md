## 题目描述

对于两个位数相等的二进制串$\red{A=(a_1,a_2,}$…,$\red{a_n ),B=(b_1,b_2,}$…,$\red{b_n)}$

它们之间的不相似度由以下函数定义：

$\red{DIF(A,B)=\sum_{i=1}^{n}{[a_i≠b_i ]} }$

注：$\red{[m=n]}$表示：如果$\red{m=n}$成立，值为$\red{1，}$否则值为$\red{0}$

如$\red{A=11001,B=01011，}$则$\red{DIF(A,B)=2}$。

给定$\red{n}$个长度为$\red{L}$的二进制串$\red{S_1,S_2,}$…,$\red{S_n}$

你的任务是找出长度为$\red{L}$的二进制串$\red{T，}$使得
$\red{min\{DIF(S_1,T),DIF(S_2,T),…,DIF(S_n,T)\}}$最大

满足条件的$\red{T}$可能存在多个，你需要输出满足条件的$\red{T}$的个数和这个最大值。

## 输入格式

第一行两个整数$\red{n,L(1≤}$$\red{N≤}$$\red{10^5,1≤}$$\red{L≤}$$\red{20)}$。

然后$\red{n}$行，每行一个长度为$\red{L}$的字符串$\red{S_i}$

## 输出格式

输出一行，包含两个整数，满足条件的$\red{T}$的个数和$\red{min\{DIF(S_1,T),DIF(S_2,T),…,DIF(S_n,T)\}}$的最大值，用空格隔开。

## 样例

#### 输入样例

```c
3 5
01001
11100
10111
```

#### 输出样例

```c
1 3
```

## 提示

只有当$\red{T = 00010}$时，$\red{min\{DIF(S_1,T),DIF(S_2,T),…,DIF(S_n,T)\}}$达到最大，为$\red{3}$。
