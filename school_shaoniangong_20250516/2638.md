

# Description

有一个铁人三项运动员，他的弱点是就是游泳，正好他家旁边有 n 条流速不同的河流，他想锻炼 m 天。

在这 m 天中，他每天都有不同的体力，他只能在流速小于他体力的河流里游，不然他就会被冲到太平洋（？！！！！！！）。

他想知道每天他能在多少条河里游泳。

# Format

## Input

一行，两个整数 n 和 m。1 ≤ n，m ≤ 10^5**1**0**5**（竟然有这么多条河？！！！！！）（他要游这么多天？！！！！！！！！）

## Output

m 行，每行一个整数，表示河流的数量。

# Samples

```input1
10 
1 1 1 2 2 2 2 3 4 4 
5 
3 
2 
5 
4 
1
```

```output1
7 
3 
10 
8 
0
```
