## 题目描述

农夫约翰和贝西又在玩游戏了。这与水槽有关。农夫约翰在谷仓后面隐藏了 $\red{N (1 <= N <= 20) }$个槽，并在其中一些槽中装满了食物。

$\red{Bessie }$已经问了 $\red{M (1 <= M <= 100) }$个表格问题，"这个列表（她背诵的）中有多少个槽被填满？"。$\red{Bessie }$需要您的帮助来推断哪些槽实际上已被填满。

考虑一个有四个槽的例子，$\red{Bessie }$提出了这些问题（并收到了指示的答案）：$\red{1）}$"这些槽中有多少被填满：槽 $\red{1}$"$\red{-> 1 }$个槽被填满 $\red{2）}$"这些槽中有多少已填满：槽 $\red{2 }$和 $\red{3}$" $\red{--> 1 }$个槽已填满 $\red{3) }$"这些槽中有多少已填满：槽 $\red{1 }$和 $\red{4}$" $\red{--> 1 }$个槽 已填满 $\red{4) }$"

## 输入格式

第 $\red{1 }$行：两个空格分隔的整数：$\red{N }$和 $\red{M }$

第 $\red{2..M+1 }$行：波谷的子集，指定为连续的 $\red{N 0 }$和 $\red{1 }$的序列，后跟一个整数，表示波谷的数量被填充的指定子集。

## 输出格式

第 $\red{1 }$行：

单行：

如果没有可能的填充槽集与 $\red{Farmer John }$的答案兼容，则字符串"$\red{IMPOSSIBLE}$"。

如果 $\red{Bessie }$无法从给定数据中准确确定填充了哪些波谷，则字符串"$\red{NOT UNIQUE}$"。

否则，一系列连续的 $\red{N 0 }$和 $\red{1 }$指定填充哪些波谷。

## 样例

#### 输入样例

```c
4 4
1000 1

```
#### 输出样例

1010
