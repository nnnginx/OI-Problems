# Description

给出一串正整数数列以及一个正整数 C，要求计算出所有满足 A - B = C 的数对的个数（不同位置的数字一样的数对算不同的数对）。

# Format

## Input

输入共两行。

第一行，两个正整数 N,C。

第二行，N个正整数，作为要求处理的那串数。

## Output

一行，表示该串正整数中包含的满足 A - B = C的数对的个数。

# Samples

```input1
4 1
1 1 2 3
```

```output1
3
```

# Limitation

对于 75\% 的数据，1≤**N**≤**2**0**0**0。

对于 100\% 的数据，1≤**N**≤**2**×**1**0^5，
0≤**a**i**​**<**2^3**0，
1≤**C**<**2**^30。

例如：N=5，C=2，5个整数是：2 2 4 8 10。答案是3。具体方案：第3个数减第1个数；第3个数减第2个数；第5个数减第4个数。
