## 题目描述

天空神殿是一个 $n\times n$ 的矩阵，里面有若干个宝藏。

第 $i$ 行第 $j$ 列的格子有三种情况：

- `.`：这一格是空地；
- `$`：这里藏着一个宝藏；
- `X`：这里是一堵墙，不能通过。

墙将天空神殿分成了若干个区域。小云只能在其中一个区域寻宝，且只能往上下左右四个方向走。试问她最多能寻得多少个宝藏？

## 输入格式

输入第一行一个整数 $n$。

接下来 $n$ 行，每行 $n$ 个字符，描述天空神殿的形态。

## 输出格式

输出一行一个整数，表示最多能寻得多少宝藏。

## 样例 #1

### 样例输入 #1

```
3
XX.
$.X
XX$
```

### 样例输出 #1

```
1
```

## 样例 #2

### 样例输入 #2

```
4
X$.$
.XXX
..XX
$XXX
```

### 样例输出 #2

```
2
```

## 提示
对于 $20\%$ 的数据，保证 $1\leq n\leq 5$。

对于另外 $20\%$ 的数据，保证不存在 `X`。

对于 $100\%$ 的数据，$1\le n\le 100$。
