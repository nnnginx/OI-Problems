## 题目描述

输入若干个整数（可以是正整数、负数或者零），输出它们的乘积的末$\red{3}$位。这些整数中会混入一些由大写字母组成的字符串，你的程序应当忽略它们。

## 输入格式

`第一行`，正整数$\red{n}$（不大于$\red{1000}$），表示有$\red{n}$个整数；

接下来共`n行`，每行一个混入了大写字母的整数，每行不超过$\red{250}$个字符，当然整数位数也不超过$\red{250}$位。

## 输出格式

$\red{n}$个整数乘积的末$\red{3}$位，对于乘积不足$\red{3}$位的情况，就输出完整的乘积，对于乘积为负数的情况，左侧不必加上`-`。

## 样例

#### 输入样例

```C
5
A12
C5DF
-4H
6S7P8YZ
P1001D
```

#### 输出样例

```C
720
```

## 提示

样例说明：

共$\red{5}$个整数相乘，$\red{12*5*(-4)*678*1001=-162882720}$，乘积末$\red{3}$位为$\red{720}$。

提示： 由于只求乘积的末$\red{3}$位，每个整数提取末$\red{3}$位即可（题中说明了整数位数不超过$\red{250}$位）在计算过程中也只保留末$\red{3}$位，

注意区分结果为$\red{0}$和末$\red{3}$位为$\red{0}$的情形。
