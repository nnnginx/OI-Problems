# 题目描述

小柒经常玩一个游戏叫“我的世界”，一天，她在游戏中开设了一个n*n(1<=n<1000)的矩形方阵，方阵中每一个格点都对应某一整数行和某一整数列，现在她有 k (k< n * n)个萤石，每个萤石可以照亮该位置的上、下、左、右、左上、右上、左下、右下 这八个方向的格点，如下图所示。
![image](./3105/file/kyNR_Pp4InH1GJqeKuXzZ.jpeg)

小柒趁着白天把这 k 个萤石放在矩形方阵中的一些位置，第 i 块萤石放在了（x<sub>i</sub> , y<sub>i</sub>）的位置上，等到天黑时，没有萤石或者光亮的格点就会生长出怪物来。小柒想知道到了天黑，这个矩形方阵中一共生长了多少个怪物？
## 输入

第一行，输入两个整数 n 和 k ，n表示矩形方阵的行和列数，k 表示萤石的数量

接下来，输入k行，每行两个整数 x,y 表示萤石摆放的格点位置。

## 输出
输出一个整数，表示矩形方阵中生长的怪物数量

# 样例

```输入

5 1
3 3

```
```输出
16
```

样例1说明：

如下图，5行5列的矩阵中，在（3,3）的位置放入萤石，会照亮该格点附近八个方向的格点，剩余没有光亮的格点数是16个
