## 题目描述

小明最近迷上了化学，几乎天天在实验室做实验，但是很多实验生成的化学产物的相对分子质量令他很困惑，不知如何计算，请你编程帮他计算。

## 输入格式

输入的`第一行`是一个正整数$\red{n}$，表示有$\red{n}$组测试数据。 接下来`n行`每行输入一个字符串，表示某个分子式，分子式中只包含大写字母和数字。

注意： 输入样例只包含$\red{8}$种元素，而这8种元素的相对原子质量如下： `H(1)`，`C(12)`，`N(14)`，`O(16)`，`F(19)`，`P(31)`，`S(32)`，`K(39)`。

## 输出格式

对于每组输入，输出相对分子质量。

## 样例

#### 输入样例

```C
4
H2O
KOH
CH4
SO2
```

#### 输出样例

```C
18
56
16
64
```
