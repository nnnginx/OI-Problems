## 题目描述

`HDU 1007`

最接近点对问题的提法是：给定平面上$\red{n}$个点，找其中的一对点，使得在$\red{n}$个点的所有点对中，该点对的距离最小。严格地说，最接近点对可能多于$\red{1}$对。为了简单起见，这里只限于找其中的一对。

## 输入格式

输入第一行为点的个数$\red{n（2≤n≤60 000）}$；接下来$\red{n}$行，每行两个实数$\red{x}$和$\red{y}$，表示一个点的行坐标和列坐标，中间用一个空格隔开。

## 输出格式

输出仅一行，为一个实数，表示最短距离的一半，精确到小数点后面$\red{2}$位。

## 样例

#### 输入样例

```c
3                                              

1 1

1 2

2 2
```

#### 输出样例

```c
0.50
```
