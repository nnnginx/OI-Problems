## 题目描述

如今涉及到计算庞大数量级、高精度数据的问题已经是司空见惯了。比如说，计算国债对于很多计算机设备来说是压力山大啊。

我们这个问题需要你编程来计算一下$\red{R^n}$（$\red{R}$是实数，$\red{0.0<R<99.999}$，$\red{n}$是整数，$\red{0<n<=25}$）的精确值。保证$\red{R}$的宽度小于等于$\red{6}$位。

## 输入格式

输入只有一行数据，$\red{R}$和$\red{n}$，用空格隔开

## 输出格式

输出只有一行，即$\red{R^n}$的精确值。

去掉整数部分的前导$\red{0}$和小数部分的后缀$\red{0}$。如果结果是整数，不要输出小数点；

如果结果有小数且整数部分为$\red{0}$，则从小数点开始输出。

## 样例

#### 输入数据

```C
0.100 10
```

#### 输出数据

```C
.0000000001
```

## 提示

把实数转字符数组可以用`sprintf`(字符数组,”`%lf`”,`double`数)。
