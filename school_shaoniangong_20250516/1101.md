## 题目描述

输入一行数字，如果我们把这行数字中的‘$\red{5}$’都看成空格，那么就得到一行用空格分割的若干非负整数
（可能有些整数以‘$\red{0}$’开头，这些头部的‘$\red{0}$’应该被忽略掉，除非这个整数就是由若干个‘$\red{0}$’组成的，这时这个整数就是$\red{0}$）。

你的任务是：对这些分割得到的整数，依从小到大的顺序排序输出。

## 输入格式

输入包含多组测试数据。每组输入样例只有一行数字（数字之间没有空格），这行数字的长度不大于$\red{1000}$。

输入样例保证：分割得到的非负整数不会大于$\red{100000000}$，输入样例不可能全由‘$\red{5}$’组成。

## 输出格式

对于每个测试用例，输出分割得到的整数排序的结果，

相邻的两个整数之间用一个空格分开，每组输出占一行。

## 样例

#### 输入样例

```C
0051231232050775
```

#### 输出样例

```C
0 77 12312320
```
