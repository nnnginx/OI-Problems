## 题目描述

给定一个只包含加法和乘法的算术表达式,请你编程计算表达式的值。

## 输入格式

输入仅有一行,为需要你计算的表达式,表达式中只包含数字、加法运算符$\red{“+”}$和乘法运算符$\red{“*”}$,且没有括号,所有参与运算的数字均为 $\red{0}$ 到 $\red{2 ^ {31} -1}$ 之间的整数。输入数据保证这一行只有$\red{ 0\sim 9、+、*}$这 $\red{12}$ 种字符。

## 输出格式

输出只有一行,包含一个整数,表示这个表达式的值。注意:当答案长度多于$ \red{4}$ 位时,请只输出最后 $\red{4}$ 位,前导 $\red{0}$ 不输出。

## 样例

#### 样例输入

```C
1+1*3+4
```

#### 样例输出

```C
8
```

## 数据范围与提示

对于 $\red{30\%}$的数据,$\red{0≤}$表达式中加法运算符和乘法运算符的总数$\red{≤100}$;

对于 $\red{80\%}$的数据,$\red{0≤}$表达式中加法运算符和乘法运算符的总数$\red{≤1000}$;

对于 $\red{100\%}$的数据,$\red{0≤}$表达式中加法运算符和乘法运算符的总数$\red{≤100000}$。
