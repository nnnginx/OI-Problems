## 题目描述

给定 $n,m,k$，要求构造长度为 $n$ 的数组 $a$，使其满足：
$1≤a_i≤m$；
对于所有满足 $1≤j≤n-k+1$ 的 $j$，都满足 $a_j⊕a_{j+1}⊕⋯⊕a_{j+k-1}=0$。
其中，$⊕$是按位异或运算。
如有多种方案，输出任意一种皆可。

## 输入格式

本题有多组测试数据。

第一行一个整数 $T$，表示测试数据组数。

接下来 $T$ 行，每行三个整数 $n,m,k$ 表示一组测试数据。

## 输出格式

对于每组测试数据，输出 $n$ 个用空格隔开的整数表示构造的数组 $a$。

如无法构造，输出 $-1$。

### 输入样例 1

```
2
3 3 3
1 100000 1
```

### 输出样例 1

```
3 1 2
-1
```

### 输入样例 2

```
2
4 10 3
6 10 0
```

输出样例 2

```
4 6 2 4
1 1 4 5 1 4
```

本题输出量可能较大，请使用较快的输出方式。
