## 题目描述

古代人认为$\red {666}$是属于魔鬼的数。

不但如此，只要某数字的十进制表示中有三个连续的$\red {6}$，古代人也认为这是个魔鬼的数，比如$\red {666,1666,6663,16666,6660666}$等等。

古代典籍中经常用“第$\red {X}$小的魔鬼的数”来指代这些数，这给研究人员带来了极大的不便。

现在请编写一个程序，可以实现输入$\red {X}$，输出对应的魔鬼数。

## 输入格式

第一行包含整数$\red {T}$，表示共有$\red {T}$组测试数据。

每组测试数据占一行，包含一个整数$\red {X}$。

## 输出格式

每组测试数据占一行，输出一个魔鬼数。

## 样例

#### 输入样例

```c
3
2
3
187
```

#### 输出样例

```c
1666
2666
66666
```

## 提示

$\red {1≤T≤1000}$,

$\red {1≤X≤5\times 10^7}$
