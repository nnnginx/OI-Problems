## 题目描述

给定一张 $\red {N\times M}$ 的棋盘，有一个机器人处于$\red {(x,y)}$位置。

这个机器人可以进行很多轮行动，每次等概论地随机选择停在原地、向左移动一格、向右移动一格或向下移动一格。

当然机器人不能移出棋盘。

求机器人从起点走到最后一行的任意一个位置上，所需行动次数的数学期望值。

## 输入格式

第一行包含两个整数$\red N$ 和 $\red M$。

第二行包含两个整数 $\red x $和$\red y$，表示机器人的初始位置。

设定棋盘左上角为$\red {(1,1)}$，右下角为$\red {(N,M)}$。

## 输出格式

输出一个实数，表示数学期望，结果保留四位小数。

## 样例

#### 输入样例

```c
10 14 
5 14
```

#### 输出样例

```c
18.0038
```

## 提示

$\red {1≤N,M≤1000}$
