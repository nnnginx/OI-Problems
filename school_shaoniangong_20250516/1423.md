## 题目描述

今天老师给天天留了一道编程题，题目是这样的，给你一个仅由大写字母组成的字符串，该字符串是由一个$\red n$位的字符子串所重复若干次组成。

现在想要你找出这个字符子串，然后统计该字符子串的字符个数$\red n$，并将$\red n$输出。

天天想了很久也没思路，你能否帮天天实现一下呢。

## 输入格式

一个由大写字母组成的字符串（长度不超过$\red{10^4}$），该字符串是仅由一个字符子串重复若干次后组成。

## 输出格式

一个整数，即字符子串的字符个数。

## 样例

#### 输入样例

```C
ABABABABABABAB
```

#### 输出样例

```C
2
```

## 提示

所有数据均满足：
字符串（长度不超过$\red{10^4}$）
