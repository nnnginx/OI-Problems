## 题目描述

金明今天很开心，家里购置的新房就要领钥匙了，新房里有一间他自己专用的很宽敞的房间。

更让他高兴的是，妈妈昨天对他说：“你的房间需要购买哪些物品，怎么布置，你说了算，只要不超过 $\red{N}$ 元钱就行”。

今天一早金明就开始做预算，但是他想买的东西太多了，肯定会超过妈妈限定的 $\red{N}$ 元。对于一个想买的物品，金明至多只能买一个。

于是，他把每件物品规定了一个重要度，分为五等：用整数 $\red{1}$ `~` $\red{5}$ 表示，第五等最重要。

他还从因特网上查到了每件物品的价格（都是整数元）。

他希望在不超过 $\red{N}$ 元的前提下，使每件物品的价格与重要度的乘积的总和最大。

设第j件物品的价格为 $\red{v[j]}$，重要度为 $\red{w[j]}$，共选中了 $\red{k}$ 件物品，编号依次为 $\red{j_1, j_2, \cdots, j_k}$，则所求的总和为：

$\red{ v[j_1]*w[j_1]+v[j_2]*w[j_2]+ …+v[j_k]*w[j_k] }$

请你帮助金明设计一个满足要求的购物单。

## 输入格式

第一行有两个正整数 $\red{N}$ 和 $\red{m}$， 其中 $\red{N（<30000}$）表示总钱数，$\red{m}$（$\red{<25}$）为希望购买物品的个数。
从第 $\red{2}$ 行到第 $\red{m+1}$ 行，第 $\red{j}$ 行给出了编号为 $\red{j-1}$ 的物品的基本数据，每行有$\red{2}$个非负整数 $\red{v}$ 和 $\red{p}$，其中 $\red{v}$ 表示该物品的价格( $\red{v \le 10000}$ )，$\red{p}$ 表示该物品的重要度( $\red{1}$`~` $\red{5}$ )）

## 输出格式

输出只有一个正整数，为不超过总钱数的物品的价格与重要度乘积的总和的最大值。

## 样例

#### 输入样例

```c
1000 5
800 2
400 5
300 5
400 3
200 2
```

#### 输出样例

```c
3900
```

## 提示

$\red{ N \leq 30000}$，$\red{m \leq 25}$，$\red{v \leq 10000}$，$\red{1 \leq p \leq 5}$。

保证答案不超过 $\red{10^9}$
