## 题目描述

动物王国中有三类动物 $\red A$ , $\red B$ , $\red C$，这三类动物的食物链构成了有趣的环形。

$\red A$ 吃 $\red B$，$\red B$ 吃 $\red C$，$\red C$ 吃 $\red A$。

现有 $\red N$ 个动物，以 $\red{1\sim N}$编号。

每个动物都是 $\red A$ , $\red B$ , $\red C$ 中的一种，但是我们并不知道它到底是哪一种。

有人用两种说法对这 $\red N$ 个动物所构成的食物链关系进行描述：

第一种说法是”`1 X Y`”，表示 $\red X$ 和 $\red Y$是同类。

第二种说法是”`2 X Y`”，表示 $\red X$ 吃 $\red Y$。

此人对 $\red N$ 个动物，用上述两种说法，一句接一句地说出 $\red K$ 句话，这 $\red K$ 句话有的是真的，有的是假的。

当一句话满足下列三条之一时，这句话就是假话，否则就是真话。

$\red{1）}$ 当前的话与前面的某些真的话冲突，就是假话；

$\red{2）}$ 当前的话中 $\red X$ 或 $\red Y$ 比 $\red N$ 大，就是假话；

$\red{3）}$ 当前的话表示 $\red X$ 吃 $\red X$，就是假话。

你的任务是根据给定的 $\red N$ 和 $\red K$ 句话，输出假话的总数。

## 输入格式

第一行是两个整数 $\red N$ 和 $\red K$，以一个空格分隔。

以下 $\red K$ 行每行是三个正整数 $\red D$，$\red X$，$\red Y$，两数之间用一个空格隔开，其中 $\red D$ 表示说法的种类。

若 $\red{D=1}$，则表示 $\red X$ 和 $\red Y$ 是同类。

若 $\red {D=2}$，则表示 $\red X$ 吃  $\red Y$。

## 输出格式

只有一个整数，表示假话的数目。

## 样例

#### 输入样例

```c
100 7
1 101 1 
2 1 2
2 2 3 
2 3 3 
1 1 3 
2 3 1 
1 5 5
```

#### 输出样例

```c
3
```


## 提示

$\red{1\le N\le 50000}$,

$\red{0\le K\le 100000}$
