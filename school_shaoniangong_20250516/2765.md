## 题目描述

快到奶牛冠军杯足球赛了，今年在$\red{J}$队与$\red{H}$队之间将会出现十分激烈的对抗．

作为今年牛奶生产创记录的奖励，约翰同意他的奶牛们观看这场比赛．$\red{N(1≤}$$\red{N≤}$$\red{2500)}$头牛已经在仓房排好队．他们将被挨个儿地接上车，直到约翰喊停．

之后下一辆车继续挨个儿接奶牛．最终，奶牛将都被送上车．一些牛是$\red{J}$队的球迷，另一些是$\red{H}$队的球迷，竞争对手之间往往相处得很糟．所以，约翰不能让一辆 汽车上载过多的$\red{J}$队球迷或$\red{H}$队球迷，这样另一支队的球迷在途中会受到恐吓．

因此，他得保证一辆车中，两队球迷的个数差的绝对值在$\red{I(1≤}$$\red{I≤}$$\red{N)}$内．除非那辆车上只有$\red{J}$队或$\red{H}$队的球迷．

给出奶牛上车的次序，请计算出最少几辆汽车可以解决问题．

## 输入格式

第$\red{1}$行输入两个分开的整数$\red{N}$和$\red{J}$；

接下来$\red{N}$行表示奶牛们在仓房中排队的顺序．用$\red{J}$和$\red{H}$表示她们是$\red{J}$队和$\red{H}$队昀球迷．

## 输出格式

一个整数，表示最少汽车的数量．

## 样例

```c
14 3
H
J
H
H

J
H
J

H

H
H
```

#### 输出样例

```c
3
```

## 提示

有多种方案，如：除最后$\red{5}$只外，其余皆坐一辆车；最后$\red{5}$只坐第二辆车
