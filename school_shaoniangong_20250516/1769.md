## 题目描述

`POJ 2192`

天顶星人使用量子纠缠技术传送信息,量子纠缠超越了我们生活的四维时空,不受四维
时空的约束,其传输信息的速度至少比光速快$\red{10000}$倍。换句话说,即使传送双方远在字
宙的两端,信息也可以瞬间到达。但天顶星人传送的信息有真有假,判断真假的方式是对三
个字符串进行验证,即给三个字符串,判断第三个字符串是否由前两个字符串的一部分序列
顺序组成,例如字符串$\red{A}$为"$\red{cat}$",字符串$\red{B}$为"$\red{tree}$",字符串$\red{C}$为"$\red{tcraete}$" ,字符串$\red{C}$由字
符串$\red{A}$和字符串$\red{B}$组成,则信息为真。

再比如字符串$\red{A}$为"$\red{cat}$",字符串$\red{B}$为"$\red{tree}$" ,字符串$\red{C}$为"$\red{carttee}$" ,由于组成的序列顺
序不对,所以信息应该是假。

## 输入格式

第一行有一个整数$\red{N,}$大小在$\red{1\sim1 000,}$代表后续有$\red{N}$行,每行有三组字符串数据,每
组字符串由空格分隔开,第三组字符串数据的长度总是前两组字符串数据的长度总和。前
两行字符串的长度在$\red{1\sim200}$。

## 输出格式

每行如果信息为真,则打印$\red{Data ~set~ n: ~yes}$

每行如果信息为假,则打印$\red{Data~ set ~n: ~no }$

$\red{n}$代表序号。

## 样例

#### 输入样例

```c
3
cat tree tcraete
cat tree catrtee
cat tree cttaree
```

#### 输出样例

```c
Data set 1: yes
Data set 2: yes
Data set 3: no
```
