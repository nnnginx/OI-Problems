* ## 题目描述

在清华校园里，没有自行车，上课办事会很不方便。
但实际上。并非去办任何事情都是骑车快,因为骑车总要找车、开锁、停车、锁车等,这要耽误一些时间。假设找到自行车，开锁并骑上自行车的时间为$\red{27}$秒；

停车锁车的时间为$\red{23}$秒；步行每秒行走$\red{1.2}$米，骑车每秒行走$\red{3.0}$米。

请判断走不同的距离去办事,是骑车快还是走路快。如果骑车快，输出一行"`Bike`"；

如果走路快，输出一行"`Walk`"；如果一样快，输出一行"`All`"。

## 输入格式

输入一行，包含一个实数，表示一次办事要行走的距离,单位为米。

## 输出格式

- 输出一行,如果骑车快,输出一行"`Bike`";
- 如果走路快,输出一行"`Walk`";
- 如果一样快,输出一行"`All`"。

## 样例

#### 输入数据

```C
120
```

#### 输出数据

```C
Bike
```

## 提示

请使用`float`
