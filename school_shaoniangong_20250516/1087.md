## 题目描述

假定图书馆新进了$\red{m}$（$\red{10 ≤ m ≤ 999}$）本图书，它们都是由$\red{n}$（$\red{1 ≤ n ≤ 26}$）个作者独立或相互合作编著的。

假设m本图书编号为整数（$\red{1}$`~`$\red{999}$），作者的姓名为字母（$\red{A}$`~`$\red{Z}$），请根据图书作者列表找出参与编著图书最多的作者和他的图书列表。

## 输入格式

第一行为所进图书数量$\red{m}$，其余`m行`，每行是一本图书的信息，其中第一个整数为图书编号，接着一个空格之后是一个由大写英文字母组成的没有重复字符的字符串，
$\red{
每个字母代表一个作者。输入样例保证仅有一个作者出书最多。

## 输出格式

输出有多行：

`第一行`为出书最多的作者字母；

`第二行`为作者出书的数量；

其余各行为作者参与编著的图书编号（按输入顺序输出）。

