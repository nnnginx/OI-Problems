## 题目描述

每天早上，农夫约翰的奶牛们被挤奶的时候，都会站成一个$\red{R}$行$\red{C}$列的方阵。

现在在每个奶牛的身上标注表示其品种的大写字母，则所有奶牛共同构成了一个$\red{R}$行$\red{C}$列的字符矩阵。

现在给定由所有奶牛构成的矩阵，求它的最小覆盖子矩阵的面积是多少。

如果一个子矩阵无限复制扩张之后得到的矩阵能包含原来的矩阵，则称该子矩阵为覆盖子矩阵。

## 输入格式

第$\red{1}$行：输入两个用空格隔开的整数，$\red{R}$和$\red{C}$。

第$\red{2..R+1}$行：描绘由奶牛构成的$\red{R}$行$\red{C}$列的矩阵，每行$\red{C}$个字符，字符之间没有空格。

## 输出格式

输出最小覆盖子矩阵的面积。（每个字符的面积为$\red{1}$）

## 样例

#### 输入样例

```c
2 5
ABABA
ABABA
```

#### 输出样例

```c
2
```

## 提示

$\red{1≤R≤10000}$,

$\red{1≤C≤75}$
