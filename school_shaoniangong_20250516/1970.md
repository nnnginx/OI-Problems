## 题目描述

小明发明了一种数，叫完美数。

完美数的定义是：对于一个正整数 $x$，如果存在大于 $1$ 的整数 $y,z$ 且 $x=y^z$，则 $x$ 为完美数。

现在小明想知道 $n$ 是不是完美数，你能帮帮他吗？

## 输入格式

多组数据。

第一行一个正整数 $T$，为数据的组数。

接下来 $T$ 行，每行一个正整数 $x$。

## 输出格式

对于每个 $x$，如果它是完美数，输出"YES"，否则输出"NO"（均不含引号）。

## 样例 #1

### 样例输入 #1

```
10
1
2
3
4
5
6
8
9
72
114514
```

### 样例输出 #1

```
NO
NO
NO
YES
NO
NO
YES
YES
NO
NO
```

## 提示

$1\le T\le100,1\le n<2^{31}$。
