## 题目描述

有一长度为$\red{N}$($\red{1< =Ｎ< =10}$)的地板，给定两种不同瓷砖：一种长度为$\red{1}$，另一种长度为$\red{2}$，数目不限。要将这个长度为$\red{N}$的地板铺满，一共有多少种不同的铺法？

例如，长度为$\red{4}$的地面一共有如下$\red{5}$种铺法：

<pre>
4=1+1+1+1
4=2+1+1
4=1+2+1
4=1+1+2
4=2+2
</pre>

编程用递归的方法求解上述问题。

## 输入格式

只有一个数$\red{N}$，代表地板的长度

## 输出格式

输出一个数，代表所有不同的瓷砖铺放方法的总数

#### 输入样例

```C
4
```

#### 输出样例

```C
5
```
