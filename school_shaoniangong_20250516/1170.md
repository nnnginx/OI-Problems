## 题目描述

求$\red{n}$个字符串的最长公共子串，$\red{n<20}$，字符串长度不超过$\red{255}$。

## 输入格式

第一行一个整数$\red{n}$，接下来`n行`，每行一个字符串。

## 输出格式

输出最长公共子串，如果没有公共子串则输出“`no`”。

## 样例

#### 输入样例

```C
3
what is local bus?
Name some local bus.
local bus is high speed I/O bus close to the processor.
```

#### 输出样例

```C
local bus
```
