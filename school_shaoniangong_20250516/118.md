## 题目描述

有一个球形空间产生器能够在 $\red n$ 维空间中产生一个坚硬的球体。

现在，你被困在了这个 $\red n$ 维球体中，你只知道球面上 $\red{n+1}$ 个点的坐标，你需要以最快的速度确定这个 $\red n$ 维球体的球心坐标，以便于摧毁这个球形空间产生器。

## 输入格式

第一行是一个整数 $\red n$。

接下来的 $\red{n+1}$ 行，每行有 $\red n$ 个实数，表示球面上一点的 $\red n$ 维坐标。

每一个实数精确到小数点后 $\red 6$ 位，且其绝对值都不超过 $\red{20000}$。

## 输出格式

有且只有一行，依次给出球心的 $\red n$ 维坐标（ $\red n$ 个实数），两个实数之间用一个空格隔开。

每个实数精确到小数点后 $\red 3$ 位。

数据保证有解。

## 样例

#### 输入样例

```c
2
0.0 0.0
-1.0 1.0
1.0 0.0
```

#### 输出样例

```c
0.500 1.500
```

## 提示

$\red{1\leq n\leq 10}$
