## 题目描述

输入若干个单词，输出它们的平均长度。单词只包含大写字母和小写字母，用一个或多个空格隔开。

## 输入格式

一行，若干个单词的句子，单词只包含大写字母和小写字母，用一个或多个空格隔开。

单词长度$\red{ \le 2*10^5}$

## 输出格式

所有单词的平均长度，保留一位小数。

## 样例

#### 输入样例

```C
John  reed was a  schoolboy of fourteen years old
```

#### 输出样例

```C
4.3
```

## 提示

$\blue{小知识}$
全球最长英语单词长达`189819`个字母。
