## 题目描述

科学家在热带森林中发现了一种特殊的昆虫，这种昆虫的繁殖能力很强。

每对成虫过$\red{x}$个月后，每个月产$\red{y}$对卵，每对卵要过两个月长成成虫。

假设每个成虫不死，第一个月只有一对成虫，且卵长成成虫后过$\red{x}$个月产卵，问过$\red{z}$个月以后，共有成虫多少对?

## 输入格式

$\red{x}$,$\red{y}$,$\red{z}$的数值。

## 输出格式

过$\red{z}$个月以后，共有成虫对数。

## 样例

#### 输入样例

```C
1 2 8
```

#### 输出样例

```C
37
```

## 数据范围与提示

$\red{0≤x≤20}$,

$\red{1≤y≤20}$,

$\red{x≤z≤50}$
