#### 题目限制

1000 ms    256 M

#### 题目描述

已知正整数 $i,j$ 的最小公倍数为 $n$ ，求 $max(i,j)$ 的最小值。

#### 输入格式

输入一行：一个数 $n$ （ $n\le 1e12$ ）

#### 输出格式

输出一行一个数，表示 $max(i,j)$ 的最小值。

#### 数据范围

对于 $10\%$ 的数据， $1 \le n \le 30$ ；

对于 $38\%$ 的数据， $1 \le n \le 200000$ ；

对于 $100\%$ 的数据， $1 \le n \le 10^{12}$ ；

#### 输入样例 1

```
6
```

#### 输出样例 1

```
3
```
