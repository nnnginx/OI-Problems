## 题目描述

由于邪狼编写的万进制高精度除法有小小的缺陷，导致修罗王打开最后一道牢门时触发了陷阱，修罗王和邪狼因此落入一个类似于$\red{M\times N}$的网格棋盘中，修罗王和邪狼必须要从左下角$\red{（1，1）}$开始逃到右上角$\red{（M，N）}$的安全位置，才可以摆脱狱警的追踪，但是修罗王和邪狼每次只能向上或向右走，试问有多少种不同的走法？已知$\red{1≤N<1040，0≤M≤1000}$。

## 输入格式

两个整数$\red{M、N}$。

## 输出格式

一个整数，即路径数。

## 样例

#### 输入样例

```c
2 2
```

#### 输出样例

```c
2
```
