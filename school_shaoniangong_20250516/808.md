## 题目描述

第一行一个整数 $\red{n}$。

第二行 $\red{n}$ 个整数 $\red{a_1 , a_2 , .... , a_n}$。

## 输入格式

一行一个整数 $\red {a_1 \bigoplus a_2 \bigoplus .... \bigoplus a_n}$ 。其中 $\red \bigoplus $ 是 异或

## 输出格式

输出所有数据异或($\red{\bigoplus}$)结果

## 样例

#### 输入样例

```C
2
2 2
```

#### 输出样例

```C
0
```

## 提示

对于$\red {100\%} $ 的数据 ，$\red { n = 10^7 , -2^{31} \le a_i \lt  2^{31} } $
