## 题目描述

给你一个整数$\red{n}$和一个整数$\red{k,}$想要把$\red{n}$分成$\red{k}$个不同的整数之和,有多少种不同的方案数(数据保证$\red{n}$能分成$\red{k}$个不同的整数之和)(交换顺序算是同一种方法，比如
$\red{2022 = 1000+ 1022}$和$\red{2022 = 1022 + 1000}$ )

## 输入格式

一行输入两个整数$\red{n}$和$\red{k}$

## 输出格式

输出一个整数表示方案数

## 样例

#### 输入样例1

```c
5 2
```

#### 输出样例1

```c
2
```

#### 输入样例2

```c
5 3
```

#### 输出样例2

```c
0
```

## 提示

数据范围

对于$\red{30\%}$的数据， $\red{n≤}$$\red{10,k≤}$$\red{3}$

对于$\red{100\%}$的数据， $\red{n≤}$$\red{2022,k≤}$$\red{10}$
