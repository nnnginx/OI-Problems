## 题目描述

众所周知，骰子是一个六面分别刻有一到六点的立方体,每次投掷骰子，从理论上讲
得到一点到六点的概率都是$\red{1/6}$。

今有骰子一颗,连续投掷$\red{N}$次,间点数总和大于等于$\red{X}$的概率是多少?

## 输入格式

输人文件一行两个整数，分别表示$\red{n}$和$\red{x，}$其中$\red{1≤}$$\red{N≤}$$\red{24,0≤}$$\red{x<150}$。

## 输出格式

输出一行一个分数,要求以最简的形式精确地表达出连续投掷$\red{N}$次骰子,总点
数大于等于$\red{X}$的概率。如果是$\red{0/1}$就输出$\red{0,}$如果是$\red{1/1}$就输出$\red{1}$。

## 样例

#### 输入样例

```c
3 9
```

#### 输出样例

```c
20/27
```
