## 题目描述

有 $\red{n }$个盘子。盘子被生产出来后，被按照某种顺序摞在一起。初始盘堆中如果一
个盘子比所有它上面的盘子都大，那么它是安全的，否则它是危险的。称初始盘堆为
$\red{A，}$另外有一个开始为空的盘堆 $\red{B}$。为了掩盖失误，生产商会对盘子序列做一些"处
理"，每次进行以下操作中的一个：

$\red{(1)}$将 $\red{A }$最上面的盘子放到 $\red{B }$最上面；

$\red{(2)}$将 $\red{B }$最上
面的盘子给你。

在得到所有 $\red{n }$个盘子之后，你需要判断初始盘堆里是否有危险的盘子。

## 输入格式

输入文件包含多组数据（不超过 $\red{10 }$组）
每组数据的第一行为一个整数 $\red{n}$
接下来 $\red{n }$个整数，第 $\red{i }$个整数表示你收到的第 $\red{i }$个盘子的大小

## 输出格式

对于每组数据，如果存在危险的盘子，输出"$\red{J}$"，否则输出"$\red{Y}$"

## 样例

#### 输入样例

```c
3
2 1 3
3
3 1 2
```

#### 输出样例

```c
Y
J
```

## 提示

$\red{20\%}$的数据保证 $\red{n<=8}$

$\red{80\%}$的数据保证 $\red{n<=1,000}$

$\red{100\%}$的数据保证 $\red{1<=n<=100,000，}$$\red{0<}$盘子大小$\red{<1,000,000,000 }$且互不相等
