## 题目描述

考虑如下定义的特殊表达式:

$\red{(1)}$数字$\red{1,2,3,5,7}$都是合法表达式;

$\red{(2)}$若$\red{a}$是合法表达式,$\red{a!}$也是;

$\red{(3)}$若$\red{a,b}$都合法,则$\red{(a+b),(a \times b),(a \^~b)}$都是合法表达式。

对一个特定的值,超素表达式给出按以上规则定义的由最少数字组成的表达式。例
如:$\red{(((3 \times (2 \times 2))-2) \times 5)}$和$\red{3!!}$都是$\red{720}$的合法表达式,但只有后者才是超素表达式。

现在请你编个程序,对于输人的整数，输出它的超素表达式。

## 输入格式

输入一行一个整数$\red{n(0<n≤}$$\red{20 000)}$。

## 输出格式

输出一行,包含一个要求的超索表达式。

注意:结果可能不唯一,你只要输出任何一个。

## 样例

#### 输入样例

```c
14
```

#### 输出样例

```c
(2*7)
```
