
###题目描述 
小修回忆起整个爬山过程，最难忘的就是当时天空中的云朵，看起来很漂亮，挺想摘下
那样美的几朵云，做成棉花糖。
给你云朵的个数 N，再给你 M 个关系，表示哪些云朵可以连在一起。现在小修要把所有
云朵连成 K 个棉花糖，一个棉花糖最少要用掉一朵云，小修想知道他怎么连，花费的代价 最小。
### 输入格式
第一行三个正整数 N,M,K。
接下来 M 行，每行三个数 X,Y,L，表示 X 云和 Y 云可以通过 L 的代价连在一起。
### 输出格式
输出一行，仅一个整数，表示最小的代价。 如果怎么连都连不出K个棉花糖，输出”No Answer”(不含引号)。
### 样例
#### 输入样例1
```
3 1 2
1 2 1
```
#### 输出样例1
``` 
1
```
#### 输入样例2
```
3 1 1
1 2 1
```
#### 输出样例2
```
No Answer
```
### 提示
对于 $\red{30%}$的数据，$\red{1<=N<=100，1<=M<=1000}$;

对于 $\red{100%}$的数据，$\red{1<=N<=10^3，1<=M<=10^4，1<=K<=10，1<=X,Y<=N，0<=L<10^4}$;
