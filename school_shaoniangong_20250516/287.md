## 题目描述

在一块 $\red {N\times M}$ 的网格状地面上，有一些格子是泥泞的，其他格子是干净的。

现在需要用一些宽度为$\red {1}$、长度任意的木板把泥地盖住，同时不能盖住干净的地面。

每块木板必须覆盖若干个完整的格子，木板可以重叠。

求最少需要多少木板。

## 输入格式

第一行包含两个整数 $\red {N}$ 和 $\red {M}$。

接下来$\red {N}$行，每行$\red {M}$个字符，用来描述地面，$\red {‘*’}$表示泥泞格子，$\red {‘.’}$表示干净格子，字符之间没有空格。

## 输出格式

输出一个整数，表示结果。

## 样例

#### 输入样例

```c
4 4
*.*.
.***
***.
..*.
```

#### 输出样例

```c
4
```

## 提示

$\red {1≤N,M≤50}$
