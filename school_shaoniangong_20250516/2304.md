## 题目描述

佩奇有$\red{n}$个空盒子，对于每一个 $\red{i，}$第$\red{ i}$个盒子是一个边长为 $\red{a_i}$的正方体。

如果满足以下条件，佩奇可以将盒子 放入另一个盒子 中：

第 $\red{i}$个盒子没有放进另一个盒子里；
第 $\red{j}$个盒子不包含任何其他盒子；
第$\red{i }$个盒子比第$\red{j}$个盒子小$\red{( a_i<a_j)}$。

佩奇可以将盒子互相放置任意次数。 他希望旧能减少可以看见的盒子的数量。

如果一个盒子没有被放入另一个盒子中，则该盒子为可见的。

现请你帮助佩奇确定可见的盒子的最少数量。

## 输入格式

共两行。

第一行包含一个整数 $\red{n，}$表示佩奇有的空盒子数量。

第二行包含$\red{n }$个整数 $\red{a_1, a_2... a_n}$表示 $\red{n}$个盒子的边长。

## 输出格式

一个整数，表示答案。

## 样例

#### 输入样例1

```c
3
1 2 3
```

#### 输出样例1

```c
1
```

#### 输入样例2

```c
4
4 2 4 3
```

#### 输出样例2

```c
2
```

## 提示

对于$\red{30\%}$的数据，有$\red{1≤}$$\red{n≤}$$\red{10}$;

对于$\red{50\%}$的数据，有$\red{1≤}$$\red{n≤}$$\red{100}$;

对于$\red{100\%}$的数据，有$\red{1≤}$$\red{n,  a_i≤}$$\red{100000}$。
