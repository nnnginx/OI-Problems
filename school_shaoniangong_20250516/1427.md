## 题目描述

面面是个爱学习的孩子，面面做数学题的时候遇到这样一道难题，对于一个长度为n的正整数序列，$\red{a_1,a_2......a_n}$满足  $\red{1 \le a_1,a_2.....a_n \le n}$;

聪明的你是否可以帮助他快速的从$\red n$个数中找出一个重复的数字呢，数据保证有且仅有一个数是重复的。

## 输入格式

共两行；
第一行包括一个正整数$\red n$；
第二行包含n个正整数$\red{a_1,a_2......a_n}$;用空格隔开

## 输出格式

共一行，为重复的正整数；

## 样例

#### 输入样例

```C
5
4 2 1 3 1
```

#### 输出样例

```C
1
```

## 提示

对于所有的数据满足:

$\red{1 \le n \le 10^6}$;

$\red{1 \le a_1,a_2.....a_n<n}$;
