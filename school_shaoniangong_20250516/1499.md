## 题目描述

写一程序读入二次方程的系数$\red{a,b,c}$,计算并输出二次方程的二个根。二次方程为：$\red{ax^2+bx+c=0}$

## 输入格式：

共一行，三个数$\red{a,b,c}$。

## 输出格式：

输出按以下格式，字符$\red{x}$为小写，保留小数点后两位。

## 样例

#### 输入样例 1

```c
1 2 1
```

#### 输出样例 1

```c
x=-1.00
```

#### 输入样例 2

```c
1 2 8
```

#### 输出样例 2

```c
No root.
```

#### 输入样例 3

```c
1 25 6
```

#### 输出样例 3

```c
x1=-0.24
x2=-24.76
```
