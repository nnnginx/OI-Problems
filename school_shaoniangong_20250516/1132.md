## 题目描述

编写程序，求给定字符串$\red{s}$的亲朋字符串$\red{s_1}$。

亲朋字符串$\red{s_1}$定义如下：给定字符串s的第一个字符的`ASCII`值加第二个字符的`ASCII`值，得到第一个亲朋字符；

给定字符串$\red{s}$的第二个字符的`ASCII`值加第三个字符的`ASCII`值，得到第二个亲朋字符；

依此类推，直到给定字符串$\red{s}$的倒数第二个字符。亲朋字符串的最后一个字符由给定字符串$\red{s}$的最后一个字符`ASCII`值加$\red{s}$的第一个字符的`ASCII`值。

## 输入格式

输入`一行`，一个长度大于等于$\red{2}$，小于等于$\red{100}$的字符串。字符串中每个字符的`ASCII`值不大于$\red{63}$。

## 输出格式

输出`一行`，为变换后的亲朋字符串。

输入保证变换后的字符串只有一行。

## 样例

#### 输入样例

```C
1234
```

#### 输出样例

```C
cege
```
