# 题目描述

小柒正在大棚种植果蔬。

一共有N盆果蔬，排放成一排。从左至右依次编号 1−N，每盆果蔬相距1米。为了让果蔬长的更好，小柒购买了一批灯泡对果蔬进行照射。灯泡的照射距离为K米，如果灯泡吊在编号为X的果蔬上面，在X−K 到 X+K范围内(包括两个端点)的果蔬都能够被照射。但是第一批购买的灯泡有限，不能将所有果蔬都照射到，需要再购买一批灯泡。

请问：至少需要增加多少个灯泡，使得所有果蔬都在照射范围内。

## 输入

第一行一个整数 N，表示有 N 盆果蔬。

第二行一个整数 M，表示有 M 个灯泡。

第三行一个整数 K，表示灯泡照射的距离为 K 米。

接下来 M 个数，表示已经购买的灯泡安放的位置。

## 输出

输出一个整数，添加灯泡的最少数量。

# 样例

```
26
3
3
3
19
26
```

```
2
```

# 数据说明

对于100%的数据，1≤N≤1000，1≤M≤N，0≤K≤N。
