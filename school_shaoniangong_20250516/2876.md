## 题目描述

给出两个数$\red{A,B(B>=A)，}$ 问有多少个序列满足以下条件：

$\red{1. }$序列是递增的。

$\red{2. }$所有数字属于区间 $\red{[A,B] （}$包括 $\red{A }$和 $\red{B ，}$ $\red{B - A <= 100 ）}$ 。

$\red{3. }$序列中的所有数字两两互质

## 输入格式

一行输入两个数 $\red{A, B}$。其中$\red{1≤}$$\red{A≤}$$\red{B≤}$$\red{1e18，}$$\red{B-A≤}$$\red{100}$。

## 输出格式

输出对应的答案。

## 样例

#### 输入样例

```c
2 4
```

#### 输出样例

```c
5
```

## 提示

对于$\red{25}$的数据 $\red{1 ≤}$ $\red{A ≤}$ $\red{B ≤}$ $\red{10^7 ~B - A ≤}$ $\red{30}$

对于$\red{40}$的数据 $\red{1 ≤}$ $\red{A ≤}$ $\red{B ≤}$ $\red{10^{12}~ B - A ≤}$ $\red{50}$

对于$\red{100}$的数据 $\red{1 ≤}$ $\red{A ≤}$ $\red{B ≤}$ $\red{10^{18}~ B - A ≤}$ $\red{100}$

样例解释

$\red{2,3,4,(2,3),(3,4)}$
