## 题目描述

虽然骑自行车有点累，终于还是到达了神庙门口。

守门的人说他遇到一个想不明白的问题，如果你能帮他解决，他就放你进去。

他家的鸭第一天生了`一个小鸭`，第二天生了`两个小鸭`，第三天生了`三个小鸭`......第$\red n$天生了`n个小鸭`，他想知道他家现在有多少个小鸭。

机智的你赶紧帮他解决这个问题吧！

## 输入格式

输入整数$\red N$天

## 输出格式

输出总的小鸭数量$\red M$

例如$\red {N=3}$，总共的小鸭数量是$\red {1+2+3=6}$

## 样例

#### 输入样例

```C
3
```

#### 输出样例

```C
6
```
