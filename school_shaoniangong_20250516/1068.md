## 题目描述

输入三个自然数$\red{N}$，$\red{i}$，$\red{j}$ （$\red{1<=i<=N，1<=j<=N}$），

输出在一个`N*N`格的棋盘中（行列均从1开始编号），与格子（$\red{i，j}$）同行、同列、同一对角线的所有格子的位置。

如：$\red{n=4}$，$\red{i=2}$，$\red{j=3}$表示了棋盘中的第二行第三列的格子，如下：

|       | 第一列 | 第二列 | 第三列 | 第四列 |
| ----- | --------- | --------- | :---------: | --------- |
| 第一行 | . |  | . |  |
| 第二行 | |.  | (2,3) |.|
| 第三行 |.||.||
| 第四行 ||.||.|

当$\red{n=4}$，$\red{i=2}$，$\red{j=3}$时，输出的结果是：

($\red{2,1}$) ($\red{2,2}$) ($\red{2,3}$) ($\red{2,4}$)                        同一行上格子的位置

($\red{1,3}$) ($\red{2,3}$) ($\red{3,3}$) ($\red{4,3}$)                        同一列上格子的位置

($\red{1,2}$) ($\red{2,3}$) ($\red{3,4}$)                                      左上到右下对角线上的格子的位置

($\red{4,1}$) ($\red{3,2}$) ($\red{2,3}$) ($\red{1,4}$ )                       左下到右上对角线上的格子的位置

## 输入格式

一行，三个自然数$\red{N}$，$\red{i}$，$\red{j}$，相邻两个数之间用单个空格隔开。$\red{1 <= N <= 10}$。

## 输出格式

四行：

- `第一行`：从左到右输出同一行格子位置；
- `第二行`：从上到下输出同一列格子位置；
- `第三行`：从左上到右下输出同一对角线格子位置；
- `第四行`：从左下到右上输出同一对角线格子位置。
  其中每个格子位置用如下格式输出：($\red{x,y}$)，$\red{x}$为行号，$\red{y}$为列号，采用英文标点，中间无空格。

相邻两个格子位置之间用单个空格隔开。

## 样例

#### 输入样例

```C
4 2 3
```

#### 输出样例

```C
(2,1) (2,2) (2,3) (2,4)
(1,3) (2,3) (3,3) (4,3)
(1,2) (2,3) (3,4)
(4,1) (3,2) (2,3) (1,4)
```
