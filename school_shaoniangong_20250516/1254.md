## 题目描述

`小BX`正在玩一个解密类游戏，但是他遇到了一个难题，这里有一些数，想要进入到下一关的话，就需要从数$\red{K}$中删去要求的$\red{N}$个数字使剩下的数字组成的数最大，为了解决这个问题，`小BX`请你动用编程的力量为他解决这一难题。

## 输入格式

有多组测试数据。

第一行有一个数$\red{T}$，表示有$\red{T}$组测试数据。

接下来的`T行`，每一行有两个数$\red{K}$和$\red{N}$ ( $\red{0<= K <= 2^{31}}$ ,$\red{N}$小于$\red{ K}$的位数)。

## 输出格式

对每一组数据输出删除若干数字后剩下的数字组成的那个最大数。

## 样例

#### 输入样例

```C
1
917845 3
```

#### 输出样例

```C
985
```
