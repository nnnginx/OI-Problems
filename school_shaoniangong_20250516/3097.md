# Description

六一儿童节到了，huhe准备给他的N个学生发N个礼物。每个孩子都有自己的愿望，有的想要手提电脑，有的想要更多的题目......
原本，huhe只是打算按编号发礼物，例如i号同学得到i号奖品；
为了让大家都满意，他让每个人都按自己的喜欢程度，给N个礼物排了序，越在前面的越喜欢；
huhe并让他们自己分配礼物，以确保每人得到的礼物的优先级，都不低于他们最初能得到的礼物的优先级。

# Format

## Input

输入的第一行包含 N。以下 N行每行为每一位学生的清单。输入保证每行均为 1…N的一个排列。

## Output

重新分配后每位学生得到的奖品编号

# Samples

```input1
4
1 2 3 4
1 3 2 4
1 2 3 4
1 2 3 4
```

```output1
1
3
2
4
```

【样例分析】
在这个例子中，有两种可能的重新分配方式：

初始的分配方式：

同学 1收到礼物 1，同学 2收到礼物 2，同学 3收到礼物 3，同学 4收到礼物 4。


重新分配后：

同学 1收到礼物 1，同学 2收到礼物 3，同学 3收到礼物 2，同学 4收到礼物 4。



可以看出：到同学 1和 4均不可能收到比她们最初得到的礼物更好的礼物。然而，同学 2

和 3可以。

# Limitation

* 测试点 2-3 满足 **N**≤**8**。
* 测试点 4-11 没有额外限制。
