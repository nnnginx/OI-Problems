## 题目描述

佳佳碰到了一个难题，请你来帮忙解决。对于不定方程 $\red{ a_1+a_2+\cdots +a_{k-1}+a_k=g(x)}$，其中$\red{ k\ge 2 }$且$\red{ k\in \mathbb{N}}$，$\red{x}$ 是正整数，$\red{g(x)=x^x \bmod 1000}$（即$\red{ x^x}$ 除以$\red{ 1000 }$的余数），$\red{x,k}$ 是给定的数。我们要求的是这个不定方程的正整数解组数。

举例来说，当$\red{ k=3,x=2 }$时，方程的解分别为：

$\red{\begin{cases} a_1=1\\ a_2=1\\ a_3=2 \end{cases} }$$\red{\ \ \ \ \begin{cases} a_1=1\\ a_2=2\\ a_3=1 \end{cases} }$$\red{\ \ \ \ \begin{cases} a_1=2\\ a_2=1\\ a_3=1 \end{cases}}$

## 输入格式

有且只有一行，为用空格隔开的两个正整数，依次为 $\red{k,x}$。

## 输出格式

有且只有一行，为方程的正整数解组数。

## 样例

#### 输入样例

```c
3 2
```



#### 输出样例

```c
3
```



## 数据范围与提示

对于$\red{ 40\% }$数据，答案不超过$\red{ 10^{16}}$；
对于全部数据，$\red{1\le k\le 100}$,$\red{1\le x\lt 2^{31}}$,$\red{k\le g(x)}$。
