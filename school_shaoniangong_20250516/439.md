## 题目描述

来自 `ICPC CERC 1999/2000`，有改动。

有$\red{ N }$个盘子，每个盘子上写着一个仅由小写字母组成的英文单词。你需要给这些盘子安排一个合适的顺序，使得相邻两个盘子中，前一个盘子上单词的末字母等于后一个盘子上单词的首字母。请你编写一个程序，判断是否能达到这一要求。如果能，请给出一个合适的顺序。

## 输入格式

多组数据。第一行给出数据组数$\red{ T}$，每组数据第一行给出盘子数量 $\red{N}$，接下去$\red{ N }$行给出小写字母字符串，一种字符串可能出现多次。

## 输出格式

若存在一组合法解输出`Ordering is possible.`，否则输出`The door caNot be opened.`。

## 样例

#### 输入样例

```c
3
2
acm
ibm
3
acm
malform
mouse
2
ok
ok
```

#### 输出样例

```c
The door caNot be opened.
Ordering is possible.
The door caNot be opened.
```

## 提示

$\red{1 \le N \le 10^5,|S| \le 1000}$
