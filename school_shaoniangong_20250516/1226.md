## 题目描述

任何一个正整数都可以用2的幂次方表示。例如：$\red{137=2^7+2^3+2^0}$

同时约定方次用括号来表示，即$\red{a^b}$可表示为$\red{a(b)}$。由此可知，$\red{137}$可表示为：$\red{2(7)+2(3)+2(0)}$

进一步：$\red{7=2^2+2+2^0}$（$\red{2^1}$用$\red{2} $表示）$\red{3=2+2^0}$

所以最后$\red{137}$可表示为：$\red{2(2(2)+2+2(0))+2(2+2(0))+2(0)}$

又如：$\red{1315=2^{10}+2^8+2^5+2+1}$

所以 $\red 1315$ 最后可表示为：$\red{2(2(2+2(0))+2)+2(2(2+2(0)))+2(2(2)+2(0))+2+2(0)}$

## 输入格式

一个正整数 $\red n$（ $\red{ n≤20000}$ ）。

## 输出格式

一行，符合约定的 $\red n$ 的 $\red 0$，$\red 2$ 表示（在表示中不能有空格）。

## 样例

#### 输入样例

```C
137
```

#### 输出样例

```C
2(2(2)+2+2(0))+2(2+2(0))+2(0)
```
