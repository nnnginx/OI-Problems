## 题目描述

老师在开学第一天就把所有作业都布置了，每个作业如果在规定的时间内交上来的话才有学分。每个作业的截止日期和学分可能是不同的。例如如果一个作业学分为 $\red{10}$，要求在 $\red{6}$ 天内交，那么要想拿到这 $\red{10}$ 学分，就必须在第 $\red{6}$ 天结束前交。

每个作业的完成时间都是只有一天。例如，假设有 $\red{7}$ 次作业的学分和完成时间如下：

| 作业号 | 期限   | 学分   |
| -------- | -------- | -------- |
| $\red{1}$ | $\red{1}$ | $\red{6}$ |
| $\red{2}$ | $\red{7}$ |
| $\red{3}$ | $\red{3}$ | $\red{2}$ |
| $\red{4}$ | $\red{1}$ |
| $\red{5}$ | $\red{2}$ | $\red{4}$ |
| $\red{6}$ | $\red{5}$ |
| $\red{7}$ | $\red{6}$ | $\red{1}$ |

最多可以获得 $\red{15}$ 学分，其中一个完成作业的次序为$\red {2,6,3,1,7,5,4}$，注意可能还有其他方法。

你的任务就是找到一个完成作业的顺序获得最大学分。

## 输入格式

第一行一个整数 $\red{N}$，表示作业的数量；

接下来 $\red{N}$ 行，每行包括两个整数，第一个整数表示作业的完成期限，第二个数表示该作业的学分。

## 输出格式

输出一个整数表示可以获得的最大学分。保证答案不超过 `C/C++` 的 `int` 范围。

## 样例

#### 输入样例

```c
7
1 6
1 7
3 2
3 1
2 4
2 5
6 1
```

#### 输出样例

```c
15
```

## 提示

对于 $\red{20\%}$ 的数据，$\red{N \leq 10^3}$；

对于 $\red{40\%}$ 的数据，$\red{N \leq 10^4}$；

对于 $\red{60\%}$ 的数据，$\red{N \leq 10^5}$；

对于 $\red{100\%}$ 的数据，$\red{N \leq 10^6}$，作业的完成期限均小于 $\red{7\times 10^5}$。
