## 题目描述

计算机是一个非常“内卷”的行业。今天，一群程序猿们在排队干饭时，居然也发生了“内卷”。

他们首先在食堂门口排成一列，准备开饭。但这时，程序猿们发现，由于大家的身高不同，有的猿会被前面的猿挡住。

现在程序猿们想知道，对于他们每一只猿，有多少排在他前面的猿比他严格矮。

## 输入格式

第一行一个正整数 $n$，表示程序猿们的数量。

随后一行 $n$ 个正整数 $a_1, a_2, a_3, \ldots, a_n$，以空格分隔，其中 $a_i$ 表示从前往后第 $i$ 只程序猿的身高。

## 输出格式

一行 $n$ 个整数 $ans_1, ans_2, ans_3, \ldots, ans_n$，以空格分隔，表示从前往后第 $i$ 只程序猿前面有 $ans_i$ 只程序猿严格比他矮。

## 样例

#### 样例输入

```
4
3 7 2 4
```

#### 样例输出

```
0 1 0 2
```

## 数据范围与提示

$1 \leq n \leq 5,000$

$1 \leq a_i \leq 10,000$
