## 题目描述

莫斯科正在举办一个大型国际会议，有$\red{n}$个来自不同国家的科学家参会。

每个科学家都只懂得一种语言。

为了方便起见，我们把世界上的所有语言用$\red{1}$到$\red{10^9}$之间的整数编号。

在会议结束后，所有的科学家决定一起去看场电影放松一下。

他们去的电影院里一共有$\red{m}$部电影正在上映，每部电影的语音和字幕都采用不同的语言。

对于观影的科学家来说，如果能听懂电影的语音，他就会很开心；如果能看懂字幕，他就会比较开心；

如果全都不懂，他就会不开心。

现在科学家们决定大家看同一场电影。

请你帮忙选择一部电影，可以让观影很开心的人最多。

如果有多部电影满足条件，则在这些电影中挑选观影比较开心的人最多的那一部。

## 输入格式

第一行输入一个整数$\red{n}$，代表科学家的数量。

第二行输入$\red{n}$个整数$\red{a_1,a_2…a_n}$,其中$\red{a_i}$表示第$\red{i}$个科学家懂得的语言的编号。

第三行输入一个整数$\red{m}$，代表电影的数量。

第四行输入$\red{m}$个整数$\red{b_1,b_2…b_m}$,其中$\red{b_i}$表示第i部电影的语音采用的语言的编号。

第五行输入$\red{m}$个整数$\red{c_1,c_2…c_m}$,其中$\red{c_i}$表示第i部电影的字幕采用的语言的编号。

请注意对于同一部电影来说，$\red{b_i≠c_i}$ 。

同一行内数字用空格隔开。

## 输出格式

输出一个整数，代表最终选择的电影的编号。
如果答案不唯一，输出任意一个均可。

## 样例

```C
3

```

#### 输出数据 1


## 提示

$\red{1≤n,m≤200000}$,

$\red{1≤a_i,b_i,c_i ≤10^9}$
