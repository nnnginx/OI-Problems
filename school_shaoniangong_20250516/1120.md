## 题目描述

奶牛们灰常享受在牛栏中牟叫，因樗们可以听到她们牟声的回音。虽然有时候并不能完全听到完整的回音。

`Bessie`曾经是一个出色的秘书，所以她精确地纪录了所有的牟叫声及其回声。她很好奇到底两个声音的重复部份有多长。

输入两个字符串（长度$\red{1}$`~`$\red{80}$个字母），表示两个牟叫声。你要确定最长的重复部份的长度。

两个字符串的重复部份指的是同时是一个字符串的前缀（从第一个字符开始的连续字符串）和另一个字符串的后缀（从最后一个字符开始的连续字符串）的字符串。

我们通过一个例子来理解题目。考虑下面的两个牟声：

<pre>
moyooyoxyzooo

yzoooqyasdfljkamo
</pre>

第一个串的最后的部份"`yzooo`"跟第二个串的第一部份重复。第二个串的最后的份"`mo`"跟第一个串的第一部份重复。
所以"`yzooo`"跟"`mo`"都是这$\red{2}$个串的重复部份。其中，"`yzooo`"比较长，所以最长的重复部份的长度就是$\red{5}$。

### 输入格式

输入只有共两行，第一行为奶牛的牟叫，第二行为奶牛牟叫的回声

### 输出格式

输出为一行，包含一个单独的整数表示输入的$\red{2}$个字符串中，一个字符串的前缀和另一个字符串的后缀的最长的重复部份的长度。

## 样例

### 输入样例

```C
abcxxxxabcxabcd
abcdxabcxxxxabcx
```

### 输出样例

```C
11
```
