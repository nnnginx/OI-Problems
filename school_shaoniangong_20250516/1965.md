## 题目描述

小明有很多张 $1$ 和 $-1$ 的数字卡片。他今天想选出 $6k$ 张数字卡片（$k$ 为正整数），让它们的和为 $n$。如果可行，输出应该选 $1$ 和 $-1$ 的张数，否则输出 "NO"（不含引号）。

## 输入格式

一行一个正整数 $n$。

## 输出格式

一行两个正整数，为 $1$ 和 $-1$ 的张数（如果有多种方案，输出总张数最少的）。

## 样例 #1

### 样例输入 #1

```
32
```

### 样例输出 #1

```
34 2
```

## 样例 #2

### 样例输入 #2

```
17
```

### 样例输出 #2

```
NO
```

## 提示

样例 $1$：$34-2=32$，可以证明这位是总张数最少的一种方案。

样例 $2$：也可以证明不存在符合要求的方案。

$1\le n\le10^5$。
