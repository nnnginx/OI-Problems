## 题目描述

给出两个正整数 $\red{a,b}$。

在十进制下重排 ，构造一个不超过 $\red{b}$的最大数，不能有前导零。允许不去重排 $\red{a}$。

## 输入格式

共两行。

第一行一个数字表示 $\red{a}$

第二行一个数字表示 $\red{b}$

## 输出格式

一个整数，表示答案。

## 样例

#### 输入样例1

```c
123 

222
```

#### 输出样例1

```c
213
```

#### 输入样例2

```c
3921 

10000
```

#### 输出样例2

```c
9321
```

#### 输入样例3

```c
4940 

5000
```

#### 输出样例3

```c
4940
```

## 提示

对于$\red{30\%}$的数据，$\red{1≤}$$\red{a≤}$$\red{b≤}$$\red{10^3}$

对于另外$\red{30\%}$的数据，$\red{len_a==len_b}$

对于$\red{100\%}$的数据，有$\red{1≤}$$\red{a≤}$$\red{b≤}$$\red{10^{18}}$。
