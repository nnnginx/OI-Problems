## 题目描述

给定一个整数，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例$\red2$）。

## 输入格式

输入共$\red1$ 行，一个整数$\red{N}$

## 输出格式

输出共$\red{1}$ 行，一个整数，表示反转后的新数。

## 样例

#### 样例输入1

```C
123
```

#### 样例输出1

```C
321
```

#### 样例输入2

```C
-380
```

#### 样例输出2

```C
-83
```

## 数据范围与提示

【数据范围】
$\red{-1,000,000,000 ≤ N ≤ 1,000,000,000 }$。
