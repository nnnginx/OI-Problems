## 题目描述

农民约翰为他的奶牛购买了一份好的《护蹄》杂志的订阅，因此在挤奶期间，奶牛们在谷仓里等待时有很多材料可以阅读。不幸的是，最新一期包含了一篇关于如何烹饪完美牛排的相当不恰当的文章，$\red{FJ}$宁 愿他的奶牛看不到这篇文章（显然，该杂志需要更好的编辑监督）。

$\red{FJ}$从杂志中获取了所有文本，以创建长度不超过$\red{10^6}$个字符的字符串。因此，他希望删除长度小于等于$\red{100}$个字符的子字符串$\red{T}$的出现，以审查不适当的内容。为此，农夫约翰在$\red{S}$中找到$\red{T}$的第一个出现点，并将其删除。然后，他再次重复该过程，再次删除第一个出现的$\red{T，}$直到$\red{S}$中不再出现$\red{T}$。请注意，删除一个出现的$\red{T}$可能会创建一个以前不存在的新出现的$\red{T}$。

审查完成后，请帮助$\red{FJ}$确定$\red{S}$的最终内容。

## 输入格式

第一行将包含$\red{S}$。

第二行将包含$\red{T}$。$\red{T}$的长度最多为$\red{S}$的长度，$\red{S}$和$\red{T}$的所有字符将是小写字母字符（在$\red{a...z}$范围内）。

## 输出格式

所有删除完成后的字符串。可以保证$\red{S}$在删除过程中不会变为空。

## 样例

#### 输入样例

```c
whatthemomooofun
moo
```

#### 输出样例

```c
whatthefun
```
