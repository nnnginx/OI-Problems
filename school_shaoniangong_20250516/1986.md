## 题目描述

小$\red{H}$与小$\red{Y}$刚刚参加完$\red{UOIP}$外卡组的初赛，就迫不及待的跑出考场对答案。

"剑我的答案和你都不一样！"，小$\red{Y}$说道，"我们去找神拿俏蚀鸢赴"。

外卡组试卷中共有$\red{m}$道判断题，小$\red{H}$与小$\red{Y}$一共从其他$\red{n}$个神哪俏柿舜鸢浮Ｖ后又从小$\red{G}$那里得知，这$\red{n}$个神闹杏$\red{p}$个考了满分，$\red{q}$个考了零分， 其他神牟晃满分或零分。这可让小$\red{Y}$与小$\red{H}$犯了难。你能帮助他们还原出标准答案吗？

如有多解则输出字典序最小的那个。无解输出$\red{-1}$。

## 输入格式

第一行四个整数$\red{n, m, p, q，}$意义如上描述。

接下来$\red{n}$行，每一行$\red{m}$个字符’$\red{N}$’或’$\red{Y}$’，表示这题这个神牡拇鸢浮

## 输出格式

仅一行，一个长度为$\red{m}$的字符串或是$\red{-1}$。

## 样例

#### 输入样例

```c
2 2 2 0
YY
YY
```

#### 输出样例

```c
YY
```

## 提示

$\red{30\% : n <= 100.}$

$\red{60\% : n <= 5000 , m <= 100.}$

$\red{100\% : 1 <= n <= 30000 , 1 <= m <= 500.  0 <= p , q }$且 $\red{p + q <= n. }$
