## 题目描述

某校大门外长度为$\red{L}$的马路上有一排树，每两棵相邻的树之间的间隔都是$\red{1}$米。我们可以把马路看成一个数轴，马路的一端在数轴$\red{0}$的位置，
另一端在$\red{L}$的位置；数轴上的每个整数点，即 $\red{0，1，2，……，L}$，都种有一棵树。

由于马路上有一些区域要用来建地铁。这些区域用它们在数轴上的起始点和终止点表示。已知任一区域的起始点和终止点的坐标都是整数，
区域之间可能有重合的部分。现在要把这些区域中的树（包括区域端点处的两棵树）移走。你的任务是计算将这些树都移走后，马路上还有多少棵树。

## 输入格式

第一行有两个整数$\red{L（1 ≤ L ≤ 10000）}$和 $\red{M（1 ≤ M ≤ 100）}$，$\red{L}$代表马路的长度，$\red{M}$代表区域的数目，$\red{L}$和$\red{M}$之间用一个空格隔开。
接下来的$\red{M}$行每行包含两个不同的整数，用一个空格隔开，表示一个区域的起始点和终止点的坐标。

## 输出格式

包括一行，这一行只包含一个整数，表示马路上剩余的树的数目。

## 样例

#### 输入样例

```c
500 3
150 300
100 200
470 471
```

#### 输出样例

```c
298
```
