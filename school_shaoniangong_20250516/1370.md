
## 题目描述
面面上小学二年级了，有一天妈妈去超市买了两件物品$\red a$、$\red b$，妈妈问面面，你能告诉我这两件商品的价格比吗？

这当然难不倒聪明的面面了，面面很快就用编程算出了两件商品的价格比，`价格比 = 商品a的价格 / 商品b的价格`。

## 输入格式
共两行，第一行一个正整数表示物品$\red a$的价格$\red x$，第二行一个正整数表示物品$\red b$的价格$\red y$


## 输出格式
一行，表示商品的价格比，保留到小数点后$\red 4$位。
## 样例
#### 输入样例
```C
1
8
```
#### 输出样例
```C
0.1250
```
## 提示
数据范围
对于所有的数据$\red {1<=x<=1000,1<=y<=100}$;
