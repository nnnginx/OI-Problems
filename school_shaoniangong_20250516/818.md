## 题目描述

一只大象口渴了，要喝$\red{20}$升水才能解渴，但现在只有一个深$\red{h}$厘米，底面半径为$\red{r}$厘米的小圆桶($\red{h}$和$\red{r}$都是整数)。

问大象至少要喝多少桶水才会解渴。

## 输入格式

输入有一行：包行两个整数，以一个空格分开，分别表示小圆桶的深$\red{h}$和底面半径$\red{r}$，单位都是厘米。

## 输出格式

输出一行，包含一个整数，表示大象至少要喝水的桶数。

## 样例

#### 输入样例

```C
23 11
```

#### 输出样例

```C
3
```

## 提示

$\red{PI = 3.14159}$

$\red{1}$升 $\red =$ $\red{1000}$毫升

$\red{1}$毫升 $\red=$ $\red{1}$ 立方厘米
