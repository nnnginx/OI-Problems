## 题目描述

某程序员开始工作，年薪$\red{N}$万，他希望在中关村公馆买一套$\red{60}$平米的房子，现在价格是$\red{200}$万，
假设房子价格以每年百分之$\red{K}$增长，并且该程序员未来年薪不变，且不吃不喝，不用交税，
每年所得$\red{N}$万全都积攒起来，问第几年能够买下这套房子？（第一年年薪N万，房价$\red{200}$万）

## 输入格式

一行，包含两个正整数$\red{N}$（$\red{10 <= N <= 50}$）, $\red{K}$（$\red{1 <= K <= 20}$），中间用单个空格隔开。

## 输出格式

如果在第20年或者之前就能买下这套房子，则输出一个整数$\red{M}$，表示最早需要在第$\red{M}$年能买下，否则输出"`Impossible`"。

## 样例

#### 输入数据

```C
50 10
```

#### 输出数据

```C
8
```
