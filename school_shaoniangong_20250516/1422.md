## 题目描述

在古时候，希腊的某个城堡里住着一位暴戾的国王，他酷爱数学，但是他有一个坏毛病，对于数学中的数字极其敏感。

特别是对于数字“`0`”，传言，去觐见的大臣们但凡奏折中出现数字$\red 0$都要被拉出去砍头。

聪明的你，能否帮帮大臣们实现数字去$\red 0$呢？

对于一个正整数$\red n$,我们要做的是一旦出现零就进行删除，输出处理后的整数

## 输入格式

共一行：
包含一个正整数$\red n$

## 输出格式

共一行：为删除$\red 0$后的正整数

## 样例

#### 输入样例

```C
1209
```

#### 输出样例

```C
129
```

## 提示

所有数据均满足：
$\red{1 \le n \lt 10^9}$。
