## 题目描述

有 $\red {N}$ 个任务排成一个序列在一台机器上等待执行，它们的顺序不得改变。

机器会把这 $\red {N}$ 个任务分成若干批，每一批包含连续的若干个任务。

从时刻$\red 0$开始，任务被分批加工，执行第 $\red {i}$ 个任务所需的时间是 $\red {T_i}$ 。

另外，在每批任务开始前，机器需要 $\red {S}$ 的启动时间，故执行一批任务所需的时间是启动时间 $\red {S}$ 加上每个任务所需时间之和。

一个任务执行后，将在机器中稍作等待，直至该批任务全部执行完毕。

也就是说，同一批任务将在同一时刻完成。

每个任务的费用是它的完成时刻乘以一个费用系数 $\red {C_i}$。

请为机器规划一个分组方案，使得总费用最小。

## 输入格式

第一行包含两个整数 $\red {N}$ 和 $\red {S}$。

接下来$\red N$行每行有一对整数，分别为 $\red {T_i}$ 和 $\red {C_i}$ ，表示第 $\red {i}$ 个任务单独完成所需的时间 $\red {T_i}$ 及其费用系数 $\red {C_i}$ 。

## 输出格式

输出一个整数，表示最小总费用。

## 样例

#### 输入样例

```c
5 1
1 3
3 2
4 3
2 3
1 4
```

#### 输出样例

```c
153
```

## 提示

$\red {1≤N≤3\times 10 ^5}$,

$\red {0≤S,C_i ≤512}$,

$\red {−512≤T_i ≤512}$
