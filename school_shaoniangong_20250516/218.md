## 题目描述

求 $\red {N}$ 个节点的无向连通图有多少个，节点有标号，编号为$\red {1\sim N}$。

例如下列图示，三个节点的无向连通图共$\red {4}$个。

![img](./218/file/3.jpg)

## 输入格式

输入包含多组测试数据。

每组数据包含一个整数$\red {N}$。

当输入为$\red {0}$时，表示输入终止。

## 输出格式

每组测试数据输出一个结果，每个结果占一行。

## 样例

#### 输入样例

```c
1
2
3
4
0
```

#### 输出样例

```c
1
1
4
38
```

## 提示

$\red {1≤N≤50}$
