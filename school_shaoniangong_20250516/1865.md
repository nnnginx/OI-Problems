## 题目描述

输人$\red{a}$，求出一个最小的$\red{n}$和$\red{k(n>k≥a)}$，使得能在$\red{1\sim n}$之间能找到一个$\red{k}$，且$\red{1\sim k-1}$的和等于$\red{k+1\sim n}$的和。

## 输入格式

输人一行一个正整数$\red{a(3≤a≤1\ 940\ 500)}$。

## 输出格式

输出一行$\red{2}$个数,表示最小的正整数$\red{k}$和$\red{n}$(严格用一个空格隔开)。

## 样例

#### 输入样例

```c
3
```

#### 输出样例

```c
6 8
```

## 提示

【样例说明】
$\red{n=8,k=6,1+2+3+4+5=15=7+8}$
