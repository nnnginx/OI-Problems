## 题目描述

企业发放的奖金根据利润提成。

- 利润低于或等于$\red{100000}$元的，奖金可提$\red{10\%}$；
- 利润高于$\red{100000}$元，利润低于$\red{200000}$元的（$\red{100000<N<≤200000}$）,低于部分按$\red{10\%}$提成，高于部分可提$\red{7.5\%}$；
- $\red{200000<N≤400000}$时，低于$\red{200000}$部分按上述方法提成（下同）。高于部分按$\red{5\%}$提成
- $\red{400000<N≤600000}$时,高于$\red{400000}$部分按$\red{3\%}$提成；
- $\red{600000<N≤1000000}$时,高于$\red{600000}$部分按$\red{1.5\%}$提成；
- $\red{N>1000000}$时，高于部分按$\red{1\%}$提成。输入利润$\red{N}$，计算奖金。

## 输入格式

共一行，一个数$\red{N}$

## 输出格式

共一行，奖金金额，保留小数点后两位

## 样例

#### 输入数据

```C
50000
```

#### 输出数据

```C
5000.00
```
