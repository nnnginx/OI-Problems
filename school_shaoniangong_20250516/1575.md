## 题目描述

大于$\red{1 }$的正整数$\red{n}$可以分解为：$\red{n=x1×x2×…×xm}$。
例如，当$\red{n=12 }$时，共有$\red{8}$ 种不同的分解式：

$\red{12=12}$；$\red{12=6×2}$；$\red{12=4×3}$；$\red{12=3×4}$；$\red{12=3×2×2}$；
$\red{12=2×6}$；$\red{12=2×3×2}$；$\red{12=2×2×3}$。

编程任务：对于给定的正整数$\red{n}$，编程计算$\red{n}$共有多少种不同的分解式。

## 输入格式

第一行有$\red{1}$ 个正整数$\red{n (1≤n≤2000000000)}$。

## 输出格式

计算出不同的分解式数

## 样例

#### 输入样例

```c
12
```

#### 输出样例

```c
8
```
