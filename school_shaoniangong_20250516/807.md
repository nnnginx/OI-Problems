## 题目描述

第一行两个整数 $\red{n , a_0}$ 。

## 输入格式

一行$\red n$个整数$\red {a_1,a_2,....,a_n}$ ，其中 $\red{a_i = rnd(a_{i-1}) }$，$\red {rnd} $ 函数的一个 $\red {C++}$ 实现如下：

```c++
int rnd(unsigned x) {
	x ^= x << 13;
	x ^= x >> 17;
	x ^= x << 5;
	return x;
    }
```

## 输出格式

输出每一个$\red{a_i}$ 下的结果，每两个结果之间用空格隔开

## 样例

#### 输入样例

```c
5 5
```

#### 输出样例

```c
1351845 336141829 -822273599 -714806461 1234782181
```

## 提示

对于$\red {100\%} $ 的数据 ，$\red { n = 10^7 , -2^{31} \le a_i \lt  2^{31} } $
