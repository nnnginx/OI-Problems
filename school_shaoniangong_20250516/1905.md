## 题目描述

给出一个由小写字母组成的字符

## 输入格式

第一行一个字符串$\red{S}$

第二行一个整数$\red{k}$串$\red{S，}$求排名为$\red{k}$的后缀。

## 输出格式

一行。排名为$\red{k}$的字符窜后缀。

## 样例

#### 输入样例

```c
dasfh
3
```

#### 输出样例

```c
fh
```

## 提示

$\red{1<=len(s)<=2000}$

$\red{1<=k<=len(s)}$

样例说明

字符串的某一个后缀是指从这个字符串的某一位一直到最后一位所组成的字符串。比如样例字符串的所有后缀为$\red{dasfh，}$$\red{asfh，}$$\red{sfh，}$$\red{fh，}$$\red{h}$。按字典序排好后第三个就是$\red{fh}$
