## 题目描述

从前，有 $\red{n }$只兔子在森林里迷了路。于是它们便安营扎寨，等待着其他兔子前来救援。
不幸的是，森林里有大灰狼，它可能会在夜晚来袭击兔子们，所以每天晚上必须要有兔子来
站岗。

出于安全方面的考虑，每天需要至少三只兔子站岗，这样才能保证站岗的兔子自身的安
全。不过站岗的兔子也不老实。假如某两只兔子已经一起站岗过超过三次了（不一定是连续
的三次），它们就会互相产生感情。这显然是对兔子的野外生存很不利的，所以要尽量避免。
现在请问兔子们最多能安全度过多少天，并且保证没有任何兔子之间产生感情。

## 输入格式

一个正整数 $\red{n ，}$表示有$\red{n }$只兔子。

## 输出格式

第一行输出一个整数 $\red{ans ，}$表示最多可以安全度过 $\red{ans }$天。

接下来 $\red{ans }$行，每行若干个整数，表示这一天派出去守夜的兔子。如果有多组解，输出
任意一组即可。

## 样例

#### 输入样例

```c
3
```

#### 输出样例

```c
3 
1 2 3 
1 2 3 
1 2 3
```

## 提示

对于 $\red{50\% }$的数据，保证$\red{n <=99 }$。

对于 $\red{100\% }$的数据，保证$\red{3<=n<=999，}$且 $\red{n }$为奇数。
