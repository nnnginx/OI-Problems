## 题目描述

的死对头 $\red{Farmer Paul }$决定破坏 $\red{Farmer John }$的挤奶设备！

挤奶设备由一排$\red{N (3 <= N <= 100,000) }$台挤奶机组成，其中第 $\red{i }$台挤奶机生产 $\red{M_i }$个单位的牛奶 $\red{(1 <= M_i <= 10,000)}$。

$\red{Farmer Paul }$计划断开这些机器的连续块$\red{--}$从第 $\red{i }$台机器到第 $\red{j }$台机器（$\red{2 <= i <= j <= N-1）}$；请注意，$\red{Farmer Paul }$不想断开第一台或最后一台机器的连接，因为这会使他的情节太容易被发现。

$\red{Farmer Paul }$的目标是尽量减少剩余机器的平均产奶量。

农民保罗计划移除至少$\red{1 }$头奶牛，即使他最好完全避免破坏。

幸运的是，农夫约翰了解了农夫保罗他的邪恶阴谋，他想知道如果阴谋得逞，他的产奶量会受到多大的影响。

如果$\red{Farmer Paul }$成功，请帮助 $\red{Farmer John }$算出剩余机器的最低平均产奶量。

## 输入格式

第$\red{1}$行：整数$\red{N}$。

第$\red{2...}$$\red{1+N}$行:$\red{i+1}$行包含$\red{M_i}$。

## 输出格式

第$\red{1}$行：一个实数， 表示平均牛奶产量的最小值， 保留三位小数 （四舍五入）。

## 样例

#### 输入样例

```c
5
5
1
7
8
2
```

#### 输出样例

```c
2.667
```

## 提示

样例说明
移去 $\red{7 }$和 $\red{8，}$剩下 $\red{5, 1, 2，}$平均值为 $\red{8/3}$。

数据规模和约定

对于 $\red{30\%}$的数据，$\red{N <= 1,000}$。
对于 $\red{50\%}$的数据，$\red{N <= 10,000}$。
对于 $\red{100\%}$的数据，$\red{3 <= N <= 100,000，}$$\red{1 <= M_i <= 10,000}$。

时空限制

$\red{0.2s/128M}$
