## 题目描述

庄力要把苹果都拿起来，苹果标号从左往右依次是 $1,2,3...n$。

他希望找到一个**最大**的 $K$，使得：

1. 第一次必须要拿第一个苹果。
2. 以后拿苹果需要与上一次取苹果隔着 $K$ 个苹果以上（以前被拿掉过的苹果也算）。

庄力最后要将所有苹果取完，他想知道最大的 $K$ 是多少。

## 输入格式

第一行一个整数 $n$，表示有 $n$ 个苹果在桌上。

## 输出格式

输出一个最大的 $K$，表示可以按上述格式取完苹果的最大整数。如果找不到最大的 $K$，请输出 `ovo`。

## 样例 #1

### 样例输入 #1

```
5
```

### 样例输出 #1

```
1
```

## 样例 #2

### 样例输入 #2

```
3
```

### 样例输出 #2

```
0
```

## 提示

#### 样例1解释

可以按 $1,3,5,2,4$ 的顺序取苹果。

#### 样例2解释

一共有 $1,2,3$ 和 $1,3,2$ 两种取法，显然这两种取法都为 $0$。

### 数据范围

对于 $20\%$ 的数据，满足 $1 \le n \le 10$。

对于 $30\%$ 的数据，满足 $1 \le n \le 200$。

对于 $40\%$ 的数据，满足 $n$ 是奇数。

对于 $100\%$ 的数据，满足 $0 \le K \le n,1 \le n \le 10^{6}$。
