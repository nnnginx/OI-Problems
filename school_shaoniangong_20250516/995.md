## 题目描述

在一个序列(下标从$\red{1}$开始)中查找一个给定的值，输出第一次出现的位置。

## 输入格式

`第一行`包含一个正整数$\red{n}$，表示序列中元素个数。$\red{1 ≤n≤ 10000}$。

`第二行至第n+1行`包含$\red{n}$个整数，依次给出序列的每个元素。元素的绝对值不超过$\red{10000}$。

`第n+2行`包含一个整数$\red{x}$，为需要查找的特定值。x的绝对值不超过$\red{10000}$。

## 输出格式

若序列中存在$\red{x}$，输出$\red{x}$第一次出现的下标；

否则输出$\red{-1}$。

## 样例

#### 输入样例

```C
5
2 
3 
6 
7 
3
3
```

#### 输出样例

```C
2
```
