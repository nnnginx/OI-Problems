## 题目描述

牧羊人$\red{A}$和牧羊人$\red{B}$总是很无聊，所以他们要玩一个游戏。

$\red{A}$有$\red{a}$只羊，$\red{B}$有$\red{b}$只羊。他们想要知道$\red{a^b}$的因子和是多少。

这就很为难两个牧羊人了，由于答案太大，你能不能告诉我答案取模$\red{9901}$的数。

## 输入格式

仅一行，为两个正整数$\red{a}$和$\red{b }$

## 输出格式

$\red{a^b}$的因子和对$\red{9901}$的余数。

## 样例

#### 输入样例

```c
2 3
```

#### 输出样例

```c
15
```

## 提示

数据范围

对于$\red{100\%}$的数据，$\red{0≤}$$\red{a,b≤}$$\red{50000000}$
