## 题目描述

为了准备一个独特的颁奖典礼，组织者在会场的一片矩形区域（可看做是平面直角坐标系的第一象限）铺上一些矩形地毯。

一共有 $\red{n}$ 张地毯，编号从 $\red{1}$ 到 $\red{n}$ 。现在将这些地毯按照编号从小到大的顺序平行于坐标轴先后铺设，后铺的地毯覆盖在前面已经铺好的地毯之上。

地毯铺设完成后，组织者想知道覆盖地面某个点的最上面的那张地毯的编号。

注意：在矩形地毯边界和四个顶点上的点也算被地毯覆盖。

## 输入格式

输入共 $\red{n+2}$ 行。

第一行，一个整数 $\red{n}$，表示总共有 $\red{n}$ 张地毯。

接下来的 $\red{n}$ 行中，第 $\red{i+1}$ 行表示编号 $\red{i}$ 的地毯的信息，包含四个正整数 $\red{a，b，g，k}$ ，每两个整数之间用一个空格隔开，分别表示铺设地毯的左下角的坐标 $\red{(a,b)}$ 以及地毯在 $\red{x}$ 轴和 $\red{y}$ 轴方向的长度。

第 $\red{n+2}$ 行包含两个正整数 $\red{x}$ 和 $\red{y}$ ，表示所求的地面的点的坐标 $\red{(x,y)}$ 。

## 输出格式

输出共 $\red{1}$ 行，一个整数，表示所求的地毯的编号；若此处没有被地毯覆盖则输出 $\red{-1}$ 。

## 样例

## 样例输入1

```C
3
1 0 2 3
0 2 3 3
2 1 3 3
2 2
```

## 样例输出1

```C
3
```

## 样例一说明

如下图，$\red1$ 号地毯用实线表示，$\red2$ 号地毯用虚线表示，$\red3$ 号用双实线表示，覆盖点$\red{（2， 2）}$的最上面一张地毯是 $\red3$ 号地毯。

## 样例输入2

```C
3
1 0 2 3
0 2 3 3
2 1 3 3
4 5
```

## 样例输出2

```C
-1
```

## 样例二说明

如上图，$\red{1}$ 号地毯用实线表示，$\red{2}$ 号地毯用虚线表示，$\red{3}$ 号用双实线表示，点$\red{（4，5）}$ 没有被地毯覆盖，所以输出$\red{-1}$。

## 数据范围与提示

对于$\red{30\%}$ 的数据，有 $\red{n\leq2}$；

对于$\red{50\%}$ 的数据，$\red{a, b, g, k }$；

对于$\red{100\%}$ 的数据，有 $\red{0\leq n\leq10,000}$，$\red{0\leq a, b, g, k \leq10,000}$ 。
