## 题目描述

在一个$\red{2}$维平面上有两条传送带,每一条传送带可以看成是一条线段。两条传送带
分别为线段$\red{AB}$和线段$\red{CD}$。$\red{lxhgww}$在$\red{AB}$上的移动速度为$\red{P}$，在$\red{CD}$上的移动速度为$\red{Q,}$在平面上的移动速度$\red{R}$。现在$\red{lxhgww}$想从$\red{A}$点走到$\red{D}$点,他想知道最少需要走多长时间。

## 输入格式

输人数据第一行是$\red{4 }$个整数,表示$\red{A}$和$\red{B}$的坐标，分别为$\red{Ax,Ay,Bx,By}$;

第二行是$\red{4}$个整数,表示$\red{C}$和$\red{D}$的坐标，分别为$\red{Cx,Cy,Dx,Dy}$;

第三行是$\red{3}$个整数,分别是$\red{P,Q,R}$。

## 输出格式

输出数据为一行,表示$\red{lxhgww}$从$\red{A}$点走到$\red{D}$点的最短时间,保留到小数点后$\red 2$位。

## 样例

#### 输入样例

```c
0 0 0 100
100 0 100 100
2 2 1
```

#### 输出样例

```c
136.60
```

## 提示

【数据范围】
对于$\red{100\%}$的数据满足:$\red{1≤Ar,Ay,Bxr,By,Cx,Cy,Dx,Dy≤1000;1≤P,Q,R≤10}$
