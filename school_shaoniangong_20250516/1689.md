## 题目描述

-只蚂蚁落入一个$\red{M \times N}$的矩形绝境，其中M和N至少有一个数是奇数。矩形中有$\red{1}$到
$\red{MN- 1}$个可以滑动的方块，$\red{0}$代表空地，例如当$\red{M=4}$和$\red{N=3}$时，矩阵可能`如图7.7`所示。

![img](./1689/file/1.png)

通过移动空地周围的方块，移成`如图7.8`所示的状态才可逃脱。

![img](./1689/file/11.png)

例如$\red{M=4}$和$\red{N=3}$时的绝境移动顺序`如图7.9`所示。

![img](./1689/file/22.png)

现在给一个$\red{M× N}$的矩形绝境，试计算能否逃脱。

## 输入格式

包含多组测试数据，每组数据第一行为两个整数$\red{M}$和$\red{N（2≤M，N≤ 999）}$，随后M行为各组数据。全部测试数据结束后，以$\red{0~0}$结尾。

## 输出格式

每组测试数据给出答案，即是否能逃脱。能则`“YES”`，否则`“NO”`。

## 样例

#### 输入样例

```c
3 3
1 0 3
4 2 5
7 8 6
4 3
1 2 5
4 6 9
11 8 10
3 7 0
0 0
```

#### 输出样例

```c
YES
NO
```
