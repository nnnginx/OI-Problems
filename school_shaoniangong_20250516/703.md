## 题目描述

一元 $\red{n }$次多项式可用如下的表达式表示：

$\red{ f(x)=a_nx^n + a_{n-1}x^{n-1} + ... +a_1x+a_0，a_n≠0}$

其中，$\red{a_ix^i}$称为i次项，$\red{a_i}$称为$\red{i}$次项的系数。

给出一个一元多项式各项的次数和系数，请按照如下规定的格式要求输出该多项式：

- `1.` 多项式中自变量为$\red{x}$，从左到右按照次数递减顺序给出多项式。
- `2.` 多项式中只包含系数不为$\red{0}$的项。
- `3.` 如果多项式n次项系数为正，则多项式开头不出现“$\red{+}$”号，如果多项式$\red{n}$次项系数为负，则多项式以“$\red{-}$”号开头。
- `4.` 对于不是最高次的项，以“$\red{+}$”号或者“$\red{-}$”号连接此项与前一项，分别表示此项系数为正或者系数为负。紧跟一个正整数，表示此项系数的绝对值（如果一个高于$\red{0}$次的项，其系数的绝对值为$\red{1}$，则无需输出$\red{1}$）。如果$\red{ x}$的指数大于$\red 1$，则接下来紧跟的指数部分的形式为“$\red{x^b}$”，其中$\red{b}$为$\red{x}$的指数；如果$\red{x}$的指数为$\red{1}$，则接下来紧跟的指数部分形式为“$\red{x}$”； 如果x的指数为$\red{0}$，则仅需输出系数即可。
- `5.` 多项式中，多项式的开头、结尾不含多余的空格。

## 输入格式

共有`2 行`：

`第一行` $\red{1}$ 个整数 $\red{n}$，表示一元多项式的次数。

`第二行`有 $\red{n+1}$ 个整数，其中第 $\red{i}$ 个整数表示第 $\red{n-i+1}$ 次项的系数，每两个整数之间用空格隔开。

$\red{1 ≤ n ≤ 100}$，多项式各次项系数的绝对值均不超过$\red{100}$。

## 输出格式

共`1行`，按题目所述格式输出多项式。

## 样例

#### 输入样例1

```C
5
100 -1 1 -3 0 10
```

#### 输出样例1

```C
100x^5-x^4+x^3-3x^2+10
```

#### 输入样例2

```C
3
-50 0 0 1
```

#### 输出样例2

```C
-50x^3+1
```
