## 题目描述

有一队人,共$\red{30}$人，要过河，因为人数太多，要分批过河。

分批规则是所有人排成一队，排队的位置即为他们的编号。

然后进行报数，从 $\red 1$ 开始，数到 $\red {k(0<k<15)}$ 的人离开队伍，如此循环，直到队伍仅剩 $\red{15}$人为止。

小明想在第一批过河，请问他的第一次排队位置可以是哪些编号？

输出编号用英文逗号分隔，一行输出。

## 输入格式

输入一个$\red K$

## 输出格式

输出第一次过河的所有编号,

编号从小到大输出。

## 样例

#### 输入样例

```C
9
```

#### 输出样例

```C
1,2,3,4,10,11,13,14,15,17,20,21,25,28,29,
```
