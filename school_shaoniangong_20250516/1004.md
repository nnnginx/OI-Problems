## 题目描述

有$\red{N}$个人提着不同大小的水桶来接水，因为水桶大小不同，每个桶接满的时间也各不相同。

现在问你，给你每个人接水需要的时间，如何安排接水，让所有人等待的时间最少(等待时间为排队时间+接水时间）。

$\red{1<=N<=1000}$。

## 输入格式

第一行一个整数 $\red{N}$

后面有`N行`，每行一个整数$\red{A_i}$($\red{1<=A<=100}$)，第$\red{i}$个人接满水需要的时间，单位是秒。

## 输出格式

所有人把桶接满，等待时间的累加和。

## 样例

#### 输入样例

```C
3
2
8
3
```

#### 输出样例

```C
20
```

## 提示

解析：( 先第一个人，再第三个，最后第二个)
