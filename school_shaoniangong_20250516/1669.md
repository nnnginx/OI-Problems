## 题目描述

有两个长度都为$\red{n}$的正整数序列$\red{A}$和$\red{B}$,从$\red{A}$和$\red{B}$中各取其中的一个数
相加一共可以得到$\red{n^2}$个和。要求输出这$\red{n^2}$个和中最小的$\red{n}$个。

## 输入格式

第一行,一个正整数$\red{n}$。

第二行,$\red{n}$个用空格隔开的正整数,代表$\red{A}$序列。

第三行,$\red{n}$个用空格隔开的正整数,代表$\red{B}$序列。

## 输出格式

一行,依次是从小到大输出$\red{n}$个最小的和,每两个数之间用一个空格隔开。

## 样例

#### 输入样例

```c
3
266
148
```

#### 输出样例

```c
3 67
```

## 提示

$\red{50\%}$的数据:$\red{ n≤500,0< A_i,B_i≤1000000000}$;

$\red{100\%}$的数据: $\red{n≤100000}$。
