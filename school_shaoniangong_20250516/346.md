## 题目描述

在一个$\red{n}$个方格的国际象棋棋盘上，马（骑士）可以攻击的棋盘方格如图所示。棋盘 上某些方格设置了障碍，骑士不得进入。

![img](./346/file/20200108155147_47459.png)

对于给定的$\red{n}$个方格的国际象棋棋盘和障碍标志，计算棋盘上最多可以放置多少个骑 士，使得它们彼此互不攻击。

## 输入格式

第一行有$\red{2}$ 个正整数$\red{n}$ 和$\red{m ( 1 <= n <= 200, 0 <= m < n^2 )}$， 分别表示棋盘的大小和障碍数。接下来的$\red{m}$ 行给出障碍的位置。每行$\red{2}$ 个正整数，表示障 碍的方格坐标。

## 输出格式

计算出共存骑士数。

## 样例

#### 输入样例

```c
3 2
1 1
3 3
```

#### 输出样例

```c
5
```
