## 题目描述

英语老师布置了一个作业，要求把给定的两篇英文文章作比较，统计两篇文 章共同出现的不重复的单词数量，单词中出现的英文字母不区分大小写，

例如 `The` 和 `the` 是一个单词。

英语老师的作业开始把小明给难住了，手工一个一个地去查找比较，到深更 半夜也做不完啊!有没有什么好办法呢?

对了，最近不是学习 `python` 编程吗? 正好可以练练手。

不一会儿，小明就把 `python` 程序调试出来了，顺利完成任务， 总算可以早点睡觉了。

如果你是小明你能用 `python` 做出来吗?现在试一试吧! 

## 输入格式

两行，每行一个字符串，表示一篇英文文章。文章里的单词之间使用单个空格或 者英文的逗号、分号、句号、感叹号、问号隔开，除此之外，单词前后没有其他 字符或者多余的分隔符号
## 输出格式

一行一个整数，表示两篇文章共有不重复单词的数量 

## 样例
#### 输入样例
```C
Rose and I must stop speaking Arabic to each other.Right,Rose?
Rose nodded,and then said something in Arabic.
```
#### 输出样例
```C
3
```
