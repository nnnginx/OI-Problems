## 题目描述

循环数是那些不包括$\red{0}$且没有重复数字的整数(比如$\red{81362}$)并且还应同时具有一个有趣的性质, 就像这个例子:

如果你从最左边的数字开始(在这个例子中是$\red{8}$)向右数最左边这个数(如果数到了最右边就回到最左边),你会停止在另一个新的数字(如果停在一个相同的数字上，这个数就不是循环数).就像: $\red{8\ 1\ 3\ 6\ 2}$ 从最左边接下去数$\red{8}$个数字: $\red{1\ 3\ 6\ 2\ 8\ 1\ 3\ 6}$ 所以下一个数字是$\red{6}$

重复这样做 (这次从$\red{“6”}$开始数$\red{6}$个数字) 并且你会停止在一个新的数字上: $\red{2\ 8\ 1\ 3\ 6\ 2}$, 也就是$\red{2}$

再这样做 (这次数两个): $\red{8\ 1}$

再一次 (这次一个): $\red{3}$

又一次: $\red{6\ 2\ 8}$

这时你回到了起点,在经过每个数字一次后回到起点的就是循环数。如果你经过每一个数字一次以后没有回到起点, 你的数字不是一个循环数。

给你一个数字 $\red{M}$ (在$\red{1}$到$\red{9}$位之间), 找出第一个比 $\red{M}$大的循环数, 输出数据保证结果能用一个无符号长整型数（$\red{21}$亿）装下。 （追加提醒：循环数每个数位都必须要访问到）

## 输入格式

仅仅一行, 包括$\red{M}$

## 输出格式

仅仅一行，输出第一个比$\red M$大的循环数。

## 样例

#### 输入样例

```c
81361
```

#### 输出样例

```c
81362
```

## 提示

对于 $\red{100\%}$ 的数据，$\red{1≤m≤10^9}$。
