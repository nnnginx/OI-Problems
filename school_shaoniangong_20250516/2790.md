## 题目描述

奶牛们接到了寻找一种新型挤奶机的任务，为此它们准备依次经过$\red{N(1≤}$$\red{N≤}$$\red{50000)}$颗行星，在行星上进行交易．

为了方便，奶牛们已经给可能出现的$\red{K(1≤}$$\red{K≤}$$\red{1000)}$种货物进行了由$\red{1}$到$\red{K}$的标号．由于这些行星都不是十分发达．没有流通的货币，所以在每个市场里都只能用固定的一种货物去换媛另一种货物．

奶牛们带着一种上好的饲料从地球出发，希望进行最少的交易，最终得到所需要的机器．饲料的标号为$\red{1，}$所需要的机器的标号为$\red{K.}$如果任务无法完成，输出$\red{-1.}$

## 输入格式

第$\red{1}$行是两个数字$\red{N}$和$\red{K.}$

第$\red{2}$到$\red{N+1}$行，每行是两个数字$\red{Ai}$和$\red{Bi，}$表示第$\red{i}$颗行星愿意提供$\red{Ai}$为得到$\red{Bi}$．

## 输出格式


第$\red{1}$行输出最小交换次数


## 样例

#### 输入样例

```c
6 5
1 3
3 2
2 3
3 1
2 5
5 4
```

#### 输出样例

```c
3
```
