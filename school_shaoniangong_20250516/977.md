## 题目描述

读入两个整数$\red{m}$、$\red{n}$。编程输出最大公约数和最小公倍数。

## 输入格式

（`多组数据`）。每组两个数$\red{m}$、$\red{n}$（$\red{1≤m,n≤32000}$）

## 输出格式

（`多组输出`） 每组两个数，即最大公约数和最小公倍数。保证答案输出不超过$\red{1000000000}$。两数之间有一空格。

## 样例

#### 输入数据

```C
24 16
101 205
```

#### 输出数据

```C
8 48
1 20705
```

## 提示

输入数据最多有$\red{200000}$组。
