## 题目描述

龙骑士非常喜欢树的问题，而为了刁难大家，他反手就丢了一棵有$\red{N}$个结点的$\red{1}$为根的树。

但是为了进一步刁难大家，龙骑士把每一个结点分为黑色和白色两种颜色。他现在让你选择恰好$\red{m}$个
黑色的结点，使得这$\red{m}$个黑点之间的距离的最大值最小，输出最小距离。

## 输入格式

第一行输入两个整数$\red{n}$和$\red{m,(1<=m<=n<=100)，}$分别代表结点的个数以及你需要选取的黑色结点
的个数。

接下来的一行输入$\red{n}$个整数$\red{p_1,p_2...p_n(0<=p_i<=1)}$。如果$\red{p_i=1，}$那么说明第$\red{i}$个结点是黑色
的，否则则是白色的。输入的数据保证黑色结点至少有$\red{m}$个。

输入的数据同时保证输入的图一定是一棵树。

## 输出格式

输出一行，代表答案。

## 样例

#### 输入样例

```c
6 3
1 1 0 1 1 1
1 2
1 3
1 4
3 5
3 6
```

#### 输出样例

```c
2
```

## 提示

在第一组样例中，唯一的选择是选择结点$\red{1，}$$\red{2，}$$\red{4，}$他们的最大值是$\red{2}$。
