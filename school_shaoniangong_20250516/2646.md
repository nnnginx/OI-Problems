## 题目描述

公牛在数学上比母牛好得多。他们可以将巨大的整数相乘并得到完全精确的答案

至少他们是这么说的。农夫约翰想知道他们的答案是否正确。

帮助他检查公牛的答案。读入两个正整数（每个不超过 $\red{40 }$位）并计算它们的乘积.将其输出为普通数字（没有额外的前导零）。

$\red{FJ }$要求您自己执行此操作；不要使用特殊的库函数进行乘法运算。输入两个数，输出其乘积

## 输入格式

第 $\red{1...2 }$行：每行包含一个十进制数。

## 输出格式

第 $\red{1 }$行：两个输入行的精确乘积

## 样例

#### 输入样例

```c
11111111111111
1111111111
```




#### 输出样例

```c
12345679011110987654321
```
