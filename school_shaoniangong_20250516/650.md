## 题目描述

输入两个正整数 $\red{x_0，y_0}$ ，求出满足下列条件的 $\red{P，Q}$ 的个数：

1. $\red{P，Q}$ 是正整数
2. 要求 $\red{P，Q}$ 以 $\red{x_0}$ 为最大公约数，以 $\red{y_0}$ 为最小公倍数。

试求：满足条件的所有可能的 $\red{P、Q}$ 的个数。

## 输入格式

一行两个正整数 $\red{x_0}$，$\red{y_0}$ 。

## 输出格式

一个整数，即满足条件的 $\red{P、Q}$ 的个数

## 样例

#### 输入样例

```c
3 60
```

#### 输出样例

```c
4
```

## 提示

$\red{P，Q}$有$\red 4$种：

- $\red{3, 60}$
- $\red{15, 12}$
- $\red{12, 15}$
- $\red{60, 3}$

对于$\red{100\%}$的数据，$\red{2 \le x_0, y_0 \le {10}^5}$。
