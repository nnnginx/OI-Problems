## 题目描述

小球钟是一个通过不断在轨道上移动小球来度量时间的设备。每分钟，一个转动臂将一个小球从小球队列的底部挤走，并将它上升到钟的顶部并将它安置在一个表示分钟，$\red{5}$分钟，$\red{15}$分钟和小时的轨道上。这样可以显示从$\red{1:00}$到$\red{24:59（}$这正是奇怪之处）范围内的时间，若有$\red{3}$个球在分钟轨道，$\red{1}$个球在$\red{5}$分钟轨道，$\red{2}$个球在$\red{15}$分钟轨道及$\red{15}$个球在小时轨道上，就显示时间$\red{15:38}$。

当小球通过钟的机械装置被移动后，它们就会改变其初始次序。仔细研究它们次序的改变，可以发现相同的次序会不断出现。由于小球的初始次序最后迟早会被重复，所以这段时间的长短是可以被度量的，这完全取 决于所提供的小球的总数。

每分钟，最近最少被使用的那个小球从位于球钟底部的小球队列被移走，并将上升安置于显示分钟的轨道上，这里可以放置$\red{4}$个小球。当第$\red{5}$个小球滚入该轨道，它们的重量使得轨道倾斜，原先在轨 道上的$\red{4}$个小球按照与它们原先滚入轨道的次序相反的次序加入到钟底部的小球队列。引起倾斜的第$\red{5}$个小球滚入显示$\red{5}$分钟的轨道。该轨道可以放置$\red{2}$个球。当第$\red{3}$个小球滚入该轨道，它们的重量使得轨道倾斜，原先$\red{2}$个小球同样以相反的次序加入钟底部的小球队列。而这第$\red{3}$个小球滚入了显示$\red{15}$分钟的轨道。这里可以放置$\red{3}$个小球。当第$\red{4}$个 小球滚入该轨道，它们的重量使得轨道倾斜，原先在轨道上的$\red{3}$个小球按照与它们原先滚入轨道的次序相反的次序加入钟底部的小球队列，而这第$\red{4}$个小球滚入了显示小时的轨道。该轨道同样可以放 置$\red{23}$个球，但这里有一个外加的固定的不能被移动的小球，这样小时的值域就变为$\red{1}$到$\red{24}$。从$\red{15}$分钟轨道滚入的第$\red{24}$个小球将使小时轨道倾斜，这$\red{23}$个球同样以相反的次序加入钟底部的小球队列，然后那第$\red{24}$个小球同样加入钟底部的小球队列。

## 输入格式

输入定义了一序列的小球时钟。每个时钟都按照前面描述的那样运作。所有时钟的区别仅在于它们在$\red{1:00}$时钟启动时刻小球初始个数的不同。在输入的每行上给出一个时钟的小球数，它并不包括那个在小时 轨道上的固定的小球。合法的数据应在$\red{33}$$\red{\sim 250}$。$\red{0}$表明输入的结束。

## 输出格式

输出中每一行只有一个数，表示对应的输入情形中给出的小球数量的时钟在经过多少天的运行可以回到它的初始小球序列。

## 样例

#### 输入样例

```c
33　

55

0
```

#### 输出样例

```c
22

50
```
