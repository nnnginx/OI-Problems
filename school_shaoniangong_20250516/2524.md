## 题目描述

约翰的邻居鲍勃控告约翰家的牛们太会叫．

约翰的$\red{N(1≤}$$\red{N≤}$$\red{10000)}$只牛在一维的草场上的不同地点吃着草．她们都是些爱说闲话的奶牛，每一只同时与其他$\red{N-1}$只牛聊着天．

一个对话的进行，需要两只牛都按照和她们间距离等大的音量吼叫，因此草场上存在着$\red{\frac{N（N-1）}{2}}$个声音．

请计算这些音量的和．

## 输入格式

第$\red{1}$行输入$\red{N，}$接下来输入$\red{N}$个整数，表示一只牛所在的位置．

## 输出格式

一个整数，表示总音量．

## 样例

#### 输入样例

```c
5
1
5
3
2
4
```

#### 输出样例

```c
40
```

## 提示

输入详细信息：

在位置$\red{1}$、$\red{5}$、$\red{3}$、$\red{2}$和$\red{4}$有五头牛。

输出详细信息：

$\red{1}$岁的奶牛贡献$\red{1+2+3+4=10，}$$\red{5}$岁的奶牛贡献$\red{4+3+2+1=10，}$$\red{3}$岁的奶牛贡献$\red{2+1+1+2=6，}$$\red{2}$岁的奶牛贡献$\red{1+1+2+3=7，}$$\red{4}$岁的奶牛贡献$\red{3+2+1+1=7}$。总体积为$\red{（10+10+6+7+7）}$$\red{=40}$。
