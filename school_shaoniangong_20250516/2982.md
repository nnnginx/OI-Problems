# Background

方乐行整理  数据有问题

# Description

给定两个只包含小写字母的字符串$s1$,$s2$，求$s2$是否是$s1$的字串，如果是，输出$s2$在$s1$中出现的次数，否则输出`No`。

# Format

## Input

共两行，一行一个字符串，分别表示$s1$和$s2$。

## Output

如果$s2$是$s1$的子串，输出$s2$在$s1$中出现的次数，否则输出`No`。

# Samples

```input1
ababababab
ab
```

```output1
5
```

```input2
jinitaimei
baby
```

```output2
No
```

# Limitation

对于100%的数据，$1<=|s2|<=|s1|<=10^6$
