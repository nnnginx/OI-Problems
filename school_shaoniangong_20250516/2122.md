## 题目描述

为了赚点外快，奶牛们在他们的谷仓里开了一家专门卖奶昔的餐厅。餐厅连续有 $\red{N }$个座位 $\red{(1 <= N <= 500,000)}$。最初，它们都是空的。

在一天中，餐厅会依次发生 $\red{M }$个不同的事件（$\red{1 <= M <= 300,000）}$。可能发生的两种类型的事件是：

一个大小为 $\red{p }$的聚会到达 $\red{(1 <= p <= N)}$。$\red{Bessie }$想把派对安排在一个连续的 $\red{p }$个空座位区。如果这是可能的，她会在座位列表中旧能处于最低位置。如果不可能，党就被拒之门外。

给定范围 $\red{[a,b] (1 <= a <= b <= N)，}$该范围内的每个人都离开。

请帮助 $\red{Bessie }$计算一天中被拒绝的聚会的总数。

有一排$\red{n}$个座位，$\red{m}$次操作。$\red{A}$操作：将$\red{a}$名客人安置到最左的连续$\red{a}$个空位中，没有则不操作。$\red{L}$操作：$\red{[a,b]}$的客人离开。

求$\red{A}$操作的失败次数。

## 输入格式

第 $\red{1 }$行：两个空格分隔的整数，$\red{N }$和 $\red{M}$。

第 $\red{2..M+1 }$行：每行描述一个事件。它要么是"$\red{A p}$"（表示大小为 $\red{p }$的队伍到达）或"$\red{L a b}$"（表示 $\red{[a, b] }$范围内的所有奶牛离开）形式的行。

## 输出格式

第 $\red{1 }$行：被拒之门外的人数。

## 样例

#### 输入样例

```c
10 4 
一个 6
大号 2 4
5
A2
```

#### 输出样例

```c
1
```

## 提示

有$\red{10}$个座位，$\red{4}$个项目。首先，一群 $\red{6 }$头奶牛来了。然后座位 $\red{2..4 }$中的所有奶牛离开。接下来，一个 $\red{5 }$人的聚会到达，然后是一个 $\red{2 }$人的聚会。

派对#$\red{3 }$被拒之门外。其他各方均就座。
