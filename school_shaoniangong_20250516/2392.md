## 题目描述

$\red{Farmer John }$有太多的工作要做啊！！！！！！！！为了让农场高效运转，他必须靠他的工作赚钱，每项工作花一个单位时间。

他的工作日从$\red{0}$时刻开始，有$\red{1000000000}$个单位时间（！）。在任一时刻，他都可以选择编号$\red{1\sim}$$\red{N}$的$\red{N(1 <= N <= 100000)}$项工作中的 任意一项工作来完成。 因为他在每个单位时间里只能做一个工作，而每项工作又有一个截止日期，所以他很难有时间完成所有$\red{N}$个工作，虽然还是有可能。

对于第$\red{i}$个工作，有一个截止时间$\red{D_i(1 <= D_i <= 1000000000)，}$如果他可以完成这个工作，那么他可以获利$\red{P_i( 1<=P_i<=1000000000 ). }$

在给定的工作利润和截止时间下，$\red{FJ}$能够获得的利润最大为多少呢？答案可能会超过$\red{32}$位整型。

## 输入格式

第$\red{1}$行：一个整数$\red{N. }$第$\red{2}$~$\red{N+1}$行：第$\red{i+1}$行有两个用空格分开的整数：$\red{D_i}$和$\red{P_i.}$

## 输出格式

输出一行，里面有一个整数，表示最大获利值。

## 样例

#### 输入样例

```c
3
2 10
1 5
1 7
```


#### 输出样例

```c
17
```

## 提示

第$\red{1}$个单位时间完成第$\red{3}$个工作$\red{(1,7)，}$


然后在第$\red{2}$个单位时间完成第$\red{1}$个工作$\red{(2,10)}$以达到最大利润
