## 题目描述

　从键盘输入一个字符串（长度$\red{<=40}$个字符），并以字符 ’`.`’ 结束。

编辑功能有：

- `1`、 $\red{D}$：删除一个字符，命令的方式为： $\red{D}$ $\red{a}$   其中$\red{a}$为被删除的字符，例如：$\red{D}$ $\red{s}$ 表示删除字符 $\red{s}$ ，若字符串中有多个 $\red{s}$，则删除第一次出现的。
- `2`、 $\red{I}$：插入一个字符，命令的格式为：$\red{I}$ $\red{a_{1} a_{2}}$   其中$\red{a_1}$表示插入到指定字符前面，$\red{a_2}$表示将要插入的字符。
  例如：$\red{I}$ $\red{s}$ $\red{d}$ 表示在指定字符$\red{s}$ 的前面插入字符$\red{d}$ ，若原串中有多个$\red{s}$ ，则插入在最后一个字符的前面。
- `3`、 $\red{R}$：替换一个字符，命令格式为：$\red{R}$ $\red{a_{1} a_{2}}$   其中$\red{a_1}$为被替换的字符，$\red{a_2}$为替换的字符，若在原串中有多个$\red{a_1}$则应全部替换。

在编辑过程中，若出现被改的字符不存在时，则输出提示信息"指定字符不存在"

## 输入格式

总共两行

第一行一个字符串（长度$\red{<=40}$个字符），并以字符 ’`.`’ 结束。

第二行输入编辑的命令

## 输出格式

输出编辑完后的字符串

## 样例

#### 输入样例

```C
This is a book.
D s
```

#### 输出样例

```C
Thi is a book.
```
