## 题目描述

相传韩信才智过人，从不直接清点自己军队的人数，

只要让士兵先后以`3人`一排、`5人`一排、`7人`一排的变换队形，而他每次只要掠一眼队伍的排尾就知道总人数了。

请问：韩信的士兵的最小值是多少？

## 输入格式

一行，共三个整数，包含$\red{3}$个非负整数$\red{a}$,$\red{b}$,$\red{c}$，表示每种队形排尾的人数（$\red{a<3,b<5,c<7}$），整数之间用空格隔开。

## 输出格式

一行，输出韩信的总的士兵的数量的最小值。

如果无解，则输出字符串`no answer`

## 样例

#### 输入数据

```C
1 2 2
```

#### 输出数据

```C
37
```

## 提示

注意韩信的总的兵的范围 $\red{10<n<10^{6}}$
