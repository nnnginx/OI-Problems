## 题目描述

`FJ`在沙盘上写了这样一些字符串：

- $\red{A_1}$ `=` “`A`”
- $\red{A_2}$ `=` “`ABA`”
- $\red{A_3}$ `=` “`ABACABA`”
- $\red{A_4}$ `=` “`ABACABADABACABA`”
- … …
  你能找出其中的规律并写所有的数列$\red{A_N}$吗？

## 输入格式

仅有一个数：$\red{N ≤ 26}$。

## 输出格式

请输出相应的字符串$\red{A_N}$，以一个换行符结束。输出中不得含有多余的空格或换行、回车符。

## 样例

#### 输入样例

```C
3
```

#### 输出样例

```C
ABACABA
```
