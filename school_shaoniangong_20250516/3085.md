# 题目描述

小柒一向有存钱的习惯，她会从妈妈给的零花钱中取出一小部分作为自己的购物储备金。

购物节来临，小柒在网上购物软件上挑选了 n 件商品放入了购物车，每件商品的价格可能不同，第i件商品的价格是 a<sub>i</sub>

假设小柒的购物储备金是 m 元，请问她能否把选好的 n 件商品都买下来，如果可以的话，全部买下来后请算出小柒剩余多少钱；如果钱不够，请问她还差多少钱才能买下这 n 件商品

# Format

## 输入格式

第一行，输入一个整数 m，表示小柒的购物储备金

第二行，输入一个整数 n，表示 n 件商品

第三行，输入 n 个整数，表示每件商品的价格

## Output

输出一个整数，表示把n件商品全部买下后剩余的钱数，或者表示储备金不够是还差多少钱能买下全部商品

# Samples

```input1
10
3
3 5 2
```

```output1
0
```

# Limitation

1s, 1024KiB for each test case.
