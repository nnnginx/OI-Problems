## 题目描述

$\red{Farmer John }$打算修建一座花园，他需要移动不少泥土。

花园由 $\red{N }$个花坛组成（$\red{1≤}$$\red{N≤}$$\red{100）}$，其中花坛 $\red{i }$包含 $\red{A_i}$单位的泥土。$\red{FJ }$希望花坛 $\red{i }$包含 $\red{B_i}$单位的泥土，保证 $\red{0 \leq A_i,B_i \leq 10}$

为了达到这个目标，他可以做这几件事情：

购买一单位的泥土，放在指定的花坛中，费用为 $\red{X}$。

从任意一个花坛中移走一单位泥土，费用为 $\red{Y}$。

从花坛 $\red{i }$运送一单位泥土到花坛 $\red{jj，}$费用为 $\red{Z|i-j|}$。

请你帮 $\red{FJ }$计算移动泥土的最歇销。

## 输入格式

第一行四个整数 $\red{N,X,Y,Z（}$$\red{0≤}$$\red{X,Y,Z≤}$$\red{1000）}$。

接下来 $\red{N }$行，第 $\red{i }$行两个整数 $\red{A_i,B_i}$。

## 输出格式

输出移动泥土的最歇销。

## 样例

#### 输入样例

```c
4 100 200 1 
1 4 
2 3 
3 2 
4 0
```

#### 输出样例

```c
210
```

## 提示

按下面的方案，最小花费为 $\red{210，}$可以证明不存在开销更小的方案。

移除 $\red{4 }$号花坛的一单位泥土，花费 $\red{200}$。

将 $\red{4 }$号花坛的三单位泥土移到 $\red{1 }$号花坛，花费 $\red{3×}$$\red{3=9}$。

将 $\red{3 }$号花坛的一单位泥土移到 $\red{2 }$号花坛，花费 $\red{1×}$$\red{1=1}$。
