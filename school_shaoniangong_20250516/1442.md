## 题目描述

有一个俄罗斯方块的游戏，但是这是只有三列的，并且对于每个木块，你只能先旋转确定形状和插到哪一列以后，直接往下落，中途不能再进行操作了。

木块一共有以下$\red {7}$种：
![img](./1442/file/1.png)

现在依次有$\red {n}$ 个木块下来，求最终最大高度的最小值。

## 输入格式

第一行一个整数$\red n$，表示掉落下来的木块个数。

接下来$\red n$行，每行一个整数，表示第$\red i$个掉下来的木块编号。

## 输出格式

输出只有一行，表示叠放最高的木块最小高度。

## 样例

#### 输入样例

```C
3
4
5
1
```

#### 输出样例

```C
5
```



## 提示

$\red {1<N<=100}$
