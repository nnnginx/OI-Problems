# HCH的祝福

## 题目背景
HCH为了庆祝陆进入复赛。
HCH写了一个字符串，来祝贺陆。
HCH认为一个字符串的祝福力度就是这个字符串中有多少个子序列可以形成$AKJS$

现在陆想知道，HCH的这个字符串中一共有多少个子序列可以形成$AKJS$

## 题目描述

输出HCH的这个字符串中一共有多少个子序列可以形成$AKJS$

## 输入格式

第一行n代表长度
输入第2行包含一个长度为n的字符串

## 输出格式

输出这个字符串的祝福力度

## 样例 #1

### 样例输入 #1

```
12
AAKJQQKJSSDD
```

### 样例输出 #1

```
12
```

## 提示

对于$100%$的数据字符串长度$len\le 10^{7}$
