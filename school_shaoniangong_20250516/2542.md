## 题目描述

农夫$\red{JOHN}$准备把他的 $\red{N（}$$\red{1 <= N <= 10,000）}$头牛排队以便于行动。因为脾气大的牛有可能会捣乱，$\red{JOHN}$想把牛按脾气的大小排序。

每一头牛的脾气都是一个在$\red{1}$到$\red{100，}$$\red{000}$之间的整数并且没有两头牛的脾气值相同。在排序过程中，$\red{JOHN }$可以交换任意两头牛的位置。因为脾 气大的牛不好移动，$\red{JOHN}$需要$\red{X+Y}$秒来交换脾气值为$\red{X}$和$\red{Y}$的两头牛。

请帮$\red{JOHN}$计算把所有牛排好序的最短时间。

## 输入格式

第$\red{1}$行： 一个数， $\red{N}$。

第$\red{2...N+1}$行： 每行一个数，第$\red{i+1}$行是第$\red{i}$头牛的脾气值。

## 输出格式

第$\red{1}$行： 一个数，把所有牛排好序的最短时间。

## 样例

#### 输入样例

```c
3
2
3
1
```

#### 输出样例

```c
7
```

## 提示

输入解释：

队列里有三头牛，脾气分别为 $\red{2，}$$\red{3，}$ $\red{1}$。

输出解释：
$\red{2 3 1 : }$初始序列

$\red{2 1 3 : }$交换脾气为$\red{3}$和$\red{1}$的牛$\red{(}$时间$\red{=1+3=4). }$




$\red{1 2 3 : }$交换脾气为$\red{1}$和$\red{2}$的牛$\red{(}$时间$\red{=2+1=3). }$
