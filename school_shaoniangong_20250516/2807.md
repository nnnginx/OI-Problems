## 题目描述

给出一个$\red{1,...,2n}$的排列$\red{p_i，}$并给出下述两种操作：

$\red{1}$、将$\red{p_1}$和$\red{p_2}$、$\red{p_3}$和$\red{p_4}$、$\red{...}$、$\red{p_{2n-1}}$和$\red{p_{2n}}$交换；

$\red{2}$、将$\red{p_1}$和$\red{p_{n+1}}$、$\red{p_2}$和$\red{p_{n+2}}$、$\red{...}$、$\red{p_{n}}$和$\red{p_{2n}}$交换。

请问最少多少步操作能够使得排列$\red{p_i}$成为从小到大排序的呢？

## 输入格式

第一行输入一个正整数$\red{n}$；

第二行输入$\red{2n}$个正整数表示排列$\red{p}$

## 输出格式

输出一个正整数表示答案。

如果上述操作无法实现排序，则输出$\red{-1.}$

## 样例

#### 输入样例1

```c
3
6 3 2 5 4 1
```

#### 输出样例1

```c
3
```

#### 输入样例2

```c
2
3 4 2 1
```

#### 输出样例2

```c
-1
```

## 提示

对于$\red{40\%}$的数据满足$\red{1<=n<=10}$；

对于$\red{100\%}$的数据满足$\red{n<=1000}$

对于样例$\red{1：}$

第一步：$\red{3~ 6 ~5 ~2 ~1 ~4}$

第二步：$\red{2~ 1 ~4 ~3 ~6 ~5}$

第三步：$\red{1~ 2 ~3 ~4 ~5 ~6}$
