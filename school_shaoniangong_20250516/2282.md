## 题目描述

虽然科学界已经深入解决了鸡为什么过马路的古老问题，但令人惊讶的是，关于牛过马路相关主题的研究文献中几乎没有发表。

$\red{Farmer John }$非常清楚这个问题的重要性，当当地一所大学联系他时，他非常激动，要求他协助对奶牛过马路的原因进行科学研究。

他热切地自愿提供帮助。

作为研究的一部分，农夫约翰被要求记录每头奶牛过马路的次数。

他仔细记录了有关奶牛位置的数据，在一天内进行了一系列$\red{N }$观察。每个观察记录一头奶牛的$\red{ID }$号（一个介于 $\red{1…}$$\red{10 }$的整数，因为 $\red{Farmer John }$有 $\red{10 }$头奶牛），以及奶牛在路的哪一边。

根据$\red{Farmer John }$记录的数据，请帮他统计确认过境的总数。当连续看到一头牛将其放在道路的不同侧面时，就会发生确认的交叉。

## 输入格式

输入的第一行包含观察数$\red{N，}$一个最大为$\red{100}$的正整数。

接下来的每条$\red{N}$线包含一个观察值，由一个$\red{cow ID}$号和由零或一（道路一侧为零，另一侧为一）指示的位置组成。

## 输出格式

请计算已确认的过境点总数。

## 样例

#### 输入样例

```c
8
3 1
3 0
6 0
2 1
4 1
3 0
4 0
3 1
```




#### 输出样例

```c
3
```

## 提示

在这个例子中，奶牛$\red{3}$穿过两次--她首先出现在第$\red{1}$面，然后出现在第$\red{0}$面，然后仍然出现在第$\red{1}$面。



$\red{4}$号奶牛肯定过一次。$\red{2}$号和$\red{6}$号奶牛似乎没有杂交。
