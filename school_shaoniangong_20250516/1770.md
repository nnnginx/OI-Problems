## 题目描述

“报告，我军已探出地雷阵中所有的地雷位置，并且还发现了一份使用说明书。”一个黑暗军团的小兵匆忙跑来，交给修罗王一张纸。

只见这张纸上面写道：“我是一颗萌萌的地雷，拆除我很容易，看到我身上标着的整数$\red Ｎ$了吗？你只要输入这个$\red Ｎ$位数中有多少个数中有偶数个数字$\red 3$就可以把我拆除哦，加油！你行的。”

## 输入格式

一个整数$\red Ｎ$。

## 输出格式

输出这个$\red Ｎ$位数中有多少个数中有偶数个数字$\red ３$。

## 样例

#### 输入样例

```c
２
```

#### 输出样例

```c
73
```
