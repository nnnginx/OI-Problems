# 题目背景

小ao在mc的一个服务器里玩RPG。

# 题目描述

小ao获得了一把剑，这把剑的伤害是一个数 $a_i |$ 或 $\&$ 上 m 个数。他用这把剑砍了 $n$ 下，他想知道这 $n$ 下能造成多少伤害。

## 输入格式

输入第一行输入 $n,m$
第2到 $n+1$ 行，每行输入一个数 $a_i$ , $a_i$ 用2进制表示
第 $n+1$到 $n+1+m$ 行，每行输入一个字符和 $k$ ，如果这个字符为 'y' ,则与上 $k$，如果是 'h' 则或上 $k$，$k$ 也用二进制输入。

## 输出格式

输出 $n$ 行，每行输出 $a_i$ 的结果。

# 样例

```input1
2 2
1010 
101
y 1111
h 1111
```

```output1
1111
1111
```

# 提示说明

30%的数据
$a_i,k \le 2^{50}$
$n,m \le 10^2$

50%的数据
$a_i,k \le 2^{200}$
$n,m \le 10^2$

100%的数据
$a_i,k \le 2^{200}$
$n,m \le 10^4$

# 样例解释

将1010与上1111再或上1111得1111

将101与上1111再或上1111得1111
