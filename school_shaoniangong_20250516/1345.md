## 题目描述

农夫知道一头牛的位置，想要抓住它。农夫和牛都位于数轴上，农夫起始位于点$\red{N}$($\red{0<=N<=100000}$)，牛位于点$\red K$($\red{0<=K<=100000}$)。

农夫有两种移动方式：

`1`、从$\red{X}$移动到$\red{X-1}$或$\red{X+1}$，每次移动花费一分钟

`2`、从$\red{X}$移动到$\red{2*X}$，每次移动花费一分钟

假设牛没有意识到农夫的行动，站在原地不动。农夫最少要花多少时间才能抓住牛？

## 输入格式

输入两个整数，$\red{N}$和$\red{K}$输出一个整数，

## 输出格式

农夫抓到牛所要花费的最小分钟数

## 样例

#### 输入样例

```C
5 17
```

#### 输出样例

```C
4
```
