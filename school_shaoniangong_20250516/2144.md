## 题目描述

作为所有寒冷天气运动的爱好者(其是那些涉及奶牛的运动)，$\red{Farmer John}$想要旧能多地记录下即将到来的冬季$\red{Moolympics}$。

$\red{Moolympics}$的电视节目表由$\red{N}$个节目$\red{(1 <= N <= 150)}$组成，每个节目都有一个指定的开始时间和结束时间。

$\red{FJ}$有一个双调谐记录仪，可以同时 记录两个程序。
请帮助他确定他能记录的最大程序数。

农民约翰热衷于所有寒冷天气的运动（尤其是涉及到牛的运动）， 农民约翰想录下旧能多的电视节目。

$\red{moolympics }$的节目时间表有 $\red{N}$个不同的节目 $\red{(1 <= N <= 150)，}$每个节目给定开始时间和结束时间。

$\red{FJ }$有一个双调谐器录音机，可以同时录制两个节目。 请帮助他确定他能录制的节目的最大数量。

## 输入格式

第 $\red{1}$行：整数 $\red{N}$。

第 $\red{2 }$到第 $\red{N+1 }$行：每行包含单个节目的开始和结束时间（范围为 $\red{0…}$$\red{10000000000 }$的整数）。

## 输出格式

仅一行，$\red{FJ}$可以记录的最大节目数量。

## 样例

#### 输入样例

```c
6
0 3
6 7
3 10
1 5
2 8
1 9

```


#### 输出样例

```c
4
```

## 提示

输入详细信息:
$\red{Moolympics}$广播由$\red{6}$个节目组成。第一个从时间$\red{0}$运行到时间$\red{3，}$依此类推。

输出详细信息:
$\red{FJ}$最多可以录制$\red{4}$个节目。例如，他可以在第一个调谐器上连续记录程序$\red{1}$和$\red{3，}$在第二个调谐器上连续记录程序$\red{2}$和$\red{4}$。
