## 题目描述

太空梯工程作为一个系统,由$\red{n}$个部件串联而成，只要有一个部件故障，系统就不能正
常运行，为提高系统的可靠性,每一件都装有备用件,一旦原部件故障,备用件就自动进人
系统。显然备用件越多,系统可靠性越高,但费用也越大,那么在一定总费用限制下,系统的
最高可靠性等于多少?

给定一些系统备用件的单价$\red{C_k,}$以及当用$\red{M_k}$个此备用件时部件的正常工作概率
$\red{P_k(M_k),}$总费用上限$\red{C}$。求系统可能的最高可靠性。

## 输入格式

第一行:$\red{n ~C}$

第二行: $\red{C_1 ~P_1(0) ~P_1(1)...P_1(X_1) (0≤}$$\red{X_1≤}$$\red{[C/C_k])}$

...

第$\red{n}$行: $\red{C~_n ~P_n(0) ~P_n(1)...P_n(X_n) (0≤}$$\red{X_n≤}$$\red{[C/C_n])}$

## 输出格式

输出系统可能的最大可靠性,保留$\red{4}$位有效数字。

## 样例

#### 输入样例

```c
2 20
3 0.6 0.65 0.7 0.75 0.8 0.85 0.9
5 0.7 0.75 0.8 0.8 0.9 0.95
```

#### 输出样例

```c
0.6375
```
