## 题目描述

贝西喜欢下载游戏在手机上玩，尽管她确实觉得小触摸屏与她的大蹄子配合使用相当麻烦。

她对目前正在玩的游戏特别感兴趣。游戏从$\red{N}$个正整数序列开始$\red{（2≤}$$\red{N≤}$$\red{262144）}$，每个在$\red{1…}$$\red{40}$范围内。在一个动作中，贝西可以取两个值相等的相邻数字，并 将其替换为一个值大一点的单个数字（例如，她可能将两个相邻的$\red{7}$替换为$\red{8）}$。目标是在游戏结束时使序列中出现的最大数字的值最大化。请帮助贝西获得旧能高的分数。

## 输入格式

第一行输入包含$\red{N，}$接下来的$\red{N}$行在游戏开始时给出了$\red{N}$个数字的序列

## 输出格式

请输出贝西可以生成的最大整数。

## 样例

#### 输入样例

```c
4
1
1
1
2
```

#### 输出样例

```c
3
```

## 提示

在此示例中，贝西首先合并第二个和第三个$\red{1}$以获得序列$\red{1 2 2，}$然后她将$\red{2}$合并为$\red{3}$。请注意，将前两个$\red{1}$连接起来并不是最佳选择。
