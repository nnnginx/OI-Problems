## 题目描述

农夫约翰已经从他的牧场中取得了数不清块数的正方形草皮，草皮的边长总是整数（有时农夫约翰割草皮的刀法不合适，甚至切出了边长为$\red{0}$的正方形草皮），他已经把草皮放在了一个奶牛贝茜已经知道的地方。

贝茜总是希望把美味的草皮放到她的秘密庄园里，她决定从这些草皮中取出恰好$\red{4}$块搬到她的秘密庄园中，然后把它们分成$\red{1×}$$\red{1}$的虚，组成一个面积为$\red{N(1<=N<=10,000)}$个单位面积的部分。

贝茜对选出这样四块草皮的方法数很感兴趣，如果她得到了一个$\red{4}$个单位面积的部分，那么她可以有$\red{5}$中不同的方法选$\red{4}$块草皮：

$\red{(1,1,1,1),(2,0,0,0),(0,2,0,0),(0,0,0,2).}$顺序是有效的：$\red{(4,3,2,1)}$和$\red{(1,2,3,4)}$是不同的方法。

## 输入格式

第一行：一个单独的整数$\red{N}$。

## 输出格式

单独的一行包含一个整数，表示贝茜选四块草皮的方案数。

## 样例

#### 输入样例

```c
4
```

#### 输出样例

```c
5
```
