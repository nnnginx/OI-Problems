## 题目描述

$\red{Farmer John }$的 $\red{N }$头奶牛$\red{（1 <= N <= 50,000）}$位于他的二维牧场的不同点。牧场中间是一个大的圆形粮仓。筒仓对面的奶牛无法看到对方，因为筒仓挡住了它们的视线。

请确定可以通过直接视线看到对方的奶牛对的数量。粮仓以原点 $\red{(0,0) }$为中心，半径为 $\red{R}$。没有奶牛位于对应于筒仓的圆上或圆内，并且没有两头奶牛位于与筒仓的切线上。

$\red{R }$的值在 $\red{1..1,000,000 }$范围内，每头奶牛都生活在整数坐标在 $\red{-1,000,000..+1,000,000 }$范围内的点上。

在$\red{(0,0)}$点半径为$\red{R}$的圆挡了这些牛的视线，问住有多少对牛是相互观察的。

## 输入格式

第 $\red{1 }$行：两个整数：$\red{N }$和 $\red{R}$。

第 $\red{2..1+N }$行：每行包含两个整数，指定牛的 $\red{(x,y) }$坐标。

## 输出格式

第 $\red{1 }$行：可以看到彼此的奶牛对的数量。

## 样例

#### 输入样例

```c
4 5
0 10
0 -10
10 0
-10 0

```
#### 输出样例

```c
4
```

## 提示

输入详细信息：

在 $\red{(0,10)}$、$\red{(0,-10)}$、$\red{(10,0) }$和 $\red{(-10,0) }$位置有 $\red{4 }$头奶牛。筒仓以 $\red{(0,0) }$为中心，半径为 $\red{5}$。

输出细节：

所有 $\red{6 }$对奶牛都可以互相看到，除了位于筒仓相对两侧的奶牛：$\red{(-10,0) }$和 $\red{(10,0) }$处的奶牛不能互相看到，并且$\red{(0,-10) }$和 $\red{(0,10) }$处的奶牛无法看到彼此。
