# Description

小明在排队，由于他来的很早，所以他排在了队头，但是他觉得自己很厉害，需要压轴出场，所以聪明的他找来了一个栈:所有排队的人都要再入栈出栈形成新的顺序。现在小明知道出栈和入栈操作的顺序，他想知道最后出栈的序列。

# Format

## Input

第一行一个数N，为排队的总人数。

接下来一行2N个用空格分隔的0和1表示出入栈操作的序列，其中0表示入栈，1表示出栈。输入保证合法且入栈出栈各有N次。

## Output

一行N个整数。第i个整数表示第i个出栈的人是第几个入栈的。

# Samples

```input1
4
0 1 0 0 1 0 1 1
```

```output1
1 3 4 2
```
