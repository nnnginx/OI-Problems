# 题目描述

给定一个长度为 n 的序列，初始全为 0，依次执行n次操作：

* 指定一个位置 p<sub>i</sub> ，保证序列的第   p<sub>i</sub> 位为 0，然后将该数赋值为 1

每次操作后，回答在当前操作位置上，向左右两端连续的 1  的个数。

## 输入

第一行一个整数 n ，表示序列的长度。

接下来 n 行，每行一个整数 p<sub>i</sub>，表示每次操作的位置。

## 输出

输出 n 行，每行一个整数，表示当前位置连续  1 的个数。

# 样例

```
6
1
3
2
5
6
4
```

```
1
1
3
1
2
6
```



# 数据范围

对于 40% 的数据： n <=1000

对于 100% 的数据： n <=10<sup>5</sup>
