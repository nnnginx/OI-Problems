## 题目描述

对于一个长度为n的正整数序列，$\red{a_1,a_2......a_n}$,输入$\red{m}$表示$\red{m}$组测试数据，对于每组数据输入$\red i$,$\red j$，输出$\red {a_i}$到$\red {a_j}$的和。

## 输入格式

`第一行`包含一个正整数$\red n$;

`第二行`包含$\red n$个正整数，用空格隔开；

`第三行`包含一个正整数$\red m$；

接下来的m行数据每行包含两个数据$\red i$,$\red j$，以空格隔开；

## 输出格式

共$\red m$行，
每行一个正整数表示从$\red{a_i}$到$\red{a_j}$的和

## 样例

#### 输入样例

```C
5
10 20 30 40 50
3
2 3
2 5
1 3
```

#### 输出样例

```C
50
140
60
```

## 提示

对于所有的数据满足:
$\red{1 \le n \le 100, m < n ;}$

$\red{i < n , i < j < n}$;
