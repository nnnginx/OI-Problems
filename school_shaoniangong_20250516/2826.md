## 题目描述

平面上有 $\red{n }$个没有公共点 ．．．．．的圆。你要从点$\red{(x1,y1)}$走到$\red{(x2,y2)}$。问你最少要经过多少圆
的边界。保证这两个点都不在圆的边界上。

## 输入格式

第一行一个整数 $\red{n，}$$\red{1<=n<=50}$。

接下来三行每行 $\red{n }$个整数，分别表示 $\red{n }$个圆的圆心和半径，格式如下：

$\red{x1,x2,...,xi,...,xn}$

$\red{y1,y2,...,yi,...,yn}$

$\red{r1,r2,...,ri,...,rn}$

$\red{-1000<=xi,yi<=1000,1<=ri<=1000}$

最后一行四个整数 $\red{X1,Y1,X2,Y2，}$$\red{-1000<=x1,y1,x2,y2<=1000}$。

## 输出格式

一个整数，意义如上。

## 样例

#### 输入样例1

```c
1
0
0
2
-5 1 5 1
```

#### 输出样例1

```c
0
```

#### 输入样例2

```c
3
0 -6 6
0 1 6
2 2 2
-5 1 5 1
```

#### 输出样例2

```c
1
```
