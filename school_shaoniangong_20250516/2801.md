## 题目描述

给出一个正整数$\red{n，}$请你找出$\red{n}$个正整数，使得它们的按照数位相加之和（比如"$\red{11}$"相加之后的结果是$\red{2}$）是相等的，那么这$\red{n}$个数之和最小是多少呢？

## 输入格式

输入一个正整数$\red{n}$

## 输出格式

输出一个正整数表示答案

## 样例

#### 输入样例1

```c
2
```

#### 输出样例1

```c
11
```

#### 输入样例2

```c
3
```

#### 输出样例2

```c
33
```

## 提示

对于$\red{100\%}$的数据，满足$\red{1<=n<=5000}$

对于$\red{40\%}$的数据，满足$\red{1<=n<=100}$

样例1解释：
10 + 1

样例2解释：
2 + 11 + 20
