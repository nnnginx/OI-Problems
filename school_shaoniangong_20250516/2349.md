## 题目描述

给定一个仅包含字符 $\red{1 }$或 $\red{0 }$或 ? 字符串$\red{s}$。定义一个子串是不稳定的当且仅当子串中任意相邻两数均
不相同，如$\red{101010...101010...}$或$\red{010101...010101...}$。其中 $\red{?}$ 为通配符，可以当成$\red{1}$和$\red{0}$。请求出给
定的$\red{s}$中不稳定子串个数。

## 输入格式

一行一个字符串，表示$\red{s}$。

## 输出格式

输出一行一个整数，表示答案。

## 样例

#### 输入样例1

```c
0?10
```

#### 输出样例1

```c
8
```

#### 输入样例2

```c
?10??1100
```

#### 输出样例2

```c
25
```

## 提示

对于$\red{50\%}$的数据，$\red{1<=s<=50}$。

对于$\red{100\%}$的数据，$\red{a<=s<=2\times 10^5}$。
