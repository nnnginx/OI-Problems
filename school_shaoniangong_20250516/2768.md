## 题目描述

$\red{Moo}$大学有$\red{N(1≤}$$\red{N≤}$$\red{1000)}$只奶牛今年想要组建一支体操队，每只奶牛都有一个正整数的高度和重量且小于$\red{100000.}$你的目标是组建一支人数旧能多的体操队．

但是这支队伍必须满足一个条件：每只队伍中的奶牛的高度日和重量$\red{W}$必须满足

$\red{A×}$$\red{(H -h)+B×}$$\red{(W- w)≤}$$\red{C}$

这里$\red{h}$和$\red{w}$叫表示这支队伍中奶牛的最低高度和最轻重量．$\red{A,B，}$$\red{C}$是给出的小于$\red{10000}$的正整数．计算这支队伍中奶牛数目的最大值．

## 输入格式

第$\red{1}$行：一个整数$\red{N.}$

第$\red{2}$行：三个由空格隔开的整数$\red{A，}$$\red{B，}$$\red{C.}$

第$\red{3}$到$\red{N+2}$行：两个空格隔开的整数，这头奶牛的高度和重量．


## 输出格式


一个整数表示队伍中奶牛最多可以有几头．


## 样例

#### 输入样例

```c
4
78
```


#### 输出样例

```c
5
```

## 提示

样例说明
这支队伍包括奶牛$\red{1，}$$\red{2，}$$\red{3，}$$\red{4，}$$\red{7}$．不存在更大的队伍
