## 题目描述

给定四个算术运算符 $\red{+、-、×、/}$，任意输入$\red{N}$个正整数（限定$\red{N=5）}$，在每相邻两个数之间填入上述运算符，构成一个算术表达式，同时给定一个整数$\red{M}$，编程求出所有表达式值为M的表达个数或输出无解信息（即不存在满足条件表达式）除法用整除处理。所有的数不超过$\red{100000000}$。

## 输入格式

共二行，第一行两个整数，$\red{N}$和$\red{M}$，接下来一行是$\red{N}$个整数

## 输出格式

满足条件的表达式的个数或`No Answer`。

## 样例

#### 输入样例1

```c
1 1 1 1 1 5
```

#### 输出样例1

```c
1
```

#### 输入样例2

```c
1 1 1 1 1 8
```

#### 输出样例2

```c
No Answer
```
