# 题目描述

小柒遇到了一个数学问题，这个问题是要找出长度为 n （1<n<10000）的正整数数列中的次大值，且数列中的数字可能重复出现。

次大值指的是一个数列中第二大的数字。

如果不存在次大值，则输出 No。

请你编程帮小柒解决这个难题

## 输入格式

第一行，输入一个整数 n ,表示数列的长度

第二行，输入 n 个正整数，表示数列中的元素

## 输出格式

输出一个整数，表示数列中的次大值。

# 样例

```input1
5
1 2 3 5 5
```

```output1
3
```

# Limitation

1s, 1024KiB for each test case.
