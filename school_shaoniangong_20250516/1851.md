## 题目描述

`gty`又虐了一场比赛,被虐的人们决定吊打`gty`。`gty`见大势不好机智的分出了$\red{n}$
个分身,但还是被人多势众的人们抓住了。人们将$\red{n}$个`gty`吊在$\red{n}$根绳子上,每根绳子
穿过天台的一个洞。这$\red{n}$根绳子有一个公共的绳结$\red{x}$。吊好`gty`后人们发现由于每个
`gty`重力不同,绳结$\red{x}$在移动。`wangxz`脑洞大开的决定计算出$\red{x}$最后停留处的坐
标，由于他太弱了决定向你求助。

不计摩擦,不计能量损失,由于`gty`足够矮所以不会掉到地上。

## 输入格式

输人第一行为一个正整数$\red{n(1≤}$$\red{n≤}$$\red{10 000),}$表示`gty`的数目。

接下来$\red{n}$行,每行三个整数$\red{x_i,y_i,w_i,}$表示第$\red{i}$个`gty`的横坐标,纵坐标和重力。

对于$\red{100\%}$的数据,$\red{1≤}$$\red{n≤}$$\red{10 000,- 100 000≤}$$\red{x_i,y_i≤}$$\red{100000}$。

## 输出格式

输出一行两个浮点数（保留到小数点后$\red{3}$位,中间用$\red{1}$个空格隔开）,表示最终$\red{x}$的
横、纵坐标。

## 样例

#### 输入样例

```c
3
0 0 1
0 2 1
1 1 1
```

#### 输出样例

```c
0.577 1.000
```
