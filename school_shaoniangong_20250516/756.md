## 题目描述

有两个仅包含小写英文字母的字符串 $ \red{A} $ 和 $ \red{B} $。现在要从字符串 $ \red{A} $ 中取出 $ \red{k} $ 个互不重叠的非空子串，然后把这 $ \red{k} $ 个子串按照其在字符串 $ \red{A} $ 中出现的顺序依次连接起来得到一个新的字符串,请问有多少种方案可以使得这个新串与字符串 $\red{ B }$ 相等？

注意：子串取出的位置不同也认为是不同的方案。

## 输入格式

第一行是三个正整数 $ \red{n,m,k} $，分别表示字符串 $ \red{A} $ 的长度，字符串 $ \red{B} $ 的长度，以及问题描述中所提到的 $ \red{k} $，每两个整数之间用一个空格隔开。

第二行包含一个长度为 $ \red{n }$ 的字符串，表示字符串 $ \red{A} $。

第三行包含一个长度为 $ \red{m }$ 的字符串，表示字符串 $ \red{B} $。

## 输出格式

输出共一行，包含一个整数，表示所求方案数。由于答案可能很大，所以这里要求输出答案对 $ \red{1,000,000,007} $ 取模的结果。

## 样例

#### 样例输入 1

```C
6 3 1
aabaab
aab
```

#### 样例输出 1

```C
2
```

#### 样例输入 2

```C
6 3 2
aabaab
aab
```

#### 样例输出 2

```C
7
```

#### 样例输入 3

```C
6 3 3
aabaab
aab
```

#### 样例输出 3

```C
7
```

## 数据范围与提示

对于第 $\red{ 1} $ 组数据：$ \red{1 \leq n \leq 500,1 \leq m \leq 50,k = 1} $；

对于第 $ \red{2} $ 组至第 $\red{ 3} $ 组数据：$\red{ 1 \leq n \leq 500,1 \leq m \leq 50,k = 2} $；

对于第 $ \red{4 }$ 组至第 $ \red{5} $ 组数据：$\red{ 1 \leq n \leq 500,1 \leq m \leq 50,k = m }$；

对于第 $\red{ 1 }$ 组至第 $\red{ 7} $ 组数据：$\red{ 1 \leq n \leq 500,1 \leq m \leq 50,1 \leq k \leq m }$；
}
对于第 $ \red{1} $ 组至第 $ \red{9 }$ 组数据：$\red{ 1 \leq n \leq 1000,1 \leq m \leq 100,1 \leq k \leq m }$；

对于所有 $\red{ 10} $ 组数据：$\red{ 1 \leq n \leq 1000,1 \leq m \leq 200,1 \leq k \leq m }$。
