# 题目描述

在生活中常见的数是十进制数，而计算机采用二进制数据处理。有个问题引起了小柒同学的思考――“十进制的数字可不可以转为二进制的数字呢？”。

请写出一个C++程序帮小柒解决这个问题，要求输出 32 位二进制数，不足 32 位请补足 32 位。


## 输入

第一行，输入一个整数数字 n ，表示十进制的数字

## 输出

输出十进制数字 n 转为 32 位二进制数字的结果。

# 样例

```input1
5
```

```output1
00000000000000000000000000000101
```

# Limitation

对于 100% 的数据： n在 int 类型范围内
