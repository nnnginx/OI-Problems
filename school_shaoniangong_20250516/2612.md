## 题目描述

贝茜和其他一些人去滑雪．贝茜发现她自己站在一块$\red{R \times C(1≤}$$\red{R，}$$\red{C≤}$$\red{100)}$的区域中，区域中的每一块都有一个高度值$\red{E(-25 <= E <= 25)}$．为了参加大家的聚会，贝茜想要眷到达右下角．

贝茜每一步只能向正东，正西，正南，正北前进一步．贝茜以初速度$\red{V(I≤}$$\red{V≤}$$\red{106)}$前进，她发现了一个她的速度和高度的关系．

当贝茜从高度$\red{n}$移动到高度$\red{b，}$她的速度就乘上了一个数$\red{2^{a-b}}$．贝茜移动一步的速度取决于她在前一格时的速度．

请找出贝茜移动所需的最小时间．

## 输入格式

第$\red{1}$行：$\red{3}$个用空格隔开的整数$\red{V，}$$\red{R，}$$\red{C，}$分别表示贝茜的初速度和区域的长度和宽度．

第$\red{2}$到$\red{R+1}$行：以矩阵的形式表示该区域中各块的高度．

## 输出格式

输出一个实数（保留$\red{2}$位小数），表示贝茜达到目的地最少需要的时间．

## 样例

#### 输入样例

```c
1 3 3
1 5 3 
6 3 5
2 4 3
```

#### 输出样例

```c
29.00
```

## 提示

输出详细信息：

贝西的最佳路线是：

以$\red{1,1}$时间$\red{0}$速度$\red{1}$启动

东至$\red{1,2}$时间$\red{1}$速度$\red{1/16}$

南至$\red{2,2}$时间$\red{17}$速度$\red{1/4}$

南至$\red{3,2}$时间$\red{21}$速度$\red{1/8}$

东至$\red{3,3}$时间$\red{29，}$速度$\red{1/4}$
