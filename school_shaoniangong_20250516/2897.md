## 题目描述

小 $\red{T }$有一个字符串 $\red{S，}$然后小 $\red{T }$把 $\red{S }$复制了一份在后面形成了 $\red{SS，}$然后又往里面的某个地方插入了一个字符变成了 $\red{T}$。

然后小 $\red{T }$忘了他插入的字符是什么以及在哪里了，你可以帮它复原 $\red{S }$吗？

## 输入格式

一行一个字符串，表示字符串 $\red{T，}$其长度为 $\red{n}$。

## 输出格式

输出字符串 $\red{S}$。

如果 $\red{S }$有不止一种，输出 $\red{-1}$。

如果 $\red{S }$不存在，输出 $\red{-2}$。

## 样例

#### 输入样例1

```c
ABCDXABCD
```

#### 输出样例1

```c
ABCD
```

#### 输入样例2

```c
ABCDE
```

#### 输出样例2

```c
-2
```

#### 输入样例3

```c
ABABA
```

#### 输出样例3

```c
-1
```

## 提示

对于 $\red{35\% }$的数据，$\red{2\le N\le 2001}$。

对于全部的数据，$\red{2\le N\le2,000,001}$。
