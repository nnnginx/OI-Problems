## 题目描述

$\red{Bessie }$想要驾驶她的宇宙飞船穿过一个危险的小行星带，该小行星带呈 $\red{N \times N }$网格 $\red{(1 <= N <= 500) }$的形状。网格包含 $\red{K }$个小行星 $\red{(1 <= K <= 10,000)，}$它们方便地位于网格的格点。

幸运的是，$\red{Bessie }$拥有强大的武器，可以一枪将网格中任何给定行或列中的所有小行星蒸发。


这把武器很贵，所以她想少用。给定场地中所有小行星的位置，找出 $\red{Bessie }$需要发射的最小数量来消除所有小行星。

## 输入格式

第$\red{1}$行：两个整数$\red{N}$和$\red{K，}$由单个空格分隔。

第$\red{2...K+1}$行：

每行包含两个空间分隔整数$\red{R}$和$\red{C（}$$\red{1<=R，}$$\red{C<=N）}$，分别表示小行星的行和列坐标。

## 输出格式

第$\red{1}$行：表示贝西必须拍摄的最小次数的整数。

## 样例

#### 输入样例

```c
3 4
1 1
1 3
2 2
3 2

```




#### 输出样例

```c
2
```

## 提示

输入详细信息：下图表示数据，其中"$\red{X}$"是一个小行星和"."是空白：

```
X.X
.X.
.X.
```

输出细节： $\red{Bessie }$可能会在第 $\red{1 }$行发射以摧毁 $\red{(1,1) }$处的小行星，并且 $\red{(1,3)，}$然后她可能会向第 $\red{2 }$列发射以摧毁小行星 在 $\red{(2,2) }$和 $\red{(3,2) }$处。
