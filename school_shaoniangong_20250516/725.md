## 题目描述

有一位使者要游历各国，他每到一个国家，都能学到一种文化，但他不愿意学习任何一种文化超过一次（即如果他学习了某种文化，则他就不能到达其他有这种文化的国家）。不同的国家可能有相同的文化。不同文化的国家对其他文化的看法不同，有些文化会排斥外来文化（即如果他学习了某种文化，则他不能到达排斥这种文化的其他国家）。

现给定各个国家间的地理关系，各个国家的文化，每种文化对其他文化的看法，以及这位使者游历的起点和终点（在起点和终点也会学习当地的文化），国家间的道路距离，试求从起点到终点最少需走多少路。

## 输入格式

第一行为五个整数 $\red{N，K，M，S，T}$，每两个整数之间用一个空格隔开，依次代表国家个数（国家编号为 $\red{1}$ 到 $\red{N}$），文化种数（文化编号为 $\red{1}$ 到 $\red{K}$），道路的条数，以及起点和终点的编号（保证 $\red{S}$ 不等于$\red{ T}$）；

第二行为 $\red{N}$ 个整数，每两个整数之间用一个空格隔开，其中第$\red{i}$ 个数 $\red{C_i}$，表示国家$\red{ i}$的文化为 $\red{C_i}$。

接下来的 $\red{K}$ 行，每行 $\red{K }$个整数，每两个整数之间用一个空格隔开，记第 $\red{i}$ 行的第 $\red{j}$ 个数为 $\red{a_{i,j}，a_{i,j}= 1}$ 表示文化$\red{ i }$排斥外来文化 $\red{j}$（$\red{i}$ 等于 $\red{j }$时表示排斥相同文化的外来人），$\red{a_i= 0}$ 表示不排斥（注意 $\red{i }$排斥 $\red{j}$ 并不保证 $\red{j}$ 一定也排斥 $\red{i}$）。

接下来的 $\red{M}$ 行，每行三个整数 $\red{u，v，d，}$每两个整数之间用一个空格隔开，表示国家 $\red{u}$与国家 $\red{v}$ 有一条距离为 $\red{d}$ 的可双向通行的道路（保证 $\red{u}$ 不等于 $\red{v}$，两个国家之间可能有多条道路）。

## 输出格式

输出只有一行，一个整数，表示使者从起点国家到达终点国家最少需要走的距离数（如果无解则输出$\red{-1}$）。

## 样例

## 样例输入

```C
2 2 1 1 2
1 2
0 1
1 0
1 2 10
```

#### 样例输出

```C
-1
```

## 数据范围与提示

【输入输出样例说明】

由于到国家 $\red{2}$ 必须要经过国家 $\red{1}$，而国家 $\red{2 }$的文明却排斥国家$\red{ 1}$ 的文明，所以不可能到达国家$\red{ 2}$。

【输入输出样例 $\red{2}$】

输入：

```C
2 2 1 1 2
1 2
0 1
0 0
1 2 10
```

输出：

```C
10
```

【输入输出样例说明】

路线为 $\red{1 -> 2}$。

【数据范围】

对于 $\red{20\%}$的数据，有 $\red{2≤N≤8，K≤5}$；

对于 $\red{30\%}$的数据，有 $\red{2≤N≤10，K≤5}$；

对于 $\red{50\%}$的数据，有 $\red{2≤N≤20，K≤8}$；

对于$\red{ 70\%}$的数据，有 $\red{2≤N≤100，K≤10}$；

对于 $\red{100\%}$的数据，有 $\red{2≤N≤100，1≤K≤100，1≤M≤N2，1≤ki≤K，1≤u, v≤N，1≤d≤1000，S≠T，1 ≤S, T≤N}$。
