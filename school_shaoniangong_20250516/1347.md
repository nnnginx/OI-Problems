## 题目描述

如题，现在有一个并查集，你需要完成合并和查询操作。

## 输入格式
第一行包含两个整数 $\red N$,$\red M$ ,表示共有 $\red N$ 个元素和 $\red M$ 个操作。

接下来 `M 行`，每行包含三个整数 $\red {Z_i,X_i,Y_i}$。

当 $\red{Z_i=1}$ 时，将 $\red{X_i}$ 与 $\red{Y_i}$ 所在的集合合并。

当 $\red{Z_i=2}$时，输出 $\red{X_i}$与 $\red{Y_i}$是否在同一集合内，是的输出 `Y` ；否则输出 `N` 。

## 输出格式

对于每一个 $\red{Z_i=2}$ 的操作，都有一行输出，每行包含一个大写字母，为 `Y` 或者 `N` 。

## 样例
#### 输入样例
```C
4 7
2 1 2
1 1 2
2 1 2
1 3 4
2 1 4
1 2 3
2 1 4
```
#### 输出样例
```C
N
Y
N
Y
```
## 提示


对于 $\red{100\%}$ 的数据，$\red{1\le N \le 10^4，1\le M \le 2\times 10^5
 ，1 \le X_i, Y_i \le N ，Z_i ∈{1,2}。}$
