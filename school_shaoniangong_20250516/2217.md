## 题目描述

奶牛贝西是纸牌游戏的超级粉丝，这很令人惊讶，因为她没有对生拇指。不幸的是，牛群中的其他奶牛都不是好对手。事实上，他们是如此糟糕，以至于他们总是以一种完全可以预测的方式打球。尽管如此，贝西仍 然面临着如何获胜的挑战。

贝西和她的朋友埃尔西目前正在玩一个简单的纸牌游戏，他们拿一副$\red{2N}$张牌，方便地编号为$\red{1…}$$\red{2N，}$然后将它们分成$\red{N}$张给贝西的牌和$\red{N}$张给埃尔西的牌。然后两人打$\red{N}$轮，每轮贝西和埃尔西都打一张牌。在前$\red{N/2}$轮中，牌数最高的玩家获得一分，在最后$\red{N/2}$轮中，规则切换，牌数最低的玩家获得一分。

鉴于贝西可以预测埃尔西打牌的顺序，请确定贝西可以赢得的最大点数。

## 输入格式

输入的第一行包含$\red{N}$的值$\red{（2≤}$$\red{N≤}$$\red{50,000}$; $\red{N}$将是偶数）。
接下来的$\red{N}$行包含$\red{Elsie}$将在游戏的每个连续回合中玩的牌。注意，根据这些信息很容易确定贝西的卡片。

## 输出格式

输出一条线，给出 $\red{Bessie }$可以得分的最大点数。

## 样例

#### 输入样例

```c
4
1
8
4
3
```

#### 输出样例

```c
2
```

## 提示

在这里，$\red{Bessie }$手上必须有 $\red{2}$、$\red{5}$、$\red{6 }$和 $\red{7 }$牌，她最多可以用这些牌赢得 $\red{2 }$分，方法是将她的"$\red{2}$"牌保存到下半场的其中一张牌中
