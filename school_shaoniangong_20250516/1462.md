## 题目描述

给定一个点集$\red U$以及之间的关系图$\red G$：
![img](./1462/file/1.png)

你需要找出满足下列两个条件的子集的个数

- `1`	这个子集内的任意两个点都没有边相连
- `2`	不能在加入一个点使得子集还是满足条件$\red 1$

## 输入格式

一个数$\red n$

## 输出格式

不同方案数

## 样例

#### 输入样例

```C
5
```

#### 输出样例

```C
4
```

## 提示


$\red {30\%}$		$\red{P<=20}$

$\red {100\%}$	$\red{P<=1000}$
