## 题目描述

给你$\red{k }$个串，每个串长度都是$\red{n ，}$

请问是否有可能这些串是同一个串交换两个位置的字符所产生的。

如果有可能，则输出"$\red{YES}$",否则输出"$\red{NO}$"

## 输入格式

每个测试点包含多组数据。

第一行一个整数 表示数据组数。

对于每组数据，共 $\red{k+1}$行。

第一行包含两个整数 和 表示字符串个数和字符串的长度。

接下来的$\red{k }$行包含字符串 $\red{s_1,s_2,s_3,...,s_k}$每个字符串均由 $\red{n}$个小写拉丁字母组成。

## 输出格式

无

## 样例

#### 输入样例

```c
3
3 4 
abac 
caab 
acba 
3 4 
kbbu 
kbub 
ubkb 
5 4 
abcd 
dcba 
acbd 
dbca 
zzzz
```

#### 输出样例

```c
YES 
YES 
NO
```

## 提示

对于$\red{30\%}$的数据，有$\red{1≤}$$\red{n≤}$$\red{10}$;

对于$\red{60\%}$的数据，有$\red{1≤}$$\red{n≤}$$\red{70}$;

对于$\red{100\%}$的数据，有$\red{1≤}$$\red{T≤}$$\red{5,1≤}$$\red{N,K≤}$$\red{2×}$$\red{10^3,k×}$$\red{n≤}$$\red{2×}$$\red{10^3.}$
