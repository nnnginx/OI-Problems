## 题目描述

原题来自：`POI 2008`

`Byteotia` 城市有 $\red{n}$ 个城镇，$\red{m}$ 条双向道路。每条道路连接两个不同的城镇，没有重复的道路，所有城镇连通。
输出 $\red{n }$个数，代表如果把与第$\red{ i }$个点连接的所有边去掉，将有多少对点不能互通。

## 输入格式

输入$\red{ n,m}$ 及 $\red{m}$ 条边。

## 输出格式

输出$\red{ n}$ 个数，代表如果把第$\red{ i }$个点连接的所有边去掉，将有多少对点不能互通。

## 样例

#### 输入样例

```c
5 5
1 2
2 3
1 3
3 4
4 5
```

#### 输出样例

```c
8
8
16
14
8
```

## 提示

$\red{n\le 10^5, m\le 5×10^5}$。
