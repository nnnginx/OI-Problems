## 题目描述

$\red {N}$个小朋友（编号为$\red {0,1,2,…,N-1}$）一起玩石头剪子布游戏。

其中一人为裁判，其余的人被分为三个组（有可能有一些组是空的），第一个组的小朋友只能出石头，第二个组的小朋友只能出剪子，第三个组的小朋友只能出布，而裁判可以使用任意手势。

你不知道谁是裁判，也不知道小朋友们是怎么分组的。

然后，孩子们开始玩游戏，游戏一共进行$\red {M}$轮，每轮从$\red {N}$个小朋友中选出两个小朋友进行猜拳。

你将被告知两个小朋友猜拳的胜负结果，但是你不会被告知两个小朋友具体使用了哪种手势。

比赛结束后，你能根据这些结果推断出裁判是谁吗？

如果可以的话，你最早在第几轮可以找到裁判。

## 输入格式

输入可能包含多组测试用例

每组测试用例第一行包含两个整数$\red {N}$和$\red {M}$。

接下来$\red {M}$行，每行包含两个整数$\red {a,b}$，中间夹着一个符号$\red {(‘>’,’=’,’<’)}$，表示一轮猜拳的结果。

两个整数为小朋友的编号,$\red {”a>b”}$表示$\red {a}$赢了$\red {b，”a=b”}$表示$\red {a}$和$\red {b}$平手，$\red {”a<b”}$表示$\red {a}$输给了$\red {b}$。

## 输出格式

每组测试用例输出一行结果，如果找到裁判，则输出裁判编号和确定轮数。

如果存在一个裁判但不能确定其具体编号则输出`“Can not determine”`。

如果根据输入得到的结果为确定没有裁判或者确定有多个裁判，则输出`“Impossible”`。

具体格式可参考样例。

## 样例

#### 输入样例

```c
3 3
0<1
1<2
2<0
3 5
0<1
0>1
1<2
1>2
0<2
4 4
0<1
0>1
2<3
2>3
1 0
```

#### 输出样例

```c
Can not determine
Player 1 can be determined to be the judge after 4 lines
Impossible
Player 0 can be determined to be the judge after 0 lines
```

## 提示

$\red {1≤N≤500}$,

$\red {0≤M≤2000}$
