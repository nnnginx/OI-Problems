## 题目描述

曾经发明了脑洞治疗仪与超能粒子炮的发明家 SHTSC 又公开了他的新发明：超能粒子炮・改——一种可以发射威力更加强大的粒子流的神秘装置。

超能粒子炮・改相比超能粒子炮，在威力上有了本质的提升。它有两个参数$\red{ n、k}$，它会向每个编号为 $\red{0 }$到 $\red{k}$（包含两端）的位置$\red{ i}$ 发射威力为$\red{ \mathrm{C}_n^i \mathbin{\mathrm{mod}} 2333 }$的粒子流。

现在 SHTSC 给出了他的超能粒子炮・改的参数，让你求出其发射的粒子流的威力之和除以$\red{ 2333 }$所得的余数。

## 输入格式

第一行一个整数$\red{ t }$表示数据组数。
之后$\red{ t}$ 行，每行两个整数 $\red{n、k}$，含义如题面描述。

## 输出格式

$\red{t }$行，每行一个整数，表示其粒子流的威力之和模$\red{ 2333 }$的值。

## 样例

#### 输入样例

```c
3
5 5
10 7
1145 14
```

#### 输出样例

```c
32
968
763
```
## 数据范围与提示

对于$\red{ 10\% }$的数据，$\red{t = 1}$，$\red{n, k \leq 1000}$；
对于$\red{ 30\% }$的数据，$\red{t = 1}$，$\red{n, k \leq 1000000}$；
对于 $\red{50\% }$的数据，$\red{t = 1}$，$\red{n \leq 10^{18}}$,$\red{ k \leq 1000}$；
对于 $\red{70\%}$ 的数据，$\red{t = 100}$，$\red{n, k \leq 10^{18}}$；
对于 $\red{100\% }$的数据，$\red{t = 100000}$，$\red{n, k \leq 10^{18}}$。
