## 题目描述

输入基数$\red{b}$（$\red{2<=b<=10}$）和正整数$\red{n}$（$\red{b}$进制），输出$\red{n}$的十进制表示。

## 输入格式

一行两个数，基数$\red{b}$（$\red{2<=b<=10}$）和正整数$\red{n}$（$\red{n}$的位数不超过九位)。

## 输出格式

$\red{n}$的十进制表示。

## 样例

#### 输入数据

```C
3 212
```

#### 输出数据

```C
23
```

## 提示

$\red{b}$进制整数转换为十进制整数

按权展开求和：把$\red{b}$进制数首先写成加权系数展开式，然后按十进制加法规则求和。
