


## 题目描述
在神秘的霍格沃茨魔法学校，新入学的学生们正在参加分院仪式。每位小巫师都有一个独特的"魔法年龄"，这个年龄决定了他们最适合学习哪种魔法。校长邓布利多想知道这批新生的平均魔法年龄，以便合理安排魔法课程。

## 输入格式
- 第一行是一个整数 `n` (1 ≤ n ≤ 100)，表示新生人数
- 接下来的 `n` 行，每行一个整数，表示每位学生的魔法年龄

## 输出格式
输出一个浮点数，表示所有新生的平均魔法年龄，**保留两位小数**

## 样例

### 输入样例
```
2
18
17
```

### 输出样例
```
17.50
```

## 魔法提示
1. 之前的题目，有可以完美满足保留两位小数的提示
2. 平均年龄的计算公式是：所有年龄之和 ÷ 人数
3. 记得使用浮点类型来存储总和，避免整数除法的问题



## 趣味知识
在魔法世界中，17岁被认为是成年的年龄，小巫师们从这个年龄开始可以合法地在校外使用魔法哦！
