## 题目描述

餐桌上，每一对筷子和勺子都有属于自己的编号。

并且每一对筷子和勺子都有一样的对应的编号。

现在有一对筷子的丢了，剩下都是正常的。

找出弄丢的筷子，并输出的筷子的编号

## 输入格式

第一行：一个整数$\red{N}$，有$\red{N}$个整数

第二行：$\red{N}$个数$\red{A_i}$（用空格隔开）

## 输出格式

数组中，只有一个数出现一次，剩下都出现三次，找出出现一次的数

结果肯定是正整数

## 样例

#### 输入样例：

```C
7
1 1 1 2 3 2 2
```

#### 输出样例：

```C
3
```

## 提示

对于$\red{40\%}$的数据: $\red{N <= 10^4}$,

对于$\red{100\%}$的数据: $\red{N <=10^6}$ , $\red{A_i < 2^{32}}$;
