## 题目描述

$\red{A、B、C、D、E}$为竞赛前五名，他们在名次公布之前猜名次。

$\red{A}$说：$\red{B}$得第三，$\red{C}$得第五；

$\red{B}$说：$\red{D}$得第二，$\red{E}$得第四；

$\red{C}$说：$\red{B}$得第一，$\red{E}$得第四；

$\red{D}$说：$\red{C}$得第一，$\red{B}$得第二；

$\red{E}$说：$\red{D}$得第二，$\red{A}$得第三。

公布结果后发现：每人只猜对了一半。请编程确定他们的名次。

## 输入格式

无 。

## 输出格式

$\red{A、B、C、D、E}$的名次（名次之间有一空格）。

## 样例

#### 输出样例

```c
无
```
