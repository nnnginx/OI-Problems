
#### 题目限制

1000 ms    256 M

#### 题目描述

我们将 $a$ 到 $z$ 这 $26$ 个字母的排列打乱顺序，得到一个新的字符串 $s$ （长度 $26$ ， 包括 $a$ 到 $z$ ）。再给出一个字符串 $t$ 。我们可以将 $s$ 重复任意多次，直到满足 $t$ 是 $s$ 的子序列为止，问最少需要让 $s$ 重复多少次？

例如： $s$ 为 $abcdefghijklmnopqrstuvwxyz$ ， $t$ 为 $aaa$ ，那么最少需要让 $s$ 重复 $3$ 次，才满足 $t$ 是 $s$ 的子序列。

#### 输入格式

第一行：包含 $26$ 个小写字母 ' $a'$ 到 ' $z'$ 的字符串 $s$ 。
第二行：一个只包含小写字母的字符串 $t$ ， $t$ 的长度小于等于 $1000$ 。

#### 输出格式

输出 $s$ 需要重复的最小次数。

#### 数据范围

对于 $10\%$ 的数据， $|t| \le 5$ 。

对于 $50\%$ 的数据，保证字母表顺序与通常的英文字母表相同。

对于 $100\%$ 的数据， $|t| \le 1000$ 。

#### 输入样例 1

```
abcdefghijklmnopqrstuvwxyz
mood
```

#### 输出样例 1

```
3
```
