## 题目描述

笨小猴的词汇量很小，所以每次做英语选择题的时候都很头疼。但是他找到了一种方法，经试验证明，用这种方法去选择选项的时候选对的几率非常大！

这种方法的具体描述如下：假设 $\red{maxn}$ 是单词中出现次数最多的字母的出现次数， $\red{minn}$ 是单词中出现次数最少的字母的出现次数，如果 $\red{maxn-minn }$是一个质数，那么笨小猴就认为这是个 $\red{Lucky\ Word}$ ，这样的单词很可能就是正确的答案。

## 输入格式

只有一行，是一个单词，其中只可能出现小写字母 。

## 输出格式

共两行，第一行是一个字符串，假设输入的的单词是 $\red{Lucky\ Word}$，那么输出$\red{“Lucky\ Word”}$，否则输出$\red{“No\ Answer”}$；

第二行是一个整数，如果输入单词是 $\red{Lucky\ Word}$ ，输出 $\red{maxn-minn}$ 的值，否则输出 $\red{0}$ 。

## 样例

#### 输入样例1

```c
error
```

#### 输出样例1

```c
Lucky Word
2
```

#### 输入样例2

```c
olympic
```

#### 输出样例2

```c
No Answer
0
```

## 提示

对于$\red{100\%}$ 的数据，单词长度小于 $\red{100}$。

【样例一说明】
单词 $\red{error}$ 中出现最多的字母 $\red{r}$ 出现了 $\red{3}$ 次，出现次数最少的字母出现了 $\red{1}$ 次， $\red{3-1=2}$ ， $\red{2}$ 是质数。
【样例二说明】
单词 $\red{olympic}$ 中出现最多的字母出现了 $\red{1}$ 次，出现次数最少的字母出现了 $\red{1}$ 次， $\red{1-1=0}$ ， $\red{0}$ 不是质数。
