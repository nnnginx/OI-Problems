## 题目描述

从文件读入两城市之间的距离（单位为千米）及旅行的速度（单位为千米/小时）和从某一城市出发的时间，编程计算出到达另一城市时间。输入时间用四位整数表示，最高两位表示小时，后两位表示分。如$\red{1805}$表示$\red{18}$点$\red{05}$分，到达时间用$\red{18-5}$表示，即时与分之间用`“-”`分开

## 输入格式

共一行，三个整数，分别表示出发时间、两城市之间距离及速度。

## 输出格式

共一行，到达时间与分之间用`“-”`连接，无小数位。

## 样例

#### 输入样例

```c
1805 100 50
```

#### 输出样例

```c
20-5
```
