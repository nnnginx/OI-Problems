## 题目描述

给你一个正整数 $\red{n}$（不带前导零），你能够删除$\red{n }$中若干个数位上的数，得到一个新的不带前导零的正整数。

求出最少删除多少位数使得最后的数是个平方数。

如果无法得到平方数，则输出$\red{-1}$。

## 输入格式

一个正整数$\red{n }$。

## 输出格式

一个正整数，最少需要删除多少位。

## 样例

#### 输入样例1

```c
8314
```

#### 输出样例1

```c
2
```

#### 输入样例2

```c
333
```

#### 输出样例2

```c
-1
```

## 提示

对于$\red{100\% }$数据，满足$\red{1<n<2\times10^9, }$
