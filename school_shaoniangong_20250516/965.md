## 题目描述

亚索被敌方英雄追击，一旦被他追上，亚索就会被控住，然后送人头。

所以亚索开始逃跑，亚索的速度为$\red{v_1}$，敌方英雄的速度为$\red{v_2}$，

只有亚索跑到防御塔内才能躲开，此时亚索和敌方英雄相距$\red{s_2}$，与防御塔相距$\red{s_1}$，

每当亚索判断会被敌方英雄追上时，就会用出`E`技能，即位移，位移的距离是$\red{15}$个单位长度。

请问亚索最少要用多少次位移才能逃开？（假设忽略亚索蓝的损耗）

## 输入格式

四个整数，分别是$\red{v_1}$,$\red{v_2}$,$\red{s_1}$,$\red{s_2}$（每个整数用一个空格隔开）

## 输出格式

一个整数，即亚索最少位移的次数

## 样例

#### 输入数据

```C
3 2 10 3
```

#### 输出数据

```C
0
```

## 提示



对于前$\red{25\%}$的数据，$\red{v_1 > v_2}$ 且$\red{s_1 < s_2}$

对于后$\red{75\%}$的数据，$\red{v_1 < v_2 < 15}$且$\red{s_1 < s_2}$且$\red{s_1}$,$\red{s_2}$皆在`int`的范围内
（当亚索位移时，他无法用原本的速度移动）
