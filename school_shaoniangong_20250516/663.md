## 题目描述

国际乒联现在主席沙拉拉自从上任以来就立志于推行一系列改革，以推动乒乓球运动在全球的普及。

其中$\red{11}$分制改革引起了很大的争议，有一部分球员因为无法适应新规则只能选择退役。

华华就是其中一位，他退役之后走上了乒乓球研究工作，意图弄明白$\red{11}$分制和$\red{21}$分制对选手的不同影响。

在开展他的研究之前，他首先需要对他多年比赛的统计数据进行一些分析，所以需要你的帮忙。

华华通过以下方式进行分析，首先将比赛每个球的胜负列成一张表，然后分别计算在$\red{11}$分制和$\red{21}$分制下，双方的比赛结果（截至记录末尾）。 

比如现在有这么一份记录，（其中$\red{W}$表示华华获得一分，$\red{L}$表示华华对手获得一分）： 

$\red{WWWWWWWWWWWWWWWWWWWWWWLW}$ 

在$\red{11}$分制下，此时比赛的结果是华华第一局$\red{11}$比$\red{0}$获胜，第二局$\red{11}$比$\red{0}$获胜，正在进行第三局，当前比分$\red{1}$比$\red{1}$。

而在$\red{21}$分制下，此时比赛结果是华华第一局$\red{21}$比$\red{0}$获胜，正在进行第二局，比分$\red{2}$比$\red{1}$。

如果一局比赛刚开始，则此时比分为$\red{0}$比$\red{0}$。 

你的程序就是要对于一系列比赛信息的输入（$\red{WL}$形式），输出正确的结果。注意：如果1局结束，两人的比分相差超过2才算结束。例如$\red{11}$分制：$\red{10：11}$不算结束，必须$\red{10：12}$才算结束。

## 输入格式

每个输入文件包含若干行字符串（每行至多$\red{20}$个字母），字符串由大写的$\red{W}$、$\red{L}$和$\red{E}$组成。

其中$\red{E}$表示比赛信息结束，程序应该忽略$\red{E}$之后的所有内容。

## 输出格式

输出由两部分组成，每部分有若干行，每一行对应一局比赛的比分（按比赛信息输入顺序）。

其中第一部分是$\red{11}$分制下的结果，第二部分是$\red{21}$分制下的结果，两部分之间由一个空行分隔。

## 样例

#### 输入样例

```c
WWWWWWWWWWWWWWWWWWWW
WWLWE
```

#### 输出样例

```c
11:0
11:0
1:1

21:0
2:1
```
