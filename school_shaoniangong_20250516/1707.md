## 题目描述

楚继光骑着战马站在一个$\red{N\times M}$的棋盘上的$\red{P}$点(设$\red{P}$点为纵轴中点)，而另一个点$\red{Q}$
(设$\red{Q}$点与$\red{P}$点横轴相同)为楚继光要到达的目的地,同时约定$\red{Q}$在$\red{P}$的右边,如图所
示,规则为马走“日”字,且只能向右走,请找出马从$\red{P}$到$\red{Q}$点所有通路的条数。

![img](./1707/file/11.png)

## 输入格式

两个整数,即$\red{M}$和$\red{N}$。

## 输出格式

通道数。

## 样例

#### 输入样例

```c
4 2
```

#### 输出样例

```c
2
```

## 提示

$\red{30\%}$数据中,$\red{N,M≤100}$

$\red{30\%}$数据中,$\red{N,M≤500}$

$\red{40\%}$数据中,$\red{N,M≤1000}$
