## 题目描述

$\red{FJ }$正在调查他的牛群以寻找最普通的奶牛。

他想知道这头"中位数"奶牛的产奶量：一半的奶牛产奶量与中位数一样多或更多；一半给予或多或少。给定奇数奶牛 $\red{N (1 <= N < 10,000) }$和它们的产奶量 $\red{(1...1,000,000)，}$求奶量的中位数，使得至少一半奶牛产奶量相同或更多，并且至少有一半给出相同或更少。

输入$\red{N}$个数，输出升序排列后中间那个数。

## 输入格式

第 $\red{1 }$行：单个整数 $\red{N }$

第 $\red{2...N+1 }$行：每行包含一个整数，表示一头奶牛的产奶量。

## 输出格式

第 $\red{1 }$行：一个整数，表示牛奶产量的中位数。

## 样例

#### 输入样例

```c
5
2
4
1
3
5

```


#### 输出样例

```c
3
```

## 提示

输入详细信息：五头产奶量为$\red{1...5}$的奶牛

输出详细信息：$\red{1}$和$\red{2}$低于$\red{3}$；$\red{4}$和$\red{5}$高于$\red{3}$。
