## 题目描述

原题来自：`POI 2006`

串是有限个小写字符的序列，特别的，一个空序列也可以是一个串。一个串 $\red{P}$ 是串 $\red{A}$ 的前缀，当且仅当存在串 $\red{B}$，使得 $\red{A=PB}$。如果 $\red{P \not=  A}$ 并且 $\red{P}$ 不是一个空串，那么我们说 $\red{P}$ 是 $\red{A}$ 的一个 $\red{proper}$ 前缀。

定义 $\red{Q}$ 是 $\red A$ 的周期，当且仅当 $\red{Q}$ 是 $\red{A}$ 的一个 $\red{proper}$ 前缀并且 $\red{A}$ 是 $\red{Q}$ 的前缀（不一定要是 $\red{proper}$ 前缀）。比如串 `abab` 和 `ababab` 都是串 `abababa` 的周期。串 $\red{A}$ 的最大周期就是它最长的一个周期或者是一个空串（当 $\red{A}$ 没有周期的时候），比如说，`ababab` 的最大周期是 `abab`。串 `abc` 的最大周期是空串。

给出一个串，求出它所有前缀的最大周期长度之和。

## 输入格式

第一行一个整数 $\red{k}$，表示串的长度。

接下来一行表示给出的串。

## 输出格式

输出一个整数表示它所有前缀的最大周期长度之和。

## 样例

#### 输入样例

```
8
babababa
```

#### 输出样例

```
24
```

## 提示

对于全部数据，$\red{1<k<10^6}$。
