## 题目描述

小明读一年级了，他拿了$\red M$块钱去买文具，笔记本$\red 5$元，铅笔$\red 3$元，橡皮$\red 1$元三个，如果他刚好能把钱花完，并且刚好买了$\red M$件物品，请输出他能有多少种购物方案。

例如输入$\red {100}$元，可以买$\red 4$本笔记本，$\red {18}$支铅笔，$\red {78}$块橡皮，方案可以写成$\red {4,18,78}$，除此方案外还可以$\red {0,25,75}$，$\red {8,11,81}$，$\red {12,4,84}$共$\red 4$种方案，输出$\red 4$。

## 输入格式

输入一行，表示钱的数量

## 输出格式

输出可行性方案数

## 样例

#### 输入样例

```C
100
```

#### 输出样例

```C
4
```
