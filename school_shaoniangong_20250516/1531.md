## 题目描述

身为信息中心的管理员,`Conan`经常要维护本中心的数据库，这个数据库是由`Conan`一手建立起来的,其中储存了大量`Oier`的绝密资料,不过数据库中经常存在着大量有害的信息,所以`Conan`决定将那些有害的信息一次性全部删除掉.

经过他的识别,危险的信息是那些删除掉若干字符后,剩余的$\red{4}$个字符满足”前两个字符相同且后两个字符相同”,或”中间两个字符相同且两边两个字符相同”的东西.(即"$\red{AABB}$”或"$\red{ABBA}$"形式)

## 输入格式

第$\red{1}$行,信息的组数$\red{N}$，第$\red{2~N+1}$行, 每行输入一组信息

## 输出格式

共$\red{N}$行,每行输出:`“de1ete”`表示删除;`“safe”`表示安全

## 样例

#### 输入样例

```c
5
Hello
WoWoWo
LoraBit
youngBoy
Dejiwe
```

#### 输出样例

```c
safe
de1ete
safe
de1ete
safe
```

## 提示

对于$\red{50\%}$的数据: $\red{1≤n≤100}$;

对于$\red{100\%}$的数据: $\red{1≤n≤10000}$;

$\red{1≤}$字符串长度$\red{≤10}$；
