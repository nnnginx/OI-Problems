## 题目描述

期末考试成绩出来了，我们把它分下类：

- $\red{90}$`~`$\red{100}$为"`Great`"
- $\red{70}$`~`$\red{89}$为"`Good`"
- $\red{60}$`~`$\red{69}$为"`Average`"
- $\red{0}$`~`$\red{59}$为"`Poor`"

## 输入格式

一行一个整数 $\red{n}$ ，为分数

## 输出格式

输出对应的等级

## 样例

#### 输入样例

```C
77
```

#### 输出样例

```C
Good
```

## 数据范围与提示

分数保证在$\red{0}$`~`$\red{100}$范围内
