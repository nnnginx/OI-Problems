## 题目描述

贝茜把$\red{N}$ ($\red{1 <= N <= 80}$)粒蓝色和橙色的珠子连成了一串，问有多少对珠子（相邻的）是不同颜色的。

## 输入格式

`第一行`: 一个整数$\red{N}$；

`第二行`: $\red{N}$个数字（$\red{0}$或$\red{1}$），其中$\red{0}$表示`橙色`，$\red{1}$表示`蓝色`。

## 输出格式

输出相邻两粒珠是不同颜色的对数。

## 样例

#### 输入数据

```C
6 
1 0 0 1 1 1
```

#### 输出数据

```C
2
```

## 提示

注意边界问题（入只有一颗珠子）
