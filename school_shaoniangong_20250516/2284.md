## 题目描述

每天，农夫约翰都会穿过牧场，检查每头奶牛的降状况。在他的农场里，他养了两头奶牛，霍尔斯泰牛和根西岛牛。他的$\red{H}$霍尔斯泰犬方便地编号为$\red{1…}$$\red{H，}$他的$\red{G}$根西岛犬方便地编 号为$\red{1…}$$\red{G（}$$\red{1≤}$$\red{H≤}$$\red{1000,1≤}$$\red{G≤}$$\red{1000). }$每个牛位于$\red{2D}$平面中的一个点上（不一定不同）。

农民约翰从霍尔斯泰因$\red{1}$号开始他的旅行，到霍尔斯泰因$\red{H}$号结束。他想沿途参观每头奶牛，为了方便维护他迄今为止参观过的奶牛清单，他想按编号顺序参观霍尔斯泰因和根西岛。在他访问的所有$\red{H+G}$奶牛的序列中，编号为$\red{1}$…$\red{H}$的霍尔斯泰牛应显示为（不一定是连续的）子序列，格恩西岛也是如此。否则，所有$\red{H+G}$奶牛的序列应通过将编号为$\red{1…}$$\red{H}$的霍尔斯泰牛列表与编号为$\red{1…}$$\red{G}$的格恩西岛牛列表交错排列而成。

当$\red{FJ}$从一头奶牛移动到另一头奶牛，移动距离为$\red{D}$时，他消耗$\red{D^2}$能量。请帮助他根据上述旅行确定拜访所有奶牛所需的最低能量。

## 输入格式

第一行输入包含$\red{H}$和$\red{G，}$由空格分隔。接下来的$\red{H}$线包含霍尔斯泰山脉的$\red{x}$和$\red{y}$坐标，之后的$\red{G}$线包含格恩西岛的坐标。每个坐标是$\red{0…}$$\red{1000}$范围内的整数

## 输出格式

写一行输出，给出$\red{FJ}$巡视所有奶牛所需的最小能量

## 样例

#### 输入样例

```c
3 2
0 0
1 0
2 0
0 3
1 3
```

#### 输出样例

```c
20
```
