## 题目描述

$\red{FJ}$的$\red{N(1 <= N <= 100)}$头奶牛们最近参加了场程序设计竞赛。在赛场上，奶牛们按$\red{1..N}$依次编号。

每头奶牛的编程能力不尽相同，并且没有哪两头奶牛的水平不相上下，也就是说，奶牛们的编程能力有明确的排名。 整个比赛被分成了若干轮，每一轮是两头指定编号的奶牛的对决。

如果编号为$\red{A}$的奶牛的编程能力强于编号为$\red{B}$的奶牛$\red{(1 <= A <= N}$; $\red{1 <= B <= N}$; $\red{A != B) ，}$那么她们的对决中，编号为$\red{A}$的 奶牛总是能胜出。

$\red{FJ}$想知道奶牛们编程能力的具体排名，于是他找来了奶牛们所有 $\red{M(1 <= M <= 4,500)}$轮比赛的结果，希望你能根据这些信息，推断出旧能多的奶牛的编程能力 排名。

比赛结果保证不会自相矛盾。

## 输入格式

第$\red{1}$行: $\red{2}$个用空格隔开的整数：$\red{N }$和 $\red{M}$

第$\red{M+1}$行: 每行为$\red{2}$个用空格隔开的整数$\red{A}$、$\red{B，}$描述了参加某一轮比赛的奶 牛的编号，以及结果

（编号为$\red{A，}$即为每行的第一个数的奶牛为 胜者）

## 输出格式

第$\red{1}$行: 输出$\red{1}$个整数，表示排名可以确定的奶牛的数目

## 样例

#### 输入样例

```c
5 5
4 3
4 2
3 2
1 2
2 5
```

#### 输出样例

```c
2
```

## 提示

输出说明:

编号为$\red{2}$的奶牛输给了编号为$\red{1}$、$\red{3}$、$\red{4}$的奶牛，也就是说她的水平比这$\red{3}$头奶牛都差。

而编号为$\red{5}$的奶牛又输在了她的手下，也就是说，她的水平比编号为$\red{5}$的奶牛强一些。

于是，编号为$\red{2}$的奶牛的排名必然为第$\red{4，}$编号为$\red{5}$的奶牛的水平必然最差。

其他$\red{3}$头奶牛的排名仍无法确定。
