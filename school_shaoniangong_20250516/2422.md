## 题目描述

$\red{Farmer John}$决定为他的所有奶牛都配备手机，以此鼓励她们互相交流。不过，为此$\red{FJ}$必须在奶牛们居住的$\red{N(1 <= N <= 10,000)}$块草地中选一些建上无 线电通讯塔，来保证任意两块草地间都存在手机信号。

所有的$\red{N}$块草地按$\red{1..N }$顺次编号。 所有草地中只有$\red{N-1}$对是相邻的，不过对任意两块草地$\red{A}$和$\red{B(1 <= A <= N}$; $\red{1 <= B <= N}$; $\red{A != B)，}$都可以找到一个以$\red{A}$开头以$\red{B}$结尾的草地序列，并且序列中相邻的编号所代表的草地相邻。

无线电通讯塔只能建在草地上，一座塔的服务范围为它所在的那块草地，以及与那块草地相邻的所有草地。

请你帮$\red{FJ}$计算一下，为了建立能覆盖到所有草地的通信系统，他最少要建多少座无线电通讯塔。

## 输入格式

第$\red{1}$行: $\red{1}$个整数，$\red{N}$

第$\red{2..N}$行: 每行为$\red{2}$个用空格隔开的整数$\red{A}$、$\red{B，}$为两块相邻草地的编号

## 输出格式

第$\red{1}$行: 输出$\red{1}$个整数，即$\red{FJ}$最少建立无线电通讯塔的数目

## 样例

#### 输入样例

```c
5
1 3
5 2
4 3
3 5
```

#### 输出样例

```c
2
```

## 提示

输入说明:

$\red{Farmer John}$的农场中有$\red{5}$块草地：草地$\red{1}$和草地$\red{3}$相邻，草地$\red{5}$和草地$\red{2}$、草地
$\red{4}$和草地$\red{3，}$草地$\red{3}$和草地$\red{5}$也是如此。更形象一些，草地间的位置关系大体如下：

（或是其他类似的形状）

$\red{4      2}$
$\red{|     |}$
$\red{1--3--5}$

输出说明:

$\red{FJ}$可以选择在草地$\red{2}$和草地$\red{3，}$或是草地$\red{3}$和草地$\red{5}$上建通讯塔。
