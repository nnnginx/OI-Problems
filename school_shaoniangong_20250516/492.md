## 题目描述

原题来自：`POI 2004`

`John` 打算驾驶一辆汽车周游一个环形公路。公路上总共有 $\red{n}$ 车站，每站都有若干升汽油（有的站可能油量为零），每升油可以让汽车行驶一千米。`John` 必须从某个车站出发，一直按顺时针（或逆时针）方向走遍所有的车站，并回到起点。在一开始的时候，汽车内油量为零，`John` 每到一个车站就把该站所有的油都带上（起点站亦是如此），行驶过程中不能出现没有油的情况。

任务：判断以每个车站为起点能否按条件成功周游一周。

## 输入格式

第一行是一个整数$\red{ n}$，表示环形公路上的车站数；

接下来$\red{ n}$ 行，每行两个整数 $\red{p_i,d_i}$​，分别表示表示第$\red{ i}$ 号车站的存油量和第$\red{ i}$ 号车站到下一站的距离。

## 输出格式

输出共 $\red{n}$ 行，如果从第 $\red{i }$号车站出发，一直按顺时针（或逆时针）方向行驶，能够成功周游一圈，则在第$\red{ i}$ 行输出 `TAK`，否则输出 `NIE`。

## 样例

#### 输入样例

```c
5
3 1
1 2
5 2
0 1
5 4
```

#### 输出样例

```c
TAK
NIE
TAK
NIE
TAK
```

## 提示

对于全部数据，$\red{3\le n\le 10^6,0\le p_i\le 2\times 10^9,0\lt d_i\le 2\times 10^9}$。
