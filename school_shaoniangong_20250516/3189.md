# 题目描述

众所周知，流萤变身萨姆时衣服是不见了的，准确来说流萤的衣服是被「火萤IV型战略强袭装甲」散发的火焰给烧没的。而流萤的衣服并没有被完全烧到灰都不剩，而是变成了一块块碎片。

这次流萤变身后衣服变成了一行 $n$ 块碎片，每块碎片都有一种颜色。每当有连续的超过 $k$ 片碎片，这些碎片就都会融化。如今开拓者手里有无数流萤的衣服碎片（别问哪来的，问就是偷偷在每次流萤变身完收集的）。

她想知道最少要在这行碎片中插入多少片以前收集的碎片才可以让这些碎片全部融化，算完后她会给你很多的橡木蛋糕卷作为奖励哦。

# 输入格式

第一行，两个整数 $N,K$；

第二行，$N$ 个数 $a_i$，不同的数表示不同的颜色。

# 输出格式

一个整数，代表答案。

# 样例

```input1
6 3
1 1 4 5 1 4
```

```output1
6
```

```input2
7 4
1 9 1 9 8 1 0
```

```output2
13
```

# 数据范围

对于所有数据，$1 \le N \le 100，2 \le K \le 5，1 \le a_i \le 100$。
