## 题目描述

求正整数$\red{N}$和$\red{M}$之间具有最多个真因子的数。对于整数$\red{P}$来说，真因子是指这样的因数：包含$\red{1}$但不包含$\red{P}$，对于特殊的整数$\red{1}$，我们认为它的真因子就是它本身。

## 输入格式

$\red{n}$,$\red{m}$

## 输出格式

真因子个数最多的数及真因子的个数(这样的数可能不止一个，输出其中最小的一个)

## 样例

#### 输入样例

```c
200 300
```

#### 输出样例

```c
240 19
```

## 提示

$\red{(1<=n<m<999999999,m-n<999999)}$
