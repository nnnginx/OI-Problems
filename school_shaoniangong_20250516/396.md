## 题目描述

`Lweb` 面对如山的英语单词，陷入了深深的沉思，「我怎么样才能快点学完，然后去玩三国杀呢？」。这时候睿智的凤老师从远处飘来，他送给了 `Lweb` 一本计划册和一大缸泡椒，然后凤老师告诉 `Lweb` ，我知道你要学习的单词总共有 $\red{n}$ 个，现在我们从上往下完成计划表，对于一个序号为 $\red{x}$ 的单词（序号 $\red{1 \ldots x-1 }$都已经被填入）：

`1.` 如果存在一个单词是它的后缀，并且当前没有被填入表内，那他需要吃 $\red{n×n}$ 颗泡椒才能学会；
`2.` 当它的所有后缀都被填入表内的情况下，如果在 $\red{1 \ldots x - 1}$ 的位置上的单词都不是它的后缀，那么他吃 $\red{x}$ 颗泡椒就能记住它；
`3.` 当它的所有后缀都被填入表内的情况下，如果$\red{ 1 \ldots x - 1 }$的位置上存在是它后缀的单词，所有是它后缀的单词中，序号最大为 $\red{y}$，那么他只要吃 $\red{x - y}$ 颗泡椒就能把它记住。

`Lweb` 是一个吃到辣辣的东西会暴走的奇怪小朋友，所以请你帮助 `Lweb`，寻找一种最优的填写单词方案，使得他记住这 $\red{n}$ 个单词的情况下，吃最少的泡椒。

## 输入格式

输入一个整数 $\red{n}$，表示 `Lweb` 要学习的单词数。接下来 $\red{n}$ 行，每行有一个单词（由小写字母构成，且保证任意单词两两互不相同）。

## 输出格式

`Lweb` 吃的最少泡椒数。

## 样例

#### 输入样例

```c
2
a
ba
```

#### 输出样例

```c
2
```

## 提示

$\red{1 \leq n \leq 100000}$，所有字符的长度总和 $\red{1 \leq |\text{len}| \leq 510000}$
