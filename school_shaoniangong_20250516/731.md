## 题目描述

一条单向的铁路线上,依次有编号为 $\red{1, 2, ..., n}$ 的 $\red{n} $个火车站。每个火车站都有一个级别,最低为 $\red{1}$ 级。现有若干趟车次在这条线路上行驶,每一趟都满足如下要求:如果这趟车次停靠了火车站$\red{x}$,则始发站、终点站之间所有级别大于等于火车站 $\red{x}$ 的都必须停靠。
(注意:起始站和终点站自然也算作事先已知需要停靠的站点)
例如,下表是 $\red{5}$ 趟车次的运行情况。其中,前 $\red{4} $趟车次均满足要求,而第$\red{ 5}$ 趟车次由于停靠了 $\red{3 }$号火车站($\red2$ 级)却未停靠途经的 $\red{6}$ 号火车站(亦为$\red{ 2}$ 级)而不满足要求。

![img](./731/file/1964.gif)

现有 $\red{m}$ 趟车次的运行情况(全部满足要求),试推算这 $\red{n }$个火车站至少分为几个不同的级别。

## 输入格式

第一行包含 $\red{2}$ 个正整数 $\red{n, m}$,用一个空格隔开。
第 $\red{i + 1}$ 行$\red{(1 ≤ i ≤ m)}$中,首先是一个正整数$\red{ s_i (2 ≤ s_i ≤ n)}$,表示第$\red{ i}$ 趟车次有 $\red{s_i }$个停靠站;接下来有$\red{ s_i }$个正整数,表示所有停靠站的编号,从小到大排列。每两个数之间用一个空格隔开。输入保证所有的车次都满足要求。

## 输出格式

输出只有一行,包含一个正整数,即$\red{ n }$个火车站最少划分的级别数。

## 样例

## 样例输入

```C
9 2
4 1 3 5 6
3 3 5 6
```

## 样例输出

```C
2
```

## 数据范围与提示

对于 $\red{20\%}$的数据,$\red{1 ≤ n, m ≤ 10}$;
对于$\red{ 50\%}$的数据,$\red{1 ≤ n, m ≤ 100}$;
对于 $\red{100\%}$的数据,$\red{1 ≤ n, m ≤ 1000}$
