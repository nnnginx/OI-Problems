## 题目描述

原题来自：`USACO 2005 Feb. Gold`

农夫约翰建造了一座有 $\red{n}$ 间牛舍的小屋，牛舍排在一条直线上，第 $\red{i}$ 间牛舍在 $\red{x_i}$ 的位置，但是约翰的 $\red{m}$ 头牛对小屋很不满意，因此经常互相攻击。约翰为了防止牛之间互相伤害，因此决定把每头牛都放在离其它牛尽可能远的牛舍。也就是要最大化最近的两头牛之间的距离。

牛们并不喜欢这种布局，而且几头牛放在一个隔间里，它们就要发生争斗。为了不让牛互相伤害。`John` 决定自己给牛分配隔间，使任意两头牛之间的最小距离尽可能的大，那么，这个最大的最小距离是多少呢？

## 输入格式

第一行用空格分隔的两个整数 $\red{n}$ 和 $\red{m}$；

第二行为 $\red{n}$ 个用空格隔开的整数，表示位置 $\red{x_i}$。

## 输出格式

输出仅一个整数，表示最大的最小距离值。

## 样例

#### 输入样例

```c
5 3
1 2 8 4 9
```

#### 输出样例

```c
3
```

把牛放在 $\red{1}$, $\red{4}$ ,$\red{8}$ 这样最小距离是 $\red{3}$ 。

## 提示

$\red{2 \leq n \leq 10^5, 0 \leq x_i \leq 10^9, 2\leq m \leq n}$。
