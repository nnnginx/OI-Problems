## 题目描述

判断某年是否是闰年。

## 输入格式

输入只有一行，包含一个整数 $\red{a(0 < a < 30000)}$

## 输出格式

一行，如果公元 $\red a$ 年是闰年输出 "$\red{Y}$"，否则输出 " $ \red{N} $ "。

## 样例

#### 样例输入

```C
2006
```

#### 样例输出

```C
N
```

## 提示

闰年是公历中的名词。闰年分为普通闰年和世纪闰年。

普通闰年:能被$\red{4}$整除但不能被$\red{100}$整除的年份为普通闰年。（如$\red{2004}$年就是闰年，$\red{1900}$年不是闰年）；

世纪闰年:能被$\red{400}$整除的为世纪闰年。（如$\red{2000}$年是世纪闰年，$\red{1900}$年不是世纪闰年）；
