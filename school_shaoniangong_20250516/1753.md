## 题目描述

`URAL 1167`

邪狼负责管理所自的战马,每天,他放出所有战马,任它们奔跑嬉戏。到了晚上,邪狼把
所有马带回马既.邪狼把它们排在一条直线上然后跟着他回厩。由于马儿们很累.邪狼尽量
不让它们移动。所以他发明了一个算法:他把前$\red{P}$匹马放在第一个马厩，下面$\red{P,}$匹马放
在第二个马厩，等等,而且.他不想$\red{K}$个马厩任何一个是空的,并且没有马被留在外边。邪
狼有黑白两种颜色的马,两种颜色的马相处不好。如果有$\red{i}$匹黑马和$\red{j}$匹白马同在一个马厩
中，那么这个马既的忧愁系数为$\red{i\times j，}$总忧愁系数是所有马既忧愁系数的和。忧愁系数过
大会表现在马儿互相打架或者马儿彻夜长嘶,这会引起修罗王的不满，邪狼可能要被惩罚，
后果您也可以想象出来,很严重的!故请决定一种方法把$\red{N}$匹马放进$\red{K}$个马厩使得总忧愁
系数最小。

## 输入格式

第一行是$\red{N(1≤}$$\red{N≤}$$\red{500)}$和$\red{K(1≤}$$\red{K≤}$$\red{N),}$下面$\red{N}$行有$\red{N}$个数第$\red{i}$行是第$\red{i}$匹马的颜
色,$\red{1}$代表黑色,$\red{0}$代表白色。

## 输出格式

最小的总忧愁系数.

## 样例

#### 输入样例

```c
63
1
1
0
1
0
1
```

#### 输出样例

```c
2
```

## 提示

将前$\red{2}$匹马放在第一个马厩，下面$\red{3}$匹马放在第二个马厩，最后一匹马放在第三个
马厩。
