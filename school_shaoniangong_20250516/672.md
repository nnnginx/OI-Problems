## 题目描述

我们可以把由$\red{“0”}$和$\red{“1”}$组成的字符串分为三类：全$\red{“0”}$串称为$\red{B}$串，全$\red{“1”}$串称为$\red{I}$串，既含$\red{“0”}$又含$\red{“1”}$的串则称为$\red F$串。

`FBI`树是一种二叉树，它的结点类型也包括$\red{F}$结点，$\red{B}$结点和$\red{I}$结点三种。由一个长度为$\red{2^N}$的$\red{“01”}$串$\red{S}$可以构造出一棵`FBI`树$\red T$，递归的构造方法如下：

- $\red{1}$. $\red{T}$的根结点为$\red{R}$，其类型与串$\red{S}$的类型相同；
- $\red{2}$. 若串$\red{S}$的长度大于$\red{1}$，将串$\red{S}$从中间分开，分为等长的左右子串$\red{S_1}$和$\red{S_2}$；由左子串$\red{S_1}$构造$\red{R}$的左子树$\red{T_1}$，由右子串$\red{S_2}$构造$\red{R}$的右子树$\red{T_2}$。

现在给定一个长度为$\red{2^N}$的$\red{“01”}$串，请用上述构造方法构造出一棵`FBI`树，并输出它的后序遍历
![img](./672/file/1.png)

## 输入格式

第一行是一个整数$\red{N（0 <= N <= 10）}$，

第二行是一个长度为$\red{2^N}$的$\red{“01”}$串。

## 输出格式

包括一行，这一行只包含一个字符串，即`FBI`树的后序遍历序列。

## 样例

#### 输入样例

```c
3
10001011
```

#### 输出样例

```c
IBFBBBFIBFIIIFF
```

## 提示

对于$\red{40\%}$的数据，$\red{N <= 2}$；

对于全部的数据，$\red{N <= 10}$。
