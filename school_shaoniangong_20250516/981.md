## 题目描述

孙老师讲授的《`C++`》这门课期中考试刚刚结束，

他想知道考试中取得的最高分数。因为人数比较多，他觉得这件事情交给计算机来做比较方便。

你能帮孙老师解决这个问题吗？

## 输入格式

输入两行，

`第一行`为整数$\red{n}$（$\red{1 ≤ n < 100000}$），表示参加这次考试的人数.

`第二行`是这$\red{n}$个学生的成绩，相邻两个数之间用单个空格隔开。

所有成绩均为$\red{0}$到$\red{100}$之间的整数。

## 输出格式

输出一个整数，即最高的成绩。

## 样例

#### 输入数据

```C
5
85 78 90 99 60
```

#### 输出数据

```C
99
```
