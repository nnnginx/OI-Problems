## 题目描述

小 $\red{T }$的画布上有一张无向图，每条边初始时都是白色的。

每当小 $\red{T }$发现一个节点的所有边中只有一条边是白色的，他就会把这条边涂黑。

显然，有的时候小 $\red{T }$并不能把所有的边涂黑，所以你可以使用你的魔法让其中的某些边变为黑色。

由于使用魔法是很累的，你想知道你最少需要使用多少次魔法。

## 输入格式

第一行两个正整数 $\red{n,m，}$表示无向图的点数与边数。

之后 $\red{m }$行，每行两个整数 $\red{u,v，}$表示一条边的两个端点。

## 输出格式

输出一个整数，表示最少使用魔法的次数

## 样例

#### 输入样例1

```c
3 3
1 2
2 3
1 3
```

#### 输出样例1

```c
1
```

#### 输入样例2

```c
3 2
1 2
2 3
```

#### 输出样例2

```c
0
```



## 提示

对于 $\red{30\% }$的数据，保证 $\red{n\le5}$。

对于另外 $\red{20\% }$的数据，保证是一个森林。

对于全部的数据，保证 $\red{1\le n,m\le 10^5，}$$\red{1\le u,v\le n，}$没有重边和自环。
