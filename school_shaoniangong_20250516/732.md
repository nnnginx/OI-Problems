## 题目描述

$\red{n}$ 个小伙伴（编号从 $\red{0}$ 到 $\red{n-1}$）围坐一圈玩游戏。按照顺时针方向给 $\red{n}$ 个位置编号，从 $\red{0}$ 到 $\red{n-1}$。最初，第 $\red{0}$ 号小伙伴在第 $\red{0}$ 号位置，第 $\red{1}$ 号小伙伴在第 $\red1$ 号位置，……，依此类推。

游戏规则如下：每一轮第 $\red{0}$ 号位置上的小伙伴顺时针走到第 $\red{m}$ 号位置，第 $\red{1}$ 号位置小伙伴走到第 $\red{m+1}$ 号位置，……，依此类推，第 $\red{n−m}$ 号位置上的小伙伴走到第 $\red{0}$ 号位置，第 $\red{n-m+1}$ 号位置上的小伙伴走到第 $\red{1}$ 号位置，……，第 $\red{n-1}$ 号位置上的小伙伴顺时针走到第 $\red{m-1}$ 号位置。


现在，一共进行了 $\red{10^k}$ 轮，请问 $\red{x}$ 号小伙伴最后走到了第几号位置。

## 输入格式

输入共 $\red{1}$ 行，包含 $\red{4}$ 个整数 $\red{n}$、$\red{m}$、$\red{k}$、$\red{x}$，每两个整数之间用一个空格隔开。

## 输出格式

输出共 $\red{1}$ 行，包含 $\red{1}$ 个整数，表示 $\red{10^k}$ 轮后 $\red{x}$ 号小伙伴所在的位置编号。

## 样例

## 样例输入 $\red1$

```C
10 3 4 5
```

## 样例输出 $\red1$

```C
5
```

## 数据范围与提示

对于 $\red{30\%}$ 的数据，$\red{0 < k < 7}$；

对于 $\red{80\%}$ 的数据，$\red{0 < k < 10^7}$；

对于$ \red{100\%}$ 的数据，$\red{1 < n < 10^6}$，$\red{0 < m < n}$，$\red{1 \leq x \leq n}$，$\red{0 < k < 10^9}$。
