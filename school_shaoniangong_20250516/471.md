## 题目描述

帅帅经常和同学玩一个矩阵取数游戏：对于给定的$\red{ n\times m}$ 的矩阵，矩阵中每个元素 $\red{a_{ij}​}$ 均为非负整数。游戏规则如下：

$\red{1}$. 每次取数时必须从每行各取走一个元素，共$\red{ n }$个，$\red{m }$次取完所有元素。

$\red{2}$. 每次取走的各个元素只能是该元素所在行行首或行尾。

$\red{3}$. 每次取数都有一个的分值，为每行取数得分之和，每行取数得分==被取走元素值$\red{\times 2^i}$，其中 $\red{i}$ 表示第 $\red{i}$ 次取数，从$\red{ 1 }$开始计数。

$\red{4}$. 游戏结束时，总得分为$\red{ m}$ 次取数得分之和。

帅帅想让你帮忙写一个程序，对于任意矩阵，可以求出取数后的最大得分。

## 输入格式

输入包括$\red{ n+1 }$行。 第一行两个空格隔开的正整数$\red{ n,m}$ 接下来$\red{ n }$行每行$\red{ m }$个用空格隔开的整数。

## 输出格式

输出为一个整数，为所输入矩阵取数后的最大得分

## 样例

#### 输入样例 1

```c
2 3
1 2 3
3 4 2
```

#### 输出样例 1

```c
82
```

第一次：第一行取行首元素，第二行取行尾元素，本次得分为$\red{ 1\times 2^1+2\times 2^1=6}$； 第二次：两行均取行首元素，本次得分为 $\red{2\times 2^2+3\times 2^2=20}$； 第三次：本次得分为 $\red{3\times 2^3+4\times 2^3=56}$，总得分为$\red{ 6+20+56=82}$。

#### 输入样例 2

```c
1 4
4 5 0 5
```

#### 输出样例 2

```c
122
```

#### 输入样例 3

```c
2 10
96 56 54 46 86 12 23 88 80 43
16 95 18 29 30 53 88 83 64 67
```

#### 输出样例 3

```c
316994
```

## 提示

对于$\red{ 60\% }$的数据，$\red{1\le n,m\le 30}$，答案不超过 $\red{10^{16}}$；

对于$\red{ 100\% }$的数据，$\red{1\le n,m\le 80,0\le a_{i,j}\le 1000}$。
