## 题目描述

给定整数$\red{n，}$问$\red{[\frac{n}{i}]}$的结果有多少个不同的数字。（$\red{1 ≤}$$\red{i≤}$$\red{n, i}$为整
数。)

比如$\red{n=5}$时，$\red{[\frac{5}{1}]}$$\red{= 5,[\frac{5}{2}]}$$\red{=2,[\frac{5}{3}]= 1,[\frac{5}{4}]}$$\red{= 1,[\frac{5}{5}]}$$\red{= 1,}$所以结果一
共有三个不同的数字。

注意$\red{32}$位整数的表示范围。

## 输入格式

一行一个整数$\red{n}$。

## 输出格式

一行一个整数答案。

## 样例

#### 输入样例

```c
5
```

#### 输出样例

```c
3
```

## 提示

对于$\red{30\%}$的数据，满足$\red{1≤}$$\red{n≤}$$\red{1000}$

对于$\red{60\%}$的数据，满足$\red{1≤}$$\red{n≤}$$\red{10^5 }$

对于$\red{100\%}$的数据，满足$\red{1≤}$$\red{n≤}$$\red{10^{12}}$。
