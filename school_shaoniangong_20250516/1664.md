## 题目描述

请你帮助解决形如$\red{ax^3＋bx^2＋cx＋d＝0}$这样的一个一元三次方程。给出该方程中各项的系数（$\red{a，b，c，d}$均为实数），并约定该方程存在三个不同实根（根的范围在$\red{-100}$至$\red{100}$，且根与根之差的绝对值$\red{≥1}$）。要求由小到大依次在同一行输出这三个实根（根与根之间留有空格，并精确到小数点后$\red{2}$位）。

## 输入格式

输入包含四个实数即$\red{a、b、c、d}$。

## 输出格式

输出为排好序的三个实根。

## 样例

#### 输入样例

```c
1   -5   -4   20
```

#### 输出样例

```c
-2.00   2.00   5.00
```
