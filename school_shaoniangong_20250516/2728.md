## 题目描述

贝西有两个酥脆的红苹果要送给牛群中的两个朋友。当然，她会走$\red{C (1 <= C <= 200,000) }$条牛道，这些牛道排列成通常的图表，连接 $\red{P (1 <= P <= 100,000) }$个牧场，方便地从 $\red{1..P }$编号：没有牛道从 $\red{a}$牧场到自己，牛道是双向的，每条牛道都有一个相关的距离，最重要的是，总是有可能从任何牧场到达任何其他牧场。

每条牛路连接两个不同的牧场$\red{P1_i (1 <= P1_i <= P) }$和 $\red{P2_i (1 <= P2_i <= P)，}$它们之间的距离为 $\red{D_i}$。所有距离 $\red{D_i }$的总和不超过 $\red{2,000,000,000}$。$\red{Bessie }$从牧场 $\red{PB (1 <= PB <= P) }$到牧场 $\red{PA1 (1 <= PA1 <= P) }$和 $\red{PA2 (1 < = PA2 <= P) }$以任何顺序排列。当然 ，这三个牧场都是不同的。考虑这张带括号的牧场数量和距离的牛道地图：

![img](./2728/file/1.png)

如果$\red{Bessie }$从牧场 $\red{[5] }$开始，将苹果送到牧场 $\red{[1] }$和 $\red{[4]，}$她的最佳路径是：$\red{5 -> 6-> 7 -> 4* -> 3 -> 2 -> 1* }$总共距离 $\red{12}$。

$\red{CLJ}$从$\red{b}$点（家）出发，既往$\red{Pa1}$点$\red{NOI}$赛场拿牌，也要去$\red{Pa2}$点$\red{CMO}$赛场拿牌。 $\red{CLJ}$当然可以总指挥家也可以派出$\red{CMO，}$当然可以先去$\red{NOI，}$当然可以先去。

## 输入格式

第$\red{1 }$行：第 $\red{1 }$行包含五个用空格分隔的整数：$\red{C}$、$\red{P}$、$\red{PB}$、$\red{PA1 }$和 $\red{PA2 }$

第$\red{2..C+1 }$行：第 $\red{i+1 }$行通过命名它连接的两个牧场以及它们之间的距离来描述牛径 $\red{i : P1_i, P2_i, D_i}$

## 输出格式

第$\red{1 }$行：$\red{Bessie }$运送两个苹果必须经过的最短距离

## 样例

#### 输入样例

```c
9 7 5 1 4
5 1 7
6 7 2
4 7 2
5 6 1
5 2 4
4 3 2
1 2 3
3 2 2
2 6 3
```

#### 输出样例

```c
12
```
