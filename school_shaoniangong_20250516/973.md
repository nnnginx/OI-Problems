## 题目描述

监护室每小时测量一次病人的血压，

若收缩压在$\red{90}$`-`$\red{140}$之间并且舒张压在$\red{60}$`-`$\red{90}$之间(包含端点值)则称之为正常，

现给出某病人若干次测量的血压值，计算病人保持正常血压的最长小时数。

## 输入格式

第一行为一个正整数$\red{n}$，其后有$\red{n}$行，每行$\red{2}$个正整数，分别为一次测量的收缩压和舒张压。

## 输出格式

输出仅一行，血压连续正常的最长小时数。

## 样例

#### 输入数据

```C
4
100 80
90 50
120 60
140 90
```

#### 输出数据

```C
2
```

## 提示

$\red{1 \le n<100}$
