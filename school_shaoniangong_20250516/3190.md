
#### 题目限制

1000 ms    256 M

#### 题目描述

给出了三个长度为 $N$ 的序列： $A,B,C$ ，由 $1$ 和 $N$ （包括 $1$ 和 $N$ ）之间的整数组成。求有多少整数对 $(i,j)$ 满足 $A_i=B_{c_j}$

#### 输入格式

第一行： $1$ 个数 $N$ ，表示顶点的数量。（ $1\le N\le 100000$ ）
之后 $N$ 行：每行 $3$ 个数，分别对应 $A_i,B_i,C_i$ 。

#### 输出格式

输出对应的答案

#### 数据范围

对于 $23\%$ 的数据， $1 \le N \le 5$ ；

对于 $100\%$ 的数据， $1 \le N \le 10^5, 1 \le A_i, B_i, C_i \le N$ 。

#### 输入样例 1

```
3
1 2 2
3 1 2
2 3 2
```

#### 输出样例 1

```
4
```
