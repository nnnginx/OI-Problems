## 题目描述

牛牛最近学习了C++入门课程，这门课程的总成绩计算方法是：

总成绩=作业成绩&times;$\red{20\%}$+小测成绩&times;$\red{30\%}$+期末考试成绩&times;$\red{50\%}$

牛牛想知道，这门课程自己最终能得到多少分。

## 输入格式

输入文件只有$\red{1}$行，包含三个非负整数$\red{A、B、C}$，分别表示牛牛的作业成绩、小测成绩和期末考试成绩。相邻两个数之间用一个空格隔开，三项成绩满分都是$\red{100}$分。

## 输出格式

输出文件只有$\red{1}$行，包含一个整数，即牛牛这门课程的总成绩，满分也是$\red{100}$分

## 样例

### 样例输入1

```c
100 100 80
```

### 样例输出1

```c
90
```

## 提示

对于$\red{30\%}$的数据，$\red{A=B=0}$。

对于另外$\red{30\%}$的数据，$\red{A=B=100。}$

对于$\red{100\%}$的数据，$\red{0\leq A、B、C \leq  100}$且$\red{A、B、C}$都是$\red{10}$的整数倍。
