## 题目描述

给定一个由不同的小写字母组成的字符串，输出这个字符串的所有全排列。

我们假设对于小写字母有$\red{ a < b < ... < y < z}$，而且给定的字符串中的字母已经按照从小到大的顺序排列。

## 输入格式

输入只有一行，是一个由不同的小写字母组成的字符串，已知字符串的长度在$\red{1}$ 到$\red 6$之间。

## 输出格式

输出这个字符串的所有排列方式，每行一个排列。要求字母序比较小的排列在前面。字母序如下定义：
已知$\red{ S = s_1 s_2 ...s_k , T = t_1 t_2 ...t_k }$，则$\red{S < T}$ 等价于，
存在$\red{p (1 <= p <= k)}$，使得
$\red{s_1 = t_1 , s_2 = t 2 , ..., sp - 1 = t p - 1 , sp < tp}$成立。

## 样例

#### 输入样例

```C
abc
```

#### 输出样例

```C
abc
acb
bac
bca
cab
cba
```
