## 题目描述

给定一个$\red{M}$行$\red{N}$列的$\red{01}$矩阵（只包含数字$\red{0}$或$\red{1}$的矩阵），再执行$\red{Q}$次询问，每次询问给出一个$\red{A}$行$\red{B}$列的$\red{01}$矩阵，求该矩阵是否在原矩阵中出现过。

## 输入格式

第一行四个整数$\red{M,N,A,B}$。

接下来一个$\red{M}$行$\red{N}$列的$\red{01}$矩阵，数字之间没有空格。

接下来一个整数$\red{Q}$。

接下来$\red{Q}$个$\red{A}$行$\red{B}$列的$\red{01}$矩阵，数字之间没有空格。

## 输出格式

对于每个询问，输出$\red{1}$表示出现过，$\red{0}$表示没有出现过。

## 样例

#### 输入样例

```c
3 3 2 2
111
000
111
3
11
00
11
11
00
11
```

#### 输出样例

```c
1
0
1
```

## 提示

$\red{A≤100，M,N,B≤1000，Q≤1000}$
