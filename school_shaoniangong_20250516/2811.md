## 题目描述

约翰的$\red{N(1≤}$$\red{N≤}$$\red{10^5)}$头奶牛站成一排，奶牛$\red{i}$的身高是$\red{Hi(1≤}$$\red{Hi≤}$$\red{1,000,000)}$．现在，每只奶牛都在向左看齐．对于奶牛$\red{i，}$如果奶牛$\red{j}$满足$\red{i<j}$且$\red{Hi<Hj，}$我们可以说奶牛$\red{i}$可以仰望奶牛$\red{j}$．    求出每只奶牛离她最近的仰望对象．

## 输入格式

第$\red{1}$行输入$\red{N，}$之后每行输入一个身高．

## 输出格式

共$\red{N}$行，按顺序每行输出一只奶牛的最近仰望对象．如果没有仰望对象，输出$\red{0}$．

## 样例

#### 输入样例

```c
6
3
2
6
1
1
2
```

#### 输出样例

```c
3
3
0
6
6
0
```

## 提示

$\red{N <= 10^5}$
