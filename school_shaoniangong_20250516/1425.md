## 题目描述

大面面是广州某中学的学生， 由于成绩优异办事能力强，获得了老师的青睐，成为了老师的左膀右臂。

经历过期末考试后，班主任想要统计该班总分排名靠前的学生的分数，于是把这个艰巨的任务交给了大面面，面面一筹莫展。

聪明的你能够帮助面面解决这个“世纪难题”吗？

## 输入格式

共两行

`第一行`为一个正整数$\red n$，表示该班的学生总数

`第二行`为$\red{n}$个正整数，$\red{a_i}$表示第$\red i$位学生本次期末考的总分数，用空格间隔

## 输出格式

共一行，表示本次学生的成绩，从高分往低分输出，每个分数间以空格隔开。

## 样例

#### 输入样例

```C
5
800 900 600 700 500
```

#### 输出样例

```C
900 800 700 600 500
```

## 提示

#### 提示

对于所有的数据
$\red{10 \le n \le 100,学生成绩0 \le a_i \le 1000}$
