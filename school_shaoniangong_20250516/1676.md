## 题目描述

修罗王解决了计算机的内存限制问题，终于可以使用电脑进行大型的魔法运算了，他交给邪狼的第一个任务是计算两个非负整数$\red{A、B}$的和，其中$\red{A}$和$\red{B}$的位数在$\red{5000}$位以内。

## 输入格式

共两行数据，第一行为一个非负整数$\red{A}$，第二行为一个非负整数$\red{B}$，$\red{A、B}$的位数均在$\red{5000}$以内。

## 输出格式

输出一个非负数，即两数之和。

## 样例

#### 输入样例

```c
1111111111

2222222222
```

#### 输出样例

```c
3333333333
```
