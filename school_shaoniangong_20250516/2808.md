## 题目描述

给你$\red{b}$个黑色和$\red{w}$个白色的的$\red{1\times 1}$的方块，如果想用他们密铺成黑白相间的正方形方格棋盘，能够得到的棋盘最大尺寸是多少呢？

## 输入格式

输入两个整数$\red{b}$和$\red{w}$

## 输出格式

输出一个正整数表示最大正方形棋盘的边长，如果不能得到正方形棋盘，则输出"$\red{Impossible}$".

## 样例

#### 输入样例1

```c
12 15
```

#### 输出样例1

```c
5
```

#### 输入样例2

```c
0 0
```

#### 输出样例2

```c
Impossible
```

## 提示

对于$\red{100\%}$的数据，满足$\red{0<=b,w<=10000}$
