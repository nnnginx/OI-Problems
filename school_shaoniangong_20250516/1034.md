## 题目描述

已知一个已经从小到大排序的数组，这个数组的一个平台（`Plateau`）就是连续的一串值相同的元素，并且这一串元素不能再延伸。

例如，在 `1，2，2，3，3，3，4，5，5，6 `中 `1` ，`2−2`，`3−3−3`，`4`，`5−5`，`6`都是平台。

试编写一个程序，接收一个数组，把这个数组最长的平台找出来。在上面的例子中`3−3−3`就是最长的平台。

## 输入格式

`第一行`有一个整数$\red{n}$，为数组元素的个数，$\red{n<1000}$。

`第二行`有$\red{n}$个整数，整数之间以一个空格分开。

## 输出格式

输出最长平台的长度。

## 样例

#### 输入样例

```C
10
1 2 2 3 3 3 4 5 5 6
```

#### 输出样例

```C
3
```
