## 题目描述

在一只大灰狼偷偷潜入$\red{Farmer Don}$的牛群被群牛发现后，贝西现在不得不履行着她站岗的职责。从她的守卫塔向下t望简直就是一件烦透了的事情。她决定做一些开发智力的小练习，防止她睡着了。

想象牧场是一个$\red{X，}$$\red{Y}$平面的网格。她将$\red{N}$只奶牛标记为$\red{1…}$$\red{N (1 <= N <= 100,000)，}$每只奶牛的坐标为$\red{X_i,Y_i (-100,000 <= X_i <= 100,000}$;$\red{-100,000 <= Y_i <= 100,000}$; $\red{1 <= i <=N)}$。然后她脑海里想象着所有可能由奶牛构成的三角形。

如果一个三角形完全包含了原点$\red{(0,0)，}$那么她称这个三角形为"黄金三角形"。原点不会落在任何一对奶牛的连线上。

另外，不会有奶牛在原点。给出奶牛的坐标，计算出有多少个"黄金三角形"。顺便解释一下样例，考虑五只牛，坐标分别为$\red{(-5,0), (0,2), (11,2), (-11,-6), (11,-5)}$ 。下图是由贝西视角所绘出的图示。
![img](./2656/file/1.jpg)

## 输入格式

第一行:一个整数: $\red{N }$

第$\red{2}$到第$\red{N+1}$行: 每行两个整数$\red{X_i，}$$\red{Y_i，}$表示每只牛的坐标

## 输出格式

第一行: 一行包括一个整数，表示"黄金三角形的数量"

## 样例

#### 输入样例

```c
5
-5 0
0 2
11 2
-11 -6
11 -5
```

#### 输出样例

```c
5
```
