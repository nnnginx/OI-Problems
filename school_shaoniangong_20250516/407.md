## 题目描述

`Farmer John` 要把他的牛奶运输到各个销售点。运输过程中，可以先把牛奶运输到一些销售点，再由这些销售点分别运输到其他销售点。

运输的总距离越小，运输的成本也就越低。`Farmer John` 期望低成本的运输，但他并不想让他的竞争对手知道他具体的运输方案，所以他希望采用费用第二小的运输方案而不是最小的。

现在请你帮忙找到该运输方案。

## 输入格式

第一行是两个整数 $\red{N,M}$，表示顶点数和边数；
接下来 $\red{M}$ 行每行$\red{3}$ 个整数，$\red{x,y,z}$，表示一条路的两端 $\red{x,y}$ 和距离 $\red{z}$。

## 输出格式

仅一行，输出第二小方案。

## 样例

#### 输入样例

```c
4 4
1 2 100
2 4 200
2 3 250
3 4 100
```

#### 输出样例

```c
450
```

## 提示

$\red{1 \le N \le 500,1 \le M \le 10^4,1 \le z \le 10^9}$，数据可能有重边。
