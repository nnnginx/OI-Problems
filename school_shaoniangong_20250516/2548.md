## 题目描述

农夫约翰有$\red{N(1≤}$$\red{N≤}$$\red{1000)}$头奶牛，每一头奶牛都有一个确定的独一无二的正整数产奶率．约翰想要让这些奶牛按产奶率从高到低排序．

约翰已经比较了$\red{M(1≤}$$\red{M≤}$$\red{10000)}$对奶牛的产奶率，但他发现，他还需要再做一张关于另外$\red{C}$对奶牛的产奶率比较，才能推断出所有奶牛的产 奶率排序．

请帮他确定$\red{C}$的最小值．

## 输入格式


第$\red{1}$行包含两个用空格分开的整数$\red{N}$和$\red{M.}$接下来$\red{M}$行，每行有两个用空格分开的整数$\red{X}$和$\red{Y(1≤}$$\red{X，}$$\red{y≤}$$\red{1000)，}$表示奶牛$\red{X}$的产奶率高于奶牛$\red{Y.}$


## 输出格式

$\red{C}$的最小值．

## 样例

#### 输入样例

```c
5 5
2 1
1 5
2 3
1 4
3 4
```

#### 输出样例

```c
3
```

## 提示

从输入样例中可以发现，约翰已经知道的排名有奶牛$\red{2>}$奶牛$\red{1>}$奶牛$\red{5}$和奶牛$\red{2>}$奶牛$\red{3>}$奶牛$\red{4，}$奶牛$\red{2}$排名第一．

但是他还需要知道奶牛$\red{1}$的名次是否高于奶牛$\red{3}$来确定排名第$\red{2}$的奶牛，假设奶牛$\red{1}$的名次高于奶牛$\red{3}$．

接着，他需要知道奶牛$\red{4}$和奶牛$\red{5}$的名次，假设奶牛$\red{5}$的名次高于奶牛$\red{4}$．在此之后，他还需要知道奶牛$\red{5}$的名次是否高于奶牛$\red{3}$．

所以，他至少仍需要知道$\red{3}$个关于奶牛的排名．

输入详细信息：

$\red{FJ}$正在比较$\red{5}$头奶牛，并且已经确定奶牛$\red{2>}$奶牛$\red{1}$、奶牛$\red{1>}$奶牛$\red{5}$、奶牛$\red{2>}$奶牛$\red{3}$、奶牛$\red{1>}$奶牛$\red{4}$和奶牛$\red{3>}$奶牛$\red{4（}$其中"$\red{>}$"符号表示"产奶更快"）。
