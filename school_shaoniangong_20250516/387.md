## 题目描述

原题来自：`BalticOI 2009`

给你一个字符串，它是由某个字符串不断自我连接形成的。但是这个字符串是不确定的，现在只想知道它的最短长度是多少。

## 输入格式

第一行给出字符串的长度 $\red{L}$，第二行给出一个字符串，全由小写字母组成。

## 输出格式

输出最短的长度。

## 样例

#### 输入样例

```c
8
cabcabca
```

#### 输出样例

```c
3
```

对于样例，我们可以利用 `abc` 不断自我连接得到 `abcabcabc`，读入的 `cabcabca` 是它的子串。

## 提示

对于全部数据，$\red{1\le L\le 10^6}$。
