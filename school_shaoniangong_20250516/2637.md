

# Description

（PS：帅的人已经做对这题了，而丑的人还在观望着）

从前有一个肥宅黄先生，一天他心血来潮地想去锻炼（其实是为了他的女神），然而他每天的体力值都不一样，他希望你能帮他找出他一天可以走多少条路。

# Format

## Input

有 n（1 ≤ n ≤ 10^5**1**0**5**）段路程，接下来 n 个数代表走完这段路需要的体力值 si（1 ≤ si ≤ 10000000）。

有 m（1 ≤ n ≤ 10^5**1**0**5**）天，接下来 m 行代表他每天的体力值 k（1 ≤ k ≤ 10000000）他一天可以走完小于等于他体力值 k 的路。

## Output

有 m 行，接下来每个数代表他那天可以走多少段路程。

# Samples

```input1
10 
1 1 2 2 2 3 3 3 4 4 
3 
1 
2 
3
```

```output1
2 
5 
8 
```
