## 题目描述

手写体的 "$\red{uu}$" 和 "$\red{w}$"，"$\red{nn}$" 和 "$\red{m}$" 简直让人难以辨认。

某人拿到了一个手写体字符串$\red{s ，}$她只能按照这个字符串某种可能的形式将其记
录下来。具体来说，对于字符串内的一个子串 "$\red{uu}$"，她有可能记录成 "$\red{w}$"，对于
子串 "$\red{w}$"，可能记录成 "$\red{uu}$"。同样对于子串 "$\red{nn}$"，可能记录成 "$\red{m}$"，对于子串
"$\red{m}$"，可能记录成 "$\red{nn}$"。而对于 $\red{u,w,n,m }$以外的其它字符，则不会出现记录错误。
更糟糕的是，她记录完后，下一个人按照她的记录进行再一次记录时又会产生同
样的错误。

现在拿到牛妹拿到了不知道被反复记录了多少次的字符串 $\red{t，}$万幸的是她知道了
这个字符串原本的长度为$\red{n，}$即$\red{|s|= n}$。她现在希望你帮她恢复出这个字符串$\red{s，}$
如果有多种可能性，任意一种符合题意的$\red{s}$都会被认为是正确的。

## 输入格式

第一行一个整数$\red{T，}$表示数据组数。

接下来每两行代表一组数据。对于每一组数据，

其第一行，一个正整数$\red{n ，}$表示原串$\red{s }$的长度；

其第二行，一个字符串$\red{t ，}$表示$\red{s}$经过反复记录后的字符串。保证$\red{t}$串仅由小写字
母组成。

## 输出格式

输出$\red{T}$行，第$\red{i}$行对应第$\red{i}$组数据的原串$\red{s}$。如果有多种可能性，任意一种符合题意
的$\red{s}$都会被认为是正确的。

数据保证存在至少一个答案。

## 样例

#### 输入样例

```c
3
5
abcw
7
xuwuxnmnx
3
wm
```

#### 输出样例

```c
abcuu
xwwxmmx
uum
```

## 提示

第一组数据，"$\red{abcuu}$" 被记录成 "$\red{abcw}$"。

第二组数据， "$\red{xwwxmmx}$" 被记录成 "$\red{xuuuuxnnnnx}$" ，再次被记录成
"$\red{xuwuxnmnx}$"。

第三组数据，"$\red{uum}$" 被记录成 "$\red{wm}$"，还有另一个可能的答案是 "$\red{wnn}$"。

数据范围

对于 $\red{20\%}$数据，满足$\red{1 ≤}$ $\red{n, | t |≤}$ $\red{10}$。

对于 $\red{50\%}$数据，满足$\red{1 ≤}$ $\red{n, | t |≤}$ $\red{1000}$。

对于另外 $\red{10\%}$数据，满足字符串$\red{t}$中不含有$\red{u, w, n, m}$。

对于 $\red{100\%}$数据，满足$\red{1 ≤}$ $\red{n, | t |≤}$ $\red{100000, 1 ≤}$ $\red{T ≤}$ $\red{10}$。

所有数据保证$\red{t}$中仅含有小写字母，保证至少存在一个答案。
