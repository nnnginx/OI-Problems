## 题目描述

运输公司对用户计算运费，路程(`km`)越远，每吨每公里运费越低。标准如下：

- $\red{s<250   \ \ \ \  \ \ \ \  }$  没有折扣
- $\red{250≤s<500  \ \   }$        $\red{ \qquad\ \ \ \ \     2\%}$折扣
- $\red{500≤s<1000}$   $\red{ 5\%}$折扣
- $\red{1000≤s<2000}$   $\red{  \ \ 8\%}$折扣
- $\red{2000≤s<3000 }$   $\red{10\%}$折扣
- $\red{3000≤s   }$   $\red{ \ \ \ 15\%}$折扣

设每吨货物每千米的运费$\red{p}$，货物重为$\red{w}$，运输距离为$\red{s}$，折扣为$\red{d}$，

则计算运费的公式为：$\red{cost=p * w * s * (1-d)}$

## 输入格式

共一行，三个数$\red{p}$、$\red{s}$、$\red{w}$

## 输出格式

共一行，运费，保留小数点后两位

## 样例

#### 输入样例

```C
100 300 20
```

#### 输出样例

```C
588000.00
```
