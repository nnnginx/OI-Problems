## 题目描述

给出一个电话列表，如果列表中存在其中一个号码是另一个号码的前缀这一情况，那么就称这个电话列表是不兼容的。

假设电话列表如下：

・$\red{Emergency 911}$

・$\red{Alice 97 625 999}$

・$\red{Bob 91 12 54 26}$

在此例中，报警电话号码$\red{（911）}$为$\red{Bob}$电话号码$\red{（91 ~12~ 54 ~26）}$的前缀，所以该列表不兼容。

## 输入格式

第一行输入整数$\red{t}$，表示测试用例数量。

对于每个测试用例，第一行输入整数$\red{n}$，表示电话号码数量。

接下来$\red{n}$行，每行输入一个电话号码，号码内数字之间无空格，电话号码不超过$\red{10}$位。

## 输出格式

对于每个测试用例，如果电话列表兼容，则输出$\red{”YES”}$。

否则，输出$\red{”NO”}$。

## 样例

#### 输入样例

```c
2
3
911
97625999
91125426
5
113
12340
123440
12345
98346
```

#### 输出样例

```c
NO
YES
```

## 提示

$\red{1≤t≤40}$,
$\red{1≤n≤10000}$
