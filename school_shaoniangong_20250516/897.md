## 题目描述

`Larry`今年毕业并找到了一份工作。

他赚很多钱，但似乎总是不够。

`Larry`认为他需要控制他的投资以解决自己的财务问题。

`Larry`拿到了自己的银行账户详单，想看看自己有多少钱。

请帮助`Larry`写一个程序，通过过去$\red{12}$个月中每月的月末结余，计算平均结余。

## 输入格式

输入包含`12行`，

每行包含一个数，为某个月的结余。

每个数都是不大于$\red{1,000,000}$的正数，

保留两位小数，省略"$"符。

## 输出格式

输出一个数，为这$\red{12}$个月的平均月末结余，

保留到小数点后第二位，并在最前面加一个"`$`"符。

## 样例

#### 输入数据

```C
100.00
489.12
12454.12
1234.10
823.05
109.20
5.27
1542.25
839.18
83.99
1295.01
1.75
```

#### 输出数据

```C
$1581.42
```
