## 题目描述

机器人灰灰在写作文，他的思路是这样的

在开始时，作文纸上什么都没有。

主人给他五种指令，如下：

- `1`、“`I x`”，在光标处插入数值$\red x$。
- `2`、“`D`”，将光标前面的第一个元素删除，如果前面没有元素，则忽略此操作。
- `3`、“`L`”，将光标向左移动，跳过一个元素，如果左边没有元素，则忽略此操作。
- `4`、“`R`”，将光标向右移动，跳过一个元素，如果右边没有元素，则忽略次操作。
- `5`、“`Q` `k`”，假设此刻光标之前的序列为$\red {a_1,a_2,…,a_n}$,输出$\red {max_{1≤i≤k}S_i}$，其中$\red {S_i=a_1+a_2+…+a_i}$。

## 输入格式

第一行包含一个整数$\red N$，表示主人有多少命令

接下来$\red Q$行，每行一个指令，具体指令格式如题目描述。

## 输出格式

每一个“`Q` `k`”指令，输出一个整数作为结果，每个结果占一行。

## 样例

#### 输入样例

```C
8
I 2
I -1
I 1
Q 3
L
D
R
Q 2
```

#### 输出样例

```C
2
3
```

## 提示

$\red {1≤Q≤10^6}$ ,
$\red {|x|≤10^3}$ ,
$\red {1≤k≤n}$
