## 题目描述

经过了几周的辛苦工作,贝茜终于迎来了一个假期.作为奶牛群中最会社交的牛,她希望去拜访$\red{N(1<=N<=50000)}$个朋友.

这些朋友被标号为$\red{1...N.}$这些奶牛有一个不同寻常的交通系统,里面有$\red{N-1}$条路,每条路连接了一对编号为$\red{C1}$和$\red{C2}$的奶牛$\red{(1 <= C1 <= N}$; $\red{1 <= C2 <= N}$; $\red{C1<>C2).}$这样,在每一对奶牛之间都有一条唯一的通路.

$\red{FJ}$希望贝茜眷的回到农场.于是,他就指示贝茜,如果对于一条路直接相连的两个奶牛,贝茜只能拜访其中的一个.

当然,贝茜希望她的假期越长越好,所以她想知道她可以拜访的奶牛的最大数目.

## 输入格式

第$\red{1}$行:单独的一个整数$\red{N }$

第$\red{2...N}$行:每一行两个整数,代表了一条路的$\red{C1}$和$\red{C2.}$

## 输出格式

单独的一个整数,代表了贝茜可以拜访的奶牛的最大数目.

## 样例

#### 输入样例

```c
7
6 2
3 4
2 3
1 2
7 6
5 6
```

#### 输出样例

```c
4
```

## 提示

输入详细信息：

贝西认识$\red{7}$头牛。奶牛$\red{6}$和$\red{2}$通过道路直接连接，如奶牛$\red{3}$和$\red{4}$、奶牛$\red{2}$和$\red{3}$等。下图描述了连接奶牛的道路：

```
                       1--2--3--4
                          |
                       5--6--7
```

输出详细信息：

贝西可以看望四头牛。

最好的组合包括两头奶牛在第一排，两个在底部。

从那以后她就不能去看望$\red{6}$号奶牛了这将排除拜访$\red{5}$和$\red{7}$头牛的可能性；因此，她访问了$\red{5}$和$\red{7.}$她还可以看望第一排的两头奶牛：$\red{\{1,3\}}$,$\red{\{1,4\}}$或$\red{\{2,4\}.}$
