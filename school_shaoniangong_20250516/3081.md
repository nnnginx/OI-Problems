# 题目描述

如下图所示，在一个坐标系中由实线和虚线标记出了一个边长为 n 的正方形，正方形的平行两条边存在 A点（0，y1）和 B点（n,y2）。

现在，从动点 A 沿着正方形的边缘往上或者往下走到定点 B ，不能回头。请判断出应该往哪个方向（往上或者往下）走到定点 B 的距离最近呢？

如果往上走的距离更近，那么输出“UP!”;

如果往下走的距离更近，那么输出“Down!”

如果往上和往下走的距离一样，那么输出“The same!”

![image](./3081/file/JLLmy6SOwPIRQ6TWYvpBv.jpeg)

## 输入

第一行，输入三个整数 n , y1 , y2 ，表示正方形的边长，A点距离 X 轴的高度和 B 点距离 X 轴的高度。

## 输出

如果往上走的距离更近，那么输出“UP!”; 如果往下走的距离更近，那么输出“Down!” ；如果往上和往下走的距离一样，那么输出“The same!”

# 样例

```input1
10 5 8
```

```output1
UP!
```
