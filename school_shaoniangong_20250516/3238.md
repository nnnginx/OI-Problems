## 题目描述

呼和同终于到达聚会场所了。

但是人太多，他去找自己的朋友。

可是每一个人都有属于自己的昵称(昵称不会超过10个单词，每个单词不回超过10个字符)

在参加聚会的时候他跟她的同学规定，昵称当中一定要有一个单词在互相名字当中出现才行。

聚会当中总共有M个人 (M 的范围 <= 10000)

请找出所有呼和有可能是呼和同学的好友

## 输入格式

第一行输入一个昵称。
第二行一个数M
接下来M行，每行一个昵称

## 输出格式

请输出huhe同学的所有好友数量

## 输入样例

```
hu he hao te 
5
hu bei
bei jing
he nan
nan jing 
te ke si
```

## 输出样例

```
3
```

## 提示

呼和同学的好友分别是hu bei","he nan" , "te ke si"

所有聚会昵称均为小写
