## 题目描述

给定一个$\red{m*n}$的矩阵$\red{A}$和$\red{r*s}$的矩阵$\red{B}$,其中$\red{0 < r ≤ m, 0 < s ≤ n}$，$\red{A}$、$\red{B}$所有元素值都是小于$\red{100}$的正整数。

求$\red{A}$中一个大小为$\red{r*s}$的子矩阵$\red{C}$，使得$\red{B}$和$\red{C}$的对应元素差值的绝对值之和最小，这时称$\red{C}$为最匹配的矩阵。

如果有多个子矩阵同时满足条件，选择子矩阵左上角元素行号小者，行号相同时，选择列号小者。

## 输入格式

`第一行`是$\red{m}$和$\red{n}$，以一个空格分开。

之后`m行`每行有$\red{n}$个整数，表示A矩阵中的各行，数与数之间以一个空格分开。

`第m+2`行为$\red{r}$和$\red{s}$，以一个空格分开。

之后`r行`每行有$\red{s}$个整数，表示$\red{B}$矩阵中的各行，数与数之间以一个空格分开。

（$\red{1 ≤ m ≤ 100,1 ≤ n ≤ 100}$）

## 输出格式

输出矩阵$\red{C}$，一共`r行`，每行$\red{s}$个整数，整数之间以一个空格分开。

## 样例

#### 输入样例

```C
3 3
3 4 5
5 3 4
8 2 4
2 2
7 3
4 9
```

#### 输出样例

```C
4 5 
3 4
```
