## 题目描述

一只乌龟由于智商低下，它只会向左或向右走，不过它会遵循主人小$\red{h}$的指令：$\red{F}$（向前走一步），$\red{T}$（掉头）。现在小$\red{h}$给出一串指令，由于小$\red{h}$有高超的计算能力，他可以马上知道乌龟最后走到哪里。为了难倒小$\red{h，}$他的好朋友小$\red{c}$就说，现在让你修改其中$\red{n}$个指令，使得乌龟移动到离起点最远的地方。（修改是指"$\red{T}$"变成"$\red{F}$"，或"$\red{F}$"变成"$\red{T}$",可以对同一个指令多次修改）。乌龟一开始在$\red{0}$点。

## 输入格式

第一行一个字符串$\red{S}$代表指令

接下来一行一个整数$\red{n，}$表示要修改的指令个数

## 输出格式

一行一个整数，表示乌龟所能移动到的最远距离。

## 样例

#### 输入样例1

```c
FT
1
```

#### 输出样例1

```c
2
```

#### 输入样例2

```c
FFFTFFF
2
```

#### 输出样例2

```c
6
```

## 提示

$\red{1<=len(s)<=100}$
$\red{1<=n<=50}$
