## 题目描述

一次考试之后，分获数学、物理、化学和英语第一的考生在一起猜测考试结果。

$\red{A}$说：$\red{D}$获得英语第一；

$\red{B}$说：$\red{C}$获得物理第一；

$\red{C}$说：$\red{A}$的数学成绩不会超过第三名；

$\red{D}$说：没有人的化学成绩能超过$\red{B}$。

公布结果后发现：只有获得数学、英语第一的人猜对了。请编程求出他们各获得哪科第一。

## 输入格式

无 。

## 输出格式

$\red{A、B、C、D}$各获得哪科第一（格式见样例）。

## 样例

#### 输出样例

```c
A:2 B:3 C:4 D:1
```

## 提示

$\red{1、2、3、4}$分别代表数学、物理、化学和英语
