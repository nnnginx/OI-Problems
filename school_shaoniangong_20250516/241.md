## 题目描述

奶牛们讨厌黑暗。

为了调整牛棚顶的电灯的亮度，`Bessie`必须建一座干草堆使得她能够爬上去够到灯泡。

一共有$\red {N}$大包的干草(从$\red {1}$到$\red {N}$编号)依靠传送带连续的传输进牛棚来。

第$\red {i}$包干草有一个宽度$\red {W_i}$。

所有的干草包的厚度和高度都为$\red {1}$。

`Bessie`必须利用所有$\red {N}$包干草来建立起干草堆。

她可以想放多少包就放多少包来建立起草堆的地基（当然是紧紧的放在一行中）。

接下来她可以将下一个草包放在之前一级的上方来建立新的一级。

## 输入格式

第$\red 1$行：一个整数$\red {N}$。

第$\red {2..N+1}$行：第$\red {i+1}$行包含整数$\red {W_i}$。

## 输出格式

输出一个整数，表示草包堆的最高的高度。

## 样例

#### 输入样例

```c
3
1
2
3
```

#### 输出样例

```c
2
```

## 提示

$\red {1≤N≤100000}$,

$\red {1≤W_i ≤10000}$

注意：每一级不能比下面的一级宽。

她持续的这么放置，直到所有的草包都被安置完成。

她必须按照草包进入牛棚的顺序堆放草包。

说得更清楚一些：一旦她将一个草包放在第二级 ，那么她不能将接下来的草包放在第一级上。

`Bessie`的目标是建立起最高的草包堆。
