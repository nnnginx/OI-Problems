## 题目描述

`POJ 2287`

“田忌赛马”是历史上有名地揭示如何善用自己的长处去对付对手的短处，从而在竞技中获胜的事例。当时田忌和齐王赛马，他们各派出$\red{N}$匹马$\red{（N≤2 000）}$。每场比赛，输的一方将要给赢的一方$\red{200}$两黄金，如果是平局的话，双方都不必拿出钱。

每匹马的速度值是固定而且已知的，而齐王出马也不管田忌的出马顺序。请问田忌该如何安排自己的马去对抗齐王的马，才能赢最多的钱？

## 输入格式

有多组数据，每一组数据第一行为一个正整数$\red{N（N≤1 000）}$，表示双方马的数量。第二行有$\red{N}$个整数表示田忌的马的速度。第三行的$\red{N}$个整数为齐王的马的速度。全部数据结束以$\red{0}$表示。

## 输出格式

每组数据输出一行，为田忌赛马可能赢得的最多的钱，结果有可能为负。

## 样例

#### 输入样例

```c
3

92 83 71

95 87 74

2

20 20

20 20

2

20 19

22 18

0
```

#### 输出样例

```c
200

0

0
```
