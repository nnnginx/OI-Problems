## 题目描述放苹果

`POJ 1664`

小光要把$\red{M}$个同样的苹果放在$\red{N}$个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的分法（$\red{5}$、$\red{1}$、$\red{1}$和$\red{1}$、$\red{5}$、$\red{1 }$是同一种分法）？

## 输入格式

第$\red{1}$行为一个整数$\red{t（}$$\red{0≤}$$\red{t≤}$$\red{20）}$，表示有$\red{t}$组测试数据，随后$\red{t}$行，每行为两个整数即$\red{M}$和$\red{N（}$$\red{1≤}$$\red{M，}$$\red{N≤}$$\red{20）}$。

## 输出格式

输出有多少种不同分法。

## 样例

#### 输入样例

```c
1　　
7 3
```

#### 输出样例

```c
8
```
