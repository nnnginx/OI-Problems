## 题目描述

设$\red{x1,x2,…,xn}$是直线上的$\red{n}$个点。用固定长度的闭区间覆盖这$\red{n}$个点，至少需要多少个这样的闭区间？设计解此问题的有效算法，并证明算法的正确性。
编程任务：对于给定的实直线上的$\red{n}$个点和闭区间的长度$\red{k}$，编程计算覆盖点集的最少区间数。

## 输入格式

第一行二个整数$\red{n,k}$，表示有$\red{n}$个点且固定闭区间的长度为$\red{k}$。接下来的一行是$\red{n}$个整数，表
示实直线上的点的坐标(可能相同)。

## 输出格式

最少闭区间数。

## 样例

#### 输入样例

```c
7 3
1 2 3 4 5 -2 6
```

#### 输出样例

```c
3
```
