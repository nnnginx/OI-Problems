## 题目描述

学校实行学分制。

每门的必修课都有固定的学分，同时还必须获得相应的选修课程学分。

学校开设了$\red { N }$门的选修课程，每个学生可选课程的数量 $\red {M}$ 是给定的。

学生选修了这 $\red {M}$ 门课并考核通过就能获得相应的学分。

在选修课程中，有些课程可以直接选修，有些课程需要一定的基础知识，必须在选了其他的一些课程的基础上才能选修。

例如《Windows程序设计》必须在选修了《Windows操作基础》之后才能选修。

我们称《Windows操作基础》是《Windows程序设计》的先修课。

每门课的直接先修课最多只有一门。

两门课可能存在相同的先修课。

你的任务是为自己确定一个选课方案，使得你能得到的学分最多，并且必须满足先修条件。

假定课程之间不存在时间上的冲突。

## 输入格式

输入文件的第一行包括两个整数$\red {N、M}$（中间用一个空格隔开）其中$\red {1≤N≤300,1≤M≤N}$。

接下来$\red {N}$行每行代表一门课，课号依次为$\red {1，2，…，N}$。

每行有两个数（用一个空格隔开），第一个数为这门课先修课的课号（若不存在先修课则该项为$\red {0}$），第二个数为这门课的学分。

学分是不超过$\red {10}$的正整数。

## 输出格式

输出一个整数，表示学分总数。

## 样例

#### 输入样例

```c
7 4
2 2
0 1
0 4
2 1
7 1
7 6
2 2
```

#### 输出样例

```c
13
```
