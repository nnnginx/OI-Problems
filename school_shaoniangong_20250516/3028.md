# Description

专家们曾经用各种理论证明拥有极高科技的外星人是不会入侵其他星球的，但他们忘了一句至理名言，凡事皆有可能!

例如牛皮无敌星人的灭绝是因为不者华会死星人发现牛皮无敌星人虽然一无所长，但那身怎么吹也吹不破的生皮可以做高级皮靴；之所以不作不死星人论为吃货星人的家畜，是因为吃货星人发现不作不死星人虽然没啥能耐还爱没事找事，但那一身的精肉还是鲜嫩可口的，而天顶星人入侵其他星球的目的是要通过特殊设备抽取他人的生命力(命源)以延续自己的生命。因此天顶星人自称农夫把被抽取生命力的外星球人称之为奶牛。已知一个天顶星人农夫有N(N≤1000)头奶牛，可由于产奶(抽取生命力)太少，他决定把当天产奶最少的牛杀掉，但他有点舍不得，如果当天不只一头奶生产奶最少，他便放过它们，这些奶牛生产奶是周期性的，他想知道有多少奶牛可幸免于难（可能全部被杀）,每头奶生周期不会超过10(每头奶生产奶量≤250)

# Format

## Input

第一行为一个整数T,表示数据总数，1≤T≤500:

每组数据的第一行为一个整数N,N≤1000,表示奶牛总数：随后N行为每头奶牛的周期天数以及每天的产奶数。

## Output

幸免于难的奶牛数以及最后一头牛被杀的天数。

# Samples

```input1
1
4
4 7 1 2 9
1 2
2 7 1
1 2
```

```output1
2 6
```

# TIPS

输出结果，2指的是最后剩下2头奶牛，6是指最后一头是第六天被杀的
