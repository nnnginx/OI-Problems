## 题目描述

又考试了，这次考试的人数特别多，每个人的学号很特别，是用字符串表示的（不超过
$\red{30 }$位），每次考试结束后，成绩统计是一件很重要的事情。

老师们都很关心学生的成绩，于是他们把学生的成绩按学号排列$\red{}$(字典顺序，学号全为
小写字母，从小到大排列$\red{}$)（不排成绩），并统计各个分数，及分数段的人数，以及满分人员
（满分要奖励 $\red{XXX }$奖学金的）。

## 输入格式

第一行：一个数 $\red{n }$（$\red{n<=130000 }$人）。

以下 $\red{n }$行：每行两个信息，分别为学号，分数（$\red{1\sim 150 }$分）。

## 输出格式

第一行：各个分数段（空格隔开）（例如 $\red{1\sim 10, 11\sim 20, ....}$(见例样，不包括 $\red{150 }$分
的人数）。

第二行：各个分数段的人数（空格隔开，没有则输出 $\red{0）}$。

接下来的 $\red{n }$行，分别为 $\red{n }$个学生的学号，成绩，（空格隔开）。

再接下来的一行为满分的人的人数 $\red{x（}$如果没有则为 $\red{0）}$（保证 $\red{x }$不超过 $\red{10000）}$。

接下来的 $\red{x }$行为满分人的学号（如果 $\red{x }$为 $\red{0 }$则为一行‘$\red{No}$’）（按字典序从小到大排序）。
、
注意：一行若有多个数据，每一个数据之间用空格隔开，最后一个数据后不允许有空格。
分数段中没有 $\red{0 }$分和 $\red{150 }$分的。

## 样例

#### 输入样例

```c
3
bid 50
wxy 85
alc 55
```

#### 输出样例

```c
1~9 10~19 20~29 30~39 40~49 50~59 60~69 70~79 80~89 90~99 100~109 110~119 120~129 
130~139 140~149
0 0 0 0 0 2 0 0 1 0 0 0 0 0 0
alc 55
bid 50
wxy 85
0
No
```
