## 题目描述

今天是贝茜的生日，为了庆祝自己的生日，贝茜邀你来玩一个游戏．

贝茜让$\red{N(1≤}$$\red{N≤}$$\red{100000)}$头奶牛坐成一个圈．除了$\red{1}$号与$\red{N}$号奶牛外，$\red{i}$号奶牛与$\red{i-l}$号和$\red{i+l}$号奶牛相邻．$\red{N}$号奶牛与$\red{1}$号奶牛相邻．农夫约翰用很多纸条装满了一个桶，每一张包含了一个独一无二的$\red{1}$到$\red{1,000,000}$的数字．

接着每一头奶牛$\red{i}$从柄中取出一张纸条$\red{Ai}$．每头奶牛轮流走上一圈，同时拍打所有编号能整除在纸条上的数字的牛的头，然后做回到原来的位置．

牛们希望你帮助他们确定，每一头奶牛需要拍打的牛．

## 输入格式

第$\red{1}$行包含一个整数$\red{N，}$

接下来第$\red{2}$到$\red{N+1}$行每行包含一个整数$\red{Ai}$．

## 输出格式


第$\red{1}$到$\red{N}$行，每行的输出表示第$\red{i}$头奶牛要拍打的牛数量．


## 样例

#### 输入样例

```c
5  
2
1
2
3
4
```



#### 输出样例

```c
2
0
2
1
3
```

## 提示

输入详细信息：

这$\red{5}$头牛分别被赋予数字$\red{2}$、$\red{1}$、$\red{2}$、$\red{3}$和$\red{4}$。

输出详细信息：


第一头牛拍拍第二头牛和第三头牛；第二只奶牛不拍奶牛；等
