## 题目描述

去年偶们湖南遭受$\red{N}$年不遇到冰冻灾害，现在芙蓉哥哥则听说另一个骇人听闻的消息： 一媛流星雨即将袭击整个霸中，由于流星体积过大，它们无法在撞击到地面前燃烧 殆尽， 届时将会对它撞到的一切东西造成毁灭性的打击。

很自然地，芙蓉哥哥开始担心自己的 安全问题。以霸中至$\red{In}$型男名誉起誓，他一定要在被流星砸到前，到达一个安全的地方 （也就是说，一块不会被任何流星砸到的土 地）。如果将霸中放入一个直角坐标系中， 芙蓉哥哥现在的位置是原点，并且，芙蓉哥哥不能踏上一块被流星砸过的土地。

根据预 报，一共有$\red{M}$颗流星$\red{(1 <= M <= 50,000)}$会坠落在霸中上，其中第$\red{i}$颗流星会在时刻 $\red{T_i (0 <= T_i <= 1,000)}$砸在坐标为$\red{(X_i, Y_i) (0 <= X_i <= 300}$；$\red{0 <= Y_i <= 300) }$的格子里。流星的力量会将它所在的格子，以及周围$\red{4}$个相邻的格子都化为焦土，当然 芙蓉哥哥也无法再在这些 格子上行走。

芙蓉哥哥在时刻$\red{0}$开始行动，他只能在第一象限中， 平行于坐标轴行动，每$\red{1}$个时刻中，他能移动到相邻的（一般是$\red{4}$个）格子中的任意一个， 当然目标格子要没有被烧焦才行。如果一个格子在时刻$\red{t}$被流星撞击或烧焦，那么芙蓉哥哥 只能在$\red{t}$之前的时刻在这个格子里出现。

请你计算一下，芙蓉哥哥最少需要多少时间才能到 达一个安全的格子。

## 输入格式

第$\red{1}$行: $\red{1}$个正整数：$\red{M}$

第$\red{2..M+1}$行:

第$\red{i+1}$行为$\red{3}$个用空格隔开的整数：$\red{X_i，}$$\red{Y_i，}$以及$\red{T_i}$

## 输出格式

输出$\red{1}$个整数，即芙蓉哥哥逃生所花的最少时间。

如果芙蓉哥哥无论如何都无法在流星雨中存活下来，输出$\red{-1}$

## 样例

#### 输入样例

```c
4
0 0 2
2 1 2
1 1 2
0 3 5
```

#### 输出样例

```c
5
```

## 提示

输入说明:

一共有$\red{4}$颗流星将坠落在霸中，它们落地点的坐标分别是$\red{(0, 0)，}$$\red{(2, 1)，}$$\red{(1, 1)}$
以及$\red{(0, 3)，}$时刻分别为$\red{2，}$$\red{2，}$$\red{2，}$$\red{5}$。
