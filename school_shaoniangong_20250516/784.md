## 题目描述

$\red Z$ 国有 $\red{n}$ 座城市，$\red{n-1}$ 条双向道路，每条双向道路连接两座城市，且任意两座城市都能通过若干条道路相互到达。

$\red Z$ 国的国防部长小 $\red Z$ 要在城市中驻扎军队。驻扎军队需要满足如下几个条件：

- 一座城市可以驻扎一支军队，也可以不驻扎军队。
- 由道路直接连接的两座城市中至少要有一座城市驻扎军队。
- 在城市里驻扎军队会产生花费，在编号为 $\red{i}$ 的城市中驻扎军队的花费是 $\red{p_i}$。

小 $\red Z$ 很快就规划出了一种驻扎军队的方案，使总花费最小。但是国王又给小 $\red Z$ 提出了 $\red{m}$ 个要求，每个要求规定了其中两座城市是否驻扎军队。小 $\red Z$ 需要针对每个要求逐一给出回答。具体而言，如果国王提出的第 $\red{j}$ 个要求能够满足上述驻扎条件（不需要考虑第 $\red{j}$ 个要求之外的其它要求），则需要给出在此要求前提下驻扎军队的最小开销。如果国王提出的第 $\red{j}$ 个要求无法满足，则需要输出 $\red{-1}$。现在请你来帮助小 $\red Z$。

## 输入格式

第一行包含两个正整数 $\red{n,m}$ 和一个字符串 $\red{\text{type}}$，分别表示城市数、要求数和数据类型。$\red{\text{type}}$ 是一个由大写字母 $\red{A，B}$ 或 $\red{C}$ 和一个数字 $\red{1，2，3}$ 组成的字符串。它可以帮助你获得部分分。你可能不需要用到这个参数。这个参数的含义在「数据范围与提示」中有具体的描述。
第二行 $\red{n}$ 个整数 $\red{p_i}$，表示编号 $\red{i}$ 的城市中驻扎军队的花费。
接下来 $\red{n-1}$ 行，每行两个正整数 $\red{u,v}$，表示有一条 $\red{u}$ 到 $\red{v}$ 的双向道路。
接下来 $\red{m}$ 行，第 $\red{j}$ 行四个整数 $\red{a,x,b,y}$ ($\red{a\neq b}$)，表示第 $\red{j}$ 个要求是在城市 $\red{a}$ 驻扎 $\red{x}$ 支军队，在城市 $\red{b}$ 驻扎 $\red{y}$ 支军队。其中，$\red{x,y}$ 的取值只有 $\red{0}$ 或 $\red{1}$：若 $\red{x}$ 为 $\red{0}$，表示城市 $\red{a}$ 不得驻扎军队，若 $\red{x}$ 为 $\red{1}$，表示城市 $\red{a}$ 必须驻扎军队；若 $\red{y}$ 为 $\red{0}$，表示城市 $\red{b}$ 不得驻扎军队，若 $\red{y}$ 为 $\red{1}$，表示城市 $\red{b}$ 必须驻扎军队。

输入文件中每一行相邻的两个数据之间均用一个空格分隔。

## 输出格式

输出共 $\red{m}$ 行，每行包含一个整数，第 $\red{j}$ 行表示在满足国王第 $\red{j}$ 个要求时的最小开销，如果无法满足国王的第 $\red{j}$ 个要求，则该行输出 $\red{-1}$。

## 样例

#### 输入样例

```c
5 3 C3
2 4 1 3 9
1 5
5 2
5 3
3 4
1 0 3 0
2 1 3 1
1 0 5 0
```

#### 输出样例

```c
12
7
-1
```

#### 样例说明

对于第一个要求，在 $\red{4}$ 号和 $\red{5}$ 号城市驻扎军队时开销最小。

对于第二个要求，在 $\red{1}$ 号、$\red{2}$ 号、$\red{3}$ 号城市驻扎军队时开销最小。

第三个要求是无法满足的，因为在 $\red{1}$ 号、$\red{5}$ 号城市都不驻扎军队就意味着由道路直接连接的两座城市中都没有驻扎军队。

## 提示

对于全部数据，$\red{n=m\le 10^5,1\le p_i\le 10^5}$。

| 测试点编号 | $\red{\text{type}}$ | $\red{n=}$ |
|:-:|:-:|:-:|
| $\red{1,2}$ | $\red{A3}$ | $\red{10}$ |
| $\red{3,4}$ | $\red{C3}$ | $\red{10}$ |
| $\red{5,6}$ | $\red{A3}$ | $\red{100}$ |
| $\red{7}$ | $\red{C3}$ | $\red{100}$ |
| $\red{8,9}$ | $\red{A3}$ | $\red{2\times 10^3}$ |
| $\red{10,11}$ | $\red{C3}$ | $\red{2\times 10^3}$ |
| $\red{12,13}$ | $\red{A1}$ | $\red{10^5}$ |
| $\red{14\sim 16}$ | $\red{A2}$ | $\red{10^5}$ |
| $\red{17}$ | $\red{A3}$ | $\red{10^5}$ |
| $\red{18,19}$ | $\red{B1}$ | $\red{10^5}$ |
| $\red{20,21}$ | $\red{C1 }$| $\red{10^5}$ |
| $\red{22}$ | $\red{C2}$ | $\red{10^5}$ |
| $\red{23\sim 25}$ | $\red{C3}$ | $\red{10^5}$ |

数据类型的含义：

$\red{A}$：城市 $\red{i}$ 与城市 $\red{i+1}$ 直接相连。
$\red{B}$：任意城市与城市 $\red{1}$ 的距离不超过 $\red{100}$（距离定义为最短路径上边的数量），即如果这棵树以 $\red{1}$ 号城市为根，深度不超过 $\red{100}$。
$\red{C}$：在树的形态上无特殊约束。
$\red 1$：询问时保证 $\red{a=1,x=1}$，即要求在城市 $\red{1}$ 驻军。对 $\red{b,y}$ 没有限制。
$\red 2$：询问时保证 $\red{a,b}$ 是相邻的（由一条道路直接连通）
$\red 3$：在询问上无特殊约束。
