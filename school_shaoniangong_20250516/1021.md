## 题目描述

在线性代数、计算几何中，向量点积是一种十分重要的运算。

给定两个$\red{n}$维向量 $\red{a =(a_1 ,a_2 ,...,a_n )}$和 $\red{b =(b_1 ,b_2 ,...,b_n )}$，

求点积 $\red{a ・ b = a_1*b _1 +a_2*b _2 +...+ a_n*b _n}$ 。

## 输入格式

`第一行`是一个整数n。$\red{1 <= n <= 1000}$。

`第二行`包含$\red{n}$个整数$\red{a_1,a_2,...,a_n}$。

`第三行`包含$\red{n}$个整数$\red{b_1,b_2,...,b_n}$。

相邻整数之间用单个空格隔开。

每个整数的绝对值都不超过$\red{1000}$。

## 输出格式

一个整数，即两个向量的点积结果。

## 样例

#### 输入样例

```C
3
1 4 6
2 1 5
```

#### 输出样例

```C
36
```
