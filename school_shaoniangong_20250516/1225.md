## 题目描述

给$\red{N}$个整数 $\red{X_i}$（十进制）

消去$\red{X_i}$（二进制下）的最后一位$\red{1}$

输出消去最后一个$\red{1}$的$\red{X_i}$（十进制）

## 输入格式

第一行一整数 $\red{N}$

接下来`N行`每行一个$\red{X_i}$

## 输出格式

输出`N行`结果

## 样例

#### 输入样例

```C
5
5
233
56
99
64
```

#### 输出样例

```C
4
232
48
98
0
```



## 数据范围与提示

- $\red{5}$(`110`) $\red{ ->}$ $\red{4}$(`100`)
- $\red{233}$(`11101001`) $\red{->}$ $\red{232}$(`11101000`)
- $\red{56}$(`111000`) $\red{->}$ $\red{48}$(`110000`)
- $\red{99}$(`1100011`) $\red{->}$ $\red{98}$(`1100010`)
- $\red{64}$(`1000000`) $\red{->}$ $\red{0}$(`0000000`)

$\red{60\%}$数据 ，  $\red{N <= 10^{4}}$

$\red{100\%}$数据 ， $\red{N <= 10^{6}}$， $\red{X_i}$在`int`内
