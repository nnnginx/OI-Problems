## 题目描述

$\red{Chen.Jr}$大学毕业后进入了一家生产纸箱子的工厂上班。 所有的纸箱子最初都是一整块纸板通过剪裁和折叠生产出来的。在工作的过程中，他遇到了一个非常复杂的问题：他拿到了一个尺寸为$\red{w\times h}$的纸板，那么你能不能帮他判断一下可否通过剪裁和折叠得到一个$\red{a\times b\times c}$的纸箱呢？

例如他有一个$\red{5\times 6}$的纸板想要得到$\red{1\times 2\times 3}$的纸箱方案可以如下所示：

![img](./2573/file/1.png)

可以参考如下所示的展开方式：

![img](./2573/file/2.png)

## 输入格式

输入包含两行；

第一行包含三个整数$\red{a,b,c}$;

第二行包含两个整数$\red{w,h.}$

## 输出格式

如果可以的话，输出"$\red{Yes}$"；否则输出"$\red{No}$"

## 样例

#### 输入样例1

```c
1 2 3
6 5
```

#### 输出样例1

```c
Yes
```

#### 输入样例2

```c
1 2 3
5 5
```

#### 输出样例2

```c
No
```

## 提示

对于$\red{100\%}$的数据所有的整数都不超过$\red{10^8}$
