## 题目描述

有些数可以表示成若干个不同阶乘的和。

例如，`9=1！+2！+3！`。小明对这些数很感兴趣，所以他给你一个正整数$\red{n}$，想让你告诉他这个数是否可以表示成若干个不同阶乘的和。

## 输入格式

输入包含多组测试数据。

每组输入为一个非负整数$\red{n}$（$\red{n<=1000000}$），当$\red{n}$为负数时，输入结束。

## 输出格式

对于每组输入，如果$\red{n}$可以表示成若干个不同阶乘的和，则输出`YES`，否则输出`NO`。

## 样例

#### 输入样例

```C
9
-1
```

#### 输出样例

```C
YES
```
