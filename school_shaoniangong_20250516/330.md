## 题目描述

机器人`Rob`可在一个树状路径上自由移动。给定树状路径$\red {T}$上的起点$\red {s}$ 和终点$\red {t}$，机器
人`Rob`要从$\red {s}$运动到$\red {t}$。树状路径$\red {T}$上有若干可移动的障碍物。由于路径狭窄，任何时刻在
路径的任何位置不能同时容纳$\red {2 }$个物体。每一步可以将障碍物或机器人移到相邻的空顶点
上。设计一个有效算法用最少移动次数使机器人从$\red {s}$运动到$\red {t}$。
编程任务：
对于给定的树$\red {T}$，以及障碍物在树$\red {T}$中的分布情况。计算机器人从起点$\red {s}$ 到终点$\red {t}$的最
少移动次数。

## 输入格式

由文件$\red {input.txt}$提供输入数据。文件的第$\red {1}$ 行有$\red {3}$个正整数$\red {n，s}$和$\red {t}$，分别表示树$\red {T}$的
顶点数，起点$\red {s}$的编号和终点$\red {t}$ 的编号。
接下来的$\red {n}$ 行分别对应于树$\red {T}$ 中编号为$\red {0，1，…，n-1}$ 的顶点。每行的第$\red {1}$ 个整数$\red {h}$
表示顶点的初始状态，当$\red {h=1}$ 时表示该顶点为空顶点，当$\red {h=0}$ 时表示该顶点为满顶点，其
中已有$\red {1}$ 个障碍物。第$\red {2 }$个数$\red {k}$表示有$\red {k}$个顶点与该顶点相连。接下来的$\red {k}$个数是与该顶点
相连的顶点编号。

## 输出格式

程序运行结束时，将计算出的机器人最少移动次数输出到文件$\red {output.txt}$ 中。如果无法
将机器人从起点移动到终点，输出`“No solution！”`。

## 样例

#### 输入样例

```c
5 0 3
1 1 2
1 1 2
1 3 0 1 3
0 2 2 4
1 1 3
```

#### 输出样例

```c
3
```
