## 题目描述

给定一个大小为$\red{n≤10^6}$的数组。

有一个大小为$\red{k}$的滑动窗口，它从数组的最左边移动到最右边。

您只能在窗口中看到$\red{k}$个数字，$\red{ 1 <= k <= n}$。

每次滑动窗口向右移动一个位置。

以下是一个例子：

该数组为$\red{[1~3 ~-1~ -3~5~3~6~7]}$，$\red{k}$为$\red{3}$。

| 窗口位置                     | 最小值 | 最大值 |
| ------------------------------ | -------- | -------- |
| $\red{[1~3~ -1] ~-3~ 5~ 3~ 6~ 7}$          | $\red{-1}$ |$\red{ 3}$ |
| $\red{1~ [3 ~-1~ -3]~ 5 ~3 ~6 ~7}$          | $\red{-3}$ |$\red{ 3}$ |
| $\red{1 ~3~ [-1~ -3~ 5]~ 3 ~6 ~7}$          | $\red{-3}$ |$\red{ 5}$ |
| $\red{1 ~3~ -1 ~[-3~ 5 ~3]~ 6 ~7}$          | $\red{-3}$ |$\red{ 5}$ |
| $\red{1 ~3~ -1 ~-3 ~[5 ~3 ~6]~ 7}$          | $\red{3 }$ |$\red{ 6}$ |
| $\red{1 ~3~ -1~ -3~ 5~ [3~ 6~ 7]}$          | $\red{3 }$ |$\red{ 7}$ |

您的任务是确定滑动窗口位于每个位置时，窗口中的最大值和最小值。

## 输入格式

输入包含两行。

第一行包含两个整数$\red{n}$和$\red{k}$，分别代表数组长度和滑动窗口的长度。

第二行有$\red{n}$个整数，代表数组的具体数值。

同行数据之间用空格隔开。

## 输出格式

输出包含两个。

第一行输出，从左至右，每个位置滑动窗口中的最小值。

第二行输出，从左至右，每个位置滑动窗口中的最大值。

## 样例

#### 输入样例

```c
8 3
1 3 -1 -3 5 3 6 7
```

#### 输出样例

```c
-1 -3 -3 -3 3 3
3 3 5 5 6 7
```
