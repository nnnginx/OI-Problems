## 题目描述

不甘心失败的修罗王派出大小各不相同的一队妖兽站在河左岸的石墩（记为$\red{A}$）上，要过到对岸的石礅（记为$\red{D}$）上去执行军事任务。如图所示，河心有几片荷叶（分别记为$\red{Y1…Ym}$）和几个石墩（分别记为$\red{S1…Sn}$）。

![img](./1779/file/5.png)

妖兽站队和移动方法规则如下：

$\red{（１）}$每只妖兽只能站在荷叶、石墩或者比它大一号的妖兽背上（统称为合法的落脚点）；

$\red{（２）}$一只妖兽只有背上没有其他妖兽的时候才能够从一个落脚点跳到另一个落脚点；

$\red{（３）}$妖兽允许从左岸$\red{A}$直接跳到河心的石墩、荷叶和右岸的石墩$\red{D}$上，允许从河心的石墩和荷叶跳到右岸的石墩$\red{D}$上；

$\red{（４）}$妖兽在河心的石墩之间、荷叶之间以及石墩和荷叶之间可以来回跳动；

$\red{（５）}$妖兽在离开左岸石墩后，不能再返回左岸；到达右岸后不能再跳回；

$\red{（６）}$假定石墩承重能力很大，允许无论多少只妖兽都可待在上面。但是由于石墩面积不大，至多只能有一只妖兽直接站在上面，而其他的妖兽只能依规则$\red{1}$落在比它大一号的妖兽背上；

$\red{（７）}$荷叶不仅面积不大，而且负重能力也有限，至多只能有一只妖兽站在上面；

$\red{（８）}$每一步只能移动一只妖兽，并且移动后需要满足站队规则；

$\red{（９）}$在一开始的时候，妖兽均站在$\red{A}$上，最大的一只妖兽直接站在石墩上，而其他的妖兽依规则$\red{6}$站在比其大一号的妖兽背上。

妖兽希望最终能够全部移动到$\red{D}$上，并完成站队。

设河心有$\red{M}$片荷叶和$\red{N}$个石墩，请求出这队妖兽至多有多少只，在满足站队和移动规则的前提下，能从$\red{A}$过到$\red{D}$。

例如当河心有一片荷叶和一个石墩时，此时最多有$\red{4}$只妖兽跳动$\red{9}$步能够过河。如图所示。

![img](./1779/file/6.png)

## 输入格式

文件仅有两行，每一行仅包含一个整数和一个换行/回车符。第一行数字为河心的石墩数$\red{N（0≤N≤25）}$,第二行为荷叶数$\red{M（0≤M≤25）}$。

## 输出格式

文件中仅包含一个数字和一个换行/回车符。该数字为在河心有$\red{Ｎ}$个石墩和$\red{Ｍ}$片荷叶时，最多能够过河的妖兽只数。

## 样例

#### 输入样例

```c
1
1
```

#### 输出样例

```c
4
```
