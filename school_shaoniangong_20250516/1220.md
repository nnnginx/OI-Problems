## 题目描述

未来是`超导计算机`、`纳米计算机`、`光计算机`、`DNA计算机`和`量子计算机`的天下。

但是基于集成电路的计算机短期内还不会退出历史的舞台。

`2000年`，上海大学金翊教授从构造计算机的基本原则和光的基本特性出发，首次将光强度与偏振方向结合起来表示三值信息，利用液晶的旋光效应和偏振器来实现三种光学状态的相互转换和迁移，提出了一种全新的[光计算机](http://baike.baidu.com/view/381305.htm)理论――三值光学计算机(`Ternary Optical Computer`，简称`TOC`)。

三值光学计算机用两个正交的偏振态和无光态表示信息，

所以，它的一个[数据位](http://baike.baidu.com/view/190231.htm)上有三个取值，除$\red{0}$和$\red{1}$之外，还可以再取一个值。而普通的集成电路计算机的数据位上只有两个值$\red{0，1}$。

已知普通计算机上的数据位只有两个值$\red{0}$和$\red{1}$。

所以任何数据都可以用$\red{0}$和$\red{1}$表示。你的目标是求出已知数据的二进制表示中$\red{1}$的个数。

并对其进行统计。

## 输入格式

多组测试数据，每组测试数据各占两行。

`第一行`一个整数$\red{N}$，表示接下来的下一行包含多少个整数。

`第二行`包含$\red{N}$个整数。

（所有的输入都以十进制表示，$\red{1<N<=1000}$，整数范围[$\red{-2^{31},2^{31}-1}$]）

请读到文件末尾。

## 输出格式

根据每组测试数据，请统计出$\red{N}$个整数的二进制表示中$\red{1}$的个数。

## 样例

#### 输入样例

```C
5
1 2 3 4 5
3
2 1 0
1
-1
```

#### 输出样例

```C
Case #1: 7
Case #2: 2
Case #3: 32
```

## 提示

以上储存的数据都为`int`型，每个数据在存储中使用$\red{32}$个数据位。
