## 题目描述

$\red{Farmer Jhon }$决定去一次跨国旅游度假。为了不让他的奶牛们感到被抛弃，他决定租一辆大卡车来带他的奶牛们一起旅行。

这辆卡车有一个很大的油箱，可以装下$\red{G}$个单位的油$\red{(1 <= G <= 1,000,000), }$不幸的是，卡车的耗油量也很大，卡车每运动一个单位的距离，就要消耗一个单位的油。$\red{Farmer Jhon }$要在他 的旅程中走$\red{D}$个单位的距离。$\red{(1 <= D <= 1,000,000,000)}$

因为$\red{FJ}$直到他可能要几次在旅途中停下，给油箱加油，所以他把在旅途沿路上的$\red{N}$个加油站的记录做成了表格。对于第$\red{i}$个加油站，他记录了加油站与起点的距离$\red{X_i(0 <= X_i <= D),}$以及加油站中每单位油的价格$\red{Y_i(1 <= Y_i <= 1,000,000)}$。

已知以上所给的信息，以及$\red{FJ}$在路途中实际使用的油的数量$\red{B(0 <= B <= D)，}$请计算出$\red{FJ}$到达目的地时花费的油费用的最小值。如果$\red{FJ}$无法到达旅途的终点，那么轻输出$\red{-1}$。本题的答案可能无法使用$\red{32}$位整数储存。

## 输入格式

第$\red{1}$行： 四个整数： $\red{N,G,B,D}$

第$\red{2}$~$\red{1+N}$行： 每一行都有两个整数$\red{X_i}$与$\red{Y_i，}$意义如上所述

## 输出格式

一个整数，如果$\red{FJ}$无法到达旅途的终点，那么输出$\red{-1，}$否则输出$\red{FJ}$到达目的地时花费的油费用的最小值。

## 样例

#### 输入样例

```c
4 10 3 17
2 40
9 15
5 7
10 12
```

#### 输出样例

```c
174
```

## 提示

样例解释：$\red{FJ}$先移动$\red{2}$个单位，然后停下购买$\red{2}$个单位的油（要花费$\red{40 \times20）}$。然后一直前进到距离起点$\red{5}$个单位的地方，此时油箱为空。这时向油箱里加满油（要花费$\red{7\times 10）}$。再向前走$\red{5}$个单位，加$\red{2}$个单位的油（花费$\red{12\times 2）}$。最后一直走到终点。此时总花费是$\red{174}$.
