## 题目描述

$\red{n}$ 个整数 $\red{x_1,x_2,…,x_n}$，以及一个整数 $\red{k（k＜n）}$。从 $\red{n }$个整数中任选 $\red{k}$ 个整数相加，可分别得到一系列的和。例如当 $\red{n=4}$，$\red{k＝3}$，$\red{4}$ 个整数分别为 $\red{3，7，12，19}$ 时，可得全部的组合与它们的和为：

　　$\red{3＋7＋12=22}$　　

$\red{3＋7＋19＝29}$

　　$\red{7＋12＋19＝38}$

　　$\red{3＋12＋19＝34}$

现在，要求你计算出和为素数共有多少种。
例如上例，只有一种的和为素数：$\red{3＋7＋19＝29}$。

## 输入格式

第一行两个空格隔开的整数 　$\red{n , k （1<=n<=20，k＜n）}$
第二行 $\red{n}$ 个整数，分别为 $\red{x_1,x_2,\cdots,x_n}$ $\red{（1<=xi<=5000000）}$

## 输出格式

一个整数（满足条件的种数）。

## 样例

#### 输入样例

```c
4 3
3 7 12 19
```

#### 输出样例

```c
1
```

## 提示

$\red{（1<=n<=20，k＜n）}$

$\red{（1<=xi<=5000000）}$
