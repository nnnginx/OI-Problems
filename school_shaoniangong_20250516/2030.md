## 题目描述

小$\red{Z}$有一个数字序列$\red{a_1,a_2,...,a_n,}$长度为$\red{n,}$小$\red{Z}$只有一个操作:选
定$\red{p(2≤}$$\red{p≤}$$\red{n),}$然后把$\red{a_p}$与$\red{a_1}$交换。

比如$\red{a}$序列为$\red{2,3,1,5,4, p=3,}$会交换$\red{1}$和$\red{2}$把序列变为$\red{1,3,2,5,4.}$

现在给你两个序列$\red{a}$和$\red{b，}$问你至少要操作多少步使得由序列$\red{a}$变成
序列$\red{b}$。

## 输入格式

第一行一个整数$\red{n,}$表示两个序列的长度。

第二行$\red{n}$个整数，表示序列$\red{a_o}$

第三行$\red{n}$个整数，表示序列$\red{b_o}$。

## 输出格式

一行一个整数，表示答案。

## 样例

#### 输入样例

```c
4
3 2 1 4
2 1 4 3
```

#### 输出样例

```c
3
```

## 提示

对于$\red{30\%}$的数据,$\red{n≤}$$\red{5}$。

对于$\red{60\%}$的数据，$\red{n<7}$

对于$\red{100\%}$的数据, $\red{1≤}$$\red{a_i≤}$$\red{9, 1≤}$$\red{n≤}$$\red{9}$。
