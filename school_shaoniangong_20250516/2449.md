## 题目描述

约翰想要计算他那$\red{N(1≤}$$\red{N≤}$$\red{1000)}$只奶牛的名字的能量．每只奶牛的名字由不超过$\red{1000}$个字待构成，没有一个名字是空字体串，  约翰有一张" 能量字符串表"，上面有$\red{M(1≤}$$\red{M≤}$$\red{100)}$个代表能量的字符串．每个字符串由不超过$\red{30}$个字体构成，同样不存在空字符串．

一个奶牛的名字蕴含多少个能量字符串，这个名字就有多少能量．所谓"蕴含"，是指某个能量字符串的所有字符都在名字串中按顺序出现（不一定一个紧接着一个）．

所有的大写字母和小写字母都是等价的．比如，在贝茜的名字"$\red{Bessie}$"里，蕴含有"$\red{Be}$""$\red{sI}$""$\red{EE}$"以及"$\red{Es}$"等等字符串，但不蕴含"$\red{lS}$"或"$\red{eB}$"．

请帮约翰计算他的奶牛的名字的能量．

## 输入格式

第$\red{1}$行输入两个整数$\red{N}$和$\red{M，}$

之后$\red{N}$行每行输入一个奶牛的名字，

之后$\red{M}$行每行输入一个能量字符串．

## 输出格式

一共$\red{N}$行，每行一个整数，依次表示一个名字的能量．

## 样例

#### 输入样例

```c
5 3
Bessie
Jonathan
Montgomery
Alicia
Angola
se
nGo
Ont
```


#### 输出样例

```c
1
1
2
0
1
```

## 提示

输入详细信息：

有$\red{5}$头牛，它们的名字是"贝西"，"乔纳森"，"蒙哥马利"、"艾丽西亚"和"安哥拉"。$\red{3}$个好字符串是"$\red{se}$"，"$\red{nGo}$"和"$\red{Ont}$"。

输出详细信息：


"$\red{Bessie}$"包含"$\red{se}$"，"$\red{Jonathan}$"包含"$\red{Ont}$"，"$\red{Montgomery}$"包含"$\red{nGo}$"和"$\red{Ont}$"，$\red{Alicia}$都不包含任何好字符串，并且"安哥拉"包含"非政府组织"。
