## 题目描述


小明买了一部汽车。

今天他第一次开自己的车，他非常激动。

今天他要去 $n$ 米远的地方，这段路上没有任何汽车。但这段路的限速是 $m$ 米每秒。并且小明的车有一个缺陷：就是不能自如地加速减速。他每秒只能比上一秒走多 $1$ 米、走少 $1$ 米或保持上一秒的速度。

小明最终车的速度一定要是 $0$ 米每秒，他想知道他去到目的地最少要多少秒。

## 输入格式

一行两个正整数 $n,m$。

## 输出格式

输出最小时间。

## 样例 #1

### 样例输入 #1

```
10 3
```

### 样例输出 #1

```
6
```

## 提示

第一秒走 $1$ 米，第二秒走 $2$ 米，第三秒走 $3$ 米，第四秒走 $2$ 米，第五、六秒走 $1$ 米。

$1\le n\le10^5,1\le m\le100$。
