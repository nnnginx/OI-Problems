## 题目描述

农民约翰的$\red{N}$头牛$\red{(1 <= N <= 50000)}$在他的二维牧场上不同的点上。牧场的中央是一个圆形的大粮仓。站在谷仓两边的奶牛看不见对方，因为谷仓挡住了它们的视线。请确定有多少对奶牛可以通过直接的视线看到对方。
粮仓以原点$\red{(0,0)}$为圆心，半径为$\red{r}$。粮仓对应的圆上及圆内没有奶牛，粮仓切线上没有两头奶牛。$\red{R}$的取值范 围是$\red{1..1,000,000，}$每头牛都生活在一个点上，其整数坐标范围为$\red{-1,000,000..+1,000,000}$。
农民约翰的$\red{n}$头奶牛（$\red{1 < = n＝}$$\red{50000）}$位于他的二维牧场的不同地点。在牧场的中部是一个大的圆形谷物筒 仓。在筒仓对面的奶牛不能看到对方，因为筒仓挡住了他们的视线。请确定一对能直接通过视线看到的牛的数目。
粮食筒仓的中心在原点（$\red{0,0）}$，半径$\red{R}$没有牛位于或圈内相应的筒仓，没有两牛躺在切线的筒仓。$\red{r}$的值在$\red{1...}$$\red{1000000}$的范围内，而每一头奶牛的生命值都在整数坐标的范围内$\red{-1000000±}$$\red{1000000}$。

## 输入格式

第一行:两个整数:$\red{N}$和$\red{R}$。

第$\red{2 . .1+N}$行：每一行包含两个整数，指定奶牛的$\red{(x,y)}$坐标。

## 输出格式

第一行:可以看到彼此的奶牛的数量。

## 样例

#### 输入样例

```c
4 5
0 10
0 -10
10 0
-10 0
```

#### 输出样例

```c
4
```

## 提示

$\red{4}$头奶牛分别位于$\red{(0,10)}$、$\red{(0，}$$\red{-10)}$、$\red{(10,0)}$和$\red{(-10,0)}$的位置。筒仓以$\red{(0,0)}$为中心，半径为$\red{5}$。

所有$\red{6}$对奶牛都能看到对方，除了位于筒仓对面的奶牛:位于$\red{(-10,0)}$和$\red{(10,0)}$的奶牛不能看到对方，位于$\red{(0，}$$\red{-10)}$和$\red{(0,10)}$的奶牛不能看到对方。
