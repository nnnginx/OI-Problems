## 题目描述

约翰准备教他的奶牛们弹一首歌.这首歌由$\red{N(1≤}$$\red{N≤}$$\red{50000)}$个音阶组成，第$\red{i}$个音阶要敲
击$\red{B_i(1≤}$$\red{B_i≤}$$\red{10000)}$次.奶牛从第$\red{0}$时刻开始弹，因此他从$\red{0}$时刻到$\red{B_1 - 1}$时刻都是敲第$\red{1}$个音阶，
然后他从$\red{B_1}$时刻到$\red{B_1 + B_2- 1}$时刻敲第$\red{2}$个音阶，从$\red{B_1 + B_2}$时刻到$\red{B_1 + B_2+ B_3- 1}$时刻敲第$\red{3}$个
音.....现在有$\red{Q(1≤}$$\red{Q≤}$$\red{50000)}$个问题:在时间段区间$\red{[T,T + 1)}$内，奶牛敲的是哪个音阶?

看下面的一首歌，第$\red{1}$个音阶持续$\red{2}$个单位时间，第$\red{2}$个音阶持续$\red{1}$个单位时间，第$\red{3}$个音阶持
续$\red{3}$个单位时间:

![img](./2820/file/1.png)

以下是一些询问和回答:

|询问|回答的音阶|
|---|---|
|$\red{2}$|$\red{2}$|
|$\red{3}$|$\red{3}$|
|$\red{4}$|$\red{3}$|
|$\red{0}$|$\red{1}$|
|$\red{1}$|$\red{1}$|

## 输入格式

第$\red{1}$行：两个整数$\red{N，}$$\red{Q.}$

第$\red{2}$到$\red{N+1}$行：第$\red{i+l}$行只有一个整数$\red{B_i}$．

第$\red{N+2}$到$\red{N+Q+I}$行：第$\red{N+i+l}$行只有一个整数$\red{T_i}$．

## 输出格式

第$\red{1}$到$\red{Q}$行：对与每个询问，在词问的时间内，奶牛敲击的是哪个音阶？

## 样例

#### 输入样例

```c
3 5
2
1
3
2
3
4
0
1
```

#### 输出样例

```c
2
3
3
1
1
```
