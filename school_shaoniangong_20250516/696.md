## 题目描述

在初赛普及组的“阅读程序写结果”的问题中，我们曾给出一个字符串展开的例子：如果在输入的字符串中，含有类似于$\red{“d-h”}$或$\red{“4-8”}$的子串，我们就把它当作一种简写，输出时，用连续递增的字母或数字串替代其中的减号，即，将上面两个子串分别输出为$\red{“defgh”}$和$\red{“45678”}$。在本题中，我们通过增加一些参数的设置，使字符串的展开更为灵活。具体约定如下：

- $\red{（1）}$遇到下面的情况需要做字符串的展开：

在输入的字符串中，出现了减号$\red{“-”}$，减号两侧同为小写字母或同为数字，且按照$\red{ASCII}$码的顺序，减号右边的字符严格大于左边的字符。

- $\red{（2）}$参数$\red {p_1}$：展开方式。
  
  - $\red{p_1=1}$时，对于字母子串，填充小写字母；
  - $\red{p_1=2}$时，对于字母子串，填充大写字母。这两种情况下数字子串的填充方式相同；
  - $\red{p_1=3}$时，不论是字母子串还是数字子串，都用与要填充的字母个数相同的星号$\red{“*”}$来填充。
- $\red{（3）}$参数$\red {p_2}$：填充字符的重复个数。$\red{p_2=k}$表示同一个字符要连续填充$\red k$个。

例如，当$\red{p_2=3}$时，子串$\red{“d-h”}$应扩展为$\red{“deeefffgggh”}$。减号两侧的字符不变。

- $\red{（4）}$参数$\red{p_3}$：是否改为逆序。$\red{p_3=1}$表示维持原有顺序，$\red{p_3=2}$表示采用逆序输出，注意这时仍然不包括减号两端的字符。

例如当$\red{p_1=1、p_2=2、p_3=2}$时，子串$\red{“d-h”}$应扩展为$\red{“dggffeeh”}$。

- $\red{（5）}$如果减号右边的字符恰好是左边字符的后继，只删除中间的减号，例如：$\red{“d-e”}$应输出为$\red{“de”}$，“$\red{3-4}$”应输出为“$\red{34}$”。如果减号右边的字符按照$\red{ASCII}$码的顺序小于或等于左边字符，输出时，要保留中间的减号，例如：“$\red{d-d}$”应输出为“$\red{d-d}$”，“$\red{3-1}$”应输出为“$\red{3-1}$”。

## 输入格式

包括两行：

第 $\red 1$ 行为用空格隔开的 $\red 3$ 个正整数，依次表示参数$\red{p_1，p_2，p_3}$。

第 $\red 2$ 行为一行字符串，仅由数字、小写字母和减号$\red{“-”}$组成。行首和行末均无空格。

## 输出格式

输出只有一行，为展开后的字符串。

## 样例

#### 输入样例1

```c
1 2 1
abcs-w1234-9s-4zz
```

#### 输出样例1

```c
abcsttuuvvw1234556677889s-4zz
```

#### 输入样例2

```c
2 3 2
a-d-d
```

#### 输出样例2

```c
aCCCBBBd-d
```

#### 输入样例3

```c
3 4 2
di-jkstra2-6
```

#### 输出样例3

```c
dijkstra2************6
```

## 提示

$\red{40\%}$ 的数据满足：字符串长度不超过 $\red 5$ ；

$\red {100\%}$ 的数据满足：$\red {1\leq p_1\leq 3}$，$\red {1\leq p_2\leq 8}$，$\red {1\leq p_3\leq 2}$，字符串长度不超过 $\red {100}$。
