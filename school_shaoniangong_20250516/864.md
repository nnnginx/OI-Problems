## 题目描述

给定一个整数，判断它能否被$\red{3}$，$\red{5}$，$\red{7}$整除，并输出以下信息：

`1、`能同时被$\red{3}$，$\red{5}$，$\red{7}$整除（直接输出$\red{3}$ $\red{5}$ $\red{7}$，每个数占一行）；

`2、`只能被其中两个数整除（输出两个数，小的在前，大的在后。例如：$\red{3}$ $\red{5}$或者 $\red{3}$ $\red{7}$或者$\red{5}$ $\red{7}$,每个数占一行）；

`3、`只能被其中一个数整除（输出这个除数）；

`4、`不能被任何数整除，输出小写字符‘$\red{n}$’,不包括单引号。

## 输入格式

输入一行，包括一个整数。

## 输出格式

输出若干行，按照描述要求给出整数被$\red{3}$，$\red{5}$，$\red{7}$整除的情况。

## 样例

#### 样例输入

```C
105
```

#### 样例输出

```none
3
5
7
```
