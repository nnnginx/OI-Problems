## 题目描述

面面是个成绩优异的孩子，一天，在课堂上老师提问到，当前有一个正整数$\red n$,哪位同学可以快速的说出,比$\red n$大一倍的数，以及所有小于$\red n$的因子。

聪明的面面很快就回答出了正确答案，你可以告诉我面面是怎么做到的吗，擅长编程的你，快用编程实现吧。

## 输入格式

共一行；
包含一个正整数$\red n$;

## 输出格式

共两行；
第一行包括一个正整数k,表示比$\red n$大一倍的数；
第二行包括多个正整数，表示$\red n$的因子，用空格间隔；

## 样例

#### 输入样例

```C
10
```

#### 输出样例

```C
20
1 2 5
```

## 提示

对于所有的数据满足;
$\red{1 \le n \le 1000}$
