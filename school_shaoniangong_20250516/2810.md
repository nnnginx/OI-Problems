## 题目描述

集训队里面有$\red{n}$个人，大家都在$\red{A}$和$\red{B}$两个网站上做题，$\red{A}$网站和$\red{B}$网站都有自己的$\red{rating}$。

集训队里面的每个人都想证明自己比别人强，所以当自己的其中一个网站的分数比另一个人高的时候就会觉得自己比那个人强。如果$\red{A}$比$\red{B}$强，$\red{B}$比$\red{C}$强的话，$\red{A}$也会认为他比$\red{C}$强，

以此类推，如果对于一个序列$\red{P_1,P_2,...,P_k}$都满足$\red{P_i}$比$\red{P_{i+1}}$强的话，那么$\red{P_1}$就认为比$\red{P_k}$强。

能否帮每个人计算一下他比多少人强呢？

## 输入格式

第一行输入一个整数$\red{n}$；

接下来$\red{n}$行，每行两个整数$\red{a_i,b_i}$分别表示这个人在$\red{A}$网站和$\red{B}$网站的$\red{rating}$。

## 输出格式

输出$\red{n}$行，每行表示这个人比多少人强。

## 样例

#### 输入样例

```c
4
2 3
3 2
1 1
4 5
```

#### 输出样例

```c
2
2
0
3
```

## 提示

对于$\red{100\%}$的数据，$\red{1<=n<=100000,1<=a_i,b_i<=10^6}$;

对于$\red{40\%}$的数据，$\red{1<=n<=1000.}$
