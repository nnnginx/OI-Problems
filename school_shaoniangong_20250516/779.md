## 题目描述

春春是一名道路工程师，负责铺设一条长度为 $\red{n}$ 的道路。

铺设道路的主要工作是填平下陷的地表。整段道路可以看作是 $\red{n}$ 块首尾相连的区域，一开始，第 $\red{i}$ 块区域下陷的深度为 $\red{d_i}$。

春春每天可以选择一段连续区间 $\red{[L,R]}$ ，填充这段区间中的每块区域，让其下陷深度减少 $\red{1}$。在选择区间时，需要保证，区间内的每块区域在填充前下陷深度均不为 $\red{0}$ 。

春春希望你能帮他设计一种方案，可以在最短的时间内将整段道路的下陷深度都变为 $\red{0}$ 。

## 输入格式

输入文件包含两行，第一行包含一个整数 $\red{n}$，表示道路的长度。
第二行包含 $\red{n}$ 个整数，相邻两数间用一个空格隔开，第 $\red{i}$ 个整数为 $\red{d_i}$。

## 输出格式

输出文件仅包含一个整数，即最少需要多少天才能完成任务。

## 样例

#### 输入 #1

```
6   
4 3 2 5 3 5
```

#### 输出 #1

```
9
```

## 提示

#### 【样例解释】

一种可行的最佳方案是，依次选择： $\red{[1,6]}$、$\red{[1,6]}$、$\red{[1,2]}$、$\red{[1,1]}$、$\red{[4,6]}$、$\red{[4,4]}$、$\red{[4,4]}$、$\red{[6,6]}$、$\red{[6,6]}$。

#### 【数据规模与约定】

对于 $\red{30\%}$ 的数据，$\red{1 ≤ n ≤ 10}$ ；
对于 $\red{70\%}$ 的数据，$\red{1 ≤ n ≤ 1000}$ ；
对于 $\red{100\%}$ 的数据，$\red{1 ≤ n ≤ 100000 , 0 ≤ d_i ≤ 10000}$。
