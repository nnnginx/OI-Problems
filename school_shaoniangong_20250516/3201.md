

# Description

CSP-J 2023中，小Y的苹果全被好朋友小苞拿走了，现在小Y决定夺回属于自己的苹果！

小苞一开始有原本属于小Y的N个苹果。每天小Y会从小苞那里拿回一半的苹果，且小苞不会把他们再拿回去了，但小Y不想破坏苹果，如果拿之前剩下的苹果是奇数个的时候，小Y就要选择是多拿半个还是少拿半个。作为一个选择困难症，小Y只想尽量少的做出选择，所以她想知道，她至少要做出几次选择才能拿完全部的苹果？

# Format

## Input

第一行一个整数T，表示测试数据组数。

接下来T行，每行一个整数N。

## Output

输出T行，每行一个整数，表示对应N的情况下，小Y最小选择次数。

# Samples

```input1
3
8
7
13
```

```output1
1
2
3
```
