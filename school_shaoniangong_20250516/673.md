## 题目描述

人类终于登上了火星的土地并且见到了神秘的火星人。人类和火星人都无法理解对方的语言，但是我们的科学家发明了一种用数字交流的方法。这种交流方法是这样的，首先，火星人把一个非常大的数字告诉人类科学家，科学家破解这个数字的含义后，再把一个很小的数字加到这个大数上面，把结果告诉火星人，作为人类的回答。

火星人用一种非常简单的方式来表示数字――掰手指。火星人只有一只手，但这只手上有成千上万的手指，这些手指排成一列，分别编号为$\red{1，2，3……}$。火星人的任意两根手指都能随意交换位置，他们就是通过这方法计数的。

一个火星人用一个人类的手演示了如何用手指计数。如果把五根手指――拇指、食指、中指、无名指和小指分别编号为$\red{1，2，3，4}$和$\red{5}$，当它们按正常顺序排列时，形成了$\red{5}$位数$\red{12345}$，当你交换无名指和小指的位置时，会形成$\red{5}$位数$\red{12354}$，当你把五个手指的顺序完全颠倒时，会形成$\red{54321}$，在所有能够形成的$\red{120}$个$\red{5}$位数中，$\red{12345}$最小，它表示$\red{1}$；$\red{12354}$第二小，它表示$\red{2}$；$\red{54321}$最大，它表示$\red{120}$。下表展示了只有$\red{3}$根手指时能够形成的$\red{6}$个$\red{3}$位数和它们代表的数字：

| 三进制数 | $\red{123}$ | $\red{132}$ | $\red{213}$ | $\red{231}$ | $\red{312}$ | $\red{321}$ |
| ---------- | ----- | ----- | ----- | ----- | ----- | ----- |
| 代表数字 | $\red 1$   | $\red 2$   | $\red 3$   | $\red 4$   | $\red 5$   | $\red 6$   |

现在你有幸成为了第一个和火星人交流的地球人。一个火星人会让你看他的手指，科学家会告诉你要加上去的很小的数。你的任务是，把火星人用手指表示的数与科学家告诉你的数相加，并根据相加的结果改变火星人手指的排列顺序。输入数据保证这个结果不会超出火星人手指能表示的范围。

## 输入格式

三行，第一行有一个正整数$\red{N}$，表示火星人手指的数目$\red{（1 ≤ N ≤ 10000）}$。
第二行是一个正整数$\red{M}$，表示要加上去的小整数$\red{（1 ≤ M ≤ 100）}$。
下一行是$\red{1}$到$\red{N}$这$\red{N}$个整数的一个排列，用空格隔开，表示火星人手指的排列顺序。

## 输出格式

只有一行，这一行含有$\red{N}$个整数，表示改变后的火星人手指的排列顺序。每两个相邻的数中间用一个空格分开，不能有多余的空格。

## 样例

#### 输入样例

```c
5
3
1 2 3 4 5
```

#### 输出样例

```c
1 2 4 5 3
```

## 提示

对于$\red{30 \%}$的数据，$\red{N \leq 15}$；
对于$\red{60 \%}$的数据，$\red{N \leq 50}$；对于全部的数据，$\red{N \leq 10000}$。
