## 题目描述

牛们要到太空去了！他们打算建造一座太空电梯来送他们进入轨道．

有$\red{K(1≤}$$\red{K≤}$$\red{400)}$神方块，第$\red{i}$种有一个特定的高度$\red{hi(l≤}$$\red{hi≤}$$\red{100)，}$一定的存量$\red{ci(l≤}$$\red{ci≤}$$\red{10).}$为防宇宙射线的破坏，第$\red{i}$种方块的任何部分不能超过高度$\red{ai(l≤}$$\red{ai≤}$$\red{40000)}$．

请用这些方块堆出最高的太空电梯．

## 输入格式

第$\red{1}$行输入一个整数$\red{K.}$

接下来$\red{K}$行，每行输入三个整数$\red{hi，}$$\red{ai，}$$\red{ci}$．


## 输出格式


一个整数，表示最大高度．


## 样例

#### 输入样例

```c
3
7 40 3
5 23 8
2 52 6
```


#### 输出样例

```c
48
```

## 提示

从底部开始，先放$\red{3}$个方块$\red{2，}$之后$\red{3}$个方块$\red{1，}$接下来$\red{6}$个方块$\red{3}$．不能把$\red{3}$个方块$\red{1}$堆到$\red{4}$个方
块$\red{2}$上，因为这样最高的方块$\red{1}$的顶部高度超过了$\red{40.}$
