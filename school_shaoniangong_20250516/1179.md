## 题目描述

输入`1行`句子（不多于200个单词，每个单词长度不超过$\red{100}$），只包含字母、空格和逗号。

单词由至少一个连续的字母构成，空格和逗号都是单词间的间隔。

试输出`第1个`最长的单词和`第1个`最短单词。

## 输入格式

一行句子。

## 输出格式

两行输出：

`第1行`，第一个最长的单词。

`第2行`，第一个最短的单词。

## 样例

#### 输入样例

```C
I am studying Programming language C in Peking University
```

#### 输出样例

```C
Programming
I
```

## 提示

如果所有单词长度相同，那么第一个单词既是最长单词也是最短单词。
