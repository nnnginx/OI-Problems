## 题目描述

奶牛们打算通过锻炼来培养自己的运动细胞，作为其中的一员，贝茜选择的运动方式是每天进行$\red{N(1 <= N <= 10,000)}$分钟的晨跑。在每分钟的开始，贝茜会选择下一分钟是用来跑步还是休息。

贝茜的体力限制了她跑步的距离。更具体地，如果贝茜选择在第$\red{i}$分钟内跑步，她可以在这一分钟内跑$\red{D_i(1 <= D_i <= 1,000)}$米，并且她的疲劳度会增加 $\red{1}$。

不过，无论何时贝茜的疲劳度都不能超过$\red{M(1 <= M <= 500)}$。如果贝茜选择休息，那么她的疲劳度就会每分钟减少$\red{1，}$但她必须休息到疲劳度恢复到$\red{0}$为 止。

在疲劳度为$\red{0}$时休息的话，疲劳度不会再变动。晨跑开始时，贝茜的疲劳度为$\red{0}$。 还有，在$\red{N}$分钟的锻炼结束时，贝茜的疲劳度也必须恢复到$\red{0，}$否则她将没有足够的精力来对付这一整天中剩下的事情。

请你计算一下，贝茜最多能跑多少米。

## 输入格式

第$\red{1}$行: $\red{2}$个用空格隔开的整数：$\red{N }$和 $\red{M}$

第$\red{2..N+1}$行: 第$\red{i+1}$为$\red{1}$个整数：$\red{D_i}$

## 输出格式

第$\red{1}$行: 输出$\red{1}$个整数，表示在满足所有限制条件的情况下，贝茜能跑的最大 距离

## 样例

#### 输入样例

```c
5 2
5
3
4
2
10
```

#### 输出样例

```c
9
```

## 提示

输出说明:

贝茜在第$\red{1}$分钟内选择跑步（跑了$\red{5}$米），在第$\red{2}$分钟内休息，在第$\red{3}$分钟内跑步（跑了$\red{4}$米），剩余的时间都用来休息。


因为在晨跑结束时贝茜的疲劳度必须为$\red{0，}$所以她不能在第$\red{5}$分钟内选择跑步。
