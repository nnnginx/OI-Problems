## 题目描述

写一个程序，输入一个形如$\red{N/D}$的分数($\red{N}$是分子，$\red{D}$是分母)，输出它的小数形式。 如果小数有循环节的话，把循环节放在一对圆括号中。

例如, $\red{1/3 =0.33333333}$写成$\red{0.(3)}$, $\red{41/333 = 0.123123123...}$写成$\red{0.(123)}$, 用$\red{XXX.0}$ 等表示整数。 典型的转化例子：

$\red{1/3 = 0.(3)}$
$\red{22/5 = 4.4}$
$\red{1/7 = 0.(142857)}$
$\red{2/2 = 1.0}$
$\red{3/8 = 0.375}$
$\red{45/56 = 0.803(571428)}$

## 输入格式

单独的一行包括被空格分开的$\red{N}$和$\red{D(1 <= N,D <= 100000)}$。

## 输出格式

按照上面规则计算出的小数表达式.如果结果长度大于$\red{76}$,每行输出$\red{76}$个字符.

## 样例

#### 输入样例

```c
45 56
```

#### 输出样例

```c
0.803(571428)
```
