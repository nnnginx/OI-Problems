## 题目描述

出题是一件痛苦的事情！

相同的题目看多了也会有审美疲劳，于是我舍弃了大家所熟悉的 `A+B Problem`，改用` A-B` 了哈哈！

好吧，题目是这样的：给出一串数以及一个数字 $\red {C}$，要求计算出所有 $\red {A-B=C}$ 的数对的个数（不同位置的数字一样的数对算不同的数对）。

## 输入格式

输入共两行。

第一行，两个整数 $\red {N}$，$\red{C}$。

第二行，$\red{N}$ 个整数，作为要求处理的那串数。

$\red{1\le n\le 2\times 10^5}$，所有输入的整数绝对值均小于 $\red {2^{30}}$。

## 输出格式

一行，表示该串数中包含的满足 $\red {A-B=C}$ 的数对的个数。

## 样例

#### 输入样例

```C
4 1
1 1 2 3
```

#### 输出样例

```C
3
```
