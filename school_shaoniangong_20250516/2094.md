## 题目描述

有 $n$ 只白羊排成一排，从左到右分别标号为 $1, 2, 3, \cdots, n$。现在，我们要对白羊们染上颜色。

我们共会进行 $m$ 次染色。每次选择两个正整数 $l$ 和 $r$，然后把从编号 $l$ 到编号 $r$ 的之间的所有白羊都染成 **数据删除** 色。

在所有染色结束后，若若子想知道现在还有多少只白羊呢？

## 输入格式

第一行两个正整数 $n$ 和 $m$，用一个空格分隔，表示白羊的数量和染色的次数。

随后 $m$ 行，每行两个正整数 $l$ 和 $r$，用空格分隔，表示此次染色的区间范围。

## 输出格式

一行一个整数 $ans$，表示答案。

## 样例

#### 样例输入

```
9 1
5 8
```

#### 样例输出

```
5
```

## 数据范围与提示

$1 \leq n \leq 10^6$

$1 \leq m \leq 100$

$1 \leq l, r \leq n$
