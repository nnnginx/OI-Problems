## 题目描述

$\red {(n,k)01}$ 串定义为：长度为$\red {n}$ 的$\red {01}$ 串，其中不含$\red {k}$ 个连续的相同子串。对于给定的正整数$\red {n }$和$\red {k}$，计算$\red {(n,k)01}$ 串的个数。
对于给定的正整数$\red {n}$和$\red {k}$，计算$\red {(n,k)01}$串的个数。

## 输入格式

第一行有$\red {2}$ 个正整数$\red {n}$和$\red {k，1≤k，n≤40}$。

## 输出格式

将计算出的$\red {(n,k)01}$ 串的个数输出。

## 样例

#### 输入样例

```c
2 3
```

#### 输出样例

```c
4
```
