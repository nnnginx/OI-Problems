## 题目描述

计算 $\red{A}$ `/` $\red{B}$ 的精确值，设 $\red{A}$，$\red{B}$ 是以一般整数输入，计算结果精确小数后20位

(若不足$\red{20}$位，末尾不用补$\red{0})$ 。

## 输入格式

输入两个数字，$\red{A}$、$\red{B}$

## 输出格式

输出一行，先输出“$\red{A}$`/`$\red{B}$`=`”在输出结果。结果精确到小数点后$\red{20}$位

## 样例

#### 输入样例1

```C
4  3
```

#### 输出样例1

```C
4/3=1.33333333333333333333
```

#### 输入样例2

```C
6  5
```

#### 输出样例2

```C
6/5=1.2
```
