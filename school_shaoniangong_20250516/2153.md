## 题目描述


小明去文具店买文具，一共花了 $n$ 元。

小明是个富豪，他有面值为 $1,2,3,4...$ 元的纸币无限张。他知道他有很多很多种付钱的方式，但他非常好奇到底有多少种付钱方式。于是他向你求助了。

## 输入格式

多组数据。

第一行一个正整数 $T$，表示有 $T$ 组数据。

接下来 $T$ 行，每行一个正整数 $n$。

## 输出格式

共 $T$ 行，第 $i$ 行为第 $i$ 组数据的答案。

## 样例 #1

### 样例输入 #1

```
5
1
2
3
4
5
```

### 样例输出 #1

```
1
2
3
5
7
```

## 提示

$1\le T\le10,1\le n\le64$。
