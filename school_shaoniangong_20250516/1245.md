## 题目描述

现有 $\red{n}$个正整数， $\red{n≤10000}$，要求出这 $\red{n}$个正整数中的第 $\red{k}$个最小整数（相同大小的整数只计算一次）， $\red{k≤1000}$，正整数均小于 $\red{30000}$。

## 输入格式

`第一行`为 $\red{n}$和 $\red{k}$; 第二行开始为 $\red{n}$个正整数的值，整数间用空格隔开。

## 输出格式

第 $\red{k}$个最小整数的值；

若无解，则输出“`NO RESULT`”。

## 样例

#### 输入样例

```C
10 3 
1 3 3 7 2 5 1 2 4 6
```

#### 输出样例

```C
3
```

## 数据范围与提示

$\red{n≤10000}$
