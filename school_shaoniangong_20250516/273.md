## 题目描述

`B城`有 $\red {n}$ 个城镇，$\red {m}$ 条双向道路。

每条道路连结两个不同的城镇，没有重复的道路，所有城镇连通。

把城镇看作节点，把道路看作边，容易发现，整个城市构成了一个无向图。

## 输入格式

第一行包含两个整数 $\red {n }$和 $\red {m}$。

接下来$\red {m}$行，每行包含两个整数 $\red {a}$ 和 $\red {b}$，表示城镇$\red { a}$ 和 $\red {b}$ 之间存在一条道路。

## 输出格式

输出共$\red {n}$行，每行输出一个整数。

第 $\red {i}$ 行输出的整数表示把与节点 $\red {i}$ 关联的所有边去掉以后（不去掉节点 $\red {i}$ 本身），无向图有多少个有序点$\red {（x，y）}$，满足 $\red {x}$ 和 $\red {y}$ 不连通。

## 样例

#### 输入样例

```c
5 5
1 2
2 3
1 3
3 4
4 5
```

#### 输出样例

```c
8
8
16
14
8
```

## 提示

$\red {n≤100000,m≤500000}$
