## 题目描述

如图所示，这是一个纵横火柴棒游戏。在$\red{3\times 4}$的格子中，游戏的双方轮流放置火柴棒。

![img](./1810/file/2.png)

其规则是：

$\red{（１）}$不能放置在已经放置火柴棒的地方（即只能在空格中放置）。

$\red{（２）}$火柴棒的方向只能是竖直或水平放置。

$\red{（３）}$火柴棒不能与其他格子中的火柴“连通”。所谓连通是指两根火柴棒可以连成一条直线，且中间没有其它不同方向的火柴“阻拦”。

例如：上图所示的局面下，可以在$\red{C2}$位置竖直放置（为了方便描述格子位置，图中左、下都添加了标记），但不能水平放置，因为会与$\red{A2}$连通。同样道理，$\red{B2，B3，D2}$此时两种方向都不可以放置。但如果$\red{C2}$竖直放置后，$\red{D2}$就可以水平放置了，因为不再会与$\red{A2}$连通（受到了$\red{C2}$的阻挡）。

$\red{（４）}$游戏双方轮流放置火柴，不可以弃权，也不可以放多根。直到某一方无法继续放置，则该方为负（输的一方）。

游戏开始时可能已经放置了多根火柴。

你的任务是：编写程序，读入初始状态，计算出对自己最有利的放置方法并输出。

## 输入格式

第一行一个整数，表示有多数数据，以下各行描述初始状态。

## 输出格式

输出答案。

## 样例

#### 输入样例

```c
2  (表示有两组数据)

0111（“0”表示空，“1”表示竖直放置，用“-”表示水平放置。）

-000

-000

1111(这是第二组数据)

----

0010
```

#### 输出样例

```c
00-　（对第一个局面，在第0行第0列水平放置，答案非唯一，输出一种即可）
```
