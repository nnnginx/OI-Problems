## 题目描述

原题来自：SDOI 2011

你被要求设计一个计算器完成以下三项任务：

1. 给定$\red{ y,z,p}$，计算$\red{  y^z\bmod p}$ 的值；
2. 给定$\red{ y,z,p}$，计算满足 $\red{ x\times y\equiv z\ (\bmod p\ )}$ 的最小非负整数 x；
3. 给定$\red{ y,z,p}$，计算满足 $\red{ y^x\equiv z\ (\bmod p\ ) }$的最小非负整数 x。

## 输入格式

输入包含多组数据。

第一行包含两个正整数$\red{  T,K }$分别表示数据组数和询问类型（对于一个测试点内的所有数据，询问类型相同）；
以下 T 行每行包含三个正整数$\red{ y,z,p}$，描述一个询问。

## 输出格式

对于每个询问，输出一行答案。

对于询问类型$\red 2 $和$\red 3$，如果不存在满足条件的，则输出`Orz, I cannot find x!`，注意逗号与 `I` 之间有一个空格。

## 样例 1

#### 输入样例 1

```c
3 1
2 1 3
2 2 3
2 3 3
```

#### 输出样例 1

```c
2
1
2
```

#### 输入样例 2

```c
3 2
2 1 3
2 2 3
2 3 3
```

#### 输出样例 2

```c
2
1
0
```


## 数据范围与提示

对于全部数据，$\red{1\le y,z,p\le 10^9}$,$\red{1\le T\le 10}$，且保证 $\red p$为质数。
