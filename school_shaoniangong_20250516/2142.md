## 题目描述

$\red{Farmer John }$对于农场里昏暗的灯光很不满，于是他刚刚装上一个装饰精美的新吊灯。

这个吊灯由$\red{N(3 <= N <= 16)}$个灯组成，并排成一个圆圈。

奶牛们对于这个新的发光装置很有兴趣。他们喜欢玩这样的一个游戏：

在时间$\red{T}$时，他们会操作在时间$\red{T-1}$的时左边的灯是开的的灯的灯泡的开关。

他们会在$\red{B}$个单位的时间里一直进行这样的操作$\red{( 1 <= B <= 10^{15} )}$

请注意，$\red{B}$可能会超过一般的$\red{32}$位整数的范围。

现在已知每一个灯的初始状态，请计算出在经过$\red{B}$个单位的时间后，每一个灯的状态

## 输入格式

第$\red{1}$行是两个整数，$\red{N}$和$\red{B}$

第$\red{2}$~$\red{1+N}$中，第$\red{i+1}$行描述了灯的初始状态，用$\red{0(}$关$\red{）}$与$\red{1（}$开$\red{）}$表示

## 输出格式

$\red{N}$行，第$\red{i}$行应输出一个整数，描述灯的最终状态，用$\red{0(}$关$\red{)}$与$\red{1(}$开$\red{)}$表示

## 样例

#### 输入样例

```c
5 6
1
0
0
0
0
```

#### 输出样例

```c
1
1
1
0
1
```

```
Time T=2: 1 0 1 0 0

Time T=4: 1 0 0 0 1
Time T=5: 0 1 0 0 1
Time T=6: 1 1 1 0 1
```
