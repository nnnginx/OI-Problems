## 题目描述

大家都知道扫雷游戏

![img](./1127/file/pimg3927_1.jpg)

这个是一个扫雷的局面，我们可以发现，上面的数字表示该位置周围$\red{8}$个方格有多少个雷，例如第二行第二列的“$\red{2}$”表示周围的$\red{8}$个方格里有$\red{2}$个雷，没有标明数字也没有标明雷的格子表示周围的方格里没有雷。

现在小明突发奇想，如果把原来不是雷的地方都改成雷，是雷的地方标上数字，（标上的数字意思见前面叙述），他想知道这样变换之后局面上的数字之和是多少，你能帮助他么？

## 输入

先输入$\red{T}$，表示输入$\red{T}$个测试数据，对于每个测试数据，

第一行输入整数$\red{n}$,$\red{m}$表示局面的行数，列数($\red{0<n,m<=1000}$)。

接下来输入一个$\red{n}$*$\red{m}$矩阵，矩阵元素表示意思如下
数字$\red{0 \sim 8}$，表示周围雷的个数
星号“`*`”，表示该位置是雷。

输入的数据保证数字和雷不矛盾。

## 输出

对于每个测试数据，输出一行，一个数字，表示变换后局面上数字之和。

## 样例

#### 输入样例

```C
1
3 3
*2*
132
01*
```

#### 输出样例

```C
9
```

## 提示

对于样例输入，将其变换后得到局面如下

<pre>
3 * 3

* * *

* * 3
</pre>

因此结果为`3+3+3=9`，输出$\red{9}$。
