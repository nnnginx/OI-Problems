### 题目描述

在给定$\red{N}$个数字。

求区间和


### 输入格式

输入一个$\red{N （N <= 100）}$

接下来有$\red{N}$个数字。

分别是$\red{a_1,a_2....a_n}$;

输入一个$\red{M (M <= 100 )}$

接下来每一行有$\red{L , R ( 0 <= L , R <= N )}$

### 输出格式

$\red{M}$行正整数 

每一行分别是$\red{L , R }$的区间和 

### 样例
#### 输入数据 
```
5
1 2 3 4 5
3
1 2
2 3
1 5
```
#### 输出数据
```
3
5
15
```
