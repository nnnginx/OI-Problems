## 题目描述

检查一个如下的 $\red{6\times 6}$ 的跳棋棋盘,有六个棋子被放置在棋盘上,使得每行,每列,每条对角线(包括 两条主对角线的所有对角线)上都至多有一个棋子.

```c
0   1   2   3   4   5   6
  -------------------------
1 |   | O |   |   |   |   |
  -------------------------
2 |   |   |   | O |   |   |
  -------------------------
3 |   |   |   |   |   | O |
  -------------------------
4 | O |   |   |   |   |   |
  -------------------------
5 |   |   | O |   |   |   |
  -------------------------
6 |   |   |   |   | O |   |
  -------------------------
```

上面的布局可以用序列$\red{2\ 4\ 6\ 1\ 3\ 5}$来描述，第$\red i$个数字表示在第$\red i$行的相应位置有一个棋子，如下：

行号 $\red{1\ 2\ 3\ 4\ 5\ 6}$

列号 $\red{2\ 4\ 6\ 1\ 3\ 5}$

这只是跳棋放置的一个解。请编一个程序找出所有跳棋放置的解。并把它们以上面的序列方法输出。解按字典顺序排列。请输出前$\red 3$个解。最后一行是解的总个数。

特别注意: 对于更大的$\red N$(棋盘大小$\red{N\times N)}$你的程序应当改进得更有效。不要事先计算出所有解然后只输出(或是找到一个关于它的公式），这是作弊。

## 输入格式

一个数字$\red{N (6 <= N <= 13)}$ 表示棋盘是$\red{N\times N}$大小的。

## 输出格式

前三行为前三个解，每个解的两个数字之间用一个空格隔开。第四行只有一个数字，表示解的总数。

## 样例

#### 输入样例

```c
6
```

#### 输出样例

```c
2 4 6 1 3 5 
3 6 2 5 1 4 
4 1 5 2 6 3 
4
```
