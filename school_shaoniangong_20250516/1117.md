## 说明

在情报传递过程中，为了防止情报被截获，往往需要对情报用一定的方式加密，简单的加密算法虽然不足以完全避免情报被破译，但仍然能防止情报被轻易的识别。

我们给出一种最简的的加密方法，对给定的一个字符串，把其中从$\red{a}$`~`$\red{y}$，$\red{A}$`~`$\red{Y}$的字母用其后继字母替代，把$\red{z}$和$\red{Z}$用$\red{a}$和$\red{A}$替代，其他非字母字符不变，则可得到一个简单的加密字符串。

## 输入格式

输入一行，包含一个字符串，长度小于$\red{80}$个字符。

## 输出格式

输出每行字符串的加密字符串。

## 样例

#### 输入数据

```C
Hello! How are you!
```

#### 输出数据

```C
Ifmmp! Ipx bsf zpv!
```
