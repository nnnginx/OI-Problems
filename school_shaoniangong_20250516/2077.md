## 题目描述

农夫约翰让他的 $\red{n （}$$\red{1 <= n <= 100,000）}$ 头牛在长度为 $\red{c }$的跑道上进行跑 $\red{l }$圈的比赛，所有牛从同一起点，以不同的速度开始跑。直到当跑得最快的那一头牛跑完 $\red{l }$圈时，所有牛才同时停下。

约翰发现在跑圈过程中发生了几次"超越事件"。其定义是：在比赛结束前某时刻，奶牛 $\red{x }$已经超越了奶牛 $\red{y }$整整一圈，则称做一次"超越事件"。（注： 至少一圈 ，超越了$\red{1/2}$圈，或者超越了$\red{1/4}$圈等等都不算。且对于同一对奶牛$\red{（x，}$$\red{y）}$不会重复计算次数。）

约翰想知道比赛过程中发生了多少次"超越事件"。

（注：可能原文章表达有误或某些其他原因，各种翻译方式过来的题意都有问题，给人误导很大，这里是根据题目数据和样例解释写的正确的题意，而不是原文）

## 输入格式

第一行：三个整数：$\red{n，}$$\red{l，}$$\red{c（}$$\red{1 <= l,c <= 25,000)}$

第二行到第 $\red{n+1 }$行：每行一个整数 $\red{vi ，}$表示奶牛$\red{i}$的速度$\red{（1 <= vi <= 1,000,000）}$

## 输出格式

第一行：一个整数，表示总共发生的"超越事件"的次数

## 样例

#### 输入样例

```c
4 2 100 
20 
100 
70 
1
```

#### 输出样例

```c
4
```

## 提示

有 $\red{4 }$头奶牛在长度为 $\red{100 }$的圆形轨道上跑 $\red{2 }$圈。奶牛的速度分别为 $\red{20}$、$\red{100}$、$\red{70 }$和 $\red{1}$。

比赛持续 $\red{2 }$个单位的时间，因为这是最快的母牛（母牛 $\red{2）}$完成的时间。在这段时间内，有 $\red{4 }$次交叉事件：奶牛 $\red{2 }$超过奶牛 $\red{1 }$和 $\red{4，}$奶牛 $\red{3 }$超过奶牛 $\red{1 }$和 $\red{4}$。
