## 题目描述

$\red{Dye}$认为虽然代码能够$\red{AC，}$但是还不够完美，所以在$\red{Dye}$提交代码之前，$\red{Dye}$打算旧能的缩写他的变量名。$\red{Dye}$的代码里有$\red{n}$个变量名，变量名两两互不相同，且只包含小写字母'$\red{a}$'$\red{-}$'$\red{z}$'。

他希望用非空前缀替换每个变量名，使得这些新变量名仍然是互不相同的（但是，某些变量的新名称可以与另一个或同一个变量的旧名称重合）。

他想找到一种新名字总长度最小的方法。

字符串$\red{a }$是字符串$\red{b}$的前缀，表示你可以从$\red{b}$的末尾删除若干个(或$\red{0}$个)字符并获得$\red{a }$。

请找出新变量名的最猩能总长度。

## 输入格式

第一行包含一个整数$\red{N(1≤}$$\red{N≤}$$\red{10^5)，}$表示变量的个数

接着$\red{N}$行表示$\red{N}$个变量名，变量名只包含小写字母'$\red{a}$'$\red{-}$'$\red{z}$'。

变量名的总长度不超过$\red{10^5，}$且互不相同。

## 输出格式

输出一行，包含一个整数――新变量名的最小可能总长度

## 样例

#### 输入样例

```c
5
abba
abb
ab
aa
aacada
```

#### 输出样例

```c
11
```

## 提示

我们用"$\red{aac}$"作为最后一个变量名，将"$\red{a}$"作为第一个变量名，剩下的变量不需要改变，得到最小总长度为$\red{11}$
