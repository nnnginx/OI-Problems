## 题目描述

小 $\red{K}$ 同学向小 $\red{P}$ 同学发送了一个长度为 $\red{8}$ 的 $\red{01}$ 字符串来玩数字游戏，小 $\red{P}$ 同学想要知道字符串中究竟有多少个 $\red{1}$。

注意：$\red{01}$ 字符串为每一个字符是 $\red{0}$ 或者 $\red{1}$ 的字符串，如$\red{“101”}$（不含双引号）为一个长度为 $\red{3}$ 的$\red{ 01}$ 字符串。

## 输入格式

输入文件只有一行，一个长度为 $\red{8}$ 的 $\red{01}$ 字符串 $\red{s}$。

## 输出格式

输出文件只有一行，包含一个整数，即 $\red{01}$ 字符串中字符 $\red{1}$ 的个数。

## 样例

#### 输入样例

```c
00010100
```

#### 输出样例

```c
2
```

#### 输入样例

```c
11111111
```

#### 输出样例

```c
8
```

## 提示

【输入输出样例 $\red{1}$ 说明】

该 $\red{01}$ 字符串中有 $\red{2}$ 个字符 $\red{1}$。

【输入输出样例 $\red{2}$ 说明】

该 $\red{01}$ 字符串中有 $\red{8}$ 个字符 $\red{1}$。

【数据规模与约定】

对于 $\red{20\%}$ 的数据，保证输入的字符全部为 $\red{0}$。

对于 $\red{100\%}$ 的数据，输入只可能包含字符 $\red{0}$ 和字符 $\red{1}$，字符串长度固定为 $\red{8}$。
