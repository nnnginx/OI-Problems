## 题目描述

输入两个正整数$\red{m}$和$\red{k}$，其中($\red{1<m<100000}$，$\red{1<k<5}$) ，

判断$\red{m}$ 能否被$\red{19}$整除，且恰好含有$\red{k}$个$\red{3}$，

如果满足条件，则输出`YES`，否则，输出`NO`。

## 输入格式

$\red{m}$ 和 $\red{k}$ 的值，中间用单个空格间隔。

## 输出格式

满足条件时输出 `YES`，不满足时输出 `NO`。

## 样例

#### 输入数据1

```C
43833 3
```

#### 输出数据1

```C
YES
```

#### 输入数据2

```C
39331 3
```

#### 输出数据2

```C
NO
```

## 提示

`样例2`当中：

尽管有$\red{3}$个$\red{3}$，但不能被$\red{19}$整除，也不满足条件，应输出`NO`
