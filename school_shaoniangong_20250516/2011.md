## 题目描述

在一条无限长的跑道上有 $\red{N }$头牛，每头牛有自己的初始位置及奔跑的速度。牛之间不能互
相穿透。当一只牛追上另一只牛时，它不得不慢下来，成为一个群体。求 $\red{T }$分钟后一共有几
个群体。

## 输入格式

第一行 $\red{N,T(N<=100000,1 <= T <= 1,000,000,000)}$

下面 $\red{N }$行每行两个数，表示一头牛的位置 $\red{p }$和速度 $\red{v(0<=p<=1000000,1<=v<=1000000)}$

每头牛的位置不同，且输入保证位置递增。

## 输出格式

表示有多少个群体。

## 样例

#### 输入样例

```c
5 3
0 1
1 2
2 3
3 2
6 1
```

#### 输出样例

```c
3
```
