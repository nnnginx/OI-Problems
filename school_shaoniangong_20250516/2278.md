## 题目描述

牛为什么要过马路？

嗯，一个原因是 $\red{Farmer John }$的农场有很多道路，这使得他的奶牛在不经过许多道路的情况下不可能四处走动。 $\red{FJ }$的农场排列成 $\red{N×}$$\red{N }$方格的田地$\red{（2≤}$$\red{N≤}$$\red{100）}$，某些相邻的田地（例如南北或东西）由道路隔开，高高的围栏围绕着整个网格的外围，防止奶牛离开农场。

奶牛可以从任何田地自由移动到任何其他相邻的田地（北、东、南或西），但除非绝对必要，否则它们不喜欢过马路。

$\red{FJ}$的农场有$\red{K}$奶牛$\red{（1≤}$$\red{K≤}$$\red{100，}$$\red{K≤}$$\red{N^2）}$，每头奶牛分布在不同的田地。

如果为了让一头母牛拜访另一头母牛，必须穿过至少一条马路，则称一对母牛是"遥远的"。

请帮 $\red{FJ }$数一数远处的奶牛对的数量。

## 输入格式

第一行输入包含$\red{N}$、$\red{K}$和$\red{R.}$

接下来的$\red{R}$行描述了存在于相邻场对之间的$\red{R}$道路。

每条线的形式为$\red{rc r′}$$\red{c′}$（范围为$\red{1…}$$\red{N}$的整数），表示（$\red{r}$行，$\red{c}$列）中的字段与（$\red{r′}$，$\red{c'}$列）中的相邻字段之间的道路。



最后的$\red{K}$线表示$\red{K}$奶牛 的位置，每一个都以行和列的形式指定。

## 输出格式

打印相距较远的奶牛对数。

## 样例

#### 输入样例

```c
3 3 3
2 2 2 3
3 3 3 2
3 3 2 3
3 3
2 2
2 3
```



#### 输出样例

```c
2
```
