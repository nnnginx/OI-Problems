## 题目描述

任何一个大于$\red{1}$的自然数$\red{n}$，总可以拆分成若干个小于$\red{n}$ ($\red{1<n<=50}$)的自然数之和。

当$\red{n=7}$共$\red{14}$种拆分方法：

`7=1+1+1+1+1+1+1

7=1+1+1+1+1+2

7=1+1+1+1+3

7=1+1+1+2+2

7=1+1+1+4

7=1+1+2+3

7=1+1+5

7=1+2+2+2

7=1+2+4

7=1+3+3

7=1+6

7=2+2+3

7=2+5

7=3+4

total=14

## 输入格式

输入`n`。

## 输出格式

按字典序输出具体的方案。

## 样例

#### 输入样例

```C
7
```

#### 输出样例

```C
7=1+1+1+1+1+1+1

7=1+1+1+1+1+2

7=1+1+1+1+3

7=1+1+1+2+2

7=1+1+1+4

7=1+1+2+3

7=1+1+5

7=1+2+2+2

7=1+2+4

7=1+3+3

7=1+6

7=2+2+3

7=2+5

7=3+4

total=14
```
