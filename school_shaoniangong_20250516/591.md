## 题目描述

考虑排好序的$\red{N(N<=31)}$位二进制数。

他们是排列好的，而且包含所有长度为$\red{N}$且这个二进制数中$\red{1}$的位数的个数小于等于$\red{L(L<=N)}$的数。

你的任务是输出满足长度为 $\red{N}$，且$\red{ 1}$ 的个数小于等于 $\red{L}$ 的第 $\red{i}$ 小的那个二进制数（保证这个数存在）。

（例：$\red{100101}$中，$\red{N=6}$，含有位数为$\red{1}$的个数为$\red{3}$）。

## 输入格式

共一行，用空格分开的三个整数$\red{N，L，i}$。

## 输出格式

共一行，输出满足条件的第$\red{i}$小的二进制数。

## 样例

#### 输入样例

```c
5 3 19
```

#### 输出样例

```c
10011
```
