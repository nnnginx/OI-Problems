## 题目描述

逆时针给出$\red{n}$个凸多边形的顶点坐标,求它们交的面积。

## 输入格式

第一行有一个整数$\red{n}$。表示凸多边形的个数,$\red{n≤}$$\red{10}$。

以下依次描述各个多边形。第$\red{i}$个多边形的第一行包含一个整数$\red{m,}$表示多边形的
边数,$\red{m_i≤}$$\red{100}$。以下$\red{m_i}$行每行两个整数,逆时针给出各个顶点的坐标，每个坐标的绝对
值均小于等于$\red{1 000}$。

## 输出格式

输出一行一个实数,表示相交部分的面积，保留三位小数。

## 样例

#### 输入样例

```c
2
6
-2 0
-1 -2
1 -2
2 0
1 2
-1 2
4
0 -3
1 -1
2 2
-1 0
```

#### 输出样例

```c
5.233
```
