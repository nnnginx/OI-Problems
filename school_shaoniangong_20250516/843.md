## 题目描述

利用公式 $\red{C=5×(F−32)÷9}$ (其中$\red C$表示摄氏温度，$\red{F}$表示华氏温度)进行计算转化,输入华氏温度$\red{F}$。

输出摄氏温度$\red{C}$，要求精确到小数点后$\red{5}$位。

## 输入格式

输入一行，包含一个实数$\red{F}$，表示华氏温度。（ $\red{F ≥ −459.67}$ ）

## 输出格式

输出一行，包含一个实数，表示对应的摄氏温度，要求精确到小数点后$\red{5}$位。

## 样例

#### 输入样例

```C
41
```

#### 输出样例

```C
5.00000
```
