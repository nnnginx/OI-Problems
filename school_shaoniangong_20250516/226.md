## 题目描述

给定一个整数数组$\red {a_1 ,a_2 ,…,a_n}$。

定义数组第 $\red {i}$ 位上的减操作：把$\red {a_i}$和

$\red {a_{i+1}}$换成

$\red {a_i −a_{i+1}}$。

用

$\red {con(a,i)}$表示减操作，可以表示为：

$\red {con(a,i)=[a_1 ,a_2~ ,…,a_{i−1} ,a_i~ −a _{i+1} ,a _{i+2} ,…,a_n ]}$

长度为 $\red {n}$

的数组，经过 $\red {n−1 }$次减操作后，就可以得到一个整数 $\red {t}$。

例如数组$\red {[12,10,4,3,5]}$经过如下操作可得到整数$\red {4}$：

$\red {con([12,10,4,3,5],2) = [12,6,3,5]}$

$\red {con([12,6,3,5] ,3) = [12,6,-2]}$

$\red {con([12,6,-2] ,2) = [12,8]}$

$\red {con([12,8] ,1) = [4]}$

现在给定数组以及目标整数，求完整操作过程。

## 输入格式

第$\red {1}$行包含两个整数$\red {n}$和$\red {t}$。

第$\red {2..n+1}$行：第$\red {i}$行包含数组中的第 $\red {i }$个整数$\red {a_i}$。

## 输出格式

输出共$\red {n-1}$行，每行包含一个整数，第 $\red {i}$ 行的整数表示第$\red { i }$次减操作的操作位置。

## 样例

#### 输入样例

```c
5 4
12
10
4
3
5
```

#### 输出样例

```c
2
3
2
1
```

## 提示

$\red {1≤n≤100}$,

$\red {−10000≤t≤10000}$,

$\red {1≤a_i ≤100}$
