## 题目描述

$\red{Chen.Jr}$注册了一个电子邮箱，可是他时常忘记查收自己的邮件，导致他的邮箱非常的凌乱。

在他和别人通过邮件交流的时候，第一封邮件是一个只包含小写字母的非空字符串，回复的邮件就会加上"$\red{Re: }$"前缀，再次回信就又加上"$\red{Re: }$"前缀，这样称为一组邮件往来。

比如一开始邮件是"$\red{subj}$"，第二封邮件就是"$\red{Re: subj}$"，第三封邮件就是"$\red{Re: Re: subj}$"，以此类推。

很不幸的是，他的邮箱里面的部分邮件丢失了，他猜测他原来有$\red{n}$封邮件，可是现在在他的邮箱中只剩下了$\red{k}$封邮件，你能不能帮他判断一下他的猜测是否有可能是对的呢？

## 输入格式

一行输入两个整数$\red{n}$和$\red{k}$

下面$\red{k}$行分别表示现在剩下的邮件。

## 输出格式

如果有可能是正确的，则输出"$\red{YES}$"；否则输出"$\red{NO}$"

## 样例

#### 输入样例1

```c
7 3 
Re: Re: Re: hello 
Re: world 
hello
```

#### 输出样例1

```c
YES
```

#### 输入样例2

```c
3 2 
Re: Re: pleasehelp
me
```

#### 输出样例2

```c
NO
```

## 提示

对于$\red{100\%}$的数据 $\red{1<=k<=n<=100}$
