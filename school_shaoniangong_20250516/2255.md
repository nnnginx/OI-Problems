## 题目描述

$\red{Farmer John }$收到了一份需要立即装满 $\red{M }$单位牛奶 $\red{(1≤}$$\red{M≤}$$\red{1,000) }$的订单。

不幸的是，他的花哨的挤奶机刚刚坏了，他只有三个整数大小的牛奶桶 $\red{X}$、$\red{Y }$和 $\red{M (1≤}$$\red{X<Y<M)}$。

所有三个桶最初都是空的。

使用这三个桶，他可以执行任意数量的以下两种类型的操作：

$\red{- }$他可以用 $\red{X }$个单位的牛奶将最小的桶（$\red{X }$号）完全装满，然后倒入 $\red{M }$号桶中，如只要这不会导 致 $\red{M }$号桶溢出。

$\red{- }$他可以将$\red{Y}$个单位的牛奶完全装满中号桶（$\red{Y}$号），然后倒入$\red{M}$号桶中，只要这不会导致$\red{M}$ 号桶溢出。

虽然 $\red{FJ }$意识到他可能无法完全装满 $\red{M }$号桶，但请帮助他确定他可以添加到这个桶中的最大牛奶量。

## 输入格式

第一行（也是唯一的一行）包含$\red{X}$、$\red{Y}$和$\red{M，}$用空格分隔。

## 输出格式

输出$\red{FJ}$可能添加到$\red{M}$桶大小的最大牛奶量。

## 样例

#### 输入样例

```c
17 25 77
```




#### 输出样例

```c
76
```

## 提示

在本例中，$\red{FJ}$填充$\red{17}$号桶三次，填充$\red{25}$号桶一次，总共累积$\red{76}$单位牛奶。
