## 题目描述

棋盘上$\red{A}$点有一个过河卒，需要走到目标B点。

卒行走的规则：可以向下、或者向右。同时在棋盘上$\red{C}$点有一个对方的马，该马所在的点和所有跳跃一步可达的点称为对方马的控制点。因此称之为“马拦过河卒”。

棋盘用坐标表示，$\red{A}$点$\red{(0, 0)}$、$\red{B}$点$\red{(n, m)}$($\red{n, m}$为不超过$\red{20}$的整数)，同样马的位置坐标是需要给出的。

现在要求你计算出卒从$\red{A}$点能够到达$\red{B}$点的路径的条数，假设马的位置是固定不动的，并不是卒走一步马走一步，$\red{C≠A}$且$\red{C≠B}$。

![img](./659/file/6.gif)

## 输入格式

一行四个数据，分别表示$\red{B}$点坐标和马的坐标。

## 输出格式

一个数据，表示所有的路径条数。

## 样例

#### 输入样例

```c
6 6 3 3
```

#### 输出样例

```c
6
```
