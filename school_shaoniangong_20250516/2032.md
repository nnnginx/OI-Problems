## 题目描述

$\red{3}$个元素的集合$\red{\{5,1,2\}}$的所有子集的和分别是$\red{0,1,2,3,5,6,7,8}$。发
现最小的不能由该集合子集拼出的数字是$\red{4}$。

现在给你一个$\red{n}$个元素的集合，问你最小的不能由该集合子集拼出的
数字是多少。

注意$\red{32}$位数字表示范围。

## 输入格式

第一行一个整数$\red{n}$。

第二行$\red{n}$个正整数$\red{a_i,}$表示集合内的元素。

## 输出格式

一行一个整数答案。

## 样例

#### 输入样例

```c
3
5 1 2
```

#### 输出样例

```c
4
```

## 提示

对于$\red{30\%}$的数据，满足$\red{n≤}$$\red{15}$。

对于$\red{60\%}$的数据，满足$\red{n≤}$$\red{1000}$。

对于$\red{100\%}$的数据，满足$\red{n≤}$$\red{100000,1≤}$$\red{a_i≤}$$\red{10^9}$。
