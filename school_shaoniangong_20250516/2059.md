## 题目描述

$\red{FJ}$有$\red{N (1 <= N <= 100)}$个农场，每个农场具有独立的整数坐标$\red{(x_i, y_i)}$。他需要一个物资配送路线，从第$\red{1}$个农场出发，依次经过农场$\red{1，}$农场$\red{2，}$农场$\red{3}$…，最后从农场$\red{N}$回到农场$\red{1.}$

$\red{FJ}$每次只能朝东南西北四个方向行走，没行走一个单位长度需要$\red{1}$分钟，除了农场$\red{1，}$其他农场能且仅能到达一次。

请计算$\red{FJ}$的最小时间花费。

## 输入格式

第 $\red{1 }$行：农场数量，$\red{N}$。

第 $\red{2..1+N }$行：第 $\red{i+1 }$行包含两个以空格分隔的整数，$\red{x_i }$和 $\red{y_i (1 <= x_i, y_i <= 1,000,000)}$。

## 输出格式

第 $\red{1 }$行：$\red{FJ }$完成他的配送路线所需的最少分钟数，如果不可能找到一条可行的配送路线，可以准确地访问每个农场一次（农场 $\red{1 }$除外），则为 $\red{-1}$。

## 样例

#### 输入样例

```c
4 
2 2 
2 4 
2 1 
1 3
```

#### 输出样例

```c
12
```

## 提示

$\red{FJ}$可以在$\red{12}$分钟内完成他的送货路线：从农场$\red{1}$到农场$\red{2}$需要$\red{2}$分钟，从农场$\red{2}$到农场$\red{3}$需要$\red{5}$分钟（绕过农场$\red{1）}$，从农场$\red{3}$到农 场$\red{4}$需要$\red{3}$分钟，然后$\red{2}$分钟返回农场$\red{1}$。
