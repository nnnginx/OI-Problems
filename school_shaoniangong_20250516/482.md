## 题目描述

由于科协里最近真的很流行数字游戏，某人又命名了一种取模数，这种数字必须满足各位数字之和 $\red{\bmod N }$为$\red{ 0}$。现在大家又要玩游戏了，指定一个整数闭区间 $\red{[a,b]}$，问这个区间内有多少个取模数。

## 输入格式

题目有多组测试数据。每组只含三个数字 $\red{a,b,N}$。

## 输出格式

对于每个测试数据输出一行，表示各位数字和$\red{ \bmod N}$ 为 $\red{0}$ 的数的个数。

## 样例

#### 输入样例

```c
1 19 9
```

#### 输出样例

```c
2
```

## 提示

对于全部数据，$\red{1\le a,b\le 2^{31}-1,1\le N\lt 100}$。
