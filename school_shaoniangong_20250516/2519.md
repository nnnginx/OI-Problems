## 题目描述

这些奶牛被派去执行一项任务，为他们的谷仓购买一台新的挤奶机。它们飞过包含 $\red{N (1 <= N <= 50,000) }$颗行星的星团，每颗行星都有一个交易站。

奶牛已经确定了星团中每个行星需要 $\red{K (1 <= K <= 1,000) }$种类型的物体（编号 $\red{1..K）}$中的哪一种，以及它们必须交易的产品。没有一个星球发展了货币，所 以它们在易货系统下运作：所有交易都由每一方交易的每一方都只交易一个对象（可能是不同类型的）。

奶牛们从地球开始带着一罐优质干草（第 $\red{1 }$项），他们想要一台新的挤奶机（第 $\red{K }$项）。帮助他们找到在集群中的行星上进行一系列交易以获得物品 $\red{K }$的最佳方法。如果此任务不可能完成，则输出 $\red{-1}$。

## 输入格式

第 $\red{1 }$行：两个空格分隔的整数，$\red{N }$和 $\red{K}$。

第 $\red{2..N+1 }$行：
第 $\red{i+1 }$行包含两个空格分隔的整数，分别为 $\red{a_i }$和 $\red{b_i，}$它们是行星 $\red{i }$的交易交易产品。行星将给予项目 $\red{b_i }$以接收项目 $\red{a_i}$。

## 输出格式

第 $\red{1 }$行：获得挤奶机的最小交易次数多 $\red{1 }$次，即项目 $\red{K（}$如果奶牛无法获得项目 $\red{K，}$则为 $\red{-1）}$。

## 样例

#### 输入样例

```c
6 5
1 3
3 2
2 3
3 1
2 5
5 4
```

#### 输出样例

```c
4
```

## 提示

$\red{//6}$个星球,希望得到$\red{5,}$开始时你手中有$\red{1}$号货物.

$\red{//1}$号星球,希望得到$\red{1}$号货物,将给你$\red{3}$号货物

输出详细信息：

奶牛总共拥有$\red{4}$件物品：首先，它们用物品$\red{1}$换取对象$\red{3，}$然后对象$\red{3}$表示对象$\red{2，}$然后对象$\red{2}$表示对象$\red{5}$。
