## 题目描述

$\red{LHX}$教主要来$\red{X}$市指导$\red{OI}$学习工作了。为了迎接教主，在一条道路旁，一群$\red{Orz}$教主$\red{er}$穿着文化衫站在道路两旁迎接教主，每件文化衫上都印着大字。一旁的$\red{Orzer}$ 依次摆出"欢迎欢迎欢迎欢迎……"的大字，但是领队突然发现，另一旁穿着"教"和"主"字文化衫的$\red{Orzer}$却不太和谐。
为了简单描述这个不和谐的队列，我们用"$\red{j}$"替代"教"，"$\red{z}$"替代"主"。而一个"$\red{j}$"与"$\red{z}$"组成的序列则可以描述当前的队列。为了让教主看得尽量舒服，你必须调整队列，使得"$\red{jz}$"子串尽量多。每次调整你可以交换任意位置上的两个人，也就是序列中任意位置上的两个字母。而因为教主马上就来了，时间仅够最多作$\red{K}$次调整（当然可以调整不满$\red{K}$次），所以这个问题交给了你。

## 输入格式

输入文件$\red{welcome.in}$的第$\red{1}$行包含$\red{2}$个正整数$\red{N}$与$\red{K，}$表示了序列长度与最多交换次数。
第$\red{2}$行包含了一个长度为$\red{N}$的字符串，字符串仅由字母"$\red{j}$"与字母"$\red{z}$"组成，描述了这个序列。

## 输出格式

输出文件$\red{welcome.out}$仅包括一个非负整数，为调整最多$\red{K}$次后最后最多能出现多少个"$\red{jz}$"子串。

## 样例

#### 输入样例

```c
5 2
```

#### 输出样例

```c
2
```

## 提示

样例说明

第$\red{1}$次交换位置$\red{1}$上的$\red{z}$和位置$\red{4}$上的$\red{j，}$变为$\red{jzzzj}$；

第$\red{2}$次交换位置$\red{4}$上的$\red{z}$和位置$\red{5}$上的$\red{j，}$变为$\red{jzzjz}$。

最后的串有$\red{2}$个"$\red{jz}$"子串。

 数据规模

对于$\red{10\%}$的数据，有$\red{N≤}$$\red{10}$；

对于$\red{30\%}$的数据，有$\red{K≤}$$\red{10}$；

对于$\red{40\%}$的数据，有$\red{N≤}$$\red{50}$；

对于$\red{100\%}$的数据，有$\red{N≤}$$\red{500，}$$\red{K≤}$$\red{100}$
