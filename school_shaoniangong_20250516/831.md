## 题目描述

输入圆的半径$\red{R}$，求圆的面积和周长。

$\red{p_i=3.1415926535}$

面积$\red{=p_i\times r\times r}$

周长$\red{=2\times p_i\times r}$

## 输入格式

输入一个实数$\red{R}$，表示圆的半径

## 输出格式

第一行，输出圆的面积（保留两位小数）

第二行，输出圆的周长（保留两位小数）

## 样例

#### 输入样例

```C
3
```

#### 输出样例

```C
28.27
18.85
```

## 提示

圆周率可以取$\red{pi=3.14159265}$参与运算
