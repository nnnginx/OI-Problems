## 题目描述

呼和同学，在新年期间去聚会。地点在天河区。

聚会地点和呼和同学家距离为S米。

开车的话需在路途当中必须加油，每次加油时间是K秒，总共需要加油N次。

车每秒可以行驶M米。

到地方进行停车需要P秒。

请问能否在2小时之内到达聚会地点。

如果能到达可以请输出"YES",不可以输出"NO"。

## 输入格式

一行输入5个数，S代表总距离，K代表每一次加油时间。N代表加油次数。M代表每秒行驶距离，P代表停车时间。

## 输出格式

输出一行，"YES" 或者 "NO"

## 输入样例

```
1000 120 40 1 10
```

## 输出样例

```
YES
```

## 输入样例

```
1000 120 50 1 300
```

## 输出样例

```
NO
```

## 数据范围

所有数据在int 范围
