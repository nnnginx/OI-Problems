## 题目描述

输入$\red{N}$个数，计算数字$\red{M}$的出现次数，并输出$\red{M}$的第一次出现的位置

## 输入格式

总共`N+2行`数据。第一行为数字N。接下来`N行`，每行一个整数。最后一行为数字$\red{M}$。
在上述的$\red{N}$的数据中查找$\red{M}$第一次出现的位置及总共出现的次数。

## 输出格式

输出$\red{M}$首次出现的位置和次数。未找到的位置输出$\red{0}$,次数输出$\red{0}$

## 样例

#### 输入样例

```C
5
52
18
18
654
18
18
```

#### 输出样例

```C
2 3
```

## 提示

$\red{n<=100000}$
