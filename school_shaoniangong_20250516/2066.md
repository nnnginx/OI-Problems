## 题目描述

$\red{FJ}$给他的奶牛用二进制进行编号，每个编号恰好包含$\red{K }$个"$\red{1}$" $\red{(1 <= K <= 10)，}$且必须是$\red{1}$开头。$\red{FJ}$按升序编号，第一个编号是由$\red{K}$个"$\red{1}$"组成。

请问第$\red{N(1 <= N <= 10^7)}$个编号是什么。

## 输入格式

第 $\red{1 }$行：两个以空格分隔的整数 $\red{N }$和 $\red{K}$。

## 输出格式

无

## 样例

#### 输入样例

```c
7 3
```

#### 输出样例

```c
10110
```
