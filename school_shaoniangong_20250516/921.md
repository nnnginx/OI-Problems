## 题目描述

给出一组样本数据，包含$\red{n}$个浮点数，计算其均值，精确到小数点后$\red{4}$位。

## 输入格式

输入有两行，

`第一行`包含一个整数$\red{n}$（$\red{n}$小于$\red{100}$），代表样本容量；

`第二行至第n+1行`包含$\red{n}$个绝对值不超过$\red{1000}$的浮点数，代表各个样本数据。

## 输出格式

输出一行，包含一个浮点数，表示均值，精确到小数点后$\red{4}$位。

## 样例

#### 输入数据

```C
2
1.0 
3.0
```

#### 输出数据

```C
2.0000
```
