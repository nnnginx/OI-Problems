## 题目描述

已知$\red{m}$、$\red{n}$为整数,且满足下列两个条件：

- （`1`） $\red{m、n∈1,2,3,……,k}$，即$\red{1≤m,n≤k}$
- （`2`） $\red{(n^2-mn-m^2)^2=1}$

你的任务是：编程由键盘输入正整数$\red{k}$（$\red{1≤k≤10^9}$），求一组满足上述两个条件的$\red{m}$、$\red{n}$，并且使$\red{m^2+n^2}$的值最大。

## 输入格式

一个正整数$\red{k}$。

## 输出格式

输出两行，按照格式输出对应的$\red{m}$和$\red{n}$的值

## 样例

#### 输入样例

```C
1995
```

#### 输出样例

```C
m=987 
n=1597
```
