## 题目描述

`ROMA城`中有一些古典的印度式建筑,这些建筑和周围的欧洲建筑风格格格不人。这些伪装成教堂的建筑其实是某国特工的基地。`Tomas `接受了一项任务,就是从某个教堂出发,逐个访问这些教堂,搞清楚每一个教堂的内部结构。并回到出发的地方。这些教堂很有规律地构成了一个$\red{m \times n}$的矩形，每个教堂和它的八个方向的教堂有直接的路径相连。水平或垂直方向相邻的教堂之间的路程均为$\red{1}$。请问`Tomas`至少需要走多远的路,才能完成这个危险而艰巨的任务呢?

## 输入格式

输人一行两个整数$\red{m}$和$\red{n(m,n≤}$$\red{10 000)}$。

## 输出格式

输出一行一个实数,表示最少需要走的路程,保留两位小数。

## 样例

#### 输入样例

```c
2 3
```

#### 输出样例

```c
6.00
```
