## 题目描述

在高为 $\red H$ 的天花板上有$\red n$ 个小球，体积不计，位置分别为$\red{ 0，1，2，…．n-1}$。在地面上有一个小车（长为 $\red L$，高为 $\red K$，距原点距离为 $\red{S_1}$）。已知小球下落距离计算公式为 $\red{d＝1/2\times g\times (t^2)}$，其中 $\red {g=10}$，$\red t$ 为下落时间。地面上的小车以速度 $\red V$ 前进。

　　如下图：
![img](./661/file/1.jpg)

　　小车与所有小球同时开始运动，当小球距小车的距离 $\red{<= 0.00001}$ 时，即认为小球被小车接受（小球落到地面后不能被接受）。

　　请你计算出小车能接受到多少个小球。

## 输入格式

键盘输人：

　　$\red{H，S_1，V，L，K，n （L<=H，n <=100000）}$

## 输出格式

屏幕输出：

　　小车能接受到的小球个数。

## 样例

#### 输入样例

```c
5.0 9.0 5.0 2.5 1.8 5
```

#### 输出样例

```c
1
```
