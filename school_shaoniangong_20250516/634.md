## 题目描述

农夫约翰的奶牛们喜欢通过电邮保持联系,于是她们建立了一个奶牛电脑网络,以便互相交流.这些机器用如下的方式发送电邮:如果存在一个由 $\red c$ 台电脑组成的序列 $\red{a1,a2,...,a(c)}$,且 $\red{a1}$ 与 $\red{a2}$ 相连,$\red{a2}$ 与 $\red{a3}$ 相连,等等,那么电脑 $\red{a1}$ 和 $\red{a(c)}$就可以互发电邮.

很不幸,有时候奶牛会不小心踩到电脑上,农夫约翰的车也可能碾过电脑,这台倒霉的电脑就会坏掉.这意味着这台电脑不能再发送电邮了,于是与这台电脑相关的连接也就不可用了.

有两头奶牛就想:如果我们两个不能互发电邮,至少需要坏掉多少台电脑呢？

请编写一个程序为她们计算这个最小值和与之对应的坏掉的电脑集合.

以如下网络为例:

<pre>
  1* 
 / 
3 - 2* 
</pre>

这张图画的是有$\red 2$条连接的$\red 3$台电脑.我们想要在电脑$\red 1$和$\red 2$之间传送信息.电脑$\red 1$与$\red 3$、$\red 2$与$\red 3$直接连通.如果电脑$\red 3$坏了,电脑$\red 1$与$\red 2$便不能互发信息了.

## 输入格式

第$\red 1$行: 四个由空格分隔的整数:$\red{N,M,c1,c2}$.$\red N$ 是电脑总数$\red{(1<=N<=100)}$,电脑由$\red 1$到$\red N$ 编号.$\red M$ 是电脑之间连接的总数$\red{(1<=M<=600)}$.最后的两个整数 $\red{c1}$ 和 $\red{c2}$ 是上述两头奶牛使用的电脑编号.连接没有重复且均为双向的(即如果 $\red{c1}$ 与 $\red{c2}$ 相连,那么 $\red{c2}$ 与 $\red{c1}$ 也相连).两台电脑之间至多有一条连接.电脑 $\red{c1}$ 和 $\red{c2}$ 不会直接相连.

第 $\red 2$ 到 $\red{M+1}$ 行: 接下来的 $\red M$ 行中,每行包含两台直接相连的电脑的编号.

## 输出格式

输出共有两行.

第一行是使电脑 $\red{c1}$和 $\red{c2}$不能互相通信需要坏掉的电脑数目的最小值.

第二行是排好序的坏掉的电脑的编号列表.注意 $\red{c1}$ 和 $\red{c2}$ 都不能坏掉.如果有多种可能情况,输出第一个数最小的一种,如果第一个数相同,则输出第二个数最小的一种,依此类推.

## 样例

#### 输入样例

```c
3 2 1 2 
1 3 
2 3
```

#### 输出样例

```c
1 
3
```
