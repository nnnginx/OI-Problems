# Description

huhe最近新买了一本字典,他十分高兴,因为这上面的单词都十分的和谐,他天天查字典

# Format

## Input

1个整数N,表示字典里面的单词数量。

接下来N行,每行一个字符串,表示一个单词。

然后第N+2行,一个整数M,表示要查的单词数。

接下来M行,每行一个字符串,表示一个要查的单词。

## Output

对于每一个要查的单词,如果在字典里面,就输出’Yes’,否则输出’No’,一行一个。

# Samples

```input1
2

i

you

1

love
```

```output1
No
```

# Data Size & Hint

1<N<10000

1<M<100

每个单词的长度[1..100]

保证都是小写字母。
