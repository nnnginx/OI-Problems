## 题目描述

`7`月`17`日是`Mr.W`的生日，`ACM-THU`为此要制作一个体积为$\red{Nπ}$的$\red{M}$层生日蛋糕，每层都是一个圆柱体。

设从下往上数第$\red{i}$层蛋糕是半径为$\red{R_i}$, 高度为$\red{H_i}$的圆柱。

当$\red{i < M}$时，要求$\red{R_i > R_{i+1}}$且$\red{H_i > H_{i+1}}$。

由于要在蛋糕上抹奶油，为尽可能节约经费，我们希望蛋糕外表面（最下一层的下底面除外）的面积$\red{Q}$最小。

令$\red{Q = Sπ}$ ，请编程对给出的$\red{N}$和$\red{M}$，找出蛋糕的制作方案（适当的$\red{R_i}$和$\red{H_i}$的值），使$\red{S}$最小。

除$\red{Q}$外，以上所有数据皆为正整数 。

## 输入格式

输入包含两行，第一行为整数$\red{N（N <= 10000）}$，表示待制作的蛋糕的体积为$\red{Nπ}$。

第二行为整数$\red{M(M <= 20)}$，表示蛋糕的层数为$\red{M}$。

## 输出格式

输出仅一行，是一个正整数$\red{S}$（若无解则$\red{S = 0}$）。

## 样例

#### 输入样例

```c
100
2
```

#### 输出样例

```c
68
```

## 提示

$\red{1≤N≤10000}$,
$\red{1≤M≤20}$
