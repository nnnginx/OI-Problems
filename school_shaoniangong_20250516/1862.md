## 题目描述

兔子们在玩两个串的游戏。给定两个字符串$\red{S}$和$\red{T ,}$兔子们想知道$\red{T}$在$\red{S}$中出现了
几次,分别在哪些位置出现。注意$\red{T}$中可能有$\red{"?"}$字符,这个字符可以匹配任何字符。

## 输入格式

两行两个字符串，分别代表$\red{S}$和$\red{T}$。

## 输出格式

第一行一个正整数$\red{k,}$表示$\red{T}$在$\red{S}$中出现了几次。
接下来$\red{k}$行正整数,分别代表$\red{T}$每次在$\red{S}$中出现的开始位置。按照从小到大的顺序
输出,$\red{S}$下标从$\red{0}$开始。

## 样例

#### 输入样例

```c
ababcadaca
a? a
```

#### 输出样例

```c
0
5
```

## 提示

对于$\red{10\%}$的数据，$\red{S}$和$\red{T}$的长度不超过$\red{100}$;

对于另外$\red{20\%}$的数据,$\red{T}$中无$\red{“?”}$;

对于$\red{100\%}$的数据,$\red{S}$长度不超过$\red{10*5}$,$\red{T}$长度不会超过$\red{S}$。$\red{S}$中只包含小写字母,$\red{T}$
中只包含小写字母和$\red{“?”}$。
