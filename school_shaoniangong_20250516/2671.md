## 题目描述

$\red{nodgd}$写了一篇文章，自认为这是一篇好文章。

$\red{nodgd}$的文章由$\red{n}$个小写英文字母组成。文章的一个子串指的是文章中的一段连续的字母，子串的长度就是这一段的字母个数。

$\red{nodgd}$在文章中用了排比、对偶、前后照应之类的手法，所以就有很多个子串是相同或者相近的。为了向大家证明这是一篇好文章，$\red{nodgd}$决定给自己的文章进行 评分。

$\red{nodgd}$首先确定了一个整数$\red{m}$，然后统计出文章中有多少个不相同的长度为$\red{m}$的子串，这个数量就是文章的评分。

然而，$\red{nodgd}$懒得老老实实计算这个评分了，就把任务丢给了你。

## 输入格式

第一行包含两个整数$\red{n}$,$\red{m}$，表示文章的长度和需要统计的子串长度。

第二行包含一个长度为$\red{n}$的只包含小写字母的字符串。

## 输出格式

输出一行一个整数，表示文章的评分。

## 样例

#### 输入样例1

```c
5 3

aaaab
```

#### 输出样例1

```c
2
```

#### 输入样例2

```c
9 3

abcabacba
```

#### 输出样例2

```c
7
```

## 提示

数据范围

对于$\red{30\%}$的数据，$\red{1≤}$$\red{m≤}$$\red{n≤}$$\red{200}$；

对于$\red{50\%}$的数据，$\red{1≤}$$\red{m≤}$$\red{n≤}$$\red{2000}$；

对于另外$\red{20\%}$的数据，$\red{1≤}$$\red{m≤}$$\red{50≤}$$\red{n≤}$$\red{200000}$；

对于$\red{100\%}$的数据，$\red{1≤}$$\red{m≤}$$\red{n≤}$$\red{200000}$。
