

# Description
小华参加了编程比赛。比赛规则如下：共有若干道题目，每道题目的答案可以随时提交评测。如果答案正确，则记录通过该题的时间；如果答案错误，则记录一次错误次数。当某道题目被正确解答时，该题的每次错误次数会在通过时间上增加20分钟的罚时。比赛结束后，小华查看了N支队伍的成绩，比赛共有M道题目。现在，他需要计算每支队伍完成所有题目所用的总时间。

# Format

## Input
第一行包含两个整数N和M，分别表示队伍数量和题目总数。

接下来N行，每行描述一支队伍的成绩：

第一个整数K表示该队伍通过的题目数量。

第二个整数P表示该队伍的总罚时次数（即所有题目的错误次数之和）。

接下来是K个形如h:m:s的时间，表示该队伍通过每道题目所用的时间（小时:分钟:秒）。每个时间部分不包含前导零，且用单个空格分隔。

## Output
输出N行，每行对应一支队伍的总用时，格式为h:m:s。其中，小时、分钟和秒都不包含前导零，且分钟和秒的值不超过59。

# Samples

```input1
2 3
1 5 0:38:56
3 0 0:1:2 1:2:3 2:3:4
```

```output1
2:18:56
3:6:9
```

# Limitation
队伍数量N：1 ≤ N ≤ 5000

通过题目数量K：1 ≤ K ≤ M ≤ 12

罚时次数P：0 ≤ P ≤ 10K

时间部分：0 ≤ h, m, s ≤ 59
