## 题目描述

呼和找到了所有的好友，

找到好友之后，他们准备玩儿一个有意思的游戏

这个游戏规则是。只能够上下左右移动。

在屏幕上随机出现一个4*4的矩阵数字。

同时在下方随机出现一个方向字母，分别是“L”，“R”，“U”，“D”

代表左右上下

通过4个方向的移动，进行相加相同数字。

相加以后的数字此次不在进行相加。

通过M次操作以后，请输出最终结果

## 输入格式

输入4*4的矩阵。

第四行输入一个M（<=15）

接下来的M行，每一行有一行字母，代表移动方向

## 输出格式

输出最终矩阵形态。

## 输入样例

```
2 2 2 0
2 2 4 4
2 2 4 8
0 2 2 0
4
R
U
L
D
```

## 输出样例

```
0 0 0 0 
4 0 0 0 
8 2 0 0 
4 16 4 0
```

## 提示

第一次"R"操作以后

```
0	0	2	4
0	0	4	8
0	4	4	8
0	0	0	4
```

第二次"U"操作以后

```
0	4	2	4
0	0	8	16
0	0	0	4
0	0	0	0
```

第三次"L"操作以后

```
4	2	4	0
8	16	0	0
4	0	0	0
0	0	0	0
```

第四次"D"操作以后

```
0	0	0	0
4	0	0	0
8	2	0	0
4	16	4	0
```
