## 题目描述

有很多显示数字的$\red{LED}$灯管，每一位数字都由$\red{7}$段构成，如下所示：

![img](./2574/file/1.png)

现在你能够点亮且正好点亮$\red{n}$段灯管，那么你能够点亮的所有数字之和最大是多少呢？

## 输入格式

一行输入$\red{1}$个整数$\red{n}$

## 输出格式

输出一个整数代表答案。

## 样例

#### 输入样例1

```c
4
```

#### 输出样例1

```c
4
```

#### 输入样例2

```c
7
```

#### 输出样例2

```c
11
```

## 提示

对于$\red{30\%}$的数据 $\red{0<=n<=100}$

对于$\red{100\%}$的数据 $\red{0<=n<=1000000}$
