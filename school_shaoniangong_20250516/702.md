## 题目描述

给你$\red{ n}$ 根火柴棍，你可以拼出多少个形如“ $\red{A+B=C}$ ”的等式？等式中的 $\red{A 、 B 、 C }$是用火柴棍拼出的整数（若该数非零，则最高位不能是 $\red0$ ）。

用火柴棍拼数字 $\red{0-9}$ 的拼法如图所示：

![img](./702/file/49.png)

注意：

$\red{1}$. 加号与等号各自需要两根火柴棍

$\red{2}$. 如果 $\red{A\ne B}$，则 $\red{A+B=C}$ 与 $\red{B+A=C}$ 视为不同的等式$\red{A,B,C\geq 0}$）

$\red{3}$. $\red{n}$ 根火柴棍必须全部用上

## 输入格式

共一行，有一个整数 $\red{n}$ 。

## 输出格式

共一行，表示能拼成的不同等式的数目。

## 样例

## 样例输入1

```C
14
```

## 样例输出1

```C
2
```

## 样例输入2

```C
18
```

## 样例输出2

```C
9
```

## 样例二说明

$\red{9}$ 个等式为：

$\red{0+4=4}$

$\red{0+11=11}$

$\red{1+10=11}$

$\red{2+2=4}$

$\red{2+7=9}$

$\red{4+0=4}$

$\red{7+2=9}$

$\red{10+1=11}$

$\red{11+0=11}$

## 数据范围与提示

对于$\red{100\%}$ 的数据，$\red{n\leq24}$。
