


## 题目描述
面面是一位密室爱好者，一天面面与同学去玩密室逃脱，面对紧锁的大门，面面一筹莫展。经过一番的寻找，面面发现了门下有一个电子密码锁，密码锁会随机生成两个正整数$\red a$,$\red b$，在密码栏需要输入对面的密码才能解锁大门。

同行的明明发现了一张线索图，图上记录着破解密码的方法，经过推导得出密码就是这两位数$\red a$,$\red b$的最小公倍数。面面清楚的记得自己曾写过求最小公倍数的代码，现在你能帮他把代码复原吗？

## 输入格式
共两行；

`第一行`包括一个正整数$\red a$，

`第二行`包括一个正整数$\red b$；

## 输出格式
共一行，为一个整数是密码锁的密码。
## 样例
#### 输入样例
```C
6 
8
```
#### 输出样例
```C
24
```

## 提示
对于所有的数据满足：
$\red{1<=a<=100,1<=b<=100}$;
