## 题目描述

有$\red{N}$个正方形，依次给出它们的边长，要求可以换掉一些正方形：如果变长为$\red{A_i}$的正方形换成边长为$\red{B_i}$的正方形所需代价为：$\red{|A_i-B_i| \times |A_i-B_i|}$。问至少花费多少代价使得这$\red{N}$个正方形面积总和为$\red{M}$？若无论如何也不可能使这$\red{N}$个正方形面积总和为$\red{M}$则输出$\red{-1}$。

## 输入格式

第 $\red{1 }$行：两个空格分隔的整数，$\red{N (1<=N<=10) }$和 $\red{M}$
$\red{(1<=M<=10,000)}$。

第 $\red{2..1+N }$行：每行包含整数 $\red{A_1 }$到 $\red{A_N }$之一，描述输入正方形的边长
$\red{(1<=A_i<=100)}$。

## 输出格式

第 $\red{1 }$行：交换瓷砖以获得 $\red{M }$单位总面积的最低成本，如果不可能，则为 $\red{-1}$。

## 样例

#### 输入样例

```c
3 6 
3 
3
1
```

#### 输出样例

```c
5
```

## 提示

有$\red{3}$个瓷砖。两个是边长为 $\red{3 }$的正方形，一个是边长为 $\red{1 }$的正方形。我们想将它们交换为总面积为 $\red{6}$。

将一个边 $\red{3 }$方格交换为边 $\red{2 }$方格，并将另一个边 $\red{3 }$方格交换为边 $\red{1 }$方格。这给出了 $\red{4+1+1=6 }$的所需面积，成本为 $\red{4+1=5 }$个单位。
