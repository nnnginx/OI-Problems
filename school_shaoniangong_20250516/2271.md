## 题目描述

农民约翰正在把他的牛排成一行拍照$\red{（1≤}$$\red{N≤}$$\red{100,000).}$

顺序中第二头奶牛的高度为$\red{hi，}$所有奶牛的高度都不同。和所有他的奶牛照片一样，$\red{FJ}$希望这张照片看起来旧能漂亮 。

他认为，如果丽丽和丽丽的差异超过$\red{2}$倍，那么第二头牛看起来"不平衡"，其中丽丽和丽丽分别是她左边和右边比第二头牛高的 奶牛数。

也就是说，如果$\red{Li}$和$\red{Ri}$中的较大者严格地大于这两个数字中较小者的两倍，则$\red{i}$是不平衡的。$\red{FJ}$希望 他的奶牛不要太多不平衡。

请帮助$\red{FJ}$计算不平衡奶牛的总数。

## 输入格式

第一行输入包含$\red{N.}$接下来的$\red{N}$行包含$\red{h1…}$$\red{hN   }$每行最多为$\red{100000000}$个非负整数。

## 输出格式

请输出不平衡奶牛数。

## 样例

#### 输入样例

```c
7
34
6
23
0
5
99
2
```




#### 输出样例

```c
3
```

## 提示

在本例中，高度$\red{34}$、$\red{5}$和$\red{2}$的奶牛不平衡。
