## 题目描述

在社交媒体上，经常会看到针对某一个观点同意与否的民意调查以及结果。例如，对某一观点表示支持的有 $\red{1498}$ 人，反对的有 $\red{902}$人，那么赞同与反对的比例可以简单的记为$\red{1498:902}$。


不过，如果把调查结果就以这种方式呈现出来，大多数人肯定不会满意。因为这个比例的数值太大，难以一眼看出它们的关系。对于上面这个例子，如果把比例记为 $\red{5:3}$，虽然与真实结果有一定的误差，但依然能够较为准确地反映调查结果，同时也显得比较直观。

现给出支持人数 $\red{A}$，反对人数 $\red{B}$，以及一个上限 $\red{L}$，请你将 $\red{A }$比 $\red{B}$ 化简为 $\red{A}$’比 $\red{B}$’，要求在 $\red{A}$’和 $\red{B}$’均不大于 $\red{L }$且 $\red{A}$’和 $\red{B}$’互质（两个整数的最大公约数是 $\red1$）的前提下，$\red{A’/B’ ≥ A/B}$且 $\red{A’/B’ - A/B}$ 的值尽可能小。

## 输入格式

输入共一行，包含三个整数 $\red{A，B，L}$，每两个整数之间用一个空格隔开，分别表示支持人数、反对人数以及上限。

## 输出格式

输出共一行，包含两个整数 $\red{A’，B’}$，中间用一个空格隔开，表示化简后的比例。

## 样例

## 输入样例

```
1498 902 10
```
## 输出样例

```
5 3
```

## 数据范围与提示

对于 $\red{100\%}$的数据，$\red{1  \leq  A  \leq  1,000,000，1  \leq   B  \leq   1,000,000，1  \leq   L  \leq   100， A/B  \leq   L。}$
