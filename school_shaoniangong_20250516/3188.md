# 题目描述

众所周知，散兵非常喜欢骂人，所以在他游历完提瓦特大陆后把认识的人分为两类：凡人和蝼蚁（doge）。散兵认识的每个人他都给了个编号，如果一个人的编号与另外一个人的编号至少有一个相同的数字(不要求在相同的位置），那么k们就被散兵认为是一对凡人。如今散兵认识了$n$ 个人，他太懒了，强制你帮他算一下里面共有多少对凡人，算完后他会给你 $1600$ 纠缠之缘作为奖励。

# 输入格式

第一行一个整数 $n$，代表散兵认识的人数。

接下来 $n$ 行，每行一个整数 $a_i$，代表散兵认识的人的编号。

# 输出格式

一个整数 $s$，代表散兵认识的人中有多少对凡人。

# 样例


```input1
8
1
14
5
141
9
19
8
10
```

```output1
11
```

# 数据范围

对于 $100\%$ 的数据，$1 \le n \le 10^4$，$1 \le a_i \le 10^{18}$。
