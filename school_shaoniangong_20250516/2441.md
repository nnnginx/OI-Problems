## 题目描述

$\red{Farmer John}$打算将电话线引到自己的农场，但电信公司并不打算为他提供免费服务。于是，$\red{FJ}$必须为此向电信公司支付一定的费用。

$\red{FJ}$的农场周围分布着$\red{N(1 <= N <= 1,000)}$根按$\red{1..N}$顺次编号的废弃的电话线杆，任意两根电话线杆间都没有电话线相连。一共$\red{P(1 <= P <= 10,000)}$对电话线杆间可以拉电话线，其余的那些由于隔得太远而无法被连接。

第$\red{i}$对电话线杆的两个端点分别为$\red{A_i}$、$\red{B_i，}$它们间的距离为 $\red{L_i (1 <= L_i <= 1,000,000)}$。数据中保证每对$\red{\{A_i，}$$\red{B_i\}}$最多只出现$\red{1}$次。编号为$\red{1}$的电话线杆已经接入了全国的电话网络，整个农场的电话线全都连到了编号为$\red{N}$的电话线杆上。

也就是说，$\red{FJ}$的任务仅仅是找一条将$\red{1}$号和$\red{N}$号电话线杆连起来的路径，其余的电话线杆并不一定要连入电话网络。 经过谈判，电信公司最终同意免费为$\red{FJ}$连结$\red{K(0 <= K < N)}$对由$\red{FJ}$指定的电话线杆。

对于此外的那些电话线，$\red{FJ}$需要为它们付的费用，等于其中最长的电话线的长度（每根电话线仅连结一对电话线杆）。如果需要连结的电话线杆不超过 $\red{K}$对，那 么$\red{FJ}$的总支出为$\red{0}$。

请你计算一下，$\red{FJ}$最少需要在电话线上花多少钱。

## 输入格式

第$\red{1}$行: $\red{3}$个用空格隔开的整数：$\red{N，}$$\red{P，}$以及$\red{K}$

第$\red{2..P+1}$行: 第$\red{i+1}$行为$\red{3}$个用空格隔开的整数：$\red{A_i，}$$\red{B_i，}$$\red{L_i}$

## 输出格式

第$\red{1}$行: 输出$\red{1}$个整数，为$\red{FJ}$在这项工程上的最小支出。如果任务不可能完成， 输出$\red{-1}$

## 样例

#### 输入样例

```c
5 7 1
1 2 5
3 1 4
2 4 8
3 2 3
5 2 9
3 4 7
4 5 6
```

#### 输出样例

```c
4
```

## 提示

输入说明:

一共有$\red{5}$根废弃的电话线杆。电话线杆$\red{1}$不能直接与电话线杆$\red{4}$、$\red{5}$相连。电话 线杆$\red{5}$不能直接与电话线杆$\red{1}$、$\red{3}$相连。其余所有电话线杆间均可拉电话线。电信 公司可以免费为$\red{FJ}$连结一对电话线杆。

输出说明:

$\red{FJ}$选择如下的连结方案：$\red{1->3}$；$\red{3->2}$；$\red{2->5，}$这$\red{3}$对电话线杆间需要的电话线的长度分别为$\red{4}$、$\red{3}$、$\red{9}$。$\red{FJ}$让电信公司提供那条长度为$\red{9}$的电话线，于是，
他所需要购买的电话线的最大长度为$\red{4}$。
