## 题目描述

奶牛冰壶是$\red{Moolympics}$上一项受欢迎的寒冷天气运动。

和普通的冰壶运动一样，这项运动有两支队伍，每支队伍都要在冰面上滑动$\red{N}$块重石头$\red{(3 <= N <= 5}$万$\red{)}$。

在游戏的最后，冰上有$\red{2N}$个石头，每个都位于不同的$\red{2D}$点。

然而，在奶牛版的冰壶比赛中得分有点奇怪。如果一块石头被放在一个三角形中，而这个三角形的边角是对手所有的石头，那么这块石头就被称为"俘获"了（一块位于三角形边界上的石头也算作俘获）。一个队伍的得分是指捕获对手石头的数量。

请帮助计算一场奶牛冰壶比赛的最终得分，给出所有$\red{2N}$块石头的位置。

有两支队伍在比赛，一队可以一次取出$\red{3}$个点来，所围成的三角形覆盖的区域可以"捕获"对方的点，问两支队伍各能够捕获对 方多少个点。




## 输入格式

第一行:整数$\red{N}$。

第$\red{2 . .1+N}$行:每一行包含$\red{2}$个整数，指定$\red{a}$组一块石头的$\red{x}$和$\red{y}$坐标（每个坐标位于范围$\red{-40,000 ..+ 40000}$）


第 $\red{2 + N. .1+2N}$行:每一行包含$\red{2}$个整数，指定$\red{B}$组一块石头的$\red{x}$和$\red{y}$坐标每个坐 标位于范围$\red{-40,000 ..+ 40000}$。

## 输出格式

第一行:两个空格分隔的整数，给出团队$\red{A}$和团队$\red{B}$的分数。

## 样例

#### 输入样例

```c
4
0 0
0 2
2 0
2 2
1 1
1 10
-10 3
10 3

```



#### 输出样例

```c
1 2
```

## 提示

每队拥有$\red{4}$颗宝石。

$\red{A}$队在$\red{(0,0)}$、$\red{(0,2)}$、$\red{(2,0)}$和$\red{(2,2)}$处有石子，$\red{B}$队 在$\red{(1,1)}$、$\red{(1,10)}$、$\red{(-10,3)}$和$\red{(10,3)}$处有石子。


$\red{A}$队在$\red{(1,1)}$处夺取对手的宝石。$\red{B}$队在$\red{(0,2)}$和$\red{(2,2)}$处夺取对手的宝石。
