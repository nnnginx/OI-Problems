## 题目描述

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

![image](./3092/file/ntAz7XznattuLoeDYnIbz.png)

## 输入格式

`第一行`包含一个正整数$\red{n}$，表示序列中元素个数。$\red{1 ≤n≤ 10000000}$。

`第二行`包含$\red{n}$个整数，依次给出序列的每个元素，相邻两个整数之间用单个空格隔开。

## 输出格式

输出一个整数

## 样例

#### 输入样例

```C
12
0 1 0 2 1 0 1 3 2 1 2 1
```

#### 输出样例

```C
6
```
