## 题目描述

输入$\red{a}$,$\red{b}$。输出$\red{a*b}$的结果

## 输入格式

一行两个数$\red{a}$,$\red{b}$

## 输出格式

一行一个整数，$\red{a*b}$的结果

结果里不能有多余的前导$\red{0}$，即如果结果是$\red{342}$，那么就不能输出为$\red{0342}$。

## 样例

#### 输入样例

```C
2 3
```

#### 输出样例

```C
6
```

## 数据范围与提示

$\red{a}$,$\red{b}$两数的位数不超过$\red{500}$位

保证输入的两数开头第一位不是$\red{0}$
