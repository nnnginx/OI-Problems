# 题目描述

给定一个 n  行 m 列的区域，`'.'` 表示肥沃土地 `'*'` 表示贫瘠的山地。对于上下左右相邻的肥沃土地，由于交通便利，可以认为属于连通的区域。

小图是个非常勤奋的人，他决定开垦一块贫瘠的山地，将其变为肥沃的土地，作为自己的领土。这份勤劳感动了上帝，于是上帝对他说：“去做吧，然后与这块土地连通的所有肥沃土地，都是你的应许之地。”

小图想知道，如果他选择最优的开垦方案，能获得的最大的“应许之地”有多大。

## 输入

第一行两个整数 n, m，表示这块区域的大小。

接下来 n 行，每行一个长度为 m  的字符串，表示这块区域的地形。

## 输出

输出一行，一个整数，表示最优方案下的应许之地的大小。

# 样例

```input1
3 3
...
***
...
```

```output1
7
```

# 数据说明

对于所有测试点，保证 1<=n,m<=1000，至少存在一块贫瘠的山地。
