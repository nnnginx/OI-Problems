## 题目描述

贝西正在和她的朋友埃尔西建一座堡垒。像任何好的堡垒一样，这座堡垒需要从坚固的框架开始。贝西想建造一个一米宽的矩形框架，她将在其上建造堡垒。

贝西已经选择了一个建造堡垒的地点一块面积为$\red{N}$米$\red{×}$$\red{M}$米的土地$\red{(1≤}\red{N,M≤}$$\red{200)}$

不幸的是，该场地有一些沼泽地区无法用于支撑框架。

请帮助贝西确定她可以用堡垒覆盖的最大区域（框架支撑的矩形区域），这样框架就不会坐在任何沼泽地上。

## 输入格式

第$\red{1}$行包含整数$\red{N}$和$\red{M}$。

接下来的$\red{N}$行每个都包含$\red{M}$字符，形成一个描述站点的网格。

"."的字符表示普通草，而"$\red{X}$"表示沼泽点。

## 输出格式

一个整数，表示贝西可以用她的堡垒覆盖的最大面积。

## 样例

#### 输入样例

```c
5 6
......
..X..X
X..X..
......
..X...
```


#### 输出样例

```c
16
```

## 提示

在该示例中，最优帧的位置由下面的“f”表示：

```
.ffff.
.fX.fX
Xf.Xf.
.ffff.
..X...
```
