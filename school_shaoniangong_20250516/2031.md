## 题目描述

小$\red{Z}$有一个数字序列$\red{a_1,a_2,...,a_n,}$长度为$\red{n,}$小$\red{Z}$只有一个操作:选
定$\red{p(1≤}$$\red{p≤}$$\red{n)，}$然后把$\red{a_p}$从序列里拿出，然后再插入到序列中任意位置。

比如$\red{a}$序列为$\red{1,2,4,5,3, p=5,}$可以取出$\red{3,}$然后在任意位置插入，可
以变为$\red{1,2,3,4,5}$。

现在给你一个序列$\red{a,}$问你是否可以通过一次操作把整个序列从小到大
排好序变成不降的。

## 输入格式

第一行一个整数$\red{n,}$第二行空格隔开的$\red{n}$个整数，代表$\red{a}$序列。

## 输出格式

如果可以一次操作可以排好序，输出"$\red{YES}$",否则输出"$\red{NO}$"。

## 样例

#### 输入样例

```c
5
1 2 4 5 3
```

#### 输出样例

```c
YES
```

## 提示

对于$\red{30\%}$的数据，满足$\red{n≤}$$\red{1000}$。

对于$\red{60\%}$的数据，满足$\red{n≤}$$\red{10^5}$。

对于$\red{100\%}$的数据，满足$\red{n≤}$$\red{10^6,1≤}$$\red{a_i≤}$$\red{10^6}$。
