## 题目描述

编程找出$\red{m}$到$\red{n}$之间的整数满足下列条件的四位数。

- `1.`这四位数是形如$\red{abcd}$，$\red{abcd}$是$\red{11}$的倍数；
- `2.`它的每一位数是互不相等的自然数；
- `3.`$\red{b+c=a}$；
- `4.`$\red{bc}$是完全平方数。

## 输入格式

一行两个整数$\red{m}$和$\red{n}$。（$\red{1000≤m、n≤9999}$）

## 输出格式

`共二行`，

`第一行`是所有满足条件的特殊自然数，数之间有一空格。

`第二行`特殊自然数的个数；如果没有则输出只有一行，“`No Answer.`”

## 样例

#### 输入数据1

```C
9200 9500
```

#### 输出数据1

```C
9361
1
```

#### 样例输入2

```C
1000 1001
```

#### 样例输出2

```C
No Answer.
```
