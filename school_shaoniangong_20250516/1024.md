## 题目描述

一个长度为$\red{n}$（$\red{n>0}$）的序列中存在“`有趣的跳跃`”当前仅当相邻元素的差的绝对值经过排序后正好是从$\red{1}$到($\red{n-1}$)。

例如，`1 4 2 3`存在“`有趣的跳跃`”，因为差的绝对值分别为`3,2,1`。

当然，任何只包含单个元素的序列一定存在“`有趣的跳跃`”。

你需要写一个程序判定给定序列是否存在“`有趣的跳跃`”。

## 输入格式

`多组输入` 第一个数是$\red{n}$（$\red{0 < n < 3000}$），为序列长度，接下来有$\red{n}$个整数，依次为序列中各元素，

各元素的绝对值均不超过$\red{1,000,000,000}$。



## 输出格式

若该序列存在“`有趣的跳跃`”，输出"`Jolly`"，否则输出"`Not jolly`"。

## 样例

#### 输入样例

```C
4 1 4 2 3
```

#### 输出样例

```C
Jolly
```
