## 题目描述

给定含有$\red{n}$个元素的多重集合$\red{S}$，每个元素在$\red{S}$中出现的次数称为该元素的重数。多重集$\red{S}$中重数最大的元素称为众数。
例如，$\red{S={1，2，2，2，3，5}}$。多重集$\red{S}$的众数是$\red{2}$，其重数为$\red{3}$。
编程任务：对于给定的由$\red{n}$ 个自然数组成的多重集$\red{S}$，编程计算$\red{S}$ 的众数及其重数。

## 输入格式

文件的第$\red{1}$行多重集$\red{S}$中元素个数$\red{n(1<=n<=1300000)}$；接下来的$\red{n}$ 行中，每行有一个自然数,不超过$\red{65535}$。

## 输出格式

输出文件有$\red{2}$ 行，第$\red{1}$ 行给出众数，第$\red{2}$ 行是重数。

## 样例

#### 输入样例

```c
6               
1               
2
2
2
3
5
```

#### 输出样例

```c
2
3
```

## 提示

最后两组数据测试$\red{2}$秒。
