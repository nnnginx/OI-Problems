## 题目描述

已知有 $n$ 个在一条线上的点（如图，$n=3$），现在让你求出有多少条线段。

![图片](https://cdn.luogu.com.cn/upload/image_hosting/iqxvlna0.png)

图片中有 $3$ 个点，则有 $AB$、$BC$、$AC$ 三条线段。

## 输入格式

一行，一个整数 $n$，表示点的个数。

## 输出格式

一行，一个整数，表示有多少条线段。

## 样例 #1

### 样例输入 #1

```
3
```

### 样例输出 #1

```
3
```

## 样例 #2

### 样例输入 #2

```
5
```

### 样例输出 #2

```
10
```

## 提示

### 数据范围

对于 $100\%$ 的数据，$1≤n≤10^8$

### 样例说明

#### 样例组 #1

设 $3$ 个点为$A$、$B$、$C$，则有三条线段：$AB$、$BC$、$AC$

#### 样例组 #2

设 $5$ 个点为$A$、$B$、$C$、$D$、$E$，则有$10$条线段：

```AB,AC,AD,AE,BC,BD,BE,CD,CE,DE```
