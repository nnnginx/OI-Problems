

# Description

任何一个正整数都可以用2的幂次方表示。例如:137 =2^7 + 2^3 + 2^0

，同时约定次方用括号来表示，即a^b可表示为 a(b)。

由此可知，137 可表示为：2(7) + 2(3) + 2(0)，进一步：7 =2^2 + 2 + 2^0（2^1 用 2 表示），3 = 2 +2^0， 所以最后 137 可表示为：2(2(2)+2+2(0)) + 2(2+2(0)) + 2(0)。

又如：1315 =2^10 + 2^8+ 2^5+ 2 + 1，所以 1315 最后可表示为：2(2(2+2(0))+2) + 2(2(2+2(0))) + 2(2(2) + 2(0)) + 2 + 2(0)。

# Format

## Input

一行，一个正整数 n。（n ≤ 20000）.

## Output

一行，为符合约定的 n 的 0，2 表示。（在表示中不能有空格）

# Samples

```input1
137
```

```output1
2(2(2)+2+2(0))+2(2+2(0))+2(0)
```
