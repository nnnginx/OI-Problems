## 题目描述

小明想要堆积木，他手中的积木都是边长为$\red{1}$的正方体，并且他希望他堆得越高越好。但是为了满足稳定性的结构，堆积木的时候一定要边和边紧贴，不可以交错，如下所示：

![img](./2804/file/1.png)

左侧两种例子是可行的，而右侧的三种方式都不行。

同时需要让每一个堆上去的方块足够稳定，对于每一个方块放上去的时候必须保证他的正下方以及他正下方的前后左右四个位置也有方块，如下所示：

![img](./2804/file/2.png)

左侧三种是可行的，右侧两种不可行。

他有$\red{n}$个方块，请问最多能堆多高呢？

## 输入格式

输入仅包含一个整数$\red{n}$

## 输出格式

输出一个整数表示答案

## 样例

#### 输入样例1

```c
6
```

#### 输出样例1

```c
2
```

#### 输入样例2

```c
5
```

#### 输出样例2

```c
1
```

#### 输入样例3

```c
20
```

#### 输出样例3

```c
3
```

## 提示

对于$\red{100\%}$的数据满足$\red{1<=n<=10^9}$；

其中$\red{40\%}$的数据满足$\red{n<=100}$
