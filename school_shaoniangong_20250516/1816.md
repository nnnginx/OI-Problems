## 题目描述

监狱里除了修罗王以外,其他的囚犯都喜欢拉帮结派以壮大势力,但拉帮结派并不是无.
原则的.而是要看发展对象是不是看上去够顺眼，所谓顺眼.即是发展对象拥有的一些个性
特征符合帮派的要求。例如$\red{N= 3,}$表示符合帮派的个性特性共有$\red{3}$个，假设以$\red{a,b,c}$定义。
我们称$\red{(a,b,c)}$为集合$\red{S.}$面发展对象只要拥有其中的一个或多个特征都会被帮派选中，即
符合子集$\red{(S)= \{(),(a),(b),(c),(a,b),(b,c),(a,c),(a,b,c)\}}$中的任何一个均可。当然，
你可能要问了.这空集没有任何特征也符合条件?对啊."和光同尘"也是一种很受大家欢迎
的生活态度啊。

现输入$\red{N,}$求$\red{(S)}$的子集.

## 输入格式

一个整数即$\red{N}$。

## 输出格式

输出所有子集,每个子集占一行。

#### 输入样例

```c
2
```

#### 输出样例

```c

```
