## 题目描述

奶嘘又在外面活动蹄子了！有$\red{N}$头嘘在无限长的单车道轨道上慢跑$\red{（1≤}$$\red{N≤}$$\red{100000）}$。每头奶嘘都从赛道上一个不同的位置出发，有些奶嘘慢跑以不同的速度。

由于跑道上只有一条车道，奶嘘无法相互超越。当奶嘘追上另一头奶嘘的速度越快，她必须减速以避免与另一头奶嘘相遇，成为同一跑步组的一员。奶嘘将奔跑$\red{T}$分钟$\red{（1<=T<=100000000）}$。请帮助 农民约翰确定此时还剩多少组。

如果两头奶嘘处于同一组，则应将其视为同一组的一部分$\red{T}$分钟结束时的相同位置。

## 输入格式

第一行输入包含两个整数$\red{N}$和$\red{T}$。

以下$\red{N}$行分别包含一头奶嘘。位置为非负整数，速度为正整数这两个数字最多为$\red{10}$亿。所有奶嘘都从不同的位置，这些位置将以递增的顺序在输入。

## 输出格式

一个整数，指示$\red{T}$分钟后剩余的组数。

## 样例

#### 输入样例

```c
5 3
0 1
1 2
2 3
3 2
6 1
```

#### 输出样例

```c
3
```
