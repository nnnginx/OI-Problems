
#### 题目限制

1000 ms    256 M

#### 题目描述

给出一个数 $L$ ，求最小的长度为 $L$ 的十进制数 $N$ （没有前导 $0$ ），满足 $N$ 是（ $2,3,5,7$ ）的公倍数。如果不存在，则输出  $-1$ 。

#### 输入格式

输入一个数 $L$ （ $1\le L\le 10^5)$

#### 输出格式

输出对应的答案，如果不存在则输出 $-1$ 。

#### 数据范围

对于 $42\%$ 的数据， $1 \le L \le 100$ ；

对于 $100\%$ 的数据， $1 \le L \le 10^5$ 。

#### 输入样例 1

```
1
```

#### 输出样例 1

```
-1
```
