## 题目描述

给定一个仅由$\red{0}$和$\red{1}$组成的字符串$\red{s，}$$\red{|s|}$是它的长度。

你需要选择一个整数$\red{k(k>0)，}$然后找到一个长度为$\red{k}$序列$\red{a}$满足：

$\red{1＜}$$\red{a_1＜}$$\red{a_2＜}$...$\red{<a_k<=|s|}$;

$\red{a_{i-1}+1<a_i(2<=i<=k)}$;

将字符串$\red{s}$中$\red{a_1,a_2,...,a_k}$位置的字符移除，其它字符位置不变进行重新拼合组成字符串$\red{s'}$。

是否存在一个序列$\red{a}$使得字符串$\red{s'}$满足$\red{s'}$$\red{_{i-1}<=s'}$$\red{_i(2<=i<=|s'}$$\red{) ?}$

给定字符串$\red{s，}$如果可以找到一个序列$\red{a}$满足条件输出 $\red{YES ，}$否则输出 $\red{NO }$。

## 输入格式

第一行一个整数$\red{t，}$表示数据组数。

对于每组数据，输入一行字符串$\red{S}$

## 输出格式

对于每组数据，$\red{YES}$或$\red{NO}$。

## 样例

#### 输入样例

```c
5
10101011011
0000
11111
110
1100
```

#### 输出样例

```c
YES
YES
YES
YES
NO
```

## 提示

对于$\red{50\%}$的数据，有$\red{2<=|s|<=10}$；

对于$\red{100\%}$的数据，有$\red{2<=|s|<=100,1<=t<=1000}$。
