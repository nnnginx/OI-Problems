# Background

一题多解，请务必用DFS解决

# Description

写一个程序来查找从最高点到底部任意处结束的路径，使路径经过数字的和最大。每一步可以走到左下方的点也可以到达右下方的点。

![image](./3033/file/2PEStxbzl3UGKITlFrVOw.jpeg)

# Format

## Input

第一个行包含  **R** ( **1** <= **R** <= **1000** ),表示行的数目。

后面每行为这个数字金字塔特定行包含的整数。

所有的被供应的整数是非负的且不大于 **100** 。

## Output

单独的一行,包含那个可能得到的最大的和。

# Samples

```input1
5
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5

```

```output1
30

```

# Limitation

对于 **100%** 的数据， **1** ≤ **r** ≤ **1000** ，所有输入在 **[ **0** , **100** ]** 范围内。
