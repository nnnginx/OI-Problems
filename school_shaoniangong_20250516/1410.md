## 题目描述

有一条奶牛冲出了围栏，来到了一处圣地（对于奶牛来说），上面用牛语写着一段文字。

现用汉语翻译为：

有 $\red{N}$ 个区间，每个区间 $\red{x,y}$表示提供的 $\red{x∼y}$ 共 $\red{y−x+1}$ 堆优质牧草。你可以选择任意区间但不能有重复的部分。

对于奶牛来说，自然是吃的越多越好，然而奶牛智商有限，现在请你帮助他。

## 输入格式

第一行一个整数 $\red{N}$。

接下来 $\red{N}$ 行，每行两个数 $\red{x,y}$，描述一个区间。

## 输出格式

输出最多能吃到的牧草堆数。

## 样例

#### 输入样例

```c
3
1 3
7 8
3 4
```

#### 输出样例

```c
5
```

## 提示

$\red{1≤ n ≤1.5×10^5，0≤ x ≤ y ≤3×10^6}$。
