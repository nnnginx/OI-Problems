## 题目描述

编程实现输入四个整数$\red {a,b,c,d }$，当$\red {a+b}$大于$\red{10}$时，输出$\red {a*b}$的值,否则当$\red {b+c>5}$时
输出$\red {c*d }$的值，否则当$\red {d<10}$或者$\red {a*c>100}$时输出"`Yes`"，否则输出"`No`"

## 输入格式

输入$\red{4}$个整数$\red{a}$,$\red{b}$,$\red{c}$,$\red{d}$，以空格隔开

## 输出格式

输出相应结果

## 样例

样例输入

```
1 6 9 7
```

样例输出

```
63
```
