## 题目描述

读入一个`双精度`浮点数，分别按输出格式“`%lf`”，“`%lf`”保留$\red{5}$位小数，“`%le`”和“`%lg`”的形式输出这个整数，每次在单独一行上输出。

## 输入格式

一个双精度浮点数。

## 输出格式

`第一行`是按“`%lf`”输出的双精度浮点数；

`第二行`是按“`%lf`”保留$\red 5$位小数输出的双精度浮点数；

`第三行`是按“`%le`”输出的双精度浮点数；

`第四行`是按“`%lg`”输出的双精度浮点数。

## 样例

#### 输入样例

```C
12.3456789
```

#### 输出样例

```C
12.345679
12.34568
1.234568e+001
12.3457
```
