
## 题目描述

给出N堆石子，`Alice`和`Bob`在玩去石子游戏，游戏规则为每次从一堆是石子从取出若干石子，并且至少要取一颗，而且一个人能取i堆中的石子当且仅当$\red {1..i-1}$堆的石子都已经取完。

`Alice`先手，拿到最后一颗石子的人算赢，问最后谁会赢。

## 输入格式

第一行一个数$\red N$，表示石子堆数。

第二行$\red N$个数，第i个数表示第i堆石子中含有的石子数目

## 输出格式

仅一行，输出赢家的姓名。

## 样例

#### 输入样例

```C
2
2 1
```

#### 输出样例

```C
Alice
```

## 提示

$\red {N<=50}$,每堆石子的个数 $\red{<= 10^9}$
