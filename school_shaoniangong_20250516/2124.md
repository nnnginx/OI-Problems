## 题目描述

农夫约翰的$\red{N(1 <= N <= 100,000)}$只奶牛排成了一队，每只牛都用编上了一个"血统编号"，该编号为范围$\red{0...1,000,000,000}$的整数。血统相同的奶牛有相同的编号，也就是可能有多头奶牛是相同的"血统编号"。

约翰觉得如果连续排列的一段奶牛有相同的血统编号的话，奶牛们看起来会更具有威猛。为了创造这样的连续段，约翰最多能选出$\red{k}$种血统的奶牛，并把他们全部从队列中赶走。

请帮助约翰计算这样做能得到的由相同血统编号的牛构成的连续段的长度最大是多少？

## 输入格式

第 $\red{1 }$行：两个空格分隔的整数：$\red{N }$和 $\red{K}$。

第 $\red{2..1+N }$行：第 $\red{i+1 }$行包含品种 $\red{ID B(i)}$。

## 输出格式

第 $\red{1 }$行：连续奶牛块的最大尺寸

$\red{FJ }$可以创建的相同品种 $\red{ID}$。

## 样例

#### 输入样例

```c
9 1 
2 
7 
3 
7
7
3
7
5 
7
```

#### 输出样例

```c
4
```

## 提示

阵容中有 $\red{9 }$头奶牛，品种 $\red{ID }$为 $\red{2}$、$\red{7}$、$\red{3}$、$\red{7}$、$\red{7}$、$\red{3}$、$\red{7}$、$\red{5}$、$\red{7}$。$\red{FJ }$想从该阵容中删除最多 $\red{1 }$个品种 $\red{ID}$。

通过删除所有品种 $\red{ID }$为 $\red{3 }$的奶牛，阵容减少到 $\red{2}$、$\red{7}$、$\red{7}$、$\red{7}$、$\red{7}$、$\red{5}$、$\red{7}$。在这个新阵容中，有一个连续的 $\red{4 }$头奶牛具有相 同的品种 $\red{ID (7)}$。
