## 题目描述

在完成了古越州圆盘密码的研究之后，考古学家小布来到了南美大陆的西部。相传很久以前在这片土地上生活着两个部落，一个部落崇拜闪电，另一个部落崇拜高山，他们分别用闪电和山峰的形状作为各自部落的图 腾。小布的团队在山洞里发现了一幅巨大的壁画，壁画上被标记出了$\red{N}$个点，经测量发现这$\red{N}$个点的水平位置和竖直位置是两两不同的。小布认为这幅壁画所包含的信息仅与这$\red{N}$个点的相对位置有关，因此不妨设坐标分别为$\red{(1, y1) , (2, y2), ..., (n, yn)，}$其中$\red{y1}$~$\red{yn}$是$\red{1}$~$\red{N}$的一个排列。小布的团队打算研究在这幅壁画中包含着多少个图腾，其中闪电图腾的定义图示如下（图腾的形式只与$\red{4}$个纵坐标值的相对大小排列顺序有关）：
![img](./1994/file/1.png)
小布的团队希望知道，这$\red{N}$个点中闪电图腾数目，由于该值可能较大，本题中只需输出该值对$\red{16777216}$的余数。

## 输入格式

第一行包含一个整数$\red{N，}$为点的数目。接下来一行包含$\red{N}$个整数，分别为$\red{y1, y2, }$…, $\red{yn}$。保证$\red{y1, y2, }$…, $\red{yn}$是$\red{1}$~$\red{N}$的一个排列。

## 输出格式

仅包含一个数，表示闪电图腾数目对$\red{16777216}$的余数。

## 样例

#### 输入样例1

```c
5
1 5 3 2 4
```

#### 输出样例1

```c
0
```

## 样例

#### 输入样例2

```c
4
1 2 4 3
```

#### 输出样例2

```c
1
```

$\red{1.n<=600}$
