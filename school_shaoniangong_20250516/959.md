## 说明

问题描述

利用字母可以组成一些美丽的图形，下面给出了一个例子：

<pre>
A B C D E F G

B A B C D E F

C B A B C D E

D C B A B C D

E D C B A B C

</pre>

这是一个$\red 5$行$\red 7$列的图形，请找出这个图形的规律，并输出一个$\red n$行$\red m$列的图形。

## 输入格式

输入一行，包含两个整数$\red n$和$\red m$，分别表示你要输出的图形的行数的列数。

## 输出格式

输出$\red n$行，每个$\red m$个字符，为你的图形。

## 样例

#### 样例输入

```C
5 7
```

#### 输出样例

```C
ABCDEFG
BABCDEF
CBABCDE
DCBABCD
EDCBABC
```

## 提示

$\red{1 < = n, m < = 26 }$。
