## 题目描述

在地图上散落着 $\red{n }$个车轮，小 $\red{J }$想用它们造一辆车。要求如下：

$\red{1. }$一辆车需要四个车轮，且四个车轮构成一个正方形

$\red{2. }$车轮不能移动
你需要计算有多少种造车的方案（两个方案不同当且仅当所用车轮不全相同，坐
标相同的两个车轮视为不同车轮）。

## 输入格式

第一行一个整数 $\red{n}$

接下来 $\red{n }$行，每行两个整数 $\red{x y，}$表示在$\red{(x,y)}$处有一个车轮

## 输出格式

一行一个整数，表示方案数

## 样例

#### 输入样例

```c
9
0 0
1 0
2 0
0 2
1 2
2 2
0 1
1 1
2 1
```

#### 输出样例

```c
6
```

## 提示

$\red{30\%}$的数据保证 $\red{n ≤}$ $\red{30}$
$\red{100\%}$的数据保证 $\red{1 ≤}$ $\red{n ≤}$ $\red{1000}$; $\red{|x|, |y| < 20000}$
