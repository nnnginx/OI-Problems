## 题目描述

给出一个长度为 $\red{n}$ 的序列 $\red{a}$，选出其中连续且非空的一段使得这段和最大。

## 输入格式

第一行是一个整数，表示序列的长度$\red{ n}$。

第二行有 $\red{n}$ 个整数，第$\red{ i}$ 个整数表示序列的第$\red{ i }$个数字 $\red{a_i}$。

## 输出格式

输出一行一个整数表示答案。

## 样例

#### 输入样例

```c
7
2 -4 3 -1 2 -4 3
```

#### 输出样例

```c
4
```

## 提示

对于100%的数据满足$\red{1 <= n <= 2 * 10^5}$,$\red{−10^4≤a_i≤10^4}$
