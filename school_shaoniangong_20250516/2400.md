## 题目描述

自从卡门在弹珠游戏中被贝茜彻底击败，他一直在想找机会复仇，这会儿，他邀贝茜去玩一个电脑游戏.

游戏中，贝茜在$\red{(BX,BY)(-1000≤}$$\red{BX,BY≤}$$\red{1000)}$处开始行动，这时时刻为$\red{0.}$她要试图逃离，她的速度为$\red{(BVX，}$$\red{BVY)(-100≤}$$\red{BVX，}$$\red{BVY≤}$$\red{100)}$每秒，也就是说$\red{t=1}$时她在$\red{(BX+ BVX BY+BVY,t=1.5}$时她在$\red{(BX+1.5×}$$\red{BVX, BY+1.5×}$$\red{BVY）}$

不幸的是，卡门为了复仇，放出$\red{N(1<N<50000)}$个杀手追击贝茜，在$\red{t=0}$时，杀手$\red{i}$的位置是$\red{(X_iY_i)(-1000≤}$$\red{X_iY_i≤}$$\red{1000)，}$他的速度是$\red{(VX_iVY_i)(-1000<VX_i}$，$\red{VY_i}$$\red{≤}$$\red{1000)}$每秒。

由于每个杀手配备了手枪，手枪的射程是$\red{R(1<R<2500)，}$也就是说贝茜要与这个杀手的距离保持超过$\red{R，}$否则有性命之虞。

然而，贝茜还有一件秘密武器，盾，但是，她不想过多地消耗盾的能量，所以，她想知道在逃脱过程中，某一个时刻她在最多为多少个杀手的射程内，当然这个时刻不一定是整数 。

为了防止出现精度误差，数据保证在$\red{R}$士$\red{0.0001}$时也能得出正确结果。

## 输入格式

第$\red{1}$行输入$\red{N，}$$\red{R，}$$\red{BX，}$$\red{BY, BVX，}$$\red{BVY，}$之后$\red{N}$行每行输入四个整数$\red{Xi，}$$\red{Yi，}$$\red{VXi，}$$\red{VYi}$．

## 输出格式

```
一个整数，表示在逃脱过程中，某一个时刻最多有这个数理的杀手可以射杀贝茜．
```

## 样例

#### 输入样例

```c
3 1 0 0 0 2
0 -3 0 4
1 2 -1 1
1 -2 2 -1
```

#### 输出样例

```c
2
```

## 提示

输出详细信息：在时间$\red{1.5}$时，贝西处于$\red{（0，}$$\red{3）}$点，三名撞伤者在点（$\red{0，}$$\red{3）}$，$\red{（-0.5，}$$\red{3.5）}$和$\red{（4，}$$\red{-3.5）}$处。

前两头牛瘀伤者距离贝西不到$\red{1}$个单位，而第三个永远不会在贝西的$\red{1}$个单位内，因此$\red{2}$是最可行的。
