## 题目描述

某侦察队长接到了一项紧急任务，要他在代号为$\red{A、B、C、D、E、F}$的六个队员中选若干队员去执行一项紧急任务，人员的配备必须按以下要求执行：

`（1）`$\red{A、B}$两人至少去一人；

`（2）`$\red{A、D}$不能一起去；

`（3）`$\red{A、E、F}$三人中要去两人；

`（4）`$\red{B、C}$两人都去或都不去；

`（5）`$\red{C、D}$两人中去一人；

`（6）`若$\red{D}$不去，则$\red{E}$也不去。

请编程确定谁去。

## 输入格式

无。

## 输出格式

哪些队员去（字符间有一空格，格式见样例）。

## 样例

#### 输出样例

```c
A B C F
```
