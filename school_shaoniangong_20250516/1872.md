## 题目描述

琦琦出了一道益智游戏，给定$\red{n（}$$\red{1<=n<=17）}$个数的排列$\red{p_i}$。例如$\red{n=5}$时，$\red{\{1,4,2,5,3\}}$是其 中一个排列。
对 $\red{p }$相邻元素相加得到下一行的$\red{n-1}$个元素。
重复直到仅剩下一个元素 $\red{s}$。
例如

$\red{1 ~4 ~2 ~5 ~3}$

$\red{5 ~6 ~7 ~8}$

$\red{11 ~13 ~15}$

$\red{24 ~28}$

$\red{52}$

$\red{52}$即是最后一个元素 $\red{s}$。
现在给出 $\red{s ，}$请逆推导出排列 $\red{p，}$如有多个解，输出最小字典序的解。
数据保证有解。

## 输入格式

第一行，两个整数，分别是 $\red{n，}$$\red{s}$。

## 输出格式

共一行，第一行，$\red{n}$个整数，表示 $\red{p_i}$。

## 样例

#### 输入样例

```c
5 52
```

#### 输出样例

```c
1 4 2 5 3
```

## 提示

$\red{20\% n<=8}$

$\red{30\% n<=10}$

$\red{70\% n<=15}$

$\red{100\% n<=17}$
