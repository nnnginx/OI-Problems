## 题目描述

给定一棵$\red {N}$个节点的树，要求增加若干条边，把这棵树扩充为完全图，并满足图的唯一最小生成树仍然是这棵树。

求增加的边的权值总和最小是多少。

## 输入格式

第一行包含整数$\red {t}$，表示共有$\red {t}$组测试数据。

对于每组测试数据，第一行包含整数$\red {N}$。

接下来$\red {N-1}$行，每行三个整数$\red {X,Y,Z}$，表示$\red {X}$节点与$\red {Y}$节点之间存在一条边，长度为$\red {Z}$。

## 输出格式

每组数据输出一个整数，表示权值总和最小值。

每个结果占一行。

## 样例

#### 输入样例

```c
2
3
1 2 2
1 3 3
4
1 2 3
2 3 4
3 4 5
```

#### 输出样例

```c
4
17
```

## 提示

注意： 树中的所有边权均为整数，且新加的所有边权也必须为整数。

$\red {1≤N≤6000}$

$\red {1≤Z≤100}$
