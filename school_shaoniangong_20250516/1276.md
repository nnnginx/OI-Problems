## 题目描述

`B市`和`T市`之间有一条长长的高速公路，这条公路的某些地方设有路标，但是大家都感觉路标设得太少了，相邻两个路标之间往往隔着相当长的一段距离。

为了便于研究这个问题，我们把公路上相邻路标的最大距离定义为该公路的“`空旷指数`”。

现在政府决定在公路上增设一些路标，使得公路的“空旷指数”最小。他们请求你设计一个程序计算能达到的最小值是多少。

请注意，公路的起点和终点保证已设有路标，公路的长度为整数，并且原有路标和新设路标都必须距起点整数个单位距离。

## 输入格式

`第1行`包括三个数$\red L$、$\red N$、$\red K$，分别表示公路的长度，原有路标的数量，以及最多可增设的路标数量。

`第2行`包括$\red N$个整数，分别表示原有的$\red N$个路标的位置。路标的位置用距起点的距离表示，且一定位于区间$\red {[0,L]}$内。

## 输出格式

输出`1行`，包含一个整数，表示增设路标后能达到的最小“`空旷指数`”值。

## 样例

#### 输入样例

```C
101 2 1
0 101
```

#### 输出样例

```C
51

```
## 提示
$\red{50\% 的数据中，2≤N≤100，0≤K≤100}$。

$\red{100\% 的数据中，2≤N≤100000, 0≤K≤100000。}$

$\red{100\% 的数据中，0<L≤10000000。
}$
