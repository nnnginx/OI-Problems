## 题目描述

农夫约翰的奶牛在公路上旅行！他们车上的里程表显示一个整数英里值，从行程开始时的$\red{X（}$$\red{100<=X<=10^{16}）}$英里开始，到行程结束时的$\red{Y（}$$\red{X<=Y<=10^{16}）}$英里结束。

每当里程表显示一个"有趣"的数字时（包括在旅行开始和结束时），奶牛就会哞哞叫。一个数字是"有趣的"，如果你看它的所有数字，除了前导零，所有这些都是相同的，除了一个不同的单个数字。

例如，数字$\red{33323}$和$\red{110}$是有趣的，而数字$\red{9779}$和$\red{55555}$不是。帮助$\red{FJ}$计算奶牛在旅途中会鸣叫多少次 。

对于一半的测试用例，约束$\red{X<=Y<=10^{16}}$将保持不变。

请注意，要存储$\red{10^{16}}$的数字，您需要一个$\red{64}$位整数数 据类型，例如$\red{C/C++}$中的"$\red{long-long}$"。

## 输入格式

第$\red{1}$行：第一行包含两个整数，$\red{X}$和$\red{Y，}$分开按空格。

## 输出格式

第$\red{1}$行：一个整数，包含奶牛鸣叫的次数在旅途中。

## 样例

#### 输入样例

```c
110
133
```

#### 输出样例

```c
13
```

## 提示

输入详细信息：
行程从$\red{110}$开始，到$\red{133}$结束。

输出详细信息：
当里程表读数为$\red{110}$、$\red{112}$、$\red{113}$、$\red{114}$、$\red{115}$、$\red{116}$时，奶牛发出哞哞声，$\red{117}$、$\red{118}$、$\red{119}$、$\red{121}$、$\red{122}$、$\red{131}$和$\red{133}$。
