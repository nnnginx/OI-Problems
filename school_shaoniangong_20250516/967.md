## 题目描述

农夫约翰在去年赚了一大笔钱！

他想要把这些钱用于投资，并对自己能得到多少收益感到好奇。

已知投资的复合年利率为$\red{R}$（$\red{0 \le R \le 20 }$)。

约翰现有总值为$\red{M}$的钱（$\red{100 \le M \le 1,000,000}$)。

他清楚地知道自己要投资$\red{Y}$年（$\red{0 \le Y \le 400}$）。

请帮助他计算最终他会有多少钱，并输出它的整数部分。

## 输入格式

一行包含三个整数$\red{R}$，$\red{M}$，$\red{Y}$，相邻两个整数之间用单个空格隔开。

## 输出格式

一个整数，即约翰最终拥有多少钱（整数部分）。

## 样例

#### 输入数据

```C
5 5000 4
```

#### 输出数据

```C
6077
```

## 提示

在样例中，

第一年后: $\red{1.05 * 5000 = 5250}$

第二年后: $\red{1.05 * 5250 = 5512.5}$

第三年后: $\red{1.05 * 5512.50 = 5788.125}$

第四年后: $\red{1.05 * 5788.125 = 6077.53125}$

$\red{6077.53125}$的整数部分为$\red{6077}$。
