## 题目描述

给出两幅相同大小的黑白图像（用$\red{01}$矩阵）表示，求它们的相似度。

说明：若两幅图像在相同位置上的像素点颜色相同，则称它们在该位置具有相同的像素点。

两幅图像的相似度定义为相同像素点数占总像素点数的百分比。

## 输入格式

第`一行`包含两个整数$\red{m}$和$\red{n}$，表示图像的行数和列数，中间用单个空格隔开。$\red{1 <= m <= 100, 1 <= n <= 100}$。

之后`m行`，每行$\red{n}$个整数$\red{0}$或$\red{1}$，表示第一幅黑白图像上各像素点的颜色。

相邻两个数之间用单个空格隔开。

之后`m行`，每行$\red{n}$个整数$\red{0}$或$\red{1}$，表示第二幅黑白图像上各像素点的颜色。

相邻两个数之间用单个空格隔开。

## 输出格式

一个实数，表示相似度（以百分比的形式给出），精确到小数点后两位。

## 样例

#### 输入样例

```C
3 3
1 0 1
0 0 1
1 1 0
1 1 0
0 0 1
0 0 1
```

#### 输出样例

```C
44.44
```
