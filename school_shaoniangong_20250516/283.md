## 题目描述

给定一个$\red {N}$行$\red {M}$列的棋盘，已知某些格子禁止放置。

问棋盘上最多能放多少个不能互相攻击的。

放在格子里，攻击范围与中国象棋的“”一致。

## 输入格式

第一行包含三个整数$\red {N,M,T}$，其中$\red {T}$表示禁止放置的格子的数量。

接下来$\red {T}$行每行包含两个整数$\red {x}$和$\red {y}$，表示位于第$\red {x}$行第$\red {y}$列的格子禁止放置，行列数从$\red {1}$开始。

## 输出格式

输出一个整数，表示结果。

## 样例

#### 输入样例

```c
8 8 0
```

#### 输出样例

```c
8
```

## 提示

$\red {1≤N,M≤200}$
