## 题目描述

麻雀帕西和青蛙弗洛格是好玩伴，它们经常一起比赛唱歌。但冬天来了，青蛙弗洛格冬眠了，它的睡眠深度是$\red D$。麻雀帕西觉得好无聊，于是它想办法要唤醒弗洛格。

麻雀帕西只会唱$\red N$首歌，第i首歌的音量是$\red {S_i}$。每听完一首歌，青蛙弗洛格的睡眠深度就会减少，减少的值等于它听到的歌的音量。当青蛙弗洛格的睡眠深度大于$\red 0$的时候，它会继续冬眠，当睡眠深度小于或者等于$\red 0$时，它就会被唤醒了。

麻雀帕西会从第$\red 1$首歌开始唱，唱完第$\red 1$首歌后如果弗洛格还没醒就接着唱第$\red 2$首歌，如果唱完第$\red 2$首歌弗洛格还没醒就接着唱第$\red 3$首歌，依次类推，如果唱完第$\red N$首歌后弗洛格还没醒，那么麻雀帕西又重新从第$\red 1$首歌开始唱，就像循环播放音乐一样，一直到青蛙弗洛格被唤醒为止，那么麻雀帕西总共唱了多少首歌？

## 输入格式

第一行，两个整数：$\red D$和$\red N$。$\red{1<=D<=2 * 10^{9}}$,$\red{1<=N<=1000000}$

第二行，$\red N$个整数，空格分开，第i个整数就是第i首歌的音量$\red {S_i}$.

## 输出格式

一个整数，麻雀帕西总共唱了多少首歌后，弗洛格会被唤醒?

## 样例

#### 输入样例1

```C
13 3
5 2 4
```

#### 输出样例1

```C
4
```

#### 输入样例2

```C
3 3 
5 2 4
```

#### 输出样例2

```C
1
```

#### 输入样例3

```C
21 3 
2 1 3
```

#### 输出样例3

```C
11
```
