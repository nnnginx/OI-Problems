## 题目描述

原题来自：Romania OI 2002

求$\red{ A^B}$ 的所有约数之和$\red{ \bmod\ 9901}$。

## 输入格式

输入两个整数$\red{ A,B}$。

## 输出格式

输出答案$\red{ \bmod 9901}$。

## 样例

#### 输入样例

```c
2 3
```

#### 输出样例

```c
15
```

$\red{2^3=8 }$的所有约数为$\red{ 1,2,4,8}$，$\red{1+2+4+8=15}$,$\red{15\bmod 9901=15}$，因此输出 $\red{15}$。

## 数据范围与提示

对于全部数据，$\red{0\le A}$,$\red{B\le 5\times 10^7}$。
