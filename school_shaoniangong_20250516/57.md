## 题目描述

给定$\red m$个序列，每个包含$\red n$个非负整数。

现在我们可以从每个序列中选择一个数字以形成具有$\red m$个整数的序列。

很明显，我们一共可以得到$\red {n^m}$个这种序列， 然后我们可以计算每个序列中的数字之和，并得到$\red {n^m}$个值。

现在请你求出这些序列和之中最小的$\red n$个值。

## 输入格式

第一行输入一个整数$\red T$，代表输入中包含测试用例的数量。

接下来输入$\red T$组测试用例。

对于每组测试用例，第一行输入两个整数$\red m$和$\red n$。

接下在$\red m$行输入$\red m$个整数序列，数列中的整数均不超过$\red {10000}$。

## 输出格式

对于每组测试用例，均以递增顺序输出最小的$\red n$个序列和，数值之间用空格隔开。

每组输出占一行。

## 样例

#### 输入样例

```C
1
2 3
1 2 3
2 2 3
```

#### 输出样例

```C
3 3 4
```

## 提示

$\red {0<m≤1000}$,
$\red {0<n≤2000}$
