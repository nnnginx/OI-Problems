### 题目描述

小 P 考完 NOIP 后，在 OIerDb 上看了看他们学校的成绩单，突然发觉大神 hjh NOIP 拿了 $400pts$！于是他想要拜谢 hjh。

他了解到，hjh 喜欢吃苹果，于是他想要送给 hjh 一些苹果。然而 hjh 吃不了这么多苹果，于是他想要小 P 把这些苹果分成 $k$ 堆送给机房的其他人。

出于强迫症，hjh 喜欢在分物品的时候根据物品的大小 $a$ 来分组，并且他要求有 $p$ 堆中物品的大小和为偶数，剩下的 $k-p$ 堆中物品的大小和为奇数。

hjh 希望小 P 也这么分苹果，但是小 P 太m了，因此他找你来帮忙确定一个分苹果的方案。

### 输入格式

第一行输入三个整数 $n,k,p$。

第二行 $n$ 个整数 $a_i$ 表示每个苹果的大小。

### 输出格式

如果没法做到像 hjh 要求的那样分苹果，输出一行 `NO`，如果可以做到则输出一行 `YES`。

然后输出 $k$ 行，每行表示一堆物品。对于每一堆，第一个数字表示这一堆中苹果的个数 $c$，然后输出 $c$ 个数表示这一堆中每个苹果的大小。每堆至少有一个苹果。

### 样例 #1

#### 样例输入 #1

```
5 3 1
21 6 13 17 15
```

#### 样例输出 #1

```
YES
1 6
3 15 21 13
1 17
```

### 样例 #2

#### 样例输入 #2

```
5 5 3
7 14 2 9 5
```

#### 样例输出 #2

```
NO
```

### 提示

对于所有数据，$1\le k\le n \le10^6,0\le p\le k,1\le a_i\le 10^9$。

因为并不知道暴力怎么做因此随便分的部分分。

| 数据组数 | $n$ |  特殊情况 |
| :----------: | :----------: | :----------: |
| $1$ | $\le5$ |  |
| $2$ | $\le10^3$ | $\text{A}$ |
| $3$ | $\le10^3$ | $\text{B}$ |
| $4\sim6$ | $\le10^5$ |  |
| $7$ | $\le10^6$ | $\text{A}$ |
| $8\sim10$ | $\le10^6$ |  |

特殊性质 $\text{A}$：$p=0$。

特殊性质 $\text{B}$：$k=1$。
