## 题目描述

给出一个小于$\red{2^{32}}$的正整数。

这个数可以用一个$\red{32}$位的二进制数表示（不足$\red{32}$位用$\red{0}$补足）。

我们称这个二进制数的前$\red{16}$位为“`高位`”，后$\red{16}$位为“`低位`”。

将它的高低位交换，我们可以得到一个新的数。

试问这个新的数是多少（用十进制表示）。

例如，数`1314520`用二进制表示为`0000 0000 0001 0100 0000 1110 1101 1000`,（添加了$\red{11}$个前导$\red{0}$补足为$\red{32}$位），

其中前$\red{16}$位为`高位`，即`0000 0000 0001 0100`；

后$\red{16}$位为`低位`，即`0000 1110 1101 1000`。

将它的高低位进行交换，我们得到了一个新的二进制数`0000 1110 1101 1000 0000 0000 0001 0100`。

它即是十进制的`249036820`。

## 输入格式

只有一行，为一个小于$\red{2^{32} }$的正整数。

## 输出格式

只有一行为交换后的数

## 样例

#### 输入样例

```C
1314520
```

#### 输出样例

```C
249036820
```
