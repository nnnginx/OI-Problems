## 题目描述

考古学家发现了一座千年古墓，墓碑上有神秘的字符。经过仔细研究，发现原来这是开启古墓入口的方法。

墓碑上有`2行`字符串，其中第一个串的长度为偶数，现在要求把第$\red{2}$个串插入到第一个串的正中央，如此便能开启墓碑进入墓中。

## 输入格式

首先给出一个整数$\red{n}$，表示测试数据的组数。

然后是$\red{n}$组数据，每组数据`2行`，

每行一个字符串，长度大于$\red{0}$，小于$\red{50}$，并且第一个串的长度必为偶数。

## 输出格式

请为每组数据输出一个能开启古墓的字符串，每组输出占一行。

## 样例

#### 输入样例

```C
2
CSJI
BI
AB
CMCLU
```

#### 输出样例

```C
CSBIJI
ACMCLUB
```
