## 题目描述

把一个正整数$\red N$表示成如下表达式的一系列正整数的和，叫做整数$\red N$的一个划分。

某个正整数$\red N$的不同表达式的个数称为整数$\red N$的分划数。编程输出所有的划分。

## 输入格式

一个整数$\red N$。

## 输出格式

多行数据，前面多行输出划分情况，最后一行一个整数，即划分数。

## 样例

#### 输入样例

```C
4
```

#### 输出样例：

```C
5
```

## 提示

样例解释：

$\red{4=4}$;

$\red{4=3+1;}$

$\red{4=2+2;}$

$\red{4=2+1+1;}$

$\red{4=1+1+1+1}$
