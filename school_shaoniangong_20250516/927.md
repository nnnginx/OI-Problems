## 题目描述

输入一个自然数，现在请你判断是否是质数。如果是的话输出"`Yes`"否则输出"`No`"。
一个自然数是质数当且仅当其因子只有$\red{1}$和其本身。

## 输入格式

输入有多组数据，第一行为一个整数$\red{Q}$，表示有$\red{Q}$组数据。
接下来有$\red{Q}$行，每行一个整数，为待判断的数字。

## 输出格式

输出$\red{Q}$行，每行输出"`Yes`"或者"`No`"，表示是否是质数。注意大小写。

## 样例

#### 输入数据

```C
3
5
6
7
```

#### 输出数据

```C
Yes
No
Yes
```

## 提示

对于$\red{80\%}$的数据，$\red{0 <= }$ `需要判断的数` $\red{<= 10^5}$

对于$\red{100\%}$的数据，$\red{1 <= Q <= 10, 0 <= }$ `需要判断的数` $\red{<= 10^8}$
