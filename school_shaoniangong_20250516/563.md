## 题目描述

农民约翰的母牛总是产生最好的肋骨。 你能通过农民约翰和美国农业部标记在每根肋骨上的数字认出它们。

农民约翰确定他卖给买方的是真正的质数肋骨,是因为从右边开始切下肋骨,每次还剩下的肋骨上的数字都组成一个质数,

举例来说: $\red{7\ 3\ 3\ 1\ }$全部肋骨上的数字 $\red{7331}$是质数;三根肋骨 $\red{733}$是质数;二根肋骨 $\red{73}$ 是质数;当然,最后一根肋骨 $\red{7}$ 也是质数。 $\red{7331}$ 被叫做长度 $\red{4}$ 的特殊质数。

写一个程序对给定的肋骨的数目 $\red{N (1<=N<=8)}$,求出所有的特殊质数。 数字$\red{1}$不被看作一个质数。

## 输入格式

单独的一行包含$\red{N}$。

## 输出格式

按顺序输出长度为 $\red{N}$ 的特殊质数,每行一个。

## 样例

#### 输入样例

```c
4
```

#### 输出样例

```c
2333
2339
2393
2399
2939
3119
3137
3733
3739
3793
3797
5939
7193
7331
7333
7393
```

## 提示

对于 $\red{100\%}$ 的数据，$\red{1≤n≤8}$。
