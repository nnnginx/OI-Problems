## 题目描述

面面是个爱生气的孩子，并且面面有较强的强迫症，例如面面遇到字符串$\red a$的时候，如果该字符串字母的顺序没有按照字母序来排列，面面就会大发雷霆，聪明的你快来帮助面面解决这一个问题吧。

## 输入格式

共一行；
第一行包含字符串$\red a$(全部由大写字母组成),长度小于$\red {10^6}$；

## 输出格式

共一行；
按照字母序排好序的字符串。

## 样例

#### 输入样例

```C
DABAC
```

#### 输出样例

```C
AABCD
```

## 提示

对于所有的数据满足：
字符串的长度小于等于$\red {10^6}$
