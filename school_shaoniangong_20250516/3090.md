# Background

$\mathbb{S}\mathbb{A}\mathbb{B}\mathbb{R}\mathbb{I}\mathbb{N}\mathbb{A}$ 终于抵达了 $\tt Deducción\,Fiscal$ 王国。

$\tt Deducción\,\,Fiscal$ 王国的街头茶会上有很多甜点。

# Description

由于 $\tt Deducción\,\,Fiscal$ 王国除了热衷于 $\tt Deducción\,\,Fiscal$ 还喜欢魔方，所以他们用魔方来代替货币！真是优雅的文明(?)

$\tt Deducción\,\,Fiscal$ 王国上由于物资匮乏，王国上只有有 $N$ 种不同的魔方，第 $i$ 种甜点的价值是 $a_i$ 。大富翁 $\mathbb{S}\mathbb{A}\mathbb{B}\mathbb{R}\mathbb{I}\mathbb{N}\mathbb{A}$ 手上每个款式的魔方都有无数个。

因为该王国上的人 (?) 有 $K$ 个手指，所以他们使用的是 $K$ 进制！此外，$\tt Deducción\,\,Fiscal$ 王国上的人都有着崇高的信仰，他们认为数字 $P$ (在 $K$ 进制下) 是 $\mathbb{H}\mathbb{O}\mathbb{L}\mathbb{Y}$ 的！所以，如果凑出来的魔方的价值在 $K$ 进制中最后一位数字是 $P$ , $\tt Deducción\,\,Fiscal$ 王国的人会很愉快！（愉快的话会免费送你甜点）但不幸的是，$\mathbb{S}\mathbb{A}\mathbb{B}\mathbb{R}\mathbb{I}\mathbb{N}\mathbb{A}$ 忘了 $P$ 是多少！所以，$\mathbb{S}\mathbb{A}\mathbb{B}\mathbb{R}\mathbb{I}\mathbb{N}\mathbb{A}$ 想知道他能凑出的所有可能让$\tt Deducción\,\,Fiscal$ 王国的人愉快 (然后送她甜点) 的 $P$ 的值。

# Format

## Input

第一行，输入两个整数 $ n $ 和 $ k $ ( $ 1 \le n \le 100,000 $ , $ 2 \le k \le 100,000 $ )

第二行输入 $ n $ 个整数数 $ a_1, a_2, \ldots, a_n $ ( $ 1 \le a_i \le 10^9 $ )

## Output

第一行，输出 $\mathbb{S}\mathbb{A}\mathbb{B}\mathbb{R}\mathbb{I}\mathbb{N}\mathbb{A}$ 可以让 $\tt Deducción\,Fiscal$ 王国人愉快的 $P$ 值的数量。

第二行，以递增的顺序输出所有这些值。

## 样例 #1

### 样例输入 #1

```
2 8
12 20
```

### 样例输出 #1

```
2
0 4
```
