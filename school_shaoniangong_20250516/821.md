## 题目描述

对于多项式$\red{f(x)=ax ^3+bx ^2+cx+d}$和给定的$\red{a,b,c,d,x}$，

计算$\red{f(x)}$的值，保留到小数点后$\red{7}$位。

## 输入格式

输入仅一行，包含$\red{5}$个实数，分别是$\red{x}$，及参数$\red{a、b、c、d}$的值。

每个数都是绝对值不超过$\red{100}$的双精度浮点数。

数与数之间以一个空格分开。

## 输出格式

输出一个实数，即$\red{f(x)}$的值，保留到小数点后$\red{7}$位。

## 样例

#### 输入样例

```C
2.31 1.2 2 2 3
```

#### 输出样例

```C
33.0838692
```
