## 题目描述

有$\red{N}$头奶牛，每头那牛都有一个标号$\red{Pi，}$$\red{1 <= Pi <= M <= N <= 40000}$。现在$\red{Farmer John}$要把这些奶牛分成若干段，定义每段的不河蟹度为：



 若这段里有$\red{k}$个不同的数，那不河蟹度为$\red{k×}$$\red{k}$。那总的不河蟹度就是所有段的不河蟹度的总和。

## 输入格式

第一行：两个整数$\red{N，}$$\red{M}$

第$\red{2..N+1}$行：$\red{N}$个整数代表每个奶牛的编号

## 输出格式

一个整数，代表最小不河蟹度
