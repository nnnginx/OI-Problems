## 题目描述

由于无敌的凡凡在$\red {2005}$年世界英俊帅气男总决选中胜出，`Yali Company`总经理`Mr.Z`心情好，决定给每位员工发奖金。公司决定以每个人本年在公司的贡献为标准来计算他们得到奖金的多少。于是`Mr.z`下令召开$\red {m}$方会谈。每位参加会谈的代表提出了自己的意见：“我认为员工$\red {a}$的奖金应该比$\red {b}$高！”`Mr.Z`决定要找出一种奖金方案，满足各位代表的意见，且同时使得总奖金数最少。每位员工奖金最少$\red {100}$元。

## 输入格式

第一行两个整数$\red {n,m}$，表示员工总数和代表数；以下$\red {m}$行，每行$\red {2}$个整数$\red {a,b}$，表示某个代表认为第$\red {a}$号员工奖金应该比第$\red {b}$号员工高。

## 输出格式

若无法找到合法方案，则输出`“Poor Xed”`；否则输出一个数表示最少总奖金。

## 样例

#### 输入样例

```c
2 1

1 2
```

#### 输出样例

```c
201
```

## 提示

$\red {80％}$的数据满足$\red {n<=1000，m<=2000；100％}$的数据满足$\red {n<=10000，m<=20000}$。
