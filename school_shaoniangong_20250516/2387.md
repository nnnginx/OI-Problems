## 题目描述

$\red{Farmer John}$尝试通过和奶牛们玩益智玩具来保持他的奶牛们思维敏捷. 其中一个大型玩具是牛栏中的灯.

$\red{N (2 <= N <= 100,000) }$头奶牛中的每一头被连续的编号为$\red{1..N, }$站在一个彩色的灯下面.刚到傍晚的时候, 所有的灯都是关闭的.

奶牛们通过$\red{N}$个按钮来控制灯的开关; 按第$\red{i}$个按钮可以改变第$\red{i}$个灯的状态.奶牛们执行$\red{M (1 <= M <= 100,000)}$条指令, 每个指令都是两个整数中的一个$\red{(0 <= }$指令号 $\red{<= 1).}$

第$\red{1}$种指令$\red{(}$用$\red{0}$表示$\red{)}$包含两个数字$\red{S_i}$和$\red{E_i (1 <= S_i <= E_i <= N), }$它们表示起始开关和终止开关. 奶牛们只需要把从$\red{S_i}$到$\red{E_i}$之间的按钮都按一次, 就可以完成这个指令. 第$\red{2}$种指令$\red{(}$用$\red{1}$表示$\red{)}$同样包含两个数字$\red{S_i}$和$\red{E_i (1 <= S_i <= E_i <= N), }$不过这种指令是询问从$\red{S_i}$到$\red{E_i}$之间的灯有多少是亮着的.

帮助$\red{FJ}$确保他的奶牛们可以得到正确的答案.

## 输入格式

第 $\red{1 }$行: 用空格隔开的两个整数$\red{N}$和$\red{M}$

第 $\red{2..M+1 }$行: 每行表示一个操作, 有三个用空格分开的整数: 指令号, $\red{S_i }$和 $\red{E_i}$

## 输出格式

第 $\red{1..}$询问的次数 行: 对于每一次询问, 输出询问的结果.

## 样例

#### 输入样例

```c
4 5
0 1 2
0 2 4
1 2 3
0 2 4
1 1 4
```

#### 输出样例

```c
1
2
```
