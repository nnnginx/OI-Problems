## 题目描述

世博会志愿者的选拔工作正在 $\red{A}$市如火如荼的进行。为了选拔最合适的人才，$\red{A}$市对所有报名的选手进行了笔试，笔试分数达到面试分数线的选手方可进入面试。面试分数线根据计划录取人数的 $\red{150\%}$ 划定，即如果计划录取$\red{m}$名志愿者，则面试分数线为排名第$\red{m}$×$\red{150\%}$（向下取整）名的选手的分数，而最终进入面试的选手为笔试成绩不低于面试分数线的所有选手。

现在就请你编写程序划定面试分数线，并输出所有进入面试的选手的报名号和笔试成绩。

## 输入格式

第一行，两个整数$\red{n，m（5 ≤ n ≤ 5000，3 ≤ m ≤ n）}$，中间用一个空格隔开，其中$\red{n}$ 表示报名参加笔试的选手总数，$\red{m}$ 表示计划录取的志愿者人数。输入数据保证$\red{m×}$$\red{150\%}$向下取整后小于等于$\red{n}$。
第二行到第$\red{ n+1 }$行，每行包括两个整数，中间用一个空格隔开，分别是选手的报名号$\red{k（1000 ≤ k ≤ 9999）}$和该选手的笔试成绩$\red{s（1 ≤ s ≤ 100）}$。数据保证选手的报名号各不相同。

## 输出格式

第一行，有两个整数，用一个空格隔开，第一个整数表示面试分数线；第二个整数为进入面试的选手的实际人数。

从第二行开始，每行包含两个整数，中间用一个空格隔开，分别表示进入面试的选手的报名号和笔试成绩，按照笔试成绩从高到低输出，如果成绩相同，则按报名号由小到大的顺序输出。

## 样例

## 输入样例

```C
6 3
1000 90
3239 88
2390 95
7231 84
1005 95
1001 88
```

## 输出样例

```C
88 5
1005 95
2390 95
1000 90
1001 88
3239 88
```

## 样例说明

$\red{m×}$ $\red{150\%}$ = $\red{3}$× $\red{150\%}$ $=\red{4.5}$，向下取整后为$\red{4}$。保证$\red{4}$ 个人进入面试的分数线为$\red{88}$，但因为$\red{88}$有重分，所以所有成绩大于等于$\red{88}$ 的选手都可以进入面试，故最终有$\red{5}$ 个人进入面试。
