# Background

小明来参加ACM比赛了。ACM的赛制是这样的：总共有若干道题目，每题答案可以随时提交评测，如果评测通过记录通过此题的时间，如果评测未通过则会记录成该题的错误次数。当通过该题目的时候，该题的每次错误次数会在该题通过时间上罚时20分钟。比赛过程中还可以随时查看目前的排行榜单，现在比赛快结束了，小明查看了这次比赛N个队伍的成绩，比赛共M题，他想知道每队完成的题目共计算了多少时间。

# Format

## Input

先是一行，包括两个整数N M。

接下来N行，每队一行，先是该队通过题数K，然后是该队总共有效的罚时次数P，后面跟K个形如 h: m : s的用时，每个时间表示该队通过的一题用时h小时m分钟s秒。每个用时的部分保证不包含前导0用单个空格分隔。

## Output

输出N行，每队一行，为该队用时，按h:m:s的格式输出，每个数字不包含前导0，且m和s不应大于59。

# Samples

```input1
2 3
1 5 0:38:56
3 0 0:1:2 1:2:3 2:3:4
```

```output1
2:18:56
3:6:9
```
