

# Description

有三根柱 A、B、C，在 A 柱上有 n 块盘片，所有盘片都是大片在下面，小片放在大片上面。并依次编好序号。现要将 A 上的 n 块盘片移到 C 柱上，每次只能移动一片，而且在同一根柱子上必须保持上面的盘片比下面的盘片小，请输出移动方法。

# Format

## Input

仅一个整数 n（n ≤ 20），表示 A 柱上的盘片数。.

## Output

输出盘片的移动步骤。

# Samples

```input1
3
```

```output1
A-1-C

A-2-B

C-1-B

A-3-C

B-1-A

B-2-C

A-1-C
```
