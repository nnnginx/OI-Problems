# Background
huhe的农场种植了n块试验田，每块田的作物产量各不相同。为了提高农场整体收益，梁老师建议huhe选择一块连续的试验田区域进行土壤改良。经过分析发现：

改良后的连续至少k块田地，其平均产量会有显著提升

huhe希望找到平均产量最高的连续不少于k块的田地

为了精确评估，需要计算到小数点后三位

# Description
给定n块田地的产量数据和一个整数k，请找出所有长度至少为k的连续田地段的最高平均产量（保留三位小数，输出时去掉小数点，如12.345输出12345）。

# Format

## Input
第一行两个整数n和k（1 ≤ k ≤ n ≤ 100000）
第二行n个整数，表示每块田地的产量（0 ≤ 产量 ≤ 2000）

## Output
一个整数，表示最高平均产量×1000后的整数部分

# Samples

```input1
5 3  
10 20 30 25 15
```

```output1
25000
```

# Example Explanation


10,20,30 → 平均(10+20+30)/3=20.000 → 20000

20,30,25 → 平均25.000 → 25000

30,25,15 → 平均23.333 → 23333
