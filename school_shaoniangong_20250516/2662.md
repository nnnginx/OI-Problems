## 题目描述

有一个未知的序列$\red{x，}$长度为$\red{n}$。它的$\red{K-}$划分序列$\red{y}$指的是每连续$\red{K}$个数的和得到划分序列，$\red{y[1]=x[1]+x[2]+....+x[K]，}$$\red{y[2]=x[K+1]+x[K+2]+....+x[K+K]....}$。

若$\red{n}$不被$\red{K}$整除，则$\red{y[n/K+1]}$可以由少于$\red{K}$个数加起来。比如$\red{n=13，}$$\red{K=5，}$则$\red{y[1]=x[1]+...+x[5]，}$$\red{y[2]=x[6]+....+x[10]，}$$\red{y[3]=x[11]+x[12]+x[13]}$。

若小$\red{A}$只确定$\red{x}$的$\red{K[1]}$划分序列以及$\red{K[2]}$划分序列....$\red{K[M]}$划分序列的值情况下，问她可以确定$\red{x}$多少个元素的值。

## 输入格式

第一行输入两个正整数$\red{n，}$$\red{M}$。

第二行输入$\red{M}$个正整数表示$\red{K[1],K[2].....K[M]}$。

## 输出格式

输出$\red{1}$个整数，表示能确定的元素

## 样例

#### 输入样例1

```c
3 1

2
```

#### 输出样例1

```c
1
```

#### 输入样例2

```c
6 2

2 3
```

#### 输出样例2

```c
2
```

#### 输入样例3

```c
123456789 3

5 6 9
```

#### 输出样例3

```c
10973937
```

## 提示

数据范围

对于$\red{20\%}$的数据，$\red{3 <= N <= 2000,M<=3}$。

对于$\red{40\%}$的数据，$\red{3 <= N <= 5\times 10^6}$。

对于$\red{100\%}$的数据，$\red{3 <= N <= 10^9 , 1 <= M <= 10，}$$\red{2 <= K[i] < N}$。
