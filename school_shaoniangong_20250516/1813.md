## 题目描述

将一个给定的矩形划分为一个个正方形，其规则是先旧能多地从矩形中划分一块正方形，接下来，在剩下的矩形中旧能多的划分一块正方形……，例如，图中所示是一个$\red{3}$$\red{×4}$的矩阵，可最少划分为$\red{4}$个正方形。

![img](./1813/file/11.png)

也就是说，取走一个$\red{3}$$\red{×3}$的正方形后，将问题规模变成$\red{3}$$\red{×1，}$然后变成$\red{2}$$\red{×1，}$最后变成$\red{1}$$\red{×1}$。规模每缩小一次，正方形的个数加$\red{1}$。试计算能划分的正方形的个数。

## 输入格式

输入为两个整数，即长和宽。

## 输出格式

输出为一个整数，即正方形个数。

## 样例

#### 输入样例

```c
3　4
```

#### 输出样例

```c
4
```

## 提示

对于$\red{20\%}$的数据 $\red{1 <= n <= m <= 100}$

对于$\red{40\%}$的数据 $\red{1 <= n <= m <= 1000}$

对于$\red{60\%}$的数据 $\red{1 <= n <= m <= 100000}$

对于$\red{100\%}$的数据 $\red{1 <= n <= m <= 10000000}$
