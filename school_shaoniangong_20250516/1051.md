## 题目描述

把一张矩形的纸按平行于边的方向剪开，会得到很多矩形小纸片。如下图所示，最外围的矩形表示纸张，中间的线条表示切口。

以下图中的切割方法，竖直方向上被切为$\red 5$部分，水平方向上被切为$\red 3$部分。如果我们记竖直方向上每部分的宽度为$\red a$，竖直方向上每部分的高度为$\red b$的话，这个切割方案就可以唯一确定了。

![img](./1051/file/1.png)

你的任务是，给定一个切割方案，请编写程序计算一下：在此切割方案之下，最终得到的矩形小纸片有多少种（同样大小的记为一种）。

## 输入格式

第一行为两个整数$\red {m、n<=100}$，分别表示竖直方向和水平方向切割几部分。

第二行为$\red m$个整数，分别表示竖直方向上每个部分的宽度a，每个整数都不小于$\red 1$，不大于$\red{50}$。

第三行为$\red n$个整数，分别表示水平方向上每个部分的高度$\red b$，每个整数都不小于$\red 1$，不大于$\red {50}$。

## 输出格式

输出只有一个整数，表示得到的小纸片的种数。

## 样例

#### 输入样例

```C
3 3
1 2 3
1 3 4
```

#### 输出样例

```C
8
```
