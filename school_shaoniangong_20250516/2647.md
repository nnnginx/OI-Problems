## 题目描述

农夫约翰去年赚了！他想好好投资，但不知道他能赚多少钱。他知道他的银行每年复利的利率 $\red{R}$（$\red{0 }$到 $\red{20 }$之间的整数）。他有一个整数 $\red{M }$在 $\red{100...1,000,000 }$。

他知道他打算将钱投资于银行多少年$\red{（0...400）}$。通过计算他每年储蓄的利息来帮助他了解他将来会有多少钱。打印一个不四舍五入的整数答案。

保证测试数据的答案适合带符号的 $\red{32 }$位整数。

## 输入格式

第 $\red{1 }$行：三个空格分隔的整数：$\red{R}$、$\red{M }$和 $\red{Y }$

## 输出格式

第 $\red{1 }$行：一个整数，表示 $\red{FJ }$在 $\red{Y }$年后将拥有的美元数。

## 样例

#### 输入样例

```c
5 5000 4
```




#### 输出样例

```c
6077
```

## 提示

输入详细信息：$\red{5\%}$年息，$\red{5000}$元，$\red{4}$年

输出详细信息：

第$\red{1}$年：$\red{1.05\times5000=5250}$

第$\red{2}$年：$\red{1.05\times5250=5512.5}$

第$\red{3}$年：$\red{1.05\times5512.50=5788.125}$

第$\red{4}$年：$\red{1.05\times5788.125=6077.531256077.53125}$的整数部分为$\red{6077}$。
