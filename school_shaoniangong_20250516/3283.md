
# Description
任何大于 1 的自然数 N，都可以写成若干个大于等于2且小于等于 N 的质数之和表达式(包括只有一个数构成的和表达式的情况)，并且可能有不止一种质数和的形式。

例如9的质数和表达式就有四种本质不同的形式:
9= 2+5+2= 2+3+2+2 = 3+3+3 =  2+7

# Format

## Input
文件中的每一行存放一个自然数N，2≤N≤200。

## Output
依次输出每一个自然数N的本质不同的质数和表达式的数目

# Samples

```input1
2
```

```output1
1
```
```input2
200
```

```output2
9845164
```
