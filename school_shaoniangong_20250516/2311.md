# Description

一矩形阵列由数字0到9组成,数字1到9代表细胞,0是间隔，细胞的定义为沿细胞数字上下左右还是细胞数字则为同一细胞,求给定矩形阵列的细胞个数。如:

阵列

```
4 10
0234500067
1034560500
2045600671
0000000089
```

有4个细胞。

# Format

## Input

第一行为矩阵的行n和列m;

下面为一个n×m的矩阵。

## Output

细胞个数

# Samples

```c
4 10
0234500067
1034560500
2045600671
0000000089
```

```c
4
```

# Limitation

1s, 1024KiB for each test case.

m,n<=60
