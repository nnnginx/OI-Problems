## 题目描述

$\red{Farmer John}$新买的干草打包机的内部结构大概算世界上最混乱的了，它不象普通的机器一样有明确的内部传动装置，而是，$\red{N (2 <= N <= 1050)}$个齿轮互相作用，每个齿轮都可能驱动着多个齿轮。

$\red{FJ}$记录了对于每个齿轮$\red{i，}$记录了它的$\red{3}$个参数：$\red{X_i，}$$\red{Y_i}$表示齿轮中心的位置坐标$\red{(-5000 <= X_i <= 5000}$; $\red{-5000 <= Y_i <= 5000)}$；$\red{R_i}$表示该齿轮的半径$\red{(3 <= R_i <= 800)}$。


驱动齿轮的位置为$\red{0,0，}$并且$\red{FJ}$也知道最终的工作齿轮位于$\red{X_t,Y_t}$。 驱动齿轮顺时针转动，转速为$\red{10,000}$转$\red{/}$小时。

你的任务是，确定传动序列中所有齿轮的转速。传动序列的定义为，能量由驱动齿轮传送到工作齿轮的过程中用到的所有齿轮的集合。对能量传送无意义的齿轮都应当被忽略。



在一个半径为$\red{Rd，}$转速为$\red{S}$转$\red{/}$每小时的齿轮的带动下，与它相接的半径为$\red{Rx}$的齿轮的转速将为$\red{-S*Rd/Rx}$转$\red{/}$小时。$\red{S}$前 的负号的意思是，一个齿轮带动的另一个齿轮的转向会与它的转向相反。

$\red{FJ}$只对整个传动序列中所有齿轮速度的绝对值之和感兴趣，你的任务也就相应转化成求这个值。机器中除了驱动齿轮以外的所有齿轮都被另外某个齿轮带动，并且不会出现$\red{2}$个不同的齿轮带动同一个齿轮的情况。 相信你能轻易地写个程序来完成这些计算:$\red{)}$

## 输入格式

第$\red{1}$行: $\red{3}$个用空格隔开的整数：$\red{N，}$$\red{X_t，}$$\red{Y_t}$

第$\red{2..N+1}$行: 第$\red{i+1}$描述了齿轮$\red{i}$的位置及半径：$\red{X_i，}$$\red{Y_i，}$以及$\red{R_i}$

## 输出格式

第$\red{1}$行: 输出所有在传动中起到作用的齿轮转速的绝对值，包括驱动齿轮和 工作齿轮。

只需要输出答案的整数部分

## 样例

#### 输入样例

```c
4 32 54
0 0 10
0 30 20
32 54 20
-40 30 20

```




#### 输出样例

```c
20000
```

## 提示

输出说明:


齿轮 位置  半径     转速
$\red{1 (0,0)     10     10,000}$
$\red{2 (0,30)    20     -5,000}$
$\red{3 (32,54)   20      5,000}$
$\red{------}$
齿轮转速绝对值之和:$\red{20,000}$

机器里一共有$\red{4}$个齿轮，位于$\red{0,0}$的是半径为$\red{10}$的驱动齿轮，它带动了位于 $\red{0,30}$的，半径为$\red{20}$的某个齿轮。这个齿轮又间接带动了位于$\red{32,54，}$半径为$\red{20}$的 工作齿轮，以及一个位于$\red{-40,30，}$半径同样为$\red{20}$的冗余的齿轮。
