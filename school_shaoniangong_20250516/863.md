## 题目描述

输入$\red{x}$和$\red{y}$，输出$\red{x}$年$\red{y}$月有多少天，闰年分为普通闰年和世纪闰年，普通闰年是能被$\red{4}$整除但不能被$\red{100}$整除，世纪闰年是可以被$\red{400}$整除。

## 输入格式

两行两个正整数$\red{x}$和$\red{y}$，分别表示年份和月份。

$\red{x}$在int范围以内，$\red{y}$为$\red{ 1 \sim 12}$。

## 输出格式

一个整数，表示该年该月有多少天。

## 样例

#### 输入样例

```C
2000 
3
```

#### 输出样例

```C
31
```
