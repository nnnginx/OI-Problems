## 题目描述

给出两个字符串`s1`,`s2`（(只有大写字母），求`s1`在`s2`中出现多少次。

例如：`s1="ABA"`,`s2="ABAABA"`,答案为`2`。

## 输入格式

输入`T`组数据，每组数据输出结果。

## 输出格式

如题述。

## 样例

#### 输入样例

```c
3
BAPC
BAPC
AZA
AZAAZAAZA
VEEDI
AVERDXIVYERDLAN
```

#### 输出样例

```c
1
3
0
```

## 提示

$\red {1≤s_1}$ 的长度 $\red{≤10^4}$​​ ，

$\red{1≤s_2}$的长度 $\red{≤10^6}$​ 。
