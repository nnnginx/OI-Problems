## 题目描述

给出正整数 $\red n$ 和 $\red k$，计算$\red{j(n, k)=k}$ $\red{mod}$ $\red{1 + k}$ $\red{mod}$ $\red{2 + k}$ $\red{mod}$ $\red 3$ $\red{+ … + k}$  $\red{mod}$ $\red n$的值。

例如$\red{j(5, 3)=3}$ $\red{mod}$ $\red{1 + 3}$ $\red{mod}$ $\red{2 + 3}$ $\red{mod}$ $\red{3 + 3}$ $\red{mod}$ $\red{4 + 3}$ $\red{mod}$ $\red{5=0+1+0+3+3=7}$。

## 输入格式

输入仅一行，包含两个整数$\red n$, $\red k$。

## 输出格式

输出仅一行，即$\red{j(n, k)}$。

## 样例

#### 输入样例

5 3

#### 输出样例

7

## 提示

$\red{1\leq n,k\leq 10^9}$
