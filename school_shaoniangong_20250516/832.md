## 题目描述

对于半径为 $\red{r}$ 的球，其体积的计算公式为 $\red{V=\frac{ 4 } { 3 }πr ^3}$ ，这里取 $\red{π=3.14}$。现给定 $\red{r}$，即球半径，类型为`double`，求球的体积$\red{V}$，

保留到小数点后$\red{2}$位。

## 输入格式

输入为一个不超过 $\red{100}$ 的非负实数，即球半径，类型为`double`。

## 输出格式

输出一个实数，即球的体积，保留到小数点后 $\red{2}$位。

## 样例

#### 输入样例

```C
4
```

#### 输出样例

```C
267.95
```
