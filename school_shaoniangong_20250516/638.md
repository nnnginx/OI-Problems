## 题目描述

现代数学的著名证明之一是`Georg Cantor`证明了有理数是可枚举的。

他是用下面这一张表来证明这一命题的：

$\red{1/1\  1/2\  1/3\  1/4 \ 1/5\ …}$

$\red{2/1\  2/2\  2/3\  2/4\  …}$

$\red{3/1\  3/2\  3/3\  …}$

$\red{4/1\  4/2\  …}$

$\red{5/1\  …}$

$\red{…}$

我们以$\red Z$字形给上表的每一项编号。
第一项是$\red{1/1}$，然后是$\red{1/2}$，$\red{2/1}$，$\red{3/1}$，$\red{2/2}$，$\red{…}$

## 输入格式

输入：整数$\red{N（1≤N≤1e7）}$

## 输出格式

输出：表中的第$\red N$项

#### 输入样例

```c
7
```

#### 输出样例

```c
1/4
```
