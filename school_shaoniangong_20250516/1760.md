## 题目描述

`ZJU 2271`

墨老师:"做了这么多题，同学们上课都挺累的吧,我给大家讲个故事放松一下,故事是
这样的,很久很久以前,魔法世界有一个喜爱旅游的公主,她所在的国家由$\red{n \times n}$个城镇组
成,$\red{n=2\times k-1}$且$\red{2≤}$$\red{k≤}$$\red{50,}$如图为$\red{n=5}$时的情况。

![img](./1760/file/11.png)

其中最左上方的城镇假设坐
标为$\red{(0,0) ,}$公主的初始位置为$\red{(n/2,n/2),}$即$\red{A}$处，每天公主随心所欲地从一个城镇到另一
个城镇,她的移动方向为上、下、左、右。王子的初始位置为$\red{(-1,n/2),}$即$\red{B}$处,但王子每天
只能由左到右移动到下一个城镇。请问:在王子移动出这个国家之前,会有多大的概率遇
到公主?"

"老师你够了!"

## 输入格式

一个整数，即棋盘大小$\red{n}$。

## 输出格式

王子遇到主的概率值。

## 样例

#### 输入样例

```c
3
```

#### 输出样例

```c
0.6667
```
