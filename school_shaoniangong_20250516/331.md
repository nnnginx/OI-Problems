## 题目描述

在一个有$\red {m\times n}$ 个方格的棋盘中，每个方格中有一个正整数。现要从方格中取数，使任
意$\red {2}$ 个数所在方格没有公共边，且取出的数的总和最大。试设计一个满足要求的取数算法。
编程任务：
对于给定的方格棋盘，按照取数要求编程找出总和最大的数。

## 输入格式

由文件$\red {input.txt}$提供输入数据。文件第$\red {1}$ 行有$\red {2}$ 个正整数$\red {m}$和$\red {n}$，分别表示棋盘的行数
和列数。接下来的$\red {m}$行，每行有$\red {n}$个正整数，表示棋盘方格中的数。
$\red {n,m<=30}$。

## 输出格式

程序运行结束时，将取数的最大总和输出到文件$\red {output.txt}$中。

## 样例

#### 输入样例

```c
3 3
1 2 3
3 2 3
2 3 1
```

#### 输出样例

```c
11
```
