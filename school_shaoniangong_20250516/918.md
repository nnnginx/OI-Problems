## 题目描述

谓角谷猜想，是指对于任意一个正整数，

- 如果是奇数，则乘$\red{3}$加$\red{1}$，
- 如果是偶数，则除以$\red{2}$，得到的结果再按照上述规则重复处理，最终总能够得到$\red{1}$。

如，假定初始整数为$\red{5}$，计算过程分别为$\red{16、8、4、2、1}$。程序要求输入一个整数，将经过处理得到$\red{1}$的过程输出来。

## 输入格式

一个正整数$\red{N}$($\red{N≤ 2,000,000}$ )。

## 输出格式

从输入整数到$\red{1}$的步骤，每一步为一行，每一部中描述计算过程。

最后一行输出"`End`"。如果输入为$\red{1}$，直接输出"`End`"。

## 样例

#### 输入数据

```C
5
```

#### 输出数据

```C
5*3+1=16
16/2=8
8/2=4
4/2=2
2/2=1
End
```
