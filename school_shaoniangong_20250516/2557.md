## 题目描述

农夫约翰有$\red{N(1≤}$$\red{N≤}$$\red{5000)}$只牛站成一排，有一些很乖的牛朝前站着．但是有些不乖的牛却朝后站着．农夫约翰需要让所有的牛都朝前站着．幸运的是约翰最近买了一个自动转身机．

这个神奇的机器能使$\red{K(1≤}$$\red{K≤}$$\red{N)}$只连续的牛转身．  因为约翰从来都不改变$\red{K}$的价值，请帮助他求出$\red{K，}$使旋转次数$\red{M}$达到最小．同时要求出对应的$\red{M.}$

## 输入格式

第$\red{1}$行：整数$\red{N.}$

第$\red{2}$行到第$\red{N+1}$行：第$\red{i+l}$行表示牛$\red{j}$的朝向，$\red{F}$表示朝前，$\red{B}$表示朝后．


## 输出格式

一行两个数，分别是$\red{K}$和$\red{M，}$中间用空格隔开

## 样例

#### 输入样例

```c
7
B
B
F
B
F
B
B

```



#### 输出样例

```c
3 3
```

## 提示

输入详细信息：有七头牛，它们面朝后，向后，向前，向后、向前、向后和向后。
输出详细信息：对于$\red{K=3，}$机器必须操作三次：转动奶牛（$\red{1,2,3}$），（$\red{3,4,5}$），最后（$\red{5,6,7}$）：

```
B > F   F   F
      B > F   F   F
      F > B > F   F
      B   B > F   F
      F   F > B > F
      B   B   B > F
      B   B   B > F
```

当$\red{K=3}$时神奇的机器旋转$\red{3}$次：$\red{(1，}$$\red{2，}$$\red{3)，}$$\red{(3，}$$\red{4，}$$\red{5)，}$和$\red{(5，}$$\red{6，}$$\red{7)}$
