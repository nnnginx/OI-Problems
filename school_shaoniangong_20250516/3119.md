# 题目描述

MJ是一家著名珠宝店，在2056年6月的某天凌晨时分，有大量珠宝被国际大盗偷走了。警察正在全力追查此案件，并且找到了珠宝店的负责人一起查看珠宝店内的监控系统。

已知，珠宝店可视作一个n*m（ n 和 m表示矩阵的长和宽）的矩阵，起初珠宝店装修的时候，为了安全起见在店内安装了一批摄像头，每个摄像头都能监控店内的一个矩形区域，

该矩形区域是由左上角坐标(x,y)，右下角坐标（l,r）围成。

当天，警察想要查看店内某些区域的监控收集线索，请问他们能看到区域的每一个位置吗？

## 输入

第一行， 输入三个整数 n , m , p ，依次表示珠宝店的长和宽，摄像头的数量

接下来的 p 行，每行四个整数: x  , y  , l , r ,表示第i个摄像头的监控区域矩阵的左上角坐标(x ,y)， 右下角坐标（l,r）.

接下来，输入一行一个整数 q , 表示查看监控区域的数量。

接下来的 q 行，每行四个整数: x<sub>1</sub> , y<sub>1</sub> , x<sub>2</sub> , y<sub>2</sub> ,表示第i个摄像头的监控区域矩阵的左上角坐标(x<sub>1</sub>,y<sub>1</sub>)， 右下角坐标（x<sub>2</sub>,y<sub>2</sub>）.

## 输出

q 行，若能查看该区域所有的位置，输出 YES ，否则输出 NO

# 样例

```输入
6 6 3
2 2 4 4 
3 3 5 6 
5 1 6 2
2
3 2 5 4 
1 5 6 5
```

```output1
YES
NO
```

# 数据说明

对于30% 的数据：1 ≤ n,m ≤ 50,1 ≤ x  ≤l ≤ n , 1 ≤ y  ≤  r ≤ m,1 ≤ p ≤ 100,1 ≤ q ≤ 100

对于100% 的数据：1 ≤ n,m ≤ 1000,1 ≤ x<sub>1</sub> ≤x<sub>2</sub> ≤ n , 1 ≤ y<sub>1</sub> ≤  y<sub>2</sub> ≤ m,1 ≤ p ≤ 10<sup>6</sup>,1 ≤ q ≤ 10<sup>6</sup>
