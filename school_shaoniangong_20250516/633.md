## 题目描述

一个正方形的镇区分为 $\red{N2}$ 个小方块$\red{（1 <= N <= 7）}$.农场位于方格的左上角,集市位于左下角.贝茜穿过小镇,从左上角走到左下角,刚好经过每个方格一次.当 $\red{N=3}$ 时,贝茜的漫游路径可能如下图所示:

<pre>
----------------
|    |    |    |
| F**********  |
|    |    | *  |
------------*---
|    |    | *  |
|  *****  | *  |
|  * | *  | *  |
---*---*----*---
|  * | *  | *  |
|  M | ******  |
|    |    |    |
----------------
</pre>

写一个程序,对于给出的 $\red N$ 值,计算贝茜从农场走到集市有多少种唯一的路径.

## 输入格式

$\red 1$行: 一个整数 $\red {N （1 <= N <= 7）}$

## 输出格式

只有一行.输出一个整数表示唯一路径的数量.

## 样例

#### 输入样例

```c
3
```

#### 输出样例

```c
2
```
