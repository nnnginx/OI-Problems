## 题目描述

王老师正在批改小明的试卷。

试卷满分为 $\red{n}$分，小明得了 $\red{m}$ 分。规定得分是满分 $\red{85\%}$ 或以上为 $\red{A}$ 等级，$\red{75\%}$ 或以上为 $\red{B}$ 等级，$\red{60\%}$ 或以上为 $\red{C}$ 等级，$\red{60\%}$ 以下（不含）为 $\red{D}$ 等级。王老师想知道小明的等级，但她数学不好，只好求助你了。

## 输入格式

两个整数 $\red{n,m}$。

## 输出格式

一个字符，为小明的等级。

## 样例

#### 输入样例1

```c
85 100
```

#### 输出样例1

```c
A
```

#### 输入样例2

```c
34 50
```

#### 输出样例2

```c
C
```

## 提示

样例一小明得了 $\red{85}$ 分，满分为 $\red{100}$ 分，得分是满分的 $\red{85\%}$，因此他是 $\red{A}$ 等级。

$\red{1\le n \le m \le 150}$
