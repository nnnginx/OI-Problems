## 题目描述

原题来自：`USACO 2008 Dec. Gold`


贝茜正在领导奶牛们逃跑。为了联络，奶牛们互相发送秘密信息。

信息是二进制的，共有 $\red{M}$ 条。反间谍能力很强的约翰已经部分拦截了这些信息，知道了第 $\red{i}$ 条二进制信息的前 $\red{b_i}$​ 位。他同时知道，奶牛使用 $\red{N}$ 条密码。但是，他仅仅了解第 $\red{j}$ 条密码的前 $\red{c_j}$​ 位。

对于每条密码 $\red{j}$ ，他想知道有多少截得的信息能够和它匹配。也就是说，有多少信息和这条密码有着相同的前缀。当然，这个前缀长度必须等于密码和那条信息长度的较小者。

## 输入格式

第一行输入 $\red{N}$ 和 $\red{M}$，之后 $\red{N}$ 行描述秘密信息，之后 $\red{M}$ 行描述密码．每行先输入一个整数表示信息或密码的长度，之后输入这个信息或密码。

所有数字之间都用空格隔开。

## 输出格式

共 $\red{M}$ 行，输出每条密码的匹配信息数。

## 样例

#### 输入样例

```c
4 5
3 0 1 0
1 1
3 1 0 0
3 1 1 0
1 0
1 1
2 0 1
5 0 1 0 0 1
2 1 1
```

#### 输出样例

```c
1
3
1
1
2
```

$\red{4 }$条信息，$\red{5 }$条密码

截获的信息前缀是 $\red{010, 1, 100, 110}$，可能的密码前缀是 $\red{0, 1, 01, 01001, 11}$。

$\red{0}$ 只配对$\red{ 010}$；

$\red{1 }$配对$\red{ 1, 100, 110}$；

$\red{01 }$只配对 $\red{010}$；

$\red{01001 }$配对 $\red{010}$；

$\red{11 }$配对 $\red{1,110}$。

## 提示

对于 $\red{100\%}$的数据， $\red{1\le M\le 50000,1\le N\le 50000,1\le b_i\le 10000,1\le c_j\le 10000}$，位的总数即 $\red{\sum B_i+\sum C_i​ }$不会超过 $\red{500000}$。
