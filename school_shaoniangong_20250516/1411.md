## 题目描述

桌子上有一个$\red{m}$行$\red{n}$列的方格矩阵，将每个方格用坐标表示，行坐标从下到上依次递增，列坐标从左至右依次递增，左下角方格的坐标为$\red{(1,1)}$，则右上角方格的坐标为$\red{(m,n)}$。

小明是个调皮的孩子，一天他捉来一只蚂蚁，不小心把蚂蚁的右脚弄伤了，于是蚂蚁只能向上或向右移动。小明把这只蚂蚁放在左下角的方格中，蚂蚁从左下角的方格中移动到右上角的方格中，每步移动一个方格。蚂蚁始终在方格矩阵内移动，请计算出不同的移动路线的数目。

对于$\red{1}$行$\red{1}$列的方格矩阵，蚂蚁原地移动，移动路线数为$\red{1}$；对于$\red{1}$行$\red{2}$列（或$\red{2}$行$\red{1}$列）的方格矩阵，蚂蚁只需一次向右（或向上）移动，移动路线数也为$\red{1}$……对于一个$\red{2}$行$\red{3}$列的方格矩阵，如下图所示：

\-----------------------

|$\red{(2,1)}$|$\red{(2,2)}$|$\red{(2,3)}$|

\-----------------------

|$\red{(1,1)}$|$\red{(1,2)}$|$\red{(1,3)}$|

\-----------------------

蚂蚁共有$\red{3}$种移动路线：

路线$\red{1}$：$\red{(1,1)→ (1,2) → (1,3) → (2,3)}$

路线$\red{2}$：$\red{(1,1)→ (1,2) → (2,2) → (2,3)}$

路线$\red{3}$：$\red{(1,1)→ (2,1) → (2,2) → (2,3)}$

## 输入格式

输入只有一行，包括两个整数$\red{m}$和$\red{n（0<m+n<=20）}$，代表方格矩阵的行数和列数，$\red{m、n}$之间用空格隔开

## 输出格式

输出只有一行，为不同的移动路线的数目。

## 样例

#### 输入样例

```c
2 3
```

#### 输出样例

```c
3
```
