## 题目描述

有$\red{n}$个正整数$\red{a[i]，}$设它们乘积为$\red{p，}$你可以给$\red{p}$乘上一个正整数$\red{q，}$使$\red{p\times q}$刚好为正整数$\red{m}$的阶乘，求$\red{m}$的最小值。

## 输入格式

共两行。

第一行一个正整数$\red{n}$。

第二行$\red{n}$个正整数$\red{a[i]}$。

## 输出格式

共一行
一个正整数$\red{m}$。

## 样例

#### 输入样例

```c
1
6
```

#### 输出样例

```c
3
```

## 提示

对于$\red{10\%}$的数据，$\red{n<=10}$

对于$\red{30\%}$的数据，$\red{n<=1000}$

对于$\red{100\%}$的数据，$\red{n<=100000，}$$\red{a[i]<=100000}$
