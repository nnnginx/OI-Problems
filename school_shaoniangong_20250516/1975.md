##题目描述
话说$\red{CD}$比较欠扁，他表示在课室的日子没有教主在旁边打他的日子太寂寞了，所以这一晚，他终于来到了电脑室被打。由于$\red{CD}$是大家的宠物，于是大家都来打$\red{CD}$了。电脑室里有$\red{n}$个人，第$\red{i}$个人希望打$\red{CD ai}$下。但是太多人打$\red{CD，}$他又会不爽，于是他规定只能有$\red{K}$个人打到他，并且为了公平起见，最终$\red{K}$个人打他的次数都必须是相同的，$\red{CD}$规 定这个次数就是这$\red{K}$个人希望打他的次数的最大公约数。为什么是最大公约数呢？因为他觉得被打的次数是$\red{GCD}$的话他才会变成$\red{Glad CD}$。之前说了，$\red{CD}$比较欠扁，于是$\red{CD}$希望，$\red{K}$个人打他的次数的和最大。你能告诉他他最后总共会被打多少下么？

## 输入格式

第一行两个正整数$\red{n，}$$\red{k}$。
第二行$\red{n}$个正整数，表示每个人希望打$\red{CD}$多少下。

## 输出格式

输出一个正整数表示$\red{CD}$会被打多少下。

## 样例

#### 输入样例

```c
3 1
1 2 3
```

#### 输出样例

```c
3
```

##提示
对于$\red{30\%}$的数据，保证$\red{k≤}$$\red{n≤}$$\red{20}$。

对于$\red{50\%}$的数据，保证输入中所有数小于$\red{5000}$。

对于$\red{100\%}$的数据，保证输入中所有数小于$\red{500000，}$$\red{k≤}$$\red{n}$。
