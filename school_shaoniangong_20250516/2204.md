## 题目描述

农民约翰为他的奶牛购买了一份好的《护蹄》杂志的订阅，因此在挤奶期间，奶牛们在谷仓里等待时有很多材料可以阅读。不幸的是，最新一期包含了一篇关于如何烹饪完美牛排的相当不恰当的文章，$\red{FJ}$宁 愿他的奶牛看不到这篇文章（显然，该杂志需要更好的编辑监督）。

$\red{FJ}$从杂志中获取了所有文本，以创建长度不超过$\red{10^5}$个字符的字符串。他有一个经过审查的单词列表$\red{t_1...t_N}$他希望从$\red{S}$中删除。为此，农民约翰在$\red{S}$中找到最早出现的经过审查的单词（具有最早的开始索引），并从$\red{S}$中删除该单词实例。然后他再次重复该过程，从$\red{S}$中删除最早出现的经过审查的单词，重复，直到$\red{S}$中不再出现经过审查的单词。请注意，删除一个经过审查的单词可能会使以前不存在的经过审查的单词再次出现。

$\red{Farmer John}$指出，经过审查的单词具有这样的属性：没有经过审查的单词会作为另一个经过审查的单词的子字符串出现。特别是，这意味着在$\red{S}$中具有最早索引的经过审查的词是唯一定义的。

审查完成后，请帮助$\red{FJ}$确定$\red{S}$的最终内容。

## 输入格式

第一行将包含$\red{S}$。

第二行将包含$\red{N，}$即经过审查的字数。

接下来的$\red{N}$行包含字符串$\red{t_1...t_N}$每个字符串将包含小写字母字符（在$\red{a...z}$范围内），所有这些字符串的组合长度最多为$\red{10^5}$。

## 输出格式

所有删除完成后的字符串。可以保证$\red{S}$在删除过程中不会变为空。

## 样例

#### 输入样例

```c
begintheescapexecutionatthebreakofdawn
2
escape
execution
```

#### 输出样例

```c
beginthatthebreakofdawn
```
