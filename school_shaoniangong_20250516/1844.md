## 题目描述

字母$\red{A,B,C}$的所有可能的组合（按字典顺序排序）是:$\red{A,AB,ABC,AC,B,BC,C}$。
每个组合都对应一个字典顺序的序号,如下所示:

$\red{1~~~A}$

$\red{2~~~AB}$

$\red{3~~~ABC}$

$\red{4~~~AC}$

$\red{5~~~B}$

$\red{6~~~BC}$

$\red{7~~~C}$

找出某个字母组合的字典序号。例如，上例中$\red{AC}$的字典序号是$\red{4}$。

注:假设某个字母组合为$\red{X_1X_2X_3...X_K,}$保证$\red{X_1<X_2<X_3<...<X_K}$。

## 输入格式

输人包括$\red{2}$行:

第一行:$\red{N，}$表示字母组合由字母表中前$\red{N(N≤}$$\red{26)}$个字母组成;

第二行:$\red{K}$,求编号为$\red{K}$的字母组合;

## 输出格式

该字母组合;均为大写字母。

## 样例

#### 输入样例

```c
3
2
```

#### 输出样例

```c
AB
```
