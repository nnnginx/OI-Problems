## 题目描述

$\red{Farmer John }$正在继续思考前面两个问题中介绍的奶牛过马路的问题。

他现在意识到友好的门槛比他之前考虑的要微妙一些--如果 $\red{|a-b|≤}$$\red{K，}$品种 $\red{a }$和 $\red{b }$现在是友好的，否则不友好。

给定通过$\red{FJ}$农场的道路两侧的田地顺序，请计算不友好的品种交叉对的数量，其中交叉品种对的定义如前面的问题。

## 输入格式

第一行输入包含$\red{N（}$$\red{1≤}$$\red{N≤}$$\red{100,000）}$和$\red{K（}$$\red{0≤}$$\red{K<N）}$。

接下来的 $\red{N }$行按照品种 $\red{ID }$描述了道路一侧的田地的顺序；每个品种 $\red{ID }$都是 $\red{1…}$$\red{N}$的整数。

最后 $\red{N }$行按品种 $\red{ID }$描述道路另一边的田地的顺序。

每个品种 $\red{ID }$在每个排序中仅出现一次。

## 输出格式

请输出不友好杂交品种的数量。

## 样例

#### 输入样例

```c
4 1
4
3
2
1
1
4
2
3

```

#### 输出样例

```c
2
```

## 提示

在这个例子中，品种$\red{1}$和$\red{4}$是不友好和交叉的，就像品种$\red{1}$和$\red{3}$一样。
