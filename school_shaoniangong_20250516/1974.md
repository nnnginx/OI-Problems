## 题目描述

$\red{moreD}$和$\red{moreD}$的宠物$\red{CD}$正在玩一个日历游戏，开始时，他们从$\red{1900}$年$\red{1}$月$\red{1}$日到$\red{2012}$年$\red{12}$月$\red{22}$日（你懂的……）选一个日期开始，依次 按照如下规则之一向后跳日期：
$\red{1.}$跳到日历上的下一天。
$\red{2.}$跳到日历上的下个月的同一天（如果不存在，则不能这么做）。
要是谁正好到达$\red{2012}$年$\red{12}$月$\red{22}$日那么他就赢了，如果到达这天之后的日期那他就输了$\red{--}$原因你也懂的。
每次都是$\red{moreD}$先走的。
现在，给你一个日期，请问$\red{moreD}$一定能赢吗？

## 输入格式

输入共$\red{T}$行，每行三个整数，$\red{Y}$、$\red{M}$、$\red{D，}$分别表示年、月、日。日期在$\red{1900}$年$\red{1}$月$\red{1}$日到$\red{2012}$年$\red{12}$月$\red{22}$日之间（包含两端）。
$\red{T}$并不在输入数据当中。

## 输出格式

要是$\red{moreD}$一定能赢，输出一行$\red{YES，}$否则输出$\red{NO}$。

## 样例

#### 输入样例1

```c
2012 12 20
```

#### 输出样例1

```c
NO
```

#### 输入样例2

```c
2012 12 21
```

#### 输出样例2

```c
YES
```

【提示】
对于$\red{50\%}$的数据，是$\red{1949}$年$\red{1}$月$\red{1}$日后的日期。 $\red{T <= 5}$

对于$\red{100\%}$的数据，是$\red{1900}$年$\red{1}$月$\red{1}$日后的日期。$\red{T <= 10}$
