

## 题目描述
面面从小酷爱编程，并且在课余时间会登录`OJ`网站练习。一天面面遇到了这样一道题，输入一个随机数$\red n$,如果当前的随机数从左往右读和从右往左读是一样的，则说明该数是一个数字回文数，如果是回文数则输出“`Yes`"否则输出"`No`”，这些可把面面难住了，聪明的你快来帮帮他吧。

## 输入格式
共一行，包含一个正整数$\red n$;

## 输出格式
共一行，如果是数字回文数则输出“`Yes`"否则输出“`No`"
## 样例
#### 输入样例
```C
12321
```
#### 输出样例
```C
Yes
```

## 提示
对于所有的数据满足：
$\red{1<=n<=10^9}$；
