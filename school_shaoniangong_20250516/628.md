## 题目描述

一些学校连入一个电脑网络。那些学校已订立了协议：每个学校都会给其它的一些学校分发软件（称作“接受学校”）。注意即使 $\red B$ 在 $\red A$ 学校的分发列表中， $\red A$ 也不一定在 $\red B$ 学校的列表中。

你要写一个程序计算，根据协议，为了让网络中所有的学校都用上新软件，必须接受新软件副本的最少学校数目（子任务 $\red A$）。更进一步，我们想要确定通过给任意一个学校发送新软件，这个软件就会分发到网络中的所有学校。为了完成这个任务，我们可能必须扩展接收学校列表，使其加入新成员。计算最少需要增加几个扩展，使得不论我们给哪个学校发送新软件，它都会到达其余所有的学校（子任务 $\red B$）。一个扩展就是在一个学校的接收学校列表中引入一个新成员。

## 输入格式

输入文件的第一行包括一个整数 $\red N$：网络中的学校数目$\red{（2 <= N <= 100）}$。学校用前 $\red N$ 个正整数标识。

接下来 $\red N$ 行中每行都表示一个接收学校列表（分发列表）。第 $\red{i+1}$ 行包括学校 $\red i$ 的接收学校的标识符。每个列表用 $\red 0$ 结束。空列表只用一个 $\red 0$ 表示。

## 输出格式

你的程序应该在输出文件中输出两行。

第一行应该包括一个正整数：子任务 $\red A$ 的解。

第二行应该包括子任务 $\red B$ 的解。

## 样例

#### 输入样例

```c
5
2 4 3 0
4 5 0
0
0
1 0
```

#### 输出样例

```c
1
2
```

## 提示

$\red{2≤N≤100}$。
