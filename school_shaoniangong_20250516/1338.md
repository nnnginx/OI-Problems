### 题目描述
`Alien` 们过着群居的生活，有 $\red{N}$ 个居住的地点(可以看作是平面上的坐标)。 

为了方便考虑问题，我们把两个群居部落的距离，定义为部落中距离最近的那两个居住点的距离。

现在 `Alien` 们总共被分为了 $\red{K}$ 个部落。`Alien` 们委托你求出一种部落划分的方法，使
靠得最近的两个部落尽可能远离。
### 输入格式
第一行四个整数$\red{ N，K}$，意义如上。
接下来 $\red{N}$ 行，每行两个数 $\red{X_i, Y_i}$，表示一个点的坐标。 

### 输出格式
一行一个数 ，表示答案，精确到小数点后两位。
### 样例
#### 输入样例
```
4 2
0 0
0 1
1 1
1 0
```
#### 输出样例
```
1.00
```
### 提示
对于 $\red{100\%}$数据 $\red{N,K≤1000}$
