# Description

huhe的拍照技术是很强的？？？？
他去仙本那期间，拍了不少猴子，在仙本那的红树林，有长尾猴和短尾猴两种猴子，huhe就是喜欢去拍猴子，那里有很多猴子。
huhe是有追求的人，他会给N只猴子拍照，他的照片中一定会有三只或者以上的猴子，他只愿意给
**连续**的三只或以上的猴子拍照，但不愿意猴子序列中**只有一只同类**的猴子，例如只有一只长尾猴或者一只短尾猴。
如果出现了这种情况，他会把这张照片删掉。

# Format

## Input

第一行输入包含N，表示有N只猴子给huhe拍照

第二行包含一个字符串N字符。G代表长尾猴，H代表短尾猴

## Output

huhu删掉的照片数量

# Samples

```input1
5
GHGHG
```

```output1
3
```

【样例解释】

这个例子中的每一个长为 3 的子串均恰好包含一只长尾猴或者一只短尾猴――所以这些子串是huhe会删掉的照片，所有更长的子串（GHGH、HGHG 和 GHGHG）则不会。

# Limitation

* 测试用例 2 到 4 具有N≤50
* 测试用例 5 到 10 具有N≤5000
* 测试用例 11 没有其他约束。
  请注意， 这种情况的答案可能太大，无法放入标准的 32 位 整数，并且可能需要使用更大的整数类型（例如，64 位“ long long int“）。
