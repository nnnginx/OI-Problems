## 题目描述

在黑板上写了$\red {N}$ 个正数组成的一个数列，进行如下操作：每一次擦去其中$\red {2}$个数设为$\red {a}$和$\red {b}$，然后在数列中加入一个数$\red {a\times b+1}$，如此下去直至黑板上只剩下一个数。在所有按这种操作方式最后得到的数中，最大的数记为$\red {max}$，最小的数记为$\red {min}$，则该数列的极差$\red M$ 定义为$\red {M=max-min}$。
对于给定的数列，编程计算出其极差$\red {M}$。

## 输入格式

第一行是数列的长度$\red {N}$ （不超过$\red {2000}$），第二行起是数列中的$\red {N }$个数，相邻$\red {2}$个数由空格分隔。。

## 输出格式

结果应分两行输出，第一行是数$\red {M}$的位数，第二行是数$\red {M}$。

## 样例

#### 输入样例

```c
3

1 1 1
```

#### 输出样例

```c
1

0
```
