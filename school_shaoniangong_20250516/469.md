## 题目描述

`Hecy` 又接了个新任务：$\red{BE}$ 处理。$\red{BE}$ 中有一类被称为$\red{ GBE}$。

以下是 $\red{GBE}$ 的定义：

$\red{1}$. 空表达式是 $\red{GBE}$

$\red{2}$. 如果表达式 `A` 是 $\red{GBE}$，则 `[A]` 与 `(A)` 都是 $\red{GBE}$

$\red{3}$. 如果 `A` 与 `B` 都是$\red{ GBE}$，那么 `AB` 是$\red{ GBE}$

下面给出一个$\red{BE}$，求至少添加多少字符能使这个 $\red{BE}$ 成为 $\red{GBE}$。

## 输入格式

输入仅一行，为字符串 $\red{BE}$。

## 输出格式

输出仅一个整数，表示增加的最少字符数。

## 样例

#### 输入样例

```c
[])
```

#### 输出样例

```c
1
```

## 提示

对于 $\red{100\%}$ 的数据，输入的字符串长度小于 $\red{100}$。
