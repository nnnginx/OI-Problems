## 题目描述

计算非负整数$\red{m}$到$\red{n}$(包括$\red{m}$和$\red{n}$)之间的所有奇数的和，( $\red {m \le n \le 300} $)

例如 $\red{m=3}$, $\red{n=12}$, 其和则为：$\red{3+5+7+9+11=35}$。

## 输入格式

两个数 $\red{m}$ 和 $\red{n}$，两个数以一个空格分开，其中 $\red{0 ≤ m ≤ n ≤ 300}$ 。

## 输出格式

输出一行，包含一个整数，表示$\red{m}$ 到 $\red{n}$（包括$\red{m}$ 和 $\red{n}$ ）之间的所有奇数的和

## 样例

#### 输入数据

```C
7 15
```

#### 输出数据

```C
55
```
