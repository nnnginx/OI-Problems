## 题目描述

`POJ 1650`

给定一个浮点数$\red{A}$和一个整数$\red{L}$,求在范围$\red{[1,L]}$内的两个整数$\red{n}$和$\red{d}$.

使得$\red{n/d}$能近似等于$\red{A}$.且使误差$\red{|A- n/d|}$最小。

## 输入格式

第一行为一个浮点数$\red{A}$.第二行为一个整数$\red{L}$。

## 输出格式

两个整数$\red{n}$和$\red{d}$。

## 样例

#### 输入样例

```c
3.14159265358979
10000
```

#### 输出样例

```c
355 113
```
