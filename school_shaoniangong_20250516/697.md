## 题目描述

每一本正式出版的图书都有一个$\red{ISBN}$号码与之对应，$\red{ISBN}$码包括$\red{9}$位数字、$\red{1}$位识别码和$\red{3}$位分隔符，其规定格式如$\red{“x-xxx-xxxxx-x”}$，其中符号$\red{“-”}$是分隔符（键盘上的减号），最后一位是识别码，例如$\red{0-670-82162-4}$就是一个标准的$\red{ISBN}$码。$\red{ISBN}$码的首位数字表示书籍的出版语言，例如$\red{0}$代表英语；第一个分隔符$\red{“-”}$之后的三位数字代表出版社，例如$\red{670}$代表维京出版社；第二个分隔之后的五位数字代表该书在出版社的编号；最后一位为识别码。

识别码的计算方法如下：

首位数字乘以$\red{1}$加上次位数字乘以$\red{2}$……以此类推，用所得的结果$\red{mod 11}$，所得的余数即为识别码，如果余数为$\red{10}$，则识别码为大写字母$\red{X}$。例如$\red{ISBN}$号码$\red{0-670-82162-4}$中的识别码$\red{4}$是这样得到的：对$\red{067082162}$这$\red{9}$个数字，从左至右，分别乘以$\red{1，2，…，9}$，再求和，即$\red{0×1+6×2+„„+2×9=158}$，然后取$\red{158}$ $\red{mod 11}$的结果$\red{4}$作为识别码。

你的任务是编写程序判断输入的$\red{ISBN}$号码中识别码是否正确，如果正确，则仅输出$\red{“Right”}$；如果错误，则输出你认为是正确的$\red{ISBN}$号码。

## 输入格式

只有一行，是一个字符序列，表示一本书的$\red{ISBN}$号码（保证输入符合$\red{ISBN}$号码的格式要求）。

## 输出格式

共一行，假如输入的$\red{ISBN}$号码的识别码正确，那么输出$\red{“Right”}$，否则，按照规定的格式，输出正确的$\red{ISBN}$号码（包括分隔符$\red{“-”}$）。

## 样例

#### 输入样例1

```c
0-670-82162-4
```

#### 输出样例1

```c
Right
```

#### 输入样例2

```c
0-670-82162-0
```

#### 输出样例2

```c
0-670-82162-4
```
