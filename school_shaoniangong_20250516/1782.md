## 题目描述

修罗王终于找到了埋藏上古神器的山洞，但没想到山洞入口还有一个自称神器守护者的家伙，他必须要解决守护者提出的一个问题才可以进入。该问题是：一个实数数列共有$\red{n}$项，已知：$\red{a_i=(a_{i-1}－a_{i+1})/2+d(1<i<n，n<60)}$。请编程从键盘输入：$\red{n，d，a_1，a_n，m}$(输入数据均不需判错)，输出$\red{a_m}$。

## 输入格式

五个数，即$\red{n，d，m，a_1，a_n}$。

## 输出格式

一个整数，即$\red{a_m}$。

## 样例

#### 输入样例

```c
3 2 2　1 -3
```

#### 输出样例

```c
4（强制转化为整型）
```
