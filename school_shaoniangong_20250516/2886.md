## 题目描述

小 $\red{T }$的花园里有很多朵花，由于数量实在是非常多，小 $\red{T }$只能告诉你花最少有 $\red{L}$
朵，最多有 $\red{R }$朵。

然而小 $\red{T }$是一个强迫症患者，所以他的花园里面花都是一排一排的，而他可以告诉
你每一排都有恰好 $\red{k }$朵花。

可是花的排数也同样太多了，所以小 $\red{T }$想问你花的数量有多少种可能的情况。

## 输入格式

一行三个整数 $\red{L, R, k}$。

## 输出格式

一个整数，表示可能的情况数量。

## 样例

#### 输入样例1

```c
2 5 3
```

#### 输出样例1

```c
1
```

#### 输入样例2

```c
2 6 3
```

#### 输出样例2

```c
3
```

## 提示

对于 $\red{40\% }$的数据保证 $\red{L, R, k ≤}$ $\red{10^5}$。

对于另外 $\red{20\% }$的数据保证 $\red{L, R }$都能被 $\red{k }$整除。

对于所有测试点保证 $\red{1 ≤}$ $\red{L ≤}$ $\red{R ≤}$ $\red{10^9，}$$\red{1 ≤}$ $\red{k ≤}$ $\red{10^9}$。
