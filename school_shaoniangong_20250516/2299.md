## 题目描述

佩奇有个机器人在$\red{（0，}$$\red{0）}$上，有四种指令 $\red{U,D,L,R}$分别表示让机器人向上、向下、向左、向右平移一个单 位。

由于一些原因，机器人停留在原地，佩奇认为这是因为部分指令没有被机器人识别，

因此它想要知道，若机器人在最后仍然待在原地，它最多执行了多少次命令？

## 输入格式

第一行一个正整数 $\red{n}$表示指令序列的长度。

第二行一个长度为 $\red{n}$的字符串，表示指令序列。

## 输出格式

一个整数，表示机器人最多执行了多少次命令。

## 样例

#### 输入样例1

```c
4
LDUR
```

#### 输出样例1

```c
4
```

#### 输入样例2

```c
5
RRRUU
```

#### 输出样例2

```c
0
```

## 提示

对于$\red{50\%}$的数据，有$\red{1≤}$$\red{n≤}$$\red{10}$;

对于$\red{100\%}$的数据，有$\red{1≤}$$\red{n≤}$$\red{100}$。
