## 题目描述

从前有个变量 $\red{x，}$它的初始值已给出。

你会依次执行 $\red{n }$次操作，每次操作有 $\red{p\% }$的概率令 $\red{x = x \times   2，}$$\red{(100 - p)\% }$的概率令 $\red{x= x + 1}$。

假设最后得到的值为 $\red{w，}$令 $\red{d }$为 $\red{w }$的质因数分解中 $\red{2 }$的次数，求 $\red{d }$的期望。

## 输入格式

从文件 $\red{exp.in }$中读入数据。

第一行三个整数 $\red{x, n, p，}$含义见题目描述。

## 输出格式

输出到文件 $\red{exp.out }$中。

一行一个实数，表示 $\red{d }$的期望。

如果你的答案与标准答案的误差不超过 $\red{10-6，}$则判定为正确。

## 样例

#### 输入样例1

```c
1 1 50
```

#### 输出样例1

```c
1.0000000000
```

#### 输入样例2

```c
5 3 0
```

#### 输出样例2

```c
3.0000000000
```

#### 输入样例3

```c
5 3 25
```

#### 输出样例3

```c
1.9218750000
```

## 提示

对于 $\red{20\% }$的数据，$\red{n ≤}$ $\red{20}$；

对于 $\red{30\% }$的数据，$\red{n ≤}$ $\red{50}$；

对于 $\red{50\% }$的数据，$\red{n ≤}$ $\red{100}$；

对于 $\red{100\% }$的数据，$\red{x ≤}$ $\red{10^9, n ≤}$ $\red{200, 0 ≤}$ $\red{p ≤}$ $\red{100}$。
