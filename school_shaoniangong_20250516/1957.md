# Description

瓦杂幸涣境担他要从家去邮局。他家距邮局d公里。瓦缘某岛懿睿它每行驶k公里就需要修理t秒后才能重新发动。开一公里汽车瓦孕枰花a秒，步行一公里他需要花费b秒(a<b)。你的任务是求瓦缘酱镉示值淖疃淌奔洹Ｍ钥梢栽谌魏问焙蚶肟车开始步行。

# Format

## Input

第一行包含5个正整数d，k，a，b，t。

## Output

求瓦缘酱镉示值淖疃淌奔洹

# Samples

```input1
5 2 1 4 10
```

```output1
14
```

```input2
5 2 1 4 5
```

```output2
13
```

在第一个例子瓦孕枰开2公里的车（2秒），然后步行3公里（12秒）。所以答案等于14秒。

在第二个例子瓦孕枰开2公里的车（2秒），然后修理自己的汽车（5秒）和再2公里的车（2秒）。之后，他需要步行1公里（4秒）。所以答案等于13秒。

# Limitation

对于30%的数据：1≤d≤10^3,1≤k,a,b,t≤100,a<b

对于100%的数据：1≤d≤10^12,1≤k,a,b,t≤1000000,a<b
