## 题目描述

大部分元素是$\red{0}$的矩阵称为稀疏矩阵，假设有$\red{k}$个非$\red{0}$元素，则可把稀疏矩阵用`K*3`的矩阵简记之，其中第一列是行号，第二列是列号，第三列是该行、该列下的非$\red{0}$元素的值。如：

<pre>
　　　　0  0  0  5               写简记成： 1  4  5      //第1行第4列有个数是5

　　　　0  2  0  0                         2  2  2      //第2行第2列有个数是2

　　　　0  1  0  0                         3  2  1      //第3行第2列有个数是1
</pre>

试编程读入一稀疏矩阵，转换成简记形式，并输出。

## 输入格式

第一行两个整数$\red{n}$和$\red{m}$，$\red{1≤n,m≤1000}$；

接下来共`n行`，每行$\red{m}$个整数，表示一个`n*m`的稀疏矩阵，其中稀疏矩阵非$\red{0}$元素个数为$\red{k}$，$\red{1≤k≤100}$。

（注意看清输入格式哦，教材中的程序需要修改！！）

## 输出格式

`k行`，每行三个整数，描述稀疏矩阵一个非$\red{0}$元素。

## 样例

#### 输入数据

```C
3 5
0 0 0 0 5
0 0 4 0 0
1 0 0 0 1
```

#### 输出数据

```C
1 5 5
2 3 4
3 1 1
3 5 1
```
