## 题目描述

农场被划分为$\red{5\times 5}$的格子，每个格子中都有一头奶牛，并且只有荷斯坦(标记为$\red{H}$)和杰尔西（标记为$\red{J}$）两个品种．如果一头奶牛在另一 头上下左右四个格子中的任一格里，我们说它们相连．

奶牛要大选了．现在有一只杰尔西奶牛们想选择$\red{7}$头相连的奶牛，划成一个竞选区，使得其中它们品种的奶牛比荷斯坦的多．  要求你编写一个程序求出方案总数．

## 输入格式

第$\red{1...5}$行：

五行中的每一行包含五个字符，每个字符为"$\red{H}$"或"$\red{J}$"。没有空格。

## 输出格式

第$\red{1}$行：

由$\red{7}$头相连奶牛组成的不同地区的数量，使得该地区的球衣数量超过了霍尔斯泰牛

## 样例

#### 输入样例

```c
HHHHH
JHJHJ
HHHHH
HJHHJ
HHHHH
```

#### 输出样例

```c
2
```
