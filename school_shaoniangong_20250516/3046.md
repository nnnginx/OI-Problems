# Description

小Z是一个海港的海关工作人员，每天都有许多船只到达海港，船上通常有很多来自不同国家的乘客。
小Z对这些到达海港的船只非常感兴趣，他按照时间记录下了到达海港的每一艘船只情况：对于第i艘到达的船，他记录了这艘船到达的时间$\red{t_i}$（单位：秒），船上的乘客数量$\red{k_i}$，以及每名乘客的国籍$\red{x_{i1}}$，$\red{x_{i2}}$，……，$\red{x_{iki}}$。
小Z统计了$\red{n}$艘船的信息，希望你帮忙计算出以每一艘到达时间为止的$\red{24小时（24小时=86400秒）}$内所有乘船到达的乘客来自多少个不同的国家。
形式化地讲，你需要计算$\red{n}$条信息，对于输出的第$\red{i}$条信息，你需要统计满足$\red{t_i-86400<t_p<=t_i}$的船只$\red{p}$，在所有的$\red{x_{pi}}$中，总共有多少个不同的数。

# Format

## Input

第一行一个正整数$\red{n}$，表示小Z统计了$\red{n}$艘船的信息。
接下来的$\red{n}$行，每行描述一艘船的信息：
前两个整数$\red{t_i}$和$\red{k_i}$分别看表示这艘船到达海港的时间和船上的乘客数量，接下来$\red{k_i}$个整数$\red{x_{i,j}}$表示船上乘客的国籍。
保证输入的$\red{t_i}$是递增的，单位是秒；表示从小Z第一次上班开始计时，这艘船在第$\red{t_i}$秒到达海港。

## Output

输出$\red{n}$行，第$\red{i}$行输出一个整数表示第$\red{i}$艘船到达后的统计信息。
