## 题目描述

有时大型游戏玩腻味了，小$\red{Z}$也会玩玩弱智级的小游戏，比如连连看。但小$\red{Z}$眼力不行，常常得分很低。于是，他找到了你，希望你能帮他找出一种获胜方案。
连连看的游戏规则很简单：玩家可以将 $\red{2 }$个相同图案的牌连接起来，连接线不多于 $\red{3 }$条线段，就可以成功将两个牌消除。将游戏界面上的牌全部消除掉，玩家就胜利了。

## 输入格式

输入文件$\red{card.in}$的第一行为两个正整数$\red{m}$和$\red{n，}$表示连连看游戏的矩阵大小。
接下来的$\red{m}$行，每行$\red{n}$个英文大写字母（为了计算方便，只用$\red{A}$~$\red{E}$五个字母），表示牌的种类，相同字母表示同种牌。

## 输出格式

输出文件$\red{card.out}$。
若能获胜，则输出共一行，包括$\red{m×}$$\red{n/2}$个英文大写字母，第$\red{i}$个字母表示第$\red{i}$次消除的牌的种类。若有多解，输出字典顺序最小的解。
若不能获胜，则输出共一行，包括字符串"$\red{Game over.}$"。

## 样例

#### 输入样例1

```c
2 1
A
A
```

#### 输出样例1

```c
A
```

#### 输入样例2

```c
2 1
A
B
```

#### 输出样例2

```c
Game over
```

## 提示

$\red{100\%}$的数据满足：$\red{m×}$$\red{n<=12}$
