## 题目描述

你刚刚继承了流行的“破锣摇滚”乐队录制的尚未发表的$\red{N(1 <= N <= 20)}$首歌的版权。你打算从中精选一些歌曲，发行$\red{M(1 <= M <= 20)}$张`CD`。每一张`CD`最多可以容纳$\red{T(1 <= T <= 20)}$分钟的音乐，一首歌不能分装在两张`CD`中。`CD`数量可以用完，也可以不用完

不巧你是一位古典音乐迷，不懂如何判定这些歌的艺术价值。于是你决定根据以下标准进行选择：

- $\red{1.}$歌曲必须按照创作的时间顺序在所有的`CD`盘上出现。(注：第$\red i$张盘的最后一首的创作时间要早于第$\red{i+1}$张盘的第一首)
- $\red{2.}$选中的歌曲数目尽可能地多。

## 输入格式

- 第一行： 三个整数：$\red N$, $\red T$, $\red M$.
- 第二行： $\red N$个整数，分别表示每首歌的长度，按创作时间顺序排列。

## 输出格式

一个整数，表示可以装进$\red M$张`CD`盘的乐曲的最大数目。

## 样例

#### 输入样例

```c
4 5 2
4 3 4 2
```

#### 输出样例

```c
3
```
