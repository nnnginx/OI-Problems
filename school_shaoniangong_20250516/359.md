## 题目描述

明明做作业的时候遇到了 $\red{n}$ 个二次函数 $\red{S_i(x)= ax^2 + bx + c}$，他突发奇想设计了一个新的函数 $\red{F(x) = \max\{S_i(x)\}, i = 1\ldots n}$。

![img](./359/file/5b39a8000e615.jpg)

明明现在想求这个函数在 $\red{[0,1000]}$ 的最小值，要求精确到小数点后四位，四舍五入。

## 输入格式

输入包含 $\red{T}$ 组数据，每组第一行一个整数 $\red{n}$；

接下来 $\red{n}$ 行，每行 $\red{3}$ 个整数 $\red{a}$, $\red{b}$, $\red{c}$ ，用来表示每个二次函数的 $\red{3}$ 个系数。注意：二次函数有可能退化成一次。

## 输出格式

每组数据输出一行，表示新函数 $\red{F(x)}$ 的在区间 $\red{[0,1000]}$ 上的最小值。精确到小数点后四位，四舍五入。

## 样例

#### 输入样例

```c
2
1
2 0 0
2
2 0 0
2 -4 2
```

#### 输出样例

```c
0.0000
0.5000
```

## 提示

对于 $\red{50\%}$ 的数据，$\red{1 \leq n \leq 100}$；

对于 $\red{100\%}$ 的数据，$\red{1 \leq T \leq 10, 1 \leq n \leq 10^5, 0 \leq a \leq 100, 0 \leq |b| \leq 5000, 0 \leq |c| \leq 5000}$。
