## 题目描述

通过计算这个时空的坐标，张琪曼和李旭林发现她们被修罗王的“时空陷”送到了$\red{40}$年前的过去，她们只能通过随身携带的魔法钟先传送一个人回到现在再想办法搭救另一个人。已知魔法钟是在一个$\red{3×3}$矩阵空间摆立着九个钟如图所示，通过一定的方法旋转时钟指针，使所有时钟的指针都指向$\red{12}$点即可施展魔法回到现在。

![img](./1822/file/4.png)

允许旋转时钟指针的方法有$\red{9}$种，每一种移动用一个数字号$\red{（1，2，…，9）}$表示。图中表示出$\red{9}$个数字号与相应的受控制的时钟，这些时钟在图中以灰色标出，其指针将顺时针旋转$\red{90}$度。

![img](./1822/file/5.png)

## 输入格式

由输入文件读$\red{9}$个数码，这些数码给出了$\red{9}$个时钟时针的初始位置。数码与时刻的对应关系为：

$\red{0→12}$点

$\red{1→3}$点

$\red{2→6}$点

$\red{3→9}$点

本题图例对应下列输入数据：

$\red{330}$

$\red{222}$

$\red{212}$

## 输出格式

将一个最短的移动序列（数字序列）写入输出文件中，该序列要使所有的时钟指针指向$\red{12}$点，若有等价的多个解，仅需给出其中一个。

## 样例

#### 输入样例

```c
330

222

212
```

#### 输出样例

```c
5489
```
