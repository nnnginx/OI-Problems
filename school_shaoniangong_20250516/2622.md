

# Description

通过Farmer John农场的漫长的道路上有 N 条人行横道，编号为 1 到 N（1 ≤ N ≤ 100000）。为了让奶牛通过这些人行横道，Farmer John安装交通信号灯，当允许奶牛过马路时，会显示绿色牛图标，否则为红色。不幸的是，一场大风暴损坏了他的一些信号。给出损坏信号的列表，请计算 Farmer John最少需要修复多少信号灯，才能存在至少 K 个连续的工作信号灯。

# Format

## Input

第一行输入包含 N，K 和 B（1 ≤ B，K ≤ N）。

接下来 B 行，每行描述一个断开的信号灯的编号。 .

## Output

请计算需要修复的信号的最小数量，以便在道路上的某处存在连续的 K 个工作信号灯。

# Samples

```input1
10 6 5 
2 
10 
1 
5 
9
```

```output1
1
```
