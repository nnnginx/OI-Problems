# 题目描述

一些整数可能拥有以下的性质：

性质 1：是偶数；
性质 2：大于 4 且不大于 12。
哈哈 喜欢这两个性质同时成立的整数；嘻嘻 喜欢这至少符合其中一种性质的整数；嗯嗯 喜欢刚好有符合其中一个性质的整数；嗯呐 喜欢不符合这两个性质的整数。现在给出一个整数 x，请问他们是否喜欢这个整数？

## 输入

输入一个整数x(0≤x≤1000)

## 输出

输出这 4个人是否喜欢这个数字，如果喜欢则输出 1，否则输出 0，用空格分隔。输出顺序为：哈哈、嘻嘻、嗯嗯、嗯呐

# 样例

```输入
12
```

```输出
1 1 0 0
```

# Limitation

1s, 1024KiB for each test case.
