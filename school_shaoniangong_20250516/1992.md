## 题目描述

$\red{Jasio }$是一个三岁的小男孩,他最喜欢玩玩具了,他有$\red{n }$个不同的玩具,它们都被放在了很高的架子上所以$\red{Jasio }$拿不到它们. 为了让他的房间有足够的空间,在任何时刻地板上都不会有超过$\red{k }$个玩具. $\red{Jasio }$在地板上玩玩具. $\red{Jasio}$'的妈妈则在房间里陪他的儿子. 当$\red{Jasio }$想玩地板上的其他玩具时,他会自己去拿,如果他想玩的玩具在架子上,他的妈妈则会帮他去拿,当她拿玩具的时候,顺便也会将一个地板上的玩具放上架子使得地板上有足够的空间. 他的妈妈很清楚自己的孩子所以他能够预料到$\red{Jasio }$想玩些什么玩具. 所以她想尽量的使自己去架子上拿玩具的次数尽量的少,应该怎么安排放玩具的顺序呢?

## 输入格式

第一行三个整数: $\red{n, k, p (1 <= k <= n <= 100.000, 1 <= p <= 500.000), }$分别表示玩具的总数,地板上玩具的最多个数以及$\red{Jasio }$他想玩玩具的序列的个数,接下来$\red{p}$行每行描述一个玩 具编号表示$\red{Jasio }$想玩的玩具.

## 输出格式

一个数表示$\red{Jasio }$的妈妈最少要拿多少次玩具.

## 样例

#### 输入样例

```c
3 2 7
1
2
3
1
3
1
2
```

#### 输出样例

```c
4
```

数据保证梯度
