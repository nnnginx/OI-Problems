## 题目描述

$\red{Farmer John}$最近发明了一个游戏，来考验自命不凡的贝茜。

游戏开始的时 候，$\red{FJ}$会给贝茜一块画着$\red{N (2 <= N <= 200)}$个不重合的点的木板，其中第$\red{i}$个点 的横、纵坐标分别为$\red{X_i}$和$\red{Y_i (-1,000 <= X_i <=1,000}$； $\red{-1,000 <= Y_i <= 1,000)}$。

贝茜可以选两个点画一条过它们的直线，当且仅当平面上不存在与画出直线 平行的直线。游戏结束时贝茜的得分，就是她画出的直线的总条数。

为了在游戏 中胜出，贝茜找到了你，希望你帮她计算一下最大可能得分。

## 输入格式

第$\red{1}$行: 输入$\red{1}$个正整数：$\red{N}$

第$\red{2..N+1}$行:

第$\red{i+1}$行用$\red{2}$个用空格隔开的整数$\red{X_i}$、$\red{Y_i，}$描述了点$\red{i}$的坐标

## 输出格式

第$\red{1}$行: 输出$\red{1}$个整数，表示贝茜的最大得分，即她能画出的互不平行的直线数

## 样例

#### 输入样例

```c
4
-1 1
-2 0
0 0
1 1

```




#### 输出样例

```c
4
```

## 提示
输出说明: 贝茜能画出以下$\red{4}$种斜率的直线：$\red{-1，}$$\red{0，}$$\red{1/3}$以及$\red{1}$。
