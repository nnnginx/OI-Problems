## 题目描述

某乡有$\red{N}$个村庄$\red{（1<N<40）}$,有一个售货员，他要到各个村庄去售货，各村庄之间的路程为$\red{S(0<S<1000)}$是已知的，且$\red{A}$村到$\red{B}$村与$\red{B}$村到$\red{A}$村的路大多不同。为了提高效率，他从商店出发到每个村庄一次，然后返回商店所在地，假设商店所在的村庄为$\red{1}$，他不知道选择什么样的路线才能使他所走的路线最短。请你帮他选择一条最短的路。

## 输入格式

村庄数$\red{N}$及各村之间的路程，均为整数。

## 输出格式

输出最短路程。

## 样例

#### 输入样例

```c
3 
0 2 1  
1 0 2  
2 1 0
```

#### 输出样例

```c
3
```

## 说明

输入样例各行代表的意义

$\red{3}$ {村庄数}

$\red{0~2 ~1}$  {村庄$\red{1}$到各村的路程}

$\red{1 ~0 ~2}$  {村庄$\red{2}$到各村的路程}

$\red{2 ~1~ 0}$  {村庄$\red{3}$到各村的路程}
