## 题目描述

给定一个 $\red {n}$ 个点 $\red {m}$ 条边的有向图，现在要求图中任意两点$\red {u}$和$\red {v}$，均可满足$\red {u}$能通往$\red {v}$或$\red {v}$能通往$\red {u}$，请你判断要求是否能够成立。

## 输入格式

第一行包含整数$\red {T}$，表示共有$\red {T}$组测试数据。

每组测试数据，第一行包含两个整数 $\red {n}$ 和 $\red {m}$。

接下来 $\red {m}$ 行，每行包含两个整数 $\red {u}$ 和 $\red {v}$，表示点 $\red {u}$ 和点 $\red {v}$ 之间存在一条有向边。

## 输出格式

每组数据输出一个结果，如果要求成立，则输出`“Yes”`，否则输出`“No”`。

## 样例

#### 输入样例

```c
1
3 3
1 2
2 3
3 1
```

#### 输出样例

```c
Yes
```

## 提示

$\red {0<n<1001}$,

$\red {m<6000}$
