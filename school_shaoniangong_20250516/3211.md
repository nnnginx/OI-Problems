## 题目背景


水中月是天上月，眼前人是心上人。

## 题目描述

BG 喜欢和 GB 一起在夜晚欣赏圆月。

聪慧的 BG 立马想到了一个问题：如果把眼前的图景看成一个平面，把天上的圆月看成点 $(x_1,y_1)$，把水面看成一条直线 $Ax+By+C=0$，那么这个点关于这条直线的**对称点** $(x_2,y_2)$（即月亮在水中的像）在哪里呢？

由于 BG 和 GB 的数学都很弱，请你帮他们计算一下。

## 输入格式

输入共有两行。

第一行 $2$ 个三位小数 $x_1, y_1$。

第二行 $3$ 个三位小数 $A, B, C$。

## 输出格式

输出一行， $2$ 个三位小数 $x_2, y_2$，向下取整。

## 样例 #1

### 样例输入 #1

```
0.000 1.000
1.000 1.000 0.000
```

### 样例输出 #1

```
-1.000 0.000
```
