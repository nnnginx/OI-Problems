## 题目描述

输入一个自然数$\red{N}$,对于一个最简分数$\red{a/b}$（分子和分母互质的分数）,满足$\red{1<=b<=N,0<=a/b<=1}$,请找出所有满足条件的分数。

这有一个例子，当$\red{N=5}$时，所有解为：

$\red{0/1,1/5,1/4,1/3,2/5,1/2,3/5,2/3,3/4,4/5,1/1}$

给定一个自然数$\red{N(1<=N<=160)}$，请编程按分数值递增的顺序输出所有解。

注：

$\red{1}$、$\red{0}$ 和任意自然数的最大公约数就是那个自然数。

$\red{2}$、互质指最大公约数等于$\red{1}$的两个自然数。

## 输入格式

单独的一行一个自然数 $\red{n}$

## 输出格式

每个分数单独占一行，按照大小次序排列

## 样例

#### 输入样例

```c
5
```

#### 输出样例

```c
0/1
1/5
1/4
1/3
2/5
1/2
3/5
2/3
3/4
4/5
1/1
```
