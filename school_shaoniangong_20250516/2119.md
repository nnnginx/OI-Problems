## 题目描述

农民$\red{John}$有$\red{N}$头牛需要挤奶$\red{(1 <= N <= 10,000)，}$每头牛只需要一个单位时间挤奶。
有些奶牛是没有耐心的动物，如果农夫约翰等太久才给它们挤奶，它们就会拒绝挤奶。更具体地说，奶牛$\red{i}$生产$\red{g_i}$加仑的牛奶$\red{(1 <= g_i <= 1000)，}$但只有在最后期限$\red{d_i (1 <= d_i <= 10,000)}$之前挤奶。时间从$\red{t=0}$开始，所以在截止时间$\red{t=x}$之前，最多可以挤奶$\red{x}$头奶牛。

请帮助农夫约翰确定如果他以最佳的方式挤牛奶，他可以获得的最大牛奶量。

$\red{FJ}$有$\red{N(1 <= N <= 10,000)}$头牛要挤牛奶，每头牛需要花费$\red{1}$单位时间。

奶牛很厌烦等待，奶牛$\red{i}$在它的截止时间$\red{d_i (1 <= d_i <= 10,000)}$前挤$\red{g(1 <= g_i <= 1000)}$的 奶，否则将不能挤奶。时间$\red{t}$开始时为$\red{0，}$即在时间$\red{t=x}$时，最多可以挤$\red{x}$头奶牛。

请计算$\red{FJ}$的最大挤奶量。

## 输入格式

第一行:$\red{N}$的值。

第$\red{2 . .1+N:}$第$\red{i+1}$行包含整数$\red{g_i}$和$\red{d_i}$。

## 输出格式

第一行:农夫约翰所能获得的最大牛奶加仑数。

## 样例

#### 输入样例

```c
4
10 3
7 5
8 1
2 1
```

#### 输出样例

```c
25
```
