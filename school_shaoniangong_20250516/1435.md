## 题目描述


如果一个只包含大写字母的非空字符串旋转180度和原串一模一样，那么就叫它`Ambigram串`。

比如“`MOW`”，“`XXXX`”，“`HXHXH`”等。

要求的就是把一个任意的只包含大写字母的非空字符串修改成`Ambigram串`最少要多少花费。

操作有以下两种：

- `1`）	删除一个字符，需要的花费为这个字符与'`A`'和'`Z`'的差的较小值加1。
- `2`）	修改一个字符，需要的花费为两个字符的差。

旋转一个仍然和本身一样的字符有：`H,X,O,Z,N,I,S`

$\red M$旋转以后是$\red W$，$\red W$旋转以后是$\red M$。

## 输入格式

输入只包含一行字符串，表示需要被修改的串。

## 输出格式

输出只有一行，表示修改后的`Ambigram串`。

要保证花费最小。

如果答案有多个输出较长的，如果有多个一样长的输出字典序最小的。

## 样例

#### 输入样例

```C
XIXHZMOAOSHXIX
```

#### 输出样例

```C
XIXHMOOWHXIX
```

## 提示

输入字符串长度小于等于$\red {50}$。
