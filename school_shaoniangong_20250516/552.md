## 题目描述

如果一个数从左往右读和从右往左读都是一样，那么这个数就叫做“回文数”。例如，$\red{12321}$就是一个回文数，而$\red{77778}$就不是。当然，回文数的首和尾都应是非零的，因此$\red{0220}$就不是回文数。

事实上，有一些数（如$\red{21}$），在十进制时不是回文数，但在其它进制（如二进制时为$\red{10101}$）时就是回文数。

编一个程序，读入两个十进制数$\red{N (1 <= N <= 15)S (0 < S < 10000)}$然后找出前$\red{N}$个满足大于$\red{S}$且在两种或两种以上进制（二进制至十进制）上是回文数的十进制数，输出到文件上。

本问题的解决方案不需要使用大于$\red{32}$位的整型

## 输入格式

只有一行，用空格隔开的两个数$\red{N}$和$\red{S}$。

## 输出格式

$\red{N}$行, 每行一个满足上述要求的数，并按从小到大的顺序输出。

## 样例

#### 输入样例

```c
3 25
```

#### 输出样例

```c
26
27
28
```
