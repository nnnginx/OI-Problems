## 题目描述

今年是国际数学联盟确定的“$\red{2000}$`DD`世界数学年”，又恰逢我国著名数学家华罗庚先生诞辰 $\red{90}$周年。在华罗庚先生的家乡江苏金坛，组织了一场别开生面的数学智力竞赛的活动，你的一个好朋友`XZ `也有幸得以参加。活动中，主持人给所有参加活动的选手出了这样一道题目：

设有一个长度为 $\red{N}$ 的数字串，要求选手使用 $\red{K}$ 个乘号将它分成 $\red{K+1}$ 个部分，找出一种分法，使得这 $\red{K+1}$ 个部分的乘积能够为最大。

同时，为了帮助选手能够正确理解题意，主持人还举了如下的一个例子：

有一个数字串：$\red{312}$， 当 $\red{N=3,K=1}$ 时会有以下两种分法：

- $\red{3 \times 12=36}$
- $\red{31 \times 2=62}$

这时，符合题目要求的结果是: $\red{31 \times 2 = 62}$。

现在，请你帮助你的好朋友 `XZ` 设计一个程序，求得正确的答案。

## 输入格式

第一行共有两个自然数 $\red{N,K}$。

第二行是一个长度为 $\red{N}$ 的数字串。

## 输出格式

结果显示在屏幕上，相对于输入，应输出所求得的最大乘积（一个自然数）。

## 样例

#### 输入样例

```c
4 2
1231
```

#### 输出样例

```c
62
```

## 提示

$\red{6\le N\le 40,1\le K\le 6}$。
