## 题目描述

有一个正方形，四个角的坐标（ $\red{x}$,$\red{y}$ )
分别是($\red{1}$，$\red{-1}$)，($\red{1}$，$\red{1}$)，($\red{-1}$，$\red{-1}$)，($\red{-1}$，$\red{1}$)，

$\red{x}$是横轴，$\red{y}$是纵轴。写一个程序，判断一个给定的点是否在这个正方形内(包括正方形边界)。

如果点在正方形内，则输出`yes`，否则输出`no`。

## 输入格式

输入一行，包括两个整数$\red{x}$、$\red{y}$，以一个空格分开，表示坐标($\red{x}$,$\red{y}$)。

## 输出格式

输出一行，如果点在正方形内，则输出`yes`，否则输出`no`。

## 样例

#### 输入数据

```C
1 1
```

#### 输出数据

```C
yes
```
