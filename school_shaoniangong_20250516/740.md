## 题目描述

一个 $\red{n}$ 行 $\red{n}$ 列的螺旋矩阵可由如下方法生成：

从矩阵的左上角（第 $\red{1}$ 行第 $\red{1}$ 列）出发，初始时向右移动；如果前方是未曾经过的格子，则继续前进，否则右转；重复上述操作直至经过矩阵中所有格子。根据经过顺序，在格子中依次  填入 $\red{1, 2, 3,\sim , n}$ ，便构成了一个螺旋矩阵。

下图是一个 $\red{n = 4}$  时的螺旋矩阵。

![img](./740/file/1.jpg)

现给出矩阵大小 $\red{n}$ 以及 $\red{i}$ 和 $\red{j}$，请你求出该矩阵中第 $\red{i}$ 行第 $\red{j}$ 列的数是多少 。

## 输入格式

输入共一行，包含三个整数$\red{n，i，j}$，每两个整数之间用一个空格隔开，分别表示矩阵大小、待求的数所在的行号和列号。

## 输出格式

输出共一行，包含一个整数，表示相应矩阵中第 $\red{i}$ 行第 $\red{j}$ 列的数。

## 样例

#### 输入样例

```
4 2 3
```

#### 输出样例

```
14
```

## 数据范围与提示

$对于 \red{50\%的数据，1  \leq  n  \leq  100;}$

$对于 \red{100\%的数据，1  \leq  n  \leq  30,000，1  \leq  i  \leq  n，1  \leq  j  \leq  n。}$
