## 题目描述

农民约翰正在建造一个景观优美的花园，在这个过程中需要移走大量的泥土。

花园由$\red{N}$个花坛组成$\red{（1≤}$$\red{N≤}$$\red{100000）}$，其中花坛$\red{i}$最初包含$\red{Ai}$单位的污垢。农民约翰想重新美化花园，使每个花坛都有两个单位的泥土。$\red{Ai}$和$\red{Bi}$都是$\red{0…}$$\red{10}$范围内的整数。

为了美化花园，农民约翰有几种选择：他可以购买一个单位的泥土，并将其放置在他选择的花坛上，花费$\red{X}$个单位的钱。他可以从自己选择的花坛上取下一个单位的污垢，然后用$\red{Y}$个单位的钱把它运 走。他还可以将一个单位的泥土从花坛$\red{i}$运送到花坛$\red{j，}$费用为$\red{Z}$乘以$\red{|i-j |}$。请计算农民约翰完成绿化项目的最低总成本。

## 输入格式

第一行输入包含$\red{N}$、$\red{X}$、$\red{Y}$和$\red{Z（}$$\red{0≤}$$\red{X}$、 $\red{Y≤}$$\red{10^8}$;$\red{0≤}$$\red{Z≤}$$\red{1000). }$第$\red{i+1}$行包含整数$\red{Ai}$和$\red{Bi}$。

## 输出格式

请打印$\red{FJ}$需要花费在绿化上的最低总成本。

## 样例

#### 输入样例

```c
4 100 200 1
1 4
2 3
3 2
4 0
```

#### 输出样例

```c
210
```
