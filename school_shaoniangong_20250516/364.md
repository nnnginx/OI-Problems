## 题目描述

将整数 $\red{n}$ 分成 $\red{k}$ 份，且每份不能为空，问有多少种不同的分法。当 $\red{n=7, k=3}$ 时，下面三种分法被认为是相同的：

$\red{1,1,5}$;

$\red{1,5,1}$;

$\red{5,1,1}$;

## 输入格式

一行两个数 $\red{n}$ , $\red{k}$。

## 输出格式

一行一个整数，即不同的分法数。

## 样例

#### 输入样例

```c
7 3
```

#### 输出样例

```c
4
```

四种分法为：$\red{1,1,5；1,2,4；1,3,3；2,2,3}$。

## 提示

$\red{6 \leq n \leq 200, 2 \leq k \leq 6}$。
