## 题目描述

小渊是个聪明的孩子，他经常会给周围的小朋友讲些自己认为有趣的内容。最近，他准备给小朋友讲解立体图，请你帮他画出立体图。

小渊有一块面积为$\red{m\times n}$的矩形区域，上面有$\red{m\times n}$个边长为$\red{1}$的格子，每个格子上堆了一些同样大小的积木（积木的长宽高都是$\red{1}$），小渊想请你打印出这些格子的立体图。我们定义每个积木为如下格式，并且不会做任何翻转旋转，只会严格以一种形式摆放。

```c
+---+
 /   /|高
+---+ |
|   | +
|   |/宽
+---+
 长
```

每个顶点用$\red{1}$个加号$\red{"+"}$表示,长用$\red{3}$个$\red{"-"}$表示,宽用$\red{1}$个$\red{"/"}$表示,高用两个$\red{"|"}$表示。符号$\red{"+","-","/","|"}$的$\red{ASCII}$码分别是$\red{43,45,47,124}$。字符$\red{"."}$($\red{ASCII}$码$\red{46}$)需要作为背景输出，即立体图中的空白部分需要用$\red{"."}$来代替。立体图的画法如下面的规则：

若两块积木左右相邻，图示为：

```c
..+---+---+  
./   /   /|  
+---+---+ |  
|   |   | +  
|   |   |/.  
+---+---+..
```

若两块积木上下相邻，图示为：

```c
..+---+  
./   /|  
+---+ |  
|   | +  
|   |/|  
+---+ |  
|   | +  
|   |/.  
+---+..
```

若两块积木前后相邻，图示为：

```c
....+---+
.../   /|
..+---+ |
./   /| +
+---+ |/.
|   | +..
|   |/...
+---+....
```

立体图中，定义位于第$\red{(m,1)}$的格子（即第$\red{m}$行第$\red{1}$列的格子）上面的自底向上的第一块积木（即最下面的一块积木）的左下角顶点为整张图最左下角的点。

## 输入格式

第一行有用空格隔开的$\red{2}$个整数$\red{m}$和$\red{n}$，表示有$\red{m\times n}$个格子$\red{（1≤m,n≤50)}$。

接下来的$\red{m}$行，是一个$\red{m\times n}$的矩阵，每行有$\red{n}$个用空格隔开的整数，其中第$\red{i}$行第$\red{j}$列上的整数表示第$\red{i}$行第$\red{j}$列的格子上摞有多少个积木（$\red{1≤}$每个格子上的积木数$\red{≤100}$)。

## 输出格式

包含题目要求的立体图，是一个$\red{K}$行$\red{L}$列的字符矩阵，其中$\red{K}$和$\red{L}$表示最少需要$\red{K}$行$\red{L}$列才能按规定输出立体图。

## 样例

#### 输入样例

```c
3 4 
2 2 1 2 
2 2 1 1 
3 2 1 2
```

#### 输出样例

```c
......+---+---+...+---+
..+---+  /   /|../   /|
./   /|-+---+ |.+---+ |
+---+ |/   /| +-|   | +
|   | +---+ |/+---+ |/|
|   |/   /| +/   /|-+ |
+---+---+ |/+---+ |/| +
|   |   | +-|   | + |/.
|   |   |/  |   |/| +..
+---+---+---+---+ |/...
|   |   |   |   | +....
|   |   |   |   |/.....
+---+---+---+---+......
```
