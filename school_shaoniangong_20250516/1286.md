## 题目描述

给你一个正整数$\red{n}$，要求你找到最小的$\red{x}$（$\red{x>0}$）满足$\red{2^x\ mod\ n\ =\ 1}$。

## 输入格式

输入包含多组测试数据。每行一个正整数，代表$\red{n}$的值。

## 输出格式

如果最小的$\red{x}$存在，则输出$\red{2^x \ mod\  n\  =\  1}$（注意$\red x$和$\red n$要用具体的值代替），否则输出$\red{2^?\ \ mod\ \ n\ \ =\ 1}$。

## 样例

#### 输入样例

```C
2
5
```

#### 输出样例

```C
2^? mod 2 = 1
2^4 mod 5 = 1
```
