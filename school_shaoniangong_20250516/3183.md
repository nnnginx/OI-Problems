#### 题目限制

1000 ms    256 M

#### 题目描述

给出一个只包含 $G$ 和 $H$ 的字符串，求有多少个长度大于等于 $3$ 的子串（连续的），只包含 $1$ 个 $H$ 或 只包含 $1$ 个 $G$ 。

#### 输入格式

第一行：一个数 $N$ 表示字符串的长度。（ $N\le 500000$ ）
第二行：一个长为 $N$ 的字符串。数据保证该字符串中只包括 $H$ 和 $G$ 。

#### 输出格式

输出长度大于等于 $3$ ，且只包含 $1$ 个 $H$ 或 只包含 $1$ 个 $G$ 的子串数量。

#### 数据范围

对于 $36\%$ 的数据， $N \le 50$ ；

对于 $91\%$ 的数据， $N \le 5000$ ；

对于 $100\%$ 的数据， $N \le 500000$ 。

#### 输入样例 1

```
5
GHGHG
```

#### 输出样例 1

```
3
```
