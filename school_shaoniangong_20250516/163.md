## 题目描述

给定一个有 $\red N$ 个点（编号 $\red 0 $ , $\red 1$ , $\red{......}$ , $\red{N-1}$）的树，每条边都有一个权值（不超过 $\red{1000}$）。

树上两个节点 $\red x$ 与 $\red y$ 之间的路径长度就是路径上各条边的权值之和。

求长度不超过 $\red K$ 的路径有多少条。

## 输入格式

输入包含多组测试用例。

每组测试用例的第一行包含两个整数 $\red N$ 和 $\red K$。

接下来 $\red{N-1}$ 行，每行包含三个整数 $\red u$ , $\red v$ , $\red l$，表示节点 $\red u$ 与 $\red v$ 之间存在一条边，且边的权值为 $\red l$。

当输入用例 $\red{N=0}$，$\red{K=0}$ 时，表示输入终止，且该用例无需处理。

## 输出格式

每个测试用例输出一个结果。

每个结果占一行。

## 样例

#### 输入样例

```c
5 4
0 1 3
0 2 1
0 3 2
2 4 1
0 0
```

#### 输出样例

```c
8
```


## 提示

$\red{N\le 10000}$
