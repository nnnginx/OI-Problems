## 题目描述

有一个取数的游戏。初始时，给出一个环，环上的每条边上都有一个非负整数。这些整数中至少有一个$\red{0}$。然后，将一枚硬币放在环上的一个节点上。两个玩家就是以这个放硬币的节点为起点开始这个游戏，两人轮流取数，取数的规则如下：
（$\red{1）}$选择硬币左边或者右边的一条边，并且边上的数非$\red{0}$；
（$\red{2）}$将这条边上的数减至任意一个非负整数$\red{(}$至少要有所减小$\red{)}$；
（$\red{3）}$将硬币移至边的另一端。
如果轮到一个玩家走，这时硬币左右两边的边上的数值都是$\red{0，}$那么这个玩家就输了。



如下图，描述的是$\red{Alice}$和$\red{Bob}$两人的对弈过程，其中黑色节点表示硬币所在节点。结果图$\red{(d)}$中，轮到$\red{Bob}$走时，硬币两边的边上都是$\red{0，}$所以$\red{Alcie}$获胜。

![img](./1976/file/1.png)

现在，你的任务就是根据给出的环、边上的数值以及起点（硬币所在位置），判断先走方是否有必胜的策略。

## 输入格式

第一行一个整数$\red{N（}$$\red{N≤}$$\red{20）}$，表示环上的节点数。
第二行$\red{N}$个数，数值不超过$\red{30，}$依次表示$\red{N}$条边上的数值。硬币的起始位置在第一条边与最后一条边之间的节点上。

## 输出格式

仅一行。若存在必胜策略，则输出"$\red{YES}$"，否则输出"$\red{NO}$"。

## 样例

#### 输入样例1

```c
4
2 5 3 0
```

#### 输出样例1

```c
YES
```

#### 输入样例2

```c
3
0 0 0
```

#### 输出样例2

```c
NO
```
