## 题目描述

文件名:"lango.in/lango.out"

$\red{Lagno }$是一种二人智力游戏。游戏设有一个黑方和一个白方。游戏桌面是正方形的，包
含 $\red{8 }$行 $\red{8 }$列。

如果黑方玩家走出这样一步棋：将一枚黑子放在任一空格上，而在这个空格的八个方
向（上、下、左、右和 $\red{4 }$个对角线方向）的至少一个方向上有一排白子被夹在这枚新下的
黑子和其他黑子之间，任何方向，在新黑子和原来黑子之间的所有白子都要变成黑子。

为这个游戏设计一个程序，计算一步棋中黑方能转变的白子数量的最大值。

## 输入格式

共 $\red{8 }$行

每行 $\red{8 }$个字符；"."代表一个空格；"$\red{B}$"代表黑子，"$\red{W}$"代表白子

## 输出格式

共一行，有一个整数，表示一步中黑方能吃掉白子的最大数，如果

无法吃掉就输出"$\red{0}$"。

## 样例

#### 输入样例1

```c
........
........
........
...BW...
...WB...
........
........
........
```

#### 输出样例1

```c
1
```

#### 输入样例2

```c
........
........
...BB...
...B....
..BBW...
..WWW...
....WB..
........
```

#### 输出样例2

```c
2
```
