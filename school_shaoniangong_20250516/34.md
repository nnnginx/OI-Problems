## 题目描述

格格兰郡的$\red N$名士兵随机散落在全郡各地。

格格兰郡中的位置由一对($\red {x,y}$)整数坐标表示。

士兵可以进行移动，每次移动，一名士兵可以向上，向下，向左或向右移动一个单位（因此，他的$\red x$或$\red y$坐标也将加1或减1）。

现在希望通过移动士兵，使得所有士兵彼此相邻的处于同一条水平线内，即所有士兵的$\red y$坐标相同并且$\red x$坐标相邻。

请你计算满足要求的情况下，所有士兵的总移动次数最少是多少。

需注意，两个或多个士兵不能占据同一个位置。

## 输入格式

第一行输入整数$\red N$，代表士兵的数量。

接下来的N行，每行输入两个整数$\red x$和$\red y$，分别代表一个士兵所在位置的x坐标和y坐标，第$\red i$行即为第$\red i$个士兵的坐标($\red {x[i],y[i]}$)。

## 输出格式

输出一个整数，代表所有士兵的总移动次数的最小值。

## 样例

#### 输入样例

```C
5
1 2
2 2
1 3
3 -2
3 3
```

#### 输出样例

```C
8
```

## 提示

$\red {1≤N≤10000,−10000≤x[i],y[i]≤10000}$
