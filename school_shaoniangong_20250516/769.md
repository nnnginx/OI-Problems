## 题目描述

小凯手中有两种面值的金币，两种面值均为正整数且彼此互素。每种金币小凯都有无数个。在不找零的情况下，仅凭这两种金币，有些物品他是无法准确支付的。现在小凯想知道在无法准确支付的物品中，最贵的价值是多少金币？注意：输入数据保证存在小凯无法准确支付的商品。

## 输入格式

输入数据仅一行，包含两个正整数 $\red{a}$ 和 $\red{b}$，它们之间用一个空格隔开，表示小凯手中金币的面值。

## 输出格式

输出文件仅一行，一个正整数 $\red{N}$，表示不找零的情况下，小凯用手中的金币不能准确支付的最贵的物品的价值。

## 样例

#### 样例输入

```C
3 7
```

#### 样例输出

```C
11
```

#### 样例说明

小凯手中有面值为$\red{3}$和$\red{7}$的金币无数个，在不找零的前提下无法准确支付价值为 $\red{1、2、4、5、8、11}$的物品，其中最贵的物品价值为$\red{11}$。
比$\red{11}$贵的物品都能买到，比如：

- $\red{ 12 = 3 \times 4 + 7 \times 0}$
- $\red{13 = 3 \times 2 + 7 \times1}$
- $ \red{14 = 3 \times 0 + 7 \times 2} $
- $ \red{15 = 3 \times 5 + 7 \times 0}$

## 提示

对于 $\red{30\%}$ 的数据：$\red{ 1 \le a,b \le 50}$;

对于 $\red{60\%}$ 的数据： $\red{1 \le a,b \le 10,000}$;

对于 $\red{100\%}$ 的数据：$\red{1 \le a,b \le 1,000,000,000}$。
