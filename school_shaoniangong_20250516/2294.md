## 题目描述

给一个长为 $\red{n}$ 的字符串，每当有两个元音字母相邻时，删除后一个元音，输出修改后的字符串。 $\red{（a,e,i,o,u,y为元音字母） }$

## 输入格式

共两行。

第一行一个正整数 表示字符串长度$\red{n}$。

第二行一个长度为由小写字母构成的字符串 。

## 输出格式

一个字符串表示答案。

## 样例

#### 输入样例1

```c
5
weird
```

#### 输出样例1

```c
werd
```

#### 输入样例2

```c
5
aaeaa
```

#### 输出样例2

```c
a
```

#### 输入样例3

```c
4
word
```

#### 输出样例3

```c
word
```

## 提示

1s, 1024KiB for each test case.

对于$\red{30\%}$的数据，$\red{1 <= n <= 10}$

对于$\red{100\%}$的数据,$\red{1 <= n <=100}$
