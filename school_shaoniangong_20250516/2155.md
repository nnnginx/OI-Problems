## 题目描述


小明的班级里有 $n$ 个人，第 $i$ 个人的能力值为 $a_i$。他们会膜拜所有能力值比他高 $m$ 或以上的人。求所有人膜拜人数总和是多少。

## 输入格式

第一行两个正整数 $n,m$。

第二行 $n$ 个正整数 $a_i$。

## 输出格式

输出所有人膜拜人数总和。

## 样例 #1

### 样例输入 #1

```
6 2
1 4 2 8 5 7
```

### 样例输出 #1

```
12
```

## 提示

第一个人膜拜 $2,4,5,6$；

第二个人膜拜 $4,6$；

第三个人膜拜 $2,4,5,6$；

第四个人不会膜拜任何人；

第五个人膜拜 $4,6$；

第六个人不会膜拜任何人。

对于 $55\%$ 的数据，$1\le n\le 2\times10^4$。

对于 $100\%$ 的数据，$1\le n\le 2\times10^5,1\le m,a_i<2^{30}$。
