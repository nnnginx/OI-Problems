## 题目描述

给出圆的半径，求圆的直径、周长和面积。

输入圆的半径实数$\red{r}$，输出圆的直径、周长、面积，每个数保留小数点后$\red{4}$位。圆周率取值为$\red{3.14159}$。

## 输入格式

输入包含一个实数$\red{r}$（$\red{0 < r ≤ 10,000 }$ ），表示圆的半径。

## 输出格式

输出一行，包含三个数，分别表示圆的直径、周长、面积，数与数之间以一个空格分开，每个数保留小数点后$\red{4}$位。

## 样例

#### 输入数据

```C
3.0
```

#### 输出数据

```C
6.0000 18.8495 28.2743
```
