## 题目描述

有一种奇怪的语言叫做“二五语言”.它的每个单词都由 $\red{A \sim Y}$这 $\red{25}$个字母各一个组成.合法的
二五语言单词必须满足这样一个条件:把它的 $\red{25}$ 个字母排成一个 $\red{5 \times 5}$ 的矩阵,它的每一行和每一列都必须是递增的.比如单词 $\red{ACEPTBDHQUFJMRWGKNSXILOVY}$,它排成的矩阵如下所示：

<pre>
A  C  E  P  T
B  D  H  Q  U
F  J  M  R  W
G  K  N  S  X
I  L  O  V  Y
</pre>

它每行每列都是递增的,是一个合法的单词.而 $\red{YXWVUTSRQPONMLKJIHGFEDCBA}$ 则不合法.
由于单词太长存储不便,需要给每一个单词编一个码.编码方法如下:写出单词中字母 $\red A$ 在矩阵中的
行号和列号,再写出 $\red B$ 的行号和列号……依此类推.得到的序列叫做行列序列.如上面的合法单词的
行列序列是:$\red{11\ 21\ 12\ 22\ 13\ 31\ 41\ 23\ 51\ 32\ 42\ 52\ 33\ 43\ 53\ 14\ 24\ 34\ 44\ 15\ 25\ 54\ 35\ 45\ 55}$.

然后把所有合法单词的行列序列排序.一个单词的行列序列排序后的位置,就是这个单词的编码.比
如,单词 $\red{ABCDEFGHIJKLMNOPQRSTUVWXY}$ 的编码为 $\red 1$,而单词 $\red{ABCDEFGHIJKLMNOPQRSUTVWXY}$ 的编码为 $\red 2$.

现在,你需要编一个程序,完成单词与编码间的转换.

## 输入格式

第一行为一个字母 $\red N$ 或 $\red W$.

$\red N$ 表示把编码转换为单词,$\red W$ 表示把单词转换为编码.

若第一行为 $\red N$,则第二行为一个整数,表示单词的编码.若第一行为 $\red W$,则第二行为一个合法的单词.

## 输出格式

每行一个整数或单词.

## 样例

#### 输入样例1

```c
N 
2
```

#### 输出样例1

```c
ABCDEFGHIJKLMNOPQRSUTVWXY
```

#### 输入样例2

```c
W 
ABCDEFGHIJKLMNOPQRSUTVWX
```

#### 输出样例2

```c
2
```
