## 题目描述

为了使电文保密，往往按一定规律将其转换成密文，收报人再按约定的规律将其译回原文。现按以下规律将电文变成密文：

将字母$\red{A}$变成$\red{E}$，$\red{a}$变成$\red{e}$，即变成其后的第四个字母，$\red{W}$变成$\red{A}$，$\red{X}$变成$\red{B}$，$\red{Z}$变成$\red{D}$，非字母字符不变。编程输入一行文字(包含空格），将其变成相应密文。

## 输入格式

长度不超过$\red{1000000}$字符的原文。

## 输出格式

密文。

##样例

#### 输入样例

```c
China!
```

#### 输出样例

```c
Glmre!
```
