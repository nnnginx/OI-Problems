## 题目描述

写一程序读入三角形三个边$\red{a}$,$\red{b}$,$\red{c}$,计算三角形的面积$\red{S}$，

可利用下面公式计算：

$$
\red{
S=\sqrt {p(p-a)(p-b)(p-c)}
}
$$

其中：$\red{p=(a+b+c)/2}$

## 输入格式

共一行，三个数$\red{a}$,$\red{b}$,$\red{c}$，表示三角形的三边长

## 输出格式

判断能否构成三角形，如果能则输出三角形的面积（保留小数点后二位），否则输出“`No Solution.`”

## 样例

#### 输入数据1

```C
3 4 5
```

#### 输出数据1

```C
6.00
```

#### 输入数据2

```C
3 4 1
```

#### 输出数据2

```C
No Solution.
```
