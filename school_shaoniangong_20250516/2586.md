## 题目描述

$\red{John}$在他的农场中闲逛时发现了许多虫洞。虫洞可以看作一条十分奇特的有向边，并可以使你返回到过去的一个时刻（相对你进入虫洞之前）。

$\red{John}$的每个农场有$\red{M}$条小路（无向边}连接着$\red{N }$（从$\red{1..N}$标号）块地，并有$\red{W}$个虫洞。其中$\red{1<=N<=500,1<=M<=2500,1<=W<=200}$。 现在$\red{John}$想借助这些虫洞来回到过去（出发时刻之前），请你告诉他能办到吗。

$\red{John}$将向你提供$\red{F(1<=F<=5)}$个农场的地图。没有小路会耗费你超过$\red{10000}$秒的时间，当然也没有虫洞回帮你回到超过$\red{10000}$秒以前。

## 输入格式

第一行: 一个整数 $\red{F, }$表示农场个数。

每个农场的第一行: 三个整数 $\red{N, M, W}$。

每个农场的$\red{2..M+1 }$行: 三个数$\red{（S, E, T）}$。表示在标号为$\red{S}$的地与标号为$\red{E}$的地中间有一条用时$\red{T}$秒的小路。

每个农场的$\red{M+2..M+W+1}$行: 三个数$\red{（S, E, T）}$。表示在标号为$\red{S}$的地与标号为$\red{E}$的地中间有一条可以使$\red{John}$到达$\red{T}$秒前的虫洞。

## 输出格式

第$\red{1..F}$行: 如果$\red{John}$能在这个农场实现他的目标，输出"$\red{YES}$",否则输出"$\red{NO}$"。

## 样例

#### 输入样例

```c
2
3 3 1
1 2 2
1 3 4
2 3 1
3 1 3
3 2 1
1 2 3
2 3 4
3 1 8
```

#### 输出样例

```c
NO
YES
```
