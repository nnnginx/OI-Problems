## 题目描述

佩奇要吃 $\red{x}$块鸡块，有$\red{3 }$块一盒的小鸡块和 $\red{7}$块一盒的大鸡块，能不能买恰好$\red{x }$块鸡块？

## 输入格式

每个测试点包含多组数据

第一行一个正整数 $\red{T}$表示数据组数。

对于每组数据，共一行，第一行一个整数 $\red{x(1≤x≤100)}$如上所述。

## 输出格式

对于每组数据，输出"$\red{YES}$"或"$\red{NO}$"表示答案

## 样例

#### 输入样例

```c
2
6
5
```

#### 输出样例

```c
YES 
NO
```

## 提示

对于$\red{50\%}$的数据，有$\red{1≤}$$\red{x≤}$$\red{20}$;

对于$\red{100\%}$的数据，有$\red{1≤}$$\red{x≤}$$\red{100}$。
