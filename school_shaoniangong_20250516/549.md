## 题目描述

一块$\red{N \times N（1<=N<=10）}$正方形的黑白瓦片的图案要被转换成新的正方形图案。写一个程序来找出将原始图案按照以下列转换方法转换成新图案的最小方式：

- $\red{1}$：转$\red{90}$度：图案按顺时针转$\red{90}$度。
- $\red{2}$：转$\red{180}$度：图案按顺时针转$\red{180}$度。
- $\red{3}$：转$\red{270}$度：图案按顺时针转$\red{270}$度。
- $\red{4}$：反射：图案在水平方向翻转（以中央铅垂线为中心形成原图案的镜像）。
- $\red{5}$：组合：图案在水平方向翻转，然后再按照$\red 1$到$\red 3$之间的一种再次转换。
- $\red{6}$：不改变：原图案不改变。
- $\red{7}$：无效转换：无法用以上方法得到新图案。

如果有多种可用的转换方法，请选择序号最小的那个。

只使用$\red{1\sim 7}$中的一个步骤来完成这次转换。

## 输入格式

第一行： 单独的一个整数$\red{N}$。

第二行到第$\red{N+1}$行： $\red{N}$行每行$\red{N}$个字符（不是$\red{“@”}$就是$\red{“-”}$）；这是转换前的正方形。

第$\red{N+2}$行到第$\red{2\times N+1}$行： $\red{N}$行每行$\red{N}$个字符（不是$\red{“@”}$就是$\red{“-”}$）；这是转换后的正方形。

## 输出格式

单独的一行包括$\red{1}$到$\red{7}$之间的一个数字（在上文已描述）表明需要将转换前的正方形变为转换后的正方形的转换方法。

## 样例

#### 输入样例

```c
3
@-@
---
@@-
@-@
@--
--@
```

#### 输出样例

```c
1
```
