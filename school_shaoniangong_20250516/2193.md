## 题目描述

农民约翰的$\red{N}$头奶牛
$\red{(1 <= N <= 50,000)}$似乎在蹒跚而行
$\red{FJ}$农场前面的路，但他们实际上只是在跑
看哪头奶牛跑得最快。

从上面看，每头奶牛都由一个单位长度的水平面表示
线段，由其左角点的坐标指定
时间$\red{t=0}$。例如，（$\red{-3,6）}$将指定在$\red{t=0}$时
由从（$\red{-3,6）}$到（$\red{-2,6）}$的段表示。每头奶牛都在向
右侧（在$\red{+x}$方向）以特定速率，由
她向右移动$\red{1}$个单位所需的整数时间。

$\red{FJ}$对他的奶牛在外面奔跑并不特别兴奋
而不是在谷仓里生产牛奶。他计划用
比赛结束后的严厉演讲。为了确定
他的奶牛正在参加比赛，$\red{FJ}$将自己定位在（$\red{0,0）}$
并沿$\red{+y}$方向延伸的光线进行观察。作为比赛
展开后，如果$\red{FJ}$是这条路上第一头看到的奶牛，她会看到一头奶牛
射线也就是说，如果另一头牛在"前面"，则可能看不到一头牛
在整个过程中，她跨越了$\red{FJ}$的视线。

请计算$\red{FJ}$在整个比赛中能看到的奶牛数量。

## 输入格式

输入的第一行包含$\red{N}$行。以下$\red{N}$行中的每一行
描述具有三个整数$\red{x y r}$的$\red{cow，}$对应于$\red{cow}$
其左端点在时间$\red{t=0}$时位于（$\red{x，}$$\red{y）}$，在$\red{a}$处向右移动
每$\red{r}$个时间单位的$\red{1}$个距离单位的连续速度。这个
$\red{x}$的值在$\red{-1000}$范围内$\red{-1，}$$\red{y}$的值在范围内
$\red{1...1000000（}$每头奶牛都不同，以防止任何可能的
碰撞），并且$\red{r}$的值在$\red{1...}$$\red{1,000,000.}$

## 输出格式

单个整数，指定$\red{FJ}$在测试过程中可以看到的奶牛数量
整个比赛（从$\red{t=0}$开始）。

## 样例

#### 输入样例

```c
3
-2 1 3
-3 2 3
-5 100 1
```

#### 输出样例

```c
2
```

## 提示

$\red{FJ}$可以看到奶牛$\red{1}$和$\red{2，}$但不能看到奶牛$\red{3}$。
