## 题目描述

`sgu223`

修罗王准备将他的猛兽军团放置在$\red{N\times N}$的方阵中,但是猛兽们均有自己的地盘,它们
会攻击自身周围八个格子的任何目标,现猛兽有$\red{K}$只,要求猛兽之间不能互相攻击,问有多
少种可行方案?

## 输入格式

两个整数$\red{N(1≤}$$\red{N≤}$$\red{10)}$和$\red{K(0≤}$$\red{K≤}$$\red{N^2)}$。

## 输出格式

可行放置方案个数。

## 样例

#### 输入样例1

```c
3 2
```

#### 输出样例1

```c
16
```

#### 输入样例2

```c
4 4
```

#### 输出样例2

```c
79
```
