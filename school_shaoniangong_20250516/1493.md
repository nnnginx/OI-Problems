## 题目描述

写一程序读入二次方程的系数$\red{a,b,c}$,计算并输出二次方程的二个根。二次方程为：$\red{ax^2+bx+c=0}$

## 输入格式：

共一行，三个数$\red{a,b,c}$。

## 输出格式：

共二行，每行由二部分组成，字符串和实根 （保留小数点后二位） 组成，冒号后有一空格，且第一根的值大于等于第二个根。

## 样例

#### 输入样例

```c
1 2 1
```

#### 输出样例

```c
The first root is: -1.00 
The second root is: -1.00
```

## 提示 

输入数据保证方程一定有解。
