## 题目描述

原题来自：`SCOI 2010`

在一个 $\red{2}$ 维平面上有两条传送带，每一条传送带可以看成是一条线段。两条传送带分别为线段$\red{\text{AB}}$ 和线段 $\red{\text{CD}}$。`lxhgww `在$\red{\text{AB}}$ 上的移动速度为 $\red{P}$ ，在 $\red{\text{CD}}$ 上的移动速度为 $\red{Q}$ ，在平面上的移动速度 $\red{R}$。

现在 `lxhgww `想从 $\red{A}$ 点走到 $\red{D}$ 点，他想知道最少需要走多长时间。

## 输入格式

输入数据第一行是 $\red{4}$ 个整数，表示 $\red{A}$ 和 $\red{B}$ 的坐标，分别为 $\red{A_x,A_y,B_x,B_y}$；

第二行是 $\red{4}$ 个整数，表示 $\red{C}$ 和 $\red{D}$ 的坐标，分别为 $\red{C_x,C_y，D_x,D_y}$；

第三行是 $\red{3}$ 个整数，分别是 $\red{P,Q,R}$。

## 输出格式

输出数据为一行，表示 `lxhgww `从 $\red{A}$ 点走到 $\red{D}$ 点的最短时间，保留到小数点后 $\red{2}$ 位。

## 样例

#### 输入样例

```c
0 0 0 100
100 0 100 100
2 2 1
```

#### 输出样例

```c
136.60
```

## 提示

对于 $\red{100\%}$ 的数据，$\red{1 \leq A_x,A_y,B_x,B_y,C_x,C_y,D_x,D_y \leq 1000,1 \leq P,Q,R \leq 10}$。
