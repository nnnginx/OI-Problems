## 题目描述

有两台机器 $\red {A，B}$ 以及 $\red {K}$ 个任务。

机器 $\red {A}$ 有 $\red {N}$ 种不同的模式（模式$\red {0\sim N-1}$），机器 $\red {B}$ 有 $\red {M}$ 种不同的模式（模式$\red {0\sim M-1}$）。

两台机器最开始都处于模式$\red {0}$。

每个任务既可以在$\red {A}$上执行，也可以在$\red {B}$上执行。

对于每个任务 $\red {i}$，给定两个整数 $\red {a[i]}$ 和 $\red {b[i]}$，表示如果该任务在 $\red {A}$ 上执行，需要设置模式为$\red { a[i]}$，如果在$\red { B}$ 上执行，需要模式为 $\red {b[i]}$。

任务可以以任意顺序被执行，但每台机器转换一次模式就要重启一次。

求怎样分配任务并合理安排顺序，能使机器重启次数最少。

## 输入格式

输入包含多组测试数据。

每组数据第一行包含三个整数 $\red {N, M, K}$。

接下来$\red {k}$行，每行三个整数 $\red {i, a[i]}$和$\red {b[i]，i }$为任务编号，从$\red {0}$开始。

当输入一行为$\red {0}$时，表示输入终止。

## 输出格式

每组数据输出一个整数，表示所需的机器最少重启次数，每个结果占一行。

## 样例

#### 输入样例

```c
5 5 10
0 1 1
1 1 2
2 1 3
3 1 4
4 2 1
5 2 2
6 2 3
7 2 4
8 3 3
9 4 3
0
```

#### 输出样例

```c
3
```

## 提示

$\red {N,M<100,K<1000}$

$\red {0≤a[i]<N}$

$\red {0≤b[i]<M}$
