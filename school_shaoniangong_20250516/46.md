## 题目描述

输入一个长度为$\red n$的整数序列，从中找出一段长度不超过$\red m$的连续子序列，使得子序列中所有数的和最大。

## 输入格式

第一行输入两个整数$\red {n,m}$。

第二行输入$\red n$个数，代表长度为$\red n$的整数序列。

同一行数之间用空格隔开。

## 输出格式

输出一个整数，代表该序列的最大子序和。

## 样例

#### 输入样例

```C
6 4
1 -3 5 1 -2 3
```

#### 输出样例

```C
7
```

## 提示

注意： 子序列的长度至少是$\red 1$。

$\red {1≤n,m≤300000}$
