## 题目描述

农夫约翰的$\red{N（}$$\red{2 <= N<=10,000）}$头奶牛，编号为$\red{1.. N，}$一共会流利地使用$\red{M（}$$\red{1<= M <=30,000）}$种语言，编号从$\red{1 .. M.，}$ 第$\red{i}$头，会说$\red{K_i（}$$\red{1 <= K_i<= M）}$种语言，即$\red{L_{i1}, L_{i2},..., L_{iK_i} (1 <= L_{ij} <= M)}$。 $\red{FJ}$的奶牛不太聪明，所以$\red{K_i}$的总和至多为$\red{100,000}$。

两头牛，不能直接交流，除非它们都会讲某一门语言。然而，没有共同语言的奶牛们，可以让其它的牛给他们当翻译。换言之，牛$\red{A}$和$\red{B}$可以谈话，当且仅当存在 一个序列奶牛$\red{T_1，}$$\red{T_2，...，}$$\red{T_k，}$$\red{A}$和$\red{T_1}$都会说某一种语言，$\red{T_1}$和$\red{T_2}$也都会说某一种语言……，并且$\red{T_k}$和$\red{B}$会说某一种语言。

农夫约翰希望他的奶牛更加团结，所以他希望任意两头牛之间可以交流。他可以买书教他的奶牛任何语言。作为一个相当节俭的农民，$\red{FJ}$想要购买最少的书籍，让所有他 的奶牛互相可以说话。

帮助他确定：

$\red{*}$他必须购买的书籍的最低数量

## 输入格式

第$\red{1}$行：两个用空格隔开的整数：$\red{N}$和$\red{M}$

第$\red{2.. N +1}$行：第$\red{i +1}$行描述的牛$\red{i}$的语言，$\red{K_{i+1}}$个空格隔开的整数：$\red{K_i ，L_{i1}}$
$\red{L_{i2}，...，}$$\red{L_I{K_i}}$。

## 输出格式

第$\red{1}$行：一个整数，$\red{FJ}$最少需要购买的书籍数量。

## 样例

#### 输入样例

```c
3 3
2 3 2
1 2
1 1
```

#### 输出样例

```c
1
```

## 提示

给三号牛买第二本书即可
