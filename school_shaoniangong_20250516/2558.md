## 题目描述

最近，$\red{Farmer John}$的奶牛们越来越不满于牛棚里一塌糊涂的电话服务 于是，她们要求$\red{FJ}$把那些老旧的电话线换成性能更好的新电话线。 新的电话线架设在已有的$\red{N(2 <= N <= 100,000)}$根电话线杆上， 第$\red{i}$根电话线杆的高度为$\red{height_i}$米$\red{(1 <= height_i <= 100)}$。

电话线总是从一根电话线杆的顶端被引到相邻的那根的顶端 如果这两根电话线杆的高度不同，那么$\red{FJ}$就必须为此支付 $\red{C*}$电话线杆高度差$\red{(1 <= C <= 100)}$的费用。

当然，你不能移动电话线杆， 只能按原有的顺序在相邻杆间架设电话线。$\red{Farmer John}$认为 加高某些电话线杆能减少架设电话线的总花费，尽管这项工作也需要支出一定的费用。

更准确地，如果他把一根电话线杆加高$\red{X}$米的话，他得为此付出$\red{X^2}$的费用。

请你帮$\red{Farmer John}$计算一下，如果合理地进行这两种工作，他最少要在这个电话线改造工程上花多少钱。

## 输入格式

第$\red{1}$行: $\red{2}$个用空格隔开的整数：$\red{N}$和$\red{C}$

第$\red{2..N+1}$行: 第$\red{i+1}$行仅有一个整数：$\red{height_i}$

## 输出格式

第$\red{1}$行: 输出$\red{Farmer John}$完成电话线改造工程所需要的最小花费

## 样例

#### 输入样例

```c
5 2
2
3
5
1
4
```




#### 输出样例

```c
15
```

## 提示

输入说明:

一共有$\red{5}$根电话线杆，在杆间拉电话线的费用是每米高度差$\red{2}$美元。
在改造之前，电话线杆的高度依次为$\red{2，}$$\red{3，}$$\red{5，}$$\red{1，}$$\red{4}$米。

输出说明:

最好的改造方法是：$\red{Farmer John}$把第一根电话线杆加高$\red{1}$米，把第四根加高$\red{2}$米，使得它们的高度依次为$\red{3，}$$\red{3，}$$\red{5，}$$\red{3，}$$\red{4}$米。这样花在加高电线杆上的钱是$\red{5}$美元。




此时，拉电话线的费用为$\red{2\times(0+2+2+1) = }$美元$\red{10，}$总花费为$\red{15}$美元。
