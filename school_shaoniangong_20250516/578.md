## 题目描述

有些公司是其他公司的部分拥有者，因为他们获得了其他公司发行的股票的一部分。(此处略去一句废话）据说，如果至少满足了以下三个条件之一，公司$\red{A}$就可以控制公司$\red{B}$了：

- $\red{1}$.公司$\red{A}$ = 公司$\red{B}$。
- $\red{2}$.公司$\red{A}$拥有大于$\red{50\%}$的公司B}$的股票。
- $\red{3}$.公司$\red{A}$控制$\red{K(K >= 1)}$个公司，记为$\red{C_1}$, ..., $\red{C_K}$，每个公司$\red{C_i}$拥有$\red{x_i\%}$的公司$\red{B}$的股票，并且$\red{x_1+ .... + x_K > 50\%}$。

给你一个表，每行包括三个数$\red{(i,j,p)}$；表明公司$\red{i}$享有公司$\red{j}$的$\red{p\%}$的股票。计算所有的数对$\red{(h,s)}$，表明公司$\red{h}$控制公司$\red{s}$。至多有$\red{100}$个公司。

写一个程序读入$\red{N}$组数$\red{(i,j,p)，i,j}$和$\red{p}$是都在范围$\red{(1..100)}$的正整数，并且找出所有的数对$\red{(h,s)}$，使得公司$\red{h}$控制公司$\red{s}$。

## 输入格式

第一行： $\red{N}$，表明接下来三个数的数量，即$\red{(i,j,p)}$的数量。

第二行到第$\red{N+1}$行： 每行三个整数作为一个三对数$\red{(i,j,p)}$，表示$\red{i}$公司拥有$\red{j}$公司 $\red{p\%}$的股份。

## 输出格式

输出零个或更多个的控制其他公司的公司。每行包括两个整数$\red{A、B}$,表示$\red{A}$公司控制了$\red{B}$公司。将输出的数对以升序排列。

请不要输出控制自己的公司(应该是不输出自己，互相控制的公司还是要输出的  $\red{A≠B}$）。

## 样例

#### 输入样例

```c
3
1 2 80
2 3 80
3 1 20
```

#### 输出样例

```c
1 2
1 3
2 3
```
