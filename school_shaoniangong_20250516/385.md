## 题目描述

不同的雪花往往有不同的形状。在北方的同学想将雪花收集起来，作为礼物送给在南方的同学们。一共有 $\red{n}$ 个时刻，给出每个时刻下落雪花的形状，用不同的整数表示不同的形状。在收集的过程中，同学们不希望有重复的雪花。你可以从任意 $\red{a}$ 时刻开始，在 $\red{b}$ 时刻停止。$\red{a}$ 到 $\red{b}$ 时刻中间的雪花也都将被收集。他们希望收集的雪花最多。

## 输入格式

第一行一个正整数 $\red{n}$；

第 $\red{2}$ 行 $\red{n}$ 个非负整数表示 $\red{n}$ 个时刻雪花的形状。

## 输出格式

最多能收集雪花的数量。

## 样例

#### 输入样例

```c
5
1 2 3 2 1
```

#### 输出样例

```c
3
```

## 提示

对于 $\red{97}$ 分的数据，$\red{1\le n \le 10^6, 0\le x_i \le 10^8}$。（为原始数据）

应用户要求，加入 $\red{3 }$分的数据，$\red{1\le n\le 10^6,0\le x_i\le 10^9}$。
