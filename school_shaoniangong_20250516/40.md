## 题目描述

这里有$\red n$列火车将要进站再出站，但是，每列火车只有$\red 1$节，那就是车头。

这$\red n$列火车按$\red 1$到$\red n$的顺序从东方左转进站，这个车站是南北方向的，它虽然无限长，只可惜是一个死胡同，而且站台只有一条股道，火车只能倒着从西方出去，而且每列火车必须进站，先进后出。

也就是说这个火车站其实就相当于一个栈，每次可以让右侧头火车进栈，或者让栈顶火车出站。

车站示意如图：

<pre>
出站<――       <――进站
	|车|
        |站|
        |__|
</pre>

现在请你按《字典序》输出前20种可能的出栈方案。

## 输入格式

输入一个整数$\red n$，代表火车数量。

## 输出格式

按照《`字典序`》输出前$\red {20}$种答案，每行一种，不要空格。

## 样例

#### 输入样例

```C
3
```

#### 输出样例

```C
123
132
213
231
321
```

## 提示

$\red {1≤n≤20}$
