## 题目描述

原题来自：`USACO 2015 Feb. Silver`

给出两个字符串 $\red{S}$ 和 $\red{T}$，每次从前往后找到 $\red{S}$ 的一个子串 $\red{A=T}$ 并将其删除，空缺位依次向前补齐，重复上述操作多次，直到 $\red{S}$ 串中不含 $\red{T}$ 串。输出最终的 $\red{S}$ 串。

## 输入格式

第一行包含一个字符串 $\red{S}$，第二行包含一个字符串 $\red{T}$。

## 输出格式

输出处理后的 $\red{S}$ 串。

## 样例

#### 输入样例

```c
whatthemomooofun
moo
```

#### 输出样例

```c
whatthefun
```

## 提示

对于全部数据，$\red{1\le |T|\le |S|\le 10^6}$，保证字符串中只出现小写字母。
