## 题目描述

贝西又开始弹琴了。她发现通过改变字母顺序，她可以使一些字符串在字典顺序上排在所有其他字符串之前（字典排序）。

例如，$\red{Bessie }$发现对于字符串"$\red{omm}$"、"$\red{moo}$"、"$\red{mom}$"和"$\red{ommnom}$"，她可以使用标准字母表让"$\red{mom}$"首先出现，并且她可以使用字母表让"$\red{omm}$"首先出现" $\red{abcdefghijklonmpqrstuvwxyz}$"。

然而，$\red{Bessie }$想不出任何办法让"$\red{moo}$"或"$\red{ommnom}$"首先出现。帮助 $\red{Bessie }$通过重新排列字母顺序来计算输入中的哪些字符串可以按字典顺序 排列。

要计算字符串$\red{X}$是否在字符串$\red{Y}$之前按字典顺序排列，请找到它们不同的第一个字符的索引，即$\red{j}$。如果不存在此类索引，则如果$\red{X}$短于$\red{Y，}$则$\red{X}$在$\red{Y}$之前按词典顺序排列。否则，如果$\red{X[j]}$在字母表中出现的时间早于$\red{Y[j]，}$则$\red{X}$在字母表上在$\red{Y}$之前。

给定$\red{n}$个总长不超过$\red{m}$的互不相同的字符串，现在你可以任意指定字符之间的大小关系。问有多少个串可能成为字典
序最小的串，并输出这些串。$\red{n <= 30,000 , m <= 300,000}$

## 输入格式

第 $\red{1 }$行：包含 $\red{N (1 <= N <= 30,000) }$的单行，即 $\red{Bessie }$正在弹奏的琴弦数。

第 $\red{2..1+N }$行：每行包含一个非空字符串。所有字符串中的字符总数将不超过 $\red{300,000}$。输入中的所有字符都是小写字符"$\red{a}$"到"$\red{z}$"。输入将不 包含重复的字符串。

## 输出格式

第 $\red{1 }$行：包含 $\red{K }$的单行，可按字典顺序排在首位的字符串数。

第 $\red{2..1+K }$行：第 $\red{(1+i) }$行应包含按字典顺序排列的第 $\red{i }$个字符串。字符串应该按照它们在输入中给出的顺序输出。

## 样例

#### 输入样例

```c
4
omm
moo
mom
ommnom

```


#### 输出样例

```c
2
omm
mom
```

输入细节：

问题陈述中的示例。

只能先订购"$\red{omm}$"和"$\red{mom}$"。
