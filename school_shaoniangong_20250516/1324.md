### 题目描述
`Farmer John`已经决定把水灌到他的$\red{n(1<=n<=300)}$块农田，农田被数字$\red{1}$到$\red{n}$标记。

把一块土地进行灌水有两种方法，从其他农田饮水，或者这块土地建造水库。 

建造一个水库需要花费$\red{w_i(1<=w_i<=100000)}$,连接两块土地需要花费$\red{P_{ij}(1<=p_{ij}<=100000,p_{ij}=p_{ji},p_{ii}=0)}$. 计算`Farmer John`所需的最少代价。

### 输入格式

第一行：一个数$\red{n}$

第二行到第$\red{n+1}$行：第$\red{i+1}$行含有一个数$\red{w_i}$

第$\red{n+2}$行到第$\red{2n+1}$行：第$\red{n+1+i}$行有$\red n$个被空格分开的数，第$\red{j}$个数代表$\red{p_{ij}}$。

输出格式

第一行：一个单独的数代表最小代价.

### 样例

#### 样例输入
```			
4
5
4
4
3
0 2 2 2
2 0 3 3
2 3 0 4
2 3 4 0
```
#### 样例输出
```		
9
```





### 提示：

`Farmer John`在第四块土地上建立水库，然后把其他的都连向那一个，这样就要花费$\red{3+2+2+2=9}$
