## 题目描述

小王在考试中遇到一道难题：方程$\red{a1+a2+……}$$\red{+an=m}$的非负整数解有几个，请你帮他算一下（这也可以算作他作弊吧）。

## 输入格式

一行，两个以空格隔开的数$\red{n，}$$\red{m，}$表示方程$\red{a1+a2+……}$$\red{+an=m}$

## 输出格式

一行，一个数，表示非负整数解的个数。

## 样例

#### 输入样例

```c
3 4
```

#### 输出样例

```c
15
```

## 提示

$\red{0,0,4 ~~~ 0,1,3 ~~~ 0,2,2 ~~~ 0,3,1 ~~~ 0,4,0}$

$\red{1,0,3 ~~~ 1,1,2 ~~~ 1,2,1 ~~~ 1,3,0}$

$\red{2,0,2  ~~~2,1,1 ~~~ 2,2,0}$

$\red{3,0,1 ~~~ 3,1,0}$

$\red{4,0,0     ~~~            (total=5+4+3+2+1=15)}$
