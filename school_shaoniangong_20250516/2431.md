## 题目描述

了解奶牛们的人都知道，奶牛喜欢成群结队．

观察约翰的$\red{N(1≤}$$\red{N≤}$$\red{100000)}$只奶牛，你会发现她们已经结成了几个"群"．每只奶牛在吃草的时候有一个独一无二的位置坐标$\red{Xi，}$$\red{Yi(l≤}$$\red{Xi，}$$\red{Yi≤}$$\red{[1...}$$\red{10^9]}$；$\red{Xi，}$$\red{Yi∈}$整数．当满足下列两个条件之一，两只奶牛$\red{i}$和$\red{j}$是属于同一个群的：

$\red{1}$．两只奶牛的曼哈顿距离不超过$\red{C(1≤}$$\red{C≤}$$\red{10^9)，}$即$\red{|Xi - xi|+|Yi - Yi|≤}$$\red{C.}$

$\red{2}$．两只奶牛有共同的邻居．即，存在一只奶牛$\red{k，}$使$\red{i}$与$\red{k，}$$\red{j}$与$\red{k}$均同属一个群．

给出奶牛们的位置，请计算草原上有多少个牛群，以及最大的牛群里有多少奶牛


## 输入格式

第$\red{1}$行输入$\red{N}$和$\red{C，}$之后$\red{N}$行每行输入一只奶牛的坐标．

## 输出格式

仅一行，先输出牛群数，再输出最大牛群里的牛数，用空格隔开．

## 样例

#### 输入样例

```c
4 2
1 1
3 3
2 2
10 10
```




#### 输出样例

```c
2 3
```

## 提示

第$\red{1}$行：

具有两个空格分隔的整数的单行：奶牛社区的数量和最大奶牛的大小邻里

输出详细信息：

有两个社区，一个由前三头奶牛和另一头是最后一头奶牛。

因此最大的社区尺寸为$\red{3}$。
