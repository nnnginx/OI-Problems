## 题目描述

一家工厂的流水线正在生产一种产品，这需要两种操作：操作$\red A$和操作$\red B$。每个操作只有一些机器能够完成。

![img](./611/file/1.png)

上图显示了按照下述方式工作的流水线的组织形式。$\red A$型机器从输入库接受工件，对其施加操作$\red A$，得到的中间产品存放在缓冲库。$\red B$型机器从缓冲库接受中间产品，对其施加操作$\red B$，得到的最终产品存放在输出库。所有的机器平行并且独立地工作，每个库的容量没有限制。每台机器的工作效率可能不同，一台机器完成一次操作需要一定的时间。

给出每台机器完成一次操作的时间，计算完成$\red A$操作的时间总和的最小值，和完成$\red B$操作的时间总和的最小值。

## 输入格式

第一行 三个用空格分开的整数：$\red N$，工件数量 $\red{(1<=N<=1000)}$;$\red{M1}$，$\red A$型机器的数量 $\red{(1<=M1<=30)}$;$\red{M2}$，$\red B$型机器的数量 $\red{(1<=M2<=30)}$。

第二行…等 $\red{M1}$个整数（表示$\red A$型机器完成一次操作的时间，$\red{1..20}$），接着是$\red{M2}$个整数（$\red B$型机器完成一次操作的时间，$\red{1..20}$）

## 输出格式

只有一行。输出两个整数：完成所有$\red A$操作的时间总和的最小值，和完成所有$\red B$操作的时间总和的最小值（$\red A$操作必须在$\red B$操作之前完成）。

## 样例

#### 输入样例

```c
5 2 3
1 1 3 1 4
```

#### 输出样例

```c
3 5
```
