## 题目描述

我们把只有$\red{0}$和$\red{1}$组成的字符串叫做比特字符串。如果比特字符串中$\red{1}$的个数为奇数，则称这个比特字符串是奇性的。

如果比特字符串中$\red{1}$的个数为偶数，则称这个比特字符串是偶性的。
注意：

- `1`、$\red{0}$是偶数，所以不包含$\red{1}$的比特字符串是偶性的。
- `2`、$\red{0}$的个数不影响比特字符串的奇偶性。

## 输入格式

输入包含多组测试数据。

每组数据由$\red{1}$`~`$\red{31}$`个`$\red{0}$、$\red{1}$组成，最后跟一个小写字母`e`或`o`，`e`表示此比特字符串应为偶性，$\red{o}$表示此比特字符串应为奇性。

当输入`#`时，表示输入结束。

## 输出格式

每组输入对应一个输出，你应该将最后的字母替换成$\red{0}$或$\red{1}$，使得此比特字符串符合输入时确定的奇偶性。

## 样例

#### 输入样例

```C
101e
010010o
1e
000e
110100101o
#
```

#### 输出样例

```C
1010
0100101
11
0000
1101001010
```
