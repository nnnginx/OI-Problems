## 题目描述

原题来自：`SCOI 2009`

`Windy` 定义了一种 `Windy` 数：不含前导零且相邻两个数字之差至少为$\red{ 2}$ 的正整数被称为 `Windy` 数。

`Windy` 想知道，在 $\red{A }$和$\red{ B }$之间，包括$\red{ A}$ 和$\red{ B}$，总共有多少个 `Windy` 数？

## 输入格式

一行两个数，分别为$\red{ A,B}$。

## 输出格式

输出一个整数，表示答案。

## 样例

#### 输入样例 1

```c
1 10
```

#### 输出样例 1

```c
9
```

#### 输入样例 2

```c
25 50
```

#### 输出样例 2

```c
20
```

## 提示

$\red{20\%}$ 的数据，满足 $\red{1\le A\le B\le 10^6}$；

$\red{100\%}$ 的数据，满足 $\red{1\le A\le B\le 2\times 10^9}$。
