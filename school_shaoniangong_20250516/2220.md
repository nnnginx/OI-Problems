## 题目描述

农民约翰最近建造了一个巨大的谷仓，由$\red{N×}$$\red{N}$个房间组成$\red{（2≤}$$\red{N≤}$$\red{100）}$，编号从（$\red{1,1）}$到（$\red{N，}$$\red{N）}$。由于有点怕黑，奶牛贝西想在旧能多的 房间里开灯。

贝西从房间（$\red{1,1）}$开始，这是唯一一个最初亮起的房间。在一些房间里，她会找到可以用来切换其他房间灯光的开关；例如，房间（$\red{1,1）}$中可能有一个开关，用于切换房间（$\red{1,2）}$中的灯光。贝西只能穿过有灯光的房间，她只能从一个房间（$\red{x，}$$\red{y）}$移动到四个相邻的房间（$\red{x-}$$\red{1，}$$\red{y）}$、（$\red{x+1，}$$\red{y）}$、（$\red{x，}$$\red{y-}$$\red{1 ）}$ 和（$\red{x，}$$\red{y+1）}$（如果该房间位于网格边界上，则可能更少的邻居）。

请确定贝西可以照亮的最大房间数。

## 输入格式

第一行输入包含整数$\red{N}$和$\red{M（}$$\red{1≤}$$\red{M≤}$$\red{20,000)}$。

接下来的$\red{M}$行分别描述了一个具有四个整数$\red{x，}$$\red{y，}$$\red{a，}$$\red{b}$的单灯开关，房间（$\red{x，}$$\red{y）}$中的开关可用于切换房间（$\red{a，}$$\red{b）}$中的灯。任何房间中都可能存在多个开关，多个开关可以切换任何房间的灯光。

## 输出格式

一条线给出了 $\red{Bessie }$可以照亮的最大房间数。

## 样例

#### 输入样例

```c
3 6 
1 1 1 2 
2 1 2 2 
1 1 1 3 
2 3 3 1 
1 3 1 2 
1 3 2 1
```

#### 输出样例

```c
5
```

## 提示

在这里，贝西可以使用（$\red{1,1）}$中的开关打开（$\red{1,2）}$和（$\red{1,3）}$中的灯。然后她可以走到（$\red{1,3）}$并打开（$\red{2,1）}$中的灯，从中她可以打开（$\red{2,2）}$中的灯。（$\red{2,3）}$中的开关在一个没有照明的房间里，她无法接近。因此，她最多可以照亮$\red{5}$个房间。
