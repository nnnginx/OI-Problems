## 题目描述

贝西喜欢棋盘游戏和角色扮演类游戏所以她说服$\red{Farmer John}$把她带到玩具店，在那里，她购买了三个不同的骰子，这三个质量均匀的骰子，分别有$\red{S1,S2,S3}$个面。

$\red{(2 <= S1 <= 20}$; $\red{2 <= S2 <= 20}$; $\red{2 <= S3 <= 40). }$贝西掷啊掷啊掷啊，想要知道出现几率最大的和是多少。

问题给出三个骰子的面数，让你求出出现几率最大的和是多少。

如果有很多种和出现的几率相同，那么就输出小的那一个。

## 输入格式

第一行：三个由空格隔开的整数：$\red{s1,s2,s3}$

## 输出格式

第一行：所要求的解

## 样例

#### 输入样例

```c
3 2 3
```




#### 输出样例

```c
5
```

## 提示

输出详解:

这里是所有可能的情况.

$\red{1 1 1 -> 3 1 2 1 -> 4 2 1 1 -> 4 2 2 1 -> 5 3 1 1 -> 5 3 2 1 -> 6}$

$\red{1 1 2 -> 4 1 2 2 -> 5 2 1 2 -> 5 2 2 2 -> 6 3 1 2 -> 6 3 2 2 -> 7}$

$\red{1 1 3 -> 5 1 2 3 -> 6 2 1 3 -> 6 2 2 3 -> 7 3 1 3 -> 7 3 2 3 -> 8}$

$\red{5}$和$\red{6}$出现的几率都是最大的，所以输出$\red{5.}$
