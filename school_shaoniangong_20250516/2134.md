## 题目描述

$\red{Bessie }$和她的朋友们正在玩一种独特版本的扑克游戏，其中包含 $\red{N (1 <= N <= 100,000) }$个不同等级的牌组，方便编号为 $\red{1..N（}$普通牌组有 $\red{N = 13）}$。在这个游戏中，奶牛只 能玩一种类型的牌：一个人可以选择标有 $\red{i }$的牌和标有 $\red{j }$的牌，并从 $\red{i }$到 $\red{j }$打出一张牌。这种类型的手称为"顺子"。

$\red{Bessie }$的手上目前持有 $\red{a_i }$等级为 $\red{i (0 <= a_i <= 100000) }$的牌。帮助她找出她必须玩的最少手数才能摆脱她所有的牌。

一个牛有$\red{N}$堆牌，每堆数量不等。一只牛一次可以将第$\red{i}$张到第$\red{j}$张各打一张出去，问最少几次打完

## 输入格式

第 $\red{1 }$行：整数 $\red{N}$。

第 $\red{2..1+N }$行：第 $\red{i+1 }$行包含 $\red{a_i }$的值。

## 输出格式

第 $\red{1 }$行：$\red{Bessie }$必须打出的最小顺子数才能摆脱她的所有牌。

## 样例

#### 输入样例

```c
5 
2 
4 
1 
2
3
```

#### 输出样例

```c
6
```

## 提示

$\red{Bessie }$可以玩 $\red{1 }$到 $\red{5 }$的顺子、$\red{1 }$到 $\red{2 }$的顺子、$\red{4 }$到 $\red{5 }$的顺子、$\red{2 }$到 $\red{2 }$的两个顺子和 $\red{5 }$到 $\red{5 }$的顺子，总共需 要 $\red{6 }$轮才能摆脱她所有的卡片。
