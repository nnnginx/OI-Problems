## 题目描述

已知线段的两个端点的坐标$\red{A(X_a,Y_a)，B(X_b，Y_b)}$，求线段$\red{AB}$的长度，保留到小数点后$\red{3}$位。

## 输入格式

第一行是两个实数$\red{X_a，Y_a}$，即$\red{A}$的坐标。

第二行是两个实数$\red{X_b，Y_b}$，即$\red{B}$的坐标。

输入中所有实数的绝对值均不超过$\red{10000}$。

## 输出格式

一个实数，即线段$\red{AB}$的长度，保留到小数点后$\red{3}$位。

## 样例

#### 输入样例

```C
1 1
2 2
```

#### 输出样例

```C
1.414
```
