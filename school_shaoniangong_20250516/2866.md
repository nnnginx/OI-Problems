## 题目描述

牛牛最近在观察月相，他把月亮的大小分为 $\red{0}$$\red{\sim15 }$这$\red{16 }$个大小等级，月亮会在
每个月先每天放大$\red{1，}$从$\red{0}$缓慢变大到$\red{15，}$然后再每天减小$\red{1，}$从$\red{15}$逐渐变成$\red{0，}$
以此循环往复。

牛牛连续观察了$\red{2}$天月亮的大小，牛牛想知道下一天月亮的大小是多少。

## 输入格式

第一行输入两个整数$\red{a, b，}$表示牛牛观察的月亮的大小，分别是第一天、第二天
的结果。

对于 $\red{60\%}$的数据有$\red{1 ≤}$ $\red{a, b ≤}$ $\red{14}$。

对于 $\red{100\%}$的数据有$\red{0 ≤}$ $\red{a, b ≤}$ $\red{15}$。

## 输出格式

一行一个整数，表示下一天的月亮的大小。

## 样例

#### 输入样例

```c
15 14
```

#### 输出样例

```c
13
```

## 提示

前两天分别是 $\red{15, 14, }$故牛牛知道月亮在逐渐变小，所以下一天是 $\red{13}$。
