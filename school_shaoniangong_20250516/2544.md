## 题目描述

每天,农夫 $\red{John }$的$\red{N(1 <= N <= 50,000)}$头牛总是按同一序列排队. 有一天, $\red{John }$决定让一些牛们玩一场飞盘比赛. 他准备找一群在对列中为置连续的牛来进行比赛.

但是为了避免水平悬殊,牛的身高不应该相差太大. $\red{John }$准备了$\red{Q (1 <= Q <= 180,000) }$个可能的牛的选择和所有牛的身高 $\red{(1 <= }$身高 $\red{<= 1,000,000). }$他想知道每一组里面最高和最低的牛的身高差别.

注意: 在最大数据上, 输入和输出将占用大部分运行时间.

## 输入格式

第一行: $\red{N }$和 $\red{Q. * }$第$\red{2..N+1}$行: 第$\red{i+1}$行是第$\red{i}$头牛的身高.

第$\red{N+2..N+Q+1}$行: 两个整数, $\red{A }$和 $\red{B (1 <= A <= B <= N), }$表示从$\red{A}$到$\red{B}$的所有牛.

## 输出格式

第$\red{1..Q}$行: 所有询问的回答 $\red{(}$最高和最低的牛的身高差$\red{), }$每行一个.

## 样例

#### 输入样例

```c
6 3
1
7
3
4
2
5
1 5
4 6
2 2
```


#### 输出样例

```c
6
3
0
```
