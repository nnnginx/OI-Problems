## 题目描述

一个走廊的拐角处如图所示（俯视图）。组成这个拐角的两条走廊的宽度分别为$\red{a}$和$\red{b}（$图中已标出）。现有一个长和宽分别为$\red{c}$和$\red{d}$的箱子（不考虑高度），请问箱子是否能够顺利 通过这个走廊？要求是箱子的任何部分不得触碰到走廊墙壁的任何部分。箱子可以水平旋转，但是不可以倾斜，它可认为是一个必须在运动过程中保持与地面平行的矩形。

![img](./2025/file/1.png)

## 输入格式

每个输入文件给出了一组$\red{a}$、$\red{b}$和多组$\red{c}$、$\red{d}$的值，也就是说每个输入文件给出了一个走廊拐角和多个箱子。

输入文件中第一行给出了两个用空格隔开的实数，为$\red{a}$、$\red{b}$（也就是两条走廊的宽度），第二行给出了一个整数$\red{n，}$表示给出箱子的组数。从第三行起，每行给出了两个用空格隔开的实数，表示这一组$\red{c}$、$\red{d}$的值（也就是这一个箱子的长和宽）。

## 输出格式

依次输出各组箱子能否通过走廊（按照输入的次序，每行输出一组）。

如果能，输出"$\red{YES}$"，否则输出"$\red{NO}$"（不包括引号）。最后输出一个回车$\red{/}$换行符。

## 样例

#### 输入样例

```c
0.9 1.0
2
3.0 0.8
0.5 0.6
```

#### 输出样例

```c
NO
YES
```

## 提示

$\red{n<=50}$

其他数值不超过$\red{double}$类型所能表示的范围。
