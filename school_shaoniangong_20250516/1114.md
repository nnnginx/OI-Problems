## 题目描述

`紫荆学姐`喜欢用英文写作文，尤其喜欢写化学式（显得比较有逼格）。

他觉得键盘上的空格键很大，敲击的手感比回车好，于是他花了一些钱改进了空格键的手感。

因此，紫荆学姐就把所有的换行都换成了空格。有一天，紫荆学姐突发奇想，如果把英文文章的所有空格都变成换行，那会怎么样。

于是他准备了一些文章，让你编程把这些文章的空格全部变成换行。

输入一行字符串$\red{a}$，将这个字符串的空格转换成回车输出

## 数据量很大

## 输入格式

一行字符

## 输出格式

将这个字符串的空格转换成回车输出

## 样例

#### 输入样例1

```C
Hello World!
```

#### 输出样例1

```
Hello
World!
```

#### 输入样例2

```C
I like C++
```

#### 输出样例2

```C
I
like
C++
```

## 提示

可以使用 `while(cin>>a)`也可用 `cin.getline()` 函数

多个空格当作一个空格处理!!!
