## 题目描述

奶牛们在玩按字母表顺序排列的拼图谜题，每道谜题有$\red{R(1<=R<=10)}$列$\red{C1<=C<=10)}$行的拼图块，它们边缘是由字母或封闭边界组成，完成后的整副拼图外围是边界线，中间的边界是字母.

每块拼图块都有一个序列号和$\red{4}$个字母或者数字表示边界线$\red{(}$顺序为上右下左$\red{)，}$在输入中，数字$\red{0}$充当边界线.

拼图可以换位和旋转，完成后的拼图在边缘的块上靠近外围的是边界线，拼图完成后，一块拼冬若与另一块相邻，它们的边界字母必须相同，以下是一系列已经成功完成的拼冬谜 题，共$\red{6}$块.
![img](./2435/file/3.png)

## 输入格式

第$\red{1}$行是两个整数$\red{R}$和$\red{C，}$用空格隔开．

从第$\red{2}$行到第$\red{R\times C+I}$行，每一行包含$\red{1}$个数字和$\red{4}$个代表边界的符号（可以是字母或数字$\red{0）}$．

## 输出格式

输出换位及旋转后的拼图．共$\red{RxC}$行，每行第一个数字代表第几块拼图块，后面四个字符，按顺序代表$\red{4}$个边界字母（边界线仍用$\red{0}$表示）．

答案可能有多组，输出一组即可．

## 样例

#### 输入样例

```c
2 3
1 c d 0 0
2 0 d b 0
3 c 0 d a
4 b a b 0
5 d 0 0 e
6 0 0 b e
```

#### 输出样例

```c
1 0 c d 0
3 0 d a c
5 0 0 e d
2 d b 0 0
4 a b 0 b
6 e 0 0 b
```
