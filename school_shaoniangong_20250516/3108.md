# 题目描述

小明是一个有强迫症的男生，不管他看到什么都要按照一定的准则整理一番。有一次，小明随便写了一个生产随机数的程序，当他看到屏幕上

随意跳动的杂乱数据，强迫症迫使他当即写了一个排序算法的程序，按照数字单调递增的方式优雅的排列起来。
## 输入

第一行，输入一个整数 n（n<100000）， 表示 n 个数字

第二行，输入 n 个整数

## 输出

按照数字单调递增的方式输出，数字之间用逗号",”隔开

# 样例

```输入
5
6 2 4 7 1
```

```输出
1,2,4,6,7
```

# Limitation

1s, 1024KiB for each test case.
