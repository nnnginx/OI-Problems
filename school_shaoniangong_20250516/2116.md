## 题目描述

$\red{John }$在他的农场中闲逛时发现了许多虫洞。虫洞可以看作一条十分奇特的有向边，并可以使你返回到过去的一个时刻（相对你 进入虫洞之前）。

$\red{John }$的每个农场有 $\red{m }$条小路（无向边）连接着 $\red{n }$块地（从 $\red{1 \sim n}$标号），并有 $\red{w }$个虫洞。

现在 $\red{John }$希望能够从某块地出发，走过一条路径回到出发点，且同时也回到了出发时刻以前的某一时刻。请你告诉他能否做到。

## 输入格式

输入的第一行是一个整数 $\red{T，}$代表测试数据的组数。

每组测试数据的格式如下：

每组数据的第一行是三个用空格隔开的整数，分别代表农田的个数 $\red{n，}$小路的条数 $\red{m，}$以及虫洞的个数 $\red{w}$。

每组数据的第 $\red{2 }$到第 $\red{(m+1) }$行，每行有三个用空格隔开的整数 $\red{u,v,p，}$代表有一条连接 $\red{u }$与 $\red{v }$的小路，经过这条路需要花费 $\red{p }$的时间。

每组数据的第 $\red{(m+2)}$到第 $\red{(m+w+1) }$行，每行三个用空格隔开的整数 $\red{u,v,p，}$代表点 $\red{u }$存在一个虫洞，经过这个虫洞会到达点 $\red{v，}$并回到 $\red{p }$秒之前。

## 输出格式

对于每组测试数据，输出一行一个字符串，如果能回到出发时刻之前，则输出 $\red{YES，}$否则输出 $\red{NO}$。

## 样例

#### 输入样例

```c
2
3 3 1
1 2 2
1 3 4
2 3 1
3 1 3
3 2 1
1 2 3
2 3 4
3 1 8
```

#### 输出样例

```c
NO
YES
```
