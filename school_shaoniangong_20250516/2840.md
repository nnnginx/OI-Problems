## 题目描述

大家都知道 $\red{n }$阶奇数幻方吧？如下为一个 $\red{5 }$阶幻方：

$\red{17\ 24\ 1\  8\  15}$

$\red{23\ 5\ 7\ 14\ 16}$

$\red{4\ 6\ 13\ 20\ 22}$

$\red{10\ 12\ 19\ 21\ 3}$

$\red{11\ 18\ 25\ 2\ 9}$

现在，输入奇数 $\red{n，}$$\red{n<1000000，}$输出该奇数幻方最右下角的元素值。

## 输入格式

一行一个数 $\red{n}$。

## 输出格式

一行一个数，表示该奇数幻方右下角的值。

## 样例

#### 输入样例

```c
5
```

#### 输出样例

```c
9
```
