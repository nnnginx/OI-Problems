## 题目描述

一个正整数,如果它能被$\red{7}$整除,或者它的十进制表示法中某一位上的数字为$\red{7}$,则称其为与$\red{7}$相关的数.

现求所有小于等于$\red{n}$($\red{n < 100}$)的与$\red{7}$无关的正整数的`平方和`.

## 输入格式

输入为一行,正整数$\red{n}$($\red{n < 100}$)

## 输出格式

输出一行，包含一个整数，即小于等于$\red{n}$的所有与$\red{7}$无关的正整数的平方和。

## 样例

#### 输入数据

```C
21
```

#### 输出数据

```C
2336
```
