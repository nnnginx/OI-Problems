
## 题目描述
面面是班里的数学科代表，经历完期末考试后，数学老师要面面帮忙求班级的数学平均分，已知班级的人数是$\red n$,$\red n$位同学的成绩分别是$\red{a_1,a_2....a_n}$，聪明的你能帮帮面面求出班级的平均分吗？

## 输入格式
共两行；

第一行一个正整数$\red n$；

第二行为$\red n$个同学的成绩$\red{a_1,a_2....a_n}$，以空格分隔

## 输出格式
共一行；
为班级的平均分，保留到小数点后两位
## 样例
#### 输入样例
```C
5
10 20 30 40 50
```
#### 输出样例
```C
30.00
```

## 提示
对于所有的数据满足：
$\red{1 <= n <= 100}$; $\red{1<=a_i<=100}$;
