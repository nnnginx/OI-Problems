## 说明

给定两个正整数$\red{a}$（$\red{a>1}$)和$\red{b}$。可以知道一定存在整数$\red{x}$，使得

$\red{x <= log_{a}^{b} < x + 1}$ 或者 $\red{a^x <= b < a^x+1}$

请计算x。

## 输入格式

两行，第一行是$\red{a}$，第二行是$\red{b}$。每个整数均不超过$\red{150}$位。

## 输出格式

一行，即对应的$\red x$。输入数据保证$\red{x}$不大于$\red{20}$。

## 样例

#### 输入数据

```C
10000
1000000000001
```

#### 输出数据

```C
3
```
