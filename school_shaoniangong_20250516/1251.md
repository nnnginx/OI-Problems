## 题目描述

众数是指在一组数据中，出现次数最多的数。例如：$\red{1}$, $\red{1}$, $\red{3}$ 中出现次数最多的数为 $\red{1}$，则众数为 $\red{1}$。 给定一组数，你能求出众数吗？

## 输入格式

输入数据有多组（数据组数不超过 $\red{50}$），到 `EOF` 结束。

对于每组数据：

`第 1 行`输入一个整数 $\red{n}$ ($\red{1 <= n <= 10000}$)，表示数的个数。

`第 2 行`输入 $\red{n}$ 个用空格隔开的整数 $\red{A_i}$ ($\red{0 <= A_i <= 1000}$)，依次表示每一个数。

## 输出格式

对于每组数据，在一行中输出一个整数，表示这组数据的众数。

## 样例

#### 输入样例

```C
3
1 1 3
5
0 2 3 1 2
```

#### 输出样例

```C
1
2
```

## 数据范围与提示

数据保证有众数存在。
