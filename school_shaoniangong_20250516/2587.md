## 题目描述

农夫$\red{John}$想到镇上买些补给。为了高效地完成任务，他想使硬币的转手次数最少.即使他交付的硬币数与找零得到的的硬币数最少。

$\red{John}$想要买价格为$\red{T(1<=T<=10000)}$的东西。有$\red{N(1<=n<=100)}$种货币参与流通，面值分别为$\red{V1,V2..Vn (1<=Vi<=120)}$。$\red{John}$有$\red{Ci}$个面 值为$\red{Vi}$的硬币$\red{(0<=Ci<=10000)}$。

我们假设店主有无限多的硬币，并总按最优方案找零。

## 输入格式

第一行: 两个整数 $\red{N }$与 $\red{T }$。

第二行: $\red{N }$个数，表示 $\red{V1, V2, ..Vn}$。

第三行: $\red{N }$个数，表示 $\red{C1, C2, ..Cn}$。

## 输出格式

第一行: 一个整数，表示最优方案的转手次数，如无解输出$\red{-1}$。

## 样例

#### 输入样例

```c
3 70
5 25 50
5 2 1
```

#### 输出样例

```c
3
```
