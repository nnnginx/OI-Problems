## 题目描述

每一头牛的愿望就是变成一头最受欢迎的牛。现在有 $\red{N}$ 头牛，给你 $\red{M}$ 对整数 $\red{(A,B)}$，表示牛 $\red{A}$ 认为牛 $\red{B}$ 受欢迎。这种关系是具有传递性的，如果 $\red{A}$ 认为 $\red{B}$ 受欢迎，$\red{B}$ 认为 $\red{C}$ 受欢迎，那么牛 $\red{A}$ 也认为牛$\red{C}$ 受欢迎。你的任务是求出有多少头牛被除自己之外的所有牛认为是受欢迎的。

## 输入格式

第一行两个数 $\red{N,M}$；
接下来 $\red{M}$ 行，每行两个数 $\red{A,B}$，意思是 $\red{A}$ 认为 $\red{B}$ 是受欢迎的（给出的信息有可能重复，即有可能出现多个 $\red{A,B}$）。

## 输出格式

输出被除自己之外的所有牛认为是受欢迎的牛的数量。

## 样例

#### 输入样例

```c
3 3
1 2
2 1
2 3
```

#### 输出样例

```c
1
```

## 提示

#### 样例解释:

只有第三头牛被除自己之外的所有牛认为是受欢迎的。

#### 数据范围:

对于全部数据，$\red{1≤N≤10^4,1≤M≤5×10^4}$。
