## 题目描述

给定一个自然数$\red {N}$，要求把$\red {N}$拆分成若干个正整数相加的形式，参与加法运算的数可以重复。

注意:

拆分方案不考虑顺序；
至少拆分成$\red { 2 }$个数的和。
求拆分的方案数 $\red {mod2147483648}$ 的结果。

## 输入格式

一个自然数$\red {N}$。

## 输出格式

输入一个整数，表示结果。

## 样例

#### 输入样例

```c
7
```

#### 输出样例

```c
14
```

## 提示

注意:

* 拆分方案不考虑顺序；
* 至少拆分成$\red {2}$个数的和。

求拆分的方案数 $\red {mod 2147483648}$的结果。

$\red {1≤N≤4000}$
