
## 题目描述
面面是位游戏天才，一天他去游乐园发现了一个有趣的游戏，游戏是这样的，游戏开始的时候`GM`（游戏管理员）会从盲盒中随机抽取一个数字$\red n$,紧接着便会提问参与者，这个数字的密钥是多少。

几轮游戏下来，聪明的面面发现了一个规律，密钥是由$\red 0$，$\red 1$组合而成的 ，并且很快知道了解题的窍门。你一定也知道游戏的窍门，并且推导出密钥的对吧？

## 输入格式
共一行，包含一个正整数$\red n$;

## 输出格式
共一行，为n对应的密钥
## 样例
#### 输入样例
```C
20
```
#### 输出样例
```C
10100
```
## 提示
对于所有的数据满足
$\red {1<=n<=100}$;
