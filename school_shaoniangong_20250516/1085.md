## 题目描述

一颗彗星的后面有一个不明飞行物（`UFO`），这个`UFO`经常到地球上来寻找忠实的追随者，把他们带到宇宙中去。

但由于舱内空间有限，它们每一趟只能带一组追随者。

尽管如此，外星人仍然想出了一个妙法来决定带谁走：以$\red{A}$代表$\red{1}$，$\red{B}$代表$\red{2}$，……$\red{Z}$代表$\red{26}$，`USACO`即`21*19*1*3*15=17955`，

倘若此组人的组名所代表的数字与彗星的名字所代表的数字分别除以$\red{47}$，余数相同，则彗星名与组名相匹配， `UFO`带此组人飞向宇宙，余数不同则不匹配，故不带。

编程任务： 写一程序，打印出彗星名与组名是否相匹配，是打印"`GO`"，否打印"`STAY`"；同时打印出两者的余数。

## 输入格式

输入文件包含两行，第一行为慧星名，第二行为组名，名字字符串小于$\red{500}$字符。

## 输出格式

输出共两行：

第一行输出："`GO`" 或者 "`STAY`"

第二行：

如果两者的余数一样输出“`r1=r2=`”之后还有余数的值。

若不一样输出各自的余数值:"`r1=`余数 `r2=`余数“ `r1` `r2` 之间用空格隔开

## 样例

#### 输入样例

```C
COMETHALEBOPP
HEAVENSGATE
```

#### 输出样例

```C
GO
r1=r2=17
```
