## 题目描述

农民约翰的$\red N$头奶牛（编号为$\red {1..N}$）计划逃跑并加入马戏团，为此它们决定练习表演杂技。

奶牛们不是非常有创意，只提出了一个杂技表演：

叠罗汉，表演时，奶牛们站在彼此的身上，形成一个高高的垂直堆叠。

奶牛们正在试图找到自己在这个堆叠中应该所处的位置顺序。

这N头奶牛中的每一头都有着自己的重量$\red {W_i}$以及自己的强壮程度$\red {S_i}$。

一头牛支撑不住的可能性取决于它头上所有牛的总重量（不包括它自己）减去它的身体强壮程度的值，现在称该数值为风险值，风险值越大，这只牛撑不住的可能性越高。

您的任务是确定奶牛的排序，使得所有奶牛的风险值中的最大值尽可能的小。

## 输入格式

第一行输入整数$\red N$，表示奶牛数量。

接下来$\red N$行，每行输入两个整数，表示牛的重量和强壮程度，第$\red i$行表示第$\red i$头牛的重量$\red {W_i}$以及它的强壮程度$\red {S_i}$。

## 输出格式

输出一个整数，表示最大风险值的最小可能值。

## 样例

#### 输入样例

```C
3
10 3
2 5
3 3
```

#### 输出样例

```C
2
```

## 提示

$\red {1≤N≤50000}$,
$\red {1≤W_i≤10,000}$,
$\red {1≤S_i≤1,000,000,000}$
