## 题目描述

给出一个整数 $x$，求 $⌊\frac{x}{10}⌋$。

$⌊\frac{x}{10}⌋$ 的意思是 $x$ 除 $10$ 的结果向下取整，例如：$⌊4.7⌋=4,⌊−2.4⌋=−3,⌊5⌋=5$。

## 输入格式

输入一个整数 $x$。

## 输出格式

输出 $⌊\frac{x}{10}⌋$ 的结果。

```input1
47
```

```output1
4
```

```input2
-24
```

```output2
-3
```

```input3
50
```

```output3
5
```

```input4
-30
```

```output4
-3
```

```input5
987654321987654321
```

```output5
98765432198765432
```

## 数据范围

* $−10^{18}≤x≤10^{18}$
