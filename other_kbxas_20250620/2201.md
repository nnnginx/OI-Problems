## 题目描述

羊村的供水系统搞砸了，隔壁牛村捐赠的的矿泉水刚刚送达，村长让喜羊羊们排队领水，已知有 $n$ 个羊村村民正在排队取水，懒羊羊不知道他在队伍的具体哪个位置，但他知道有不少于 $a$ 个人在他前面，有不多于 $b$ 个人在他后面，你能帮忙计算一下懒羊羊有多少个可能的位置吗？

## 输入格式

输入一行包含三个整数 $n,a,b$（$0 \le a,b<n \le 100$）。

## 输出格式

输出一行包含一个整数表示可能的位置数。

```input1
3 1 1
```

```output1
2
```

```input2
10 1 3
```

```output2
4
```
