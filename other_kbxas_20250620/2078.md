## 题目描述

给定一个以十进制表示的分数 $a/b$，保证 $a<b$，请将它化成一个二进制的小数。若存在循环节，用圆括号表示循环节。例如：

$1/4=0.01$

$7/8=0.111$

$1/5=0.(0011)$

## 输入格式

两个整数：表示 $a$ 与 $b$。

## 输出格式

一串数字：表示 $a/b$ 的二进制小数表示。

```input1
1 4
```

```output1
0.01
```

```input2
7 8
```

```output2
0.111
```

```input3
1 5
```

```output3
0.(0011)
```

## 数据范围

* 对于 $30\%$ 的数据，$1≤a<b≤30$；
* 对于 $60\%$ 的数据，$1≤a<b≤5,000$；
* 对于 $100\%$ 的数据，$1≤a<b≤2,000,000$；
