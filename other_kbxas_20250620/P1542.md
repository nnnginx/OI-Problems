## 题目描述

给定一个非负整数数组，统计里面每一个数的出现次数。我们只统计到数组里最大的数。

假设$Fmax（Fmax<10000）$是数组里最大的数，那么我们只统计{$0,1,2.....Fmax$}里每个数出现的次数。

## 输入格式

第一行$n$是数组的大小。$1 ≤ n ≤ 10000$。

紧接着一行是数组的$n$个元素。

## 输出格式

按顺序输出每个数的出现次数，一行一个数。如果没有出现过，则输出$0$。

```input1
5
1 1 2 3 1
```

```output1
0
3
1 
1
```

## 样例说明

样例中的数组，最大的数是$3$，因此我们只统计{$0,1,2,3$}的出现频数。
