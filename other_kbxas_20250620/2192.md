# Background

小Q空的时候挺喜欢玩玩电脑游戏的。自从编程技术提高后，他就想，要是自己也能开发出一款游戏来，那该多好啊！不过，小Q也布着急，先练好基本功再说。Windows中就有一款叫扫雷的小游戏，挺好玩的，不过想编出一个来。还真不容易。小Q就自己想了一种简单的扫雷游戏:在n行2列的方格棋盘。左列某些方格内埋有地雷，而右列每个方格中都有一个数字（0-3），第I格的数字表示：左列第I-1、I、I+1格（即:上、中、下三格）中埋雷的总数。如下图所示：左图是初始状态，右图是扫雷完成的状态（插小旗的方格内有雷）。

![](http://www.bitcode.vip/upload/pimg1171_1.png)

你的任务是：根据右列的数字分析出左列格子中的地雷（0表示无雷。1表示有雷），并且统计出左列格子中地雷的总数。小Q想，如果这样的任务完成了，相信输出更复杂的扫雷游戏也就为期不远了。

## Input

共2行，第一行，一个整数N（2≤N≤40），第二行有N个数字（以一个空格相隔），表示右列格子中的数字，输入数据保证正确有解。

## Output

共2行，第一行是N个0、1数字（没有空格相隔），表示左列每格中有无地雷。第二行一个整数，表示地雷总数。

# Samples

```input1
7
1 2 3 2 2 2 2
```

```output1
0111011
5
```
