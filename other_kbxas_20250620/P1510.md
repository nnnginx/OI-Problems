## 题目描述

有$N(4≤N≤100)$名同学站成一队，已知每名同学的身高 (单位：厘米)，请比较该队中任意相邻两名同学的身高差 (取正值)并输出最大的身高差。

例如：$N=8$，$8$名同学的身高分别是$151,120,144,182,160,158,147,161$，相邻两名同学的身高差依次是$31,24,38,22,2,11,14$，其中最大身高差是$38$。

## 输入格式

第一行输入一个正整数$N(4≤N<100)$，表示站队同学人数；

第二行输入$N$个正整数($80\le$正整数$\le220$)，表示的队列中学生的身高(单位：厘米)，相邻两个正整数之间用一个空格隔开。

## 输出格式

输出一个非负整数，表示相邻两名同学中最大身高差。

```input1
8
151 120 144 182 160 158 147 161
```

```output1
38
```
