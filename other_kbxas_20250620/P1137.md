## 题目描述

小明刚刚学习了每月有多少天，以及如何判断平年和闰年，想到可以使用编程方法求出给定的月份有多少天。你能做到吗？

## 输入格式

输入一行，包含两个整数，分别表示一个日期的年、月。

## 输出格式

输出一行，包含一个整数，表示输入月份有多少天。

```input1
1926 8
```

```output1
31
```

```input2
2000 2
```

```output2
29
```

## 提示

闰年的判定方法有两种：

①能被4整除，但是不能被100整数的年份是闰年

②能被400整除的年份是闰年

数据保证 $1000 \leq y \leq 3000$，$1 \leq m \leq 12$。
