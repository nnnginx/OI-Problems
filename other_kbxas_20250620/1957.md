## 题目描述

编写程序，计算下列分段函数 $y=f(x)$ 的值。结果保留到小数点后两位。

$y=x+1; \quad  0<x<100$

$y=x-1; \quad 100≤x<200$

$y=-1; \quad \quad  其他$

## 输入格式

一个浮点数 $x$。

## 输出格式

输出 $x$ 对应的分段函数值：$f(x)$。结果保留到小数点后两位。

```input1
50
```

```output1
51.00
```

```input2
150
```

```output2
149.00
```

```input3
400
```

```output3
-1.00
```
