# Background

美羊羊给喜羊羊和沸羊羊出了一道难题，说谁能先做出来，我就奖励给他我自己做的一样礼物。沸羊羊这下可乐了，于是马上答应立刻做出来，喜羊羊见状，当然也不甘示弱，向沸羊羊发起了挑战。

可是这道题目有一些难度，喜羊羊做了一会儿，见沸羊羊也十分头疼，于是就来请教你。

题目是这样的：

把自然数Ｎ（N<＝100）分解为若干个自然数之和，求出有几种情况。

如N＝5时，有7种情况

5=1+1+1+1+1

5=1+1+1+2

5=1+1+3

5=1+2+2

5=1+4

5=2+3

5=5

怎么样？你要加油帮助喜羊羊哦！


## Input

一个自然数N(N<＝100）

## Output

 无序拆分的种数

# Samples

```input1
5
```

```output1
7
```
