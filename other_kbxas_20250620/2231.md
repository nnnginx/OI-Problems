## 题目描述

小爱投入在学习上的总时间为 $n$ 秒。请将它转换成以小时、分钟和秒组成的时间格式，中间以冒号 `:` 分割。

例如 $n=3600$，输出 `1:0:0`，因为 $3600$ 秒恰好为一小时。

## 输入格式

单个整数：表示 $n(0≤n≤1,000,000,000)$。

## 输出格式

三个整数，由 `:` 分隔。第一个数表示小时，第二个数表示分钟，第三个数表示秒数。

```input1
100
```

```output1
0:1:40
```

```input2
3662
```

```output2
1:1:2
```

```input3
35999999
```

```output3
9999:59:59
```
