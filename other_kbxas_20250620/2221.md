## 题目描述

小明的数学老师喜欢玩数字游戏，一天他在课上给同学们板书了一条等式 $A×B+C=N$，他问小明，如果老师告诉你 $N$ 是多少，你能否快速得出到底有多少组正整数 $(A,B,C)$ 满足这条等式。小明的数学水平一般，请你帮小明完成这个问题。

## 输入格式

输入一行一个正整数 $N$。

## 输出格式

输出一行一个整数，表示满足条件的正整数组 $(A,B,C)$ 的个数。

```input1
3
```

```output1
3
```

### 样例 $1$ 解释

$(A,B,C)=(1,1,2),(1,2,1),(2,1,1)$

```input2
100
```

```output2
473
```

```input3
1000000
```

```output3
13969985
```

## 数据范围

对于 $100\%$ 的数据

* $2≤N≤10^6$
* 所有的输入都是整数
