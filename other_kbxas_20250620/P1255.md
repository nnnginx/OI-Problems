## 题目描述

已知线段的两个端点的坐标 $A(x_a,y_a)$，$B(x_b，y_b)$，线段长度计算公式为：$\sqrt{(x_a-x_b)^2+(y_a-y_b)^2}$求线段$AB$的长度，保留到小数点后$3$位。

## 输入格式

第一行是两个实数 $x_a，y_a$，即 $A$ 的坐标。

第二行是两个实数 $x_b，y_b$，即 $B$ 的坐标。

输入中所有实数的绝对值均不超过 $10000$。

## 输出格式

一个实数，即线段 $AB$ 的长度，保留到小数点后 $3$ 位。

```input1
1 1
2 2
```

```output1
1.414
```

## 提示

在C++中，使用函数sqrt()求平方根，具体表达式为：$sqrt((x_a-x_b)*(x_a-x_b)+(y_a-y_b)*(y_a-y_b))$
