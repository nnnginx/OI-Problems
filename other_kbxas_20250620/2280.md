## 题目描述

输入两个不相等的四位正整数 $N$（$1000 \le N \le 9999$）和 $M$（$1000 \le M \le 9999$），其中 $N<M$，中间以一个空格隔开，输出 $N$ 与 $M$ 之间（包含 $N$ 和 $M$）所有满足要求的正整数且正整数之间以一个英文逗号隔开。

要求每个正整数的各个位上的数都为偶数（注：$0$ 为偶数）

## 输入格式

在一行输入两个不相等的四位正整数 $N$ 和 $M$（$N$ 和 $M$ 之间以一个空格隔开）。

## 输出格式

输出 $N$ 与 $M$ 之间（包含 $N$ 和 $M$）所有满足要求的正整数且正数之间以一个英文逗号隔开。

```input1
4000 4008
```

```output1
4000,4002,4004,4006,4008
```
