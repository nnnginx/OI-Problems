## 题目描述

给出 $n$ 个数 $a_1,a_2,...,a_n$，求两个数相加的绝对值的最小值，即求 $|a_i+a_j|$ 的最小值，其中 $i$ 不等于 $j$。

## 输入格式

第 $1$ 行 $1$ 个正整数 $n$；

接下来 $1$ 行 $n$ 个整数，表示 $a_1,a_2,...,a_n$，每两个数之间用一个空格隔开。

## 输出格式

一行一个整数，表示答案。

```input1
5
-2 6 7 7 8
```

```output1
1
```

## 数据范围

对于 $40\%$ 的数据满足：$n\le10^3,-10^6 \le a_i \le 10^6$；

对于 $80\%$ 的数据满足：$n\le10^4,-10^6 \le a_i \le 10^6$；

对于 $100\%$ 的数据满足：$n\le10^5,-10^6 \le a_i \le 10^6$；
