## 题目描述

最近小明买了一本非常有趣的书。他知道，把这本书读完需要 $t$ 秒。小明想要尽可能快地完成阅读。

但他每天可能有一些工作和其他事情要做。未来几天，如果有空闲时间，他就可以进行阅读。请你帮助小明确定阅读完成的最低天数（从第一天开始最少需要到第几天）。

小明保证自己可以在不超过 $n$ 天内读完这本书。

请记住每一天有 $86400$ 秒。

## 输入格式

第一行包含两个整数 $n$ 和 $t$，表示天数和读的书所需的时间。

第二行包含 $n$ 个整数 $a_i$，表示未来几天他每天需要工作和做其他事情的时间。

## 输出格式

打印出最少需要到第几天，小明可以完成阅读本书。

确保答案不超过 $n$。

```input1
2 2 
86400 86398
```

```output1
2
```

### 样例1解释

第一天，小明都没有空；第二天有 $2$ 秒钟空，刚好完成阅读。

```input2
2 86400 
0 86400
```

```output2
1
```
### 样例2解释

第一天，小明所有时间可以都用来阅读 $1$ 天就可以完成。

## 数据范围

对于 $100\%$ 的数据，$1≤n≤100$，$1≤t≤10^6$，$0≤a_i≤86400$。
