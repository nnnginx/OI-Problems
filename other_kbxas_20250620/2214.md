## 题目描述

加、减、乘、除是小学数学中最基本的四则运算，现在小明同学在此基础上又定义了一种新运算，运算符为“Δ”，运算规则是:

$aΔb=a×b×(b+1)÷2$。

如: $5Δ3=5×3×(3+1)÷2=30$。

想必这样的问题不会难倒你，请你编程输出 $aΔb$ 的结果。

## 输入格式

输入只有一行，是用空格隔开的两个整数 $a$ 和 $b$。

## 输出格式

输出也只有一行，表示 $aΔb$ 的结果。

```input1
5 3
```

```output1
30
```

## 数据范围

$0≤a,b≤10000$，$a$ 和 $b$ 以及运算的结果都为整数。
