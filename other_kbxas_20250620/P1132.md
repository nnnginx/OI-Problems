## 题目描述

输入一个年份，判断这一年是否是闰年，如果是输出$1$，否则输出$0$。

闰年满足以下条件的任何一个即可。

1.能被$4$整除并且不能被$100$整除

2.能被$400$整除

## 输入格式

输入一个正整数$n$，表示年份。

## 输出格式

输出一行。如果输入的年份是闰年则输出$1$，否则输出$0$。

```input1
1926
```

```output1
0
```

```input2
1900
```

```output2
0
```

```input3
2000
```

```output3
1
```

```input4
1996
```

```output4
1
```

## 数据范围

数据保证，$1582 \leq n \leq 2020$ 且年份为自然数。
