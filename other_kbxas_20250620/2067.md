## 题目描述

给定四个整数 $a,b,c,d$，现在有两个整数 $x,y$，其中 $a≤x≤b,c≤y≤d$，请输出 $x−y$ 可能的最大值。

## 输入格式

输入四个整数 $a,b,c,d$。

## 输出格式

输出题目要求的答案。

```input1
0 10
0 10
```

```output1
10
```

### 样例 $1$ 解释

$(x,y)=(10,0)$，最大値 $x−y=10$。

```input2
-100 -100
100 100
```

```output2
-200
```

```input3
-100 100
-100 100
```

```output3
200
```

## 数据范围

对于 $100\%$ 的数据，$−100 \le a \le b \le 100,−100 \le c \le d \le 100$
