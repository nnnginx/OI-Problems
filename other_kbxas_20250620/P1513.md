## 题目描述

$bob$ 是一个不爱运动的人，他平常最喜欢喝的就是肥宅快乐水――可乐。但 $alice$ 觉得奶茶才是最好喝的饮料，某超市有卖这两种饮料，可乐 $3$ 元一瓶，雪碧 $5$ 元一瓶。现在他们有 $n$ 元钱，问有多少种不同的买饮料方案？（假设库存无限多）

## 输入格式

第一行为一个正整数 $n$ ，表示有 $n$ 块钱。

## 输出格式

输出买饮料的方案总数。

```input1
5
```

```output1
3
```

### 样例解释

方案 $(0,0),(1,0),(0,1)$。

## 数据范围

对于 $100\%$ 的数据，$n \leq 10^7$。
