## 题目描述

幼儿园有$28$个小朋友，每个小朋友分一个蛋糕吃，因此一共需要$28$个蛋糕。

一个大盒子可以装$8$块蛋糕，一个小盒子可以装$3$块蛋糕。现在有$n$个大盒子和$m$个小盒子，现在给每个小朋友一块蛋糕，问能剩下几个？数据保证蛋糕数量大于$28$。

## 输入格式

一行，两个整数$n,m$，具体意义见题目描述。

## 输出格式

一行，表示剩余蛋糕的数量。

```input1
2 5
```

```output1
3
```

```input2
2 4
```

```output2
0
```

## 提示

对于全部数据：$0\le n,m\le100$
