## 提示信息

阶乘定义：一个正整数的阶乘是所有小于及等于该数的正整数的乘积。

例如：$3$ 的阶乘 $6$（也就是 $1*2*3$ 的结果）

例如：$5$ 的阶乘 $120$（也就是 $1*2*3*4*5$ 的结果）

## 题目描述

输入一个正整数 $N$（$3 \le N \le 20$），输出 $1$ 到 $N$ 之间（包含 $1$ 和 $N$）所有正整数阶乘的和。

例如：输入为 $3$，$1$ 的阶乘为 $1$，$2$ 的阶乘为 $2$，$3$ 的阶乘为 $6$，$1+2+6=9$，则输出 $9$。

## 输入格式

输入一个正整数 $N$（$3 \le N \le 20$）。

## 输出格式

输出 $1$ 到 $N$ 之间（包含 $1$ 和 $N$）所有正整数阶乘的和。

```input1
3
```

```output1
9
```
