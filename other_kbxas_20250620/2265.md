## 题目描述

给定一个正整数 $n$，定义一个数字的根为它的十进制数字之和，例如 $1234$ 的数根为 $1+2+3+4$，请判定$n$ 的数根能否整除 $n$。

## 输入格式

单个整数表示 $n$

## 输出格式

如果 $n$ 的数根可以除尽 $n$，输出 `Yes`，否则输出 `No`

```input1
1234
```

```output1
No
```

```input2
333
```

```output2
Yes
```

## 数据范围

$30\%$ 的数据，$1≤n≤10^6$

$60\%$ 的数据，$1≤n≤10^{18}$

$100\%$ 的数据，$1≤n≤10^{100,000}$
