## 题目描述

一个交替01串被定义为一个字符串，只有字符“0”和“1”组成，且任意相邻两个字符都是不同的。例如：”1“，”10101“，”0101010“都是交替01串

现在给定一个01字符串s，问其中最长的连续子串，是交替01串的长度是多少？

## 输入格式

输入共一行，一个字符串s，只有字符”0“或者”1“组成。

## 输出格式

输出一个整数，表示最长的交替01串长度。

```input1
1110111
```

```output1
3
```

## 数据范围

对于30%的数据，s的长度<=100

对于70%的数据，s的长度<=1000

对于100%的数据，s的长度<=1000000
