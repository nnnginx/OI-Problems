## 题目描述

有一块矩形蛋糕，小明从蛋糕的正上方横向和纵向各切 $N$ 刀 $(1 < N < 100)$，每刀互不重合。计算出蛋糕被切割成了几块。

例如：$N = 2$，蛋糕被切割成$9$块。

![](http://www.csp-js.com.cn/file/2/%E5%88%87%E5%89%B2%E8%9B%8B%E7%B3%95.jpg)

## 输入格式

输入一个正整数 $N(1 < N < 100)$，表示蛋糕被横向和纵向各切割的刀数。

## 输出格式

输出一个整数，表示蛋糕最后被切割成了几块。

```input1
2
```

```output1
9
```
