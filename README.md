## English Version

### Heads up  

This article has important info―make sure to read it all! We’ll abbreviate the Olympiad in Informatics as **OI** here.  

### About This Project  

This is a repository packed with high-quality OI problems (more than 60,000 questions). We’re using crawlers to gather questions from a wide range of sources, including school databases, popular online judges, and classic competitions.  

Each problem source gets its own folder. We clearly label the platform type, platform name, and crawl time. Every update ensures that no existing questions are deleted; we just add new ones, except for duplicates.

This project uses the **GPL3 license**, with an extra note: **Commercial or profit-driven use is not allowed**.  

### Known Issues  
1. **Duplicate data**: We’ve removed identical problems *within the same platform*, keeping only one copy.  
2. **Missing test data & constraints**: Use the built-in problem viewer to jump to the original site for full details or to submit solutions.  
3. **Missing problem titles**: Sadly, we noticed this after collecting 10k+ problems. But you can search titles via the problem viewer.  
4. **Encoding conflicts**: Some files may show garbled text. Try switching encoding (GB2312 or UTF-8)―we’ve confirmed files are intact.  
5. **Broken problems**: A few have missing/corrupted descriptions due to source issues. We’ve cleaned most junk data―report any leftovers.  
6. **External content**: Images/PDFs linked from original sites won’t load here. Use the source links (see below) to view them manually.  
7. **Multi-language questions**: Some questions may have prompts in multiple languages, but we weren't able to scrape them all, so we only kept one language. Users can go back to the original site to check out the other languages.

## 中文版本

### 前言

这篇文章有一些重要的信息，所以一定要读完！本文将信息学奥林匹克竞赛缩写为 OI。

### 项目简介

这是一个存储大量优质 OI 题目的仓库（超过六万题），我们将使用爬虫从广泛的来源收集题目，包括学校题库、著名的评测系统题库和经典比赛。

每个题库都会有独立的文件夹，我们将明确标注平台类别、平台名称、爬取时间，每次更新保证不会删除原有题目，只会添加新题目，去重过滤除外。

该项目采用 GPL3 协议，特别附加的是，不能用于商业目的或牟利。

### 潜在问题

1. 重复数据：我们已经尽力将同平台内的相同题目删除，仅保留一个。
2. 缺少评测数据和限制数据：用户可通过提供的题目查看器快速跳转至原网站，方便查看完整题目信息并提交评测。
3. 缺少题目名称：遗憾的是在已有上万个题目后我们才发现这一点，但用户可通过提供的题目查看器检索题目名称。
4. 编码不统一：在打开文件时可能遇到乱码，请尝试更改编码（GB 2312 或 UTF-8），我们已确认文件正常。
5. 无用题目：部分题目因原平台问题，没有正确的题面，我们已经尽力删除垃圾数据，如果你认为有这样的题目，请反馈给开发者。
6. 内容加载：由于某些题目引用了来自原始网站的图片或PDF，无法正确加载，因此我们将提供原始网站链接（自行查阅本文其他章节），以便手动查看文件。
7. 多语言题面：某些题目可能包含多语言题面，我们未能全部爬取，因此只保留了一种语言，用户可回到原始网站查看其他语言。
