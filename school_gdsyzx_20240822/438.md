
### 问题描述

《三国演义》中曹军不适水性，采用铁索连环战术。

现在，给出 $N$ 艘战船的战力 $S_i$，给出 $M$ 个命令，每个命令：指定两条战船的编号 $A$ 和 $B$ ，让这 $2$ 条战船所在的战船群合并连在一起成为新的战船群。
请你求出有多少个战船群（一艘战船也算），并从小到大输出每个战船群的战力。

### 输入格式

第一行，两个正整数 $N,M。(0<N<=1000,0<M<=N*N)$

第二行，$N$ 个整数 $S。(0<S<=10000)$

第3..M+2行， $M$ 个指示，两个整数 $A,B$，$A$ 和 $B$ 表示编号。$(0<=A,B < N)$

### 输出格式

第一行，一个正整数，表示战舰群的个数。

第二行，多个用空格隔开的正整数，从小到大表示战舰群的战力。

### 输入样例1

```
3 3
1 1 1
0 1
1 2
2 0
```

### 输出样例

```
1
3
```

### 输入样例2

```
4 1
1 1 1 1
1 2
```

### 输出样例2

```
3
1 1 2
```
