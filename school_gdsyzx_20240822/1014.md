# Description
Farmer John 有 N（1 ≤ N ≤ 50000）头奶牛沿着一维的栅栏吃草，第 i 头奶牛在目标点 x（i），它的身高是 h（i）（1 ≤ x（i），h（i） ≤ 1000000000）。

当一头奶牛左边 D 距离内而且右边 D 距离内有身高至少是它的两倍的奶牛（1 ≤ D ≤ 1000000000），它就会觉得拥挤。

请计算觉得拥挤的奶牛的数量。

# Format

## Input
一行，两个整数 N 和 D。

第 2 至 第 n + 1 行，每行两个整数 x（i）和 h（i），所有牛的位置都是不同的。

## Output
一行，觉得拥挤的奶牛的数量。

# Samples

```input1
6 4 
10 3 
6 2 
5 3 
9 7 
3 6 
11 2 
```

```output1
2
```

# Explanation
共有 6 头牛，当距离在 4 以内时会感到拥挤，奶牛 1 在x = 10 的位置，高度 h = 3，依此类推。x = 5 和 x = 6 位置的奶牛都很拥挤。
