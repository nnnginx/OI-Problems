# Description
在足球比赛中，各国主要以联赛方式进行：

联赛中，每个队伍都会两两进行主客场交手，获得胜利的队伍全取 3 分；失利的队伍不得分；若双方战平，则各得 1 分。而统计球队的积分，则难坏了小明，他想要请你开发一个软件，帮他统计一下整个赛季结束之后各个球队的积分情况，以便向外界公布。

# Format

## Input
第一行一个整数 N，表示共有 N 个球队。2 ≤ N ≤ 50。

接下来 N 行，表示每个球队在主场作战的情况。每行共 N 个元素，“W”表示主队获胜，“D”表示双方战平，“L”表示主队失利。自己和自己用“-”表示。

## Output
共 1 行。N 个数，表示 N 个球队的积分情况，中间用空格分隔，最后一个球队后不要有空格。

# Samples

```input1
3
-WW
W-W
WW-
```

```output1
6 6 6
```
