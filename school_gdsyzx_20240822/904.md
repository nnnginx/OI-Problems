# Description

在smoj网站上，有很多针对小学信息学入门的课程，把这些入门课程的题都刷一遍并理解之后，你就算正式的信息学选手啦。例如课程9的某一道题是这样的：

输入两个正整数 a 和 b，表示开始的年份和结束的年份，问从 a 年到 b 年有多少闰年？闰年只需要满足如下两个条件之一：

1、能整除 400。

2、能整除 4，但不能整除 100。

# Format

## Input

第一行 2 个整数 a 和 b，范围在 [1，1000000000]。

## Output

一个整数。

# Samples

```input1
1 10000
```

```output1
2425
```

# Limitation

对于 80% 的数据，b ≤ 100000000。
