# Description
Farmer John有两队不和的牛群，分别命名为M和C。现在M队里有一名叫Romeo（简称为R）的公牛喜欢上了C队里叫Juliet（简称为J）的母牛，R很想能偶遇到J，但他又不希望他的队员知道。

R和J希望能沿着草场围栏的一个尽可能大的区域一起会面，但这个区域里不能有太多的R的队友，因此他们俩很难寻找到机会。R已经知道沿着草场围栏有 N（1 ≤ N ≤ 1000）个队员在吃草，而这个围栏共建有 P 条木柱（标号为 1 至 P）（1 ≤ P ≤ 1000）。R的队友就在相邻的木柱间吃草（同一区域可能有多只奶牛）。R希望找出一个不超过 C（0 ≤ C ≤ 1000）名队友的最长的区域来跟J会面。


# Format

## Input
第一行，三个空格分开的整数：N P C。

接下来 N 行，每行一个整数 X（范围是 1 至 P - 1），表示每只奶牛在 X 与 X + 1 之间的区域吃草。

## Output
一行，一个整数，表示不超过 C 名队员的最长的区域。

# Samples

```input1
2 6 1
2
3
```

```output1
3
```
