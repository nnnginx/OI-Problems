# Description
我们已经知道 A 和 B 两个变量的交换方法有一种是：

用临时变量轮换：temp=A; A=B; B=temp;

![img](./1033/file/数列轮换2.png)

现在有 N 个数的数列，要求使用类似算法轮换。即每个数都向前移动一位（第 1 个数移动到最后一位），例如：N = 3

原数列 5 8 4

temp =a[0];

a[0]=a[1]; 8 8 4

a[1]=a[2]; 8 4 4

a[2]=temp; 8 4 5

请使用这种方法编程求出新的数列。

# Format

## Input
第一行 1 个正整数 N，范围在 [1，100]。

第二行 N 个正整数，范围在 [1，10000]。

## Output
一行 N 个整数。

# Samples

```input1
4
3 4 1 2
```

```output1
4 1 2 3
```
