# Description
冬令营结束了，同学们都很高兴，于是决定 n（1 ≤ n ≤ 100000）个人在操场围成一圈，按照顺时针的顺序 1，2，3，...，n 进行编号：从第一个人开始报一个数字 k（1 ≤ k ≤ 99），然后顺时针的下一个人会报数字 k + 1，...... 以此类推，当某个人报的数字含有 m 或者是 m 的倍数（2 ≤ m ≤ 9）时，这个人就退出游戏，其他人接着报数，如果数到 100，则变为 k 重新开始数，直到剩下最后一个同学，他就是赢家！刚刚参加冬令营完毕的你能找出来最后一个人是谁吗？

# Format

## Input
一行，三个整数 n，m，k，含义如题目所述。

## Output
一行，一个整数，为最后一个人的编号。

# Samples

```input1
5 2 3
```

```output1
3
```
