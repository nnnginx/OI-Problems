# Description
假设在周末舞会上，男士们和女士们进入舞厅时，各自排成一队。跳舞开始时，依次从男队和女队的队头上各出一人配成舞伴。规定每个舞曲能有一对跳舞者。若两队初始人数不相同，则较长的那一队中未配对者等待下一轮舞曲。现要求写一个程序，模拟上述舞伴配对问题。

# Format

## Input
第一行男士人数 m 和女士人数 n；

第二行舞曲的数目 k。

## Output
共 k 行，每行两个数，表示配对舞伴的序号，男士在前，女士在后。

# Samples

```input1
4 3
6
```

```output1
1 1
2 2
3 3
4 1
1 2
2 3
```
