# Description

字符串 S 由“0”、“1”构成，你把 S 任意**相邻两个字符**构成的子串全部抄写下来。例如：

S = "0010111111"，那么抄写下来的子串有 9 个，分别是："00"，"01"，"10"，"01"，"11"，"11"，"11"，"11"，"11"。这 9 个子串分成三大类：

第 1 类：子串不含“1”。

第 2 类：子串恰好含一个“1”。

第 3 类：子串含两个“1”。

现在给出三个整数：a，b，c，表示如果把字符串 S 的所有**相邻两个字符**构成的子串全部抄写下来，那么第 1 类的子串有 a 个，第 2 类的子串有 b 个，第 3 类的子串有 c 个，求满足上述条件的字符串 S。

数据保证：至少有一个字符串 S 满足题意，如果有多个 S满足题意，请输出字典序最小那个。

# Format

## Input

一行，3 个整数：a，b，c。0 ≤ a，b，c ≤ 100， a + b + c > 0。

## Output

一个字符串 S。

# Samples

```input1
1 3 5
```

```output1
0010111111
```

# Explanation

样例一说明：

这是题目描述中的例子，例如 1110011110 也是符合题意的，但 0010111111 才是字典序最小的。
