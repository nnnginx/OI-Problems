# Description
给出 n 个区间，请合并所有重叠的区间，输出合并后的区间数。

# Format

## Input
第 1 行：1 个正整数 N，不超过 1000。

第 2 至第 2 +  N - 1 行：每行 2 个整数，范围 [1，10000]，表示一个区间的开始和结束数。

## Output
输出一个整数。

# Samples

```input1
4
1 3
2 6
8 10 
15 18
```

```output1
3
```
```input2
2
1 3
4 5
```

```output2
1
```

# Explanation
样例一说明：

合并后的区间有：[1，6]，[8，10]，[15，18]。

样例二说明：

合并后的区间有：[1，5]。
