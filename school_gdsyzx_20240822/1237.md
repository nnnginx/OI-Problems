# Description
队列操作题。根据输入的操作命令，操作队列（ 1 ）入队、（ 2 ）出队并输出、（ 3 ）计算队中元素个数并输出。

# Format

## Input
第一行一个数字 N。

下面 N 行，每行第一个数字为操作命令（ 1 ）入队、（ 2 ）出队并输出、（ 3 ）计算队中元素个数并输出。

## Output
若干行，每行显示一个2或3命令的输出结果。

注意：2 出队命令可能会出现空队出队（下溢），请输出“ no ”，并退出。

# Samples

```input1
7
1 19
1 56
2
3
2
3
2
```

```output1
19
1
56
0
no
```
