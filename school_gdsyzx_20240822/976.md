# Description

在百度百科中说道：奇数是一种数学术语，整数中，能被 2 整除的数是偶数，不能被 2 整除的数是奇数，偶数可用 2k 表示，奇数可用 2k + 1 表示，这里 k 是整数。奇数包括正奇数、负奇数。其中注意：0 不是奇数，是偶数。

# Format

## Input

输入文件共 1 行，用 2 个空格分隔的整数 A 和 B（-20000 ≤ A ≤ B ≤ 20000）

## Output

输出文件包含 N + 1 行。第 1 行一个整数 N，表示在 A 到 B 的区间（包括 A 和 B）中共有 N 个奇数。接下来的 N 行，每行一个区间 A 到 B 中的奇数（从小到大输出）。

# Samples

```input1
2 5
```

```output1
2
3
5
```

```input2
-1 8
```

```output2
5
-1
1
3
5
7
```

# Limitation

对于 30% 的数据，-200 ≤ A ≤ B ≤ 200；

对于 50% 的数据，-2000 ≤ A ≤ B ≤ 2000；

对于 70% 的数据，-10000 ≤ A ≤ B ≤ 10000；

对于 100% 的数据，-20000 ≤ A ≤ B ≤ 20000。
