#### **题目描述**

从前有一个肥宅黄先生，一天他心血来潮地想去锻炼（其实是为了他的女神），然而他每天的体力值都不一样，他希望你能帮他找出他一天可以走多少条路。

#### **输入**

有$n(1<=n<=10^5)$段路程，接下来$n$个数代表走完这段路需要的体力值$s_i(1<=s<=10000000)$。

有$m(1<=m<=10^5)$天，接下来m行代表他每天的体力值$k(1<=k<=10000000)$他一天可以走完小于等于他体力值k的路。

#### **输出**

有$m$行，接下来每个数代表他那天可以走多少段路程。

#### **样例输入**

```
10 
1 1 2 2 2 3 3 3 4 4 
3 
1 
2 
3
```

#### **样例输出**

```
2 
5 
8
```
