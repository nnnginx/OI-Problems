
### 【问题描述】

有一个 $n$ 行 $m$ 列的矩阵 $G$ ，矩阵中每个格子有一个整数 $0$ 或者 $1$，一个或多个相邻的整数为 $1$ 的格子，可以组成正方形（单独 $1$ 个格子也是正方形），请你找出由1组成的正方形中面积最大的那个，输出这个正方形的面积（每个格子为 $1$ 个单位面积）。

### 【输入格式】

第一行有 $2$ 个用空格隔开的整数 $n$ 和 $m$，表示矩阵有 $n$ 行 $m$ 列（$1<=n,m<=100$）。

接下来的n行是一个 $n*m$ 的矩阵，矩阵中第 $i$ 行 $j$ 列的有一个 $1$ 或 $0$ 的整数。每行的 $m$ 个整数之间用空格隔开。

### 【输出格式】

共一行，包含一个整数，表示路径中经过的格子的数的总和最大值。

### 【输入样例】

```
3 3
1 1 0
1 1 1
0 1 0
```

### 【输出样例】

```
4
```

### 【限制】

​	30%的数据满足： $1<=m,n<=30$

​	100%的数据满足： $1<=m,n<=100$
