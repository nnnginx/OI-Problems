# Last Duel

## 题面描述

给定 $n$ 种咒语，每种咒语上有一个伤害值 $a_i$ 且每个咒语只能施放一次。

哈利要先施一种咒语给伏地魔。假设当前最后一次施放的魔法伤害值为 $x$，接下来，由于伏地魔施了一种防御魔法，你每次只能施魔法值为 $x$ 或 $(x + 1) \bmod m$ 的咒语。

你希望最后剩下的未施放的咒语的伤害值总和最低(意思就是对伏地魔要最大的伤害)

## 【输入格式】

第一行两个数 $n$，$m$。

接下来 $n$ 个数，表示伤害值 $a_i$。

## 【输出格式】

输出最小和值。

## 【数据范围】

$1 \le n \le 2 \times 10^5$

$2 \le m \le 10^9$

保证 $0 \le a_i < m$。

## 样例 #1

### 样例输入 #1

```
9 7
3 0 2 5 5 3 0 6 3
```

### 样例输出 #1

```
11
```

## 样例 #2

### 样例输入 #2

```
1 10
4
```

### 样例输出 #2

```
0
```

## 样例 #3

### 样例输入 #3

```
20 20
18 16 15 9 8 8 17 1 3 17 11 9 12 11 7 3 2 14 3 12
```

### 样例输出 #3

```
99
```

## 提示

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 2\ \leq\ M\ \leq\ 10^9 $
- $ 0\ \leq\ A_i\ \lt\ M $
- 输入皆为整数
