#### **题目描述**

输入$4$个整数，$a, b, c, d$。求$a$，$b$的最大公约数$r1$，再求$c，d$的最大公约数$r2$，输出$r1$和$r2$中较大的值。

#### **输入格式**

一行$4$个整数$a, b, c, d：$范围在$[1,2000000000]$。

#### **输出格式**

一个整数,$ r1$和$r2$中较大的值

#### **输入样例1**

```
20 15 2 3
```

#### **输出样例1**

```
5
```

#### **输入样例2**

```
32 80 24 12
```

#### **输出样例2**

```
16
```

#### **提示**

```
求最大公约数可使用辗转相除法：

int gcd(int x, int y){

       while(y){

              int r=x %y;

              x=y; y=r;

       }

       return x;

}
```
