# Description

在一个无穷大的桌面上，一开始没有小球，现在小明从左往右放小球，重复如下步骤无穷次。

1、在当前所有小球的最右边放 R 个红球。

2、在当前所有小球的最右边放 B 个黑球。

你的任务是计算从左往右看，前面的 N 个小球当中，有多少个红色小球。

# Format

## Input

第一行，3 个整数，N，R，B。1 ≤ N ≤ $10^{18}$，R ≥ 0，B ≥ 0，0 < R + B ≤ $10^{18}$。

## Output

一个整数。

# Samples

```input1
8 3 4
```

```output1
4
```

```input2
8 0 4
```

```output2
0
```
