# Description

本学期小明学习了很多新单词，为了将本学期的新单词整理有序，小明想按单词的第一个字母将新单词分类（不区分大小写）。这样他就可以方便找到他的新单词。

由于单词的数量比类多，现在就请你来帮小明完成这个任务。

# Format

## Input

输入数据有若干行。

第一行，是一个整数 n（0 ＜ n ≤ 100000），表示新单词的个数。

接下来有 n 行由大写或小写字母组成的字符串，长度小于 10。

## Output

输出数据有输入数据有若干行。

第一行输出这 n 个单词一共有多少类。

接下来每行输出一个大写字母，和一个整数，表示以这个字母开头的新单词的个数。

# Samples

```input1
5
aabe
Bace
Fgbeg
bedgd
Aibed
```

```output1
3
A 2
B 2
F 1
```
