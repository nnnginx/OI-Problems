# Description
给出一个 n 个结点的二叉树，请求出二叉树的前序遍历，中序遍历和后序遍历。

# Format

## Input
第一位一个整数 n（0 ＜ n ≤ 26），表示二叉树有 n 个结点，以下 n 行，每行第一个为一个大写字母表示结点，后面为两整数，第一个表示左儿子序号，第二个表示右儿子序号，如果该序号为 0 表示没有。

## Output
共三行，第一行为二叉树的前序遍历，第二行为中序遍历，第三行为后序遍历。

# Samples

```input1
7
F 2 3
C 4 5
E 0 6
A 0 0
D 7 0
G 0 0
B 0 0
```

```output1
FCADBEG
ACBDFEG
ABDCGEF
```
