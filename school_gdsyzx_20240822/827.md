# Capture Hogwarts

## 题目描述

给出霍格沃兹的地图。这是一个 $n$ 个点 $m$ 条边的无向图。每条边有一个权值 $w \in \{0, 1\}$。$w = 0$ 表示这条边无法通过，已被许多霍格沃兹师生把守，$w = 1$ 则可以通过。

有 $k$ 个点上可以施古老而又邪恶的翻转魔法，第 $i$ 个点是 $s_i$。

伏地魔现在位于 $1$ 号点。每次，伏地魔可以做两件事情中的一件：

1. 移动。移到相邻的一个地方上，注意这条路一定是可以通行的。
2. 施翻转魔法。此时，全部路的边权取反。即：$w = 0$ 变成 $1$，$w = 1$ 变成 $0$。

请问你是否能够到达 $n$ 号点。如果可以，求出最少移动次数。

【输入格式】

第一行三个数 $n, m, k$。

接下来 $m$ 行，每行三个数 $u_i, v_i, w_i$表示一条连接 $u_i$ 与 $v_i$ 的边。

最后一行 $k$ 个数，表示可以施翻转魔法的位置。

【输出格式】

如果无法到达，输出 $-1$。否则输出最少移动次数。

【数据范围】

$2 \le n \le 2 \times 10^5$

$1 \le m \le 2 \times 10^5$

$1 \le k \le n$

保证 $1 \le u_i, v_i \le n$，且 $u_i \ne v_i$。

保证 $1 \le s_1 < s_2 < \cdots < s_k \le n$。

## 样例 #1

### 样例输入 #1

```
5 5 2
1 3 0
2 3 1
5 4 1
2 1 1
1 4 0
3 4
```

### 样例输出 #1

```
5
```

## 样例 #2

### 样例输入 #2

```
4 4 2
4 3 0
1 2 1
1 2 0
2 1 1
2 4
```

### 样例输出 #2

```
-1
```

## 提示

- $ 2\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ M\ \leq\ 2\ \times\ 10^5 $
- $ 0\ \leq\ K\ \leq\ N $
- $ 1\ \leq\ u_i,\ v_i\ \leq\ N $
- $ u_i\ \neq\ v_i $
- $ a_i\ \in\ \lbrace\ 0,\ 1\rbrace $
- $ 1\ \leq\ s_1\ \lt\ s_2\ \lt\ \cdots\ \lt\ s_K\ \leq\ N $
- 输入皆为整数
