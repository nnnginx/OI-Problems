# Description
有 N 个头奶牛判断依次通过一个小桥，小桥比较短，每次最多同时有 2 头奶牛在上面。已知每个奶牛的重量都不超过 200 公斤，重量都放在一个数组中。小桥最大承载是 350 公斤。

可是小萌萌发现队伍中有 2 头相邻的奶牛的重量和大于了 350 公斤，如果这样过桥会损坏小桥。小萌萌自己是 50 公斤，他决定自己插入到这 2 头奶牛之间充当一只“小奶牛”，这样就可以安全通过小桥了。请输出新的队伍重量的数组。

# Format

## Input
第一行 1 个正整数 N，范围在 [1，200]。有且只有 1 对相邻数和大于 350。

第二行 N 个不同的正整数，范围在 [1，10000]。

## Output
N + 1 个正整数。把 50 插入都恰当的位置。

# Samples

```input1
4
150 160 198 130
```

```output1
150 160 50 198 130
```
