#### **题目描述**

计算五边形的面积。

提示：从图中可以看出，五边形的面积是三个三角形面积。

![img](http://ssoi.noip.space/upload/201508/wubianxing.png)

#### **输入**

输入$b1,b2,b3,b4,b5,b6,b7$，共$7$个整数。

#### **输出**

输出五边形的面积，结果保留$2$位小数

#### **样例输入**

```
3 3 3 5 4 5 4
```

#### **样例输出**

```
18.00
```

#### **提示**

提示：由三边长求三角形面积的海伦公式：$S=sqrt((a+b+c)(a+b-c)(a+c-b)(b+c-a))/4$。请用$double$类型！！！
