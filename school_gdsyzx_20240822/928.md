# Description
加普有一个非常忙碌的周末。因为，下个周末是万圣节，他计划参加尽可能多的聚会。因为是万圣节，而且这些派对都是化装舞会，加普总是会选择与朋友们融为一体的着装，也就是说，当他参加由漫画迷朋友们安排的派对时，他会带着超人的服装去，但当派对是由竞赛伙伴们安排的时候，他会穿“中国邮递员”的服装。

因为他要参加万圣节晚上的一些派对，并穿上相应的服装，所以他会多次更换服装。因此，为了让事情变得简单一些，他可能会穿上一套又一套的服装再出发。（比如说用邮递员的服装盖住超人的服装）。每次排队前，他都可以脱下或穿上新的服装，也就是说，如果他先穿超人服装再穿邮递员服装，然后他想穿超人服装去参加派对，他可以脱下邮递员服装，也可以穿上新的超人服装。

但是要记住，加普不喜欢不先洗衣服就穿衣服，所以，在脱下邮递员服装后，他不能在万圣节晚上再用那件衣服，如果真的再次需要邮递员服装，他就得用新的。他可以脱下任意数量的服装，如果他脱下其中的 k 件服装，那就必须是最后的 k 件。（例如，他在 B 件服装前穿了 A 件服装，要脱下 A 件服装，就必须先脱下 B 件服装）。

考虑到派对和服装店情况，找到加普在万圣节晚上需要的服装的最小数量。

# Format

## Input
一行，一个整数 n（1 ≤ n ≤ 100），为派对的数量。

接下来包含 n 个整数，其中第 i 个整数 ci（1 ≤ ci ≤ 100），表示他将在派对 i 上要穿的衣服。他会按照顺序参加派对。

## Output
输出所需服装的最小数量。

# Samples

```input1
4
1 2 1 2
```

```output1
3
```
```input2
7
1 2 1 1 3 2 1
```

```output2
4
```
