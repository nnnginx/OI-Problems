# Description
桐桐把一个自然数 N 的正因子个数记为 F（N），例如 18 的所有正因子为 1、2、3、6、9、18，所以 F（18）= 6。现在给出 K，桐桐想求出所有满足 F（N）= K 的 N 中最小的数，你能帮助她吗？

# Format

## Input
第 1 行为 K，其中 0 ＜ K ≤ 80。

## Output
如果存在不大于 20000 的解，则输出这个 N，并输出相应的 K 个因子；否则输出“NO SOLUTION”。

# Samples

```input1
9
```

```output1
36
1 2 3 4 6 9 12 18 36
```
