### 【题目描述】

高斯在他还是小P孩的时候就求出$1+2+3+4+……+n=n*(n+1)/2$

LT在他还是小P孩的时候就求出$1/（1*2）+……+1/((n-1)*n)=1-1/n$;

现在，在你还是小P孩的时候，你要求出：
![image](./651/file/QSOuvBVpCDkQ0LSBkRKvm.jpeg)

### 【输入文件】

输入两个整数$n、m$

### 【输出文件】

输出占两行，第一行一个整数$X$，第二行整数$Y$，表示$S=X/Y$，且$X,Y$互质。

### 【样例输入】

```
1 2
```

### 【样例输出】

```
1
2
```

### 【数据规模】

$m>1,n>0$;

$50\%$的数据满足$n<=50;$

$100\%$的数据满足$n+m<=500$。
