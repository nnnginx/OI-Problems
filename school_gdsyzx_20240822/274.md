#### **题目描述**

五一劳动节来了，华华去商场去购物，他有$S$ $（1<=s<=100,000,000）$元钱，他有好多好多东西要买,不过肯定没有那么多钱可以全部买啦- -！。。。他想在他的钱财范围内可以买多少件物品。不过他很笨，每次只会挑选多钱的买，一旦不够钱，他就不买了。他想在$1$秒中之内可以知道自己买物品后会还剩多少钱，于是他找到了$Oier$的你，你能帮帮他吗？

#### **输入**

第一行$s$ 和$ n (1<=n<=100,000)$ 第二行到第$N+1$行，价钱，$(1<=$价钱$<=10^7)$

#### **输出**

一个数，为还剩下多少元

#### **样例输入**

```
6 3 
1 
2 
3
```

#### **样例输出**

```
0
```
