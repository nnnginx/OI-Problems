# Description

一个数列中有 n 个整数，其中一个数是好数的条件是：去掉该数后剩下的数如果从前往后数，奇数位置的数之和跟偶数位置的数之和相等，那么该数是“好数”。

给出 n 个整数的数列，问该数列中有多少个“好数”。

# Format

## Input

第一行，一个正整数 n。

第二行，n 个整数，表示该数列中的数 ai。

## Output

输出好数的个数。

# Samples

```input1
7
5 5 4 5 5 5 6
```

```output1
2
```

# Limitation

对于 40% 的数据，1 ≤ n ≤ $10^4$；ai ≤ 1000；

对于 100% 的数据，1 ≤ n ≤ 2 × $10^5$；ai ≤ 100000。
