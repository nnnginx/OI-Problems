# Description

运动场修好后，体艺节马上要开始了。体艺节上要用 N 个计时器，每个计时器是一个中间显示秒数的方形电子表。如下图是显示 5 秒和 201 秒的两个计时器：

![img](./891/file/计时器1.png)

为了检查这些计时器的质量，晨晨创新地开发设计出一个新游戏，鼓动了 M 个同学报名参加。游戏开始时她把这 N 个计时器排成一行，开始时间都是 0。然后她对这 M 个同学每个人发一条命令：

把第 si 个到第 ti 个计时器上时间都加 1 秒。

M 个同学完成指令后，N 个计时器上时间几乎都变了。晨晨要通过编计算机程序来计算所有计时器上的数字长度（位数）总和。例如下面 N = 3 个计时器的数字长度分别是 1、3、2，长度和是 6（= 1 + 3 + 2）。

![img](./891/file/计时器2.png)

# Format

## Input

第 1 行，2 个正整数 N 和 M。（N，M 均为大于 0 小于 200000 的整数）

下面有 M 行：每行 2 个整数 s 和 t。（0 ＜ s ＜ t ≤ N）

## Output

一整数，表示 N 个数总位数长度。

# Samples

```input1
5  10
1  3
2  4
2  3
1  3
1  4
1  4
1  4
2  4
2  4
2  4
```

```output1
7
```

# Explanation

样例一说明：

最后计时器上的数字分别是：5、10、10、7、0。

数字长度：1 + 2 + 2 + 1 + 1 = 7
