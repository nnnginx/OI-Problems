# 问题 F: 动态维护中位数

**内存限制：128 MB**    **时间限制：1 S**

#### 题目描述

**给出一个长度为N的非负整数序列**Ai**，对于所有**1≤k≤(N+1)/2**，输出**A1,A1∼A3,…,A1∼A2k−1**的中位数。即前**1,3,5,…**个数的中位数。**

#### 输入

**第11行为一个正整数N，表示了序列长度。 第22行包含N个非负整数Ai(Ai≤10^9)。**

#### 输出

**共**(N+1)/2**行，第i行为**A1,  A3 ,…,  A2k−1**的中位数。**

#### 样例输入

```
7
1 3 5 7 9 11 6S
```

#### 样例输出

```
1
3
5
6
```

#### 提示

**数据范围：10, 100, 1000, 5000, 8000, 10000, 20000, 30000, 50000, 70000, 100000**
