# Description

桐桐很喜欢研究数字，特别喜欢研究质数。一天，桐桐发现有一些数字可以表示成两个质数相乘的形式，比如，10 = 2 × 5，2，5 都是质数，所以 10 是一个“桐桐数”。所以桐桐决定考考你，她告诉你一个数 n，请你判断 n 是不是“桐桐数”。

# Format

## Input

一个数 n（1 ≤ n ≤ $2^{31}$-1）。

## Output

输出一行，如果 n 是一个“桐桐数”，则输出“It's a Tongtong number.”，否则输出“It's not a Tongtong number.”

# Samples

```input1
10
```

```output1
It’s a Tongtong number.
```
