#### **题目描述**

中央电视台的“幸运$52$”栏目深受观众喜爱，其中的“猜商品价格”的节目更是脍炙人口，现在请你编一个程序模拟这一游戏：由计算机随机产生$200$至$5000$之间的一个整数x，作为某件商品的价格，然后由你有$10$次猜数的机会，若你猜的数大了，则计算机输出提示“$gao$”，若你猜的数小了，则计算机输出提示“$di$”，若你猜对了，计算机会提示“$Ok$”，并输出你是第几次猜中，若$10$次都没猜中，计算机会提示“$No$”。

#### **输入**

第一行：输入一个整数$x(200≤n≤5000)$,表示当前商品的价格； 第二行：有$10$个数，表示猜的数。

#### **输出**

第一行：猜的结果； 第二行：若猜中，输出第几次猜中。若$10$次都没猜中，输出“$No$”。

#### **样例输入**

```
【输入样例1】
860 
1000 500 700 800 900 850 870 860 880 899 

【输入样例2】
860 
1000 500 700 800 900 850 870 850 855 859
```

#### **样例输出**

```
【输出样例1】
gao di di di gao di gao Ok 
8 

【输出样例2】
gao di di di gao di gao di di di 
No
```
