# Description
有 N 个整数从左到右排成一行，如果某个数等于它前面的 2 个数的和，就称这个数是可以表示的数。问给定的数列里有多少个数是可以表示的数。

# Format

## Input
第一行 1 个整数 N，表示数列有多少个整数。1 ≤ N ≤ 10000。

第二行 N 个正整数，每个正整数不超过 10000。

## Output
一个整数，有多少可表示的数。

# Samples

```input1
8
5 2 2 3 4 8 7 16
```

```output1
3
```

# Explanation
样例一说明：

4 = 2 + 2；8 = 5 + 3；7 = 3 + 4。
