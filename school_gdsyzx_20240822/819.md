# Correct charms

## 题目描述

给出 $N\,(1 \le N \le 52)$ 个长度为 $2$ 的咒语 $M_1,M_2,\dots ,M_N$，判断该 $N$ 个咒语是否满足以下条件：

- 第一个字符为 `H`（表示治疗），`D`（表示伤害），`C`（表示消灭），`S`（表示蛇佬腔） 其中之一。
- 第二个字符为 `A`，`2`，`3`，`4`，`5`，`6`，`7`，`8`，`9`，`T`，`J`，`Q`，`K` 其中之一，表示咒语在整类中的编号。
- 咒语两两不相等。即 $\forall i \not=j,S_i \not= S_j$。

若满足以上条件，输出 `Yes`，否则输出 `No`。

## 样例 #1

### 样例输入 #1

```
4
CK
H3
D5
S2
```

### 样例输出 #1

```
Yes
```

## 样例 #2

### 样例输入 #2

```
5
H3
DA
CK
H3
FK
```

### 样例输出 #2

```
No
```
