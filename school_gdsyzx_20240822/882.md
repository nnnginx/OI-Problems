# Description


小明班经常会进行语、数、英三科的统一考试，每次只要学生的考试总分比前一次高，老师就会发信息给学生家长进行表扬。现给出小明每次考试的三科成绩，问小明共有多少次能受到老师的表扬。

# Format

## Input

第一行，一个整数 n（n ≤ 100）；

接下来 n 行，每行三个 100 以内的整数。

## Output

一个整数，代表小明受表扬的次数。

# Samples

```input1
5
60 65 70
80 82 85
78 80 80
82 76 90
88 90 90
```

```output1
3
```

# Explanation

样例一说明：

第二次、第四次和第五次受到表扬。
