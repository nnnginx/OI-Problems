# [C++一本通-图论算法]例4-11 最短网络agrinet

【题目描述】

农民约翰被选为他们镇的镇长！他其中一个竞选承诺就是在镇上建立起互联网，并连接到所有的农场。当然，他需要你的帮助。约翰已经给他的农场安排了一条高速的网络线路，他想把这条线路分享给其他农场。为了用最小的消费，他想铺设最短的光纤去连接所有的农场。你将得到一份各农场之间连接费用的列表，你必须找出能连接所有农场并所用光纤最短的方案。每两个农场之间的距离不会超过100000.

【输入格式】

第一行：农场的个数N（3<=N<=100）。

第2行至最后一行：后来的行包含了一个N*N的矩阵，表示每个农场之间的距离。两个数字之间用一个空格隔开。

理论上，它们是N行，每行由N个用空格隔开的数组成，实际上，它们限制在80个字符，因此，某些行会紧接着另一些行。当然，对角线将会是0，因为不会有线路从第i个农场到它本身。

【输出格式】

只有一个输出，其中包含连接到每个农场的光纤的最小长度。

【输入样例】

```
4
0 4 9 21
4 0 8 17
9 8 0 16
21 17 16 0
```

【输出样例】

```
28
```
