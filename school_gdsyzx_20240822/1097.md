# Description
有 n 个数保存在数组 a[1...n]，从里面选出两个数，使得这两个数相加的和是 21，问有多少种不同的选法。

# Format

## Input
第一行，一个整数 n。1 ≤ n ≤ 50。

第二行，n 个整数，第 i 个整数是 a[i]。1 ≤ a[i] ≤ 20。

## Output
一个整数。

# Samples

```input1
2
1 20
```

```output1
1
```
```input2
4
10 10 11 11
```

```output2
4
```
```input3
1
8
```

```output3
0
```
