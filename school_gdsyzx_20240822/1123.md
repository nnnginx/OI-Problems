# Description
小明决定徒步完成一个 N 天的远足登山活动。第一天他带上一个大睡袋和足够的食物，从大本营出发。大本营的海拔高度为 0，在第 N 天，他必须回到大本营。在这 N 天中，他可以在他的大睡袋和简易的帐篷中入睡。

在每天的登山活动中，他可以选择向山顶出发，也可以选择下山（同一天只能选择一种方式），而且有如下两条规则：

1. 在登山过程中，他不能走的太快，海拔的升高的过程，空气的含氧量会发生变化，因此每天只能上升 A 米的海拔。

2. 在下山过程中，他也不能走的太快，否则他的膝盖无法承受冲击。每天他只能下降 B 米的海拔。

例如：某天早晨小明处在 470 米的海拔位置，A = 100 且 B = 200，那么他在这天中可以向上到达 570 米的位置，也可以下山达到海拔 270 米的位置，因此一天结束后，他可能处在的海拔位置在 270 至 570 米之间。

现在，给定 N，A 和 B，问小明最多能到达海拔多少米的高度。

# Format

## Input
第一行三个整数，N，A 和 B，表示 N 天的登山远足活动，每天可以向上爬海拔 A 米，也可以下山下降海拔 B 米。

## Output
输出一个整数，表示在第 N 天中能到达的最高海拔高度。

# Samples

```input1
3 7 10
```

```output1
10
```
```input2
5 40 30
```

```output2
80
```
# Explanation

样例一说明：

第一天向上爬 7，第二天向上爬 3 到达海拔 10，第三天回到大本营。

样例二说明：

第一天和第二天分别向上爬 40，后面3天下山分别 30 + 30 + 20。


# Limitation

对于 50% 的数据，2 ≤ n ≤ 20，1 ≤ A，B ≤ 50；

对于 100% 的数据，2 ≤ n ≤ 50，1 ≤ A，B ≤ 50。
