# Description
小明今天生日，邀请了一些朋友过来开生日会。妈妈专门去买了一个大蛋糕，蛋糕为一个 n × m 的矩形，现在想把这个蛋糕分成 1 × 2 的小块，并且要求必须是完整的小块，不能拼接。问一共能分多少块？

# Format

## Input
一行，两个正整数 n，m。

## Output
一行，一个整数 x，表示最多能分多少块。

# Samples

```input1
7 8
```

```output1
28
```

# Limitation

对于 50% 的数据，0 ＜ n，m ＜ 10000； 

对于 100% 的数据，0 ＜ n，m ＜ $10^9$。
