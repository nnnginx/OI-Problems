# Description

光头强是一名伐木工，此番来森林伐木。美丽的大森林面临前所未有的危机，为了保护家园，一场旷日持久的家园保卫战开始了。看着森林被毁，熊兄弟决定决定把森林里最珍贵的果树移植到安全地方。移植前，熊二写下树的年龄是一个$2$​位数$a$，可是细心的熊大发现熊二把数写反了。请帮助熊二写出果树的正确年龄。

# Format

## Input

一行，一个正整数，表示$a$，范围在$[11...99]$。

## Output

一个正整数。

# Samples

```input1
89
```

```output1
98
```

```input2
27
```

```output2
72
```
