# Description
对于给定金额的人民币（以元为单位，如 345），输出支付该金额的各种面额的人民币数量，显示100 元，50 元，20 元，10 元，5 元，1 元各多少张，要求尽量使用大面额的人民币。

# Format

## Input
一个小于 1000 的正整数。

## Output
输出分行，每行显示一个整数，从上到下分别表示 100 元，50 元，20 元，10 元，5 元，1 元人民币的张数。

# Samples

```input1
735
```

```output1
7
0
1
1
1
0
```
