# Description
你参加了一个交换游戏。游戏开始你和另外两个人各拿了写有号码 1、2、4 的牌，然后节目主持人开始发一些指令，指令的内容是：持有 A 号牌的和持有 B 号牌的人，把牌交换。

请问 N 次这样的指令后，你持有的牌是几号？

# Format

## Input
第一行 1 个 N 整数，范围在 [1，100]。

下面有 N 行，每行 2 个不同的整数，A 和 B 是{1，2，4} 其中的 2 个，表示一次交换指令。

## Output
1 个整数（1、2、或 4）。

# Samples

```input1
3
1 4
2 4
4 1
```

```output1
2
```

# Explanation

本题要求不得使用数组。
