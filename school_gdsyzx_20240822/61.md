
### 【问题描述】

一个只包含 $A，B，C$ 三种字符的字符串，如果其中有连续的 $3$ 个由 $A，B，C$ 各一个组成的子串，则称这个字符串为 $fstring$ 字符串。

例如：$BAACAACCBAAA$ 就是一个 $fstring$ 字符串，而 $AABBCCAABB$ 则不是。

你的任务就是计算只包含 $A，B，C$ 三种字符且长度为 $n$ 的这种字符串有多少个不是 $fstring$字符串。

### 【输入格式】

一个整数 $n (1≤n≤30)$。

### 【输出格式】

一个整数。

### 【输入样例1】

```
2
```

### 【输出样例1】

```
9
```

### 【输入样例2】

```
3
```

### 【输出样例2】

```
21
```
