# Description
输入 2 个整数 N、M，输出 N 行，每行 M 个 “*” 组成的平行四边形。例如：N = 4，M = 5 时：

```
*****

_*****

__*****

___*****
```

（注：上面使用下划线“_”表示空格，以避免看不清造成误解）

# Format

## Input
第一行 2 个正整数：N、M，范围在[1，20]。

## Output
N 行，每行 M 个 “*” 号的平行四边形。

# Samples

```input1
3 4
```

```output1
****
_****
__****
```
