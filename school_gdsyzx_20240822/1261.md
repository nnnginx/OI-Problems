# Description
有 N 个任务，每个任务有一个最晚结束时间以及一个对应的奖励。在结束时间之前完成该任务，就可以获得对应的奖励。完成每一个任务所需的时间都是 1 个单位时间。有时候完成所有任务是不可能的，因为时间上可能会有冲突，这需要你来取舍。求能够获得的最高奖励。

# Format

## Input
第 1 行，一个数 N，表示任务的数量（2 ≤ N ≤ 50000）；

第 2 至第 N + 1 行，每行 2 个数，中间用空格分隔，表示任务的最晚结束时间 E[i] 以及对应的奖励 W[i]。（1 ≤ E[i] ≤ $10^9$，1 ≤ W[i] ≤ $10^9$）

## Output
输出能够获得的最高奖励。

# Samples

```input1
7
4 20
2 60
4 70
3 40
1 30
4 50
6 10
```

```output1
230
```
