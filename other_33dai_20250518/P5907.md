## 题目背景
[原题链接](https://www.luogu.com.cn/problem/P4948)

## 题目描述
给定 $n,a,k$，求：

$$\sum_{i=1}^n i^k a^i$$
答案对 $998244353$ 取模。

## 输入格式
输入一行三个正整数 $n,a,k$。

## 输出格式
输出一行一个整数表示答案。

```input1
6 4 3
```

```output1
1030980
```

```input2
233333333 114514 1919810
```

```output2
3191086
```

## 提示
【数据范围】    
对于 $20\%$ 的数据，$1\le n,k \le 10^5$。  
对于 $100\%$ 的数据，$1\le k \le 10^7$，$1\le n,a \le 998244352$。
