### 题目描述

鼹鼠在地下挖了一条长长的通道，在这个通道里，他们选了$n$个位置作为鼹鼠的家，第$i$窝鼹鼠的家位于坐标$i$处，家里一共有$c_i$口鼹鼠。

为了方便在洪水来临时逃生，鼹鼠们决定在通道里选择$k$个整数坐标建立逃生出口，洪水来临时，所有鼹鼠都会往最近的逃生出口跑。

定义逃生时间为所有鼹鼠在洪水来临时的逃生时间之和。

问：在最优的策略下， 所有鼹鼠的逃生时间之和最小是多少，在此基础上，输出字典序最小的一组解。

### 输入格式

$n\ k\\$
$c_1\ c_2\ \cdots\ c_n\\$

$1\leq k\leq n\leq 300,0\leq c_i\leq 10^9$。

### 输出格式

懒得写，见样例

### 样例输入1

```
5 2
1 1 1 1 1
```

### 样例输出1

```
3
1 4
```

### 样例输入2

```
5 3
1 1 1 1 1
```

### 样例输出2

```
2
1 2 4
```
