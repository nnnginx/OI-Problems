## 题目描述
求 $1\sim N$ 的一个给定全排列在所有 $1\sim N$ 全排列中的排名。结果对 $998244353$ 取模。

## 输入格式
第一行一个正整数 $N$。

第二行 $N$ 个正整数，表示 $1\sim N$ 的一种全排列。

## 输出格式
一行一个非负整数，表示答案对 $998244353$ 取模的值。

```input1
3
2 1 3
```

```output1
3
```

```input2
4
1 2 4 3
```

```output2
2
```

## 提示
对于$10\%$数据，$1\le N\le 10$。

对于$50\%$数据，$1\le N\le 5000$。

对于$100\%$数据，$1\le N\le 1000000$。
