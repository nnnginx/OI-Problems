## 题目描述

给你一个长度为$N$的字符串$S$，每个位置不是$.$就是$Y$。

你可以执行下面的操作最多$K$次：

选择相邻的两个字符，然后交换他们。

问：当你执行完上述操作后，最多能找到多长的一段连续的字符$Y$。

## 输入格式

第一行一个字符串$S$。

第二行一个数字$K$。

## 输出格式

输出一个数字表示答案。

## 样例 #1

### 样例输入 #1

```
YY...Y.Y.Y.
2
```

### 样例输出 #1

```
3
```

## 样例 #2

### 样例输入 #2

```
YYYY....YYY
3
```

### 样例输出 #2

```
4
```

## 数据范围

Subtask 1（30分）：$|S|\leq 20$。

Subtask 2（30分）：$|S|\leq 1000$。

Subtask 3（40分）：$|S|\leq 200000$。

保证$K\leq 10^{18}$
