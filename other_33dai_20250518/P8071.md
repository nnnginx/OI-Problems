## 题目描述
给定 $24$ 小时制下的时间 $H:M$，输出 $24$ 小时制下比该时间早 $45$ 分钟对应的时刻。

## 输入格式
第一行，两个正整数 $H,M$。

## 输出格式
输出两个整数，表示早 $45$ 分钟后的时刻。

```input1
10 10
```

```output1
9 25
```

```input2
0 30 
```

```output2
23 45
```

```input3
23 40
```

```output3
22 55
```

## 提示
**【数据规模与约定】**

- 对于 $100\%$ 的数据，$0 \le H \le 23$，$0 \le M \le 59$。
