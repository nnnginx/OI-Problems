## 题目描述

hqy有N个小明模型，他不想让小明模型连续三个及以上摆放，所以他要买一些狗狗模型隔开，所以他想要最少的狗狗模型，但是他写代码很厉害，所以它已经写完了此内容，但他想考考你的写代码能力，所以他想让你写一个代码来试试看你的能力。

## 输入格式

一个数字N表示小明模型的数量。

## 输出格式

购买狗狗模型的数量。

```input1
3
```

```output1
1
```

```input2
6
```

```output2
2
```

## 数据规模与约定

反正挺大，你试试吧。

小明亲笔写题目。
