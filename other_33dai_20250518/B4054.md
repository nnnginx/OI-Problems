## 题目描述
在一场战斗中，使用 $K, D, A$ 三个参数来衡量战士的实力，一个战士的**实力**如下：

- 若 $K-D\ge10$，则他的**实力**为 $K\times(K-D)+A$。
- 否则，若 $K\ge D$，则他的**实力**为 $(K-D+1)\times 3+A$。
- 否则，他的**实力**为 $A\times 2$。

有 $n$ 名战士参与了这场战斗，**实力最高**的战士会被选为领袖。保证实力最高的战士只有一位。请你告诉小 Z，领袖是**第几位**战士。

## 输入格式
第一行一个整数 $n$，表示战士的数量。

接下来 $n$ 行，第 $i$ 行**用 `/` 隔开**的三个整数 $K_i, D_i, A_i$，表示第 $i$ 个战士的参数。具体可见样例。

## 输出格式
一行一个整数 $ans$，表示第 $ans$ 位战士为领袖。

```input1
3
5/3/8
3/4/7
3/3/13
```

```output1
1
```

```input2
2
10/1/9
10/0/1
```

```output2
2
```

```input3
2
9/6/3
8/5/2
```

```output3
1
```

## 提示
提示：对于一个字符串 `K/D/A`（其中字母对应位置代表其对应的整数）可以使用 `scanf("%d/%d/%d",&K,&D,&A)` 的方式读入它们的值。

### 样例 #1 解释

第 $1$ 位战士的 $K, D, A$ 分别为 $5,3,8$，他的实力为 $(5-3+1)\times3+8=17$。

第 $2$ 位战士的 $K, D, A$ 分别为 $3,4,7$，他的实力为 $7\times 2=14$。

第 $3$ 位战士的 $K, D, A$ 分别为 $3,3,13$，他的实力为 $(3-3+1)\times3+13=16$。

第 $1$ 位战士实力最高，故他被选为领袖。

### 数据范围

对于 $100\%$ 的数据，$2\le n\le 100000,0\le K_i,D_i,A_i\le 10000$，保证实力最高的战士只有一位。

对于前 $20\%$ 的数据，$n=2$。

对于前 $40\%$ 的数据，$n\le 1000,0\le K_i,D_i,A_i\le 1000$。

另有 $30\%$ 的数据，$K_i\ge D_i$。
