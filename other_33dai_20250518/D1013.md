## 题目描述

输入 $n$ 个 $1\sim 2000$ 范围内的整数 $a_1\sim a_n$，保证任意两个元素各不相同。对下标排序输出。

即输出 $id_1\sim id_n$，以保证 $a_{id_1} \sim a_{id_n}$ 就是从小到大排好序后的原数组。 

## 输入格式

第一行一个整数 $n$。  
接下来一行 $n$ 个整数，含义如题意所述。

## 输出格式

一行 $n$ 个空格隔开的整数，含义如题意所述。

```input1
5
3 2 7 6 8
```

```output1
2 1 4 3 5
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\le n \le 1000$。
