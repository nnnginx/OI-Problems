## 题目背景
YSGH 牛逼

## 题目描述
给定长度非零的非空 01 串 $S$。

找出 $S$ 的非空连续子串 $T$ 满足串中 0 的个数减去 1 的个数最大。

你只需要输出最大值即可。

## 输入格式
一行一个 01 串表示 $S$。

## 输出格式
一行一个数表示答案。

```input1
0111100101
```

```output1
2
```

## 提示
设 $|S| = n$。

| 数据点编号 | $n \le$ |
| :--: | :--: |
| $1 \sim 2$ | $10$ |
| $3 \sim 6$ | ${10}^3$ |
| $7 \sim 10$ | ${10}^5$ |

对于 $100\%$ 的数据，$1 \le n \le {10}^5$。
