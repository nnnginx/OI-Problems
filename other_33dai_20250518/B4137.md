## 题目描述
小明认为，如果以下条件之一成立，则正整数 $x$ 就是一个幸运数字：

1. $x$ 是 $4$ 的倍数。
1. $x$ 是 $7$ 的倍数。
1. $x$ 转换为字符串后，包含子串 `44` 或 `77`。

例如，$105$（$7$ 的倍数）、$442$（包含`44`）、$1284$（$4$ 的倍数）都是幸运数字，而 $474$ 则不是。编程统计 $1$ 到 $n$ 中幸运数字的数量。

## 输入格式
一行，一个整数 $n$。

## 输出格式
一行，一个整数，$1,2,3,\dots n$ 中幸运数字的数量。

```input1
32
```

```output1
11
```

```input2
987654
```

```output2
406754
```

## 提示
对于 $100\%$ 的数据，$1\le n\leq 10^6$。

>本题原始满分为 $10\text{pts}$。
