## 题目背景
原题链接：<https://oier.team/problems/J3A>。

## 题目描述
输入自然数 $a$，判断 $37\times a$ 是否各个数位都相等。

## 输入格式
输入含一行一个自然数 $a$。

## 输出格式
如果 $37a$ 各个数位都相等，则输出 `Yes`，否则输出 `No`。

```input1
6

```

```output1
Yes
```

```input2
33

```

```output2
No

```

## 提示
**【样例解释 #1】**

$37\times 6 = 222$，其三个数位都是 $2$，符合题意。

**【样例解释 #2】**

$37\times 33 = 1221$，有的数位不相等（如千位和百位），不符合题意。

**【数据范围】**

本题共 $20$ 个测试点，每个 $5$ 分。

保证 $0\le a \le 3737$。
