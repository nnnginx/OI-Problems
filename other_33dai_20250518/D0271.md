## 题目描述

输入整数 $p,q$，输出离 $\frac{p}{q}$ 最接近的整数。

如果同时存在两个数 $x,y$ 与 $\frac{p}{q}$ 的距离相同，则输出 $x,y$ 中较小的一个。

## 输入格式

两个整数 $p,q$。

## 输出格式

如题所述，输出对应的整数。

```input1
9 4
```

```output1
2
```

```input2
-11 -4
```

```output2
3
```

```input3
10 4
```

```output3
2
```

```input4
-9 4
```

```output4
-2
```

```input5
11 -4
```

```output5
-3
```

```input6
-10 4
```

```output6
-3
```

## 数据规模与约定

对于 $100\%$ 的数据，$-10^9 \le p,q \le 10^9$。

- 子任务 1（30 分）：保证 $p$ 除以 $q$ 的余数为 $0$。
- 子任务 2（30 分）：保证 $0\lt p,q$。
- 子任务 3（40 分）：没有特殊限制。
