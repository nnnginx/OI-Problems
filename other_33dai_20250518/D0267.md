## 题目描述

输入一个 $8$ 位数 $n$，输出有几个数位是 $3$。

## 输入格式

输入一行，为一个 $8$ 位数 $n$。

## 输出格式

输出 $n$ 里面有几个数位是 $3$。

```input1
12345678
```

```output1
1
```

```input2
12456789
```

```output2
0
```

```input3
12341234
```

```output3
2
```


## 数据规模与约定

对于 $100\%$ 的数据，$10000000\le n \le 99999999$。
