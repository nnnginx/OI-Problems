## 题目描述
定义第一类好数为所有数位上的数字均相同的数。

定义第二类好数为满足如下任意一个条件的数：
- 为第一类好数。
- 设 $t$ 为数的位数，则有 $t-1$ 位数上的数字相同，且无前导零。

现给定一个 $x$，求出 $\ge x$ 的第一个第 $k+1$ 类好数 $y$。

## 输入格式
第一行一个整数 $x$。

第二行一个整数 $k$。

## 输出格式
一行一个整数 $y$。

```input1
700
0
```

```output1
777
```

```input2
700
1
```

```output2
700
```

## 提示
对于所有子任务 $1\le x\le 10^{17}$，$k\in\{0,1\}$。

| 子任务编号 |        特殊限制         | 分值 |
| :--------: | :---------------------: | :--: |
|    $1$     | $1\le x\le 10^5$，$k=0$ | $15$ |
|    $2$     |          $k=0$          | $20$ |
|    $3$     |    $1\le x\le 10^5$     | $21$ |
|    $4$     |       无特殊限制        | $44$ |
