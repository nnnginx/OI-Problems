## 题目描述

输入有若干组，每组都是一行为空格隔开的 $a,b$，对于每组输入输出一行 $a+b$。

## 输入格式

输入有若干组，每组都是一行为空格隔开的 $a,b$

## 输出格式

对于每组输入输出一行 $a+b$

## 样例

```input1
123 456
-123 456
33221 11233
```

```output1
579
333
44454
```

## 数据范围

对于 $60\%$ 的数据：$-10^9\le a,b \le 10^9$；

对于 $100\%$ 的数据：$-4\times 10^{18}\le a,b \le 4\times 10^{18}$，输入 $a,b$ 的组数不超过 $10$。
