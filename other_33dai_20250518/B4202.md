## 题目背景
搬运自 <http://czoj.com.cn/p/438>。数据为民间数据。

## 题目描述
[](VFhwSmVVNXFTWGM9)
小 X 把一本书摊开在桌上。这本书的页码是连续的，从 $1$ 到 $n$。  

![](https://cdn.luogu.com.cn/upload/image_hosting/78pc5h3o.png)

现在他告诉你现在桌面上书本左右两页的页码是什么，请你回答他当他翻到第 $A$ 页时，第 $A$ 页会在左边还是右边。 

## 输入格式
第一行 $1$ 个正整数 $n$，表示这本书的页码从 $1$ 标号到 $n$。   
第二行 $2$ 个正整数，表示现在书本左右两页的页码 $P_1$ 和 $P_2$。   
第三行 $1$ 个正整数，表示小 X 想问你第 $A$ 页在左边还是右边。

## 输出格式
如果第 $A$ 页在左边，输出 `L`；如果第 $A$ 页在右边，输出 `R`。

```input1
5
2 3
1
```

```output1
R
```

```input2
5
2 3
4
```

```output2
L
```

## 提示
### 样例解释
![](https://cdn.luogu.com.cn/upload/image_hosting/msx7hn2l.png)

一开始书本状态为第 $2$ 张图。可以看到，第 $1$ 页在右，第 $4$ 页在左。   
### 数据范围
本题共有 $5$ 个测试点。   
对于所有测试点：$1 \le P1,P2,A \le n \le10^3$。
|测试点编号|特殊性质|
|:-:|:-:|
|$1\sim 2$|$A\in\{P_1,P_2\}$|
|$3\sim 5$|无|
