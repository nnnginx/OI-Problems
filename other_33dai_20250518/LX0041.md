### 题目描述

有$n$个数字，从上到下是依次递增的。

但是其中一些位，被模糊成了$?$。

现在的问题是，给定的这$n$个数字是否合法，如果合法，输出一个所有数字都最小的解。

### 输入格式

懒得写，见样例。

$n\leq 1000$，每个数字最多$300$位。

### 输出格式

懒得写，见样例


```input1
3
?
18
1?
```

```output1
YES
1
18
19
```

```input2
2
??
?
```

```output2
NO
```

```input3
5
12224
12??5
12226
?0000
?00000
```


```output3
YES
12224
12225
12226
20000
100000
```
