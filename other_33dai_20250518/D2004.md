## 题目描述

对于一个质数，如果他的各个数位之和仍然是一个质数，千夜老师就认为他是一个二重质数。

比如 $23$ 是一个质数，$2+3=5$ 也是一个质数，因此 $23$ 就是一个二重质数。

现在千夜老师想知道 $1\sim n$ 之间有多少个二重质数。

## 输入格式

输入一行，为一个整数，即 $n$。

## 输出格式

输出 $1\sim n$ 中所有的二重质数，每行一个。

## 样例

```input1
20
```

```output1
2
3
5
7
11
```

## 数据范围

对于 $60\%$ 的数据，$2\le n\le 1000$

对于 $100\%$ 的数据，$2\le n\le 5\times 10^6$
