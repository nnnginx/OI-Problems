## 题目描述

有 $n$ 位同学，编号从 $1\sim n$，每位同学都有一个自己的积分值，且互相都能看到对方的积分值。

现在同学们按照编号顺序依次进入教室。

如果第 $i$ 位同学进入教室后，发现教室内“比自己积分更高的同学数量”**没超过**“比自己积分更低的同学数量”。那么他就会很开心，请你统计一下有多少个人会很开心。

## 输入格式

输入第一行为一个正整数 $n$，即总人数。

接下来一行位空格隔开的 $n$ 个正整数，即 $a_1\sim a_i$，表示每位同学的积分值。

## 输出格式

输出一行为一个整数，即开心的人数。

## 样例

```input1
8
3 2 7 6 8 1 4 5
```

```output1
6
```


```input2
8
3 3 3 1 1 1 2 2 
```

```output2
5
```
## 数据范围

对于 $60\%$ 的数据，$1\le n\le 1000$。

对于 $100\%$ 的数据，$1\le n\le 100000$，$1\le a_i \le 100$。
