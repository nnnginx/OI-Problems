## 题目描述
定义 Handsome 数：这个正整数中任意两个相邻的数位的奇偶性不同。

给你一个正整数 $n$，求与 $n$ 最近的 Handsome 数。如果有多个按从小到大的顺序输出。

## 输入格式
一个正整数 $n$。

## 输出格式
与 $n$ 最近的 Handsome 数。

```input1
13
```

```output1
12 14
```

```input2
5801001
```

```output2
5810101
```

## 提示
对于 $100\%$ 的数据，$1 \leq n \leq 10^{1000}$。

译自 [COCI 2014/2015 CONTEST #5](https://hsin.hr/coci/archive/2014_2015/contest5_tasks.pdf)。
