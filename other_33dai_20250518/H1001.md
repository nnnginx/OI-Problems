## 题目描述

小明在过年时收到了很多红包，小明有一个癖好，就是如果红包比之前的小，就看不上这个红包。你作为小明的家长，现将所有红包的金额数告诉了你，你将把这些红包派发给小明，你想让小明收到的 $money$ 最大，请你写一个程序来计算。

## 输入格式

给你一个 $n$，表示你一知 $n$ 个红包的金额。
后面一行 $n$ 个整数，表示每个红包的 $money$ 数。

## 输出格式

一行 $1$ 个数，为 小明最多能拿到的红包数。

```input1
7
4 2 3 7 6 5 8
```

```output1
4
```

## 数据规模与约定

对于 $100\%$ 的数据，$0 \le n \le 10^6$。
