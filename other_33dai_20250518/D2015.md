## 题目描述

33DAI 想知道 $l\sim r$ 中的每个数字的每个数位一共有多少个 $1$。

## 输入格式

输入第一行为一个整数 $n$，表示有 $n$ 个问题。

接下来 $n$ 行，每行都是一个问题，为空格隔开的 $l,r$

## 输出格式

输出 $n$ 行，即 $n$ 个问题的答案。

## 样例

```input1
3
1 2
2 9
1 21
```

```output1
1
0
13
```

## 数据范围

对于 $60\%$ 的数据：$1\le n,l,r \le 1000$；

对于 $100\%$ 的数据：$1\le n,l,r \le 10^6$
