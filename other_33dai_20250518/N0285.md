### 题目描述

小明从西安旅游回来后就迷上了打标签，有一天，他给班里的每个同学打上了$n$个属性的标签，比如某$qx$的颜值$=0$，身材$=0$，搞笑程度$=5$（满分$5$分制）。

假设满分是$k$分制，每个人每个属性的标签都是一个$[0,k]$之间的整数，那么就会有最多$(k+1)^n$类人。

定义两个人可以交朋友，当且仅当这两个人每个属性的差都不超过$1$。

然后呢，小明就安安心心的数了一下，对于这$(k+1)^n$中的每一类人，如果他作为新生来到这个班级，他能在班里交多少个朋友。

但是呢，小明在西安学坏了，他也给你出了个谜题：告诉你每一类学生作为新生，加入班级以后能交到的朋友个数，请你告诉小明，班里每一类人到底有多少个。

### 输入格式

第一行输入$n,k$，如题所述。

第二行输入$(k+1)^n$个数字，按$k+1$进制下从小到大的顺序表示每一类人来班级以后能交到的朋友个数。

### 输出格式

输出$(k+1)^n$个自然数，表示答案，如果有多解，输出任意一个都可以。

### 样例输入1

```
1 6
0 1 2 2 2 1 1
```

### 样例输出1

```
0 0 1 1 0 1 0
```
