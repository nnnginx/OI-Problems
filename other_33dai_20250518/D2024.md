## 题目描述

输入长度相等的两个由小写英文字母构成的字符串 $s_1$、$s_2$，假设两个字符串相同位置上的字符的 ascii 相加的和恒等，我们就说这两个字符串相关。

如果两个字符串相关，请输出 $\text{恒等值} - 194$，否则请输出 $-1$

## 输入格式


输入两行，分别为 $s_1$、$s_2$。

## 输出格式

如果两个字符串相关，请输出 $\text{恒等值} - 194$，否则请输出 $-1$

## 样例

```input1
abcdefg
nmlkjih
```

```output1
13
```

```input2
abcdefg
nmlkoih
```

```output2
-1
```

## 样例 1 解释

两个字符串的字符对应的 ascii 如下：

```
s1: 97 98 99 100 101 102 103
s2: 110 109 108 107 106 105 104
```

两个字符串相同位置上的字符的 ascii 相加的结果都是恒等值 $207$，因此输出 $13$。

## 数据范围

对于 $100\%$ 的数据：$1\le |s_1|、|s_2| \le 10$。
