## 题目描述

有一个长度为$n$的传送带，第$i$个位置要么是`>`，要么是`<`。

有一个人，一开始会从传送带的某个位置进入传送带。

如果这个位置是`<`，则他会向左走一格，然后把刚才那个位置改成`>`。

如果这个位置是`>`，则他会向右走一格，然后把刚才那个位置改成`<`。

当他走出界外，则结束本次旅程。

问：他从每一个位置进入传送带，经过几次会出去。

## 输入格式

第一行输入$n$。

接下来一行输入一个长度为$n$的字符串，每个位置不是`<`就是`>`。

## 输出格式

输出$n$个数字表示答案。

### 样例输入 #1

```
3
><<
```

### 样例输出 #1

```
3 6 5
```

### 样例输入 #2

```
6
<><<<>
```

### 样例输出 #2

```
1 4 7 10 8 1
```

## 数据范围

对于30%的数据：保证$1\leq n\leq 500$。

对于60%的数据：保证$1\leq n\leq 5000$。

对于另20%的数据：保证最多有$20$个`<`。

对于100%的数据：保证$1\leq n \leq 5\times 10^5$。
