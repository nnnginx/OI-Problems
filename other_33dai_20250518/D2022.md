## 题目描述

33DAI 定义了一种表达式为 AJ 表达式，这种表达式的规则是：
- 只包含加减号及 `'a'` 到 `'j'` 的小写英文字符；
- 表达式的值为将 `'a'` 到 `'j'` 分别转换为 `0` 到 `9` 后的数学表达式的值（保证转换后的表达式符合数学规范）。
- AJ 表达式的长度不超过 $10000$，且连续的英文字母长度不超过 $10$。

## 输入格式

输入一个 AJ 表达式。

## 输出格式

输出 AJ 表达式的值。

## 样例

```input1
cab+b-ba
```

```output1
192
```

## 样例 1 解释

```
201+1-10 = 192
```


## 数据范围

对于 $100\%$ 的数据：连续的英文字母长度不超过 $10$。
