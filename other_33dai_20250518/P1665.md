## 题目描述
给定平面上 $N$ 个点，你需要计算以其中 $4$ 个点为顶点的正方形的个数。注意这里的正方形边不一定需要和坐标轴平行。


## 输入格式
第一行一个数 $N$。

接下来 $N$ 行每行两个数，表示点的坐标。


## 输出格式
一个数表示正方形的个数。


```input1
7
0 0
0 1
1 0
1 1
1 2
2 1
2 2
```

```output1
3
```

## 提示
对于 $20\%$ 的数据，满足 $1\leq N\leq 20$。

对于 $100\%$ 的数据，满足 $1\leq N\leq 500$，$-50\leq X_i, Y_i\leq 50$，点不会重合。
