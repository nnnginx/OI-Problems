### 题目描述

小明在玩一个消除游戏。

游戏规则是这样的，你有$n$个小方块，第$i$个小方块的颜色是$a_i$。

一开始，你有一个空的方块序列，第$i$回合，你可以选择把第$i$个小方块放在序列的左侧或者右侧。

一旦序列的长度$\geq 2$，且序列中**左右两端**颜色一样，那么整个序列的小方块就都会被消除掉。

请你构造一个游戏方案，使得游戏结束时序列是空的。

### 输入格式

第一行输入$T$，表示$T$组数据。

对于每组数据：

第一行输入$n$。

第二行输入$n$个正整数$a_i$。

### 输出格式

如果无解，输出`-1`，否则输出一个长度为$n$的序列，`L，R`分别表示放在左侧或者右侧。

### 样例输入 #1

```
1
3
1 2 1
```

### 样例输出 #1

```
LRR
```

### 样例解释 #1

放入第三个方块的时候，序列刚好是$[1,2,1]$，消完啦。

### 样例输入 #2

```
1
4
1 1 2 2
```

### 样例输出 #2

```
LRLR
```

### 样例输入 #3

```
1
5
2 1 1 2 2
```

### 样例输出 #3

```
LLRLR
```

### 样例解释 #3

前三个数字被放进来的时候，序列是：$[1,2,1]$，消掉了，所以后两个随便怎么放了。

### 样例输入 #4

```
1
5
2 2 1 1 1
```

### 样例输出 #4

```
-1
```

### 数据范围

对于30%的数据：$1\leq n\leq 20$。

对于60%的数据：$1\leq n \leq 300$。

对于另25%的数据：保证有解。

对于100%的数据：$1\leq T\leq 20,1\leq n \leq 10^5,1\leq a_i\leq n$。
