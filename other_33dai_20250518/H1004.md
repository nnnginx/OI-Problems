## 题目描述

小明非常喜欢抽卡片，每盒只有一包好卡，好卡比普卡重，他带了一个秤，用来分辨好卡和普卡，求至少秤多少次，可以称出好卡。

## 输入格式

一个数 $n$ ，表示这盒有 $n$ 包卡片。

## 输出格式

一个整数，表示至少要称多少次。

```input1
1
```

```output1
0
```

```input2
5
```

```output2
2
```
## 数据规模与约定

对于样例一

当只有一包卡片时，它一定是好卡的，不需要称。

对于样例二

将5包卡片分成2,2,1三组，称量2,2两组。若秤平衡，则剩下那包为好卡;若不平衡，则质量大的卡包在数量为2的组中，需要再称量一次。因此最坏情况下需要2次称量。

数据范围 1 ≤ n ≤ 10^17
