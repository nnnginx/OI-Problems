## 题目背景

由于世界局势巨变，各国纷纷使用了强有力的 A 武器，导致世界上诞生了许多许多亘古未见的悬崖。

小 C 作为一位探险家，自然不会放过这些壮丽的景色。为了一览众山小，小 C 来到了最高的一座悬崖，开始史诗般的攀登。

## 题目描述

已知小 C 每天能向上爬 $a$ 米，但是晚上睡觉的时候小 C 又会应为不可抗力的自然因素向下掉 $b$ 米。悬崖一共高 $c$ 米。

求小 C 需要多少天才能登上悬崖，如果不能则输出 `1mpossible`。

## 输入格式

本题共有 $t$ 组数据。

第一行一个数 $t$ 表示有 $t$ 组数据。

此后 $t$ 行每行三个数，$a$、$b$、$c$，意思如题。

## 输出格式

输出共 $t$ 行，每行表示每次的答案。

## 样例

```input1
4
2 1 5
5 1 6
100 99 1000000000
514 514 1919810
```

```output1
4
2
999999901
1mpossible
```

## 提示

### 数据范围

对于 $20\,\%$ 的数据 $1 \leq a,\, b,\, c \leq 100$。

对于 $50\,\%$ 的数据 $t = 1$，$1 \leq a,\,b,\,c \leq 10^4$。

对于所有数据，$1 \leq t \leq 5 \times 10^5$，$1 \leq a,\,b,\,c \leq 10^{18}$。
