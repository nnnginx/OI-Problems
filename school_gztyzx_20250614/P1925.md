## 问题描述

给你 $n$ 根火柴棍，你可以拼出多少个形如 `A+B=C` 的等式？等式中的 $A、B、C$ 是用火柴棍拼出的整数（若该数非零，则最高位不能是 $0$）。用火柴棍拼数字 `0-9` 的拼法如图所示：

![image](./957/file/diodKMYqlp-YUi2j1Kv2N.png)

注意：

1. 加号与等号各自需要两根火柴棍
2. 如果 $A≠B$，则 $A+B=C$ 与 $B+A=C$ 视为不同的等式（$A、B、C\geq 0$）
3. $n$ 根火柴棍必须全部用上

## 输入格式

输入共一行，又一个整数 $n(n\leq 24)$。

## 输出格式

输出共一行，表示能拼成的不同等式的数目。

```input1
14
```

```output1
2
```

## 样例1解释

2个等式为0+1=1和1+0=1。

```input2
18
```

```output2
9
```

## 样例2解释

9个等式为：

0+4=4

0+11=11

1+10=11

2+2=4

2+7=9

4+0=4

7+2=9

10+1=11

11+0=11
