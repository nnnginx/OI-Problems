## 题目描述

小$A$有一个$n$行$m$列的矩阵$A$。

小$A$认为一个$2 \times 2$的矩阵是好的，当且仅当$D_{1,1} \times D_{2,2}=D_{1,2}\times D_{2,1}$。

其中$D_{i,j}$表示矩阵$D$的第$i$行第$j$列的元素。

小$A$想知道$A$中有多少个好的子矩阵。

## 输入格式

第一行，输入两个整数$n,m$；

接下来$n$行，每行$m$个整数$A_{i,1},A_{i,2}...,A_{i,m}$。

## 输出格式

一行，一个整数，表示$A$中好的子矩阵的数量。

```input1
3 4
1 2 1 0
2 4 2 1
0 3 3 0
```

```output1
2
```
