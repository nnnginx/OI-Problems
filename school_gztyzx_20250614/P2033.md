## 问题描述

给一个长度为 $n$ 整数序列 $n$，和 $q$ 个询问，每个询问包含两个整数 $L,R$，询问 $a_L, a_{L+1},\cdots , a_R$ 这些数字的方差是多少。

因为不希望有实数输出，所以你应当将输出乘 $(r -l＋1)^2$ 转为整数。

对于 $n$ 个数字 $x_1,x_2,\cdots,x_n$，令 $\mu$ 为 $n$ 个数字的平均数，其方差的定义为 $ var=\frac{\sum_{i=1}^n(x_i-\mu)^2}{n}$

此题需注意 $32$ 位整数表示范围。

## 输入格式

第一行两个整数 $n，q$。

接下来一行 $n$ 个数字，表示 $n$ 序列。

接下来 $q$ 行，每行两个整数 $L,R$ 表示询问。

## 输出格式

输出共 $q$ 行，表示每个询问的答案。

```input1
3 4
1 2 2
1 1
1 2
1 3
2 3
```

```output1
0
1
2
0
```

## 样例解释

对于 $1\ 3$ 询问，$\mu=\frac{1+2+2}{3}=\frac{5}{3}$。

$var=((1-\frac{5}{3})^2+(2-\frac{5}{3})^2+(2-\frac{5}{3})^2)/3*3*3=2$

## 数据规模和约定

对于 $30\%$ 的数据，$n ≤1000,q ≤1000,a_i≤ 100$。

对于 $60\%$ 的数据，$n<10^6,q≤ 10^6 ,a_i≤10,R-L≤1000$。

对于 $100\%$ 的数据，$n≤10^6,q≤10^6, a_i≤ 10^3,1≤L≤R≤n$。
