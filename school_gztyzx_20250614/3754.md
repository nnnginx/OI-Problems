## 题目描述

小C要开始军训了，军训的第一件事情当然是整队，需要按照从队伍头到队伍尾、从低到高的顺序排列学生。
一开始队伍里有$n$个人，从前往后数第$i$个人的身高为$h_i$。

每次小C可以从队伍中挑选出一个人让他站到队伍最前面。现在小C想知道他至少需要这样做几次才能让整个队伍是从前往后是从低到高排的。

## 输入格式

第一行一个整数 $n$，表示队伍人数。

第二行 $n$ 个整数，表示队伍 $h_i$，第$i$个整数$h_i$表示初始队伍从前往后数第$i$个人。保证 $1\le h_i\le n$ 且所有数互不相同。

## 输出格式

输出一个整数，表示最小的次数。

```input1
5
4 5 2 1 3
```

```output1
3
```

```input2
25
18 11 17 19 10 9 22 23 8 24 25 21 16 20 5 3 12 13 6 14 2 4 1 15 7
```

```output2
21
```

## 数据范围

对于所有数据，$1\le n\le 2\times10^5$，且 $h_i$ 为排列。

| 数据编号 |        $n$         |        特殊性质        |
| :------: | :----------------: | :--------------------: |
|    1     |      $\le 2$       |                        |
|    2     |      $\le 10$      |                        |
|    3     |      $\le 10$      |                        |
|    4     |     $\le 200$      |                        |
|    5     |     $\le 200$      |                        |
|    6     | $\le 2\times 10^3$ | $A$ 初始为从大到小排序 |
|    7     | $\le 2\times 10^3$ |                        |
|    8     | $\le 2\times 10^5$ | $A$ 初始为从大到小排序 |
|    9     | $\le 2\times 10^5$ |                        |
|    10    | $\le 2\times 10^5$ |                        |
