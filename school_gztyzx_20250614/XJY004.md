## 题目描述

他想要找到她。

前方的路，迷雾笼罩着，等着他去揭开。在月光的照耀下，它闪烁着淡淡的光泽。宛如一层轻纱，为夜色增添了几分朦胧的美感，给大自然增添了一抹诗意；也如一团白色的鬼魅，悄然弥漫在空气中，将万物都收入其翼下，让人感到迷茫和困惑。

迷雾漫漫，却遮不住心中那片明朗的天空和理想。

峡谷中，迷雾形成了一个 $n\times m$ 块组成的迷阵，可以由某一块移动到相邻的四块。第 $1$ 行的 $m$ 个房间都是迷阵的入口；第 $n$ 行有 $m$ 个机关需全部打开。每块都有毒，将会对进入房间的人造成伤害：设 $i$，$j$ 均为整数且 $i\in [1,n]$，$j\in [1,m]$，则每块造成的伤害值为 $a_{i,j}$（$a_{1,j}=a_{n,j}=0$）。当然，他希望以最小伤害代价进入迷阵并且打开全部机关。

对了，他有召唤精灵的能力（逐渐离谱）。任意多个的精灵可以从任意的入口进入，最终必须到达第 $n$ 行。在一路上所有伤害的最大值即为精灵的伤害值，而他的精灵总伤害值为每个精灵中的最大伤害值。你需要求出最小的总伤害值。

## 输入格式

第一行，两个整数 $n$，$m$。

接下来 $n$ 行，每行 $m$ 个数，表示 $a_{i,j}$。

## 输出格式

一行一个整数，表示最小伤害值。

```input1
3 3
0 0 0
1 1 4
0 0 0
```

```output1
1
```

## 提示

#### **样例解释**

召唤 $5$ 个精灵从第 $1$ 行第 $1$ 列出发，往右走 $1$ 格，往左走 $1$ 格，往右走 $1$ 格，往左走 $1$ 格，往右走 $1$ 格，往右走 $1$ 格，往左走 $2$ 格，再往下走 $2$ 格。这只是其中一种走法。

#### **数据范围**

$50\%$ 的数据，$n,m\leq10^2$；

$100\%$ 的数据，$n,m,a_{i,j}\leq10^3$。
