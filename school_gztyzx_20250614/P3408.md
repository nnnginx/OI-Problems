## 问题描述

很显然，这是比赛中最弱的一道题（人口普查）。

![](/upload/image/20210128/20210128095003_40249.jpg)

求 $n$ 个点有多少种构成标号无重边无自环无向图的方法（不要求连通）。

## 输入格式

第一行一个数 $T$，表示有 $T$ 组数据。

对于每组数据，有两个数 $n，k$，意义如题目所述，$k$ 没用。

## 输出格式

每行输出一个数 $ans$ 表示构成题目中所述的图的数量。

由于答案可能很大，请输出 $ans$ 对 $10000007$ 取模的结果。

```input1
1
3 2
```

```output1
8
```

## 数据范围

对于 $10\%$ 的数据，是彩蛋。

对于 $100\%$ 的数据，$T\leq  100, k \leq  n \leq 60$。
