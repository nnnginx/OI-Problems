## 题目描述

在数学课上，老师给孩子们布置了一道非常简单的题目：给你一个数字，你需要输出这个数字的$19$倍！

如给出的数字是$2$，那么你需要输出$38$，十分的简单！

而在上个星期，$JY$老师刚刚和同学们分享了二进制的内容，于是$JY$老师把$19$倍的题目稍微修改了一下，变成了这个样子：给你一个二进制数$n$，请你求出这个二进制数的$19$倍是多少，结果也要用二进制输出！

对于刚刚学完二进制的你，应该如何解决这个问题呢？

## 输入格式

一个二进制表示的数$n$。

## 输出格式

$19*n$的二进制表示。

```input1
10110111
```

```output1
110110010101
```

## 样例解释

$10110111$相当于十进制的$183$，于是$183*19=3477$，二进制形式是$110110010101$。

## 数据范围

$30\%$的数据$N$的位数小于$25$位。

$50\%$的数据$N$的位数小于$50$位。

$100\%$的数据$N$的位数小于$1000$位。
