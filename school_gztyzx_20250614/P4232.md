## 【题目描述】

P神喜欢一种数叫做P数，P数中的每一位是3的倍数的数字之和 % 7 = 5 。

P神给你 $q$ 个询问，让你计算 $a_i$ 是否是P数。

如$39$，该数中的每一位是$3$的倍数的数字之和为$12$，12%7=5，所以该数是$p$数，输出$Y$。

## 【输入格式】

第一行一个数 $q$ 。
第二行共 $q$ 个数，第 $i$ 个数是 $a_i$ 。

## 【输出格式】

请输出$1$行共 $q$ 个字符，每个字符可能是`Y,N`，代表 $a_i$ 是否是P数。

```input1
3
33 114514 39
```

```output1
NNY
```

## 【数据范围】

100%测试数据，满足 $q \leq 10^{5};a_i < 10^{5}$
