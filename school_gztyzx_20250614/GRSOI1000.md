# 三点三线 I

## 题目背景

xiaosi4081 正在从宿舍前往机房。

## 题目描述

从宿舍到机房的路很奇怪，它分成 $n$ 个小段。第 $i$ 个小段对应一个 $a_i$，这也就意味着经过第 $i$ 段的时间可以为 $[1,a_i]$ 间的任意整数，并且通过整段路的时间为在每一段路上经过的时间的乘积。比如说共有两段路，经过第一段的时间为 $2$，经过第二段的时间为 $3$，则经过整段路的时间为 $2\times 3=6$。

xiaosi4081 想要知道到达机房的时间的期望。答案对 $1000000007$ 取模。

---

期望：在概率论和统计学中，数学期望（或均值，亦简称期望）是试验中每次可能结果的概率乘以其结果的总和。

也就是说，期望相当于枚举所有情况，将每一种情况所需的时间记录下来并且加起来 ，然后除以情况数。

如何对分数取模：对于一个分数 $\dfrac x y$ ，其对 $p$ 取模的结果为 $xy^{-1}$ ，也就是 $x$ 乘以 $y$ 的逆元。

求逆元的代码给各位贴出来：

```c++
void exgcd(int a,int b,int &x,int &y){
	if(!b)return x=1,y=0,void();
	else{
		exgcd(b,a%b,y,x);
		y-=(a/b)*x;
	}
}
```

求对于 $a$ 在模 $p$ 意义下的逆元：

```c++
int tmp,ny;
exgcd(a,p,ny,tmp);
ny=(ny%p+p)%p;
//ny是a在模p意义下的逆元
```

## 输入格式

第一行，一个数 $n$ 。

接下来有 $n$ 个数，第 $i$ 个数表示 $a_i$ 。

## 输出格式

输出对 $1000000007$ 取模后的期望值。

## 样例 #1

### 样例输入 #1

```
3
1 1 2
```

### 样例输出 #1

```
500000005
```

## 提示

对于 $5 \%$ 的数据，$n\leq 5,\prod a_i \leq 10^6$ 。

对于另外 $10 \%$ 的数据，$\prod a_i \leq 10^6$ 。

对于另外 $5 \%$ 的数据，所有 $a_i$ 相等。

对于 $100 \%$ 的数据，$n \leq 10^6,a_i \leq 10^8$

对于样例【1】的解释：

第一种方案是 $[1,1,1]$，即在第一段路上用 $1$ 时间，在第二段路上用 $1$ 时间，在第三段路上用 $1$ 时间，耗费的时间为 $1\times 1\times 1=1$。

第二种方案是 $[1,1,2]$，耗费的时间为 $1 \times 1 \times 2$。

不存在其他的方案。两种方案的和为 $1+2=3$。其期望为 $\frac 3 2$，$2$ 的逆元为 $500000004$，期望模完之后的结果为 $500000005$。
