## 题目描述

小远拿着一个鸡腿走进了$TY$的机房，然后小丁趁他不注意，把他的鸡腿锁进了密码箱！密码箱的密码非常离谱，你需要解答谜题才能获取密码。

谜题是这样的，密码箱的显示屏会显示一大串数字，这串数字非常之大，你要做的是快速地求出这串数字中有多少段的数字是$2019$的整数倍数，每一段数可以用一个二元组 $ (i,j) $来表示，含义是把数字串中区间$[i,j]$的数字拎出来形成一个新的数字串，判断该数字串是否是$2019$的倍数，如果是则段数$+1$，这个段数就是密码箱的答案！

小远求你帮他求出这个密码，鸡腿成功拿出来之后，他会跟你五五分成！

## 输入格式

一行一个整数 $ S $

## 输出格式

求数字串中满足条件的二元组 $ (i,j) $ ($ 1\ <\ =\ i\ <\ =\ j\ <\ =\ |S| $) 的数量。

```input1
1817181712114
```

```output1
3
```

## 样例说明

有三对$(i,j)$使得条件成立：$ (1,5),\ (5,9),\ (9,13) $。

```input2
14282668646
```

```output2
2
```

```input3
2119
```

```output3
0
```

## 数据范围

- $ 1\ <\ =\ |S|\ <\ =\ 200000 $
- $ S $保证是$1-9$的数字序列
