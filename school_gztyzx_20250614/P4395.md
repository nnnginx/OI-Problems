## 题目描述

小浩是个足球迷，他以克里斯蒂亚诺・罗纳尔多$（Cristiano Ronaldo）$为自己的偶像，经常在集训结束后踢足球，因此他非常喜欢数字$7$，为此他还构建了一个$777$序列，这个序列所有的数字都是由数字$7$构成的数，形式如下：$7、77、777、7777、77777、777777.......$。

恰好，小浩今天在$TY$的奥信课上学习了模数的概念，因此对于输入的某一个数$k$，小浩想找到$777$序列中最小的数$x$，使得$x \ mod  \ k$等于$0$，即$x\%k=0$。

请你找到最小的数字$x$，并输出$x$的数位长度即可。

如果$777$序列中没有这样的数，则输出 `-1`。

## 输入格式

一行一个整数，表示模数$k$。

## 输出格式

一行一个整数，表示满足条件的数字$x$的长度，如果无解则输出`-1`。

```input1
101
```

```output1
4
```

### 样例解释

$777$序列中，$7777$可以使得$7777\%101=0$，因此输出$7777$的长度$4$。
