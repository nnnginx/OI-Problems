## 问题描述

8岁的阿尔和10岁的爱德华是两兄弟，从小就对炼金术很有兴趣。阿尔最近开始学习复杂的数学（其实就是加法和乘法的混合运算），在一旁的哥哥爱德华为了显露他的才能，利用伟大的炼金术，除了能把物体炼成以外还把算式的结果也一样瞬间炼成。那么聪明的你能够写一个程序，瞬间地对输入的一条算式，求出算式的结果吗？（提示：结果总小于maxlongint。）

## 输入格式

一字符串 $s$ 表示算式（算式中只包含数字和运算符 `+` 和 `*`）

## 输出格式

一行，算式结果

```input1
1+1*1
```

```output1
2
```

## 数据范围

$S$ 长度小于 $255$
