## 题目描述

在二维平面上有一个以中心点 $(X,Y)$，半径 $R$ 的圆。请计算出在圆内或圆周上的格点（即 $x$ 和 $y$ 坐标均为整数的点）的个数。


## 输入格式

输入一行三个数字 $ X $ 、$ Y $、 $ R $。

## 输出格式

输出答案。

```input1
0.2 0.8 1.1
```

```output1
3
```

```input2
100 100 1
```

```output2
5
```

```input3
42782.4720 31949.0192 99999.99
```

```output3
31415920098
```

## 数据范围

- $ |X|\ \le\ 10^5 $
- $ |Y|\ \le\ 10^5 $
- $ 0\ \lt\ R\ \le\ 10^5 $
- $ X,\ Y,\ R $最多精确到小数点后$4$位。
