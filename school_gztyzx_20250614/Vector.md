# Vector

## 题目描述

众所周知，向量是一个非常有用的东西。

无论是在数学，亦或是在物理，甚至在OI中，向量都有着不同寻常的作用。

由于你可能不太了解向量，这里提供了一个给你了解平面向量初步的地方：https://book.pep.com.cn/1421001122191/mobile/#p=9

众所周知，有三种常见的家禽：鸡、鸭、鹅。

他们一天在一起玩一个游戏：按照一定的轨迹做周期运动。

一开始他们的运动轨迹是椭圆，但是由于向心力的影响，他们现在做圆周运动。并且，他们越玩越开心，因此，他们都做加速运动

小鸡喜欢唱，跳，rap，篮球，所以他的运动很不稳定，做变速运动。

小鸭喜欢研究物理，他最近学习了匀变速直线运动，因此，他做匀变速运动。

小鹅喜欢研究数学，所以他做匀速运动，这样能更好的研究数学。

一开始，他们的坐标用数量来表示，但是，鸭在学习了一些物理知识后，认为用向量来表示位置很帅，于是提出要用向量来表示每只动物的坐标。但鹅觉得这样太复杂了，便拒绝了。

"I hate you! 不和你玩了！" 鸭以此来表明自己的不满。

没有办法，如果少了一只动物，这个游戏就没办法继续下去了。鹅只能答应了鸭的提议。

鹅的数学很好，他想，如果用向量来表示每只动物的位置，那就可以van一些好van的数学问题了。于是......

现在，你知道鸡，鸭，鹅分别按照半径为 $r_1$ , $r_2$ , $r_3$ 的圆做圆周运动，且这三个圆是同心圆。你站在这三个圆的圆心 $O$ 上，用向量来表示这三只动物的位置。向量的起点为你的位置，终点为那只动物的位置。我们用 $A$ 代表鸡，$B$ 代表鸭， $C$ 代表鹅，求下列式子的最小值：

$$
\vec {OA} \cdot \vec {OB} +\vec {OB} \cdot \vec {OC}  +\vec {OC}  \cdot \vec {OA}
$$

## 输入格式

一行，三个整数 $r_1$ , $r_2$ , $r_3$ ，含义如题所示。

## 输出格式

一行，一个实数 $ans$ ，表示题目中式子的最小值。

结果保留2位小数

## 样例 #1

### 样例输入 #1

```
1 3 5
```

### 样例输出 #1

```
-17.00
```

## 提示

对于 $100\%$ 的数据， $0 < r_1 \le r_2 \le r_3 \le 10^8$
