## 问题描述

约翰已经忘记他自己养了多少只奶牛了，而且他也不好意思去一个个数，因为他不想让奶牛们知道他对它们如此得不关心。所以，他准备偷偷地在每一个牛棚里装一个麦克风，通过收集到的声音来判断有多少奶牛。

约翰的 $N(1\leq  N \leq 100)$ 个牛棚分布在一条直线上。奶牛的品种有 $B$ 种($1\leq B \leq 20$)，第 $i$ 种奶牛的叫声是 $v_i$ 分贝($1\leq v_i \leq 100$)，每个牛棚了可能有多种不同的牛。因为构造的问题，牛棚的声音会向右扩散，假设在某个牛棚里牛的叫声共 $x$ 个分贝，那么下一个牛棚就能接收到 $x-1$ 个分贝的声音（依次类推）。

给你每个牛棚接收到的声音，你能计算出约翰最少有几头牛么？

## 输入格式

第一行输入 $N$ 和 $B$；

接下来输入 $B$ 行，每一行输入一个值 $v_i$，表示对应品种奶牛声音的分贝值；

接下来 $N$ 行，每行输入一个值 $x$，表示第 $i$ 个牛棚收集到的分贝数。

## 输出格式

输出约翰可能拥有的奶牛数量的最小值，如果问题无解，输出 $-1$。

```input1
5 2
5
7
0
17
16
20
19
```

```output1
4
```

## 样例解释

第2个牛棚有两只奶牛是品种1的，一只奶牛是品种2的，这样牛棚2的分贝数为17；

第3个牛棚的16个分贝是第2个牛棚传过来的；

第4个牛棚有一只品种1的奶牛，加上从牛棚3传来的15分贝，总共的分贝是20；

第5个牛棚的19分贝是第4个牛棚传来的；
