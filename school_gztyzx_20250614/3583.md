## 题目背景

同学给了小N一道难题，想让他做。小N想在同学面前展现自己，但是自己又不会做，想求助你。

## 题目描述

问题如下：

给定 $n$ 和 $m$ 以及 $n$ 个数 $a_{1\sim n}$ 和 $m$ 个数 $b_{1\sim m}$ ，求出以下式子的值：

$$
\sum_{i=1}^{n}\sum_{j=1}^{m}\max(a_i\ ,\ b_j)
$$

因为最终的值太大，所以最后只需求出答案$\mod (10^9+7)$ 的值。

如果你帮助小N，他会给你**丰厚的奖励**。

## 输入格式

第一行，输入两个正整数 $n$ 和 $m$ 。

第二行，输入 $n$ 个整数 $a_i$ 。

第三行，输入 $m$ 个整数 $b_i$ 。

## 输出格式

输出一个整数，表示式子的值。

## 样例 #1

### 样例输入 #1

```
4 5
1 2 3 4
2 2 2 2 5
```

### 样例输出 #1

```
64
```

## 提示

### 样例解释

![](https://cdn.luogu.com.cn/upload/image_hosting/hbewy71u.png)

所有数的和为 $64$ 。

### 数据范围

对于 $10\%$ 的数据，$1\le n,m\le 10$，$0\le a_i,b_i\le 100$ 。

对于 $30\%$ 的数据，$1\le n,m\le 10^3$，$0\le a_i,b_i\le 10^5$ 。

对于 $60\%$ 的数据，$1\le n,m\le 10^4$，$0\le a_i,b_i\le 10^9$ 。

对于 $100\%$ 的数据，$1\le n,m\le 5\times 10^5$，$0\le a_i,b_i\le 10^9$ 。
