## 题目描述

在$TY$机房上面有两张书桌，专门用来放置与编程相关的书籍，书桌$1$上有$n$本书，书桌$2$上有$m$本书，由于还没有购买书架，这些书都是横放且堆叠在一起的。

为了保证书本能摆放整齐，同学们读书时只能从上往下读，只有读完最上面的书本，才能继续阅读压在下面的书本，每本书都有一定的阅读时间，只有阅读完毕后，才能阅读下一本书。

对于书桌$1$上面的课本，从上往下堆叠的第$i$本书，其阅读时间为$a_i$，对于书桌$2$上面的课本，从上往下堆叠的第$i$本书，其阅读时间为$b_i$。

现在小东只有$k$个单位的阅读时间，他想知道自己最多能够看多少本书呢，请你帮他计算可阅读书本的数量。

## 输入格式

第一行输入三个整数$n,m,k$，分别表示书桌$1$书本的数量，书桌$2$书本的数量，小东的阅读时间。

第二行输入$n$个整数$a_1 \cdots a_n$，表示书桌$1$中$n$本书的阅读时间。

第二行输入$m$个整数$b_1 \cdots b_m$，表示书桌$2$中$m$本书的阅读时间。

## 输出格式

输出小东在$K$时间内，最多可读的书本数量。

```input1
3 4 240
60 90 120
80 150 80 150
```

```output1
3
```

```input2
3 4 730
60 90 120
80 150 80 150
```

```output2
7
```

```input3
5 4 1
1000000000 1000000000 1000000000 1000000000 1000000000
1000000000 1000000000 1000000000 1000000000
```

```output3
0
```

## 数据范围

- $ 1\ \leq\ n,\ m\ \leq\ 200000 $
- $ 1\ \leq\ k\ \leq\ 10^9 $
- $ 1\ \leq\ a_i,\ b_i\ \leq\ 10^9 $
