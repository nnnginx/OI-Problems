## 题目描述

小林刚吃完饭，准备回机房，发现门口的锁变成了密码锁！上面有一个字符串 $S$，长度是奇数的。

你需要判断这个字符串是不是超级回文串，所谓超级回文串就是字符串当且仅当全部满足以下几点条件：

- $S$ 是一个回文串，设 $n$ 是 $S$ 的长度。
- $S$的前$(\frac{(n-1)}{2})$个字符所组成的字符串是回文的。
- $S$的后$(n-\frac{(n+3)}{2}+1)$个字符所组成的字符串是回文的。

请你判断字符串$S$是不是超级回文串。

所谓回文串是指：正读和反读都一样的字符串，如$abcba$。

## 输入格式

一行一个字符串 $S$。

## 输出格式

如果 $S$ 是超级回文串，输出 $Yes$；否则输出 $No$。

```input1
akasaka
```

```output1
Yes
```

```input2
level
```

```output2
No
```

```input3
atcoder
```

```output3
No
```

## 数据范围

$S$只包含小写字母。

$S$的长度是一个属于$[3,99]$的奇数。
