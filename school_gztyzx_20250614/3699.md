## 题目描述

Memory和他的朋友Lexa在一款流行的电脑游戏中竞争更高的分数。

Memory从评分$ a $开始，Lexa从评分$ b $开始。在单个回合中，Memory和Lexa都获得$ [-k;k] $范围内的某个整数(即$ -k,-k+1,-k+2,…,-2,-1,0,1,2,…,k-1,k $)中的一个整数)。并将它们添加到当前分数中。

游戏有$ t $个回合。然而，Memory和Lexa并不擅长这个游戏，所以它们总是在轮到自己时得到$[-k,k]$中的一个随机整数。

Memory想知道有多少次游戏能够让他以比Lexa更高的分数结束游戏。如果在至少一个回合中至少有一个玩家获得不同的分数，则认为两个游戏是不同的。

总共有$ (2k+1)^{2t} $次游戏。由于答案可能非常大，您应该以$ 10^{9}+7 $为模输出。请为Memory解决此问题。

## 输入格式

第一行也是唯一一行输入包含四个整数$ a $， $ b $， $ k $和$ t $ ($ 1<=a,b<=100 $， $ 1<=k<=1000 $， $ 1<=t<=100 $)。

―$a$和$b$表示Memory和Lexa开始时的数量，数字$ k $表示取值区间，$t$是每次游戏的回合数。

## 输出格式

在一行中输出Memory赢得胜利的可能方案数$\% 1000000007 $。

```input1
1 2 2 1
```

```output1
6
```

```input2
1 1 1 2
```

```output2
31
```

```input3
2 12 3 1
```

```output3
0
```

## 提示

在第一个示例测试中，Memory从$ 1 $开始，Lexa从$ 2 $开始。如果Lexa选择$ -2 $，Memory可以选择$ 0 $、$ 1 $或$ 2 $获胜。如果Lexa选择$ -1 $，Memory可以选择$ 1 $或$ 2 $获胜。如果Lexa选0美元，Memory可以选2美元获胜。如果Lexa选择1美元或2美元，Memory不可能赢。因此，有$ 3+2+1= $ 6种可能的游戏，其中Memory获胜。

## 数据范围

对于$100\%$的数据，$ 1<=a,b<=100 $， $ 1<=k<=1000 $， $ 1<=t<=100 $。
