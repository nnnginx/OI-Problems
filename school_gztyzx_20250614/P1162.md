## 问题描述

FJ的 $N$ 头奶牛(奶牛编号为 $1..N$)围绕着篝火的 $N$ 把椅子(椅子编号也为 $1..N$)按顺时针方向坐成一圈，初始时第 $i$ 头奶牛坐在第 $i$ 把椅子上，FJ给奶牛们讲述昔日的故事。一个故事结束后，FJ建议他们进行奶牛消防演习。

在奶牛的消防演习中，当轮到奶牛 $i$ 移动时，她沿顺时针方向移动到途中遇到的第 $i$ 把椅子。比如轮到编号为 $3$ 的奶牛移动，她会从编号为 $3$ 的椅子开始移动，并移动到编号为 $6$ 的椅子处（如果 $N≥6$）。

当第 $i$ 头奶牛到达她的新椅子上，到达的那张椅子上的原来那头奶牛会开始移动，给奶牛 $i$ 腾出空间，然后奶牛 $i$ 坐下。这个过程一直持续，直到一头牛坐到空椅子上或者直到某头奶牛被要求第二次移动。在上述任何一种情况下，游戏结束。编号为 $1$ 的奶牛总是第一个开始，所以她的椅子总是空的。

最后结束比赛的奶牛（坐在编号为 $1$ 的椅子上或者到达一只已经移动过的牛那里），将收到一种特殊的嫩草奖励。

帮助FJ提前了解哪只奶牛会得到嫩草。


## 输入格式

输入只有一行一个整数 $N$；

## 输出格式

输出只有一行一个整数，结束比赛的那只奶牛的编号；

```input1
5
```
```output1
3
```


## 数据范围

对于 $20\%$ 的数据：$1≤N≤100$；

对于 $40\%$ 的数据：$1≤N≤250$；

对于 $60\%$ 的数据：$1≤N≤1,000$；

对于 $100\%$ 的数据：$1≤N≤500,000$；
