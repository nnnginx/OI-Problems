## 问题描述

Bessie买了 $N$ 本书，第 $i$ 本书的价格是 $Price[i]$。书店老板有个规定，如果把任意的 $3$ 本书放在一起打包，那么这 $3$ 本书之中价格最低的那本书可以不用给钱。那么Bessie应该如何打包，才能用最少的钱把这 $N$ 本书买回来？注意：是每 $3$ 本书都可以打包，你不能尝试着把 $4$ 本书放在一起打包，也不能把 $2$ 本书放在一起打包。

## 输入格式

第一行，一个整数 $N$。$ 1\leq N\leq 100000$。

接下来有 $N$ 行，第 $i$ 行是一个整数，表示第 $i$ 本书的价钱 $Price[i]$。 $1\leq Price[i]\leq 100000$。

## 输出格式

一个整数，最少的钱。

```input1
4
3
2
3
2
```

```output1
8
```

## 样例1解释

可以把前 $3$ 本书打包，这样第 $2$ 本书不用给钱。



```input2
6
6
4
5
5
5
5
```

```output2
21
```



## 样例2解释

分两次打包，可以省下 $9$ 元。

## 数据范围

有 $50\%$ 的数据，$N \leq 2000$。
