## 题目描述

小董$AK$了今年的$CSPS-2023$，因此他来到了$TY$奥信队$B$班的机房里分享经验，机房总是充满了快活的气息，偶尔会传出几句难懂的话，什么“TLE”不算错也，“骗分导论兮”，“我被卡肠乎”，“听取蛙声一片”之类的话，引得众人都哄笑起来。因此同学们特别讨厌青蛙，也特别不愿意看到单词$WRONG$。

然而，当英语老师来给集训的同学补习时，英语老师写下了长度为$n$的字符串$s$，这个字符串居然都是由$W$或者$R$组成的，其中不乏出现了大家厌恶的$WR$的成分，因此集训队的队员们希望你能帮助他们去除$WR$的子串！

现在你被赋予了修改字符串的能力，每次使用修改能力后，你可以使用以下的两个操作之一来修改字符串的任意字符：

- 1、交换字符串中的任意两个字符；
- 2、修改字符串中的任意一个字符，可以将$W$修改为$R$，或者将$R$修改为$W$。

由于队员们是非常讨厌`WR`，所以希望你使用最少的修改次数，使得字符串中不存在`WR`这样的子串。

## 输入格式

输入两行数据，

第一行输入一个正整数 $n$，

第二行输入长度为 $n$ 的字符串 $s$。

## 输出格式

输出一行一个非负整数，即使得字符串中不存在`WR`子串的最少操作次数。

```input1
4
WWRR
```

```output1
2
```

### 样例1说明

第一次修改，使用操作1，交换 $s_1$ 和 $s_3$，让 $s$ 变为`RWWR`；

第二次修改，使用操作2，修改 $s_4$ 为`W`，此时花费了最少的操作次数，使得字符串形成了`RWWW`，满足条件。

```input2
2
RR
```

```output2
0
```

```input3
8
WRWWRWRR
```

```output3
3
```

## 数据范围

对于$100\%$的数据， $2 \le n \le 200000$； $c_i$ 是`W`或`R`中的一个。
