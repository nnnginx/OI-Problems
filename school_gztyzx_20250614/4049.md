# Function

## 题目描述

求 $f(x)$ 函数值

$$
\large
f(x) =
\begin{cases}
f(x-1)*3-f(x-2)*2 & \text{x≥2} \cr
p
& \text{x=0}\cr
q & \text{x=1}
\end{cases}
$$

其中 $p$ ， $q$ 给定

由于答案可能非常大，你只需要输出 $\%10000007$ 后的结果

## 输入格式

第一行三个整数，$T$，$p$，$q$

接下来 $T$ 行，每行一个整数 $a_i$

## 输出格式

共 $T$ 行 , 第 $i$ 行为答案 $f(a_i) \% 10000007$

## 样例 #1

### 样例输入 #1

```
3 1 2
4
2
9
```

### 样例输出 #1

```
16
4
512
```

## 提示

对于 $20\%$ 的数据，$T=1$，$1≤p,q,a_i≤40$

对于 $40\%$ 的数据，$T=1$，$1≤a_i≤30000$

对于 $60$ 的数据，$T=1$，$1≤a_i≤1000000$

对于另外$20\%$的数据，$1≤T≤1000$，$1≤a_i≤10000$

对于$100\%$的数据，$1≤T≤1000000$，$1≤p,q≤32767$，$1≤a_i≤10000000$
