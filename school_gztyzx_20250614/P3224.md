## 问题描述

我们设本题场景开始的时刻为 $1$，结束的时刻为 $n$。

猫咪小S很爱吃糖．如果她在时刻 $i$ 吃糖，那么她以前最近一次吃糖的时刻必定不早于时刻 $s_i$。特别地，如果 $s_i=0$，那么小S可以从开始到时刻 $i-1$ 都不吃糖，而时刻 $i$ 吃糖。

但是小S只能从主人那里偷糖来吃。她要间隔一段时间才能偷糖。因此，同样地，如果小S在时刻 $i$ 吃糖，那么她以前最近一次吃糖的时刻必定不迟于时刻 $t_i$。

特别地，$n$ 是场景结束的时刻，小S也要在不早于 $s_n$、不迟于 $t_n$ 的时间段内吃一次糖，这样小S在结束的时刻才不会感到饥饿．由于小S在时刻 $n$ 不再偷糖，我们保证 $t_n=n-1$。

每个时刻的糖都有一定的美味度，记为 $d_i$。

虽然小S爱吃糖，但她也会感到厌倦；当然也不排除她越来越喜欢吃糖的可能。当她在时刻 $i$ 吃糖时，她的满足度为 $(k∙i+b)$。

小S的总喜悦值 $r$ 为她所有吃糖时刻的糖的美味度和其满足度的总和。即，$r=\sum_{P(i)},[d_i+(k∙i+b)]$，其中 $P(i)$ 表示小S在时刻 $i$ 吃了糖。注意，小S只可能在时刻 $1$ 到时刻 $n-1$ 之间的时间（含 $1$ 和 $n-1$ ）吃糖。

你的任务是求出小S的最大可能总喜悦值。

如果你不知道题目中符号的含义，你可以参考下面的定义。

> “$\sum$”是求和符号，设 $Q(i)$ 为对于 $i$ 的一个条件，$f(i)$ 为关于 $i$ 的一个函数（你也可以把 $f(i)$ 理解成一个普通的式子），则 $\sum_{Q(i)}f(i)=f(A_1)+f(A_2)+f(A_3)+⋯+f(A_r)$ 。其中，对于所有的整数 $i(1≤i≤r)$，条件 $Q(A_i)$ 都满足；而对于任意的 $x$，若不存在 $i$ 使 $A_i=x$ 成立，那么条件 $Q(x)$ 都不满足；对于所有的整数 $i,j(i≠j,1≤i,j≤r)，A_i≠A_j$。

## 输入格式

输入的第一行是三个整数，$n、k、b$，意义见“问题描述”。

接下来有 $(n-1)$ 行。这 $(n-1)$ 行中的第 $i (1≤i≤n-1)$ 行包含三个整数，分别为 $d_i、s_i$ 和 $t_i$，意义见“问题描述”.

输入的最后一行是两个整数，$s_n$ 和 $t_n$，意义见“问题描述”。

## 输出格式

有且仅有一行，为一个整数，表示小S的最大可能总喜悦值。

```input1
5 1 2
6 0 0
-50 1 1
9 0 1
5 1 2
4 4
```

```output1
20
```

```input2
5 0 0
4 0 0
5 0 1
6 1 2
7 2 3
3 4
```

```output2
22
```

## 样例1解释

小S在时刻 $1$、时刻 $4$ 吃糖，可以获得最大的总喜悦值，为 $[6+(1×1+2)]+[5+(1×4+2)]=20$。 可以证明，她不可能获得更大的总喜悦值。

## 数据范围及约定

下表列出了题中各个变量在各个测试点中的取值范围，以及一些特殊性质。

| 测试点编号 | $n≤$     | 特殊性质 |
| ------------ | --------- | ---------- |
|          1 |      $10$ | 无       |
| 2,3        |     $100$ |          |
| 4-7        | $10^3$    |          |
| 8,9        | $3×10^5$ | 性质A    |
| 10,11      | $3×10^5$ | 性质B    |
| 12-17 | $3×10^5$ | 性质C    |
| 18-25      | $3×10^5$ | 无       |

表中的特殊性质：

性质A：$k=0$。

性质B：对于所有的整数 $i (1≤i≤n)，t_i-s_i≤1$。

性质C：对于所有的整数 $i (1≤i≤n-1)，s_i≤s_(i+1),t_i≤t_(i+1)$。

## 数据范围

$1≤n≤3×10^5,|k|≤10^5,|b|≤10^9$。

对于所有的整数 $i (1≤i≤n-1)，|d_i |≤10^9$。

对于所有的整数 $i (1≤i≤n)，0≤s_i≤t_i<i$。

$t_n=n-1$。

题中出现的数字都是整数。
