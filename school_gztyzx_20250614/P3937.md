## 问题描述

给出字符串 $s$ 和 $n$ 个字符 $c_i$ （可以重复）。对于每个字符 $c_i$ ，你可以选择一个 $j$ ，将 $s_j$ 替换成 $c_i$，也可以不操作。求替换后的字典序最大的 $s$ 。

注意：一个 $c_i$ 至多替换一次。

## 输入格式

输入共三行：

第一行一个整数 $n$ ，表示字符个数。

第二行一个字符串 $s$ ，表示给出字符串。

第三行 $n$ 个不以空格分开的字符 $c_i$ ，表示 $n$ 个可供替换的字符。

## 输出格式

输出共一行一个字符串，表示替换后的字典序最大的 $s$。

```input1
1 a
bb
c
```

```output1
cbb
```

```input2
2 f
fg
ab
```

```output2
ffg
```

## 数据范围

对于 $30\%$ 的数据，$1\leq |s| \leq 10$。

对于 $60\%$ 的数据，$1\leq |s| \leq 5000$。

对于 $100\%$ 的数据，$1\leq |s|\leq 5\times 10^5 ，0\leq n\leq 5\times 10^5$ ， $s_i$和$c_i$ 均为小写英文字母。
