## 题目描述

FJ有$N(1 \leq  N \leq 1,000)$头牛在野外，他需要去给牛安装栏杆以防止们受到伤害，每头牛在野外每分钟受到一点伤害，假设所有牛和FJ所在地都在一条直线上，第$i$头牛的位置是$P_i(-500,000 \leq P_i\leq  500,000, P_i \not = 0)$，FJ在0点，他每移动一个单位都需要一分钟，假设安装栏杆不需要时间，他想知道这些牛受到最少的总伤害是多少。

## 输入格式

第一行一个整数$N$，接下来$N$行每行一个$P_i$。

## 输出格式

输出一行一个整数表示最少的伤害总量。

```input1
4
-2
-12
3
7
```

```output1
50
```
