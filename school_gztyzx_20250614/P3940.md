## 问题描述

给出两个仅由$ ( $和$ ) $组成的字符串 $s，t$ ，长度都为 $n$ ，你需要将两个字符串交错合并起来（具体见样例解释），使其成为一个合法的括号序列。问有多少种方案，对 $10^9+7$ 取模。

## 输入格式

输入一共两行：

第一行一个字符串 $s$，第二行一个字符串 $t$，表示两个括号序列。

## 输出格式

输出共一行一个整数，表示合法序列个数。

```input1
(()
())
```

```output1
19
```

对于 $30\%$ 的数据，$1\leq |s|，|t|\leq 12$。
