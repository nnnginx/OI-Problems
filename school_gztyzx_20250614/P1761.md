## 问题描述

人民在雷莫兰德共和国每年庆祝独立日。 不过，很久以前，没有人会记得什么时候。 人们唯一可以记得的是，今天自独立以来过的天数（$D$）是一个完全平方数，而且这个数字最大的可以形成一个小于或等于 $n$ 的不同数字的乘积。

由于Remoteland的年份有 $1,000,000,007$ 天，他们的公民只需要 $D$ 模 $1,000,000,007$。 请注意，他们对最大的 $D$ 感兴趣，而不是最大的 $D$ 模 $1,000,000,007$。

## 输入格式

每个测试用例由整数 $n(1\leq n \leq 10,000,000)$ 的单行描述。 输入以包含 $0$ 的行结束。

## 输出格式

对于每个测试用例，输出共和国成为独立天数，对 $1,000,000,007$ 取模，每行一个。

```input1
4
9348095
6297540
0
```
```output1
4
177582252
644064736
```
