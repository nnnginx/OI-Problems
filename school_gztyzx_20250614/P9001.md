## 题目背景

CodingCow 终于找齐了角色突破所需的材料。成功的突破了他最喜欢的一个角色：雷电将军。突破之后，CodingCow 需要使用经验书。但是由于 CodingCow 玩的是老版本，没有自动填充的功能。为了尽可能的减少经验的溢出，CodingCow 需要计算出需要使用多少本各种经验书。

## 题目描述

已知 CodingCow 已经攒了很久资源，所以拥有无限的绿、蓝、紫三种经验书。CodingCow 的雷电将军升级需要 $x$ 点经验，绿、蓝、紫三种经验书能提供的经验分别为 $10$，$50$，$120$。求不溢出的，经验值最大的一种方案。多种方案的存在并不影响评测。

## 输入格式

第一行一个整数，$x$，意思同题目。

## 输出格式

输出三个整数，分别表示绿、蓝、紫三种经验书的使用数量。

## 样例

```input1
101
```

```output1
5 1 0
```

```input2
514
```

```output2
3 0 4
```

```input3
1145141938
```

```output3
0 1 9542849
```

### 数据范围

对于 $100\, \%$ 的数据，$1 \leq x \leq 10^{18}$。
