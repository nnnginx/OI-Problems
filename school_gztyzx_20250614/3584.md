## 题目背景

小N最近在学 $\operatorname{mod}$ 运算，老师想看看小N有没有好好学习，于是出了一道题来考考小N。

## 题目描述

老师给出一个只含有字符 $0\sim9$ 和 $?$ 的字符串 $n$ 。

小N需要在 $?$ 中填入数字（$0\sim9$），使填完后的数是 $p$ 的倍数。（可以有前导零）

问总共有多少种方案。小N很菜，希望你能帮助他。

## 输入格式

第一行，输入一个字符串 $n$ 。

第二行，输入一个正整数 $p$ 。

## 输出格式

输出总方案数 $mod\ 10^9+7$ 的值。

## 样例 #1

### 样例输入 #1

```
?44
11
```

### 样例输出 #1

```
1
```

## 样例 #2

### 样例输入 #2

```
?1?1
7
```

### 样例输出 #2

```
14
```

### 数据范围

对于 $20\%$ 的数据，$|n|\le5$，$2\le p\le 100$ ，且字符串 $n$ 中只含有字符 $?$ 。

对于 $40\%$ 的数据，$|n|\le10$，$2\le p\le 30$ 。

对于 $60\%$ 的数据，$|n|\le10^3$，$2\le p\le50$ 。

对于 $100\%$ 的数据，$|n|\le10^5$，$2\le p\le 100$ 。
