## 问题描述

小马面前一共有 $n$ 堆积木，第 $i$ 堆积木的高度为 $A[i]$。

他每回合可以往任意一堆积木顶部再放一个零件，使这堆积木的高度增加 $1$。

为了使每堆积木的高度都不一样，小马最少需要几回合呢。

## 输入格式

第一行输入一个非负整数 $n$；

第二行输入 $n$ 个非负整数，表示 $A[i]$；

## 输出格式

输出一个非负整数，表示最少需要的回合数

```input1
3
2 3 2
```

```output1
2
```

## 样例1解释

例如：一共有 $3$ 堆积木，高度分别是 $2\ 3\ 2$，则最少需要 $2$ 回合，使高度变成 $2\ 4\ 3$ 或者 $2\ 3\ 4$



```input2
5
3 7 5 2 6
```

```output2
0
```

```input3
7
2 1 4 7 8 9 1
```

```output3
2
```



## 数据范围

对于 $30\%$ 的数据，$1\leq n\leq 10$ ；

对于 $75\%$ 的数据，$0\leq A[i]\leq 10$；

对于 $100\%$ 的数据， $1\leq n\leq 40000,0\leq A[i]\leq 40000$
