## 题目描述

各位同学们终于比赛完了。为了庆祝比赛结果，同学们将吃蛋糕庆祝。但是因为一个班级的人数众多所以不得不郁闷了起来。

本来想去蛋糕店给每一位同学买蛋糕。但是蛋糕店老板却说，他这里只有$N$边形蛋糕。

所以为了方便，他们只好买$N+2$边形蛋糕了。这样刚好能够切出$N$边形蛋糕，每一个蛋糕刚好是三角形，并且刚好包含$N$变形的三个角。

但是因为蛋糕的形状不同，及老板的手艺不娴熟。每一个蛋糕的每一个边角都不一定是均匀，并且除了顶点，其余地方没有味道。

每一个边角都有一个相应的美味值（$<1e11$）。有些蛋糕的边的味道更好。而同学们只想尝一下某一个边角。

现在同学们很懊恼，不知如何分的话，能够让同学们尝到的美味最多？

现在把问题交给了你，每一位同学拿到一个三角形的蛋糕以后，他们品尝的味道是蛋糕三角的三个角的乘积。老师让你能否想到一个方案，使得班级里面的所有同学拿到的蛋糕的美味值最大？

（一个边角不管划分多少次，只有被某一个同学拿到了，味道自然就会留在当前同学拿到的三角形当中）

## 输入格式

输入第一行为$N$表示蛋糕的边个数 $N\leq 50$.

第二行依次为$1$ 至$N $的边角的美味值。

## 输出格式

输出仅一行，为这些每一位同学拿到的味道值的和。

```input1
5
121 122 123 245 231
```

```output1
12214884
```

## 数据范围

保证结果在$long \ long$范围。
