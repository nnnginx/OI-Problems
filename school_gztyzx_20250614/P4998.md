## 题目背景

这是什么，操作题？做一下。

## 题目描述


一个序列共有 $n$ 个数，我们称第 $i$ 个数为 $a_i$，最大值为 $mx$，平均值为 $avg$。每一轮操作为：

- 对于每一个数 $i(1 \le i \le n)$ 若 $a_i<mx$，则将 $a_i$ 增加 $1$，更新 $avg$。

现在给出一个数 $x$，请求出至少多少（整数）轮操作，才能使得 $a_x\ge avg$。保证最初 $a_x\le avg$。


## 输入格式

第一行两个数 $n$，$x$，含义见题面。

接下来一行，$n$ 个数，第 $i$ 个数为 $a_i$。

## 输出格式

一行一个数，表示答案。

## 样例 #1

```input1
6 2
1 1 4 5 1 4
```

```output1
4
```

## 样例 #2

```input2
6 3
4 8 4 7 6 1
```

```output2
3
```

## 样例 #3

```input3
6 3
97 46 12 87 28 6
```

```output3
84
```

## 提示

**【样例一解释】**

第三次操作后序列为`4 4 5 5 4 5`，平均数为 $4.5$，第五次操作后序列为`5 5 5 5 5`，平均数为 $5$。

**【样例二解释】**

第二次操作后序列为`6 8 6 8 8 3`，平均数为 $6.5$，第五次操作后序列为`7 8 7 8 8 4`，平均数为 $7$。

**【数据范围】**

对于 $100\%$ 的数据：$1\leq n\leq 10^6,1\leq a_i\leq 10^9$

| 数据点 |        $n$         |                  $a_i$                  |
| :----: | :----------------: | :-------------------------------------: |
| $1\sim2$  | $\le 10^3$ |          $a_i \le 10^3$           |
| $3\sim4$  |  |          $a_i \le 10^6$           |
|  $5$   |  |           所有 $a$ 中的数相等           |
|  $6$   |  | $\max^{n}_{i=1}a_i-\min^{n}_{i=1}a_i=1$ |
| $7\sim10$ |  |  |

留空表示无额外限制。
