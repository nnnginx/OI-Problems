## 题目描述

对 $200$ 这个数情有独钟的苹果先生拜托你来解决这个问题。

给定一个长度为 $N$ 的正整数序列 $A$，请求出所有满足 $1 \leq i < j \leq N$ 且 $A_i-A_j$ 为 $200$ 的倍数的二元组 $(i, j)$ 的个数。

## 输入格式

第一行输入整数$ N $ 。

第二行输入$N$个正整数，$ A_1 $ 、$ A_2 $ $ \dots $ $ A_N $。

## 输出格式

输出答案。

```input1
6
123 223 123 523 200 2000
```

```output1
4
```

### 样例1解释

二元组$(i,j)$的一种方案是$(1,3)$，选择序列中的数字$a_1$和$a_3$，$a_1-a_3=0$是$200$的倍数。

二元组$(i,j)$的第二种方案是$(1,4)$，选择序列中的数字$a_1$和$a_4$，$a_1-a_4=-400$是$200$的倍数。

二元组$(i,j)$的第三种方案是$(3,4)$，选择序列中的数字$a_3$和$a_4$，$a_3-a_4=-400$是$200$的倍数。

二元组$(i,j)$的第四种方案是$(5,6)$，选择序列中的数字$a_5$和$a_6$，$a_5-a_6=-1800$是$200$的倍数。

```input2
5
1 2 3 4 5
```

```output2
0
```

```input3
8
199 100 200 400 300 500 600 200
```

```output3
9
```

## 数据范围

- $ 2\ \le\ N\ \le\ 2\ \times\ 10^5 $
- $ 1\ \le\ A_i\ \le\ 10^9 $
