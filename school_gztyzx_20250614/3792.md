## 题目描述

定义$\textrm{TY}$数为位数不大于 $n$ 且各数位之和不超过 $k$ 的正整数。

给定 $n,k$，求$\textrm{TY}$数的个数。

## 输入格式

第一行输入两个整数 $n,k$。

## 输出格式

输出一行一个整数，表示答案。

```input1
3 2
```

```output1
9
```

## 样例解释

在所有三位数中，有 $1,2,10,11,20,100,101,110,200$ 共 $9$ 个数的数位和不超过 $2$。

## 数据规模与约定

对于 $30\%$ 的数据，$n=1$。

对于 $100\%$ 的数据，$1 \leq n \leq 7$，$1 \leq k \leq 100$。
