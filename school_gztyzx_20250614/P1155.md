## 问题描述

一天，晨晨的数学老师布置了一道题目，大意如下：用 $1×1$ 和 $2×2$ 的磁砖不重叠地铺满 $n×3$ 的地板，共有多少种方案？

例如：

$n=1$ 时：$1×3$ 的地板方法就一个，直接由三个 $1×1$ 的磁砖铺满。

$n=2$ 时：$2×3$ 的地板可以由下面 $3$ 种方案铺满：

![image](./163/file/zHNRB0guX4dvZkGbXw15a.png)

## 输入格式

第一行：一个整数 $n(1≤n≤50)$。

## 输出格式

输出铺满 $n×3$ 的地板的方案数。

```input1
3
```

```output1
5
```

## 数据范围

对于 $100\%$ 的数据，$1\leq n\leq 50$;
