# 问题描述

信息组有N个学生，被标号为1......N，现在正站成一排。

每个学生分别属于预备、初一和初二这三个年级，用数字1、2、3来表示。

现在老师想让你帮他数出在给定区间中每个年级学生的数量。

# 输入格式

第一行两个整数N, Q($1<=N<=10^5$, $1<=Q<=10^5$)

接下来N行，每行一个整数1，2或3，表示这个学生的年级

接下来Q行，每行两个整数a,b表示要求的区间 [a,b]  中，每个年级学生的数量 ($a<=b$)。

# 输出格式

共Q行，每行三个整数，分别对应每一个询问中预备年级(数字1)，初一(数字2)，初二(数字3)的学生的数量。

```Input1
6 3
2
1
1
3
2
1
1 6
3 3
2 4
```

```Output1
3 2 1
1 0 0
2 0 1
```
