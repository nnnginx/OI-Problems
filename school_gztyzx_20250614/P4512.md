## 题目描述

给出三个整数$N,M,K$，请你计算有多少个长度为 $N$ 的整数序列 $A=(A_1,\ldots,A_N)$ 满足下面的所有条件？

- $1\le A_i \le M$ $(1 \le i \le N)$
- $|A_i - A_{i+1}| \geq K$ $(1 \le i \le N - 1)$

由于计数可能非常庞大，因此求出

$998244353$ 的模数。

通俗地讲，你需要从$[1,M]$中取出$N$个数字组成一个序列$A_i\ | \ 1\leq i \leq N$，这个序列中相邻的两个数的差值大于等于$K$，即$|A_i - A_{i+1}| \geq K$ $(1 \le i \le N - 1)$。

请你求出不同序列的方案数。

## 输入格式

输入三个整数：$N$ ，$M$， $K$。

## 输出格式

输出方案数 $\%$$998244353$ 。

```input1
2 3 1
```

```output1
6
```

下列 $6$ 序列满足条件。

- $(1,2)$
- $(1,3)$
- $(2,1)$
- $(2,3)$
- $(3,1)$
- $(3,2)$

```input2
3 3 2
```

```output2
2
```

下列 $2$ 序列满足条件。

- $(1,3,1)$
- $(3,1,3)$

```input3
100 1000 500
```

```output3
657064711
```

## 数据范围

- $2 \leq N \leq 1000$
- $1 \leq M \leq 5000$
- $0 \leq K \leq M-1$
- 输入值均为整数。
