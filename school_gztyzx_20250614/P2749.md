## 问题描述

程序龙在校园里捡到一张纸条，纸条上写着一个数学算式 $A+1=B$，看得出$A$ 和 $B$ 都是两位正整数，但是因为纸条经过多次涂画，已经看不清$A$和$B$的具体数值了，只能看清$A$ 最左边的数字是 $a$，而 $B$最左边的数字是 $b$。有好奇心的程序龙下定决心要还原出这个等式，但他试了很多次都失败了。无奈之下只能请你判断下用 $a$ 和 $b$ 能否还原出 $A$ 和 $B$，使得算式 $A+1=B$ 成立。如果能成立，输出 $Yes$，否则输出 $No$。

## 输入格式

仅一行，正整数 $a$ 和 $b$ ($1\leq a,b\leq 9$)。

## 输出格式

$Yes$ 或 $No$

```input1
1 2
```

```output1
Yes
```

```input2
6 3
```

```output2
No
```

## 样例解释

样例 1 说明：$A$ 可以为 $19$，$B$ 为 $20$；<br />
样例 2 说明：$6$ 开头的任何一个两位数加 $1$ 的结果不可能以 $3$ 开头。
