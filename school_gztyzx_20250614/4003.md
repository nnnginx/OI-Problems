## 题目描述

白浅妹妹今天学习了求余运算，她很好奇求余运算和乘法运算结合起来会是什么样子，于是她设计了这样一道题目。

给定数字 $n$，你可以任选一个数字 $k(1 ≤ k ≤ n)$，然后计算出 $𝑛 \% 𝑘$ 的值（其中 $\%$ 为求余运算），记为 $𝑞$，请问 $𝑘 ∗ 𝑞$ 的最大值是多少。

## 输入格式

输入仅包含一个正整数 $n$。

## 输出格式

输出一行一个整数表示答案。

```input1
4
```

```output1
3
```

## 样例解释

选择 $k = 3$，那么用 $4$ 求余 $3$ 得到余数 $1$，最终算出答案为 $1 ∗ 3 = 3$
