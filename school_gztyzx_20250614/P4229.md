## 【题目描述】

给你一堆数，第 $i$ 个数表示为 $a_i$，请你计算 $a_i\times a_j$ 的最大值 $(i\neq j)$ 。

## 【输入格式】

共两行。
第一行有一个数 $n$，表示数的个数，
第二行有 $n$ 个数，第 $i$ 个数为 $a_i$。

## 【输出格式】

见题意。

```input1
5
1 2 3 4 5
```

```output1
20
```

```input2
3
0 2 4
```

```output2
8
```

## 【数据范围】

对于$100\%$的数据，有：

$n \leq 10^7$

$-10^9 \leq a_i \leq 10^9$

## 【提示】

注意数据范围
