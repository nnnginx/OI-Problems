## 题目背景

摩尔斯电码（Morse code）也被称作摩斯密码，是一种时通时断的信号代码，通过不同的排列顺序来表达不同的英文字母、数字和标点符号。它的代码包括五种：短促的点信号`.`，保持一定时间的长信号`-`，表示点和划之间的停顿、每个词之间中等的停顿，以及句子之间长的停顿。

## 题目描述

你截获了敌人的一串摩斯电码，现在你要破译它。

敌人也不是傻，他们把摩斯译文在加密了一下，就是会有 $n$ 条字母替换信息，有两个大写字母 $x,y$ ，要你把字符串中所有的 $x$ 替换成 $y$ 。但他不知道他们的电报员之中有一个卧底，卧底会给你 $n$ 个消息，告诉你 $x,y$ 。

## 输入格式

第一行，输入 $n$，表示有 $n$ 条信息。

第二行，输入一个字符串表示你截获的密文，含 `.` 、`-` 和空格。

注意：字符串行末有空格

接下来 $n$ 行，每行输入两个大写字母 $x,y$ ，表示卧底给你发的加密信息。

## 输出格式

输出一行，一个字符串表示译文，包含大写字母，无空格，无数字。

## 样例 #1

### 样例输入 #1

```
3
... --- ... 
S A
O W
A Q
```

### 样例输出 #1

```
QWQ
```

## 提示

对于所有的 $n  \le 10^5$ ，密文长度不超过 $5 \times 10^5$ 。
