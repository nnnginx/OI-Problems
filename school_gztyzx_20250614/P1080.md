## 题目描述

$TY$机房原来的密码是一串小写英文字母，然而一些同学破解了密码，导致他们不按作息要求可以自由进出，这让$chxulong$老师非常的生气，所以他请了小昱同学为$TY$的密码进行加密。

加密的规则是这样的：要是原密码中出现两个以上（包括两个）连续的相同的字母，则把他们替换成这个字母的大写形式，后面紧跟相同字母的个数，并把它之前跟之后的两段字串调换，例如出现$bcaaaaaaef$，则加密后新字符串变成：$efA6bc$。然后从新扫描字串，继续对相同字符加密，直到没有出现相同小写字母为止。

## 输入格式

第一行输入一行字符穿：原始密码字符串。

## 输出格式

输出加密后的新字符串。

```input1
bcaaaaaaef
```

```input2
cmmmcefffg
```

```output1
efA6bc
```

```output2
gM3cF3ce
```
