## 题面翻译

在$TYOI$冬令营中，同学们做了一个文化课的测试，测试总共有 $n$道题目（$3 \le n \le 100,000$），每道题目的成绩在 $0 \sim 10,000$ 之间。

$JY$老师将会按照求平均值的方式来计算你的最终成绩：去掉你最低的一个成绩，然后将剩下的成绩求和，再求平均值作为你的最终成绩。

然而你准备交卷时，你的宠物狗一口咬掉了前 $k$个题目和答案！（$1 \le k \le N-2$）这有可能使得你丢失这一部分的分数！

通过你的狡辩，$JY$老师居然相信了你的鬼故事，决定不理会被咬掉的$k$个题目，也就是对剩下的$[k+1,n]$道题目继续按上述的计分方式来计分――首先去掉最低的成绩（如果有多个最低成绩，则只去掉其中一个），然后取剩余成绩的平均值作为成绩。

你当然希望你的成绩是越高越好的，因此请你根据这一成绩的计算方案，按升序输出所有可以使你最终成绩最高的 $K$ 的值，相邻输出之间用**换行**隔开。



## 输入格式

第一行输入一个整数$n$；

第二行输入$n$个整数，表示$n$个题目的成绩。

## 输出格式

请按升序输出所有可以使你最终成绩最高的 $K$ 的值，相邻输出之间用**换行**隔开。


```input1
5
3 1 9 2 7
```

```output1
2
```

## 数据范围

对于$100\%$的数据，$3 \le n \le 100,000$） $0 \leq 题目分数 \leq  10,000，1 \le k \le N-2。$
