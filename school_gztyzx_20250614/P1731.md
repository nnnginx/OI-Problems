## 问题描述

oiclass网站的基地中有一些奇怪的美式据点，这些据点和周围的中国据点风格格格不入。这些伪装成据点的建筑其实是漂亮国的特工基地！

这些据点很有规律地构成了一个 $m*n$ 的矩形，每个据点和它的八个方向的据点有直接的路径相连。水平或垂直方向相邻的据点之间的路程均为 $1$，斜边方向距离为 $1.41421$。

程序龙给你发放了一项任务，就是从某个据点出发，逐个访问这些据点，必须把所有的据点都全部访问，并回到出发的地方。

请问你至少需要走多远的路，才能完成这个危险而艰巨的任务呢？

## 输入格式

输入一行两个整数 $m$ 和 $n(m，n≤10000)$

## 输出格式

输出一行一个实数，表示最少需要走的路程，保留两位小数。

保留两位小数参考以下代码：

```CPP
#include<iomanip>

cout<<setprecision(2)<<fixed<<a;
```



```input1
2 3
```

```output1
6.00
```

## 样例解释

样例输入的 $2\ 3$，即为 $2*3$ 的矩阵，据点位置如下图所示

![image](./762/file/TGfISKc4-xH2XB2HqHMSz.png)

程序龙可以任选其中的某一个据点出发，然后访问其他的所有据点，最少路程是 $6.00$
