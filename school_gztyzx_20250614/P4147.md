## 问题描述

最近，$lily$ 玩画图画上瘾了，画着画着她觉得画不过瘾，于是她用 $C++$ 自己写了一个画图程序。<br />
她想画一个指定高度的三角形，用从 $1$ 开始的连续自然数逆时针填充三角形的边，即沿着边依次写出从 $1$ 开始的自然数。但是 $lily$ 腐朽太久了，打代码不够熟练了，正好你颓废又被她抓住了，她让你帮她写出来这个画图程序，不然就要写检讨（6000字的那种！），你还是赶紧开始干吧……

## 输入格式

一个整数，即三角形的高度。

## 输出格式

题目所要求的三角形，空白处用"."填充。

```input1
5
```

```output1
....1
...2.1
..3...2
.4.....1
567891011
```

## 数据范围

$1\leqslant n \leqslant 100$
