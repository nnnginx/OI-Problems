## 问题描述

铁一创智空间有许多迷你机器人，根据机器人的脚的数量可以分为人型机器人和犬型机器人，其中人型机器人有 $2$ 只脚，犬型机器人有 $4$ 只脚。有一天，无聊的tyoier建议玩一个游戏，游戏是已经知道创智空间里面的脚的总数 $n$，问创智空间里面最少有多少个机器人，最多有多少个机器人。

## 输入格式

一行，包括一个正整数 $n$ 。

## 输出格式

一行，共两个正整数，第一个是最少的机器人数量，第二个是最多的机器人数量，两个正整数用空格分开。如果没有满足要求的情况出现，则输出 $2$ 个 $0$

```input1
3
```

```output1
0 0
```

```input2
6
```

```output2
2 3
```

```input3
20
```

```output3
5 10
```

## 数据范围

$n$ 在 $20$ 亿以内，注意输出的两个数中间有一个空格。
