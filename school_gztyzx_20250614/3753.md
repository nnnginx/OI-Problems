## 【问题描述】

小C家门前有一条由$n$块地砖构成的走廊，其中第$i$块地砖的颜色要么为黑色要么为白色，黑色用$1$表示，白色用$0$表示，则整条走廊可以用一个长度为$n$的$01$串表示。现在想要让黑色的地砖（如果有的话）连成一个大块，为此他可以把一些白地砖染成黑色，现在小C想知道他至少需要将多少个白地砖染成黑色才能实现目标。

## 【输入格式】

一行，一个字符串，保证字符串仅包含$0、1$。

## 【输出格式】

一行，包含一个整数，表示至少需要将多少个白地砖染成黑色

```input1
010011
```

```input2
0
```

```input3
1111000
```

```output1
2
```

```output2
0
```

```output3
0
```

## 【数据范围】

对于$100\%$ 数据，保证字符串的长度$len\le 10^5$。

## 样例解释

对于输入$1$，需要将中间的两个白地砖染成黑色，变成$011111$。
