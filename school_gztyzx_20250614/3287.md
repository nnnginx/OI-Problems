## 题目描述

放暑假啦！阿潮带着他的小宇出去玩！小宇在海豚岛旁边发现了一些海豚！由于阿潮很喜欢数学，他想到了一个问题：现在有 $4$ 只海豚，把这 $4$ 只海豚看作一个长方形的四个点，它们可以组成一个整体的大长方形。如图所示：

![](https://cdn.luogu.com.cn/upload/image_hosting/mps7msdr.png)

但是由于不同时候海豚的位置都不同，所以他想知道这 $4$ 只海豚在 $n$ 个时刻内它们整体的长方形在合并成一个大长方形后的整体边长之和。举个当 $n=3$ 时其中一个的例子：
![](https://cdn.luogu.com.cn/upload/image_hosting/qs8kg4id.png)

将会合并为

![](https://cdn.luogu.com.cn/upload/image_hosting/dui4oxep.png)

## 输入格式

第一行，一个正整数 $n$ ，表示有 $n$ 个海豚游动时刻。

接下来 $n$ 行，每行输入四个整数，表示每次 $4$ 只海豚中左下角的海豚坐标 $(x_1,y_1)$ 和右上角的海豚坐标 $(x_2,y_2)$ 。

## 输出格式

一行，一个正整数 $ans$ ，表示 $4$ 只海豚们在经过 $n$ 个时刻后组成长方形的合体长方形边长之和。由于这个数可能很大，你只需要输出它对 $998244353$ 取模的值。

## 样例 #1

### 样例输入 #1

```
3
-3 1 -1 4
-2 -2 3 2
0 -3 4 0
```

### 样例输出 #1

```
28
```

## 提示

对于 $20\%$ 的数据， $n \le 10^3$ 。

对于 $100\%$ 的数据， $1 \le n \le 10^4 ， -10^4 \le x_1,y_1,x_2,y_2 \le 10^4$ 。
