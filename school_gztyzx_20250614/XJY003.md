## 题目描述

因式分解都知道吧？

---

**前置芝士：**

因式分解：将一个式子转为若干个**单项式**的乘积形式。

有很多种方法可以做到：

1. 提取公因式法：$2ab^2+8b=2b(ab+4)$；
2. 公式法：
   * 完全平方公式：$a^2\pm2ab+b^2=(a\pm b)^2$；
   * 平方差公式：$a^2-b^2=(a+b)(a-b)$；
3. 十字相乘法：$a^2+2a-3=(a-1)(a+3)$。

因式分解可以结合起来用，譬如：

$18a^2-8b^2-3a+2b$

$=2(9a^2-4b^2)-(3a-2b)$

$=2(3a+2b)(3a-2b)-(3a-2b)$

$=(6a+4b-1)(3a-2b)$。

---

有没有被上面的话吓到？

**问题：**

$\textup{\textbf{In\ fact}}$，你只需要考虑提取公因式的情况就可以啦。

输入一个式子，输出它**仅使用**提取公因式法之后因式分解的结果。

$\textup{\textbf{Patient\ is\ bitter,but\ it's\ fruit\ is\ sweet.}}$

## 输入格式

一行，一个字符串。

## 输出格式

一行，一个字符串。

注意每一个单项式的顺序均按原顺序，提出的公因式中字母升序排序。

```input1
2ab^2+8b
```

```output1
2b(ab+4)
```

```input2
48o+48i
```

```output2
48(o+i)
```

## 提示

#### **数据范围：**

对于 $12\%$ 的数据，输入不含字母。

对于 $100\%$ 的数据，$|s|\leq100$ 且出现的常数均 $<10^6$。

#### **几个保证：**

* 数据保证加、减分别为 `+`、`-`，`-` **仅**代表减号；
* 数据保证数字和字母之间相乘时忽略乘号且数字在字母前，字母和字母之间相乘时忽略乘号；
* 数据保证未知数仅有小写字母；
* 数据保证式子**最简**；
* 为保证强度，数据都是手造的；
* 数据保证**字母**不会作为某个数的乘方出现。
