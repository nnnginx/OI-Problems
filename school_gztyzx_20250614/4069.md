## 题目描述

XQ 喜欢随机说话。

XQ 每次会说一句话，包含三个数 $a,b,c(a<b<c)$。这些数看似是随机的，实际上需要满足 $\gcd(a,b)=\gcd(a,c)=\gcd(b,c)=1$ 的要求。

XQ 受到了 SS 的神秘限制，现在他每次说的数要在 $l\sim r$ 之间，并且**一段话中一定不能说一个数两遍**。比如上次 XQ 说了 $(1,2,3)$ 则这次他不能说 $(1,3,4)$ 或者 $(3,5,7)$。

XQ 很喜欢随机说话，他想问你他最多可以说几句话？

## 输入格式

有 $T\le 100$ 组测试数据，第一行一个整数 $T$ 表示数据组数。

接下来 $T$ 行每行输入两个整数 $l,r$。

## 输出格式

对于每一组数据，输出一个整数表示答案。一共输出 $T$ 行。

## 输入输出样例 #1

### 输入 #1

```
2
1 3
3 7
```

### 输出 #1

```
1
1
```

## 说明/提示

第一组测试数据，说 $1,2,3$。

第二组测试数据，说 $3,5,7$。

## 数据范围

$1\le l\le r \le 10^9$。
