## 问题描述

有 $N$ 奶牛排成一排，农夫约翰要把连续一段的奶牛安排进一个宿舍。

但是，安排进一个宿舍的规则比较特别，要么保证整个宿舍都是同一性别的奶牛，要么公牛和母牛的性别差不超过 $M$。农夫约翰想知道，至少需要分进多少个宿舍。

## 输入格式

输入文件第一行包括 $N$ 和 $M$。

之后 $N$ 行，每行一个整数，`1` 表示公牛，`2` 表示母牛。

## 输出格式

输出一个整数，表示所需的最少宿舍数量。

```input1
5 1
2
2
1
2
2
```

```output1
2
```


## 数据规模

对于 $30\%$ 的数据，有 $1 ≤ N ,M≤ 50$；

对于 $100\%$ 的数据，有 $1 ≤ N,M ≤ 2500$。
