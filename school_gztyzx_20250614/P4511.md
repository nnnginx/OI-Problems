## 题面翻译

有 $N$ 个孩子，编号为 $1, 2, \ldots, N$ 。

他们决定一起分享 $K$ 粒糖果。在这里，对于每个 $i$ ( $1 \leq i \leq N$ )，第 $i$ 个孩子至少分得 $0$ 颗，至多分得 $a_i$ 颗。此外，糖果不能剩余。

求他们分享糖果的方法数，模数为 $10^9 + 7$ 。在这里，如果有一个孩子分到的糖果数量不同，则这两种分法就称为不同分法。

## 输入格式

输入内容由标准输入法提供，格式如下：

$N$ ，$K$；

$a_1$， $a_2$， $\ldots$ $a_N$。

## 输出格式

输出孩子们分享糖果的方式数，模数为 $10^9 + 7$ 。

```input1
3 4
1 2 3
```

```output1
5
```

孩子们有以下五种分享糖果的方式：

- $(0, 1, 3)$
- $(0, 2, 2)$
- $(1, 0, 3)$
- $(1, 1, 2)$
- $(1, 2, 1)$

这里，在每个序列中， 第$i$ 个元素代表孩子 $i$ 收到的糖果数量。

```input2
1 10
9
```

```output2
0
```

```input3
2 0
0 0
```

```output3
1
```

孩子们有一种分享糖果的方法，如下所示：

- $(0, 0)$

```input4
4 100000
100000 100000 100000 100000
```

```output4
665683269
```

## 数据范围

- 输入值均为整数。
- $1 \leq N \leq 100$
- $0 \leq K \leq 10^5$
- $0 \leq a_i \leq K$
