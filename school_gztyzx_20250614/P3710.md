## 问题描述

为了备考 D 站面试，小 V 不得不掏出手机复习表情包。 D 站的表情包放的乱糟糟的，为了表现自己的决心，小 V 决定整理表情包。于是，他请来了卓越的你。他还有以下要求：

1. 对表情包进行从小到大排序。
2. 为了难住你，他希望排序的是表情包的所有因数之和（包括 $1$ 和它本身）。
3. 另外他不喜欢 $3$ 这个数字，所以他希望因数中没有 $3$ 的倍数。

## 输入格式

第一行一个整数 $n$，代表表情包的个数。

接下来一行 $n$ 个整数 $a_i$ ，代表每个表情包。

## 输出格式

输出就一行， $n$ 个整数 $a_i$ ，代表满足条件的表情包序列。

```input1
5
2 4 9 3 5
```

```output1
1 1 3 6 7
```

## 样例解释

共有 $5$ 个表情包。

$2$：$1 + 2 = 3$

$4$：$1 + 2 + 4 = 7$

$9$：$1 + 3 + 9 - 3 - 9 = 1$（$3$ 和 $9$ 都是 $3$ 的倍数）

$3$：$1 + 3 - 3 = 1$（$3$ 是 $3$ 的倍数）

$5$：$1 + 5 = 6$

排序后变为：$1,1, 3, 6, 7$

## 数据范围

对于 $50\%$ 的数据，$1 \le n \le 10^2$。

对于 $100\%$ 的数据，$1 \le n \leq 10^3$，$2 \le a_i \le 10^3$。
