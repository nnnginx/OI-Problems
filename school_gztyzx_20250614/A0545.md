## 题目描述

输入一个“$n$ 位的二进制数” 的二进制原码 $s$，请输出它对应的补码。

- 正数：
  - 三种编码一样，最高位为 $0$，后面是这个数的二进制。
- 负数：
  - 原码：最高位为 $1$，后面是这个数绝对值的二进制。
  - 反码：除了最高位外，每一位都是原码取反。
  - 补码：在反码基础上加 $1$。
## 输入格式

输入 $s$。

## 输出格式

输出对应的补码。

```input1
011100001111
```

```output1
011100001111
```

```input2
1111000011111
```

```output2
1000111100001
```

```input3
1000
```

```output3
0000
```


```input4
1100
```

```output4
1100
```

```input5
1010
```

```output5
1110
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\le n \le 100$。
