## 问题描述

给定两个数 $x,y(0\leq x,y \leq 10^9)$，有以下两种操作：

1. 将其中一个数 $+1$ 或 $-1$，代价为 $a$
2. 将两个数同时 $+1$ 或 $-1$，代价为 $b$

求将$x,y$变为$0$的最小代价。

## 输入格式

四个数：$x,y,a,b$。

## 输出格式

一个数，将 $x,y$ 变为 $0$ 的最小代价。

```input1
1 3 391 555
```

```output1
1337
```

```input2
11 47 8 101
```

```output2
464
```

## 数据范围

对于 $30\%$ 的数据满足 $1\leq x,y\leq100$；

对于$100\%$的数据满足$1\leq x,y,a,b\leq10^9$。
