## 问题描述
写一个程序，输入一个形如 $N/D$ 的分数（$N$ 是分子，$D$ 是分母），输出它的小数形式。如果小数有循环节的话，把循环节放在一对括号中。

例如：
$1/3=0.33333…$ 写成 $0.(3)$；

$41/333=0.123123123…$ 写成 $0.(123)$。

用 $*.0$ 表示整数，如 $2/2=1.0$

## 输入格式

单独的一行包括被空格隔开的 $N$ 和 $D$。（$1≤N,D≤100000$）

## 输出格式

输出一行，即分数 $N/D$ 的小数形式。

```input1
45 56
```

```output1
0.803(571428)
```

```input2
3 8
```

```output2
0.375
```

```input3
8 2
```

```output3
4.0
```
