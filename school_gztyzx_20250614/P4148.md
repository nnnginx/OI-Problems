## 问题描述

Lily自拍了一张照片，但她并不满意，所以她决定对照片进行美颜。美颜照片的基本操作是对照片进行高斯模糊，因为手头没有比较好用的P图软件，精通算法的Lily决定自己写程序实现模糊效果。她的算法是这样的：

一张 $n\times m$ 的像素的图片，每一个像素有一个颜色值。转化成 $n\times m$ 位图，我们设每个像素的颜色值为以他为中心的 $3\times 3$ 范围内的所有像素的颜色值相加并除以 $3\times 3$ 范围内的像素个数 （注意可能少于九个）再向下取整。

因为Lily忙于自拍没空实现这个算法，请你帮帮忙。现在给出 $n$ 和 $m$ 和一张 $n\times m$ 的图的每个像素点的颜色值，请输出转化后的位图的像素值。

## 输入格式

输入的第一行包含两个整数 $n, m$ 。

第 $2$ 行到第 $n + 1$ 行每行 $m$ 个整数，表示每个像素的颜色值。

## 输出格式

输出 $n$ 行，每行 $m$ 个整数，表示转化完后的位图。

```input1
3 4
0 0 0 255
0 0 255 0
0 30 255 255
```

```output1
0 42 85 127
5 60 116 170
7 90 132 191
```

## 数据规范

$1\leqslant n，m\leqslant 10^3$

保证所有颜色值不大于 $10^4$
