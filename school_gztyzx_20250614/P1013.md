## Description

水是生命之源”，某市供水公司为了鼓励企业与居民节约用水，减少排污，对水费实行阶梯收费，即：

不超过 $40$ 吨的部分：$1.8$ 元每吨

超过 $40$ 吨的部分：$2.2$ 每吨

另：每吨用水加收 $0.2$ 元的城市污水处理费

请你编一个计算水费的程序，保留小数点后二位。

## 输入格式

输入水的重量 $x (x \leq10^5)$

## 输出格式

把水费输出，保留小数点后二位。

```input1
35.5
```

```output1
71.00
```

```input2
41
```

```output2
82.40
```
