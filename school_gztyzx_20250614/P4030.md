## 问题描述

请构造一个 $n \times n$ 的由大写字母组成的字符矩阵，使其中横着找找到 QWQ 与竖着找找到 QWQ 的次数和最大。

## 输入格式

**本题多测。**

第一行一个整数 $T$。

接下来 $T$ 行，每行一个整数 $n$。

## 输出格式

对于每一个 $n$，输出你的构造方案。

如果有多个满足要求的构造方案，任意输出一个即可。

```input1
2
3
5
```

```output1
QWQ
WAW
QWQ
QWQWQ
WQWQW
QWQWQ
WQWQW
QWQWQ
```

## 样例解释

第一个矩阵横着找能找到 $2$ 个 `QWQ`，竖着找能找到 $2$ 个 `QWQ`。

第二个矩阵横着找能找到 $8$ 个 `QWQ`，竖着找能找到 $8$ 个 `QWQ`。



## 数据范围

对于 $100\%$ 的数据，$1 \le T \le 100$，$1 \le n \le 100$。

## 评测说明

**本题使用 SPJ。**
`samples\QWQ ` 目录下会提供 `checker.exe`，以方便选手检查答案正确性。

选手可以使用 `cmd` 输入 `(checker.exe 的绝对路径) (输入文件路径) (输出文件路径) (答案文件路径)`这条命令来检查答案正确性。

如果 `cmd` 输出了一个绿色的 `ok`，就代表你的答案正确。

如果 `cmd` 输出了一个红色的 `Wrong Answer`，就代表你的答案错误。

`Martix size is not enough.` 表示你的字符矩阵大小不够。

`Your martix is wrong` 表示你的矩阵构造错了。

否则，就是一些其他问题。
