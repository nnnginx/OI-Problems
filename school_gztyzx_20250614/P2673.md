## 问题描述

派大星想吃海绵宝宝做的蟹黄堡，海绵宝宝向派大星提出了一道问题，只有派大星回答正确才能吃到蟹黄堡，派大星希望得到大家的帮助吃到蟹黄堡，现在问题如下：

海绵宝宝给派大星一个字符串 $s$ 和一个整数 $k$，现在海绵宝宝从字符串中任意选出 $k$ 个字符删去，然后将新的字符串再交给派大星，请问派大星是否一定能确定删去的所有字符在原串中的位置。如果一定可以，输出 `Certain`，否则输出 `Uncertain`。

其中，$1 \leq k \leq 100$，$1 \leq 字符串s长度 \leq 100$，$k \leq n$，$s$ 仅包含小写字母。

## 输入格式

第一行一个整数 $T$，表示接下来有 $T$ 组测试数据，$1 \leq T \leq 10$。

第 $2n$ ($n = 1， 2  ...  T$) 行一个字符串表示 $s$ ，$1 \leq 字符串s长度 \leq 100$，$s$ 仅包含小写字母。 

第 $2n + 1$($n = 1， 2  ...  T$) 行一个正整数表示 $K$ ，$1 \leq k \leq 100$。

## 输出格式

输出 $T$ 行，一行一个字符串。

如果一定可以，输出 `Certain`，否则输出 `Uncertain`。

```input1
1
abcda
2
```

```input2
1
abcacd
2
```

```input3
1
wsaox
2
```

```output1
Certain
```

```output2
Uncertain
```

```output3
Certain
```

## 样例解释

样例2，从 `abcacd` 中删掉 $2$ 个字符，无法还原字符位置，例如删去 `ca` 或 `ac`，剩余字符串都是 `abcd`

## 数据范围

对于 $50\%$ 的数据，$s长度 \leq 10$

对于 $75\%$ 的数据，$s长度 \leq 50$

对于 $100\%$ 的数据，$s长度 \leq 100$，$1\leq T \leq 10$
