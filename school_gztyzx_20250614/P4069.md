## 问题描述

**本题为弱化版，良心的出题人专门为你们准备的。**

广州现在是冬天，这意味着要下雪(?)！从宿舍到机房的道路上有N块瓷砖，编号为 $1…N$ ，瓷砖 $i$ 有深 $f_i$ 的雪覆盖。在他宿舍的宝箱里，小龙有一双 $B$ 号靴子，编号为 $1…B$ 。有些靴子比其他靴子更耐用，有些靴子比其他靴子更灵活。具体地说，第 $i$ 双靴子能够让小龙在至多 $s_i$ 深的积雪中行走，能够让小龙每步至多前进 $d_i$个瓷砖 。

小龙从瓷砖 $1$ 开始，必须到达瓷砖 $N$ 才能颓废。瓷砖 $1$ 被宿舍屋顶遮挡，瓷砖 $N$ 被实验楼屋顶遮挡，因此这两块瓷砖都没有雪。帮助小龙确定哪双雪地靴能顺利到达机房。

## 输入格式

第一行包含两个空格分隔的整数 $N$ 和 $B$ 。

第二行包含 $N$ 个空格分隔的整数；第 $i$ 个为 $f_i$，即 $i$ 号瓷砖的积雪深度。输入保证 $f_1=f_N=0$ 。

下面 $B$ 行，每行包含两个空格分隔的整数。第 $i+2$ 行的第一个数为 $s_i$ ，表示第 $i$ 双靴子能够承受的最大积雪深度。第 $i+2$ 行的第二个数为 $d_i$ ，表示第 $i$ 双靴子的最大步长。

## 输出格式

输出包含 $N$ 行。第 $i$ 行包含一个字符串：如果小龙能够穿着第 $i$ 双靴子从 $1$ 号地砖走到 $N$ 号地砖，输出 `true` ，否则输出 `false` 。

```input1
7 8
0 2 0 6 8 5 0
0 1
0 2
0 4
0 5
4 5
5 3
7 1
7 4
```

```output1
false
false
true
true
true
true
false
true
```

## 样例解释

对于 `0 2` 的数据： 小龙无法从宿舍踏出一步，因为他能走到的瓷砖都积了多于 $0$ 的雪。

对于 `4 5` 的数据： 小龙从宿舍可以走到第 $5$ 个瓷砖，再走到机房。

## 数据范围

对于 30% 的数据 $1 \le N \le 10^4$

对于 100% 的数据 $1 \le N,B \le 10^5$ , $1 \le f_i,s_i \le 10^9$,$1 \le d_i \le N$
