## 问题描述

小V有许多弹珠和一个弹珠盒，他会放弹珠到弹珠盒，也会拿出来。这时候他要将 $n$ 个弹珠放入或拿出。

## 输入格式

输入有多行，第一行一个数 $t$，表示有 $t$ 组测试数据。 对于每个测试数据，第一行一个数 $n$，表示进行了 $n$ 次操作。 之后一行 $n$ 个数，若为 $-1$，表示取出最顶上的弹珠，否则表示放入编号为 $k$ 的弹珠。

## 输出格式

对于每个测试数据，输出许多行，表示取出的弹珠编号。

```input1
1
3
3 -1 2
```

```output1
3
```

```input2
1
4
4 2 -1 -1
```

```output2
2
4
```
