## 问题描述

狐狸尼克深知光靠以前那点伎俩在江湖上已混不下去了，为了能在江湖中有一席之地，于是开始研究高科技了。

狐狸尼克最近在上化学课，他需要使用到N种化学物质来进行他的实验。在做实验的时候，他需要将所有化学物质放在桌面上，按次序排成-条直线。

然而每一种化学物质都是危险品，对于第i个化学物质，如果有其它的化学物质与它的距离小于 $ A_i $ 那么就会发生爆炸。

狐狸尼克想知道如果要安全的完成他的实验，桌子最短可以多短。


## 输入格式

第 $1$ 行一个整数 $ N $ ，表示化学物质的个数; 

第 $2$ 行有 $ N $ 个整数，第 $ i $ 个整数 $ A_i $ 表示第 $ i $ 个化学物质必须与其它化学物质保持的距离。

## 输出格式

输出共一行一个整数，表示能够让狐狸尼克安全完成实验的桌子最小长度。

```input1
3
3 1 2
```

```output1
5
```

## 样例说明

$ N $ 个化学物质必须按照题且给出的顺序以一条直线的形式依次摆放在桌面上。

## 数据规模

对于 $30\%$ 的数据: $ 2≤N≤100 $ ;

对于 $60\%$ 的数据: $ 2≤N≤1,000 $ ;

对于 $100\%$ 的数据:  $ 2≤N \leq 1,000,000; 1≤A_i≤100,000 $ ;
