## 问题描述

给定一个序列 $a$ 包含 $n$ 个整数，求在其中取两个数并计算它们的乘积，使得乘积最大或最小。不能选下标相同的两个数。

## 输入格式

第一行一个整数 $n$

第二行 $n$ 个整数，表示序列 $a$

## 输出格式

输出两个整数，表示序列取数的最大值和最小值。

```input1
5
2 3 4 5 6
```

```output1
30 6
```

## 数据范围

对于前 $20 \%$ 的数据，$n\le 10$

对于前 $40 \%$ 的数据，$n\le 1000,a_i\le 1e4$

对于前 $70 \%$ 的数据，$n\le 100000$

对于 $100 \%$ 的数据，$2\le n\le 7000000,\;1\le a_i\le 1e9$

因为数据范围较大，这里提供快读代码：

```CPP
int read()
{
	int x=0,y=1;
	char c=getchar();
	for(;!isdigit(c);c=getchar()) if(c=='-') y=-1;
	for(;isdigit(c);c=getchar()) x=x*10+c-'0';
	return x*y;
}
```

使用方法：`int x=read();` 诸如之类
