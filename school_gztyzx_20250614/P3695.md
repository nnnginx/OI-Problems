## 问题描述

HIM 在参加完生日会后的头一天晚上，就接收到了oi总部发来的命令（其实HIM是oi总部 S 级特工，没想到吧~ ），要求 HIM 马上敲出一个程序，用来提前预知地球状况，解决3478年的三体入侵问题（梦幻联动，双厨狂喜）。

3470年，Mrs. Ye向三体人发送了地球坐标(万恶之源)，于是三体人联合地球上的某些组织，高呼着“消灭人类暴政，世界属三体！”的口号，启动了“向地转移计划”。

整个计划包括四只“人畜无害”的“水滴”，而它们会来到地球，挡住太阳光，所以在“水滴”靠近和远离地球时，会出现下面几种情况：

一、当所有水滴远离地球时，地球处于“温暖状态”；

二、当只有一只水滴靠近地球时，地球处于“寒冷状态”；

三、当有两只水滴靠近地球时，地球处于“极冻状态”；

四、当有三只水滴靠近地球时，地球处于“冰封状态”；

五、当所有水滴靠近地球时，地球处于“绝对零度状态”；

而对于以下四种情况，地球文明将灭绝：

一、寒冷状态持续$7$年；

二、极冻状态持续$5$年；

三、冰封状态持续$3$年；

四、绝对零度状态持续$1$年；

同时，良心的三体人（是三体人，OK ? 不是出题者我 ）为了使题目难度不至于太高或太低，制定了以下“黑暗森林法则”：

①地球不会移动；

②“水滴”只有“靠近行星 ―― $1$ ”和“远离行星 ―― $0$ ”两种状态；

③因为没有永动机，所以第 $i$ 只“”每保持现有状态 $s_i$ 年就会回到快乐老家，即从“靠近行星”变为“远离行星”或从“远离行星”变为“靠近行星”，去补充燃料；

④地球文明毁灭后会立刻重建下一个文明（神奇的功能又增加了？），即文明编号从第 $m$ 号变为第 $(m+1)$ 号；

【第一年时文明编号为$1$】

虽然 HIM 在数论方面是位F佬，但他在其他地方却是个Xm，所以他找到了你。

现在 HIM 给你第一年时各只“水滴”的状态和 $s_i$，想让你帮忙求出多少年后文明编号会变成 $K$ ，以便更好地布置战术，联合民众（全世界的无产者，联合起来！――《觉醒年代》），组织军队，对抗三体！为了地球，加油吧！！！

## 输入格式

输入数据共三行：

第一行共 $4$ 个数 $a_i$，表示第 $i$ 只“水滴”最开始时的状态（$0$或$1$）；

第二行共 $4$ 个整数 $s_i$，含义如题所示；

第三行一个整数 $K$，含义如题所示。

保证有解！

## 输出格式

输出数据就一行

输出多少年后文明编号会变成 $K$。

```input1
1 1 0 1
1 2 3 4
2
```

```output1
11
```

## 样例解释

打个表：

```
年份 状态 星一 星二 星三 星四      
 1   冰封   1   1   0   1     
 2   极冻   0   1   0   1       
 3   极冻   1   0   0   1     
 4   极冻   0   0   1   1     
 5   冰封   1   1   1   0     
 6   极冻   0   1   1   0     
 7   寒冷   1   0   0   0     
 8   温暖   0   0   0   0     
 9   冰封   1   1   0   1     
10   冰封   0   1   1   1     
11   冰封   1   0   1   1
```

从第 $9$ 年开始到第 $11$ 年，连续 $3$ 年持续冰封状态，所以文明 $1$ 灭亡，从而开始文明 $2$ 。所以输出 $11$ 。

（是不是很简单 ！）

## 数据说明

对于 $100\%$ 的数据，$1 \leq s_i \leq 10000 , 2 \leq k \leq 1000$
