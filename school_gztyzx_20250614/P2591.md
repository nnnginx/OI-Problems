## 问题描述

快期末考试了，英语老师为了检查学生背单词的情况，在课室前画了一个 $H×W$ 的网格，每个格子上面都有一个字母。每个学生要进入课室之前都必须通过这个网格。起点和终点可以任选，每次学生可以选择当前区域的上方或右方的任意一个格子跳过去（距离有多远都没关系，假设跳远的能力足够强）。只有途中经过的格子中的字母连起来是一个单词，这位学生才能被允许进入课室。学生是排队通过这个网格的，排在后面的学生不能走排在前面的学生走过的路径。只要走过的路径不同，即使走过的路径上面的字母连起来恰巧是同一个单词也是允许的。

例如，现在有个 $3×4$ 的网格，假设 `TO` 和 `OX` 是单词：

```
TXXO
TXQT
XTXQ
```

有四条可行的跳跃路线，都产生了 `TO`：

1. 以 $(1,1)$ 为起点，$(1,4)$ 为终点；
2. 以 $(2,1)$ 为起点，$(1,4)$ 为终点；
3. 以 $(2,4)$ 为起点，$(1,4)$ 为终点；
4. 以 $(3,2)$ 为起点，$(1,4)$ 为终点。

`OX` 虽然是一个合法的单词，但跳出 `OX` 要以 $(1,4)$ 为起点然后往左跳DD这是不允许的。

关于哪些词是单词，我们会给出一个英语词典供你参考。

但学生对此很不满，他们觉得这个太难了，以至于排在后面的学生根本找不到合法的跳跃路线，进不了课室了。老师说这是他们没有用心去找。他请你帮他写一个程序，证明他画的网格是有很多合法跳跃路线的。

## 输入格式

第一行两个空格分隔的整数，$H、W$，表示网格有 $H$ 行 $W$ 列。接下来 $H$ 行，每行 $W$ 个大写字符（中间没有任何空格），描述这个网格。

接下来的若干行是英语单词词典。每行是一个长度至多为 $4$ 的非空字符串，表示一个单词。单词只包含大写字母。

## 输出格式

仅一行，上面一个整数，表示有几条不同的合法跳跃路线。

```input1
3 4
TXXO
TXQT
XTXQ
TO
OX
```

```output1
4
```

## 数据范围

对于 $40\%$ 的数据，$1\leq  H,W \leq 15$；

对于 $100\%$ 的数据，$1\leq H,W \leq 30$。

对于 $100\%$ 的数据，英语词典中至多有 $2500$ 个单词。
