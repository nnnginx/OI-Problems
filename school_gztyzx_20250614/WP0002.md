# Background

**宪法的重要地位**

①我国宪法是党和人民意志的集中体现，是公民权利的保障书、治国安邦的总章程。

②宪法是一切组织和个人根本的活动准则。

③宪法是国家的根本法，具有最高的法律地位、法律权威和法律效力。

④所以，坚持依法治国首先要坚持依宪治国；依法执政首先要坚持依宪执政。

# Description

有一个无限长的序列 $A$。

定义 $A_i=\begin{cases}n&i=1\\ \sum\limits_{j=1}^{i-1} A_j +\gcd(i,c)&i>1\end{cases}$

在这个序列中任取一些数（必须要取一个数）求和加入集合 $S$ （集合具有去重性）中，求在 $S$ 中第 $k$ 小的数是多少。

答案 $\bmod \ 10^9+7$。

多测。

# Format

## Input

第一行一个整数 $t$ 表示有多少组数据。

接下来 $t$ 行，每行三个整数 $n,c,k$。

## Output

输出 $t$ 行表示答案。

# Samples

```input1
5
2 3 8
20 63 5
32 6 5
90 6 6
21 10 2
```

```output1
14
64
101
277
23
```

# Limitation

样例解释：

n=2,c=3,k=8;

A数组：2,3,8,14……

S:2,3,5,8,10,11,13,14,16,17……

第8小为14

对于 $30\%$ 的数据，$t\leq 10$，$1\leq k\leq 20$，$1\leq n,c\leq 10^{9}$。

对于 $100\%$ 的数据，$t\leq 10^5$，$1\leq n,c,k\leq 10^{18}$。

$\gcd(i,j)$ 表示 $i$ 和 $j$ 的最大公约数，指的是 $i$ 和 $j$ 中共有约数中最大的一个。
