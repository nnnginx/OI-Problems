## 问题描述

植物大战僵尸这款游戏中，有很多种植物武器供玩家选择，每种植物有固定的攻击力，玩家可以直接把植物摆放在草坪上打僵尸，为了简化问题，我们假设草坪现在只有 $1$ 行 $n$ 列，现在给你 $n$ 棵植物武器，第 $i$ 棵植物的攻击力为 $a_i$，现在你把这 $n$ 棵植物摆放到草坪的 $1$ 到 $n$ 的位置中，每棵植物的位置由你决定，请问有没有可以通过摆放，使得对于一切 $0 \lt i \leq n C 2$，第 $i$ 个位置上植物的攻击力 $b_i$  满足条件（$(b_i \lt b_{i + 1}）and（b_{i + 1} \gt  b_{i + 2}））or（（b_i \gt b_{i + 1}）and（b_{i + 1} \lt  b_{i + 2}）$）。

## 输入格式

第一行是一个整数 $d$，表示有 $d$ 组数

对于每一个组数，首先第一行一个正整数 $n$，接下来 $n$ 行，每行一个整数表示 $a_i$ 。

## 输出格式

对于每一组数，如果可以通过摆放而满足题目的条件则输出 `Yes`，否则输出 `No`

```input1
2
5
3
3
3
5
2
6
3
3
1
5
2
2
```

```output1
No
Yes
```

## 数据范围

对于 $30\%$ 的数据：$d\leq 10, 3\leq n\leq 100$

对于 $100\%$ 的数据：$d \leq 100，3 \leq n \leq 10000，1 \leq a \leq 10^9$
