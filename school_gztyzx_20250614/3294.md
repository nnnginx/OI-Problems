## 题目背景

小大在其他人的带领下，来到了这里最大的城市，小大惊奇的发现这个城市的村落分布很像他之前学过的一种数据结构――树。

小大从别的居民那里了解到关于这些村落排布的原因。本来这些村落之间每个都与其他大部分村落直接相连，来往比较方便。但有一天，市长学到了一种算法：最小生成树。他按照在这个算法的思想将所有的道路删到只剩村落树减一条。很明显这有一个缺点：中间的道路压力很大。

现在又有一名经济学家找到了你，并向你提出了以下问题：

## 题目描述

给出一颗有 $n$ 个节点，$n-1$ 条边的树。树根是 $1$，最初所有点点权为 $0$。现在有下面两个操作：

- `1 c v`：将编号为 $c$ 的节点及其子树的点权值增加 $v$。
- `2 c`：求出编号为 $c$ 的节点及其子树中的点权最大值和最小值的差。

对于每个操作`2`，输出结果。

## 输入格式

第一行两个数 $n$，$q$，表示树上有 $n$ 个结点，共 $q$ 次操作。

接下来一行 $n-1$ 个数，第 $i$ 个数表示在树上，编号为 $i+1$ 的节点的父亲的节点编号。

接下来 $q$ 行，每行描述一个操作，格式见题面。

## 输出格式

共若干行，第 $i$ 行一个数表示第 $i$ 次操作`2`的结果。

## 样例 #1

### 样例输入 #1

```
6 5
1 2 2 3 3
1 3 4
1 1 2
2 2
1 5 3
2 3
```

### 样例输出 #1

```
4
3
```

## 提示

**【样例解释】**

```
1                如图，
     	    |             对于第一次赋值，点权为[0,0,4,0,4,4]
     	  	2               对于第二次赋值，点权为[2,2,6,2,6,6]
            /\            第一次询问，最大值为6，最小值为2，差为4
           /  \            对于第三次赋值，点权为[2,2,6,2,9,6]
          4    3           第二次询问，最大值为9，最小值为6，差为3
              /\
             /  \
            5    6
```

**【数据范围】**

设 $s_i$ 为节点 $i$ 的儿子节点的数量。  

| 数据点  |             $n$             |            $q$             |           特殊性质           |
| :-----: | :-------------------------: | :------------------------: | :--------------------------: |
|  $1-3$  |     $1 \le n \le 10^3$      |     $1 \le q \le 10^4$     |         保证$1+1=2$          |
|   $4$   |     $1 \le n \le 10^5$      | $1 \le q \le 2\times 10^5$ | 保证操作1的数量不大于 $100$  |
|  $5-6$  |     $1 \le n \le 10^5$      | $1 \le q \le 2\times 10^5$ | 保证操作2的数量不大于 $100$  |
|   $7$   | $1 \le n \le 5\times 10^5$  | $1 \le q \le 5\times 10^5$ | 保证 $s_1=n-1$，其余 $s_i=0$ |
|  $8-9$  |     $1 \le n \le 10^4$      | $1 \le q \le 5\times 10^5$ |      保证整个图为一条链      |
| $10-11$ | $1 \le n \le 2\times 10^5$  | $1 \le q \le 2\times 10^5$ |  保证 $s_i(i\neq 1)\leq 1$   |
| $12-13$ | $1 \le n \le 2\times 10^5$  | $1 \le q \le 5\times 10^5$ |       保证 $s_i\leq 2$       |
| $14-20$ | $1 \le n \le 5 \times 10^5$ | $1 \le q \le 5\times 10^5$ |   防止卡常，时间限制$1.2s$   |

对于 $100\%$ 的数据：$1 \le n \le 5 \times 10^5,1 \le q \le 5\times 10^5,1 \le c \le n,0 \le v \le 10^9$
