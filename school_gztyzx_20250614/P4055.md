## 问题描述

给出一个 $b$ 进制正整数 $n$（进制中 $10$ 用大写字母 $A$ 表示，$11$ 用 $B$ 表示，以此类推），把每一个数字拆下来选择若干个，再拼接到一起，求能够组成多少个不同的数（不允许有前导 $0$）（对$998244353$ 取模）。

## 输入格式

一行两个正整数，$n,b$。

## 输出格式

一行一个非负整数，表示答案。

```input1
101 2
```

```output1
6
```

## 样例1解释

共有 $6$ 个不同的数：$0,1,10,11,101,110$

```input2
114514IAKIOI1919810 36
```

```output2
644490570
```

## 数据范围

设 $n$ 的位数为 $k$，则

对于 $10\%$ 的数据，$k≤5$

对于 $20\%$ 的数据，$k≤10$

对于 $50\%$ 的数据，$k≤17$

对于另外 $30\%$ 的数据，$n$ 中没有数字 $0$

对于 $100\%$ 的数据，$2≤b≤36,1≤k≤1000$

【提示】

如果要求 $a/b\%p$（$p$是质数），可以求 $a*b^{(p-2)}\%p$
