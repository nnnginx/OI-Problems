## 问题描述

多多刚做完了“100以内数的加减法”这部分的作业，请你帮他检查一下多多算得对不对。每道题目（包括答案）的格式为 $a + b = c$ 或者 $a C b = c$，其中 $a$ 和 $b$ 是作业中给出的，均为不超过 $100$ 的非负整数；$c$ 是多多算出的答案，是不超过 $200$ 的非负整数；且每个符号间有一个空格，即 `+` 的前后各有一个空格，`=` 前后各有一个空格。

## 输入格式

第一行一个整数 $N（N\leq 100）$，表示共有 $N$ 道算式。

接下来 $N$ 行，每行一条算式，格式保证符合上述规定，输入的所有整数均不含前导 $0$。

## 输出格式

输出多多算对了多少题。

```input1
3
1 + 2 = 3
3 + 4 = 5
90 C 10 = 80
```

```output1
2
```
