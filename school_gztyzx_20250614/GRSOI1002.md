## 题目背景

饱餐一顿后，xiaosi4081 在回宿舍的路上玩一款叫做《NOIP》的游戏。

## 题目描述

你说的对，但是《NOIP》是一款游戏。玩家需要在《NOIP》游戏中的比赛取得尽可能高的分数。

你的程序可以浓缩成一个数 $c$。会有 $n$ 个毒瘤数据点，每个数据点都会执行一个操作，有三种形如 $mode_i ~x_i$ 的操作：

$1~x$：此时相当于把 $c$ 按位与上 $x$ 。

$2~x$：此时相当于把 $c$ 按位或上 $x$ 。

$3~x$：此时相当于把 $c$ 按位异或上 $x$ 。

程序 $c$ 获得的分数为进行完上面的所有操作后 $c$ 的值。你需要决定你的程序浓缩成的数 $c$ 使得其经过 $n$ 个数据点后获得的分数最高。但是由于各种题目限制，你必须要保证 $c \leq m$ 。

## 输入格式

第 $1$ 行有两个数，$n$ 和 $m$。

接下来 $n$ 行，每行两个数，表示第 $i$ 个操作模式 $mode_i$ 和操作数 $x_i$。

## 输出格式

一个数，表示你的程序浓缩成的数 $c$ 的分数。

## 样例 #1

### 样例输入 #1

```
3 1
1 1
3 1
2 1
```

### 样例输出 #1

```
1
```

## 提示

$1 \leq n \leq 2\times 10^5,1 \leq m,x_i \leq 10^{18}$

对于 $5\%$ 的数据，乱搞可以通过。
