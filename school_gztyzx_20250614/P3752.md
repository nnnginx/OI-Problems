## 问题描述

给一个 $n$ 个点的图，第 $i$ 个点上有一个区间 $[l_i,r_i]$。两个点之间有边当且仅当它们区间的交集非空。 求有多少个点集的非空子集，满足这些点以及它们之间的边构成了一棵树。

## 输入格式

第一行一个整数 $n$。

接下来 $n$ 行，每行两个整数 $l_i,r_i$。

## 输出格式

一行一个整数表示答案模 $10^9 + 7$ 的结果。

```input1
3
1 3
2 3
3 4
```

```output1
6
```

## 数据范围

对于 $20\%$ 的数据，有 $n\le 18$。

对于 $40\%$ 的数据，有 $n\le 50$。

对于 $60\%$ 的数据，有 $n\le 200$。

对于 $100\%$ 的数据，有 $1\le n\le 2000,1\le l_i < r_i\le 4000$。
