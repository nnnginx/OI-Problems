## 问题描述

所谓亲和数就是：

定义数对 $(x,y)$ 为亲和数对当且仅仅当 $x、y$ 为不同正整数，且 $x、y$ 各自的所有非自身正因子之和等于另一个数。例如 $(220,284)$ 和 $(280,224)$ 都是亲和数对，因为：

$220$ 的所有非自身正因子之和为：$1 + 2 + 4 + 5 + 10 + 11 + 20 + 22 + 44 + 55 + 110 = 284$

$284$ 的所有非自身正因子之和为：$1 + 2 + 4 + 71 + 142 = 220$

数对 $(x,y)$ 跟 $(y,x)$ 被认为是同一数对，所以我们只考虑 $x\lt y$ 的情况。

任务：tenshi对某个范围内的亲和数对的数量非常感兴趣，所以希望你能帮她编写一个程序计算给定范围内的亲和数对的数量。给定一个范围 $A$ 到 $B$，如果 $A≤ x ≤ B$，则我们称 $(x,y)$ 在范围 $[A,B]$ 内。

## 输入格式

从文件的第一行分别读入正整数 $A$ 和 $B$，其中 $A、B$ 满足
$1 ≤ A ≤ B ≤ 10^8$; 且 $B-A ≤ 10^5$

## 输出格式

输出文件只有一行，就是 $[A,B]$ 内亲和数对的数量

```input1
200 1200
```

```output1
2
```

## 样例解释

$[200,1200]$ 内的数对只有两个，分别是 $(220,284)$ 和 $(1184,1210)$
