## 问题描述

现在知道CPP有 $n$ 把锄头，每一把锄头都有一个耐久值 $a_i$。由于CPP准备使用1个锄头去耕面积为 $k$ 平方米的地，每耕1平方米的地需要使用1耐久。由于他不想太浪费，所以请你来帮他看看，他应该用哪个锄头最好。

## 输入格式

输入共 $n+1$ 行

第一行，两个整数 $n$ 和 $k$。

接下来 $n$ 行，每行1个整数 $a_i$，表示第 $i$ 个锄头的耐久值。

## 输出格式

输出仅一行，一个整数 $q$，表示第 $q$ 个锄头是最好的。如果有多个最符合条件并且耐久值一样的锄头，输出最前的那个。如果没有符合题目条件的锄头，输出 `NoHaveGood`。

```input1
4 3
4
2
3
3
```

```output1
3
```

## 样例1解释

CPP需要耕 $3$ 平方米的地，所以选择 $3$ 平方米的是最好的(刚好用完)并且第三个和第四个锄头是一样的，所以选择第三个锄头。

```input2
4 5
4
2
3
3
```

```output2
NoHaveGood
```

## 样例2解释

CPP需要耕 $5$ 平方米的地，没有任何一个符合条件的锄头，所以输出 `NoHaveGood`

## 数据提示

对于 $60\%$ 的数据， $n \leqslant 33$。

对于 $100\%$ 的数据， $n \leqslant k \leqslant 10^8$，$a_i \leqslant 10^9$。
