# Math

## 题目描述

你是一名数学课代表

由于老师讲的题你都会，而且你认为这些题太简单了，没有听的必要。于是，你觉得在数学课上自己研究一下别的东西。

但是，最近老师抓的很严，即使你的侦查意识很强，但仍然逃不过被老师收走书本的悲剧。因此，你决定在老师转头讲课时再研究。但是这样可能会导致你的研究效率降低。因此，你向老师提出了建议，在数学课上讲你想研究的东西。

为了保证班级的数学成绩，老师只能答应。但是毕竟老师会照顾全班同学，你认为有些东西还是太简单了。于是，你还是会在老师转头讲课时研究一些知识点。

你现在有 $n$ 个知识点要在这节数学课上研究完，每个知识点都能使你的智慧值加 $val_i$ ，而这个东西如果让老师讲，则会花费 $time_i$ 分钟，并且，如果让老师讲第 $i$ 个知识点，你就会损失 $f_i$ 的精力值。

当老师在讲授一个新的知识点时会一直转头讲题，在此过程中，你可以花费一些时间来研究一些新的知识点，由于你很聪明，每个知识点只需要花费 $1$ 分钟去研究。

由于你是数学课代表，因此你可以随意变动数学课的时长，并且你可以指定老师讲的题目和讲知识点的顺序。但是，你想研究完所有知识点再下课，因此，你决定在花费最小精力值的情况下，研究完所有的知识点。

求出这个最小值。

注意：老师讲的过程中，即使你没听（在自己研究），你也能“研究”完这一个知识点。

## 输入格式

第一行，一个整数 $n$ ，表示总共的知识点个数。

接下来 $n$ 行，每行两个整数 $time_i$ 和 $f_i$ ，含义如题所示。

## 输出格式

一行，一个整数 $ans$ ，表示你研究完所有知识点最小需要花费的精力值。

## 样例 #1

### 样例输入 #1

```
2
2 10
1 5
```

### 样例输出 #1

```
5
```

## 提示

对于 $10\%$ 的数据， $1 \le n \le 10$ 。

对于 $20\%$ 的数据， $1 \le n \le 50$ 。

对于 $25\%$ 的数据， $1 \le n \le 100$ 。

对于 $40\%$ 的数据， $1 \le n \le 500$ 。

对于 $90\%$ 的数据， $1 \le n \le 2000$ ， $0 \le time_i \le 2000$ ， $1 \le f_i \le 10^6$ 。

对于 $100\%$ 的数据， $1 \le n \le 2000$ ， $0 \le time_i \le 2000$ ， $1 \le f_i \le 10^8$ 。
