## 问题描述

奶牛Bessie要去参加舞蹈比赛，它只能在矩形方框内跳舞，方框由 $1*1$ 的方格组成。它有四只脚，对应为：`FL`,`FR`,`RL`,`RR`(前左脚，前右脚，后左脚，后右脚)，每只脚占用一个 $1*1$ 的方格。一开始时，Bessie头朝北，四只脚在相邻的四个 $1*1$ 的方格内。跳舞时，它接受一系列的指令，每条指令由三个字符组成，前二个字符代表哪只脚 $I$，第三个字代表这只脚的操作指令，操作指令为告诉它第 $I$ 只脚向哪个方向 $（F=forward,B=back,R=right,L=left）$ 挪动一格，或者整体以第 $I$ 只脚为轴，顺时针旋转 $90$ 度。比如： $FLF$,表示前左脚向前挪动一格，$RRR$，后右脚向右挪动一格，$FLP$，代表以前左脚为轴，顺时针旋转 $90$ 度。

现在给定 $N$ 条操作序列，求出最小要给Bessie设定多大的方框，才能满足它的任意一只脚不会挪动到方框外面来，如果在跳舞过程中，出现两只脚在同一个 $1*1$ 的方格内，则输出 `-1`。

## 输入格式

第一行为 $N$，代表操作序列的总数；

以下 $N$ 行，每行由三个字符组成，表示一条操作指令。

## 输出格式

输出最小的满足条件的方框大小。一旦有两只脚处于同一格内，则输出 `-1`。

```input1
3
FRF
FRP
RLB
```

```output1
16
```

## 样例解释

```
.. .. .. ..
.. .. .. .. (facing north)
.. .. FL FR
.. .. RL RR

After FRF:

.. .. .. ..
.. .. .. FR (facing north)
.. .. FL ..
.. .. RL RR

After FRP:

.. RL FL ..
.. RR .. FR (facing east)
.. .. .. ..
.. .. .. ..

After RLB:

RL .. FL ..
.. RR .. FR (facing east)
.. .. .. ..
.. .. .. ..
```

## 数据规模

对于 $50\%$ 的数据：$1≤N≤20$；

对于 $100\%$ 的数据：$1≤N≤1,000$；
