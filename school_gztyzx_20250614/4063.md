## 题目背景

用来计算的计算机如果用来计算就可以像如果用来计算的用来计算的计算机一样计算应该计算的计算机计算。

做不出这道题就会「数据删除」。

## 题目描述

给定 $n$ 个数 $a_{i}$，求 
$$
\frac{\sum_{k=i+1}^{j}{a_k^2}}{j^2-i^2}(0\le i < j\le n)
$$ 
的 $\max$ 与 $\min$。

## 输入格式

第一行一个整数 $n$。

第二行 $n$ 个整数 $a_i$。

## 输出格式

输出式子的 $\max$ 与 $\min$。**(保留三位小数)**

## 输入输出样例 #1

### 输入 #1

```
3 
1 2 3
```

### 输出 #1

```
1.800 1.000
```

## 说明/提示

## 保留三位小数的方法如下

```cpp
cout << fixed << setprecision(3) << ans;

printf ("%.3f",ans); // double

printf ("%.3Lf",ans); // long double
```

## 数据范围

$1\le n\le10^5$ ，$-10^5\le a_i\le10^5$.

「数据删除」
