## 题目描述

小C很喜欢成双成对的东西，一天，小C发现有一些数字可以表示成一个正整数乘以它本身，小C认为这种数字也是成双成对的。例如，$16=4\times 4$，因此16是成双成对的，除此之外，$25，36，49$也是成双成对的，而$41，15，33$不是成双成对的。

现在小C拿到了一个数$n$，小C想要把它变成成双成对的，因此小C决定删除$n$的某些数位，从而得到一个新的正整数，该正整数是成双成对的。由于小C很严谨，所以他不希望的得到的数有前导零（例如013就是有前导零）。

现在小C想知道他至少需要删除多少个数位才能得到一个成双成对的数，当然，如果$n$本来就是成双成对的，那么就不需要删除，而如果无法得到成双成对的数，则输出$-1$。

## 输入格式

一个正整数$n$。

## 输出格式

一个正整数，最少需要删除多少位才能得到成双成对的数。

```input1
8314
```

```output1
2
```

```input2
333
```

```output2
-1
```

## 样例解释

样例1：删除3和4，得到$81=9\times 9$，一共两位。

## 数据范围

对于$100 \%$的数据满足 $1 \leq n \leq 2 \times 10 ^ 9$
