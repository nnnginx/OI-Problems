## 题目描述

给定一函数 $f$ 以及 $L$、$R$，求：

$$
\int^R_L f(x)\, \textrm{d}x
$$

## 输入格式

从文件 `int.in` 中读入数据。

输入第一行一个字符串，表示 $f(x)$。

第二行两个实数 $L$ 和 $R$。

## 输出格式

输出到文件 `int.out` 中。

输出一个实数，所取得值，保留 $6$ 位小数。

```input1
x^2+1
1 14
```

```output1
927.333333
```

```input2
0.5*x+114
1 1000
```
```output2
363885.750000
```
```input3
x^(1/2)*(x^2*12-115)
1 10000
```

```output3
342857066190549.428558
```

## 提示

### 数据范围

对于 $100 \,\%$ 的数据，输入字符串长度
$\leq 2500$，$0 \leq |L|,\,|R| \leq 10^4$，保证给定的 $L,\,R \in$
函数定义域，值域 $\in \mathbf{R}$，保证函数可积，且答案在 `long double`
范围内。

规定输入的表示函数的字符串的格式如下：

- 函数为多项式函数。
- 函数中只包含自变量 $x$。
- 只使用
  $+$（`+`）、$-$（`-`）、$\times$（`*`）、$\div$（`/`）、^（   `^`）五种运算符，其中    ^
  表示乘方，运算优先级最高；乘除其次；加减最低。
- 函数中会使用小括号以规定运算顺序，乘方的指数可能不是一个整数。
- 函数中包含正常表达的常数，不省略乘号。
- 保证多项式中的指数均为整数或 $\frac{1}{2}$，且 $\leq 5$。
- 保证在给定的范围内，积分不发散。
