<h2>Description</h2>
又是一年毕业季，大山中学共有N个教授，共有N 2 个课室。第1个教授使用编号为&nbsp; 1, N + 1, 2 N + 1, …, N 2 − N + 1 的课室、第2个教授使用编号为 2,N + 2, 2 N + 2, …, N 2 − N + 2 的课室、……、第n个教授使用编号为&nbsp; N, 2 N, 3 N, …, N 2 的课室。<br />
教授们发现，有些懒学生根本没有来上过他的课。因此他们决定，学生必须来过每个教授的至少一个教室，才能给他发毕业证。<br />
有N个学生，他们我行我素，按照自己的喜好来上课。第1个学生去过所有教室，第2个学生只去过编号能被2整除的教室，第3个学生只去过编号能被3整除的教室，……，第n个学生只去过编号能被n整除的教室。<br />
最后学校统计出来发现，他们总共给K个人发放了毕业证。请你找到一个最小的N，然后这N个学生按照他们喜好上课后，收到毕业证的人数恰好为K。<br />

<h2>Input Format</h2>
一个整数K
<h2>Output Format</h2>
输出一个满足题目要求的最小的正整数N。如果不存在这样的N，输出0。
<pre><code class="language-input1">
【样例输入1】
8
【样例输入2】
3
</code></pre><pre><code class="language-output1">
【样例输出1】
15
【样例输出2】
0
</code></pre>
<h2>Hint</h2>
对于10%的数据，K≤50<br />
对于40%的数据，K≤10000<br />
对于100%的数据，K≤2∗10^9<br />
