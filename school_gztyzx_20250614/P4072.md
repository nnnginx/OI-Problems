## 问题描述

现在有一个杨辉三角。

```
\\杨辉三角是一个这样的东西：
    1
   1 1
  1 2 1
 1 3 3 1
1 4 6 4 1
 ......
```

即一个二维数组 $a$，第 `i` 行有 `i` 个数，每一个数都是上方两个数之和。

现在给三个数 $n,m,k (m \le n)$ ，求杨辉三角第 $n$ 行第 $m$ 列及其左下方连续 $k$ 个数之和 $mod\  1e9+7$ 的值。多组数据。具体看样例

## 输入格式

第一行一个数 $T$ ，表示有T组数据。

接下来 $T$ 行每行三个正整数 $n,m,k$ 表示第 $n$ 行第 $m$ 列及其左下方连续 $k$ 个数。

## 输出格式

共 $T$ 行，每行一个正整数表示答案。

```input1
1
3 2 3
```

```output1
9
```

## 样例解释

```
\\样例解释
    1
   1 1
  1 2 1
 1 3 3 1
1 4 6 4 1
```

分别是第三行第二个数 $2$ ，第四行第二个数 $3$ ，第五行第二个数 $4$ ，加起来是 $9$ 。

注意：本题最后 $10\%$ 数据轻微超纲，这里给一个逆元代码。

逆元的作用就是在 $mod\;\;p$ 意义下，若 $p$ 是质数，则除以一个数 $x$ 和乘以 $x$ 的逆元在 $mod\;\;p$ 意义下相等。

这是代码：

```CPP
long long pow_(int x,int y,int p) // 关于 mod p 的x的y次方 
{ 
	long long s=1;
	for(;y;y/=2,x=x*x%p) if(y%2) s=s*x%p;
	return s;
}
long long ny(int x,int p) // x 关于 mod p 的逆元 
{
	return pow_(x,p-2,p); 
}
```
