## 题目描述

碎镜中的火光缓缓熄灭，其中出现了无限多的各种各样的卡牌。聪明的你发现，这些神秘的卡牌上的数字都是完全平方数，如 $0，1，4，…，$并且任意完全平方数都可以找到。现在 $ASH$ 给你两个整数 $L$ 和 $R$ ，问 $L≤x≤R$ 中有多少个整数 $x$ 满足存在两张卡牌的差为 $x$ 。

## 输入格式

第一行为一个正整数 $T$，表示数据组数。

接下来 $T$ 组数据，每组数据一行包含两个正整数 $L$，$R$，用一个空格分隔。

## 输出格式

对于每组数据，输出一行包含一个整数满足给定条件的 $x$ 的数量。

## 输入数据1

```
2
1 6
1 10
```

## 输出数据1

```
4
7
```

## 输入数据2

```
2
301890482136 925797157308
162 688
```

## 输出数据2

```
467930006380
395
```

## 数据范围

对于$30％$的数据，$1≤L，R≤5000，T≤10$。

对于$60％$的数据，$1≤L，R≤10^9，T≤100$。

对于$100％$的数据，$1≤L，R≤10^{18}，T≤10000$。
