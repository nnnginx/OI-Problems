## 问题描述

学习信息学的人都熟悉 $2$ 进制，但有没有人想过
$-2$ 进制！那样的数字就不需要符号了！！

$2$ 进制从低位到高位(即从右向左)的位权是 $1、2、4、8、16、....$。

$-2$ 进制的从右向左的位权当然就是 $1、－2、4、－8、16、....$。

$-2$ 进制是可以表示任何整数的。如：

$1, 110, 111, 100, 101, 11010, 11011, 11000, 11001,...$

表示 $1，2，3，4，5，6，7，8，9,....$

而 $11, 10, 1101, 1100,1111,...$ 则表示 $-1,-2,-3,-4,-5,....$

现在给你一个十进制的整数 $n$，请求出它的 $-2$ 进制数。

## 输入格式

只一行，一个十进制整数 $n$。

## 输出 格式

一个 $-2$ 进制数。如果数字不为 $0$，不能有前导 $0$。

```input1
-13
```

```output1
110111
```

## 样例说明

从右向左：

$1*1 + 1*-2 + 1*4 + 0*-8 +1*16 + 1*-32 = -13$

## 数据范围

$-2,000,000,000\leq n\leq 2,000,000,000$
