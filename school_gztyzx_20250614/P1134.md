## 问题描述

有 $1$ 元、$5$ 元、$10$ 元、$50$ 元、$100$ 元、$500$ 元的硬币各 $c1、c5、c10、c50、c100、c500$ 枚。现在要用这些硬币来支付 $A$ 元，最少需要多少枚硬币？假定本题至少存在一种支付方案。

## 输入格式

一行 $c1、c5、c10、c50、c100、c500、A$，中间用空格隔开。

## 输出格式

最少的硬币数量。

```input1
3 2 1 3 0 2 620
```

```output1
6
```

## 样例解释

500元硬币1枚，50元硬币2枚，10元硬币1枚，5元硬币2枚，合计6枚。

## 数据范围

$0\leq  c1、c5、c10、c50、c100、c500\leq 10^9$

$0\leq A\leq 10^9$
