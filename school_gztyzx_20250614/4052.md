# 寒山秋

## 题目背景

山寒，云起。

天暗，叶红。

路是人走的，因此白云缭绕，却因烟火人家失去了它的虚无缥缈。

满山如火如荼，秋叶胜春花，生命蠢蠢欲动。天色暗了，但他停下欣赏生命的纯美，而非远处只能看见的人家。

寒山不寒。

> 远上寒山石径斜，白云生处有人家。
>
> 停车坐爱枫林晚，霜叶红于二月花。

每一个字都平和醇美，少了曾经怀才不遇的愤懑，和借古讽今的犀利。最简单的笔触，却使寒山秋色如此动人。

杜甫有沉郁顿挫，李白却有海涛天风；李商隐有巴山夜雨，他却有枫林叶红。

他与命运和解，怡然且从容。受冷落，又何必悲秋？一生不被重用又如何，改变能改变的，接受无法改变的，就已足够自在洒脱。

$“\textrm{Lord,\ grant\ me\ the\ strength\ to\ change\ the\ things\ I\ can,}$

$\textrm{the\ courage\ to\ accept\ the\ things\ I\ can't,}$

$\textrm{and\ the\ wisdom\ to\ know\ the\ difference.}"$

## 题目描述

为什么要放这个题目背景，因为这是唐氏（诗）题目！

[$\color{black}\textup{\textmd{\textrm{xyy}}}$](https://oiclass.com/d/puji/p/1205/solution/64d48bf059462429d9282f78) 开始玩弄数组，并且给一个有 $n$ 个元素的数组起名为 $a$！

但是它太菜了，所以它的数组的每个 $a_i=i$。

因为它刚刚了解了约数，所以又用了一个数组 $x$，定义为：对于每个 $num$，$x_{num}$ 为 $num$ 第二大的约数。然后为了证明自己不是只会一点点数组的小白，它对于每个 $i$ 作死**依次**交换了每！个！$a_i$ 和 $a_{x_{a_i}}$……

然后一波操作后现在它又不会了，所以请你告诉它全部交换完后原本 $a_1$ 所处的位置。对了，小白 $\textup{\textmd{\textrm{xyy}}}$ 作死了 $t$ 次。

## 输入格式

第一行，一个整数 $t$ 表示数据组数；

对于每组数据，输入一个整数 $n$。

## 输出格式

$t$ 行，每行一个整数表示答案。

```input1
4
1
4
114514
19491001
```

```output1
1
4
65536
16777216
```

## 提示

对于 $30\%$ 的数据，$1\leq t\leq10$，$1\leq n\leq10^3$；

对于另外 $10\%$ 的数据，$t=1$；

对于全部数据，$1\leq t\leq3\times10^4$，$1\leq n\leq10^{18}$。
