## 问题描述

程序龙有一系列开关，他把开关按照满二叉树的形态摆放起来，开关从上到下从左到右的编号为 $1，2，3，……$。现在，他摆放了K层的开关，然后，他在开关1的地方放了一个小球，小球会从上往下落。

初始时，全部开关处于关闭状态，当每次有小球落到开关上时，开关的状态都会发生改变，即是由开变关或者由关变开。当一个小球落在开关上时，如果开关关闭，小球则往左走，否则往右走，直到走到叶子开关时停止。

现在，程序龙准备了 $N$ 个小球，他一个一个地将小球从开关1处往下放，看着小球落下的痕迹他感到很开心。但是慢慢地他就厌倦了，因为小球数太多了，他受不了这种折腾，他现在只想知道最后一个小球会落在哪个开关上，然后他就可以结束这个游戏了。

考虑到你的语文水平和智商水平，程序龙决定玩一次给你看，然后要求你帮他求解答案。

这里 $K$ 为 $3$，$N$ 也为 $3$，第一次小球落下：

所以，最后一个小球落在了开关 $5$ 上，输出 $5$ 即可。

## 输入格式

输入只有一行共有二个正整数：$K\ N$ （其中 $K$ 为开关的深度，$N$ 为小球个数，$1\leq  N \leq 2^{K-1}$ ）

## 输出格式

输出只有一行且只有一个正整数：第 $N$ 个小球最后所在的开关编号

```input1
16 12345
```

```output1
36358
```

## 数据规模

$100\%$ 的数据：$1\leq K \leq  40$
