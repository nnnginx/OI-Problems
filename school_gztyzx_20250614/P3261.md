## 问题描述

因为一些机缘巧合，淘淘和蓝蓝拥有了一大块土地，土地上有 $N$ 个木桩，第 $i$ 个木桩的位置为 $(x_i,Y_i)$ ，由于两个人的工作量有限，他们打算先选一个木桩 $p(X_p,Y_p)$ ，然后再选一个 $X_i=X_p$ 的木桩 $i$ ，最后选一个 $Y_j=Y_p$ 的木桩 $j$ ，由这三个木桩围成一个三角形的田地。

他们希望你能告诉他们所有这样的三角田地的面积总和乘 $2$ 是多少，由于答案可能非常大，你只需输出答案模$1000000007$ 的余数即可。

## 输入格式

第一行一个整数$N$ 。

接下来 $N$ 行，每行两个整数$X_i,Y_i$ 表示一个木桩 $i$ 的位置上。

## 输出格式

一行一个整数表示答案。

```input1
4 
0 0 
0 1
1 0
1 2
```

```output1
3
```

## 样例解释

$(0,0),(1,0),(1,2)$ 组成一个面积为 $1$ 的三角田地 ，$(0,0),(1,0),(0,1)$ 组成一个面积为 $0.5$ 的三角田地。所以答案为 $2*(1+0.5)=3$。

## 数据范围

对于 $40\%$ 的数据，满足 $N\leq 100$

对于 $60\%$ 的数据，满足 $N\leq 2000$

对于另外 $15\%$ 的数据，满足 $N\leq 40000$，且对于所有 $i$ 为偶数，第 $i$ 个木桩的位置是 $(-10000，-10000+\frac{i}{2}$；对于所有 $i$ 为奇数，第 $i$ 个木桩的位置是 $(-10000+\frac{(i-1}{2},-10000$ 。

对于 $100\%$ 的数据，满足 $3\leq N \leq 100000$，$-10000\leq X_i,Y_i\leq 10000$ 。
