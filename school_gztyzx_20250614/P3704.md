## 问题描述

参加奥运会是体操运动员乔本的梦想，为了能够顺利参加奥运会，赢得金牌，乔本进行了多天的训练。这一天，乔本所在的训练队伍要对每个人进行训练考核（满分100分），并根据分数划分了五种等级：

不合格（输出“FAIL”，不含引号）：分数小于60分。

合格（输出“PASS”，不含引号）：分数达到60分及以上，小于70分。

良好（输出“GOOD”，不含引号）：分数达到70分及以上，小于80分。

优秀（输出“EXCELLENT”，不含引号）：分数达到80分及以上，小于90分。

完美（输出“PERFECT”，不含引号）：分数达到90分及以上。

乔本想知道自己的考核等级，但是又忙于训练，于是找到了你，请你帮他计算训练等级。

## 输入格式

第一行一个数整 $n$，表示桥本的训练分数。

## 输出格式

输出桥本的训练等级。

```input1
95
```

```output1
PERFECT
```

## 数据规范

对于 $100\%$ 的数据：$0≤n≤100$。
