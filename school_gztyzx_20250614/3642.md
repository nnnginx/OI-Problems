## 题目描述

有一个小罐子，它的容积为 $v$ 。旁边摆上若干个物品，在这之中，有三种物品分别是：网球 $x$ 个，石子$y$个，小弹珠$z$个。

先往罐子里面放网球，直到罐子放不下网球。

满了吗？没有。

抓起石子，往罐子里放，直到罐子放不下石子。

满了吗？没有。

拿起小弹珠，又往罐子里面放，不几下的功夫，罐子又装满了小弹珠；

这下彻底满了。

我们的人生就像这个罐子，网球代表重要的事情，比如家庭，健康，和你的热情等；而小石子则代表其他重要的事情比如你的车子，工作，房屋等，而小弹珠代表其他东西，一些小事情。

人都要活一次，如果我们善用时间的话，我们能做任何事。如果我们先把弹珠倒进罐子里的话，就没有空间去放石子了，人生也是一样，如果你把时间和精力都耗在小事情上，你就不会有时间去做真正重要的事情。”专注在那些让你真正感到高兴的事情上，先考虑网球，在考虑石子，也就是先把重要的事情做好。把优先事项列好，其他都是像小弹珠一样的小事。现在，让我们按照正确的人生观去刚好填满这个罐子，尽可能多点重要的事情。

## 输入格式

共四行。
第一行是四个整数 $v,x,y,z$ ，分别表示罐子的容积和三种物品的数量。

第二行包含 $x$ 个整数$a_i$，第 $i$ 个整数 $a_i(0\leq a_i\leq v)$，表示第 $i$ 个网球的的体积。

第三行包含 $y$ 个整数$b_i$，第 $i$ 个整数 $b_i(0\leq b_i\leq v)$，表示第 $i$ 个石子的的体积。

第四行包含 $z$ 个整数$c_i$，第 $i$ 个整数 $c_i(0\leq c_i\leq v)$，表示第 $i$ 个小弹珠的的体积。

## 输出格式

三个整数，也就是网球，石子，小弹珠分别在这个罐子里的体积，以空格隔开。自然的，这三个数的和为 $v$ 。

## 样例 #1

### 样例输入 #1

```
10 2 3 2
2 3
4 3 2
2 1
```

### 样例输出 #1

```
5 5 0
```

## 提示

$1 \le x,y,z \le 1000$

$0 \le v \le 2000$
