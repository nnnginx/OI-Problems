**【题目背景】**

蛙想要找另一只蛙。于是它开始往地下挖，但是它发现它出不来了。帮帮它吧！

**【题目描述】**

蛙一共往下挖了 $n$ 米，现在它位于洞底。当它位于 $x$ 米深时，头顶的蓝天和小鸟激励着它往上跳，它能往上跳 $y$ 米，其中 $y\in(\mathbb{N}\cap[0,a_x])$。

但是每次跳跃之后，蛙的腿都会很酸，蛙将会休息一会儿。当蛙位于 $x$ 米深时，休息会下滑 $b_x$ 米。

蛙求蛙心切，它想知道它能否跳出它挖出的洞以及最少需要跳多次才能跳出。由于蛙只是一只蛙，所以蛙只好来求助你啦。

**【输入格式】**

从文件 `forg.in` 中读入数据。

第一行一个整数 $n$。

接下来一行 $n$ 个整数，第 $i$ 个整数为 $a_i$。

最后一行 $n$ 个整数，第 $i$ 个整数为 $b_i$。

**【输出格式】**

输出到文件 `forg.out` 中。

如果蛙无法跳出它挖的洞，则输出 $-1$；

否则输出两行，第一行一个数 $k$ 表示蛙最少要跳的次数，第二行 $k$ 个整数，第 $i$ 个整数表示蛙跳 $i$ 次之后（下滑前）位于的深度。多解则输出任意解。

**【样例输入 1】**

```
3
0 2 2
1 1 0
```

**【样例输出 1】**

```
2
1 0
```

**【数据范围】**

对于 $30\%$ 的数据，$n\le 10^3$。

对于 $100\%$ 的数据，$1\le n\le 2\times10^6$，$0\le a_i\le i$，$0\le b_i\le n-i$。
