## 题目描述

大家都知道“田忌赛马”的故事，但是其中的奥妙你们真的理解了吗？！

现在小东和小西准备比拼运筹帷幄，他们各派出$n$匹战马参加比赛。对于每次对局，输的一方将要给赢的一方$200$块，如果是平局的话，双方都不必拿出钱。

现在每匹战马的速度值是固定而且已知的，并且每次都是小西先派出战马，而小西出马也不管小东的出马顺序。

请问小东该如何安排自己的战马去对抗小西的马，才能赢取最多的钱呢，这个最多的钱是多少？

## 输入格式

第一行为一个正整数$n (n \leq 10000$) ，表示双方战马的数量。

第二行有$n$个整数表示小东的战马的速度。

第三行的$n$个整数为小西的战马的速度。

## 输出格式

仅有一行，为小东可能赢得的最多的钱，结果有可能为负。

```input1
3
92 83 71
95 87 74
```

```output1
200
```

## 数据范围

对于$100\%$的数据，$1 \leq n \leq 10000$.
