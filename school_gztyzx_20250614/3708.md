## 题目描述

小C和有一块 $H\times W$ 的棋盘，不幸的是这个棋盘上有 $N$ 个洞，问有多少个每个元素均为正整数的三元组 $(x,y,n)$，满足：
$x+n−1\le H$，$y+n−1\le W$，以 $(x,y)$ 为左上角、 $(x+n−1,y+n−1)$ 为右下角的矩阵每个位置都没有洞。

## 输入格式

输入第一行包含 $3$ 个整数 $H, W, N$。

接下来 $N$ 行，每一行包含两个整数 $x_i, y_i$ 表示第 $x_i$ 行第 $y_i$ 列有洞。

## 输出格式

输出 $1$ 个整数表示答案。

```input1
2 3 1
2 3
```

```output1
6
```

```input2
3 2 6
1 1
1 2
2 1
2 2
3 1
3 2
```

```output2
0
```

```input3
3000 3000 0
```

```output3
9004500500
```

## 样例解释

样例1：$(1, 1, 1),(1, 1, 2), (1, 2, 1), (1, 3, 1), (2, 1, 1), (2, 2, 1)$ 共 $6$ 个。

## 数据规模与约定

* $1 ≤ H, W ≤ 3000$
* $0 ≤ N ≤ min(H × W, 10^5)$
* $1 ≤ x_i​ ≤ H$
* $1 ≤ y_i​ ≤ W$
* 对于其中的 $30$ 分，有 $1\le H,W,N\le 100$
