## Description

ty奥信队为了营造即将到来的春节，所有同学会装饰树木。为了方便描述，我们用一个有根的数字树来表示一棵真正的树木。这个数字树有N（1&lt;=N&lt;=100&#44;000）个元素，从1～N标号，1号元素为树根。每个非树根元素（编号大于1的元素）都有一个父元素$P_e$，1号元素没有父亲（在输入数据中用-1表示）。<br />
每个元素i都有一棵以此元素为根的相应的子树。我们在装饰树木的时候希望元素i对应的子树至少拥有$C_i（0<=C_i<=10,000,000）$个装饰物。同时，我们也希望放置这些装饰物所花费的总时间尽量少些（将$k$个装饰物放置在元素i的位置需要花费$k×T_i$个单位时间，$1<=T_i<=100$）。
请你帮忙计算一下，最少需要花费多少时间，才能完成一棵树的装饰工作。

## Input Format

输入数据共若干行。第一行包含一个整数N。接下来N行，每行包含三个用空格隔开的整数P_i，C_i和T_i。

## Output Format

输出数据仅一个整数，表示所需的最少时间。结果可能会超过32位整数的范围，但不会超过64位整数的范围。

```input1
5
-1 9 3
1 2 2
5 3 2
5 1 4
2 3 3
```

```output1
20
```
