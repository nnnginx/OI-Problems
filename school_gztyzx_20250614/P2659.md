## 问题描述

数学老师课堂上组织大家玩一个小游戏，他一共有n位学生，首先请大家站成一排，然后每个人随便选一个数字 $A[i]$。此时任意选择两名学生，他们与他们之间学生可以作为一组，组内数字之和便称为这些学生的总分数。求所有组的总分数之和。由于数值较大，结果输出 $\mod\ 1000000007$ 的结果。

$3$ 个学生，分别选了三个数 $1,2,3$，可能的小组包括：

$\{1\} \{2\} \{3\} \{1\ 2\} \{2\ 3\} \{1\ 2\ 3\}$，这些组的组内分数为 $1\ 2\ 3\ 3\ 5\ 6$，这些小组的总分数就是 $20$。

## 输入格式

第一行输入一个非负整数 $n$；

第二行输入 $n$ 个非负整数，表示 $A[i]$；

## 输出格式

输出一个非负整数，表示总分数

```input1
3
1 2 3
```

```output1
20
```

```input2
5
3 7 5 2 6
```

```output2
162
```

```input3
7
2 1 4 7 8 9 1
```

```output3
433
```

## 样例解释

$3$ 个学生，分别选了三个数 $1,2,3$，可能的小组包括：

$\{1\}\ \{2\}\ \{3\}\ \{1\ 2\}\ \{2\ 3\}\ \{1\ 2\ 3\}$，这些组的组内分数为 $1\ 2\ 3\ 3\ 5\ 6$，这些小组的总分数就是 $20$。

## 数据范围

对于 $30\%$ 的数据，$1\leq n\leq 100$;

对于 $50\%$ 的数据，$1\leq n\leq 1000$;

对于 $100\%$ 的数据，$1\leq n\leq 100000，0\leq A[i]\leq 50000$
