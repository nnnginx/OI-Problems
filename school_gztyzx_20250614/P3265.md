<h2>Description</h2>
Czhou为了提高机房里各种神牛的身体素质，决定在每次训练后举行篮球比赛。为了保持比赛公平，Czhou要将神牛们分成两队。首先神牛们赛前都要排成固定的队伍；然后Czhou将队伍分成一半（前一半和后一半队伍人数可以不等），再分别从两个队伍中选出一些人进行篮球比赛。为了保持公平性，Czhou要求第一个队伍参加比赛的神牛能力的XOR值等于第二个队伍参加比赛的神牛能力的and值。为了增加比赛趣味，每次比赛的参加神牛们不能一样，Czhou现在想知道可以举办多少天的比赛。(很明显参加比赛的人数不能为0)<br />
Xor即为亦或&#44; 0 xor 0 = 0&#44; 0 xor 1 = 1&#44; 1 xor 0 = 1 &#44; 1 xor 1 = 0。<br />
And即为与&#44; 0 and 0 = 0&#44; 0 and 1 = 0&#44; 1 and 0 = 0&#44; 1 and 1 = 1。<br />
举个例子10 and 2 = 10，10 xor 2 = 8， 10 = (1010)2&nbsp; 2 = (10)2&nbsp; 8 =(1000)2<br />

<h2>Input Format</h2>
第一行n，表示机房有n个神牛。<br />
第二行有n个数a_i，表示各个神牛的能力值，并且这是赛前各个神牛的排队方式。<br />
<div>
	<br />
</div>
<h2>Output Format</h2>
就一个数字，表示可以举办多少天比赛。由于天数会比较多，输出结果模1000000007。
<pre><code class="language-input1">
【样例输入1】
3
1 2 3
【样例输入2】
4
1 2 3 3
</code></pre><pre><code class="language-output1">
【样例输出1】
1
【样例输出2】
4
</code></pre>
<h2>Hint</h2>
样例1说明：1 xor 2 = 3<br />
样例2说明：可以举办四天比赛，参加比赛的双方队伍分别是（1&#44;2）（3）；（1&#44;2）（3）；(1&#44;2)(3&#44;3)；（3）（3）这里虽然能力值相同，但是指的是不同的牛。&nbsp;<br />
对于(1&#44;2)(3&#44;3)来说，队伍分为两个队伍：(1&#44;2)(3&#44;3)，再从各自队伍选出全部选手参加比赛<br />
对于（3）（3）来说，队列分为两个队伍：（1&#44;2&#44;3）(3)，再从各自队伍中选出3进行比赛<br />
<br />
数据范围：<br />
0&lt;=n&lt;=10^3<br />
0 &lt;= a_i &lt;1024<br />
