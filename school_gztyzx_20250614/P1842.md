## 题目描述

小于今天格外帅气，因为他准备去音乐厅表演节目啦！然而由于他过于紧张，化妆的时间用了太多了！刘老师打电话过来说表演马上开始，小于必须以最快速度赶到音乐厅！<br />
小于赶紧编程爬了一张到达音乐厅之间的地图，巧合的是（真的是巧合吗）地图大小又是$n*m$，地图中小于处于最左上角的位置，音乐厅处于右下角（真是个偶然的巧合）。

地图上$.$代表马路，$*$代表房屋。当然了小于只能沿着$.$行走（正常来说是不能踏入$*$房屋的），并且行走的方向只能是上下左右的相邻格子。为了赶上演出的时间，小于在整个过程中可能会使用一招“萌混过关”（也可能不使用，但最多只能使用一次），使用萌混过关技能后，小于可以走进房屋一次（也就是在全程的行走中最多可以走一个$*$，注意，只有一次机会）； <br />
现在小于想要知道最少需要走多少步才能抵达音乐厅，小于可怜地请求你的帮助，按照惯例，事后小于会请你吃ty的大鸡腿！<br />

## 输入格式

第一行，2个正整数 $n$和$m$，表示一个$n*m$的矩阵,

接下来$n$行，每行m个字符，字符一定为 $.$ 或者是$*$ ，分别代表马路和房屋。

输入数据保证左上角和右下角都为$.$。

## 输出格式

一行，如果可以到达，则输入需要行走的最少步数（使用“萌混过关”当然也像正常行走一样，也记为一步）

如果不可以到达，则输出`no`.

```input1
3 3
.*.
...
...
```

```input2
3 3
.**
***
**.
```

```output1
4
```

```output2
no
```

## 数据范围

$0<N M  \leq 1000$
