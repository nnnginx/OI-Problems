## 题目描述

走廊上有 $N$ 面镜子，每面镜子中有一把钥匙，每把钥匙有一个编号 $a_i $，总共有 $M$ 个编号，编号可能相同。

你进入的镜子必须是连续的，即必须说明两个数字 $L$ 和 $R$，代表你要进入第 $L$ 面至第 $R$ 面镜子（包含 $L$，$R$）取钥匙。每进入一面镜子都会受到困在镜子中的畸形血仆的攻击而导致生命值下降 $x_i$ 点。

你希望在生命值消耗尽可能小的情况下找到所有编号的钥匙，打开面前的大门。

请求出你进入的镜子时选择的 $L$ ，$R$ 和需要消耗的最小生命值。

若存在多组解，输出 $L$ 最小的那组。

## 输入格式

第一行两个整数 $N$，$M$，分别表示走廊上镜子的总数和钥匙的编号总数。

第二行包含 $N$ 个整数 $a_i$ ，代表第 $i$ 面镜子里的钥匙编号。

第三行包含 $N$ 个整数 $x_i$ ，代表第 $i$ 面镜子下降的生命值。

## 输出格式

第一行两个整数 $L$，$R$，含义如题目描述。

第二行一个整数，表示需要消耗的最小生命值。

## 输入数据1

```

```

## 数据范围

对于$30％$的数据，$N≤200，M≤20$。

对于$60％$的数据，$N≤10^5，M≤10^3$。

对于$100％$的数据，$1≤ a_i ≤ M≤N≤10^6，0 ≤ x_i ≤ 10^9$。
