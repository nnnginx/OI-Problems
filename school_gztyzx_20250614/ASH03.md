## 题目描述

暗格里有 $N$ 根蜡烛，分别从 $1$~$N$ 被标上号码。

这里有四个咒语：

咒语1：当念出此咒语，将改变所有蜡烛的状态：本来亮着的蜡烛就熄灭，本来熄灭的蜡烛被点亮。

咒语2：当念出此咒语，将改变所有奇数号的蜡烛

咒语3：当念出此咒语，将改变所有偶数号的蜡烛

咒语4：当念出此咒语，将改变所有序号是 $3k+1$（k$\in$[0,+∞)$\cap$$\mathbb{Z}$）的蜡烛，即：$1,4,7,10,...$

初始所有蜡烛都是亮着的，你有 $x$ 次念咒语的机会。

当念完 $x$ 句咒语后，你将得到经过 $x$ 次操作后某些蜡烛的状态。

你需要预测其他蜡烛的状态，并把所有可能的情况不重复地列举出来。

## 输入格式

第一行一个正整数 $N$，表示蜡烛的根数。

第二行一个非负整数 $x$，表示念咒语的次数。

第三行若干个正整数，表示最后亮着的蜡烛，以 $0$ 结束。

第四行若干个正整数，表示最后熄灭着的蜡烛，以 $0$ 结束。

保证不会有蜡烛在输入中出现两次。

## 输出格式

每一行是所有蜡烛可能的最后状态，不能重复。

每一行有 $N$ 个字符，第 $i$ 个字符表示第 $i$ 号蜡烛的状态。$0$ 表示熄灭，$1$ 表示亮着。

答案看作二进制数从小到大排列输出。

如果没有可能的状态，则输出一行 `!ASH!` 。

## 输入数据1

```

```


## 数据范围

对于$100％$的数据，$10≤N≤100，0≤x≤10^4$。
