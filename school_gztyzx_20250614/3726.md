## 题目背景

你夜观天象，发现关注 $\text {ZYY}$ 会让你 $\text {AK}$ 这场比赛。

## 题目描述

给你一个长度为 $n$ 的序列 $b$，序列中的第 $i$ 个数代表第 $i$ 天的 **「星象」** ，这个序列中的数在 $1 \sim a$ 之间。

你还有一本占卜书，上面记载了一个 $a \times a$ 的矩阵 $k$ 。

$k_{i,i}=i，k_{i,j}=k_{j,i}$

对于所有 $1 \le i,j \le a$ ， $1 \le k_{i,j}\le a$ 。

有一种叫做 **「占卜」** 的运算，记作 $\uplus$ ，它的运算法则：$x \uplus y=k_{x,y}$ 。$\uplus$ 运算满足结合律。

你要进行 $m$ 次 **「连续占卜」**，每次给你两个数 $x$ 和 $y$  要你输出 $b_x \uplus b_{x+1} \uplus b_{x+2} ..... b_{y-1} \uplus b_y$ 。

## 输入格式

一行三个数，$n,m,a$。

然后一行一个序列 $b$。

然后 $a$ 行，每行 $a$ 个数，表示矩阵 $k$。

然后 $m$ 行，每行两个数，表示询问

## 输出格式

每个询问输出一行答案。

## 样例 #1

### 样例输入 #1

```
4 2 4
1 2 3 4
1 4 2 3
4 2 3 1
2 3 3 2
3 1 2 4
2 2
1 3
```

### 样例输出 #1

```
2
2
```

## 提示

对于 $30\%$ 的数据， $n \le 5000,m\le 1000,a \le 1000$ 。

对于 $100\%$ 的数据，$n \le 5 \times 10^5,m\le 10^6,a \le 1000$ 。
