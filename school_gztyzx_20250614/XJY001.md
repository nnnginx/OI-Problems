人比黄华瘦。

待到山花烂漫时，

还来就菊花，

采菊东篱下。

田园真好，生活是晨起暮落，日子是柴米油盐，透着烟火气的平淡与真实。

## 题目描述

农作物生长需要适宜的温度。

* 小麦生长需要温度在 $a$ 到 $b$ 摄氏度之间；
* 水稻生长需要温度在 $c$ 到 $d$ 摄氏度之间；
* 热带水果生长需要温度在 $e$ 到 $f$ 摄氏度之间。

（区间有可能重叠）。

现在给出每一带的温度，$k$ 次询问，每次你需要求出 $l$ 和 $r$ 带之间适宜种小麦的温度个数，适宜种水稻的温度个数和适宜种热带水果的温度个数。

## 输入格式

第一行，六个整数 $a$，$b$，$c$，$d$，$e$，$f$。

第二行，两个整数 $n$ 和 $k$，表示温度个数和询问次数。

第三行，$n$ 个整数 $p_1,\dots,p_n$，表示每个温度。

接下来 $k$ 行，每行两个整数 $l$ 和 $r$。

## 输出格式

$k$ 行，每行三个整数，表示适宜种小麦的温度个数，适宜种水稻的温度个数和适宜种热带水果的温度个数。

```input1
12 14 18 25 24 24
5 3
0 15 16 25 30
1 2
1 4
2 5
```

```output1
0 0 0
0 1 0
0 1 0
```

## 提示

$50\%$ 的数据，$1\leq l\leq r\leq n\leq4\times10^4$，$k\leq7\times10^3$；

$100\%$ 的数据，$1\leq a,b,c,d,e,f\leq10^9$，$1\leq l\leq r\leq n\leq10^6$，$k,p_i\leq10^6$。

数据保证 $a\leq b$，$c\leq d$ 和 $e\leq f$。
