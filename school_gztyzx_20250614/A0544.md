## 题目描述

33DAI 画了一幅画：

```
./\.
/..\
\../
.\/.
```

请输出 $n\times m$ 个这样的画，格式参考样例。

## 输入格式

第一行一个整数 $n,m$。

## 输出格式

输出如果描述的画。

```input1
2 3
```

```output1
./\../\../\.
/..\/..\/..\
\../\../\../
.\/..\/..\/.
./\../\../\.
/..\/..\/..\
\../\../\../
.\/..\/..\/.
```

```input2
3 3
```

```output2
./\../\../\.
/..\/..\/..\
\../\../\../
.\/..\/..\/.
./\../\../\.
/..\/..\/..\
\../\../\../
.\/..\/..\/.
./\../\../\.
/..\/..\/..\
\../\../\../
.\/..\/..\/.
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\le n,m\le 4$。
