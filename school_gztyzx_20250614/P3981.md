## 问题描述

有一家商店鲜奶的售价规则如下：

鲜奶刚上架时，一瓶鲜奶售价为 $x$ 元，有效期限为 $h$ 个小时。接著，在之后的十个小时，每经过一小时鲜奶的售价会减少 $1$ 元；在第十一至二十小时，每经过一小时鲜奶的售价会减少 $2$ 元，依此类推。

也就是说，对于所有小于 $h$ 的正整数 $i$，在经过第 $i$ 小时后，鲜奶售价会再减少 $\lceil \frac{i}{10} \rceil$ 元。（$\lceil v \rceil$ 的意思是不小于 v 的最小整数。）并且经过 h 小时后，鲜奶会被下架（因为过期了），顾客不能再购买了。

举例来说，当 $x = 1000,h = 30$，若鲜奶上架时就立刻买下（也就是经过 $0$ 个小时），需要花 $1000$ 元。而经过 $1$ 个小时后才买下，售价会减少 $1$，变为 $999$ 元。经过 $10$ 小时后才买下，售价则会减少 $10$ 次 $1$，所以此时只需花 $990$ 元。而若是第 $11$ 个小时买下，售价会再比第 $10$ 小时少 $2$ 元，此时售价为 $988$ 元。

小明想要买一瓶鲜奶，但他的预算有限，只有 $y$ 元，请问小明至少要等鲜奶上架几个小时后购买才能符合预算？也就购买鲜奶时的花费不会超过 $y$ 元。

## 输入格式

输入只有一行，包含 $3$ 个整数值，整数间以一个空白隔开。第一个整数代表 $x$，第二个整数代表 $h$，第三个整数代表 $y$，这三个数均为正整数，且 $x,y \le 10^6$，并保证在过程中鲜奶售价也总是正整数。

## 输出格式

输出也只有一行，若在鲜奶有效期间内（恰经过 $h$ 小时也不能购买），小明无法买到符合预算的鲜奶，请输出`IMPOSSIBLE`。否则，请输出一个非负整数，代表至少要上架后几个小时，小明才能买到符合他的预算的鲜奶。

```input1
1000 100 989
```

```output1
11
```

```input2
514 1 1000000
```

```output2
0
```

```input3
514 1 513
```

```output3
IMPOSSIBLE
```

## 数据范围

对于 $20\%$ 的数据，$h = 1$。

对于 $60\%$ 的数据，$1 \le h \le 10$。

对于 $100\%$ 的数据，$1 \le h \le 1000$。
