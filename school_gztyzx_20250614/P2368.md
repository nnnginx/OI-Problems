## 问题描述

程序龙有 $N$ 个盒子（ $N$ 足够大，可装入任何数量的球），分别编号 $1，2，……，N$。同时有 $S$ 个小球 $(0<S<1000)$，要求你将 $S$ 个小球装入到盒子中去。

装球时必须遵守以下的规则：

1. 第 $1$个盒子不能为空；
2. 第 $1$ 个盒子和最后一个有球的盒子中间不可以有空的盒子；
3. 装入必须严格按递增顺序进行，即每个有球盒子中的小球数必须比它前面的盒子中的小球数多；
4. 要求有球的盒子尽可能多；
5. 装完后，相邻盒子中球个数差的绝对值之和最小（未装的盒子不计）。

如：当 $S=8$ 时，装入方法有 $1，2，5$ 或 $1，3，4$。装入法 $1，2，5$ 时差的绝对值之和为 $ABS(2-1)+ABS(5-2)=4$；装入法 $1，3，4$ 时差的绝对值之和为 $ABS(3-1)+ABS(4-3)=3$。所以装入法 $1，3，4$ 为正确的装入方法。

你能做到吗？

## 输入格式

输入一个整数 $S$，$S$ 表示小球的个数。

## 输出格式

输出满足上述游戏规则的装入方法。

```input1
8
```

```output1
1 3 4
```

## 数据范围

$0<S<1000$
