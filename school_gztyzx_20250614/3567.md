
## 题目描述

给定一个仅包含字符 `1` 或 `0` 或 `?` 字符串 $s$。定义一个非空子串是不稳定的当且仅当子串中任意相邻两数均不相同，如 $101010...101010... $或 $010101...010101...$。其中 `?` 为通配符，可以当成 $1$ 和$0$ 。请求出给定的 $s$ 中不稳定子串个数。

## 输入格式

一行一个字符串，表示 $s$。

## 输出格式

输出一行一个整数，表示答案。

```input1
0?10
```

```output1
8
```

## 样例解释1

对于样例$0?10$，他的非空子串分别为：$\{0\}$、$\{?\}$、$\{1\}$、$\{0\}$、$\{0?\}$、$\{?1\}$、$\{10\}$、$\{0?1\}$、$\{?10\}$、$\{0?10\}$。

其中长度为$1$的子串$\{0\}$、$\{?\}$、$\{1\}$、$\{0\}$显然是不稳定的；

长度为$2$的子串：$\{0?\}$、$\{?1\}$、$\{10\}$，无论$?$取$0$还是$1$，那么都是不稳定的。

长度为$3$的子串，$\{0?1\}$无论$?$取$0/1$，都是稳定的。$\{?10\}$当$?$取$0$时，是不稳定的，所以$\{?10\}$是不稳定的。

长度为$4$的子串，$\{0?10\}$是稳定的。

因此，共有$8$个不稳定的子串。


```input2
?10??1100
```

```output2
25
```

该样例长度为$\ 8$的子串，有$0$个是不稳定的；

该样例长度为$9$的子串，有$0$个是不稳定的；

## 数据范围

对于 $20\%$ 的数据，$1≤ s≤ 50$。
对于 $100\%$ 的数据，$1 \le s \le 2 * 10^5$。
