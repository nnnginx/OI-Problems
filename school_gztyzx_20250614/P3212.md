<h2>Description</h2>
给出一个长度为N的数列{a[n]}，1&lt;=a[i]&lt;=M(1&lt;=i&lt;=N)。<br />
现在问题是，对于1到M的每个整数d，有多少个不同的数列b[1]&#44; b[2]&#44; ...&#44; b[N]，满足：<br />
(1)1&lt;=b[i]&lt;=M(1&lt;=i&lt;=N)；<br />
(2)gcd(b[1]&#44; b[2]&#44; ...&#44; b[N])=d；<br />
(3)恰好有K个位置i使得a[i]&lt;&gt;b[i](1&lt;=i&lt;=N)<br />
注：gcd(x1&#44;x2&#44;...&#44;xn)为x1&#44; x2&#44; ...&#44; xn的最大公约数。<br />
输出答案对1&#44;000&#44;000&#44;007取模的值。<br />
<div>
	<br />
</div>
<h2>Input Format</h2>
第一行包含3个整数，N，M，K。<br />
第二行包含N个整数：a[1]&#44; a[2]&#44; ...&#44; a[N]。&nbsp;<br />
<div>
	<br />
</div>
<h2>Output Format</h2>
输出M个整数到一行，第i个整数为当d=i时满足条件的不同数列{b[n]}的数目mod 1&#44;000&#44;000&#44;007的值。
<pre><code class="language-input1">
【样例输入1】
3 3 3
3 3 3
【样例输入2】
3 5 3
1 2 3
</code></pre><pre><code class="language-output1">
【样例输出1】
7 1 0
【样例输出2】
59 3 0 1 1
</code></pre>
<h2>Hint</h2>
【样例1解释】<br />
当d=1，{b[n]}可以为：(1&#44; 1&#44; 1)&#44; (1&#44; 1&#44; 2)&#44; (1&#44; 2&#44; 1)&#44; (1&#44; 2&#44; 2)&#44; (2&#44; 1&#44; 1)&#44; (2&#44; 1&#44; 2)&#44; (2&#44; 2&#44; 1)。<br />
当d=2，{b[n]}可以为：(2&#44; 2&#44; 2)。<br />
当d=3，因为{b[n]}必须要有k个数与{a[n]}不同，所以{b[n]}不能为(3&#44; 3&#44; 3)，满足条件的一个都没有。<br />
【数据说明】<br />
对于30%的数据，1&lt;=N&lt;=20&#44; 1&lt;=M&lt;=2。<br />
对于50%的数据，1&lt;=N&#44;M&lt;=1000。<br />
对于70%的数据，1&lt;=N&#44;M&lt;=10000。<br />
对于100%的数据，1&lt;=N&#44;M&lt;=300000&#44; 1&lt;=K&lt;=N&#44; 1&lt;=a[i]&lt;=M。<br />
