## 问题描述

牛牛喜欢数数，不过因为他现在只能认识 $0\sim 9$ 的 $10$ 个数字，所以他在从 $0$ 数到$9$ 以后下个数字又从 $0$ 开始重新数起。

现在有一个字符画打印成边长为N的等边三角形，例如 $N = 5$ 时，三角形如下所示。

```
.....
....
...
..
.
```

现在牛牛从上至下，从左至右依次数过去，并且他总是从 $0$ 开始数起，为了确认他数的结果是正确的，他需要你告诉他，当他数到第 $x$ 行第 $y$ 列时，他数到的数字应该是几。

## 输入格式

仅一行，三个正整数 $N, x, y$，分别表示三角形的边长，以及牛牛想要确认结果的坐标是第 $x$ 行第 $y$ 列。

## 输出格式

仅一个 $0\sim 9$ 的数字，表示问题的答案。

```input1
5 1 1
```

```output1
0
```

```input2
5 2 1
```

```output2
5
```

```input3
7803917 6501657 800542
```

```output3
3
```

## 数据范围

对于 $25\%$ 的测试数据，保证 $1≤ N ≤ 10^3$。

对于 $50\%$ 的测试数据，保证 $1 ≤ N ≤ 10^9$。

对于 $100\%$ 的测试数据，保证 $1 ≤ N ≤ 10^{18},1 ≤ x,y ≤ N$ 且 $2 ≤ x + y ≤ N + 1$。
