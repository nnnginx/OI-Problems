## 问题描述

为了测试小W的数学水平，果果给了小W $N$ 个点，问他这 $N$ 个点能构成的三角形个数。

## 输入格式

第一行一个整数 $N$，代表点数。

接下来 $N$ 行，每行两个非负整数 $X、Y$，表示一个点的坐标。

## 输出格式

一个非负整数，即构成三角形个数。

```input1
5
0 0
1 0
2 0
0 1
1 1
```

```output1
9
```

## 数据范围

对于 $20\%$ 的数据：$N=3$

对于另外 $40\%$ 的数据：保证任意 $3$ 点不在同一直线上

对于 $100\%$ 的数据：$N\leq 100$，保证任意两点不重合，坐标$\leq 10000$
