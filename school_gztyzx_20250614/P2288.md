## 题目描述

$TY$的机房最近正在装修，总共购买了$n \times m$块双面镜子。

调皮的小赵看到了这些双面镜，于是他把这些镜子放置在一个$n \times m$的方格矩阵中。矩阵中的每一个方格都会布置一面镜子，小赵把每一面镜子都沿着方格的对角线布置下去，因此每个格子上的镜子都会有两种布置方式，第一种由字符'/'表示，也就是把镜子放在方格的左下角到右上角的位置，第二种是由字符'\\'表示，也就是把镜子放在方格的左上角到右下角的位置。

小赵想用这些镜子来测试光的折射。

于是到了晚上，小赵拿出一支激光笔，站在方格矩阵的外围，沿着水平或垂直于方格矩阵的一行或一列的方向用激光笔射出一束光，光束在方格矩阵中的镜子之间反射。

小赵是把镜子沿对角线方向放置的，所以呢一束水平的光线经过一面镜子反射后将变成竖直方向的，而一束竖直方向的光经过一面镜子反射后，将会变成水平方向的。

小赵现在把方格矩阵中每一面镜子的摆放方式告诉你，他想知道他发出的一束光，最多可以在方格矩阵中反射多少次，请你帮他计算出最多的反射次数！

## 输入格式

第1行输入两个整数，$n$和$m$，由空格隔开。

接下来输入$n$行，每行输入$m$个字符'/'或'\\'：

对于第$i$行第$j$列的字符，表示方格矩阵中格子$(i,j)$所布置的镜子的方向。<br />

## 输出格式

输出一个整数，表示从场地外水平或垂直射入方格矩阵的光束可以被反射的最大次数的整数。

如果这束光会被无限次的反射，请输出`-1`。

```input1
3 3
/\\
\\\
/\/
```

```output1
3
```

小赵可以由上往下朝中间第$2$列发射光束，光束来到格子$(1,2)$，被'\\'形式的镜子反射成向右的水平光线，向右移动。

接下来光束来到格子$(1,3)$，光线被'\\`形式的镜子反射成向下的竖直光线，向下移动。

然后光束来到格子$(2,3)$，光线再被'\\'形式的镜子反射成向右的水平光线，向右移动，射出矩阵以外，终止。

所以，光束会被反射$3$次，这是最多的次数。

## 数据范围

对于$100\%$的数据，$n,m \leq 1000$。
