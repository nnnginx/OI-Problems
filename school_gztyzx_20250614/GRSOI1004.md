## 题目背景

xiaosi4081 正在从宿舍前往机房。

## 题目描述

从宿舍到机房的路很奇怪，它分成 $n$ 个小段。第 $i$ 个小段对应一个 $a_i$，这也就意味着经过第 $i$ 段的时间可以为 $[1,a_i]$ 间的任意整数，并且通过整段路的时间为在每一段路上经过的时间的乘积。

xiaosi4081 是一个奇怪的人，他想要把每个从宿舍走到机房的方案都走一遍。定义两个方案相同当且仅当两个方案中经过每一个小段路的时间都相等。

现在，xiaosi4081 想知道把每个从宿舍走到机房的方案都走一遍所需要的总时间。答案对 $10^9+7$ 取模。

我们假设 xiaosi4081 有一个超级先进的工具可以使得他每次走到机房之后都能瞬移回到宿舍。

形式化地说，相当于求下面这个式子的值：

$$
\sum_{i_1=1}^{a_1}
\sum_{i_2=1}^{a_2}
\cdots 
\sum_{i_n=1}^{a_n} i_1\times i_2 \times \cdots \times i_n \bmod (10^9+7)
$$

## 输入格式

第一行，一个数 $n$ 。

接下来有 $n$ 个数，第 $i$ 个数表示 $a_i$ 。

## 输出格式

输出对 $1000000007$ 取模后的答案。

## 样例 #1

### 样例输入 #1

```
3
1 1 2
```

### 样例输出 #1

```
3
```

## 提示

对于 $5 \%$ 的数据，$n\leq 5,\prod a_i \leq 10^6$ 。

对于另外 $10 \%$ 的数据，$\prod a_i \leq 10^6$ 。

对于另外 $5 \%$ 的数据，所有 $a_i$ 相等。

对于 $100 \%$ 的数据，$n \leq 10^6,a_i \leq 10^8$

对于样例【1】的解释：

第一种方案是 $[1,1,1]$，即在第一段路上用 $1$ 时间，在第二段路上用 $1$ 时间，在第三段路上用 $1$ 时间，耗费的时间为 $1\times 1\times 1=1$。

第二种方案是 $[1,1,2]$，耗费的时间为 $1 \times 1 \times 2$。

不存在其他的方案。两种方案的和为 $1+2=3$。
