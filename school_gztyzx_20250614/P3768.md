## 问题描述

程序龙的脑洞总是很清奇，正常人的阅读习惯是从左到右阅读，可他却喜欢从右往左看。这天他很无聊，就在纸上写了一行$n$个正整数，然后用他的方式求出这$n$个整数的和。例如写了$3$个整数：$123\ 45\ 60$ ，结果是 $06+54+321=381$。

现在，他用同样的方式考考你，你能接受考验吗？

## 输入格式

第一行$1$个整数$n$，$1\leq n \leq 100$。

第二行$n$个正整数，每个正整数不超过$1000000$。

## 输出格式

一个整数。

```input1
3
123 45 60
```

```output1
381
```
