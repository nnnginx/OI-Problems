# 问题描述

最近老师教了狗狗怎么算组合数，狗狗又想到了一个问题。。。

狗狗定义 $C(N,K)$ 表示从 $N$ 个元素中不重复地选取 $K$ 个元素的方案数。

狗狗想知道的是 $C(N,K)$ 的奇偶性。

当然，$N$ 和 $K$ 都可能相当大。

解决这个问题。

# 输入格式

第 $1$ 行：一个正整数 $t$，表示数据的组数。

第 $2\sim 2+t-1$ 行：两个非负整数 $N$ 和 $K$。（保证 $k\leq n$）

# 输出格式

每一组输入，如果 $C(N,K)$ 是奇数则输出 $1$，否则输出 $0$。

```input1
3
1 1
1 0
2 1
```

```output1
1
1
0
```


## 数据范围

对于 $30\%$ 的数据，$n\leq 10^2$,$t\leq 10^4$

对于 $50\%$ 的数据，$n\leq 10^3$,$t\leq 10^5$

对于 $100\%$ 的数据，$n\leq 10^9$, $t\leq 10^5$
