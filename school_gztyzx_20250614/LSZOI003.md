终于到国庆假期了，皮皮决定去找同学们一起玩，经过了激烈的讨论，他们最后决定去玩密室逃脱。在大家的共同努力下，他们顺利到达了最后一关。

最后一关是一个密码门，门上有着 $T$ 个仅由数字、字母和三种括号组成的字符串，每个字符串后都有一个数字，在旁边写着一个提示：

将所有字符串依次按规则展开，展开后每个字符串取出对应位置的字符，组合得到的结果就是打开门的密码。
字符串的展开规则如下：

首先找到一对括号，中间不含有任何其他括号，如果为小括号，则将里面的内容重复 $K$ 遍，$K$ 为左括号后面紧挨着的数字 $（1 \le K \le 9）$；如果是中括号，则将里面的字母进行大小写转换；如果为大括号，则将里面的内容进行翻转。上述操作完成后，将这一对括号删除。依次执行上述操作，直到字符串没有任何括号存在的时候，展开结束。

皮皮和同学们在经过了前面的重重关卡后，现在已经完全想不出来该如何去做了，于是他来求助你，你能帮助他们从密室中顺利逃脱么？

## 输入格式

第一行，一个整数 $T$，表示字符串及位置的个数

接下来 $T$ 行，每行一个字符串 $S_i$
和一个整数 $P_i$
，表示待处理的字符串及字符的对应位置，字符串保证仅由大小写字母、小括号"()"、中括号"[]"、大括号"{}"以及数字组成。

## 输出格式

仅一行，一个长度为 $T$ 的字符串，为每个字符串展开后取出对应位置的结果。

## 样例 #1

```input1
2
{Once[Again]} 3
(2[{code}]) 8
```

```output1
AC
```

## 提示

第一个字符串首先展开成"{OnceaGAIN}"，最后得到"NIAGaecnO"，第3个字符为'A'。

第二个字符串首先展开成"(2[edoc])"，然后展开成"(2EDOC)"，最后得到"EDOCEDOC"，第8个字符为'C'。

设定括号嵌套层数为 $L$，原字符串长度为 $N$，展开后字符串长度为 $M$；

对于 $50\%$ 的数据，$1 \le L \le 100,1 \le N,M \le 10000$；

对于 $100\%$ 的数据，$1 \le L,N \le 10^6,1 \le M \le 10^{18},1 \le T \le10$，保证 $1\le P_i\le M$ 。
