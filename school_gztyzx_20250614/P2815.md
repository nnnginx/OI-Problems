<h2>Description</h2>
在一个坐标平面中，有n个点，定义第i个点的坐标是(i&#44; yi)。<br />
现在给小C一个点集，小C的任务是确定是否有可能绘制出两条平行且不重叠的直线，使集合中的每个点都恰巧可以落在其中一条直线上，并且保证每条直线上至少有一个点。<br />
现在请你来帮助小C解决这个问题。<br />
<div>
	<br />
</div>
<h2>Input Format</h2>
多组测试数据，第一行输入测试的组数T，1 &lt;= T &lt; 10<br />
第2n（n = 1&#44; 2&#44; 3 ... T）行：一个正整数n（3 &lt;= n &lt;= 1000），代表点的数量<br />
第2n + 1（n = 1&#44; 2&#44; 3 ... T）行：有n个用空格隔开的整数y1&#44; y2 ... yn（|yi| &lt;= 1000000000），代表每个点的纵坐标<br />
<div>
	<br />
</div>
<h2>Output Format</h2>
输出T行分别表示T组数据的判断结果，如果小C可以找到这样的两条线，输出"Yes"，如果找不到这样的两条线，输出"No"
<pre><code class="language-input1">
样例一：
1
5
100 0 0 0 0
样例二：
1
6
2 2 3 3 4 4
样例三：
1
5
2 3 4 5 6
</code></pre><pre><code class="language-output1">
样例一：
Yes
样例二：
Yes
样例三：
No
</code></pre>
<h2>Hint</h2>
样例解释<br />
样例一：<br />
&nbsp;（1，100）在一条直线上，<br />
（2，0）（3，0）（4，0）在一条直线上，并且两条直线平行 <br />
样例二： <br />
（1，2）（3，3）（5，4）在一条直线上，<br />
（2，2）（4，3）（6，4）在一条直线上，并且两条直线平行<br />
数据范围<br />
对于50%的数据，3 &lt;= n &lt;= 50&nbsp; <br />
对于75%的数据，3 &lt;= n &lt;= 500 <br />
对于100%的数据，3 &lt;= n &lt;= 1000，1 &lt;= T &lt; 10<br />
