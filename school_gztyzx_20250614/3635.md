## 题目背景

你看到了一个告示。上面说：“解出这两道题，就会有奖励。”

## 题目描述

有一个序列，一个平面。

一、序列有以下特征：

1，单调递增，所有数都大于0。

2，序列里有不小于 $k$ 对数成倍数关系。

3，前缀异或和单调不递减。

4，序列里偶数数量与奇数数量的差的绝对值大于序列长度的一半。

5，后缀异或和单调不递减。

要求你给他一个满足以上条件且和最小的序列。”

你求得这个序列的和模 $10007$ 就可以了。

---

二、如图，有一个平面直角坐标系，两个重要的点和 **「数学射线」** 。
![image](./3635/file/LeQgRh4z7SysKlxpDjF_P.png)

如图：

蓝色点是 **「数学射线」** 发射点，记作点 $A(x_1,y_1)$ 。

紫色点是目标点，记作点 $B(x_2,y_2)$ 。

我们要让 **「数学射线」** 射到目标点，并且 **「数学射线」** 必须反射 $2$ 次。

两个反射点记作点 $P(a,0)$ 和 $Q(0,b)$ 。

求得 $a,b$ 即可通过。

## 输入格式

接下来 $T$ 行，每行一个正整数 $op$。当 $op=1$ ，输入 $n,k$ ；否则，输入四个正整数 $x_1,y_1,x_2,y_2$。

## 输出格式

输出 $T$ 行，特征序列和 或 $a,b$ 保留三位小数。

## 样例 #1

### 样例输入 #1

```
4
2 1 2 2 1
2 7 3 1 1
2 1 2 3 4
1 30 2
```

### 样例输出 #1

```
1.000 1.000
1.000 0.500
0.333 0.500
730
```

## 提示

数据保证 $\frac{x_1}{x_2}\ne \frac{y_1}{y_2},1 \leq k < n$ 。

| 测试点 | 数据范围 | 特殊性质 |
| :-----------: | :-----------: | :-----------: |
| $1$ | $n \leq 10$ | $A$ |
| $2$~$3$ | $n \leq 10^7$ | $A$ |
| $4$ | $x_1,y_1,x_2,y_2 \leq 100$ | $B$ |
| $5$~$6$ | $x_1,y_1,x_2,y_2 \leq 10^{12}$ | $B$ |
| $7$~$8$ | $n \leq 10$ , $x_1,y_1,x_2,y_2 \leq 100$ | **无** |
| $9$~$10$ | $n \leq 10^7$ ,  $x_1,y_1,x_2,y_2 \leq 10^{12}$ | **无** |

**特殊性质A :** 对于每个测试数据 $op=1$ 。

**特殊性质B :** 对于每个测试数据 $op=2$ 。

##### 提示：数学射线是第二场比赛T2的升级版
