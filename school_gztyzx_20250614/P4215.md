# 题目描述

求

$$
\sum_{i = 1}^n\sum_{j = 1}^n \operatorname{lcm}(i, j)
$$

对 $p$ 取模的值。

# 输入格式

一行两个整数 $n, p$。

# 输出格式

一行一个整数表示答案。

# 输入输出样例

```input1
2 6
```

```output1
1
```

```input2
114514 1919810
```

```output2
1359761
```

```input3
1919810 114514
```

```output3
63539
```

# 数据范围

对于 $20\%$ 的数据，$1 \le n \le 2 \times 10^3$。


对于 $100\%$ 的数据，$1 \le n \le 2 \times 10^7$，$2 \le p \le 10^9$。
