## 问题描述

给出一个长度为 $n$ 的序列 $a$，请问它有没有一个子段和为 $k$？

## 输入格式

第一行两个整数 $n$，$k$。

第二行 $n$ 个整数，表示 $a$。

## 输出格式

一个字符串，为 `YES` 或 `NO`，表示有或没有。

```input1
6 10
1 1 4 5 1 4
```

```output1
YES
```

```input2
7 19
1 9 1 9 8 1 0
```

```output2
YES
```

```input3
10 47
2 1 4 7 4 8 3 6 4 7
```

```output3
NO
```

## 样例解释

第一个样例，可以选择 $[2, 4]$，$[3,5]$，$[4,6]$。

第二个样例，可以选择 $[2,4]$，$[3,6]$，$[3, 7]$。

第三个样例并没有选择方案。

## 注意事项

本题数据范围较大，建议使用快读。

快读模板：

```CPP
inline long long read(){
    long long x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9'){
        if (ch == '-') f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') x = (x << 1) + (x << 3) + (ch ^ 48), ch = getchar();
    return x * f;
}
```
