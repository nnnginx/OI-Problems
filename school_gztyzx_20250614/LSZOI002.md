## 题目背景

(此题为ACM)

小 $n$ 喜欢学习数字知识，他对大数字特别感兴趣。有一天，他认为数学课上学的知识太简单了，就开始想他昨晚做的题

## 题目描述

对于一个正整数，如果其及其各个数字之和的差（指其减去各个数字之和，以十进制表示）不小于s，便是“最好数”。

他需要计算不大于 $n$ 的“最好数” 个数。

小 $n$ 想自己计算，但很快意识到
这对他来说太难了。你可以帮他吗?

## 输入格式

第一行两个数， $n$ 和 $s$

## 输出格式

一个数，表示  $1$ ~ $n$ 之间有多少个“最好数”。

```input1
12 1
```

```output1
3
```

```input2
10 9
```

```output2
1
```

## 提示

对于 $100\%$ 的数据，$10^{16} \le n \le 10^{18}$,$1 \le s \le n$。
