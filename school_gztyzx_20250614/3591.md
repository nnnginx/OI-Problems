## 题目描述

"A+B problem"对你们来说太简单了，所以我们将它加强了！

给出两个数 $a,b$，询问是否有两个数 $a',b'($**1**$ \le a' \le a)($**1**$ \le b' \le b)$ 使得 $x*(a'+b')=a+b$ ($x$ 为任意整数) ，如果有，请输出最大的 $x$ ，否则输出 $-1$。

## 输入格式

一行两个数 $a,b$ ，含义见题。

## 输出格式

一行一个数 表示最大的 $x$。

## 样例 #1

### 样例输入 #1

```
4 2
```

### 样例输出 #1

```
3
```

## 样例 #2

### 样例输入 #2

```
24 27
```

### 样例输出 #2

```
17
```

## 样例 #3

### 样例输入 #3

```
20 11
```

### 样例输出 #3

```
1
```

## 提示

对于第一个样例，$a'=b'=1$ 时，$x$ 最大。

对于第二个样例，$a'=1,b'=2$ 时，$x$ 最大。

对于第三个样例，注意不能取 $a'=0,b'=1$。

【数据范围】

对于 $30\%$ 的数据，$1 \le a,b \le 1000$

对于 $60\%$ 的数据，$1 \le a,b \le 10^6$

对于 $100\%$ 的数据，$1 \le a,b \le 10^9$
