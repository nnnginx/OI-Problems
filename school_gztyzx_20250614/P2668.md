## 问题描述

马拉多纳是一个神奇的地方，这里的人们沿着一条笔直的山脚建立了自己的房子，一共有 $n$ 座房子，$n$ 座房子的位置分别为 $p$。但是这些居民十分的宅，一辈子只会在以自己家为圆心的一个固定的半径 $r$ 内活动，这就导致了会有某一些居民，永远都不会有所交流。

请问马拉多纳一共有多少对人家一辈子都不会有所交流呢？

## 输入格式

第$1$行：一个数 $N$，表示房子的数量($1 \leq N \leq 50000$)

第$2$ - $N + 1$行：每行 $2$ 个数 $P$，$R$ 中间用空格分隔，$P$ 表示房子的位置，$R$ 表示这家住户的活动范围半径($1\leq P; R \leq 10^9$)

## 输出格式

输出共有多少对永远不会有所交流的人家。

```input1
4
1  1
2  1
3  2
4  1
```

```input2
2
1  2
3  4
```

```input3
3
2  2
1  3
5  5
```

```output1
1
```

```output2
0
```

```output3
0
```

## 样例解释

$4$ 座房子分别位于 $1,2,3,4$ 的位置，活动范围半径分别为 $1, 1, 2, 1$，那么 $\{1, 2\}, \{1, 3\},\{2, 3\}, \{2, 4\}, \{3,4\}$ 这 $5$ 对居民活动范围都有交点，只有 $\{1, 4\}$ 是永远都没有交流的。

## 数据范围

对于 $10\%$ 的数据，$1≤N≤10$；

对于 $40\%$ 的数据，$1≤N≤2000$；

对于 $100\%$ 的数据，$1≤N≤50000$。
