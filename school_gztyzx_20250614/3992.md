## 题目背景

你正在玩回合制游戏《铁》。

## 题目描述

初始时，场上有 $6$ 个角色，编号为 $1 \sim 6$，全部存活。其中编号 $1 \sim 3$ 为我方角色，编号 $4 \sim 6$ 为敌方角色。

- ### 行动顺序

我们用**行动序列**表示每个人的行动顺序。

初始时，行动序列为 $1 \ 2 \ 3\ 4\ 5\ 6$，数字表示角色编号。

在对局开始或上个人行动结束后，**序列首位的元素先被移至序列末位，然后此元素对应的角色开始行动。**

在任意时刻，若有角色死亡，则角色对应的编号从行动序列中移除。

所有角色将遵守此规则行动直至对局结束。

- ### 角色及其行动方式

每个角色都有名称 $a$ 和血量 $b$ 两个参数，角色血量 $\leqslant 0$ 视为死亡。

不同名称的角色拥有不同的技能，在开始行动时使用一次。名称为以下名称的一种：


- **垃圾桶（bin）**：攻击血量最低的敌方角色，扣除 $3$ 点血量。

-  **希儿（welder）**：攻击血量最低的敌方角色，扣除 $2$ 点血量。若受到攻击的角色死亡，则触发 **【再现】**。

   **【再现】**：再使用一次技能，但不会触发 **【再现】**。

-  **白露（bailu）**：治疗除自身外血量最低的我方角色，增加 $2$ 点血量。若我方只剩自己一个角色，则不使用技能，行动结束。

-  **冰斧哥（winter）**：攻击行动序列中位置最靠前的敌方角色，扣除其 $1$ 点血量并将它在行动序列的位置与后一位交换。

技能无法对死亡的角色使用；若有多个血量最低的角色，选择行动序列中最靠前的。

不能再使用技能时，角色行动结束。

特别地，我方角色名称一定不为 **winter**，敌方角色名称一定不为 **welder** 或 **bailu**，且我方不会出现多个名称为 **welder** 或 **bailu** 的角色。

- ### 结束条件

在任意时刻，当场上一方角色全部死亡时，对局结束。保证所有角色的总行动次数小于 $3\times 10^6$。

------------
在对局开始前，你得知了 $6$ 个角色的名称和血量，请求出对局结束时的行动序列和所有存活角色的血量。

## 输入格式

输入共两行。

第一行 $6$ 个字符串，第 $i$ 个字符串表示 $a_i$。

第二行 $6$ 个整数，第 $i$ 个整数表示 $b_i$。

## 输出格式

输出共两行。

第一行若干个整数，表示对局结束时的行动序列。

第二行若干个整数，按**编号**升序顺序输出所有存活角色的血量。

```input1
bin welder bailu winter bin bin
3 3 3 3 3 3
```

```output1
1
2
```

## 样例 1 解释

对于第一个样例，前四个人行动后的行动序列与所有存活角色的血量如下：

```
2 3 5 6 1 
3 3 3 3 3 

3 5 6 1 2 
3 1 3 3 3 

5 6 1 2 3 
1 3 5 3 3 

6 1 3 5 
3 5 3 1 
```
血量仍按编号顺序给出。


```input2
bin welder bailu bin winter bin
5 5 5 5 5 5
```

```output2
3 2 1
6 1 5
```

```input3
bin bin welder bin winter winter
400000 500000 500000 1000000 1000000 1000000
```

```output3
6
599963
```

## 数据范围

$\forall 1\leqslant i \leqslant 3,a_i\in \{\text{bin},\text{welder},\text{bailu}\},1\leqslant b_i \leqslant 10^6$，不存在多个 $a_i = 2$ 或 $3$。

$\forall 4\leqslant i \leqslant 6,a_i\in \{\text{bin},\text{winter}\},1\leqslant b_i \leqslant 10^6$。

保证所有角色的总行动次数小于 $3\times 10^6$。
