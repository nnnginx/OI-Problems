**【题目背景】**

蛛有一颗珠，这颗珠有望远镜功能。

有一天蛛用珠发现了一颗生活着许多蛛的星球――蛛星！

**【题目描述】**

蛛星上一共有 $n$ 只蛛，第 $i$ 只蛛有 $a_i$ 条腿。蛛发现一只有 $\alpha$ 条腿的蛛和一只有 $\beta$ 条腿的蛛会织出攻击力为 $\alpha\operatorname{xor}\beta$ 的网，其中 $\operatorname{xor}$ 表示按位异或运算。

蛛星已经”蛛满为患“，好心蛛通过”珠牌信号发射器“和蛛星上的蛛取得了联系，它们计划占领另一颗星球――猩星！

猩星是一颗居住着许多猩猩的星球，每一只猩猩的防御力都为 $k$。为了打败这些猩猩，蛛们需要选出一些蛛参战，且需要保证这些蛛中任意一对蛛都能织出防御力至少为 $k$ 的网。

为了令蛛们看起来更具压迫力，它们想选出更多的蛛参战。

但是蛛觉得自己太蠢了，选不出最优方案，所以它使用”珠牌信号发射器”联系了你，一只精通算法的蛛，让你帮它决定参战的蛛。

**【输入格式】**

从文件 `spader.in` 中读入数据。

第一行两个整数 $n,k$。

接下来一行 $n$ 个整数，第 $i$ 个整数为 $a_i$。

**【输出格式】**

输出到文件 `spader.out` 中。

如果蛛们无法参战，输出 `More_and_more_vegetable,what_should_ckj_do?`；

否则输出两行，第一行为最多可以选出蛛的只数 $l$，第二行 $l$ 个整数表示选出的蛛的编号，多解则任意输出一个。

**【样例输入 1】**

```
6 8
2 8 4 16 10 14
```

**【样例输出 1】**

```
3
1 5 4
```

**【样例输入 2】**

```
6 1024
1 2 3 1 4 0
```

**【样例输出 2】**

```
More_and_more_vegetable,what_should_ckj_do?
```

**【数据范围】**

对于 $10\%$ 的数据，$n\le 5$，$a_i,k\le 10$。

对于 $30\%$ 的数据，$n\le 10^3$，$a_i,k\le 10^3$。

对于 $100\%$ 的数据，$2\le n\le 3\times10^5$，$0\le a_i,k\le 10^9$。
