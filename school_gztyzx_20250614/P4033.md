## 问题描述

对于长度为 $n$ 的数组 $s$ 与 $a$，有 $s_1 = a_1$ $+$ $a_2$，$s_2 = a_2$ $+$ $a_3$，……，$s_{n - 1} = a_{n - 1}$ $+$ $a_n$，$s_n = a_n$ $+$ $a_1$。

现在，给你 $s$，请你求出 $a$。

## 输入格式

第一行一个整数 $n$。

第二行 $n$ 个整数，第 $i$ 个整数为 $s_i$。

## 输出格式

$n$ 个整数，第 $i$ 个整数为 $a_i$，若有多解，输出任意一种即可。

```input1
3
8 12 10
```

```output1
3 5 7
```

## 样例解释

$3 + 5 = 8$，$5 + 7 = 12$，$7 + 3 = 10$。

## 数据范围

**本题采用捆绑测试。**

对于 $100\%$ 的数据：

$3 \le n \le 10^6$，且保证 $0<a_i \le 10^9$，所有 $s$ 均可以得到所有元素均为正整数的 $a$。

|          | 数据点范围     | 分值   | $n$        | 特殊性质          |
| ---------- | ---------------- | -------- | -------------- | ------------------- |
| Subtask1 | $1\sim 5$   | $20$ | $n$ 是奇数 | 无                |
| Subtask2 | $6\sim 10$  | $20$ | 无特殊性质   | $a_i\leq 100$ |
| Subtask3 | $11\sim 20$ | $60$ | 无特殊性质   | 无                |
