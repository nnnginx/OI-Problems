# Description
小武有 2 个方程，x | y = A，x + y = B，其中 | 为二进制或符号，x 和 y 是未知数，A 和 B 已知，小武想知道这个方程是否有非负整数解。

# Format

## Input
第一行一个整数 T 表示数据组数。

接下来 T 行，每行两个数 A 和 B。

## Output
T 行，若方程有解输出 Possible，否则输出 Impossible

# Samples

```input1
4
14 14
2 5
0 9
10 10
```

```output1
Possible
Impossible
Impossible
Possible
```

# Explanation
样例一说明：

第一组的一个解为 14 和 0；

第二组无解；

第三组无解；

第四组的一个解为 10 和 0。

# Limitation
对于 30% 的数据，A，B ≤ 1000；

对于 50% 的数据，A，B ≤ $10^9$；

对于 100% 的数据，T ≤ 10，0 ≤ A，B ≤ $10^{18}$。
