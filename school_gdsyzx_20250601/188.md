#### **题目描述**

水仙花数$（Narcissistic number）$也被称为超完全数字不变数$（pluperfect digital invariant, PPDI）$、 自恋数、 自幂数、阿姆斯壮数或阿姆斯特朗数$（Armstrong number）$。 水仙花数是指一个 $n$ 位数$（n≥3 ）$，它的每个位上的数字的$ n$ 次幂之和等于它本身（例如$153$是三位的水仙花数, 因为： $1^3 + 5^3+ 3^3= 153$）。

求$[a..b]$之间的所有水仙花数,如果没有输出'$NO$'。

#### **输入**

输入两个整数$a,b$, 其中$100< a <= b <1000$（即$a,b$都为三位数）。

#### **输出**

输出$a \sim b$之间所有的水仙花数。每行一个，若没有输出$“NO”$。

#### **样例输入**

```
100 500
```

#### **样例输出**

```
153 
370 
371 
407
```
