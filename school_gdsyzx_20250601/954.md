# Description
梯形的面积公式大家都知道吧！如下图： 

![img](./954/file/梯形面积.png)

那么面积：$\frac{(a + b) × h}{2}$。

现已知梯形的上底 a，下底 b 和高 h，求梯形的面积s。

# Format

## Input
输入三个正整数：a、b、h，之间用一个空格隔开。

## Output
输出一个整数，梯形的面积s。

# Samples

```input1
2 3 2
```

```output1
5
```
```input2
4 6 3
```

```output2
15
```

# Limitation

对于 50% 的数据，a、b、h 均小于100；

对于 100% 的数据，a、b、h 均小于1000；

数据保证能够整除。
