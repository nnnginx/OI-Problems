# Description
有一个包含 n 个元素的数组 a[1...n]，然后把这 n 个元素按照如下的次序写在白纸上：

a1 a3 ...... a4 a2

例如：n = 7，a = [3，1，4，1，5，9，2]，那么白纸上的数是：[3，4，5，2，9，1，1]。

现在给出写在白纸的 n 个数，你的任务是还原并输出 a 数组。

# Format

## Input
第一行，一个整数 n，1 ≤ n ≤ 300。

第二行，n 个整数，每个整数范围 [1，$10^9$]。

## Output
一行，n 个整数，表示 a 数组。

# Samples

```input1
11
8 4 3 1 2 7 8 7 9 4 2
```

```output1
8 2 4 4 3 9 1 7 2 8 7
```
