# Description

给出一个数组 a[1...n]，对于 1 ≤ i ≤ n，如果 a[i] 满足下面的条件：

存在两个下标 x 和 y，1 ≤ x < y ≤ n，使得 a[i] = a[x] +  a[x+1] +...+ a[y]。

那么 a[i] 就是“好数”。求 a 数组有多少个“好数”。

# Format

## Input

多组测试数据。

第一行，一个整数 t，表示有 t 组测试数据。1 ≤ t ≤ 1000。

每组测试数据格式如下：

第 1 行，一个整数 n。1 ≤ n ≤ 8000。

第 2 行，n 个整数，第 i 个整数是 a[i]。1 ≤ a[i] ≤ n。

数据保证：所有 t 组测试数据的 n 的总和不超过 8000。

## Output

共 t 行，每行一个整数。

# Samples

```input1
5
9
3 1 4 1 5 9 2 6 5
3
1 1 2
5
1 1 1 1 1
8
8 7 6 5 4 3 2 1
1
1
```

```output1
5
1
0
4
0
```

# Explanation

样例一说明：

第一组测试样例解释：a[3]，a[5]，a[6]，a[8]，a[9] 是“好数”。
