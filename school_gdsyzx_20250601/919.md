# Description

Limak 是只想学习英语字母的小熊。他试图按照字母表顺序从“a”一直读到“z”。你的工作是阻止他犯错，一旦他犯错，你立刻让他停止。你能检查他正确说了多少个首字母吗？

# Format

## Input

一个字符串 S，全部由小写英文字母构成，长度不超过 26。

## Output

一个整数，表示Limak在停止前正确说出的英文字母个数。

# Samples

```input1
abctyf
```

```output1
3
```

```input2
abcdefghijklmnopqrstuvwxyz
```

```output2
26
```

```input3
abcdegfhijk
```

```output3
5
```

# Explanation

样例一说明：

说对了前 3 个字母。

样例二说明：

所有字母都按顺序说对了。
