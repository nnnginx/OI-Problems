# Description
Farmer John 的 N 头奶牛（1 ≤ N ≤ 80000）正处于糟糕头发的日子！由于每头奶牛都对自己凌乱的发型产生了意识，因此 Farmer John 想计算能看到其他奶牛头顶的奶牛的数量。

每头奶牛 i 都有一个指定的高度 h i（1 ≤ hi ≤ 1000000000），并且站成一排面向东方（在我们的图表中向右）。因此，只要前面的这些奶牛严格矮于奶牛 i，那么她可以看到她面前奶牛的头顶（即奶牛 i + 1、i + 2 等）。

奶牛 #4 看不到奶牛的发型

奶牛 #5 可以看到奶牛的发型奶牛 #6

奶牛 #6 根本看不到奶牛！

# Format

## Input
令 ci 表示从奶牛i可以看到其发型的奶牛数量；请计算 c1 到 cN 的总和。对于此示例，所需的答案是 3 + 0 + 1 + 0 + 1 + 0 = 5。

## Output
第 1 行：奶牛数量 N。

第 2 至第 N + 1 行：第 i + 1 行包含一个整数，即奶牛 i 的高度。
