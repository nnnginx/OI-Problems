# A数列

### 【题目描述】

一个简单的数列问题：给定一个长度为n的数列，求这样的三个元素ai,aj,ak的个数，满足ai < aj > ak,且i < j < k.

### 【输入格式】

第一行是一个整数n（1<=n<=50000）

接下来n行，每行一个元素ai（0<=ai<=32767）。

### 【输出格式】

一个数，满足ai < aj >ak(i<j<k)的个数

### 【样例输入】

```
5
1
2
3
4
1
```

### 【样例输出】

```
6
```

### 【数据范围】

对于30%的输入数据，有$n<=200$；

对于80%的输入数据，有$n<=10000$。
