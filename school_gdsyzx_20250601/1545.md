# Description

给你云朵的个数 $N$，再给你 $M$ 个关系，表示哪些云朵可以连在一起。现在小杉要把一些云朵连在一起，做成 $K$ 个棉花糖，一个棉花糖最少要用掉一朵云，小杉想知道他怎么连，花费的代价最小。

# Format

## Input
每组测试数据的第一行有三个数 $N，M，K$。$ 1 ≤ N ≤ 1000；1 ≤ M ≤ 10000；1 ≤ K ≤ 10$。

接下来 $M$ 个数每行三个数 $X，Y，L$，表示 $X$ 云和 $Y$ 云可以通过 $L$ 的代价连在一起。

## Output
对每组数据输出一行，仅有一个整数，表示最小的代价。如果怎么连都连不出 $K$ 个棉花糖，请输出 $No\ Answer$。

```input1
3 1 2
1 2 1
```

```output1
1
```

```input2
3 1 1
1 2 1
```

```output2
No Answer
```
