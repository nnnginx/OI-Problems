# Description
周末黄子韬想去剧场看演出，但他没有票。这时，救世主李国豪出现了，他慷慨地愿意卖给黄子韬一些票。李国豪手上共有 n 张票，但每张票的费用都不一样，贪心的黄子韬想要得到尽可能多的票，但又想花费最少，慷慨的李国豪愿意给连续的 m 张票。黄子韬希望你能帮助他在花钱范围内取得最大的票数。

# Format

## Input
第一行是 2 个整数 n，f，其中 2 ≤ n ≤ 1000000，表示票的数目；10 ≤ f ≤ 10000，表示黄子韬身上的钱。

接下来的一行，有 n 个整数 a（1 ≤ a ≤ 30），表示每一张票的票价。

## Output
仅一行整数 m，表示黄子韬能得连续的最大票数。

# Samples

```input1
5 10
2 3 1 6 7
```

```output1
3
```

# Limitation
对于 50% 的数据，2 ≤ n ≤ 10000；

对于 100% 的数据，2 ≤ n ≤ 1000000。
