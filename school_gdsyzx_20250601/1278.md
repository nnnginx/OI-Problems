# Description
排列与组合是常用的数学方法，桐桐刚刚学会了全排列，就想试试组合，组合就是从 n 个元素中抽出 r 个元素（不分顺序且 r ≤ n），我们可以简单地将 n 个元素理解为自然数 1，2，...，n，从中任取 r 个数。

# Format

## Input
两个整数 n 和 r（1 ≤ r ≤ n ≤ 20）。

## Output
输出所有的组合，每一个组合占一行且其中的元素按由小到大的顺序排列，所有的组合也按字典顺序。

# Samples

```input1
5 3
```

```output1
1 2 3
1 2 4
1 2 5
1 3 4
1 3 5
1 4 5
2 3 4
2 3 5
2 4 5
3 4 5
```
