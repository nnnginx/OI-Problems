# Can you answer on these queries III

### 【问题描述】

给定长度为N的数量A，以及M条指令，每条指令可能是以下两种之一：

1. "1 x y",查询区间[x,y]中的最大连续字段和。
2. “2 x y",把A[x] 改为y。

对于每个查询指令，输出一个整数表示答案。即$ max_{x<=l<=r<=y}{\sum_{i=l}^r A[i]}$

# 【输入格式】

第一行两个整数N,M。

第二行N个整数A[i]。

接下来M行每行3个整数k，x，y, k=1表示查询（此时如果x>y,请交换x,y),k=2表示修改。

# 【数据范围】

N<=500000 , M<=100000

# 【输出格式】

对于每个查询指令输出一个整数表示答案

每个答案占一行

# 【输入样例】

```
5 3
1 2 -3 4 5
1 2 3
2 2 -1
1 3 2
```

# 【样例输出】

```
2
-1
```
