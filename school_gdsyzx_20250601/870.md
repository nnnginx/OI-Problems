# Description
琳琳刚学会使用数组，老师就给了他一个这样的问题：有两个数组 A[1...n] 和 B[1...n] 他们之间存在数学关系式是： B[i] = （A[1] + A[2] + ... + A[i]）/ i（注意是整除），其中 1 ≤ i ≤ n。

例如： A 数组是： 1，3，2，6，8。

那么 B 数组是：

![img](./870/file/0000661数列.png)

可以算出 B 数组是： 1，2，2，3，4。

# Format

## Input
输入数据有两行：

第一行，一个整数 n。 1 ≤ n ≤ 100。

第二行，包含 n 个整数，第 i 个整数是 B[i]。1 ≤ B[i] ≤ $10^9$。

## Output
输出数据仅一行，共 n 个整数，每个数之间用一个空格分开，第 i 个整数表示 A[i]。

数据保证 A[i] 一定是整数，而且保证 1 ≤ A[i] ≤ $10^9$。

# Samples

```input1
4
3 2 3 5
```

```output1
3 1 5 11
```
