# Description
一家银行计划安装一台提款机。这台机器能够为要求的现金数额提供适当的金钱。机器使用 N 种不同的钞票面额， 对于每种面额 Dk，机器会有 nk 张钞票供应。例如：

N = 3，n1 = 10，D1 = 100，n2 = 4，D2 = 50，n3 = 5，D3 = 10，表示机器有 10 张 100 元的钞票，4 张 50 元的钞票，5 张 10 元的钞票。

请你根据提款机收到的应交付的现金请求量，编写一个程序，根据机器的可供应钞票，计算能够提供的最接近（即小于或等于现金请求量）的最大金额。

# Format

## Input
一行，分别为 cash，N，n1，D1，n2，D2，...，nN，DN，其中cash为现金请求量，其余含义如题所述。其中 0 ≤ cash ≤ 100000，0 ≤ N ≤ 10，0 ≤ nk ≤ 1000，1 ≤ Dk ≤ 1000。

## Output
一行，为最终能提取的最大金额。

# Samples

```input1
735 3 4 125 6 5 3 350
```

```output1
735
```
```input2
633 4 500 30 6 100 1 5 0 1
```

```output2
630
```
