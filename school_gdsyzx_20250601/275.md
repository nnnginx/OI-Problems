#### **题目描述**

$402$班准备在学校的体艺节里进行合唱演出，为了准备这一次演出，老师从班里挑出$n$名学生$（n$为奇数）并为她们安排合唱队形。合唱队形是中间高两端低，老师是这样安排队形的：先让挑出来的所有同学按高个儿在前的顺序排成一队。然后，最高的那位同学单独站出来，这是合唱队形的中心，再让第二位同学站在她的左手边，让第三位同学站在她的右手边，再依次向两端安排其他人…… 给定$402$班所有学生的身高，请输出她们站成合唱队形之后的身高顺序。 输入格式： 第一行是一个整数$n$，表示合唱队的总人数，已知$n$为奇数，且$n<500$ 。 第二行是$n$个整数，表示以厘米为单位的所有人的身高。

#### **输入**

第一行为$n$，表示班里有$n$个同学，第二行$n$个整数，表示$n$个学生的具体身高

#### **输出**

$n$个整数，表示她们按老师的要求站成合唱队形之后的身高顺序。

#### **样例输入**

```
7 
154 160 157 162 159 152 163
```

#### **样例输出**

```
152 157 160 163 162 159 154
```
