#### **题目描述**

输入 $1$ 个整数$ N$，输出 $N$ 行的数字三角形。第 $1$ 行 $1$ 个$’1’$，第 $2$ 行 $3$ 个$’2’$，第 $3$ 行 $5$ 个$’3’$，…。例如：$N=4$ 时：

___1

__121

_12321

1234321

(注：上面使用下划线’_’表示空格，以避免看不清造成误解)

#### **输入**

第一行 $1$ 个正整数：$N$，范围在$[1,9]$。

#### **输出**

$N$ 行的数字三角形。

#### **样例输入**

```
3
```

#### **样例输出**

```
  1
 121
12321
```
