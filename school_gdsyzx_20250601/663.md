# 不要倍数

### 题目描述

神 $zhy$ 要你求 $1$ 到 $n$ 之间（闭区间）所有整数中除了 $a$ 和 $b$ 的倍数之外的整数之和。

### 输入格式

一行的三个整数 $N,A,B$

### 输出格式

一行一个整数表示答案

### 样例

#### 输入 #1

```cpp
10 3 5
```

#### 输出 #1

```cpp
22
```

#### 输入 #2

```cpp
1000000000 314 159
```

#### 输出 #2

```cpp
495273003954006262
```

### 数据范围

$ 1\ \leq N, A,B \leq\ 10^9 $
