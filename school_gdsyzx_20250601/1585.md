# Description
在漫长的挤奶过程中，奶牛Bessie喜欢盯着自家谷仓的窗外看街对面的两块巨大的矩形广告牌，上面写着“Farmer Alex的美味苜蓿”和“Farmer Greg的大谷物”。在Bessie看来，广告牌上这两种奶牛饲料产品的照片比农场里的草要美味得多。

有一天，当Bessie凝视窗外时，她惊恐地看到一辆巨大的矩形卡车停在街对面。

卡车的侧面有一则“Farmer Smith的上等牛排”的广告，Bessie对此并不十分理解，但她最担心的是，卡车可能会挡住她最喜欢的两个广告牌的视线。

考虑到两个广告牌的位置和卡车的位置，请计算仍然可见的两个广告牌的总面积。

注意：卡车可能既不遮挡，也不遮挡，或者只遮挡其中一个广告牌。

# Format

## Input
第一行输入包含四个空格分隔的整数：$x_1，y_1，x_2，y_2$，其中（$x_1，y_1$）和（$x_2，y_2$）是Bessie视野中第一块广告牌左下角和右上角的坐标。

下一行还包含四个整数，类似地指定了第二个公告牌的左下角和右上角。

第三行也是最后一行输入包含四个整数，指定卡车的左下角和右上角。

所有坐标都在 $-1000000000$ 到 $+1000000000000$ 之间。这两个广告牌之间保证没有任何正面重叠区域。

## Output
请输出仍然可见的两个广告牌的总面积。

# Samples

```input1
1 2 3 5
6 0 10 4
2 1 8 3
```

```output1
17
```

# Explanation
样例一说明：

第一块广告牌的 5 个单位面积和第二块广告牌的 12 个单位面积仍然可见。
