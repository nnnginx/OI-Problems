# Description
琳琳正在研究一个只有加法和乘法的算术式，假设她可以合法的任意加括号，你能帮她能求出这个
算术式可能得到的最大值和最小值吗？参与运算的数据项都是正整数，数据项的个数不超过 10（即每个数据项的大小为 1 到 10）。

# Format

## Input
输入数据只有一行，是一个合法的算术式，两项之间都有一个空格。

## Output
输出数据有两行，第一行为能得到的最大值，第二行为能得到的最小值。

# Samples

```input1
2 + 3 * 5 + 4 =
```

```output1
45
21
```

# Explanation
样例一说明：

（2 + 3） × （5 + 4）= 45

2 + 3 × 5 + 4 = 21
