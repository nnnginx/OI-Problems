# AT_kupc2021_c Gacha

## 题目描述

一条直线上有 $N$ 个游戏机（Gacha）和 $N$ 枚硬币，第 $i$ 个游戏机的坐标是 $A_i$，第 $j$ 枚硬币的坐标是 $B_j$，这些坐标均为正整数。你现在在坐标 $0$ 点，你可以在直线上自由移动。

- 向着硬币的方向走，经过时自动捡起硬币。
- 向着游戏机的方向走，游玩游戏机时消费一枚硬币。

你希望把 $N$ 个游戏机都游玩一遍。请求出游玩所有的游戏机所需要移动的最小距离。

## 输入格式

根据以下的格式从标准输入进行输入。

> $N$  
> $A_1 A_2 \dots A_N$  
> $B_1 B_2 \dots B_N$

## 输出格式

输出一行，为游玩所有游戏机所需要移动的最短距离。

## 输入输出样例 #1

### 输入 #1

```
4
1 6 7 12
3 5 10 11
```

### 输出 #1

```
21
```

## 输入输出样例 #2

### 输入 #2

```
2
1 2
1 1000000000
```

### 输出 #2

```
1999999998
```

## 说明/提示

-   $1 \leq N \leq 10^5$
-   $1 \leq A_i \leq 10^9$
-   $A_i \lt A_{i+1}$ $(1 \leq i \leq N-1)$
-   $1 \leq B_j \leq 10^9$
-   $B_j \lt B_{j+1}$ $(1 \leq j \leq N-1)$
