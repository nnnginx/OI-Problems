# AT_abc172_d [ABC172D] Sum of Divisors

## 题目描述

### 题意翻译
- $f(x)$ 表示正整数 $x$ 的因数个数
- 现在给你一个正整数 $ N $ ，让你求出 $  \sum_{K=1}^N\ K\times\ f(K) $  。
----------


-------------

## 输入格式

无

## 输出格式

- 共一行，输出 $ \sum_{K=1}^N\ K\times\ f(K) $ 的结果
- 最后别忘了输出回车

## 输入输出样例 #1

### 输入 #1

```
4
```

### 输出 #1

```
23
```

## 输入输出样例 #2

### 输入 #2

```
100
```

### 输出 #2

```
26879
```

## 输入输出样例 #3

### 输入 #3

```
10000000
```

### 输出 #3

```
838627288460105
```

## 说明/提示

-  $ 1\ \leq\ N\ \leq\ 10^7 $
- $f(1)=1 $ ,  $ f(2)=2 $ ,  $ f(3)=2 $ ,  $ f(4)=3 $ , 所以答案为  $ 1\times\ 1\ +\ 2\times\ 2\ +\ 3\times\ 2\ +\ 4\times\ 3\ =23 $ 。
