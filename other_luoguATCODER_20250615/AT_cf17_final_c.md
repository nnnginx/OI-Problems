# AT_cf17_final_c Time Gap

## 题目描述

### 题目大意
给定$n$个数$d$，每个$d$都可以选择变成$24-d$或者保留数值，求任意两个数之间差的最小值中的最大值。

## 输入格式

第一行：$n$\
第二行：$d_1$,$d_2$……$d_n$

## 输出格式

任意两个数之间差的最小值中的最大值。

## 输入输出样例 #1

### 输入 #1

```
3
7 12 8
```

### 输出 #1

```
4
```

## 输入输出样例 #2

### 输入 #2

```
2
11 11
```

### 输出 #2

```
2
```

## 输入输出样例 #3

### 输入 #3

```
1
0
```

### 输出 #3

```
0
```

## 说明/提示

1$\leq$$n$$\leq$50\
0$\leq$$d_i$$\leq$12
