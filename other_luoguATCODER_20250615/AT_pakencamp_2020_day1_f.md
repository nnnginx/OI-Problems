# AT_pakencamp_2020_day1_f Fibonaccyan

## 题目描述

给定正整数 $P$，求能被 $P$ 整除的最小斐波那契数。输出答案是斐波那契数列中的第几个数。如果无解，输出 `-1`。

## 输入格式

输入一行一个正整数 $P$。

## 输出格式

输出一行一个正整数，表示答案。

## 输入输出样例 #1

### 输入 #1

```
3
```

### 输出 #1

```
4
```

## 输入输出样例 #2

### 输入 #2

```
10
```

### 输出 #2

```
15
```

## 输入输出样例 #3

### 输入 #3

```
3000
```

### 输出 #3

```
1500
```

## 说明/提示

对于所有数据，$1\leqslant P\leqslant 3000$。
