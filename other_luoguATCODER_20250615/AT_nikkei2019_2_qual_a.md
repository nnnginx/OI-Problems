# AT_nikkei2019_2_qual_a Sum of Two Integers

## 题目描述

[problemUrl]: https://atcoder.jp/contests/nikkei2019-2-qual/tasks/nikkei2019_2_qual_a

合計が $ N $ となるように相異なる $ 2 $ つの正整数を選ぶ方法は何通りあるでしょうか (順序は考慮しません)。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

答えを出力せよ。

## 输入输出样例 #1

### 输入 #1

```
4
```

### 输出 #1

```
1
```

## 输入输出样例 #2

### 输入 #2

```
999999
```

### 输出 #2

```
499999
```

## 说明/提示

### 制約

- $ 1\ ≦\ N\ ≦\ 10^6 $
- $ N $ は整数である。

### Sample Explanation 1

合計が $ 4 $ となるように相異なる $ 2 $ つの正整数を選ぶ方法は、$ 1 $ と $ 3 $ を選ぶ $ 1 $ 通りのみです。($ 3 $ と $ 1 $ を選ぶことはこれと区別しません。)
