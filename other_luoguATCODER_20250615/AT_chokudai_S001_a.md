# AT_chokudai_S001_a 最大値

## 题目描述

请输出排列 $a$ 的最大值。

## 输入格式

第一行一个正整数 $n$，表示排列中有 $n$ 个数。

第二行 $n$ 个正整数，表示排列 $a$。

## 输出格式

一行一个正整数，表示排列 $a$ 中的最大值。

## 输入输出样例 #1

### 输入 #1

```
5
3 1 5 4 2
```

### 输出 #1

```
5
```

## 输入输出样例 #2

### 输入 #2

```
6
1 2 3 4 5 6
```

### 输出 #2

```
6
```

## 输入输出样例 #3

### 输入 #3

```
7
7 6 5 4 3 2 1
```

### 输出 #3

```
7
```

## 输入输出样例 #4

### 输入 #4

```
20
19 11 10 7 8 9 17 18 20 4 3 15 16 1 5 14 6 2 13 12
```

### 输出 #4

```
20
```

## 说明/提示

对于 $100\%$ 的数据，$1 \le n \le 100$。
