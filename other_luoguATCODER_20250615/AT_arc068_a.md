# AT_arc068_a [ABC053C] X: Yet Another Die Game

## 题目描述

一个骰子，初始的时候任意的一个面朝上，接下来每一轮可以翻转骰子到相邻的一面，并获得此面的得分（那一面的数值即是得分）。问至少要几轮才可以获得够 $x$ 分。

**注**：所有数均在$longlong$范围内。

## 输入格式

一个数 $x$ 。

## 输出格式

输出也是一个数，即最少的轮数。

## 输入输出样例 #1

### 输入 #1

```
7
```

### 输出 #1

```
2
```

## 输入输出样例 #2

### 输入 #2

```
149696127901
```

### 输出 #2

```
27217477801
```
