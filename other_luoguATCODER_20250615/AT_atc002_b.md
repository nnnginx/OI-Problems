# AT_atc002_b n^p mod m

## 题目描述

求 $n^p$ $mod$ $m$ 的值

## 输入格式

一行，为整数 n,m,p(注意顺序)

## 输出格式

一行，为 $n^p$ $mod$ $m$ 的值

## 输入输出样例 #1

### 输入 #1

```
12 15 7
```

### 输出 #1

```
3
```

## 输入输出样例 #2

### 输入 #2

```
123456789 234567894 6574837563712
```

### 输出 #2

```
120678297
```

## 说明/提示

对于100%的数据 $1≤n,m≤10^9$ $1≤p≤10^{14}$
