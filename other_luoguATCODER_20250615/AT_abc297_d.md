# AT_abc297_d [ABC297D] Count Subtractions

## 题目描述

给你 $A,B$ 两个数，进行以下操作 :

1. 如果 $A>B$,那么 $A=A-B$ 。
1. 如果 $A<B$,那么 $B=B-A$ 。

请输出经过多少次操作后 $A=B$ 。

## 输入格式

两个整数。
>A B

## 输出格式

答案，一个整数。

by [X_Z_Y](https://www.luogu.com.cn/user/645323)

## 输入输出样例 #1

### 输入 #1

```
3 8
```

### 输出 #1

```
4
```

## 输入输出样例 #2

### 输入 #2

```
1234567890 1234567890
```

### 输出 #2

```
0
```

## 输入输出样例 #3

### 输入 #3

```
1597 987
```

### 输出 #3

```
15
```
