# AT_ttpc2019_i I hate P

## 题目描述

已知 $f(x)=\left\{\begin{matrix}
  f\left(\dfrac{x}{P}\right)&\text{if}\ x\equiv 0\pmod{P}  \\
  x&\text{otherwise}
\end{matrix}\right.$。求 $\text{ans}=\left(\prod\limits_{i=L}^{R}{f(i)}\right)\pmod{Q}$，$P,Q,L,R$ 给定。

## 输入格式

```
P Q
L R
```

## 输出格式

```
ans
```

## 输入输出样例 #1

### 输入 #1

```
2 5
6 9
```

### 输出 #1

```
4
```
