# AT_abc254_a [ABC254A] Last Two Digits

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc254/tasks/abc254_a

$ 100 $ 以上の整数 $ N $ が与えられます。$ N $ の下 $ 2 $ 桁を出力してください。

ただし、$ N $ の下 $ 2 $ 桁とは十の位と一の位をこの順に並べたものを言います。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

答えを出力せよ。

## 输入输出样例 #1

### 输入 #1

```
254
```

### 输出 #1

```
54
```

## 输入输出样例 #2

### 输入 #2

```
101
```

### 输出 #2

```
01
```

## 说明/提示

### 制約

- $ 100\ \le\ N\ \le\ 999 $
- $ N $ は整数である。

### Sample Explanation 1

$ 254 $ の下 $ 2 $ 桁は $ 54 $ であるため、$ 54 $ を出力します。

### Sample Explanation 2

$ 101 $ の下 $ 2 $ 桁は $ 01 $ であるため、$ 01 $ を出力します。
