# AT_abc157_a [ABC157A] Duplex Printing

## 题目描述

高桥想要打印一有 $N$ 面的文档，因为他用的是双面打印机，所以一张纸可以打印 $2$ 面，求他需要几张纸？

## 输入格式

仅一个整数，表示文档的面数 $N$。

## 输出格式

仅一个整数，表示高桥需要几张纸。

## 输入输出样例 #1

### 输入 #1

```
5
```

### 输出 #1

```
3
```

## 输入输出样例 #2

### 输入 #2

```
2
```

### 输出 #2

```
1
```

## 输入输出样例 #3

### 输入 #3

```
100
```

### 输出 #3

```
50
```

## 说明/提示

$N$ 为整数，

$1\leq N \leq100$。
