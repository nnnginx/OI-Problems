# AT_abc111_a [ABC111A] AtCoder Beginner Contest 999

## 题目描述

某位OIer在练习写字，但是他不小心把1和9写反了，也就是说他把1写成了9,9写成了1。现在请你帮他改过来。

## 输入格式

输入他写的字$n$，只包含1和9。

## 输出格式

输出一行，是修正好的字。

## 输入输出样例 #1

### 输入 #1

```
119
```

### 输出 #1

```
991
```

## 输入输出样例 #2

### 输入 #2

```
999
```

### 输出 #2

```
111
```

## 说明/提示

$111 \le n \le 999$
