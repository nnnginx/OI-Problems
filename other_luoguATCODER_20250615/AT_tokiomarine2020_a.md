# AT_tokiomarine2020_a Nickname

## 题目描述

无

## 输入格式

一个字符串 $S(3\le|S|\le20)$

## 输出格式

将这个字符串中的任意连续三个字符输出，别忘了换行。

## 输入输出样例 #1

### 输入 #1

```
takahashi
```

### 输出 #1

```
tak
```

## 输入输出样例 #2

### 输入 #2

```
naohiro
```

### 输出 #2

```
nao
```
