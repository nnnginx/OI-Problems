# AT_pakencamp_2021_day2_d 選択問題の正答はすべて同じ選択肢で… / 选择题正确答案都是同一个选项

## 题目描述

某次考试共有 $N$ 道题，$M$ 个选项，试卷上印着“这次考试选择题正确答案都是同一个选项”。

但是小 A 没看到这句话，他每一道题的答案分别写的是 $A_i$（$1 \le A_i \le M$）。

请计算出小 A 可能的最低和最高得分（每道题 $1$ 分）。

## 输入格式

第一行，两个正整数 $N,M$。

第二行，$N$ 个正整数 $A_1,A_2,\ldots,A_N$。

## 输出格式

输出两个非负整数，第一个整数代表小 A 可能的最低得分，第二个整数代表小 A 可能的最高得分。

## 输入输出样例 #1

### 输入 #1

```
3 2
1 1 2
```

### 输出 #1

```
1 2
```

## 输入输出样例 #2

### 输入 #2

```
4 5
5 1 1 5
```

### 输出 #2

```
0 2
```

## 说明/提示

### 样例解释

第一组样例，小 A 选了 $2$ 个 $1$ 和 $1$ 个 $2$。如果正确答案是 $1$，那么小 A 得 $2$ 分。如果正确答案是 $2$，那么小 A 得 $1$ 分。

第二组样例，如果正确答案是 $1$ 或 $5$，小 A 得 $2$ 分，否则得 $0$ 分。

### 数据范围

$1 \le N \le 2 \times {10}^5,\ 1 \le M \le {10}^9$
