# AT_abc081_a [ABC081A] Placing Marbles

## 题目描述

读入一个字符串s，求它的数字和是多少

## 输入格式

一行，一个字符串s

## 输出格式

一行，一个数sum，表示s字符串的数字和（个各位数相加的和）。
### **输入样例**
```
101
```
### **输出样例**
```
2
```

## 输入输出样例 #1

### 输入 #1

```
101
```

### 输出 #1

```
2
```

## 输入输出样例 #2

### 输入 #2

```
000
```

### 输出 #2

```
0
```
