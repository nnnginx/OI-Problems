# AT_abc051_b [ABC051B] Sum of Three Integers

## 题目描述

有两个整数 $K$ , $S$

求有几种方案使得三个非负整数 $X$ , $Y$ , $Z$ 之和= $S$ 并且均 $\leq K$

## 输入格式

无

## 输出格式

一个整数，表示方案数

## 输入输出样例 #1

### 输入 #1

```
2 2
```

### 输出 #1

```
6
```

## 输入输出样例 #2

### 输入 #2

```
5 15
```

### 输出 #2

```
1
```
