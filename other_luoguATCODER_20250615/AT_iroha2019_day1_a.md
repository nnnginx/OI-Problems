# AT_iroha2019_day1_a 一問目

## 题目描述

[problemUrl]: https://atcoder.jp/contests/iroha2019-day1/tasks/iroha2019_day1_a

文字列 $ S $ が与えられます。$ S $ の頭文字を出力してください。

## 输入格式

文字列 $ S $ が $ 1 $ 行に与えられます。

> $ S $

## 输出格式

$ S $の頭文字を出力してください。最後に改行を入れてください。

## 输入输出样例 #1

### 输入 #1

```
Iroha
```

### 输出 #1

```
I
```

## 输入输出样例 #2

### 输入 #2

```
AtCoder
```

### 输出 #2

```
A
```

## 说明/提示

### 制約

- $ 1\ \leq\ |S|\ \leq\ 100 $
- $ S $ は半角英数字からなる文字列
