# AT_abc230_e [ABC230E] Fraction Floor Sum

## 题目描述

求 $\large{\sum\limits_{i=1}^N ⌊\frac{N}{i}⌋}$ 的值

## 输入格式

一行一个整数 $N$

## 输出格式

一行一个整数，你的答案

## 输入输出样例 #1

### 输入 #1

```
3
```

### 输出 #1

```
5
```

## 输入输出样例 #2

### 输入 #2

```
10000000000
```

### 输出 #2

```
231802823220
```
