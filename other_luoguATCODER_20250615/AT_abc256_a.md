# AT_abc256_a [ABC256A] 2^N

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc256/tasks/abc256_a

$ N $ が与えられます。$ 2^N $ を出力してください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ N $

## 输出格式

答えを出力せよ。

## 输入输出样例 #1

### 输入 #1

```
3
```

### 输出 #1

```
8
```

## 输入输出样例 #2

### 输入 #2

```
30
```

### 输出 #2

```
1073741824
```

## 说明/提示

### 制約

- $ 0\ \leq\ N\ \leq\ 30 $
- $ N $ は整数である

### Sample Explanation 1

$ 2^3=8 $ です。
