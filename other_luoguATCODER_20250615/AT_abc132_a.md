# AT_abc132_a [ABC132A] Fifty-Fifty

## 题目描述

有长度为 $4$ 的由大写英文字母构成的串 $S$ ，问其是否恰好含有两种不同字母且这两种字母都恰好出现两次。

## 输入格式

$S$ 。

## 输出格式

如果符合要求输出`Yes`，否则输出`No`。

## 输入输出样例 #1

### 输入 #1

```
ASSA
```

### 输出 #1

```
Yes
```

## 输入输出样例 #2

### 输入 #2

```
STOP
```

### 输出 #2

```
No
```

## 输入输出样例 #3

### 输入 #3

```
FFEE
```

### 输出 #3

```
Yes
```

## 输入输出样例 #4

### 输入 #4

```
FREE
```

### 输出 #4

```
No
```
