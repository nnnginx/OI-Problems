# AT_abc059_b [ABC059B] Comparison

## 题目描述

给定两个正整数 $a, b$，比较他们的大小。

## 输入格式

输入有两行，第一行为数 $a$，第二行为数 $b$。

## 输出格式

如果 $a > b$，输出 "GREATER"；如果 $a = b$，输出 "EQUAL"，如果 $a < b$，输出 "LESS"。

## 输入输出样例 #1

### 输入 #1

```
36
24
```

### 输出 #1

```
GREATER
```

## 输入输出样例 #2

### 输入 #2

```
850
3777
```

### 输出 #2

```
LESS
```

## 输入输出样例 #3

### 输入 #3

```
9720246
22516266
```

### 输出 #3

```
LESS
```

## 输入输出样例 #4

### 输入 #4

```
123456789012345678901234567890
234567890123456789012345678901
```

### 输出 #4

```
LESS
```

## 说明/提示

$1 \le a, b \le 10^{100}$，保证 $a, b$ 均无前导零。
