# AT_abc236_a [ABC236A] chukodai

## 题目描述

无

## 输入格式

输入一个字符串 $S(2\le|S|\le10)$和两个正整数 $a,b(1\le a,b\le|s|)$。

## 输出格式

将这个字符串的第 $a$ 和第 $b$ 个字符调换后输出。

**记得换行！！**

## 输入输出样例 #1

### 输入 #1

```
chokudai
3 5
```

### 输出 #1

```
chukodai
```

## 输入输出样例 #2

### 输入 #2

```
aa
1 2
```

### 输出 #2

```
aa
```

## 输入输出样例 #3

### 输入 #3

```
aaaabbbb
1 8
```

### 输出 #3

```
baaabbba
```
