# AT_abc133_a [ABC133A] T or T

## 题目描述

无

## 输入格式

输入三个数 $N$,$A$,$B$。

## 输出格式

输出 $N* A$ 和 $B$ 中最小的那个数。如果相同就随便输出一个。

**记得换行。**

## 输入输出样例 #1

### 输入 #1

```
4 2 9
```

### 输出 #1

```
8
```
