# AT_abc169_a [ABC169A] Multiplication 1

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc169/tasks/abc169_a

$ A\ \times\ B $ を求めてください。

## 输入格式

入力は以下の形式で標準入力から与えられる。

> $ A $ $ B $

## 输出格式

値 $ A\ \times\ B $ を整数として出力せよ。

## 输入输出样例 #1

### 输入 #1

```
2 5
```

### 输出 #1

```
10
```

## 输入输出样例 #2

### 输入 #2

```
100 100
```

### 输出 #2

```
10000
```

## 说明/提示

### 制約

- $ 1\ \leq\ A\ \leq\ 100 $
- $ 1\ \leq\ B\ \leq\ 100 $
- 入力は全て整数である。

### Sample Explanation 1

$ 2\ \times\ 5\ =\ 10 $ です。
