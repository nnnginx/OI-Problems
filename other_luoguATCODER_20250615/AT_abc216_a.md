# AT_abc216_a [ABC216A] Signed Difficulty

## 题目描述

给你一个实数 $X.Y$ ，其中 $Y$ 是一个数码。

请你按如下格式输出：

- 若 $0 \le Y \le 2$ ，输出 $X-$ ；
- 若 $3 \le Y \le 6$ ，输出 $X$ ；
- 若 $7 \le Y \le 9$ ，输出 $X+$ 。

---

## 输入格式

输入一个实数 $X.Y$ 。

---

## 输出格式

输出答案。

$\text{Translated by @nr0728.}$

## 输入输出样例 #1

### 输入 #1

```
15.8
```

### 输出 #1

```
15+
```

## 输入输出样例 #2

### 输入 #2

```
1.0
```

### 输出 #2

```
1-
```

## 输入输出样例 #3

### 输入 #3

```
12.5
```

### 输出 #3

```
12
```

## 说明/提示

- $1 \le X \le 15$
- $0 \le Y \le 9$
- $X$ 和 $Y$ 都是整数。
