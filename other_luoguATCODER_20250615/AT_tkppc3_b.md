# AT_tkppc3_b 鰻と忖度

## 题目描述

给出一个整数 $n$，你需要判断它能否被 $6$ 或 $11$ 整除。

## 输入格式

一行一个整数 $n(0 < n <10^{200000})$。

## 输出格式

两行，第一行表示 $n$ 能否被 $6$ 整除，第二行表示 $n$ 能否被 $11$ 整除。如果可以的话输出 `yES`，否则输出 `nO`。

## 输入输出样例 #1

### 输入 #1

```
11
```

### 输出 #1

```
nO
yES
```
