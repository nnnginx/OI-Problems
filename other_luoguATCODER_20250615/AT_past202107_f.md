# AT_past202107_f 双重预订

## 题目描述

你现在需要参加 $n$ 个在线会议，第 $i$ 个在线会议会在 $d_i$ 天，从 $s_i$ 点开到 $t_i$ 点。你再次决定编写程序，帮你判断是否有会议的时间是重复的。如果第 $a$ 个会议的开始时间与第 $b$ 个会议的结束时间相同，则不视这两个会议的时间重复（可参考输入输出样例 $2$）。

## 输入格式

输入 $(n+1)$ 行。第一行输入一个整数 $n$，接下来的 $n$ 行中，按照 $i=1,2,...,n$ 的顺序依次输入 $d_i,s_i$ 和 $t_i$。

## 输出格式

输出一行一个字符串。如果会议时间有重复请输出`Yes`；否则请输出`No`。

## 输入输出样例 #1

### 输入 #1

```
3
1 10 15
1 11 12
1 14 16
```

### 输出 #1

```
Yes
```

## 输入输出样例 #2

### 输入 #2

```
3
1 20 22
1 22 24
2 0 2
```

### 输出 #2

```
No
```

## 输入输出样例 #3

### 输入 #3

```
5
2 0 24
1 0 10
3 0 1
1 12 23
3 22 24
```

### 输出 #3

```
No
```

## 说明/提示

### 数据规模与约定

对于全部测试点，数据保证：

- $1 \le n \le 10^5$；
- $1 \le d_i \le 10^5$；
- $1 \le s_i < t_i \le 24$；
- 所有输入均为整数。

### 样例解释 #1
第一次和第二次会议重叠，第一次和第三次会议也重叠。

### 样例解释 #2
第一次会议之后紧接着是第二次会议，第二次会议之后紧接着是第三次会议，但我们不认为它们是重叠的。
