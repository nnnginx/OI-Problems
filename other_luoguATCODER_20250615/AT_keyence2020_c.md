# AT_keyence2020_c Subarray Sum

## 题目描述

输入三个整数，N，K，S。
请你找到一个N个元素的整数序列，其中每一个元素在1，10的9次方之间。这个序列满足K个子序列的和为S。

## 输入格式

无

## 输出格式

无

## 输入输出样例 #1

### 输入 #1

```
4 2 3
```

### 输出 #1

```
1 2 3 4
```

## 输入输出样例 #2

### 输入 #2

```
5 3 100
```

### 输出 #2

```
50 50 50 30 70
```
