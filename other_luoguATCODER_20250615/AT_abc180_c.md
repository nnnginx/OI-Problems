# AT_abc180_c [ABC180C] Cream puff

## 题目描述

### 题目大意：

给定一个数 $n\ (1 \leq n \leq 10^{12})$，从小到大输出它的因数，注意：包括 $1$ 和 $n$。

## 输入格式

一个正整数 $n$。

## 输出格式

$n$ 的因数，输出完一个就换行。

## 输入输出样例 #1

### 输入 #1

```
6
```

### 输出 #1

```
1
2
3
6
```
