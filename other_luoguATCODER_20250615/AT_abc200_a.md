# AT_abc200_a Century

## 题目描述

给定一个年份 $N$，求它在哪个世纪。

> 什么是世纪？
>
> 一个世纪是一个 $100$ 年的时期。第 $1$ 个世纪由公元 $1$ 年到公元 $100$ 年组成，第二个由公园 $101$ 年到公元 $200$ 年组成，以此类推。

## 输入格式

一行一个正整数 $N$。

## 输出格式

一行一个正整数，代表它在哪个世纪。

## 输入输出样例 #1

### 输入 #1

```
2021
```

### 输出 #1

```
21
```

## 输入输出样例 #2

### 输入 #2

```
200
```

### 输出 #2

```
2
```

## 说明/提示

数据范围：$1 \le N \le 3000$。
