# AT_abc318_a [ABC318A] Full Moon

## 题目描述

**题意简述**

满月第一次出现是在第 $M$ 天，之后每个 $P$ 天出现一次。问在 $N$ 天以内有多少次满月。

## 输入格式

一行 $3$ 个整数 $N,M,P$。

## 输出格式

一个整数，如题意。

## 输入输出样例 #1

### 输入 #1

```
13 3 5
```

### 输出 #1

```
3
```

## 输入输出样例 #2

### 输入 #2

```
5 6 6
```

### 输出 #2

```
0
```

## 输入输出样例 #3

### 输入 #3

```
200000 314 318
```

### 输出 #3

```
628
```

## 说明/提示

- $1 \le N \le 2 \times 10^5$
- $1 \le M \le P \le 2 \times 10^5$

**样例一解释**

有 $3$ 次，天数分别为 $3,8,13$。
