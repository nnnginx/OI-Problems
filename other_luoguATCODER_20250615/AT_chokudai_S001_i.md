# AT_chokudai_S001_i 和がNの区間

## 题目描述

[problemUrl]: https://atcoder.jp/contests/chokudai_S001/tasks/chokudai_S001_i

数列 $ a $ に含まれる連続した区間のうち、和が $ N $ になるものがいくつ存在するかを出力しなさい。

## 输入格式

无

## 输出格式

无

## 输入输出样例 #1

### 输入 #1

```
5
3 1 5 4 2
```

### 输出 #1

```
1
```

## 输入输出样例 #2

### 输入 #2

```
6
1 2 3 4 5 6
```

### 输出 #2

```
2
```

## 输入输出样例 #3

### 输入 #3

```
7
7 6 5 4 3 2 1
```

### 输出 #3

```
2
```

## 输入输出样例 #4

### 输入 #4

```
20
19 11 10 7 8 9 17 18 20 4 3 15 16 1 5 14 6 2 13 12
```

### 输出 #4

```
3
```

## 说明/提示

### 制約

- $ 1\ \leq\ N\ \leq\ 100,000 $
