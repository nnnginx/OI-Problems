# AT_abc137_a [ABC137A] +-x

## 题目描述

## 题意：

给出两个整数 $A$ 和 $B$。求出 $A + B$，$A\times B$ 和 $ A - B$ 中的最大值。


------------

## 输入格式

一行输入两个整数：$A$ 和 $B$。


------------

## 输出格式

输出 $A + B$，$A\times B$ 和 $ A - B$ 中的最大值。


------------


## 样例输入输出：

无


------------
## 数据规模与约定：

满足所有输入为整数。

$-100 \le A,B \le 100$。

## 输入输出样例 #1

### 输入 #1

```
-13 3
```

### 输出 #1

```
-10
```

## 输入输出样例 #2

### 输入 #2

```
1 -33
```

### 输出 #2

```
34
```

## 输入输出样例 #3

### 输入 #3

```
13 3
```

### 输出 #3

```
39
```
