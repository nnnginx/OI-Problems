# AT_cf17_final_a AKIBA

## 题目描述

输入一个字符串S，高桥君在其中任意位置插入A，问是否可以变成AKIHABARA

## 输入格式

一个字符串S

## 输出格式

可以输出YES，否则输出NO

## 输入输出样例 #1

### 输入 #1

```
KIHBR
```

### 输出 #1

```
YES
```

## 输入输出样例 #2

### 输入 #2

```
AKIBAHARA
```

### 输出 #2

```
NO
```

## 输入输出样例 #3

### 输入 #3

```
AAKIAHBAARA
```

### 输出 #3

```
NO
```
