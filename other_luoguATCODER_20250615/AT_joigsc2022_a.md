# AT_joigsc2022_a リレー競技 (Relay)

## 题目描述

给定 $N$ 和长度为 $N$ 的数列 $A$ 和 $B$。

你需要选择三个互不相同的下标 $i,j,k$，求出对应的 $A_i+\max(B_i,B_j)+A_j+\max(B_j,B_k)+A_k$ 的最小值。

## 输入格式

第一行一个正整数 $N$。

接下去 $N$ 行，每行两个正整数 $A_i$ 和 $B_i$。

## 输出格式

一行一个正整数，表示你的答案。

## 输入输出样例 #1

### 输入 #1

```
4
1070 90
1080 70
1050 60
1020 100
```

### 输出 #1

```
3320
```

## 输入输出样例 #2

### 输入 #2

```
5
1000 28
1000 14
1000 21
1000 20
1000 14
```

### 输出 #2

```
3034
```

## 输入输出样例 #3

### 输入 #3

```
9
3 1
4 1
5 9
2 6
5 3
5 8
9 7
9 3
2 3
```

### 输出 #3

```
13
```

## 说明/提示

对于全部数据，$3\le N\le 2\times10^5$，$1\le A_i,B_i\le10^8$。

| 子任务编号 | 附加限制 | 分值 |
| :-: | :-: | :-: |
| $1$ | $N\le 100$ | $25$ |
| $2$ | $N\le 3000$ | $33$ |
| $3$ | $A_1=A_2=\cdots A_N$ | $10$ |
| $4$ | 无额外约束 | $32$ |
