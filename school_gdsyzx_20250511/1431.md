# Description
面包师Lavrenty打算用馅料做几个面包卖。

Lavrenty有 n 克面团以及 m 种不同的馅料，馅料类型分别从 1 到 m 进行编号。Lavrenty知道他的第 i 种馅料还剩下 ai 克，然后需要花费 bi 克的馅料以及ci 克面团去制作一个对应馅料的面包。这样的面包可以卖出去 di 克的钱。

当然他也可以做不带馅料的面包。每个这样的面包需要 c0 克面团，然后可以卖出去 d0 克的钱。所以Lavrenty可以做任意数量带不同馅料的面包，也可以不用馅料，直到面团和馅料用完。做完面包后Lavrenty就会将所有多余的材料给扔掉。

请你找出面包师Lavrenty最多能赚多少钱。

# Format

## Input
第一行包括四个整数，n，m，c0，d0（1 ≤ n ≤ 1000，1 ≤ m ≤ 10，1 ≤ c0，d0 ≤ 100）。

接下来 m 行，每行四个整数，分别是ai，bi，ci 和 di。（1 ≤ ai，bi，ci，di ≤ 100）。

## Output
一行，一个整数，输出最多可以赚多少钱。

# Samples

```input1
10 2 2 1
7 3 2 100
12 3 1 10
```

```output1
241
```
```input2
100 1 25 50
15 5 20 10
```

```output2
200
```

# Explanation
样例一说明：他可以做 2 个带第 1 和第 4 种馅料的面包，4 个带第 2 种馅料的面包以及 1 个不带任何馅料的面包。

样例二说明：他可以做 4 个不带馅料的面包。
