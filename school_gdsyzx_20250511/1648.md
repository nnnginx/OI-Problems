

#### 【问题描述】

求方程 $ax^3+bx^2+cx+d=0$ 在 $[L ，R]$ 区间的近似根，误差为$10^{-7}$，保留小数点后 $6$ 位输出。


#### 【输入格式】

6行，每行一个数。分别代表 $a,b,c,d,L,R$ 。（输入数据保证方程在 [$L，R]$ 区间有近似根，$-100<L<=R<100$）

#### 【输出格式】

一个保留小数点后 $6$ 位的实数 $x$，为方程 $ax^3+bx^2+cx+d=0$ 在 $[L ，R]$ 区间的近似根。

#### 【样例输入1】

```
0
1
-3
-18
1
10
```

#### 【样例输出1】

```
6.000000
```

#### 【样例输入2】

```
3
4
1
3
-5
5
```

#### 【样例输出2】

```
-1.538962
```
