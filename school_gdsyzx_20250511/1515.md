# Description
有一天，琪琪想去拜访她的一位朋友。由于容易晕车，她想尽快赶到朋友家。

现在给你一张城市交通路线的地图，以及琪琪家附近可以乘坐的车站。您可以认为琪琪可以在任何车站换乘巴士。请找出琪琪需要花费的最少时间。为方便起见，如果城市有 n 个公交车站，则车站将表示为整数 1，2，3，...，n。

# Format

## Input
多组测试数据。

每组测试数据以三个整数 n、m 和 s 开头，（n ＜ 1000，m ＜ 20000，1 ≤ s ≤ n）。n 代表这个城市的公交车站数，m 代表之间有向路数公交车站。（也许两个公交车站之间有几种方式换乘。）s 代表靠近琪琪朋友家的公交车站。

然后接下来 m 行，每行包含三个整数 p，q，t（0 ＜ t ≤ 1000）。意味着从站 p 到站 q 有一条路，它会花费 t 分钟。

然后是一个整数 w（0 ＜ w ＜ n）的行，表示琪琪一开始可以走的站数。然后接下来一行是 w 个整数代表这些站。

## Output
对于每组数据，输出包含一行：琪琪需要花费的最少时间，如果找不到这样的路线，则输出“-1”。
