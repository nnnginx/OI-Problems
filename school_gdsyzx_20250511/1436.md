# Description
奶牛们要去太空了！他们计划通过建造一个太空电梯来实现，这个电梯就是一座巨大的积木塔。他们有 K（1 ≤ K ≤ 400）种不同类型的积木来建造，第 i 种类型的积木高度为 hi（1 ≤ hi ≤ 100），数量为 ci（1 ≤ ci ≤ 10）。由于宇宙射线可能会造成破坏，第 i 种方块的任何部分不能超过高度 ai（1 ≤ ai ≤ 40000）。

请你根据上述规则，帮助奶牛计算可能通过堆叠积木建造的最高的太空电梯。

# Format

## Input
一行，一个整数 K。

第二至第 K + 1 行，每行包含三个空格分隔整数 hi，ai 和 ci。

## Output
一行，一个整数 H，表示电梯可以建造的最大高度。

# Samples

```input1
3
7 40 3
5 23 8
2 52 6
```

```output1
48
```

# Explanation
样例说明：从底部开始，第二类积木有 3 块，第一类的积木放置 3 块以下，第三类的积木放置 6 块以下。如果是堆叠第二类积木 4 块，第一类积木 3 块是不合法的，因为最后会使得第 1 类积木的顶部超出 40 的高度。
