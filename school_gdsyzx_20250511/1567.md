# Description
小G在一个 n × m 的棋盘上随意放上了一些黑色的棋子，然后又在剩下所有没有放棋子的格子里放上了白色的棋子。现在小G想知道他是否能通过以下两种变换将整个棋盘上的棋子全部变成白色。

变幻 1：选择一列，将这一列的棋子全部反色，即黑变白，白变黑。

变幻 2：选择一行，将这一行的棋子全部反色。

如果能将整个棋盘上的棋子全部变成白色，则输出最少需要的变幻次数。否则输出经过若干次变幻后，棋盘上最少的黑子个数。

# Format

## Input
第一行，两个正整数 n，m，含义见题面。

接下里 n 行，每行 m 个字符，“0” 表示白子，“1” 表示黑子。

## Output
一行，一个整数 ans，含义见题面。

# Samples

```input1
3 4
0111
0111
1000
```

```output1
3
```

# Limitation
对于 30% 的数据，1 ≤ n，m ≤ 9；

对于 60% 的数据，1 ≤ n，m ≤ 15；

对于 100% 的数据，1 ≤ n ≤ 16，1 ≤ m ≤ 20。
