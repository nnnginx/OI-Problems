# Description
突破蝙蝠的包围，yifenfei来到一处悬崖面前，悬崖彼岸就是前进的方向，好在现在的yifenfei已经学过御剑术，可御剑轻松飞过悬崖。

现在的问题是：悬崖中间飞着很多红，黄，蓝三种颜色的珠子，假设我们把悬崖看成一条长度为 n 的线段，线段上的每一单位长度空间都可能飞过红，黄，蓝三种珠子，而yifenfei必定会在该空间上碰到一种颜色的珠子。如果在连续3段单位空间碰到的珠子颜色都不一样，则yifenfei就会坠落。
比如经过长度为 3 的悬崖，碰到的珠子先后为“红黄蓝”，或者“蓝红黄”等类似情况就会坠落，而如果是“红黄红”或者“红黄黄”等情况则可以安全到达。如图所示。

![img](./1272/file/0000098彼岸.jpg)

现在请问：yifenfei安然抵达彼岸的方法有多少种？

# Format

## Input
一个正整数 n（n ＜ 40）。

## Output
一个整数，表示yifenfei安然抵达彼岸的方法数。

# Samples

```input1
2
```

```output1
9
```
```input2
3
```

```output2
21
```
