# Description
wyh学长特别喜欢斐波那契数列，F（0）= 0，F（1）= 1，F（n) =F（n-1）+ F（n-2）（n ≥ 2）.一天他突发奇想，想求 F（$a^b$）% c。

# Format

## Input
输入第一行一个整数 T（1 ≤ T ≤ 100），代表测试组数。

接下来 T 行，每行三个数 a，b，c（a，b ≤ $2^{64}$)（1 ＜ c ＜ 1000）

## Output
输出第 $a^b$ 项斐波那契数对 c 取余的结果。

# Samples

```input1
3
1 1 2
2 3 1000
32122142412412142 124124124412124 123
```

```output1
1
21
3
```
