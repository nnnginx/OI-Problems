# Description
给定一个长度为 n 的整数序列 a1，a2，...，an。

请你选出一个该序列的严格上升子序列，要求所选子序列的各元素之和尽可能大。

请问这个最大值是多少？

# Format

## Input
第一行包含整数 n。

第二行包含 n 个整数 a1，a2，...，an。

## Output
输出最大的上升子序列和。

# Samples

```input1
2
100 40
```

```output1
100
```
```input2
4
1 9 7 10
```

```output2
20
```

# Explanation

对于样例 1，我们只选取 100。

对于样例 2，我们选取 1，9，10。

# Limitation
对于 30% 的数据，1 ≤ n ≤ 4；

对于 100% 的数据，1 ≤ n ≤ $10^5$，1 ≤ ai ≤ $10^9$。
