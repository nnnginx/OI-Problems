# Description
今天是伊格纳修斯的生日，他邀请了很多朋友。现在是晚饭时间，伊格纳修斯想知道他至少需要多少张桌子。你需要注意的是，并不是所有的朋友都相互认识，所有的朋友都不想和陌生人坐在一起。所以，这个问题的一个很重要的规则是：如果已知 A 认识 B，B 认识 C，那意味着 A，B，C相互或间接认识，所以他们可以围坐在一张桌子上。

例如，已知 A 认识 B，B 认识 C，D 认识 E，那么 A，B，C 可以围坐在一张桌子上；D，E 必须围坐在一张桌子上，所以伊格纳修斯至少需要两张桌子。

# Format

## Input
第一行，两个整数 N 和 M（1 ≤ N，M ≤ 1000）。N 表示朋友的数量，分别编号 1 至 N。

接下来 M 行，每行两个整数 A 和 B，表示 A 认识 B。

## Output
一行，一个整数，为伊格纳修斯至少需要多少张桌子。

