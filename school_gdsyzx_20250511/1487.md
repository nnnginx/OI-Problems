# Description
在珍珠王国里，人人都喜欢珍珠。一家名为“皇家珍珠”的公司生产了许多镶有珍珠的首饰。“皇家珍珠”之所以得此名，原因是它专门服务于珍珠王国里的皇室。当然，他也会为普通人生产手镯和项链等，不过珍珠的质量自然比皇室的低得多。在这些珠光宝气中，珍珠被分为 100 种不同的质量等级。一个质量等级是由该等级当中的其中珍珠的价格决定的，这个价格是独一无二的，并且总是高于较低等级的珍珠的价格。

每个月，皇家珍珠公司的库存经理都会准备一份清单，列出每个质量等级所需的珍珠数量。这些珍珠是在当地的珍珠市场购买的。每种质量等级的每颗珍珠都有自己的价格，但对于某一质量等级的每一笔完整交易，买家必须额外支付相当于该等级中十颗珍珠的钱，这是为了防止游客只买一颗珍珠。

此外，皇家珍珠公司也正遭受全球经济放缓的影响，因此公司需要提高效率。首席财务官（CFO）发现，他有时可以通过购买比实际需要的质量更高的珍珠来省钱，毕竟没有顾客会责怪皇家珍珠公司在手镯里放了更好的珍珠，当然前提是价格不变。

例如，10 欧元区需要 5 颗珍珠，20 欧元区需要 100 颗珍珠，这通常需要（5 + 10） × 10 +（100 + 10） × 20 = 2350 欧元，但购买 20 欧元区的 105 颗珍珠只需要（5 + 100 + 10） × 20 = 2300 欧元。

然而，首席财务官知道如何在一个更高质量的级别上买多少颗珍珠之前，他需要做大量的计算工作，于是向你求助。

给定一份不同质量等级的珍珠数量和每颗珍珠的价格清单，算出购买清单上所有物品所需的最低价格。珍珠可以按要求购买，也可以按更高质量等级购买，但不能按较低的质量等级购买。

# Format

## Input
第一行，一个整数 c，代表质量等级的数量。（1 ≤ c ≤ 100）

接下来 c 行，每行两个整数 ai 和 pi。ai 表示该等级珍珠的数量（1 ≤ ai ≤ 1000），pi 表示该等级每颗珍珠的价格（1 ≤ pi ≤ 1000）。保证给定的数据（价格）是按照质量等级升序排列给定。所有的输入数字都是整数。

## Output
一行，一个整数，为购买清单上所有商品所需的最低价钱。

# Samples

```input1
2
100 1
100 2
```

```output1
330
```
```input2
3
1 10
1 11
100 12
```

```output2
1344
```
