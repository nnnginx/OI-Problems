# Description
第一节编程课上，老师要求大家写一个程序计算两个正整数的和。

看到小X不屑的眼神后，老师决定给小X增加难度。以求 $12$ 和 $3$ 的和为例，老师在 $12+3$ 这个原始式子里加入一些无用的空格，再把它交给小X。

这下小X傻眼了，希望你帮帮他。

# Format

## Input
一行，包含一个字符串，表示老师给小X的式子。

## Output
若式子的结果不超过 $10^8$，则第一行包含一个整数，表示式子的结果；否则第一行包含一个字符串 $Large$。

# Samples

```input1
1 2 +   3
```

```output1
15
```
```input2
23456789+98765432
```

```output2
Large
```

# Limitation
对于 $30\%$ 的数据，式子中不包含无用的空格，式子的结果不超过 $10^8$。

对于 $100\%$ 的数据，字符串长度不超过 $100$。
