# Description

对于字符串 S 来说，如果它的长度不超过 L，而且每一个字符都是小写英文字母，那么字符串 S 称为“基本串”。设字符串 A 和字符串 B 都是“基本串”，它们的大小关系是按照经典的字典序比较的：

先比较第 1 个字符的ASCII码，如果 A[1] < B[1]，那么 A < B；如果 A[1] > B[1] 那么 A > B；如果 A[1] = B[1]，那么继续比较第 2 个字符，以此类推就能比较出大小关系。如果在比较的过程中，A 的每一个字符都和 B 对应位置的字符相同，但 B 长度更大，那么大小关系是 A < B；如果在比较的过程中，B 的每一个字符都和 A 对应位置的字符相同，但 A 长度更大，那么大小关系是 B < A。例如：“car”<“cat”，因为“r”<“t”。例如：”cat”<“cats”。

现在给出两个“基础串”X 和 Z，已知 X < Z。求有多少个不同的“基础串”Y，使得 X < Y < Z。

# Format

## Input

第 1 行，一个整数 L。1 ≤ L ≤ 10。

第 2 行，一个“基础串”X。

第 3 行，一个“基础串”Z。

## Output

一个整数。

# Samples

```input1
2
ay
c
```

```output1
28
```

```input2
10
abc
xyzzy
```

```output2
135069320654491
```

# Explanation

样例一说明：

Y 可以是“az”、“b”、”ba”、“bb”、...、“bz”。

# Limitation

对于 60% 的数据，1 ≤ L ≤ 4。
