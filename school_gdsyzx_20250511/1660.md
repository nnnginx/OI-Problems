# Description
监狱有连续编号为 1 到 n 的 n 个房间，每个房间关押一个犯人。有 m 种宗教，每个犯人可能信仰其中一种。如果相邻房间的犯人信仰的宗教相同，就可能发生越狱。求有多少种状态可能发生越狱。

# Format

## Input
输入两个整数 m 和 n。

## Output
可能越狱的状态数，对 100003 取余。

# Samples

```input1
2 3
```

```output1
6
```

# Explanation
样例说明：

所有可能的 6 种状态为：{0，0，0}，{0，0，1}，{0，1，1}，{1，0，0}，{1，1，0}，{1，1，1}。

# Limitation
对于 100% 的数据，1 ≤ m ≤ $10^8$，1 ≤ n ≤ $10^{12}$。
