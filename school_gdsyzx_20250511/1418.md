# Description
你即将开车出远门，当然希望在车上能聆听一些美好的音乐。你的车上只有播放录音带的设备，但是你最喜欢的音乐却都存放在CD上。所以你需要把CD上的音乐转录到录音带上。现在你必须解决的问题是：你的空白录音带时长共 N 分钟，你如何选择CD上的歌使得尽可能的利用录音带的空间。以下是一些关于此问题的假设：

1. CD上的歌不会超过 20 首。

2. 没有任何一首歌的长度超过 N 分钟。

3. 刻录在录音带上的歌不能重复。

4. 每首歌的长度以一整数表达。

5. N 也是一个整数。

你的程序必须找出该放哪些CD上的歌到录音带上（按CD上的顺序），使得录音带的空白空间最小。

# Format

## Input
每组测试数据一行，第一个整数为 N，代表空白录音带的长度。第二个整数 T 代表CD上共有多少首歌。接下来的 T 个整数分别代表CD上每首歌的长度。

## Output
输出一行，分别为要放进录音带的各首歌的长度（答案并非唯一，如样例五中，43 2 也是正确的答案。只要是正确答案都可被接受），以及总长度。

# Samples

```input1
5 3 1 3 4
```

```output1
1 4 sum:5
```

```input2
10 4 9 8 4 2
```

```output2
8 2 sum:10
```

```input3
20 4 10 5 7 4
```

```output3
10 5 4 sum:19
```

```input4

```

```output4

```
