#### **题目描述**

将有$N$个元素的有序的数组，和有$M$个元素的有序数组合并为一个有序数组。 例如： 将有序数组${2,3,3}$和有序数组${1，1，2，3，3，4}$，合并后结果为：${1，1，2，2，3，3，3，3，4}$ 现已知两个有序数组，你能将他们按顺序合并吗?

#### **输入**

输入的数据有三行。

第一行有两个数据，分别是$N$和$M（1<=N，M< 1000000）$。

第二行有$N$个从小到大的正整数,正整数的范围不超过$10^9$。

第二行有$M$个从小到大的正整数,正整数的范围不超过$10^9$。

#### **输出**

输出结果将$N+M$个正整数从小到大的输出。

#### **样例输入**

```
3 6
2 3 3 
1 1 2 3 3 4
```

#### **样例输出**

```
1 1 2 2 3 3 3 3 4
```
