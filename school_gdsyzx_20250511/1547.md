# Description
给出一个字符串 S，其中满足 S 的每一个字符都是数字字符，你要删除 S 的连续一段字符（也可以删除 0 个字符），使得剩下的字符依次连接起来的字符串是 “2020”，可以做到吗？如果可以做到输出 “YES”，否则输出 “NO”。

# Format

## Input
第一行，一个整数 G，表示有 G 组测试点。1 ≤ G ≤ 10。

每组测试格式如下：

第一行，一个整数 n，表示字符串 S 的长度。1 ≤ n ≤ 200。

第二行，字符串 S。

## Output
共 G 行，每行一个字符串，“YES” 或 “NO”，双引号不用输出。

# Samples

```input1
5
8
20192020
8
22019020
4
2020
5
20002
6
729040
```

```output1
YES
YES
YES
NO
NO
```
