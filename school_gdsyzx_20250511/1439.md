# Description
你现在拿到了许多的礼物，你要把这些礼物放进袋子里。你只有一个最多装下 V 体积物品的袋子，你不能全部放进去。你也拿不动那么重的东西。你估计你能拿的最大重量为 G。现在你了解了每一个物品的完美值、重量和体积，你当然想让袋子中装的物品的完美值总和最大，你又得计划一下了。

# Format

## Input
第一行，G 和 V 表示最大重量和体积。 （0 ＜ G,V ≤ 400）

第二行，N 表示拿到 N 件礼物。 （0 ＜ N ≤ 400）

第三到第 N + 2 行，每行 3 个数，Ti，Gi，Vi，表示各礼物的完美值、重量和体积。 

## Output
输出共一个数，表示可能获得的最大完美值。

# Samples

```input1
6 5  
4  
10 2 2  
20 3 2
40 4 3
30 3 3
```

```output1
50
```
