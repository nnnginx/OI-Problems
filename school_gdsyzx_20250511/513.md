**内存限制：64 MB**	**时间限制：1 S**

#### 题目描述

逆波兰表达式是一种把运算符前置的算术表达式，例如普通的表达式$2 + 3$的逆波兰表示法为$+ 2\ 3$。逆波兰表达式的优点是运算符之间不必有优先级关系，也不必用括号改变运算次序，例如$(2 + 3) * 4$的逆波兰表示法为$* + 2\ 3\ 4$。本题求解逆波兰表达式的值，其中运算符包括$+ - * /$四个。

#### 输入

输入为一行，其中运算符和运算数之间都用空格分隔，运算数是浮点数。

#### 输出

输出为一行，表达式的值。
可直接用 printf("%f ", $v$) 输出表达式的值$v$。

#### 样例输入

```plain
* + 11.0 12.0 + 24.0 35.0
```

#### 样例输出

```plain
1357.000000
```

#### 提示

可使用atof(str)把字符串转换为一个double类型的浮点数。atof定义在math.h或stdlib.h中。
