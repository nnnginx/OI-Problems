# Description
在ACM举办任何活动之前，都必须要准备好预算并获得必要的财政支持。其主要收入来源于存钱。背后的处理是很简单的：每当某位ACM成员有零钱时，他就会把所有的硬币存进小猪钱罐，这个过程是不可逆的，因为如果要从小猪钱罐里面拿钱，既要打破它。在足够长的时间之后，小猪钱罐里就有足够的钱支付活动所需要的使用的款项。

但是小猪钱罐有个大问题就是无法确定里面有多少钱，如果我们把打碎了钱罐却发现里面的钱不够，不仅办不了活动，还亏了一个钱罐。显然，我们希望避免这种令人不愉快的情况。所以，现在唯一的可能是称一下小猪钱罐的重量，然后大概猜一猜里面有多少硬币，这样才好下手。假设我们能够准确地确定小猪钱罐的重量，并且知道所有不同种类的硬币的重量，那么我们就可以确定存钱罐所存的最小金额的钱。你的任务是找出最差的情况，确定小猪钱罐里的最小现金量，不要再有提前打烂的情况辣！

# Format

## Input
第一行，两个整数 E 和 F，代表空的时候小猪钱罐的重量以及小猪钱罐装了钱后的重量。单位均为克。小猪钱罐的重量不会超过 10 kg，说明 1 ≤ E ≤ F ≤ 10000。

第二行为一个整数 N（1 ≤ N ≤ 500），代表硬币的总数量。接下来 N 行，每行两个整数，分别代表不同种类硬币的价值和重量 P 和 W。（1 ≤ P ≤ 50000，1 ≤ W ≤ 10000）。单位为克。

## Output
一行，如果能够精确地计算出最小现金量，输出“The minimum amount of money in the piggy-bank is X.”，其中X为所获得的的最小现金量。如果不能精确地计算，则输出“This is impossible.”
