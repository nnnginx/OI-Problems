# Description
给定两个向量 v1 =（x1，x2，...，xn）和 v2 =（y1，y2，...，yn），它们的标量乘积为：x1y1 + x2y2 +...+ xnyn。

假设你可以任意地排列向量 v1 和向量 v2 中的分量坐标（即任意排列 x1 至 xn 和任意排列 y1 至 yn），从而得到两个新的向量，使得两个新的向量的标量乘积最小。

# Format

## Input
共有三行：

第一行为向量的维数 n（0 ＜ n ≤ 100）；

第二行由 n 个整数组成，它们之间由空格隔开，分别表示向量 1 的分量坐标（即 x1 至 xn 的值，-2000 ≤ xi ≤ 2000）；

第三行由 n 个整数组成，它们之间由空格隔开，分别表示向量 2 的坐标（即 y1 至 yn 的值，-2000 ≤ yi ≤ 2000）。

## Output
仅为一行，为所求的最小的标量乘积的值。

# Samples

```input1
3
1 3 -5
-2 4 1
```

```output1
-25
```
