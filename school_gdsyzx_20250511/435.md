
### 题目描述

城市有 $n$ 个节点，编号为 $1..n$ 。小A正在城市中游荡，不管他在哪个节点，当听到他的朋友小B在城市中的某个节点时，就会想快速的去找小B玩耍。现在给出城市的 $n$ 个节点的邻接矩阵，然后提出 $m$ 次询问，每次询问小A从节点 $x$ ，到小B所在节点 $y$ 的最短距离。

### 输入格式

第一行为整数 $n$，表示城市节点数。

接下来的 $n$ 行是一个 $n*n$ 的矩阵，矩阵中第 $i$ 行 $j$ 列的整数表示第 $i$ 点到第 $j$ 点之间的直接连线（0表示两点无直接连线），每行的 $n$ 个整数之间用空格隔开。 ($0<n<=100$)

接下来一个整数 $m$，表示有 $m$ 次询问。

接下来 $m$ 行，每行两个整数，表示小 A 在 $x$ 节点，小B在 $y$ 节点。

### 输出格式

$m$ 行，每行为该组询问的最短距离。

### 输入样例

```
3
0 1 10
1 0 2
10 2 0
2
1 3
2 3
```

### 输出样例

```
3
2
```
