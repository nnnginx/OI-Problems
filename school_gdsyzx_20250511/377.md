#### **题目描述**

小飞最近迷上了一个叫生死狙击的游戏。不过他的技术并不好，假设他的战斗力是$•x$。不过，他的战斗力是由心情来定的，每天都不同。现有一群$(1<=n<=100000)$也是玩生死狙击的朋友，他们也有战斗力。小飞想找一些比自己强的人玩，提高自己的战斗力，如：小烨，小祖，小明等。 小飞一共玩了$m$天，假设其他人的战斗力不变。问：小飞每天可以找到多少比自己强的人。

#### **输入**

第一行输入一个$n$，表示有$n$个朋友；$(1<=n<=100000)$

第二行输入$n$个朋友的战斗力；每个朋友战斗力不超过$long\ int$;（已有序）；

第三行输入一个$m$，表示有$m$天；$(1<=m<=100000)$

接下来$m$行，表示小飞每天的战斗力；小飞战斗力不超过$long\ int$;

#### **输出**

共$m$行，每一行小飞每天找到的人数；

#### **样例输入**

```
5 
1 1 3 5 5 
3 
0 
2 
5
```

#### **样例输出**

```
5 
3 
0
```
