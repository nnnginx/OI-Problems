# Description
暑假来了，hrdv又要留学校在参加ACM集训了，集训的生活非常Happy（ps：你懂得），可是他最近遇到了一个难题，让他百思不得其解，他非常郁闷。亲爱的你能帮帮他吗？

问题是我们经常见到的整数划分，给出两个整数 n，m，要求在 n 中加入 m - 1 个乘号，将 n 分成 m 段，求出这 m 段的最大乘积。

# Format

## Input
每行有两个正整数 n，m（1 ≤ n ＜ $10^{19}$，0 < m ≤ n 的位数）。

## Output
一行，一个整数，为最大乘积。

# Samples

```input1
111 2
```

```output1
11
```
```input2
1111 2
```

```output2
121
```
