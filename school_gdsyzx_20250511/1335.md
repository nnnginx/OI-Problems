# Format

## Input
输入的第一行包含两个正整数 T，M，分别表示方程数和系数绝对值的上界。

接下来 T 行，每行包含三个整数 a，b，c。

## Output
输出 T 行，每行包含一个字符串，表示对应询问的答案，格式如题面所述。

<b>每行输出的字符串中间不应包含任何空格</b>。

# Samples

```input1
9 1000
1 -1 0
-1 -1 -1
1 -2 1
1 5 4
4 4 1
1 0 -432
1 -3 1
2 -4 1
1 7 1
```

```output1
1
NO
1
-1
-1/2
12*sqrt(3)
3/2+sqrt(5)/2
1+sqrt(2)/2
-7/2+3*sqrt(5)/2
```

# Limitation
对于所有测试数据有：1 ≤ T ≤ 5000，1 ≤ M ≤ $10^3$,|a|,|b|,|c| ≤ M，a ≠ 0。


![img](./1335/file/0001594一元二次方程4.png)

其中：

特殊性质A：保证 b = 0；

特殊性质B：保证 c = 0；

特殊性质C：如果方程有解，那么方程的两个解都是整数。
