文章阅读(read)

## 题目描述

给出 $T$ 篇仅由大小写英文字母组成的文章，分别求问每篇文章中有多少个在字母中出现的封闭图形。

例如 `Watch Out OP` 这篇文章中的封闭图形数量为 $4$ ，因为字母 `a,O,P` 中分别出现了 $1,1,1$ 个封闭图形，而 `O` 出现了两次，所以答案为 $1+1\times 2+1=4$ 。
- 特别的，为了统一字母写法，小写 `g` 算作两个封闭图形，小写 `k` 算作零个。
- 数量为 $1$ ：`A,D,O,P,Q,R,a,b,d,e,o,p,q` 。
- 数量为 $2$ ：`B,g` 。

## 输入格式

- 从`read.in`中读入数据。
- 第一行一个数：$T$ 。
- 接下来 $T$ 行每行一篇文章：
	- 每行文章开头会有一个数字 ：$n$ 表示这篇文章的单词数量。
    - 在 $n$ 之后，有 $n$ 个仅包含大小写字母的单词 $S$，即字符串。

## 输出格式

- 输出到`read.out`中。
- 对于每一篇文章，输出对应出现的封闭图形个数。

## 样例 #1

### 样例输入 #1

```
1
3 Watch Out OP
```

### 样例输出 #1

```
4
```

## 提示

|  测试点   | $n\le$ | 特殊性质 |
| :-------: | :----: | :------: |
| $1\sim4$  |  $1$   |          |
|    $5$    | $10^2$ |   $A$    |
|    $6$    | $10^2$ |   $B$    |
| $7\sim10$ | $10^2$ |          |

特殊性质 $A$ ：文章中不包含小写字母。

特殊性质 $B$ ：文章中不包含大写字母。

对于所有数据保证：$1\le T\le10^2,|S|\le10^2$ 。
