# Description
国王将金币作为工资，发放给忠诚的骑士。

第一天，骑士收到一枚金币；之后两天（第二天和第三天），每天收到两枚金币；之后三天（第四、五、六天），每天收到三枚金币；之后四天（第七、八、九、十天），每天收到四枚金币...；这种工资发放模式会一直这样延续下去：当连续 N 天每天收到 N 枚金币后，骑士会在之后的连续 N + 1 天里，每天收到 N + 1 枚金币。

请计算在前 K 天里，骑士一共获得了多少金币。

# Format

## Input
只有 1 行，包含一个正整数 K，表示发放金币的天数。

## Output
只有 1 行，包含一个正整数，即骑士收到的金币数。

# Samples

```input1
6
```

```output1
14
```
```input2
1000
```

```output2
29820
```

# Explanation
样例一说明：

骑士第一天收到一枚金币；第二天和第三天，每天收到两枚金币；第四、五、六天，每天收到三枚金币。因此一共收到 1 + 2 + 2 + 3 + 3 + 3 = 14 枚金币。

# Limitation
对于 100% 的数据，1 ≤ K ≤ 10000。
