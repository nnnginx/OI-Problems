# Description
MIS R有 m 颗糖，编号为 1，2，...，m。他打算发给 n 个小朋友。发完糖之后，为了促进小朋友之间的友谊，他会安排 r 次互换活动。每次互换活动，会让第 x 个小朋友和第 y 个小朋友交换手里的糖果。现在MIS R想知道最终每个小朋友手中有哪些糖果。

# Format

## Input
第一行两个整数 n，m，r（2 ≤ n ≤ 1000，1 ≤ m，r ≤ $10^6$），表示 n 个小朋友，m 颗糖，r 次互换安排。

第二行有 m 个整数，表示第 i 个糖果发给了第 ai 个小朋友。

接下来 r 行，每行两个整数 x，y，表示第 x 个小朋友和第 y 个小朋友互换手中的糖果。

数据保证 1 ≤ ai，x，y ≤ n，x ≠ y。

## Output
n 行，每行以一个整数 ci 开头，表示第 i 个小朋友含有的糖果数，接下来是一个长度为 ci 的递增编号序列，整数之间用空格隔开，表示第 i 个小朋友手中糖果的编号。

# Samples

```input1
3 6 1
1 3 2 1 3 2
2 3
```

```output1
2 1 4
2 2 5
2 3 6
```
