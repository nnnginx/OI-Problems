#### 题目描述

有一个长为 $n$ 的数组 $a$ ，初始时所有 $a_p=p$ 。

给出一个 $1\sim n$ 的排列 $s$ ，$a$ 数组会根据 $s$ 进行 $k$ 次变化，每次 $a$ 数组中的所有元素都会**同时**进行以下操作：

- 将 $a$ 数组位置 $i$ 上的元素替换为 $a_{s_i}$ 。

现在给出 $k$ ，求有多少个整数 $i\in[1,k]$ 使得变换完 $i$ 次后 $a$ 数组回到初始状态（即所有 $a_p=p$）。

#### 输入格式

- 从`permutation.in`中读入数据。
- 第一行两个正整数： $n,k$ 。
- 第二行一行 $n$ 个数表示 $s$ 数组。

#### 输出格式

- 输出到`permutation.out`中。
- 输出一行一个数表示答案。

#### 样例 #1

#### 样例输入 #1

```
4 1
2 1 4 3
```

#### 样例输出 #1

```
0
```

#### 样例 #2

#### 样例输入 #2

```
10 1000
3 5 4 6 7 2 9 10 1 8
```

#### 样例输出 #2

```
125
```

#### 提示

**数据约定**

|  测试点   | $n\le$ |  $k\le$   |
| :-------: | :----: | :-------: |
| $1\sim4$  | $10^2$ |  $10^5$   |
| $5\sim6$  |  $3$   | $10^{18}$ |
| $7\sim10$ | $10^6$ | $10^{18}$ |

对于所有数据：$s$ 数组元素互不相同且满足 $1\le s_i\le n$ 。

样例解释：

- ```a:1 2 3 4```
- ```s:2 1 4 3```

第一次变换：

- $a^{'}_1=a_{s_1}=a_2=2$
- $a^{'}_2=a_{s_2}=a_1=1$
- $a^{'}_3=a_{s_3}=a_4=4$
- $a^{'}_4=a_{s_4}=a_3=3$

其中 $a^{'}_i$ 表示 $i$ 位置变化完之后的值，所以第一次操作后数组为 ```2 1 4 3``` ，并非初始状态，故答案为 $0$ 。
