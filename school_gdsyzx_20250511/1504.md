# Description
给你一些已经确定的元素之间的关系，请你判断是否能从这些元素关系中推断出其他的元素关系。

# Format

## Input
首先是一个正整数 m（m ≤ 100），表示给定元素关系的个数。

接下来 m 行，每行一个元素关系，格式为：

元素 1 ＜ 元素 2 或者元素 1 ＞ 元素 2。

元素用一个大写字母表示，输入中不会包含冲突的关系。

## Output
接下来输出所有推断出的新的元素关系，按照字典序从小到大排序，格式为：

元素 1 ＜ 元素 2

每个元素关系占一行，输入中给定的元素关系不要输出。

如果没有新的元素关系推断出来，则输出 NONE。

# Samples

```input1
3
A<B
C>B
C<D
```

```output1
A<C
A<D
B<D
```
```input2
2
A<B
C<D
```

```output2
NONE
```
