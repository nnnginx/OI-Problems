# Description
研究中心正在研究一种数列，该数列由整数组成，每次可以对相邻的两个数进行相减操作，并删去这两个数，将运算结果替换进去。直到此数列只剩一个数为止。现在此研究中心欲使此最后剩下的数字为目标数，请你帮助他们完成这个任务。（假设：在运算过程中的所有数都在int范围内）

若此数列为{12，10，4，3，5}，目标数为 4，E（n）表示对第 n 和第 n + 1 数进行相减并替换。

E（2）={12，6，3，5}（6 = 10 - 4）

E（3）={12，6，-2}（-2 = 3 - 5）

E（2）={12，8}（8 = 6 - (-2) ）

E（1）={4}（4 = 12 - 8）

则最后达到目的，数列仅剩一目标数，则此过程为：2，3，2，1。

　　

# Format

## Input
第一行两个整数N（3 ≤ N ≤ 10）和 M，为数列中数的个数和目标数。

第二行 N 个整数，为此数列中的数。

## Output
一行，输出总共可行方案的数量。

# Samples

```input1
5 4
12 10 4 3 5
```

```output1
3
```
