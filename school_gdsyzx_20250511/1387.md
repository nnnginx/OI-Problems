# Description
小明最近学习了差的绝对值，|a - b| 表示 a - b 的绝对值，若 a - b ≥ 0，则 |a - b| = a - b；若 a - b ＜ 0，则 |a - b| = -（a - b）。

经过几次练习，小明已经熟练掌握了差的绝对值，现在他找来了 N 个整数，开始任意取出两个数，求差的绝对值，再将所有差的绝对值相加。例如 N = 4，有 4 个整数，分别是 1，2，3，4。任取两个数有 6 种取法，|1 - 2| = 1，|1 - 3| = 2，|1 - 4| = 3，|2 - 3| = 1，|2 - 4| = 2，|3 - 4| = 1，它们的和就是 10。

由于运算量太大，累坏了小明。请你写一个程序帮他计算一下吧。

# Format

## Input
第一行一个整数 n，表示有 n 个整数；

第二行 n 个整数，表示小明写下的 n 个整数。

## Output
输出一个数，表示任意两数差的绝对值之和。

# Samples

```input1
3
3 1 2
```

```output1
4
```
```input2
4
1 2 3 4
```

```output2
10
```

# Explanation

样例一说明：

|3 - 1| = 2，|3 - 2| = 1，|1 - 2| = 1，答案为 4。

# Limitation

对于 40% 的数据， n ≤ 1000，0 ≤ 每个数 ≤ 1000；

对于 70% 的数据， 输入数据保证第二行的 n 个数字从小到大有序。

对于 100% 的数据， n ≤ 1000000，0 ≤ 每个数 ≤ 1000000000。
