## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc226/tasks/abc226_a

小数第三位までで表すことのできるg数 $ X $ が、小数第三位まで入力されます。  
 $ X $ を小数第一位で四挝迦毪筏拷Y果として得られる整数を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $

## 输出格式
$ X $ を小数第一位で四挝迦毪筏频盲椁欷胝数を出力せよ。

## 题目大意
给一个100以内的正三位小数，将其四舍五入到整数后输出。

```input1
3.456
```

```output1
3
```

```input2
99.500
```

```output2
100
```

```input3
0.000
```

```output3
0
```

## 提示
### 制s

- $ 0\ \leq\ X\ <\ 100 $
- $ X $ は小数第三位までで表F可能である。
- $ X $ は小数第三位まで与えられる。

### Sample Explanation 1

$ 3.456 $ の小数第一位は $ 4 $ であるので、$ 3.456 $ を小数第一位で四挝迦毪筏は $ 3 $ となります。
