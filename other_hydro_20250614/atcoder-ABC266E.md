## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc266/tasks/abc266_e

サイコロを使ったゲ`ムをします。ゲ`ムは最大 $ N $ 回のタ`ンからなり、各タ`ンは次のようにM行します。

- $ 1,\ldots,6 $ の目が等_率で出る $ 6 $ 面ダイスを振り、出目を $ X $ とする(出目は各タ`ンで独立とする)。
- F在が $ N $ タ`ン目なら、**スコア** を $ X $ とし、ゲ`ムをK了する。
- そうでないとき、ゲ`ムをA行するかK了するかxkする。
  - ゲ`ムをK了する龊稀スコアを $ X $ とし、残りのタ`ンは行わずにゲ`ムをK了する。

スコアの期待が最大になるように行婴筏郡趣、スコアの期待を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。  
 なお、真の解との~`差または相`差が $ 10^{-6} $ 以下であれば正解としてQわれる。

## 题目大意
你有一个普通均匀的正方体骰子，六个面写有 $1,2,3,4,5,6$。你在玩一个游戏，每次丢骰子之后，你可以：
- 如果这是你的第 $N$ 次投掷，那么你应当结束游戏。
- 否则你可以选择重新投，或者结束游戏。

给定 $N$，计算如果你希望**最后一次投掷**时朝上面的**期望**最大，那么这个期望是多少。

```input1
1
```

```output1
3.5000000000
```

```input2
2
```

```output2
4.2500000000
```

```input3
10
```

```output3
5.6502176688
```

### 制s

- $ 1\ \leq\ N\ \leq\ 100 $
