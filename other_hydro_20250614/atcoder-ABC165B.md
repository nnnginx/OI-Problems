## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc165/tasks/abc165_b

高颏んはAtCodery行に $ 100 $ 窑蝾Aけています。

AtCodery行では、澳觐A金~の $ 1 $ % の利子がつきます(}利、小数点以下切り韦)。

利子以外の要因でA金~が浠することはないと⒍à筏郡趣、高颏んのA金~が初めて $ X $ 乙陨悉摔胜毪韦虾文赆幛扦筏绀Δ。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $

## 输出格式
高颏んのA金~が初めて $ X $ 乙陨悉摔胜毪韦虾文赆幛を出力せよ。

## 题目大意
输入正整数$n$,找一个自然数$k$，使$100*1.01^k \geq n$

```input1
103
```

```output1
3
```

```input2
1000000000000000000
```

```output2
3760
```

```input3
1333333333
```

```output3
1706
```

## 提示
### 制s

- $ 101\ \le\ X\ \le\ 10^{18} $
- 入力はすべて整数

### Sample Explanation 1

\- $ 1 $ 年後のA金~は $ 101 $ 窑扦埂 - $ 2 $ 年後のA金~は $ 102 $ 窑扦埂 - $ 3 $ 年後のA金~は $ 103 $ 窑扦埂 したがって、A金~が初めて $ 103 $ 乙陨悉摔胜毪韦 $ 3 $ 年後です。
