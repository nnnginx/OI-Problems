## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc113/tasks/arc113_b

正の整数 $ A,B,C $ が与えられます。$ A^{B^C} $ の $ 10 $ M法での $ 1 $ の位を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $

## 输出格式
$ A^{B^C} $ の $ 10 $ M法での $ 1 $ の位を出力せよ。

## 题目大意
### 题目描述
这里有 $3$ 个正整数。请求出 $ A^{B^C} $ 的个位。
### 输入格式

按以下格式输入。

> $ A $ $ B $ $ C $

### 输出格式

输出 $ A^{B^C} $ 个位上的数。

### 样例输入 #1

```
4 3 2
```

### 样例输出 #1

```
4
```

## 样例 #2

### 样例输入 #2

```
1 2 3
```

### 样例输出 #2

```
1
```

## 样例 #3

### 样例输入 #3

```
3141592 6535897 9323846
```

### 样例输出 #3

```
2
```

### 提示

- $ 1\leq\ A,B,C\ \leq\ 10^9 $
- $ A,B,C $ 是整数。

### 样例说明 1

$ 4^{3^2}=4^9=262144 $ 的个位是 $ 4 $ 。

```input1
4 3 2
```

```output1
4
```

```input2
1 2 3
```

```output2
1
```

```input3
3141592 6535897 9323846
```

```output3
2
```
