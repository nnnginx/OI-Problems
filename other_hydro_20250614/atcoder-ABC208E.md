## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc208/tasks/abc208_e

$ N $ 以下の正の整数のうち、各桁の数字のeが $ K $ 以下であるものは何ありますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
条件を氦郡拐数の数を出力せよ。

## 题目大意
求有多少个不大于 $N$ 的正整数，使得该正整数各位乘积不大于 $K$。

$N \le 10^{18},K \le 10^9$

```input1
13 2
```

```output1
5
```

```input2
100 80
```

```output2
99
```

```input3
1000000000000000000 1000000000
```

```output3
841103275147365677
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^{18} $
- $ 1\ \leq\ K\ \leq\ 10^9 $
- 入力は全て整数である。

### Sample Explanation 1

$ 13 $ 以下の正の整数のうち、各桁の数字のeが $ 2 $ 以下であるものは $ 1,2,10,11,12 $ の $ 5 $ つです。

### Sample Explanation 2

$ 100 $ 以下の正の整数のうち、$ 99 $ 以外のものが条件を氦郡筏蓼埂

### Sample Explanation 3

答えが $ 32 $ bit 整数にГ蓼椁胜た赡苄预あることに注意してください。
