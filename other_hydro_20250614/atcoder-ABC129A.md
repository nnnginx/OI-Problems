## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc129/tasks/abc129_a

空港 A, B, C があり、それぞれの空港のgでは、双方向にw行Cが\航しています。

空港 A, B gのw行rgは片道 $ P $ rg、空港 B, C gのw行rgは片道 $ Q $ rg、空港 C, A gのw行rgは片道 $ R $ rgです。

いずれかの空港からスタ`トして他の空港にw行Cで移婴贰さらにそのどちらでもない空港にw行Cで移婴工毪瑜Δ式U路を考えます。

w行rgの和は最短で何rgになるでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ P $ $ Q $ $ R $

## 输出格式
w行rgの和の最小を出力せよ。

## 题目大意
有A,B,C三个机场，给出A到B的距离，B到C的距离，以及C到A的距离。
求把三个机场全部遍历一遍所要走的距离。

```input1
1 3 4
```

```output1
4
```

```input2
3 2 3
```

```output2
5
```

## 提示
### 制s

- $ 1\ \leq\ P,Q,R\ \leq\ 100 $
- 入力は全て整数である

### Sample Explanation 1

A, B, C のに移婴工虢U路のw行rgの和は、$ 1\ +\ 3\ =\ 4 $ rg A, C, B のに移婴工虢U路のw行rgの和は、$ 4\ +\ 3\ =\ 7 $ rg B, A, C のに移婴工虢U路のw行rgの和は、$ 1\ +\ 4\ =\ 5 $ rg B, C, A のに移婴工虢U路のw行rgの和は、$ 3\ +\ 4\ =\ 7 $ rg C, A, B のに移婴工虢U路のw行rgの和は、$ 4\ +\ 1\ =\ 5 $ rg C, B, A のに移婴工虢U路のw行rgの和は、$ 3\ +\ 1\ =\ 4 $ rg であるので、これらの最小は $ 4 $ rgです。
