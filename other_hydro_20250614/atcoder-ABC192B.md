## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc192/tasks/abc192_b

先^から奇数番目の文字が全て英小文字であり、かつ、先^から偶数番目の文字が全て英大文字であるような文字列を **iみにくい文字列** と呼びます。

文字列 $ S $ がiみにくい文字列かどうか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ がiみにくい文字列なら `Yes`、iみにくい文字列でないなら `No` を出力せよ。

## 题目大意
定义一个字符串是难以阅读的，当且仅当所有偶数位上是大写字母，奇数位上是小写字母。

给定一个字符串，判断是否是难以阅读的。如果是，则输出 `Yes`，否则输出 `No`。

下标从 $1$ 开始。

```input1
dIfFiCuLt
```

```output1
Yes
```

```input2
eASY
```

```output2
No
```

```input3
a
```

```output3
Yes
```

## 提示
### 制s

- $ S $ は英大文字及び英小文字のみからなる
- $ S $ のLさは $ 1 $ 以上 $ 1000 $ 以下

### Sample Explanation 1

先^から奇数番目の文字が全て英小文字であり、かつ、先^から偶数番目の文字が全て英大文字であるので、iみにくい文字列です。

### Sample Explanation 2

先^から $ 3 $ 番目の文字が英小文字ではないので、iみにくい文字列ではありません。
