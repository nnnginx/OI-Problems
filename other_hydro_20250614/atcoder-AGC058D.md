## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc058/tasks/agc058_d

整数 $ A,B,C $ が与えられます． `A`, `B`, `C` からなる文字列 $ S $ であって，以下の条件をすべて氦郡工猡韦数を $ 998244353 $ で割った余りを求めてください．

- $ S $ に含まれる `A`, `B`, `C` の数はそれぞれ $ A,B,C $ である．
- $ S $ は（BAする）部分文字列として，`ABC`, `BCA`, `CAB` のいずれも含まない．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ A $ $ B $ $ C $

## 输出格式
答えを出力せよ．

## 题目大意
给出 $a,b,c$，求由 $a$ 个 A，$b$ 个 B，$c$ 个 C 构成的字符串数量，使得不存在子串 `ABC`，`BCA` 和 `CAB`。$1 \leq a,b,c \leq 10^6$

```input1
1 1 1
```

```output1
3
```

```input2
2 2 2
```

```output2
42
```

```input3
96 11 46
```

```output3
818015722
```

```input4
125132 102271 152064
```

```output4
128086069
```

- $ 1\ \leq\ A,B,C\ \leq\ 10^6 $
- 入力されるはすべて整数である

条件を氦郡刮淖至肖希`ACB`, `CBA`, `BAC` の $ 3 $ つです．
