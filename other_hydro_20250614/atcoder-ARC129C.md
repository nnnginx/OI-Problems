## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc129/tasks/arc129_c

整数 $ N $ が与えられます．

`1`, `2`, $ \cdots $, `9` からなる文字列 $ s $ であって，以下の条件を氦郡工猡韦蛞护那螭幛皮ださい．

- $ s $ のLさ $ |s| $ は $ 10^6 $ 以下．
- 次の条件を氦郡拐数のM $ (l,r) $ ($ 1\ \leq\ l\ \leq\ r\ \leq\ |s| $) の数がちょうど $ N $ である．
  - $ s $ の $ l $ 文字目から $ r $ 文字目までを取り出して数としてたとき，$ 7 $ で割り切れる．

なお，この}の制sより，解が必ず存在することが^明できます．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $

## 输出格式
条件を氦郡 $ s $ を出力せよ． 解が}数存在する龊希どれを出力しても正解とみなされる．

## 题目大意
给定 $N$，请构造一个长度不超过 $10^6$，且仅包含 1~9 的字符串 $s$，使其恰好有 $N$ 对 $(l,r)(1\leq l\leq r\leq |s|)$ 满足 $s$ 的第 $l$ 到第 $r$ 个字符顺次拼接组成的数是 7 的倍数。

输出任意一组解即可。

```input1
2
```

```output1
142
```

```input2
3
```

```output2
77
```

- $ 1\ \leq\ N\ \leq\ 10^6 $
- 入力されるはすべて整数である

$ (l,r)=(1,2),(2,3) $ の $ 2 $ つが条件を氦郡筏蓼梗

### Sample Explanation 2
