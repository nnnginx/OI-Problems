## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc137/tasks/arc137_b

$ 0,1 $ からなるLさ $ N $ の整数列 $ A=(A_1,A_2,\cdots,A_N) $ が与えられます．

あなたはこれから，次の操作をちょうど $ 1 $ 回行います．

- $ A $ の**BAする**部分列をxび，そこに含まれる要素を flip する．つまり，$ 0 $ ならば $ 1 $ に浃ǎ$ 1 $ ならば $ 0 $ に浃à耄 なお，ここでxぶ部分列は空であることもSされ，その龊 $ A $ の要素は全く浠しない．

あなたのスコアは，$ A $ に含まれる $ 1 $ の数です． あなたが取るスコアとしてあり得るが何通りあるか求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ A_1 $ $ A_2 $ $ \cdots $ $ A_N $

## 输出格式
答えを出力せよ．

## 题目大意
给定一个长度为 $n$ 的由 $0,1$ 组成的整数序列 $A=(A_1,A_2,\cdots,A_n)$ 。你可以做以下的操作**一次且仅一次**：

- 选择 $A$ 的一个连续的子段，对该子段进行反转操作，也就是将 $0$ 变成 $1$ ,将 $1$ 变成 $0$ 。注意，你可以选择一个空字段，这就相当于你什么都没有做。

最后 $A$ 中的 $1$ 的个数，是你能获得的分数。请问你有多少种可能的得分。

```input1
4
0 1 1 0
```

```output1
4
```

```input2
5
0 0 0 0 0
```

```output2
6
```

```input3
6
0 1 0 1 0 1
```

```output3
3
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 3\ \times\ 10^5 $
- $ 0\ \leq\ A_i\ \leq\ 1 $
- 入力されるはすべて整数

### Sample Explanation 1

スコアとしてあり得るは，$ 0,1,2,3 $ の $ 4 $ 通りです． 例えば，$ A $ の $ 2 $ 番目から $ 4 $ 番目までの要素を flip すると，$ A=(0,0,0,1) $ となり，スコアは $ 1 $ になります．
