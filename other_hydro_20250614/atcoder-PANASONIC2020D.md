## 题目描述
[problemUrl]: https://atcoder.jp/contests/panasonic2020/tasks/panasonic2020_d

この}では、英小文字からなる文字列のみを考えます。

文字列 $ s,\ t $ は以下の条件を氦郡工趣 **同型** であるといいます。

- $ |s|\ =\ |t| $ である。
- 任意の $ i,\ j $ にし次のいずれかが成立する。
  - $ s_i\ =\ s_j $ かつ $ t_i\ =\ t_j $
  - $ s_i\ \neq\ s_j $ かつ $ t_i\ \neq\ t_j $

たとえば、`abcac` と `zyxzx` は同型ですが、`abcac` と `ppppp` は同型ではありません。

文字列 $ s $ は以下の条件を氦郡工趣 **市** であるといいます。

- 任意の $ s $ と同型な文字列 $ t $ にし、$ s\ \leq\ t $ が成立する。ただしここで $ \leq $ は辞での比^を表す。

たとえば、`abcac` は市韦扦工、`zyxzx` はそれより辞で小さい `abcac` と同型のため市韦扦悉りません。

整数 $ N $ が与えられます。 Lさ $ N $ の市韦挝淖至肖蛉て、辞でNで出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
Lさ $ N $ の市韦挝淖至肖 $ K $ あり、辞で $ w_1,\ \ldots,\ w_K $ であるとする。 このとき以下の形式で出力せよ。

> $ w_1 $ $ : $ $ w_K $

## 题目大意
若等长的字符串 $s,t$ 满足以下条件，则称 $s,t$ 同构：

- $\forall s_i=s_j$，$t_i=t_j$。
- $\forall s_i\neq s_j$，$t_i \neq t_j$。

如果字符串 $s$ 满足除 $s$ 外所有与它同构的字符串的字典序均大于其本身，则称 $s$ 是 **标准型** 字符串。

请你求出长度为 $n$ 的所有标准型字符串。

$n \leq 10$
