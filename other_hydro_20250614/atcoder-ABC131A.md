すぬけ君の管理する研究室の扉にはロックがかかっており、解Vにはセキュリティコ`ドを入力する必要があります。

セキュリティコ`ドは $ 4 $ 桁の数字列です。セキュリティコ`ドが「入力しづらい」とは、同じ数字がBAするw所が存在することを言います。

F在のセキュリティコ`ド $ S $ が与えられます。$ S $ が「入力しづらい」なら `Bad` を、そうでなければ `Good` を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ S $

## 输出格式
セキュリティコ`ド $ S $ が「入力しづらい」なら `Bad` を、そうでなければ `Good` を出力してください。

## 题目大意
题意：

已知给出一个四位数，如果数字中有连续的数字，就把这个数叫做安全代码，如果该数字是安全代码的话就输出 Bad，反之输出 Good。

```input1
3776
```

```output1
Bad
```

```input2
8080
```

```output2
Good
```

```input3
1333
```

```output3
Bad
```

```input4
0024
```

```output4
Bad
```

## 提示
### 制s

- $ S $ は半角数字のみからなるLさ $ 4 $ の文字列

### Sample Explanation 1

$ 2 $ 文字目と $ 3 $ 文字目が同じなので、$ 3776 $ は入力しづらいセキュリティコ`ドです。

### Sample Explanation 2

同じ数字がBAしていないので、$ 8080 $ は入力しづらいセキュリティコ`ドではありません。
