## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc264/tasks/abc264_a

文字列 `atcoder` の $ L $ 文字目から $ R $ 文字目までを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ L $ $ R $

## 输出格式
答えを出力せよ。

## 题目大意
请从字符串 `atcoder` 的第 $L$ 个字符输出到第 $R$ 个字符。

```input1
3 6
```

```output1
code
```

```input2
4 4
```

```output2
o
```

```input3
1 7
```

```output3
atcoder
```

## 提示
### 制s

- $ L,R $ は整数
- $ 1\ \le\ L\ \le\ R\ \le\ 7 $

### Sample Explanation 1

`atcoder` の $ 3 $ 文字目から $ 6 $ 文字目までを出力すると `code` となります。
