## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc060/tasks/abc060_b

あなたは、正の整数をいくつかxび、それらのt和を求めます。

xぶ数の上限や、xぶ整数の数に制限はありません。 どんなに大きな整数をxんでもよいですし、整数を $ 5000 $ 兆xんでもよいです。 ただし、xぶ数はすべて $ A $ の倍数でなくてはいけません。 また、少なくとも $ 1 $ つは整数をxばなくてはいけません。

そしてt和を $ B $ で割ったあまりが $ C $ となるようにしたいです。 こうなるように整数をxぶことが出来るか判定してください。

出来るならば `YES`、そうでないならば `NO` を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $

## 输出格式
`YES` か `NO` を出力する。

## 题目大意
问在A的倍数里有没有除B余C的，如果有输出"YES"(双引号不输出)，否则输出"NO"(双引号不输出)

```input1
7 5 1
```

```output1
YES
```

```input2
2 2 1
```

```output2
NO
```

```input3
1 100 97
```

```output3
YES
```

```input4
40 98 58
```

```output4
YES
```

```input5
77 42 36
```

```output5
NO
```

## 提示
### 制s

- $ 1\ Q\ A\ Q\ 100 $
- $ 1\ Q\ B\ Q\ 100 $
- $ 0\ Q\ C $

### Sample Explanation 1

たとえば $ 7,\ 14 $ をxぶとt和は $ 21 $ となり、これを $ 5 $ で割ったあまりは $ 1 $ となります。

### Sample Explanation 2

偶数をいくつ足したとしても、けっして奇数になることはありません。

### Sample Explanation 3

$ 1 $ の倍数、つまりすべての整数がxべるので、$ 97 $ をxべば良いです。
