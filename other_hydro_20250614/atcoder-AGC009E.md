## 题目描述
[problemUrl]: https://atcoder.jp/contests/jrex2017/tasks/agc009_e

\板に、$ N $ の $ 0 $ と $ M $ の $ 1 $ がかれています。 この状Bから、\板にかれている有理数のうち $ K $ をxんで消し、それら $ K $ の有理数の平均を新たにき加える操作をRり返します。 ただし、$ N+M-1 $ は $ K-1 $ で割り切れるものとします。

このとき、操作ができなくなるまでこの操作をRり返すと最K的に\板には $ 1 $ つの有理数がかれた状Bになります。

この残った有理数のとしてありうるものの数を $ 10^9\ +\ 7 $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ K $

## 输出格式
最後に残った有理数のとしてありうるものの数を $ 10^9\ +\ 7 $ で割ったあまりを出力せよ。

## 题目大意
黑板上有 $n$ 个 $0$ 和 $m$ 个 $1$，我们每次选择 $k$ 个数字将其擦除，然后把它们的平均数写上去，这样一直操作直到只剩下一个数字，问剩下的这个数字有多少种不同的情况。

答案对 $10^9+7$ 取模。

$1 \leq n,m \leq 2000,2 \leq k \leq 2000$

保证 $n+m-1$ 能被 $k-1$ 整除。

```input1
2 2 2
```

```output1
5
```

```input2
3 4 3
```

```output2
9
```

```input3
150 150 14
```

```output3
937426930
```

## 提示
### 制s

- $ 1\ Q\ N,\ M\ Q\ 2000 $
- $ 2\ Q\ K\ Q\ 2000 $
- $ N+M-1 $ は $ K-1 $ で割り切れる。

### Sample Explanation 1

最後に残る有理数としてありうるものは、$ \frac{1}{4},\ \frac{3}{8},\ \frac{1}{2},\ \frac{5}{8},\ \frac{3}{4} $ の $ 5 $ 通りです。 例えば $ \frac{3}{8} $ は、以下のような操作で最後に残ります。 - $ 0,1 $ を消して $ \frac{1}{2} $ をく。 - $ \frac{1}{2},1 $ を消して $ \frac{3}{4} $ をく。 - $ 0,\frac{3}{4} $ を消して $ \frac{3}{8} $ をく。
