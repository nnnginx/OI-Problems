## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc292/tasks/abc292_d

点に $ 1 $ から $ N $ の番号が、xに $ 1 $ から $ M $ の番号がついた $ N $ 点 $ M $ xのo向グラフが与えられます。x $ i $ は点 $ u_i $ と点 $ v_i $ をYんでいます。

このグラフのすべてのBY成分が次の条件を氦郡工どうかを判定してください。

- そのBY成分に含まれる点の数とxの本数が等しい。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ u_1 $ $ v_1 $ $ \vdots $ $ u_M $ $ v_M $

## 输出格式
すべてのBY成分が条件を氦郡工胜椁 `Yes` と、そうでなければ `No` と出力せよ。

## 题目大意
- 有 $n$ 个顶点。
- 有 $m$ 条无向边。
- 第 $i$ 条无向边连接 $u_i,v_i$。
- 若 $(u_i,v_i)$ 和 $(u_j,v_j)$ 是同一条边，当且仅当 $i=j$。
- 问是否所有连通块里的点数和边数都相等。
- $1\le n,m\le 2\times 10^5$，$1\le u_i\le v_i\le n$。

```input1
3 3
2 3
1 1
2 3
```

```output1
Yes
```

```input2

1 2
2 3
3 4
3 5
1 5
```

```output2
Yes
```


```output3
No
```

## 提示
### 注

**o向グラフ** とは、xに向きのoいグラフのことをいいます。  
あるグラフの **部分グラフ** とは、元のグラフのいくつかの点といくつかのxをxんでできるグラフのことをいいます。  
グラフが **BY** であるとは、グラフに含まれるすべての点同士がxをU由して互いに行き来できることをいいます。  
**BY成分** とは、BYな部分グラフのうち、そのグラフを含んだより大きいBYな部分グラフが存在しないものをいいます。

### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 0\ \leq\ M\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ u_i\ \leq\ v_i\ \leq\ N $
- 入力はすべて整数

### Sample Explanation 1

このグラフには点 $ 1 $ のみからなるBY成分と点 $ 2,3 $ からなるBY成分があります。 前者には $ 1 $ 本のx(x $ 2 $ )が、後者には $ 2 $ 本のx(x $ 1,3 $ )が含まれており、条件を氦郡筏蓼埂
