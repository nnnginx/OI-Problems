## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc046/tasks/agc046_b

k $ A $ マス横 $ B $ マスのマス目があり、そのすべてのマスは白くTられています。このマス目に、以下の操作をRり返し行います。

- F在のマス目がk $ a $ マス横 $ b $ マスであるとする。kまたは横をxぶ。
  - kをxんだ龊悉膝蕙鼓郡紊悉 $ 1 $ 行を追加し、k $ a+1 $ マス横 $ b $ マスのマス目にする。
  - 横をxんだ龊悉膝蕙鼓郡斡窑 $ 1 $ 列を追加し、k $ a $ マス横 $ b+1 $ マスのマス目にする。
- これにより追加されたマスのうちちょうど $ 1 $ マスを\くTり、追加された残りのマスを白くTる。

最K的にマス目がk $ C $ マス横 $ D $ マスになったとするとき、最K的なマス目のなるTられ方としてありうるものの数を $ 998244353 $ で割った余りを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ D $

## 输出格式
最K的なマス目のなるTられ方としてありうるものの数を $ 998244353 $ で割った余りを出力せよ。

```input1
1 1 2 2
```

```output1
3
```

```input2
2 1 3 4
```

```output2
65
```

```input3
31 41 59 265
```

```output3
387222020
```

## 提示
### 制s

- $ 1\ \leq\ A\ \leq\ C\ \leq\ 3000 $
- $ 1\ \leq\ B\ \leq\ D\ \leq\ 3000 $
- $ A,B,C,D $ は整数である

### Sample Explanation 1

左下以外の $ 3 $ マスの中の任意の $ 2 $ マスが\くTられているようなTられ方が条件を氦郡筏蓼埂
