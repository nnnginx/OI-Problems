## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc091/tasks/abc091_a

小学生の高蚓はj店にやってきました。

高蚓は $ A $ 矣藏と $ B $ 矣藏の $ 2 $ 枚を持っており，$ C $ 窑违モチャをIいたいと思っています。高蚓はオモチャをIうことができるでしょうか？

なお，高蚓は高蛲豕に住んでいるため，日本窑扦悉りえないような硬を持っていることもあります。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $

## 输出格式
オモチャをIえるならば `Yes`，Iえないならば `No` を出力せよ。

## 题目大意
输入$A,B,C$

小$K$有一张$A$元的硬币和一个$B$元的硬币

问他的钱是否够买个$C$元的东西(不是刚好等于$C$，大于也可以)

如果可以输出$Yes\ \ \ \ $否则输出$No$

```input1
50 100 120
```

```output1
Yes
```

```input2
500 100 1000
```

```output2
No
```

```input3
19 123 143
```

```output3
No
```

```input4
19 123 142
```

```output4
Yes
```

## 提示
### 制s

- 入力は全て整数である
- $ 1\ \leq\ A,\ B\ \leq\ 500 $
- $ 1\ \leq\ C\ \leq\ 1000 $

### Sample Explanation 1

高蚓は$ 50\ +\ 100\ =\ 150 $页证盲皮い毪郡幔$ 120 $窑韦もちゃがIえます。

### Sample Explanation 2

高蚓は$ 500\ +\ 100\ =\ 600 $页证盲皮い蓼工，$ 1000 $窑韦もちゃはIえません。

### Sample Explanation 3

高蛲豕には$ 19 $矣藏や$ 123 $矣藏も存在します。使いにくいですね。
