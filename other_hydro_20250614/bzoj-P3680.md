## 题目描述

gty 又虐了一场比赛，被虐的Xm们决定吊打 gty。gty 见大势不好机智的分出了 $n$ 个分身，但还是被人多势众的Xm抓住了。Xm们将 $n$ 个 gty 吊在 $n$ 根绳子上，每根绳子穿过天台的一个洞。这 $n$ 根绳子有一个公共的绳结 $x$。吊好 gty 后Xm们发现由于每个 gty 重力不同，绳结 $x$ 在移动。Xm wangxz 脑洞大开的决定计算出 $x$ 最后停留处的坐标，由于他太弱了决定向你求助。
不计摩擦，不计能量损失，由于 gty 足够矮所以不会掉到地上。

## 输入格式

输入第一行为一个正整数 $n(1\le n\le 10000)$，表示 gty 的数目。
接下来 $n$ 行，每行三个整数$x_i,y_i,w_i$，表示第 $i$ 个 gty 的横坐标，纵坐标和重力。

## 输出格式

输出 $1$ 行两个浮点数（保留到小数点后 $3$ 位），表示最终 $x$ 的横、纵坐标。

```input1
3
0 0 1
0 2 1
1 1 1
```

```output1
0.577 1.000
```

## 数据规模与约定

对于 $20\%$ 的数据，gty 排列成一条直线。

对于 $50\%$ 的数据，$1\le n\le 1000$。

对于 $100\%$ 的数据，$1\le n\le 10000,-100000\le x_i,y_i\le 100000$
