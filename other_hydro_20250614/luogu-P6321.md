## 题目背景
Mirko 在拿放大镜看杂志。

## 题目描述
他正在看一面 $r$ 行 $c$ 列的杂志。放大镜有两个参数 $zr,zc$，表示把每个字符都转换成一个 $zr$ 行 $zc$ 列的矩阵。

给出原始杂志，请你帮他找出杂志在放大镜下的影像。

## 输入格式
输入第一行四个整数 $r,c,zr,zc$。

接下来的 $r$ 行，每行 $c$ 个字符。字符可能为数字，字母或 `.`。

## 输出格式
输出一个 $r\times zr$ 行，$c\times zc$ 列的矩阵，表示放大镜下的影像。

```input1
3 3 1 2
.x.
x.x
.x. 
```

```output1
..xx..
xx..xx
..xx.. 
```

```input2
3 3 2 1
.x.
x.x
.x.
```

```output2
.x.
.x.
x.x
x.x
.x.
.x.
```

## 提示
#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1\le r,c\le 50$，$1\le zr,zc\le 5$。
