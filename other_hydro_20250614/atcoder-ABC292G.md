## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc292/tasks/abc292_g

数字( `0123456789` )と `?` からなるLさ $ M $ の文字列の列 $ S_1,\ldots,S_N $ が与えられます。

`?` を独立に数字に置きQえる方法は $ 10^q(q $ は $ S_1,\ldots,S_N $ に含まれる `?` の数の合$ ) $ 通りありますが、そのうち置きQえ後の文字列をそれぞれ整数とみなしたときに $ S_1\lt\ S_2\ \lt\ \ldots\ \lt\ S_N $ が成り立つようなものが何通りあるかを $ 998244353 $ で割った余りを求めてください。

なお、`?` を置きQえた後の $ S_i $ は先^に $ 1 $ 以上の `0` がBAしていてもいません。例えば、`0000000292` を整数とみなすと $ 292 $ となります。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ S_1 $ $ \vdots $ $ S_N $

## 输出格式
答えを出力せよ。

## 题目大意
你有 $n$ 个数，每个数长度为 $m$。

不过这 $n$ 个数中，可能有某些位不确定，需要你在每个 `?` 位置上 $0$ 到 $9$ 之间填一个数。设你填出来的序列是 $\{S_i\}$。

请你求出，在所有可能的填数方案中，有多少种满足 $S_1 < S_2 < \dots < S_n$？对 $998244353$ 取模。允许前导零存在。

数据范围：$n,m \le 40$。

```input1
3 2
?0
??
05
```

```output1
4
```

```input2
2 1
0
0
```

```output2
0
```

```input3
68?7?6?800
?3??2???23
?442312158
??2??921?8
????5?96??
```

```output3
137811792
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 40 $
- $ 1\ \leq\ M\ \leq\ 40 $
- $ N,M $ は整数
- $ S_i $ は数字と `?` からなるLさ $ M $ の文字列
 
### Sample Explanation 1

条件を氦郡怪盲Qえ方は以下の $ 4 $ 通りです。 - $ S_1 $ の $ 1 $ 文字目の `?` を `0` に、$ S_2 $ の $ 1,2 $ 文字目の `?` をそれぞれ `0`, `1` に置きQえる。 - $ S_1 $ の $ 1 $ 文字目の `?` を `0` に、$ S_2 $ の $ 1,2 $ 文字目の `?` をそれぞれ `0`, `2` に置きQえる。 - $ S_1 $ の $ 1 $ 文字目の `?` を `0` に、$ S_2 $ の $ 1,2 $ 文字目の `?` をそれぞれ `0`, `3` に置きQえる。 - $ S_1 $ の $ 1 $ 文字目の `?` を `0` に、$ S_2 $ の $ 1,2 $ 文字目の `?` をそれぞれ `0`, `4` に置きQえる。

### Sample Explanation 3

答えを $ 998244353 $ で割った余りを求めてください。
