## 题目描述
小 A 有一个口袋，里面可以装整数。他从 $1$ 开始，按从小到大的顺序，依次将每个整数装入口袋。

但是口袋是有限的，大小为 $n$，这就是说，口袋里所有的数字的和不能够超过 $n$。


## 输入格式
输入只有一行一个整数，表示整数 $n$。

## 输出格式
输出若干行，每行一个整数，按从小到大的顺序输出口袋里的数字。

```input1
15
```

```output1
1
2
3
4
5
```

```input2
9
```

```output2
1
2
3
```

## 提示
### 数据规模与约定

- 对于 $40\%$ 的数据，$n\le 10^3$；
- 对于 $100\%$ 的数据，$1\le n \le 10^9$。
