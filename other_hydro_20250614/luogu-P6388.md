## 题目描述
给定一个字符串，请你将其分成三段，并将每一段倒序后按照分段的顺序输出。由于方案有多种，请输出答案字典序最小的那一种。

## 输入格式
输入一行为一个字符串，表示待分段的串。

## 输出格式
输出一行为一个字符串，表示答案串，需要保证字典序最小。

```input1
dcbagfekjih
```

```output1
abcdefghijk
```

```input2
mobitel
```

```output2
bometil
```

```input3
anakonda
```

```output3
aanadnok
```

## 提示
#### 数据规模与约定

对于 $100\%$ 的数据，保证输入的字符串长度在 $3\sim 50$ 之间。
