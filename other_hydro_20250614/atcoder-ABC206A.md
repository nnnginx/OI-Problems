## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc206/tasks/abc206_a

ABC 国の消M税率は $ 8 $ パ`セントです。  
 ABC 国にはエナジ`ドリンク屋さんがあります。ここでは、エナジ`ドリンク $ 1 $ 本を、税iき $ N $ 窑秦婴筏皮い蓼埂  
 ここに消M税を加算した後の金~は $ \lfloor\ 1.08\ \times\ N\ \rfloor $ 窑趣胜辘蓼埂￥郡坤贰g数 $ x $ にし、$ \lfloor\ x\ \rfloor $ は $ x $ 以下の最大の整数を表します。   
 この金~が定の $ 206 $ 窑瑜臧菠い胜 `Yay!` 、定と等しいなら `so-so` 、定より高いなら `:(` と出力して下さい。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
输入正整数 $n$ ，求 $1.08×n$ 的**整数部分**与 $206$ 的大小关系。

如果它比 $206$ 小，输出 Yay!

如果它与 $206$ 相等，输出 so-so

如果它比 $206$ 大，输出 :(

```input1
180
```

```output1
Yay!
```

```input2
200
```

```output2
:(
```

```input3
191
```

```output3
so-so
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 300 $
- $ N $ は整数

### Sample Explanation 1

$ N=180 $ であるとき、税z格は $ \lfloor\ 180\ \times\ 1.08\ \rfloor\ =\ 194 $ 窑扦工、この金~は定の $ 206 $ 窑瑜臧菠い扦埂

### Sample Explanation 3

この龊稀⑺稗z格がちょうど定の $ 206 $ 窑纫恢陇筏蓼埂
