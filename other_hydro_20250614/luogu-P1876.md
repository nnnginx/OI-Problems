## 题目描述
首先所有的灯都是关的（注意是关！），编号为 $1$ 的人走过来，把是 $1$ 的倍数的灯全部打开，编号为 $2$ 的人把是 $2$ 的倍数的灯全部关上，编号为 $3$ 的人又把是 $3$ 的倍数的灯开的关上，关的开起来……直到第 $N$ 个人为止。

给定 $N$，求 $N$ 轮之后，还有哪几盏是开着的。

## 输入格式
一个数 $N$，表示灯的个数和操作的轮数。

## 输出格式
若干数，表示开着的电灯编号。

```input1
5
```

```output1
1 4
```

## 提示
**【数据范围】**

对于 $100 \%$ 的数据，$1 \le N \le 2^{40}$。
