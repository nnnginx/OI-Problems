<p>Consider the equation

</p><blockquote><p align="justify">

</p><p><i>x</i><sup>2</sup> - <i>n</i><i>y</i><sup>2</sup> = 1
</p></blockquote><p align="justify">

</p><p>where <i>n</i> is some integer.<br>

</p><p>Find the smallest strictly positive integer solutions (<i>x</i>, <i>y</i>) for a given <i>n</i>.

</p><h3>Input</h3>
<p>The number of test cases <i>t</i> (around 30), followed by a list of <i>t</i> values of <i>n</i> (2 ¡Ü <i>n</i> ¡Ü 1000). You can assume that the equation can be solved for all values of <i>n</i> in the input set.

</p><h3>Output</h3>
<p>For every test case, the values of <i>x</i> and <i>y</i> separated by a space character, on separate lines.

</p><h3>Example</h3>

<pre><b>Input:</b>
