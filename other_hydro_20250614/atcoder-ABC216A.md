## 题目大意
### 题目描述

给你一个实数 $X.Y$ ，其中 $Y$ 是一个数码。

请你按如下格式输出：

- 若 $0 \le Y \le 2$ ，输出 $X-$ ；
- 若 $3 \le Y \le 6$ ，输出 $X$ ；
- 若 $7 \le Y \le 9$ ，输出 $X+$ 。

---

### 数据范围

- $1 \le X \le 15$
- $0 \le Y \le 9$
- $X$ 和 $Y$ 都是整数。

---

### 输入格式

输入一个实数 $X.Y$ 。

---

### 输出格式

输出答案。

$\text{Translated by @nr0728.}$

```input1
15.8
```

```output1
15+
```

```input2
1.0
```

```output2
1-
```

```input3
12.5
```

```output3
12
```

## 提示
### 制s

- $ 1\ \leq\ X\ \leq\ 15 $
- $ 0\ \leq\ Y\ \leq\ 9 $
- $ X $ と $ Y $ は整数

### Sample Explanation 1

`15 +` のような出力は不正解とみなされます。 $ X $ と `+` のgや、$ X $ と `-` のgには空白を入れずに出力してください。

### Sample Explanation 2

`1.00` や `1` などのが入力として与えられることはありません。
