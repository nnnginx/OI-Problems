## 题目描述
[problemUrl]: https://atcoder.jp/contests/tenka1-2019/tasks/tenka1_2019_c

$ N $ の石が一列にKんでおり、すべての石は白か\でTられています。 石の状BはLさ $ N $ の文字列 $ S $ で表され、$ S $ の $ i $ 文字目が `.` のとき左から $ i $ 目の石が白であり、`#` のとき左から $ i $ 目の石が\であることを表します。

高蚓は、$ 0 $ 以上の石の色を\または白に涓し、\い石のすぐ右に白い石があるようなw所がないようにしたいです。 色を涓する必要のある石の数の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
色を涓する必要のある石の数の最小を出力せよ。

## 题目大意
有 N 块石头连续排列，每块石头都被漆成白色或黑色。 字符串 S 代表石头的颜色。 如果 S 的第i个字符是 `.`，则从左起第 i 个石头是白色；如果字符是 `＃`，则第 i 个石头是黑色。

高桥希望将某些石头的颜色更改为黑色或白色，以使黑色石头的右侧不会立即出现白色石头。 找到需要重新着色的最少数量的石头。

```input1
3
#.#
```

```output1
1
```

```input2
5
#.##.
```

```output2
2
```

```input3
9
.........
```

```output3
0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ S $ は `.`, `#` のみからなるLさ $ N $ の文字列である

### Sample Explanation 1

例えば、$ 1 $ 目の石の色を白に涓すればよいです。
