## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc237/tasks/abc237_h

英小文字からなる文字列 $ S $ があります。  
 叭栈匚膜韦长趣肖りを考えている高虿┒啶んは、$ S $ の部分文字列のうち回文となっているものをいくつかxび、小}S子さんに教えることにしました。

小}S子さんは、教えられた回文のうち $ 2 $ つであって、一方が他方の部分文字列になっているようなものが存在すると、怒ります。

小}S子さんが怒らないという条件のもとで、高虿┒啶んは最大でいくつの回文をxぶことができますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
答えを出力せよ。

## 题目大意
给定一个字符串, 你需要从中选出若干回文子串, 并且使得选出的串不存在某一个是另一个的子串, 问最多能选出多少子串.

```input1
ababb
```

```output1
3
```

```input2
xyz
```

```output2
3
```

```input3
xxxxxxxxxx
```

```output3
1
```

## 提示
### 注

$ S $ の**部分文字列**とは、$ S $ の先^から $ 0 $ 文字以上、末尾から $ 0 $ 文字以上削除して得られる文字列のことをいいます。  
 例えば、`ab` は `abc` の部分文字列ですが、`ac` は `abc` の部分文字列ではありません。

### 制s

- $ 1\ \leq\ |S|\ \leq\ 200 $
- $ S $ は英小文字からなる

### Sample Explanation 1

`aba` 、`bab` 、`bb` の $ 3 $ つの回文をxぶことができます。

### Sample Explanation 2

`x` 、`y` 、`z` の $ 3 $ つの回文をxぶことができます。
