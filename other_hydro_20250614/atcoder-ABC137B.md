## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc137/tasks/abc137_b

数直上に $ 2000001 $ の石が置かれています。これらの石の座摔 $ -1000000,\ -999999,\ -999998,\ \ldots,\ 999999,\ 1000000 $ です。

これらの石のうち、あるBAする $ K $ の石が\でTられており、それ以外の石は白でTられています。

また、座 $ X $ にある石は\でTられていることが分かっています。

\でTられている石が置かれている可能性のある座摔颏工伽啤⑿・丹ろに出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ K $ $ X $

## 输出格式
\でTられている石が置かれている可能性のある座摔颏工伽啤⒖瞻驻乔切って小さいに出力せよ。

## 题目大意
已知有连续的 $K$ 个整数，其中一个是 $X$ ，按升序输出所有可能出现在这 $K$ 个整数中的数。

```input1
3 7
```

```output1
5 6 7 8 9
```

```input2
4 0
```

```output2
-3 -2 -1 0 1 2 3
```

```input3
1 100
```

```output3
100
```

## 提示
### 制s

- $ 1\ \leq\ K\ \leq\ 100 $
- $ 0\ \leq\ X\ \leq\ 100 $
- 入力中のはすべて整数である。

### Sample Explanation 1

\でTられた石の数が $ 3 $ であることと、座 $ 7 $ の石が\でTられていることが分かっています。このとき、次の $ 3 $ 通りの可能性が考えられます。 - \でTられた $ 3 $ の石は座 $ 5,6,7 $ に置かれている。 - \でTられた $ 3 $ の石は座 $ 6,7,8 $ に置かれている。 - \でTられた $ 3 $ の石は座 $ 7,8,9 $ に置かれている。 よって、\でTられている石が置かれている可能性のある座摔 $ 5,6,7,8,9 $ の $ 5 $ つです。

### Sample Explanation 2

の座摔它\でTられている石が置かれている可能性もあります。
