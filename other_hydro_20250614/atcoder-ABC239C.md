## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc239/tasks/abc239_c

$ xy $ 座似矫嫔悉 $ 2 $ つの格子点 $ (x_1,\ y_1),\ (x_2,\ y_2) $ からの距xがともに $ \sqrt{5} $ である格子点は存在しますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ x_1 $ $ y_1 $ $ x_2 $ $ y_2 $

## 输出格式
条件を氦郡垢褡拥悚存在する龊悉 `Yes` を、存在しない龊悉 `No` を出力せよ。

## 题目大意
在平面直角坐标系上有两个点，第 $i$ 个点的坐标为 $(x_i,y_i)$ 。现在依次输入 $x_1,y_1,x_2,y_2$ ，问：是否存在一个点，使得这个点与前文所述的两个点的距离都是 $\sqrt5$ ?

```input1
0 0 3 3
```

```output1
Yes
```

```input2
0 1 2 3
```

```output2
No
```

```input3
1000000000 1000000000 999999999 999999999
```

```output3
Yes
```

## 提示
### 注

$ xy $ 座似矫嫔悉摔る点のうち、$ x $ 座摔 $ y $ 座摔ともに整数である点を格子点と呼びます。  
 また、$ xy $ 平面上の $ 2 $ 点 $ (a,\ b),\ (c,\ d) $ の距xをユ`クリッド距x $ \sqrt{(a\ -\ c)^2\ +\ (b-d)^2} $ として定xします。

参考として、$ xy $ 平面の $ (0,\ 0) $ の上に\丸を、$ (0,\ 0) $ からの距xが $ \sqrt{5} $ となる格子点の上に白丸をいた恧弦韵陇韦瑜Δ摔胜辘蓼埂($ x,y $ いずれかが整数となる部分に目盛りを引いています。)

### 制s

- $ -10^9\ \leq\ x_1\ \leq\ 10^9 $
- $ -10^9\ \leq\ y_1\ \leq\ 10^9 $
- $ -10^9\ \leq\ x_2\ \leq\ 10^9 $
- $ -10^9\ \leq\ y_2\ \leq\ 10^9 $
- $ (x_1,\ y_1)\ \neq\ (x_2,\ y_2) $
- 入力はすべて整数である。

### Sample Explanation 1

\- 点 $ (2,1) $ と $ (x_1,\ y_1) $ の距xは $ \sqrt{(0-2)^2\ +\ (0-1)^2}\ =\ \sqrt{5} $ - 点 $ (2,1) $ と $ (x_2,\ y_2) $ の距xは $ \sqrt{(3-2)^2\ +\ (3-1)^2}\ =\ \sqrt{5} $ - 点 $ (2,\ 1) $ は格子点 なので点 $ (2,\ 1) $ は条件を氦郡筏蓼埂￥瑜盲 `Yes` を出力します。 なお、点 $ (1,\ 2) $ も条件を氦郡工长趣同にして_Jできます。

### Sample Explanation 2

条件を氦郡垢褡拥悚洗嬖冥筏蓼护蟆￥瑜盲 `No` を出力します。

### Sample Explanation 3

点 $ (10^9\ +\ 1,\ 10^9\ -\ 2) $ および点 $ (10^9\ -\ 2,\ 10^9\ +\ 1) $が条件を氦郡筏蓼埂
