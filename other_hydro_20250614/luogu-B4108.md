## 题目描述
JOI 君有 $A$ 日元，一根铅笔 $5$ 日元，问 JOI 君最多可以买多少根铅笔。

## 输入格式
一行一个正整数 $A$。

## 输出格式
一行一个整数，表示答案。

```input1
9
```

```output1
1
```

```input2
10
```

```output2
2
```

```input3
3
```

```output3
0
```

```input4
100
```

```output4
20
```

## 提示
### 样例解释
$1$ 根铅笔 $5$ 日元，$2$ 根铅笔 $10$ 日元，$3$ 根铅笔 $15$ 日元。

### 数据范围

$1\leq A\leq 100$。
