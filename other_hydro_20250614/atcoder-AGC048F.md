## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc048/tasks/agc048_f

すぬけくんは大きな\板をもらいました． これに喜んだすぬけくんは，まず，\板にいくつかの正整数をきました． 次にすぬけくんは，\板にかれている整数がなくなるまで，以下の操作をRり返し行いました．

- \板にかれている整数を $ 1 $ つxび，消す． 消した整数を $ x $ とする． 次に，$ x $ を $ 2 $ で割ったあまりをノ`トにhする． 最後に，$ x\ \geq\ 2 $ である龊悉希新たに $ x-1 $ を\板にき加える．

すぬけくんのhは，`0` と `1` からなる文字列 $ S $ によって表されます． これは，すぬけくんが $ i $ 回目の操作でxんだ整数を $ 2 $ で割ったあまりが $ S_i $ であることを表します．

すぬけくんは，最初に\板にいた正整数のMみ合わせを忘れてしまいました． 文字列 $ S $ の情螭ら，最初に\板にいた正整数のMみ合わせとしてありうるものが何通りあるか求めてください． ここで，正整数のMみ合わせ $ a $ と $ b $ がなるとは，ある整数 $ v $ が存在して，$ a $ に含まれる $ v $ の数と $ b $ に含まれる $ v $ の数がなることを意味します． なお，答えは非常に大きくなることがあるので，$ 10^9+7 $ で割ったあまりを求めてください． また，すぬけくんのhがg`っており，条件を氦郡拐整数のMみ合わせが存在しないこともありますが，その龊悉gに $ 0 $ と答えてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ S $

## 输出格式
最初に\板にいた正整数のMみ合わせとしてありうるものの数を $ 10^9+7 $ で割ったあまりを出力せよ．

```input1
101
```

```output1
2
```

```input2
100
```

```output2
0
```

```input3
010101
```

```output3
3
```

```input4
11101000111110111101001011110010111110101111110111
```

```output4
3904
```

## 提示
### 制s

- $ 1\ \leq\ |S|\ \leq\ 300 $
- $ S $ は `0` と `1` からなる文字列．

### Sample Explanation 1

最初に\板にいた整数のMみ合わせとしてあり得るのは，$ \{1,2\},\ \{3\} $ の $ 2 $ つです．

### Sample Explanation 2

最初に\板にいた整数のMみ合わせとしてあり得るものはありません．

### Sample Explanation 3

最初に\板にいた整数のMみ合わせとしてあり得るのは，$ \{2,2,2\},\ \{2,4\},\ \{6\} $ の $ 3 $ つです．
