## 题意简述
有 $N$ 名选手，编号为 $1 \sim N$，一起参加了有 $M$ 道题目的比赛，题目编号 $1 \sim M$。$S_{ij}$ 为 `o` 或 `x` 分别表示选手 `i` 能否解出题目 `j`。请你求出有多少种组成一个队伍的方式，使得这个队伍能共同解出 $M$ 道题。共同解出指，一道题目至少被队伍中的一个选手解出。

一个队伍有且仅有 $2$ 人。

```input1
5 5
ooooo
oooxx
xxooo
oxoxo
xxxxx
```

```output1
5
```

```input2
3 2
ox
xo
xx
```

```output2
1
```

```input3
2 4
xxxx
oxox
```

```output3
0
```
