## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc249/tasks/abc249_e

英小文字のみからなる文字列 $ X $ にし、以下の手Aきによって文字列を得ることを考えます。

- $ X $ をなる文字がOり合っている部分で分割する。
- 分割した各文字列 $ Y $ にして、$ Y $ を $ Y $ を成する文字と $ Y $ のLさをげた文字列に置きQえる。
- 元の番を保ったまま、置きQえた文字列をすべてげる。

例えば、`aaabbcccc` の龊稀`aaa`,`bb`,`cccc` に分けられ、それぞれを `a3`,`b2`,`c4` に置きQえ、その番のままげることにより `a3b2c4` を得ます。また、`aaaaaaaaaa` の龊稀`a10` になります。

Lさ $ N $ の英小文字のみからなる文字列 $ S $ のうち、上の手Aきによって得られた文字列 $ T $ とのLさを比べたとき、$ T $ の方が短いものの数を $ P $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ P $

## 输出格式
答えを出力せよ。

## 题目大意
给定一种字符串压缩算法：对于连续的相同字母，会压缩成 该字母 + 出现次数 的形式，例如 `aaabbcccc` 会被压缩成 `a3b2c4`，`aaaaaaaaaa` 会被压缩成 `a10`。

字符集为英文小写字母，给定 $ n, p $，求对于所有长度为 $ n $ 的字符串，有多少满足压缩后的字符串长度严格小于原字符串。对 $ p $ 取模。保证 $ p $ 为质数。

```input1
3 998244353
```

```output1
26
```

```input2
2 998244353
```

```output2
0
```

```input3
5 998244353
```

```output3
2626
```

```input4
3000 924844033
```

```output4
607425699
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 3000 $
- $ 10^8\ \le\ P\ \le\ 10^9 $
- $ N,P $ は整数
- $ P $ は素数

### Sample Explanation 1

$ 1,2,3 $ 文字目が全て等しい文字列のみが条件を氦郡筏蓼埂 例えば、`aaa` は `a3` となり条件を氦郡筏蓼工、`abc` は `a1b1c1` となり条件を氦郡筏蓼护蟆

### Sample Explanation 2

`aa` → `a2` のように、Lさが等しいものは条件を氦郡丹胜い长趣俗⒁猡筏皮ださい。

### Sample Explanation 3

`aaabb` や `aaaaa` などが条件を氦郡筏蓼埂

### Sample Explanation 4

条件を氦郡刮淖至肖数を $ P $ で割ったあまりを求めてください。
