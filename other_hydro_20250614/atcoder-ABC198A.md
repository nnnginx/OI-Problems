## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc198/tasks/abc198_a

$ N $ の互いに区eできないお子を、A君とB君で分け合います。 I者とも $ 1 $ 以上の整数のお子を得るような分け方は何通りありますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを整数として出力せよ。

## 题目大意
输入正整数 $n$ ,输出 $n-1$ 的值。

```input1
2
```

```output1
1
```

```input2
1
```

```output2
0
```

```input3
3
```

```output3
2
```

### 制s

- $ N $ は整数
- $ 1\ \leq\ N\ \leq\ 15 $

A君が $ 1 $ 、B君が $ 1 $ 取る方法のみ存在します。
