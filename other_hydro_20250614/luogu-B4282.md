## 题目描述
给定两个正整数 $N$ 和 $M$（$0 < N < 200$，$0 < M < 200$，$N \neq M$），比较两个正整数的大小，然后将较大的一个正整数输出。例如：$N = 145$，$M = 100$，比较后 $145$ 大于 $100$，故输出 $145$。

## 输入格式
输入两个正整数 $N$ 和 $M$（$0 < N < 200$，$0 < M < 200$，$N \neq M$），正整数之间一个空格隔开。


## 输出格式
输出一个正整数，表示 $N$ 和 $M$ 中较大的一个正整数。

```input1
145 100
```

```output1
145
```
