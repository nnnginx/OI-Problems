<p>A positive integer is said to be bitwise prime if the sum of all the bits in its binary representation is a prime number.&nbsp;</p>
<p>You are given two integers a and b. You have to output number of bitwise prime numbers between a and b (inclusive).</p>
<p>&nbsp;</p>
<h3>Input</h3>
<p>First line contains T : number of test cases. (1 &lt;= T &lt;= 10^5)</p>
<p>For next T lines, each test case contains two space seperated integers a and b. (a &lt;= b). 1 &lt;= a, b &lt;= 10^19.</p>
<h3>Output</h3>
<p>For each test case, print the&nbsp;number of bitwise prime numbers between a and b (inclusive).</p>
<p>&nbsp;</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
2<br>1 2<br>1 3

<strong>Output:</strong>
0<br>1&nbsp;</pre>
