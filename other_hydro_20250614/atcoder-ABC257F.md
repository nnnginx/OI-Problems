## 题目大意
存在 $ n $ 个小镇，$ m $ 条传送通道，第 $ i $ 条双向连结 $ u_i, v_i $ 两个小镇，经过每个传送通道需要花费 $ 1 $ 分钟。特别地，可能存在 $ u_i = 0 $，表示该条传送通道只规定了一端，另一端待定。存在 $ n $ 个独立询问，对于 $ i = 1, 2, \cdots, n $，钦定所有未确定的 $ u_i $ 均为 $ i $，求从小镇 $ 1 $ 到小镇 $ n $ 最小耗费的时间。若无法到达输出 $ -1 $。

```input1
3 2
0 2
1 2
```

```output1
-1 -1 2
```

```input2

```
