


## 题目描述
给定一个 d 维空间椭球：
求其与原点 0 的最近距离。
## 输入格式
第一行一个整数 d;
接下来 d 行描述了一个系数矩阵 faijg，满足 aij = aji。
## 输出格式
一行一个实数，表示答案。答案与标准答案误差不超过 10^- 8 即视为正确。

```input1
3
1 0 0
0 0.01 0
0 0 0.04

```

```output1
1.00000000
```

## 提示
对于 100% 的数据，满足 d <= 15; 0 < aii < 1;  |aij|  < 2；
