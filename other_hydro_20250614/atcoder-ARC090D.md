## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc090/tasks/arc090_d

正の整数 $ n $ にし、$ f(n) $ を $ n $ の $ 10 $ M法での桁数と定めます。

整数 $ S $ が与えられます。 正の整数のM $ (l,\ r) $ ($ l\ \leq\ r $) であって、$ f(l)\ +\ f(l\ +\ 1)\ +\ ...\ +\ f(r)\ =\ S $ を氦郡工猡韦数を $ 10^9\ +\ 7 $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
答えを出力せよ。

## 题目大意
对于一个正整数 $n$，设 $f(n)$ 表示它在十进制下的位数。如：$f(1234)=4, f(33)=2, f(101)=3$。给定 $k\ (k\le 10^8)$ ，求正整数对 $(l,r)\ (l\le r)$ 的个数，使得$\sum_{i=l}^{r} f(i)=k$。例子：当 $k=1$时，有九组：$(1,1),(2,2),...,(9,9)$。答案对 $10^9+7$ 取模。

```input1
1
```

```output1
9
```

```input2
2
```

```output2
98
```

```input3
123
```

```output3
460191684
```

```input4
36018
```

```output4
966522825
```

```input5
1000
```

```output5
184984484
```

## 提示
### 制s

- $ 1\ \leq\ S\ \leq\ 10^8 $

### Sample Explanation 1

条件を氦郡 $ (l,\ r) $ のMは $ (1,\ 1) $, $ (2,\ 2) $, $ ... $, $ (9,\ 9) $ の $ 9 $ あります。

### Sample Explanation 2

条件を氦郡 $ (l,\ r) $ のMは $ (1,\ 2) $ や $ (33,\ 33) $ など $ 98 $ あります。
