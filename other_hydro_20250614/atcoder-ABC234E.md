以下の条件を氦郡拐の整数 $ n $ を、 **等差数** と呼びます。

- ($ n $ を先^に余な $ 0 $ を付けずに $ 10 $ M法で表したH、) $ n $ の上から $ i $ 桁目を $ d_i $ とする。このとき、 $ n $ が $ k $ 桁の整数であったとすると、 $ (d_2-d_1)=(d_3-d_2)=\dots=(d_k-d_{k-1}) $ が成立する。
  - この条件は、「 数列 $ (d_1,d_2,\dots,d_k) $ が等差数列である」と言いQえることができる。
  - 但し、 $ n $ が $ 1 $ 桁の整数であるr、 $ n $ は等差数であるものとする。

たとえば、 $ 234,369,86420,17,95,8,11,777 $ は等差数ですが、 $ 751,919,2022,246810,2356 $ は等差数ではありません。

等差数のうち、 $ X $ 以上で最小のものを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $

## 输出格式
答えを整数として出力せよ。

## 题目大意
一个正整数 $n$ 被称为等差数需要满足以下条件：

- 令 $d_i$ 表示 $n$ 在十进制下从最高位开始的第 $i$ 位数字，即 $n = \overline{d_1d_2\cdots d_k}$，其中 $k$ 表示 $n$ 的位数。
- 并且，序列 $(d_1, d_2, \dots, d_k)$ 为等差数列。
- 若 $n$ 是一位整数，则它也被视作为等差数。

例如，$234,369,86420,17,95,8,11,777$ 是等差数，然而 $751,919,2022,246810,2356$ 不是等差数。

请你求出不小于 $X$ 的最小等差数。

```input1
152
```

```output1
159
```

```input2
88
```

```output2
88
```

```input3
8989898989
```

```output3
9876543210
```

## 提示
### 制s

- $ X $ は $ 1 $ 以上 $ 10^{17} $ 以下の整数である

### Sample Explanation 1

$ 152 $ 以上で最小の等差数は $ 159 $ です。

### Sample Explanation 2

$ X $ 自身が等差数である龊悉猡ります。
