## 题目描述
[problemUrl]: https://atcoder.jp/contests/code-festival-2017-quala/tasks/code_festival_2017_quala_d

k $ H $ 行、横 $ W $ 列のマス目があります。 上から $ i $ 行目、左から $ j $ 列目のマスを $ (i,\ j) $ と表します。 また、マス $ (i_1,\ j_1) $ と $ (i_2,\ j_2) $ のgの距xを $ |i_1\ -\ i_2|\ +\ |j_1\ -\ j_2| $ と定xします。

すぬけ君は各マスを 赤 / 黄 / v / 青 のいずれかの色でTろうとしています。 このとき、正の整数 $ d $ にして、次の条件が成り立つようにします。

- 距xがちょうど $ d $ であるようなマスのペアには、なる色がTられている。

条件を氦郡股のTり方をひとつ求めてください。 解は必ず存在することが示せます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ H $ $ W $ $ d $

## 输出格式
条件を氦郡股のTり方をひとつ出力せよ。 色のTり方は次のフォ`マットで出力せよ。 マス $ (i,\ j) $ の色が 赤 / 黄 / v / 青 ならば、$ c_{ij} $ はそれぞれ `R` / `Y` / `G` / `B` とせよ。

> $ c_{11} $$ c_{12} $$ ... $$ c_{1W} $ $ : $ $ c_{H1} $$ c_{H2} $$ ... $$ c_{HW} $

## 题目大意
### 题目描述

给定一个$H \times W$的网格，试给其中所有格子染`RYGB`四种颜色之一，使得网格上任意一对曼哈顿距离为$d$的格子颜色不同。

### 输入格式

一行两个正整数$H ,W , d$

$1 \leq H,W \leq 500 , 1 \leq d \leq H + W - 2$

### 输出格式

输出一个$H \times W$的字符矩阵表示染色方案

```input1
2 2 1
```

```output1
RY
GR
```

```input2
2 3 2
```

```output2
RYB
RGB
```

## 提示
### 制s

- $ 2\ <\ =\ H,\ W\ <\ =\ 500 $
- $ 1\ <\ =\ d\ <\ =\ H\ +\ W\ -\ 2 $

### Sample Explanation 1

距xがちょうど $ 1 $ であるようなマスのペアは、次の $ 4 $ Mです。 右趣耸兢筏郡瑜Δ恕どのペアにもなる色がTられています。 - $ (1,\ 1) $ と $ (1,\ 2) $ : `R` と `Y` - $ (1,\ 2) $ と $ (2,\ 2) $ : `Y` と `R` - $ (2,\ 2) $ と $ (2,\ 1) $ : `R` と `G` - $ (2,\ 1) $ と $ (1,\ 1) $ : `G` と `R`

### Sample Explanation 2

距xがちょうど $ 2 $ であるようなマスのペアは、次の $ 6 $ Mです。 右趣耸兢筏郡瑜Δ恕どのペアにもなる色がTられています。 - $ (1,\ 1) $ と $ (1,\ 3) $ : `R` と `B` - $ (1,\ 3) $ と $ (2,\ 2) $ : `B` と `G` - $ (2,\ 2) $ と $ (1,\ 1) $ : `G` と `R` - $ (2,\ 1) $ と $ (2,\ 3) $ : `R` と `B` - $ (2,\ 3) $ と $ (1,\ 2) $ : `B` と `Y` - $ (1,\ 2) $ と $ (2,\ 1) $ : `Y` と `R`
