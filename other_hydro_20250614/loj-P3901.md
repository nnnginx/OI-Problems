## 题目描述

**译自 [COCI 2022/2023 Contest #2](https://hsin.hr/coci/archive/2022_2023) T4「[Prijateljice](https://hsin.hr/coci/archive/2022_2023/contest2_tasks.pdf)」**

在克尔克的一个夏天晚上，Leona 和 Zoe 去散步。在散步路上，她们发现了一堆单词！她们十分激动并且迅速收集了这些单词：Leona 收集了 $n$ 个，Zoe 收集了 $m$ 个。在欣赏了一会儿之后，她们决定用这些单词玩一个游戏：

在一轮中玩家说出她的词堆中的一个单词。玩家按满足如下条件的方式选择单词：这个单词的字典序比之前的单词大，并且要么这个单词的开头字母与之前单词的开头字母相同，要么这个单词的开头字母是上个单词开头字母在英语字母表中的下一个。如果玩家在一轮中不能说出单词，那么她就输了。

她们交替在每轮中说单词，Leona 首先说出她词堆中字典序最小的单词。如果 Leona 和 Zoe 按最优方式进行游戏，确定谁会赢。

注：如果单词 $A$ 是单词 $B$ 的前缀，或者对于单词 $A$ 和 $B$ 不同的第一个字母，单词 $A$ 的字母在字母表的顺序比单词 $B$ 的靠前，则称单词 $A$ 的字典序比单词 $B$ 小。

## 输入格式

输入第一行包含两个整数 $n,m\ (1\le n,m\le 10^5)$，分别表示 Leona 和 Zoe 的单词数。

接下来 $n$ 行，每行一个字符串，表示 Leona 的单词。

接下来 $m$ 行，每行一个字符串，表示 Zoe 的单词。

输入中，单词均为仅由小写英文字母构成的字符串，所有单词互不相同，它们的总长度不超过 $10^6$，并且 Leona 和 Zoe 的单词均已经按字典序排好序了。

## 输出格式

输出胜者的名字，`Leona` 或 `Zoe`。

```input1
2 1
avokado
dabar
brazil

```

```output1
Zoe

```

```input2
3 3
ananas
atlas
banana
albatros
cikla
nogomet

```

```output2
Leona

```

```input3
2 2
hrvatska
zastava
bijeli
galeb

```

```output3
Leona

```
