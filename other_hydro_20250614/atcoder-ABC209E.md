## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc209/tasks/abc209_e

高虼には $ N $ のgZがdっており、$ i\,\ (1\ \leq\ i\ \leq\ N) $ 番目のgZは $ s_i $ です。

高蚓と青木君は高虼を使って $ 3 $ しりとりをします。 $ 3 $ しりとりのル`ルは以下です。

- 高蚓と青木君は、高蚓から始めて交互にgZを言い合っていく。
- 各プレ`ヤ`は前の人が言ったgZの最後の $ 3 $ 文字で始まるgZを言わなければならない。例えば、前の人が `Takahashi` と言った龊稀⒋韦稳摔 `ship`、`shield` などを言うことができ、`Aoki`、`sing`、`his` などを言うことはできない。
- 大文字と小文字は区eされる。例えば、`Takahashi` のあとに `ShIp` を言うことはできない。
- 言うgZがなくなった方がける。
- 高虼にdっていないgZを言うことはできない。
- 同じgZは何度でも使ってよい。

各 $ i\,\ (1\ \leq\ i\ \leq\ N) $ について、高蚓が $ 3 $ しりとりをgZ $ s_i $ から始めたときどちらが伽膜を判定してください。ただし、二人とも最善に行婴工毪趣筏蓼埂＞咛宓膜摔稀⒆苑证けないことを最先し、その次に相手をかすことを先します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ s_1 $ $ s_2 $ $ \vdots $ $ s_N $

## 输出格式
$ N $ 行出力せよ。$ i\,\ (1\ \leq\ i\ \leq\ N) $ 行目には、高蚓が $ 3 $ しりとりをgZ $ s_i $ から始めたとき、高蚓が伽膜胜 `Takahashi`、青木君が伽膜胜 `Aoki`、しりとりが永hにAき・Qまらないなら `Draw` と出力せよ。

## 题目大意
给定 $N$ 个单词，Takahashi 和 Aoki 二人使用这 $N$ 个单词进行成语接龙，二者轮流说出单词，接龙的规则如下：

1. Takahashi 先手， Aoki 后手；
2. 当一个人说出单词后，另一个人必须选择一个单词满足：前一个人说的单词的后三个字母等于此单词的前三个字母，并说出它，如果这个人没有合法的单词可以说出，他输了;
3. 单词可以重复使用，区分大小写。

请你输出先手 Takahashi 从第 $i\ (1\le i\le N)$ 个单词开始说的情况下，且二者都绝顶聪明，游戏的胜负情况。若先手必胜输出 `Takahashi`，后手必胜输出 `Aoki`，平局输出 `Draw`。

```input1
3
abcd
bcda
ada
```

```output1
Aoki
Takahashi
Draw
```

```input2
1
ABC
```

```output2
Draw
```

```input3
5
eaaaabaa
eaaaacaa
daaaaaaa
eaaaadaa
daaaafaa
```

```output3
Takahashi
Takahashi
Takahashi
Aoki
Takahashi
```

## 提示
### 制s

- $ N $ は $ 1 $ 以上 $ 2\ \times\ 10^5 $ 以下の整数
- $ s_i $ は英小文字と英大文字のみからなるLさ $ 3 $ 以上 $ 8 $ 以下の文字列

### Sample Explanation 1

高蚓が `abcd` から始めたとき、次に青木君が `bcda` と言って高蚓は言うgZがなくなります。よって青木君が伽膜韦 `Aoki` と出力してください。 高蚓が `bcda` から始めたとき、次に青木君は言うgZがなくなります。よって高蚓が伽膜韦 `Takahashi` と出力してください。 高蚓が `ada` から始めたとき、二人とも `ada` をRり返すのでしりとりがKわることはありません。よって `Draw` と出力してください。同じgZを何度でも使用できることに注意してください。
