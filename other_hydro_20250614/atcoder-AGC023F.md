## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc023/tasks/agc023_f

すぬけ君は、$ N $ 点からなる根付き木を持っています。 点には $ 1 $ から $ N $ までの番号が振られています。 点 $ 1 $ はこの木の根です。 点 $ i $ ( $ 2\leq\ i\ \leq\ N $ ) のHは点 $ P_i $ ( $ P_i\ <\ i $ ) です。 各点には、$ 0 $ または $ 1 $ がかれていて、点 $ i $ にかれている数は $ V_i $ です。

すぬけ君は、この木の点を横一列にKべたいと考えています。 そのH、どの点についても、その点より右趣摔饯雾点の先祖となる点がないようにします。

点をKべKえたあと、点にかれている数を点のKびに沿ってKべた数列を $ X $ とします。 すぬけ君は、$ X $ の倒数 ( ※ ) を最小化したいです。 $ X $ の倒数の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ P_2 $ $ P_3 $ $ ... $ $ P_N $ $ V_1 $ $ V_2 $ $ ... $ $ V_N $

## 输出格式
数列 $ X $ の倒数の最小を出力せよ。

## 题目大意
- 给出一棵 $n$ 个节点的树，以及一个空序列。
- 每个节点上有一个取值在 $\{0, 1\}$ 中的数。
- 每次你可以选择没有父亲节点的点删除，并且将这个节点上的数字放在当前数列末尾。
- 请你求出这个数列可能得到的最小逆序对数。
- $n \leq 2 \times 10^5$

```input1
6
1 1 2 3 3
0 1 1 0 0 0
```

```output1
4
```

```input2
1

0
```

```output2
0
```

```input3
15
1 2 3 2 5 6 2 2 9 10 1 12 13 12
1 1 1 0 1 1 0 0 1 0 0 1 1 0 0
```

```output3
31
```

## 提示
### 注

あるLさ $ N $ の数列 $ Z $ の倒数とは、整数 $ i,\ j $ ( $ 1\ \leq\ i\ <\ j\ \leq\ N $ ) のMであって、$ Z_i\ >\ Z_j $ を氦郡工猡韦数を意味します。

### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ P_i\ <\ i $ ( $ 2\ \leq\ i\ \leq\ N $ )
- $ 0\ \leq\ V_i\ \leq\ 1 $ ( $ 1\ \leq\ i\ \leq\ N $ )
- 入力はすべて整数である。

### Sample Explanation 1

点を $ 1,\ 3,\ 5,\ 6,\ 2,\ 4 $ のにKべると、$ X\ =\ (0,\ 1,\ 0,\ 0,\ 1,\ 0) $ となり、 倒数は $ 4 $ になります。 これ以上倒数を小さくすることは出来ないので、$ 4 $ を出力します。

### Sample Explanation 2

$ X\ =\ (0) $ で、倒数は $ 0 $ です。
