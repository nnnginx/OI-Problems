## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc011/tasks/agc011_c

高蚓は，$ N $ 点 $ 1 $, $ 2 $, ..., $ N $ からなるo向グラフをもらいました． このグラフのxは $ (u_i,\ v_i) $ で表されます． このグラフには，自己xや多重xは存在しません．

高蚓は，このグラフをもとに，$ N^2 $ 点 $ (a,\ b) $ ($ 1\ \leq\ a\ \leq\ N $, $ 1\ \leq\ b\ \leq\ N $) からなるグラフを作ることにしました． このグラフのxは，次のtで定まります．

- 元のグラフにおいて $ a $ と $ a' $ のgおよび $ b $ と $ b' $ のgのI方にxがあるとき，またそのときに限り，$ (a,\ b) $ と $ (a',\ b') $ のgにxをる．

このようにして作ったグラフのBY成分の数を求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ u_1 $ $ v_1 $ $ u_2 $ $ v_2 $ : $ u_M $ $ v_M $

## 输出格式
高蚓の作ったグラフのBY成分の数を出力せよ．

## 题目大意
给定一张有 $n$ 个点，$m$ 条边的原图，现构成一张新图，其中每个点都是一个二元组 $(a, b)$。

两个二元组 $(a, b),(c, d)$ 有边当且仅当 $a$ 和 $c$ 有边且 $b$ 和 $d$ 有边。

求新图联通块个数。

```input1
3 1
1 2
```

```output1
7
```

```input2
7 5
1 2
3 4
3 5
4 5
2 6
```

```output2
18
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 100,000 $
- $ 0\ \leq\ M\ \leq\ 200,000 $
- $ 1\ \leq\ u_i\ <\ v_i\ \leq\ N $
- $ u_i\ =\ u_j $ かつ $ v_i\ =\ v_j $ を氦郡工瑜Δ十なる $ i,\ j $ のMは存在しない
