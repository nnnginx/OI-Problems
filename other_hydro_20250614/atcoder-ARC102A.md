## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc108/tasks/arc102_a

整数 $ N,K $ が与えられます。$ N $ 以下の正の整数のM $ (a,b,c) $ であって、$ a+b,b+c,c+a $ がすべて $ K $ の倍数であるようなものの数を求めてください。 ただし、$ a,b,c $ の番を入れ替えただけのMもなるものとして数えます。また、$ a,b,c $ の中に同じものがあってもいません。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
$ N $ 以下の正の整数のM $ (a,b,c) $ であって、$ a+b,b+c,c+a $ がすべて $ K $ の倍数であるようなものの数を出力せよ。

## 题目大意
给您整数 $N$ 和 $K$。 找出不大于 $N$ 的正整数的三元组 $(a,b,c)$ 的数量，以使 $a+b$，$b+c$ 和 $c+a$ 均为 $K$ 的倍数。 $a,b,c$ 的顺序确实很重要，其中一些可以相同。

```input1
3 2
```

```output1
9
```

```input2
5 3
```

```output2
1
```

```input3
31415 9265
```

```output3
27
```

```input4
35897 932
```

```output4
114191
```

## 提示
### 制s

- $ 1\ \leq\ N,K\ \leq\ 2\times\ 10^5 $
- $ N,K $ は整数である
