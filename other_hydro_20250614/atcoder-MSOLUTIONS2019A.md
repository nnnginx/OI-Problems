## 题目描述
[problemUrl]: https://atcoder.jp/contests/m-solutions2019/tasks/m_solutions2019_a

$ 3 $ 以上の整数 $ N $ が与えられます。 正 $ N $ 角形の内角の和を求めてください。

ただし、度数法を用いて、g位は出力しないでください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
正 $ N $ 角形の内角の和を表す整数を出力せよ。

## 题目大意
给定一个不小于 $3$ 的整数 $N$，求具有 $N$ 边的正多边形的内角之和。

按度打印答案，但不打印单位。（译者：也就是只输出度数）

```input1
3
```

```output1
180
```

```input2
100
```

```output2
17640
```

## 提示
### 制s

- $ 3\ \leq\ N\ \leq\ 100 $

### Sample Explanation 1

正三角形の内角の和は $ 180 $ 度です。
