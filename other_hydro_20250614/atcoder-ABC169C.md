## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc169/tasks/abc169_c

$ A\ \times\ B $ の小数点以下を切り韦啤⒔Y果を整数として出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
答えを整数として出力せよ。

## 题目大意
【题目翻译】

给定两个有理数$a$，$b$（$0≤a≤10^{15} ，0≤b<10，a∈\Z，b$为两位小数），求$a×b$，结果向下取整。

translated by @[Lhy114514](https://www.luogu.com.cn/user/749988)。

```input1
198 1.10
```

```output1
217
```

```input2
1 0.01
```

```output2
0
```

```input3
1000000000000000 9.99
```

```output3
9990000000000000
```

## 提示
### 制s

- $ 0\ \leq\ A\ \leq\ 10^{15} $
- $ 0\ \leq\ B\ <\ 10 $
- $ A $ は整数
- $ B $ は小数第 $ 2 $ 位まで与えられる

### Sample Explanation 1

$ 198\ \times\ 1.10\ =\ 217.8 $ なので、小数点以下を切り韦皮拼黏à $ 217 $ となります。
