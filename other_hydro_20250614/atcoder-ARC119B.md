## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc119/tasks/arc119_b

いま、光魇景澶 `0` と `1` から成るLさ $ N $ の文字列 $ S $ が表示されています。

あなたは次の操作を何回でも行うことができます。なお、ここでは光魇景澶吮硎兢丹欷皮い胛淖至肖 $ i $ $ (1\ \leq\ i\ \leq\ N) $ 文字目を $ S_i $ と表します。

> **操作** 整数 $ (l,\ r) $ $ (1\ \leq\ l\ <\ r\ \leq\ N) $ であって、次の条件のうちいずれかを氦郡工猡韦 $ 1 $ Mxび、$ S_l $ と $ S_r $ を入れ替える。
> 
> - $ S_l= $ `0` かつ $ S_{l+1}=\cdots=S_r= $ `1` を氦郡埂
> - $ S_{l}=\cdots=S_{r-1}= $ `1` かつ $ S_r= $ `0` を氦郡埂

光魇景澶吮硎兢丹欷皮い胛淖至肖 $ T $ に一致させることができるか判定し、可能な龊悉喜僮骰厥として考えられる最小のを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ S $ $ T $

## 输出格式
光魇景澶吮硎兢丹欷皮い胛淖至肖 $ T $ にすることが不可能な龊悉稀`-1` を出力してください。

可能な龊悉稀⒉僮骰厥として考えられる最小のを出力してください。

## 题目大意
给定两个长度为$ \textrm N $的0/1串$ \textrm S $和$ \textrm T $，每次对第一个串：选一个0移动到相邻的一段1之前或之后或中间，求最少经过多少次能使两个串相同。
## 输入
三行，分别对应$ \textrm N,S,T $；
## 输出
一行，最小的操作次数，若无法使两个串相同则输出$ \textrm -1 $.

```input1
7
1110110
1010111
```

```output1
2
```

```input2

11111000000000011111
11111000000000011111
```

```output2
0
```

```input3
6
111100
111000
```

```output3
-1
```

```input4
11111111111111111111111111011111101011111011110111110010100101001110111011110111111111110010011111101111111101110111011
```

```output4
22
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 500000 $
- $ S $ は `0` と `1` からなるLさ $ N $ の文字列である
- $ T $ は `0` と `1` からなるLさ $ N $ の文字列である

### Sample Explanation 1

例えば以下のように操作を行えば、$ 2 $ 回の操作で光魇景澶吮硎兢丹欷皮い胛淖至肖 `1010111` にすることができます。 - $ (l,\ r)\ =\ (2,\ 4) $ をxんで操作を行う。そのとき、光魇景澶挝淖至肖 `1110110` から `1011110` に浠する。 - $ (l,\ r)\ =\ (4,\ 7) $ をxんで操作を行う。そのとき、光魇景澶挝淖至肖 `1011110` から `1010111` に浠する。

### Sample Explanation 2

操作を行う前のr点で、光魇景澶吮硎兢丹欷皮い胛淖至肖 $ T $ であるため、答えは $ 0 $ となります。

### Sample Explanation 3

どのように操作を行っても、光魇景澶宋淖至 $ T $ を表示させることが不可能な龊悉稀`-1` と出力してください。
