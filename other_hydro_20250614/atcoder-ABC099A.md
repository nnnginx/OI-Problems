## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc099/tasks/abc099_a

AtCoder Beginner Contestが始まってから早数十年。

コンテストは $ 1 $ 回目からに `ABC001`,`ABC002`,$ ... $ と名付けられてきましたが、 $ 999 $ 回目のコンテスト `ABC999` をKえ、これからのコンテストの名前をどうするかという}が生じました。

そこで、$ 1000 $ 回目から $ 1998 $ 回目のコンテストをに `ABD001`,`ABD002`,$ ... $,`ABD999` と名付けることとなりました。

$ 1 $ 以上 $ 1998 $ 以下の整数 $ N $ が与えられるので、$ N $ 回目のコンテストの名前の最初の $ 3 $ 文字を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ 回目のコンテストの名前の最初の $ 3 $ 文字を出力せよ。

## 题目大意
### 题目描述
判断一个数是否大于 999 。如果不大于，输出`ABC`，否则输出`ABD`。

```input1
999
```

```output1
ABC
```

```input2
1000
```

```output2
ABD
```

```input3
1481
```

```output3
ABD
```

### 制s

- $ 1\ \leq\ N\ \leq\ 1998 $
- $ N $ は整数

$ 999 $ 回目のコンテストの名前は `ABC999` です。

$ 1000 $ 回目のコンテストの名前は `ABD001` です。
