## 题目描述

给定两个整数 $l$ 和 $r$，对于任意 $x$，满足 $l\leq x\leq r$，把 $x$ 的所有约数全部写下来。对于每个写下来的数，只保留最高位的那个数码。求$[1,9]$中每个数码出现的次数。

## 输入格式

输入一行两个整数 $l$ 和 $r$。

## 输出格式

一共 $9$ 行。

第 $i$ 行，输出数码 $i$ 出现的次数。

```input1
1 4
```

```output1
4
2
1
1
0
0
0
0
0
```

## 数据范围与提示

$1\leq l \leq r \leq 10^9$
