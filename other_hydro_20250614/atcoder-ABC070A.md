## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc070/tasks/abc070_a

$ 3 $ 桁の正の整数 $ N $ が与えられます。  
 $ N $ が 回文数であるかを判定してください。   
 回文数とは、$ 10 $ M法において数字を逆からKべても元の数と同じになる数のことを表します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ が回文数である龊悉 `Yes`、そうでない龊悉 `No` を出力せよ。

## 题目大意
给定一个 $100\sim 999$ 的数，若百位和个位相同，则输出 `Yes`，否则输出 `No`。

感谢 @da32s1da 提供的翻译

```input1
575
```

```output1
Yes
```

```input2
123
```

```output2
No
```

```input3
812
```

```output3
No
```

## 提示
### 制s

- $ 100QNQ999 $
- $ N $ は整数

### Sample Explanation 1

$ N=575 $ は逆から数字をKべても $ 575 $ となるため回文数です。したがって、`Yes` と出力します。

### Sample Explanation 2

$ N=123 $ は逆から数字をKべると $ 321 $ となり、回文数ではないため `No` と出力します。
