## 输入格式

第一行两个正整数 $N,K$。

接下来一行 $N$ 个非负整数，第 $i$ 个表示 $A_i$。

接下来一行 $N$ 个非负整数，第 $i$ 个表示 $B_i$。

## 输出格式

一行两个整数，表示子问题 1 和 2 的答案.

```input1
5 3
5 5 4 5 5
3 0 3 2 3
```

```output1
295164 295326
```

## 数据范围

对于 $100\%$ 的数据，$N\le 50,A_i\le 10^{18},Bi\le 10^{18},K\le 25$。
