<p>&nbsp;</p>
<p>Given two strings s1 and s2. We can merge characters of two strings (with order in original strings) to get a new string.</p>
<p>For example : s1 = 'ab' and s2 = 'ba'.</p>
<p>We can merge to get st = 'a<strong>b</strong>b<strong>a</strong>' but not st = 'a<strong>ab</strong>b'.</p>
<p><strong>The Problem :</strong>&nbsp;Given two string contain only lowercase letters, count the number of palindrome by merging in different ways.&nbsp;</p>
<p>Ex : 'aba', 'abba' are palindrome, 'abc' and 'abca' aren't.</p>
<h3>Input</h3>
<p>- Two lines, each line contains a string, string's length isn't over 500.</p>
<h3>Output</h3>
<p>- A single integer is the number of palindrome after <strong>modul 3210121</strong>.</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
ab</pre>
<pre>ba

<strong>Output:</strong>
4</pre>
