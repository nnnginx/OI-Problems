<p>In number theory, the totient ¦Õ of a positive integer n is defined to be the number of positive integers less than or equal to n that are coprime to n.</p>
<p>Given an integer n (1 &lt;= n &lt;= 10^6). Compute the value of the totient ¦Õ.</p>
<h3>Input</h3>
<p>First line contains an integer T, the number of test cases. (T &lt;= 20000)</p>
<p>T following lines, each contains an integer n.</p>
<h3>Output</h3>
<p>T lines, one for the result of each test case.</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
5
1
2
3
4
5

<strong>Output:</strong>
1
1
2
2
4

</pre>
