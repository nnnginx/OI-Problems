## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc218/tasks/abc218_e

$ N $ 点 $ M $ xのBYo向グラフがあります。  
 点には $ 1 $ から $ N $ の番号が、xには $ 1 $ から $ M $ の番号がついており、x $ i $ は点 $ A_i $ と $ B_i $ をYんでいます。

高蚓は、このグラフから $ 0 $ 以上のxを取り除こうとしています。

x $ i $ を取り除くと、$ C_i\ \geq\ 0 $ のとき $ C_i $ の蟪辘虻谩$ C_i\ <\ 0 $ のとき $ |C_i| $ のP金をBいます。

xを取り除いたあともグラフがBYでなければならないとき、高蚓が得られる蟪辘巫畲を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ A_1 $ $ B_1 $ $ C_1 $ $ A_2 $ $ B_2 $ $ C_2 $ $ \vdots $ $ A_M $ $ B_M $ $ C_M $

## 输出格式
答えを出力せよ。

## 题目大意
给一个无向图，让你从中选出几个边，要求选出的边权总和最大并且剩下的图要是一个连通图，输出最大的边权。

```input1
4 5
1 2 1
1 3 1
1 4 1
3 2 2
4 2 2
```

```output1
4
```

```input2
3 3
1 2 1
2 3 0
3 1 -1
```

```output2
1
```

```input3
2 3
1 2 -1
1 2 2
1 1 3
```

```output3
5
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ N-1\ \leq\ M\ \leq\ 2\times\ 10^5 $
- $ 1\ \leq\ A_i,B_i\ \leq\ N $
- $ -10^9\ \leq\ C_i\ \leq\ 10^9 $
- 与えられるグラフはBYである
- 入力に含まれるは全て整数である

### Sample Explanation 1

x $ 4,5 $ を取り除くことで合 $ 4 $ の蟪辘虻盲椁欷蓼埂￥长欷瑜甓啶の蟪辘虻盲毪长趣悉扦ないため、答えは $ 4 $ となります。

### Sample Explanation 2

蟪辘であるようなxが存在することもあります。

### Sample Explanation 3

多重xや自己ル`プが存在することもあります。
