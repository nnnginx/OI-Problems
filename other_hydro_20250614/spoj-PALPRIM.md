<p>A Palindromic number is a number without leading zeros that remains the same when its digits are reversed. For instance <strong>5, 22, 12321, 101101</strong> are Palindromic numbers where as <strong>10, 34, 566, 123421</strong> are not. A Prime number is a positive integer greater than <strong>1</strong> that has no positive divisors other than <strong>1</strong> and itself. For example, <strong>2, 31, 97</strong> are Prime numbers but <strong>1, 10, 25, 119</strong> are not. A Palindromic Prime number is both palindromic and prime at the same time. <strong>2, 3, 131</strong> are Palindromic Prime numbers but <strong>6, 17, 3333</strong> are not. Given a positive integer <strong>N</strong>, output the largest palindromic prime number not greater than <strong>N</strong>.&nbsp;</p>
<h3>Input</h3>
<p>The first line contains an integer <strong>T</strong> denoting the number of test cases. Each of the subsequent <strong>T</strong> lines contain a single integer <strong>N</strong> without leading/trailing spaces.&nbsp;</p>
<h3>Output</h3>
<p>Print <strong>T</strong> lines. For each test case, print a single integer denoting the largest palindromic prime number which does not exceed <strong>N.&nbsp;</strong></p>
