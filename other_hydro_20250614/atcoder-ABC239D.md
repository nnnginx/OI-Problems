## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc239/tasks/abc239_d

高蚓と青木君が次のようなゲ`ムをします。

- まず、高蚓が $ A $ 以上 $ B $ 以下の好きな整数をxび、青木君に护à
- 次に、青木君が $ C $ 以上 $ D $ 以下の好きな整数をxぶ
- 二人のxんだ整数の和が素数なら青木君の伽痢そうでなければ高蚓の伽

二人が最mな槁预蛉・毪趣、どちらが伽沥蓼工？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ D $

## 输出格式
二人が最mな槁预颏趣盲郡趣、高蚓が伽膜胜 `Takahashi`、青木君が伽膜胜 `Aoki` を出力せよ。

## 题目大意
高桥和青木玩下面这个游戏。开始时定下四个正整数 $a,b,c,d$ （这四个数由输入给出），然后两人开始选数。高桥先选定一个正整数 $x$ （ $a≤x≤b$ ），青木再选定一个正整数 $y$ （ $c≤y≤d$ ）。如果 $x+y$ 的值是质数则青木获胜，否则高桥获胜。当双方都采取最佳策略时，问谁会获胜？若高桥获胜输出`Takahashi`，若青木获胜输出`Aoki`。

```input1
2 3 3 4
```

```output1
Aoki
```

```input2
1 100 50 60
```

```output2
Takahashi
```

```input3
3 14 1 5
```

```output3
Aoki
```

## 提示
### 制s

- $ 1\ \leq\ A\ \leq\ B\ \leq\ 100 $
- $ 1\ \leq\ C\ \leq\ D\ \leq\ 100 $
- 入力に含まれるは全て整数である

### Sample Explanation 1

例えば高蚓が $ 2 $ をxんだときは、青木君は $ 3 $ をxぶことで、和を素数である $ 5 $ にすることができます。

### Sample Explanation 2

最mな槁预蛉・毪雀蚓が必ず伽沥蓼埂
