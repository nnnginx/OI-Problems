## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc258/tasks/abc258_g

$ N $ 点go向グラフ $ G $ が与えられます。

$ G $ は $ N $ 行 $ N $ 列のO接行列 $ A $ によって与えられます。つまり、$ A_{i,j} $ が $ 1 $ である龊悉享点 $ i,j $ gにxがあることを、$ 0 $ である龊悉摔限xがないことを意味します。

$ 1\ \le\ i\ <\ j\ <\ k\ \le\ N $ を氦郡拐数のM $ (i,j,k) $ のうち、点 $ i,j $ gにも点 $ j,k $ gにも点 $ i,k $ gにもxがあるようなものの数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_{1,1}A_{1,2}\dots\ A_{1,N} $ $ A_{2,1}A_{2,2}\dots\ A_{2,N} $ $ \vdots $ $ A_{N,1}A_{N,2}\dots\ A_{N,N} $

## 输出格式
答えを出力せよ。

## 题目大意
给你一个简单的无向图，其中有 $N$ 个顶点。用一个 的 $N\times N$ 邻接矩阵 $A$ 来表示。如果 $A_{i,j}=1$ ,则表示 $i$ 到 $j$ 有边相连，如果 $A_{i,j}=0$ ,则表示 $i$ 到 $j$ 无边相连。

求三角形 $(i,j,k)$ 的个数，满足 $1\leq i < j < k\leq n$，且 $i$ 与 $j$ 有边相连，$i$ 与 $k$ 有边相连，$j$ 与 $k$ 有边相连。

```input1
4
0011
0011
1101
1110
```

```output1
2
```

```input2
10
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
```

```output2
0
```

## 提示
### 制s

- $ 3\ \le\ N\ \le\ 3000 $
- $ A $ はgo向グラフ $ G $ のO接行列である。
- 入力はすべて整数。

### Sample Explanation 1

$ (i,j,k)=(1,3,4),(2,3,4) $ が条件を氦郡筏蓼埂 $ (i,j,k)=(1,2,3) $ は、点 $ 1,2 $ gにxがないため条件を氦郡筏蓼护蟆 よって、解は $ 2 $ です。
