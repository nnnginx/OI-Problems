## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc050/tasks/agc050_c

左右にo限にAくマスの列があります。 これを用いて、あなたとすぬけ君は以下のゲ`ムをプレイします。

- 判が、`B` と `S` からなる「タ`ン文字列」$ t $ を作り、二人にせる。
- まず、すぬけ君がマスのうち $ 1 $ つの上に立つ。
- そして、各 $ i\ =\ 1,\ ...,\ |t| $ について、この番に以下が行われる。
  - $ t $ の $ i $ 文字目が `B` のとき、あなたのタ`ンである。あなたは、他のブロックやすぬけ君を含まないマスを $ 1 $ つxび、ブロックを置く。O置後、すぬけ君のIOのマスにともにブロックが置かれている龊稀あなたの倮でゲ`ムがK了する。
  - $ t $ の $ i $ 文字目が `S` のとき、すぬけ君のタ`ンである。すぬけ君は、Oの空きマスに移婴工毪、何もしない。
- このr点でゲ`ムがK了していない龊稀すぬけ君の倮でゲ`ムがK了する。

`B`, `S`, `?` からなる文字列 $ s $ が与えられます。 $ s $ に含まれる `?` の数が $ Q $ であるとき、`?` をそれぞれ `B` または `S` で置きQえてタ`ン文字列とする方法は $ 2^Q $ 通り存在します。 これらの $ 2^Q $ のタ`ン文字列のうち、Iプレイヤ`が最mに行婴筏郡趣にあなたが倮するようなものは何あるでしょうか。 この答えを $ 998,244,353 $ で割った余りを求めてください。

## 输入格式
入力は嗜肓Δら以下の形式で与えられる。

> $ s $

## 输出格式
答えを出力せよ。

## 题目大意
有一个由房间组成的序列，其向左右两端无限延伸。你和 Snuke 在玩下面的游戏：
- 裁判给出一个由 `B` 和 `S` 组成的字符串 $t$，名为回合决定串，并将它展示给两名玩家。
- 首先，Snuke 站在其中一个房间里。
- 随后，对于每个 $i = 1,2,\dots,|t|$，发生如下的事件：
  - 如果 $t$ 中第 $i$ 个字母为 `B`，则这是你的回合。你会选择一个其中没有障碍或 Snuke 的房间，在其中放置一个障碍。这之后，如果 Snuke 所在房间左右两侧的房间均放置有障碍，则你胜利，游戏结束。
  - 如果 $t$ 中第 $i$ 个字母为 `S`，则这是 Snuke 的回合。他可以移动到与当前所在房间相邻且无障碍的房间，或什么都不做。
- 如果 $|t|$ 轮后游戏仍未结束，则 Snuke 胜利，游戏结束。

给定字符串 $s$，其中仅含有 `B`、`S` 与 `?` 三种字符。设 $s$ 中含有 $q$ 个 `?`，有 $2^q$ 种填法将 $s$ 中的 `?` 替换为 `B` 或 `S`，得到一个回合决定串。假设两名玩家都按最优策略操作，在这 $2^q$ 个串中，有多少种填法使得你能胜利呢？

请输出答案模 $998244353$ 的值。

$1\le |s|\le 10^6$。

```input1
BSSBS
```

```output1
0
```

```input2
?S?B????S????????B??????B??S??
```

```output2
16777197
```

## 提示
### 制s

- $ 1\ \leq\ |s|\ \leq\ 10^6 $
- $ s $ は `B`, `S`, `?` からなる。

### Sample Explanation 1

$ 1,\ 4 $ タ`ン目があなたのタ`ンで、$ 2,\ 3,\ 5 $ タ`ン目がすぬけ君のタ`ンです。 この龊稀I者が最mに行婴工毪趣工踏本が伽膜长趣わかります。
