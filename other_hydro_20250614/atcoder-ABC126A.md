## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc126/tasks/abc126_a

`A`, `B`, `C` からなるLさ $ N $ の文字列 $ S $ と、$ 1 $ 以上 $ N $ 以下の整数 $ K $ が与えられます。 文字列 $ S $ の $ K $ 文字目を小文字にきQえ、新しくできた $ S $ を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ S $

## 输出格式
文字列 $ S $ の $ K $ 文字目を小文字にきQえ、新しくできた $ S $ を出力せよ。

## 题目大意
## 输入格式
输入共两行。第一行两个整数 $n,k$ 。

第二行一个由 `A`,`B`,`C` 组成字符串  $s$ 。

$n$ 表示字符串的长度。

把字符串 $s$ 的第 $k-1$ 个字符变成小写。（字符串 $s$ 的第一个字符的下标为 $0$ ）

## 输出格式
一行一个字符串 $s$ ，为转换后的字符串。

```input1
3 1
ABC
```

```output1
aBC
```

```input2
4 3
CABA
```

```output2
CAbA
```

## 提示
### 制s

- $ 1\ <\ =\ N\ <\ =\ 50 $
- $ 1\ <\ =\ K\ <\ =\ N $
- $ S $ は `A`, `B`, `C` からなるLさ $ N $ の文字列
