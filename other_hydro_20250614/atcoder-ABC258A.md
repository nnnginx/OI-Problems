AtCoder Beginner Contest は通常、日本rで $ 21 $ rちょうどに始まり $ 100 $ 分gにわたって行われます。

$ 0 $ 以上 $ 100 $ 以下の整数 $ K $ が与えられます。$ 21 $ rちょうどから $ K $ 分後のr刻を `HH:MM` の形式で出力してください。ただし、`HH` は $ 24 $ rg制でのrgを、`MM` は分を表します。rgまたは分が $ 1 $ 桁のときは、先^に $ 0 $ を追加して $ 2 $ 桁の整数として表してください。

## 题目大意
- 给定一个整数 $K$，请使用 `HH:MM` 的形式输出 $21:00$ 之后 $K$ 分钟的时间。

- 其中 `HH` 是 $24$ 小时制的小时，`MM` 是分钟（如果只有一位需要补充前导零）。

- 具体可参考样例理解。

- 数据范围：$1\le K< 100$。

- Translated by Pitham（脾土蛤蟆）。

```input1
63
```

```output1
22:03
```

```input2
45
```

```output2
21:45
```

```input3
100
```

```output3
22:40
```
