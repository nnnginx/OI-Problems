## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc259/tasks/abc259_d

$ xy $ -平面上の $ N $ の窑与えられます。 $ i\ =\ 1,\ 2,\ \ldots,\ N $ について、$ i $ 番目の窑系 $ (x_i,\ y_i) $ を中心とする半径 $ r_i $ の窑扦埂

$ N $ の窑韦Δ辽伽胜とも $ 1 $ つ以上の窑抑苌悉摔る点のみを通って、点 $ (s_x,\ s_y) $ から点 $ (t_x,\ t_y) $ に行くことができるかどうかを判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ s_x $ $ s_y $ $ t_x $ $ t_y $ $ x_1 $ $ y_1 $ $ r_1 $ $ x_2 $ $ y_2 $ $ r_2 $ $ \vdots $ $ x_N $ $ y_N $ $ r_N $

## 输出格式
点 $ (s_x,\ s_y) $ から点 $ (t_x,\ t_y) $ に行くことができる龊悉 `Yes` を、そうでない龊悉 `No` を出力せよ。 ジャッジは英小文字と英大文字を密に区eすることに注意せよ。

## 题目大意
已给出 $n$ 个在平面直角坐标系上的圆。对于每个 $i \in \{1, 2, 3, ..., n\}$，有第 $i$ 个圆的圆心是 $(x_i, y_i)$，半径是 $r_i$。你需要回答你是否能从 $(s_x, s_y)$ 在圆上连续地走到 $(t_x, t_y)$。

```input1
4
0 -2 3 3
0 0 2
2 0 2
2 3 1
-3 3 3
```

```output1
Yes
```

```input2
3
0 1 0 3
0 0 1
0 0 2
0 0 3
```

```output2
No
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 3000 $
- $ -10^9\ \leq\ x_i,\ y_i\ \leq\ 10^9 $
- $ 1\ \leq\ r_i\ \leq\ 10^9 $
- $ (s_x,\ s_y) $ は $ N $ の窑韦Δ辽伽胜とも $ 1 $ つ以上の窑抑苌悉摔る
- $ (t_x,\ t_y) $ は $ N $ の窑韦Δ辽伽胜とも $ 1 $ つ以上の窑抑苌悉摔る
- 入力はすべて整数

### Sample Explanation 1

!\[\](https://img.atcoder.jp/abc259/7b850385b9d67dc150435ffc7818bd94.png) 例えば、下のU路で点 $ (0,\ -2) $ から点 $ (3,\ 3) $ へ行くことができます。 - 点 $ (0,\ -2) $ から $ 1 $ つ目の窑抑苌悉蚍r回りに通って点 $ (1,\ -\sqrt{3}) $ へ行く。 - 点 $ (1,\ -\sqrt{3}) $ から $ 2 $ つ目の窑抑苌悉r回りに通って点 $ (2,\ 2) $ へ行く。 - 点 $ (2,\ 2) $ から $ 3 $ つ目の窑抑苌悉蚍r回りに通って点 $ (3,\ 3) $ へ行く。 よって、`Yes` を出力します。

### Sample Explanation 2

!\[\](https://img.atcoder.jp/abc259/924efa40ff28e5d7125841da2710d012.png) 少なくとも $ 1 $ つ以上の窑抑苌悉摔る点のみを通って点 $ (0,\ 1) $ から点 $ (0,\ 3) $ に行くことはできないので `No` を出力します。
