AliceとBrownはゲ`ムをするのが好きです。今日は以下のゲ`ムを思いつきました。

$ 2 $つの山があり、はじめに$ X,\ Y $の石が置かれています。 AliceとBrownは哎咯`ン以下の操作を交互に行い、操作を行えなくなったプレイヤ`はけとなります。

- 片方の山から $ 2i $ の石を取り、そのうち $ i $ の石を韦啤⒉肖辘 $ i $ の石をもう片方の山に置く。ここで、整数 $ i\ (1Qi) $ のは山に十分な数の石がある欷亲杂嗓诉xぶことができる。

Aliceが先手で、二人とも最mにプレイすると⒍à筏郡趣、与えられた $ X,\ Y $ にしどちらのプレイヤ`が伽膜求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ Y $

## 输出格式
Aliceが伽膜趣`Alice`と、Brownが伽膜趣`Brown`と出力せよ。

## 题目大意
大意：

现有两堆石子，Alice和Brown以此进行游戏，规则如下：

* `Alice先手`，两方分别按照`回合制`取石子

* 每个人每回合可以任意从任一堆中取出`2的倍数个`石子（前提是该堆里有这么多石子），扔掉其中的一半，将另一半放入另一堆中。

* 当一方`无法进行取石子操作`时视作此方失败。

现给出石子数量n,m$(0≤n,m≤10^{18})$，请你输出胜方名称(Alice|Brown)。

```input1

```

```output1
Brown
```

```input2

```

```output2

```

```input3

```

```output3
Brown
```

```input4

```

```output4

```
