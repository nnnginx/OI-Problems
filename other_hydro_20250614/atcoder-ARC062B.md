## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc046/tasks/arc062_b

シカのAtCoDeerくんは友_のTopCoDeerくんとあるゲ`ムをして椁筏皮い蓼埂 このゲ`ムは $ N $ タ`ンからなります。各タ`ンではそれぞれのプレイヤ`はじゃんけんのグ`かパ`を出します。ただし、各プレイヤ`は次の条件を氦郡贡匾があります。

(※) 各タ`ンの後で、(今までにパ`を出した回数)$ Q $(今までにグ`を出した回数) を氦郡

このゲ`ムでの各プレイヤ`の得点は、(伽盲骏咯`ンの数) $ - $ (けたタ`ンの数) です。 AtCoDeerくんは特殊能力を持っているので、ゲ`ムが始まる前にTopCoDeerくんの出す $ N $ タ`ンの手を全て知ることが出来ました。 AtCoDeerくんの各タ`ンでの手をQめて、AtCoDeerくんの得点を最大化してください。 TopCoDeerくんの出す手の情螭衔淖至 $ s $ で与えられます。 $ s $ の $ i(1QiQN) $ 文字目が `g`のときは $ i $ タ`ン目でTopCoDeerくんがグ`を出すことを、 `p`のときはパ`を出すことを表します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ s $

## 输出格式
AtCoDeerくんの得点の最大を出力せよ。

## 题目大意
你和对手都只有两种出拳方式：石头（$g$）和布（$p$），规则一样，赢了得一分，平局不得分，输了减一分。
现给你对手的出拳方式，设你到 $i$ 位置共出了 $x_i$ 次石头，$y_i$ 次布，在对于任意位置 $i$ 满足 $x_i \geq y_i$ 的条件下，输出你能得到的最大分数。

```input1
gpg
```

```output1
0
```

```input2
ggppgggpgg
```

```output2
2
```

## 提示
### 制s

- $ 1QNQ10^5 $
- $ N=|s| $
- $ s $ の各文字は`g`か`p`
- $ s $ で表される手は、条件(※)を氦郡筏皮い

### Sample Explanation 1

常に相手とあいこになるように手を出すことで、$ 0 $点を取ることができて、これが最大です。

### Sample Explanation 2

例えばグ`,パ`,グ`,パ`,グ`,グ`,パ`,パ`,グ`,パ` と出すことで、 $ 3 $回伽盲$ 1 $回けているので得点は$ 2 $点になり、これが最大です。
