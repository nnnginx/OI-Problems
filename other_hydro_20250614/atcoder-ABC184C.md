## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc184/tasks/abc184_c

o限に冥る $ 2 $ 次元グリッドがあり、マス $ (r_1,\ c_1) $ にx「超oR」が置かれています。  
 このxは、 $ 1 $ 手で次のような婴ができます。

![](https://cdn.luogu.com.cn/upload/vjudge_pic/AT_abc184_c/5bfa39c442ba1e792051275842c037db26123fff.png)

より正_には、超oRがマス $ (a,\ b) $ にあるとき、以下のいずれかの条件を氦郡攻蕙 $ (c,\ d) $ に婴すことができます。

- $ a\ +\ b\ =\ c\ +\ d $
- $ a\ -\ b\ =\ c\ -\ d $
- $ |a\ -\ c|\ +\ |b\ -\ d|\ \le\ 3 $

超oRを $ (r_1,\ c_1) $ から $ (r_2,\ c_2) $ に婴すのに必要な最小手数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ r_1 $ $ c_1 $ $ r_2 $ $ c_2 $

## 输出格式
超oRを $ (r_1,\ c_1) $ から $ (r_2,\ c_2) $ に婴すのに必要な最小手数を出力せよ。

## 题目大意
有一个无限大的棋盘，棋盘上的点 $(a,b)$ 能一步走到 $(b,c)$，当且仅当下面的三条条件至少一条成立：

+ $a+b=c+d$
+ $a-b=c-d$
+ $|a-c|+|b-d|\le 3$

求 $(r_1,c_1)$ 走到 $(r_2,c_2)$ 的最少步数。

```input1
1 1
5 6
```

```output1
2
```

```input2
1 1
1 200001
```

```output2
2
```

```input3
2 3
998244353 998244853
```

```output3
3
```

```input4
1 1
1 1
```

```output4
0
```

## 提示
### 制s

- 入力は全て整数
- $ 1\ \le\ r_1,\ c_1,\ r_2,\ c_2\ \le\ 10^9 $

### Sample Explanation 1

例えば、 $ (1,\ 1)\ \rightarrow\ (5,\ 5)\ \rightarrow\ (5,\ 6) $ と婴すと $ 2 $ 手になります。

### Sample Explanation 2

例えば、 $ (1,\ 1)\ \rightarrow\ (100001,\ 100001)\ \rightarrow\ (1,\ 200001) $ と婴すと $ 2 $ 手になります。

### Sample Explanation 3

例えば、 $ (2,\ 3)\ \rightarrow\ (3,\ 3)\ \rightarrow\ (-247,\ 253)\ \rightarrow\ (998244353,\ 998244853) $ と婴すと $ 3 $ 手になります。
