## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc112/tasks/abc112_d

整数 $ N,\ M $ が与えられます。

$ a_1\ +\ a_2\ +\ ...\ +\ a_N $ = $ M $ となる正整数からなるLさ $ N $ の数列 $ a $ において、$ a_1,\ a_2,\ ...,\ a_N $ の最大公s数のとり得る最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $

## 输出格式
条件を氦郡故列 $ a_1,\ a_2,\ ...,\ a_N $ の最大公s数のとり得る最大を出力せよ。

## 题目大意
给你 $2$ 个整数 $N,M$。

你需要构造一个有 $N$ 个数的正整数序列 $a$，满足以下条件：

+ $\sum\limits_{i=1}^{N}a_i=M$

求 $\gcd\limits_{i=1}^{N}a_i$ 可能的最大值。

$1\le N\le M\le 10^{9}$

```input1
3 14
```

```output1
2
```

```input2
10 123
```

```output2
3
```

```input3
100000 1000000000
```

```output3
10000
```

## 提示
### 制s

- 入力はすべて整数である
- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ N\ \leq\ M\ \leq\ 10^9 $

### Sample Explanation 1

$ (a_1,\ a_2,\ a_3)\ =\ (2,\ 4,\ 8) $ としたときこれらの最大公s数が $ 2 $ となり最大です。
