## 题目描述
对于一个整数 $n$，若存在两个自然数 $x, y$，满足 $n = x^2 - y^2$，则称 $n$ 是 Couple number。

给出 $a,b$，请求出 $[a, b]$ 范围内有多少个 Couple number。

## 输入格式
输入只有一行两个整数，分别表示 $a, b$。

## 输出格式
输出一行一个整数表示答案。

```input1
1 10

```

```output1
7

```

## 提示
#### 数据规模与约定

对于全部的测试点，保证 $-10^7 \le a < b \le 10^7$ ，$b - a \le 10^7$。
