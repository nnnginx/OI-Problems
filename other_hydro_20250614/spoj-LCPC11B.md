<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">Given a number N, you are asked to count the number of integers between A and B inclusive which are relatively prime to N. </span></p>
<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">Two integers are said to be co-prime or relatively prime if they have no common positive divisors other than 1 or, equivalently, if their greatest common divisor is 1. The number 1 is relatively prime to every integer. </span></p>
<p><span style="font-size: 12.000000pt; font-family: 'Times'; font-weight: 700">Input Specification </span></p>
<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">The first line on input contains T (0 &lt; T &lt;= 100) the number of test cases, each of the next T lines contains three integers A, B, N where (1 &lt;= A &lt;= B &lt;= 10</span><span style="font-size: 7.000000pt; font-family: 'Verdana'; vertical-align: 5.000000pt">15</span><span style="font-size: 11.000000pt; font-family: 'Verdana'">) and (1 &lt;= N &lt;= 10</span><span style="font-size: 7.000000pt; font-family: 'Verdana'; vertical-align: 5.000000pt">9</span><span style="font-size: 11.000000pt; font-family: 'Verdana'">). </span></p>
<p><span style="font-size: 12.000000pt; font-family: 'Times'; font-weight: 700">Output Specification </span></p>
<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">For each test case, print the number of integers between A and B inclusive which are relatively prime to N. Follow the output format below. </span></p>
<p><span style="font-size: 12.000000pt; font-family: 'Times'; font-weight: 700">Sample Input </span></p>
<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">2</span></p>
<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">1 10 2</span></p>
<p><span style="font-size: 11.000000pt; font-family: 'Verdana'">3 15 5 </span></p>
<p><span style="font-size: 12.000000pt; font-family: 'Times'; font-weight: 700">Sample Output </span></p>
<p><span style="font-size: 12.000000pt; font-family: 'Verdana'">Case #1: 5</span></p>
<p><span style="font-size: 12.000000pt; font-family: 'Verdana'"> Case #2: 10 </span></p>
<p><span style="font-size: 9.000000pt; font-family: 'Verdana'">In the first test case, the five integers in range [1,10] which are relatively prime to 2 are {1,3,5,7,9}. </span></p>
