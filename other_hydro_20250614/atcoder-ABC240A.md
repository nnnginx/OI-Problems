## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc240/tasks/abc240_a

下の画像で示す恧摔いて、$ a $ 番の点と $ b $ 番の点がで直接Yばれているかを答えてください。

![](https://cdn.luogu.com.cn/upload/vjudge_pic/AT_abc240_a/f5adeb25247ba30397bc0cea3bcc56cd667df4d2.png)

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ a $ $ b $

## 输出格式
$ a $ 番の点と $ b $ 番の点がで直接Yばれている龊悉 `Yes` を出力し、Yばれていない龊悉 `No` を出力せよ。  
 ジャッジは英大文字と英小文字を密に区eすることに注意せよ。

## 题目大意
给定如上图的环并给定 $a,b$，问 $a,b$ 是否连接。

Translated by ShanCreeper.

```input1
4 5
```

```output1
Yes
```

```input2
3 5
```

```output2
No
```

```input3
1 10
```

```output3
Yes
```
