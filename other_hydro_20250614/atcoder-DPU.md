## 题目描述
[problemUrl]: https://atcoder.jp/contests/dp/tasks/dp_u

$ N $ 羽のうさぎたちがいます。 うさぎたちには $ 1,\ 2,\ \ldots,\ N $ と番号が振られています。

各 $ i,\ j $ ($ 1\ \leq\ i,\ j\ \leq\ N $) について、うさぎ $ i $ と $ j $ の相性が整数 $ a_{i,\ j} $ によって与えられます。 ただし、各 $ i $ ($ 1\ \leq\ i\ \leq\ N $) について $ a_{i,\ i}\ =\ 0 $ であり、各 $ i,\ j $ ($ 1\ \leq\ i,\ j\ \leq\ N $) について $ a_{i,\ j}\ =\ a_{j,\ i} $ です。

太郎君は、$ N $ 羽のうさぎたちをいくつかのグル`プへ分けようとしています。 このとき、各うさぎはちょうど $ 1 $ つのグル`プに属さなければなりません。 グル`プ分けのY果、各 $ i,\ j $ ($ 1\ \leq\ i\ <\ j\ \leq\ N $) について、うさぎ $ i $ と $ j $ が同じグル`プに属するならば、太郎君は $ a_{i,\ j} $ 点を得ます。

太郎君のt得点の最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_{1,\ 1} $ $ \ldots $ $ a_{1,\ N} $ $ : $ $ a_{N,\ 1} $ $ \ldots $ $ a_{N,\ N} $

## 输出格式
太郎君のt得点の最大を出力せよ。

## 题目大意
对 $N$ 个物品任意分组，如果第 $i$ 个物品和 第 $j$ 个物品分在一组，会产生 $a_{i,j}$ 的得分，最大化得分之和。  

$(i,j)$ 和 $(j,i)$ 的贡献只计算一次。

```input1
3

10 0 -100
20 -100 0
```

```output1
20
```

## 提示
### 制s

- 入力はすべて整数である。
- $ 1\ \leq\ N\ \leq\ 16 $
- $ |a_{i,\ j}|\ \leq\ 10^9 $
- $ a_{i,\ i}\ =\ 0 $
- $ a_{i,\ j}\ =\ a_{j,\ i} $

### Sample Explanation 1

$ \{1,\ 3\},\ \{2\} $ とグル`プ分けすればよいです。

### Sample Explanation 2

$ \{1\},\ \{2\} $ とグル`プ分けすればよいです。

### Sample Explanation 3

$ \{1,\ 2,\ 3,\ 4\} $ とグル`プ分けすればよいです。 答えは 32-bit 整数型にГ蓼椁胜龊悉あります。
