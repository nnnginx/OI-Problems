## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc191/tasks/abc191_b

Lさ $ N $ の整数列 $ A $ と整数 $ X $ が与えられます。  
 $ A $ から $ X $ と等しい要素を全て取り除き、残った要素をそのままの序でKべた数列 $ A' $ を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X $ $ A_1 $ $ A_2 $ $ A_3 $ $ \dots $ $ A_N $

## 输出格式
$ A' $ の要素を空白区切りでに出力せよ。

## 题目大意
给定长度为 $N$ 的整数数列 $A$ 和整数 $X$ ，从数列 $A$ 中删除所有与 $X$ 相等的数字，并输出剩余按原本顺序排列的数字。

```input1
5 5
3 5 6 5 4
```

```output1
3 6 4
```

```input2
3 3
3 3 3
```

```output2

```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 10^5 $
- $ 1\ \le\ X\ \le\ 10^9 $
- $ 1\ \le\ A_i\ \le\ 10^9 $
- 入力は全て整数

### Sample Explanation 1

$ [3,\ 5,\ 6,\ 5,\ 4] $ から $ 5 $ を取り除くと、$ [3,\ 6,\ 4] $ になります。

### Sample Explanation 2

$ A' $ が要素数 $ 0 $ の数列となる龊悉あります。この龊稀⒖招肖虺隽Δ工毪坤堡いません。
