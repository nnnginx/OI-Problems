## 题目背景
公元二零一四年四月十七日，小明参加了省赛，在一路上，他遇到了许多问题，请你帮他解决。


## 题目描述
已知车上有 $N$ 排座位，有 $2N$ 个人参加省赛，每排座位只能坐两人，且每个人都有自己想坐的排数，问最多使多少人坐到自己想坐的位置。

## 输入格式
第一行，一个正整数 $N$。

第二行至第 $2N+1$ 行，每行两个正整数 $S_{i, 1},S_{i, 2}$，为每个人想坐的排数。

## 输出格式
一个非负整数，为最多使得多少人满意。


```input1
4
1 2
1 3
1 2
1 3
1 3
2 4
1 3
2 3

```

```output1
7
```

## 提示
对于 $10\%$ 的数据，$n \le 10$；

对于 $30\%$ 的数据，$n \le 50$；

对于 $60\%$ 的数据，$n \le 200$；

对于 $100\%$ 的数据，$n \le 2000$。
