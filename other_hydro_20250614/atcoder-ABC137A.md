## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc137/tasks/abc137_a

整数 $ A $, $ B $ があります。

$ A\ +\ B $, $ \ A\ -\ B $, $ \ A\ \times\ B $ の中で最大の数を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
$ A\ +\ B $, $ \ A\ -\ B $, $ \ A\ \times\ B $ の中で最大の数を出力せよ。

## 题目大意
## 题意：

给出两个整数 $A$ 和 $B$。求出 $A + B$，$A\times B$ 和 $ A - B$ 中的最大值。


------------

## 输入格式：

一行输入两个整数：$A$ 和 $B$。


------------

## 输出格式：

输出 $A + B$，$A\times B$ 和 $ A - B$ 中的最大值。


------------


## 样例输入输出：

无


------------
## 数据规模与约定：

满足所有输入为整数。

$-100 \le A,B \le 100$。

```input1
-13 3
```

```output1
-10
```

```input2
1 -33
```

```output2
34
```

```input3
13 3
```

```output3
39
```

## 提示
### 制s

- 入力は全て整数である。
- $ -100\ \leq\ A,\ B\ \leq\ 100 $

### Sample Explanation 1

$ A\ +\ B\ =\ -10 $, $ \ A\ -\ B\ =\ -16 $, $ \ A\ \times\ B\ =\ -39 $ の中で最大の数は $ -10 $ です。

### Sample Explanation 2

$ A\ +\ B\ =\ -32 $, $ \ A\ -\ B\ =\ 34 $, $ \ A\ \times\ B\ =\ -33 $ の中で最大の数は $ 34 $ です。

### Sample Explanation 3

$ A\ +\ B\ =\ 16 $, $ \ A\ -\ B\ =\ 10 $, $ \ A\ \times\ B\ =\ 39 $ の中で最大の数は $ 39 $ です。
