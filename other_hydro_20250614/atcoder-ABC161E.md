## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc161/tasks/abc161_e

高蚓は明日からの $ N $ 日gのうち $ K $ 日をxんでPくことにしました。

整数 $ C $ と文字列 $ S $ が与えられるので、次の $ 2 $ つの条件を氦郡工瑜Δ摔筏Pく日をxびます。

- ある日Pいたら、その直後の $ C $ 日gはPかない
- $ S $ の $ i $ 文字目が `x` のとき、今日から $ i $ 日後にはPかない

高蚓が必ずPく日をすべて求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ C $ $ S $

## 输出格式
高蚓が必ずPく日をNに改行区切りですべて出力せよ。

## 题目大意
高桥打算在$N$天中工作$K$天。给定一个串$S$。

①如果$S_i$是'x'，那么在第$i$天高桥不工作；

②如果高桥在某一天工作了，那么接下来他就会休息$C$天。

请问，高桥在哪些天他**不得不工作以达到在$N$天中恰好工作$K$天的要求**? 详见输出格式。

```input1
11 3 2
ooxxxoxxxoo
```

```output1
6
```

```input2
5 2 3
ooxoo
```

```output2
1
5
```

```input3
5 1 0
ooooo
```

```output3

```

```input4
16 4 3
ooxxoxoxxxoxoxxo
```

```output4
11
16
```

### Sample Explanation 1

高蚓は $ 11 $ 日gのうち $ 3 $ 日Pこうとしています。ある日Pいたらその後 $ 2 $ 日gはPきません。 Pく日としてありえるMみ合わせは「$ 1,6,10 $ 日目」「$ 1,6,11 $ 日目」「$ 2,6,10 $ 日目」「$ 2,6,11 $ 日目」の $ 4 $ 通りです。 したがって、$ 6 $ 日目に必ずPきます。

### Sample Explanation 2

Pく日としてありえるMみ合わせは「$ 1,5 $ 日目」のみです。

### Sample Explanation 3

必ずPく日が存在しないこともあります。
