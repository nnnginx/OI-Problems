## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc271/tasks/abc271_a

`0123456789` に加えて $ 10,11,12,13,14,15 $ に辘工胧字として `ABCDEF` を使う $ 16 $ M表では、$ 0 $ 以上 $ 255 $ 以下の整数は $ 1 $ 桁または $ 2 $ 桁になります。  
 例えば、$ 0 $ や $ 12 $ は $ 16 $ M表では `0` や `C` と $ 1 $ 桁になり、$ 99 $ や $ 255 $ は $ 16 $ M表では `63` や `FF` と $ 2 $ 桁になります。

$ 0 $ 以上 $ 255 $ 以下の整数 $ N $ を、必要に辘袱葡阮^に `0` を加えることで**ちょうど $ 2 $ 桁**の $ 16 $ M表にQしてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
将十进制整数 $N$ 转换为十六进制。

$0 \le N \le 255$

```input1
99
```

```output1
63
```

```input2
12
```

```output2
0C
```

```input3
0
```

```output3
00
```

```input4
255
```

```output4
FF
```

## 提示
### 注

英大文字と英小文字は区eされます。特に、$ 16 $ M表の数字として `ABCDEF` の代わりに `abcdef` を使うことは**出来ません**。

### 制s

- $ 0\ \leq\ N\ \leq\ 255 $
- $ N $ は整数

### Sample Explanation 1

$ 99 $ は $ 16 $ M表で `63` です。

### Sample Explanation 2

$ 12 $ は $ 16 $ M表で `C` です。 要求されているのはちょうど $ 2 $ 桁の $ 16 $ M表にQすることなので、`C` の先^に `0` を加えた `0C` が答えです。
