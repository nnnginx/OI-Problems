## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc127/tasks/arc127_c

すぬけくんは\板に $ 1 $ 以上 $ (2^N-1) $ 以下の整数をすべてきました． ただし，整数は $ 2 $ M表できました．

\板にかれた整数を文字列としてたr，辞で $ X $ 番目に小さい文字列を求めてください．

なお，入力において $ N $ は $ 10 $ M法で与えられますが，$ X $ は $ 2 $ M法で与えられます．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ X $

## 输出格式
答えを出力せよ．

## 题目大意
给出 $n,x$，请求出 $[1,2^n-1]$ 中字典序第 $x$ 小的数是什么。

```input1
3
101
```

```output1
11
```

```input2
10
10100011
```

```output2
1001001111
```

```input3
1000000
11111
```

```output3
1000000000000000000000000000000
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^6 $
- $ 1\ \leq\ X\ \leq\ 2^N-1 $
- $ X $ は $ 2 $ M法で与えられる．

### Sample Explanation 1

\板にかれた文字列を辞にKべると，`1`,`10`,`100`,`101`,`11`,`110`,`111` です． また $ X=101(2\mathrm{M})=5(10\mathrm{M}) $ です． よって，答えは `11` になります．
