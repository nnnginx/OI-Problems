## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc045/tasks/abc045_b

A さん、B さん、C さんの $ 3 $ 人が以下のようなカ`ドゲ`ムをプレイしています。

- 最初、$ 3 $ 人はそれぞれ `a`、`b`、`c` いずれかの文字がかれたカ`ドを、何枚か持っている。これらは入力で与えられた番に持っており、途中でKべ替えたりしない。
- $ A $ さんのタ`ンから始まる。
- F在自分のタ`ンである人がカ`ドを $ 1 $ 枚以上持っているならば、そのうち先^のカ`ドを韦皮搿￥饯吾帷韦皮椁欷骏`ドにかれているアルファベットと同じ名前の人 (例えば、カ`ドに `a` とかれていたならば A さん) のタ`ンとなる。
- F在自分のタ`ンである人がカ`ドを $ 1 $ 枚も持っていないならば、その人がゲ`ムの僬撙趣胜辍ゲ`ムはK了する。

$ 3 $ 人が最初に持っているカ`ドがそれぞれ先^からに与えられます。 具体的には、文字列 $ S_A $、$ S_B $、$ S_C $ が与えられます。文字列 $ S_A $ の $ i $ 文字目 ( $ 1\ \leq\ i\ \leq\ |S_A| $ ) にかれている文字が、A さんの持っている中で先^から $ i $ 番目のカ`ドに かれている文字です。文字列 $ S_B $、 $ S_C $ についても同です。

最K的にlがこのゲ`ムの僬撙趣胜毪を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S_A $ $ S_B $ $ S_C $

## 输出格式
A さんが伽膜胜 `A`、B さんが伽膜胜 `B`、C さんが伽膜胜 `C` と出力せよ。

## 题目大意
### 题面描述

Alice、Bob 和 Charlie 在玩 _Card Game for Three_。

- 开始时，每名玩家有一叠由卡牌组成的牌堆。每张牌上有一个字母 $\texttt a, \texttt b$ 或 $\texttt c$。 卡牌的顺序不能被改变。
- Alice 先开始游戏。
- 玩家的牌堆中至少有一张牌，当前玩家从牌堆顶抽出一张牌，这张牌代表的玩家进行下一回合（$\texttt a$ 代表 Alice，$\texttt c$ 代表 Bob，$\texttt c$ 代表 Charlie）。
- **从左往右抽牌**（牌堆顶在左边）。
- 如果当前玩家的牌堆空了，游戏结束，这名玩家胜利。

你得到了每名玩家最初的牌堆 $S_a, S_b, S_c$，求出胜者。

### 数据范围

对于 $100 \%$ 的数据，保证 $1 \leq |S_a|, |S_b|, |S_c| \leq 100$ 且 $S_a, S_b, S_c$ 仅由 $\texttt{abc}$ 三个小写拉丁字母组成。

```input1
aca
accc
ca
```

```output1
A
```

```input2
abcb
aacb
bccc
```

```output2
C
```

## 提示
### 制s

- $ 1\ \leq\ S_A\ \leq\ 100 $
- $ 1\ \leq\ S_B\ \leq\ 100 $
- $ 1\ \leq\ S_C\ \leq\ 100 $
- $ S_A $、$ S_B $、$ S_C $ に含まれる文字はそれぞれ `a`、`b`、`c` のいずれか

### Sample Explanation 1

ゲ`ムは以下のようにM行します。 - A さんが、持っている中で最初のカ`ド `a` を韦皮搿４韦 A さんの番となる。 - A さんが、持っている中で最初のカ`ド `c` を韦皮搿４韦 C さんの番となる。 - C さんが、持っている中で最初のカ`ド `c` を韦皮搿４韦 C さんの番となる。 - C さんが、持っている中で最初のカ`ド `a` を韦皮搿４韦 A さんの番となる。 - A さんが、持っている中で最初のカ`ド `a` を韦皮搿４韦 A さんの番となる。 - A さんはもう持っているカ`ドがない。よって A さんの倮となり、ゲ`ムはK了する。
