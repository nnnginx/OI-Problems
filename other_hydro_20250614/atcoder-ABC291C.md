## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc291/tasks/abc291_c

二次元平面上に高蚓がいます。高蚓は原点から移婴 $ N $ 回行いました。

$ N $ 回の移婴祥Lさ $ N $ の文字列で表され、意味は次の通りです。

- $ i $ 回目の高蚓の移俞幛巫摔稀⒁忧挨巫摔 $ (x,y) $ として、
  - $ S $ の $ i $ 文字目が `R` であるとき $ (x+1,y) $
  - $ S $ の $ i $ 文字目が `L` であるとき $ (x-1,y) $
  - $ S $ の $ i $ 文字目が `U` であるとき $ (x,y+1) $
  - $ S $ の $ i $ 文字目が `D` であるとき $ (x,y-1) $
 
$ N $ 回の移 (始点とK点を含む) で、高蚓が同じ座摔摔い郡长趣あるかどうかを判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
$ N $ 回の移 (始点とK点を含む) で、高蚓が同じ座摔摔い郡长趣あれば `Yes`、なければ `No` と出力せよ。

## 题目大意
高桥从 $(0,0)$ 的位置开始走，将移动前的坐标记为 $(x,y)$，对于字符串：
- 如果第 $i$ 个字母是 `R` 移动到 $(x+1,y)$ 的位置。
- 如果第 $i$ 个字母是 `L` 移动到 $(x-1,y)$ 的位置。
- 如果第 $i$ 个字母是 `U` 移动到 $(x,y + 1)$ 的位置。
- 如果第 $i$ 个字母是 `D` 移动到 $(x,y - 1)$ 的位置。  
如果移动过程中重复经过了同一个位置（包括起点和终点）则输出 `Yes`，否则输出 `No`。  
第一行输入一个 $N$，第二行输入一个长度为 $N$ 的字符串。

```input1
5
RLURU
```

```output1
Yes
```

```input2
20
URDDLLUUURRRDDDDLLLL
```

```output2
No
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ N $ は整数
- $ S $ は `R`, `L`, `U`, `D` のみからなるLさ $ N $ の文字列
 
### Sample Explanation 1

高蚓のいる座摔 $ (0,0)\to\ (1,0)\to\ (0,0)\to\ (0,1)\to\ (1,1)\to\ (1,2) $ と浠します。
