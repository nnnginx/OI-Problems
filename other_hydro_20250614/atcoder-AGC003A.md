## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc003/tasks/agc003_a

高蚓はo限に冥 $ 2 $ 次元平面上に住んでいて、$ N $ 日gの旅行をします。 高蚓の旅程はLさ $ N $ の文字列 $ S $ であり、はじめは家にいます。$ i(1\ Q\ i\ Q\ N) $ 日目には、

- $ S $ の $ i $ 文字目が `N` なら北に
- $ S $ の $ i $ 文字目が `W` なら西に
- $ S $ の $ i $ 文字目が `S` なら南に
- $ S $ の $ i $ 文字目が `E` なら|に

正の距xだけ移婴筏蓼埂

高蚓は、各日の移泳嚯xはQめていません。各日の移泳嚯xをうまくQめることで、 高蚓が $ N $ 日gの旅程をすべて消化したときに家にいるようにできるかどうか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
高蚓が旅程をすべて消化したときに家にいるようにできる龊悉 `Yes`, そうでない龊悉 `No` を出力せよ。

## 题目大意
高桥君在一个平面上走路N次。

他第1次从（0，0）出发，第i次从上一次的结束点出发（2<=i<=N），且每次走路的方向已经确定，但是距离没有确定。

请确定每次走路的距离（至少为1），使高桥君在这N次走路后可以回到（0，0）。

输入格式：一个长度为N的字符串S，第i个字符表示第i次走路的方向（N,E,W,S）。

输出格式：若可以确定每次走路的距离（至少为1），使高桥君在这N次走路后可以回到（0，0），输出“Yes”，否则输出“No”。

```input1
SENW
```

```output1
Yes
```

```input2
NSNNSNSN
```

```output2
Yes
```

```input3
NNEW
```

```output3
No
```

```input4
W
```

```output4
No
```
