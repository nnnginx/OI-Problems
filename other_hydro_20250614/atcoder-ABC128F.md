## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc128/tasks/abc128_f

o限に冥る池があり、数直とみなせます。この池には $ N $ のが浮かんでおり、それらは座 $ 0,1,2,....N-2,N-1 $ にあります。

あなたは、最初座$ 0 $ のの上にいます。あなたは、以下の手に兢盲匹博`ムを行うことにしました。

- 1.正の整数 $ A,B $ をQめる。得点ははじめ $ 0 $ である。
- 2.F在の位置を $ x $ として、$ y=x+A $とする。$ x $ にあるを消して、$ y $ に移婴工搿
  
  
  - $ y=N-1 $ ならば、ゲ`ムがK了する。
  - そうでなくて、$ y $ にがあるならば、得点が $ s_y $ 加する。
  - そこにがないならば、あなたは溺れる。得点が $ 10^{100} $ p少して、ゲ`ムがK了する。
- 3.F在の位置を $ x $ として、$ y=x-B $とする。$ x $ にあるを消して、$ y $ に移婴工搿
  
  
  - $ y=N-1 $ ならば、ゲ`ムがK了する。
  - そうでなくて、$ y $ にがあるならば、得点が $ s_y $ 加する。
  - そこにがないならば、あなたは溺れる。得点が $ 10^{100} $ p少して、ゲ`ムがK了する。
- 4.手2にる。

あなたは、最K得点をできるだけ大きくしたいです。 最mに $ A,B $ のをQめたときの最K得点はいくらになるでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ s_0 $ $ s_1 $ $ ...... $ $ s_{N-1} $

## 输出格式
最mに $ A,B $ のをQめたときの最K得点を出力してください。

## 题目大意
有一排荷花漂浮于水中，用它们表示一个数列 $s$，坐标 $0$ 到 $n-1$。你现在的分数为 $0$，当前为于坐标 $0$。

进行下列操作：

1.选择两个数 $A$ 和 $B$。

2.设 $y=x+A$，分三种情况：
- 若 $y = n-1$ 游戏结束。
- 若 $y \not= n-1$ 且 $y$ 点有荷花，你的分数加上 $s_{i}$。
- 若 $y \not= n-1$ 但否则你淹死，游戏结束

3.设 $y=x-B$，同上

请输出所能达到的最大分数。

```input1
5
0 2 5 1 0
```

```output1
3
```

```input2
6
0 10 -7 -4 -13 0
```

```output2
0
```

```input3
11
0 -4 0 -99 31 14 -15 -39 43 18 0
```

```output3
59
```

## 提示
### 制s

- $ 3\ \leqq\ N\ \leqq\ 10^5 $
- $ -10^9\ \leqq\ s_i\ \leqq\ 10^9 $
- $ s_0=s_{N-1}=0 $
- 入力はすべて整数である。

### Sample Explanation 1

$ A\ =\ 3,\ B\ =\ 2 $ としたとき、ゲ`ムは次のようにM行します。 - 座 $ 0\ +\ 3\ =\ 3 $ に移婴贰⒌玫悚 $ s_3\ =\ 1 $ 加する。 - 座 $ 3\ -\ 2\ =\ 1 $ に移婴贰⒌玫悚 $ s_1\ =\ 2 $ 加する。 - 座 $ 1\ +\ 3\ =\ 4 $ に移婴贰⒌玫 $ 3 $ でゲ`ムがK了する。 得点 $ 4 $ 以上でゲ`ムをK了することはできないため、答えは $ 3 $ です。座 $ 2 $ にあるに\ってその後溺れずにgますことはできないことに注意してください。

### Sample Explanation 2

ここでの最mな槁预稀$ A\ =\ 5 $ をxんで ($ B $ のは不) ただちに最後のに\ることです。
