高蚓はo限に冥欷彼惚恧紊悉摔い蓼埂

欷彼惚恧违蕙 $ (i,j) $ には整数 $ i\ \times\ j $ がかれており、高蚓は最初 $ (1,1) $ にいます。

高蚓は $ 1 $ 回の移婴 $ (i,j) $ から $ (i+1,j) $ か $ (i,j+1) $ のどちらかにのみ移ることができます。

整数 $ N $ が与えられるので、$ N $ がかれているマスに到_するまでに必要な移踊厥の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
整数 $ N $ がかれているマスに到_するまでに必要な移踊厥の最小を出力せよ。

## 题目大意
有一个平面直角坐标系，你在点$(1,1)$ ,每次可以向上走一步或向右走一步，求你走到横纵坐标之积为 $N\ (2 \leq N \leq 10^{12})$
 的点的最小步数

```input1
10
```

```output1
5
```

```input2
50
```

```output2
13
```

```input3
10000000019
```

```output3
10000000018
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 10^{12} $
- $ N $ は整数である。

### Sample Explanation 1

$ 5 $ 回の移婴 $ (2,5) $ に到_することができます。$ 5 $ 回未氦我婴扦 $ 10 $ がかれたマスに到_することは出来ません。

### Sample Explanation 2

$ 13 $ 回の移婴 $ (5,10) $ に到_できます。

### Sample Explanation 3

入出力とも非常に大きなになる可能性があります。
