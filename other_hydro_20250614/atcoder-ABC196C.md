## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc196/tasks/abc196_c

整数 $ N $ が与えられます。  
 以下の条件を氦郡 $ 1 $ 以上 $ N $ 以下の整数 $ x $ は何あるでしょうか？

- $ x $ の十M表 (先^に $ 0 $ を付けない) は偶数桁であり、その前半と後半は文字列として等しい。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
给出一个整数 $ N $。  
在 $ 1 $ 到 $ N $ 中满足以下条件的整数 $ x $ 有多少个？
* $ x $ 的十进制表示（不带前导零）有偶数位数字，并且它的前半部分和后半部分作为字符串相等。

```input1
33
```

```output1
3
```

```input2
1333
```

```output2
13
```

```input3
10000000
```

```output3
999
```

## 提示
### 制s

- $ N $ は整数
- $ 1\ <\ =\ N\ <\ 10^{12} $

### Sample Explanation 1

$ 11,\ 22,\ 33 $ の $ 3 $ が条件を氦郡筏蓼埂

### Sample Explanation 2

例えば $ 1313 $ は、十M表が $ 4 $ 桁で、その前半も後半も $ 13 $ であるため条件を氦郡筏蓼埂
