## 题目描述
小 R 今天学习了余角有关的数学知识，请你帮帮他计算一个角的余角吧！

一个角的余角的计算公式如下：

$$\angle B=90\degree -\angle A$$

其中 A 是给出的角，B 是你要计算的角。

## 输入格式
一行一个整数 $x$，表示 $\angle A=x\degree$。

## 输出格式
一行一个整数 $y$，表示 $\angle B=y\degree$。你需要严格按照公式计算。

```input1
100
```

```output1
-10
```

## 提示
对于 $100\%$ 的数据，$x$ 在 C++ 语言的 `int` 范围内。即，$x\in[-2^{31},2^{31}-1]$。
