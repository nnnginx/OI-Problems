## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc240/tasks/abc240_g

高蚓はo限に冥る三次元グリッドのマス $ (0,\ 0,\ 0) $ にいます。

高蚓は瞬g移婴摔瑜盲匹蕙工らマスへ移婴工肽芰Δ虺证盲皮い蓼埂 マス $ (x,\ y,\ z) $ にいるとき、瞬g移婴 $ 1 $ 回行うと $ (x+1,\ y,\ z),\ (x-1,\ y,\ z),\ (x,\ y+1,\ z),\ (x,\ y-1,\ z),\ (x,\ y,\ z+1),\ (x,\ y,\ z-1) $ のいずれかのマスに移婴筏蓼埂＃ē蕙 $ (x,\ y,\ z) $ にとどまることは出来ないことに注意してください。）

ちょうど $ N $ 回の瞬g移婴蛐肖盲酷幛衰蕙 $ (X,\ Y,\ Z) $ にいるような高蚓の移咏U路が何通りあるかを求めてください。

すなわち、整数の $ 3 $ つMを $ N+1 $ Kべた列 $ \big(\ (x_0,\ y_0,\ z_0),\ (x_1,\ y_1,\ z_1),\ (x_2,\ y_2,\ z_2),\ \ldots,\ (x_N,\ y_N,\ z_N)\big) $ であって、下の $ 3 $ つの条件をすべて氦郡工猡韦数を求めてください。

- $ (x_0,\ y_0,\ z_0)\ =\ (0,\ 0,\ 0) $
- $ (x_N,\ y_N,\ z_N)\ =\ (X,\ Y,\ Z) $
- $ i\ =\ 1,\ 2,\ \ldots,\ N $ について、$ |x_i-x_{i-1}|\ +\ |y_i-y_{i-1}|\ +\ |z_i-z_{i-1}|\ =\ 1 $

ただし、答えは非常に大きくなることがあるので、答えを $ 998244353 $ で割った余りを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X $ $ Y $ $ Z $

## 输出格式
答えを $ 998244353 $ で割った余りを出力せよ。

## 题目大意
在一个空间直角坐标系中移动，每步可以沿着坐标轴正/负方向移动一个单位的长度。

给定 $N,X,Y,Z$ ，求：

恰好 $N$ 步，从点 $(0,0,0)$ 走到点 $(X,Y,Z)$ 的方案数。

答案对 $998244353$ 取模。

```input1
3 2 0 -1
```

```output1
3
```

```input2
1 0 0 0
```

```output2
0
```

```input3
314 15 92 65
```

```output3
106580952
```
