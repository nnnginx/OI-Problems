## 题目描述
给一个长度为 $n$ 的字符串 $S$ ，它首尾相连形成一个环，按照顺时针排列。我们想要把它切成连续的 $K$ 段，假设每段分别为 $S_1,S_2,\cdots,S_K$，我们想让 $\max\{S_1,S_2,\cdots,S_K\}$ 最
小，求其最小值。

## 输入格式
第一行 $2$ 个数 $n,k$。

第二行字符串 $S$。

## 输出格式
一行一个字符串，表示想求的最小值。

## 样例输入
```plain
20 3
bbbaaaaabbabaabbbbaa
```

## 样例输出
```plain
aaabbabaabbbbaabbb
```

## 提示
对于 $100\%$ 的数据 $n \le 2000$，$k \le n$。
