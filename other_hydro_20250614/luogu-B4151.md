## 题目背景
本试题为 2022 年厦门市小学生 C++ 语言**复赛**试题，数据为洛谷自造。

**初赛**为笔试。

## 题目描述
期中考结束，高老师想知道班级学生成绩两极分化的情况，以便采取有效的措施来解决此问题。已知高老师班上共有 $n$ 个学生，每个学生有一个 $0\sim 100$ 分的整数成绩，现在高老师请会编程的你帮忙计算班里学生成绩最高分与最低分的差值。

## 输入格式
第一行，一个整数，表示班级的学生人数。 

接下来 $n$ 行，每行一个整数，表示一个学生的期中考成绩。

## 输出格式
输出一行，包括一个整数，表示最高分与最低分的差值。

```input1
5
68
57
76
85
80
```

```output1
28
```

## 提示
### 样例解释

最高分为 $85$，最低分为 $57$ ，即 $85-57=28$。

### 数据范围

- 对于 $20\%$ 的数据，$n\leq 5$。
- 对于 $60\%$ 的数据，$n\leq 30$。
- 对于 $100\%$ 的数据，$n\leq 50$。
