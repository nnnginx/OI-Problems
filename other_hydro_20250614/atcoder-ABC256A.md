## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc256/tasks/abc256_a

$ N $ が与えられます。$ 2^N $ を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
给你一个 $N(0\leq N\leq 30)$，求 $2^n$ 的值。

```input1
3
```

```output1
8
```

```input2
30
```

```output2
1073741824
```

## 提示
### 制s

- $ 0\ \leq\ N\ \leq\ 30 $
- $ N $ は整数である

### Sample Explanation 1

$ 2^3=8 $ です。
