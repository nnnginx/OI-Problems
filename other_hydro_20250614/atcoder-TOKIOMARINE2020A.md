あなたは同じ授Iを受けている男性に名前をい亭蓼筏俊１摔 $ S $ と名\りました。$ S $ は英小文字からなるLさ $ 3 $ 以上 $ 20 $ 以下の文字列です。 あなたは $ S $ からm当にBAする $ 3 $ 文字をxんで、彼のあだ名とすることにしました。彼のあだ名としてm切な文字列を $ 1 $ つ出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
答えを出力せよ。

## 题目大意
# 输入格式
一个字符串 $S(3\le|S|\le20)$

# 输出格式
将这个字符串中的任意连续三个字符输出，别忘了换行。

```input1
takahashi
```

```output1
tak
```

```input2
naohiro
```

```output2
nao
```
