## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc289/tasks/abc289_h

数直上に人 $ 1 $, 人 $ 2 $, 人 $ 3 $ がいます。r刻 $ 0 $ のr点で、人 $ 1 $ は地点 $ A $ に、人 $ 2 $ は地点 $ B $ に、人 $ 3 $ は地点 $ C $ にいます。  
 ここで $ A,\ B,\ C $ はすべて整数で、$ A\ \equiv\ B\ \equiv\ C\ \pmod{2} $ が成り立ちます。

$ 3 $ 人はr刻 $ 0 $ からランダムウォ`クを行います。しくh明すると、r刻 $ t $ ( $ t $ は非整数 ) のr点で地点 $ x $ にいる人は、r刻 $ t+1 $ に地点 $ x-1 $ と地点 $ x+1 $ のいずれか一方に等_率で移婴筏蓼埂(すべての移婴工敕较颏芜xkは、ランダムかつ独立です。)

このとき、r刻 $ 0 $ 以降で、r刻 $ T $ に初めて $ 3 $ 人が同じ地点にいる状Bになる_率を $ \text{mod\ }\ 998244353 $ で算してください。

  有理数 $ \text{mod\ }998244353 $ とは 求める_率は必ず有理数となることが^明できます。 またこの}の制s下では、そのを互いに素な $ 2 $ つの整数 $ P $, $ Q $ を用いて $ \frac{P}{Q} $ と表したとき、$ R\ \times\ Q\ \equiv\ P\pmod{998244353} $ かつ $ 0\ \leq\ R\ \lt\ 998244353 $ を氦郡拐数 $ R $ がただ一つ存在することが^明できます。この $ R $ を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ T $

## 输出格式
r刻 $ T $ に初めて $ 3 $ 人が同じ地点にいる状Bになる_率を $ \text{mod\ }\ 998244353 $ で算して、答えを出力せよ。

## 题目大意
一个数轴上有 $3$ 个人，分别在点 $A,B,C$ 上，且 $A,B,C$ 奇偶性相同。这 $3$ 个人从时刻 $0$ 开始移动，每经过一个时刻，一个人就会从原先所在的位置 $x$ 走到位置 $x+1$ 或 $x-1$，问在 $T$ 时刻有多大的概率 $3$ 个人 **第一次** 走到相同的位置，答案对 $998244353$ 取模，$0\le A,B,C,T\le 10^5$。

```input1
1 1 3 1
```

```output1
873463809
```

```input2
0 0 0 0
```

```output2
1
```

```input3
0 2 8 9
```

```output3
744570476
```

```input4
47717 21993 74147 76720
```

```output4
844927176
```

## 提示
### 制s

- $ 0\ \leq\ A,\ B,\ C,\ T\ \leq\ 10^5 $
- $ A\ \equiv\ B\ \equiv\ C\ \pmod{2} $
- $ A,\ B,\ C,\ T $ は整数
 
### Sample Explanation 1

r刻 $ 1 $ に初めて $ 3 $ 人が同じ地点にいる状Bになる_率は $ \frac{1}{8} $ です。$ 873463809\ \times\ 8\ \equiv\ 1\ \pmod{998244353} $ なので $ 873463809 $ を出力します。

### Sample Explanation 2

r刻 $ 0 $ のr点ですでに $ 3 $ 人が同じ地点にいる龊悉猡ります。
