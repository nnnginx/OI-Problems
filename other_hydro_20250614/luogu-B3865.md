## 题目描述
小杨想要构造一个 的 X 字矩阵（ 为奇数），这个矩阵的两条对角线都是半角加号 `+` ，其余都是半角减号 `-` 。例如，一个 $5 \times 5$ 的 X 字矩阵如下：

```plain
+---+
-+-+-
--+--
-+-+-
+---+
```

请你帮小杨根据给定的 打印出对应的“X 字矩阵”。

## 输入格式
一行一个整数 （ $5 \le N \le 49$，保证为奇数）。

## 输出格式
输出对应的“X 字矩阵”。

请严格按格式要求输出，不要擅自添加任何空格、标点、空行等任何符号。你应该恰好输出 $N$ 行，每行除了换行符外恰好包含 $N$ 个字符，这些字符要么是 `+`，要么是 `-`。

```input1
5
```

```output1
+---+
-+-+-
--+--
-+-+-
+---+
```

```input2
7
```

```output2
+-----+
-+---+-
--+-+--
---+---
--+-+--
-+---+-
+-----+
```
