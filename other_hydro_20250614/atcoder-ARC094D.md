## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc094/tasks/arc094_d

`a`,`b`,`c` からなる文字列 $ S $ が与えられます。次の操作を $ 0 $ 回以上Rり返して作ることのできる文字列としてありうるものの数を $ 998244353 $ で割ったあまりを求めてください。

- $ 1\leq\ i\leq\ |S|-1 $ かつ $ S $ の $ i $ 文字目と $ i+1 $ 文字目がなるような整数 $ i $ をxぶ。$ S $ の $ i $ 文字目と $ i+1 $ 文字目をI方、(`a`,`b`,`c` のうち)そのどちらともなる文字で置きQえる。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
操作をRり返して作ることのできる文字列としてありうるものの数を $ 998244353 $ で割ったあまりを出力せよ。

## 题目大意
给你一个仅由 $\texttt{abc}$ 组成的字符串。问你通过零次或多次下面的操作能够得到的本质不同的字符串有多少个，对 $998244353$ 取模。

- 选择相邻的两个不同的字母, 将他们同时替换为那个与他们都不相同的字符（只能从 $\texttt{abc}$ 中选择），例如选择 $\texttt{ab}$，将其替换为 $\texttt{cc}$。

```input1
abc
```

```output1
3
```

```input2
abbac
```

```output2
65
```

```input3
babacabac
```

```output3
6310
```

```input4
ababacbcacbacacbcbbcbbacbaccacbacbacba
```

```output4
148010497
```
