## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc154/tasks/arc154_a

先^の桁が $ 0 $ でない $ N $ 桁の正整数 $ A,B $ が与えられます。

あなたは、以下の操作を好きな回数（$ 0 $ 回でもよい）Rり返すことができます。

- $ 0\ \le\ i\ \le\ N-1 $ を氦郡拐数 $ i $ をxび、$ A,B $ の $ 10^{i} $ の位の数字を交Qする。
 
操作をKえたときの $ A\ \times\ B $ の最小を $ 998244353 $ で割ったあまりを求めてください。

$ A\ \times\ B $ を $ 998244353 $ で割ったあまりの最小を求めるのではないことに注意してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $

## 输出格式
答えを $ 1 $ 行に出力せよ。

## 题目大意
有两个 $n$ 位数，可以交换任意次对应位上的数（可以是 $0$ 次），希望最小化交换后两个数的乘积，输出对 $998244353$ 取模的结果。

```input1
2
13
22
```

```output1
276
```

```input2
8
20220122
21002300
```

```output2
54558365
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 200000 $
- $ A,B $ は先^の桁が $ 0 $ でない $ N $ 桁の正整数
 
### Sample Explanation 1

以下のように $ 1 $ 回操作を行うと $ A\ \times\ B $ を $ 276 $ にすることが出来ます。 - $ i=0 $ をxび、$ A,B $ の $ 1 $ の位の数字を交Qする。$ A=12,B=23 $ となる。 $ A\ \times\ B $ を $ 275 $ 以下にすることは出来ないので、答えは $ 276 $ です。

### Sample Explanation 2

$ 998244353 $ で割ったあまりを求めてください。
