## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc158/tasks/abc158_a

AtCoder 市には $ 3 $ つのkがあり、$ 1,\ 2,\ 3 $ の番号がつけられています。

これらのkは、それぞれ道会社A, Bのいずれかが管理しています。管理状rはLさ $ 3 $ の文字列 $ S $ で表され、k $ i $ は $ S_i $ が `A` のとき道会社 A が、`B` のとき道会社 B が管理しています。

道会社 A が管理しているkと、道会社 B が管理しているkのgには、交通の便のためにバスを\行することになりました。

gHにバスが\行することになるkのMみ合わせが存在するかどうかを判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
バスが\行することになるkのMみ合わせが存在する龊悉 `Yes` を、存在しない龊悉 `No` を出力せよ。

## 题目大意
输入一个长度为 $3$ 的字符串 $S$，如果 $S$ 的每一个字符都相等，输出 `No`，否则输出 `Yes`。

```input1
ABA
```

```output1
Yes
```

```input2
BBA
```

```output2
Yes
```

```input3
BBB
```

```output3
No
```

## 提示
### 制s

- $ S $ は `A` または `B` から成る
- $ |S|\ =\ 3 $

### Sample Explanation 1

k $ 1,\ 3 $ は道会社 A が、k $ 2 $ は道会社 B が管理しています。 k $ 1,\ 2 $ gとk $ 2,\ 3 $ gでバスが\行するので、`Yes` を出力してください。

### Sample Explanation 2

k $ 1,\ 2 $ は道会社 B が、k $ 3 $ は道会社 A が管理しています。 k $ 1,\ 3 $ gとk $ 2,\ 3 $ gでバスが\行するので、`Yes` を出力してください。

### Sample Explanation 3

kは全て道会社 B が管理しているので、バスは\行しません。よって `No` を出力してください。
