## 题目描述
给定正整数 $N,K$ 和一个 $N$ 位数，求在 $N$ 位数中删除 $K$ 位后剩下的数的最大值。

## 输入格式
第一行，两个整数 $N,K$。

第二行，一个 $N$ 位整数。保证没有前导 $0$。

## 输出格式
输出剩下的数的最大值。

```input1
4 2
1924
```

```output1
94
```

```input2
7 3
1231234
```

```output2
3234
```

```input3
10 4
4177252841
```

```output3
775841
```

## 提示
**【数据规模与约定】**

- 对于 $50\%$ 的数据，$N \le 1000$。
- 对于 $100\%$ 的数据，$1 \le K \lt N \le 5 \times 10^5$。
