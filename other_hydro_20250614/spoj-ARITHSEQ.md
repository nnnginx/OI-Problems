<p>
Given an arithmetic sequence a, a + d, a + 2*d, ..., a + n*d.
</p>

<h3>Request</h3>
<p>
Count the number of digit x appearing in the given sequence.
</p>

<h3>Input</h3>
<p>
- A line containing a, d, n, x.
</p>

<h3>Output</h3>
<p>
- The result found.
</p>

<h3>Example</h3>

<pre><b>Input:</b>
10 5 2 0

<b>Output:</b>
2
</pre>

<h3>Limitations</h3>
<p>
- 1 ≤ a, n ≤ 10<sup>9</sup>.<br>
- 1 ≤ d ≤ 20,000.
</p>
