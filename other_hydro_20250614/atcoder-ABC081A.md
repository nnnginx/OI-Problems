## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc081/tasks/abc081_a

すぬけ君は $ 1,2,3 $ の番号がついた $ 3 $ つのマスからなるマス目を持っています。 各マスには `0` か `1` がかれており、マス $ i $ には $ s_i $ がかれています。

すぬけ君は `1` がかれたマスにビ`玉を置きます。 ビ`玉が置かれるマスがいくつあるか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ s_{1}s_{2}s_{3} $

## 输出格式
答えを出力せよ。

## 题目大意
### **题目描述**
读入一个字符串s，求它的数字和是多少
### **输入格式**
一行，一个字符串s
### **输出格式**
一行，一个数sum，表示s字符串的数字和（个各位数相加的和）。
### **输入样例**
```
101
```
### **输出样例**
```
2
```

```input1
101
```

```output1
2
```

```input2
000
```

```output2
0
```

## 提示
### 制s

- $ s_1,\ s_2,\ s_3 $ は `1` あるいは `0`

### Sample Explanation 1

\- マス $ 1,3 $ にビ`玉が置かれます

### Sample Explanation 2

\- ビ`玉はどのマスにも置かれません
