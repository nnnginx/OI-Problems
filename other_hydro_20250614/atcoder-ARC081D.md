## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc081/tasks/arc081_d

$ H\ \times\ W $ のマス目があります． マス目の各マスは\か白かにTられており，上から $ i $ 番目，左から $ j $ 番目のマスは，$ S_i $ の $ j $ 文字目が `#` のとき\マス，`.` のとき白マスです．

すぬけ君は，マス目にして次の操作を好きな回数行うことができます．

- マス目の任意の行または列をxび，その行または列のすべてのマスの色を反する (すなわち，\でTられたマスを白に，白でTられたマスを\にTり替える)．

操作の後，すぬけ君はマス目に沿ったL方形を $ 1 $ xびます．このとき，xんだL方形に含まれるすべてのマスは\でTられていなければなりません．

うまく操作を行うとき，すぬけ君がxぶことができる最大のL方形の面eを求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ H $ $ W $ $ S_1 $ $ S_2 $ $ : $ $ S_H $

## 输出格式
すぬけ君がxぶことができる最大のL方形の面eを出力せよ．

## 题目大意
给出一个拥有 $H\times W$ 个格子的棋盘，每个格子的颜色为黑色或白色。
Snuke 可以进行任意次下列操作：
- 选择棋盘中的一行或一列，将这一行或一列的颜色翻转（黑变成白，白变成黑）
Snuke 想知道，在他进行操作后，棋盘中最大的全黑矩形最大能为多少。

```input1
3 3
..#
##.
.#.
```

```output1
6
```

```input2
4 4
....
....
....
....
```

```output2
16
```

```input3
10 8
##...#.#
##...#.#
..###.#.
#.##.#.#
.#..#.#.
..##.#.#
##.#.#..
...#.#..
###.#.##
###..###
```

```output3
27
```

## 提示
### 制s

- $ 2\ \leq\ H\ \leq\ 2000 $
- $ 2\ \leq\ W\ \leq\ 2000 $
- $ |S_i|\ =\ W $
- $ S_i $ は `#`, `.` のみからなる．
