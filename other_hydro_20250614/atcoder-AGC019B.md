分数 : $500$ 分

## 问题陈述

您有一个由小写英文字母组成的字符串 $A = A_1 A_2 ... A_n$。

您可以选择任意两个索引 $i$ 和 $j$，使得 $1 \leq i \leq j \leq n$，并反转子字符串 $A_i A_{i+1} ... A_j$。

您最多可以执行一次此操作。

您可以获得多少个不同的字符串？

## 约束条件

- $1 \leq |A| \leq 200,000$
- $A$ 由小写英文字母组成。

## 输入

输入来自标准输入，格式如下：

> $A$

## 输出

打印您最多可以通过反转 $A$ 中的任意子字符串一次所获得的不同字符串的数量。

```input1
aatt
```

```output1
5
```

您可以获得 `aatt`（什么都不做），`atat`（反转 $A[2..3]$），`atta`（反转 $A[2..4]$），`ttaa`（反转 $A[1..4]$）和 `taat`（反转 $A[1..3]$）。

```input2
xxxxxxxxxx
```

```output2
1
```

无论您反转哪个子字符串，您总是会得到 `xxxxxxxxxx`。

```input3
abracadabra
```

```output3
44
```
