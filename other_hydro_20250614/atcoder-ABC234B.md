## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc234/tasks/abc234_b

二次元平面上に $ N $ の点があります。$ i $ 目の点の座摔 $ (x_i,y_i) $ です。

この中から $ 2 $ の点をxぶとき、それらをYぶ分のLさの最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ x_1 $ $ y_1 $ $ x_2 $ $ y_2 $ $ \hspace{0.4cm}\ \vdots $ $ x_N $ $ y_N $

## 输出格式
$ 2 $ 点をYぶ分のLさの最大を出力せよ。

想定解との~`差または相`差が $ 10^{-6} $ 以下であれば正解とみなされる。

## 题目大意
给你平面上的 $N$ 对点，输出最远点对的欧几里得距离，误差在 $10^{-6}$ 内。

```input1
3
0 0
0 1
1 1
```

```output1
1.4142135624
```

```input2
5
315 271
-2 -621
-205 -511
-952 482
165 463
```

```output2
1455.7159750446
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 100 $
- $ -1000\ \leq\ x_i,y_i\ \leq\ 1000 $
- $ (x_i,y_i)\ \neq\ (x_j,y_j)\ (i\ \neq\ j) $
- 入力はすべて整数

### Sample Explanation 1

$ 1 $ 目の点と $ 3 $ 目の点をxんだときそれらをYぶ分のLさは $ \sqrt\ 2\ =\ 1.41421356237\dots $ となり、これが最大です。
