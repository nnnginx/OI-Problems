## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc083/tasks/arc088_a

高蚓は、日の感xをzめて、お母さんに数列をプレゼントすることにしました。 お母さんにプレゼントする数列 $ A $ は、以下の条件を氦郡贡匾があります。

- $ A $ は $ X $ 以上 $ Y $ 以下の整数からなる
- すべての $ 1\leq\ i\ \leq\ |A|-1 $ にし、$ A_{i+1} $ は $ A_i $ の倍数であり、かつ $ A_{i+1} $ は $ A_i $ より真に大きい

高蚓がお母さんにプレゼントできる数列のLさの最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ Y $

## 输出格式
数列のLさの最大を出力せよ。

## 题目大意
输入两个数：x,y。求x到y之间(包括x和y)的序列a的长度(这个序列后一个数是前一个数的两倍)

感谢@hzx1 提供的翻译

```input1
3 20
```

```output1
3
```

```input2
25 100
```

```output2
3
```

```input3
314159265 358979323846264338
```

```output3
31
```
