## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc059/tasks/arc059_a

$ N $ の整数 $ a_1,a_2,..,a_N $ が与えられます。えび君はこれらをきQえて全て同じ**整数**にしようとしています。各$ a_i\ (1QiQN) $は高々一回しかきQえられません(きQえなくても良い)。整数$ x $を整数$ y $にきQえるとき、コストが$ (x-y)^2 $かかります。に$ a_i=a_j\ (i≠j) $だとしても、ひとつ分のコストで同rにきQえることは出来ません(入出力例$ 2 $ を参照)。えび君が目的を_成するのに必要なコストのt和の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_1 $ $ a_2 $ ... $ a_N $

## 输出格式
えび君が全てを同じ整数にきQえるのに必要なコストのt和の最小を出力せよ。

## 题目大意
Evi有$n$个整数，分别为$a_1$，$a_2$，直到$a_n$。

他的目标是通过改变其中的一些数来使所有数相等。

对于每个整数，他最多可以变换一次。

将一个整数$x$转换为一个整数$y$会花费他$\left(x-y\right)^2$美元。

即使第$i$个数$a_i$与第$j$个数$a_j$ $\left(i!=j\right)$相等，他仍需为了改变它们中的每个数分别花费代价(请见样例2)。

请找到能够实现他目标的最小花费。

```input1
2
4 8
```

```output1
8
```

```input2
3
1 1 3
```

```output2
3
```

```input3
3
4 2 5
```

```output3
5
```

```input4
4
-100 -100 -100 -100
```

```output4
0
```

## 提示
### 制s

- $ 1QNQ100 $
- $ -100Qa_iQ100 $

### Sample Explanation 1

全てを$ 6 $にきQえると、コストのt和は$ (4-6)^2+(8-6)^2=8 $となり、これが最小です。

### Sample Explanation 2

全てを$ 2 $にきQえると$ (1-2)^2+(1-2)^2+(3-2)^2=3 $となります。各$ a_i $ごとにきQえるので、二つの$ 1 $を一度にコスト$ (1-2)^2 $できQえられるわけではないことに注意してください。

### Sample Explanation 3

$ 4 $はきQえずに、$ 2 $と$ 5 $を共に$ 4 $にきQえることで$ (2-4)^2+(5-4)^2=5 $が_成できて、これが最小です。

### Sample Explanation 4

何もきQえなくともえび君は目的を_成しています。よってこの龊膝偿攻趣$ 0 $です。
