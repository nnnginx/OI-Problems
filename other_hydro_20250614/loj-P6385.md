## 题目描述

把一个 $n \times m$ 的棋盘染成黑白两色，要求黑色的块全部四连通，白色的块也全部四连通。问有多少种方案？

四连通：一个格子与上、下、左、右四个方向的格子有连通。

## 输入格式

输入三个数 $n, m, p$。

## 输出格式

输出答案模 $p$。

```input1
2 2 998244353
```

```output1
14
```

```input2
3 3 998244353
```

```output2
108
```

## 数据范围与提示

$1 \le m \le 8$；$1 \le n \cdot m \le 10^4$；$2 \le p \le 10^9$；$p$ 是质数。
