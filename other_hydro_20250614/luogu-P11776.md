## 题目描述
枚举平面直角坐标系具有下列性质的多边形：

- 多边形顶点为坐标系整数点。
- 多边形每条边平行于坐标轴。
- 对于顶点 $(x,y)$，满足 $0 \le x \le N,0\le y\le K$。

形状相同，位置不同的两个多边形我们认为是不同的，且根据定义，多边形的边是一条不接触自身的闭合多边形线。

## 输入格式
一行两个整数 $K,N$。

## 输出格式
一行一个整数，为多边形的数量，答案对 $10007$ 取模。

```input1
2 2 
```

```output1
13
```

```input2
3 2
```

```output2
40
```

```input3
3 3 
```

```output3
213 
```

## 提示
【数据范围与约定】

- 对于所有数据，均满足 $1\le K \le 6,1 \le N \le 1000$。

| -----------: | -----------: | -----------: | -----------: | -----------: |
| $1$ | $1$ | $1000$ | 是 | $5$  |
| $2$ | $1$ | $1000$ | 否 | $5$ |
| $3$ | $2$ | $1000$ | 是 | $5$ |

| $5$ | $3$ | $1000$ | 是 | $10$ |

- 特殊性质： $N\times K \le 20$。
