## 题目描述

约翰的 $n$ 头奶牛站成一排，奶牛i的身高是 $h_i$。

现在，每只奶牛都在向左看齐。对于奶牛 $i$，如果奶牛 $j$ 满足 $i<j$ 且 $h_i<h_j$，我们可以说奶牛 $i$ 可以仰望奶牛 $j$。

求出每只奶牛离她最近的仰望对象．

## 输入格式

第1行输入N，之后每行输入一个身高。

## 输出格式

共 $n$ 行，按顺序每行输出一只奶牛的最近仰望对象。
如果没有仰望对象，输出 $0$。

```input1
6
3
2
6
1
1
2
```

```output1
3
3
0
6
6
0
```

## 题目来源

Silver

## 数据规模与约定

对于 $100 \% $ 的数据 $1 \le n \le 10^6$，$1 \le h_i \le 10^6$。
