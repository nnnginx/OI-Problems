## 题目描述
[problemUrl]: https://atcoder.jp/contests/tenka1-2019/tasks/tenka1_2019_f

$ 0,1,2 $ のみからなるLさ $ N $ の数列であって、 どのBAする部分列にしてもそれに含まれる数のt和がちょうど $ X $ にはならないようなものの数を $ 998244353 $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X $

## 输出格式
条件を氦郡故列の数を $ 998244353 $ で割ったあまりを出力せよ。

## 题目大意
给定 $n$，请构造由 $0,1,2$ 构成的长 $n$ 的序列，求有多少个序列满足它任意的子序列都不等于 $X$，结果对 $998244353$ 取模。

```input1
3 3
```

```output1
14
```

```input2
8 6
```

```output2
1179
```

```input3
10 1
```

```output3
1024
```

```input4
9 13
```

```output4
18402
```

```input5
314 159
```

```output5
459765451
```

- $ 1\ \leq\ N\ \leq\ 3000 $
- $ 1\ \leq\ X\ \leq\ 2N $
- $ N,X $ は整数である
