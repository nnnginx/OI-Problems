## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc211/tasks/abc211_e

$ N $ 行 $ N $ 列のマス目が与えられ、上から $ i $ 番目、左から $ j $ 番目のマスは、$ S_{i,\ j} $ が `#` なら\くTられており、`.` なら白くTられています。  
 あなたは白くTられたマスのうち、ちょうど $ K $ のマスをxんで赤くTります。以下の条件が氦郡丹欷Tり方は何通りありますか？

- 赤くTられたマス(以下赤マスと呼ぶ)はBYである。すなわち、どの赤マスからどの赤マスへも赤マスのみを上下左右に{って到_できる。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ S_{1,\ 1}S_{1,\ 2}\ \dots\ S_{1,\ N} $ $ S_{2,\ 1}S_{2,\ 2}\ \dots\ S_{2,\ N} $ $ \vdots $ $ S_{N,\ 1}S_{N,\ 2}\ \dots\ S_{N,\ N} $

## 输出格式
答えを出力せよ。

## 题目大意
## 题目翻译
&emsp;给你边长为 $ N $ 的且仅由字符 `#` 和 `.` 组成的正方形阵列，其中 `#` 表示黑色格子， `.` 表示白色格子。  
&emsp;你需要在白色格子中选择 $ K $ 个涂成红色，且使红色格子互相连接（仅包括上下左右相邻），求有多少种可能的方案。

## 输入格式

第一行一个整数 $ N $  
第二行一个整数 $ K $ 
以下 $ N $ 行每行 $ N $ 个字符表示给出的阵列

可能的答案

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 8 $
- $ 1\ \leq\ K\ \leq\ 8 $
- $ S_{i,\ j} $ は `#` または `.`
- $ N $ , $ K $ は整数である

### Sample Explanation 1

``` #.# #@# #@# #@# #@# @@@ .@@ @@. @@@ @@@ @@# @@# @@# .@# @.# ``` 上のように条件を氦郡Tり方が $ 5 $ 通りあります。 赤マスを `@` で表しました。 ``` #@# @.@ @@# ``` 上のTり方はBYでないので条件を氦郡筏蓼护蟆 斜めのマス同士はBYしていないことに注意してください。

### Sample Explanation 2

条件を氦郡Tり方はありません。
