## 题目描述
[problemUrl]: https://atcoder.jp/contests/wtf19/tasks/wtf19_c1

以下のような、o限に冥る三角グリッドがあります。 座摔ともに整数であるような点のそれぞれには、ランプがひとつO置されています。

![](https://cdn.luogu.com.cn/upload/vjudge_pic/AT_wtf19_c1/49b2130f65c1769a41dbab659975767a4d56bcfb.png)

はじめ、$ (X,\ 0) $ のランプのみが点灯しており、その他のランプはすべて消灯していました。 この状Bから、すぬけ君が次の操作を $ 0 $ 回以上行いました。

- $ 2 $ つの整数 $ x,\ y $ をxぶ。 $ 3 $ つのランプ $ (x,\ y),\ (x,\ y+1),\ (x+1,\ y) $ の状Bを切り替える (点灯していれば消灯させ、消灯していれば点灯させる)。

この操作のあと、$ N $ のランプ $ (x_1,\ y_1),\ \cdots,\ (x_N,\ y_N) $ が点灯しており、その他のランプはすべて消灯していました。 $ X $ を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ x_1 $ $ y_1 $ $ : $ $ x_N $ $ y_N $

## 输出格式
$ X $ を出力せよ。

## 题目大意
给你一个三角形坐标系，其中每个整点上都有一盏灯。最开始，只有 $(X,0)$ 上的灯是开着的，你将进行有限次以下操作（$0$ 次或更多）：

- 选择一个整数对 $(x,y)$，改变 $(x,y),(x,y+1),(x+1,y)$ 的开关状态。

操作完后，输入最后亮着的 $N(1 \leq N \leq 10^5)$ 盏灯的坐标 $(x_i,y_i)(10^{-17} \leq x_i,y_i \leq 10^{17})$，输出最开始的 $X$。数据保证 $X$ 唯一。

样例解释：

![](https://img.atcoder.jp/wtf19/cff6dc4d81e995e9300ccbaca5bf85de.png)

```input1
4
-2 1
-2 2
0 1
1 0
```

```output1
-1
```
