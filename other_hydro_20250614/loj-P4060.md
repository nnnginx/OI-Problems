## 题目描述

**译自 [JOI Open 2014](https://contests.ioi-jp.org/open-2014/index.html) Day1 T3「[宇宙海\](https://s3-ap-northeast-1.amazonaws.com/data.cms.ioi-jp.org/open-2014/2014-open-d1-space_pirate.pdf) / [Space Pirate](https://s3-ap-northeast-1.amazonaws.com/data.cms.ioi-jp.org/open-2014/2014-open-d1-space_pirate-en.pdf)」**

在一个遥远的星系里，有 $N$ 颗高度文明的星球，从 $1$ 到 $N$ 编号。每颗星球都提供一个传送器。每个传送器都有一个固定的目的地。传送器都是单向的。

银河帝国艺术博物馆在星系中的星球上举办艺术展览。现在，展览在 $1$ 号星球举行。下一次展览将在我们从 $1$ 号星球出发，使用传送器 $K$ 次后到达的星球举行。

太空警察发现，一个太空海盗计划偷走博物馆的藏品。太空海盗会侵入传送器系统，修改了星球 $a$ 的传送器的目的地。星球 $a$ 的传送器的目的地将被改为星球 $b$。太空海盗只会侵入一个星球的传送器系统。然而，太空警察无法找到 $a, b$ 的准确值。

为了预测下一次展览的地点，太空警察想知道对于每个 $i$，满足下一次展览将在星球 $i$ 举行的 $(a, b)$ 的对数。由于你是一个优秀的程序员，太空警察请你来帮他们计算这些对数。

给定每个传送器的目的地。对于每个 $i$，计算满足下一次展览将在星球 $i$ 举行的 $(a, b)$ 的对数。

注意：

- 星球 $i$ 的传送器的目的地可能是星球 $i$ 本身。在这种情况下，如果我们从星球 $i$ 出发，多次使用传送器，我们将留在星球 $i$。
- 即使星球 $a$ 的传送器的当前目的地是星球 $b$，太空海盗也可能侵入传送器系统，将目的地改为星球 $b$。在这种情况下，星球 $a$ 的传送器的目的地仍然是星球 $b$；在计算 $(a, b)$ 的对数时，应该包括这样的对。



## 输入格式

第一行包含两个用空格分隔的整数 $N$ 和 $K$，表示星系中有 $N$ 颗星球，下一次展览将在我们从 $1$ 号星球出发，使用传送器 $K$ 次后到达的星球举行。

接下来的 $N$ 行中的第 $i\ (1 \leq i \leq N)$ 行包含一个整数 $A_{i}\ (1 \leq A_{i} \leq N)$。表示星球 $i$ 的传送器的当前目的地是星球 $A_{i}$。

## 输出格式

输出 $N$ 行。第 $i\ (1 \leq i \leq N)$ 行包含一个整数，表示满足下一次展览将在星球 $i$ 举行的 $(a, b)$ 的对数。

```input1
5 7
5
1
4
3
2
```

```output1
1
2
3
3
16
```

## 数据范围与提示

对于所有输入数据，满足：

- $1 \leq N \leq 10^5$
- $N \leq K \leq 10^{18}$

详细子任务附加限制及分值如下表所示。

| 子任务 | 附加限制  | 分值 |
| :--: | :--: | :--: |
|  $1$  | $N\leq 100$ | $10$  |
|  $2$  | $N\leq 3000$ | $37$ |
|  $3$  | $A_{i} \neq A_{j}\ (1 \leq i<j \leq N)$ | $33$ |
|  $4$  | 无附加限制 | $20$ |
