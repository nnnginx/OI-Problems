## 题目描述
给出一列数字，需要你添加任意多个逗号将其拆成若干个严格递增的数。如果有多组解，则输出使得最后一个数最小的同时，字典序最大的解（即先要满足最后一个数最小；如果有多组解，则使得第一个数尽量大；如果仍有多组解，则使得第二个数尽量大，依次类推……）。


## 输入格式
共一行，为初始的数字。


## 输出格式
共一行，为拆分之后的数列。每个数之间用逗号分隔。行尾无逗号。


```input1
3456
```

```output1
3,4,5,6
```

```input2
3526
```

```output2
3,5,26
```

```input3
3546
```

```output3
35,46
```

```input4
0001
```

```output4
0001
```

```input5
100000101
```

```output5
100,000101
```

## 提示

### 数据范围及约定

记输入的字符串的长度为 $l$。

- 对于 $10\%$ 的数据，$l \le 5$；
- 对于 $30\%$ 的数据，$l \le 15$；
- 对于 $50\%$ 的数据，$l \le 50$；
- 对于 $100\%$ 的数据，$l \le 500$。
