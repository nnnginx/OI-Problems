## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc297/tasks/abc297_d

正整数 $ A,B $ が与えられます。

あなたは、$ A=B $ になるまで以下の操作をRり返します。

- $ A,B $ の大小vSに辘袱啤⒋韦 $ 2 $ のうちどちらかのI理を行う。
  - $ A\ >\ B $ ならば、$ A $ を $ A-B $ で置きQえる。
  - $ A\ <\ B $ ならば、$ B $ を $ B-A $ で置きQえる。
 
$ A=B $ になるまで、操作を何回行うか求めてください。ただし、有限回の操作で $ A=B $ になることが保^されます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
答えを出力せよ。

## 题目大意
给你 $A,B$ 两个数，进行以下操作 :

1. 如果 $A>B$,那么 $A=A-B$ 。
1. 如果 $A<B$,那么 $B=B-A$ 。

请输出经过多少次操作后 $A=B$ 。
## 输入格式
两个整数。
>A B

## 输出格式
答案，一个整数。

by [X_Z_Y](https://www.luogu.com.cn/user/645323)

```input1
3 8
```

```output1
4
```

```input2
1234567890 1234567890
```

```output2
0
```

```input3
1597 987
```

```output3
15
```

## 提示
### 制s

- $ 1\ \le\ A,B\ \le\ 10^{18} $
- 入力はすべて整数
 
### Sample Explanation 1

始め、$ A=3,B=8 $ です。操作は以下のように行われます。 - $ A\ <\ B $ であるため、$ B $ を $ B-A=5 $ で置きQえる。$ A=3,B=5 $ となる。 - $ A\ <\ B $ であるため、$ B $ を $ B-A=2 $ で置きQえる。$ A=3,B=2 $ となる。 - $ A\ >\ B $ であるため、$ A $ を $ A-B=1 $ で置きQえる。$ A=1,B=2 $ となる。 - $ A\ <\ B $ であるため、$ B $ を $ B-A=1 $ で置きQえる。$ A=1,B=1 $ となる。 よって、操作回数は $ 4 $ 回です。

### Sample Explanation 2

入力が 32bit 整数型にГ蓼椁胜い长趣あることに注意してください。
