## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc231/tasks/abc231_a

水Rは水深のみに依存し、水深 $ x $ \[m\] の鏊では $ \dfrac{x}{100} $ \[MPa\] になるものとします。

水深 $ D $ \[m\] の鏊の水Rは何 \[MPa\] ですか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ D $

## 输出格式
答えを出力せよ。想定解答との~`差または相`差が $ 10^{-3} $ 以下であれば正解としてQわれる。

## 题目大意
输入正整数 $n$ ，输出 $n÷100$ 的结果。误差在 $10^{-3}$ 以下算通过。

```input1
1000
```

```output1
10
```

```input2
50
```

```output2
0.5
```

```input3
3141
```

```output3
31.41
```

## 提示
### 制s

- $ 1\ \leq\ D\ \leq\ 10000 $
- $ D $ は整数である。

### Sample Explanation 1

水深 $ 1000 $ \\\[m\\\] の鏊の水Rは $ 10 $ \\\[MPa\\\] です。`10.0` や `9.999999` などを出力しても正解となります。
