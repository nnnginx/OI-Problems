## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc218/tasks/abc218_a

明日からの $ 7 $ 日gの天萦螭虮恧刮淖至 $ S $ が与えられます。  
 $ i $ 日後の予螭 $ S $ の $ i $ 文字目が `o` であるとき晴れ、`x` であるとき雨です。

$ N $ 日後の天萦螭晴れかどうかを教えてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
$ N $ 日後の天萦螭晴れであるとき `Yes` を、雨であるとき `No` を出力せよ。

## 题目大意
有一个长为 $7$ 的字符串 $s$ ，由‘o’和‘x’组成。如果字符串的第 $n$ 个字符是‘o’输出Yes，否则输出No。（字符串下标从 $1$ 开始）

```input1
4
oooxoox
```

```output1
No
```

```input2
7
ooooooo
```

```output2
Yes
```

## 提示
### 制s

- $ N $ は $ 1 $ 以上 $ 7 $ 以下の整数
- $ S $ はLさ $ 7 $ の文字列であり、`o` と `x` のみからなる

### Sample Explanation 1

明日からの $ 7 $ 日gの天萦螭享に、晴れ、晴れ、晴れ、雨、晴れ、晴れ、雨です。 特に、今日から $ 4 $ 日後の天萦螭嫌辘扦埂
