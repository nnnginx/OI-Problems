## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc157/tasks/arc157_a

`X`, `Y` からなるLさ $ N $ の文字列 $ S $ であって，以下の条件を氦郡工猡韦存在するかどうかを判定してください．

**条件:** $ S $ 中で $ 2 $ つの文字がOり合う $ (N\ -\ 1) $ w所のうち

- ちょうど $ A $ が `XX`，
- ちょうど $ B $ が `XY`，
- ちょうど $ C $ が `YX`，
- ちょうど $ D $ が `YY` である．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ A $ $ B $ $ C $ $ D $

## 输出格式
条件を氦郡刮淖至 $ S $ が存在するなら `Yes` を，存在しないなら `No` を出力せよ．

## 题目大意
确定是否存在一个由 $X$ 和 $Y$ 组成的长度为 $N$ 的字符串 $S$ ，它满足以下条件。

在 $S$ 中的 $N-1$ 对相邻字符中，正好有 $A$ 个 $XX$，$B$ 个 $XY$，$C$ 个 $YX$，$D$ 个 $YY$。

```input1
5 1 1 1 1
```

```output1
Yes
```

```input2
5 1 2 1 0
```

```output2
Yes
```

```input3
5 0 4 0 0
```

```output3
No
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ A\ \geq\ 0 $
- $ B\ \geq\ 0 $
- $ C\ \geq\ 0 $
- $ D\ \geq\ 0 $
- $ A\ +\ B\ +\ C\ +\ D\ =\ N\ -\ 1 $

たとえば $ S\ =\ {} $`XXYYX` とすると，$ 2 $ つの文字がOり合うw所は左からに `XX`, `XY`, `YY`, `YX` であり，各 $ 1 $ ずつとなって条件を氦郡筏蓼梗
