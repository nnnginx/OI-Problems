## 题目描述

给定两个正整数 $N,M$，你需要计算 $(\sum\limits_{i=1}^N\sum\limits_{j=1}^M \mu^2(\gcd(i,j))) \bmod 998244353$ 的结果。

## 输入格式

输入一行两个空格隔开的正整数，分别为 $N,M$。


## 输出格式

输出一行一个正整数表示求和取模后的结果。

```input1
2 3
```

```output1
6
```

```input2
57 71
```

```output2
3763
```

## 数据范围与提示

$1 \leq N,M \leq 10^{13}$
