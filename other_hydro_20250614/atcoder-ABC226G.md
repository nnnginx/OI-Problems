## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc226/tasks/abc226_g

重さが $ 1 $ , $ 2 $ , $ 3 $ , $ 4 $ , $ 5 $ の $ 5 $ Nの重さの荷物があり、重さが $ i $ $ (1\ \leq\ i\ \leq\ 5) $ の荷物はそれぞれ $ A_i $ あります。  
 また、体力が $ 1 $ , $ 2 $ , $ 3 $ , $ 4 $ , $ 5 $ の $ 5 $ Nの体力の人がおり、体力が $ i $ $ (1\ \leq\ i\ \leq\ 5) $ の人はそれぞれ $ B_i $ 人います。  
 それぞれの人は $ 0 $ 以上の任意の数の荷物を持つことができますが、重さの合が体力を超えるようなM合せで荷物を持つことはできません。

$ T $ のテストケ`スが与えられます。 それぞれのケ`スにして、うまく分担してすべての荷物を持つことは可能か判定してください。すなわち、各人に割り当てられた荷物の重さのt和がその人の体力を超えないように、すべての荷物をlかに割り当てることが可能か判定して下さい。荷物を $ 1 $ つも持たない人がいてもいません。

## 输入格式
入力は嗜肓Δら与えられる。入力の $ 1 $ 行目にはテストケ`ス数 $ T $ が与えられる。

> $ T $

その後、 $ T $ のテストケ`スがAく。各テストケ`スは以下の形式で与えられる。

> $ A_1 $ $ A_2 $ $ A_3 $ $ A_4 $ $ A_5 $ $ B_1 $ $ B_2 $ $ B_3 $ $ B_4 $ $ B_5 $

## 输出格式
$ T $ 行出力せよ。 $ i $ $ (1\leq\ i\leq\ T) $ 行目には、$ i $ 番目のテストケ`スについてすべての荷物を持つことが可能なら `Yes` を、そうでないならば `No` を出力せよ。

## 题目大意
### 题意概述
有五种重物和五个人，第 $i (1 ≤ i ≤ 5)$ 种重物的质量为 $A_{i}$，第 $i (1 ≤ i ≤ 5)$ 个人可搬运总质量不超过 $B_{i}$ 的重物。问是否存在一种分配方式，使所有重物均可被一次性搬运。

### 输入格式
本题具有多组测试数据。

第 $1$ 行为一个整数 $T$，表示本测试点共有 $T$ 组测试数据。

第 $2i (1 ≤ i ≤ T)$ 行为 $5$ 个整数，分别为 $A_{1}$，$A_{2}$，$A_{3}$，$A_{4}$，$A_{5}$。

第 $2i+1 (1 ≤ i ≤ T)$ 行为 $5$ 个整数，分别为 $B_{1}$，$B_{2}$，$B_{3}$，$B_{4}$，$B_{5}$。

### 输出格式
输出共 $T$ 行，若 $i (1 ≤ i ≤ T)$ 组测试数据中所有重物均可被一次性搬运，则在第 $i$ 行输出 `Yes` ，否则输出 `No`。

```input1
3
5 1 0 0 1
0 0 0 2 1
0 3 0 0 0
0 0 2 0 0
10000000000000000 0 0 0 0
0 0 0 0 2000000000000000
```

```output1
Yes
No
Yes
```

## 提示
### 制s

- $ 1\ \leq\ T\ \leq\ 5\times\ 10^4 $
- $ 0\ \leq\ A_i,B_i\ \leq\ 10^{16} $
- $ 1\ \leq\ A_1+A_2+A_3+A_4+A_5 $
- $ 1\ \leq\ B_1+B_2+B_3+B_4+B_5 $
- 入力は全て整数である。

### Sample Explanation 1

$ 1 $ つめのテストケ`スでは、例えば以下のようにすればすべての荷物を持つことができます。 - 体力 $ 4 $ の人のうちの $ 1 $ 人目が、重さ $ 1 $ の荷物を $ 4 $ つ持つ。 - 体力 $ 4 $ の人のうちの $ 2 $ 人目が、重さ $ 1 $ の荷物と重さ $ 2 $ の荷物を $ 1 $ つずつ持つ。 - 体力 $ 5 $ の人が、重さ $ 5 $ の荷物を $ 1 $ つ持つ。 $ 2 $ つめのテストケ`スでは、体力が $ 3 $ の人のどちらかが重さ $ 2 $ の荷物を $ 2 $ つ以上持つ必要があり、不可能です。
