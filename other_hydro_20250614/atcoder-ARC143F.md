## 题目描述

正の整数 $ N $ が与えられるので，次の条件を氦郡$ \{1,2,\ldots,N\} $ の部分集合 $ S $ の数を $ 998244353 $ で割ったあまりを求めてください．

- $ N $ 以下の正の整数はすべて $ S $ のいくつかの相なる要素の和として表せる．さらに，そのような表し方は高々 $ 2 $ 通りである．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $

## 输出格式
答えを出力せよ．

```input1
3
```

```output1
2
```

```input2
5
```

```output2
5
```

```input3
1000
```

```output3
742952024
```
