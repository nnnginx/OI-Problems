## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc153/tasks/abc153_d

カラカルはモンスタ`と椁盲皮い蓼埂

モンスタ`の体力は $ H $ です。

カラカルはモンスタ`を $ 1 $ 体xんで攻膜工毪长趣できます。モンスタ`を攻膜筏郡趣、攻象のモンスタ`の体力に辘袱啤⒋韦韦嗓沥椁が起こります。

- モンスタ`の体力が $ 1 $ なら、そのモンスタ`の体力は $ 0 $ になる
- モンスタ`の体力が $ X\ >\ 1 $ なら、そのモンスタ`は消绀贰⑻辶Δ $ \lfloor\ X/2\ \rfloor $ のモンスタ`が新たに $ 2 $ 体Fれる

（$ \lfloor\ r\ \rfloor $ は $ r $ を超えない最大の整数を表す）

全てのモンスタ`の体力を $ 0 $ 以下にすればカラカルの伽沥扦埂

カラカルがモンスタ`に伽膜蓼扦诵肖攻膜位厥の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ H $

## 输出格式
カラカルがモンスタ`に伽膜蓼扦诵肖攻膜位厥の最小を出力せよ。

## 题目大意
Caracal 在打怪。  
一开始，他的面前有一只怪，这个怪有 $H$ 点血量。  
每一次，他可以选择一个怪物进行攻击，具体方式如下：  
1. 如果这只怪兽的血量为 $1$ ，变为 $0$；  
2. 如果这只怪兽的血量 $X > 1$ ，那么这只怪兽将分裂成两只，每只的血量为 $\lfloor X/2 \rfloor$。  

（其中 $\lfloor r \rfloor$ 表示 $\le r$ 的整数中，最大的一个）  
当所有存在的怪物血量全部 $\le 0$ 时，Caracal 就获胜了。  
输出他在获胜之前的攻击次数。

```input1
2
```

```output1
3
```

```input2
4
```

```output2
7
```

```input3
1000000000000
```

```output3
1099511627775
```

## 提示
### 制s

- $ 1\ \leq\ H\ \leq\ 10^{12} $
- 入力中のすべてのは整数である。

### Sample Explanation 1

モンスタ`を攻膜工毪取⒃のモンスタ`は消绀贰⑻辶 $ 1 $ のモンスタ`が $ 2 $ 体Fれます。 この $ 2 $ 体のモンスタ`をそれぞれ $ 1 $ 回ずつ攻膜贰⒑嫌 $ 3 $ 回の攻膜伽膜长趣できます。
