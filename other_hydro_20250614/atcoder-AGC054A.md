## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc054/tasks/agc054_a

英小文字からなるLさ $ N $ の文字列 $ S $ が与えられます．

あなたは，$ S $ にして以下の操作を好きな回数行えます．

- 先^の文字と最後の文字がなるBAした（非空な）部分列をxび，これを削除する．

$ S $ を空文字列にすることが可能か判定し，可能な龊悉媳匾な最小の操作回数を求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ S $

## 输出格式
$ S $ を空文字列にすることが可能な龊希必要な最小の操作回数を出力せよ． 不可能な龊希$ -1 $ を出力せよ．

## 题目大意
给出一个由小写字母组成的长度为 $n$ 的字符串 $s$。

可以对 $s$ 进行以下操作：

- 删除一个首尾字符不相同的子串。

求最少多少次操作可使 $s$ 变成空串，如不能变成空串，输出 $-1$。

```input1
4
abba
```

```output1
2
```

```input2
3
aba
```

```output2
-1
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 10^5 $
- $ S $ は英小文字からなるLさ $ N $ の文字列

### Sample Explanation 1

`abba` →（`ab`をxんで削除）→ `ba` →（`ba`をxんで削除）→ 空文字列 と操作すればよいです．
