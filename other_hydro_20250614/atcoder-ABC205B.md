## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc205/tasks/abc205_b

$ 1 $ 以上 $ N $ 以下の整数からなるLさ $ N $ の数列 $ A\ =\ (A_1,\ A_2,\ \dots,\ A_N) $ が与えられます。

$ A $ が $ (1,\ 2,\ \dots,\ N) $ のKび替えによって得られるかどうか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $

## 输出格式
$ A $ が $ (1,\ 2,\ \dots,\ N) $ のKび替えによって得られるなら `Yes`、そうでないなら `No` と出力せよ。

## 题目大意
输入一个长为 $n$ 的数列 $a$ ，判断 $a$ 是否由 $1$ 到 $n$ 的这 $n$ 个正整数重新排列得到。

```input1
5
3 1 2 4 5
```

```output1
Yes
```

```input2
6
3 1 4 1 5 2
```

```output2
No
```

```input3
3
1 2 3
```

```output3
Yes
```

```input4
1
1
```

```output4
Yes
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^3 $
- $ 1\ \leq\ A_i\ \leq\ N $
- 入力は全て整数である。

### Sample Explanation 1

$ (3,\ 1,\ 2,\ 4,\ 5) $ は $ (1,\ 2,\ 3,\ 4,\ 5) $ をKび替えて得られるため、`Yes` と出力します。

### Sample Explanation 2

$ (1,\ 2,\ 3,\ 4,\ 5,\ 6) $ をどのようにKび替えても $ (3,\ 1,\ 4,\ 1,\ 5,\ 2) $ にすることはできないので、`No` と出力します。
