## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc040/tasks/agc040_a

Lさ $ N-1 $ の文字列 $ S $ が与えられます． $ S $ の各文字は `<` または `>` です．

Lさ $ N $ の非整数列 $ a_1,a_2,\cdots,a_N $ は， すべての $ i $ ($ 1\ \leq\ i\ \leq\ N-1 $) について次の条件をみたすr，**良い非整数列**と呼ばれます．

- $ S_i= $ `<` のとき： $ a_i\ <\ a_{i+1} $
- $ S_i= $ `>` のとき： $ a_i\ >\ a_{i+1} $

良い非整数列の要素のt和としてありうる最小のを求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ S $

## 输出格式
良い非整数列の要素のt和としてありうる最小のを出力せよ．

## 题目大意
给定一个长度为 $N-1$ 的字符串 $S$，每个元素为 `>` 或 `<`。

我们称一个长度为 $N$ 的非负整数序列 $\{a_n\}$ 为好的，当且仅当对于任意的 $1\le i\le N-1$，都满足以下条件：

- 若 $S_i = $ `>` ，则 $a_i>a_{i+1}$

- 若 $S_i = $ `<` ，则 $a_i<a_{i+1}$

定义一个好的序列的权值为序列中所有元素的和，求所有好的序列中可能的最小权值。

```input1
<>>
```

```output1
3
```

```input2
<>>><<><<<<<>>><
```

```output2
28
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 5\ \times\ 10^5 $
- $ S $ は `<` と `>` のみから成るLさ $ N-1 $ の文字列．

### Sample Explanation 1

$ a=(0,2,1,0) $ は良い非整数列であり， この龊悉我素のt和は $ 3 $ になります． 要素のt和が $ 3 $ より小さい良い非整数列は存在しません．
