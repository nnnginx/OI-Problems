## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc158/tasks/arc158_b

$ 0 $ でない整数 $ x_1,\ \ldots,\ x_N $ が与えられます．$ i,j,k $ を $ 1\leq\ i\ <\ j\ <\ k\leq\ N $ を氦郡拐数とするとき，$ \dfrac{x_i+x_j+x_k}{x_ix_jx_k} $ としてありうる最小と最大を求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられます．

> $ N $ $ x_1 $ $ \ldots $ $ x_N $

## 输出格式
$ \dfrac{x_i+x_j+x_k}{x_ix_jx_k} $ としてありうる最小と最大を，それぞれ 1 行目，2 行目に出力してください．

~`差または相`差が $ 10^{-12} $ 以内であれば，正解と判定されます．

## 题目大意
给定 $n$ 个非零数 $x_1,x_2,\cdots x_n$，求出三元组 $(i,j,k)$ 满足 $1\le i<j<k\le n$ 并且 $\frac{x_i+x_j+x_k}{x_ix_jx_k}$ 最小或者最大。输出这个最小值和最大值。

```input1
4
-2 -4 4 5
```

```output1
-0.175000000000000
-0.025000000000000
```

```input2
4
1 1 1 1
```

```output2
3.000000000000000
3.000000000000000
```

```input3
5
1 2 3 4 5
```

```output3
0.200000000000000
1.000000000000000
```

## 提示
### 制s

- $ 3\leq\ N\leq\ 2\times\ 10^5 $
- $ -10^6\leq\ x_i\ \leq\ 10^6 $
- $ x_i\neq\ 0 $
 
### Sample Explanation 1

$ \dfrac{x_i+x_j+x_k}{x_ix_jx_k} $ としてありうるは次の $ 4 $ 通りです． - $ (i,j,k)\ =\ (1,2,3) $：$ \dfrac{(-2)\ +\ (-4)\ +\ 4}{(-2)\cdot\ (-4)\cdot\ 4}\ =\ -\dfrac{1}{16} $． - $ (i,j,k)\ =\ (1,2,4) $：$ \dfrac{(-2)\ +\ (-4)\ +\ 5}{(-2)\cdot\ (-4)\cdot\ 5}\ =\ -\dfrac{1}{40} $． - $ (i,j,k)\ =\ (1,3,4) $：$ \dfrac{(-2)\ +\ 4\ +\ 5}{(-2)\cdot\ 4\cdot\ 5}\ =\ -\dfrac{7}{40} $． - $ (i,j,k)\ =\ (2,3,4) $：$ \dfrac{(-4)\ +\ 4\ +\ 5}{(-4)\cdot\ 4\cdot\ 5}\ =\ -\dfrac{1}{16} $． これらの最小は $ -\dfrac{7}{40} $，最大は $ -\dfrac{1}{40} $ です．
