## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc016/tasks/agc016_b

$ N $ 匹の猫がいます。 猫には $ 1 $ から $ N $ まで番号が振られています。

各猫はある色の帽子を被っています。 猫 $ i $ は「自分を除く $ N-1 $ 匹の猫が被っている帽子の色のN数はちょうど $ a_i $ である」と言っています。

すべての猫のk言と矛盾しないような帽子の色のM合せが存在するか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_1 $ $ a_2 $ $ ... $ $ a_N $

## 输出格式
すべての猫のk言と矛盾しないような帽子の色のM合せが存在するならば、`Yes` を出力せよ。 存在しないならば、`No` を出力せよ。

## 题目大意
```
有$N$只猫，每只猫带着某种颜色的帽子，给出每只猫能看到（即其他$N-1$只猫）的颜色种数$a[i]$，问是否可以构造出合法序列。
```
有$N$只猫，每只猫带着某种颜色的帽子，给出每只猫能看到（即其他$N-1$只猫）的颜色种数$a[i]$，问是否可以构造出合法序列。

```input1
3
1 2 2
```

```output1
Yes
```

```input2
3
1 1 2
```

```output2
No
```

```input3
5
4 3 4 3 4
```

```output3
No
```

```input4
3
2 2 2
```

```output4
Yes
```

```input5
4
2 2 2 2
```

```output5
Yes
```

```input6
5
3 3 3 3 3
```

```output6
No
```

## 提示
### 制s

- $ 2\ <\ =\ N\ <\ =\ 10^5 $
- $ 1\ <\ =\ a_i\ <\ =\ N-1 $

### Sample Explanation 1

例えば、猫 $ 1 $, $ 2 $, $ 3 $ の帽子の色がそれぞれ赤、青、青ならば、すべての猫のk言と矛盾しません。

### Sample Explanation 2

猫 $ 1 $ のk言から、猫 $ 2 $, $ 3 $ の帽子の色は同一です。 また、猫 $ 2 $ のk言から、猫 $ 1 $, $ 3 $ の帽子の色は同一です。 よって、猫 $ 1 $, $ 2 $ の帽子の色は同一ですが、これは猫 $ 3 $ のk言に矛盾します。
