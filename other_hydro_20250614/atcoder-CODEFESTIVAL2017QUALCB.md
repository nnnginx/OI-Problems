## 题目描述
[problemUrl]: https://atcoder.jp/contests/code-festival-2017-qualc/tasks/code_festival_2017_qualc_b

$ 2 $ つのLさ $ N $ の整数列 $ x_1,\ x_2,\ ...,\ x_N $ と $ y_1,\ y_2,\ ...,\ y_N $ が「似ている」とは、 任意の $ i $ ($ 1\ \leq\ i\ \leq\ N $) にして $ |x_i\ -\ y_i|\ \leq\ 1 $ が成り立つことをいうものとします。

とくに、どの整数列もその数列自身と似ていると考えます。

整数 $ N $ およびLさ $ N $ の整数列 $ A_1,\ A_2,\ ...,\ A_N $ が与えられます。

$ A $ と似ている整数列 $ b_1,\ b_2,\ ...,\ b_N $ であって、すべてののe $ b_1\ b_2\ ...\ b_N $ が偶数となるものはいくつあるか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $

## 输出格式
条件を氦郡拐数列の数を出力せよ。

## 题目大意
给定一个数组 $a$，求有多少个数组 $b$，使得数组 $b$ 的每一个值都与数组 $a$ 相似，且数组 $b$ 的所有值之积为偶数。

```input1
2
2 3
```

```output1
7
```

```input2
3
3 3 3
```

```output2
26
```

```input3
1
100
```

```output3
1
```

```input4
10
90 52 56 71 44 8 13 30 57 84
```

```output4
58921
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10 $
- $ 1\ \leq\ A_i\ \leq\ 100 $
