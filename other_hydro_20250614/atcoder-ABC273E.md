## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc273/tasks/abc273_e

整数列 $ A $ とノ`トがあります。ノ`トには $ 10^9 $ 枚のペ`ジがあります。

$ Q $ のクエリが与えられます。各クエリは下の $ 4 $ Nのいずれかです。

> ADD $ x $ : 整数 $ x $ を $ A $ の末尾に追加する。

> DELETE : $ A $ の末尾の要素を削除する。ただし、$ A $ が空である龊悉虾韦猡筏胜ぁ

> SAVE $ y $ : ノ`トの $ y $ ペ`ジ目にかれている数列を消し、代わりにF在の $ A $ を $ y $ ペ`ジ目にきzむ。

> LOAD $ z $ : $ A $ をノ`トの $ z $ ペ`ジ目にかれている数列で置きQえる。

はじめ、$ A $ は空列であり、ノ`トのすべてのペ`ジには空列の情螭かれています。 その初期状Bから、$ Q $ のクエリを与えられるにg行し、各クエリのg行直後における $ A $ の末尾の要素を出力してください。

なお、入出力の量が多くなる龊悉あるので、高速な方法で入出力を行うことを推Xします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ Q $ $ \mathrm{query}_1 $ $ \mathrm{query}_2 $ $ \vdots $ $ \mathrm{query}_Q $

## 输出格式
下の形式にしたがい、$ i\ =\ 1,\ 2,\ \ldots,\ Q $ について、$ i $ 番目までのクエリをg行した直後の $ A $ の末尾の要素 $ X_i $ を（ $ A $ が空の龊悉 $ X_i\ :=\ -1 $ とする）出力せよ。

> $ X_1 $ $ X_2 $ $ \ldots $ $ X_Q $

## 题目大意
有一个版本保存系统，共有 $10^9$ 个版本，每个版本初始都为空列表，还需要维护一个列表（后称为“当前列表”）。  
您需要实现如下四种操作：
- `ADD x`：在当前列表的末尾添加 $x$
- `DELETE`：如果当前列表非空，把当前列表的末尾最后一个数删除。否则，什么也不做。
- `SAVE x`：把当前列表保存至第 $x$ 版本（在此后完成的操作不会在第 $x$ 版本中出现，而且保存后当前列表不清空）
- `LOAD x`：把当前列表变成第 $x$ 版本（直接赋值，而不是添加，而且保存后第 $x$ 版本不清空）  
给定 $q$ 次操作，每次操作是以上四种操作，求每次操作**后**的当前列表的末尾最后一个数（若数组为空输出 $-1$）。

```input1
11
ADD 3
SAVE 1
ADD 4
SAVE 2
LOAD 1
DELETE
DELETE
LOAD 2
SAVE 1
LOAD 3
LOAD 1
```

```output1
3 3 4 4 3 -1 -1 4 4 -1 4
```

```input2
21
ADD 4
ADD 3
DELETE
ADD 10
LOAD 7
SAVE 5
SAVE 5
ADD 4
ADD 4
ADD 5
SAVE 5
ADD 2
DELETE
ADD 1
SAVE 5
ADD 7
ADD 8
DELETE
ADD 4
DELETE
LOAD 5
```

```output2
4 3 4 10 -1 -1 -1 4 4 5 5 2 5 1 1 7 8 7 4 7 1
```
