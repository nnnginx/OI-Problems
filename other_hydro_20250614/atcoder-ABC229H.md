## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc229/tasks/abc229_h

$ N $ 行 $ N $ 列のグリッドがあり、各マスには白いxが $ 1 $ 置かれているか、\いxが $ 1 $ 置かれているか、何も置かれていないかのいずれかです。  
 上から $ i $ 行目、左から $ j $ 列目のマスの状Bは $ S_{i,j} $ で表され、`W` のとき白いxが置かれていて、`B` のとき\いxが置かれていて、`.` のときは何も置かれていない空マスです。

高蚓とすぬけ君がゲ`ムをします。高蚓からはじめて、交互にタ`ンが回ってきます。

高蚓のタ`ンには、

- $ 1 $ つ上に空マスがあるような**白**いxを $ 1 $ つxび、上にMめる
- 好きな**\**いxを $ 1 $ つ食べる

のいずれかの操作をします。

すぬけ君のタ`ンには、

- $ 1 $ つ上に空マスがあるような**\**いxを $ 1 $ つxび、上にMめる
- 好きな**白**いxを $ 1 $ つ食べる

のいずれかの操作をします。

操作ができなくなった方がけとします。I者が最mに行婴工毪趣、どちらが伽沥蓼工？

なお「xを上にMめる」とは、$ i $ 行 $ j $ 列目のxを $ i-1 $ 行 $ j $ 列目に移婴丹护毪长趣蛑袱筏蓼埂  
 高蚓とすぬけ君は同じ向きからP面をていて、「上」というのはI者にとって同じ向きであることに注意してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S_{1,1}S_{1,2}\ldots\ S_{1,N} $ $ S_{2,1}S_{2,2}\ldots\ S_{2,N} $ $ \vdots $ $ S_{N,1}S_{N,2}\ldots\ S_{N,N} $

## 输出格式
高蚓が伽膜胜椁 `Takahashi` と、すぬけ君が伽膜胜椁 `Snuke` と出力せよ。

## 题目大意
给定一个 $n\times n$ 的棋盘, 每个格子上有一个黑棋或一个白棋或什么也没有, 两人轮流进行操作, 无法进行操作的人输.

先手每次可以进行以下两种操作之一

1. 选一个可以向上移动的白棋, 将其上移一格.

2. 吃掉一个黑棋.

后手每次可以进行以下两种操作之一

1. 选一个可以向上移动的黑棋, 将其上移一格.

2. 吃掉一个白棋.

不能将棋子移出棋盘.

加入两人均走最优策略, 问最终胜者是谁.

(两人看棋盘的方向相同)

```input1
3
BB.
.B.
...
```

```output1
Takahashi
```

```input2
2
..
WW
```

```output2
Snuke
```

```input3
4
WWBW
WWWW
BWB.
BBBB
```

```output3
Snuke
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 8 $
- $ N $ は整数
- $ S_{i,j} $ は `W` または `B` または `.` である

### Sample Explanation 1

はじめに高蚓が $ 1 $ 行 $ 1 $ 列目の\いxを食べると、P面は下のようになります。 ``` .B. .B. ... ``` このときすぬけ君は操作を行うことができないので、高蚓の伽沥扦埂 P面の外にxを婴すことや、他のxがあるマスにxを移婴丹护毪长趣悉扦ないことに注意してください。
