## 题目描述

话说花神这天又来讲课了。课后照例有超级难的神题啦…… 我等Xm又遭殃了。

花神的题目是这样的:

设 $sum(i)$ 表示 $i$ 的二进制表示中 $1$ 的个数。给出一个正整数 $n$ ，花神要问你 $\Pi_{1}^{n}sum(i)$，也就是 $sum(1)\sim sum(n)$ 的乘积。

## 输入格式

一个正整数 $n$。

## 输出格式

一个数，答案$\mod 10^7 + 7$ 的值。

```input1
3
```

```output1
2
```

## 样例说明

对于样例一，$1 \times 1 \times 2 = 2$。

## 数据规模与约定

* 对于 $100\%$ 的数据，$n \leq 10^{15}$。
