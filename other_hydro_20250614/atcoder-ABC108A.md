## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc108/tasks/abc108_a

$ 1 $ 以上 $ K $ 以下の正の整数から、偶数と奇数ひとつずつのMをxぶ方法の数を求めてください。なお、xぶ番は考]しません。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ K $

## 输出格式
$ 1 $ 以上 $ K $ 以下の正の整数から、偶数と奇数ひとつずつのMをxぶ方法の数を出力せよ。

## 题目大意
请求出从 $1$ 以上，$K$ 以下（包括$1$和$K$）的正整数中，偶数和奇数各选一个的方法的个数。不考虑选择的顺序，即，$(a,b)$ 和 $(b,a)$ 算作同一种方案。

### 输入格式

一个正整数$K$。

### 输出格式

方案数。

### 数据范围与样例解释

$2 \le K \le 100$ 。

第一组样例：只有 $(2,1)$ 和 $(2,3)$ 两种方案符合条件。

```input1
3
```

```output1
2
```

```input2
6
```

```output2
9
```

```input3
11
```

```output3
30
```

```input4
50
```

```output4
625
```

## 提示
### 制s

- $ 2\leq\ K\leq\ 100 $
- $ K $ は整数である

$ (2,1) $ と $ (2,3) $ が条件を氦郡筏蓼埂
