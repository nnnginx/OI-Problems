$ -10^{18} $ 以上 $ 10^{18} $ 以下の整数 $ N $ が与えられます。

以下の条件を氦郡 $ 0 $ 以上 $ 998244353 $ 未氦握数 $ x $ を求めてください。なお、答えが一意に定まることが^明できます。

- $ N-x $ は $ 998244353 $ の倍数

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
您将获得一个介于 $-10^{18}$和 $10^{18}$（包括 $-10^{18}$和 $10^{18}$ ）之间的整数 $N$	

查找满足以下条件的大于或等于 $0$ 且小于 $998244353$ 的整数 $x$;

- $N-x$ 是 $998244353$ 的倍数

此外，您可以证明答案是唯一的。

```input1
998244354
```

```output1
1
```

```input2
-9982443534
```

```output2
998244349
```

## 提示

- $ N $ は $ -10^{18} $ 以上 $ 10^{18} $ 以下の整数

$ 998244354-1\ =\ 998244353 $ は $ 998244353 $ の倍数なので条件を氦郡筏蓼埂

### Sample Explanation 2
