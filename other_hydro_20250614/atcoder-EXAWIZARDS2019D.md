## 题目描述
[problemUrl]: https://atcoder.jp/contests/exawizards2019/tasks/exawizards2019_d

すぬけ君は\板と $ N $ の整数からなる集合 $ S $ を持っています。 $ S $ の $ i $ 番目の要素は $ S_i $ です。

すぬけ君は\板に整数 $ X $ をいたのち、以下の操作を $ N $ 回行います。

- $ S $ から要素を $ 1 $ つxんで取り除く。
- その後、F在\板にかれた数を $ x $、$ S $ から取り除かれた整数を $ y $ として、\板にかれた数を $ x\ \bmod\ {y} $ にきQえる。

$ S $ から要素を取り除く序は $ N! $ 通りありえます。 それぞれの龊悉摔膜い啤$ N $ 回の操作後に\板にかれている数を求め、そのt和を $ 10^{9}+7 $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X $ $ S_1 $ $ S_2 $ $ \ldots $ $ S_{N} $

## 输出格式
答えを出力せよ。

## 题目大意
Snuke有一块黑板和一组由$ N $整数组成的$ S $。$ S $中的第 $i$ 个元素是$ S_i $。

他在黑板上写了一个整数$ X $，然后执行了$ N $次以下操作：

- 从$ S $中选择一个元素并将其删除。
- 让$ x $是现在写在黑板上的数字，$ y $是从$ S $中删除的整数。用$ x \bmod{y} $替换黑板上的数字。

有 $N!$ 个可能的顺序，其中元素从 $S$ 中删除。对于每个元素，找到在 $N$ 操作之后要写在黑板上的数字，然后计算所有这些 $N!$ 的总和数字取模 $10^ {9} + 7$。

```input1
2 19
3 7
```

```output1
3
```

```input2
5 82
22 11 6 5 13
```

```output2
288
```

```input3
10 100000
50000 50001 50002 50003 50004 50005 50006 50007 50008 50009
```

```output3
279669259
```

## 提示
### 制s

- 入力は全て整数である。
- $ 1\ \leq\ N\ \leq\ 200 $
- $ 1\ \leq\ S_i,\ X\ \leq\ 10^{5} $
- $ S_i $ たちは相なる。

### Sample Explanation 1

\- $ S $ から数を取り除く序は $ 2 $ 通りあります。 - $ 3,7 $ ので取り除いたとき、\板にかれた数は $ 19\ \rightarrow\ 1\ \rightarrow\ 1 $ と浠します。 - $ 7,3 $ ので取り除いたとき、\板にかれた数は $ 19\ \rightarrow\ 5\ \rightarrow\ 2 $ と浠します。 - これらのt和である $ 3 $ を出力してください。

### Sample Explanation 3

\- t和を $ 10^{9}+7 $ で割ったあまりを求めるのを忘れずに。
