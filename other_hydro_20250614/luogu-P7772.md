## 题目背景
本题为[$\texttt{COCI 2009-2010}\ 2^\texttt{nd}\ \texttt{round}\ \text{T1 FAKTOR}$](https://hsin.hr/coci/archive/2009_2010/contest2_tasks.pdf)。

分值按原题设置，满分 $30$。

## 题目描述
给定 $A,I$，求一个最小的 $N$，使得$\lceil\frac{N}{A}\rceil\geq I$。

## 输入格式
一行两个整数 $A,I$。

## 输出格式
一行一个整数 $N$。

```input1
38 24
```

```output1
875
```

```input2
1 100
```

```output2
100
```

```input3
10 10
```

```output3
91
```

## 提示
$1\leq A,I\leq 100$。
