## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc078/tasks/abc078_b

幅 $ X $ センチメ`トルの椅子があります。 この椅子に座りたい人がたくさんおり，人は椅子に座ると必ず $ Y $ センチメ`トルの幅を使って座ります。

出来る限りたくさんの人を椅子に座らせたいですが， 人はみなシャイなので，人と人のg，また椅子の端と人のgには， 少なくとも $ Z $ センチメ`トルgを_ける必要があります。

最大で何人座ることができますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ Y $ $ Z $

## 输出格式
求めた答えを出力してください。

## 题目大意
##### 大意：椅子上坐人，椅子长为X厘米，每个人占地长为Y厘米,每人之间和座位末端的人与椅子边缘的间隔为Z厘米长，求椅子最大能坐多少人。

##### 输入：X Y Z

1 ≤ X,Y,Z ≤ 105

Y + 2*Z ≤ X

X,Y,Z都为整数

##### 输出：最大人数

## 原题:
我们有一个宽X厘米的长座位。有很多人想坐在这里。坐在座位上的人总是占据长度为Y厘米的间隔。

我们想让尽可能多的人坐下来，但他们都非常害羞，两个人之间，以及座位末端和一个人之间必须有至少Z厘米的长度差距。

最多有多少人可以坐在座位上？

```input1
13 3 1
```

```output1
3
```

```input2
12 3 1
```

```output2
2
```

```input3
100000 1 1
```

```output3
49999
```

```input4
64146 123 456
```

```output4
110
```

```input5
64145 123 456
```

```output5
109
```
