## 题目描述

**译自 [COCI 2021/2022 Contest #1](https://hsin.hr/coci/contest1_tasks.pdf) T4「Set」**

定义一个有序多元组 $a$ 的第 $i$ 项为 $a_i$。

给定一个 $n$ 个有序 $m$ 元组 $b$，要从这些 $m$ 元组中选出 $3$ 个，设这三个 $m$ 元组的下标为 $i,j,k$，他们要满足如下条件：

- $i<j<k$；
- $\forall 1\le z\le m,b_{i,z}=b_{j,z}=b_{k,z}$ 或者  $b_{i,z}\not=b_{j,z},b_{i,z}\not=b_{k,z},b_{j,z}\not=b_{k,z}$。

请问有多少种选法可以选出这个三元组。

## 输入格式

第一行为两个整数 $n,m$。

接下来 $n$ 行 $m$ 个字符，第 $i$ 行第 $j$ 个字符表示 $b_{i,j}$ 的值。

## 输出格式

仅一行一个整数，表示选择方法的种数。

```input1
3 4
1123
1322
1221
```

```output1
1
```

```input2
2 2
11
22
```

```output2
0
```

```input3
5 3
111
222
333
123
132
```

```output3
2
```

## 数据范围与提示

对于全部数据，$1\le m\le 12$，$1\le n\le 3^m$，$b_i$ 互不相同，$1\le b_{i,j}\le 3$。

| Subtask |  特殊限制  | 分数 |
| :-----: | :--------: | :--: |
|   $1$   |  $m\le 5$  | $10$ |
|   $2$   |  $m\le 7$  | $30$ |
|   $3$   | 无特殊限制 | $70$ |
