## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc115/tasks/abc115_d

とある世界では、今日はクリスマスです。

高羽氏のパ`ティで、彼は多次元バ`ガ`を作ることにしました。*レベル $ L $ バ`ガ`* ($ L $ は $ 0 $ 以上の整数) とは次のようなものです。

- レベル $ 0 $ バ`ガ`とは、パティ $ 1 $ 枚である。
- レベル $ L $ バ`ガ` $ (L\ \geq\ 1) $ とは、バン $ 1 $ 枚、レベル $ L-1 $ バ`ガ`、パティ $ 1 $ 枚、レベル $ L-1 $ バ`ガ`、バン $ 1 $ 枚、をこのに下からeみ重ねたものである。

例えば、パティを `P`、バンを `B` で表すと、レベル $ 1 $ バ`ガ`は `BPPPB` (を $ 90 $ 度回したもの)、レベル $ 2 $ バ`ガ`は `BBPPPBPBPPPBB` といったた目になります。

高羽氏が作るのはレベル $ N $ バ`ガ`です。ダックスフンドのルンルンは、このバ`ガ`の下から $ X $ 婴蚴长伽蓼 (パティまたはバン $ 1 $ 枚を $ 1 $ 婴趣筏蓼)。ルンルンはパティを何枚食べるでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X $

## 输出格式
レベル $ N $ バ`ガ`の下から $ X $ 婴撕まれるパティの枚数を出力せよ。

## 题目大意
有这样一种字符串 $str_i$：

- 当 $i=0$ 时，$str_i=\verb!P!$。
- 当 $i>0$ 时，$str_i=\verb!B!+str_{i-1}+\verb!P!+str_{i-1}+\verb!B!$。

求字符串为 $str_n$ 的前 $x$ 个字符中有多少个为 $p$。

```input1
2 7
```

```output1
4
```

```input2
1 1
```

```output2
0
```

```input3
50 4321098765432109
```

```output3
2160549382716056
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 50 $
- $ 1\ \leq\ X\ \leq\ ( $ レベル $ N $ バ`ガ`の婴尉t数 $ ) $
- $ N,\ X $ は整数である。

### Sample Explanation 1

レベル $ 2 $ バ`ガ` (`BBPPPBPBPPPBB`) の下から $ 7 $ 婴摔膝靴匹￥ $ 4 $ 枚含まれます。

### Sample Explanation 2

レベル $ 1 $ バ`ガ`の一番下の婴膝啸螭扦埂

### Sample Explanation 3

レベル $ 50 $ バ`ガ`は婴问が $ 32 $ ビット整数にГ蓼椁胜こ潭趣朔趾瘠い扦埂
