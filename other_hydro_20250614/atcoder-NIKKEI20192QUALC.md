## 题目描述
[problemUrl]: https://atcoder.jp/contests/nikkei2019-2-qual/tasks/nikkei2019_2_qual_c

$ N $ 要素からなる $ 2 $ つの整数列 $ A_1,...,A_N $ 及び $ B_1,...,B_N $ が与えられます。 以下の操作を $ N-2 $ 回まで ($ 0 $ 回でもよい) 行うことで、$ 1 $ 以上 $ N $ 以下のすべての整数 $ i $ にして $ A_i\ \leqq\ B_i $ となるようにできるかを判定してください。

- $ 1 $ 以上 $ N $ 以下の相なる整数 $ x $, $ y $ をxび、$ A_x $ のと $ A_y $ のを入れ替える。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $ $ B_1 $ $ B_2 $ $ ... $ $ B_N $

## 输出格式
可能な龊悉 `Yes` を、不可能な龊悉 `No` を出力せよ。

## 题目大意
给出由N个要素构成的22个整数列A[1]，…，A[n]以及B[1]，…，B[n]。通过进行以下操作最多N-2次（也可以是0次），对于1以上N以下的全部整数i,A[i]Q B[i].

* 选择1及以上N及以下不同的整数x，y，并交换A[x]的值和A[y]的值。

如果可以输出yes不行输出no

```input1
3
1 3 2
1 2 3
```

```output1
Yes
```

```input2
3
1 2 3
2 2 2
```

```output2
No
```

```input3
6
3 1 2 6 3 4
2 2 8 3 4 3
```

```output3
Yes
```

## 提示
### 制s

- $ 2\ Q\ N\ Q\ 10^5 $
- $ 1\ Q\ A_i,B_i\ Q\ 10^9 $

### Sample Explanation 1

$ A_2 $ のと $ A_3 $ のを入れ替えればよいです。
