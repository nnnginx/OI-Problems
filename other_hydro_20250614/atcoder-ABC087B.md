## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc087/tasks/abc087_b

あなたは、$ 500 $ 矣瘠 $ A $ 枚、$ 100 $ 矣瘠 $ B $ 枚、$ 50 $ 矣瘠 $ C $ 枚持っています。 これらの硬の中から何枚かをxび、合金~をちょうど $ X $ 窑摔工敕椒à虾瓮à辘りますか。

同じNの硬どうしは区eできません。2 通りの硬のxび方は、あるNの硬についてその硬をxぶ枚数がなるとき区eされます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ X $

## 输出格式
硬をxぶ方法の数を出力せよ。

## 题目大意
你有500日元硬币A枚、100日元硬币B枚、50日元硬币 C枚。 现在，从这些硬币中挑出数枚硬币，各个硬币价格之和刚好等于X日元的组合有多少种。 需要注意的是：

同种类（面值）的硬币是没有区别的
不同的组合方式以某个面值的硬币数量不同来区分
0 ≤ A, B, C ≤ 50
A + B + C ≥ 1
50 ≤ X ≤ 20,000
A, B, C 一定是整数
X 一定是50的倍数
输入格式

A枚500日元硬币

B枚100日元硬币

C枚50日元硬币

组合价值之和等于X元

输出格式

组合的方式个数

感谢@linhongzhao321　提供的翻译

```input1
2
2
2
100
```

```output1
2
```

```input2
5
1
0
150
```

```output2
0
```

```input3
30
40
50
6000
```

```output3
213
```

## 提示
### 制s

- $ 0\ \leq\ A,\ B,\ C\ \leq\ 50 $
- $ A\ +\ B\ +\ C\ \geq\ 1 $
- $ 50\ \leq\ X\ \leq\ 20,000 $
- $ A,\ B,\ C $ は整数である
- $ X $ は $ 50 $ の倍数である

### Sample Explanation 1

条件を氦郡惯xび方は以下の $ 2 $ 通りです。 - $ 500 $ 矣瘠 $ 0 $ 枚、$ 100 $ 矣瘠 $ 1 $ 枚、$ 50 $ 矣瘠 $ 0 $ 枚xぶ。 - $ 500 $ 矣瘠 $ 0 $ 枚、$ 100 $ 矣瘠 $ 0 $ 枚、$ 50 $ 矣瘠 $ 2 $ 枚xぶ。

### Sample Explanation 2

合金~をちょうど $ X $ 窑摔工氡匾があることに注意してください。
