## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc211/tasks/abc211_a

Эs期血R $ A $ と期血R $ B $ が与えられます。  
 平均血R $ C $ を求めてください。  
 ただし、平均血Rは以下のように定xされるとします。

- $ C\ =\ \frac{A-B}{3}\ +B $

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
$ C $ を出力せよ。  
 出力と想定解答の~`差または相`差が $ 10^{-5} $ 以下であれば正解と判定される。

## 题目大意
# [ABC211A] Blood Pressure

## 题目描述

[problemUrl]: https://atcoder.jp/contests/abc211/tasks/abc211_a

已知收缩压A和舒张压B，求平均血压C的公式如下：

- $ C\ =\ \frac{A-B}{3}\ +B $

## 输入格式

输入A和B两个整数

> $ A $ $ B $

## 输出格式

输出$ C $的值 
输出值与正确答案的误差不大于$ 10^{-5} $
  

## 样例 #1

### 样例输入 #1

```
130 100
```

### 样例输出 #1

```
110
```

## 样例 #2

### 样例输入 #2

```

```

### 样例输出 #2

```
133.3333333
```

## 样例 #3

### 样例输入 #3

```
123 123
```

### 样例输出 #3

```
123
```

## 提示

### 制s

- $ 50\ \leq\ B\ \leq\ A\ \leq\ 300 $
输入数字为整数

### Sample Explanation 1

$ C\ =\ \frac{130-100}{3}\ +100\ =\ 10\ +\ 100\ =\ 110 $ 。

### Sample Explanation 2

输出值不一定是整数

