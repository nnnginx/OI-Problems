## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc296/tasks/abc296_f

Lさ $ N $ の数列 $ A=(A_1,A_2,\ldots,A_N) $, $ B=(B_1,B_2,\ldots,B_N) $ が与えられます。

高蚓は次の操作を好きなだけ ($ 0 $ 回でも良い) Rり返す事ができます。

> $ 1 $ 以上 $ N $ 以下の、どの $ 2 $ つも互いに相なる $ 3 $ つの整数 $ i,j,k $ をxぶ。  
>  $ A $ の $ i $ 番目の要素と $ j $ 番目の要素を交Qし、$ B $ の $ i $ 番目の要素と $ k $ 番目の要素を交Qする。

高蚓がうまく操作をRり返すことによって、 $ A $ と $ B $ を一致させる事が可能ならば `Yes` を、不可能ならば `No` を出力してください。  
 ただし、$ A $ と $ B $ が一致しているとは、任意の $ 1\leq\ i\leq\ N $ について $ A $ の $ i $ 番目の要素と $ B $ の $ i $ 番目の要素が等しいことを言います。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $ $ B_1 $ $ B_2 $ $ \ldots $ $ B_N $

## 输出格式
操作をRり返すことによって、高蚓が $ A $ と $ B $ を一致させる事が可能ならば `Yes` を、不可能ならば `No` を出力せよ。

## 题目大意
zjh 有两个长度为 $n$ 的序列 $a,b$。

他每次可以选择一组互不相同的 $(i,j,k)$，满足 $1\leqslant i,j,k\leqslant n$，然后交换 $a_i,a_j$，再交换 $b_i,b_k$。

输出是否有可能使得 $a=b$。

Translated by @[Zealous_YH](https://www.luogu.com.cn/user/399150)

```input1

1 1 2
```

```output1
Yes
```

```output2

```


## 提示
### 制s

- $ 3\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ 1\leq\ A_i,B_i\leq\ N $
- 入力はすべて整数
 
### Sample Explanation 1

$ (i,j,k)=(1,2,3) $ として $ 1 $ 回操作を行うことで、$ A_1 $ と $ A_2 $、$ B_1 $ と $ B_3 $ がそれぞれ交Qされ、 $ A,B $ はともに $ (2,1,1) $ となって一致します。よって、`Yes` を出力します。

### Sample Explanation 2

どのように操作を行っても $ A $ と $ B $ を一致させることはできません。よって、`No` を出力します。
