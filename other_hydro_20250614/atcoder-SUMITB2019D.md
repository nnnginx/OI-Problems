## 题目描述
[problemUrl]: https://atcoder.jp/contests/sumitrust2019/tasks/sumitb2019_d

AtCoder 社は、オフィスの入り口に $ 3 $ 桁の暗^番号をO定することにしました。

AtCoder 社には $ N $ 桁のラッキ`ナンバ` $ S $ があります。社Lの高蚓は、$ S $ から $ N-3 $ 桁を消して残りの $ 3 $ 桁を左からiんだものを暗^番号としてO定することにしました。

このとき、O定されうる暗^番号は何Nあるでしょうか？

ただし、ラッキ`ナンバ`や暗^番号はいずれも $ 0 $ から始まっても良いものとします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ S $

## 输出格式
O定されうる暗^番号のN数を出力してください。

## 题目大意
AtCoder Inc.已决定使用 $3$ 位PIN码锁定其办公室的门。

该公司有一个 $N$ 位数的幸运数字 $S$。 社长高桥将删除 $S$ 中的 $N-3$ 位数字，并将其余 $3$ 位数字连接起来，而无需更改 PIN 码的设置顺序。

他可以通过这种方式设置多少个 PIN 码？

幸运数字和 PIN 码都可以以 $0$ 开头。

```input1
4
0224
```

```output1
3
```

```input2
6
123123
```

```output2
17
```

```input3
19
3141592653589793238
```

```output3
329
```

## 提示
### 制s

- $ 4\ \leq\ N\ \leq\ 30000 $
- $ S $ は半角数字からなるLさ $ N $ の文字列

### Sample Explanation 1

高蚓には以下のxk肢があります。 - $ S $ の $ 1 $ 桁目を消し、暗^番号を `224` とする。 - $ S $ の $ 2 $ 桁目を消し、暗^番号を `024` とする。 - $ S $ の $ 3 $ 桁目を消し、暗^番号を `024` とする。 - $ S $ の $ 4 $ 桁目を消し、暗^番号を `022` とする。 よって、O定されうる暗^番号は `022`, `024`, `224` の $ 3 $ Nです。
