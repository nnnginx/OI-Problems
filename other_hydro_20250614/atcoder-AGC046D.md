`0` と `1` のみからなる文字列 $ S $ が与えられます。以下の操作を $ 0 $ 回以上任意の回数Rり返してできる可能性のある文字列の数を $ 998244353 $ で割った余りを求めてください。

- $ S $ の先^ $ 2 $ 文字を取り除き、そのうち片方を韦啤もう片方を $ S $ の任意の位置に啡毪工搿￥长尾僮鳏稀$ S $ が $ 2 $ 文字以上からなるときのみg行できる。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
操作を $ 0 $ 回以上任意の回数Rり返してできる可能性のある文字列の数を $ 998244353 $ で割った余りを出力せよ。

## 题目大意
给定 $01$ 串 $S$，每次操作位：选择首位两个字符，删除其中一个字符，把另一个字符插到任意位置。求最后能得到多少种不同的 $01$ 串。

其中 $1\le |S|\le 300$。

```input1
0001
```

```output1
8
```

```input2
110001
```

```output2
24
```

```input3
11101111011111000000000110000001111100011111000000001111111110000000111111111
```

```output3
697354558
```
