## 题目大意
有 $N$ 个人要回答 $X$ 和 $Y$ 两个问题。其中 $A$ 个人回答了 $X$ 问题，$B$ 个人回答了 $Y$ 问题。

求出两个问题都回答的人数的最大值和最小值。

#### 数据范围：
- $1\leq N \leq 100$
- $0\leq A,B \leq N$

```input1

```

```output1

```

```input2
10 7 5
```

```output2
5 2
```

```input3
100 100 100
```

```output3

```
