## 题目描述
[problemUrl]: https://atcoder.jp/contests/cf16-exhibition-final/tasks/cf16_exhibition_final_i

以下の条件を氦郡 $ N $ 角形を一つ成してください:

- 多角形はgである。 (ノ`トをてください)
- 多角形の各xは座溯Sに平行である。
- 多角形の点の座摔 $ 0 $ 以上 $ 10^9 $ 以下の整数である。
- 多角形の点は反r回りに $ 1 $ から $ N $ まで番号がつけられている。
- $ i $ 番目の点の内角は $ a_i $ 度である。

解が}数通り考えられる龊悉稀どれを出力してもかまいません。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_1 $ : $ a_N $

## 输出格式
解が存在する龊悉稀⒁韵陇涡问饯浅隽Δ护:

> $ x_1 $ $ y_1 $ : $ x_N $ $ y_N $

ここで $ (x_i,\ y_i) $ は $ i $ 番目の点の座摔扦る。

解が存在しない龊悉稀 `-1` と出力せよ。

## 题目大意
给定 $n(3\le n\le1000)$，和数列 $\{\alpha_n\}$，

试在平面直角坐标系上构造一个多边形，满足如下条件：
* 任意两顶点不重合，且横纵坐标都为 $0\sim10^9$ 内的整数。
* 每条边平行于 $x$ 轴或 $y$ 轴；除端点外，两两边不相交。
* **顺时针**方向上第 $k$ 个顶点对应的**内角**角度为 $\alpha_k^\circ$，其中 $\alpha_k\in\{90,270\}$。

注意，要求顶点沿**顺时针**顺序输出。

特别地，如果无可行解，请输出 `-1`。

```input1
8
90
90
270
90
90
90
270
90
```

```output1
0 0
2 0
2 1
3 1
3 2
1 2
1 1
0 1
```

```input2
3
90
90
90
```

```output2
-1
```

## 提示
### ノ`ト

全てのxが正のLさを持ち、どの二つのxも (O接するxが点で接するのを除いて) 共通部分を持たないとき、多角形はgであるという。

### 制s

- $ 3\ <\ =\ N\ <\ =\ 1000 $
- $ a_i $ は $ 90 $ または $ 270 $
