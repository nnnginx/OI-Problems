## 题目描述
[problemUrl]: https://atcoder.jp/contests/tenka1-2019-beginner/tasks/tenka1_2019_b

英小文字からなるLさ $ N $ の文字列 $ S $ と整数 $ K $ が与えられます。 $ S $ の $ K $ 番目の文字となる文字全てを `*` で置きQえてできる文字列を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $ $ K $

## 输出格式
$ S $ の $ K $ 番目の文字となる文字全てを `*` で置きQえてできる文字列を出力せよ。

## 题目大意
给定长度为 N 的字符串 S，该字符串由小写字母和整数K组成。 输出一个字符串，该字符串可以通过将所有与 S 的第 K 个字符不同的字符替换为 `*` 获得。

```input1
5
error
2
```

```output1
*rr*r
```

```input2
6
eleven
5
```

```output2
e*e*e*
```

```input3
9
education
7
```

```output3
******i**
```

## 提示
### 制s

- $ 1\ \leq\ K\ \leq\ N\leq\ 10 $
- $ S $ は英小文字からなるLさ $ N $ の文字列である
- $ N,K $ は整数である

### Sample Explanation 1

$ S $ の $ 2 $ 番目の文字は `r` であり、`error` の `r` となる文字をすべて `\*` で置きQえると文字列 `\*rr\*r` になります。
