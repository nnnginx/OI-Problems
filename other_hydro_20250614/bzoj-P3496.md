## 题目描述

平面上有 $n$ 个点。重复 $k$ 次以下操作：随机选择一个点，并以这个点为圆心作半径为 rr 的圆。求最后所有圆面积并的期望。

## 输入格式

第一行三个整数 $n$,$k$,$r$.

接下来 $n$ 行，每行两个整数 $x$,$y$ 为一个点

## 输出格式

输出面积并的期望。误差在 $10^{-2}$ 以内算对。

## 样例输入

```
2 2 1
0 0
1 0
```

## 样例输出

```
4.098204131080311
```

## 数据范围

对于全部的测试点, 保证 $1 \leq n,k \leq 10^3$, $1 \leq r \leq 10^3$, $|x|, |y| \leq 10^3$.
