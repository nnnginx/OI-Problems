## 题目描述

**译自 [JOI Open 2014](https://contests.ioi-jp.org/open-2014/index.html) Day1 T2「[t和占い](https://s3-ap-northeast-1.amazonaws.com/data.cms.ioi-jp.org/open-2014/2014-open-d1-fortune_telling2.pdf) / [Fortune Telling 2](https://s3-ap-northeast-1.amazonaws.com/data.cms.ioi-jp.org/open-2014/2014-open-d1-fortune_telling2-en.pdf)」**

K 教授是国际信息学奥林匹克竞赛日本委员会的主席。他喜欢算命。他总是做各种各样的算命。今天，他决定用牌来算命，看看今年 IOI 的日本代表团的成绩。

每张牌的两面都写着一个整数。一张牌的两面的整数不一定相等。如果你把一张牌放在桌子上，你只能看到朝上那面的整数，无法看不到背面的整数。

算命的过程如下：

- 首先，K 教授会把 $N$ 张牌放在桌子上。这些牌从 $1$ 到 $N$ 编号。牌 $i$ 的一面写着整数 $A_{i}$，另一面写着整数 $B_{i}$。他会让每张牌 $i$ 的 $A_{i}$ 那面朝上。
- 有 $K$ 个操作，对于每个操作 $j=1, \ldots, K$，如果每张牌朝上那面的整数小于或等于 $T_{j}$，K 教授就会把它翻过来。
- 这 $K$ 个操作结束后，算命的结果就是桌子上的牌朝上那面的整数的和。

在某个时刻，K 教授意识到决定哪些牌要翻过来是一件无聊的事情。他最终放弃了用牌来算命。他只想知道这 $K$ 个操作结束后，桌子上的牌朝上那面的整数的和。

编写一个程序，给定牌上写的整数和操作的信息，计算所有操作结束后，桌子上的牌朝上那面的整数的和。

## 输入格式

第一行包含两个用空格分隔的整数 $N$ 和 $K$，表示有 $N$ 张牌，操作的次数是 $K$。

接下来的 $N$ 行中的第 $i\ (1 \leq i \leq N)$ 行包含两个用空格分隔的整数 $A_{i}$ 和 $B_{i}$，表示牌 $i$ 上写的整数是 $A_{i}$ 和 $B_{i}$。

接下来的 $K$ 行中的第 $j\ (1 \leq j \leq K)$ 行包含一个整数 $T_{j}$，表示在第 $j$ 次操作中，如果一张牌朝上那面的整数小于或等于 $T_{j}$，它就会被翻过来。

## 输出格式

输出写入 $K$ 次操作结束后，桌子上的牌朝上那面的整数的和。

```input1
5 3
4 6
9 1
8 8
4 2
3 7
8
2
9
```

```output1
18
```



首先，桌子上的五张牌朝上那面的整数分别是 $4,9,8,4,3$。操作如下进行。

- 如果每张牌朝上那面的整数小于或等于 $8$，就把它翻过来。操作后，桌子上的牌朝上那面的整数分别是 $6,9,8,2,7$。
- 如果每张牌朝上那面的整数小于或等于 $2$，就把它翻过来。操作后，桌子上的牌朝上那面的整数分别是 $6,9,8,4,7$。
- 如果每张牌朝上那面的整数小于或等于 $9$，就把它翻过来。操作后，桌子上的牌朝上那面的整数分别是 $4,1,8,2,3$。

所有操作结束后，桌子上的牌朝上那面的整数的和是 $4+1+8+2+3=18$。

## 数据范围与提示

对于所有输入数据，满足：

- $1 \leq N \leq 2\times 10^5$
- $1 \leq K \leq 2\times 10^5$
- $1 \leq A_{i} \leq 10^9\ (1 \leq i \leq N)$
- $1 \leq B_{i} \leq 10^9\ (1 \leq i \leq N)$
- $1 \leq T_{j} \leq 10^9\ (1 \leq j \leq K)$

详细子任务附加限制及分值如下表所示。

| 子任务 | 附加限制  | 分值 |
| :--: | :--: | :--: |
|  $1$  | $N,K\leq 1000$ | $4$  |
|  $2$  | $N,K\leq 40000$ | $31$ |
|  $3$  | 无附加限制 | $65$ |
