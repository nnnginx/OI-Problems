## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc121/tasks/abc121_b

$ N $ のソ`スコ`ドがあり、$ i $ 目のソ`スコ`ドの特栅 $ A_{i1},\ A_{i2},\ ...,\ A_{iM} $ の $ M $ の整数で表されます。

また、整数 $ B_1,\ B_2,\ ...,\ B_M $ と 整数 $ C $ が与えられます。

$ A_{i1}\ B_1\ +\ A_{i2}\ B_2\ +\ ...\ +\ A_{iM}\ B_M\ +\ C\ >\ 0 $ のときに限り、$ i $ 目のソ`スコ`ドはこの}に正答するソ`スコ`ドです。

$ N $ のソ`スコ`ドのうち、この}に正答するソ`スコ`ドの数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ C $ $ B_1 $ $ B_2 $ $ ... $ $ B_M $ $ A_{11} $ $ A_{12} $ $ ... $ $ A_{1M} $ $ A_{21} $ $ A_{22} $ $ ... $ $ A_{2M} $ $ \vdots $ $ A_{N1} $ $ A_{N2} $ $ ... $ $ A_{NM} $

## 输出格式
$ N $ のソ`スコ`ドのうち、この}に正答するソ`スコ`ドの数を出力せよ。

## 题目大意
输入 $(n+2)$ 行整数。

第一行：三个整数 $n,m,c$ 。

第二行： $m$ 个整数 $b_1,b_2,...,b_m$ 。

第三行到第 $(n+2)$ 行：每行 $m$ 个整数 $a_{i1},a_{i2},...,a_{im}$ 。如果当前是全部输入中的第 $j$ 行，则一定有 $i=j-2$ 。

输出：一行一个整数，为能使 $a_{i1}b_1+a_{i2}b_2+...+a_{im}b_m+c>0$ 成立的 $i$ 的个数（ $1≤i≤n$ ）。

```input1
2 3 -10
1 2 3
3 2 1
1 2 2
```

```output1
1
```

```input2
5 2 -4
-2 5
100 41
100 40
-3 0
-6 -2
18 -13
```

```output2
2
```

```input3
3 3 0
100 -100 0
0 100 100
100 100 100
-100 100 100
```

```output3
0
```

## 提示
### 制s

- 入力は全て整数である。
- $ 1\ \leq\ N,\ M\ \leq\ 20 $
- $ -100\ \leq\ A_{ij}\ \leq\ 100 $
- $ -100\ \leq\ B_i\ \leq\ 100 $
- $ -100\ \leq\ C\ \leq\ 100 $

### Sample Explanation 1

以下のように $ 2 $ 目のソ`スコ`ドのみがこの}に正答します。 - $ 3\ \times\ 1\ +\ 2\ \times\ 2\ +\ 1\ \times\ 3\ +\ (-10)\ =\ 0\ \leq\ 0 $ なので $ 1 $ 目のソ`スコ`ドはこの}に正答しません。 - $ 1\ \times\ 1\ +\ 2\ \times\ 2\ +\ 2\ \times\ 3\ +\ (-10)\ =\ 1\ >\ 0 $ なので $ 2 $ 目のソ`スコ`ドはこの}に正答します。

### Sample Explanation 3

全て \*Wrong Answer\* です。あなたのソ`スコ`ドは含めません。
