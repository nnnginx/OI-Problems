## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc141/tasks/abc141_e

Lさ $ N $ の文字列 $ S $ が与えられます。

非空文字列であって、$ S $ のBAする部分文字列として重ならずに $ 2 $ 回以上Fれるもののうち、最LのもののLさを答えてください。

より密には、

- $ l_1\ +\ len\ \leq\ l_2 $
- $ S[l_1+i]\ =\ S[l_2+i]\ (i\ =\ 0,\ 1,\ ...,\ len\ -\ 1) $

を氦郡拐数 $ l_1 $ , $ l_2 $ ( $ 1\ \leq\ l_1,\ l_2\ \leq\ N\ -\ len\ +\ 1 $ ) が存在するような正整数 $ len $ の最大を求めてください。そのような $ len $ が存在しないときは、$ 0 $ を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
非空文字列であって、$ S $ のBAする部分文字列として重ならずに $ 2 $ 回以上Fれるもののうち、最LのもののLさを出力せよ。そのような非空文字列が存在しないときは、$ 0 $ を出力せよ。

## 题目大意
给你一个字符串，请找到两个**互相不重叠**且**完全相同**的子串，并输出它的最大长度。

```input1
5
ababa
```

```output1
2
```

```input2
2
xy
```

```output2
0
```

```input3
13
strangeorange
```

```output3
5
```

## 提示
### 制s

- $ 2\ <\ =\ N\ <\ =\ 5\ \times\ 10^3 $
- $ |S|\ =\ N $
- $ S $ は英小文字から成る

### Sample Explanation 1

条件を氦郡刮淖至肖趣筏啤`a`, `b`, `ab`, `ba` が考えられます。これらのLさの最大 $ 2 $ が答えです。 `aba` は $ S $ のBAする部分文字列として $ 2 $ 度Fれますが、$ l_1\ +\ len\ \leq\ l_2 $ を氦郡工瑜Δ $ l_1 $ , $ l_2 $ が取れないことに注意してください。
