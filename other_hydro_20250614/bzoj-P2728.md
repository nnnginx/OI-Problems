


## 题目描述
## 输入格式
输入文件第一行是用空格隔开的四个正整数N，K，L和R，接下来的一行是 **N** 个非负整数 **A1,A2……AN** ，其含义如上所述。 ** 100%的数据满足K≤60且N≤1000,** 0<=Ai<=2^k-1,0<=L<=R<=10^18
## 输出格式
仅包含一个整数，表示[L,R]内可以被计算出的数的个数

```input1
3  3 1 4                        
3  4 5

```
```output1
4
```

## 提示
样例1中，(3 NAND 4) NADN (3 NAND 5) = 1，5 NAND 5 = 2，3和4直接可得。
