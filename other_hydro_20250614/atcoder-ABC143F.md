## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc143/tasks/abc143_f

高颏んは $ N $ 枚のカ`ドを持っています。 $ i $ 番目のカ`ドには整数 $ A_i $ がかれています。

高颏んは整数 $ K $ をxびます。そして、以下の操作を何度かRり返します。

- かれている整数が互いになるちょうど $ K $ 枚のカ`ドをxび、食べる（食べたカ`ドは消绀工耄

$ K\ =\ 1,2,\ \ldots,\ N $ のそれぞれにして、操作を行える最大の回数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $

## 输出格式
$ N $ の整数を出力せよ。 $ t\ (1\ \le\ t\ \le\ N) $ 目には、$ K=t $ のときの答えを出力せよ。

## 题目大意
有 $ n $ 个数字，每次从中取 $ k $ 个互不相同的数字，最多能取几次？请输出 $ k=1,2,3,……,n $时的答案。

```input1
3
2 1 2
```

```output1
3
1
0
```

```input2
5
1 2 3 4 5
```

```output2
5
2
1
1
1
```

```input3
4
1 3 3 3
```

```output3
4
1
0
0
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 3\ \times\ 10^5 $
- $ 1\ \le\ A_i\ \le\ N $
- 入力はすべて整数

### Sample Explanation 1

$ K\ =\ 1 $ のとき、操作を以下のように行うことができます。 - $ 1 $ 枚目のカ`ドをxび、食べる - $ 2 $ 枚目のカ`ドをxび、食べる - $ 3 $ 枚目のカ`ドをxび、食べる また、$ K\ =\ 2 $ のとき、操作を以下のように行うことができます。 - $ 1 $ 枚目のカ`ドと $ 2 $ 枚目のカ`ドをxび、食べる $ K\ =\ 3 $ のときは、操作を行うことができません。$ 1 $ 枚目のカ`ドと $ 3 $ 枚目のカ`ドを同rにxべないことに注意してください。
