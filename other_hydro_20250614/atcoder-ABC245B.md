## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc245/tasks/abc245_b

Lさ $ N $ の整数からなる数列 $ A=(A_1,\ldots,A_N) $ が与えられます。

$ A_1,\ldots,A_N $ に含まれない最小の非整数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ \ldots $ $ A_N $

## 输出格式
答えを出力せよ。

## 题目大意
给出 $n$ 个非负整数，请求出这 $n$ 个整数中不包含的最小的非负整数。

```input1
8
0 3 2 6 2 1 0 0
```

```output1
4
```

```input2
3
2000 2000 2000
```

```output2
0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2000 $
- $ 0\ \leq\ A_i\ \leq\ 2000 $
- 入力は全て整数である

### Sample Explanation 1

非整数は $ 0,1,2,3,4,\ldots $ とAきます。 $ 0,1,2,3 $ は $ A $ に含まれ、$ 4 $ は $ A $ に含まれないので、答えは $ 4 $ です。
