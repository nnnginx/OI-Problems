## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc015/tasks/agc015_a

すぬけ君は、整数を $ N $ 持っています。このうち最小のものは $ A $、最大のものは $ B $ です。すぬけ君が持っている整数のt和としてありうるは何通りあるでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $

## 输出格式
t和としてありうるの数を出力せよ。

## 题目大意
你有N个数，最小为A，最大为B，求这些数的总和的个数。

```input1
4 4 6
```

```output1
5
```

```input2
5 4 3
```

```output2
0
```

```input3
1 7 10
```

```output3
0
```

```input4
1 3 3
```

```output4
1
```

## 提示
### 制s

- $ 1\ Q\ N,A,B\ Q\ 10^9 $
- $ A,B $ は整数である

### Sample Explanation 1

$ 18=4+4+4+6,19=4+4+5+6,20=4+5+5+6,21=4+5+6+6,22=4+6+6+6 $ の $ 5 $ つのがt和として考えられます。
