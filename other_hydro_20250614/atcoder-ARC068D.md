## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc068/tasks/arc068_d

すぬけくんは $ N $ 枚のカ`ドとデック(I端キュ`)で[ぶことにしました。 カ`ドには $ 1,2,3...,N $ の数がかれており、デックははじめ空です。

すぬけくんは $ 1,2,3,...,N $ がかれたカ`ドをこのに、それぞれデックの先^あるいは末尾に啡毪筏蓼埂 その後、すぬけくんはデックの先^あるいは末尾からカ`ドを取り出して食べる、という操作を $ N $ 回行います。

食べたカ`ドにかれていた数を食べた番にKべて数列を作ることにします。このようにして作ることが可能な数列のうち、$ K $ 番目の要素が $ 1 $ であるようなものの数を $ 10^{9}\ +\ 7 $ で割った余りを求めなさい。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
答えを出力せよ。

## 题目大意
将1-n顺序加入双端队列（每次可加头可加尾），再删除（每次可删头可删尾），求有多少种删除序列，使得1是第k个被删的。

```input1
2 1
```

```output1
1
```

```input2
17 2
```

```output2
262144
```

```input3
2000 1000
```

```output3
674286644
```

## 提示
### 制s

- $ 1\ Q\ K\ Q\ N\ Q\ 2{,}000 $

### Sample Explanation 1

条件を氦郡Kびは $ 1,2 $ の $ 1 $ 通りです。例えば以下のようにして、このKびを成することが可能です。 - $ 1,2 $ のどちらもカ`ドの山の一番下に啡毪工 - カ`ドの山の一番上からカ`ドを取り出して食べることを $ 2 $ 回行う
