## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc065/tasks/abc065_b

筋力をつけたい高蚓は、AtCoder 社のトレ`ニングO浃恰トレ`ニングをすることにしました。

AtCoder 社のトレ`ニングO浃摔 $ N $ のボタンがついており、ちょうど $ 1 $ のボタンが光っています。 ボタンには、$ 1 $ から $ N $ までの番号がついています。 ボタン $ i $ が光っているときにそのボタンを押すと、ボタン $ i $ の明かりが消え、その後ボタン $ a_i $ が光ります。$ i=a_i $ であることもあります。 光っていないボタンを押しても、何も起こりません。

最初、ボタン $ 1 $ が光っています。高蚓は、ボタン $ 2 $ が光っている状Bで、トレ`ニングをやめたいと思っています。

そのようなことは可能かどうか判定し、もし可能なら最低で何回ボタンを押す必要があるかを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_1 $ $ a_2 $ : $ a_N $

## 输出格式
ボタン $ 2 $ を光らせることが不可能な龊稀$ -1 $ を出力せよ。 そうでない龊稀ボタン $ 2 $ を光らせるために必要なボタンを押す回数の最小を出力せよ。

## 题目大意
有一堆按钮，按钮的编号是从 $1$ 到 $n$ 。第 $i$ 个按钮对应第 $i$ 个灯。

你按下第 $i$ 个按钮时，如果你按下的按钮对应的灯是关闭的,那么灯 $a_i$ 会开启，灯 $i$ 会关闭； 如果按钮对应的灯是开启的,那么按下它就什么都不会发生。

最初，灯 $1$ 是关闭的，其他的灯都开着。 高桥君希望最后灯 $2$ 是关闭的。

输出他最少按下按钮的次数，如果不能则输出 $-1$ 。

```input1
3
3
1
2
```

```output1
2
```

```input2
4
3
4
1
2
```

```output2
-1
```

```input3
5
3
3
4
2
4
```

```output3
3
```

## 提示
### 制s

- $ 2\ Q\ N\ Q\ 10^5 $
- $ 1\ Q\ a_i\ Q\ N $

### Sample Explanation 1

ボタン $ 1,3 $ のに押せばよいです。

### Sample Explanation 2

ボタン $ 1 $ を押すとボタン $ 3 $ 、ボタン $ 3 $ を押すとボタン $ 1 $ が光るので、ボタン $ 2 $ が光ることはありません。
