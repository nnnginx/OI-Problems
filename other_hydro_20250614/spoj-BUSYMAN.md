<h3>Input Specifications</h3>
<p>The first line consists of an integer <b>T</b>, the number of test cases. For each test case the first line consists of an integer <b>N</b>, the number of activities. Then the next <b>N</b> lines contains two integers <b>m</b> and <b>n</b>, the start and end time of each activity.</p>

<h3>Output Specifications</h3>
<p>For each test case find the maximum number of activities that you can do.</p>

<h3>Input Constraints</h3>
<p>
1 &lt;= T &lt;=10<br>
1 &lt;= N &lt;= 100000<br>
0 &lt;= start &lt; end &lt;= 1000000
</p>
