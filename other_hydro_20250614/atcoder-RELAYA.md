## 题目描述
[problemUrl]: https://atcoder.jp/contests/cf16-relay-open/tasks/relay_a

莼芈筏摔いて、抵抗が $ R_1 $ の抵抗と抵抗が $ R_2 $ の抵抗をK列につないだとき、合成抵抗 $ R_3 $ は以下のvS式から求めることが出来ます。

- $ \frac{1}{R_1}\ +\ \frac{1}{R_2}\ =\ \frac{1}{R_3} $

$ R_1 $ と $ R_2 $ が与えられるので $ R_3 $ を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ R_1 $ $ R_2 $

## 输出格式
$ R_3 $ のを出力せよ。

~`差または相`差が $ 10^{-6} $ 以下ならば正解となる。

## 题目大意
  $\frac{1}{R_1}+\frac{1}{R_2}=\frac{1}{R_3}$  

输入R1和R2，根据上面的式子求R3

保留10位小数，误差不能超过 $10^{-6}$ 

感谢@RioBlu 提供的翻译

```input1
2 3
```

```output1
1.2000000000
```

```input2
100 99
```

```output2
49.7487437186
```

## 提示
### 制s

- $ 1\ \leq\ R_1,\ R_2\ \leq\ 100 $
- $ R_1,\ R_2 $ は整数
