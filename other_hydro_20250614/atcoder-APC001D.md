## 题目描述
[problemUrl]: https://atcoder.jp/contests/apc001/tasks/apc001_d

$ N $ 点 $ M $ xの森が与えられます。点には $ 0 $ から $ N-1 $ の番号がついています。 xは $ (x_i,y_i) $ の形で与えられます。これは点 $ x_i $ と $ y_i $ がxでつながっていることを意味します。

各点 $ i $ には $ a_i $ というが定まっています。 あなたは与えられた森にxを追加してBYにしたいです。 xを追加するときには、まずなる点二つをxkし( $ i $ , $ j $ とする)、 $ i $ と $ j $ のgにxをります。 このrコストが $ a_i+a_j $ かかります。そしてこれ以降，点 $ i $ と $ j $ は永hにxkできなくなります。

森をBYにする最小コストを求めてください。 BYにするのが不可能な龊悉`Impossible`と出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ a_0 $ $ a_1 $ $ .. $ $ a_{N-1} $ $ x_1 $ $ y_1 $ $ x_2 $ $ y_2 $ $ : $ $ x_M $ $ y_M $

## 输出格式
森をBYにする最小コストを出力せよ。ただし、不可能な龊悉`Impossible`と出力せよ。

## 题目大意
给你一个有$N$个顶点和$M$个边的森林，每个点有一个值$a_i$。在给定的森林中添加边$(i,j)$，森林变得连通，此操作花费$a_i+ a_j$美元，并且之后不能再选择点$i$和$j$。

第一行输入$N$，$M$；第二行输入$N$个数，表示$a_i$；然后依次输入$M$对数，表示这两个点联通。

找到连接森林所需的最低总成本，否则输出"Impossible"。

```input1
7 5
1 2 3 4 5 6 7
3 0
4 0
1 2
1 3
5 6
```

```output1
7
```

```input2
5 0
3 1 4 1 5
```

```output2
Impossible
```

```input3
1 0
5
```

```output3
0
```

## 提示
### 制s

- $ 1\ <\ =\ N\ <\ =\ 100,000 $
- $ 0\ <\ =\ M\ <\ =\ N-1 $
- $ 1\ <\ =\ a_i\ <\ =\ 10^9 $
- $ 0\ <\ =\ x_i,y_i\ <\ =\ N-1 $
- 与えられるグラフは森
- 入力は全て整数

### Sample Explanation 1

点 $ 0 $, $ 5 $ をつなぐとグラフがBYになり，このrかかるコストは $ 1\ +\ 6\ =\ 7 $ です。

### Sample Explanation 2

どのようにxを追加してもグラフをBYにすることはできません。

### Sample Explanation 3

最初からグラフはBYであるので，xを追加する必要はありません。
