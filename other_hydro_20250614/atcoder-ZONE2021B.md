## 题目描述
[problemUrl]: https://atcoder.jp/contests/zone2021/tasks/zone2021_b

あなたは今、高さ $ 1000 $ の非常に高いタワ`の下にいます。タワ`から距x $ D $ xれた位置の上空 $ H $ の高さに UFO がおり（入出力例 1 の恧虿握栅筏皮ださい）、あなたは UFO に波を届けたいです。  
 タワ`と UFO のgには遮蔽物が $ N $ あります。$ i $ 番目の遮蔽物はタワ`から UFO の方向に向かって距x $ d_i $ の鏊に位置していて、高さは $ h_i $ です。  
 あなたはタワ`を上って、あなたと UFO のgの直上に遮蔽物が $ 1 $ つもoい状Bにしたいです。上る必要のある最低の高さを求めてください。  
 なお、地面は凹凸のない水平面であり、タワ`及び遮蔽物は地面と垂直に建っているものとします。  
 また、あなたと UFO のgの直上にちょうど遮蔽物の上端があるとき、その遮蔽物には遮蔽されていないものとします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ D $ $ H $ $ d_1 $ $ h_1 $ $ d_2 $ $ h_2 $ $ \vdots $ $ d_N $ $ h_N $

## 输出格式
答えを出力せよ。  
 想定解答との~`差または相`差が $ 10^{-3} $ 以下であれば正解と判定される。

## 题目大意
你现在在一座高达1000层的非常高的塔楼下面。在距离塔 $D$ 远的上空的 $H$ 位有UFO（请参照输入输出1的图）

塔和飞碟之间有 $N$ 个屏蔽物。从塔向UFO方向看第i个屏蔽物位于距离 $d_i$ 处，其高度为$h_i$。

你想登上塔楼，使你和UFO之间的直线上没有一个遮蔽物。请求出需要上升的最低高度。

地面是水平的，塔和地面呈垂直，若你和UFO中间的直线正好擦过遮蔽物时，它不会遮蔽你。

```input1
1 10 10
3 5
```

```output1
2.857142857142857
```

```input2

3 2
```

```output2
0.0
```

```input3

228 59
529 310

78 266
659 391
```

```output3
245.3080684596577
```

## 提示
### スト`リ`

まずは友好の印として、UFO の操kプログラムをクラッシュさせてみよう。俺は机の上の ZONe 缶に巫挨筏 USB から、地球最のコンピュ`タウイルス「KARATE」を取り出した。  
 こいつを UFO に送りつけてやる。UFO がどんなシステムを使っているかは分からないが、あらゆるシステムをクラッシュさせてきた KARATE は、きっと郡はずだ。

### 制s

- 入力は全て整数
- $ 1\ <\ =\ N\ <\ =\ 100 $
- $ 1\ <\ =\ d_i\ <\ D\ <\ =\ 1000 $
- $ 1\ <\ =\ h_i\ <\ H\ <\ =\ 1000 $
