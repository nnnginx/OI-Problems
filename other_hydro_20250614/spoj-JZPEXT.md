<p>We call one integer beautiful, if and only if it is divisible by each of its non-zero digits. Given an interval [l, r], calculate how many beautiful numbers n satisfy l&lt;=n&lt;=r.</p>
<h3>Input</h3>
<div id="_mcePaste" style="position: absolute; left: -10000px; top: 0px; width: 1px; height: 1px; overflow-x: hidden; overflow-y: hidden;">Then t lines follow, each line two numbers l, r, representing the query interval [l, r]</div>
<p>First line, the number of test cases, t.</p>
<p>Then t lines follow, each line two numbers l, r, representing the query interval [l, r].</p>
<p>1&lt;=t&lt;=2.5*10^4</p>
<p>1&lt;=l&lt;=r&lt;=10^18</p>
<h3>Output</h3>
<p>t lines. The t-th line is the answer to the t-th query.</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
1
1 100</pre>
<pre><strong>Output:</strong>
33<span style="white-space: normal;">
</span></pre>
