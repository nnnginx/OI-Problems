## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc278/tasks/abc278_a

Lさ $ N $ の数列 $ A\ =\ (A_1,\ A_2,\ \dots,\ A_N) $ が与えられます。  
あなたは次の操作をちょうど $ K $ 回行います。

- $ A $ の先^の要素を取り除き、$ A $ の末尾に $ 0 $ を啡毪工搿

操作を行った後の $ A $ の要素をすべて出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ A_1 $ $ A_2 $ $ \dots $ $ A_N $

## 输出格式
操作を行った後の $ A $ の要素を空白区切りで $ 1 $ 行に出力せよ。

## 题目大意
给定一个有 $n$ 个整数的数组 $a$，要求进行以下 $k$ 次操作，输出操作后的数组。

操作为：将第一个数去掉，在队尾加上一个 $0$。

translate by [PineappleSummer](https://www.luogu.com.cn/user/880187)

```input1
3 2
2 7 8
```

```output1
8 0 0
```

```input2
3 4
9 9 9
```

```output2
0 0 0
```

```input3
9 5
1 2 3 4 5 6 7 8 9
```

```output3
6 7 8 9 0 0 0 0 0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ K\ \leq\ 100 $
- $ 1\ \leq\ A_i\ \leq\ 100 $
- 入力されるはすべて整数

### Sample Explanation 1

操作を行う前は $ A\ =\ (2,\ 7,\ 8) $ です。 操作を $ 1 $ 回行ったr点では $ A\ =\ (7,\ 8,\ 0) $ です。 操作を $ 2 $ 回行ったr点では $ A\ =\ (8,\ 0,\ 0) $ です。 よって $ (8,\ 0,\ 0) $ が答えです。
