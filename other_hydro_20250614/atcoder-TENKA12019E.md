## 题目描述
[problemUrl]: https://atcoder.jp/contests/tenka1-2019/tasks/tenka1_2019_e

$ N $ 次の整数S数多式 $ f(x)=a_Nx^N+a_{N-1}x^{N-1}+...+a_0 $ が与えられます。任意の整数 $ x $ にして $ p $ が $ f(x) $ を割り切るような素数 $ p $ をすべて求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_N $ $ : $ $ a_0 $

## 输出格式
任意の整数 $ x $ にして $ p $ が $ f(x) $ を割り切るような素数 $ p $ を小さいにすべて出力せよ。

## 题目大意
给定 $n$ 次多项式 $A=a_0+a_1x+a_2x^2+\cdots+a_nx^n$，问有多少个质数 $p$ 满足 $A$ 在任意整数处的点值都是 $p$ 的倍数，并升序输出。

多项式系数从 $n$ 次到 $0$ 次给定。

```input1
2
7
-7
14
```

```output1
2
7
```

```input2
3
1
4
1
5
```

```output2

```

```input3
0
998244353
```

```output3
998244353
```

## 提示
### 制s

- $ 0\ \leq\ N\ \leq\ 10^4 $
- $ |a_i|\ \leq\ 10^9(0\leq\ i\leq\ N) $
- $ a_N\ \neq\ 0 $
- 入力はすべて整数である

### Sample Explanation 1

$ 2,7 $ は例えば、$ f(1)=14 $ や $ f(2)=28 $ を割り切ります。

### Sample Explanation 2

条件を氦郡顾厥がない龊悉猡ります。
