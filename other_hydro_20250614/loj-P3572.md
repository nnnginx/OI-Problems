## 题目描述

**译自 [COCI 2021/2022 Contest #3](https://hsin.hr/coci/archive/2021_2022/contest3_tasks.pdf) T4「Ekoeko」**

给定一个长度为 $2n$ 的仅由小写字母组成的字符串 $S$，试只交换相邻的两个字母，使得 $S[1\ldots n]=S[n+1\ldots2n]$，输出最小的交换次数。

## 输入格式

第一行为一个整数 $n$。

接下来一行一个长为 $2n$ 的字符串 $S$。

## 输出格式

输出一行一个整数表示最小的交换次数。

```input1
3
koeeok

```

```output1
3

```

```input2
3
kekoeo

```

```output2
1

```

```input3
4
soolnlsn

```

```output3
4

```

## 数据范围与提示

对于全部数据，有 $1\le n\le 10^5$，$S$ 仅有小写字母组成。

| Subtask 编号 | 分值 |                      特殊限制                       |
| :----------: | :--: | :-------------------------------------------------: |
|     $1$      | $10$ | 字符串 $S$ 由 $n$ 个 `a` 和 $n$ 个 `b` 顺次连接而成 |
|     $2$      | $20$ |                每个字母最多出现两次                 |
|     $3$      | $20 $ |      前 $n$ 个字母可以与后 $n$ 个字母两两匹配，但顺序可能不同       |
|     $4$      | $20$ |                     $n\le 1000$                     |
|     $5$      | $40$ |                     无特殊限制                      |
