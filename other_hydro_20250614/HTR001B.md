## 题目背景

蒙提霍尔问题（$\text{Monty Hall problem}$），又称三门问题，最初来源于美国的一档电视游戏类节目――$\text{Let’s Make a Deal}$，因其主持人为 $\text{Monty Hall}$ 而得名。问题可以描述如下：在电视游戏节目中，主持人设置了三道门，分别为 $1$ 号、$2$ 号、$3$ 号，其中一扇门后面有一辆小轿车作为奖品，另外两扇门后各为一只山羊。竞猜者可以随机挑选一扇门，比如，选择 $1$ 号门，之后主持人在 $2$ 号门和 $3$ 号门中选择一扇藏有山羊的门打开，比如说 $3$ 号门，然后让你作出选择：你是否愿意放弃选择 $1$ 号门转而选择 $2$ 号门？

更换门的选择能够使得中奖率更高吗？

从直觉来看，有两种主要的意见：第一种意见认为剩下的两扇门必定一扇门是山羊，另外一扇门是小轿车，换选后中奖概率是 $\dfrac{1}{2}$，而不换选的中奖概率是 $\dfrac{1}{3}$；第二种意见认为只剩下一扇尚未选择的门，而门后面有小轿车的概率同样是 $\dfrac{1}{3}$，所以换选和不换选的中奖概率不变。但是通过谨慎的理论分析和计算机数据模拟都可以得出以下正确的结论：不换选的中奖概率为 $\dfrac{1}{3}$，换选的中奖概率为 $\dfrac{2}{3}$。这个结论非常反直觉，使得当时的很多读者（其中 $\dfrac{1}{10}$ 的人具有博士学位）写信给刊登这个结论的杂志，表示他（她）们拒绝相信“换选具有 $\dfrac{2}{3}$ 的中奖概率”是正确的结论。

实际上转换一下思维方式就很容易理解这个问题，从而明白正确结论就是换选门具有更高的中奖概率。假设竞猜者选择的是 $1$ 号门，则有以下可能性：

| $1$ 号门  |$2$ 号门 |	$3$ 号门 |不换选 |换选 |
| :---: | :---: |:---: |:---: |:---: |
|小轿车	|山羊	|山羊	|小轿车	|山羊|
|山羊	|小轿车	|山羊	|山羊	|小轿车|
|山羊	|山羊	|小轿车	|山羊	|小轿车|

容易看出，不换选中奖的概率为 $\dfrac{1}{3}$，换选后中奖概率为 $\dfrac{2}{3}$。如果竞猜者初始选择 $2$ 号门或者 $3$ 号门具有同样的结果。

## 题目描述

现在我们来研究蒙提霍尔问题的一个扩展。某位竞猜者参加蒙提霍尔主持的抽奖节目，主持人设置了 $n$ 扇门，并在 $a$ 扇门后设有奖品，剩余的 $n - a$ 扇门后未设置奖品。当竞猜者随机选中一扇门后，主持人打开另外 $b$ 扇门，打开的 $b$ 扇门中有 $c$ 扇门后有奖品，此时主持人询问竞猜者是否将当前选中的门换成另外一扇尚未打开的门。由于竞猜者对概率论一窍不通，无法确定换选门后的中奖概率究竟是多少，因此竞猜者将这个问题交给你来处理。

不难证明，可以将换选门后的中奖概率表示为一个最简分数 $\dfrac{p}{q}$，其中 $p$ 和 $q$ 互质。试确定一个最小的自然数 $x$，使得 $p \equiv qx \pmod P$，其中 $P=998244353$，可以证明这样的自然数 $x$ 必定存在。

## 输入格式
**输入包含多组测试数据。**

输入的第一行包含一个整数 $T$，表示测试数据的组数。接下来每行包含一组测试数据，一组测试数据包含四个整数：$n$，$a$，$b$，$c$。

## 输出格式
对于每组测试数据输出一行，包含一个符合题意要求的自然数 $x$。

## 输入输出样例

```input1
2
3 1 1 0
10 3 6 2
```

```output1
665496236
33274812
```

## 说明/提示
**样例说明**

对于样例 #1 中的第一组数据，共有 $3$ 扇门，在 $1$ 扇门后设有奖品，竞猜者选中一扇门后，主持人打开另外 $1$ 扇门，另外打开的 $1$ 扇门中有 $0$ 扇门包含奖品，此种情形在题目背景中已经说明，换选门后的中奖概率为 $\dfrac{2}{3}$，故 $p=2$ 且 $q=3$，则 $x=665496236$。

------------

**数据规模**

- 对于 $10\%$ 的数据，$3 \le n \le 10$；
- 对于 $30\%$ 的数据，$3 \le n \le 10^3$；
- 对于 $100\%$ 的数据，$3 \le n \le 10^5$；

对于 $100\%$ 的数据，$1 \le T \le 100$，$1 \le a \le n-1$，$1 \le b \le n－2$，$0 \le c \lt \operatorname{min}(a, \ b)$。
