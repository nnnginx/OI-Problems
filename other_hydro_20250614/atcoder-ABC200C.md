## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc200/tasks/abc200_c

$ 200 $ という整数が大好きなりんごさんのために、次の}を解いてください。  
 $ N $ の正整数からなる数列 $ A $ が与えられるので、以下の条件をすべて氦郡拐数のM $ (i,j) $ の数を求めてください。

- $ 1\ \le\ i\ <\ j\ \le\ N $
- $ A_i\ -\ A_j $ は $ 200 $ の倍数である。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \dots $ $ A_N $

## 输出格式
答えを整数として出力せよ。

## 题目大意
对 $200$ 这个数情有独钟的苹果先生拜托你来解决这个问题。

给定一个长度为 $N$ 的正整数序列 $A$，请求出所有满足 $1 \leq i < j \leq N$ 且 $A_i-A_j$ 为 $200$ 的倍数的二元组 $(i, j)$ 的个数。

```input1
6
123 223 123 523 200 2000
```

```output1
4
```

```input2
5
1 2 3 4 5
```

```output2
0
```

```input3
8
199 100 200 400 300 500 600 200
```

```output3
9
```

## 提示
### 制s

- 入力は全て整数
- $ 2\ \le\ N\ \le\ 2\ \times\ 10^5 $
- $ 1\ \le\ A_i\ \le\ 10^9 $

### Sample Explanation 1

例えば、$ (i,\ j)\ =\ (1,\ 3) $ のとき、$ A_1\ -\ A_3\ =\ 0 $ は $ 200 $ の倍数です。 $ (i,j)=(1,3),(1,4),(3,4),(5,6) $ の $ 4 $ つが条件を氦郡筏蓼埂
