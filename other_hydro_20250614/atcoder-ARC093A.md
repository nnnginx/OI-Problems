## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc092/tasks/arc093_a

$ x $ S上に $ N $ のQ光スポットがあり、$ 1,\ 2,\ ...,\ N $ の番号がついています。 Q光スポット $ i $ は座 $ A_i $ の点にあります。 また、$ x $ S上を座 $ a $ の点から座 $ b $ の点まで移婴工毪摔 $ |a\ -\ b| $ 窑かります。

あなたは $ x $ S上を旅行する画を立てました。 画では、最初に座 $ 0 $ の点を出kし、$ N $ のQ光スポットを番号にLれ、最後に座 $ 0 $ の点にってくることになっています。

ところが、旅行の直前に急用が入り、$ N $ すべてのQ光スポットをLれるrg的余裕がなくなってしまいました。 そこで、ある $ i $ をxび、Q光スポット $ i $ をLれるのを取りやめることにしました。 それ以外のQ光スポットは予定通り番号にLれます。 また、最初に座 $ 0 $ の点を出kし、最後に座 $ 0 $ の点にってくることについても、予定に涓はありません。

$ i\ =\ 1,\ 2,\ ...,\ N $ それぞれについて、Q光スポット $ i $ をLれるのを取りやめたときの、旅行中の移婴摔かる金~のt和を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $

## 输出格式
$ N $ 行出力せよ。 このうち $ i $ 行目には、Q光スポット $ i $ をLれるのを取りやめたときの、旅行中の移婴摔かる金~のt和を出力せよ。

## 题目大意
数轴上有 $N$ 个点，一开始在 $0$ 位置上，需要去 $N$ 个点（顺序从 $0$ 号点**按顺序走完** $N$ 个点），最后回到 $0$ 位置。

对于 $i=1, 2, \ldots, N$ 分别输出不需要去 $i$ 号点的最小路程。

```input1
3 5 -1
```

```output1

10
```


```output2

4

4
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 10^5 $
- $ -5000\ \leq\ A_i\ \leq\ 5000 $ ($ 1\ \leq\ i\ \leq\ N $)
- 入力はすべて整数である。

### Sample Explanation 1

Q光スポット $ 1,\ 2,\ 3 $ はそれぞれ座 $ 3,\ 5,\ -1 $ の点にあります。 各 $ i $ について、Q光スポット $ i $ をLれるのを取りやめた龊悉我咏U路および移婴摔かる金~は以下のようになります。 - $ i\ =\ 1 $ のとき、移咏U路は $ 0\ \rightarrow\ 5\ \rightarrow\ -1\ \rightarrow\ 0 $ となり、移婴摔かる金~は $ 5\ +\ 6\ +\ 1\ =\ 12 $ 窑趣胜辘蓼埂 - $ i\ =\ 2 $ のとき、移咏U路は $ 0\ \rightarrow\ 3\ \rightarrow\ -1\ \rightarrow\ 0 $ となり、移婴摔かる金~は $ 3\ +\ 4\ +\ 1\ =\ 8 $ 窑趣胜辘蓼埂 - $ i\ =\ 3 $ のとき、移咏U路は $ 0\ \rightarrow\ 3\ \rightarrow\ 5\ \rightarrow\ 0 $ となり、移婴摔かる金~は $ 3\ +\ 2\ +\ 5\ =\ 10 $ 窑趣胜辘蓼埂
