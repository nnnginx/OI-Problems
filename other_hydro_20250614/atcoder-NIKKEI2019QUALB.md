## 题目描述
[problemUrl]: https://atcoder.jp/contests/nikkei2019-qual/tasks/nikkei2019_qual_b

三つの文字列 $ A,\ B,\ C $ が与えられます。これらはそれぞれ、英小文字からなるLさ $ N $ の文字列です。

私たちの目摔稀これら三つの文字列をすべて等しくすることです。そのために、あなたは次の操作をRり返し行うことができます。

- 操作: 文字列 $ A,\ B,\ C $ のうち一つをxび、さらに $ 1 $ 以上 $ N $ 以下の整数 $ i $ を指定する。そして、xんだ文字列の先^から $ i $ 文字目をeの何らかの英小文字に涓する。

目摔蜻_成するためには最小で何回の操作が必要でしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $ $ C $

## 输出格式
必要な最小の操作回数を出力せよ。

## 题目大意
给定 $3$ 个长度为 $n$ 的字符串 $a,b,c$，以第 $0$ 位到第 $n-1$ 位的顺序，每次修改一个当前位上的字符，问最少多少次可以使 $a=b=c$。

```input1
4
west
east
wait
```

```output1
3
```

```input2
9
different
different
different
```

```output2
0
```

```input3
7
zenkoku
touitsu
program
```

```output3
13
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 100 $
- $ A,\ B,\ C $ はそれぞれLさ $ N $ の文字列である。
- $ A,\ B,\ C $ の各文字は英小文字である。

### Sample Explanation 1

この例では、はじめ $ A\ = $ `west`、$ B\ = $ `east`、$ C\ = $ `wait` です。以下のように $ 3 $ 回の操作を行うことで、最小の操作回数で目摔蜻_成できます。 - $ A $ の $ 2 $ 文字目を `a` に涓する。$ A $ は `wast` となる。 - $ B $ の $ 1 $ 文字目を `w` に涓する。$ B $ は `wast` となる。 - $ C $ の $ 3 $ 文字目を `s` に涓する。$ C $ は `wast` となる。

### Sample Explanation 2

はじめから $ A,\ B,\ C $ がすべて等しい龊稀⒈匾な操作回数は $ 0 $ となります。
