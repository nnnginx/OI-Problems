## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc155/tasks/abc155_e

AtCoder 王国の通は $ 10^{100}+1 $ Nの扭韦撙扦り、はそれぞれ $ 1,\ 10,\ 10^2,\ 10^3,\ \dots,\ 10^{(10^{100})} $ です。あなたは商店街で、 $ N $ のたこき器を $ 1 $ つIおうとしています。

あなたは $ N $ 以上の金~をQめて支Bいます。その後、支B~よりちょうど $ N $ だけ少ない金~を、店Tがおりとして支Bいます。

あなたと店Tが使う扭谓M合せをm切にO定するとき、I者の使う扭蚊妒の合は最小で何枚になるでしょう？

なお、あなたも店Tも任意の扭蚴分多く持っているとします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
支Bう扭蚊妒とおりとして受け取る扭蚊妒の合の最小を出力せよ。

## 题目大意
## 题目描述
给定正整数 $N$，设 $f(x)$ 表示 $x$ 在十进制下各个数位上的数的和，求一个正整数 $x$ 满足 $x\ge N$ 且最小化 $f(x)+f(x-N)$。
## 数据范围
$1\le N\le10^{1000000}$。
## 输入输出格式
### 输入格式
一行一个正整数 $N$，含义如题所述。
### 输出格式
一行一个正整数 $ans$，表示最小的 $f(x)+f(x-N)$。

```input1
36
```

```output1
8
```

```input2
91
```

```output2
3
```

```input3
314159265358979323846264338327950288419716939937551058209749445923078164062862089986280348253421170
```

```output3
243
```

## 提示
### 制s

- $ N $ は $ 1 $ 以上 $ 10^{1,000,000} $ 以下の整数

### Sample Explanation 1

あなたが $ 10 $ の $ 4 $ 枚を支Bい、店Tが $ 1 $ の $ 4 $ 枚をおりに渡すと、使う扭蚊妒は合で $ 8 $ 枚になります。 $ 8 $ 枚より少ない合枚数を_成することはできないので、答えは $ 8 $ です。

### Sample Explanation 2

あなたが $ 100 $ の $ 1 $ 枚と $ 1 $ の $ 1 $ 枚を支Bい、店Tが $ 10 $ の $ 1 $ 枚をおりに渡すと、使う扭蚊妒は合で $ 3 $ 枚になります。
