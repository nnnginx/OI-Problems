## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc235/tasks/abc235_a

$ 3 $ つの数字 $ x,y,z $ をこのにKべてできる $ 3 $ 桁の整数を $ xyz $ と表すことにします。

どの桁も $ 0 $ でない $ 3 $ 桁の整数 $ abc $ が与えられるので、$ abc+bca+cab $ を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ abc $

## 输出格式
答えを出力せよ。

## 题目大意
给定一个三位数，它的百位是 $a$ ，十位是 $b$ ，个位是 $c$ ，可以表示为 $abc$（保证三位数的每一位都不为 $0$ ）。求出 $abc+bca+cab$ 的和。

```input1
123
```

```output1
666
```

```input2
999
```

```output2
2997
```

## 提示
### 制s

- $ abc $ は どの桁も $ 0 $ でない $ 3 $ 桁の整数

### Sample Explanation 1

$ 123+231+312=666 $ となります。

### Sample Explanation 2

$ 999+999+999=2997 $ となります。
