## 题目描述
求有多少不同的包含 $n$ 个点的有标号无根树，满足：对于任何一个点 $x$，都存在点 $y$ 使得 $x$ 和 $y$ 之间有一条边且 $|x - y| = 1$。答案对 $998244353$ 取模。

## 输入格式
一行一个正整数 $n$。

## 输出格式
一行一个整数，所求答案。

```input1
4
```

```output1
4
```

```input2
209
```

```output2
21754876
```

```input3
5
```

```output3
11
```

```input4
6
```

```output4
56
```

**【数据范围】**

本题包含 $20$ 个测试点，每个测试点 $5$ 分。

| 测试点编号 | $n$ 的范围  |
| :--------: | :---------: |
|  $1 \sim 2$  |  $\leq 7$   |
|  $3 \sim 4$  |  $\leq 14$  |
|  $5 \sim 8$  |  $\leq 30$  |
|  $9 \sim 12$ | $\leq 10^3$ |
| $13 \sim 20$ | $\leq 10^5$ |

对于所有测试点满足 $n$ 为正整数且 $2 \leq n \leq {10}^5$。
