## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc105/tasks/abc105_c

整数 $ N $ が与えられるので、$ N $ の $ -2 $ M数表Fを求めてください。

ここで、$ S $ が $ N $ の $ -2 $ M数表Fであるとは、以下を全て氦郡工长趣扦埂

- $ S $ は `0` および `1` のみからなる文字列である
- $ S\ = $ `0` でなければ $ S $ の先^の文字は `1` である
- $ S\ =\ S_k\ S_{k-1}\ ...\ S_0 $ とすると、$ S_0\ \times\ (-2)^0\ +\ S_1\ \times\ (-2)^1\ +\ ...\ +\ S_k\ \times\ (-2)^k\ =\ N $ が成り立つ

なお、任意の整数 $ M $ にして $ M $ の $ -2 $ M数表Fが一意に定まることが^明できます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ の $ -2 $ M数表Fを出力せよ。

## 题目大意
对于任意一个数 $a_1a_2......a_x$ 满足 $\sum_{i=1}^{x}a_i\times base^{x-i}=A$ ， 那么称 $a_1a_2......a_x$ 为 $A$ 的 $base$ 进制表示的数，其中 $0\le a_1,a_2......,a_x< |base|$ 。   

给出一个整数 $N$ ， 满足 $-10^{9}\le N \le 10^{9}$ ，请计算出其 $-2$ 进制表示的数。

```input1
-9
```

```output1
1011
```

```input2
123456789
```

```output2
11000101011001101110100010101
```

```input3
0
```

```output3
0
```

## 提示
### 制s

- 入力はすべて整数である
- $ -10^9\ \leq\ N\ \leq\ 10^9 $

### Sample Explanation 1

$ (-2)^0\ +\ (-2)^1\ +\ (-2)^3\ =\ 1\ +\ (-2)\ +\ (-8)\ =\ -9 $ なので `1011` は $ -9 $ の $ -2 $ M数表Fです。
