## 题目描述
[problemUrl]: https://atcoder.jp/contests/cf17-relay-open/tasks/relay2_h

o限のからなる数列 $ a_1, $ $ a_2, $ $ … $ を考えます。はじめ、すべてののは $ 0 $ であり、この状Bから $ Q $ 回の操作をAけて行います。$ i $ 回目の操作 $ (1\ <\ =\ i\ <\ =\ Q) $ は次の通りです。

- すべての正の整数 $ j $ にし、$ a_{j\ ×\ m_i} $ のに $ x_i $ を加える。

これらの $ Q $ 回の操作後の最も大きいのを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ Q $ $ m_1 $ $ x_1 $ $ : $ $ m_Q $ $ x_Q $

## 输出格式
$ Q $ 回の操作後の最も大きいのを出力せよ。

## 题目大意
一个初始值均为 $0$ 的有无限项的数组 $a$，进行 $Q$ 次操作，第 $i$ 次操作如下：

-  对于每个正整数 $j$， 将 $a_{j*m_i}$ 加上 $x_i$ 。

求 $Q$ 次操作后 $a$ 数组中的最大值。

```input1
3
2 10
3 -20
6 15
```

```output1
10
```

```input2
3
10 -3
50 4
100 -5
```

```output2
1
```

```input3
5
56 114834
72 -149861
100 190757
192 -132693
240 133108
```

```output3
438699
```

## 提示
### 制s

- $ 1\ <\ =\ Q\ <\ =\ 299 $
- $ 2\ <\ =\ m_i\ <\ =\ 300 $
- $ -10^6\ <\ =\ x_i\ <\ =\ 10^6 $
- $ m_i $ はすべてなる。
- 入力はすべて整数である。

### Sample Explanation 1

数列の各 $ a_1, $ $ a_2, $ $ … $ のは以下のように浠します。 - 操作前: $ 0, $ $ 0, $ $ 0, $ $ 0, $ $ 0, $ $ 0, $ $ … $ - $ 1 $ 回目の操作後: $ 0, $ $ 10, $ $ 0, $ $ 10, $ $ 0, $ $ 10, $ $ … $ - $ 2 $ 回目の操作後: $ 0, $ $ 10, $ $ -20, $ $ 10, $ $ 0, $ $ -10, $ $ … $ - $ 3 $ 回目の操作後: $ 0, $ $ 10, $ $ -20, $ $ 10, $ $ 0, $ $ 5, $ $ … $ すべての操作後の最も大きいのは $ 10 $ です。
