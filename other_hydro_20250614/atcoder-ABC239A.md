## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc239/tasks/abc239_a

地上 $ x $ メ`トルの高さからえる水平は $ \sqrt{x(12800000+x)} $ メ`トル先にあるとするとき、 地上 $ H $ メ`トルの高さからえる水平が何メ`トル先にあるか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ H $

## 输出格式
答えを出力せよ。  
 なお、想定解との~`差または相`差が $ 10^{-6} $ 以下であれば、正解としてQわれる。

## 题目大意
给定正整数 $ x $ ，请求出 $ \sqrt{x(12800000+x)} $ 的值。

```input1
333
```

```output1
65287.907678222
```

```input2
634
```

```output2
90086.635834623
```

## 提示
### 制s

- $ 1\ \leq\ H\ \leq\ 10^5 $
- $ H $ は整数である

### Sample Explanation 1

$ \sqrt{333(12800000+333)}\ =\ 65287.9076782\ldots $ です。`65287.91` などの出力でも正解となります。

### Sample Explanation 2

$ \sqrt{634(12800000+634)}\ =\ 90086.6358346\ldots $ です。
