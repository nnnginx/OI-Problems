## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc053/tasks/agc053_c

$ 2N $ 枚のカ`ドがあり、それぞれには $ 1 $ から $ 2N $ までの番号が付いています。 このカ`ドを用いて行う、次のゲ`ムを考えます。

まず、ディ`ラ`はそれぞれの山が $ N $ 枚のカ`ドからなるように、カ`ドを $ 2 $ つの山にランダムに分けます。 このとき、ディ`ラ`は各山におけるカ`ドの序もランダムに定めます。 その後プレイヤ`は、一方の山が空になるまで次の操作をRり返し行い、最K的な操作回数がこのゲ`ムのスコアとなります。

- ある正の整数 $ k $ をxび、一方の山の上から $ k $ 枚目のカ`ドと、もう一方の山の上から $ k $ 枚目のカ`ドを比^する。（ただし、$ k $ は各山のカ`ド枚数を超えてはいけない。）そして、番号が小さい方のカ`ドをそのカ`ドを含む山から取り除く。

このゲ`ムを *チ`タ`* がプレイするとします。 つまり、各山の各カ`ドの番号を常に把握できるプレイヤ`がプレイするとします。 このプレイヤ`がスコアを最小化するよう最mにプレイしたときの、スコアの期待を $ \bmod\ 10^9+7 $ で求めてください（注参照）。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
给定 $2n$ 张卡牌，顺次编号为 $1$ 至 $2n$。考虑如下的游戏：

首先，庄家随机地将卡牌分成两堆，每堆 $n$ 张。每堆内牌的顺序也是随机的。随后，玩家会重复以下操作直到其中一堆为空：   
选择一个正整数 $k$，比较两堆中从上到下第 $k$ 张卡牌（$k$ 必须不大于牌堆的大小）。随后，移除两张牌中编号更小的牌。   
操作次数即为玩家的得分。

假设玩家是一名**作弊者**，能看到两堆牌中每张牌的编号。玩家将采用最优策略最小化得分，请输出玩家的期望得分在模 $10^9 + 7$ 意义下的值。

本题中的期望得分是一个分数。假设我们将答案表示为最简分数 $\frac xy$，你需要输出的值即为满足 $yz \equiv x\pmod {10^9 + 7}$ 的值 $z$。

$n\le 10^6$。

```input1
1
```

```output1
1
```

```input2
3
```

```output2
486111118
```

## 提示
### 注

- 求める期待は有理数となります。期待を分数 $ \frac{y}{x} $（$ x $ と $ y $ は互いに素な正の整数）で表したとき、$ x $ は $ P=10^9+7 $ と互いに素になるので、 $ xz\ \equiv\ y\ \pmod\ P $ なる $ 0 $ 以上 $ P-1 $ 以下の唯一の整数 $ z $ を出力してください。

### 制s

- $ 1\ \leq\ N\ \leq\ 10^6 $
