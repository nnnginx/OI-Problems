## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc014/tasks/agc014_a

高蚓と青木君とすぬけ君はそれぞれクッキ`を $ A,B,C $ 持っています。

この $ 3 $ 人はお互いにクッキ`を交Qすることにしました。具体的には、以下の操作をRり返します。

- $ 3 $ 人は同rに、各々が持っているクッキ`を半分ずつに分けて、残りの $ 2 $ 人にそれぞれ一方を渡す。

ただし、lかの持っているクッキ`の数が奇数になったら、そこで操作をRり返すのをやめます。

さて、クッキ`の交Qは何回行うことができるでしょうか。 ただし、o限にAけられる龊悉猡ることに注意してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $

## 输出格式
$ 3 $ 人がクッキ`の交Qを行うことができる回数を出力せよ。ただし、o限にAけられる龊悉 `-1` を出力せよ。

## 题目大意
三个人各有曲奇 $A$，$B$，$C$ 个。

每一轮三个人**同时**将自己的曲奇分成相等的两份分别给剩下的两个人。

当有人的曲奇数为奇数时结束。

求问多少轮后结束，可以无限循环输出 `-1`。

```input1
4 12 20
```

```output1
3
```

```input2
14 14 14
```

```output2
-1
```

```input3
454 414 444
```

```output3
1
```

## 提示
### 制s

- $ 1\ Q\ A,B,C\ Q\ 10^9 $

### Sample Explanation 1

はじめ、高蚓と青木君とすぬけ君はそれぞれクッキ`を $ 4,12,20 $ 持っており、 - $ 1 $ 回目の操作後は、高蚓と青木君とすぬけ君はそれぞれクッキ`を $ 16,12,8 $ 持っている。 - $ 2 $ 回目の操作後は、高蚓と青木君とすぬけ君はそれぞれクッキ`を $ 10,12,14 $ 持っている。 - $ 3 $ 回目の操作後は、高蚓と青木君とすぬけ君はそれぞれクッキ`を $ 13,12,11 $ 持っている。 $ 3 $ 回目の操作後に高蚓とすぬけ君の持っているクッキ`の数が奇数になるので、求める回数は $ 3 $ 回となります。
