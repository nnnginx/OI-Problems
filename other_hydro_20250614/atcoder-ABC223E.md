正整数 $ X,\ Y $ にし、$ 2 $ 次元平面上において以下の条件を氦郡归L方形を**良いL方形**と呼びます。

- 全てのxは $ x $ Sまたは $ y $ SにK行である。
- 全ての点にし、その $ x $ 座摔 $ 0 $ 以上 $ X $ 以下の整数であり、その $ y $ 座摔 $ 0 $ 以上 $ Y $ 以下の整数である。

面eがそれぞれ $ A $ 以上、$ B $ 以上、$ C $ 以上であるような $ 3 $ つの良いL方形を重ならないように配置することができるか判定してください。

ただし、$ 3 $ つのL方形が重ならないとは、どの $ 2 $ つのL方形についても、それらの共通部分の面eが $ 0 $ であることを指します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ Y $ $ A $ $ B $ $ C $

## 输出格式
}文で与えられた条件を氦郡工瑜Δ碎L方形を配置することができるならば `Yes`、できないならば `No` と出力せよ。

## 题目大意
### 题目大意

问能否通过摆放平面中三个面积分别**至少**为 $A,B,C$ 的小矩形使得它们能被一个 $X\times Y$ 的矩形完全包含。

### 数据范围

+ $1\le X,Y \le 10^9$；
+ $1 \le A,B,C \le 10^{18}$。

### 样例 1 解释

下图是三个小矩形的一种摆放方法。

![](https://img.atcoder.jp/ghi/abc223e_sample.png)

其中 $2 \ge A$，$3\ge B$，$3 \ge C$，所以这种摆放方式是满足条件的。

```input1
3 3 2 2 3
```

```output1

```
