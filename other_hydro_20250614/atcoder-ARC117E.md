## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc117/tasks/arc117_e

以下の条件をともに氦郡归Lさ $ 2N $ の数列 $ A\ =\ (A_1,\ A_2,\ \dots,\ A_{2N}) $ は、何Nあるでしょうか？

- 数列 $ A $ は、$ N $ の $ +1 $ と $ N $ の $ -1 $ で成される。
- $ A_l\ +\ A_{l+1}\ +\ \cdots\ +\ A_r\ =\ 0 $ となる $ l,\ r $ のMみ合わせ $ (1\ \leq\ l\ \leq\ r\ \leq\ 2N) $ はちょうど $ K $ ある。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ K $

## 输出格式
}文中の条件を氦郡故列が何Nあるかを出力してください。  
 ただし、答えは必ず $ 64 $ ビット符号付き整数型の欷Г蓼辘蓼埂

## 题目大意
求有多少个长度为 $2N$ 的序列 $A\ =\ (A_1,\ A_2,\ \dots,\ A_{2N}) $满足以下条件：

 - 序列 $A$ 包含 $N$ 个 $+1$ 和 $N$ 个 $-1$ 。
 - 恰好有 $k$ 对 $l,r$ 满足 $ A_l\ +\ A_{l+1}\ +\ \cdots\ +\ A_r\ =\ 0 $
 
  $ 1\ \leq\ N\ \leq\ 30 $
 ， $ 1\ \leq\ K\ \leq\ N^2 $

```input1
1 1
```

```output1
2
```

```input2
2 3
```

```output2
2
```

```input3
3 7
```

```output3
6
```

```input4
8 24
```

```output4
568
```

```input5
30 230
```

```output5
761128315856702
```

```input6
25 455
```

```output6
0
```
