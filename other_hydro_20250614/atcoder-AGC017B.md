## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc017/tasks/agc017_b

$ N $ のマスが一列にKんでいます． 一番左のマスには整数 $ A $ が，一番右のマスには整数 $ B $ がかれており，他のマスには何もかれていません．

青蚓は，何もかれていないマスに整数をきzみ，次の条件を氦郡工瑜Δ摔筏郡い扦梗

- どのO接する $ 2 $ マスについても，かれている整数の差は $ C $ 以上 $ D $ 以下である．

青蚓は，この条件を氦郡瓜蓼辏いくらでも大きい整数や小さい整数をきzむことができます． 青蚓が条件を氦郡工瑜Δ苏数をきzむことができるかを判定してください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $ $ C $ $ D $

## 输出格式
青蚓が条件を氦郡工瑜Δ苏数をきzむことができるなら `YES` を，できないなら `NO` を出力せよ．

## 题目大意
有N个格子排成一排，最左边的里面写了数字A，最右边的写了数字B，中间的 格子都是空的。 你需要在中间的每个格子里填上一个数字，使得这个序列中，任意相邻两个数 的差的绝对值在[C,D]之间。 问是否存在这样一种可行的填数方案，输出YES或者NO

```input1
5 1 5 2 4
```

```output1
YES
```

```input2
4 7 6 4 5
```

```output2
NO
```

```input3
48792 105960835 681218449 90629745 90632170
```

```output3
NO
```

```input4
491995 412925347 825318103 59999126 59999339
```

```output4
YES
```

- $ 0\ \leq\ C\ \leq\ D\ \leq\ 10^9 $
- 入力はすべて整数

例えば，左のマスからに $ 1,\ -1,\ 3,\ 7,\ 5 $ となるように整数をきzめばよいです．
