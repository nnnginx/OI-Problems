<p>Aliens started their journey towards our earth. They some how mastered our English language through a dictionary. However, they were unable to understand the concept of spaces present between words in a sentence. <br><br>They are continuously sending messages to our earth but it has to be converted to normal form <strong>(</strong>i.e if the aliens msg is <strong>helloworld</strong> it has to be translated into <strong>hello world</strong> with help of provided dictionary <strong>)</strong>. Scientist believes that these messages consist a very valuable information about how they planning to attack us. And it is guaranteed that aliens communicate only with the words present in the dictionary. <br><br>This small planet needs your help.... protect it with your skill.</p>
<h3>Input</h3>
<ul>
<li>The first line of the input contains an integer <strong>N</strong> denoting the size of dictionary.</li>
<li>The next following <strong>N</strong> lines consist of words present in dictionary(all letters are lowercase)</li>
<li>The next line of the input contains an integer <strong>T</strong> denoting no.of alien language words that are need to be converted</li>
<li><strong>See sample input and output for better understanding</strong></li>
</ul>
<p>&nbsp;</p>
<h3>Output</h3>
<p>&nbsp;</p>
<ul>
<li>For each test case, output a single line which contains 2 space seperated words that are present in the dictionary.</li>
<li>In case of multiple answers make sure the 1st word length to be minimum</li>
</ul>
<p>&nbsp;</p>
<h3>Constraints</h3>
<ul>
<li><strong>1</strong> ¡Ü <strong>N</strong> ¡Ü <strong>10<sup>5</sup></strong></li>
<li><strong>1</strong> ¡Ü <strong>|W|</strong> ¡Ü <strong>20</strong></li>
<li><strong>1</strong> ¡Ü <strong>T</strong> ¡Ü <strong>1000</strong></li>
<li><strong>1</strong> ¡Ü <strong>|A|</strong> ¡Ü <strong>40</strong></li>
<li>where <strong>|W|</strong> denotes the length of the largest string present in the dictionary and <strong>|A|</strong> denotes the length of the string from alien language</li>
</ul>
<p>&nbsp;</p>
