## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc106/tasks/arc106_a

整数 $ N $ が与えられます。 $ 3^A\ +\ 5^B\ =\ N $ を氦郡拐の整数のM $ (A,\ B) $ が存在するか判定し、存在する龊悉 $ 1 $ M求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
条件を氦郡菇M $ (A,\ B) $ が存在しない龊悉 `-1` と出力せよ。

存在する龊悉 $ A $ と $ B $ を空白区切りで出力せよ。答えが}数存在する龊悉悉嗓欷虺隽Δ筏皮猡まわない。

## 题目大意
输入 $n$，请找到并输出两个正整数 $a,b$，满足 $3^a+5^b=n$。如果没有符合要求的输出 `-1`。

```input1
106
```

```output1
4 2
```

```input2
1024
```

```output2
-1
```

```input3
10460353208
```

```output3
21 1
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^{18} $
- 入力はすべて整数である。

### Sample Explanation 1

$ 3^4\ +\ 5^2\ =\ 81\ +\ 25\ =\ 106 $ なので、$ (A,\ B)\ =\ (4,\ 2) $ は条件を氦郡筏蓼埂
