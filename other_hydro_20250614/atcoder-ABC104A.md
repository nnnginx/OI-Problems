## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc104/tasks/abc104_a

プログラミングコンペティションサイト *AtCode* は、プログラミングコンテストを定期的に_催しています。

AtCode で次に_催されるコンテストは ABC と呼ばれ、レ`ティングが $ 1200 $ 未氦尾渭诱撙违飑`ティングが婴筏蓼埂

その次に_催されるコンテストは ARC と呼ばれ、レ`ティングが $ 2800 $ 未氦尾渭诱撙违飑`ティングが婴筏蓼埂

そのさらに次に_催されるコンテストは AGC と呼ばれ、すべての参加者のレ`ティングが婴筏蓼埂

高颏んの AtCode でのレ`ティングは $ R $ です。彼のレ`ティングが婴工氪韦违偿螗匹攻趣虾韦扦筏绀Δ？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ R $

## 输出格式
高颏んのレ`ティングが婴工氪韦违偿螗匹攻趣蚊称（`ABC`, `ARC`, `AGC` のいずれか）を出力せよ。

## 题目大意
编程竞赛网站AtCode定期举行编程竞赛。

AtCode的第一场比赛称为ABC，该比赛的参赛者的评分必须低于1200.

第二场比赛称为ARC，该比赛的参赛者的评分必须高于或等于1200且低于2800.

第三场比赛称为AGC，该比赛的参赛者的评分必须高于或等于2800.

AtCode对高桥的评分是 R。他应该参加哪一场比赛？

即编一个程序，如果R低于1200，输出ABC；如果R高于或等于1200且低于2800，输出ARC；如果R高于或等于2800，输出AGC。
         

```input1
1199
```

```output1
ABC
```

```input2
1200
```

```output2
ARC
```

```input3
4208
```

```output3
AGC
```

## 提示
### 制s

- $ 0\ <\ =\ R\ <\ =\ 4208 $
- $ R $ は整数である。

### Sample Explanation 1

$ 1199 $ は $ 1200 $ 未氦胜韦 ABC でレ`ティングが婴筏蓼埂

### Sample Explanation 2

$ 1200 $ は $ 1200 $ 未氦扦悉胜い韦 ABC ではレ`ティングが婴筏蓼护螭、$ 2800 $ 未氦扦悉るので ARC でレ`ティングが婴筏蓼埂
