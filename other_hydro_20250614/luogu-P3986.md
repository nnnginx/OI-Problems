## 题目描述
定义一个数列：


$f(0) = a, f(1) = b, f(n) = f(n - 1) + f(n - 2)$


其中 $a, b$ 均为正整数，$n \geq 2$。


问有多少种 $(a, b)$，使得 $k$ 出现在这个数列里，且不是前两项。


由于答案可能很大，你只需要输出答案模 $10^9 + 7$ 的结果即可。


## 输入格式
一行一个整数 $k$。


## 输出格式
一行一个数，表示答案模 $10^9 + 7$ 的结果。


```input1
19260817
```

```output1
34166325
```

```input2
1000000000
```

```output2
773877569
```

## 提示
$1 \leq k \leq 10^9$
