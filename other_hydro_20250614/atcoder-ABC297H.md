## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc297/tasks/abc297_h

正整数列のうち、全てのO接している $ 2 $ がなるものを**素晴らしい整数列**と定めます。

要素のt和が $ N $ の素晴らしい整数列全てにするLさのt和を $ 998244353 $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
定义一个正整数序列是好的，当且仅当序列中相邻的元素都不相等。你需要求出序列中元素总和为 $n$ 的好序列个数 $\bmod\ 998244353$ 的值。

```input1
4
```

```output1
8
```

```input2
297
```

```output2
475867236
```

```input3
123456
```

```output3
771773807
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 2\ \times\ 10^5 $
- 入力はすべて整数
 
### Sample Explanation 1

要素のt和が $ 4 $ の素晴らしい整数列は、$ (4),(1,3),(3,1),(1,2,1) $ の $ 4 $ です。なので、答えはこれらのLさのt和の $ 1+2+2+3=8 $ です。 $ (2,2) $ や $ (1,1,2) $ はt和が $ 4 $ ですが、I方 $ 1 $ 目と $ 2 $ 目が等しいため条件を氦郡筏蓼护蟆
