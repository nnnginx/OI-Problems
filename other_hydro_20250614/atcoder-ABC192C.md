## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc192/tasks/abc192_c

$ 0 $ 以上の整数 $ x $ にして、$ g_1(x),\ g_2(x),\ f(x) $ を次のように定めます。

- $ g_1(x)= $ $ x $ を十M法で表したときの各桁の数字を大きいにKび替えてできる整数
- $ g_2(x)= $ $ x $ を十M法で表したときの各桁の数字を小さいにKび替えてできる整数
- $ f(x)=g_1(x)-g_2(x) $

例えば $ g_1(314)=431 $, $ g_2(3021)=123 $, $ f(271)=721-127=594 $ です。先^の余分な $ 0 $ はoされることに注意してください。

整数 $ N,K $ が与えられるので、$ a_0=N $, $ a_{i+1}=f(a_i)\ (i\geq\ 0) $ で定まる数列の $ a_K $ を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
$ a_K $ を出力せよ。

## 题目大意
定义如下函数：

- $g_1(x)$ 为 $x$ 在十进制下的数位按照降序排列形成的数；
- $g_2(x)$ 为 $x$ 在十进制下的数位按照升序排列形成的数；
- $f(x)=g_1(x)-g_2(x)$。

给定 $N,K$，对于序列 $a_0=N$，$a_i=f(a_{i-1})$。求出 $a_K$。

```input1
314 2
```

```output1
693
```

```input2
1000000000 100
```

```output2
0
```

```input3
6174 100000
```

```output3
6174
```

## 提示
### 制s

- $ 0\ \leq\ N\ \leq\ 10^9 $
- $ 0\ \leq\ K\ \leq\ 10^5 $
- 入力は全て整数

### Sample Explanation 1

\- $ a_0=314 $ - $ a_1=f(314)=431-134=297 $ - $ a_2=f(297)=972-279=693 $ です。

### Sample Explanation 2

\- $ a_0=1000000000 $ - $ a_1=f(1000000000)=1000000000-1=999999999 $ - $ a_2=f(999999999)=999999999-999999999=0 $ - $ a_3=f(0)=0-0=0 $ - $ \vdots $ となります。
