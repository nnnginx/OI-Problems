## 题目描述

给你一棵 $n$ 个点的树，每个结点有颜色。

你从一个点 $u$ 出发，可以走到相邻的点 $v$，当且仅当 $ (c_u + 1) \ \% C \ = c_v $，其中 $C$ 是给定的。

有以下两种操作：

- `1 u c` 将 $u$ 的颜色修改为 $c$。
- `2 u`   查询从点 $u$ 出发，能够到达多少个点。

## 输入格式

第一行三个数，$n, q, C$。分别表示点数，操作数，和颜色数。  
第二行 $n$ 个数，第 $i$ 个数表示 $c_i$。  
接下来 $n - 1$ 行，每行两个数 $u_i, v_i$，表示一条树边。  
接下来 $q$ 行。每行描述一个操作。

## 数据规模与约定

对于 $100\%$ 的数据，$n \leq 10 ^5$，$q \leq 3 \times 10 ^5$，$C \leq 100$。


| 子任务 |     特殊限制     | 分值 |
| :---: | :----------------------: | :--: |
|  $1$   | $n, q\leq 10 ^3$ | $15$ |
|  $2$   |     $C = 1$      | $5$  |
|  $3$   |     $C = 2$      | $10$ |
|  $4$   |     $C = 3$      | $15$ |
|  $5$   | 不存在 $1$ 操作  | $20$ |
|  $6$   |                  | $35$ |
