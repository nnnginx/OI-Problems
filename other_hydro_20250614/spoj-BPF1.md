<p>Brenden is a good guy, he wants to make a filter to be put in class rooms, so then no one whould be able to throw bad words to others.</p>
<p><span style="white-space: normal;">He invented a politeness filter, which could be attached to the ceil of the class room, and it will allow only good words to pass.</span></p>
<p><span style="white-space: normal;">How the filter works?</span></p>
<p><span style="white-space: normal;">The politeness filter contains several steps of filtering(layers) , each step(layer) would test the word against its own filtering settings, then send the word into its queue if the word matched its filtering string, the queue will forward those words to sepcific group of people.</span></p>
<p><span style="white-space: normal;">In another way, the configuration consist of several consequtive QUEUE tags, each one has a name,bpf properties:</span></p>
<pre><span style="white-space: normal;">&lt;QUEUE&nbsp; name="queue_name"<br></span><span style="white-space: normal;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; bpf = "bpf_string"<br></span><span style="white-space: normal;">/&gt;</span></pre>
<p>As you know, talking is not like shouting, so the filter will distinguish between normal words, and shouted words.</p>
<p><span style="white-space: normal;">The bpf string is written in the following grammer:</span></p>
<pre><span style="white-space: normal;">&lt;bpf_string&gt;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;::= "" | &lt;words_filter&gt; "|| (shout and " &lt;words_filter&gt; ")"<br></span><span style="white-space: normal;">&lt;words_filter&gt;&nbsp; &nbsp; &nbsp; &nbsp;::= "(" &lt;many_words_filter&gt; ")"<br></span><span style="white-space: normal;">&lt;many_words_filter&gt;&nbsp; ::= &lt;single_word_filter&gt; | &lt;single_word_filter&gt; " or " &lt;single_word_filter&gt;<br></span><span style="white-space: normal;">&lt;single_word_filter&gt; ::= "word " &lt;word&gt;</span></pre>
<p>(Please review the Backus¨CNaur form https://en.wikipedia.org/wiki/Backus%E2%80%93Naur_form if you didn't understand the notation).</p>
<p><span style="white-space: normal;">(Note: if the notation is a little bit hard, don't worry, the example is clear)</span></p>
<p>Ok, Your task now is to write the configuration of a politeness filter!</p>
<p>The input will start with the word "Config:" then following 4 lines with the following grammer:</p>
<pre><span style="white-space: normal;">&lt;line&gt;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;::= "&nbsp; &nbsp; &nbsp; - " &lt;queue_name&gt; "=" &lt;words_specifications&gt;<br></span><span style="white-space: normal;">&lt;words_specifications&gt; ::= "" | &lt;words_list&gt;<br></span><span style="white-space: normal;">&lt;words_list&gt;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;::= &lt;word&gt; | &lt;word&gt; "," &lt;words_list&gt;</span></pre>
<p>Please note that &lt;queue_name&gt; and &lt;word&gt; are strings that consists of combinations of lower case, upper case, numbers, underscores only (no other punctuations, no spaces), with a limit of 50 character each, and the &lt;word_list&gt; is no more than 10 words.</p>
<p>Also, you can assume that queue names are different, also the &lt;word_specificatinos&gt;s are different.</p>
<p>Final note, if the &lt;words_specifications&gt; was empty, don't print the queue at all.</p>
<p><span style="white-space: normal;">And after printing all the queues, please print a final queue with queue_name="OTHER" and an empty bpf.</span></p>
<p>Ok, that's all, here is some examples:</p>
