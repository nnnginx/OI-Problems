## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc021/tasks/agc021_a

$ N $ 以下の正の整数の $ 10 $ M法での各桁の和の最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ 以下の正の整数の $ 10 $ M法での各桁の和の最大を出力せよ。

## 题目大意
给出N，求小于等于N的正整数中用十进制表示各数位数字之和的最大值。



```input1
100
```

```output1
18
```

```input2
9995
```

```output2
35
```

```input3
3141592653589793
```

```output3
137
```

## 提示
### 制s

- $ 1\leq\ N\ \leq\ 10^{16} $
- $ N $ は整数である

### Sample Explanation 1

例えば $ 99 $ の各桁の和は $ 18 $ で、これが求める最大となります。

### Sample Explanation 2

例えば $ 9989 $ の各桁の和は $ 35 $ で、これが求める最大となります。
