## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc044/tasks/arc060_a

高蚓は、$ N $ 枚のカ`ドを持っています。 $ i\ \,\ (1\ \leq\ i\ \leq\ N) $ 番目のカ`ドには、整数 $ x_i $ がかれています。 高蚓は、これらのカ`ドの中から $ 1 $ 枚以上をxび、 xんだカ`ドにかれた整数の平均をちょうど $ A $ にしたいと考えています。 そのようなカ`ドのxび方が何通りあるか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ x_1 $ $ x_2 $ $ ... $ $ x_N $

## 输出格式
かれた整数の平均がちょうど $ A $ となるようなカ`ドのxび方のt数を出力せよ。

## 题目大意
高桥有n张卡。在i(1≤i≤n)的第一个磁卡上，卡上面写着整数x_i。

高桥从这些卡片中挑选1张以上，想把选择的卡片上写的整数的平均数变成等于A的数。问有几种方案。

读入： 第一行读入N,A； 接下来一行读入N个数

输出： 一个数，记得加回车

感谢@STEPHEN_ 提供的翻译

```input1
4 8
7 9 8 9
```

```output1
5
```

```input2
3 8
6 6 9
```

```output2
0
```

```input3
8 5
3 6 2 8 7 6 5 9
```

```output3
19
```

```input4
33 3
3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3 3
```

```output4
8589934591
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 50 $
- $ 1\ \leq\ A\ \leq\ 50 $
- $ 1\ \leq\ x_i\ \leq\ 50 $
- $ N,\,A,\,x_i $ はいずれも整数である

### 部分点

- $ 1\ \leq\ N\ \leq\ 16 $ を氦郡攻签`タセットに正解した龊悉稀$ 200 $ 点が与えられる。

### Sample Explanation 1

\- 平均が $ 8 $ となるカ`ドのxび方は、以下の $ 5 $ 通りです。 - $ 3 $ 枚目のカ`ドのみをxぶ。 - $ 1 $ 枚目と $ 2 $ 枚目のカ`ドをxぶ。 - $ 1 $ 枚目と $ 4 $ 枚目のカ`ドをxぶ。 - $ 1 $ 枚目、$ 2 $ 枚目および $ 3 $ 枚目のカ`ドをxぶ。 - $ 1 $ 枚目、$ 3 $ 枚目および $ 4 $ 枚目のカ`ドをxぶ。

### Sample Explanation 4

\- 答えは $ 32 $ ビット整数型にГ蓼椁胜龊悉あります。
