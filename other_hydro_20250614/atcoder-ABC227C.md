## 题目大意
#### 题目描述

给出正整数 $ N $。

求 $ A\leq\ B\leq\ C $ 并且 $ ABC\leq\ N $ 的正整数对 $ (A,B,C) $ 的个数。

注意，在限制的条件下，保证答案小于 $ 2^{63}$。

#### 输入格式

输入按以下格式：

> $ N $

#### 输出格式

输出答案。

#### 提示/说明

条件

- $ 1\ \leq\ N\ \leq\ 10^{11} $
- $ N $是整数

样例解释 $1$：

满足条件的组有 $5$ 组：$(1,1,1),(1,1,2),(1,1,3),(1,1,4),(1,2,2)$。

```input1
4
```

```output1
5
```

```input2
100
```

```output2
323
```

```input3
100000000000
```

```output3
5745290566750
```
