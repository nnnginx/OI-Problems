## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc024/tasks/agc024_a

高蚓、中蚓、低蚓は、それぞれ整数 $ A,B,C $ を持っています。 以下の操作を $ K $ 回行った後、高蚓の持っている整数から中蚓の持っている整数を引いたを求めてください。

- $ 3 $ 人は同rに、他の $ 2 $ 人の持っている整数の和を求める。その後、自分の持っている整数を求めた整数で置きQえる。

ただし、答えの~が $ 10^{18} $ を超える龊悉稀⒋わりに `Unfair` と出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ K $

## 输出格式
操作を $ K $ 回行った後の、高蚓の持っている整数から中蚓の持っている整数を引いたを出力せよ。 ただし、答えの~が $ 10^{18} $ を超える龊悉稀⒋わりに `Unfair` と出力せよ。

## 题目大意
输入$A_0,B_0,C_0,K$

重复$K$次操作：

$A_i=B_{i-1}+C_{i-1}$

$B_i=A_{i-1}+C_{i-1}$

$C_i=A_{i-1}+B_{i-1}$

如果$A_k-B_k$的绝对值大于$10^{18}$输出$Unfair$

否则输出$A_k-B_k$

感谢$@RioBlu$的翻译

```input1
1 2 3 1
```

```output1
1
```

```input2
2 3 2 0
```

```output2
-1
```

```input3
1000000000 1000000000 1000000000 1000000000000000000
```

```output3
0
```

## 提示
### 制s

- $ 1\ \leq\ A,B,C\ \leq\ 10^9 $
- $ 0\ \leq\ K\ \leq\ 10^{18} $
- 入力はすべて整数である

### Sample Explanation 1

$ 1 $ 回の操作後、高蚓、中蚓、低蚓の持っている整数はそれぞれ $ (5,4,3) $ となります。$ 5-4=1 $ を出力します。
