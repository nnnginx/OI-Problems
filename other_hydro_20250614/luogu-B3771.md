## 题目描述
如果一个十进制数字从左到右读时，所有数位都是**从小到大严格递增**的，我们就称它是幸运数字。例如：

- $9$、$27$、$1234$ 都是幸运数字；
- $11$、$80$、$243$、$1503$ 不是幸运数字。

对于给定的 $a$ 和 $b$，请你求出 $a,a+1,a+2,\cdots,b$ 中幸运数字的数量。

## 输入格式
输入一行空格分隔的两个整数 $a$ 和 $b$。

## 输出格式
输出一行一个整数，代表 $a,a+1,a+2,\cdots,b$ 中幸运数字的数量。

```input1
1 100
```

```output1
45
```

```input2
4096 65536
```

```output2
141
```

## 提示
对于 $100\%$ 的数据，满足 $1\leq a\leq b\leq 1,000,000$。
