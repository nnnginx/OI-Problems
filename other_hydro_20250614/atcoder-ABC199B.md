## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc199/tasks/abc199_b

Lさ $ N $ の数列 $ A\ =\ (A_1,\ A_2,\ A_3,\ \dots,\ A_N),\ B\ =\ (B_1,\ B_2,\ B_3,\ \dots,\ B_N) $ が与えられます。  
 以下の条件を氦郡拐数 $ x $ の数を求めてください。

- $ 1\ \le\ i\ \le\ N $ を氦郡谷ての整数 $ i $ について $ A_i\ \le\ x\ \le\ B_i $

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ A_3 $ $ \dots $ $ A_N $ $ B_1 $ $ B_2 $ $ B_3 $ $ \dots $ $ B_N $

## 输出格式
答えを出力せよ。

## 题目大意
**题目描述**

给你长度为 N 的两个数列 A 和 B。

求满足：大于等于 A 数列中所有数且小于等于 B 数列中所有数的整数的个数。

**输入格式**

第一行，一个整数 N。

第二行，N 个整数，表示数列 A。

第三行，N 个整数，表示数列 B。

**输出格式**

一个整数，表示满足条件的整数的个数。

```input1
2
3 2
7 5
```

```output1
3
```

```input2
3
1 5 3
10 7 3
```

```output2
0
```

```input3
3
3 2 5
6 9 8
```

```output3
2
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 100 $
- $ 1\ \le\ A_i\ \le\ B_i\ \le\ 1000 $
- 入力に含まれるは全て整数

### Sample Explanation 1

$ x $ は $ 3\ \le\ x\ \le\ 7 $ と $ 2\ \le\ x\ \le\ 5 $ のI方を氦郡丹胜堡欷肖胜辘蓼护蟆 そのような整数 $ x $ は $ 3,\ 4,\ 5 $ の $ 3 $ あります。

### Sample Explanation 2

条件を氦郡拐数 $ x $ が存在しないこともあります。
