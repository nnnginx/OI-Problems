## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc180/tasks/abc180_f

点にラベルが付きxにはラベルが付いていない $ N $ 点 $ M $ xのgともBYとも限らないグラフであって、以下の条件を氦郡工猡韦数を $ 10^9+7 $ で割ったあまりを求めてください。

- 自己ル`プを持たない
- すべての点の次数が $ 2 $ 以下である
- 各BY成分のサイズをKべたとき、その最大がちょうど $ L $ である

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ L $

## 输出格式
答えを出力せよ。

## 题目大意
求 $N$ 个点，$M$ 条边且满足以下条件的图的数量：

- 图中无自环；

- 每个点度数最多为 $2$；

- 连通块大小的最大值恰好为 $L$。

答案对 $10^9+7$ 取模。

$2\le N\le300$，$1\le M,L\le N$。

```input1
3 2 3
```

```output1
3
```

```input2
4 3 2
```

```output2
6
```

```input3
300 290 140
```

```output3
211917445
```
