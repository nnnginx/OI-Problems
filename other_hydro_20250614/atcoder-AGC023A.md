## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc023/tasks/agc023_a

Lさ $ N $ の整数列 $ A $ があります。

$ A $ の **空でない** **BAする** 部分列であって、そのt和が $ 0 $ になるものの数を求めてください。 ただし、ここで数えるのは **部分列の取り出し方** であることに注意してください。 つまり、ある $ 2 $ つの部分列が列として同じでも、取り出した位置がなるならば、それらはe々に数えるものとします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $

## 输出格式
$ A $ の空でないBAする部分列であって、そのt和が $ 0 $ になるものの数を出力せよ。

## 题目大意
输入一个数列$a$

任选$i,j$满足$i$≤$j$

问有多少种情况$a[i]+a[i+1]...+a[j]=0$

注意：数列长度≤$2*10^5$

每个数字在$[-10^9,10^9]$

感谢$@RioBlu$的翻译

```input1
6
1 3 -4 2 2 -2
```

```output1
3
```

```input2
7
1 -1 1 -1 1 -1 1
```

```output2
12
```

```input3
5
1 -2 3 -4 5
```

```output3
0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ -10^9\ \leq\ A_i\ \leq\ 10^9 $
- 入力はすべて整数である。

### Sample Explanation 1

空でないBAした部分列であって、そのt和が $ 0 $ になるのは、$ (1,3,-4) $, $ (-4,2,2) $, $ (2,-2) $ の $ 3 $ つです。

### Sample Explanation 2

この例では、列として同じだが取り出す位置のなる部分列が}数回数えられています。 例えば、$ (1,-1) $ は $ 3 $ 回数えられています。

### Sample Explanation 3

空でないBAした部分列であって、そのt和が $ 0 $ になるものはありません。
