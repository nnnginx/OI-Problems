## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc143/tasks/abc143_d

高蚓は、互いに区e出来る $ N $ 本の棒を持っています。$ i $ 本目の棒のLさは $ L_i $ です。

高蚓は、これらのうち $ 3 $ 本の棒を使って三角形を作ろうとしています。このとき、棒のLさを $ a,\ b,\ c $ として、以下の条件がすべて成り立たなければなりません。

- $ a\ <\ b\ +\ c $
- $ b\ <\ c\ +\ a $
- $ c\ <\ a\ +\ b $

作れる三角形は何Nあるでしょうか。ただし、$ 2 $ つの三角形は、そのうち一方にのみ使われている棒が存在するときになるとします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ L_1 $ $ L_2 $ $ ... $ $ L_N $

## 输出格式
作れる三角形が何Nあるかを出力せよ。

## 题目大意
【问题描述】

小$Z$有$N$条木棍，每条木棍的长度为$D_i$。他想在这$N$条木棍中选择三条，让这三条木棍能构成一个三角形。小$Z$想知道有多少种方案能构成三角形（不同的方案定义为一条木棍在其中一个三角形中出现不在另一个中出现）。

【输入格式】

输入第一行一个整数$N$，表示木棍的个数。

第二行$N$个整数$D_i$，表示木棍的长度。

【输出格式】

输出一个整数，表示构成三角形的方案数。

【样例说明】

样例$1$：只能形成一个三角形：由第一，第二和第三根木棍形成的三角形。

样例$2$：无法形成三角形。

【数据规模与约定】

对于$100\%$的数据，$3 \leq N \le 2×10^3$，$1\le D_i 
\le10^3$。


```input1
4
3 4 2 1
```

```output1
1
```

```input2
3
1 1000 1
```

```output2
0
```

```input3
7
218 786 704 233 645 728 389
```

```output3
23
```

## 提示
### 制s

- 入力は全て整数
- $ 3\ <\ =\ N\ <\ =\ 2\ \times\ 10^3 $
- $ 1\ \leq\ L_i\ \leq\ 10^3 $

### Sample Explanation 1

作れる三角形は、$ 1 $、$ 2 $、$ 3 $ 番目の棒から成る三角形のみです。

### Sample Explanation 2

作れる三角形はありません。
