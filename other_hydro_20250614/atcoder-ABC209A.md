## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc209/tasks/abc209_a

$ A $ 以上かつ $ B $ 以下の整数はいくつありますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
$ A $ 以上かつ $ B $ 以下の整数の数を出力せよ。

## 题目大意
给定 $A$，$B$，求所有的整数 $x$，满足 $A \leq x \leq B$。你只需要输出这样的 $x$ 的数量。**注意 $A$ 有可能大于 $B$，这时请输出 $0$。**

$1 \leq A \leq 100$，$1 \leq B \leq 100$。

```input1
2 4
```

```output1
3
```

```input2
10 100
```

```output2
91
```

```input3
3 2
```

```output3
0
```

## 提示
### 制s

- $ 1\ \leq\ A\ \leq\ 100 $
- $ 1\ \leq\ B\ \leq\ 100 $
- $ A,\ B $ は整数である。

### Sample Explanation 1

$ 2 $ 以上かつ $ 4 $ 以下の整数は $ 2,\ 3,\ 4 $ の $ 3 $ つなので、 $ 3 $ を出力してください。

### Sample Explanation 3

$ 3 $ 以上かつ $ 2 $ 以下の整数は存在しないので、 $ 0 $ を出力してください。
