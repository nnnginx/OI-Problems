<p><span lang="en">How many ways are there to pay <em>n</em> cents? We assume that the payment must be made with pennies (1 cent),  nickels (5 cents), dimes (10 cents), quarters (25 cents), and  half-dollars (50 cents). </span></p>
<p>For example, there are four ways to pay 13 cents, namely (13 pennies),  (2 nickels, 3 pennies), (1 nickel, 8 pennies), and (1 dime, 3 pennies).</p>
<h3>Input</h3>
<p>The input will contain multiple test cases. Each test case contains a single line with a single integer <em>n</em> (1 ¡Ü <em>n</em> ¡Ü 1000000000).</p>
<p>The input will be terminated by the end of file.</p>
<p>&nbsp;</p>
<h3>Output</h3>
<p>For each input integer <em>n</em>, output how many ways are there to pay <em>n</em> cents in a single line.</p>
<h3>Sample Input</h3>
<p><tt> </tt></p>
<pre><tt>13
100000000
</tt></pre>
<p><tt> </tt></p>
<p>&nbsp;</p>
<h3>Sample Output</h3>
<p><tt> </tt></p>
<pre><tt>4
66666793333412666685000001
</tt></pre>
<p><tt> </tt></p>
