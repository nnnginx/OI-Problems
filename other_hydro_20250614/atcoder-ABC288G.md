## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc288/tasks/abc288_g

位置 $ 0,\ 1,\ 2,\ \ldots,\ 3^N-1 $ にそれぞれ $ 0 $ あるいは $ 1 $ の爆があります。  
 また、位置 $ x $ と位置 $ y $ は $ i=0,1,\ \ldots,\ N-1 $ すべてにし以下の条件を氦郡工趣、またそのときに限り**近い位置**であるとします。

- $ x,\ y $ を $ 3 $ M表したときの $ 3^i $ の位の数字をそれぞれ $ x',\ y' $ として、$ |x'\ -\ y'|\ \leq\ 1 $ が成立する。
 
位置 $ i $ と近い位置にある爆の数が $ A_i $ であるとわかっているとき、爆の配置としてありえるものを $ 1 $ つ出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_0 $ $ A_1 $ $ \ldots $ $ A_{3^N-1} $

## 输出格式
位置 $ i $ に爆がないとき $ B_i\ =\ 0 $ 、位置 $ i $ に爆があるとき $ B_i\ =\ 1 $ として $ B_0,\ B_1,\ \ldots,\ B_{3^N-1} $ を空白区切りで出力せよ。

## 题目大意
位置 $0\sim 3^n-1$ 中有若干个雷，你需要找出这些雷，其中，每个位置至多有一个雷。

我们定义两个位置 $i$ 和 $j$ 是相邻的，当且仅当 $i$ 和 $j$ 在三进制表示下的每一位的差的绝对值都小于等于 $1$。（当然，自己也算与自己相邻）

现在，对于每一个位置 $i$，$A_i$ 表示与位置 $i$ 相邻的位置中雷的个数，请根据给定 $A$ 输出每个位置雷的个数。

```input1
1
0 1 1
```

```output1
0 0 1
```

```input2
2
2 3 2 4 5 3 3 4 2
```

```output2
0 1 0 1 0 1 1 1 0
```

```input3
2
0 0 0 0 0 0 0 0 0
```

```output3
0 0 0 0 0 0 0 0 0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 12 $
- $ A_0,\ A_1,\ \ldots,\ A_{3^N-1} $ に辘工氡の配置が存在する
- 入力はすべて整数
 
### Sample Explanation 1

$ 0 $ と近い位置は $ 0 $ と $ 1 $ で、位置 $ 0 $ と位置 $ 1 $ に爆は合で $ 0 $ あります。 $ 1 $ と近い位置は $ 0 $ と $ 1 $ と $ 2 $ で、位置 $ 0 $ と位置 $ 1 $ と位置 $ 2 $ に爆は合で $ 1 $ あります。 $ 2 $ と近い位置は $ 1 $ と $ 2 $ で、位置 $ 1 $ と位置 $ 2 $ に爆は合で $ 1 $ あります。 $ 2 $ にのみ爆があるような配置は上の条件を全て氦郡工郡帷⒄答となります。
