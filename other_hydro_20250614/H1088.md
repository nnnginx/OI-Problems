# Description

给定一个正整数 $N$ ，求 $\sum_{1 \leq M \leq N}[(N \bmod M) \mid M]$ 。  

# Format

## input

一行一个正整数 $N$ 。  

## Output

一行一个正整数，代表答案。

# Samples

```input1
5
```

```output1
4
```

解释： 对 $N = 5$ ， $M = 1, 2, 4, 5$ 都是满足条件的。   

# Limitation

$1 \leq N \leq 10^{12}$ 。
