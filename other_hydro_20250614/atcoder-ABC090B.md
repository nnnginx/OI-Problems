## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc090/tasks/abc090_b

$ A $ 以上 $ B $ 以下の整数のうち、回文数となるものの数を求めてください。 ただし、回文数とは、先^に $ 0 $ をつけない $ 10 $ M表を文字列としてたとき、前からiんでも後ろからiんでも同じ文字列となるような正の整数のことを指します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
$ A $ 以上 $ B $ 以下の整数のうち、回文数となるものの数を出力せよ。

## 题目大意
查找在整数$A$与$B$之间回文数的个数

在这里回文数是指无论从前向后读还是从后向前读都一样的数

```input1
11009 11332
```

```output1
4
```

```input2
31415 92653
```

```output2
612
```

## 提示
### 制s

- $ 10000\ \leq\ A\ \leq\ B\ \leq\ 99999 $
- 入力はすべて整数である

### Sample Explanation 1

$ 11011,11111,11211,11311 $ の $ 4 $ つが条件を氦郡筏蓼埂
