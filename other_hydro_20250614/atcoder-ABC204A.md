## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc204/tasks/abc204_a

サ`バル、フェネック、アライグマの $ 3 $ 人がじゃんけんをして、あいこになりました。

フェネックが出した手を表す文字 $ x $ とアライグマが出した手を表す文字 $ y $ が与えられます。それぞれ、`0` はグ`、`1` はチョキを、`2` はパ`を表します。

サ`バルが出した手を表す文字を出力してください。なお、答えは一意に定まります。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ x $ $ y $

## 输出格式
答えを出力せよ。出力についても、`0` はグ`、`1` はチョキを、`2` はパ`を表すものとする。

## 题目大意
三个人在玩石头剪刀布。

现在已知两个人所出的手势。若想要三人平局，那么第三个人应该出什么？

其中，``0``代表石头，``1``代表剪刀，``2``代表布。

translate by [ChrisWangZi](https://www.luogu.com.cn/user/637180)

```input1
0 1
```

```output1
2
```

```input2
0 0
```

```output2
0
```

## 提示
### 制s

- $ x $, $ y $ は `0`, `1`, `2` のいずれか

### Sample Explanation 1

フェネックが出した手はグ`、アライグマが出した手はチョキでした。あいこになるのはサ`バルがパ`を出したときです。

### Sample Explanation 2

フェネックが出した手はグ`、アライグマが出した手はグ`でした。あいこになるのはサ`バルがグ`を出したときです。
