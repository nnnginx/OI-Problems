## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc205/tasks/abc205_a

$ 100 $ mL あたり $ A $ kcal のエネルギ`を持つドリンクがあります。このドリンク $ B $ mL は何 kcal のエネルギ`を持つでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
答えを表す数を出力せよ。

なお、想定解答との~`差または相`差が $ 10^{-6} $ 以下であれば正解としてQわれる。

## 题目大意
## 题目翻译

每 $100$ mL 饮料都会有 $A$ 千卡，那么现在有 $B$ mL 饮料，这瓶饮料会有多少千卡呢？

## 输入格式

输入用以下标准输入：

> $A$ $B$

## 输出格式

给出一个数字表示答案。

你给出的答案和假定答案绝对误差或者相对误差小于 $10^{-6}$ 就视为正确的答案。

## 说明/提示

### 限制

* $0 \leq A,B \leq 1000$
* 输入的数字全是整数。

### 样例说明 1

每 $100$ mL 饮料会有 $45$ 千卡，这瓶饮料有 $200$ 毫升，所以有 $90$ 千卡的能量。

### 样例说明 2

答案可能不是整数。

```input1
45 200
```

```output1
90
```

```input2
37 450
```

```output2
166.5
```

```input3
0 1000
```

```output3
0
```

```input4
50 0
```

```output4
0
```

## 提示
### 制s

- $ 0\ \leq\ A,\ B\ \leq\ 1000 $
- 入力は全て整数である。

### Sample Explanation 1

$ 100 $ mL あたり $ 45 $ kcal のエネルギ`を持つドリンクがあり、このドリンク $ 200 $ mL は $ 90 $ kcal のエネルギ`を持ちます。

### Sample Explanation 2

答えは整数でないこともあります。
