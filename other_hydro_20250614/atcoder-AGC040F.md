## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc040/tasks/agc040_f

数直上に，区eできない $ 2 $ つのxが置かれています． どちらのxも最初，座 $ 0 $ にあります．（xは同じ座摔送rに存在できます）

これらのxにして，以下の $ 2 $ Nの操作が可能です．

- 好きなxを $ 1 $ つxび，$ 1 $ 大きい座摔艘婴工耄
- 座摔涡・丹ゑxを，座摔未螭いxの位置へと移婴工耄 なお，もともと $ 2 $ つのxが同じ座摔酥盲い皮る龊悉虾韦馄黏ないが，その龊悉扦 $ 1 $ 回の操作として数える．

以上の操作を好きな番で $ N $ 回Rり返して，$ 2 $ つのxの一方が座 $ A $，他方が座 $ B $ にあるようにしたいです． このような婴し方が何通りあるかを求めてください． ただし答えは非常に大きくなることがあるので，$ 998244353 $ で割ったあまりを求めてください．

なお，ある $ 2 $ つの婴し方 $ x,y $ がなるとは，整数 $ i $ ($ 1\ \leq\ i\ \leq\ N $) であって， $ ( $ 婴し方 $ x $ で $ i $ 回目の操作後にxの置いてある座摔渭合 $ ) $ と $ ( $ 婴し方 $ y $ で $ i $ 回目の操作後にxの置いてある座摔渭合 $ ) $ がなるものが存在することを意味します．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ A $ $ B $

## 输出格式
条件をみたすxの婴し方が何通りあるかを $ 998244353 $ で割ったあまりを出力せよ．

## 题目大意
有两个棋子初始点都在坐标 $0$ ，两个棋子之间没有区别，总共要进行 $N$ 次操作，每次操作是如下操作其中之一：

1.选择一个棋子向前移动一步。

2.选择位置较后的那个棋子，直接移动到位置较前的那个棋子的位置。

问 $N$ 次操作后两个棋子分别在位置 $A,B$ 的方案数，对 $998244353$ 取模，两种方案是相同的，当且仅当两个棋子在每一步的坐标都是相同的（注意不是每一步的操作都相同）。

```input1
5 1 3
```

```output1
4
```

```input2
10 0 0
```

```output2
1
```

```input3
10 4 6
```

```output3
197
```

```input4
1000000 100000 200000
```

```output4
758840509
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^7 $
- $ 0\ \leq\ A\ \leq\ B\ \leq\ N $
- 入力されるはすべて整数である．

### Sample Explanation 1

以下の $ 4 $ 通りの婴し方があります． なお，$ (x,y) $ で，xの座摔それぞれ $ x,y $ にある状Bを表しています． - $ (0,0)→(0,0)→(0,1)→(0,2)→(0,3)→(1,3) $ - $ (0,0)→(0,0)→(0,1)→(0,2)→(1,2)→(1,3) $ - $ (0,0)→(0,0)→(0,1)→(1,1)→(1,2)→(1,3) $ - $ (0,0)→(0,1)→(1,1)→(1,1)→(1,2)→(1,3) $
