## 题目描述
给定一幅 $R \times C$ 的地图，其中 `.` 表示该格子能走，`X` 表示该格子不能走。判断该地图是否有死胡同。

如果一个格子满足上下左右四个格子中**只有一个格子能走**，那么这个格子就是死胡同。当然，不能走到地图外。

## 输入格式
输入的第一行包含两个正整数 $R,C$。

接下来 $R$ 行，每行包含 $C$ 个字符，描述这幅地图。

## 输出格式
如果地图中有死胡同，输出 $1$，否则输出 $0$。

```input1
4 3
XXX
X.X
X.X
XXX
```

```output1
1
```

```input2
5 5
XX.XX
X...X
.....
X...X
XX.XX
```

```output2
1
```

```input3
3 9
...XXX...
.X.....X.
...XXX...
```

```output3
0
```

## 提示
#### 【数据范围】

对于 $100\%$ 的数据，$3 \le R,C \le 10$。
