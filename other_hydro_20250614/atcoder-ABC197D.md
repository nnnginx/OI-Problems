## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc197/tasks/abc197_d

$ \mathrm{x} $ Sの正の向きを右、$ \mathrm{y} $ Sの正の向きを上とする $ 2 $ 次元座似矫嫔悉恕$ p_0,\ p_1,\ p_2,\ \dots,\ p_{N\ -\ 1} $ の $ N $ の点からなる正 $ N $ 角形があります。  
 ここで $ N $ は偶数であることが保^され、点 $ p_0,\ p_1,\ p_2,\ \dots,\ p_{N\ -\ 1} $ はこのに反r回りにKんでいます。  
 $ p_i $ の座摔 $ (x_i,\ y_i) $ とします。  
 $ x_0,\ y_0,\ x_{\frac{N}{2}},\ y_{\frac{N}{2}} $ が与えられるので、$ x_1,\ y_1 $ を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ x_0 $ $ y_0 $ $ x_{\frac{N}{2}} $ $ y_{\frac{N}{2}} $

## 输出格式
$ x_1,\ y_1 $ をこのに空白区切りで出力せよ。  
 出力されたそれぞれのについて、想定解答との~`差または相`差が $ 10^{-5} $ 以下であれば正解と判定される。

```input1
4
1 1
2 2
```

```output1
2.00000000000 1.00000000000
```

```input2
6
5 3
7 4
```

```output2
5.93301270189 2.38397459622
```

## 提示
### 制s

- $ 4\ \le\ N\ \le\ 100 $
- $ N $ は偶数
- $ 0\ \le\ x_0,\ y_0\ \le\ 100 $
- $ 0\ \le\ x_{\frac{N}{2}},\ y_{\frac{N}{2}}\ \le\ 100 $
- $ (x_0,\ y_0)\ \neq\ (x_{\frac{N}{2}},\ y_{\frac{N}{2}}) $
- 入力に含まれるは全て整数である
