## 题目描述
在一个长度为 $n$ 的排列中找出逆序对数量恰好为 $c$ 的排列总数。（$\bmod \ 10^9+7$）

## 输入格式
输入共一行为两个整数 $n,c$。

## 输出格式
输出逆序对数量恰好为 $c$ 的排列总数。（$\bmod \ 10^9+7$）

```input1
10 1
```

```output1
9
```

```input2
4 3
```

```output2
6
```

```input3
9 13
```

```output3
17957
```

## 提示
#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1\le n\le 10^3$，$1\le c\le 10^4$。
