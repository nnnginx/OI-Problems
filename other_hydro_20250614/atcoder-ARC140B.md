## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc140/tasks/arc140_b

`A`,`R`,`C` からなるLさ $ N $ の文字列 $ S $ が与えられます。

あなたは、$ S $ の中にO接する $ 3 $ 文字であって `ARC`となっているものが存在する限り以下の操作を行うことができます。

- **奇数** 回目の操作では、$ S $ の中でO接する $ 3 $ 文字であって `ARC` となっているものを一つxび、`R` で置きQえる。
- **偶数** 回目の操作では、$ S $ の中でO接する $ 3 $ 文字であって `ARC` となっているものを一つxび、`AC` で置きQえる。

操作を行える回数の最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
答えを出力せよ。

## 题目大意
### 题目描述
---

给出一个仅由 $A,R,C$ 组成的长度为 $n$ $(1\le n\le 2\times 10^5)$ 的字符串 $S$，您可以进行操作：

- 如果进行第奇数次操作，您需要用 $R$ 替换一次 $S$ 中的任意一个子串 $ARC$

- 如果进行第偶数次操作，您需要用 $AC$ 替换一次 $S$ 中的任意一个子串 $ARC$

最多能进行多少次操作？

翻译 by wukaichen888

### 输入格式

输入共两行，第一行含一个正整数 $n$。

第二行包括一个长度为 $n$ 的字符串 $S$。

### 输出格式

一行，表示答案。

```input1
6
AARCCC
```

```output1
2
```

```input2
5
AAAAA
```

```output2
0
```

```input3
9
ARCARCARC
```

```output3
3
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ S $ は `A`,`R`,`C` からなるLさ $ N $ の文字列

### Sample Explanation 1

以下のように操作すると、 $ 2 $ 回操作できます。 `AARCCC` → `ARCC` → `ACC`

### Sample Explanation 2

$ S $ の中にO接する $ 3 $ 文字であって `ARC` となっているものが存在しないため、操作を一度も行えません。
