## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc093/tasks/arc093_d

$ 2^N $ 人のx手がおり、それぞれ $ 1,\ 2,\ ...,\ 2^N $ の番号がついています。 これらのx手でト`ナメントを行うことにしました。

ト`ナメントは以下のようにして行われます。

- $ 1,\ 2,\ ...,\ 2^N $ の置Q $ p_1,\ p_2,\ ...,\ p_{2^N} $ をxぶ。
- x手たちがx手 $ p_1 $, x手 $ p_2 $, $ ... $, x手 $ p_{2^N} $ のに一列にKぶ。
- 列に残っているx手が $ 1 $ 人だけになるまで、以下をRり返す。
  - 列の前から $ 1 $ 番目と $ 2 $ 番目、$ 3 $ 番目と $ 4 $ 番目、$ ... $ のx手が椁蛐肖Α けたx手は列からxれる。伽盲窟x手たちは相序を保ったまま改めて一列にKぶ。
- 最後まで残ったx手が伽扦る。

$ 2 $ 人のx手が椁筏郡趣のY果は、入力として与えられる $ M $ の 整数 $ A_1,\ A_2,\ ...,\ A_M $ を用いて以下のようにけることが分かっています。

- ある $ i $ について $ y\ =\ A_i $ のとき、x手 $ 1 $ とx手 $ y $ ($ 2\ \leq\ y\ \leq\ 2^N $) が椁工毪冗x手 $ y $ が伽摹
- どの $ i $ についても $ y\ \neq\ A_i $ のとき、x手 $ 1 $ とx手 $ y $ ($ 2\ \leq\ y\ \leq\ 2^N $) が椁工毪冗x手 $ 1 $ が伽摹
- $ 2\ \leq\ x\ <\ y\ \leq\ 2^N $ のとき、x手 $ x $ とx手 $ y $ が椁工毪冗x手 $ x $ が伽摹

このト`ナメントの僬撙稀⒆畛酩诉xぶ置Q $ p_1,\ p_2,\ ...,\ p_{2^N} $ だけに依存します。 ト`ナメントを行うHに最初にxぶ置Q $ p_1,\ p_2,\ ...,\ p_{2^N} $ であって、 ト`ナメントのY果x手 $ 1 $ が伽工毪瑜Δ胜猡韦数を $ 10^9\ +\ 7 $ で割ったあまりを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ A_1 $ $ A_2 $ $ ... $ $ A_M $

## 输出格式
答えを出力せよ。

## 题目大意
**【题意简述】**

- 有 $2^N$ 个人，按照满二叉树的形态进行淘汰赛，一开始的排列顺序为所有 $(2^N)!$ 个排列之一。
- 你是第 $1$ 个人，已知每一对人之间的实力关系，具体地说：
  - 给出 $M$ 个人 $A_1 \sim A_M$。
  - 这 $M$ 个人都打得过你。
  - 你打得过除了这 $M$ 个人之外的所有其他人。
  - 对于剩下的情况（你不参与的情况），编号小的人胜利。
- 问你在所有的 $(2^N)!$ 种情况中，有多少种情况可以取得最终胜利。答案对 ${10}^9 + 7$ 取模。

**【输入格式】**

第一行两个整数 $N, M$。  
第二行 $M$ 个整数 $A_1, A_2, \ldots , A_M$。

**【输出格式】**

输出一个数表示方案数，对 ${10}^9 + 7$ 取模。

**【数据范围】**

$1 \le N \le 16$，$0 \le M \le 16$，$2 \le A_i \le 2^N$，$A_i < A_{i + 1}$。

```input1
2 1
3
```

```output1
8
```

```input2
4 3
2 4 6
```

```output2
0
```

```input3
3 0
```

```output3
40320
```

```input4
3 3
3 4 7
```

```output4
2688
```

```input5
16 16
5489 5490 5491 5492 5493 5494 5495 5497 18993 18995 18997 18999 19000 19001 19002 19003
```

```output5
816646464
```
