## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc223/tasks/abc223_a

高蚓の布の中には $ 100 $ 矣藏が $ 1 $ 枚以上入っており、それ以外には何も入っていません。

高蚓の布の中の合金~が $ X $ 窑扦る可能性はありますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $

## 输出格式
高蚓の布の中の合金~が $ X $ 窑扦る可能性がある龊悉 `Yes`、そうでない龊悉 `No` と出力せよ。

## 题目大意
输入一个自然数 $n$ ，判断 $n$ 是否为 $100$ 的倍数且大于 $0$ 。

```input1
500
```

```output1
Yes
```

```input2
40
```

```output2
No
```

```input3
0
```

```output3
No
```

## 提示
### 制s

- $ 0\ \leq\ X\ \leq\ 1000 $
- 入力は全て整数

### Sample Explanation 1

布に $ 100 $ 矣藏が $ 5 $ 枚入っているとき、合金~は $ 500 $ 窑摔胜辘蓼埂９胜素布の中の合金~は $ X=500 $ 窑摔胜辘Δ毪郡帷`Yes` を出力します。

### Sample Explanation 3

布の中には $ 100 $ 矣藏が $ 1 $ 枚以上入っていることに注意してください。
