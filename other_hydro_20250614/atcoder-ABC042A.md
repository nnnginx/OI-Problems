## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc042/tasks/abc042_a

日本のFる美しいリズムとして、五七五というものがあります。いろはちゃんは五七五が大好きです。

$ 3 $ つの文のKびのLさがそれぞれ $ 5,7,5 $ となるようにこの番でKんでいるとき、その $ 3 $ つの文のKびは五七五であると言います。

Kび替えたい $ 3 $ つの文のLさを表す整数 $ A,B,C $ が与えられるので、それらの文をKび替えて五七五にできるか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $

## 输出格式
文のKびを五七五にすることができるなら `YES` 、そうでないなら `NO` を出力せよ。

## 题目大意
给出三个数$A,B,C$，$1≤A,B,C≤10$，判断$A,B,C$是否能重新排列为$575$。

翻译由[$@PCT2506$](https://www.luogu.com.cn/user/793625)提供

```input1
5 5 7
```

```output1
YES
```

```input2
7 7 5
```

```output2
NO
```

## 提示
### 制s

- $ 1QA,B,CQ10 $

### Sample Explanation 1

与えられる文のLさはそれぞれ $ 5,5,7 $ であり、$ 5,7,5 $ となるように文をKび替えることができます。したがって、文のKびを五七五にすることは可能といえます。
