## 题目描述
[problemUrl]: https://atcoder.jp/contests/cf17-relay-open/tasks/relay2_a

*ButCoder株式会社* は、プログラミングコンテストサイト「*ButCoder*」を\婴筏皮い蓼埂￥长违单ぅ趣扦稀ユ`ザ`にはレ`ティングという技量を表す整数が与えられ、そのはユ`ザ`がコンテストに参加するたびに婴筏蓼埂Ｐ乱ユ`ザ`のレ`ティングの初期は $ 0 $ であり、レ`ティングが $ K $ 以上に_したユ`ザ`は *皆* と呼ばれます。なお、レ`ティングはになることもあります。

低颏んというユ`ザ`が ButCoder に新たに登hしました。彼のレ`ティングは、彼が奇数回目に参加するコンテスト（$ 1 $ 回目、$ 3 $ 回目、$ 5 $ 回目$ … $）では盎 $ A $ 加し、偶数回目に参加するコンテスト（$ 2 $ 回目、$ 4 $ 回目、$ 6 $ 回目$ … $）では盎 $ B $ p少することが予yされます。

この予yによると、彼が初めて皆护摔胜毪韦虾位丐违偿螗匹攻趣瞬渭婴筏恐贬幛扦筏绀Δ、もしくは彼は永hに皆护摔胜欷胜い扦筏绀Δ？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ K $ $ A $ $ B $

## 输出格式
低颏んが永hに皆护摔胜椁胜い扔yされる龊悉稀`-1` と出力せよ。そうでない龊悉稀⒈摔初めて皆护摔胜毪蓼扦违偿螗匹攻趣位厥の予yを出力せよ。

## 题目大意
一只乌龟掉到井里了，井深 $K$ 米。

第奇数天他每天爬 $A$ 米。

第偶数天他每天下落 $B$ 米。

问他第几天可以离开井（离不开输出 $-1$）。

```input1
4000 2000 500
```

```output1
5
```

```input2
4000 500 2000
```

```output2
-1
```

```input3
1000000000000000000 2 1
```

```output3
1999999999999999997
```

## 提示
### 制s

- $ 1\ <\ =\ K,\ A,\ B\ <\ =\ 10^{18} $
- 入力はすべて整数である。

### Sample Explanation 1

低颏んがコンテストに参加するたびに、彼のレ`ティングは $ 0 $ → $ 2000 $ → $ 1500 $ → $ 3500 $ → $ 3000 $ → $ 5000 $ → … と婴工毪长趣予yされます。彼のレ`ティングが $ 4000 $ 以上に到_するのは、$ 5 $ 回目の参加の直後です。

### Sample Explanation 2

低颏んがコンテストに参加するたびに、彼のレ`ティングは $ 0 $ → $ 500 $ → $ -1500 $ → $ -1000 $ → $ -3000 $ → $ -2500 $ → … と婴工毪长趣予yされます。彼が皆护摔胜毪长趣嫌肋hにありません。

### Sample Explanation 3

入力されるや出力すべきは $ 32 $ bit 整数にГ蓼椁胜い长趣あります。
