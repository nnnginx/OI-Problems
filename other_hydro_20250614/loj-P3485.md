## 题目描述

**译自 [COCI 2020/2021 Contest #6](https://hsin.hr/coci/archive/2020_2021/contest6_tasks.pdf) T4「Geometrija」**

若两条线段 $\overline{AB}$，$\overline{CD}$，有一交点 $X$ 且 $X$ 不为 $A,B,C,D$ 中的一点，则称这两条线段相交。

给定平面上的 $n$ 个点，保证任意三点不共线。将这些点两两连线会得到 $\frac{n(n-1)}{2}$ 条线段，求出这些线段中，不与其他线段相交的线段个数。



## 输入格式

第一行为一个整数 $n$。

接下来 $n$ 行，一行两个整数 $x_i,y_i$，表示第 $i$ 个点的坐标为 $(x_i,y_i)$。

## 输出格式

仅一行一个整数，表示不与其他线段相交的线段个数。

```input1
4
1 1
-1 1
-1 -1
1 -1
```

```output1
4
```

```input2
4
-1 -1
1 -1
0 1
0 0
```

```output2
6
```

## 数据范围与提示

对于所有子任务，有 $3\le n\le 10^3$，$-10^9\le x_i,y_i\le 10^9$。

| 子任务编号 |  特殊限制  |   分值   |
| :--------: | :--------: | :------: |
|    $1$     | $n\le 40$  | $20/110$ |
|    $2$     | $n\le 200$ | $30/110$ |
|    $3$     |     无     | $60/110$ |
