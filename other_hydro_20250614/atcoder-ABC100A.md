## 题目大意
**题目大意**

一个圆形蛋糕分成16等份后,E869120要吃A块,square1001要吃B块（1<=A,B<=16），但是同一个人不能同时拿相邻的两个蛋糕.那么，他们2个人能拿到想吃的数量的蛋糕吗？

**输入**

两个数A和B，中间用空格隔开

**输出**

一行，能则输出“Yay!”，不能则输出“:(”.

```input1
5 4
```

```output1
Yay!
```

```input2
8 8
```

```output2
Yay!
```

```input3
11 4
```

```output3
:(
```
