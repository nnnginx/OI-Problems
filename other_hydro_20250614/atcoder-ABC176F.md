## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc176/tasks/abc176_f

$ 1 $ 以上 $ N $ 以下の整数のうち一つがかれた $ 3N $ 枚のカ`ドが左右一列にKんでいます。 左から $ i $ 番目のカ`ドにかれた整数は $ A_i $ です。

以下の操作を $ N-1 $ 回Rり返します。

- 左から $ 5 $ 枚のカ`ドを好きなにKび替える。その後、左から $ 3 $ 枚のカ`ドを取り除く。このとき、その $ 3 $ 枚のカ`ドにかれた整数がすべて等しければ $ 1 $ 点を得る。

$ N-1 $ 回の操作の後、残った $ 3 $ 枚のカ`ドにかれた整数がすべて等しければ追加で $ 1 $ 点を得ます。

得られる得点の最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \cdots $ $ A_{3N} $

## 输出格式
得られる得点の最大を出力せよ。

## 题目大意
hhoppitree 有 $3n$ 张卡片，其中每张卡片上都写着 $1\sim n$ 中的一个数，他会重复以下操作 $n-1$ 次：

- 将最左侧的 $5$ 张牌**任意排列**，排列后，删去最左侧的 $3$ 张牌，如果这三张牌上写着同样的数，hhoppitree 可以获得 $1$ 分。

最后，如果剩余的 $3$ 张牌上的数字一样，那么他还可以额外得到 $1$ 分。

现在，hhoppitree 想要知道，他得到的分数最高是多少。

```input1
2
1 2 1 2 2 1
```

```output1
2
```

```input2
3
1 1 2 2 3 3 3 2 1
```

```output2
1
```

```input3
3
1 1 2 2 2 3 3 3 1
```

```output3
3
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2000 $
- $ 1\ \leq\ A_i\ \leq\ N $

### Sample Explanation 1

左から $ 5 $ 枚のカ`ドをKべ替え、カ`ドにかれた整数が左からに $ 2\ 2\ 2\ 1\ 1\ 1 $ となるようにします。 左から $ 3 $ 枚のカ`ドを取り除き、このときこれら $ 3 $ 枚のカ`ドにかれた整数はすべて $ 2 $ で等しいので $ 1 $ 点を得ます。 カ`ドにかれた整数は左からに $ 1\ 1\ 1 $ となります。 残った $ 3 $ 枚のカ`ドにかれた整数はすべて $ 1 $ でこれも等しいので $ 1 $ 点を得ます。 合得点は $ 2 $ 点となり、これが最高です。
