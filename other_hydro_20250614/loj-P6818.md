## 题目描述

给你一个 $n$ 行 $m$ 列的方阵，你有 $1\sim n$ 这 $n$ 个数各 $m$ 个，要放进方阵中使得数 $i$ 不在第 $i$ 行，令方案数为 $F(n,m)$。

求 $\displaystyle \bigoplus _{i=1}^{n} (F(i,m)\bmod 998244353)$。

## 输入格式

一行两个正整数 $n,m$。

## 输出格式

一行一个整数表示答案。

```input1
3 2
```

```output1
11
```

## 数据范围与提示

| 子任务编号 |  分值  |        $nm$        | 特殊性质  |
| :---: | :--: | :----------------: | :---: |
|   1   |  $10$  |       $10^6$       | $m=1$ |
|   2   |  $10 $ |    $\leq 5000$     |   无   |
|   3   |  $40$  |    $\leq 10^5$     |   无   |
|   5   |  $40 $ |    $\leq 10^6$     |   无   |
