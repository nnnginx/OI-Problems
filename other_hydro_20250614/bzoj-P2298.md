## 题目描述

一次考试共有 $n$ 个人参加，第 $i$ 个人说：“有 $a_i$ 个人分数比我高，$b_i$ 个人分数比我低。”

问最少有几个人没有说真话（可能有相同的分数）。

## 输入格式

第一行一个整数 $n$。

接下来 $n$ 行，每行两个整数 $a_i,b_i$。

## 输出格式

一个整数，表示最少有几个人说谎。

```input1
3
2 0
0 2
2 2
```

```output1
1
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\leq n\leq 10^5$，$0\leq a_i,b_i\leq n$。
