## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc111/tasks/abc111_b

\蚓は，AtCoder Beginner Contest (ABC) にまだ参加したことがありません．

次に行われる ABC は第 $ N $ 回です． \蚓は，初めて参加する ABC を第 $ x $ 回としたときに，$ x $ の十M法表でのすべての桁の数字が同じであるようにしたいです．

\蚓が初めて参加する ABC としてふさわしいもののうち，最も早いものは第何回でしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $

## 输出格式
\蚓が初めて参加する ABC としてふさわしいもののうち，最も早いものは第何回かを出力せよ．

## 题目大意
求大于等于 $n\ (100\leq n \leq 999)$ 的三位数字都相同的最小三位数

```input1
111
```

```output1
111
```

```input2
112
```

```output2
222
```

```input3
750
```

```output3
777
```

## 提示
### 制s

- $ 100\ \leq\ N\ \leq\ 999 $
- $ N $ は整数

### Sample Explanation 1

次に行われる ABC は第 $ 111 $ 回です． これは\蚓が初めて参加する ABC としてふさわしいです．

### Sample Explanation 2

次に行われる ABC は第 $ 112 $ 回です．そのため，第 $ 111 $ 回の ABC にはもう参加することはできません． \蚓が初めて参加する ABC としてふさわしいもののうち，最も早いものは第 $ 222 $ 回です．
