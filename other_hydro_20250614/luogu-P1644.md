在爱与愁的故事第一弹第三章出来前先练练四道基本的回溯/搜索题吧……


## 题目描述
中国象棋半张棋盘如图 $1$ 所示。马自左下角 $(0,0)$ 向右上角 $(m,n)$ 跳。规定只能往右跳，不准往左跳。比如图 $1$ 中所示为一种跳行路线，并将路径总数打印出来。

![](https://cdn.luogu.com.cn/upload/image_hosting/8gkasdvs.png)


## 输入格式
只有一行：两个数 $n$，$m$。


## 输出格式
只有一个数：总方案数 $total$。


```input1
4 8
```

```output1
37
```

## 提示
对于 $100\%$ 的数据：$n, m\leq 18$
