## 题目描述
[problemUrl]: https://atcoder.jp/contests/code-festival-2017-qualc/tasks/code_festival_2017_qualc_c

英小文字のみからなる文字列 $ s $ があります。 すぬけ君は、次の操作をRり返し行うことができます。

- $ s $ の任意の位置 (先^および末尾を含む) をひとつxび、英小文字 `x` をひとつ啡毪工搿

すぬけ君の目摔稀$ s $ を回文にすることです。 すぬけ君の目摔_成可能か判定し、_成可能ならば必要な操作回数の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ s $

## 输出格式
すぬけ君の目摔_成可能ならば、必要な操作回数の最小を出力せよ。 _成不可能ならば、代わりに `-1` を出力せよ。

## 题目大意
输入一个字符串S，您可以重复执行以下操作：

在S的任意位置（包含开头和结尾），插入一个小写字母 x ， 使其变成回文的字符串。

请输出所需的最少操作次数 ，如果不能变为回文 ，输出 -1 。

输入格式：

输入文件仅有一行，一个仅包含小写字母的字符串S。

输出格式：

如果目标可实现 ，则输出所需的最少操作次数。如果不行，请输出-1。

```input1
xabxa
```

```output1
2
```

```input2
ab
```

```output2
-1
```

```input3
a
```

```output3
0
```

```input4
oxxx
```

```output4
3
```

## 提示
### 注

回文とは、前後を反しても浃铯椁胜の淖至肖韦长趣扦埂 例えば、`a`, `aa`, `abba`, `abcba` は回文ですが、`ab`, `abab`, `abcda` は回文ではありません。

### 制s

- $ 1\ \leq\ |s|\ \leq\ 10^5 $
- $ s $ は英小文字のみからなる。

### Sample Explanation 1

例えば、次のように操作を行えばよいです (新しく啡毪丹欷 `x` は太字で表されています)。 xabxa → xa\*\*x\*\*bxa → xaxbxa\*\*x\*\*

### Sample Explanation 2

どのように操作を行っても、$ s $ を回文にできません。

### Sample Explanation 3

$ s $ は最初から回文です。

### Sample Explanation 4

例えば、次のように操作を行えばよいです。 oxxx → \*\*x\*\*oxxx → \*\*x\*\*xoxxx → \*\*x\*\*xxoxxx
