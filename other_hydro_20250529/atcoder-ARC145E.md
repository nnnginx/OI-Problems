## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc145/tasks/arc145_e

Lさ $ N $ の非整数列 $ A=(A_1,A_2,\ldots,A_{N}),B=(B_1,B_2,\ldots,B_{N}) $ が与えられます。

数列 $ A $ にし以下の操作を $ 70000 $ 回以下行うことで $ A $ を $ B $ に一致させられるか判定し、可能な龊悉gHに操作手を一つ示してください。

- 整数 $ K\ (1\le\ K\ \le\ N) $ をxぶ。全ての $ i\ (2\leq\ i\ \leq\ K) $ について、$ A_i $ のを $ A_{i-1}\ \oplus\ A_i $ で置きQえる。この置きQえは全ての $ i\ (2\leq\ i\ \leq\ K) $ にして同rに行う。

ただしここで、$ \oplus $ はビットg位 $ \mathrm{XOR} $ 演算を表します。

 ビットg位 $ \mathrm{XOR} $ 演算とは非整数 $ A,B $ のビットg位 $ \mathrm{XOR} $ 演算、$ A\oplus\ B $ は、以下のように定xされます。

- $ A\oplus\ B $ を二M表したHの $ 2^k\ (k\geq\ 0) $ の位の数は、$ A,B $ を二M表したHの $ 2^k $ の位の数のうち一方のみが $ 1 $ であれば $ 1 $、そうでなければ $ 0 $ である。

例えば、$ 3\oplus\ 5\ =\ 6 $ となります（二M表すると: $ 011\oplus\ 101\ =\ 110 $)。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $ $ B_1 $ $ B_2 $ $ \ldots $ $ B_N $

## 输出格式
$ 70000 $ 回以下の操作で $ A $ を $ B $ に一致させられない龊稀`No` と出力せよ。一致させられる龊稀⒉僮骰厥を $ M $ 回とし、$ i $ 回目の操作でxんだ整数を $ K_i $ として以下の形式で出力せよ。

> Yes $ M $ $ K_1 $ $ K_2 $ $ \ldots $ $ K_M $

条件を氦郡菇猡}数存在する龊稀どれを出力しても正解とみなされる。

## 题目大意
给定长度为 $n$ 的非负整数列 $a=(a_1,a_2,\cdots, a_n),b=(b_1,b_2,\cdots, b_n)$。  

判断能否对数列 $a$ 进行以下操作最多 $70000$ 次，使得数列 $a$ 与 $b$ 一致，如果能，给出一个可行的操作步骤。
- 操作：选择整数 $k(1\le k\le n)$。对于所有 $i(2\le i\le k)$，使 $a_i\gets a_{i-1}\oplus a_i$。

其中，$\oplus$ 表示按位 $\mathrm{XOR}$（异或）运算。

```input1
3
1 2 0
1 2 3
```

```output1
Yes
2
2 3
```

```input2
2
10 100
1 0
```

```output2
No
```

```input3
2
1152921504606846975 0
1152921504606846975 0
```

```output3
Yes
0
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 1000 $
- $ 0\ \leq\ A_i,\ B_i\ <\ 2^{60} $
- 入力は全て整数

### Sample Explanation 1

この出力例では、操作によって数列 $ A $ は以下のように浠します。 - 初期状B：$ A=(1,\ 2,\ 0) $ - $ 1 $ 回目の操作後：$ A=(1,\ 3,\ 0) $ - $ 2 $ 回目の操作後：$ A=(1,\ 2,\ 3) $ $ 2 $ 回の操作後、$ A $ と $ B $ は一致しているのでこの出力は条件を氦郡筏蓼埂
