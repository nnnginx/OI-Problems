## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc268/tasks/abc268_b

英小文字のみからなる $ 2 $ つの文字列 $ S,\ T $ が与えられます。 $ S $ が $ T $ の接^辞かどうかを判定してください。

接^辞とはLさ $ N $ の文字列 $ T_1T_2\ldots\ T_N $ の接^辞とは、 $ 0\ \leq\ i\ \leq\ N $ を氦郡工る整数 $ i $ によって、$ T $ の先^ $ i $ 文字目までの文字列 $ T_1T_2\ldots\ T_i $ として表される文字列です。例えば、$ T\ = $ abc のとき、$ T $ の接^辞は、空文字列、a 、ab 、abc の $ 4 $ つです。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $ $ T $

## 输出格式
$ S $ が $ T $ の接^辞である龊悉 `Yes` を、そうでない龊悉 `No` を出力せよ。 ジャッジは英小文字と英大文字を密に区eすることに注意せよ。

## 题目大意
#### 题意 

给出两个由小写英文字母组成的字符串 $S$ 和 $T$，判断 $S$ 是否是 $T$ 的前缀。

> #### 什么是前缀?  
> 一个字符串 $S$ 被称为一个长度为 $N$ 的字符串 $T_{1},T_{2}...T_{n}$ 的前缀，  
当且仅当 $S$ 是由 $T_{1},T_{2}...T_{n}$ 的前 $i$ $(1\leq i \leq N)$ 个字母组成的。

#### 数据范围

$S$ 和 $T$ 是长度介于 $1$ 和 $100$ 之间的字符串，且只由小写英文字母组成。

---

#### 输入格式

通过标准输入以以下格式读入数据：
```
S
T
```

#### 输出格式

当 $S$ 是 $T$ 的前缀时输出 `Yes`； 否则输出 `No`。注意区分大小写。

```input1
atco
atcoder
```

```output1
Yes
```

```input2
code
atcoder
```

```output2
No
```

```input3
abc
abc
```

```output3
Yes
```

```input4
aaaa
aa
```

```output4
No
```

## 提示
### 制s

- $ S $ と $ T $ はそれぞれ英小文字のみからなるLさが $ 1 $ 以上 $ 100 $ 以下の文字列

### Sample Explanation 1

`atco` は `atcoder` の接^辞です。よって、`Yes` を出力します。

### Sample Explanation 2

`code` は `atcoder` の接^辞ではありません。よって、`No` を出力します。

### Sample Explanation 3

文字列全体もその文字列の接^辞であることに注意してください。
