## 题目描述

给定一个字符串 $S = s_1s_2s_3\dots s_n$，如果在串 $S$ 中，子串 $T(|T| = n)$ 为所有长度为 $n$ 的 $S$ 的字串中最小的（字典序比较）, 则称 $T$ 为「奇怪的字串」。你的任务就是找出这个字符串。
## 输入格式

读入两行，第一行为 $n$。

第二行为字符串 $S$。
## 输出格式

将「奇怪的字串」$T$ 输出。
## 样例输入

```plain
10
asdfasdfas
```

## 样例输出

```plain
asasdfasdf
```

## 数据规模与约定

对于 $100\%$ 的数据，保证 $n\leq 10^7$。

给定的字符串中的字符保证在 $33 \sim 254$（ASCII 码）之间。
