## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc114/tasks/abc114_c

整数 $ N $ が与えられます。$ 1 $ 以上 $ N $ 以下の整数のうち、*七五三数* は何あるでしょうか？

ここで、七五三数とは以下の条件を氦郡拐の整数です。

- 十M法で表したとき、数字 `7`, `5`, `3` がそれぞれ $ 1 $ 回以上Fれ、これら以外の数字はFれない。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ 1 $ 以上 $ N $ 以下の七五三数の数を出力せよ。

## 题目大意
你有一个整数 $N$。在 $1$ 到 $N$（含）之间的整数中，有多少个 _Shichi-Go-San_ 数字？（字面意思是 _七五三_ 数字）

此处，_Shichi-Go-San_ 数是满足以下条件的正整数：

- 用十进制数写数字时，数字 `7`、`5` 和 `3` 至少出现一次，而其他数字从不出现。


```input1
575
```

```output1
4
```

```input2
3600
```

```output2
13
```

```input3
999999999
```

```output3
26484
```

## 提示
### 制s

- $ 1\ \leq\ N $
- $ N $ は整数である。

### Sample Explanation 1

$ 575 $ 以下の七五三数は、$ 357,\ 375,\ 537,\ 573 $ の $ 4 $ です。

### Sample Explanation 2

$ 3600 $ 以下の七五三数は、上の $ 4 $ と $ 735,\ 753,\ 3357,\ 3375,\ 3537,\ 3557,\ 3573,\ 3575,\ 3577 $ の $ 13 $ です。
