## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc148/tasks/abc148_e

$ 0 $ 以上の整数 $ n $ にし、 $ f(n) $ を次のように定xします。

- $ f(n)\ =\ 1 $ ($ n\ <\ 2 $ のとき)
- $ f(n)\ =\ n\ f(n-2) $ ($ n\ >\ =\ 2 $ のとき)

整数 $ N $ が与えられます。$ f(N) $ を $ 10 $ M法で表したrに末尾に何の $ 0 $ がAくかを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ f(N) $ を $ 10 $ M法で表したrの末尾の $ 0 $ の数を出力せよ。

## 题目大意
输入一个n。

如果n是奇数那么输出0。

如果n是偶数那么输出$(\frac{n}{2})!$末尾零的个数除以二得到的商。

```input1
12
```

```output1
1
```

```input2
5
```

```output2
0
```

```input3
1000000000000000000
```

```output3
124999999999999995
```

## 提示
### 制s

- $ 0\ <\ =\ N\ <\ =\ 10^{18} $

### Sample Explanation 1

$ f(12)\ =\ 12\ ×\ 10\ ×\ 8\ ×\ 6\ ×\ 4\ ×\ 2\ =\ 46080 $ なので、末尾の $ 0 $ の数は $ 1 $ です。

### Sample Explanation 2

$ f(5)\ =\ 5\ ×\ 3\ ×\ 1\ =\ 15 $ なので、末尾の $ 0 $ の数は $ 0 $ です。
