## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc282/tasks/abc282_a

整数 $ K $ が与えられます。

英大文字を `A` からNに $ K $ げて得られる文字列を答えてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ K $

## 输出格式
答えを出力せよ。

## 题目大意
给定一个整数 $k$，保证 $1 \leq k \leq 26$。

请你输出从 `A` 开始的按升序排列的前 $k$ 个大写英文字母组成的字符串。

```input1
3
```

```output1
ABC
```

```input2
1
```

```output2
A
```

## 提示
### 制s

- $ K $ は $ 1 $ 以上 $ 26 $ 以下の整数
 
### Sample Explanation 1

英大文字は `A` からNに `A`, `B`, `C`, ... です。 `A` からNに $ 3 $ げて得られる文字列は `ABC` です。
