## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc274/tasks/abc274_b

k $ H $ マス、横 $ W $ マスのグリッドがあります。上から $ i $ 行目、左から $ j $ 列目のマスを $ (i,j) $ と表します。  
 各マスの状Bは文字 $ C_{i,j} $ で表されます。$ C_{i,j} $ が `.` ならば $ (i,\ j) $ には何も置かれておらず、 `#` ならば箱が $ 1 $ 置かれています。

$ 1\ \leq\ j\ \leq\ W $ を氦郡拐数 $ j $ にして、整数 $ X_j $ を次のように定xします。

- $ j $ 列目に置かれている箱の数。言いQえると、$ C_{i,j} $ が `#` であるような整数 $ i $ の数。

$ X_1,\ X_2,\ \dots,\ X_W $ をすべて求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ H $ $ W $ $ C_{1,1}C_{1,2}\dots\ C_{1,W} $ $ C_{2,1}C_{2,2}\dots\ C_{2,W} $ $ \vdots $ $ C_{H,1}C_{H,2}\dots\ C_{H,W} $

## 输出格式
$ X_1,\ X_2,\ \dots,\ X_W $ を以下の形式に兢盲瞥隽Δ护琛

> $ X_1 $ $ X_2 $ $ \dots $ $ X_W $

## 题目大意
一个 $H\times W$ 的矩阵，输出每一列字符`#`的个数。

```input1
3 4
#..#
.#.#
.#.#
```

```output1
1 2 0 3
```

```input2
3 7
.......
.......
.......
```

```output2
0 0 0 0 0 0 0
```

```input3
8 3
.#.
###
.#.
.#.
.##
..#
##.
.##
```

```output3
2 7 4
```

```input4
5 47
.#..#..#####..#...#..#####..#...#...###...#####
.#.#...#.......#.#...#......##..#..#...#..#....
.##....#####....#....#####..#.#.#..#......#####
.#.#...#........#....#......#..##..#...#..#....
.#..#..#####....#....#####..#...#...###...#####
```

```output4
0 5 1 2 2 0 0 5 3 3 3 3 0 0 1 1 3 1 1 0 0 5 3 3 3 3 0 0 5 1 1 1 5 0 0 3 2 2 2 2 0 0 5 3 3 3 3
```

## 提示
### 制s

- $ 1\ \leq\ H\ \leq\ 1000 $
- $ 1\ \leq\ W\ \leq\ 1000 $
- $ H,\ W $ は整数
- $ C_{i,\ j} $ は `.` または `#`

### Sample Explanation 1

$ 1 $ 列目の箱が置かれているマスは $ (1,\ 1) $ の $ 1 $ ヵ所です。よって $ X_1\ =\ 1 $ です。 $ 2 $ 列目の箱が置かれているマスは $ (2,\ 2),\ (3,\ 2) $ の $ 2 $ ヵ所です。よって $ X_2\ =\ 2 $ です。 $ 3 $ 列目の箱が置かれているマスは存在しません。よって $ X_3\ =\ 0 $ です。 $ 4 $ 列目の箱が置かれているマスは $ (1,\ 4),\ (2,\ 4),\ (3,\ 4) $ の $ 3 $ ヵ所です。よって $ X_4\ =\ 3 $ です。 よって $ (X_1,\ X_2,\ X_3,\ X_4)\ =\ (1,\ 2,\ 0,\ 3) $ が答えとなります。

### Sample Explanation 2

箱が置かれているマスが存在しない龊悉猡ります。
