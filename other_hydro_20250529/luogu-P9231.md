## 题目描述
给定 $L,R$，问 $L \leq x \leq R$ 中有多少个数 $x$ 满足存在整数 $y,z$ 使得 $x=y^2-z^2$。

## 输入格式
输入一行包含两个整数 $L,R$，用一个空格分隔。

## 输出格式
输出一行包含一个整数满足题目给定条件的 $x$ 的数量。

```input1
1 5
```

```output1
4
```

## 提示
#### 【样例说明】

* $1=1^2-0^2$
* $3=2^2-1^2$
* $4=2^2-0^2$
* $5=3^2-2^2$

#### 【评测用例规模与约定】

对于 $40 \%$ 的评测用例，$L,R \leq 5000$；

对于所有评测用例，$1 \leq L \leq R \leq 10^9$。
