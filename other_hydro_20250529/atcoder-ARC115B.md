## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc115/tasks/arc115_b

$ N $ 行 $ N $ 列の非整数を成分とする行列 $ C $ が与えられます。すべての $ (i,j) $ について $ C_{i,j}=A_i+B_j $ を氦郡工瑜Δ史秦整数列 $ A_1,A_2,\ldots,A_N $ と $ B_1,B_2,\ldots,B_N $ のMが存在するか判定し、存在するなら一つ出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ C_{1,1} $ $ C_{1,2} $ $ \ldots $ $ C_{1,N} $ $ C_{2,1} $ $ C_{2,2} $ $ \ldots $ $ C_{2,N} $ $ : $ $ C_{N,1} $ $ C_{N,2} $ $ \ldots $ $ C_{N,N} $

## 输出格式
- 条件を氦郡$ A $,$ B $ のMが存在しない龊

一行目に`No` と出力せよ。

 ```
No
```

- 条件を氦郡$ A $,$ B $ のMが存在する龊

一行目に `Yes` と出力せよ。 二行目には各要素を空白で区切って数列 $ A $ を出力せよ。 三行目には各要素を空白で区切って数列 $ B $ を出力せよ。

条件を氦郡菇猡}数存在する龊悉稀どれを出力してもよい。

> Yes $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $ $ B_1 $ $ B_2 $ $ \ldots $ $ B_N $

## 题目大意
给定 $N \times N$ 的矩阵 $C$。对于所有数对 $(i,j)$，满足 $C _ {i,j} = A _ i + B _ j$。$A, B, C$ 中元素均为非负整数。

问是否存在数列 $A, B$。

- 若存在则第一行输出 `Yes`，第二行输出 $N$ 个整数表示 $A$，第三行输出 $n$ 个整数表示 $B$。
- 否则输出 `No`。

感谢 @ChenChangxu 提供的翻译

```input1
3
4 3 5
2 1 3
3 2 4
```

```output1
Yes
2 0 1
2 1 3
```

```input2
3
4 3 5
2 2 3
3 2 4
```

```output2
No
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 500 $
- $ 0\ \leq\ C_{i,j}\ \leq\ 10^9 $

### Sample Explanation 1

$ A,B $ は非整数列であることに注意してください。
