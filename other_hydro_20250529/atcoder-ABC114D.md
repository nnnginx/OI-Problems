## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc114/tasks/abc114_d

整数 $ N $ が与えられます。$ N! $ $ (=\ 1\ \times\ 2\ \times\ ...\ \times\ N) $ のs数のうち、*七五数* は何あるでしょうか？

ここで、七五数とはs数をちょうど $ 75 $ 持つ正の整数です。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N! $ のs数であるような七五数の数を出力せよ。

## 题目大意
$N$的阶乘的因数当中，有多少个数刚好有$75$个因数

感谢$@RioBlu$的翻译

```input1
9
```

```output1
0
```

```input2
10
```

```output2
1
```

```input3
100
```

```output3
543
```

## 提示
### 注

正の整数 $ A $ が正の整数 $ B $ を割り切るとき、$ A $ を $ B $ の *s数* といいます。 例えば、$ 6 $ のs数は $ 1,2,3,6 $ の $ 4 $ です。

### 制s

- $ 1\ \leq\ N\ \leq\ 100 $
- $ N $ は整数である。

### Sample Explanation 1

$ 9!\ =\ 1\ \times\ 2\ \times\ ...\ \times\ 9\ =\ 362880 $ のs数に七五数はありません。

### Sample Explanation 2

$ 10!\ =\ 3628800 $ のs数のうち、七五数であるのは $ 32400 $ の $ 1 $ です。
