## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc084/tasks/arc084_c

有限整数列大辞典(Finite Encyclopedia of Integer Sequences)には、 $ 1 $ 以上 $ K $ 以下の整数からなる、Lさ $ 1 $ 以上 $ N $ 以下の整数列がすべてdっています。

有限整数列大辞典にdっている整数列の数が $ X $ あるとするとき、その中で辞で $ X/2 $ (小数点以下切り上げ)番目のものを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ K $ $ N $

## 输出格式
有限整数列大辞典にdっている整数列の数が $ X $ あるとするとき、 その中で辞で $ X/2 $ (小数点以下切り上げ)番目のものを、ごとに空白で区切って出力せよ。

## 题目大意
对于满足长度不超过$N$,每一个元素都满足$1\leq a_i \leq K$的序列$a$,称其为合法序列  
设现在有$X$个合法序列,请给出按照字典序排序之后排在第$\frac{X}{2}$(四舍五入)的序列

```input1
3 2
```

```output1
2 1
```

```input2
2 4
```

```output2
1 2 2 2
```

```input3
5 14
```

```output3
3 3 3 3 3 3 3 3 3 3 3 3 2 2
```

## 提示
### 制s

- $ 1\ \leq\ K,N\ \leq\ 3\ ×\ 10^5 $
- $ N,K $ は整数である

### Sample Explanation 1

有限整数列大辞典にdっている整数列は、$ (1),(1,1),(1,2),(1,3),(2),(2,1),(2,2),(2,3),(3),(3,1),(3,2),(3,3) $ の $ 12 $ です。 この中で辞で $ 12/2\ =\ 6 $ 番目のものは、$ (2,1) $ です。
