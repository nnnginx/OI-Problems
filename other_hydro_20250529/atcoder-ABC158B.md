## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc158/tasks/abc158_b

高蚓は青と赤の $ 2 $ 色のボ`ルを持っており、これらを一列にKべようとしています。

初め、列にボ`ルはありません。

根荬韦る高蚓は、次の操作を $ 10^{100} $ 回Rり返します。

- 列の末尾に、$ A $ の青いボ`ルを加える。その後、列の末尾に $ B $ の赤いボ`ルを加える。

こうしてできる列の先^から $ N $ のボ`ルのうち、青いボ`ルの数はいくつでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $

## 输出格式
列の先^から $ N $ のボ`ルのうち、青いボ`ルの数を出力せよ。

## 题目大意
高桥君要将蓝球和红球按一定次序排成一行，现在此队列中没有球。

每次他将在队列末尾放上 $A$ 个蓝球，再放上 $B$ 个红球。此操作将循环 $10^{100}$ 次。

输入整数 $N,A,B$ ，输出前 $N$ 个球中的蓝球数目。

```input1
8 3 4
```

```output1
4
```

```input2
8 0 4
```

```output2
0
```

```input3
6 2 4
```

```output3
2
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^{18} $
- $ A,\ B\ \geq\ 0 $
- $ 0\ <\ A\ +\ B\ \leq\ 10^{18} $
- 入力は全て整数である

### Sample Explanation 1

青いボ`ルを`b`、赤いボ`ルを `r`で表すと、列の先^から $ 8 $ のボ`ルは `bbbrrrrb` であるので、このうち青いボ`ルは $ 4 $ です。

### Sample Explanation 2

そもそも赤いボ`ルしかKんでいません。

### Sample Explanation 3

`bbrrrr` のうち青いボ`ルは $ 2 $ です。
