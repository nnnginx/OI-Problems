## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc157/tasks/arc157_f

`X`, `Y` からなるLさ $ N $ の文字列 $ S,\ T $ が与えられます． 各 $ i\ =\ 1,\ 2,\ \dots,\ N $ にvして，$ S $ の $ i $ 文字目と $ T $ の $ i $ 文字目を入れ替えるかどうかを自由にxべます． このとき，Y果として得られる文字列のMはのべ $ 2^N $ 通りありますが，そのいずれかの共通部分列（BAとは限らない）となる文字列のうち最Lのものを求めてください． ただし，そのような文字列が}数ある龊希そのうち辞で最初にFれるものを求めてください．

  共通部分列とは 文字列 $ S $ の**部分列**とは，$ S $ から $ 0 $ 以上の文字を削除して，残った文字を元のでKべて得られる文字列のことをいいます． 文字列 $ S,\ T $ の**共通部分列**とは，$ S $ と $ T $ のどちらの部分列でもあるような文字列のことをいいます． （出力例 1 のh明も参考にしてください．）

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ S $ $ T $

## 输出格式
入れ替え後の文字列のMの共通部分列としてあり得る最Lの文字列のうち，辞で最初にFれるものを出力せよ．

## 题目大意
给定两个长度为 $n$，且仅包含 `X` 或 `Y` 的字符串 $S,T$。对于每个整数 $i\in [1,n]$，你都可以选择交换 $S_i,T_i$ 或者不交换。

请你最大化交换完后的最长公共子序列长度，并输出一组合法的最长公共子序列。如果有多种合法答案，请输出 **字典序最小的**。$n\le 50$。

```input1
3
XXX
YYY
```

```output1
XY
```

```input2
1
X
Y
```

```output2

```

```input3
4
XXYX
YYYY
```

```output3
XYY
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 50 $
- $ S,\ T $ はそれぞれ `X`, `Y` からなるLさ $ N $ の文字列である．
 
### Sample Explanation 1

\- どの文字も入れ替えない龊希`XXX` と `YYY` の共通部分列は空文字列のみです． - $ 1 $ 文字目だけを入れ替えた龊希`YXX` と `XYY` の共通部分列は，空文字列, `X`, `Y` です． - $ 2 $ 文字目だけを入れ替えた龊希`XYX` と `YXY` の共通部分列は，空文字列, `X`, `Y`, `XY`, `YX` です． - $ 3 $ 文字目だけを入れ替えた龊希`XXY` と `YYX` の共通部分列は，空文字列, `X`, `Y` です． $ 2 $ 文字以上の入れ替えは，$ S $ と $ T $ 自体を入れ替えて考えれば上のいずれかに当します． したがって，共通部分列としてあり得る最Lの文字列は `XY` と `YX` であり，辞で最初にFれる `XY` が答えとなります．

### Sample Explanation 2

答えが空文字列となることもあります．

### Sample Explanation 3

たとえば $ 2 $ 文字目だけを入れ替えた龊悉 `XYY` が共通部分列となります． よりLい文字列，あるいは，同じLさであって辞でより早くFれる文字列は，どのように入れ替えたとしても共通部分列にはなり得ないため，これが答えとなります．
