## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc285/tasks/abc285_h

整数 $ N,K $ とLさ $ K $ の数列 $ E $ が与えられます。  
Lさ $ N $ の正整数列であって、以下の条件を全て氦郡工猡韦尉t数を $ \color{red}{10^9+7} $ で割った余りを求めてください。

- どの要素も平方数でない
- 全要素のeが $ \displaystyle\ \prod_{i=1}^{K}\ p_i^{E_i} $ である

但し、

- $ p_i $ は小さい方から $ i $ 番目の素数を指すものとします。
- ある $ 2 $ つのLさが等しい正整数列 $ A,B $ について、 $ A $ の $ i $ 目と $ B $ の $ i $ 目がなるような整数 $ i $ が存在するrに限り、 $ A $ と $ B $ はなる列であるとします。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ E_1 $ $ E_2 $ $ \dots $ $ E_K $

## 输出格式
答えを整数として出力せよ。

## 题目大意
给定 $n, k$ 和长为 $k$ 的正整数列 $E$，求长为 $n$ 的满足以下两条件的正整数列数目：

- 数列中没有完全平方数；

- 记 $p_i$ 表示从小到大第 $i$ 个质数，则数列中所有数之积为 $\prod\limits^k_{i=1}p_i^{E_i}$。

答案对 $10^9 + 7$ 取模。

```input1
3 2
3 2
```

```output1
15
```

```input2
285 10
3141 5926 5358 9793 2384 6264 3383 279 5028 8419
```

```output2
672860525
```

### Sample Explanation 1

全要素のeが $ 72=2^3\ \times\ 3^2 $ となるLさ $ 3 $ の数列は以下です。 - $ (1,1,72) $ とそのKべ替え ( $ 3 $ 通り ) ... $ 1 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (1,2,36) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 1,36 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (1,3,24) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 1 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (1,4,18) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 1,4 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (1,6,12) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 1 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (1,8,9) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 1,9 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (2,2,18) $ とそのKべ替え ( $ 3 $ 通り ) ... 条件を氦郡筏蓼埂 - $ (2,3,12) $ とそのKべ替え ( $ 6 $ 通り ) ... 条件を氦郡筏蓼埂 - $ (2,4,9) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 4,9 $ が平方数なので、条件を氦郡筏蓼护蟆 - $ (2,6,6) $ とそのKべ替え ( $ 3 $ 通り ) ... 条件を氦郡筏蓼埂 - $ (3,3,8) $ とそのKべ替え ( $ 3 $ 通り ) ... 条件を氦郡筏蓼埂 - $ (3,4,6) $ とそのKべ替え ( $ 6 $ 通り ) ... $ 4 $ が平方数なので、条件を氦郡筏蓼护蟆 よって、条件を氦郡故列は $ 15 $ です。
