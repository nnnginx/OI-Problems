<p>&nbsp; &nbsp; &nbsp;As we all know Utkarsh is very good at solving number based problems, this time Arpit thinks smartly and gives Utkarsh to solve a problem on Strings. Arpit gives Utkarsh a string and challenges him to find the length of largest substring that have its mirror string same as its original one and number of such substrings. Now Utkarsh is buzy at preparing Avishkar papers so he asks you to help him in doing this task.</p>
<p>Eg. for mirror string : Consider string "lalit" then its mirror string will be "tilal".</p>
<p>&nbsp;</p>
<p><strong>Input </strong>:</p>
<p>There are t numbers of test cases (t&lt;=200) followed t lines where each line contains a character string of lower case characters (a-z) of length l (1&gt;=l&lt;=3000).</p>
<p>&nbsp;</p>
<p><strong>Output :</strong></p>
<p>There will be two integers per line separated by space indicating the length of largest substring which have its mirror string same and number of such substrings.</p>
<p>&nbsp;</p>
<p><strong>Sample test cases :</strong></p>
<p><strong>Input :</strong></p>
<p>3</p>
<p>lalit</p>
<p>abedcdetr</p>
<p>abcde</p>
<p>&nbsp;</p>
<p><strong>Output :</strong></p>
<p>3 1</p>
<p>5 1</p>
<p>1 5</p>
