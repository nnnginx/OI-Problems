## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc139/tasks/abc139_d

整数 $ N $ にして、$ \{1,\ 2,\ ...,\ N\} $ をKべ替えた数列 $ \{P_1,\ P_2,\ ...,\ P_N\} $ をxびます。

そして、各 $ i=1,2,...,N $ について、$ i $ を $ P_i $ で割った余りを $ M_i $ とします。

$ M_1\ +\ M_2\ +\ \cdots\ +\ M_N $ の最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ M_1\ +\ M_2\ +\ \cdots\ +\ M_N $ の最大を出力せよ。

## 题目大意
## 题目描述:
对于整数 _N_ ，选择对{1，2，…，_N_}进行\
排序后的数列{_P_[1]，_P_[2]，…，
_P_[_n_]}。\
然后，关于各i＝1，2，…，_N_，将 _i_ ÷ _P_[_i_]后的余数为 _M_[_i_]。\
求 _M_[1]+ _M_[2]+…+_M_[_N_]的最大值。

### 输入格式：
一行，_N_
### 输出格式：
一行，_M_[1]+ _M_[2]+…+_M_[_N_]的最大值。否则什么也不输出。

```input1
2
```

```output1
1
```

```input2
13
```

```output2
78
```

```input3
1
```

```output3
0
```

## 提示
### 制s

- $ N $ は $ 1\ \leq\ N\ \leq\ 10^9 $ を氦郡拐数である。

### Sample Explanation 1

$ \{1,\ 2\} $ をKび替えた数列として $ \{P_1,\ P_2\}\ =\ \{2,\ 1\} $ をxぶと、$ M_1\ +\ M_2\ =\ 1\ +\ 0\ =\ 1 $ となります。
