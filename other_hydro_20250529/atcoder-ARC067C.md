## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc067/tasks/arc067_c

$ 1 $ から $ N $ までの番号のついた $ N $ 人の人がいます。 以下の二つの条件を氦郡工瑜Δ恕⒈摔椁颏いつかのグル`プに分けたいです。

- どのグル`プも、そのグル`プに含まれる人数が $ A $ 人以上 $ B $ 人以下である。
- ちょうど $ i $ 人の人が含まれるようなグル`プの数を $ F_i $ で表したとき、 すべての $ i $ について、$ F_i=0 $ または $ CQF_iQD $ が成り立っている。

このようなグル`プ分けが何通りあり得るか求めてください。 ただし、ある二つのグル`プ分けがなるとは、二人の人のMであって、 片方のグル`プ分けでは同じグル`プに含まれ、他方では同じグル`プに含まれないようなものが存在することを意味します。 なお、答えは非常に大きくなることがあるので、$ 10^9+7 $ で割った余りを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $ $ C $ $ D $

## 输出格式
あり得るグル`プ分けの数を、$ 10^9+7 $ で割った余りを出力せよ。

## 题目大意
有$N$个人，编号依次是$1,2,\cdots ,N$, 现在要将他们分成若干组，满足：

- 每一组的人数均在$[A,B]$之间。

- 记$F_i$为当前分组方案中人数为$i$的组的数量，则$F_i$应满足$F_i=0$或$C\leq F_i\leq D$.

其中$A,B,C,D$均为给定值。

求本质不同的分组方案数对$10^9+7$取模后的结果。两种方案是本质不同的当且仅当存在两个人使得在第一种方案中他们在同一组，而在第二种方案中不是。

数据范围：$1\leq N\leq 10^3,1\leq A\leq B\leq N,1\leq C\leq D\leq N$

```input1
3 1 3 1 2
```

```output1
4
```

```input2
7 2 3 1 3
```

```output2
105
```

```input3
1000 1 1000 1 1000
```

```output3
465231251
```

```input4
10 3 4 2 5
```

```output4
0
```

## 提示
### 制s

- $ 1QNQ10^3 $
- $ 1QAQBQN $
- $ 1QCQDQN $

### Sample Explanation 1

以下の $ 4 $ 通りの分け方があります。 - $ (1,2),(3) $ - $ (1,3),(2) $ - $ (2,3),(1) $ - $ (1,2,3) $ $ (1),(2),(3) $ のような分け方は、一つ目の条件は氦郡筏皮い蓼工、 二つ目の条件を氦郡筏皮い胜い郡幛耸えられません。

### Sample Explanation 2

$ 2 $ 人グル`プ、$ 2 $ 人グル`プ、$ 3 $ 人グル`プの三つに分ける以外にm切な分け方はありません。 そして、このような分け方は $ 105 $ 通りあります。

### Sample Explanation 4

答えが $ 0 $ になることもあり得ます。
