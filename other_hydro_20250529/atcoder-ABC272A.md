## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc272/tasks/abc272_a

$ N $ の整数 $ A_1,A_2,\dots,A_N $ が与えられます。

$ N $ の整数を合したを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \dots $ $ A_N $

## 输出格式
答えを出力せよ。

## 题目大意
给出 $N$ 个正整数 $A_{1\sim N}$，求这 $N$ 个数的和。

```input1
3
2 7 2
```

```output1
11
```

```input2
1
3
```

```output2
3
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 100 $
- $ 1\ \le\ A_i\ \le\ 100 $
- 入力はすべて整数

### Sample Explanation 1

$ 3 $ の整数 $ 2,7,2 $ が与えられます。 答えは $ 2\ +\ 7\ +\ 2\ =\ 11 $ です。
