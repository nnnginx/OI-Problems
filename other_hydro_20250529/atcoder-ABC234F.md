## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc234/tasks/abc234_f

文字列 $ S $ が与えられます。$ S $ の空でない、**BAするとは限らない**部分列をKび替えて得られる文字列は何Nありますか？

答えは非常に大きくなる龊悉あるので、$ 998244353 $ で割ったあまりを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ の部分列をKび替えて得られる文字列のN数を $ 998244353 $ で割ったあまりを出力せよ。

## 题目大意
给定一个仅有小写字母的字符串 $S$，你需要求出对于 $S$ 的所有**非空子序列**，将其任意重排后得到的本质不同的字符串的数量是多少。

```input1
aab
```

```output1
8
```

```input2
aaa
```

```output2
3
```

```input3
abcdefghijklmnopqrstuvwxyz
```

```output3
149621752
```

## 提示
### 制s

- $ S $ は英小文字のみからなるLさ $ 1 $ 以上 $ 5000 $ 以下の文字列

### Sample Explanation 1

$ S $ の部分列をKび替えて得られる文字列は、`a`, `b`, `aa`, `ab`, `ba`, `aab`, `aba`, `baa` の $ 8 $ Nです。

### Sample Explanation 3

$ 998244353 $ で割ったあまりを出力することに注意してください。
