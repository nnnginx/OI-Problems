## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc242/tasks/abc242_c

整数 $ N $ が与えられるので、以下の条件を全て氦郡拐数 $ X $ の数を $ 998244353 $ で割った余りを求めてください。

- $ N $ 桁の正整数である。
- $ X $ の各桁を上の位からに $ X_1,X_2,\dots,X_N $ とする。このとき以下の条件を全て氦郡埂
  - 全ての整数 $ 1\ \le\ i\ \le\ N $ にして、 $ 1\ \le\ X_i\ \le\ 9 $
  - 全ての整数 $ 1\ \le\ i\ \le\ N-1 $ にして、 $ |X_i-X_{i+1}|\ \le\ 1 $

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを整数として出力せよ。

## 题目大意
给出整数 $N$，请求出长度为 $N$ 的满足以下条件的正整数 $X$ 的情况数，除以 $998244353$ 的余数。

1.$X$长度为 $N$。

2.若 $X$ 的各个数位表示为 $x_1,x_2,……x_n$，则它们满足以下条件：

- 对于所有正整数 $i$（$1 \le\ i \le\ N$），$1 \le\ x_i \le\ 9$

- 对于所有正整数 $i$（$1 \le\ i < N$），$|x_{i+1}-x_i| \le\ 1$

```input1
4
```

```output1
203
```

```input2
2
```

```output2
25
```

```input3
1000000
```

```output3
248860093
```
