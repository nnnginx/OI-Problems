## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc055/tasks/agc055_a

Lさ $ 3N $ の文字列 $ S $ が与えられます。$ S $ は `A`, `B`, `C` をそれぞれちょうど $ N $ ずつ含みます。

文字 `A`, `B`, `C` からなる文字列 $ T $ が次の条件を氦郡工趣、$ T $ を **良い** 文字列であると呼びます。

- $ T $ のLさは $ 3 $ で割り切れる。このLさを $ 3K $ とする。
- $ T_1\ =\ T_2\ =\ \ldots\ =\ T_K $
- $ T_{K+1}\ =\ T_{K+2}\ =\ \ldots\ =\ T_{2K} $
- $ T_{2K+1}\ =\ T_{2K+2}\ =\ \ldots\ =\ T_{3K} $
- 文字 $ T_1,\ T_{K+1},\ T_{2K+1} $ は互いになる。

良い文字列の例をい菠毪取`ABC`, `BBAACC`, `AAACCCBBB` です。

$ S $ を **$ 6 $ 以下**の（BAとは限らない）部分列に分解する方法であって、各部分列が良い文字列であるような方法を一つつけてください。

これは、この}の制s下で必ず可能であることが^明できます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
`1` から `6` までの数字からなるLさ $ 3N $ の文字列を出力せよ。文字列に含まれる各 $ 1\le\ i\ \le\ 6 $ について、$ i $ を出力した位置に辘工 $ S $ の文字をKべると良い文字列が得られるようにすること。 なお、答えが}数通り存在する龊稀そのどれを出力しても正解とみなされる。

## 题目大意
**翻译**：

给定长度为 $3n (1 \le n \le 2e5)$ 的序列，其中字母 ABC 各有 $n$ 个。 

一个合法序列 $T$ 满足以下条件：

- 其长度为 $3k (1 \le k \le n)$。

-  $T_1 = T_2 = ... = T_k$

- $T_{k + 1} = T_{k + 2} = ... = T_{2k}$

- $T_{2k + 1} = T_{2k + 2} = ... = T_{3k}$

- $T_1, T_{k + 1}, T_{2k + 1}$ 互不相同。

求一个把这个序列分成**不多于** $6$ 个合法的序列的方案。

可以证明，一定存在一种**合法**的划分。

```input1
2
ABCCBA
```

```output1
111222
```

```input2
4
AABCBCAACBCB
```

```output2
111211241244
```

## 提示
### 制s

- $ 1\ \le\ N\ \le\ 2\cdot\ 10^5 $
- 文字列 $ S $ は、文字 `A`, `B`, `C` を $ N $ ずつ含む。

### Sample Explanation 1

$ S $ が部分列 `ABC`, `CBA` に分割されており、これらはそれぞれ良い文字列です。

### Sample Explanation 2

$ 1 $ の位置に辘工氩糠至肖 `AABBCC`、$ 2 $ の位置に辘工氩糠至肖 `CAB`、$ 4 $ の位置に辘工氩糠至肖 `ACB` であり、これらは全て良い文字列です。
