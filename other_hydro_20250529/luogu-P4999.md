## 输入格式
输入共$ T+1 $行，

第$ 1 $行读入$ T $。代表有$ T $组数据；

第$ 2 $~$ T+1 $行。读$ L_i $和$ R_i $

## 输出格式
输出共$ T $行，

每行输出$ L_i $和$ R_i $的区间数字和mod $ 10^9+7 $。

```input1
2
24 69
70 120
```

```output1
411
498
```

## 提示
对于 $ 50\% $ 的数据，$ 1 \leq L \leq R \leq 10^8 $；

对于 $ 100\% $ 的数据，$ 1 \leq L \leq R \leq 10^{18}  $，$ 1 \leq T \leq 20  $。
