## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc015/tasks/agc015_d

ぬけす君は、$ A $ 以上 $ B $ 以下の整数から $ 1 $ 以上xんで、それらの整数の bitwise or を取ってできる整数を持っています。 ぬけす君が持っている整数としてありうるものは何通りあるでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
ぬけす君が持っている整数としてありうるものの数を出力せよ。

## 题目大意
从 $\ge A$ 且 $\le B$ 的整数中选择一个或多个，把这些整数按位或，求一共有多少种可能的结果。

$1\le A\le B \le 2^{60}$  

```input1
7
9
```

```output1
4
```

```input2
65
98
```

```output2
63
```

```input3
271828182845904523
314159265358979323
```

```output3
68833183630578410
```

## 提示
### 制s

- $ 1\ Q\ A\ Q\ B\ <\ 2^{60} $
- $ A,B $ は整数である

### Sample Explanation 1

$ 7,8,9 $ のうちの $ 1 $ 以上の整数の bitwise or でける整数は、$ 7,8,9,15 $ の $ 4 $ つです。
