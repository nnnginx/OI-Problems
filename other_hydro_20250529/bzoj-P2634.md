## 题目描述

Xm kAc 要从 QD 去 FJ 膜拜神 CP 和 YY。地图是一个 $n \times m$ 的方格图。QD 在右上角，FJ 在右下角。
问从 QD 到 FJ 有多少条哈密尔顿路?

## 输入格式

若干行，每行两个正整数，$n,m$ 表示一组测试数据。

## 输出格式

行数同输入，每行一个整数，表示每组测试数据的答案模 `7777777`，无解输出 `YM CP&YY`。

```input1
3 2
3 3
4 10

```

```output1
YM CP&YY
2
2329
```

## 数据规模与约定

对于 $100\%$ 的数据，$N \leq 7$，$M \leq 10^9$，组数 $T \leq 20$。  
不卡常数
