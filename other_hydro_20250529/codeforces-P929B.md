## Description

<div><p>В самолёте есть <span class="tex-span"><i>n</i></span> рядов мест. Если смотреть на ряды сверху, то в каждом ряду есть <span class="tex-span">3</span> места слева, затем проход между рядами, затем <span class="tex-span">4</span> центральных места, затем ещё один проход между рядами, а затем ещё <span class="tex-span">3</span> места справа.</p><p>Известно, что некоторые места уже заняты пассажирами. Всего есть два вида пассажиров — статусные (те, которые часто летают) и обычные. </p><p>Перед вами стоит задача рассадить ещё <span class="tex-span"><i>k</i></span> <span class="tex-font-style-bf">обычных</span> пассажиров так, чтобы суммарное число соседей у статусных пассажиров было минимально возможным. Два пассажира считаются соседями, если они сидят в одном ряду и между ними нет других мест и прохода между рядами. Если пассажир является соседним пассажиром для двух статусных пассажиров, то его следует учитывать в сумме соседей дважды.</p></div><div class="input-specification"><p>В первой строке следуют два целых числа <span class="tex-span"><i>n</i></span> и <span class="tex-span"><i>k</i></span> (<span class="tex-span">1 ≤ <i>n</i> ≤ 100</span>, <span class="tex-span">1 ≤ <i>k</i> ≤ 10·<i>n</i></span>) — количество рядов мест в самолёте и количество пассажиров, которых нужно рассадить.</p><p>Далее следует описание рядов мест самолёта по одному ряду в строке. Если очередной символ равен '<span class="tex-font-style-tt">-</span>', то это проход между рядами. Если очередной символ равен '<span class="tex-font-style-tt">.</span>', то это свободное место. Если очередной символ равен '<span class="tex-font-style-tt">S</span>', то на текущем месте будет сидеть статусный пассажир. Если очередной символ равен '<span class="tex-font-style-tt">P</span>', то на текущем месте будет сидеть обычный пассажир. </p><p>Гарантируется, что количество свободных мест не меньше <span class="tex-span"><i>k</i></span>. Гарантируется, что все ряды удовлетворяют описанному в условии формату.</p></div><div class="output-specification"><p>В первую строку выведите минимальное суммарное число соседей у статусных пассажиров.</p><p>Далее выведите план рассадки пассажиров, который минимизирует суммарное количество соседей у статусных пассажиров, в том же формате, что и во входных данных. Если в свободное место нужно посадить одного из <span class="tex-span"><i>k</i></span> пассажиров, выведите строчную букву '<span class="tex-font-style-tt">x</span>' вместо символа '<span class="tex-font-style-tt">.</span>'. </p></div>

## Input

<p>В первой строке следуют два целых числа <span class="tex-span"><i>n</i></span> и <span class="tex-span"><i>k</i></span> (<span class="tex-span">1 ≤ <i>n</i> ≤ 100</span>, <span class="tex-span">1 ≤ <i>k</i> ≤ 10·<i>n</i></span>) — количество рядов мест в самолёте и количество пассажиров, которых нужно рассадить.</p><p>Далее следует описание рядов мест самолёта по одному ряду в строке. Если очередной символ равен '<span class="tex-font-style-tt">-</span>', то это проход между рядами. Если очередной символ равен '<span class="tex-font-style-tt">.</span>', то это свободное место. Если очередной символ равен '<span class="tex-font-style-tt">S</span>', то на текущем месте будет сидеть статусный пассажир. Если очередной символ равен '<span class="tex-font-style-tt">P</span>', то на текущем месте будет сидеть обычный пассажир. </p><p>Гарантируется, что количество свободных мест не меньше <span class="tex-span"><i>k</i></span>. Гарантируется, что все ряды удовлетворяют описанному в условии формату.</p>

## Output

<p>В первую строку выведите минимальное суммарное число соседей у статусных пассажиров.</p><p>Далее выведите план рассадки пассажиров, который минимизирует суммарное количество соседей у статусных пассажиров, в том же формате, что и во входных данных. Если в свободное место нужно посадить одного из <span class="tex-span"><i>k</i></span> пассажиров, выведите строчную букву '<span class="tex-font-style-tt">x</span>' вместо символа '<span class="tex-font-style-tt">.</span>'. </p>

## Samples

```input1
1 2
SP.-SS.S-S.S

```

```output1
5
SPx-SSxS-S.S

```






```input2
4 9
PP.-PPPS-S.S
PSP-PPSP-.S.
.S.-S..P-SS.
P.S-P.PP-PSP

```

```output2
15
PPx-PPPS-S.S
PSP-PPSP-xSx
xSx-SxxP-SSx
P.S-PxPP-PSP

```




## Note

<p>В первом примере нужно посадить ещё двух обычных пассажиров. Для минимизации соседей у статусных пассажиров, нужно посадить первого из них на третье слева место, а второго на любое из оставшихся двух мест, так как независимо от выбора места он станет соседом двух статусных пассажиров. </p><p>Изначально, у статусного пассажира, который сидит на самом левом месте уже есть сосед. Также на четвёртом и пятом местах слева сидят статусные пассажиры, являющиеся соседями друг для друга (что добавляет к сумме <span class="tex-span">2</span>).</p><p>Таким образом, после посадки ещё двух обычных пассажиров, итоговое суммарное количество соседей у статусных пассажиров станет равно пяти.</p>
