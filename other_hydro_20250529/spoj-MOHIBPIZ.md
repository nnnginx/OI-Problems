<p>In this problem you are given a pizza.You have to find the maximum number of slices after cutting the pizza <strong>exactly N</strong> times.</p>
<p><strong>Note that it is not necessary to make each slice equal.</strong></p>
<p>&nbsp;</p>
<p>For example if <strong>N = 1</strong>&nbsp;then we can make maximum 2 slices. Figure 1 show this.</p>
<p><img title="Fig : 01" src="file://zFtOI7f4.png" alt="Pizza 1" width="338" height="314"><img title="Figure : 02" src="file://iCY9CSsK.png" alt="Pizza 2" width="338" height="314"></p>
<p>If <strong>N = 2</strong> then we can make maximum 4 slices. Figure 2 show this.</p>
<h3>Input</h3>
<p>Input starts with an integer<strong> T (1&lt;=10^6)</strong>, denoting the number of test cases.</p>
<p>Each case contains an integer <strong>N (1 ¡Ü N ¡Ü 10^9)</strong> denoting the number of times you can cut the pizza.</p>
<h3>Output</h3>
<p>For each case of input, output only one integer the <strong>maximum number of slices</strong>.</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
<p>&nbsp; &nbsp; 2</p>
<p>&nbsp;&nbsp;&nbsp; 1</p>
<p>&nbsp;&nbsp;&nbsp; 2</p>
<strong>Output:</strong>
<p>&nbsp;&nbsp;&nbsp; 2</p>
<p>&nbsp;&nbsp;&nbsp; 4</p></pre>
