## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc167/tasks/abc167_b

$ 1 $ がかれたカ`ドが $ A $ 枚、$ 0 $ がかれたカ`ドが $ B $ 枚、 $ -1 $ がかれたカ`ドが $ C $ 枚あります。

これらのカ`ドから、ちょうど $ K $ 枚をxんで取るとき、取ったカ`ドにかれた数の和として、 ありうるの最大はいくつですか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ K $

## 输出格式
和としてありうるの最大を出力せよ。

## 题目大意
我们有上面写着 $1$ 的 $A$ 张牌，上面写着 $0$ 的 $B$ 张牌，上面写着 $-1$ 的 $C$ 张牌。   

此时我们一次性从中取出 $K$ 张牌，求所取的牌的最大和。

by [djh123456](https://www.luogu.com.cn/user/367575)    。

```input1
2 1 1 3
```

```output1
2
```

```input2
1 2 3 4
```

```output2
0
```

```input3
2000000000 0 0 2000000000
```

```output3
2000000000
```

## 提示
### 制s

- 入力は全て整数である。
- $ 0\ \leq\ A,\ B,\ C $
- $ 1\ \leq\ K\ \leq\ A\ +\ B\ +\ C\ \leq\ 2\ \times\ 10^9 $

### Sample Explanation 1

$ 1 $ がかれたカ`ドを $ 2 $ 枚、$ 0 $ がかれたカ`ドを $ 1 $ 枚取ることを考えます。 このときカ`ドにかれた数の和は $ 2 $ になり、和としてありうるの最大になります。
