## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc181/tasks/abc181_a

高颏んは今、白い服を着ています。

高颏んは、白い服を着た次の日には\い服を、\い服を着た次の日には白い服を着ます。

$ N $ 日後には何色の服を着るでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ 日後に白い服を着るなら `White` を、\い服を着るなら `Black` を出力せよ。

## 题目大意
高桥一开始穿白色衣服，穿白衣服的下一天他会穿黑衣服，同理，穿黑衣服的下一天他会穿白衣服。

问 $N$ 天后他穿什么颜色的衣服。

```input1
2
```

```output1
White
```

```input2
5
```

```output2
Black
```

## 提示
### 制s

- $ N $ は整数である
- $ 1\ \leq\ N\ \leq\ 30 $

### Sample Explanation 1

高颏んは $ 1 $ 日後に\い服を、 $ 2 $ 日後に白い服を着ます。
