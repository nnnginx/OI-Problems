## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc161/tasks/abc161_f

正整数 $ N $ が与えられます。

$ 2 $ 以上 $ N $ 以下の整数 $ K $ をQめて、$ N $ が $ K $ 未氦摔胜毪蓼谴韦尾僮鳏蚶Rり返し行います。

- 操作：$ N $ が $ K $ で割り切れるとき、$ N $ を $ N/K $ に置きQえる。そうでないとき、$ N $ を $ N-K $ に置きQえる。

最K的に $ N $ が $ 1 $ になるような $ K $ のQめ方は何通りありますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
最K的に $ N $ が $ 1 $ になるような $ K $ のQめ方が何通りあるか出力せよ。

## 题目大意
给定正整数 $n$。问 $2,3,\cdots,n$ 中有多少个 $k$ 满足如下条件：

如果 $k|n$，那么令 $n$ 的值除以 $k$，否则令 $n$ 的值减少 $k$，如此操作若干次后能得到 $1$。

$2\le n\le10^{12}$

```input1
6
```

```output1
3
```

```input2
3141
```

```output2
13
```

```input3
314159265358
```

```output3
9
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 10^{12} $
- $ N $ は整数

### Sample Explanation 1

最K的に $ N $ が $ 1 $ になるような $ K $ は $ 2,5,6 $ の $ 3 $ 通りです。 それぞれのとき、$ N $ は次のように浠します。 - $ K=2 $ のとき：$ 6\ \to\ 3\ \to\ 1 $ - $ K=5 $ のとき：$ 6\ \to\ 1 $ - $ K=6 $ のとき：$ 6\ \to\ 1 $
