<p><span id="docs-internal-guid-e9be4b85-5315-1adf-99de-18fe2315252a"> </span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">A job has been assigned to Jar Jar Binks, it goes as follows:</span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">There are N spaceships parts, each with a weight of Wi kg. Given a weight W, he has to show how many parts can be used in order to make a ship with a weight of exactly W kg. He has to show all possible solutions, of course if possible.</span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">Everybody knows Jar Jar Binks particularly because of his clumsiness, so you have to help him. Write a program that solves his problem!</span></p>
<div><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;"><br></span></div>
<p>&nbsp;</p>
<h3>Input</h3>
<p><span id="docs-internal-guid-e9be4b85-5315-7837-4cfd-5cc21608f9c2"> </span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">There will be several cases, each beginning with two integers N, Q (1&lt;=N&lt;=60, 0&lt;=Q&lt;=10000).</span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">Next there will be N positive integers representing the weights of the N spaceship parts (1&lt;=Wi&lt;=1000).</span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">Q lines will follow, each one with only one (not necessarily positive) integer W, the total weight of the spaceship.</span></p>
<p><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">End of input will be denoted with N = 0 and Q = 0. This case should not be processed</span></p>
<p>&nbsp;</p>
<h3>Output</h3>
<p><span id="docs-internal-guid-e9be4b85-5315-98f2-9e24-f9795e41f020"> </span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt; text-align: justify;" dir="ltr"><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">Print a line with K integers per query in ascending order. They must represent the amount of pieces that can be used to make a spaceship with weight W.</span></p>
<p><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">If there is no way to make a spaceship with weight W, output a line with the string </span><span style="font-size: 15px; font-family: Arial; vertical-align: baseline; white-space: pre-wrap; background-color: transparent;">¡°That's impossible!¡± (quotes to clarify)</span></p>
