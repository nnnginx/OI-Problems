## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc132/tasks/abc132_f

正の整数 $ K $ を一列にKべたものであって、O接してKんでいるどの $ 2 $ つの整数のeも $ N $ 以下であるものの数を $ 10^9+7 $ で割った余りを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
条件を氦郡沽肖数を $ 10^9+7 $ で割った余りを出力せよ。

## 题目大意
求有多少种长度为 $K$ 的正整数序列，满足相邻元素乘积不超过 $N$，答案对 $10^9 + 7$ 取模。

```input1
3 2
```

```output1
5
```

```input2
10 3
```

```output2
147
```

```input3
314159265 35
```

```output3
457397712
```

## 提示
### 制s

- $ 1\leq\ N\leq\ 10^9 $
- 1 $ 2\leq\ K\leq\ 100 $ (21:33 修正)
- $ N,K $ は整数である
