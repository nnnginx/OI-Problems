## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc232/tasks/abc232_a

$ 3 $ 文字からなる文字列 $ S $ が与えられます。$ S $ は、$ 1 $ 以上 $ 9 $ 以下の整数 $ a,\ b $ と文字 `x` を、`axb` のようににつなげて得られます。

$ a $ と $ b $ のeを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
答えを出力せよ。

## 题目大意
# 输入格式
一串形如 $a* b$ 长度为 $3$ 的字符串 $S$。

# 输出格式
输出这个字符串中 $a* b$ 的值。

**记得换行**

```input1
3x7
```

```output1
21
```

```input2
9x9
```

```output2
81
```

```input3
1x1
```

```output3
1
```

## 提示
### 制s

- $ S $ のLさは $ 3 $
- $ S $ の $ 1 $ 文字目および $ 3 $ 文字目は $ 1 $ 以上 $ 9 $ 以下の整数を表す
- $ S $ の $ 2 $ 文字目は `x`

### Sample Explanation 1

$ 3\ \times\ 7\ =\ 21 $ であるので、これを出力します。
