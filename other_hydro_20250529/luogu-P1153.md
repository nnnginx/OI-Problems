## 题目描述
平面上有一些点，你可以用直线将两点连接起来。那么有多少种方法可以把这些点连续地连起来，使得任何两个线都不交叉。

显然，三个点只有一种方法。四个点最多只有 $3$ 种方法。写一个程序计算方法总数。


## 输入格式
每一行是一个点的坐标，坐标值是整数，中间用一个空格隔开。最后一个坐标是原点。任意三点不在一条直线上。

## 输出格式
输出方案总数。


```input1
100 -10
-200 0
45 7
0 0

```

```output1
3

```

## 提示
最多只有 $10$  个点。

- 必须从一个点出发，途径所有点回到起点的路径才会被统计。

- 两个方案不相同当且仅当围成的简单多边形不同。
