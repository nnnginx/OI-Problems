## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc135/tasks/arc135_a

\板にひとつの整数 $ X $ がかれています。あなたは次の操作を、何度でも行うことができます（一度も行わなくてもよいです）。

- \板にかれている整数 $ x $ をひとつxぶ。
- $ x $ をひとつ\板から消去し、新たに $ \lfloor\ \frac{x}{2}\rfloor $ と $ \lceil\ \frac{x}{2}\rceil $ をひとつずつ\板にく。

操作後の\板にかれている整数すべてのeとしてありうる最大を、$ 998244353 $ で割った余りを答えてください。

  $ \lfloor\ \frac{x}{2}\rfloor $，$ \lceil\ \frac{x}{2}\rceil $ とは？ g数 $ x $ にして，$ x $ 以下の最大の整数を $ \lfloor\ x\rfloor $、$ x $ 以上の最小の整数を $ \lceil\ x\rceil $ ときます。したがって例えば以下が成り立ちます。

- $ x\ =\ 2 $ のとき、$ \lfloor\ \frac{x}{2}\rfloor\ =\ 1 $, $ \lceil\ \frac{x}{2}\rceil\ =\ 1 $。
- $ x\ =\ 3 $ のとき、$ \lfloor\ \frac{x}{2}\rfloor\ =\ 1 $, $ \lceil\ \frac{x}{2}\rceil\ =\ 2 $。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ X $

## 输出格式
操作後の\板にかれている整数すべてのeとしてありうる最大を、$ 998244353 $ で割った余りを出力してください。

```input1
15
```

```output1
192
```

```input2
3
```

```output2
3
```

```input3
100
```

```output3
824552442
```

## 提示
### 制s

- $ 1\leq\ X\ \leq\ 10^{18} $

### Sample Explanation 1

例えば次のように操作を行うことで、\板にかれている整数すべてのeを $ 192 $ にすることが可能です。 - はじめ、\板は次の状Bです：$ (15) $。 - $ x\ =\ 15 $ として操作を行うことで、\板は次の状Bに浠します：$ (7,\ 8) $。 - $ x\ =\ 7 $ として操作を行うことで、\板は次の状Bに浠します：$ (8,\ 3,\ 4) $。 - $ x\ =\ 4 $ として操作を行うことで、\板は次の状Bに浠します：$ (8,\ 3,\ 2,\ 2) $。 - $ x\ =\ 8 $ として操作を行うことで、\板は次の状Bに浠します：$ (3,\ 2,\ 2,\ 4,\ 4) $。 このとき、\板にかれている整数すべてのeは $ 3\times\ 2\times\ 2\times\ 4\times\ 4\ =\ 192 $ です。

### Sample Explanation 2

操作を一度も行わないことで、\板にかれている整数すべてのeを $ 3 $ にすることが可能です。

### Sample Explanation 3

操作後の\板にかれている整数すべてのeとしてありうる最大は、$ 5856458868470016 $ です。これを $ 998244353 $ で割った余りを出力します。
