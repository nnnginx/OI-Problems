## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc214/tasks/abc214_e

$ T $ のテストケ`スについて、以下の}を解いてください。

$ 1,2,\dots,10^9 $ の番号がついた $ 10^9 $ の箱と、 $ 1,2,\dots,N $ の番号がついた $ N $ のボ`ルがあります。  
 それぞれの箱に入れることのできるボ`ルの数は多くとも $ 1 $ です。  
 以下の条件を氦郡工瑜Δ恕 $ N $ のボ`ルを全て箱に入れることができるか判定してください。

- 全ての $ 1 $ 以上 $ N $ 以下の整数 $ i $ について、番号 $ i $ のボ`ルが $ L_i $ 以上 $ R_i $ 以下の番号がついた箱に入っている。

## 输入格式
入力は嗜肓Δら与えられる。入力の $ 1 $ 行目は以下の形式である。

> $ T $

その後、 $ T $ のテストケ`スがAく。各テストケ`スは以下の形式で与えられる。

> $ N $ $ L_1 $ $ R_1 $ $ L_2 $ $ R_2 $ $ \dots $ $ L_N $ $ R_N $

## 输出格式
出力は $ T $ 行からなる。  
 $ i(1\ \le\ i\ \le\ T) $ 行目には、 $ i $ 番目に入力されたテストケ`スについて、}文中の条件を氦郡工瑜Δ $ N $ のボ`ルを全て箱に入れることができるなら `Yes` 、そうでないなら `No` と出力せよ。  
 なお、正`判定器は英大文字と英小文字を区eせず、どちらも受理する。

## 题目大意
有n个球和 $10^9$ ,每个盒子最多一个球，接下有 n 个 $l_i,r_i$表示第 i 个球要在$[l_i,r_i]$, 问是否有合法方案，共 T 组数据。

```input1
2
3
1 2
2 3
3 3
5
1 2
2 3
3 3
1 3
999999999 1000000000
```

```output1
Yes
No
```

## 提示
### 制s

- $ 1\ \le\ T\ \le\ 2\ \times\ 10^5 $
- $ 1\ \le\ N\ \le\ 2\ \times\ 10^5 $
- $ 1\ \le\ L_i\ \le\ R_i\ \le\ 10^9 $
- $ 1 $ つの入力に含まれるテストケ`スについて、それらの $ N $ のt和は $ 2\ \times\ 10^5 $ を超えない。
