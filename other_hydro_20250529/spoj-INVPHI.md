<p>This task is the inverse of <a title="ETF" href="../ETF/" target="_blank">ETF</a> problem, given an integer <strong>n</strong>&nbsp;find smallest integer <strong>i</strong> such that ¦Õ(<strong>i</strong>)=<strong>n</strong>, where ¦Õ denotes <a title="Euler's_totient_function" href="http://en.wikipedia.org/wiki/Euler's_totient_function" target="_blank">Euler's totient function</a>.</p>
<h3>Input</h3>
<p>The first line is an integer <strong>T</strong>(1 ¡Ü <strong>T</strong> ¡Ü 100,000), denoting the number of test cases.&nbsp;Then, T test cases follow.</p>
<p>For each test case, there is an integer <strong>n</strong>(1 ¡Ü <strong>n</strong>&nbsp;¡Ü 100,000,000) written in one line. (one ineger per line)</p>
<h3>Output</h3>
<p>For each test case, output Smallest Inverse Euler's Totient Function of <strong>n</strong>.&nbsp;if <strong>n</strong>&nbsp;doesn't have inverse, output -1.</p>
