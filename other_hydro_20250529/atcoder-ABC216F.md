## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc216/tasks/abc216_f

Lさ $ N $ の数列 $ A\ =\ (A_1,\ \dots,\ A_N),\ B\ =\ (B_1,\ \dots,\ B_N) $ が与えられます。$ \{1,2,\ldots,N\} $ の空でない部分集合 $ S $ であって、以下の条件を氦郡工猡韦数を数えてください。

- $ \max_{i\ \in\ S}\ A_i\ \geq\ \sum_{i\ \in\ S}\ B_i $

なお、答えは非常に大きくなることがあるため、$ 998244353 $ で割ったあまりを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $ $ B_1 $ $ B_2 $ $ \ldots $ $ B_N $

## 输出格式
}文中の条件を氦郡 $ S $ の数を $ 998244353 $ で割ったあまりを出力せよ。

## 题目大意
给出两个有 $n$ 个元素的序列 $a,b$，现在在 $1 - n$ 中选一些数构成集合 $S$，使得 $max_{i \in S} \space a_i \ge \sum_{i \in S}  \space b_i$，问合法的集合 $S$ 的个数 $\mod 998244353$。
$1 \le N \le 5000,1 \le a_i,b_i \le 5000$.

```input1
2
3 1
1 2
```

```output1
2
```

```input2
2
1 1
2 2
```

```output2
0
```


```output3
476
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 5000 $
- $ 1\ \leq\ A_i,B_i\ \leq\ 5000 $
- 入力は全て整数

### Sample Explanation 1

$ \{1,2,\ldots,N\} $ の空でない部分集合としてあり得るものは、$ \{1\} $, $ \{2\} $, $ \{1,2\} $ の $ 3 $ 通りです。 - $ S=\{1\} $ のとき $ \max_{i\ \in\ S}\ A_i=3 $, $ \sum_{i\ \in\ S}\ B_i=1 $ - $ S=\{2\} $ のとき $ \max_{i\ \in\ S}\ A_i=1 $, $ \sum_{i\ \in\ S}\ B_i=2 $ - $ S=\{1,2\} $ のとき $ \max_{i\ \in\ S}\ A_i=3 $, $ \sum_{i\ \in\ S}\ B_i=3 $ であるため、}文中の条件、即ち $ \max_{i\ \in\ S}\ A_i\ \geq\ \sum_{i\ \in\ S}\ B_i $ を氦郡 $ S $ は $ \{1\} $ と $ \{1,2\} $ の $ 2 $ 通りです。

### Sample Explanation 2

条件を氦郡 $ S $ が存在しない龊悉猡ります。
