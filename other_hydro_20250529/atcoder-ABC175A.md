## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc175/tasks/abc175_a

AtCoder 町の、あるBAした $ 3 $ 日gの天荬斡hがあります。天荬斡hはLさ $ 3 $ の文字列 $ S $ で表され、$ i\ (1\ \leq\ i\ \leq\ 3) $ 日目の天荬 $ i $ 文字目が `S` のとき晴れ、`R` のとき雨でした。

天荬雨である日がBAしていた最大の日数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
天荬雨である日がBAしていた最大の日数を出力せよ。

## 题目大意
给一个字符串，求其中连续 $‘R’$ 的个数。

```input1
RRS
```

```output1
2
```

```input2
SSS
```

```output2
0
```

```input3
RSR
```

```output3
1
```

## 提示
### 制s

- $ |S|\ =\ 3 $
- $ S $ の各文字は `S` または `R` である

### Sample Explanation 1

$ 3 $ 日gのうち、$ 1,\ 2 $ 日目が雨で、最大では $ 2 $ 日g雨である日がBAしていたので $ 2 $ を出力してください。

### Sample Explanation 2

$ 3 $ 日gのうち、$ 3 $ 日とも晴れでした。雨である日はoかったため、$ 0 $ を出力してください。

### Sample Explanation 3

$ 3 $ 日gのうち、$ 1,\ 3 $ 日目が雨でした。共に $ 1 $ 日雨である日がBAしていたので、$ 1 $ を出力してください。
