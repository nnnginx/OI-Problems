## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc122/tasks/abc122_d

整数 $ N $ が与えられます。次の条件を氦郡归Lさ $ N $ の文字列の数を $ 10^9+7 $ で割った余りを求めてください。

- `A`, `C`, `G`, `T` 以外の文字を含まない。
- `AGC` を部分文字列として含まない。
- O接する $ 2 $ 文字の入れ替えを $ 1 $ 回行うことで上の条件に`反させることはできない。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
条件を氦郡刮淖至肖问を $ 10^9+7 $ で割った余りを出力せよ。

## 题目大意
您将得到一个整数 $N$ ，请找出满足下列要求的长度为 $N$ 的字符串的数目，答案对 $10^9+7$ 取模

- 该字符串仅由 `A` `C` `G` `T` 四种字符组成
- 该字符串不包含子串 `AGC` 
- 交换相邻两个字符**一次**不违反上述要求

```input1
3
```

```output1
61
```

```input2
4
```

```output2
230
```

```input3
100
```

```output3
388130742
```

## 提示
### 注

文字列 $ T $ の部分文字列とは、$ T $ の先^と末尾から $ 0 $ 文字以上を取り去って得られる文字列です。

例えば、`ATCODER` の部分文字列には `TCO`, `AT`, `CODER`, `ATCODER`, `` (空文字列) が含まれ、`AC` は含まれません。

### 制s

- $ 3\ \leq\ N\ \leq\ 100 $

### Sample Explanation 1

`A`, `C`, `G`, `T` 以外の文字を含まないLさ $ 3 $ の文字列は $ 4^3\ =\ 64 $ 通り存在し、そのうち `AGC`, `ACG`, `GAC` のみが条件に`反するため、答えは $ 64\ -\ 3\ =\ 61 $ 通りです。

### Sample Explanation 3

文字列の数を $ 10^9+7 $ で割った余りを出力することをお忘れなく。
