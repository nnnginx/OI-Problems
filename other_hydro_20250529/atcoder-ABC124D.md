## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc124/tasks/abc124_d

$ N $ 人の人が左右一列にKんでいます。

`0`, `1` からなるLさ $ N $ の文字列 $ S $ と正整数 $ K $ が与えられます。

左から $ i $ 番目の人は、$ S $ の $ i $ 文字目が `0` のとき直立し、`1` のとき逆立ちしています。

あなたは $ K $ 回まで以下の指示を行います。一度も行わなくてもいません。

**指示**: $ 1\ \leq\ l\ \leq\ r\ \leq\ N $ を氦郡拐数 $ l,\ r $ をxぶ。左から $ l,\ l+1,\ ...,\ r $ 番目の人の状Bを反する。すなわち、$ i\ =\ l,\ l+1,\ ...,\ r $ について、左から $ i $ 番目の人は直立していれば逆立ちし、逆立ちしていれば直立する。

$ K $ 回までの指示で、逆立ちした人を**BAで**最大何人Kばせることができるか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ S $

## 输出格式
$ K $ 回までの指示で、逆立ちした人をBAで最大何人Kばせることができるか出力せよ。

## 题目大意
### 题目描述

有一个长为 $n$ 的字符串 $s$，只含 $0$ 和 $1$。  
你可以进行最多 $k$ 次如下操作（$0$ 次也可以）：

- 选择字符串 $s$ 的一个子串，将其中的字符反转（$0$ 变成 $1$，$1$ 变成 $0$）。

进行不超过 $k$ 次操作后，求最长的连续的 $1$ 的长度。

### 输入格式

第一行，$2$ 个正整数 $n,k$；  
第二行，字符串 $s$。

### 输出格式

输出不超过 $k$ 次操作后，最长的连续的 $1$ 的长度。

### 数据约定

对于 $100\%$ 的数据：$1 \le n, k \le 10^5$。  
字符串 $s$ 只由 $0$ 和 $1$ 组成，长度为 $n$。

```input1
5 1
00010
```

```output1
4
```

```input2
14 2
11101010110011
```

```output2
8
```

```input3
1 1
1
```

```output3
1
```

## 提示
### 制s

- $ N $ は $ 1\ \leq\ N\ \leq\ 10^5 $ を氦郡拐数である。
- $ K $ は $ 1\ \leq\ K\ \leq\ 10^5 $ を氦郡拐数である。
- 文字列 $ S $ のLさは $ N $ である。
- 文字列 $ S $ の各文字は `0` または `1` である。

### Sample Explanation 1

以下のように指示を行えば逆立ちした人をBAして $ 4 $ 人Kばせることができ、これが最大です。 - $ l\ =\ 1,\ r\ =\ 3 $ として指示を行う。そのY果、左から $ 1,\ 2,\ 3 $ 番目の人の状Bが反する。

### Sample Explanation 3

一度も指示を行う必要はありません。
