## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc198/tasks/abc198_d

英小文字からなる文字列 $ S_1,S_2,S_3 $ が与えられます。覆面算 $ S_1+S_2=S_3 $ を解いてください。

正_には、次の $ 3 $ つの条件をすべて氦郡工瑜Δ**正の**整数のM $ N_1,N_2,N_3 $ が存在するか判定し、存在するならばそのうち $ 1 $ つを求めてください。  
 ここで、$ N_1,\ N_2,\ N_3 $ を (先^に余分な $ 0 $ をつけないで) 十M表した文字列をそれぞれ $ N'_1,\ N'_2,\ N'_3 $ とします。

- $ N'_i $ の文字数は、$ S_i $ の文字数に等しい
- $ N_1+N_2=N_3 $ を氦郡
- $ S_i $ の $ x $ 文字目と $ S_j $ の $ y $ 文字目が等しいとき、またそのrに限り、$ N'_i $ の $ x $ 文字目と $ N'_j $ の $ y $ 文字目が等しい

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S_1 $ $ S_2 $ $ S_3 $

## 输出格式
条件を氦郡工瑜Δ收整数のM $ N_1,N_2,N_3 $ が存在するならば、そのようなMの $ 1 $ つを改行区切りで出力せよ。 存在しないなら、代わりに `UNSOLVABLE` と出力せよ。

```input1
a
b
c
```

```output1
1
2
3
```

```input2
x
x
y
```

```output2
1
1
2
```

```input3
p
q
p
```

```output3
UNSOLVABLE
```

```input4
abcd
efgh
ijkl
```

```output4
UNSOLVABLE
```

```input5
send
more
money
```

```output5
9567
1085
10652
```

## 提示
### 制s

- $ S_1,S_2,S_3 $ は英小文字のみからなるLさ $ 1 $ 以上 $ 10 $ 以下の文字列

### Sample Explanation 1

$ (N_1,\ N_2,\ N_3)\ =\ (4,5,9) $ などを出力しても正解となります。$ (1,1,2) $ は $ 3 $ つ目の条件を氦郡筏皮い胜 (`a`,`b` がともに $ 1 $ に辘筏皮い) ため、不正解となります。

### Sample Explanation 2

$ (N_1,\ N_2,\ N_3)\ =\ (3,3,6) $ などを出力しても正解となります。$ (1,2,3) $ は $ 3 $ つ目の条件を氦郡筏皮い胜 ($ 1,2 $ がともに `x` に辘筏皮い) ため、不正解となります。
