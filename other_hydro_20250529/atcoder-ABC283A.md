## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc283/tasks/abc283_a

整数 $ A,\ B $ が与えられます。 $ A^B $ のを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
答えを出力せよ。

## 题目大意
输出 $a^b$。

```input1
4 3
```

```output1
64
```

```input2
5 5
```

```output2
3125
```

```input3
8 1
```

```output3
8
```

## 提示
### 制s

- $ 1\ \leq\ A,\ B\ \leq\ 9 $
- 入力はすべて整数

### Sample Explanation 1

$ 4^3\ =\ 64 $ であるので、$ 64 $ を出力します。
