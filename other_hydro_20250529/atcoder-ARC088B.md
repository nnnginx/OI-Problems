## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc083/tasks/arc088_b

`0` と `1` からなる文字列 $ S $ が与えられます。 以下の操作を好きな回数Rり返すことで $ S $ の要素をすべて `0` にできるような、$ |S| $ 以下の最大の整数 $ K $ を求めてください。

- $ S $ のLさ $ K $ 以上のBAする区g $ [l,r] $ をxぶ(すなわち、$ r-l+1\geq\ K $ が氦郡丹欷氡匾がある)。$ l\leq\ i\leq\ r $ なるすべての整数 $ i $ にし、$ S_i $ が `0` なら $ S_i $ を `1` に、$ S_i $ が `1` なら $ S_i $ を `0` に置きQえる。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
操作を好きな回数Rり返すことで $ S $ の要素をすべて `0` にできるような *最大の* (21:08 JST 修正) 整数 $ K $ を出力せよ。

## 题目大意
给出一个由0和1组成的字符串S。求不大于S的最大整数K，这样我们可以通过多次重复下面的操作将S的所有字符都变成0。



选择S中长度至少为K(即必须满足r-l+1≥K)的连续段[l,r]。对于l≤i≤r的每个整数i，执行以下操作：如果S_i为0，则用1替换；如果S_i为1，则用0替换。

```input1
010
```

```output1
2
```

```input2
100000000
```

```output2
8
```

```input3
00001111
```

```output3
4
```

## 提示
### 制s

- $ 1\leq\ |S|\leq\ 10^5 $
- $ S_i(1\leq\ i\leq\ N) $ は `0` または `1` である

### Sample Explanation 1

以下の操作で、$ S $ の要素をすべて `0` にできます。 - Lさ $ 3 $ の区g $ [1,3] $ に操作を行う。$ S $ は `101` になる。 - Lさ $ 2 $ の区g $ [1,2] $ に操作を行う。$ S $ は `011` になる。 - Lさ $ 2 $ の区g $ [2,3] $ に操作を行う。$ S $ は `000` になる。
