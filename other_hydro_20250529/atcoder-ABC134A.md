## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc134/tasks/abc134_a

半径 $ a $ の窑四诮婴工胝十二角形の面eは $ 3a^2 $ であることが知られています。

整数 $ r $ が与えられるので、半径 $ r $ の窑四诮婴工胝十二角形の面eを求めて下さい。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ r $

## 输出格式
正十二角形の面eを表す整数を出力してください。

## 题目大意
给你一个 $a$，求 $3a^2$ 的值。

```input1
4
```

```output1
48
```

```input2
15
```

```output2
675
```

```input3
80
```

```output3
19200
```

## 提示
### 制s

- $ 1\ \leqq\ r\ \leqq\ 100 $
- $ r $ は整数である。

### Sample Explanation 1

正十二角形の面eは $ 3\ \times\ 4^2\ =\ 48 $ です。
