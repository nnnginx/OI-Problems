## 问题陈述

Takahashi 生活在另一个世界。这个世界中有 $10000$ 种颜色的史莱姆（生物）。我们称这些颜色为颜色 $1, 2, ..., 10000$。

Takahashi 有 $N$ 只史莱姆，它们从左到右排列。第 $i$ 只史莱姆的颜色是 $a_i$。
如果两只相同颜色的史莱姆相邻，它们将开始合并。因为 Takahashi 喜欢较小的史莱姆，他决定用魔法改变一些史莱姆的颜色。

Takahashi 可以通过一次咒语将一只史莱姆的颜色改变为 $10000$ 种颜色中的任何一种。
为了确保没有史莱姆会开始合并，需要多少次咒语？

## 约束条件

- $2 \leq N \leq 100$
- $1 \leq a_i \leq N$
- 所有输入值均为整数。

## 输入

输入从标准输入给出，格式如下：

> $N$
> 
> $a_1$ $a_2$ $...$ $a_N$

## 输出

打印所需的最小咒语次数。

```input1
5
1 1 2 2 2
```

```output1
2
```

例如，如果我们将从左数的第二只史莱姆的颜色改为 $4$，第四只史莱姆的颜色改为 $5$，那么史莱姆的颜色将变为 $1, 4, 2, 5, 2$，这样就满足了条件。

```input2
3
1 2 1
```

```output2
0
```

尽管第一只和第三只史莱姆的颜色相同，但它们并不相邻，因此不需要咒语。

```input3
5
1 1 1 1 1
```

```output3
2
```

例如，如果我们将从左数的第二只和第四只史莱姆的颜色改为 $2$，那么史莱姆的颜色将变为 $1, 2, 1, 2, 1$，这样就满足了条件。

```input4
14
1 2 2 3 3 3 4 4 4 4 1 2 3 4
```

```output4
4
```
