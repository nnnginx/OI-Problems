## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc055/tasks/agc055_d

`A`, `B`, `C` をそれぞれちょうど $ N $ ずつ含むLさ $ 3N $ の文字列 $ T $ が次の条件を氦郡工趣、$ T $ を**良い**文字列と呼びます: $ T $ を $ N $ のLさ $ 3 $ の（BAとは限らない）部分列に分解する方法であって、各部分列が `ABC`, `BCA`, `CAB` のいずれかであるような方法が存在する。

`A`, `B`, `C`, `?` からなるLさ $ 3N $ の文字列 $ S $ が与えられます。各 `?` を `A`, `B`, `C` のいずれかに置きQえる方法であって、Y果が良い文字列となるようなものの数を求めてください。この数は非常に大きい可能性があるため、これを $ 998244353 $ で割った余りを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
答えを $ 998244353 $ で割った余りを出力せよ。

## 题目大意
称一个长度为 $3n$，只由 $\text{A,B,C}$ 组成的字符串是好的，当且仅当其能划分成 $n$ 个长度为 $3$  的子序列，每个子序列都是 $\text{ABC}$，或者 $\text{BCA}$，或者 $\text{CAB}$。求把问号替换成 $\text{A,B,C}$ 使得字符串是好的的方案数 $\bmod\ 998244353$。

```input1
1
???
```

```output1
3
```

```input2
2
AA????
```

```output2
2
```

```input3
3
?A?A?A?A?
```

```output3
0
```

```input4
9
?????????A??B??C???????????
```

```output4
331653164
```

## 提示
### 制s

- $ 1\le\ N\ \le\ 15 $
- $ S $ は、`A`, `B`, `C`, `?` からなるLさ $ 3N $ の文字列である。

### Sample Explanation 1

得られる良い文字列は、`ABC`, `BCA`, `CAB` の $ 3 $ です。

### Sample Explanation 2

得られる良い文字列は、`AABBCC`, `AABCBC` の $ 2 $ です。

### Sample Explanation 3

`A` が既に $ 4 $ 含まれるため、良い文字列を得ることはできません。
