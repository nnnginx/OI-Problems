## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc090/tasks/abc090_a

$ 3 $ 行 $ 3 $ 列の正方形状のマス目があり、各マスには英小文字がかれています。 上から $ i $ 行目、左から $ j $ 列目のマスにかれた文字は、$ c_{ij} $ です。

マス目の左上と右下をYぶような角上のマス目にかれた文字を、左上からにiんでできる $ 3 $ 文字の文字列を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ c_{11}c_{12}c_{13} $ $ c_{21}c_{22}c_{23} $ $ c_{31}c_{32}c_{33} $

## 输出格式
マス目の左上と右下をYぶような角上のマス目にかれた文字を、左上からにiんでできる $ 3 $ 文字の文字列を出力せよ。

## 题目大意
输入一个$3\ X\ 3$的正方形

从左上角依次输出到右下角（其中的一个对角线）

感谢$@RioBlu$的翻译

```input1
ant
obe
rec
```

```output1
abc
```

```input2
edu
cat
ion
```

```output2
ean
```

## 提示
### 制s

- 入力は英小文字からなる

### Sample Explanation 1

角上のマス目にかれた文字は、左上からに `a`,`b`,`c` です。これらをにiんでできる `abc` を出力します。
