<p>Waca loves maths,.. a lot. He always think that 1 is an unique number. After playing in hours, Waca suddenly realize that every integer can be represented by digit '1', plus operator and minus operator. For example, 1534 can be represented as 1111 + 1 + 111 + 111 - 11 - 11 + 111 + 111. In that case, there are total 7 operators (plus and minus).</p>
<p>Now, Waca wants to know, what is the minimum number of operators needed to achieve X</p>
<h3>Input</h3>
<p>First row is an integer T, the number of test cases.<br>Next T rows will each contain an integer, X, the number given to Waca</p>
<h3>Output</h3>
<p>For each test cases, print an integer, the minimum number of operators needed to achieve X.</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
2<br>1534<br>219

<strong>Output:</strong>
7<br>4</pre>
