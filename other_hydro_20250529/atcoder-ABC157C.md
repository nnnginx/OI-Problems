## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc157/tasks/abc157_c

以下の条件を氦郡 $ 0 $ 以上の整数が存在すれば、それらのうち最小のものを出力してください。そのような整数が存在しなければ、 `-1`と出力してください。

- 十M表で丁度 $ N $ 桁である。($ 0 $ は $ 1 $ 桁の整数とする。その他の整数については、先^に $ 0 $ をつけた表はJめない。)
- 左から数えて $ s_i $ 桁目は $ c_i $ である。$ \left(i\ =\ 1,\ 2,\ \cdots,\ M\right) $

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ s_1 $ $ c_1 $ $ \vdots $ $ s_M $ $ c_M $

## 输出格式
答えを出力せよ。

## 题目大意
定义“好数”是恰好有 $n$ 位，并且从左往右数第 $s_i$ 位是 $c_i$ 的数。

你需要输出最小的“好数”，若没有则输出```-1```。

```input1
3 3
1 7
3 2
1 7
```

```output1
702
```

```input2
3 2
2 1
2 3
```

```output2
-1
```

```input3
3 1
1 0
```

```output3
-1
```

## 提示
### 制s

- 入力は全て整数
- $ 1\ \leq\ N\ \leq\ 3 $
- $ 0\ \leq\ M\ \leq\ 5 $
- $ 1\ \leq\ s_i\ \leq\ N $
- $ 0\ \leq\ c_i\ \leq\ 9 $

### Sample Explanation 1

$ 702 $ の左から $ 1 $ 桁目は `7` であり、 $ 3 $ 桁目は `2` ですから、 $ 702 $ はの条件を氦郡筏蓼埂￥蓼俊 $ 701 $ 以下の非整数はの条件を氦郡筏蓼护蟆
