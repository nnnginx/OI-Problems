## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc178/tasks/abc178_d

整数 $ S $ が与えられます。 すべてのが $ 3 $ 以上の整数で、そのt和が $ S $ であるような数列がいくつあるか求めてください。ただし、答えは非常に大きくなる可能性があるので、 $ 10^9+7 $ で割った余りを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
答えを出力せよ。

## 题目大意
给定整数 $ S $。请求出有几个序列，所有项都是 $ 3 $ 以上的整数，其总和是 $ S $。因为答案可能会非常大，所以请输出除以$ 1e9+7 $ 后的余数。

#### 输入格式
一个整数$ S $

#### 输出格式

答案除以$ 1e9+7 $后的余数

```input1
7
```

```output1
3
```

```input2
2
```

```output2
0
```

```input3
1729
```

```output3
294867501
```

## 提示
### 制s

- $ 1\ \leq\ S\ \leq\ 2000 $
- 入力はすべて整数

### Sample Explanation 1

数列 $ \{3,4\} $,$ \{4,3\} $,$ \{7\} $ の $ 3 $ つが条件を氦郡筏蓼埂

### Sample Explanation 2

条件を氦郡故列は存在しません。
