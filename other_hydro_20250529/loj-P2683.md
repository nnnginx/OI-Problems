## 题目描述

一个数是**非回文数**当且仅当不包含长度大于 $1$ 的回文数。比如 $16276$ 是无回文数，而 $17276$ 因为含有 $727$ 而不是。

求区间内有多少个**非回文数**。

## 输入格式

一行两个整数 $L, R$（$0 \le L \le R \le 10^{18}$）。

## 输出格式

非回文数数量。

```input1
123 321
```

```output1
153
```

```input2
123456789 987654321
```

```output2
167386971
```

## 数据范围与提示

对于 $25\%$ 的测试数据，$b − a \le 100 000$.

对于所有测试数据，$0 \le L \le R \le 10^{18}$.

翻译来自 abcdabcd987。
