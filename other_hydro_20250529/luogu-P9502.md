## 题目背景
> 目标越坚定的魔法士，越幸运。――殿堂魔法士 S

## 题目描述
初级魔法士小 M 的魔法数字是 $2$。

给定一个正整数 $n$，小 M 需要找到最大的 **偶数** $m$，使得 $2^m<n$。

请你帮帮他。

## 输入格式
一行一个正整数 $n$。

## 输出格式
一行一个 **偶数** $m$，表示答案。

```input1
7
```

```output1
2
```

```input2
8
```

```output2
2
```

```input3
1024
```

```output3
8
```

## 提示
**【数据范围】**

对于前 $20\%$ 的数据，$n = 2$。

对于前 $60\%$ 的数据，$n$ 为 $2$ 的幂次。

对于 $100\%$ 的数据，$2 \le n \le 10^9$。
