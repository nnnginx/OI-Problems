## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc222/tasks/abc222_a

$ 0 $ 以上 $ 9999 $ 以下の整数 $ N $ が与えられます。

$ N $ の先^に必要なだけ $ 0 $ をつけ、$ 4 $ 桁の文字列にしたものを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
输入自然数 $N(0\le N\le9999)$ 并输出，若不足 $4$ 位，用前导 0 补足后再输出。

```input1
321
```

```output1
0321
```

```input2
7777
```

```output2
7777
```

```input3
1
```

```output3
0001
```

## 提示
### 制s

- $ 0\ \leq\ N\ \leq\ 9999 $
- $ N $ は整数

### Sample Explanation 1

$ 321 $ は $ 3 $ 桁なので、先^に $ 1 $ つ $ 0 $ をつけると $ 4 $ 桁になります。
