## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc297/tasks/abc297_b

> 高蚓は chess960 と呼ばれるゲ`ムで[んでいます。 高蚓はランダムにQめた初期配置が chess960 の条件を氦郡工_Jするプログラムをくことにしました。

Lさ $ 8 $ の文字列 $ S $ が与えられます。$ S $ には `K`, `Q` がちょうど $ 1 $ 文字ずつ、`R`, `B`, `N` がちょうど $ 2 $ 文字ずつ含まれます。 $ S $ が以下の条件を全て氦郡筏皮い毪判定してください。

- $ S $ において左から $ x,y\ (x\ <\ y) $ 文字目が `B` であるとする。このとき $ x $ と $ y $ の偶奇がなる。
- `K` は $ 2 $ つの `R` のgにある。より形式的には、$ S $ において左から $ x,y\ (x\ <\ y) $ 文字目が `R` であり、 $ z $ 文字目が `K` であるとする。このとき、 $ x\ <\ z\ <\ y $ が成り立つ。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ が条件を氦郡龊 `Yes` を、そうでない龊 `No` を出力せよ。

## 题目大意
给定一个长度为 $8$ 的字符串 $S$，其中 `R`，`B`，`N` 各有 $2$ 个，`Q` 和 `K` 各 $1$ 个 。

一个好的字符串满足两点要求：

1. `K` 在两个 `R` 之间；

2. 两个 `B` 的位置奇偶性不同。

判断 $S$ 是否为好的字符串，如果是则输出 `Yes`，否则输出 `No`。

Translate by [yanhao40340](https://www.luogu.com.cn/user/674721)。

```input1
RNBQKBNR
```

```output1
Yes
```

```input2
KRRBBNNQ
```

```output2
No
```

```input3
BRKRBQNN
```

```output3
No
```
