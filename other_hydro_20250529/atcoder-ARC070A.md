## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc056/tasks/arc070_a

o限に左右に伸びている数直上の $ 0 $ の地点にr刻 $ 0 $ にカンガル`がいます。 カンガル`はr刻 $ i-1 $ から $ i $ にかけて、なにもしないか、もしくはLさがちょうど $ i $ のジャンプを、左右どちらかの方向をxんで行えます。 つまり、r刻 $ i-1 $ に座 $ x $ にいたとすると、r刻 $ i $ には $ x-i $, $ x $, $ x+i $ のどれかに存在することが出来ます。 カンガル`の家は座 $ X $ にあります。カンガル`はできるだけ早く座 $ X $ まで移婴筏瑜Δ趣筏皮い蓼埂 カンガル`が座 $ X $ に到着するr刻の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $

## 输出格式
カンガル`が座 $ X $ に到着するr刻の最小を出力せよ。

## 题目大意
在0秒的时候有一只袋鼠在左右无限长的数轴上的原点上。在i-1到i的时间内，袋鼠可以选择不动，也可以向任意方向跳i个单位长度。也就是说，如果袋鼠在坐标x，时间i-1到i的时候，可以存在x-i,x,x+i三点之中。袋鼠的家在坐标X。袋鼠想尽快移动到它家。求袋鼠到达家的时间的最小值。

输入格式：

输入由标准输入以下列格式给出：$ X $ 

输出：

袋鼠到达坐标的最早时间


```input1
6
```

```output1
3
```

```input2
2
```

```output2
2
```

```input3
11
```

```output3
5
```

## 提示
### 制s

- $ X $ は整数
- $ 1QXQ10^9 $

### Sample Explanation 1

$ 3 $ 回右にジャンプするとr刻 $ 3 $ に家にたどり着けて、これが最小です。

### Sample Explanation 2

r刻 $ 0 $ にはなにもせず、r刻 $ 1 $ に右にジャンプすることでr刻 $ 2 $ に家にたどり着けます。
