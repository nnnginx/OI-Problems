## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc131/tasks/arc131_c

机の上に $ N $ 枚のクッキ`があります。クッキ`の表面にはそれぞれ正の整数 $ A_1,\ A_2,\ \dots,\ A_N $ がかれており、これらはすべてなります。

このクッキ`を使って 2 人でゲ`ムを行います。このゲ`ムでは、各プレイヤ`は次の行婴蚪换イ诵肖い蓼埂

> 机にあるクッキ`を 1 枚xんで食べる。  
>  そのHに、机に残ったクッキ`にかれた整数の $ \mathrm{XOR} $ が $ 0 $ になったならば、そのプレイヤ`は倮し、ゲ`ムはK了する。

あなたは E869120 君に椁蛏辘忿zみました。あなたは先手で、E869120 君は後手です。さて、I者が最mに行婴筏郡趣に、あなたは E869120 君に伽沥蓼工？

  $ \mathrm{XOR} $ とは  整数 $ A,\ B $ のビットg位 XOR、$ A\ \mathrm{XOR}\ B $ は、以下のように定xされます。

- $ A\ \mathrm{XOR}\ B $ を二M表したHの $ 2^k $ ($ k\ \geq\ 0 $) の位の数は、$ A,\ B $ を二M表したHの $ 2^k $ の位の数のうち一方のみが $ 1 $ であれば $ 1 $、そうでなければ $ 0 $ である。
 
 例えば、$ 3\ \mathrm{XOR}\ 5\ =\ 6 $ となります (二M表すると: $ 011\ \mathrm{XOR}\ 101\ =\ 110 $)。  
 一般に、$ k $ の整数 $ p_1,\ p_2,\ p_3,\ \dots,\ p_k $ のビットg位 XOR は $ (\dots\ ((p_1\ \mathrm{XOR}\ p_2)\ \mathrm{XOR}\ p_3)\ \mathrm{XOR}\ \dots\ \mathrm{XOR}\ p_k) $ と定xされ、これは $ p_1,\ p_2,\ p_3,\ \dots\ p_k $ の番によらないことが^明できます。特に $ k\ =\ 0 $ の龊稀$ \mathrm{XOR} $ は $ 0 $ となります。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ A_1 $ $ A_2 $ $ \cdots $ $ A_N $

## 输出格式
I者が最mに行婴筏郡趣にあなたが伽膜胜 `Win`、けるなら `Lose` と出力してください。

## 题目大意
给定 $n$ 堆石子，个数分别为 $A_1,A_2,\cdots,A_n$，两两不同。

两个玩家轮流在上面操作，每次操作将任意一堆石子的个数变为 $0$，当拿走后 $A_1\;\text{XOR}\;A_2\;\text{XOR}\;\cdots\;\text{XOR}\;A_n=0$，则该玩家获胜。

若先手有必胜策略，输出 `Win` ，否则输出 `Lose` 。

```input1
6
9 14 11 3 5 8
```

```output1
Lose
```

```input2
1
131
```

```output2
Win
```

```input3
8
12 23 34 45 56 78 89 98
```

```output3
Win
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 400000 $
- $ 1\ \leq\ A_i\ \leq\ 10^9\ (1\ \leq\ i\ \leq\ N) $
- $ A_1,\ A_2,\ \dots,\ A_N $ はすべてなる
- $ A_1,\ A_2,\ \dots,\ A_N $ の $ \mathrm{XOR} $ は $ 0 $ ではない
- 入力はすべて整数

### Sample Explanation 1

この例では、あなたがどんな方法を使っても、E869120 君が最mに行婴肪Aければけてしまいます。 例えば、最初に $ 11 $ がかれたクッキ`を食べるとしましょう。すると、次に E869120 君が $ 9 $ がかれたクッキ`を食べることで、残ったクッキ`にかれた数 $ 14,\ 3,\ 5,\ 8 $ の $ \mathrm{XOR} $ が $ 0 $ になるので、E869120 君が伽沥蓼埂 それ以外の行婴颏趣盲皮狻⒆罱K的には E869120 君が伽沥蓼埂

### Sample Explanation 2

この例では、あなたは最初のタ`ンで $ 131 $ がかれたクッキ`を食べることしかできません。すると、机の上からクッキ`がなくなるので、残ったクッキ`にかれた数の $ \mathrm{XOR} $ は $ 0 $ になります。したがって、E869120 君が何もできないまま、あなたが伽沥蓼埂
