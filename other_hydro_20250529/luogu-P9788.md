## 题目描述
**译自 [ROIR 2020](http://neerc.ifmo.ru/school/archive/2019-2020.html) Day2 T2.** ***[Планировка участка](http://neerc.ifmo.ru/school/archive/2019-2020/ru-olymp-regional-2020-day2.pdf)***，译者Alpha1022

*译者注：由于原题面有点令译者难以理解，所以此处直接抽象题意。*

你有四个变量 $a,b,c,d$ 须满足：
 - $a,b,c,d \in \mathbb N^*$。
 - $a \ne x, b \ne x$。
 - $a > c, b > d$。
 - $a \cdot b - c \cdot d = n$。

对于给定的 $x,n$，请求出 $a,b,c,d$ 有多少种取值方案。

## 输入格式
第一行，两个整数 $n,x$。  
若 $x=0$，则表示忽略第二个条件。

## 输出格式
一行，表示 $a,b,c,d$ 的取值方案数。

```input1
3 0
```

```output1
1
```

```input2
5 0
```

```output2
5
```

```input3
5 3
```

```output3
2
```

## 提示
#### 【样例 1 解释】
此时只有 $a=2,b=2,c=1,d=1$ 是合法的。

#### 【样例 2 解释】
此时有以下方案是合法的：
- $a=2,b=3,c=1,d=1$；
- $a=2,b=4,c=1,d=3$；
- $a=3,b=2,c=1,d=1$；
- $a=3,b=3,c=2,d=2$；
- $a=4,b=2,c=3,d=1$。

#### 【样例 3 解释】
此时有以下方案是合法的：
- $a=2,b=4,c=1,d=3$；
- $a=4,b=2,c=3,d=1$。
