## 题目描述
[problemUrl]: https://atcoder.jp/contests/panasonic2020/tasks/panasonic2020_e

すぬけ君は、文字列 $ s $ を持っています。 あぬけ君、ぶぬけ君、くぬけ君は次のような方法でそれぞれ文字列 $ a,\ b,\ c $ を得ました。

- $ s $ の空でない ($ s $ 全体であってもよい) BAな部分文字列を一つxぶ。その部分文字列のうちいくつかの文字 ($ 0 $ や全部であってもよい) を `?` で置きQえる。

たとえば、$ s $ が `mississippi` であるとき、部分文字列として `ssissip` をxび、その $ 1,\ 3 $ 文字目を `?` で置きQえることで `?s?ssip` を得ることができます。

文字列 $ a,\ b,\ c $ が与えられます。 $ s $ のLさとして考えられる最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ a $ $ b $ $ c $

## 输出格式
$ s $ のLさとして考えられる最小を出力せよ。

## 题目大意
将一字符串 $s$ 拆分为 $a,b,c$ 三个子串，并选取 $s$ 中的一个非空连续子串，并将其中的一部分字符替换为`?` 。

输入字符串 $a,b,c$ ,求字符串 $s$ 的最小长度。

```input1
a?c
der
cod
```

```output1
7
```

```input2
atcoder
atcoder
???????
```

```output2
7
```

## 提示
### 制s

- $ 1\ \leq\ |a|,\ |b|,\ |c|\ \leq\ 2000 $
- $ a,\ b,\ c $ は英小文字と `?` からなる。

### Sample Explanation 1

たとえば、$ s $ が `atcoder` のとき条件を氦郡筏蓼埂

### Sample Explanation 2

$ a,\ b,\ c $ は相なるとは限りません。
