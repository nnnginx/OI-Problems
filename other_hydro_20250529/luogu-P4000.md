## 题目描述
大家都知道，斐波那契数列是满足如下性质的一个数列：

• $f_1 = 1$

• $f_2 = 1$

• $f_n = f_{n-1} + f_{n-2}$ ($n \geq 2$ 且 $n$ 为整数)

请你求出 $f_n \mod p$ 的值。


## 输入格式
- 第 1 行：一个整数 $n$。

- 第 2 行：一个整数 $p$。


## 输出格式
- 第 1 行：$f_n \mod p$ 的值。


```input1
5
1000000007
```

```output1
5
```

```input2
10
1000000007
```

```output2
55
```

## 提示
对于 $100\%$ 的数据，$n \leq 10^{30000000}, p<2^{31}$
