## 题目描述

给出 $n$ 个点，要你从这些点中找出四个点来组成一个矩形，使其面积最大。

## 输入格式

第一行给出 $n$。

下面 $n$ 行给出这些点的坐标 $(x_i,y_i)$。

## 输出格式

最大的矩形面积。


```input1
8
-2 3
-2 -1
0 3
0 -1
1 -1
2 1
-3 1
-2 1
```


```output1
10
```

## 数据规模与约定

对于 $100\%$ 的数据，满足 $4 \leq n \leq 1500$，$-10^8\leq x_i,y_i\leq 10^8$，保证 $x,y$ 是整数。
