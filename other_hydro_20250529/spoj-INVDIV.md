<p>First, we define ¦Ò(<strong>i</strong>) = Sum of all positive divisors of&nbsp;<strong>i</strong>.</p>
<p>example: all positive divisors of 60 = {1,2,3,4,5,6,10,12,15,20,30,60}</p>
<p>so ¦Ò(60)=1+2+3+4+5+6+10+12+15+20+30+60=168</p>
<p>Now for the task: given an integer <strong>n</strong>&nbsp;find smallest integer <strong>i</strong> such that ¦Ò(<strong>i</strong>)=<strong>n</strong><strong>.</strong></p>
<h3>Input</h3>
<p>The first line is an integer <strong>T</strong>(1 ¡Ü <strong>T</strong> ¡Ü 100,000), denoting the number of test cases.&nbsp;Then, T test cases follow.</p>
<p>For each test case, there is an integer <strong>n</strong>(1 ¡Ü <strong>n</strong>&nbsp;¡Ü 100,000,000) written in one line. (one ineger per line)</p>
<h3>Output</h3>
<p>For each test case, output Smallest Inverse Sum of Divisors&nbsp;of <strong>n</strong>.&nbsp;if <strong>n</strong>&nbsp;doesn't have inverse, output -1.</p>
