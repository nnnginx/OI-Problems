## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc045/tasks/agc045_b

文字列 $ S $ が与えられます． $ S $ の各文字は，`0`,`1`,`?` のいずれかです．

$ S $ に含まれる全ての `?` を `0` か `1` に浃à疲`?` ごとにQ後の文字をxkできます），文字列 $ S' $ を作ることを考えます． ここで，$ S' $ のアンバランス度を，次のように定xします．

- $ S' $ のアンバランス度 $ =\ \max\ \{\ S' $ の $ l $ 文字目から $ r $ 文字目までに含まれる `0` の数と `1` の数の差の~ $ :\ 1\ \leq\ l\ \leq\ r\ \leq\ |S|\} $

$ S' $ のアンバランス度としてありうる最小のを求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ S $

## 输出格式
$ S' $ のアンバランス度としてありうる最小のを出力せよ．

## 题目大意
给定一个由 $0$ , $1$ 和 $?$ 组成的字符串，对于这个这个字符串，我们可以选择将 $?$ 变成 $0$ 或 $1$ 。我们规定这个字符串的不平衡度为 $S$ 。$S$ 满足 $S = max \left\{l到r中0的个数和1的个数的差的绝对值\right\}(1 \leq l,r \leq len)$


你需要最小化最大的平衡度

```input1
0??
```

```output1
1
```

```input2
0??0
```

```output2
2
```

```input3
??00????0??0????0?0??00??1???11?1?1???1?11?111???1
```

```output3
4
```
