## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc237/tasks/abc237_f

以下の条件を全て氦郡故列の数を、$ 998244353 $ で割った余りを求めてください。

- 数列のLさが $ N $
- 数列の各は $ 1 $ 以上 $ M $ 以下の整数
- 最L加部分列のLさがちょうど $ 3 $

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $

## 输出格式
答えを出力せよ。

## 题目大意
请求出满足以下条件的数列的个数：

- 数列长度为 $ N $

- 数列各项为 $ 1 $ 以上 $ M $ 以下的整数

- 最长增加部分列长度正好为 $ 3 $

请输出答案对 $998244353$ 取模的结果。

```input1
4 5
```

```output1
135
```

```input2
3 4
```

```output2
4
```

```input3
111 3
```

```output3
144980434
```

## 提示
### 注

数列の**部分列**とは、数列から $ 0 $ 以上の要素を取り除いた後、残りの要素を元の序でBYして得られる数列のことをいいます。  
 例えば、$ (10,30) $ は $ (10,20,30) $ の部分列ですが、$ (20,10) $ は $ (10,20,30) $ の部分列ではありません。

数列の**最L加部分列**とは、数列の**狭x**g{加な部分列のうち、Lさが最大のもののことをいいます。

### 制s

- $ 3\ \leq\ N\ \leq\ 1000 $
- $ 3\ \leq\ M\ \leq\ 10 $
- 入力は全て整数である

### Sample Explanation 1

例えば $ (3,4,1,5) $ は条件を氦郡故列です。 一方 $ (4,4,1,5) $ は最L加部分列のLさが $ 2 $ なので条件を氦郡筏蓼护蟆

### Sample Explanation 3

$ 998244353 $ で割った余りを求めてください。
