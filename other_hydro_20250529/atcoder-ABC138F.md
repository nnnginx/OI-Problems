## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc138/tasks/abc138_f

整数 $ L,\ R $ が与えられます。整数のM $ (x,\ y) $ $ (L\ \leq\ x\ \leq\ y\ \leq\ R) $ であって、$ y $ を $ x $ で割った余りが $ y\ \text{\ XOR\ }\ x $ に等しいものの数を $ 10^9\ +\ 7 $ で割ったあまりを求めてください。

 $ \text{\ XOR\ } $ とは 整数 $ A,\ B $ のビットごとの排他的理和 $ a\ \text{\ XOR\ }\ b $ は、以下のように定xされます。

- $ a\ \text{\ XOR\ }\ b $ を二M数表したHの $ 2^k $ ($ k\ \geq\ 0 $) の位の数は、$ A,\ B $ を二M数表したHの $ 2^k $ の位の数のうち一方のみが $ 1 $ であれば $ 1 $、そうでなければ $ 0 $ である。
 
 例えば、$ 3\ \text{\ XOR\ }\ 5\ =\ 6 $ となります (二M数表すると: $ 011\ \text{\ XOR\ }\ 101\ =\ 110 $)。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ L $ $ R $

## 输出格式
条件を氦郡拐数のM $ (x,\ y) $ $ (L\ \leq\ x\ \leq\ y\ \leq\ R) $ の数を $ 10^9\ +\ 7 $ で割ったあまりを出力せよ。

## 题目大意
找出有多少整数对 $(x,y)$ 满足 $L\leq x\leq y\leq R$，$y\bmod x=x\oplus y$。  
其中 $\oplus$ 表示异或运算。

输出答案对 $10^9+7$ 取模的结果。

```input1
2 3
```

```output1
3
```

```input2
10 100
```

```output2
604
```

```input3
1 1000000000000000000
```

```output3
68038601
```

### Sample Explanation 3
