## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc131/tasks/abc131_b

$ N $ のリンゴがあります。これらはそれぞれリンゴ $ 1 $、リンゴ $ 2 $、リンゴ $ 3 $、...、リンゴ $ N $ と呼ばれており、リンゴ $ i $ の「味」は $ L+i-1 $ です。「味」はになることもありえます。

また、$ 1 $ 以上のリンゴを材料として、アップルパイをつくることができます。その「味」は、材料となったリンゴの「味」のt和となります。

あなたはこれらのリンゴを全て材料として、アップルパイをつくる予定でしたが、おなかがすいたので $ 1 $ だけ食べることにしました。勿、食べてしまったリンゴはアップルパイの材料にはできません。

つくる予定だったアップルパイとできるだけ同じものをつくりたいので、$ N $ のリンゴ全てを材料としてできるアップルパイの「味」と、食べていない $ N-1 $ のリンゴを材料としてできるアップルパイの「味」の差の~ができるだけ小さくなるように、食べるリンゴをxぶことにしました。

このようにしてxばれたリンゴを食べたr、食べていない $ N-1 $ のリンゴを材料としてできるアップルパイの「味」を求めてください。

なお、このは一意に定まることが^明できます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ L $

## 输出格式
最mに食べるリンゴをxんだrの、食べていない $ N-1 $ のリンゴを材料としてできるアップルパイの「味」を出力してください。

## 题目大意
有 $ N $ 个苹果。苹果 $ i $ 的“味道”是 $ L+i-1 $ 。“味道”也有可能是负的。

你可以用一个以上的苹果来制作苹果派。苹果派的“味道”是作为苹果的“味道”的总和。

因为肚子饿了，所以你吃了一个苹果。吃下去的苹果不能作为苹果派的材料。

想要尽量做出与预定的苹果派相同，即用没吃的 $ N-1 $ 个苹果做出的苹果派的“味道”要与用全部 $ N $ 个苹果做出的苹果派的“味道”的绝对值的差尽可能小。 

吃掉选出的最合适苹果，计算以未吃的 $ N-1 $ 个苹果做出的苹果派的“味道”。

另外，可以证明这个值是唯一确定的。

```input1
5 2
```

```output1
18
```

```input2
3 -1
```

```output2
0
```

```input3
30 -50
```

```output3
-1044
```

## 提示
### 制s

- $ 2\ \leqq\ N\ \leqq\ 200 $
- $ -100\ \leqq\ L\ \leqq\ 100 $
- 入力は全て整数である。

### Sample Explanation 1

リンゴ $ 1,2,3,4,5 $ の「味」は、それぞれ $ 2,3,4,5,6 $ です。リンゴ $ 1 $ を食べるのが最mで、答えは $ 3+4+5+6=18 $ となります。

### Sample Explanation 2

リンゴ $ 1,2,3 $ の「味」は、それぞれ $ -1,0,1 $ です。リンゴ $ 2 $ を食べるのが最mで、答えは $ (-1)+1=0 $ となります。
