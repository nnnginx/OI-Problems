## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc269/tasks/abc269_d

以下のような、o限に冥ち角形のグリッドがあります。最初、全てのマスは白です。

![](https://cdn.luogu.com.cn/upload/vjudge_pic/AT_abc269_d/992f7292cb6316e33ee0c40605e4a519c5d857df.png)

ある六角形のマスは $ 2 $ つの整数 $ i,j $ を用いて $ (i,j) $ と表されます。  
 マス $ (i,j) $ は以下の $ 6 $ つのマスとO接します。

- $ (i-1,j-1) $
- $ (i-1,j) $
- $ (i,j-1) $
- $ (i,j+1) $
- $ (i+1,j) $
- $ (i+1,j+1) $

高颏んは、 $ N $ のマス $ (X_1,Y_1),(X_2,Y_2),\dots,(X_N,Y_N) $ を\くTりました。  
 \いマスがいくつのBY成分をなすか求めてください。  
 ただし、ある $ 2 $ つの\いマスが同じBY成分に属するとは、この $ 2 $ つの\いマスのgをいくつかのO接する\いマスを{って移婴扦ることを指します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X_1 $ $ Y_1 $ $ X_2 $ $ Y_2 $ $ \vdots $ $ X_N $ $ Y_N $

## 输出格式
答えを整数として出力せよ。

## 题目大意
题目出一张图，图中每个点和 $6$ 个点相邻，问给定的 $n$ 个点能有多少个联通块。联通指两个点通过相邻的 $6$ 个方向互相可达.

```input1
6
-1 -1
0 1
0 2
1 0
1 2
2 0
```

```output1
3
```

```input2
4
5 0
4 1
-3 -4
-2 -5
```

```output2
4
```

```input3
5
2 1
2 -1
1 0
3 1
1 -1
```

```output3
1
```
