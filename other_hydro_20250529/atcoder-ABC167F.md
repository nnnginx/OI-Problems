## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc167/tasks/abc167_f

以下のいずれかの条件を氦郡刮淖至肖**括弧列**と定xします。

1. 空文字列
2. ある括弧列 $ A $ が存在して、`(`, $ A $, `)` をこのにBYした文字列
3. ある空でない括弧列 $ A $, $ B $ が存在して、$ A $, $ B $ をこのにBYした文字列

$ N $ の文字列 $ S_i $ が与えられます。$ S_i $ 全てを好きな序でBYするとき、括弧列を成することはできますか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S_1 $ $ : $ $ S_N $

## 输出格式
$ S_i $ を任意の序でBYするとき、括弧列を成できるなら `Yes`、できないなら `No` を出力せよ。

## 题目大意
第一行给出一个 $n$ 表示有 $n$ 个只含有 $\texttt{(,)}$ 的括号序列。（$1\le n\le 10^6$）

之后 $n$ 行，每行一个括号序列。

如果可以通过排列这 $n$ 个括号序列使得其成为一个合法括号序列（即左右括号一一匹配），输出 $\text{Yes}$；如果不能，输出 $\text{No}$。

```input1
2
)
(()
```

```output1
Yes
```

```input2
2
)(
()
```

```output2
No
```

```input3
4
((()))
((((((
))))))
()()()
```

```output3
Yes
```

```input4
3
(((
)
)
```

```output4
No
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^6 $
- $ S_i $ の文字列Lの合は $ 10^6 $ 以下
- $ S_i $ は `(`, `)` のみからなる空でない文字列

### Sample Explanation 1

`(()`, `)` のにBYすると括弧列になります。
