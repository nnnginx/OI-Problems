<p>For a pair of integers a and b, the digit sum of the interval [a,b] is defined as the sum of all digits occurring in all numbers between (and including) a and b. For example, the digit sum of [28, 31] can be calculated as:</p>
<p>&nbsp;</p>
<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;2+8 &nbsp;+ &nbsp;2+9 &nbsp;+ &nbsp;3+0 &nbsp;+ &nbsp;3+1 = 28</p>
<p>&nbsp;</p>
<p>Given the numbers a and b, calculate the digit sum of [a,b].</p>
<p>&nbsp;</p>
<h3>Input</h3>
<p>On the first line one positive number: the number of test cases, at most 100.</p>
<p>After that per test case:</p>
<ul>
<li>&nbsp;one line with two space-separated integers, a and b (0 &lt;= a &lt;= b &lt;= 10^15).</li>
</ul>
<h3>Output</h3>
<p>Per test case:</p>
<ul>
<li>&nbsp;one line with an integer: the digit sum of [a,b]; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</li>
</ul>
<h3>Example</h3>
<pre><strong>Input:</strong>
3
0 10
28 31
1234 56789

<strong>Output:</strong>
46
28
1128600</pre>
