## 题目描述
把 $m$ 个同样的苹果放在 $n$ 个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的分法。（$5,1,1$ 和 $1,1,5$ 是同一种方法）


## 输入格式
第一行是测试数据的数目 $t$，以下每行均包括二个整数 $m$ 和 $n$，以空格分开。

## 输出格式
对输入的每组数据 $m$ 和 $n$，用一行输出相应的结果。


```input1
1
7 3
```

```output1
8
```

```input2
3
3 2
4 3
2 7

```

```output2
2
4
2

```

## 提示
对于所有数据，保证：$1\leq m,n\leq 10$，$0 \leq t \leq 20$。
