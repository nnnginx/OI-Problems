## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc154/tasks/abc154_b

文字列 $ S $ が与えられます。$ S $ のすべての文字を `x` で置きQえて出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ のすべての文字を `x` で置きQえて出力せよ。

## 题目大意
给出一个字符串 $S$ ，请将 $S$ 中每一个字符都替换为 ```x```，然后输出。

```input1
sardine
```

```output1
xxxxxxx
```

```input2
xxxx
```

```output2
xxxx
```

```input3
gone
```

```output3
xxxx
```

## 提示
### 制s

- $ S $ は英小文字のみからなる文字列
- $ S $ のLさは $ 1 $ 以上 $ 100 $ 以下

### Sample Explanation 1

`sardine` のすべての文字を `x` で置きQえると `xxxxxxx` となります。
