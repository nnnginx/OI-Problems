## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc130/tasks/arc130_f

$ N $ からなる正整数列 $ A\ =\ (A_1,\ A_2,\ \ldots,\ A_N) $ が与えられます。

あなたはこの数列にして、次の操作を何度でも行うことができます。

- $ 1\leq\ i\ <\ j\ <\ k\ \leq\ N $ かつ $ j\ =\ \frac{i+k}{2} $ となる整数 $ i,\ j,\ k $ をxぶ。$ A_j $ を $ \lfloor\frac{A_i+A_k}{2}\rfloor $ に置きQえる。

操作後の $ \sum_{i=1}^N\ A_i $ としてありうる最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $

## 输出格式
答えを出力してください。

## 题目大意
给定一个数组，可以进行以下操作任意多次。

选定 $1\leq i<j<k\leq N$ ，将 $a_j$ 改为 $\lfloor\frac{A_i+A_k}{2}\rfloor$ 。

求数组所有数的和的最小值。

```input1
5
2 2 5 5 4
```

```output1
13
```

```input2
5
3 1 4 1 5
```

```output2
11
```

```input3
3
3 1 3
```

```output3
7
```

```input4
3
3 5 3
```

```output4
9
```

## 提示
### 制s

- $ 3\leq\ N\leq\ 3\times\ 10^5 $
- $ 1\leq\ A_i\leq\ 10^{12} $

### Sample Explanation 1

次のように操作を行うことで、$ \sum_{i=1}^N\ A_i\ =\ 13 $ をgFできます。 - $ (i,j,k)\ =\ (1,3,5) $ として操作を行う。数列 $ A $ は $ (2,2,3,5,4) $ へと浠する。 - $ (i,j,k)\ =\ (3,4,5) $ として操作を行う。数列 $ A $ は $ (2,2,3,3,4) $ へと浠する。 - $ (i,j,k)\ =\ (2,3,4) $ として操作を行う。数列 $ A $ は $ (2,2,2,3,4) $ へと浠する。
