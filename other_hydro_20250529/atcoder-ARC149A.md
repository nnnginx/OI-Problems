## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc149/tasks/arc149_a

正整数 $ N,\ M $ が与えられます．次の条件をすべて氦郡拐整数 $ X $ のうち，最大であるものを答えてください．

- $ X $ は $ 10^N $ 未氦握整数で，$ X $ を $ 10 $ M法表したときのどの桁の数字も同じである．
- $ X $ は $ M $ の倍数である．

ただし，条件を氦郡拐整数 $ X $ が存在しない龊悉摔 `-1` と出力してください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられます．

> $ N $ $ M $

## 输出格式
条件をすべて氦郡拐整数 $ X $ のうち最大であるものを出力してください．ただし，そのような正整数 $ X $ が存在しない龊悉摔 `-1` と出力してください．

## 题目大意
问满足以下条件的小于 $10 ^ n$ 数最大是多少？

+ 每一位数字均相同；

+ 是 $m$ 的倍数。

$1\le n\le 10^5$，$1\le m\le 10^9$。

```input1
7 12
```

```output1
888888
```

```input2
9 12
```

```output2
888888888
```

```input3
1 3
```

```output3
9
```

```input4
1000 25
```

```output4
-1
```

```input5
30 1
```

```output5
999999999999999999999999999999
```

## 提示
### 制s

- $ 1\leq\ N\leq\ 10^5 $
- $ 1\leq\ M\leq\ 10^9 $

### Sample Explanation 1

条件を氦郡拐整数 $ X $ は，$ 444,\ 888,\ 444444,\ 888888 $ の $ 4 $ つです．このうち最大のものである $ 888888 $ が答となります．

### Sample Explanation 2

条件を氦郡拐整数 $ X $ は，$ 444,\ 888,\ 444444,\ 888888,\ 444444444,\ 888888888 $ の $ 6 $ つです．

### Sample Explanation 3

条件を氦郡拐整数 $ X $ は，$ 3,\ 6,\ 9 $ の $ 3 $ つです．

### Sample Explanation 4

条件を氦郡拐整数 $ X $ は存在しません．
