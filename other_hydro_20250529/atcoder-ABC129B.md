## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc129/tasks/abc129_b

$ 1 $ から $ N $ の番号がついた $ N $ の重りがあり、番号 $ i $ の重りの重さは $ W_i $ です。

ある整数 $ 1\ \leq\ T\ <\ N $ にしてこれらの重りを、番号が $ T $ 以下の重り と 番号が $ T $ より大きい重りの $ 2 $ グル`プに分けることを考え、それぞれのグル`プの重さの和を $ S_1,\ S_2 $ とします。

このような分け方全てを考えたr、$ S_1 $ と $ S_2 $ の差の~の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ W_1 $ $ W_2 $ $ ... $ $ W_{N-1} $ $ W_N $

## 输出格式
$ S_1 $ と $ S_2 $ の差の~の最小を出力せよ。

## 题目大意
有 $n$ 个正整数，第 $i$ 个数被记作 $w_i$ 。同时，还有一个正整数 $t$ （ $1≤t≤n$ ）和两个集合，分别被称作 $s_1$ 和 $s_2$ 。前 $t$ 个数会被放到 $s_1$ 里，其余的数会被放到 $s_2$ 里。记 $a$ 和 $b$ 分别为 $s_1$ 和 $s_2$ 里的所有元素之和，请编程输出 $|a-b|$ 的最小值。

```input1
3
1 2 3
```

```output1
0
```

```input2
4
1 3 1 1
```

```output2
2
```

```input3
8
27 23 76 2 3 5 62 52
```

```output3
2
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ W_i\ \leq\ 100 $
- 入力は全て整数である

### Sample Explanation 1

$ T\ =\ 2 $ としたとき、$ S_1\ =\ 1\ +\ 2\ =\ 3,\ S_2\ =\ 3 $ となり、差の~は $ 0 $ となります。

### Sample Explanation 2

$ T\ =\ 2 $ としたとき、$ S_1\ =\ 1\ +\ 3\ =\ 4,\ S_2\ =\ 1\ +\ 1\ =\ 2 $ となり、差の~は $ 2 $ です。これより差の~を小さくすることは出来ません。
