## 题目描述
[problemUrl]: https://atcoder.jp/contests/tokiomarine2020/tasks/tokiomarine2020_c

数直上に球が $ N $ Kんでおり、球には左からに $ 1 $ から $ N $ までの番号がついています。 球 $ i $ は座 $ i $ にあります。

球には光のさを表す非整数が定まっており、 座 $ x $ に光のさ $ d $ の球があるとき、その球は座 $ x-d-0.5 $ から座 $ x+d+0.5 $ までの区gを照らします。 初めは球 $ i $ の光のさは $ A_i $ です。 そこで、以下の操作を $ K $ 回Rり返し行います。

- $ 1 $ 以上 $ N $ 以下の各整数 $ i $ にし、操作rに座 $ i $ を照らしている球の数を $ B_i $ とする。そして、各球 $ i $ の光のさを $ B_i $ に涓する。

$ K $ 回の操作を行った後の各球の光のさを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $

## 输出格式
$ K $ 回の操作を行った後の球 $ i $ の光のさ $ A{'}_i $ を、以下の形式で食隽Δ顺隽Δ护琛

> $ A{'}_1 $ $ A{'}_2 $ $ \ldots $ $ A{'}_N $

## 题目大意
# Lamps

## 题目描述

[problemUrl]: https://atcoder.jp/contests/tokiomarine2020/tasks/tokiomarine2020_c

给定 $ N $ 个灯泡，其亮度分别为 $ A_i $。每个灯泡的作用范围为 $ i-A_i-0.5$ 到 $ i+A_i+0.5 $。

有 $ K $ 轮操作。

每轮操作使得每个灯泡的亮度更改为照亮它的灯泡的个数。

## 输入格式

输入格式如下 

> $ N $ 
> $ K $ 
> $ A_i $

## 输出格式

输出更改后的 $ A_i $ 

> $ A{'}_1 $ $ A{'}_2 $ $ \ldots $ $ A{'}_N $

## 样例 #1

### 样例输入 #1

```
5 1
1 0 0 1 0
```

### 样例输出 #1

```
1 2 2 1 2
```

## 样例 #2

### 样例输入 #2

```
5 2
1 0 0 1 0
```

### 样例输出 #2

```
3 3 4 4 3
```

## 提示

### 数据范围 

- $ 1\ \leqq\ N\ \leqq\ 2\ \times\ 10^5 $
- $ 1\ \leqq\ K\ \leqq\ 2\ \times\ 10^5 $
- $ 0\ \leqq\ A_i\ \leqq\ N $

### 样例1解释 

$ 1 $ 号数只有第 $ 1 $ 个数本身作用 ，第 $ 2 $ 个数有第 $ 1 $ 和 第 $ 2 $ 个数作用，以此类推。

```input1
5 1
1 0 0 1 0
```

```output1
1 2 2 1 2
```

```input2
5 2
1 0 0 1 0
```

```output2
3 3 4 4 3
```

### 制s

- $ 1\ \leqq\ K\ \leqq\ 2\ \times\ 10^5 $

### Sample Explanation 1

始めに座 $ 1 $ を照らしている球は球 $ 1 $ のみであるので、操作後の球 $ 1 $ のさは $ 1 $ になります。 また、始めに座 $ 2 $ を照らしている球は球 $ 1 $ と球 $ 2 $ であるので、操作後の球 $ 2 $ のさは $ 2 $ になります。
