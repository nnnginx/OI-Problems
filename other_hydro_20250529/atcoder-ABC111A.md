## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc111/tasks/abc111_a

猫のすぬけは文字をくをしています。 すぬけは今日、数字の `1` と `9` をくをしていたのですが、 g`えて `1` と `9` をあべこべにいてしまいました。

すぬけがいた $ 3 $ 桁の整数 $ n $ が与えられます。 $ n $ に含まれる `1` という桁をそれぞれ `9` に、 `9` という桁をそれぞれ `1` に置きQえて得られる整数を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ n $

## 输出格式
$ n $ の各桁の $ 1 $ と $ 9 $ を入れ替えた整数を出力してください。

## 题目大意
### 题目描述
某位OIer在练习写字，但是他不小心把1和9写反了，也就是说他把1写成了9,9写成了1。现在请你帮他改过来。

### 输入格式
输入他写的字$n$，只包含1和9。

### 输出格式
输出一行，是修正好的字。

## 提示
$111 \le n \le 999$

```input1
119
```

```output1
991
```

```input2
999
```

```output2
111
```

## 提示
### 制s

- $ 111\ \leq\ n\ \leq\ 999 $
- $ n $ は各桁が `1` か `9` である整数

### Sample Explanation 1

一の位の `9` を `1` に、十の位の `1` を `9` に、百の位の `1` を `9` にきQえた `991` が答えとなります．
