## 题目描述

求有多少棵大小为 $n$ 的深度为 $h$ 的二叉树。（树根深度为 $0$；左右子树有别；答案对 $10^9+7$ 取模）

## 输入格式

第一行一个整数 $T$，表示数据组数。  
以下 $T$ 行，每行两个整数 $n$ 和 $h$。

## 输出格式

共 $T$ 行，每行一个整数表示答案（对 $10^9+7$ 取模）。

```input1
2
2 1
3 2
```

```output1
2
4
```

## 数据规模和约定

对于 $100\%$ 的数据，$1 \le T \le 10$，$1 \le n \le 600$，$0 \le h \le 600$。
