## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc114/tasks/abc114_b

数字 `1`, `2`, $ ... $, `9` からなる文字列 $ S $ があります。 ダックスフンドのルンルンは、$ S $ からBAする $ 3 $ の数字を取り出し、 $ 1 $ つの整数 $ X $ としてご主人の元に持っていきます。（数字の番を浃à毪长趣悉扦ません。）

ご主人が大好きな数は $ 753 $ で、これに近い数ほど好きです。 $ X $ と $ 753 $ の差（の~）は最小でいくつになるでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ X $ と $ 753 $ の差としてありうる最小を出力せよ。

## 题目大意
给出一个整数 $n$ 。

求 $n$ 中连续三位数字按原顺序组成的三位数与 $753$ 的差的绝对值的最小值。输出它。

```input1
1234567876
```

```output1
34
```

```input2
35753
```

```output2
0
```

```input3
1111111111
```

```output3
642
```

## 提示
### 制s

- $ S $ はLさ $ 4 $ 以上 $ 10 $ 以下の文字列である。
- $ S $ の各文字は `1`, `2`, $ ... $, `9` のいずれかである。

### Sample Explanation 1

$ 7 $ 文字目から $ 9 $ 文字目までを取り出すと $ X\ =\ 787 $ となり、これと $ 753 $ との差は $ 787\ -\ 753\ =\ 34 $ です。$ X $ をどこから取り出しても、差をより小さくすることはできません。 なお、数字の番を浃à毪长趣悉扦ません。例えば、`567` を取り出して `765` にKび浃à皮悉い堡蓼护蟆 また、$ S $ からBAしていない $ 3 $ 文字を取り出すこともできません。例えば、$ 7 $ 文字目の `7`、$ 9 $ 文字目の `7` と $ 10 $ 文字目の `6` を取り出して `776` としてはいけません。

### Sample Explanation 2

`753` そのものを取り出すことができる龊稀⒋黏à $ 0 $ です。

### Sample Explanation 3

どこから $ 3 $ 文字を取り出しても $ X\ =\ 111 $ となり、差は $ 753\ -\ 111\ =\ 642 $ です。
