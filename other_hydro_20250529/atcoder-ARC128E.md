## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc128/tasks/arc128_e

Lさ $ N $ の整数列 $ A=(A_1,A_2,\cdots,A_N) $，及び整数 $ K $ が与えられます．

以下の条件をI方氦郡拐数列 $ x $ を作ることを考えます．

- 各整数 $ i $ ($ 1\ \leq\ i\ \leq\ N $) について，$ x $ はちょうど $ A_i $ の $ i $ を含む． また逆に，それ以外の整数を含まない．
- $ x $ の中でBAするどの $ K $ をても，その $ K $ のはすべてなる．

条件を氦郡 $ x $ を作ることが可能かどうか判定し，可能な龊悉咸跫を氦郡怪肖谴最小の $ x $ を求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ K $ $ A_1 $ $ A_2 $ $ \cdots $ $ A_N $

## 输出格式
条件を氦郡故列 $ x $ が作ることが不可能な龊希`-1` と出力せよ． 可能な龊希辞最小の $ x $ を出力せよ．

## 题目大意
给你一个长度为 $n$ 的整数序列 $A$ 以及一个整数 $k$ ，请你构造一个序列 $B$，满足以下两个条件 :

- 值为 $i$ 的数出现了 $A_i$ 次；
- 对于 $1 \leq i \leq n-k+1$ , 满足 $B_i \neq B_{i+1} \neq B_{i+2} \neq \cdots \neq B_{i+k-1}$。

如果无解，输出 $-1$；否则输出你能构造出的字典序最小的序列 $B$。

```input1
3 3
2 2 1
```

```output1
1 2 3 1 2
```

```input2
3 2
2 1 2
```

```output2
1 2 3 1 3
```

```input3
3 3
1 3 3
```

```output3
-1
```

## 提示
### 制s

- $ 2\ \leq\ K\ \leq\ N\ \leq\ 500 $
- $ 1\ \leq\ A_i $
- $ \sum_{1\ \leq\ i\ \leq\ N}\ A_i\ \leq\ 200000 $
- 入力されるはすべて整数である

### Sample Explanation 1

$ x=(1,2,3,1,2),(2,1,3,2,1) $ の二つが条件を氦郡罚その中で辞最小の $ (1,2,3,1,2) $ が答えになります．
