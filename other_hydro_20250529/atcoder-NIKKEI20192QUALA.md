## 题目描述
[problemUrl]: https://atcoder.jp/contests/nikkei2019-2-qual/tasks/nikkei2019_2_qual_a

合が $ N $ となるように相なる $ 2 $ つの正整数をxぶ方法は何通りあるでしょうか (序は考]しません)。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
忽略顺序，有多少种方法可以选择两个不同的正整数，它们的总和为 $N$？

```input1
4
```

```output1
1
```

```input2
999999
```

```output2
499999
```

## 提示
### 制s

- $ 1\ Q\ N\ Q\ 10^6 $
- $ N $ は整数である。

### Sample Explanation 1

合が $ 4 $ となるように相なる $ 2 $ つの正整数をxぶ方法は、$ 1 $ と $ 3 $ をxぶ $ 1 $ 通りのみです。($ 3 $ と $ 1 $ をxぶことはこれと区eしません。)
