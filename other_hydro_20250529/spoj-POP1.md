<p><span style="font-size: small;">A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself .</span></p>
<p><span style="font-size: small;">we  define here a new prime number called prime of primes number (POP) is a  prime number that consist of other prime numbers less than this number .</span></p>
<p><span style="font-size: small;"><strong>example :</strong>&nbsp;</span></p>
<p><span style="font-size: small;">1013 consist of 101 and 3 and both are primes .</span></p>
<p><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;"><strong>notes </strong>:</span></span></p>
<p><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;">2003 is not POP because leading zero not allowed .</span></span></p>
<p><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;">the POP number must contain more than or equal two primes , and overlapping not allowed .</span></span></p>
<h3>Input</h3>
<p>&nbsp;</p>
<div id="_mcePaste" style="position: absolute; left: -10000px; top: 0px; width: 1px; height: 1px; overflow: hidden;">The first line contains an integer T specifying the number of test cases. (T &lt;= 1000) followed by&nbsp;</div>
<div id="_mcePaste" style="position: absolute; left: -10000px; top: 0px; width: 1px; height: 1px; overflow: hidden;">T lines , each line contains an integer m number 0&lt;=m&lt;=10^9 .</div>
<p><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;">The first line contains an integer T specifying the number of test cases. (T &lt;= 10^4) followed by&nbsp;</span></span></p>
<p><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;">T lines , each line contains an integer m number 0&lt;=m&lt;=10^9 .</span></span></p>
<h3>Output</h3>
<p><span style="font-weight: normal;"><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;">For each test case print single line contain the first integer greater than or equal to m and is (POP) .</span></span></span></p>
<h3>Example</h3>
<p><span style="font-size: medium;"><strong><span style="font-family: arial,helvetica,sans-serif;">Input:</span></strong><br></span></p>
<pre>3</pre>
<pre>10</pre>
<pre>100</pre>
<pre>1000</pre>
<p><strong><span style="font-size: small;"><span style="font-family: arial,helvetica,sans-serif;">Output:</span></span></strong></p>
<pre><strong>23</strong></pre>
<pre><strong>113</strong></pre>
<pre><strong>1013</strong></pre>
