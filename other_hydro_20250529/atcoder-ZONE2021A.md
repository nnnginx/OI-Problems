Lさ $ 12 $ の文字列 $ S $ が与えられます。$ S $ の中に `ZONe` という文字列は (BAする部分文字列として) いくつ含まれるでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
答えを出力せよ。

## 题目大意
给出长度为 $12$ 的字符串 $S$。求 $S$ 中 $\text{ZONe}$ 这个字符串包含几个？

```input1
abcdZONefghi
```

```output1
1
```

```input2
ZONeZONeZONe
```

```output2
3
```

```input3
helloAtZoner
```

```output3
0
```
