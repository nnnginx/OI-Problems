## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc164/tasks/abc164_d

`1` から `9` までの数字のみからなる文字列 $ S $ が与えられます。

次のような条件を氦郡拐数のM $ (i,j) $ ($ 1\ <\ =\ i\ <\ =\ j\ <\ =\ |S| $) のt数を求めてください。

条件: $ S $ の $ i $ 文字目から $ j $ 文字目までを $ 10 $ M法の整数としてみると、この整数は $ 2019 $ の倍数である。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
条件を氦郡拐数のM $ (i,j) $ ($ 1\ <\ =\ i\ <\ =\ j\ <\ =\ |S| $) のt数を出力せよ。

## 题目大意
给出一长串数，求其中有多少段数是2019的整数倍。

```input1
1817181712114
```

```output1
3
```

```input2
14282668646
```

```output2
2
```

```input3
2119
```

```output3
0
```

## 提示
### 制s

- $ 1\ <\ =\ |S|\ <\ =\ 200000 $
- $ S $ は `1` から `9` までの数字のみからなる文字列

### Sample Explanation 1

条件を氦郡工韦 $ (1,5),\ (5,9),\ (9,13) $ の $ 3 $ です。

### Sample Explanation 3

条件を氦郡拐数のMは存在しません。
