## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc228/tasks/abc228_a

高蚓は、叭 $ S $ r $ 0 $ 分に部屋の荬颏膜薄叭 $ T $ r $ 0 $ 分に消します。  
 荬颏膜堡皮い腴gに日付が浃铯毪长趣猡ります。

$ X $ r $ 30 $ 分に部屋の荬ついているかどうか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $ $ T $ $ X $

## 输出格式
$ X $ r $ 30 $ 分に部屋の荬ついているならば `Yes` と、そうでなければ `No` と出力せよ。

## 题目大意
高桥家每日 $s$ 时整开灯， $t$ 时整关灯。求 $x$ 点半的时候是不是开着灯？（注： $0≤s,t,x<24$ ，有可能半夜还开着灯）

```input1
7 20 12
```

```output1
Yes
```

```input2
20 7 12
```

```output2
No
```

```input3
23 0 23
```

```output3
Yes
```

## 提示
### 制s

- $ 0\ \leq\ S,\ T,\ X\ \leq\ 23 $
- $ S\ \neq\ T $
- 入力は全て整数である。

### Sample Explanation 1

部屋の荬ついているのは $ 7 $ r $ 0 $ 分から $ 20 $ r $ 0 $ 分までのgです。$ 12 $ r $ 30 $ 分には荬ついているので、`Yes` と出力します。

### Sample Explanation 2

部屋の荬ついているのは $ 0 $ r $ 0 $ 分から $ 7 $ r $ 0 $ 分までのgと、$ 20 $ r $ 0 $ 分から（次の日の）$ 0 $ r $ 0 $ 分までのgです。 $ 12 $ r $ 30 $ 分には荬ついていないので、`No` と出力します。
