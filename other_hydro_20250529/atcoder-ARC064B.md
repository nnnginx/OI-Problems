## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc048/tasks/arc064_b

Lさ $ 3 $ 以上の文字列 $ s $ があります。 $ s $ の中に同一の文字がOり合うw所はありません。

高蚓と青木君がゲ`ムで儇します。 二人は交互に次の操作を行います。 高蚓が先手です。

- $ s $ からI端以外の文字をひとつ取り除く。 ただし、その文字を取り除くことで、$ s $ の中に同一の文字がOり合うw所ができる龊稀その文字を取り除くことはできない。

先に操作を行えなくなった人がけです。 二人が最mに行婴筏郡趣、どちらが伽膜を判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ s $

## 输出格式
先手の高蚓が伽膜胜椁 `First` を、後手の青木君が伽膜胜椁 `Second` を出力せよ。

## 题目大意
给定一个长度大于等于3的字符串$S$,$S$中没有相邻的两个字符相等。

**玩法**:

玩家$1$和玩家$2$交替进行操作――从$S$中删除一个字符,但删除后$S$不能有相邻的两个字符相等,否则另一位玩家获胜(两人都很聪明)

现在玩家$1$先操作,请问谁可以获得胜利?

输入一个字符串$S$

若玩家$1$胜利,输出`first`;玩家$2$获胜,输出`Second`



```input1
aba
```

```output1
Second
```

```input2
abc
```

```output2
First
```

```input3
abcab
```

```output3
First
```

## 提示
### 制s

- $ 3\ <\ =\ |s|\ <\ =\ 10^5 $
- $ s $ は英小文字のみからなる。
- $ s $ の中に同一の文字がOり合うw所はない。

### Sample Explanation 1

先手の高蚓は操作を行うことができません。 なぜならば、$ s $ からI端以外の文字の `b` を取り除くと、$ s $ は `aa` となって `a` がOり合うからです。

### Sample Explanation 2

先手の高蚓が $ s $ から `b` を取り除くと、$ s $ は `ac` となります。 すると、後手の青木君は操作を行うことができません。 なぜならば、$ s $ にはI端以外の文字が存在しないからです。
