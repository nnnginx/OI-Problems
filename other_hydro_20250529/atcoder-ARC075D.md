## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc075/tasks/arc075_d

正の整数 $ n $ にし、$ n $ の十M表（先^に $ 0 $ を付けない）を左右に反させて得られる整数を $ rev(n) $ と表します。例えば、$ rev(123)\ =\ 321 $, $ rev(4000)\ =\ 4 $ です。

正の整数 $ D $ が与えられます。$ rev(N)\ =\ N\ +\ D $ であるような正の整数 $ N $ はいくつ存在するでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ D $

## 输出格式
$ rev(N)\ =\ N\ +\ D $ であるような正の整数 $ N $ の数を出力せよ。

## 题目大意
给定 $D$，求满足 $\mathrm{rev}(N)=N+D$ 的 $N$ 的个数，$\mathrm{rev}(N)$ 表示十进制下将 $N$ 按位翻转并去掉前导 $0$ 后的数。

```input1
63
```

```output1
2
```

```input2
75
```

```output2
0
```

```input3
864197532
```

```output3
1920
```

## 提示
### 制s

- $ D $ は整数である。
- $ 1\ <\ =\ D\ <\ 10^9 $

### Sample Explanation 1

$ rev(N)\ =\ N\ +\ 63 $ であるような正の整数 $ N $ は、$ N\ =\ 18,\ 29 $ の $ 2 $ 存在します。

### Sample Explanation 2

$ rev(N)\ =\ N\ +\ 75 $ であるような正の整数 $ N $ は存在しません。
