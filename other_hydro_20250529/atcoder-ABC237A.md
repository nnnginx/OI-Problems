## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc237/tasks/abc237_a

整数 $ N $ が与えられます。 $ N $ が $ -2^{31} $ 以上かつ $ 2^{31} $ 未氦胜椁 `Yes` を、そうでないならば `No` を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ が $ -2^{31} $ 以上かつ $ 2^{31} $ 未氦胜椁 `Yes` を、そうでないならば `No` を出力せよ。

## 题目大意
给定 $ N $，输出 $ N $ 是否在 ```int``` 范围内，输出 ```Yes``` 或 ```No```。

```input1
10
```

```output1
Yes
```

```input2
-9876543210
```

```output2
No
```

```input3
483597848400000
```

```output3
No
```

## 提示
### 制s

- $ -2^{63}\ \leq\ N\ <\ 2^{63} $
- $ N $ は整数である。

### Sample Explanation 1

$ 10 $ は $ -2^{31} $ 以上かつ $ 2^{31} $ 未氦扦るので、`Yes` を出力します。

### Sample Explanation 2

$ -9876543210 $ は $ -2^{31} $ 未氦扦るので、`No` を出力します。

### Sample Explanation 3

$ 483597848400000 $ は $ 2^{31} $ 以上であるので、`No` を出力します。
