## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc089/tasks/abc089_a

ある学校には、$ N $ 人の生徒がいます。

生徒たちをいくつかのグル`プに分け、グル`プごとにあるテ`マについてし合ってもらうこととなりました。

あなたは、$ 2 $ 人以下のグル`プだと抗的なし合いが出来ないと考えており、なるだけ多くのグル`プを $ 3 $ 人以上にしたいです。

生徒たちを上手く分けて、$ 3 $ 人以上のグル`プの数を最大化してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ 3 $ 人以上のグル`プを最大で $ x $ 作れるとき、$ x $ を出力せよ。

## 题目大意
$AT$班有$N$人（$N$即为你输入的数）

请将他们分组，每组有三人或三人以上

问最多有多少组

感谢$@RioBlu$的翻译

```input1
8
```

```output1
2
```

```input2
2
```

```output2
0
```

```input3
9
```

```output3
3
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 1000 $
- 入力は全て整数

### Sample Explanation 1

例えば、$ 3 $ 人のグル`プと $ 5 $ 人のグル`プに分けるとよいです。

### Sample Explanation 2

どのように生徒たちを分けても $ 3 $ 人以上のグル`プを作れない龊悉猡ります。
