## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc053/tasks/abc053_b

すぬけくんは文字列 $ s $ のBAした一部分(部分文字列という)を取り出して先^が `A` であり末尾が `Z` であるような文字列を作ることにしました。 すぬけくんが作ることのできる文字列の最大のLさを求めてください。 なお，$ s $ には先^が `A` であり末尾が `Z` であるような部分文字列が必ず存在することが保^されます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ s $

## 输出格式
答えを出力せよ。

## 题目大意
题目大意：
输入一个字符串s，问从第一个A到最后一个Z之间有多少个字母（含A和Z）（包括A和Z，如：UYEAERTZUY,就是UYE AERTZ UY,A到Z之间有5个字符。数据保证了开头是A的末尾是Z的部分一定存在的。）
输入格式：
一个字符串s
输出格式：
从A到Z之间有多少个字母


感谢@Andysun06 提供的翻译

```input1
QWERTYASDFZXCV
```

```output1
5
```

```input2
ZABCZ
```

```output2
4
```

```input3
HASFJGHOGAKZZFEGA
```

```output3
12
```

## 提示
### 制s

- $ 1\ Q\ |s|\ Q\ 200{,}000 $
- $ s $ は英大文字のみからなる
- $ s $ には先^が `A` であり末尾が `Z` であるような部分文字列が必ず存在する

### Sample Explanation 1

$ 7 $ 文字目から $ 11 $ 文字目までを取り出して `ASDFZ` を作ると、先^が `A` 末尾が `Z` であるような文字列を得ることが可能です。
