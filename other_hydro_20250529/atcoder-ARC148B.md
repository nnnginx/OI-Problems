## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc148/tasks/arc148_b

`d` と `p` からなるLさ $ L $ の文字列 $ T $ にして、$ T $ を $ 180 $ 度回した文字列を $ f(T) $ と表します。より密には、$ f(T) $ を次の条件を氦郡刮淖至肖趣筏贫à幛蓼埂

- $ f(T) $ は `d` と `p` からなるLさ $ L $ の文字列である。
- $ 1\ \leq\ i\ \leq\ L $ であるすべての整数 $ i $ について、$ f(T) $ の $ i $ 文字目は $ T $ の $ L\ +\ 1\ -\ i $ 文字目となる。

例えば $ T\ = $ `ddddd` のとき $ f(T)\ = $ `ppppp`, $ T\ = $ `dpdppp` のとき $ f(T)= $ `dddpdp` です。

`d` と `p` からなるLさ $ N $ の文字列 $ S $ が与えられます。  
 あなたは次の操作を **$ 0 $ 回以上 $ 1 $ 回以下**行うことができます。

- $ 1\ \leq\ L\ \leq\ R\ \leq\ N $ である整数のM $ (L,\ R) $ を $ 1 $ つxび、$ S $ の $ L $ 文字目から $ R $ 文字目までからなる部分文字列を $ T $ とする。そして、$ S $ の $ L $ 文字目から $ R $ 文字目までを $ f(T) $ に置きQえる。

例えば $ S= $ `dpdpp`, $ (L,R)=(2,4) $ の龊稀$ T= $ `pdp`, $ f(T)= $ `dpd` なので $ S $ は `ddpdp` に浠します。

最K的な $ S $ としてあり得る文字列のうち辞最小のものを出力してください。

  辞とは？文字列 $ S\ =\ S_1S_2\ldots\ S_{|S|} $ が文字列 $ T\ =\ T_1T_2\ldots\ T_{|T|} $ より**辞で小さい**とは、下の 1. と 2. のどちらかが成り立つことを言います。 ここで、$ |S|,\ |T| $ はそれぞれ $ S,\ T $ のLさを表します。

1. $ |S|\ \lt\ |T| $ かつ $ S_1S_2\ldots\ S_{|S|}\ =\ T_1T_2\ldots\ T_{|S|} $。
2. ある整数 $ 1\ \leq\ i\ \leq\ \min\lbrace\ |S|,\ |T|\ \rbrace $ が存在して、下の $ 2 $ つがともに成り立つ。 
  - $ S_1S_2\ldots\ S_{i-1}\ =\ T_1T_2\ldots\ T_{i-1} $
  - $ S_i $ が $ T_i $ よりアルファベットで小さい文字である。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
答えを出力せよ。

## 题目大意
给出一个长度为 $N (1 \le N \le 5000)$ 的字符串 $S$，且 $\forall S_i , S_i \in \{d,p \}$。

定义 $rotate (l,r)$，对于区间 $[l,r]$  的每一个 $i$，重新赋值 $S_i$，使得 $S_i \not= S_{r+l-i}$ （即对称）。

需要执行 $0$ 或 $1$ 次 $rotate (l,r)$（$l,r$ 是任意的），使得 $S$ 字典序最小。

```input1
6
dpdppd
```

```output1
dddpdd
```

```input2
3
ddd
```

```output2
ddd
```

```input3
11
ddpdpdppddp
```

```output3
ddddpdpdddp
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 5000 $
- $ S $ は `d` と `p` からなるLさ $ N $ の文字列
- $ N $ は整数

### Sample Explanation 1

$ (L,\ R)\ =\ (2,\ 5) $ とします。$ T\ = $ `pdpp`, $ f(T)\ = $ `ddpd` なので、操作後の $ S $ は `dddpdd` になります。 得られる文字列のうち `dddpdd` が辞最小なので、これを出力します。

### Sample Explanation 2

操作を行わないことが最mな龊悉猡ります。
