## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc176/tasks/abc176_c

$ N $ 人が $ 1 $ 列にKんでおり、前から $ i $ 番目の人の身Lは $ A_i $ です。

それぞれの人の足元に、高さが $ 0 $ 以上の踏み台をO置し、全ての人が次の条件を氦郡工瑜Δ摔筏郡い扦埂

条件：踏み台をzめて身Lを比^したとき、自分より前に、自分より背の高い人が存在しない

この条件を氦郡rの、踏み台の高さの合の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ \ldots $ $ A_N $

## 输出格式
踏み台の高さの合の最小を出力せよ。

## 题目大意
有 $n$ 个数，第 $i$ 个数是 $a_i$ 。还有一个整数 $m$ ，其初始值为 $0$ 。对于每个 $a_i$ ：如果它比它前面的所有数中的某个数小，那么就将它和 $m$ 同时加上 $1$ ，直至它前面没有比它小的数为止。问在所有操作完成之后 $m$ 的最小值。（ $a_1$ 当然不用动）

```input1
5
2 1 5 4 3
```

```output1
4
```

```input2
5
3 3 3 3 3
```

```output2
0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\times\ 10^5 $
- $ 1\ \leq\ A_i\ \leq\ 10^9 $
- 入力は全て整数

### Sample Explanation 1

それぞれ、高さ $ 0,1,0,1,2 $ の踏み台を与えると、踏み台をzめた身Lは $ 2,2,5,5,5 $ となり、条件を氦郡筏蓼埂 踏み台の高さの合をこれより小さくすることはできません。

### Sample Explanation 2

全Tに高さ $ 0 $ の踏み台を与えればよいです。
