<p>Mary love painting so much, but as we know she can't draw very well.&nbsp;There is no one appreciate her works, so she come up with a puzzle with herself....</p>
<h3>Description</h3>
<p>There are only one case in each input file, the first line is a integer N (N &lt;= 1,000,00) denoted the total operations executed by Mary.&nbsp;Then following N lines, each line is one of the folling operations.</p>
<ul>
<li>D L R: draw a segment [L, R], 1 &lt;= L &lt;= R &lt;= 1,000,000,000</li>
<li>C I: clear the ith added segment. It¡¯s guaranteed that the every added segment will be cleared only once.</li>
<li>Q L R: query the number of segment sharing at least a common point with interval [L, R]. 1 &lt;= L &lt;= R &lt;= 1,000,000,000.</li>
</ul>
<h3>Input</h3>
<p>n</p>
<p>( then following n operations ... )</p>
<h3>Output</h3>
<p>(for each query, print the result on a single line ... )</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
6
D 1 3
D 2 4
Q 2 3
D 2 4
C 2
Q 2 3

<strong>Output:</strong>
2
2
</pre>
