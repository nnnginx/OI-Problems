He uses Taj¨± Kage Bunshin no <span id="qsi4p2hxo_2" style="height: 11px;">Jutsu (Multiple <span id="qsi4p2hxo_3" style="height: 11px;">Shadow Clone Technique</span>) to create maximum n number of clones and he decided to distribute maximum p/2 power to the possible number of clones. <p>Output must contains t lines consisting of the answer as mentioned in above question</p>
