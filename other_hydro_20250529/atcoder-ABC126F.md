## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc126/tasks/abc126_f

以下の条件を氦郡埂㈤Lさ $ 2^{M\ +\ 1} $ の数列 $ a $ = {$ a_1,\ a_2,\ ...,\ a_{2^{M\ +\ 1}} $} を、存在するならば $ 1 $ つBしてください。

- $ a $ は $ 0 $ 以上 $ 2^M $ 未氦握数を、それぞれちょうど $ 2 $ つずつ含む。
- $ a_i\ =\ a_j $ を氦郡谷我猡 $ i,\ j\ (i\ <\ j) $ について、$ a_i\ xor\ a_{i\ +\ 1}\ xor\ ...\ xor\ a_j\ =\ K $ である。

xor (排他的理和) とは

整数 $ c_1,\ c_2,\ ...,\ c_n $ の xor は以下のように定xされます。

- $ c_1\ xor\ c_2\ xor\ ...\ xor\ c_n $ を二M表したHの $ 2^k $ ($ k\ \geq\ 0 $) の位の数は、$ c_1,\ c_2,\ ...,\ c_n $ のうち二M表したHの $ 2^k $ の位の数が $ 1 $ となるものが奇数ならば $ 1 $、偶数ならば $ 0 $ である。

例えば、$ 3\ xor\ 5\ =\ 6 $ です (二M表すると: `011` $ xor $ `101` $ = $ `110` です)。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ M $ $ K $

## 输出格式
条件を氦郡故列 $ a $ が存在しなければ `-1` を出力せよ。

存在するならば、$ a $ の要素を空白区切りで出力せよ。

条件を氦郡故列が}数存在する龊稀どれを出力してもよい。

## 题目大意
请构造一个长度为 $2^{m+1}$ 的序列 $a$ 满足

- $\forall i \in[1, 2^{m+1}], a_i \in [0, 2^m-1]$ 且每个数都恰好出现两次。
- 对于任意一对 $(i, j)$ 满足 $a_i = a_j$，$a_i\oplus a_{i+1} \oplus \cdots \oplus a_{j-1} \oplus a_j = k$

$\oplus$ 表示按位异或。

若不存在满足要求的序列，输出```-1```。  
若存在多个满足要求的序列，输出任意一个即可。

```input1
1 0
```

```output1
0 0 1 1
```

```input2
1 1
```

```output2
-1
```

```input3
5 58
```

```output3
-1
```
