## 题目描述
2014 年江苏省“信息与未来”小学夏令营在金陵中学河西分校进行，组委会决定从河西分校的学生中，推选一位小旗手，并输出其编号。

河西分校的每一位同学都有一个汉语拼音的名字，名字中的字符全部为大写的英文字母，且无空格，例如：

- 名字：王小明
- 拼音：$\tt WANGXIAOMING$

每个大写字母都有一个对应的 ASCII 码，如下表：
| 字符 | 编码（十进制） |
| :----------: | :----------: |
| $\tt A$ | $65$ |
| $\tt B$ | $66$ |
| $\cdots$ | $\cdots$ |
| $\tt Z$ | $90$ |

名字中各字符的 ASCII 码的和即为该学生的编号。

你的任务是根据给出的学生的名字，输出对应的编号。

## 输入格式
一个大写字母组成的字符串 $s$。

## 输出格式
对应的学生编号。

```input1
ABCD
```

```output1
266
```

## 提示
$1\le |s|\le 20$。
