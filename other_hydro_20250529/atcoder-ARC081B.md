## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc071/tasks/arc081_b

$ 2\ \times\ N $ のマス目があります． すぬけ君は，このマス目に $ N $ のドミノを，重ならないように敷きめました． ここで，ドミノは，$ 1\ \times\ 2 $ または $ 2\ \times\ 1 $ のマス目を覆うことができます．

すぬけ君は，赤色，水色，v色の $ 3 $ 色を使って，これらのドミノをTることにしました． このとき，xで接しているドミノはなる色でTるようにします． ここで，必ずしも $ 3 $ 色すべてを使う必要はありません．

このようなTり方が何通りあるかを mod $ 1000000007 $ で求めてください．

ただし，ドミノの敷きめ方は，文字列 $ S_1,\ S_2 $ を用いて，次のようにして与えられます．

- 各ドミノは，それぞれなる英小文字または英大文字で表される．
- $ S_i $ の $ j $ 文字目は，マス目の上から $ i $ 番目，左から $ j $ 番目のマスにどのドミノがあるかを表す．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S_1 $ $ S_2 $

## 输出格式
ドミノをTる方法の数を mod $ 1000000007 $ で出力せよ．

## 题目大意
Snuke有一个$2\times N$的矩阵,以及$N$个多米诺骨牌,每一个骨牌是$1\times 2$或者$2 \times 1$的  
现在Snuke决定用红色、浅蓝色和绿色三种颜色来绘制这些骨牌,要保证每一个骨牌与其周围相邻的骨牌颜色都不一样  
问一共有多少种不同的方案,答案对$1e9+7$取模  

- 每一个骨牌都会用一个英文字母表示  
- 保证每一个骨牌的字母都不一样

```input1
3
aab
ccb
```

```output1
6
```

```input2
1
Z
Z
```

```output2
3
```

```input3
52
RvvttdWIyyPPQFFZZssffEEkkaSSDKqcibbeYrhAljCCGGJppHHn
RLLwwdWIxxNNQUUXXVVMMooBBaggDKqcimmeYrhAljOOTTJuuzzn
```

```output3
958681902
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 52 $
- $ |S_1|\ =\ |S_2|\ =\ N $
- $ S_1,\ S_2 $ は英小文字または英大文字からなる
- $ S_1,\ S_2 $ は正しいドミノの敷きめ方を表している
