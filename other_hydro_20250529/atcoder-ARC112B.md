## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc112/tasks/arc112_b

すぬけくんは、整数 $ B $ を持って整数やさんをLれました。 整数やさんでは、お金をBうことで、持っている整数をeの整数にしてもらうことができます。

具体的には、次の $ 2 $ Nのサ`ビスを好きなで好きなだけ入することができます。

- $ 1 $ 窑Bい、持っている整数を $ -1 $ 倍する。
- $ 2 $ 窑Bい、持っている整数から $ 1 $ を引く。

すぬけくんが $ C $ 乙阅冥亲鳏毪长趣韦扦る整数は何通りありますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ B $ $ C $

## 输出格式
答えを出力せよ。

## 题目大意
初始有一个数 $B$, 然后他通过支付一定的钱来使这个数变为另外一个数，可以使用无数次的以下两个操作，

1.支付一元让这个数乘 $-1$，

2.支付两元使这个数减 $1$，

问在最多 $C$ 元的情况下可以得到多少个数？

```input1
11 2
```

```output1
3
```

```input2
0 4
```

```output2
4
```

```input3
112 20210213
```

```output3
20210436
```

```input4
-211 1000000000000000000
```

```output4
1000000000000000422
```

## 提示
### 制s

- $ -10^{18}\le\ B\ \le\ 10^{18} $
- $ 1\le\ C\ \le\ 10^{18} $
- 入力はすべて整数

### Sample Explanation 1

以下のように、$ -11,10,11 $ の $ 3 $ 通りの数を作ることができます。 - 何もしないとき、$ 0 $ 窑蚴工盲 $ 11 $ を作ることができる - $ 11 $ を $ -1 $ 倍すると、$ 1 $ 窑蚴工盲 $ -11 $ を作ることができる - $ 11 $ から $ 1 $ を引くと、$ 2 $ 窑蚴工盲 $ 10 $ を作ることができる

### Sample Explanation 2

以下のように、$ -2,-1,0,1 $ の $ 4 $ 通りの数を作ることができます。 - 何もしないとき、$ 0 $ 窑蚴工盲 $ 0 $ を作ることができる - $ 0 $ から $ 1 $ を引くと、$ 2 $ 窑蚴工盲 $ -1 $ を作ることができる - $ 0 $ から $ 1 $ を引いて $ -1 $ 倍すると、$ 3 $ 窑蚴工盲 $ 1 $ を作ることができる - $ 0 $ から $ 1 $ を引いて $ 1 $ を引くと、$ 4 $ 窑蚴工盲 $ -2 $ を作ることができる
