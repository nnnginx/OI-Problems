## 题目描述
Paula 给 Daniel 写了一封信，她需要加粗文本的字体，以便视力恶化的 Daniel 阅读。

信可以用 `.` 和 `#` 组成的矩阵表示。对于所有信中原有的 `#`，Paula 需要将其向右下方扩展为 $2 \times 2$ 的 `#`。

## 输入格式
第一行两个整数 $n, m$，表示信的尺寸。

接下来 $n$ 行，每 $m$ 个字符，表示信的内容。

保证最后一行与最后一列中没有 `#`。

## 输出格式
$n$ 行，每行 $m$ 个字符，表示加粗后的信。

```input1
4 4
....
.#..
....
....
```

```output1
....
.##.
.##.
....
```

```input2
7 7
.......
.####..
.#...#.
.#...#.
.#...#.
.####..
.......
```

```output2
.......
.#####.
.######
.##..##
.##..##
.######
.#####.
```

```input3
9 7
.......
.####..
.#...#.
.#...#.
.####..
.#.....
.#.....
.#.....
.......
```

```output3
.......
.#####.
.######
.##..##
.######
.#####.
.##....
.##....
.##....
```

## 提示
#### 数据规模与约定

对于 $100\%$ 的数据，$2 \le n, m \le 10^3$。保证信中只包含 `.` 和 `#`。
