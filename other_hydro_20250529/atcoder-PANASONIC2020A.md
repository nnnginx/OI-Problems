## 题目描述
[problemUrl]: https://atcoder.jp/contests/panasonic2020/tasks/panasonic2020_a

次のLさ $ 32 $ の数列の $ K $ 番目のを出力してください。

 ```
1, 1, 1, 2, 1, 2, 1, 5, 2, 2, 1, 5, 1, 2, 1, 14, 1, 5, 1, 5, 2, 2, 1, 15, 2, 2, 5, 4, 1, 4, 1, 51
```

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ K $

## 输出格式
$ K $ 番目のを出力せよ。

## 题目大意
输入$K$，输入以下数列的第$K$项。
```
1, 1, 1, 2, 1, 2, 1, 5, 2, 2, 1, 5, 1, 2, 1, 14, 1, 5, 1, 5, 2, 2, 1, 15, 2, 2, 5, 4, 1, 4, 1, 51
```

```input1
6
```

```output1
2
```

```input2
27
```

```output2
5
```

## 提示
### 制s

- $ 1\ \leq\ K\ \leq\ 32 $
- 入力は全て整数である。

### Sample Explanation 1

$ 6 $ 番目のは $ 2 $ です。

### Sample Explanation 2

$ 27 $ 番目のは $ 5 $ です。
