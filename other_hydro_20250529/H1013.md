## 题目描述

给定一个 $n$ 项多项式 $P(x)$ 以及 $c, m$，请计算 $P(c^0),P(c^1),\dots,P(c^{m-1})$。所有答案都对 $998244353$ 取模。

## 输入格式

第一行三个正整数 $n,c,m$。  
第二行 $n$ 个非负整数 $a_0,a_1,\dots,a_{n-1}$，由低到高表示 $P(x)$ 的系数。

## 输出格式

一行 $m$ 个正整数，第 $i$ 个数表示 $P(c^{i-1})$。

```input1
3 3 3
3 3 3
```

```output1
9 39 273
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\le n,m\le 10^6$，$0\le c,a_i<998244353$。
