<br><br>For example, if N=12 then
+1 -2 +3 -4 +5 -6 +7 -8 +9 -1+0 -1+1 -1+2 = 5

</p><h3>Input</h3>
<p>Each line contains one integer number N (1¡Ü N ¡Ü 10<sup>15</sup>). Last line contains 0 and shouldn't be processed. Number of lines in the input does not exceed 40.

</p><h3>Output</h3>
<p>For every line in the input write the answer on a separate line.

</p><h3>Example</h3>

<pre><b>Input:</b>

0

<b>Output:</b>
