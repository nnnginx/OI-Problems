## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc178/tasks/abc178_f

Lさ $ N $ の数列 $ A $ と $ B $ が与えられます。 $ A,B $ はそれぞれNにソ`トされています。 $ B $ を好きにKべ替えてすべての $ i(1\ \leq\ i\ \leq\ N) $ について $ A_i\ \neq\ B_i $ を氦郡工瑜Δ摔扦るか判定し、できるならそのような $ B $ のKべ替え方を一つ示してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ \cdots $ $ A_N $ $ B_1 $ $ B_2 $ $ \cdots $ $ B_N $

## 输出格式
条件を氦郡Kべ替え方が存在しない龊 `No` と出力せよ。

条件を氦郡Kべ替え方が存在する龊稀⒁恍心郡 `Yes` を出力し、二行目にKべ替え方を出力せよ。 二行目にはKび替えた後の $ B $ を空白区切りで出力せよ。

条件を氦郡Kべ替え方が}数存在する龊稀そのうちどれを出力してもわない。

## 题目大意
给定两个长度为 $N$ 且升序排列的序列 $A,B$ ，问是否可以重新排列 $B$ ，使得所有 $i$ 均满足 $A_i \not = B_i$ ，可行则输出 `Yes` 并换行输出任意一组可行的 $B$ ，元素以空格隔开，否则输出 `No` 。

输入先是一行一个整数 $N$ ，再是两行两个序列 $A,B$。

```input1
6
1 1 1 2 2 3
1 1 1 2 2 3
```

```output1
Yes
2 2 3 1 1 1
```

```input2
3
1 1 2
1 1 3
```

```output2
No
```

```input3
4
1 1 2 3
1 2 3 3
```

```output3
Yes
3 3 1 2
```

## 提示
### 制s

- $ 1\leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\leq\ A_i,B_i\ \leq\ N $
- $ A,B $ はそれぞれNにソ`トされている。
- 入力はすべて整数
