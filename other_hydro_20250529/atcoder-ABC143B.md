## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc143/tasks/abc143_b

たこきフェスティバル (たこフェス) の季がやってきました！

今年のたこフェスでは $ N $ のたこきがふるまわれる予定です。このうち $ i $ 目のたこきのおいしさは $ d_i $ です。

ところで、おいしさが $ x $ と $ y $ であるたこきを一wに食べると、体力が $ x\ \times\ y $ 回亭工毪长趣一般に知られています。

たこフェスでふるまわれる $ N $ のたこきから、$ 2 $ をxぶ方法は $ \frac{N\ \times\ (N\ -\ 1)}{2} $ 通り考えられます。そのそれぞれについて、一wに食べたときの体力の回土郡蚯螭幛啤そのt和を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ d_1 $ $ d_2 $ $ ... $ $ d_N $

## 输出格式
たこフェスでふるまわれる $ N $ のたこやきから、$ 2 $ をxんで一wに食べたときの体力の回土郡尉t和を出力せよ。

## 题目大意
现在是章鱼烧烤的时间啦！！！

今年将有 $N$ 个章鱼烧（一种里面有章鱼的球状的食物）。第 $i$ 个章鱼烧的美味值是 $d_{i}$ 。   

众所周知，当你在吃同时吃美味值为 $x$ 和 $y$ 的章鱼烧时，你将回复 $x\times y$ 的生命值。    

在节日里，你有 $\frac{N\times (N-1)}{2}  $ 种选法在 $N$ 个章鱼烧中选择两个不同的章鱼烧。    

输出 $\frac{N\times (N-1)}{2}  $ 种选法所能得到的健康值之和。

by [djh123456](https://www.luogu.com.cn/user/367575) 。

```input1
3
3 1 2
```

```output1
11
```

```input2
7
5 0 7 8 3 3 2
```

```output2
312
```

## 提示
### 制s

- 入力は全て整数である。
- $ 2\ \leq\ N\ \leq\ 50 $
- $ 0\ \leq\ d_i\ \leq\ 100 $

### Sample Explanation 1

以下の $ 3 $ 通りの食べ方が考えられます。 - $ 1,~2 $ 目のたこきをxんで一wに食べる。このとき、体力の回土郡 $ 3 $ である。 - $ 2,~3 $ 目のたこきをxんで一wに食べる。このとき、体力の回土郡 $ 2 $ である。 - $ 1,~3 $ 目のたこきをxんで一wに食べる。このとき、体力の回土郡 $ 6 $ である。 体力の回土郡尉t和は $ 11 $ です。
