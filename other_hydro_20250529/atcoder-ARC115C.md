## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc115/tasks/arc115_c

整数 $ N $ が与えられます。以下の条件を氦郡归Lさ $ N $ の正の整数の列 $ A_1,A_2,\ldots,A_N $ であって、数列にFれるの最大が最小になるものを一つ出力してください。

- $ i $ が $ j $ のs数ならば、$ A_i\ \neq\ A_j $ $ (1\ \leq\ i\ <\ j\ \leq\ N) $

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
数列の各要素を空白で区切って一行に出力せよ。

条件を氦郡菇猡}数存在する龊悉稀どれを出力してもよい。

> $ A_1 $ $ A_2 $ $ \ldots $ $ A_N $

## 题目大意
构造一个长度为 $ N $ 的序列 $ A $，对于下标 $ i $, $ j \ (1 \le i < j \le N) \ $，若 $ i $ 为 $ j $ 的因数，$ A_i \neq A_j $。

你需要使这个序列中最大的数尽量小。

输出这个序列。

```input1
4
```

```output1
1 2 2 3
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^5 $

### Sample Explanation 1

この出力は以下の条件をすべて氦郡筏蓼埂 - $ A_1\ \neq\ A_2 $ - $ A_1\ \neq\ A_3 $ - $ A_1\ \neq\ A_4 $ - $ A_2\ \neq\ A_4 $ また、登訾工の最大が $ 2 $ 以下である数列であって、これらの条件をすべて氦郡工猡韦洗嬖冥筏胜い韦恰この出力はm当です。
