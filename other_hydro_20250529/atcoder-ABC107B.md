## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc107/tasks/abc107_b

k $ H $ 行、横 $ W $ 列のマス目があります。 上から $ i $ 行目、左から $ j $ 列目のマスを $ (i,\ j) $ と表します。 各マスは白または\です。 マス目の配色は、$ H $ 行 $ W $ 列の行列 $ (a_{i,\ j}) $ によって与えられます。 $ a_{i,\ j} $ が `.` ならばマス $ (i,\ j) $ は白であり、$ a_{i,\ j} $ が `#` ならばマス $ (i,\ j) $ は\です。

すぬけ君はこのマス目をRsしようとしています。 そのために、白いマスのみからなる行または列が存在するg、次の操作をRり返し行います。

- 操作: 白いマスのみからなる行または列をひとつ任意にxび、その行または列を取り除いて空白をめる。

各操作でどの行または列をxぶかによらず、最K的なマス目は一意に定まることが示せます。 最K的なマス目を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ H $ $ W $ $ a_{1,\ 1}...a_{1,\ W} $ $ : $ $ a_{H,\ 1}...a_{H,\ W} $

## 输出格式
最K的なマス目を、入力と同のフォ`マットで出力せよ。 ただし、行数および列数は省くこと。 しくはサンプルを参照せよ。

## 题目大意
给出$H*W$的图，如果这个图上一行或一列没有$#$的话将这一行或这一列删除，输出处理完后的图。
$1 ≤ H, W ≤ 100$

```input1
4 4
##.#
....
##.#
.#.#
```

```output1
###
###
.##
```

```input2
3 3
#..
.#.
..#
```

```output2
#..
.#.
..#
```

```input3
4 5
.....
.....
..#..
.....
```

```output3
#
```

```input4
7 6
......
....#.
.#....
..#...
..#...
......
.#..#.
```

```output4
..#
#..
.#.
.#.
#.#
```

## 提示
### 制s

- $ 1\ \leq\ H,\ W\ \leq\ 100 $
- $ a_{i,\ j} $ は `.` または `#` である。
- マス目全体で少なくともひとつは\いマスが存在する。

### Sample Explanation 1

元のマス目における第 $ 2 $ 行および第 $ 3 $ 列がそれぞれ取り除かれます。

### Sample Explanation 2

白いマスのみからなる行または列が存在しないので、操作は行われません。
