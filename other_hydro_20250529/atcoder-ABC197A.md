## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc197/tasks/abc197_a

Lさ $ 3 $ の文字列 $ S $ が与えられます。  
 $ S $ の先^の文字を $ S $ の末尾に移婴筏频盲椁欷胛淖至 $ S' $ を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S' $ を出力せよ。

## 题目大意
给出长度为3的字符串S，
将S的第一个字符移到末尾并输出

```input1
abc
```

```output1
bca
```

```input2
aab
```

```output2
aba
```

## 提示
### 制s

- $ S $ は英小文字のみからなるLさ $ 3 $ の文字列である

### Sample Explanation 1

`abc` の先^の文字 `a` を末尾に移婴工毪 `bca` となります。
