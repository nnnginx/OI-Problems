## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc098/tasks/arc098_b

Lさ $ N $ の整数列 $ A $ があります。

次の条件を氦郡拐数 $ l $, $ r $ ( $ 1\ \leq\ l\ \leq\ r\ \leq\ N $ ) のMの数を求めてください。

- $ A_l\ xor\ A_{l+1}\ xor\ ...\ xor\ A_r\ =\ A_l\ +\ A_{l+1}\ +\ ...\ +\ A_r $

xorのh明

整数 $ c_1,\ c_2,\ ...,\ c_m $ の $ xor $ は以下のように定xされます。

- $ xor $ のを $ X $ とおく。$ X $ を $ 2 $ M数表したときの $ 2^k $ ( $ 0\ \leq\ k $, $ k $ は整数 ) の位のは、$ c_1,\ c_2,\ ...c_m $ のうち、$ 2 $ M数表したときの $ 2^k $ の位のが $ 1 $ となるものが奇数ならば $ 1 $、偶数ならば $ 0 $ となる。

例えば、$ 3 $ と $ 5 $ の $ xor $ のは、$ 3 $ の $ 2 $ M数表が $ 011 $、$ 5 $ の $ 2 $ M数表が $ 101 $ のため、$ 2 $ M数表が $ 110 $ の $ 6 $ となります。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $

## 输出格式
条件を氦郡拐数 $ l $, $ r $ ( $ 1\ \leq\ l\ \leq\ r\ \leq\ N $ ) のMの数を出力せよ。

## 题目大意
给你一串数 $a$

求出满足$a_l+\cdots +a_r=a_l\operatorname{xor}\cdots\operatorname{xor}a_r,l\le r$ 的 $(i,j)$ 的数量

$1\le n\le 200000,\forall 1\le i\le n,0\le a_i<2^{20}(1048576)$

```input1
4
2 5 4 6
```

```output1
5
```

```input2
9
0 0 0 0 0 0 0 0 0
```

```output2
45
```

```input3
19
885 8 1 128 83 32 256 206 639 16 4 128 689 32 8 64 885 969 1
```

```output3
37
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 0\ \leq\ A_i $
- 入力はすべて整数である

### Sample Explanation 1

明らかに、$ (l,r)=(1,1),(2,2),(3,3),(4,4) $ は条件を氦郡筏蓼埂 また、$ (l,r)=(1,2) $ の龊稀$ A_1\ xor\ A_2\ =\ A_1\ +\ A_2\ =\ 7 $ となるので、これも条件を氦郡筏蓼埂 ほかに条件を氦郡菇Mはないので、答えは $ 5 $ になります。
