## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc302/tasks/abc302_a

体力が $ A $ の长います。あなたは、$ 1 $ 回攻膜颏工毪长趣长翁辶Δ $ B $ pらすことが出来ます。

长翁辶Δ $ 0 $ 以下にするためには、最小で何回攻膜颏工氡匾があるでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
答えを出力せよ。

## 题目大意
### 题目描述
一个人有 $A$ 点体力值，你可以攻击他，每次攻击使其体力值减 $B$，问要让他体力值降至 $0$ 或以下至少要攻击多少次。
### 输入格式
共一行两个整数 $A,B$。($1\leqslant A,B\leqslant 10^{18}$)
### 输出格式
共一行一个整数，表示最少攻击次数。

```input1
7 3
```

```output1
3
```

```input2
123456789123456789 987654321
```

```output2
124999999
```

```input3
999999999999999998 2
```

```output3
499999999999999999
```

## 提示
### 制s

- $ 1\ \le\ A,B\ \le\ 10^{18} $
- $ A,\ B $ は整数である。
 
### Sample Explanation 1

$ 3 $ 回攻膜工毪长翁辶Δ $ -2 $ となります。 $ 2 $ 回攻膜筏郡坤堡扦长翁辶Δ $ 1 $ であるため、$ 3 $ 回攻膜工氡匾があります。
