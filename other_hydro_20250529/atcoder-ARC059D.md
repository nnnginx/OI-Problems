## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc059/tasks/arc059_d

しぐはキ`ボ`ドをu作しました。シンプルさをO限まで追求したこのキ`ボ`ドには、`0` キ`、`1` キ`、バックスペ`スキ`の $ 3 $ つしかキ`がありません。

手始めに、しぐはこのキ`ボ`ドでgなテキストエディタを操作してみることにしました。このエディタには常に一つの文字列が表示されます（文字列が空のこともあります）。エディタを起婴筏恐贬幛扦稀⑽淖至肖峡栅扦埂％`ボ`ドの各キ`を押すと、文字列が次のように浠します。

- `0` キ`: 文字列の右端に文字 `0` が啡毪丹欷搿
- `1` キ`: 文字列の右端に文字 `1` が啡毪丹欷搿
- バックスペ`スキ`: 文字列が空なら、何も起こらない。そうでなければ、文字列の右端の $ 1 $ 文字が削除される。

しぐはエディタを起婴贰これらのキ`を合で $ N $ 回押しました。そのY果、いまエディタに文字列 $ s $ が表示されています。このようなキ`の押し方の数を $ 10^9\ +\ 7 $ で割った余りを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ s $

## 输出格式
最K的にエディタに文字列 $ s $ が表示されるような $ N $ 回のキ`の押し方の数を $ 10^9+7 $ で割った余りを出力せよ。

## 题目大意
小z得到了一个键盘，里面只有$1, 0$和退格键

键$0$可以打出一个$0$的字符串，键$1$同理

退格键可以删除前面打出的那个字符

小z可以操作这个键盘$N$次（$N\le5000$），求操作完成后打出来的字符串恰好是$S$的方案数

注意：当前没有字符也可以使用退格键

```input1
3
0
```

```output1
5
```

```input2
300
1100100
```

```output2
519054663
```

```input3
5000
01000001011101000100001101101111011001000110010101110010000
```

```output3
500886057
```

## 提示
### 制s

- $ 1\ Q\ N\ Q\ 5000 $
- $ 1\ Q\ |s|\ Q\ N $
- $ s $ は文字 `0`, `1` のみからなる。

### 部分点

- $ 1\ Q\ N\ Q\ 300 $ を氦郡攻签`タセットに正解すると、$ 400 $ 点が与えられる。

### Sample Explanation 1

バックスペ`スキ`を `B` と表すると、次の $ 5 $ 通りの押し方で最K的に表示される文字列が `0` となります: `00B`, `01B`, `0B0`, `1B0`, `BB0`。最後の押し方では、バックスペ`スキ`を押すときに何も起こりません。
