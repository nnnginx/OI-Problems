## 题目描述

**题目译自 [CEOI 2021](https://hsin.hr/ceoi/competition/) Day1 T2「[L-triominoes](https://hsin.hr/ceoi/competition/ceoi2021_day1_tasks.pdf)」**

Luka 偶然发现了一个高度为 $H$，宽度为 $W$ 并分为 $W\times H$ 个单位方格的矩形板。他迅速注意到恰好有 $K$ 个单元格缺失了。

十分有趣的是，Luka 恰好发现了无限多个 L 形三联牌。利用这些三联牌是否可以密铺这个板子呢？

![ltriominoes1.png](https://img.loj.ac.cn/2022/01/12/f0b61a2302822.png)

如果这个板子的每一个单元格都被一张三联牌覆盖，我们就认为这个板子被正确密铺了。此外，三联牌不能覆盖任何缺失的单元格，也不能互相重叠或放出板子。当然，三联牌可以旋转 $90$ 度的任意倍数。

## 输入格式

第一行包含三个整数 $W,H$ 和 $K\ (0\le K\le W\cdot H)$，意义如题目描述。

接下来 $K$ 行，每行包含两个整数 $x_i\ (1\le x_i\le W)$ 和 $y_i\ (1\le y_i\le H)$，表示第 $i$ 个缺失的单元格坐标。给出的坐标两两不同。

## 输出格式

如果 Luka 可以密铺这个板子，输出 `YES`，否则输出 `NO`。

```input1
4 3 3
1 1
1 3
4 3

```

```output1
YES

```

```input2
5 2 4
1 2
2 1
5 1
5 2

```

```output2
NO

```

```input3
2 3 0

```

```output3
YES
```
