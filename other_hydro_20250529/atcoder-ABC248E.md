## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc248/tasks/abc248_e

座似矫嫔悉 $ N $ の点が与えられます。 $ 1\leq\ i\leq\ N $ について、$ i $ 番目の点の座摔 $ (X_i,\ Y_i) $ です。

座似矫嫔悉沃本であって、$ N $ の点のうち $ K $ 以上の点を通るものの数を求めてください。  
ただし、そのようなものがo数に存在する龊悉 `Infinity` を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $ $ X_1 $ $ Y_1 $ $ X_2 $ $ Y_2 $ $ \vdots $ $ X_N $ $ Y_N $

## 输出格式
与えられた $ N $ の点のうち $ K $ 以上の点を通る直の数を出力せよ。ただし、そのようなものがo数に存在する龊悉 `Infinity` を出力せよ。

## 题目大意
给定平面直角坐标系上的 $N$ 个点，第 $i$ 个点的坐标为 $(x_i,y_i)$，问有多少条直线至少经过了 $K$ 个点。

```input1
5 2
0 0
1 0
0 1
-1 0
0 -1
```

```output1
6
```

```input2
1 1
0 0
```

```output2
Infinity
```

## 提示
### 制s

- $ 1\ \leq\ K\ \leq\ N\ \leq\ 300 $
- $ \lvert\ X_i\ \rvert,\ \lvert\ Y_i\ \rvert\ \leq\ 10^9 $
- $ i\neq\ j $ ならば $ X_i\neq\ X_j $ または $ Y_i\neq\ Y_j $
- 入力はすべて整数

### Sample Explanation 1

$ x=0 $, $ y=0 $, $ y=x\pm\ 1 $, $ y=-x\pm\ 1 $ の $ 6 $ 本の直が条件をみたします。 例えば、$ x=0 $ は、$ 1 $, $ 3 $, $ 5 $ 番目の $ 3 $ の点を通ります。 よって、$ 6 $ を出力します。

### Sample Explanation 2

原点を通る直はo数に存在します。 よって、`Infinity` を出力します。
