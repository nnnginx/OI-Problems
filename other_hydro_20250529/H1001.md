## 题目描述

给出 $n$ 个仅由小写字母组成的字符串，请你求出这些字符串中一共出现了几种不同的字符串。

## 输入格式

第一行一个整数 $n$。

接下来 $n$ 行每行一个字符串。

## 输出格式

一行一个整数表示一共出现过的字符串数量。

```input1
5
aba
aba
ababa
ababa
abababa
```

```output1
3
```

## 说明

一共出现了三种不同类型的字符串： `aba`，`ababa` 和 `abababa`。

## 数据范围

- $\texttt{Subtask 1(50pts, 1000ms):} n \le 10^5$
- $\texttt{Subtask 2(50pts, 300ms):} n = 10^6$

对于 $100\%$ 的数据，$n \ge 1,~\sum \text{字符串长度} \le 4 \times 10^6$。

较低效的语言可能无法通过此题，建议选手使用较快的输入方式并使用带优化的语言提交。
