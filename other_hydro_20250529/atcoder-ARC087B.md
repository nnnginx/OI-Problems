## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc082/tasks/arc087_b

二次元平面の原点にロボットが置かれています。 最初、ロボットは $ x $ Sの正の向きを向いています。

このロボットに命令列 $ s $ が与えられます。 $ s $ は次の $ 2 $ 文字のみからなり、先^から末尾までにg行されます。

- `F` : 今向いている向きにLさ $ 1 $ だけ移婴工搿
- `T` : r回りまたは反r回りの好きな方向に $ 90 $ 度だけ向きを浃à搿

ロボットの目摔稀⒚令列をすべてg行しKわった後に座 $ (x,\ y) $ にいることです。 この目摔_成可能か判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ s $ $ x $ $ y $

## 输出格式
目摔_成可能ならば `Yes` を出力し、_成不可能ならば `No` を出力せよ。

## 题目大意
题意描述：
现有一个机器人在坐标轴原点处，朝向x轴正方向。给定一个命令串s，s内含有两种命令：F,T。F表示向当前朝向前进1步，T表示向左**或**向右旋转90度。请你回答，按照命令串s执行，会不会到达(x,y)处。（x与y由题目输入）

```input1
FTFFTFFF
4 2
```

```output1
Yes
```

```input2
FTFFTFFF
-2 -2
```

```output2
Yes
```

```input3
FF
1 0
```

```output3
No
```

```input4
TF
1 0
```

```output4
No
```

```input5
FFTTFF
0 0
```

```output5
Yes
```

```input6
TTTT
1 0
```

```output6
No
```
