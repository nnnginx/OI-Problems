## 题目描述

小 Z 近日学会了字符串哈希，他用 C++ 写了如下的哈希函数。

```cpp
typedef __int128 bi;
bi hash(std::string s) {
    bi v = 0;
    for (char c : s) v = (v * B + c) % M;
    return v;
}
```

形式化地，字符串 $c_0c_1\cdots c_{n-1}$ 的 hash 值为 $(\sum_{i=0}^{n-1} B^{n-1-i}c_i)\bmod M$，这里的 $c$ 即各位的 ascii 码。例如，当 $B=100$，$M=998244353$ 时，$\mathrm{hash}(\texttt{abc})=979899$。

你认为这个哈希方法非常脆弱，于是打算卡掉它。对于给定的三个整数 $B$、$M$、$x$，请构造出一个长度不超过 $10000$ 的由小写字母组成的非空字符串，使得它使用如上函数算出的哈希值为 $x$。

## 输入格式

一行三个整数，$B$、$M$、$x$。

## 输出格式

一行一个由小写字母组成的非空字符串，长度不超过 $10000$。如有多解，可以输出任意一个。

```input1
200 100000000000000003 1234567890
```

```output1
aadffikdnakjaoaicmibmommolegeifpyanglichuan
```

## 数据范围与提示

对于所有数据，$B$ 在 $[130,200]$ 中均匀随机生成。$x$ 的生成方法为，随机生成一个长度为 $1000$ 的由小写字母组成的字符串，计算它的哈希值作为 $x$，因此解一定存在。

本题共有 $3$ 个子任务，每个子任务恰有 $20$ 个测试点。子任务的得分为每个测试点得分之和，每个子任务内每个测试点分值相同。

Subtask $1$（$40$ pts）：$M$ 为 $[2,10^9]$ 范围内的随机质数。

Subtask $2$（$20$ pts）：$M$ 为 $[2,10^{16}]$ 范围内的随机质数。

Subtask $3$（$40$ pts）：$M$ 为 $[2,10^{18}]$ 范围内的随机质数。
