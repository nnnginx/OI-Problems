<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">You are given an array on N elements&nbsp;<span style="box-sizing: border-box; font-weight: 700;">a[1], a[2], a[3], бн. , a[N]</span>.</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">Now you will have to answer some queries.</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">In every query, you will be given an interval,&nbsp;<span style="box-sizing: border-box; font-weight: 700;">[l, r]</span>. For this interval you have to print the total summation of happiness of all the elements of the given array between the interval&nbsp;<span style="box-sizing: border-box; font-weight: 700;">[l, r]</span>.</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">The happiness of an element&nbsp;<span style="box-sizing: border-box; font-weight: 700;">a[i]</span>&nbsp;between interval&nbsp;<span style="box-sizing: border-box; font-weight: 700;">[l, r]</span>&nbsp;is: the number of&nbsp;<span style="box-sizing: border-box; font-weight: 700;">sub-array [l<span style="box-sizing: border-box; font-size: 10.5px; line-height: 0; position: relative; vertical-align: baseline; bottom: -0.25em;">j</span>&nbsp;, r<span style="box-sizing: border-box; font-size: 10.5px; line-height: 0; position: relative; vertical-align: baseline; bottom: -0.25em;">j</span>]</span>&nbsp;where the minimum value between&nbsp;<span style="box-sizing: border-box; font-weight: 700;">[l<span style="box-sizing: border-box; font-size: 10.5px; line-height: 0; position: relative; vertical-align: baseline; bottom: -0.25em;">j</span>, r<span style="box-sizing: border-box; font-size: 10.5px; line-height: 0; position: relative; vertical-align: baseline; bottom: -0.25em;">j</span>]</span>&nbsp;is equal to&nbsp;<span style="box-sizing: border-box; font-weight: 700;">a[i]</span>. Here,&nbsp;<span style="box-sizing: border-box; font-weight: 700;"><em>l &lt;= l<span style="box-sizing: border-box; font-size: 10.5px; line-height: 0; position: relative; vertical-align: baseline; bottom: -0.25em;">j</span>&nbsp;&lt;= i &amp; i &lt;= r<span style="box-sizing: border-box; font-size: 10.5px; line-height: 0; position: relative; vertical-align: baseline; bottom: -0.25em;">j</span>&nbsp;&lt;= r.</em></span></p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">Now, you have to print the total summation of happiness of all the elements between&nbsp;<span style="box-sizing: border-box; font-weight: 700;">[l,r]</span>.</p>
<h3>Input</h3>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">The first line of the input contains the number of test cases&nbsp;<span style="box-sizing: border-box; font-weight: 700;">T</span>. The first line of each test case contains two numbers,&nbsp;<span style="box-sizing: border-box; font-weight: 700;">N</span>&nbsp;&amp;&nbsp;<span style="box-sizing: border-box; font-weight: 700;">M</span>.&nbsp;<span style="box-sizing: border-box; font-weight: 700;">N</span>&nbsp;is the number of elements in array&nbsp;<span style="box-sizing: border-box; font-weight: 700;">a</span>&nbsp;and&nbsp;<span style="box-sizing: border-box; font-weight: 700;">M</span>&nbsp;is the number of queries you need to perform.</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">The next line contains&nbsp;<span style="box-sizing: border-box; font-weight: 700;">N</span>&nbsp;integers, the array a :&nbsp;<span style="box-sizing: border-box; font-weight: 700;">a[1], a[2], a[3], бн a[N]</span>.</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">Next&nbsp;<span style="box-sizing: border-box; font-weight: 700;">M</span>&nbsp;lines contains two integers,&nbsp;<span style="box-sizing: border-box; font-weight: 700;">l</span>&nbsp;&amp;&nbsp;<span style="box-sizing: border-box; font-weight: 700;">r</span>.</p>
<h3>Constraints</h3>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">1 &lt;= T &lt;= 5</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">1 &lt;= N, M &lt;= 50000</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">1 &lt;= a[i] &lt;= 1000000000</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">1 &lt;= l &lt;= r &lt;= N</p>
<h3>Output</h3>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">For each test case, you need to print the case number on the first line in this format:&nbsp;<span style="box-sizing: border-box; font-weight: 700;">Case X:</span>&nbsp;where&nbsp;<span style="box-sizing: border-box; font-weight: 700;">X</span>&nbsp;is the case number.</p>
<p style="box-sizing: border-box; margin: 0px 0px 10px; color: #3e3f3a; font-family: 'PT Sans'; font-size: 14px; line-height: 20px;">In the next&nbsp;<span style="box-sizing: border-box; font-weight: 700;">M</span>&nbsp;lines, you need to print the total summation of happiness of all the elements between&nbsp;<span style="box-sizing: border-box; font-weight: 700;">[l, r]</span>&nbsp;of the given array.</p>

<span style="color: #3e3f3a; font-family: 'Ubuntu Mono'; font-size: 14px; line-height: 20px; white-space: pre;">2<br>5 2<br>1 3 2 4 3<br>1 3<br>2 4<br>5 2<br><span style="color: #3e3f3a; font-family: 'Ubuntu Mono'; font-size: 14px; line-height: 20px; white-space: pre;">5 3 7 6 8<br></span><span style="color: #3e3f3a; font-family: 'Ubuntu Mono'; font-size: 14px; line-height: 20px; white-space: pre;">1 4<br>2 5</span>&nbsp;</span></pre>
