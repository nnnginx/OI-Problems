## 题目描述
假如有一棵深度为 $n$ 的二叉排序树，而且它只可能有右节点，那最少要旋转多少次才能使它平衡？请输出最少的旋转次数。
## 输入格式
输入只有 1 行，只有 1 个正整数，表示 $n$。
## 输出格式
输出只有 1 行，只有 1 个正整数，表示最少要旋转的次数。

```input1
5
```

```output1
2
```

```input2
28
```

```output2
30
```

```input3
1327
```

```output3
4054
```
