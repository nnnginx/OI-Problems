## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc252/tasks/abc252_a

英小文字 `a`, `b`, $ \ldots $, `z` の ASCII 文字コ`ドはこのに $ 97,98,\ldots,122 $ です。

$ 97 $ 以上 $ 122 $ 以下の整数 $ N $ が与えられるので、ASCII 文字コ`ドが $ N $ であるような英小文字を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
输入一个数 $N$$(97<N<122)$，输出在ASCLL表中相对应的字符。

```input1
97
```

```output1
a
```

```input2
122
```

```output2
z
```

## 提示
### 制s

- $ N $ は $ 97 $ 以上 $ 122 $ 以下の整数

### Sample Explanation 1

ASCII 文字コ`ドが $ 97 $ である英小文字は `a` です。
