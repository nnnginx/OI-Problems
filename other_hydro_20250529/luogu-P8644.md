## 题目描述
X 星球的机器人表演拉拉队有两种服装，A 和 B。

他们这次表演的是搭机器人塔。

类似：

```
     A
    B B
   A B A
  A A B B
 B B B A B
A B A B B A
```

队内的组塔规则是：

A 只能站在 AA 或 BB 的肩上。

B 只能站在 AB 或 BA 的肩上。

你的任务是帮助拉拉队计算一下，在给定 A 与 B 的人数时，可以组成多少种花样的塔。

## 输入格式
输入一行两个整数 $M$ 和 $N$，空格分开（$0<M,N<N+M<231$，保证存在 $k\in \mathbb{N}$，$N+M=\frac{k(k-1)}{2}$），分别表示 A、B 的人数。

## 输出格式
要求输出一个整数，表示可以产生的花样种数。

```input1
1 2
```

```output1
3
```

```input2
3 3
```

```output2
4
```

## 提示
时限 1 秒, 256M。蓝桥杯 2016 年第七届
