## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc210/tasks/abc210_b

$ N $ 枚のカ`ドからなる山札があります。  
 それぞれのカ`ドは、「良いカ`ド」か「いカ`ド」かのどちらかです。

高蚓と青木君は、この山札を使って楗博`ムをします。  
 このゲ`ムでは、$ 2 $ 人は交互に山札の一番上のカ`ドを引いて、そのカ`ドを食べます。  
 先にいカ`ドを食べたプレイヤ`のけです。（ここで、山札には少なくとも $ 1 $ 枚のいカ`ドが含まれていることが保^されます。）

`0` と `1` からなる文字列 $ S $ が与えられます。$ i\ =\ 1,\ 2,\ \ldots,\ N $ について、

- $ S $ の $ i $ 文字目が `0` のとき、山札の上から $ i $ 番目のカ`ドが良いカ`ドであることを表します。
- $ S $ の $ i $ 文字目が `1` のとき、山札の上から $ i $ 番目のカ`ドがいカ`ドであることを表します。

高蚓が先手でゲ`ムを始めるとき、高蚓と青木君の**どちらがけるか**を答えてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
高蚓が先手でゲ`ムを始めるとき、高蚓と青木君の**どちらがけるか**を答えよ。  
 高蚓がけるならば `Takahashi` 、青木君がけるならば `Aoki` と出力せよ。

## 题目大意
输入一个长为 $n$ 的 $01$ 串 $s$ ，求 $s$ 中从左往右数的第一个 $1$ 出现在第奇数个位置上还是第偶数个位置上（串中的第一个数为第 $1$ 位）。若在奇数位上输出Takahashi，否则输出Aoki。保证串中至少有一个 $1$ 。

```input1
5
00101
```

```output1
Takahashi
```

```input2
3
010
```

```output2
Aoki
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ N $ は整数
- $ S $ は `0` と `1` からなるLさ $ N $ の文字列
- $ S $ は少なくとも $ 1 $ の `1` を含む。

### Sample Explanation 1

まず、高蚓が良いカ`ドを食べ、次に青木君が良いカ`ドを食べ、その後に高蚓がいカ`ドを食べます。 高蚓が先にいカ`ドを食べるので高蚓がけます。よって、`Takahashi` と出力します。
