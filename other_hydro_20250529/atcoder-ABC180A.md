## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc180/tasks/abc180_a

$ N $ のボ`ルが入っていた箱から $ A $ のボ`ルを取り出し、新たに $ B $ のボ`ルを入れました。今、箱にはボ`ルが何入っていますか?

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $

## 输出格式
答えを整数で出力せよ。

## 题目大意
输入 $N$ ，$A$ ，$B$ ,求 $N-A+B$ 。

```input1
100 1 2
```

```output1
101
```

```input2
100 2 1
```

```output2
99
```

```input3
100 1 1
```

```output3
100
```

## 提示
### 制s

- 入力はすべて整数
- $ 100\ \leq\ N\ \leq\ 200 $
- $ 1\ \leq\ A,B\ \leq\ 100 $

### Sample Explanation 1

箱にはもともと $ 100 $ のボ`ルが入っていましたが $ 1 $ のボ`ルを取り出し、新たに $ 2 $ のボ`ルを入れたことで、今は $ 101 $ のボ`ルが入っています。
