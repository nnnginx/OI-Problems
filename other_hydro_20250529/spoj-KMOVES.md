<p>A knight is located at the (black) origin of an infinite chessboard. Let <em>f(n) </em>define the number of black squares the knight can reach after making exactly <em>n</em> moves. Given <em>n </em>(0 &lt;= <em>n</em> &lt;= 10<sup>8</sup>), output <em>f(n)</em>.</p>
<h3>Input</h3>
<p>The first line of the input contains a single integer T, the number of test cases (1 &lt;= T &lt;= 10<sup>6</sup>). Each test case consists of a single positive integer <em>n</em>.</p>
<h3>Output</h3>
<p>For each value of <em>n</em> in the input, print a single line containing.</p>
<h3>Example</h3>
<pre><strong>Input:</strong><br>2<br>0<br>1<br><br><strong>Output:</strong><br>1<br>0</pre>
