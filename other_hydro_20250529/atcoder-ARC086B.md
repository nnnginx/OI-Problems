すぬけ君はLさ $ N $ の数列 $ a $ を持っています。$ a $ の ($ 1 $-indexedでの) $ i $ 番目の数は $ a_{i} $ です。

すぬけ君は以下の操作を何度でも行うことができます。

- 操作：$ 1 $ 以上 $ N $ 以下の整数 $ x,y $ をxび、$ a_y $ に $ a_x $ を加算する。

すぬけ君はこの操作を $ 0 $ 回以上 $ 2N $ 回以下行って $ a $ が下の条件を氦郡工瑜Δ摔筏郡い扦埂￥饯韦瑜Δ什僮魇猪の一例を示してください。 なお、この}の制s下で、条件を氦郡工瑜Δ什僮鳏问猪が必ず存在することが^明できます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_1 $ $ a_2 $ $ ... $ $ a_{N} $

## 输出格式
操作回数(これを $ m $ とする)を $ 1 $ 行目に出力せよ。 Aく $ m $ 行のうち $ i $ 行目には $ i $ 番目の操作でxんだ数 $ x,y $ を空白区切りで出力せよ。 $ m $ が $ 0 $ 以上 $ 2N $ 以下であり、$ m $ 回の操作後に $ a $ が条件を氦郡筏皮い欷姓解となる。

## 题目大意
一个 $N$ 个元素的整数序列，第 $i$ 项为 $a_i$。

小A可以做以下操作若干次：

- 选择两个整数 $i$ 和 $j$，将 $a_i$ 加到 $a_j$ 上去。

小A需要操作最多 $2N$ 次（也可能是 $0$ 次），以使得整数序列满足以下条件：

- $a_1 \le a_2 \le … \le a_N$

可以证明，上述操作序列一定存在。请输出小A的操作序列。

```input1
3
-2 5 -1
```

```output1
2
2 3
3 3
```

```input2
2
-1 -3
```

```output2
1
2 1
```

```input3
5
0 0 0 0 0
```

```output3
0
```
