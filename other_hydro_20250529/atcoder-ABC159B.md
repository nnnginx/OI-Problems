## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc159/tasks/abc159_b

Lさが奇数である文字列 $ S $ が以下の条件をすべて氦郡工趣、$ S $ は「い回文」であるといいます。

- $ S $ は回文である。
- $ N $ を $ S $ のLさとするとき、$ S $ の $ 1 $ 文字目から $ (N-1)/2 $ 文字目まで(I端含む)からなる文字列は回文である。
- $ S $ の $ (N+3)/2 $ 文字目から $ N $ 文字目まで(I端含む)からなる文字列は回文である。

$ S $ がい回文かどうかを判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ がい回文ならば `Yes` 、 い回文でないならば `No` と出力せよ。

## 题目大意
## 题目描述

一个长度为奇数的字符串 $S$ 被称为 $strong$ $palindrome$ 当且仅当以下几点全部满足。

- $S$ 是一个回文串。
- 设 $N$ 是 $S$ 的长度。$S$的前$(\frac{(N-1)}{2})$个字符所组成的字符串是回文的。
- $S$的后$(\frac{(N+3)}{2})$个字符所组成的字符串是回文的。

判断$S$是否是 $strong$ $palindrome$。

## 输入格式

标准输入。

一行一个字符串 $S$。

## 输出格式

如果 $S$ 是 $strong$ $palindrome$，输出 $Yes$；否则输出 $No$。

## 数据范围

- $S$只包含小写字母。

- $S$的长度是一个属于$[3,99]$的奇数。

```input1
akasaka
```

```output1
Yes
```

```input2
level
```

```output2
No
```

```input3
atcoder
```

```output3
No
```

## 提示
### 制s

- $ S $ は英小文字のみからなる
- $ S $ のLさは $ 3 $ 以上 $ 99 $ 以下の奇数

### Sample Explanation 1

\- $ S $ は `akasaka` - $ S $ の $ 1 $ 文字目から $ 3 $ 文字目までからなる文字列は `aka` - $ S $ の $ 5 $ 文字目から $ 7 $ 文字目までからなる文字列は `aka` これらはすべて回文であるため、$ S $ はい回文です。
