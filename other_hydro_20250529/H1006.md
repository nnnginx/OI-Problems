## 题目描述

给定一个质数 $p$，以及一个整数 $b$，一个整数 $n$，现在要求你计算一个最小的 $l$，满足 $b^l\equiv n \pmod p$。

## 输入格式

仅一行，有 $3$ 个整数，依次代表 $p,b,n$。

## 输出格式

仅一行，如果有 $l$ 满足该要求，输出最小的 $l$，否则输出 $-1$。

```input1
5 2 3
```
```output1
3
```

## 样例解释

$2^3\bmod 5=3$。

## 数据规模与约定

对于 $100\%$ 的数据，$2\leq b,n<p<10^{18}$，保证 $p-1$ 仅有 $2,3$ 两个本质不同的质因子。
