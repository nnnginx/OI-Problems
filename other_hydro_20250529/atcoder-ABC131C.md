## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc131/tasks/abc131_c

整数 $ A,B,C,D $ が与えられます。$ A $ 以上 $ B $ 以下の整数のうち、$ C $ でも $ D $ でも割り切れないものの数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ D $

## 输出格式
$ A $ 以上 $ B $ 以下の整数のうち、$ C $ でも $ D $ でも割り切れないものの数を出力せよ。

## 题目大意
给出四个整数 $A,B,C$ 和 $D$。 查找 $A$ 和 $B$（含）之间不能被 $C$ 或 $D$ 均分的整数数。

```input1
4 9 2 3
```

```output1
2
```

```input2
10 40 6 8
```

```output2
23
```

```input3
314159265358979323 846264338327950288 419716939 937510582
```

```output3
532105071133627368
```

## 提示
### 制s

- $ 1\leq\ A\leq\ B\leq\ 10^{18} $
- $ 1\leq\ C,D\leq\ 10^9 $
- 入力はすべて整数である
