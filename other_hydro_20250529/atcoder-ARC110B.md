## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc110/tasks/arc110_b

`110` を $ 10^{10} $ BYした文字列を $ S $ とします（たとえば `110` を $ 3 $ BYした文字列は `110110110` です）。

Lさ $ N $ の文字列 $ T $ があります。

$ S $ に $ T $ がBAする部分文字列としていくつ含まれるかを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ T $

## 输出格式
$ S $ に $ T $ がBAする部分文字列としていくつ含まれるかを出力せよ。

```input1
4
1011
```

```output1
9999999999
```

```input2
22
1011011011011011011011
```

```output2
9999999993
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ T $ は `0`, `1` からなるLさ $ N $ の文字列

### Sample Explanation 1

$ S $ はLいので、`110` を $ 3 $ BYした `110110110` に `1011` がいくつ含まれるかを考えます。 すると、 - $ 1 $ `1011` $ 0110 $ - $ 1101 $ `1011` $ 0 $ の $ 2 $ w所に、`1011` がBAする部分文字列として含まれています。
