## 题目描述

$T$ 个队在做 $P$ 个题。现在给出每个队解决题的个数，及每个题被通过的次数。  
问这个情况是否存在，如果存在输出字典序最小的一个解，不存在输出 `Impossible`。  

## 输入格式

第一行给出 $T$ 和 $P$。  
第二行 $T$ 个数字，其值在 $0$ 到 $90$ ，互不相同。  
第三行 $P$ 个数字，代表 $P$ 个题各有多少队通过了。  
**本题有多组测试数据**，做到 `0 0` 结束。

## 输出格式

如题

```input1
2 2
1 2
1 1
3 3
2 1 2
1 2 2
3 5
3 3 1
3 1 1 0 2
0 0
```

```output1
Impossible
NYY
NNY
YYN
YNYNY
YYNNY
YNNNN
```

## 数据规模与约定

$100\%$ 的数据满足：$1 \le T,P \le 80$。
