## 题目描述
[problemUrl]: https://atcoder.jp/contests/nikkei2019-2-qual/tasks/nikkei2019_2_qual_b

$ N $ 要素からなる整数列 $ D_1,...,D_N $ が与えられます。点に $ 1 $ から $ N $ の番号が付けられた $ N $ 点からなる木であって、 以下の条件をみたすものの数を $ 998244353 $ で割ったあまりを求めてください。

- $ 1 $ 以上 $ N $ 以下の任意の整数 $ i $ にして、点 $ 1 $ と点 $ i $ の距xが $ D_i $ である。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ D_1 $ $ D_2 $ $ ... $ $ D_N $

## 输出格式
答えを出力せよ。

## 题目大意
* 一棵有 $n$ 个节点，以 $1$ 为根的有根树是好的，当且仅当第 $i$ 个节点的深度为 $a_i$（根的深度为 $0$）。
* 计算有多少好的树，答案对 $998244353$ 取模。
* $1\leq n\leq10^5,0\leq a_i<n$

```input1
4
0 1 1 2
```

```output1
2
```

```input2
4
1 1 1 1
```

```output2
0
```

```input3
7
0 3 2 1 2 2 1
```

```output3
24
```

## 提示
### 注

- $ N $ 点の木とは $ N $ 点 $ N-1 $ xからなるBYo向グラフのことであり、$ 2 $ 点の距xとは一方から他方への最短路に用いられるxの数を指します。
- $ 2 $ つの木がなるとは、ある $ 2 $ 点 $ x $, $ y $ が存在して、$ x $ と $ y $ のgに一方の木ではxが存在し、 もう一方の木ではxが存在しないことを指します。

### 制s

- $ 1\ Q\ N\ Q\ 10^5 $
- $ 0\ Q\ D_i\ Q\ N-1 $

### Sample Explanation 1

例えば、$ (1,2) $, $ (1,3) $, $ (2,4) $ のgにxがあるような木が条件をみたします。
