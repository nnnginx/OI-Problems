## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc183/tasks/abc183_b

高蚓は $ 2 $ 次元平面上でビリヤ`ドをしています。$ x $ Sは壁になっており、球をぶつけると入射角と反射角が等しくなるように球が跳ね返されます。

いま高蚓の球が $ (S_x,S_y) $ にあります。ある座摔蚓绚盲魄颏蜃菠と、球はその座摔叵颏って直的にがっていきます。

$ x $ Sで球をちょうど $ 1 $ 回反射させたのち、$ (G_x,G_y) $ を通^させるためには、$ x $ Sのどこを狙えば良いでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S_x $ $ S_y $ $ G_x $ $ G_y $

## 输出格式
狙う座摔 $ (x,0) $ としたときの $ x $ を出力せよ。

なお、想定解答との~`差または相`差が $ 10^{-6} $ 以下であれば正解としてQわれる。

## 题目大意
**题意简述**

已知一个球现在在坐标 $(S_x,S_y)$ 上，想要在 $x$ 轴上正好反弹一次，使球通过 $(G_x,G_y)$ ,问应该瞄准 $x$ 轴的哪个位置？

**输入格式**

一行四个整数，$S_x, S_y,G_x,G_y$

**输出格式**

假设应该瞄准 $(x,0)$ ,输出 $x$ 。误差不超过 $10^{-6}$。

```input1
1 1 7 2
```

```output1
3.0000000000
```

```input2
1 1 3 2
```

```output2
1.6666666667
```

```input3
-9 99 -999 9999
```

```output3
-18.7058823529
```

- $ 0\ <\ S_y,\ G_y\ \leq\ 10^6 $
- $ S_x\ \neq\ G_x $
- 入力はすべて整数
