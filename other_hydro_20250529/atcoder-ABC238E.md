## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc238/tasks/abc238_e

高颏んは秘密の整数列 $ a $ を持っており、Fr点で、$ a $ のLさが $ N $ であることは分かっています。

$ a $ の中身を当てたいあなたにし、高颏んは以下の $ Q $ の情螭蜃芳婴怯毪à皮れることをs束しました。

- $ i\ (1\ \leq\ i\ \leq\ Q) $ 目の情: $ a_{l_i}+a_{l_i+1}+\cdots+a_{r_i} $ の

高颏んがs束を守り、$ Q $ の情螭工伽皮与えられた龊稀$ a $ に含まれる全要素のt和 $ a_1+a_2+\cdots+a_N $ を特定することは可能ですか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ Q $ $ l_1 $ $ r_1 $ $ l_2 $ $ r_2 $ $ \hspace{0.4cm}\vdots $ $ l_Q $ $ r_Q $

## 输出格式
$ a $ に含まれる全要素のt和を特定することが可能なら `Yes` を、そうでないなら `No` を出力せよ。

## 题目大意
输入一个 $n$ 和 $q$ 分别表示数组长度为 $n$，有 $q$ 次输入：

每次输入一个 $l$ 和 $r$，表示我们知道 $l$ 到 $r$ 区间的和

问你最后能否知道数组的和

如果可以输出 `Yes` ，否则输出 `No` 。

```input1
3 3
1 2
2 3
2 2
```

```output1
Yes
```

```input2
4 3
1 3
1 2
2 3
```

```output2
No
```

```input3
4 4
1 1
2 2
3 3
1 4
```

```output3
Yes
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ Q\ \leq\ \min(2\ \times\ 10^5,\frac{N(N+1)}{2}) $
- $ 1\ \leq\ l_i\ \leq\ r_i\ \leq\ N $
- $ (l_i,r_i)\ \neq\ (l_j,r_j)\ (i\ \neq\ j) $
- 入力はすべて整数

### Sample Explanation 1

$ 1 $ 目の情螭 $ 2 $ 目の情螭ら、$ a_1+a_2+a_2+a_3 $ のが分かります。そこから $ 3 $ 目の情螭摔瑜盲频盲椁欷 $ a_2 $ のを引くと、$ a_1+a_2+a_3 $ のを特定可能です。

### Sample Explanation 2

$ a $ の先^ $ 3 $ のt和を特定することは可能ですが、全要素のt和を特定することは不可能です。

### Sample Explanation 3

$ 4 $ 目の情螭摔瑜盲迫要素のt和が直接与えられています。
