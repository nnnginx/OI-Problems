正整数 $ N $ が与えられます。  
 正整数のM $ (A,B,C,D) $ であって、$ AB\ +\ CD\ =\ N $ を氦郡工猡韦数を求めてください。

なお、本の制sの下、答えが $ 9\ \times\ 10^{18} $ 以下であることが^明できます。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
### 题意简述

给定正整数 $N$，求**有序正整数**四元组 $(A,B,C,D)$ 的个数使得 $AB+CD=N$。

注：有序指**实质相同，但顺序不同的算不同的两个。**

```input1
4
```

```output1
8
```

```input2
292
```

```output2
10886
```

```input3
19876
```

```output3
2219958
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ N $ は整数
 
### Sample Explanation 1

$ (A,B,C,D) $ として以下の $ 8 $ が考えられます。 - $ (A,B,C,D)=(1,1,1,3) $ - $ (A,B,C,D)=(1,1,3,1) $ - $ (A,B,C,D)=(1,2,1,2) $ - $ (A,B,C,D)=(1,2,2,1) $ - $ (A,B,C,D)=(1,3,1,1) $ - $ (A,B,C,D)=(2,1,1,2) $ - $ (A,B,C,D)=(2,1,2,1) $ - $ (A,B,C,D)=(3,1,1,1) $
