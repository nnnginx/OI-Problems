## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc021/tasks/agc021_e

AtCoder 共和国では、カメレオン科ボ`ルタベルカメレオン属に属するスヌケカメレオンがペットとして大人荬扦埂 りんごさんは、$ N $ 匹のスヌケカメレオンの体をひとつのカゴに入れてっています。

何も食べていない状Bのスヌケカメレオンは青色です。スヌケカメレオンは、次のtで渖します。

- 青いスヌケカメレオンは、これまでに食べた青いボ`ルの数よりこれまでに食べた赤いボ`ルの数の方が真に大きくなったr、赤色に渖する。
- 赤いスヌケカメレオンは、これまでに食べた赤いボ`ルの数よりこれまでに食べた青いボ`ルの数の方が真に大きくなったr、青色に渖する。

最初、スヌケカメレオンたちはどの体も何も食べていない状Bです。りんごさんは、スヌケカメレオンたちに、以下の手を $ K $ 回Rり返すことでDをやりました。

- 赤いボ`ルまたは青いボ`ルを握る。
- 握ったボ`ルを、スヌケカメレオンたちの入ったカゴの中に投げ入れる。このとき、いずれか一匹がそのボ`ルを食べる。

りんごさんが $ K $ のボ`ルを投げ入れたところ、全ての体が赤色になっていました。りんごさんの $ K $ のボ`ルの投げ入れ方としてありうるものは何通りあるでしょうか。 $ 998244353 $ で割った余りを求めてください。ただし、$ 2 $ つの投げ入れ方がなるとは、ある $ i $ が存在し、$ i $ 目に投げ入れたボ`ルの色がなることを指します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
$ K $ のボ`ルの投げ入れ方としてありうるものの数を $ 998244353 $ で割った余りを出力せよ。

## 题目大意
有 $n$ 只变色龙，一开始都是蓝色。现在你喂了 $k$ 次球，每次指定一只变色龙吃下你指定颜色的球。

一只变色龙从蓝色变成红色当且仅当它吃的红球比蓝球多；
一只变色龙从红色变成蓝色当且仅当它吃的蓝球比红球多。

求最后能使所有变色龙都变成红色的方案数。

两个方案不同当且仅当至少一次喂的球颜色不同（**而不是喂的变色龙不同**）。

**注意：存在一次喂的变色龙不同的两个方案可能是相同的方案。**

```input1
2 4
```

```output1
7
```

```input2
3 7
```

```output2
57
```

```input3
8 3
```

```output3
0
```

```input4
8 10
```

```output4
46
```

```input5
123456 234567
```

```output5
857617983
```

## 提示
### 制s

- $ 1\ \leq\ N,K\ \leq\ 5\ \times\ 10^5 $
- $ N,K $ は整数である

### Sample Explanation 1

$ i $ 目に投げ入れるボ`ルが赤のとき `R` を、青のとき `B` をにKべた文字列を用いて投げ入れ方を表せば、 `BRRR`,`RBRB`,`RBRR`,`RRBB`,`RRBR`,`RRRB`,`RRRR` の $ 7 $ が条件を氦郡筏蓼埂
