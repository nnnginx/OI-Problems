## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc063/tasks/abc063_b

英小文字からなる文字列 $ S $ が与えられます。$ S $ に含まれる文字がすべてなるか判定してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ に含まれる文字がすべてなる龊悉 `yes`（英小文字）、そうでない龊悉 `no` と出力せよ。

## 题目大意
给定一个字符串S，判断其中字符，如果全部不相同，输出yes，否则输出no

```input1
uncopyrightable
```

```output1
yes
```

```input2
different
```

```output2
no
```

```input3
no
```

```output3
yes
```

## 提示
### 制s

- $ 2\ <\ =\ |S|\ <\ =\ 26 $, ここで $ |S| $ は $ S $ のLさを表す。
- $ S $ は英小文字のみからなる。
