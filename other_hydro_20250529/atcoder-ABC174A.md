## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc174/tasks/abc174_a

あなたは、室温が $ 30 $ 度以上のとき、またそのときに限り、冷房の源を入れます。

今の室温は $ X $ 度です。冷房の源を入れますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $

## 输出格式
冷房の源を入れるならば `Yes`、入れないならば `No` を出力せよ。

## 题目大意
输入一个数 $n$，如果 $n\ge 30$，输出 `Yes`，否则输出 `No`，末尾记得换行。

translate by @[Jasoncwx](https://www.luogu.com.cn/user/592684)。

```input1
25
```

```output1
No
```

```input2
30
```

```output2
Yes
```

## 提示
### 制s

- $ -40\ \leq\ X\ \leq\ 40 $
- $ X $ は整数である。
