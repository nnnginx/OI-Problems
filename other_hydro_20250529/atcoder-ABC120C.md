## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc120/tasks/abc120_c

机の上に $ N $ のキュ`ブがkにeまれています。Lさ $ N $ の文字列 $ S $ が与えられます。

下から $ i $ 番目のキュ`ブの色は、$ S $ の $ i $ 文字目が `0` のとき赤色、`1` のとき青色です。

あなたは、赤色のキュ`ブと青色のキュ`ブがOり合っているような部分をxんで、それら $ 2 $ のキュ`ブを取り除く操作を何度でも行えます。

このとき、取り除いたキュ`ブの上にあったキュ`ブは真下の物体の上に落下します。

最大で何のキュ`ブを取り除けるでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
最大で何のキュ`ブを取り除けるかを出力せよ。

## 题目大意
给你一段只有 `0` 和 `1` 的字符串，如果两个相邻的字符不一样，那么就可以删掉这两个字符，其他字符不变。求一共可以删除掉多少个字符。

```input1
0011
```

```output1
4
```

```input2
11011010001011
```

```output2
12
```

```input3
0
```

```output3
0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ |S|\ =\ N $
- $ S $ の各文字は `0` または `1` である。

### Sample Explanation 1

以下のに操作を行うと $ 4 $ 全てのキュ`ブを取り除けます。 - 下から $ 2 $ 番目のキュ`ブと $ 3 $ 番目のキュ`ブを取り除きます。そのY果、下から $ 4 $ 番目のキュ`ブが下から $ 1 $ 番目のキュ`ブの上に落下します。 - 下から $ 1 $ 番目のキュ`ブと $ 2 $ 番目のキュ`ブを取り除きます。
