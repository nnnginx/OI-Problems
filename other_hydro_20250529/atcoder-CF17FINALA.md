## 题目描述
[problemUrl]: https://atcoder.jp/contests/cf17-final/tasks/cf17_final_a

文字列 $ S $ が与えられます。

高蚓はこの文字列の好きな位置に好きなだけ文字 `A` を啡毪工毪长趣できます。

$ S $ を `AKIHABARA` に浃à毪长趣悉扦るでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $

## 输出格式
$ S $ を `AKIHABARA` に浃à毪长趣できる龊悉 `YES`、できない龊悉 `NO` を出力せよ。

## 题目大意
# 题目描述
输入一个字符串S，高桥君在其中任意位置插入A，问是否可以变成AKIHABARA
## 输入格式
一个字符串S
## 输出格式
可以输出YES，否则输出NO

```input1
KIHBR
```

```output1
YES
```

```input2
AKIBAHARA
```

```output2
NO
```

```input3
AAKIAHBAARA
```

```output3
NO
```
