## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc166/tasks/abc166_f

あるゲ`ムでは $ 3 $ つの涫があり、それぞれ $ A,B,C $ で表されます。

ゲ`ムのM行と共に、あなたは $ N $ 回のxkに迫られます。 それぞれのxkは文字列 $ s_i $ によって示され、 $ s_i $ が `AB` のとき、$ A $ と $ B $ のどちらかに $ 1 $ を足しもう一方から $ 1 $ を引くこと、 `AC` のとき、$ A $ と $ C $ のどちらかに $ 1 $ を足しもう一方から $ 1 $ を引くこと、 `BC` のとき、$ B $ と $ C $ のどちらかに $ 1 $ を足しもう一方から $ 1 $ を引くことを意味します。

いずれのxkの後にも、$ A,B,C $ のいずれものになってはいけません。

この条件を氦郡筏膜 $ N $ 回すべてのxkをKえることが可能であるか判定し、可能であるならそのようなxk方法をひとつ示してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $ $ C $ $ s_1 $ $ s_2 $ $ : $ $ s_N $

## 输出格式
条件を氦郡筏膜 $ N $ すべてのxkをKえることが可能である龊悉 `Yes` を、そうでない龊悉 `No` を出力せよ。

加えて、前者の龊悉暇Aく $ N $ 行にxk方法を示せ。$ i+1 $ 行目には $ i $ 回目のxkで $ 1 $ を足す涫の名前 (`A`, `B`, `C` のいずれか) を出力せよ。

## 题目大意
你开始有三个变量$A$,$B$,$C$,它们各自有一个值.

你需要依次进行$n$次操作,对于每次操作,有一个参数$s$,这个参数只可能是"AB","AC","BC"中的一个.

当s="AB"时,你可以选择以下方案中的一种:

1. $A=A+1,B=B-1$
2. $A=A-1,B=B+1$

当s="AC"时,你可以选择以下方案中的一种:

1. $A=A+1,C=C-1$
2. $A=A-1,C=C+1$

当s="BC"时,你可以选择以下方案中的一种:

1. $B=B+1,C=C-1$
2. $B=B-1,C=C+1$

若存在一种选择操作的方案使每次操作结束之后$A$,$B$,$C$均为自然数,输出"Yes",并依次输出每次操作中变大的变量名.否则输出"No".

```input1
2 1 3 0
AB
AC
```

```output1
Yes
A
C
```

```input2
3 1 0 0
AB
BC
AB
```

```output2
No
```

```input3
1 0 9 0
AC
```

```output3
No
```

```input4
8 6 9 1
AC
BC
AB
BC
AC
BC
AB
AB
```

```output4
Yes
C
B
B
C
C
B
A
A
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^5 $
- $ 0\ \leq\ A,B,C\ \leq\ 10^9 $
- $ N,\ A,\ B,\ C $ は整数である。
- $ s_i $ は `AB`, `AC`, `BC` のいずれか

### Sample Explanation 1

次のようにすることで $ 2 $ 回すべてのxkをKえることができます。 - $ 1 $ 回目のxkでは、$ A $ に $ 1 $ を足し $ B $ から $ 1 $ を引く。$ A $ のが $ 2 $ に、$ B $ のが $ 2 $ に浠する。 - $ 2 $ 回目のxkでは、$ C $ に $ 1 $ を足し $ A $ から $ 1 $ を引く。$ C $ のが $ 1 $ に、$ A $ のが $ 1 $ に浠する。
