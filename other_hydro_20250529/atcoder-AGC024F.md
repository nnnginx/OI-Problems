## 问题陈述

给定一个由 `0` 和 `1` 组成的字符串集合 $S$，以及一个整数 $K$。

找到一个最长的字符串，它是 $S$ 中 $K$ 个或更多不同字符串的子序列。如果有多个字符串满足此条件，则找到字典序最小的字符串。

这里，$S$ 的格式如下：

- 给定的数据是一个整数 $N$，以及 $N+1$ 个字符串 $X_0,X_1,...,X_N$。对于每个 $i$ $(0\leq i\leq N)$，$X_i$ 的长度为 $2^i$。
- 对于每对整数 $(i,j)$ $(0\leq i\leq N,0\leq j\leq 2^i-1)$，$X_i$ 的第 $j$ 个字符是 `1` 当且仅当 $j$ 的二进制表示（可能带有前导零）属于 $S$。这里，$X_i$ 的第一个和最后一个字符分别称为第 $0$ 个和第 $(2^i-1)$ 个字符。
- $S$ 中不包含长度为 $N+1$ 或更长的字符串。

这里，当存在一系列整数 $t_1 < ... < t_{|A|}$，使得对于每个 $i$ $(1\leq i\leq |A|)$，字符串 $A$ 的第 $i$ 个字符与字符串 $B$ 的第 $t_i$ 个字符相等时，字符串 $A$ 是字符串 $B$ 的子序列。

## 约束条件

- $0 \leq N \leq 20$
- $X_i(0\leq i\leq N)$ 是一个长度为 $2^i$ 的字符串，由 `0` 和 `1` 组成。
- $1 \leq K \leq |S|$
- $K$ 是一个整数。

## 输入

输入通过标准输入给出，格式如下：

> $N$ $K$
> 
> $X_0$
> 
> $:$
> 
> $X_N$

## 输出

打印在 $K$ 或更多不同字符串中最长字符串的字典序最小字符串。

```input1
3 4
1
01
1011
01001110
```

```output1
10
```

以下字符串属于 $S$：空字符串，`1`，`00`，`10`，`11`，`001`，`100`，`101` 和 `110`。
在四个或更多字符串中，最长字符串的字典序最小的是 `10`。

```input2
4 6
1
01
1011
10111010
1101110011111101
```

```output2
100
```

```input3
2 5
0
11
1111
```

```output3

```

答案是空字符串。
