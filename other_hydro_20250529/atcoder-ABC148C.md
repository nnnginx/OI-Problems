## 题目大意
高桥正在组织一个聚会。

在聚会上，每位客人将获得一个或多个零食。

高桥预计这次聚会的客人人数将是 $A$ 或 $B$。

在两种预测的情况下，找出可以平均分配给客人的最少件数。

我们假设一件作品无法分割并分发给多个客人。


```output1

```

```input2

```

```output2

```

```input3

```

```output3

```
