## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc295/tasks/abc295_b

k $ R $ 行横 $ C $ 列のP面があります。上から $ i $ 行目、左から $ j $ 列目のマスを $ (i,j) $ と表します。

$ (i,j) $ のF在の状Bが文字 $ B_{i,j} $ として与えられます。 `.` は空きマス、`#` は壁があるマスを表し、 `1`, `2`,$ \dots $, `9` はそれぞれ威力 $ 1,2,\dots,9 $ の爆があるマスを表します。

次の瞬gに、全ての爆が**同rに**爆kします。 爆が爆kすると、爆があるマスからのマンハッタン距xがその爆の威力以下であるような全てのマス（その爆があるマス自体を含む）が空きマスに浃铯辘蓼埂 ここで、$ (r_1,c_1) $ から $ (r_2,c_2) $ までのマンハッタン距xは $ |r_1-r_2|+|c_1-c_2| $ です。

爆k後のP面を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ R $ $ C $ $ B_{1,1}B_{1,2}\dots\ B_{1,C} $ $ \vdots $ $ B_{R,1}B_{R,2}\dots\ B_{R,C} $

## 输出格式
爆k後のP面を $ R $ 行で出力せよ。P面の表し方は入力と同じ形式を用いること ($ R $ と $ C $ を出力する必要はない)。

## 题目大意
### 题目描述

有一个 $R$ 行 $C$ 列的盘面，点 $(i, j)$ 表示第 $i$ 行第 $j$ 列的格子。上面有三种状态，`.` 代表空地，`#` 代表一堵墙，`1`, `2`, `3` ... `9` 代表炸弹。上面的 `1` ~ `9` 代表炸弹的威力值。

下面，所有的炸弹**同时爆炸**，所有曼哈顿距离小于等于每个点上的威力值的墙将变成空地。

注意：如果该点在爆炸范围内，且该点是炸弹，则不会变成空地。爆炸后，该炸弹本身会变成空地。

你需要输出爆炸后的盘面。

### 输入格式

第一行，两个数字，代表 $R$ 和 $C$。

接下来 $R$ 行，代表盘面的每一行。

### 输出格式

$R$ 行，爆炸后的盘面。

### 数据规模与约定

- $1 \le R, C \le 20$

- $R, C$ 均为整数

- 每个地方只能是 `.`, `#`, `1`, `2`, `3` ... `9` 的字符

```input1
4 4
.1.#
###.
.#2.
#.##
```

```output1
...#
#...
....
#...
```

```input2
2 5
..#.#
###.#
```

```output2
..#.#
###.#
```

```input3
2 3
11#
###
```

```output3
...
..#
```

```input4
4 6
#.#3#.
###.#.
##.###
#1..#.
```

```output4
......
#.....
#....#
....#.
```
