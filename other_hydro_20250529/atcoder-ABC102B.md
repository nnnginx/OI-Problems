## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc102/tasks/abc102_b

Lさ $ N $ の整数列 $ A $ が与えられます。 $ A $ の（添字の）なる $ 2 $ 要素の差の~の最大を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ $ ... $ $ A_N $

## 输出格式
$ A $ の（添字の）なる $ 2 $ 要素の差の~の最大を出力せよ。

## 题目大意
输入 $n$ 个数，输出这 $n$ 个数中最大数与最小数的差。

```input1
4
1 4 6 3
```

```output1
5
```

```input2
2
1000000000 1
```

```output2
999999999
```

```input3
5
1 1 1 1 1
```

```output3
0
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ A_i\ \leq\ 10^9 $
- 入力はすべて整数である。

### Sample Explanation 1

$ A_3-A_1=6-1=5 $ が、なる $ 2 $ 要素の差の最大です。
