## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc233/tasks/abc233_a

サンタさんに手を出したい高颏んは、 $ X $ 仪惺证 $ 1 $ 枚だけNられた封筒を用意しました。  
 サンタさんに手を届けるためには、Nられている切手のt~が $ Y $ 乙陨悉扦る必要があります。  
 高颏んは、この封筒に $ 10 $ 仪惺证蚝蚊钉Nり足すことで、Nられている切手のt~を $ Y $ 乙陨悉摔筏郡い扦埂  
 高颏んはこの封筒に、最小で何枚の $ 10 $ 仪惺证蛸Nり足す必要がありますか?

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ Y $

## 输出格式
答えを整数として出力せよ。

## 题目大意
给定 $x,y$，求 $x$ 需要加上多少个 $10$（可以是 $0$ 个）才能超过 $y$。

Tip：虽然题目要求的是超过 $y$，但是实际上是**大于等于** $y$ 即可。

Translated by ShanCreeper.

```input1
80 94
```

```output1
2
```

```input2
1000 63
```

```output2
0
```

```input3
270 750
```

```output3
48
```
