## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc163/tasks/abc163_c

$ N $ 人の社Tからなる会社があり、各社Tには $ 1,...,N $ の社T番号が割り当てられています。

社T番号 $ 1 $ の社T以外の全ての社Tには、自分より社T番号が小さい直属の上司がちょうど $ 1 $ 人います。

$ X $ さんが $ Y $ さんの直属の上司であるとき、$ Y $ さんは $ X $ さんの直属の部下であるといいます。

社T番号 $ i $ の社Tの直属の上司の社T番号が $ A_i $ であることが与えられます。各社Tについて直属の部下が何人いるか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_2 $ $ ... $ $ A_N $

## 输出格式
社T番号 $ 1,2,...,N $ のそれぞれの社Tについて、直属の部下が何人いるか、改行区切りで出力せよ。

## 题目大意
一个公司有 $N$ 名员工，用编号 $1...N$ 表示，编号 $1$ 的人是领导。对于每个编号 $2...N$ 的员工，有 $a_i$ 表示其上司。问每个人领导的员工有多少个？

```input1
5
1 1 2 2
```

```output1
2
2
0
0
0
```

```input2
10
1 1 1 1 1 1 1 1 1
```

```output2
9
0
0
0
0
0
0
0
0
0
```

```input3
7
1 2 3 4 5 6
```

```output3
1
1
1
1
1
1
0
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ A_i\ <\ i $

### Sample Explanation 1

社T番号 $ 1 $ の社Tの直属の部下は社T番号 $ 2,3 $ の $ 2 $ 人です。 社T番号 $ 2 $ の社Tの直属の部下は社T番号 $ 4,5 $ の $ 2 $ 人です。 社T番号 $ 3,4,5 $ の社Tには直属の部下はいません。
