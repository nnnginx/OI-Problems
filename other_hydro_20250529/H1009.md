## 题目描述

おりがみ 潜入了敌方基地内部，为了传递情报，她决定给 ことり 发一份密报。


在高端的压缩技术的加持下，这份密保仅由 $A,B,C$ 三种字符组成。由于 $おりがみ$ 手头没有什么工具，她决定用栈来发送这份密报。并且希望操作完成后栈为空。

她可以对这个栈进行三种操作:

- 压入一个字符
- 弹出一个字符
- 打印栈顶字符

为了尽快传递情报，她希望你能告诉她对于这份密报最少需要几步来打印。

## 输入格式

仅一行一个只由 $A,B,C$ 组成的字符串 $S$。

## 输出格式

仅一行，为最小的操作数。

```input1
ABCCBA

```
```output1
12
```
```input2
AAABAAB

```
```output2
13
```

## 数据规模与约定

对于 $10\%$ 的数据，$|S|\leq 10$。  
对于 $20\%$ 的数据，$|S|\leq 100$。  
对于 $30\%$ 的数据，$|S|\leq 2000$。  
对于 $100\%$ 的数据，$|S|\leq 10^6$。
