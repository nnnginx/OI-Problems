## 题目描述

给定 $n$ 个长度分别为 $a_i$ 的木棒，问随机选择 $3$ 个木棒能够拼成三角形的概率。

## 输入格式

第一行 $T$，表示数据组数。

接下来若干行描述 $T$ 组数据，每组数据第一行是 $n$，接下来一行有 $n$ 个数表示 $a_i$。

## 输出格式

$T$ 行，每行一个数，四舍五入保留 $7$ 位小数。

```input1
2
4
1 3 3 4
4
2 3 3 4
```
```output1
0.5000000
1.0000000
```

## 提示

$T \le 20,3 \le n \le 10^5,1 \le a_i \le 10^5$
