## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc190/tasks/abc190_d

整数からなる公差 $ 1 $ の等差数列のうち、t和が $ N $ であるものはいくつあるでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
- 给定一个整数 $N$，求出有多少个由整数组成的、公差为 $1$ 的等差数列各项之和为 $N$。
- 对于 $100\%$的数据，$1 \leq N \leq 10^{12}$，且 $N$ 为整数。

```input1
12
```

```output1
4
```

```input2
1
```

```output2
2
```

```input3
963761198400
```

```output3
1920
```

## 提示
### 制s

- $ 1\ <\ =\ N\ <\ =\ 10^{12} $
- $ N $ は整数

### Sample Explanation 1

\- $ [12] $ - $ [3,\ 4,\ 5] $ - $ [-2,\ -1,\ 0,\ 1,\ 2,\ 3,\ 4,\ 5] $ - $ [-11,\ -10,\ -9,\ \dots,\ 10,\ 11,\ 12] $ の $ 4 $ です。

### Sample Explanation 2

\- $ [1] $ - $ [0,\ 1] $ の $ 2 $ です。
