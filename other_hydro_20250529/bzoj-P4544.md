## 题目描述

求不定方程 $x^2+3y^2=n^2$ 整数解的个数。

## 输入格式

第一行一个整数 $T$，表示输入包含 $T$ 组数据。

接下来 $T$ 行，每行一个正整数 $n$。


## 输出格式

输出应包含 $T$ 行，每行一个整数 $N$，表示该组数据的答案。

## 样例输入

```plain
3
5
1001
191752422
```

## 样例输出

```plain
2
18
162
```

## 数据规模与约定

$1\le T \le 10$，$1\le n \le 10^{12}$。
