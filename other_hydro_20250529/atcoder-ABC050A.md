## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc050/tasks/abc050_a

joisinoおちゃんは、$ A $ $ op $ $ B $ という式のを算したいと思っています。 ここで、$ A,B $ は整数で、$ op $ は、`+` または `-` の号です。 あなたの仕事は、joisinoおちゃんの代わりにこれを求めるプログラムを作ることです。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ op $ $ B $

## 输出格式
式のを出力せよ。

## 题目大意
### **题目描述**
#### 读入a,b,x，如果x='+'就输出a+b，否则输出a-b（注意：a和x之间有一个空格，x和b之间有一个空格！！）。
### **输入格式**
```
a，空格，x，空格，b
```
### **输出格式**
#### a+b或a-b。
### **输入样例1**
```
1 + 2
```
### **输出样例1**
```
3
```
### **输出样例2**
```
5 - 7
```
### **输出样例2**
```
-2
```

```input1
1 + 2
```

```output1
3
```

```input2
5 - 7
```

```output2
-2
```

## 提示
### 制s

- $ 1QA,BQ10^9 $
- $ op $ は、`+` または `-` の号である。

### Sample Explanation 1

$ 1\ +\ 2\ =\ 3 $ なので $ 3 $ を出力します。
