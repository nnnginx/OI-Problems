## 题目背景
第二届四川省青少年 C++ 算法设计大赛小学 4-5 年级组全省总决赛试题。数据为洛谷自造。

## 题目描述
最近，学校门口的小卖部推出了一个活动：每 $a$ 个瓶盖可以兑换 $5$ 元的现金券。现在，小 X 手里有 $n$ 个瓶盖，他打算把这些瓶盖全部都拿去兑换现金券，小 X 想知道他最多能兑换得到多少元的现金券，以及此刻他手里还剩的瓶盖的数量。


## 输入格式
一行两个整数 $n,a$，如题目中所描述的含义。

## 输出格式
一行用空格隔开的两个整数 $x,y$，表示小 X 最多兑换得到了 $x$ 元的现金券并还剩 $y$ 个瓶盖。

```input1
10 3
```

```output1
15 1
```

## 提示
### 样例解释
小 X 有 $10$ 个瓶盖，每 $3$ 个瓶盖可以兑换一张 $5$ 元现金券，小 X 一共兑换得到了 $15$ 元的现金券，还剩下 $1$ 个瓶盖。

### 数据范围及约定
对于 $100\%$ 的数据，$1\leq n,a\leq 1000$。
