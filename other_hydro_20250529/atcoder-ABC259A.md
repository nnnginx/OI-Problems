## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc259/tasks/abc259_a

高蚓は $ N $ rのQ生日を迎えました。このrの彼の身Lは $ T $ cmです。  
 また、以下のことが分かっています。

- 高蚓は $ 0 $ rのQ生日(生まれた当日)から $ X $ rのQ生日までのg、澳晟黹Lが $ D $ cmずつ伸びた。より密にくと、$ i=1,2,\ldots,X $ それぞれにし、$ i-1 $ rのQ生日から $ i $ rのQ生日までのgに身Lが $ D $ cm伸びた。
- 高蚓は $ X $ rのQ生日から $ N $ rのQ生日までのg、身Lが浠していない。

高蚓の $ M $ rのQ生日のrの身Lが何cmだったかを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ X $ $ T $ $ D $

## 输出格式
答えを整数として出力せよ。

## 题目大意
题意：

高桥君正在过他$N$岁的生日。此时他身高为$Tcm$.

高桥君从出生到$X$岁生日期间，每年长高$Dcm$.（严格来说是从$i-1$岁生日到$i$岁生日之间长高$Dcm$）

求高桥君$M$岁生日时的身高。

数据范围：

+ $0\le M<N \le 100$
+ $1 \le X \le N$
+ $1 \le T \le 200$
+ $1\le D \le 100$

```input1
38 20 17 168 3
```

```output1
168
```

```input2
1 0 1 3 2
```

```output2
1
```

```input3
100 10 100 180 1
```

```output3
90
```

## 提示
### 制s

- $ 0\ \leq\ M\ \lt\ N\ \leq\ 100 $
- $ 1\ \leq\ X\ \leq\ N $
- $ 1\ \leq\ T\ \leq\ 200 $
- $ 1\ \leq\ D\ \leq\ 100 $
- 高蚓の $ 0 $ rのQ生日のrの身Lは $ 1 $ cm以上である
- 入力はすべて整数

### Sample Explanation 1

この例では、高蚓の $ 38 $ rのQ生日のrの身Lが $ 168 $ cmです。また、$ 17 $ rのQ生日から $ 38 $ rのQ生日までのg、身Lが浠していません。 このことから、$ 20 $ rのQ生日のrの身Lは $ 168 $ cmだったと言え、これが答えになります。

### Sample Explanation 2

この例において、高蚓は $ 0(=M) $ rのQ生日のrの身Lが $ 1 $ cmで、$ 1(=N) $ rのQ生日のrの身Lが $ 3(=T) $ cmです。
