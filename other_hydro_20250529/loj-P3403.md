## 题目描述

定义 $P(x)$ 表示满足 $1< y< x,y^3\equiv 1\pmod x$ 的 $y$ 的数量。

求在 $n$ 以内有多少正整数满足 $P(x)=m$。

## 输入格式

一行输入两个整数 $n,m$。

## 输出格式

输出一个数，表示答案。

```input1
10 0
```

```output1
8
```

```input2
100000000 242
```

```output2
24038
```
