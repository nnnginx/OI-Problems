## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc150/tasks/arc150_e

$ NK $ 人の人が横一列にKんでいます。左からに人 $ i\ (0\leq\ i\ \leq\ NK-1) $ と表します。

各人は常に左右どちらかの方向を向いています。r刻 $ t=0 $ において各人がどちらを向いているかは、`L`,`R` のみからなるLさ $ N $ の文字列 $ S=S_0\ S_1\ \dots\ S_{N-1} $ で表されます。r刻 $ t=0 $ において人 $ i $ は $ S_{i\ \bmod\ N} $ が `L` のとき左を、 `R` のとき右を向いています。

これらの人はr刻 $ t=0.5,\ 1.5,\ 2.5\ ,\ \dots $ において以下のtに兢盲葡颏い皮い敕较颏**同rに**浠させます。

- そのr点で左を向いている龊  
   自分が向いている方向に人が存在し、その中で^半数の人が右を向いている龊稀⑾颏い皮い敕较颏蛴窑浃à搿￥饯Δ扦胜龊舷颏い皮い敕较颏浃à胜ぁ
- そのr点で右を向いている龊  
   自分が向いている方向に人が存在し、その中で^半数の人が左を向いている龊稀⑾颏い皮い敕较颏蜃螭浃à搿￥饯Δ扦胜龊舷颏い皮い敕较颏浃à胜ぁ

r刻 $ t=0 $ から $ t=10^{100} $ までのgに、$ NK $ 人それぞれが向いている方向を浃à牖厥のt和を $ 998244353 $ で割った余りを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ K $ $ S $

## 输出格式
答えを出力してください。

## 题目大意
有 $n\times k$ 个人排成一行，从左往右按 $0,1,\cdots nk-1$ 编号。每个人初始都面对着一个方向 `L` 或 `R`。给出一个字符串 $s_{0\cdots n-1}$，则第 $i$ 个人的方向为 $s_{i \bmod n}$。

接下来进行若干轮操作，每一轮所有人 **同时进行如下操作**：

1. 若当前某人面对左边，且他左边的人中面对右边的人数超过一半，则他转头面向右边。
2. 若当前某人面对右边，且他右边的人中面对左边的人数超过一半，则他转头面向左边。

操作进行 $10^{100}$ 轮，请求出所有轮中每个人转头的次数之和。$n,k\le 2\times 10^5$。

```input1
7 1
RRLRLLL
```

```output1
9
```

```input2
4 10
LLRR
```

```output2
0
```

```input3
23 200
RLRRRLLLLLLLLRRRLLRLRRR
```

```output3
2207
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 2\ \times\ 10^5 $
- $ 1\ \leq\ K\ \leq\ 2\ \times\ 10^5 $
- $ S $ は `L`,`R` のみからなるLさ $ N $ の文字列
- 入力される数はすべて整数

### Sample Explanation 1

各r刻において $ 7 $ 人が向いている方向を文字列で表すと $ t=1 $ では `LLRLRRL` 、$ t=2 $ では `LLRLRLL` 、 $ t=3 $ では `LLLLLLL` となります。 r刻 $ t=3 $ 以降では $ 7 $ 人が向いている方向は浠しません。よって答えは $ 1+1+2+1+2+2+0=9 $ になります。
