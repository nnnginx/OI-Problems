花に $ N $ 本の花がDいており、それぞれ $ 1,2,......,N $ と番号が振られています。最初、全ての花の高さは $ 0 $ です。 数列 $ h=\{h_1,h_2,h_3,......\} $ が入力として与えられます。以下の「水やり」操作をRり返すことで、すべての $ k(1\ \leqq\ k\ \leqq\ N) $ にして花 $ k $ の高さを $ h_k $ にしたいです。

- 整数 $ l,r $ を指定する。$ l\ \leqq\ x\ \leqq\ r $ を氦郡工工伽皮 $ x $ にして、花 $ x $ の高さを $ 1 $ 高くする。

条件を氦郡工郡幛巫钚・巍杆やり」操作の回数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられます。

> $ N $ $ h_1 $ $ h_2 $ $ h_3 $ $ ...... $ $ h_N $

## 输出格式
条件を氦郡工瑜Δ首钚・巍杆やり」操作の回数を出力してください。

## 题目大意
花坛里有 $N$ 朵花，序号分别为 $1,2,...,N$ ,最初的高度都为 $0$ 。给定一个序列 $h=\{h_1,h_2,......,h_N\}$ ，你希望序号为 $k$ $ (1\ \leq\ k\ \leq\ N) $ 的花能长到 $h_k$ 的高度。你可以重复名为 &quot;浇水&quot; 的操作：

给出整数 $l$ 和 $r$ ，使所有序号 $x$ $ (l\ \leq\ x\ \leq\ r) $ 的花增长高度 $1$ 。

试求满足条件所需的最小 &quot;浇水&quot; 操作。

```input1
4
1 2 2 1
```

```output1
2
```

```input2
5
3 1 2 3 1
```

```output2
5
```

```input3
8
4 23 75 0 23 96 50 100
```

```output3
221
```

## 提示
### 制s

- $ 1\ \leqq\ N\ \leqq\ 100 $
- $ 0\ \leqq\ h_i\ \leqq\ 100 $
- 入力はすべて整数である。
