<p>&nbsp;</p>
<p><span style="color: #333300;"><span style="font-size: small;">A nice sequence is a sequence of digits in which a digit d is placed at any index iff d is 0 or any divisor of d(except 1)&nbsp; has been placed already. First digit can be anything from 1 to 9.</span></span></p>
<p><span style="color: #333300;"><span style="font-size: small;">Find the number of nice sequences of length n.</span></span></p>
<p><span style="font-size: 1.17em;">Input</span></p>
<p><span style="font-size: small;">Input consists of number of test cases t <br>Following t lines have a single line containing&nbsp; n as described in the problem statement.</span></p>
<p><span style="font-size: 1.17em;">Output</span></p>
<p><span style="font-size: small;">Print the number of nice sequences of length n modulo 1000000007&nbsp; in a seperate line .</span></p>
<h3><span style="font-size: small;">Example</span></h3>
<pre><span style="font-size: xx-small;"><strong>Input:
2
1
2

Output
9
23</strong></span></pre>
<pre></pre>
<pre><span style="font-size: small;">Explaination :
For n=2 nice sequences are :</span></pre>
<pre><span style="font-size: small;">10,20,22,24,26,28,30,33,36,39,40,44,48 and so on !
</span></pre>
<div></div>
<pre><span style="font-size: small;">Constraints:</span></pre>
<pre><span style="font-size: small;">1&lt;=n&lt;=1000</span></pre>
