## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc153/tasks/arc153_a

正整数 $ x $ が**美しい整数**であるとは，$ x $ が $ 9 $ 桁の整数であり，その $ 10 $ M法表 $ S_1\ldots\ S_9 $ （$ S_i $ は $ x $ の $ 10 $ M法表の $ i $ 文字目）が以下の条件をすべて氦郡工长趣颏いい蓼梗

- $ S_1 $ は `0` ではない
- $ S_1\ =\ S_2 $
- $ S_5\ =\ S_6 $
- $ S_7\ =\ S_9 $
 
例えば $ 998244353 $ や $ 333333333 $ は美しい整数です．$ 111112222 $ は $ S_5\ \neq\ S_6 $ であるため美しい整数ではありません．

正の整数 $ N $ が与えられます．小さい方から数えて $ N $ 番目の美しい整数を答えてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられます．

> $ N $

## 输出格式
小さい方から数えて $ N $ 番目の美しい整数を出力してください．

## 题目大意
我们定义一个 $9$ 位的正整数（设这个正整数从高位到低位的第 $i$ 位的数是 $S_i$，如 $143,446,709$，其中 $S_3 = 3$）是**美丽的正整数**，当且仅当满足：$S_1 \neq 0, S_1 = S_2, S_5 = S_6, S_7 = S_9$。

举个例子，$998244353$ 是美丽的正整数，但 $111112222$ 不是，因为 $S_5 \neq S_6$。

现在给你一个正整数 $N$，请输出第 $N$ 小的美丽的正整数。

输入保证答案一定存在。换句话说，一定存在第 $N$ 小的美丽的正整数。

```input1
3
```

```output1
110000020
```

```input2
882436
```

```output2
998244353
```

```input3
2023
```

```output3
110200222
```

## 提示
### 制s

- $ N $ は正の整数
- 美しい整数が $ N $ 以上存在する
 
### Sample Explanation 1

美しい整数を小さいにKべると，$ 110000000,\ 110000010,\ 110000020,\ \ldots $ となります．
