## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc127/tasks/abc127_c

$ N $ 枚の ID カ`ドと $ M $ のゲ`トがあります。

$ i $ 番目のゲ`トは $ L_i,\ L_i+1,\ ...,\ R_i $ 番目の ID カ`ドのうちどれか $ 1 $ 枚を持っていれば通^できます。

$ 1 $ 枚だけで全てのゲ`トを通^できる ID カ`ドは何枚あるでしょうか。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $ $ L_1 $ $ R_1 $ $ L_2 $ $ R_2 $ $ \vdots $ $ L_M $ $ R_M $

## 输出格式
$ 1 $ 枚だけで全てのゲ`トを通^できる ID カ`ドの枚数を出力せよ。

## 题目大意
# 题目描述

我们现在有 $N$ 张ID卡，有 $M$ 道门。

我们有第 $L_i$张、第 $L_i+1$张、…第 $R_i$张ID卡中的一张的时候，我们可以通过第 $i$ 道门。

总共有几张卡符合“只用一张卡就能通过全部门”？

# 输入格式

第1行有两个以空格隔开的数：$N$ 和 $M$

第2～M+1行有两个以空格隔开的数：$L_i$ 和 $R_i$

# 输出格式

输出一个整数，为符合“只用一张卡就能通过全部门”这样的ID卡数量

## 数据范围

- 所有输入都是整数
- $1\le N\le 10^5$
- $1\le M\le 10^5$
- $1\le L_i\le R_i\le N$

## 样例解释

### 样例1、样例4（相同的两个样例）

有2张ID卡满足“只用一张卡就能通过全部门”，如下：

- 第1张ID卡不能让我们通过第2道门
- 第2张ID卡能让我们通过所有的门
- 第3张ID卡能让我们通过所有的门
- 第4张ID卡不能让我们通过第1道门

所以总共满足“只用一张卡就能通过全部门”的ID卡数量为2张（第2、3张）

### Sample Explanation 1

以下のように、$ 1 $ 枚だけで全てのゲ`トを通^できる ID カ`ドは $ 2 $ 枚です。 - $ 1 $ 番目の ID カ`ドでは $ 2 $ 番目のゲ`トを通^できません。 - $ 2 $ 番目の ID カ`ドでは全てのゲ`トを通^できます。 - $ 3 $ 番目の ID カ`ドでは全てのゲ`トを通^できます。 - $ 4 $ 番目の ID カ`ドでは $ 1 $ 番目のゲ`トを通^できません。
