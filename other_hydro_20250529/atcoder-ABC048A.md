## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc048/tasks/abc048_a

すぬけ君は、AtCoder $ s $ Contest という名前のコンテストを_こうとしています。 ここで、$ s $ はLさ $ 1 $ 以上の文字列であり、$ 1 $ 文字目は英大文字、$ 2 $ 文字目以降は英小文字です。

すぬけ君は、このコンテストの略称を A$ x $C にQめました。 ここで、$ x $ は $ s $ の先^の英大文字です。

コンテストの名前が与えられるので、コンテストの略称を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> AtCoder $ s $ Contest

## 输出格式
コンテストの略称を出力せよ。

## 题目大意
输入三个字符串，求这三个字符串的首字母并输出。

翻译提供者：一条小鱼

```input1
AtCoder Beginner Contest
```

```output1
ABC
```

```input2
AtCoder Snuke Contest
```

```output2
ASC
```

```input3
AtCoder X Contest
```

```output3
AXC
```

## 提示
### 制s

- $ s $ のLさは $ 1 $ 以上 $ 100 $ 以下である。
- $ s $ の $ 1 $ 文字目は英大文字である。
- $ s $ の $ 2 $ 文字目以降は英小文字である。

### Sample Explanation 1

今あなたが参加しているコンテストです。

### Sample Explanation 2

このコンテストは存在しません。
