## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc057/tasks/abc057_c

整数 $ N $ が与えられます。   
 ここで、$ 2 $ つの正の整数 $ A,B $ にして、$ F(A,B) $ を「$ 10 $ M表における、$ A $ の桁数と $ B $ の桁数のうち大きい方」と定xします。   
 例えば、$ F(3,11) $ のは、$ 3 $ は $ 1 $ 桁、$ 11 $ は $ 2 $ 桁であるため、$ F(3,11)=2 $ となります。   
 $ 2 $ つの正の整数のM $ (A,B) $ が $ N=A×B $ を氦郡工瑜Δ婴とき、$ F(A,B) $ の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ 2 $ つの正の整数のM $ (A,B) $ が $ N=A×B $ を氦郡工瑜Δ婴ときの $ F(A,B) $ の最小を出力せよ。

## 题目大意

对于两个正整数A和B，将F ( A，B )定义为以下两者中较大的一个: max（A的位数，B的位数） 例如，F ( 3，11 ) = 2，因为3有一位，11有两位。

给你一个整数n 求F ( A，B )的最小值为 使得 N = A×B

输入 n

输出 min f(a,b)

感谢@chengni 提供的翻译

```input1
10000
```

```output1
3
```

```input2
1000003
```

```output2
7
```

```input3
9876543210
```

```output3
6
```

## 提示
### 制s

- $ 1QNQ10^{10} $
- $ N $ は整数である。

### Sample Explanation 1

$ (A,B)=(100,100) $ のときに $ F(A,B) $ は最小をとるため、$ F(100,100)=3 $ を出力します。

### Sample Explanation 2

条件を氦郡 $ (A,B) $ のMは $ (1,1000003) $ と $ (1000003,1) $ の $ 2 $ 通りで、$ F(1,1000003)=F(1000003,1)=7 $ です。
