## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc182/tasks/abc182_a

あなたは twiblr という SNS をしています。  
 twiblr では、フォロ`数が $ 2\,\times\,( $ フォロワ`数 $ )\,+\,100 $ を超えない欷钎榨━愆`数をやすことができます。  
 あなたのF在のフォロワ`数は $ A $ で、フォロ`数は $ B $ です。  
 フォロ`数はあといくつやせますか?

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
フォロ`数があといくつやせるかを出力せよ。

## 题目大意
输入两个数 $a,b$，输出 $2\times a+100-b$。

```input1
200 300
```

```output1
200
```

```input2
10000 0
```

```output2
20100
```

## 提示
### 制s

- $ 0\ \le\ A,\ B\ \le\ 10000 $
- $ B\ \le\ 2\ \times\ A\ +\ 100 $
- 入力は全て整数

### Sample Explanation 1

フォロ`数は $ 2\ \times\ 200\ +\ 100\ =\ 500 $ までやせるので、あと $ 200 $ やせます。
