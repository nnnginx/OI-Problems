## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc181/tasks/abc181_c

o限に冥 $ 2 $ 次元平面の上に $ N $ の点があります。

$ i $ 番目の点は $ (x_i,y_i) $ にあります。

$ N $ の点の中の相なる $ 3 $ 点であって、同一直上にあるものは存在するでしょうか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ x_1 $ $ y_1 $ $ \vdots $ $ x_N $ $ y_N $

## 输出格式
同一直上にある相なる $ 3 $ 点が存在するなら `Yes` を、存在しないなら `No` を出力せよ。

## 题目大意
平面直角坐标系上有 $N$ 个点，问是否有三个点共线。

```input1
4
0 1
0 2
0 3
1 1
```

```output1
Yes
```

```input2
14
5 5
0 1
2 5
8 0
2 1
0 0
3 6
8 6
5 9
7 9
3 4
9 2
9 8
7 2
```

```output2
No
```

```input3
9
8 2
2 3
1 3
3 7
1 0
8 8
5 6
9 7
0 1
```

```output3
Yes
```

## 提示
### 制s

- 入力はすべて整数
- $ 3\ \leq\ N\ \leq\ 10^2 $
- $ |x_i|,\ |y_i|\ \leq\ 10^3 $
- $ i\ \neq\ j $ ならば $ (x_i,\ y_i)\ \neq\ (x_j,\ y_j) $

### Sample Explanation 1

$ (0,\ 1),\ (0,\ 2),\ (0,\ 3) $ の $ 3 $ 点は直 $ x\ =\ 0 $ 上にあります。
