## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc127/tasks/arc127_a

整数 $ x $ を $ 10 $ M表したr，先^にKぶ $ 1 $ の数を $ f(x) $ で表すことにします． 例えば，$ f(1)=1,f(2)=0,f(10)=1,f(11)=2,f(101)=1 $ です．

整数 $ N $ が与えられるので，$ f(1)+f(2)+\cdots+f(N) $ のを求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $

## 输出格式
答えを出力せよ．

## 题目大意
当用十进制表示整数 $x$ 时，最高位开始连续的1的个数用 $f[x]$ 表示，例如 $f[1]=1$  ， $f[2]=0$ ，$f[10]=1$ ，$f[11]=2$ ， $f[101]=1$ 。

给出了整数N，求 $\sum_{n=1}^i f[i]$ 。

- $n \le 10^{15}$

```input1
11
```

```output1
4
```

```input2
120
```

```output2
44
```

```input3
987654321
```

```output3
123456789
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^{15} $
- 入力されるはすべて整数である

### Sample Explanation 1

$ f(2)=f(3)=\cdots\ =f(9)=0 $ です． 答えは，$ f(1)+f(10)+f(11)=4 $ です．
