## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc139/tasks/abc139_a

ある $ 3 $ 日gの天萦螭、Lさ $ 3 $ の文字列 $ S $ として与えられます。

$ S $ の $ i~(1\ \leq\ i\ \leq\ 3) $ 文字目が `S` のとき、$ i $ 日目の天萦螭晴れだったことを、`C` のときは窑辘坤盲郡长趣颉`R` のときは雨だったことを意味します。

また $ 3 $ 日gのgHの天荬、Lさ $ 3 $ の文字列 $ T $ として与えられます。

$ T $ の $ i~(1\ \leq\ i\ \leq\ 3) $ 文字目が `S` のとき、$ i $ 日目のgHの天荬晴れだったことを、`C` のときは窑辘坤盲郡长趣颉`R` のときは雨だったことを意味します。

$ 3 $ 日gで天萦螭的中した日が何日あるかを出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $ $ T $

## 输出格式
$ 3 $ 日gで天萦螭的中した日が何日あるかを出力せよ。

## 题目大意
求出两个字符串对应位上的字母不相等的个数。

```input1
CSS
CSR
```

```output1
2
```

```input2
SSR
SSR
```

```output2
3
```

```input3
RRR
SSS
```

```output3
0
```

## 提示
### 制s

- $ S $ および $ T $ はLさ $ 3 $ の文字列である。
- $ S $ および $ T $ は `S`, `C`, `R` のみからなる。

### Sample Explanation 1

\- $ 1 $ 日目の天萦螭窑辍gHの天荬窑辘胜韦恰この日の天萦螭系闹肖筏皮い搿 - $ 2 $ 日目の天萦螭锨绀臁gHの天荬锨绀欷胜韦恰この日の天萦螭系闹肖筏皮い搿 - $ 3 $ 日目の天萦螭锨绀臁gHの天荬嫌辘胜韦恰この日の天萦螭系闹肖筏皮い胜ぁ 以上より、このケ`スでは $ 3 $ 日gで天萦螭的中した日が $ 2 $ 日あります。
