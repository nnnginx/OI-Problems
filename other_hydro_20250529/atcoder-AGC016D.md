## 题目描述
[problemUrl]: https://atcoder.jp/contests/agc016/tasks/agc016_d

Lさ $ N $ の数列 $ a\ =\ (a_1,\ a_2,\ ...,\ a_N) $ があります。 ただし、各 $ a_i $ は $ 0 $ 以上の整数です。

すぬけ君は次の操作をRり返し行うことができます。

- $ a $ のすべての要素の XOR を $ x $ とする。 整数 $ i $ ($ 1\ <\ =\ i\ <\ =\ N $) をひとつxび、$ a_i $ を $ x $ に置きQえる。

すぬけ君の目摔稀$ a $ を数列 $ b\ =\ (b_1,\ b_2,\ ...,\ b_N) $ に一致させることです。 ただし、各 $ b_i $ は $ 0 $ 以上の整数です。

目摔_成可能か判定し、_成可能ならば必要な操作回数の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ a_1 $ $ a_2 $ $ ... $ $ a_N $ $ b_1 $ $ b_2 $ $ ... $ $ b_N $

## 输出格式
目摔_成可能ならば、必要な操作回数の最小を出力せよ。 _成不可能ならば、代わりに `-1` を出力せよ。

## 题目大意
题意:一个序列，一次操作可以将某个位置变成整个序列的异或和。 问最少几步到达目标序列。

```input1
3
0 1 2
3 1 0
```

```output1
2
```

```input2
3
0 1 2
0 1 2
```

```output2
0
```

```input3
2
1 1
0 0
```

```output3
-1
```

```input4
4
0 1 2 3
1 0 3 2
```

```output4
5
```

## 提示
### 制s

- $ 2\ <\ =\ N\ <\ =\ 10^5 $
- $ a_i $, $ b_i $ は整数である。
- $ 0\ <\ =\ a_i,\ b_i\ <\ 2^{30} $

### Sample Explanation 1

最初、$ a $ のすべての要素の XOR は $ 3 $ です。 $ a_1 $ をxんで $ 3 $ に置きQえると、$ a\ =\ (3,\ 1,\ 2) $ となります。 次に、$ a $ のすべての要素の XOR は $ 0 $ です。 $ a_3 $ をxんで $ 0 $ に置きQえると、$ a\ =\ (3,\ 1,\ 0) $ となり、$ b $ に一致します。
