<p>Daniel is a chess player. At his free time, he usually plays chess with his family or his friends. But, sometimes they have their own activities, so Daniel can't play chess. He spends his time to learn about knight's move. He has 1,000 x 1,000 chessboard, numbered from 1 to 1,000. He wants to move his knight from (a,b)  to (1,1) with minimum movement. Help Daniel to solve it.</p>
<h3>Input</h3>
<p>The input file consists of several lines.</p>
<p>The first line contains a single number t representing the number of question Daniel asked. [ 1&lt;=t&lt;=1,000 ]</p>
<p>The next t lines contains a and b representing knight's position on the chessboard. [ 1&lt;= a,b &lt;=  1,000 ]</p>
<h3>Output</h3>
<p>The output file should contains t lines.</p>
<p>The i-th line should contain one number ¨C the minimum number of knight's movement.</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
2<br>3 5<br>7 4<br><br><strong>Output:</strong>
2<br>3<br>&nbsp;</pre>
