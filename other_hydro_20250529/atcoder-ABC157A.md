## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc157/tasks/abc157_a

高蚓は、全 $ N $ ペ`ジから成るをI面印刷します。I面印刷では、$ 1 $ 枚のに $ 2 $ ペ`ジ分のデ`タを印刷することが出来ます。

最小で何枚のが必要か求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
答えを出力せよ。

## 题目大意
## 题目描述
高桥想要打印一有 $N$ 面的文档，因为他用的是双面打印机，所以一张纸可以打印 $2$ 面，求他需要几张纸？

## 输入格式
仅一个整数，表示文档的面数 $N$。

## 输出格式
仅一个整数，表示高桥需要几张纸。

## 说明提示
$N$ 为整数，

$1\leq N \leq100$。

```input1
5
```

```output1
3
```

```input2
2
```

```output2
1
```

```input3
100
```

```output3
50
```

## 提示
### 制s

- $ N $ は整数
- $ 1\ \leq\ N\ \leq\ 100 $

### Sample Explanation 1

$ 1 $ 枚目のに $ 1,\ 2 $ ペ`ジ目のデ`タを印刷し、 $ 2 $ 枚目のに $ 3,\ 4 $ ペ`ジ目のデ`タを印刷し、 $ 3 $ 枚目のに $ 5 $ ペ`ジ目のデ`タを印刷すれば、 $ 3 $枚のに全てのデ`タを印刷することが出来ます。
