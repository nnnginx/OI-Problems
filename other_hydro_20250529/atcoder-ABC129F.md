## 题目大意
有一等差数列，首项为 $A$，公差为 $B$，长度为 $L$。

现在把这个数列中所有数拼起来（e.g. $3,7,11,15,19->37111519$）。

问拼起来的这个数 $\bmod M$ 的值。

$1\le L,A,B \le 10^{18},1\le M\le 10^9$

```input1
5 3 4 10007
```

```output1
5563
```

```input2
4 8 1 1000000
```

```output2
891011
```

```input3
107 10000000000007 1000000000000007 998244353
```

```output3
39122908
```
