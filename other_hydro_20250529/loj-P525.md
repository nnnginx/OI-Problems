## 题目描述

给定一个正整数 $k$，你需要寻找一个系数均为 $0$ 到 $k-1$ 之间的非零多项式 $f(x)$，满足对于任意整数 $x$ 均有 $f(x)  \bmod  k=0$。你给出的多项式次数不能超过 $60000$，且最高次系数必须非 $0$。

## 输入格式

输入一行，包含一个正整数 $k$。

## 输出格式

若无解，则只输出一个整数 $-1$。否则首先输出一个整数 $n$ 表示你寻找的多项式的次数，随后 $n+1$ 个整数按照从低位到高位的顺序输出多项式的系数。

在此之后的输出将被忽略。

```input1
3
```

```output1
4
0 1 2 2 1
```

## 数据范围与提示

$1 \leq k \leq 30000$
