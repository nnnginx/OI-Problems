## 题目描述
[problemUrl]: https://atcoder.jp/contests/dp/tasks/dp_i

$ N $ を正の奇数とします。

$ N $ 枚のコインがあります。 コインには $ 1,\ 2,\ \ldots,\ N $ と番号が振られています。 各 $ i $ ($ 1\ \leq\ i\ \leq\ N $) について、コイン $ i $ を投げると、_率 $ p_i $ で表が出て、_率 $ 1\ -\ p_i $ でYが出ます。

太郎君は $ N $ 枚のコインをすべて投げました。 このとき、表の数がYの数を上回る_率を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ p_1 $ $ p_2 $ $ \ldots $ $ p_N $

## 输出格式
表の数がYの数を上回る_率を出力せよ。 ~`差が $ 10^{-9} $ 以下ならば正解となる。

## 题目大意
$N$ 枚硬币，第 $i$ 枚硬币有 $p_i$ 的概率正面朝上，有 $1-p_i$ 的概率反面朝上。  
扔完所有硬币，求**正面朝上的银币数比反面朝上的银币数多**的概率。

```input1
3
0.30 0.60 0.80
```

```output1
0.612
```

```input2
1
0.50
```

```output2
0.5
```

```input3
5
0.42 0.01 0.42 0.99 0.42
```

```output3
0.3821815872
```

## 提示
### 制s

- $ N $ は奇数である。
- $ 1\ \leq\ N\ \leq\ 2999 $
- $ p_i $ はg数であり、小数第 $ 2 $ 位まで与えられる。
- $ 0\ <\ p_i\ <\ 1 $

### Sample Explanation 1

表の数がYの数を上回るような各ケ`スの_率を算すると、次のようになります。 - $ (コイン\ 1,\ コイン\ 2,\ コイン\ 3)\ =\ (表,\ 表,\ 表) $ となる_率は、$ 0.3\ ×\ 0.6\ ×\ 0.8\ =\ 0.144 $ である。 - $ (コイン\ 1,\ コイン\ 2,\ コイン\ 3)\ =\ (Y,\ 表,\ 表) $ となる_率は、$ 0.7\ ×\ 0.6\ ×\ 0.8\ =\ 0.336 $ である。 - $ (コイン\ 1,\ コイン\ 2,\ コイン\ 3)\ =\ (表,\ Y,\ 表) $ となる_率は、$ 0.3\ ×\ 0.4\ ×\ 0.8\ =\ 0.096 $ である。 - $ (コイン\ 1,\ コイン\ 2,\ コイン\ 3)\ =\ (表,\ 表,\ Y) $ となる_率は、$ 0.3\ ×\ 0.6\ ×\ 0.2\ =\ 0.036 $ である。 よって、表の数がYの数を上回る_率は、$ 0.144\ +\ 0.336\ +\ 0.096\ +\ 0.036\ =\ 0.612 $ です。

### Sample Explanation 2

例えば、`0.500`, `0.500000001`, `0.499999999` などを出力しても正解となります。
