## 题目描述
给定三维空间的 $n$ 个点，输出 $\min\{\operatorname{dist}(i,j) | i\neq j \}$。

其中 $\operatorname{dist}(i,j)=(x_i-x_j)^2+(y_i-y_j)^2+(z_i-z_j)^2$。

## 输入格式
第一行一个正整数 $n$。

接下来 $n$ 行，每行三个正整数 $x_i,y_i,z_i$，表示每一个点的坐标。

## 输出格式
一行，一个数，表示最小的距离。
## 样例输入
```plain
2
1 1 1
1 1 1
```
## 样例输出
```plain
0
```
## 数据规模与约定
对于 $100\%$ 的数据，$1\leq x_i,y_i,z_i\leq 10^7$，$1\leq n\leq 10^6$。
