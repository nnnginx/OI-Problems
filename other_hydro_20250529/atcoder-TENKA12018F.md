## 题目描述
[problemUrl]: https://atcoder.jp/contests/tenka1-2018/tasks/tenka1_2018_f

整数 $ N $ からなる列 $ A_1,A_2,...,A_N $ が与えられます。

$ 1,2,...,N $ のKび替え $ p_1,p_2,...,p_N $ であって、 次の操作を何度か行うことでこの列を $ A_1,A_2,...,A_N $ にQできるようなものの数を $ 998244353 $ で割ったあまりを求めてください。

- 各 $ 1\leq\ i\leq\ N $ にし、$ q_i=min(p_{i-1},p_{i}) $ とする。ただし、$ p_0=p_N $ とする。列 $ p $ を列 $ q $ で置きQえる。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ : $ $ A_N $

## 输出格式
条件を氦郡沽肖数を $ 998244353 $ で割ったあまりを出力せよ。

```input1
3
1
2
1
```

```output1
2
```

```input2
5
3
1
4
1
5
```

```output2
0
```

```input3
8
4
4
4
1
1
1
2
2
```

```output3
24
```

```input4
6
1
1
6
2
2
2
```

```output4
0
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 3\ ×\ 10^5 $
- $ 1\ \leq\ A_i\ \leq\ N $
- 入力はすべて整数である
