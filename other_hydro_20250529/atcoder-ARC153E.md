## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc153/tasks/arc153_e

どの桁も $ 0 $ ではないような正整数 $ X $ にして，次の手により正整数 $ Y $ を得ることを考えます：

- 文字列 $ S $ を空文字列で初期化する．
- $ X $ の桁数を $ N $ とするとき，$ i\ =\ 1,\ \ldots,\ N $ のに次を行う：$ X $ の $ 10 $ M法表の $ i $ 文字目を，$ S $ の先^または末尾に啡毪工耄
- 文字列 $ S $ が表す正整数を $ Y $ とする．
 
この手により $ X $ から得ることが可能な正整数のうちで，最小のものを $ f(X) $ とくことにします．

- - - - - -

どの桁も $ 0 $ ではないような正整数 $ Y $ が与えられます．どの桁も $ 0 $ ではないような正整数 $ X $ であって $ f(X)\ =\ Y $ を氦郡工猡韦数を $ 998244353 $ で割った余りを答えてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられます．

> $ Y $

## 输出格式
どの桁も $ 0 $ ではないような正整数 $ X $ であって $ f(X)\ =\ Y $ を氦郡工猡韦数を $ 998244353 $ で割った余りを出力してください．

## 题目大意
对于一个各位数字均 $\neq 0$ 的正整数 $X$，定义 $f(X)$ 为如下过程所能得到的最小的 $Y$：

- 对于初始为空的字符串 $S$，依次将 $X$ 的十进制表示从左到右的每一位插入 $S$ 的最前端或最后端。设 $Y$ 为 $S$ 表示的正整数。

给出 $Y$，问有多少个 $X$ 满足 $f(X) = Y$。答案对 $998244353$ 取模。

```input1
1332
```

```output1
3
```

```input2
3312
```

```output2
0
```

```input3
12234433442
```

```output3
153
```
