## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc145/tasks/abc145_d

二次元グリッドの原点 $ (0,0) $ にチェスのナイトのxがあります。

ナイトのxはマス $ (i,j) $ にあるとき $ (i+1,j+2) $ か $ (i+2,\ j+1) $ のどちらかのマスにのみ婴すことができます。

ナイトのxをマス $ (X,Y) $ まで移婴丹护敕椒à虾瓮à辘りますか？

$ 10^9+7 $ で割った余りを求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ Y $

## 输出格式
ナイトのxを $ (0,0) $ から $ (X,Y) $ まで移婴丹护敕椒à问を、$ 10^9+7 $ で割った余りを出力せよ。

## 题目大意
一个二维矩阵的原点（0，0）有一个骑士棋子。

当骑士位于这个矩阵的（i，j）时，可以将其移动到（i+1，j+2）或（i+2，j+1）处。

骑士可以通过几种方式达到位置（x，y）？

输出答案对1000000007取模

```input1
3 3
```

```output1
2
```

```input2
2 2
```

```output2
0
```

```input3
999999 999999
```

```output3
151840682
```

## 提示
### 制s

- $ 1\ \leq\ X\ \leq\ 10^6 $
- $ 1\ \leq\ Y\ \leq\ 10^6 $
- 入力中のすべてのは整数である。

### Sample Explanation 1

$ (0,0)\ \to\ (1,2)\ \to\ (3,3) $ と $ (0,0)\ \to\ (2,1)\ \to\ (3,3) $ の $ 2 $ 通りが考えられます。

### Sample Explanation 2

$ (2,2) $ にナイトのxを移婴丹护毪长趣悉扦ません。

### Sample Explanation 3

方法の数を $ 10^9+7 $ で割った余りを出力してください。
