$ 1 $ から $ N $ までの番号のついた $ N $ のランプと，$ 1 $ から $ N $ までの番号のついた $ N $ のボタンがあります． 最初，ランプ $ 1,2,\cdots,A $ は点灯しており，それ以外のランプは点灯していません．

すぬけくんとりんごさんは，次のようなゲ`ムを行うことにしました．

- まず，りんごさんが $ (1,2,\cdots,N) $ の列 $ (p_1,p_2,\cdots,p_N) $ を生成する． この列は $ N! $ 通りの中から一ランダムにxばれる． すぬけくんは，この列を知らされない．
- 次に，すぬけくんは，以下の操作を好きなだけRり返す．
  
  
  - F在点灯しているランプを $ 1 $ つxぶ（そのようなランプがない龊悉喜僮鳏蛐肖à胜ぃ． xんだランプの番号を $ i $ とする． そして，ボタン $ i $ を押す． すると，ランプ $ p_i $ の状Bが反する．つまり，ランプ $ p_i $ がもともと点灯しているなら消灯し，もともと点灯していないなら点灯する．

すぬけくんは，常に，どのランプが点灯しているかを知ることができます． すぬけくんの倮条件は，すべてのランプを点灯した状Bにすることです． これが不可能と判明したr点ですぬけくんはけをJめます． すぬけくんが最mに行婴工毪趣，俾胜悉いらでしょうか？

すぬけくんの俾胜 $ w $ としたr，$ w\ \times\ N! $ は整数になります． $ w\ \times\ N! $ を $ 10^9+7 $ で割ったあまりを求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ N $ $ A $

## 输出格式
すぬけくんの俾胜 $ w $ としたr，$ w\ \times\ N! $ を $ 10^9+7 $ で割ったあまりを出力せよ．

## 题目大意
有 $N$ 盏灯与 $N$ 个开关，分别从 $1$ 到 $N$ 标号。起初，前面连续的 $A$ 盏灯是被点亮的，而后面的灯是被熄灭的。

Snuke 和 Ringo 将要玩下面这个游戏：

+ 首先，Ringo 生成一个 $1$ 到 $N$ 的排列 $(p_1,p_2,...,p_N)$。对于所有 $N!$ 种可能的排列，每一种的概率都是等价的，而 Snuke **并不知道**这个排列是什么。

+ 然后，Snuke 可以做任意次如下操作：
	+ 选择一盏已经被点亮了的灯。（如果没有，那么这个操作将不可能被执行。）假设选取了第 $i$ 盏灯，按下第 $i$ 盏灯对应的按钮，那么第 $p_i$ 盏灯的开关状态将会被改变。也就是说，如果第 $p_i$ 灯是被点亮的，那么操作后它将被熄灭，反之亦然。

每一时刻，Snuke 都可以知道每一盏灯的开关情况。如果最终所有的灯都被点亮了，那么 Snuke 取得游戏胜利；而如果事实证明他无法取得胜利，他就会认输。如果 Snuke 采取最佳策略，那么他胜利的概率是多少呢？

假设 $w$ 为胜利的概率，那么请输出 $w\times N!$ 在模 $10^9+7$ 意义下的值。

```input1
3 1
```

```output1
2
```

```input2

```

```output2
3
```

```input3

```

```output3

```

```input4

```

```output4

```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 10^7 $
- $ 1\ \leq\ A\ \leq\ \min(N-1,5000) $

### Sample Explanation 1

すぬけくんはまず，ボタン $ 1 $ を押します． ランプ $ 1 $ が消灯した龊悉悉工踏堡んのけです． そうでないときは，新しく点灯したランプに辘工毳堀骏螭蜓氦筏蓼梗 ここで残っていたランプが点灯すれば，すぬけくんの伽沥扦梗 逆に，ランプ $ 1 $ が消灯した龊希すぬけくんのけです． このゲ`ムの俾胜 $ 1/3 $ なので，$ (1/3)\times\ 3!=2 $ を出力します．
