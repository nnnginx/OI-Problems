## 题目描述

Hecy 又接了个新任务：BE 处理。BE 中有一类被称为 GBE。

以下是 GBE 的定义：
1. 空表达式是 GBE
2. 如果表达式 `A` 是 GBE，则 `[A]` 与 `(A)` 都是 GBE
3. 如果 `A` 与 `B` 都是 GBE，那么 `AB` 是 GBE

下面给出一个 BE，求至少添加多少字符能使这个 BE 成为 GBE。

## 输入格式

输入仅一行，为字符串 BE。

## 输出格式

输出仅一个整数，表示增加的最少字符数。

```input1
[])
```

```output1
1
```

## 数据范围与提示

对于 $100\%$ 的数据，输入的字符串长度小于 $100$。
