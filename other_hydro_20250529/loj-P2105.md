## 题目描述

为了提高智商，ZJY 开始学习概率论。有一天，她想到了这样一个问题：

对于一棵随机生成的 $n$ 个结点的有根二叉树（所有互相不同构的形态等概率出现），它的叶子节点数的期望是多少呢？

## 输入格式

输入一个正整数 $N$，代表有根树的结点数

## 输出格式

输出这棵树期望的叶子节点数。要求误差小于 $10^{-9}$

```input1
1
```

```output1
1.000000000
```

## 数据范围与提示

$1 \leq N \leq 10^9$
