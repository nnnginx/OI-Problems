## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc083/tasks/abc083_b

$ 1 $ 以上 $ N $ 以下の整数のうち、$ 10 $ M法での各桁の和が $ A $ 以上 $ B $ 以下であるもののt和を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A $ $ B $

## 输出格式
$ 1 $ 以上 $ N $ 以下の整数のうち、$ 10 $ M法での各桁の和が $ A $ 以上 $ B $ 以下であるもののt和を出力せよ。

## 题目大意
## 题目描述 Description
在 $1$ 以上 $N$ 以下的整数中，求所有 $10$ 进制中各位数的和在 $A$ 和 $B$ 之间的数的总和。

## 输入格式 Input
标准输入仅 $1$ 行，包括三个整数 $N,A,B$ ，当中以空格分开。

## 输出格式 Output
符合条件的数的和，**注意换行**。

## 样例 $1$ 说明 Explanation
$2+3+4+5+11+12+13+14+20=84$ 。

```input1
20 2 5
```

```output1
84
```

```input2
10 1 2
```

```output2
13
```

```input3
100 4 16
```

```output3
4554
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^4 $
- $ 1\ \leq\ A\ \leq\ B\ \leq\ 36 $
- 入力はすべて整数である

### Sample Explanation 1

$ 20 $ 以下の整数のうち、各桁の和が $ 2 $ 以上 $ 5 $ 以下なのは $ 2,3,4,5,11,12,13,14,20 $ です。これらの合である $ 84 $ を出力します。
