## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc292/tasks/abc292_b

$ 1 $ から $ N $ までの番号がついた $ N $ 人のx手がサッカ`の合をします。  
x手が反tを犯したとき、そのx手には **イエロ`カ`ド** と **レッドカ`ド** のどちらかが提示されます。  
以下の条件のうち一方を氦郡筏窟x手は **退I分** と呼ばれるペナルティを受けます。

- イエロ`カ`ドを累 2 回提示される。
- レッドカ`ドを提示される。

なお、退I分を受けたx手にそれ以降カ`ドが提示されることはありません。

あなたはこの合をQ椁筏蓼埂￥悉袱帷すべてのx手はカ`ドを 1 回も提示されていません。  
$ Q $ のイベントがk生するので、イベントでかれる|に正しく答えてください。  
イベントは 3 Nあり、`c x` ($ c $ は $ 1,\ 2,\ 3 $ のいずれか) という形式で入力から与えられます。イベントのh明は次の通りです。

- `1 x` : x手 $ x $ にイエロ`カ`ドが提示される。
- `2 x` : x手 $ x $ にレッドカ`ドが提示される。
- `3 x` : あなたはx手 $ x $ が退I分を受けたかを|される。x手 $ x $ が退I分を受けていれば `Yes` と、そうでなければ `No` と答える。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。ただし、$ \text{event}_i $ は $ i $ 番目にk生するイベントを意味する。

> $ N $ $ Q $ $ \text{event}_1 $ $ \text{event}_2 $ $ \vdots $ $ \text{event}_Q $

イベントは次の 3 つのいずれかの形式で入力される。

> 1 $ x $

> 2 $ x $

> 3 $ x $

## 输出格式
入力で与えられる $ 3 $ N目のイベントの数を $ X $ として、$ X $ 行出力せよ。  
$ i $ 行目には、$ 3 $ N目のイベントのうち $ i $ 番目のものでかれる|について、x手 $ x $ が退I分を受けていれば `Yes` を、そうでなければ `No` を出力せよ。

## 题目大意
$N$ 名选手进行足球比赛。选手犯规时会被出示黄牌或红牌，当累计被出示两次黄牌或被出示红牌是选手会被罚下场。

比赛时会发生 $Q$ 个事件，事件有三种：

1. 输入 $1$ $x$，表示选手 $x$ 被出示黄牌。
1. 输入 $2$ $x$，表示选手 $x$ 被出示红牌。
1. 输入 $3$ $x$，询问选手 $x$ 是否被罚下场。若被罚下场，输出 `Yes`，否则输出 `No`。

```input1
3 9
3 1
3 2
1 2
2 1
3 1
3 2
1 2
3 2
3 3
```

```output1
No
No
Yes
No
Yes
No
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 100 $
- $ 1\ \leq\ Q\ \leq\ 100 $
- 全てのイベントにおいて $ 1\ \leq\ x\ \leq\ N $
- $ 3 $ N目のイベントは少なくとも $ 1 $ 以上存在する
- すでに退I分を受けたx手にカ`ドが提示されることはない
- 入力されるはすべて整数
