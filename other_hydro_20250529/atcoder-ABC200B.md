## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc200/tasks/abc200_b

整数 $ N $ が与えられます。  
 以下の操作を $ K $ 回行った後の整数 $ N $ を出力してください。

- 整数 $ N $ が $ 200 $ の倍数であれば、$ N $ を $ 200 $ で割る。
- そうでなければ、整数 $ N $ を、$ N $ の後ろに文字列として $ 200 $ を付け加えた整数に置きQえる。
  - 例えば、$ 7 $ を置きQえると $ 7200 $ に、$ 1234 $ を置きQえると $ 1234200 $ になります。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
答えを整数として出力せよ。

## 题目大意
给定 $ N $, $ K $，做 $ K $ 次操作：

若 $ N $ 为 $ 200 $ 的倍数，则将 $ N $ 除以 $ 200 $。

否则，将 $ N $ 的末尾添上 $ 200 $（即 $ N \times 1000 + 200 $）。

```input1
2021 4
```

```output1
50531
```

```input2
40000 2
```

```output2
1
```

```input3
8691 20
```

```output3
84875488281
```

## 提示
### 制s

- 入力は全て整数
- $ 1\ \le\ N\ \le\ 10^5 $
- $ 1\ \le\ K\ \le\ 20 $

### Sample Explanation 1

$ N=2021 $ に $ 4 $ 回操作を行うと、$ N $ のは操作を行うごとに $ 2021\ \rightarrow\ 2021200\ \rightarrow\ 10106\ \rightarrow\ 10106200\ \rightarrow\ 50531 $ となります。

### Sample Explanation 3

答えは $ 32 $ bit 符号付き整数型にГ蓼椁胜た赡苄预あります。
