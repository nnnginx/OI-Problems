## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc059/tasks/abc059_b

$ 2 $ つの正整数 $ A,\ B $ が与えられるので、その大小を比^してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $

## 输出格式
$ A\ >\ B $ のとき`GREATER`、$ A\ <\ B $ のとき`LESS`、$ A=B $ のとき`EQUAL`と出力せよ。

## 题目大意
### 题目描述

给定两个正整数 $a, b$，比较他们的大小。

### 输入格式

输入有两行，第一行为数 $a$，第二行为数 $b$。

### 输出格式

如果 $a > b$，输出 "GREATER"；如果 $a = b$，输出 "EQUAL"，如果 $a < b$，输出 "LESS"。

### 数据范围

$1 \le a, b \le 10^{100}$，保证 $a, b$ 均无前导零。

```input1
36
24
```

```output1
GREATER
```

```input2
850
3777
```

```output2
LESS
```

```input3
9720246
22516266
```

```output3
LESS
```

```input4
123456789012345678901234567890
234567890123456789012345678901
```

```output4
LESS
```

## 提示
### 制s

- $ 1QA,\ B\ Q\ 10^{100} $
- 入力の $ A,\ B $ の先^は`0`でない。

### Sample Explanation 1

$ 36\ >\ 24 $ なので、答えは`GREATER`です。
