## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc090/tasks/arc091_a

k横にo限に冥るマス目があり、そのうちのBAする $ N $ 行 $ M $ 列のI域のすべてのマスに表Yの区eできるカ`ドが置かれています。 最初はすべてのカ`ドが表を向いています。

以下の操作を、カ`ドが置かれている全てのマスについて $ 1 $ 度ずつ行います。

- そのマスとxまたは点で接する $ 8 $ つのマスと、そのマスの合 $ 9 $ マスについて、カ`ドが存在するならY返す。

すべての操作を行った後の各カ`ドの状Bは操作を行う番に依らないことが^明できます。 すべての操作を行った後、Yを向いているカ`ドの枚数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ M $

## 输出格式
すべての操作を行った後、Yを向いているカ`ドの枚数を出力せよ。

## 题目大意
给定一个一个 $n$ 行， $m$ 列的矩形，每个格子中有一张正面朝上的卡片

现在对于其中的每一个格子，都将其自身以及周围的 $8$ 个格子（如果存在的话）中的卡片翻转

求问所有操作结束后有多少张卡片正面朝下

```input1
2 2
```

```output1
0
```

```input2
1 7
```

```output2
5
```

```input3
314 1592
```

```output3
496080
```

## 提示
### 制s

- $ 1\ \leq\ N,M\ \leq\ 10^9 $
- 入力は全て整数である

### Sample Explanation 1

$ 4 $ 回の操作のうちのどの操作でも、すべてのカ`ドをY返します。よって、すべての操作を行った後は、すべてのカ`ドが表を向いています。

### Sample Explanation 2

すべての操作を行った後は、I端以外のカ`ドがYを向いています。
