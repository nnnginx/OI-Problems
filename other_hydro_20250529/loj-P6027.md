## 题目描述

求满足$1< p \leq n$ 且 $p$ 的二进制表示最后两位为 $01$ 的质数 $p$ 有多少个。

## 输入格式

一行一个整数 $n$。

## 输出格式

一行一个整数 $\pi$ 表示答案。

```input1
20
```

```output1
3
```

```input2
100000
```

```output2
4783
```

## 数据范围与提示

对于 $30\%$ 的数据，$1\leq n\leq 10^4$

对于 $50\%$ 的数据，$1\leq n\leq 10^7$

对于 $80\%$ 的数据，$1\leq n\leq 10^{10}$

对于 $100\%$ 的数据，$1\leq n\leq 3\times 10^{10}$
