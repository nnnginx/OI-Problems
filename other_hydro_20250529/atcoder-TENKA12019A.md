## 题目描述
[problemUrl]: https://atcoder.jp/contests/tenka1-2019-beginner/tasks/tenka1_2019_a

数直上に家 $ 1,2,3 $ があり、それぞれのある座摔 $ A,B,C $ です。 家 $ 1 $ から家 $ 2 $ まで寄り道をせずにまっすぐ向かう途中で家 $ 3 $ のある座摔蛲à龊悉 `Yes` を、そうでない龊悉 `No` を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $

## 输出格式
家 $ 1 $ から家 $ 2 $ まで寄り道をせずにまっすぐ向かう途中で家 $ 3 $ のある座摔蛲à龊悉 `Yes` を、そうでない龊悉 `No` を出力せよ。

## 题目大意
数字线上有 $3$ 个房屋：房屋 $1$，$2$ 和 $3$，坐标分别为 $A$，$B$ 和 $C$。 如果在不绕行的情况下从房屋 $1$ 到房屋 $2$ 的直线路径上传递了房屋 $3$ 的坐标，则打印 `Yes`，否则打印 `No`。

```input1
3 8 5
```

```output1
Yes
```

```input2
7 3 1
```

```output2
No
```

```input3
10 2 4
```

```output3
Yes
```

```input4
31 41 59
```

```output4
No
```

## 提示
### 制s

- $ 0\leq\ A,B,C\leq\ 100 $
- $ A,B,C $ は相なる整数である

### Sample Explanation 1

座 $ 3 $ にある家から座 $ 8 $ にある家までまっすぐ向かう途中には、座 $ 5 $ を通ります。
