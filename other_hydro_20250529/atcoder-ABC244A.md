## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc244/tasks/abc244_a

英小文字からなるLさ $ N $ の文字列 $ S $ が与えられます。$ S $ の末尾の文字を出力してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
$ S $ の末尾の文字を出力せよ。

## 题目大意
输出长度为 $n$ 的字符串 $s$ 的最后一位。

```input1
5
abcde
```

```output1
e
```

```input2
1
a
```

```output2
a
```

## 提示
### 制s

- $ N $ は整数
- $ 1\ <\ =\ N\ <\ =\ 1000 $
- $ S $ は英小文字からなるLさ $ N $ の文字列

### Sample Explanation 1

$ S\ =\ {} $`abcde` です。$ S $ の末尾の文字は `e` なので、`e` を出力します。
