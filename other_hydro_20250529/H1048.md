## 题目描述

上次的密码十分轻易被破解了，导致战略情报被敌方获知。于是ことり决定重新设置一个更强的密码。新密码也需要测试，找不到おりがみ的ことり将目光投向了你……

密码形同下面这个问题:

对于给定的 $n$ 求有多少有序数对 $(a,b,c)$ 满足 $1\le a,b,c\leq n (a,b,c\in N^*)$ 且 $a,b,c$ 构成一个等比数列。

即 $b^2\!=\!ac$。

## 输入格式

输入共一行一个数，表示 $n$。

## 输出格式

输出共一行一个数，表示方案数。


```input1
10
```

```output1
18
```

```input2

```

```output2

```
