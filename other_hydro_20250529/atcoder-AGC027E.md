## 问题陈述

有一个由 `a` 和 `b` 组成的字符串 $s$。  
Snuke 可以进行以下两种操作，任意次数地按任意顺序执行：

- 选择一个 `aa` 的出现作为子串，并将其替换为 `b`。
- 选择一个 `bb` 的出现作为子串，并将其替换为 `a`。

通过这一系列操作可以得到多少个字符串 $s$？  
找到结果对 $10^9 + 7$ 的模。

## 约束条件

- $1 \leq |s| \leq 10^5$
- $s$ 由 `a` 和 `b` 组成。

## 输入

输入格式如下所示，从标准输入中给出：

> $s$

## 输出

打印可以获得的字符串数量，结果对 $10^9 + 7$ 取模。

```input1
aaaa
```

```output1
6
```

可以获得六个字符串：

- `aaaa`
- `aab`
- `aba`
- `baa`
- `bb`
- `a`

```input2
aabb
```

```output2
5
```

可以获得五个字符串：

- `aabb`
- `aaa`
- `bbb`
- `ab`
- `ba`

```input3
ababababa
```

```output3
1
```

Snuke 不能进行任何操作。

```input4
babbabaaba
```

```output4
35
```
