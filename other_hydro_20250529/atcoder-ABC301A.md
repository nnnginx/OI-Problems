## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc301/tasks/abc301_a

高颏んと青木くんが $ N $ 回の合を行いました。 これらの合のY果を表すLさ $ N $ の文字列 $ S $ が与えられます。 $ i $ 回目の合の僬撙稀$ S $ の $ i $ 文字目が `T` ならば高颏ん、`A` ならば青木くんです。

高颏んと青木くんのうち、伽盲吭合の数が多い方をt合僬撙趣筏蓼埂 ただし、伽盲吭合の数が同じである龊悉稀⑾趣摔饯伽潦に_した者をt合僬撙趣筏蓼埂 高颏んと青木くんのどちらがt合僬撙扦るか求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
t合僬撙高颏んならば `T` を、青木くんならば `A` を出力せよ。

## 题目大意
### 题目描述
两个人比赛，用字符串给出比赛的情况：字符 `A` 表示 A 赢了，字符 `T` 表示 T 赢了，最终的胜者是胜场更多的那一方，如果相同，则先达到总胜场的一方获胜。输出胜方。
### 输入格式
第一行一个正整数 $N$，表示字符串的长度。

第二行一个长度为 $N$ 的字符串。
### 输出格式
共一行一个字符，表示胜方。

```input1
5
TTAAT
```

```output1
T
```

```input2
6
ATTATA
```

```output2
T
```

```input3
1
A
```

```output3
A
```

## 提示
### 制s

- $ 1\leq\ N\ \leq\ 100 $
- $ N $ は整数
- $ S $ は `T` および `A` からなるLさ $ N $ の文字列
 
### Sample Explanation 1

高颏んは $ 3 $ 回の合に伽痢⑶嗄兢んは $ 2 $ 回の合に伽沥蓼筏俊 よって、伽盲吭合の数が多い高颏んがt合僬撙扦埂

### Sample Explanation 2

高颏んと青木くんのどちらも $ 3 $ 回の合に伽沥蓼筏俊 また、高颏んは $ 5 $ 回目の合で $ 3 $ 倌郡诉_し、青木くんは $ 6 $ 回目の合で $ 3 $ 倌郡诉_しました。 よって、先に $ 3 $ 倌郡诉_した高颏んがt合僬撙扦埂
