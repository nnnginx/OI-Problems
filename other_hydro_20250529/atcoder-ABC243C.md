## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc243/tasks/abc243_c

$ xy $ 座似矫嫔悉 $ N $ 人の人がいます。人 $ i $ は $ (X_i,\ Y_i) $ にいます。すべての人はなる地点にいます。

`L`, `R` からなるLさ $ N $ の文字列 $ S $ があります。  
 人 $ i $ は $ S_i\ = $ `R` ならば右向きに、$ S_i\ = $ `L` ならば左向きに、一扭送じ速度でiき始めます。ここで、右は $ x $ Sの正の向き、左は $ x $ Sのの向きです。

たとえば $ (X_1,\ Y_1)\ =\ (2,\ 3),\ (X_2,\ Y_2)\ =\ (1,\ 1),\ (X_3,\ Y_3)\ =(4,\ 1),\ S\ = $ `RRL` の龊悉洗韦恧韦瑜Δ婴ます。

![image](https://cdn.luogu.com.cn/upload/vjudge_pic/AT_abc243_c/dcf8faf208a98b36fba58ea0016822db31f879c4.png)

反の向きにiいている人同士が同じ地点に来ることを「n突」と呼びます。すべての人がiきAけたとき、n突はk生しますか？

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ X_1 $ $ Y_1 $ $ X_2 $ $ Y_2 $ $ \vdots $ $ X_N $ $ Y_N $ $ S $

## 输出格式
n突がk生するならば `Yes` を、k生しないならば `No` を出力せよ。

## 题目大意
平面直角坐标系上有 $n$ 个点，第 $i$ 个点的坐标为 $(x_i,y_i)$ 。有一个字符串 $s$ （ $s$ 的下标从 $1$ 开始），其长度也为 $n$ 。如果 $s_i$ 为`L`，则第 $i$ 个点向左移动；如果 $s_i$ 为`R`，则第 $i$ 个点向右移动。现在，依次输入 $n$ ，所有 $x_i$ 和 $y_i$ 以及 $s$ ，问：是否会有两个点重合在一起？

（注：向左移动是指沿 $x$ 轴向负方向移动，向右移动与此相反。）

```input1
3
2 3
1 1
4 1
RRL
```

```output1
Yes
```

```input2
2
1 1
2 1
RR
```

```output2
No
```

```input3
10
1 3
1 4
0 0
0 2
0 4
3 1
2 4
4 2
4 4
3 3
RLRRRLRLRR
```

```output3
Yes
```
