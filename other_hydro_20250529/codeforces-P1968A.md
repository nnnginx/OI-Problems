## Description

<div><p>You are given an integer $x$. Your task is to find any integer $y$ $(1\le y&lt;x)$ such that $\gcd(x,y)+y$ is maximum possible.</p><p><span class="tex-font-style-bf">Note that if there is more than one $y$ which satisfies the statement, you are allowed to find any.</span></p><p>$\gcd(a,b)$ is the Greatest Common Divisor of $a$ and $b$. For example, $\gcd(6,4)=2$.</p></div><div class="input-specification"><p>The first line contains a single integer $t$ ($1 \le t \le 1000$)&nbsp;¡ª the number of test cases.</p><p>Each of the following $t$ lines contains a single integer $x$ ($2 \le x \le 1000$).</p></div><div class="output-specification"><p>For each test case, output any $y$ ($1 \le y &lt; x$), which satisfies the statement.</p></div>
