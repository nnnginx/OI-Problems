<p>You are given two integers, <strong>n</strong> and <strong>m</strong>.</p>
<p>Find and print the number of sequences of length n which satisfy:</p>
<ul>
<li>&nbsp;All elements of the sequence are positive divisors of m</li>
<li>&nbsp;For any two adjacent elements, say p and q, there exists at least one prime which divides both of them.</li>
</ul>
<p>Print the number of such sequences modulo 10<sup>9</sup>+7</p>
<h3>Input</h3>
<p>The only line of input contains two integers, n and m.</p>
<h4>Constraints</h4>
<ul>
<li>0 &lt; n ¡Ü 10<sup>5</sup></li>
<li>0 &lt; m ¡Ü 10<sup>18</sup></li>
</ul>
<h3>Output</h3>
<p>Print the number of valid sequences modulo 10<sup>9</sup>+7</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
2 10

<strong>Output:</strong>
7
</pre>
