## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc098/tasks/arc098_a

$ N $ 人の人が|西方向に一列にKんでいます。 それぞれの人は、|または西を向いています。 lがどの方向を向いているかはLさ $ N $ の文字列 $ S $ によって与えられます。 西から $ i $ 番目にKんでいる人は、$ S_i\ = $ `E` なら|を、$ S_i\ = $ `W` なら西を向いています。

あなたは、$ N $ 人のうちlか $ 1 $ 人をリ`ダ`として任命します。 そして、リ`ダ`以外の全Tに、リ`ダ`の方向を向くように命令します。 このとき、リ`ダ`はどちらの方向を向いていてもいません。

Kんでいる人は、向く方向を浃à毪韦蛳婴盲皮い蓼埂 そのためあなたは、向く方向を浃à肴耸が最小になるようにリ`ダ`をxびたいです。 向く方向を浃à肴耸の最小を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ S $

## 输出格式
向く方向を浃à肴耸の最小を出力せよ。

## 题目大意
给定一个长为 $N$ 的只有 $E$ 或 $W$ 的字符串，选择任意一个点 $a_i$ ，将这个点左边不是 $E$ 的点都变为 $E$，将这个点右边不是 $W$ 的点都变为 $W$，每改一个点付出一点代价。请输出最小的代价。不需要改变 $a_i$ 的方向。  
输入有两行，第一行为一个整数 $N$ $(2 ≤ N ≤ 3 × 10^5)$，接下来一行为一个长为 $N$ 的字符串 $S$。  
输出一行，输出最小的代价。  
对于样例一，指定第三个点，将第一个点改为 $E$ ，代价为一。所以最小的代价为一。

```input1
5
WEEWW
```

```output1
1
```

```input2
12
WEWEWEEEWWWE
```

```output2
4
```

```input3
8
WWWWWEEE
```

```output3
3
```

## 提示
### 制s

- $ 2\ \leq\ N\ \leq\ 3\ \times\ 10^5 $
- $ |S|\ =\ N $
- $ S_i $ は `E` または `W` である

### Sample Explanation 1

西から $ 3 $ 番目にKんでいる人をリ`ダ`に任命するとします。 すると、西から $ 1 $ 番目にKんでいる人は|を向かなくてはならないので、向く方向を浃à氡匾があります。 ほかの人は向く方向を浃à氡匾がないので、この龊稀⑾颏方向を浃à肴摔 $ 1 $ 人になります。 向く方向を浃à肴摔 $ 0 $ 人にすることは出来ないので、答えは $ 1 $ になります。
