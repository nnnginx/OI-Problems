## 题目描述
[problemUrl]: https://atcoder.jp/contests/nikkei2019-2-qual/tasks/nikkei2019_2_qual_e

正の整数 $ N $ 及び $ K $ が与えられます。

以下の条件をみたすように、$ 3N $ の整数 $ K,K+1,...,K+3N-1 $ を $ 3 $ 整数からなる $ N $ のM $ (a_1,b_1,c_1),...,(a_N,b_N,c_N) $ に分割することが可能かを判定して下さい。$ K,K+1,...,K+3N-1 $ のうちどの整数も、$ N $ のMのうちちょうど $ 1 $ にFれなければなりません。

- $ 1 $ 以上 $ N $ 以下のすべての整数 $ i $ にして $ a_i\ +\ b_i\ \leqq\ c_i $ が成り立つ。

また、可能な龊悉悉饯韦瑜Δ史指瞍 $ 1 $ つ成してください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
条件をみたすように $ N $ の三つMに分割することができない龊 `-1` を出力せよ。可能な龊悉 $ N $ の三つMを以下の形式で出力せよ。

> $ a_1 $ $ b_1 $ $ c_1 $ $ : $ $ a_N $ $ b_N $ $ c_N $

## 题目大意
给定整数$N,K$

将$K,K+1,...,K+3N-1$

问能否这$3N$个整数划分为$N$个三元组
$(a1,b1,c1),.....(an,bn,cn)$

其中$(a + b ≤c)$

使得上述$3N$个整数必须在三元组中全部出现

若满足条件，输出这$N$个三元组

若无法构成，输出$-1$

题目

```input1
1 1
```

```output1
1 2 3
```

```input2
3 3
```

```output2
-1
```
