<p>&nbsp;</p>
<p>Gollum has spent half a millennium in the long darkness of Moria, where his eyes grew used to the dark, and without caring for reading or writing, he became dyslexic. Indeed, as much as he hates the Moon and the Sun, he also hates strings with long palindromes in them.</p>
<p>&nbsp;</p>
<p>Gollum has a tolerance level of K, which means that he can read a word so long as it does not contain any palindromic substring of length K or more. Given the values N and K, return how many BINARY strings of length N can Gollum tolerate reading.</p>
<p>&nbsp;</p>
<p><strong>Input (STDIN):</strong></p>
<p>The first line contains T, the number of test cases.</p>
<p>Each test case consists of one line containing 2 integers, N and K.</p>
<p>&nbsp;</p>
<p><strong>Output (STDOUT):</strong></p>
<p>For each test case, output the answer modulo 1,000,000,007.</p>
<p>&nbsp;</p>
<p><strong>Constraints:</strong></p>
<p>1 &lt;= T &lt;= 100</p>
<p>1 &lt;= N &lt;= 400</p>
<p>1 &lt;= K &lt;= 10</p>
<p>&nbsp;</p>
<p><strong>Sample Input:</strong></p>
<p>3</p>
<p>2 2</p>
<p>3 3</p>
<p>3 4</p>
<p>&nbsp;</p>
<p><strong>Sample Output:</strong></p>
<p>2</p>
<p>4</p>
<p>8</p>
<p>&nbsp;</p>
<p><strong>Notes/Explanation of Sample Input:</strong></p>
<p>For the first test case, 01 and 10 are the valid binary strings, while 00 and 11 are invalid.</p>
<p>For the second test case, 001, 011, 100, 110 are the valid binary strings.</p>
<p>For the third test case, all possible binary strings of length 3 are valid.</p>
