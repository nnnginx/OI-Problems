## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc071/tasks/arc081_a

太さがoできる棒が $ N $ 本あります． $ i $ 番目の棒のLさは $ A_i $ です．

すぬけ君は，これらの棒から $ 4 $ 本のなる棒をxび，それらの棒をxとしてL方形（正方形を含む）を作りたいです． 作ることができる最大のL方形の面eを求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ A_1 $ $ A_2 $ ... $ A_N $

## 输出格式
すぬけ君が作ることのできる最大のL方形の面eを出力せよ． ただし，L方形を作れない龊悉希$ 0 $ を出力せよ．

## 题目大意
有N根可以忽视粗细的棒。第i棒的长度是a_i。
有人想从这些棒子中选出4个不同的棒子，用这些棒子做个矩形(包括正方形)。求最大可以制作的矩形面积。

```input1
6
3 1 2 4 2 1
```

```output1
2
```

```input2
4
1 2 3 4
```

```output2
0
```

```input3
10
3 3 3 3 4 4 4 5 5 5
```

```output3
20
```

## 提示
### 制s

- $ 4\ \leq\ N\ \leq\ 10^5 $
- $ 1\ \leq\ A_i\ \leq\ 10^9 $
- $ A_i $ は整数

### Sample Explanation 1

$ 1\ \times\ 2 $ のL方形を作ることができます．

### Sample Explanation 2

L方形を作ることはできません．
