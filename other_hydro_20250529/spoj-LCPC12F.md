<pre><span style="white-space: normal;"><p>Johnny is a gambling addict. He entered a casino and started playing a game with the dealer. The game is as follows: the dealer deals a sequence of <strong><em>N</em></strong> cards, each card containing a number <strong><em>C[i]</em></strong> and asks Johnny how many pairs (<strong><em>j, k</em></strong>) such that <strong><em>j</em></strong> &lt; <strong><em>k</em></strong> and <strong><em>C[j] </em></strong>+ <strong><em>C[k] </em></strong>= <strong><em>X</em></strong>. <h2>Input Format&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</h2>
<p>The first line of input contains an integer <strong><em>T</em></strong>, the number of test cases. <strong><em>T</em></strong> test cases follow, Each test case start with the value of 0 ¡Ü <strong><em>X </em></strong>¡Ü 2*10<sup>3</sup> followed by the number of cards 0 &lt; <strong><em>N </em></strong>¡Ü 10<sup>5</sup> followed by <strong><em>N</em></strong> numbers representing the numbers on the cards 0 ¡Ü <strong><em>C[i] </em></strong>¡Ü 10<sup>3</sup>.</p>
<h2>Output Format</h2>
<p>There should be <strong><em>T</em></strong> lines, containing the following format.</p>
<p><strong><em>k. S</em></strong></p>
<p>Where <strong><em>k</em></strong> is the test case number (starting at 1), a single period, a single space and <strong><em>S</em></strong> representing the number of valid pairs (<strong><em>j</em></strong>, <strong><em>k</em></strong>) as described above.</p>
