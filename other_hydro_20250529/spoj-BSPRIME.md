<p>Binary Sequence of Prime Number is a binary sequence that created by converting prime number to base-2 (Without leading zeros):</p>
<p>(2)<sub>10</sub>=(10)<sub>2</sub><br>(3)<sub>10</sub>=(11)<sub>2</sub><br>(5)<sub>10</sub>=(101)<sub>2</sub><br>(7)<sub>10</sub>=(111)<sub>2</sub><br>...</p>
<p>If all base-2 of prime number joined, then we get the binary sequence like this: 10111011111011110110...</p>
<p>Now your task is to count how many digit '1' appear in the first <strong>N</strong>&nbsp;terms of the sequence, example:</p>
<p>--&gt;If N=3, digit '1' appear 2 times: <span style="color: #ff00ff;"><span style="text-decoration: underline;">1</span>0<span style="text-decoration: underline;">1</span><span style="color: #000000;">110...</span></span><br>--&gt;If N=10, digit '1' appear 8 times: <span style="color: #ff00ff;"><span style="text-decoration: underline;">1</span>0<span style="text-decoration: underline;">111</span>0<span style="text-decoration: underline;">1111</span></span>101...</p>
<h3>Input</h3>
<p>The first line is an integer&nbsp;<strong>T</strong>(1 ¡Ü&nbsp;<strong>T</strong>&nbsp;¡Ü 50,000), denoting the number of test cases. Then,&nbsp;<strong>T</strong>&nbsp;test cases follow.</p>
<p>For each test case, there is an integer&nbsp;<strong>N</strong>(0 ¡Ü&nbsp;<strong>N</strong>&nbsp;¡Ü 150,000,000) written in one line.</p>
<h3>Output</h3>
<p>For each test case, output the number of digit '1' appear in the first <strong>N</strong> terms of the sequence</p>
<h3>Example</h3>
<pre><strong>Input:</strong>
3</pre>
<pre>3</pre>
<pre>10</pre>
<pre>50

<strong>Output:</strong>
2</pre>
<pre>8</pre>
<pre>36</pre>
