## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc136/tasks/abc136_b

整数 $ N $ が与えられます。$ N $ 以下の正の整数のうち、(先^に $ 0 $ をつけずに十M法で表したときの) 桁数が奇数であるようなものの数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $

## 输出格式
$ N $ 以下の正の整数のうち、桁数が奇数であるようなものの数を出力せよ。

## 题目大意
给定正整数 $N$ ，求 $N$ 以内（含 $N$ ）有多少个**位数为奇数**的正整数。

```input1
11
```

```output1
9
```

```input2
136
```

```output2
46
```

```input3
100000
```

```output3
90909
```

## 提示
### 制s

- $ 1\ \leq\ N\ \leq\ 10^5 $

### Sample Explanation 1

$ 11 $ 以下の正の整数のうち、桁数が奇数であるようなものは $ 1,\ 2,\ \ldots,\ 9 $ の $ 9 $ です。

### Sample Explanation 2

$ 1,\ 2,\ \ldots,\ 9 $ に加えて、$ 100,\ 101,\ \ldots,\ 136 $ の $ 37 $ の数も桁数が奇数です。
