## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc273/tasks/abc273_b

非整数 $ X $ にし、 $ i=0,1,\dots,K-1 $ のに次の操作を行ったとき、操作を全てKえたr点での $ X $ を求めてください。

- $ X $ の $ 10^i $ の位以下を四挝迦毪工搿
  - 密には、 $ X $ を「 $ |Y-X| $ が最小となる $ 10^{i+1} $ の倍数のうち最大のもの」である $ Y $ に置きQえる。
  - 具体例をい菠搿
      - $ 273 $ の $ 10^1 $ の位以下を四挝迦毪工欷 $ 300 $ となる。
      - $ 999 $ の $ 10^2 $ の位以下を四挝迦毪工欷 $ 1000 $ となる。
      - $ 100 $ の $ 10^9 $ の位以下を四挝迦毪工欷 $ 0 $ となる。
      - $ 1015 $ の $ 10^0 $ の位以下を四挝迦毪工欷 $ 1020 $ となる。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ X $ $ K $

## 输出格式
答えを整数として出力せよ。

## 题目大意
已知一个数字 $X$，依次进行 $K$ 次操作。

每次对数字 $X$ 按 $10^i$ 进行做 **四舍五入** 操作。

例： $273$ 按 $10^2$ 做四舍五入为 $300$ ； $273$ 按 $10^1$ 做四舍五入为 $270$。

请你求出，$K$ 次操作后，数字 $X$ 最终变为多少。

其中 

$ 0 \leq X \leq 10^{15} $ 

$ 1 \leq K \leq 15 $ 

$ 1 \leq i \leq K $

```input1
2048 2
```

```output1
2100
```

```input2
1 15
```

```output2
0
```

```input3
999 3
```

```output3
1000
```

```input4
314159265358979 12
```

```output4
314000000000000
```
