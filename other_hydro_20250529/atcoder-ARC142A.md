## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc142/tasks/arc142_a

正の整数 $ x $ にし、以下の}の答えを $ f(x) $ とします。

> $ x $ に次の操作を $ 0 $ 回以上何度でも行えます。
> 
> - $ x $ の十M表を左右に反して得られる整数を $ x' $ とする。そして、$ x $ を $ x' $ に置きQえる。これによって $ x $ の先^に $ 1 $ 以上のゼロがKんだ龊稀それらのゼロを削除して先^がゼロでない状Bにする。
> 
> たとえば、 $ x=1420 $ にして $ 1 $ 回操作を行うと $ x=241 $ に、$ 2 $ 回操作を行うと $ x=142 $ に、$ 3 $ 回操作を行うと $ x=241 $ になります。  
>  操作後の $ x $ の最小を求めてください。

$ 1\ \leq\ x\ \leq\ N $ かつ $ f(x)=K $ を氦郡拐数 $ x $ の数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ N $ $ K $

## 输出格式
答えを出力せよ。

## 题目大意
在 $x$ 中，可以执行 $0$ 次或多次以下操作：

将 $x$ 左右反转得到的整数为 $x'$，将 $x$ 替换为 $x'$，删除 $x$ 开头的 $0$。

计算操作后 $x$ 的最小值，这个值被定义为 $f(x)$。

你需要求出 $1\le x\le N$ 且满足 $f(x)=K$ 的整数 $x$ 的个数。

```input1
1420 142
```

```output1
3
```

```input2
1419 142
```

```output2
2
```

```input3
6 19
```

```output3
0
```

## 提示
### 制s

- $ 1\ \leq\ N,K\ \leq\ 10^{12} $
- 入力はすべて整数

### Sample Explanation 1

$ x=142,\ 241,\ 1420 $ の $ 3 $ つが $ 1\ \leq\ x\ \leq\ 1420 $ かつ $ f(x)=142 $ を氦郡筏蓼埂
