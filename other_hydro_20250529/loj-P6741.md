## 题目描述

你有 $q$ 组询问，每组询问你需要计算出组合数 $\binom{n}{m}$ 的因子数量。

由于答案可能很大，你只需要输出将答案对 $p = 10^9 + 7$ 取模的结果即可。

## 输入格式

第一行一个正整数 $q$ 表示询问数量。

接下来 $q$ 行，每行两个整数 $n, m$，保证 $0\le m \le n$。

## 输出格式

输出 $q$ 行，每行一个整数对应该询问的答案。

```input1
3
0 0
4 2
10 3
```

```output1
1
4
16
```

## 数据范围与提示

对于 $10\%$ 的数据，保证 $q \le 10^{3}, n \le 10^{3}$。

对于 $50\%$ 的数据，保证 $q \le 10^{5}, n \le 10^{5}$。

对于 $100\%$ 的数据，保证 $q \le 5\times 10^{5}, n \le 10^{6}$。
