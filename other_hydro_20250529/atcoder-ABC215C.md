## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc215/tasks/abc215_c

文字列 $ S $ の各文字をKべ替えて作ることが可能な文字列を辞にすべて列い筏郡趣、前から $ K $ 番目にくる文字列を求めてください。

 「各文字をKべ替えて作ることが可能な文字列」とは？ 「文字列 $ A $ が文字列 $ B $ の各文字をKべ替えて作ることが可能な文字列である」とは、任意の文字が文字列 $ A $ と文字列 $ B $ に同数含まれるということを指します。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ S $ $ K $

## 输出格式
答えを出力せよ。

## 题目大意
在字符串 $S$ 的全排列中，字典序第 $K$ 小的排列组合是其中的哪种。

```input1
aab 2
```

```output1
aba
```

```input2
baba 4
```

```output2
baab
```

```input3
ydxwacbz 40320
```

```output3
zyxwdcba
```

## 提示
### 制s

- $ 1\ \le\ |S|\ \le\ 8 $
- $ S $ は英小文字のみからなる
- $ S $ の各文字をKべ替えてできる文字列は $ K $ N以上存在する

### Sample Explanation 1

文字列 `aab` の各文字をKべ替えて作ることが可能な文字列は $ \{ $ `aab`, `aba`, `baa` $ \} $ の $ 3 $ つですが、このうち辞で前から $ 2 $ 番目にくるものは `aba` です。
