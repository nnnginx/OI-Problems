<p>There is a king name Monkey. He love bananas so much. In his kingdom there is big field which contains many banana trees. We can represent this field as a 2D grid. In this grid some of the cells are contains banana tree, some of them are empty and some of the cells are are for tax area. If you select a cell which contains a banana tree then you can earn a banana and if you select a cell which is tax area then you must pay a banana to Monkey King.&nbsp;</p>
<p>Monkey king is really an intelligent man like a monkey. Today he gets an excellent idea. He wants to give you a chance to earn some bananas.But there is a condition for that. That condition is you can select only a square area from this field (2D grid).</p>
<p>Now Monkey King wants to know how many bananas (as much as possible) you can earn from this field.</p>
<p>&nbsp;</p>

<p>Input starts with an integer T(&lt;= 213), denoting the number of test cases.</p>
<p>Each case starts with a line containing two integers r and c (0 &lt;= r, c &lt;= 100), r denotes the number of rows and c denotes the number of columns of the modeled grid. Each of the next r lines contains c characters representing the field.</p>
