## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc070/tasks/abc070_b

Alice と Bob は、ロボットを制御するためのスイッチを1つずつ持っており、ロボットを婴しています。   
 Alice はロボットを婴し始めて $ A $ 秒後にスイッチを押し始め、ロボットを婴し始めて $ B $ 秒後にスイッチをxしました。   
 Bob はロボットを婴し始めて $ C $ 秒後にスイッチを押し始め、ロボットを婴し始めて $ D $ 秒後にスイッチをxしました。   
 Alice と Bob が、二人ともスイッチを押していた秒数を求めてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ A $ $ B $ $ C $ $ D $

## 输出格式
Alice と Bob が二人ともスイッチを押していた秒数を出力せよ。

## 题目大意
## 题目描述
Alice和Bob都有一个开关用来控制机器人。
Alice在A秒按下开关，移动机器人，并在B秒后释放开关。

Bob在C秒按下开关，移动机器人，并在D秒后释放开关。

求Alice和Bob都按下开关的秒数。
## 输入格式
一行，有四个正整数A，B，C，D。分别代表Alice在第A秒按下开关，在第B秒释放开关。Bob在第C秒按下开关，在第D秒释放开关。
## 输出格式
输出Alice和Bob都按下开关的秒数

```input1
0 75 25 100
```

```output1
50
```

```input2
0 33 66 99
```

```output2
0
```

```input3
10 90 20 80
```

```output3
60
```

## 提示
### 制s

- $ 0QA\ <\ BQ100 $
- $ 0QC\ <\ DQ100 $
- 入力は全て整数である。

### Sample Explanation 1

ロボットを婴肥激幛 $ 0 $ 秒後から $ 75 $ 秒後までのg、Alice はスイッチを押していました。 一方、ロボットを婴肥激幛 $ 25 $ 秒後から $ 100 $ 秒後までのg、Bob はスイッチを押していました。 したがって、二人が同rにスイッチを押していたrgは、ロボットを婴肥激幛 $ 25 $ 秒後から $ 75 $ 秒後までの $ 50 $ 秒です。

### Sample Explanation 2

Alice と Bob が同rにスイッチを押していないので、答えは $ 0 $ 秒です。
