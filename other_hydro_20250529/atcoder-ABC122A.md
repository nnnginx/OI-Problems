## 题目描述
[problemUrl]: https://atcoder.jp/contests/abc122/tasks/abc122_a

AtCoder 星には四Nのc基 `A`, `C`, `G`, `T` が存在し、`A` と `T`、`C` と `G` がそれぞれになります。

文字 $ b $ が入力されます。これは `A`, `C`, `G`, `T` のいずれかです。c基 $ b $ とになるc基を表す文字を出力するプログラムをいてください。

## 输入格式
入力は以下の形式で嗜肓Δら与えられる。

> $ b $

## 输出格式
c基 $ b $ とになるc基を表す文字を出力せよ。

## 题目大意
在 AtCoder 星球上，有四种类型的基数：`A`，`C`，`G` 和 `T`。`A` 与 `T` 结合，`C` 与 `G` 结合。

您会得到一个字母 $b$ 作为输入，它是 `A`，`C`，`G` 或者 `T`。编写一个程序，打印代表与 $b$ 结合的字母。

```input1
A
```

```output1
T
```

```input2
G
```

```output2
C
```

## 提示
### 制s

- $ b $ は文字 `A`, `C`, `G`, `T` のいずれかである。
