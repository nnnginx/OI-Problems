## 题目描述
[problemUrl]: https://atcoder.jp/contests/arc143/tasks/arc143_a

\板に $ 3 $ つの非整数 $ A $, $ B $, $ C $ がかれています． あなたは，以下の $ 2 $ つの操作を好きな序で好きな回数Rり返すことができます．

- $ 2 $ つの整数をxんで，それらから $ 1 $ を引く．
- すべての整数から $ 1 $ を引く．

あなたの目摔希\板にかれている数をすべて $ 0 $ にすることです． 目摔_成可能であるかどうか判定し，また可能であるなら必要な最小の操作回数を求めてください．

## 输入格式
入力は以下の形式で嗜肓Δら与えられる．

> $ A $ $ B $ $ C $

## 输出格式
目摔_成可能でない龊希`-1` を出力せよ．可能である龊希必要な最小の操作回数を出力せよ．

## 题目大意
黑板上写着三个整数 $A,B,C$。

你可以以任意顺序执行一下两个操作任意次：
- 选择其中两个数，并将这两个数减 $1$。
- 将三个数都减 $1$。

你的目标是使黑板上的所有数字都为 $0$。

确定目标是否可以实现。如果是，请输出最少的操作次数。

```input1
2 2 3
```

```output1
3
```

```input2
0 0 1
```

```output2
-1
```

```input3
0 0 0
```

```output3
0
```

## 提示
### 制s

- $ 0\ \leq\ A,\ B,\ C\ \leq\ 10^{18} $

### Sample Explanation 1

例えば次のように操作を行うことで，すべての数を $ 0 $ にすることができます． - $ A $ と $ C $ から $ 1 $ を引く．\板にかれた数は $ 1 $, $ 2 $, $ 2 $ となる． - $ B $ と $ C $ から $ 1 $ を引く．\板にかれた数は $ 1 $, $ 1 $, $ 1 $ となる． - すべての数から $ 1 $ を引く．\板にかれた数は $ 0 $, $ 0 $, $ 0 $ となる．
