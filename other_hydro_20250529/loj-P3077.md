## 题目描述

给出一个简单无向图，判断是否存在两个长度相同的简单环。

## 输入格式

第一行两个正整数 $n$ , $m​$ 表示点数和边数。

接下来 $m$ 行每行两个正整数描述一条边。

## 输出格式

若存在，输出 `Yes`；否则，输出 `No`。

```input1
10000 0
```

```output1
No
```

```input2
5 6
1 2
2 3
3 1
1 4
4 5
5 1
```

```output2
Yes
```

## 数据范围与提示
