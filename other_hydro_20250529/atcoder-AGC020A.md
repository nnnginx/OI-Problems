分数：$300$ 分

## 问题描述

游戏在一个由 $N$ 个连续编号为 $1$ 到 $N$ 的格子组成的条形上进行。

Alice 的棋子在格子 $A$ 上。Borys 的棋子在另一个不同的格子 $B$ 上。

玩家轮流行动，Alice 先走。每个移动的玩家必须将自己的棋子从当前格子 $X$ 移动到左邻格子 $X-1$ 或右邻格子 $X+1$。注意，不允许将棋子移动到条形之外或移动到另一个玩家的棋子所在的格子。每次移动必须恰好移动一次。

不能移动的玩家输，其他玩家赢。

两位玩家都希望获胜。如果他们都采取最优策略，谁会赢？

## 约束条件

- $2 \leq N \leq 100$
- $1 \leq A < B \leq N$
- 所有输入值均为整数。

## 输入

从标准输入中读取，格式如下：

> $N$ $A$ $B$

## 输出

如果 Alice 赢，输出 `Alice`；如果 Borys 赢，输出 `Borys`；如果没有人赢，输出 `Draw`。

```input1
5 2 4
```

```output1
Alice
```

Alice 可以将棋子移动到格子 $3$。之后，Borys 将无法将他的棋子移动到格子 $3$，因此他将不得不将棋子移动到格子 $5$。然后，Alice 将棋子移动到格子 $4$。Borys 无法再移动，最终输掉游戏。

```input2
2 1 2
```

```output2
Borys
```

Alice 无法进行第一次移动，因此她输掉游戏。

```input3
58 23 42
```

```output3
Borys
```
