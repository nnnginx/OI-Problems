## 题目描述
给定 $8$ 个数字 $X_1,X_2,...,X_8$，从中选出 $5$ 个数字，使得这 $5$ 个数字的总和最大。输出这 $5$ 个数字的和以及它们的编号。$X_i$ 的编号为 $i$。

## 输入格式
输入共 $8$ 行，第 $i$ 行包含一个正整数 $X_i$。**保证所有 $X_i$ 互不相同**。

## 输出格式
输出共两行。

第一行包含一个整数，表示最大的总和。

第二行包含 $5$ 个整数，表示 $5$ 个数的编号，**按升序排序**。

```input1
20
30
50
48
33
66
0
64
```

```output1
261
3 4 5 6 8
```

```input2
20
0
50
80
77
110
56
48
```

```output2
373
3 4 5 6 7
```

```input3
20
30
50
80
110
11
0
85
```

```output3
355
2 3 4 5 8
```

## 提示
#### 【数据范围】

对于 $100\%$ 的数据，$1 \le X_i \le 150$。

本题分值按 COCI 原题设置，满分 $50$。
