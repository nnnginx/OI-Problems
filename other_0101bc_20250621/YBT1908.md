# 问题描述

在《Harry Potter and the Deathly Hallows》中，Harry Potter他们一起逃亡，现在有许多的东西要放到赫敏的包里面，但是包的大小有限，所以我们只能够在里面放入非常重要的物品，现在给出该种物品的数量、体积、价值的数值，希望你能够算出能使背包的价值最大的组合方式，并且输出这个数值，赫敏会非常地感谢你。

# 格式

## 输入

第一行有 2个整数，物品种数 n 和背包装载体积 v。</br>
2 行到i+1 行每行 3 个整数，为第 i种物品的数量 m、体积w、价值 s。

## 输出

包含一个整数，即为能拿到的最大的物品价值总和。

# 样例

```input1
2 10
3 4 3
2 2 5
```

```output1
13
```

# 提示

选第一种一个，第二种两个。结果为 $3*1+5*2=13$
</br>

对于 30%的数据</br>
$1<=v<=500，1<=n<=2000，1<=m<=10，1<=w<=20，1<=s<=100$</br>
对于 100%的数据</br>
$1<=v<=500，1<=n<=2000，1<=m<=5000，1<=w<=20，1<=s<=100$</br>
