## 题目描述

一个正五边形 P 如下图所示。

![图片#B #S #R #60% #auto](https://m.meracode.com/upload/markdown/202411/202411060950150440.png) 判断连接点 S1 和 S2 的线段长度是否等于连接点 T1 和 T2 的线段长度。

## 输入格式

输入为以下格式：

S1S2

T1T2

## 输出格式

如果连接点 S1 和 S2 的线段长度等于连接点 T1 和 T2 的线段长度，输出 `Yes`；否则，输出 `No`。

## 输入输出样例

### 输入样例1

```
AC
EC
```

### 输出样例1

```
Yes
```

### 输入样例2

```
DA
EA
```

### 输出样例2

```
No
```

### 输入样例3

```
BD
BD
```

### 输出样例3

```
Yes
```
