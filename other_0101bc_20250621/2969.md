#### 题目描述


提示信息 楔形数指可以表示成三个不同素数的积的正整数。可以把一个楔形数表示为n = p x q x r，这里p、q、r是不同的素数因子。 例如，30 = 2 x 3 x 5，其中2、3、5是不同的三个素数，所以30是楔形数。 但20不是楔形数，虽然20 = 2 x 2 x 5，但是因子2出现了2次。 题目描述 给定两个正整数m、n，请统计出 m到n之间（含m和n）有多少个不同的楔形数。

## 输入

一行包含两个正整数m、n，整数之间以一个空格隔开。 数据范围 测试点1 \~ 5：1 ≤ m ≤ n ≤ 10000。 测试点6 \~ 10：1 ≤ m ≤ n ≤ 100000。

## 输出

一个整数，表示m到n之间（含m和n）有多少个不同的楔形数。


```input1
100 3000
```


```output1
478
```
