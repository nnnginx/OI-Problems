## 题目描述

仔仔怕黑又爱玩MC，但是玩得又菜，他在“我的世界”里盖了一个 $n \times n$ 的火柴盒，但是因为很菜，所以身上只有 $m$ 个火把和 $k$ 个萤石，分别放在 $(x_1, y_1) \sim (x_m, y_m)$ 和 $(o_1, p_1) \sim (o_k, p_k)$ 的位置，因为他很菜，所以火柴盒里没有任何物品，没有光并且没放东西的地方刷怪，请问他的火柴盒中有几个点会刷怪？

P.S. 仔仔玩的菜鸟版MC的火把照亮范围是：

```
|暗|暗| 光 |暗|暗|
|暗|光| 光 |光|暗|
|光|光|火把|光|光|
|暗|光| 光 |光|暗|
|暗|暗| 光 |暗|暗|
```

萤石：

```
|光|光| 光 |光|光|
|光|光| 光 |光|光|
光|光|萤石|光|光|
|光|光| 光 |光|光|
|光|光| 光 |光|光|
```

## 输入格式

输入共 $m + k + 1$ 行。
第一行为 $n, m, k$。
第 $2$ 到第 $m + 1$ 行分别是火把的位置 $x_i, y_i$。
第 $m + 2$ 到第 $m + k + 1$ 行分别是萤石的位置 $o_i, p_i$。

注：可能没有萤石，但一定有火把。

## 输出格式

有几个点会刷怪。

## 样例 #1

### 样例输入 #1

```
5 1 0
3 3
```

### 样例输出 #1

```
12
```

## 提示

数据保证，$1 \le n \le 100$，$1 \leq m+k \leq 25$，$1 \leq m \leq 25$，$0 \leq k \leq 5$。
