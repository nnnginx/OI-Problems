# 问题描述

给你一个整数数组 `nums` 。如果任一值在数组中出现 **至少两次** ，返回 `true` ；如果数组中每个元素互不相同，返回 `false` 。

# 格式

**示例 1：**

```input1
1 2 3 1
```

```output1
true
```

**解释：**

元素 1 在下标 0 和 3 出现。

**示例 2：**

```input2
1 2 3 4
```

```output2
false
```

**解释：**

所有元素都不同。

**示例 3：**

```input3
1 1 1 3 3 4 3 2 4 2
```

```output3
true
```
