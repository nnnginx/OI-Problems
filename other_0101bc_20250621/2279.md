# 问题描述

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

# 格式

# 样例1

```input1
100 4 200 1 3 2
```

```output1
4
```

<pre>
<strong>解释：</strong>最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。</pre>

# 样例2

```input2
0 3 7 2 5 8 4 6 0 1
```

```output2
9
```

# 样例3

```input3
1 0 1 2
```

```output3
3
```
