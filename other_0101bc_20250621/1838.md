# 说明

假设你正在爬楼梯。需要 $n$ 阶你才能到达楼顶。

每次你可以爬 $1$ 或 $2$ 个台阶。你有多少种不同的方法可以爬到楼顶呢？

# 格式

## 输入

一个整数 $n$。

## 输出

一个整数，即方法数。

# 样例

```input1
2
```

```output1
2
```

```input1
3
```

```output1
3
```

# 提示

### 样例说明

样例一：有两种方法可以爬到楼顶。

1. 1 阶 + 1 阶
2. 2 阶

样例二：有三种方法可以爬到楼顶。

1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶

### 数据范围

* $1 \le n \le 45$
