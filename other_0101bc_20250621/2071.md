## 题目描述

大聪明发明了一种密码，能用很少的长度表示很长很长的字符串。
密码的规则如下：
对于连续的若干个相同的子串 $\texttt{X}$ ，压缩为 $\texttt{[DX]}$ 的形式（$D$ 是一个整数且 $1\leq D\leq99$），比如说字符串 $\texttt{CBCBCBCB}$ 就压缩为 $\texttt{[4CB]}$ 或者$\texttt{[2[2CB]]}$，类似于后面这种压缩之后再压缩的称为二重压缩。如果是 $\texttt{[2[2[2CB]]]}$ 则是三重的。
现在给你一串压缩过的密码，请你对其进行解压缩。

## 输入格式

输入一行，一个字符串，表示压缩过的密码。

## 输出格式

输出一行，一个字符串，表示解压缩后的结果。

## 样例 #1

### 样例输入 #1

```text
AC[3FUN]
```

### 样例输出 #1

```text
ACFUNFUNFUN
```

## 提示

【数据范围】

对于 $50\%$ 的数据：解压后的字符串长度在 $1000$ 以内，最多只有三重压缩。

对于 $100\%$ 的数据：解压后的字符串长度在 $20000$ 以内，最多只有十重压缩。保证只包含数字、大写字母、`[` 和 `]`。
