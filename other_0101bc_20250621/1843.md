给定一个整数数组和一个整数 k ，请找到该数组中和为 k 的连续子数组的个数。

示例 1：

输入:nums = [1,1,1], k = 2
输出: 2
解释: 此题 [1,1] 与 [1,1] 为两种不同的情况
示例 2：

输入:nums = [1,2,3], k = 3
输出: 2

## 输入

第一个为k的值
第二个开始为数组的值

## 输出

子数组个数

# 格式

```input1
2
1 1 1
```

```output1
2
```

提示:

1 <= nums.length <= 2 \* 104
-1000 <= nums[i] <= 1000
-107 <= k <= 107
