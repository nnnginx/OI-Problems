# 问题描述

给你两个字符串 `haystack` 和 `needle` ，请你在 `haystack` 字符串中找出 `needle` 字符串的第一个匹配项的下标（下标从 0 开始）。如果 `needle` 不是 `haystack` 的一部分，则返回  `-1` 。

# 格式

## 输入

字符串 `haystack` 和 `needle` 

## 输出

在 `haystack` 字符串中找出 `needle` 字符串的第一个匹配项的下标

# 样例

```input1
sadbutsad sad
```

```output1
0
```

```input2
leetcode leeto
```

```output2
-1
```
