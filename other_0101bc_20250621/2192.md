你的任务是计算 a<sup>b</sup>对 `1337` 取模，`a` 是一个正整数，`b` 是一个非常大的正整数且会以数组形式给出。
## 输入格式

第一行一个正整数 $x$，表示要计算的数。
第二行一个整数 $n$，表示b的位数。
第三行输入n个数

## 输出格式

输出一行，表示计算的结果

# 样例1

```input1
2
1
3
```

```output1
8
```

# 样例2

```input2
2
2
1 0
```

```output2
1024
```

# 样例3

```input3
1
6
4 3 3 8 5 2
```

```output3
1
```

# 样例4

```input4
2147483647
3
2 0 0
```

```output4
1198
```


**提示：**

* `1 <= a <= 2<sup>31</sup> - 1`
* `1 <= b.length <= 2000`
* `0 <= b[i] <= 9`
* `b` 不含前导 0
