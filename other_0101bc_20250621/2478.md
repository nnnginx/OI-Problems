#### 题目描述

有n个单词（1≤n≤50），每个单词由2个小写字母组成，并约定第1个单词为龙头。
例如：n=7
7个单词为
aa
ac
ab
de
bh
hk
cd
接龙的方法为前一个单词的第2个字母和后一个单词的第1个字符相同，此时，可接的方法有：
aa―ac―cd 长度为3，即龙上有3个单词
aa―ab―bh―hk 长度为4
程序要求给出单词之后，求出最长龙的长度。

#### 输入

第一行一个整数n
接下来n行，每行2个字母表示一个单词（单词字母间无空格）

#### 输出

输出一个整数，即最长的接长度（即龙上单词的个数）。

#### 样例输入 **复制**

```
7
aa
ac
ab
de
bh
hk
cd
```

#### 样例输出 **复制**

```
4
```
