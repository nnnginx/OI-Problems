# 问题背景

2024 年 6 月 GESP C++ 一级编程第 2 题

# 题目描述

小杨有一个正整数 n ，他想知道 n 是否是一个立方数。

一个正整数 n 是立方数当且仅当存在一个正整数 x 满足 x×x×x=n 。

# 格式

## 输入

第一行包含一个正整数 n 。

## 输出

如果正整数 n 是一个立方数，输出 Yes，否则输出 No。

# 样例

```input1
8
```

```output1
YES
```

```input2
9
```

```output2
No
```

## 样例解释

对于样例1，存在正整数 2 使得 8=2×2×2 ，因此 8 为立方数。

对于样例2，不存在满足条件的正整数，因此 9 不为立方数。

## 数据范围

对于全部数据，保证有 1≤n≤1000 。
