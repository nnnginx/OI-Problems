# 题目描述

小杨同学积攒了一部分零用钱想要用来购买书籍，已知一本书的单价是13元，请根据小杨零用钱的金额，编写程序计算最多可以购买多少本书，还剩多少零用钱。

# 格式

## 输入

输入一个正整数 m ,表示小杨拥有的零用钱数。

## 输出

输出包含两行，第一行，购买图书的本书，第二行剩余的零用钱数。

# 样例

```input1
100
```

```output1
7
9
```

```input2
199
```

```output2
15
4
```

# 提示


### 数据范围

对于全部数据，保证有 0<m<200。
