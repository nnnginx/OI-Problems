# 题目描述

用户输入未来的某一天，输入格式为如 2021 6 1，编程计算这一天和2022.1.1相差多少天？（例如：今天和明天是相差一天）那一天是星期几？（注意：输出格式为 星期的数字值并在其前加“\*”，星期天用7表示)

# 格式

## 输入

共一行，三个整数：Y、M、D，分别代表未来某天的年、月、日。

## 输出

共两行 第一行一个整数，表示一个未来某天和今相差天数\$第一行一个整数，表示一个未来某天和今2022.1.1相差天数.\$第二行一个“\*”和一个整数，表示未来某天星期几。

# 样例

```input1
2022 1 2
```

```output1
1
*7
```

## 数据规模与约定

对于 100 的数据，2022≤Y≤3000、1≤M≤12、
