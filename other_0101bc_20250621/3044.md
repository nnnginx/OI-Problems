## 题目描述

在一条直线上有7个点、、、、、A、B、C、D、E、F和G,按这个顺序排列。(见下图) 相邻点之间的距离如下:

* A和B之间:3
* B和C之间:1
* C和D之间:4
* D和E之间:1
* E和F之间:5
* F和G之间:9 ![图片#369px #89px #B](https://m.meracode.com/upload/markdown/202402/202402281113110182.png) 给定两个大写英文字母p和q。p和q各自是、、、、、A、B、C、D、E、F或G之一,且p≠q。 请计算点p和点q之间的距离。

## 输入格式

输入为以下格式: p q

## 输出格式

输出点p和点q之间的距离。

## 输入输出样例

### 输入样例1

```
A C
```

### 输出样例1

```
4
```

### 输入样例2

```
G B
```

### 输出样例2

```
20
```

### 输入样例3

```
C F
```

### 输出样例3

```
10
```

## 数据范围与提示

【样例1说明】 点A和点C之间的距离是3+1=4。 【样例2说明】 点G和点B之间的距离是9+5+1+4+1=20。 【数据范围】 p和q各自是、、、、、A、B、C、D、E、F或G之一，p≠q。
