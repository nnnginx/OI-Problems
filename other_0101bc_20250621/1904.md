
# 问题描述

给定 $n$个字符串，分别判断其是否为一个回文串。只考虑字母和数字，并忽略大小写。

# 格式

## 输入

第一行一个正整数 $n$，表示有 $n$ 组数据。
接下来 $n$ 行，每行一个字符串，注意可能有空格。

## 输出

共 $n$ 行，且只包含 `yes` 或者 `no`。

# 样例

```input1
3
A man, a plan, a canal: Panama
race a car
1b , 1
```

```output1
yes
no
yes
```

# 提示

**数据范围：**
$1 \leq n \leq 1000$
字符串长度在 $10000$ 以内。
