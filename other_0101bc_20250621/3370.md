# 题目描述

小 A 喜欢坐地铁。地铁环线有n个车站，依次以1，2，...n标号。车站![image](./3370/file/2tPg7tQl6LcUNGN_p6ufk.png) 的下一个车站是车站i+1。特殊地，车站n的下一个车站是车站1。

小 A 会从某个车站出发，乘坐地铁环线到某个车站结束行程，这意味着小 A 至少会经过一个车站。小 A 不会经过一个车站多次。当小 A 乘坐地铁环线经过车站i时，小 A 会获得![image](./3370/file/_AZ9pzLTCiOIeU-dPWyu0.png) 点快乐值。请你安排小 A 的行程，选择出发车站与结束车站，使得获得的快乐值总和最大。

# 格式

## 输入

第一行，一个正整数n，表示车站的数量。

第二行，n个整数![image](./3370/file/3rxDuo2zy-Ch0hd0CG1I0.png) ，分别表示经过每个车站时获得的快乐值。

## 输出

一行，一个整数，表示小 A 能获得的最大快乐值。

# 样例

```input1
4
-1 2 3 0
```

```output1
5
```

```input2
5
-3 4 -5 1 3
```

```output2
5
```
