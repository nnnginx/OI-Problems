## 题目背景

**仔仔前排提醒：吸烟就是逊啦，请勿吸烟！**

## 题目描述

礼堂王子顶针 有 $n$ 个电子烟弹，他每抽完一个烟弹就把弹壳保存起来，因为用$k$个弹壳就能换一个新的烟弹，那么在不额外花钱的情况下，顶针总共能抽多少个烟弹？

注意这不是脑筋急转弯，顶针并不能借别人的弹壳，抽完后再还回去。

## 输入格式

每组测试数据一行包括两个整数 $n, k$（$1 < n, k \le 10^8$）。

## 输出格式

对于每组测试数据，输出一行包括一个整数表示最终烟的根数。

## 样例 #1

### 样例输入 #1

```
4 3
```

### 样例输出 #1

```
5
```

## 样例 #2

### 样例输入 #2

```
10 3
```

### 样例输出 #2

```
14
```

## 提示

对于 $100\%$ 的数据，$1<n, k \le 10^8$。
