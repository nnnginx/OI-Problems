# 问题描述

给定两个数组 `nums1` 和 `nums2` ，返回 *它们的 交集

* 。输出结果中的每个元素一定是 **唯一** 的。我们可以 **不考虑输出结果的顺序** 。

# 格式

# 样例1

```input1
1 2 2 1
2 2
```

```output1
2
```

# 样例2

```input2
4 9 5
9 4 9 8 4
```

```output2
4 9
```


**提示：**

* `1 <= nums1.length, nums2.length <= 1000`
* `0 <= nums1[i], nums2[i] <= 1000`
