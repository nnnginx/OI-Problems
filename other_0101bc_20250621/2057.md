# 说明

现在各大 oj 上有 $n$ 个比赛，每个比赛的开始、结束的时间点是知道的。

yyy 认为，参加越多的比赛，noip 就能考的越好（假的）。

所以，他想知道他最多能参加几个比赛。

由于 yyy 是Xm，如果要参加一个比赛必须善始善终，而且不能同时参加 $2$ 个及以上的比赛

# 输入格式

第一行是一个整数 $n$，接下来 $n$ 行每行是 $2$ 个整数 $a_i,b_i (a_i<b_i)$，表示比赛开始、结束的时间。

# 输出格式

一个整数最多参加的比赛数目

# 样例

```input1
3
0 2
2 4
1 3
```

```output1
2
```

# 提示

* 对于 $20\%$ 的数据，$n≤10$；
* 对于 $50\%$ 的数据，$n≤10^3$；
* 对于 $70\%$ 的数据，$n≤10^5$；
* 对于 $100\%$ 的数据，$1≤n≤10^6，0≤a_i<b_i≤10^6$。
