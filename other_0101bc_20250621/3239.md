# [GESP202403四级] 做题

## 题目描述

小杨同学为了提高自己的实力制定了做题计划，在第 k 天时，他必须要完成 k 道题，否则他就会偷懒。

小杨同学现在找到了一个题库，一共有 n 套题单，每一套题单中有一定数量的题目。但是他十分挑剔，每套题单他只会使用一次，每一天也只能使用一套题单里的题目，之后那套题单就会被弃之不用。对于每套题单，他不必完成题单内所有的题。

那么问题来了，小杨同学最多会做题几天才偷懒呢？

## 输入格式

第一行，1 个数为 n ，表示有多少套题单。

第二行， n 个整数 a1,a2,…,an ，分别表示每套题单有多少道题。

## 输出格式

输出一行，小杨同学偷懒前最多做题天数。

## 样例 #1

### 样例输入 #1

```
4
3 1 4 1
```

### 样例输出 #1

```
3
```

## 提示

### 数据范围

对于全部数据，保证有 1≤n≤106 ，1≤ai≤109。
