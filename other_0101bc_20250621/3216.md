## 题目背景

2024 年 6 月 GESP C++ 三级编程第 2 题

## 题目描述

小杨有一个包含 n 个正整数的序列 A=[a1,a2,...,an] ，他想知道是否存在 i(1≤i≤n) 使得 ai 是序列 A 中所有数 的倍数。

## 输入格式

第一行包含一个正整数 t ，代表测试用例组数。

接下来是 t 组测试用例。

对于每组测试用例，一共两行。其中，第一行包含一个正整数 n ；第二行包含 n 个正整数，代表序列 A 。

## 输出格式

对于每组测试用例，如果存在 i(1≤i≤n) 满足对于所有 k(1≤k≤n) ai 是 ak 的倍数，输出 Yes，否则输出 No。

## 样例

输入1:

```input1
2
3
1 2 4
5
1 2 3 4 5
```

输出1:

```output1
Yes
No
```

## 样例解释

对于第一组数据，对于 a3=4 ，满足 a3 是 a1 和 a2 的倍数。

## 数据范围

对于全部数据，保证有 1≤t≤10,1≤n≤105,1≤ai≤109 。
