## 题目描述

甲、乙、丙三人在操场上跑步，甲跑一圈需要 a 分钟，乙跑一圈需要 b 分钟，丙跑一圈需要 c 分钟;假设三人在某一时刻相遇，这一刻记为第 0 分钟，那么请问三人下一次相遇是在第几分钟?

## 输入

一行包含三个整数 a，b，c，分别表示甲、乙、丙三人跑一圈操场需要的时间(单位: 分钟)，整数之间以一个空格隔开。 数据范围测试点 1~~8: 1<= a，b，c <=1000 测试点 9~~10: 1<= a，b，c <=100000

## 输出

一个整数，表示三人下一次相遇是在第几分钟


```input1
2 3 5
```


```output1
30
```
