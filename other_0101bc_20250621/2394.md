# 题目描述

```
在墙角堆放着一堆完全相同的正方体小木块，如下图所示：
```

![](https://oj.czos.cn:443/upload/image/20171203/20171203140402_71121.jpg)

```
因为木块堆得实在是太有规律了，你只要知道它的层数就可以计算所有木块的数量了。
```

#### 输入

只有一个整数 n ，表示这堆小木块的层数，已知1 <= n <= 100 。

#### 输出

只有一个整数，表示这堆小木块的总数量。

#### 样例输入             

```
5
```

#### 样例输出            

```
35
```
