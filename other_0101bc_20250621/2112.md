给你二叉树的根节点 `root` 和一个表示目标和的整数 `targetSum` 。判断该树中是否存在 **根节点到叶子节点** 的路径，这条路径上所有节点值相加等于目标和 `targetSum` 。如果存在，返回 `true` ；否则，返回 `false` 。

**叶子节点** 是指没有子节点的节点。

## 输入格式

第一行一个整数 $n$，表示结点数。

之后 $n$ 行，第 $i$ 行三个整数$m$、 $l$、$r$，分别表示结点 $i$ 的权值以及左右子结点权值。若 $l=0$ 则表示无左子结点，$r=0$ 同理。

最后一行输入目标和的整数 `targetSum`

## 输出格式

输出一行，若存在路径输出true，否则输出false

## 输入输出样例 #1

# 样例1

![](https://assets.leetcode.com/uploads/2021/01/18/pathsum1.jpg)

```input1
9
5 4 8
4 11 0
8 13 4
11 7 2
7 0 0
2 0 0
13 0 0
4 0 1
1 0 0
22
```

```output1
true
```


**提示：**

* 树中节点的数目在范围 `[0, 5000]` 内
* `-1000 <= Node.val <= 1000`
* `-1000 <= targetSum <= 1000`
