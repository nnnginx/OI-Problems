# 问题描述

本例计算要处理的数据为整型数据1094861636，其二进制形式为01000001010000100100001101000100，不难发现其四个字节从低到高依次是字符'D'、'℃、'B'、'A'
的ASCII码,为了能输出这4个字节的数据，可以定义一个char类型的指针p(特殊情况)，并让它指向要处理的整数的起始地址，然后利用*p取其第一个字节内容并按“%c”格式输出，然后执行p++，让指针移动到第二个字节，依次执行相同操作，将四个字节数据打印出来。

# 格式

## 输入

无

## 输出

整数1094861636四个字节的内容

```input1
无
```

```output1
DCBA
```
