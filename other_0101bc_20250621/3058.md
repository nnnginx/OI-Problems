## 题目描述


小高和小李正在做一道加法题。给定两个正整数A和B，他们需要计算A+B（十进制）。如果计算过程中没有进位，输出"`Easy`"；如果有进位，输出"`Hard`"。

## 输入格式

输入A和B。

## 输出格式

如果计算过程中没有进位，输出"`Easy`"；如果有进位，输出"`Hard`"。

## 输入输出样例

### 输入样例1

```
229 390
```

### 输出样例1

```
Hard
```

### 输入样例2

```
123456789 9876543210
```

### 输出样例2

```
Easy
```

## 数据范围与提示

【样例1说明】 当计算229+390时，从十位到百位有进位，所以答案是"`Hard`"。 【样例2说明】 这里没有进位，所以答案是"`Easy`"。 注意，输入可能不适合32位整数。 【数据范围】 A和B是整数, 1≤A,B≤1018。
