# 问题描述

零零同学正在进行一场数字探险之旅。在这个奇妙的数字世界里，他遇到了一个有趣的挑战。有一个神秘的数字 N，零零需要找到第 N 个既不能被数字 2 整除也不能被数字 3 整除的正整数。这个数字到底会是多少呢？零零同踏上了寻找这个神秘数字的征程。

## 输入

输入为 N。

## 输出

输出为一个正整数。 如既不能被 **2**整除也不能被**3**整除的第 **N**个正整数；

| **样例1** |
| - |
| 输入：5   |
|输出：13 |

# 样例

```input1
5
```

```output1
13
```

# 提示

【数据范围】

对于 20%的数据， 保证 N<=2。

对于 40%的数据， 保证 N<=10。

对于 60%的数据， 保证 N<=10^5。

对于 100%的数据， 保证 N<=10^9。
