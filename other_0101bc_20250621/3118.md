## 题目描述

AtCoder目前举办四个系列的常规比赛：`ABC`、`ARC`、`AGC`和`AHC`。除了S1、S2和S3之外，AtCoder目前举办的常规比赛系列是什么？

## 输入格式

输入从标准输入中以下列格式给出：

S1

S2

S3

## 输出格式

输出所求答案。

## 输入输出样例

### 输入样例1

```
ARC
AGC
AHC
```

### 输出样例1

```
ABC
```

### 输入样例2

```
AGC
ABC
ARC
```

### 输出样例2

```
AHC
```

## 数据范围与提示

【样例1说明】 输入中给出了`ARC`、`AGC`和`AHC`。剩下的是`ABC`。 【数据范围】 S1、S2和S3中的每一个都是`ABC`、`ARC`、`AGC`或`AHC`，S1、S2和S3两两不同。
