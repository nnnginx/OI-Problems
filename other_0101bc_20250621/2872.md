#### 题目描述


题目描述：

成绩等级评定规则如下：

达到90分，评定为“A”；

达到60但不足90分，评定为“B”；

不足60分，评定为“C”。

给定成绩，编写程序来判定等级。


#### 输入


输入一个正整数，表示成绩。


#### 输出

一个字符（大写字母A、B或C），表示根据成绩评定的等级。

注意：

1.题目保证输入都在int范围内。

2.输入输出格式要和样例严格保持一致，不要输出任何与答案无关的内容，如“请输入：”之类的提示信息，或者在数据前面输出多余的空格或空行。

#### 样例输入              

```
90
```

#### 样例输出            

```
A
```
