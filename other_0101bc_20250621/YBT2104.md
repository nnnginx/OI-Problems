# 【题目描述】

小明在你的帮助下，破密了 Ferrari 设的密码门，正要往前走，突然又出现了一个密码

门，门上有一个算式，其中只有“(”，“)”，“0-9”，“+”，“-”，“\*”，“/”，“^”，求出的值就是

密码。小明数学学得不好，还需你帮他的忙。(“/”用整数除法)

## 【输入格式】

输入文件 calc.in 共 1 行，为一个算式。

## 【输出格式】

输出文件 calc.out 共 1 行，就是密码。

## 【样例输入】

```input1
1+(3+2)\*(7^2+6\*9)/(2)
```

## 【样例输出】

```output1
258
```

## 【提示】

100%的数据满足：算式长度<=30 其中所有数据在 231-1 的范围内。
