# 题目描述

输入一个正方形的边长（a）及一个长方形的长与宽（b、c），然后比较两个图形的面积。如果长方形面积大， 输出“RECTANGLE”；如果正方形面积大，输出“SQUARE”；如果一样大，输出“SAME”。

# 格式

## 输入

输入正整数 、、a、b、c，分别代表正方形的边长和长方形的长与宽。

## 输出

如果长方形面积大，输出“RECTANGLE”； 如果正方形面积大，输出；\$如果正方形面积大，输出“SQUARE”；\$如果一样大， 输出 “SAME”。

# 样例

```input1
5 4 6
```

```output1
SQUARE
```

## 数据规模与约定

对于 100 的数据，0<a,b,c≤1000。
