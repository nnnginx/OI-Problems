# 问题描述

**字母异位词** 是指由相同字母组成但排列顺序[不同的](https://so.csdn.net/so/search?q=%E4%B8%8D%E5%90%8C%E7%9A%84&spm=1001.2101.3001.7020)单词。例如，`"aab"`、`"aba"` 和 `"baa"` 都是字母异位词。

给定两个字符串 `s` 和 `t` ，编写一个函数来判断 `t` 是否是 `s` 的 字母异位词。

# 格式

**示例 1:**

```input1
anagram
nagaram
```

```output1
true
```

**示例 2:**

```input2
rat
car
```

```output2
false
```
