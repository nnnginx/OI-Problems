# 题目描述

四舍五入是一种常见的近似计算方法。现在，给定n个整数，你需要将每个整数四舍五入到最接近的整十数。例如， 43四舍五入后为40，58， 四舍五入后为60。

# 格式

## 输入

共n+1行，第一行，一个整数n，表示接下来输入的整数个数。

接下来n行，每行一个整数![image](./3368/file/ghq93ph3XrdFzBiP0GT7l.png) ，表示需要四舍五入的整数。

## 输出

n行，每行一个整数，表示每个整数四舍五入后的结果。

# 样例

```input1
5 
43 
58 
25 
67 
90
```

```output1
40 
60 
30 
70 
90
```
