# 题目描述

小 A 最近刚刚学习了因数的概念，具体来说，如果一个正整数 a 可以被另一个正整数 b 整除，那么我们就说 b 是 a 的因数。

请你帮忙写一个程序，从小到大输出正整数 a 的所有因数。

# 格式

## 输入

输入一行一个正整数 a。保证 a≤1000。

## 输出

输出若干行，为 a 的所有约数，从小到大排列。

# 样例

```input1
1
```

```output1
1
```

```input2
6
```

```output2
1
2
3
6
```

```input3
10
```

```output2
1
2
5
10
```
