#### 题目描述

![](https://cspoj.com/upload/cspoj.com/20220524//1223_1_20220524130841_90436.jpg)

```
汉诺塔的问题大家都已经很熟悉了，有三个柱子，每个柱子上有一些大小不一的金片，要把金片从A柱移动到C柱，可以借助B柱，请问n个金片的情况下，需要最少移动多少次？
```

#### 输入

输入一个整数n代表金片的数量（n<=20）

#### 输出

一个整数，代表n个金片的移动次数

#### 样例输入              

```
3
```

#### 样例输出             

```
7
```
