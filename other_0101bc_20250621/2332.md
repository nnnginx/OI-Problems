

# 问题描述

小杨有一棵包含 n个节点的树，这棵树上的任意一个节点要么是白色，要么是黑色。小杨认为一棵树是美丽树当且

仅当在删除所有白色节点之后，剩余节点仍然组成一棵树。

小杨每次操作可以选择一个白色节点将它的颜色变为黑色，他想知道自己最少要执行多少次操作可以使得这棵树变

为美丽树。

# 格式

## 输入

![image](./2332/file/XMxShomWCxNqD6ZirbFok.png) 

## 输出

输出一个整数，代表最少执行的操作次数。

# 样例

```input1
5 
0 1 0 1 0 
1 2
1 3 
3 4
3 5
```

```output1
2
```
