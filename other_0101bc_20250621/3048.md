## 题目描述


你得到一个由小写英文字母组成的字符串 S。找出在 S 中出现次数最多的字符。如果有多个这样的字符，报告字母顺序最靠前的那个。

## 输入格式

输入S。

## 输出格式

在 S 中出现次数最多的字符中，输出字母顺序最靠前的那个。

## 输入输出样例

### 输入样例1

```
frequency
```

### 输出样例1

```
e
```

### 输入样例2

```
atcoder
```

### 输出样例2

```
a
```

### 输入样例3

```
pseudopseudohypoparathyroidism
```

### 输出样例3

```
o
```

## 数据范围与提示

【样例1说明】 在 `frequency` 中，字母 `e` 出现了两次，比其他任何字符都多，所以应该输出 `e`。 【样例2说明】 在 `atcoder` 中，字母 `a`、`t`、`c`、`o`、`d`、`e` 和 `r` 各出现一次，所以应该输出字母顺序最靠前的，即 `a`。

【数据范围】

* 1≤|S|≤1000 (|S| 是字符串 S 的长度。)
* S 中的每个字符都是小写英文字母。
