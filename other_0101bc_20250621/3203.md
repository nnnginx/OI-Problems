## 题目描述

有2N个人站成一排，从左到右第i个位置的人穿着颜色为Ai的衣服。这里衣服有N种颜色，从1到N编号，每种颜色恰好有两个人穿。请找出有多少个整数i(1≤i≤N)满足以下条件:

* 穿着颜色i衣服的两个人之间恰好有一个人。

## 输入格式

输入从标准输入中给出，格式如下:

N

A1 A2 ... A2N

## 输出格式

输出所求答案。

## 输入输出样例

### 输入样例1

```
3
1 2 1 3 2 3
```

### 输出样例1

```
2
```

### 输入样例2

```
2
1 1 2 2
```

### 输出样例2

```
0
```

### 输入样例3

```
4
4 3 2 3 2 1 4 1
```

### 输出样例3

```
3
```
