## 题目描述

给定一个长度为 N 的整数序列 A 和一个整数 X。从 A 中删除所有等于 X 的元素，并将剩余元素按原顺序排列得到序列 A′。请输出 A′。

## 输入格式

输入从标准输入中给出，格式如下：

N X

A1 A2 A3... AN

## 输出格式

按顺序输出 A′ 的元素，元素之间用空格分隔。

## 输入输出样例

### 输入样例1

```
5 5
3 5 6 5 4
```

### 输出样例1

```
3 6 4
```

### 输入样例2

```
3 3
3 3 3
```

### 输出样例2

```

```

## 数据范围与提示

【样例1说明】 从 [3, 5, 6, 5, 4] 中删除 5 得到 [3, 6, 4]。 【样例2说明】 A′ 可以是一个空序列，这种情况下我们只需要输出一个空行。 【数据范围】 1≤N≤105,1≤X≤109,1≤Ai≤109，输入中的所有值都是整数。
