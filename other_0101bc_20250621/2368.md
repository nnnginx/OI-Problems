## 问题背景

小杨有n元钱用于购物。商品A的单价为a元，商品B的单价是b元。小杨想购买相同数量的商品A和商品B。

请你编写程序帮小杨计算他最多能购买多少个商品A和商品B。

---

## 问题描述

给定一个正整数n，代表小杨用于购物的总金额。

第二行包含一个正整数a，代表商品A的单价。

第三行包含一个正整数b，代表商品B的单价。

---

## 格式

### 输入

第一行包含一个正整数n，代表小杨用于购物的总金额。

第二行包含一个正整数a，代表商品A的单价。

第三行包含一个正整数b，代表商品B的单价。

---

### 输出

输出一个整数，代表小杨最多能购买多少个商品A和商品B。

---

## 样例

### 输入数据 1

```
12
1
2
```

### 输出数据 1

```
4
```

### 输入数据 2

```
13
1
2
```

### 输出数据 2

```
4
```

---
## 提示
对于样例1，由于需要购买相同数量的两种商品，因此小杨最多能购买4件商品A和4件商品B，共花费4×1+4×2=12元。因此，样例的答案为4。

对于样例2，由于需要购买相同数量的两种商品，因此小杨最多能购买4件商品A和4件商品B，共花费4×1+4×2=12元。如果小杨想购买5件商品A和5件商品B，则花费5×1+5×2=15元，超过了小杨的预算13元。因此，样例的答案为4。

|--|--|--|--|
| 1 | 30% | n ≤ 1000 | 1 ≤ a, b ≤ 1000 |
| 2 | 30% | n ≤ 1000 | 1 ≤ a, b ≤ 1000 |
| 3 | 40% | n ≤ 10^5 | 1 ≤ a, b ≤ 10^5 |

对于全部数据，保证：  
1 ≤ n ≤ 10^5，1 ≤ a, b ≤ 10^5
