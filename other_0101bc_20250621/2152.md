# 说明

给定一个整数，写一个函数来判断它是否是 3 的幂次方。如果是，输出 `YES` ；否则，输出 `NO` 。

整数 $n$ 是 $3$ 的幂次方需满足：存在整数 $x$ 使得 $n = 3^x$

# 输入

一行一个整数 $n$；

# 输出

`YES` 或者 `NO` 。

# 样例

```input1
27
```

```output1
YES
```

```input2
0
```

```output2
NO
```

```input3
9
```

```output3
YES
```

```input4
45
```

```output4
NO
```

# 提示

#### 数据范围

* $-2^{31} \leq n \leq 2^{31} - 1$
