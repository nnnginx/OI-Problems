## 题目描述

小新很不爱惜东西，经常弄坏东西。他有一个键盘，这个键盘只有 26 个字母键和退格键(backspace)可以使用了，一共 27 个键，并且切换大小写的按键也坏了，所以他只能输入小写字母。

当按下任何一个小写英文字母的键的时候，文本编辑器就会在当前编辑文本的末端添加对应的字母。
例如，假设当前文本是 `codin`，当她按下 `g` 键时，文本就会变成 `coding`。

当按下退格键的时候，文本编辑器就会删除当前文本的**最后一个**字母。如果当前文本是空的，则什么都不会发生。
例如，如果当前文本是 `coding`，当按下退格键后，文本就会变成 `codin`。

仔仔很好奇小新用这种键盘到底能打什么文字，于是悄悄在他电脑里安装了一个木马程序，能够记录小新每次按下的键，并将按键记录发送给自己。

但是仔仔不知道该怎么解读这个按键记录，于是他找到了编程高手的你，让你帮忙解读。

## 输入格式

第一行是一个整数 $n$（$1 \leq n \leq 100$），表示按键的次数。
第二行是 $n$ 个用空格隔开的字符串，依次表示小新按下的每个按键。

输入的每个字符串要么是一个小写字母，表示对应的按键，要么是字符串 `<bs>`，表示退格键。

## 输出格式

输出一行一个字符串，表示小新按完给定的所有按键后的文本。**数据保证输出不是空串**。

## 样例 #1

### 样例输入 #1

```
5
a b c d e
```

### 样例输出 #1

```
abcde
```

## 样例 #2

### 样例输入 #2

```
3
a <bs> b
```

### 样例输出 #2

```
b
```
