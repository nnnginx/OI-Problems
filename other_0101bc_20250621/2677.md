## 输入格式

第一行为整数 n(n<=5000)，表示数的个数。
第二行有 n 个整数，为给定的 n 个数，每个数的绝对值都小于 10000。

## 输出格式

第一行输出这些数中的最大值
第二行输出这些数中的最小值
第三行输出这些数的和
第四行输出平均值(保留两位小数)
第五行输出与平均值的差在 2(包含 2)以内的数据的数量

输入1:

```input1
3
1 2 3
```

输出1:

```output1
3
1
6
2.00 
3
```

输入2:

```input2
5
9 7 8 3 4
```

输出2:

```output2
9
3
31
6.20 
2
```
