# 问题背景

一张纸厚度为0.1毫米，将其不断地对折，问对折多少次后其厚度可以超过珠穆朗玛峰的高度？（珠穆朗玛峰高度为8848米）

# 问题描述

该问题是在理想的情况下将一张纸不断对折后
计算其厚度的问题，不难发现，其折纸厚度的规律如下：
对折1次厚度为：0.1*21=0.2毫米
对折2次厚度为：0.1*22=0.4毫米
对折3次厚度为：0.1*23=0.8毫米
….
对折n次厚度为：0.1*2n毫米
即当0.1*2n > 8848*1000时，折纸的厚度超过珠穆朗玛峰的高度，折纸结束。


# 格式

## 输入

无

## 输出

问对折多少次后其厚度可以超过珠穆朗玛峰的高度


