# 问题描述

在一个由 `'0'` 和 `'1'` 组成的二维矩阵 `matrix` 中，找到只包含 `'1'` 的最大正方形，并返回其面积。

# 格式

## 输入

第一行：n,m（分别代表行，列）
之后几行：地图

## 输出

最大正方形面积

# 样例

```input1
3 5
1 0 1 0 0
1 0 1 1 1
1 1 1 1 1
```

```output1
4
```

# 样例2

```input2
2 2
0 0
0 0
```

```output2
0
```

# 提示

- $1\leq n\leq 20$

```
解释1: 最大的只包含 '1' 的正方形边长为 2，面积为 4。

解释2: 矩阵中没有只包含 '1' 的正方形，面积为 0。
```
