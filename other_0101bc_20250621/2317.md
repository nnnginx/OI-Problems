## 问题描述

小杨有 n 个正整数，需要找出这些正整数的数位和中的最大值。“数位和” 指的是一个数字中所有数位的和，例如数字 12345，它的数位和是 1 + 2 + 3 + 4 + 5 = 15。

## 格式

### 输入

第一行包含一个正整数 n，代表正整数个数。
之后 n 行，每行包含一个正整数。

### 输出

输出这些正整数的数位和的最大值。

## 样例


```input1
3
16
81
10
```


```output1
9
```


```input2
2
234
111
```


```output2
9
```


```input3
4
456
789
123
321
```


```output3
24
```

## 提示

数据范围：1 ≤ n ≤ 10^5，每个正整数不超过 10^12。
