## 题目描述

桌面上有 n 张卡片，每张卡片上都印有一个整数，卡片上的整数即使相同，也算不同的卡片。要从这 n 张卡片中任意抽取两张，并记录两张卡片上的数之和，请统计所有组合中出现次数最多的和:如果出现次数最多的和有多个，那就输出其中最小的和。 例 1: n = 4，这4张卡片上的数分别是 1，2，3，4;从中任意抽取两张，所有的卡片组合情况如下: 1 + 2 = 3，1 + 3 = 4,1 + 4 = 5,2 + 3 = 5,2 + 4 = 6,3 + 4 = 7其中 5 作为和出现的次数最多，所以输出 5。 例 2: n = 4，这4张卡片上的数分别是 1，2，3，1; 所有的卡片组合情况如F: 1 + 2 = 3，1 + 3 = 4，1 + 1 = 2,2 + 3 = 5,2 + 1 = 3,3 + 1 = 4;其中 3 和 4 作为和出现的次数最多，按照题目要求，如果出现次数最多的和有多个，那就输出其中最小的和，所以输出 3。

## 输入

共两行; 第一行一个整数 n，表示卡片的数量; 第二行包含 n 个整数 a1，a2，a3.....an，表示每张卡片上的数，整数之间以一个空格隔开。 数据范围 测试点1\~10: 2<=n<=20，1<=ai<=100

## 输出

个整数，表示出现次数最多的和;如果出现次数最多的和有多个，那就输出其中最小的和。


```input1
4
1 2 3 4
```


```output1
5
```


```input2
4
1 2 3 1
```


```output2
3
```
