#### 题目描述

一矩形阵列由数字0到9组成,数字1到9代表细胞,细胞的定义为沿细胞数字上下左右还是细胞数字则为同一细胞,求给定矩形阵列的细胞个数。如:
阵列
4 10
0234500067
1034560500
2045600671
0000000089
有4个细胞。

#### 输入

第一行为矩阵的行n和列m; (0 ≤ n,m ≤ 1,000)
下面为一个n×m的矩阵。

#### 输出

细胞个数。

#### 样例输入 

```
4 10
0234500067
1034560500
2045600671
0000000089
```

#### 样例输出

```
4
```

#### 提示

这个题描述得相当晦涩，就是以“0”为分界线，将非0的数字分成多少个区域。样例四个细胞为A：2345-3456-456 B：67 C:12 D:671-89
