# 二叉树的锯齿形层序遍历

## 题目描述

有一个 $n(n \le 10^6)$ 个结点的二叉树。给出每个结点的两个子结点编号（均不超过 $n$），建立一棵二叉树（根节点的编号为 $1$），如果是叶子结点，则输入 `0 0`。

建好树这棵二叉树之后，求出它的 **锯齿形层序遍历**（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

## 输入格式

第一行一个整数 $n$，表示结点数。

之后 $n$ 行，第 $i$ 行两个整数 $l$、$r$，分别表示结点 $i$ 的左右子结点编号。若 $l=0$ 则表示无左子结点，$r=0$ 同理。

## 输出格式

输出一行，是这个二叉树的**锯齿形层序遍历**（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

## 输入输出样例 #1

# 样例1

```input1
5
3 5
0 0
2 4
0 0
0 0
```

```output1
1 5 3 2 4
```
