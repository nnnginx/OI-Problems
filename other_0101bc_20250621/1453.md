## 题目描述

有正整数 $x$ 与 $y$，求 $x^y$ 。
当答案超过 ${10}^9$ 时，输出 `-1` ，否则就输出正确的答案。

## 输入格式

共一行，即两个正整数 $x$ 和 $y$。

## 输出格式

一个整数，即 $x^y$ 的答案或 $-1$。

## 样例 #1

### 样例输入 #1

```
5 3
```

### 样例输出 #1

```
125
```

## 样例 #2

### 样例输入 #2

```
114514 1919
```

### 样例输出 #2

```
-1
```

## 提示

数据保证 $1 \le x, y \le 2 \times{10}^9$。
