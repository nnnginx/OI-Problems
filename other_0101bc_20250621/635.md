## 题目描述

设有一个由正整数组成的长度为 $n$ 的数列，求其中连续 $m$ 个元素的和的最小值是多少？

## 输入格式

第一行两个正整数 $n$ 和 $m$
接下来 $n$ 行为数列中的元素

## 输出格式

输出一行一个整数，表示连续 $m$ 个元素的和的最小值是多少

## 样例 #1

### 样例输入 #1

```
8 3
1
4
7
3
1
2
4
3
```

### 样例输出 #1

```
6
```

## 提示

#### 数据规模与约定

 $0 \leq m \leq n \leq 3 \times 10^3$，$1 \leq a_i \leq 100$。
