# 说明

给定 $n$ 个表示变量之间关系的字符串 ，其长度为 $4$，有以下两种形式：`"a==b"`或`"a!=b"`。在这里，`a` 和 `b` 是小写字母（可能相同），表示单字母变量名。

只有当可以将整数分配给变量名，以便满足所有给定的方程时才输出 `YES`，否则输出 `NO`。

# 格式

## 输入

第一行一个整数 $n$。
接下来 $n$ 行，每行一个长度为 $4$ 的字符串。

## 输出

`YES` 或者 `NO`。

# 样例

```input1
2
a==b
b!=a
```

```output1
NO
```

```input2
2
b==a
a==b
```

```output2
YES
```

```input3
3
a==b
b==c
a==c
```

```output3
YES
```

# 提示

## 样例说明

#### 样例一

如果我们指定，$a = 1$ 且 $b = 1$，那么可以满足第一个方程，但无法满足第二个方程。没有办法分配变量同时满足这两个方程。

#### 样例二

我们可以指定 $a = 1$ 且 $b = 1$ 以满足满足这两个方程。

## 数据范围

1. $1 \le n \le 500$
2. 字符串长度为 $4$
3. 字符串头尾字符为小写字母
4. 字符串中间部分为 `==` 或者 `!=`
