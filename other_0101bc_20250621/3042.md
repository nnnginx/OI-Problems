# 问题背景

在遥远的大海中，有一个传说中的神秘岛屿，岛上藏有无数宝藏。一位勇敢的探险家得到了一张藏宝图，上面标记了宝藏的大致位置。探险家需要编写一个程序，根据藏宝图上的坐标（x, y），计算出宝藏的确切位置。

# 问题描述

编写一个程序，输入两个整数x和y，分别代表探险家当前位置与宝藏位置的水平和垂直距离。程序需要计算出探险家到达宝藏位置所需的最短路径长度。

# 格式

## 输入

输入两个整数x和y，用空格隔开。

## 输出

输出一个整数，表示探险家到达宝藏位置的最短路径长度。

# 样例

```input1
9 4
```

```output1
11
```
