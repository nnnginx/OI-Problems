# 问题描述

yyyymmdd的日期格式想必大家都不陌生，其中yyyy表示年份，mm表示月份，dd表示日期。dd(日期)只有一种表示方法(01-28或01-29或01-30或01-31)

回文日期指的是对于一个日期格式是yyyymmdd的日期，无论从左到右还是从右到左读都是一样的，现给定两个日期（包括这两个日期本身），求有多少个回文日期存在。
例如：
2024年10月01日的表示为20241001，它不是回文日期。2010年1月2日的表示为20100102，它是回文日期。
2024年10月02日的表示为20241002，它也不是回文日期。

# 格式

## 输入

两行，每行包括一个 8 位数字。

第一行表示起始日期。

第二行表示终止日期。

保证 date1，date2都是真实存在的日期，且年份部分一定为 4 位数字，且首位数字不为 0。

保证 date1 一定不晚于 date2。

## 输出

一个整数，表示在date1 和 date2 之间，有多少个日期是回文的。

# 样例

```input1
20011023
92240912
```

```output1
280
```

```input2
20100102
20100102
```

```output2
1
```
