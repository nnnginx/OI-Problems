# 题目描述

给定一个长度为 $n$ 的整数数组 `a[]` 和一个整数 $k$，请返回其中出现频率前 $k$ 高的元素。
按 **频率高低顺序** 输出，如两个元素频率相同则元素值小的排前面。

# 格式

## 输入

第一行两个整数 $n$、$k$。
第二行 $n$ 个整数 $a_i$。

## 输出

一行 $k$ 个整数，即出现频率前 $k$ 的元素，单个空格隔开。

# 样例

```input1
6 2
1 1 1 2 2 3
```

```output1
1 2
```

```input2
1 1
1
```

```output2
1
```

# 提示

* $1 <= n <= 10^5$
* $k$ 保证不超过数组不同元素的个数。
