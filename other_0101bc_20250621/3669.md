## 题目描述


奶牛们要去太空了！它们打算用方块建造一座太空电梯。现在它们有 $N$ 种方块，第 $i$ 种方块有一个特定的高度 $h_i$，一定的数量 $c_i$。为了防止宇宙射线破坏方块，第 $i$ 种方块的任何部分不能超过高度 $a_i$。
请用这些方块堆出最高的太空电梯。

## 输入格式

第一行，一个整数 $N$；
第二行到 $N+1$ 行，第 $i+1$ 行三个整数 $h_i,a_i,c_i$，数字之间用空格分隔。

## 输出格式

共一行，一个整数，为太空电梯的高度。

## 输入输出样例 #1

### 输入 #1

```
3
7 40 3
5 23 8
2 52 6
```

### 输出 #1

```
48
```
