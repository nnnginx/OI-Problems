# 问题描述

100 可以表示为带分数的形式：100 = 3 + 69258 / 714

还可以表示为：100 = 82 + 3546 / 197

注意特征：带分数中，数字 1\~9 分别出现且只出现一次（不包含 0 ）。

类似这样的带分数，100 有 11 种表示法。

# 格式

## 输入

从标准输入读入一个正整数 N(N<1000×1000)

## 输出

程序输出该数字用数码 1\~9 不重复不遗漏地组成带分数表示的全部种数。

注意：不要求输出每个表示，只统计有多少表示法！

# 样例

```input1
100
```

```output1
11
```

# 提示

### 运行限制

* 最大运行时间：3s
* 最大运行内存: 64M
