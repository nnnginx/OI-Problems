给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

# 样例1

```input1
()
```

```output1
true
```

# 样例2

```input2
()[]{}
```

```output2
true
```

# 样例3

```input3
(]
```

```output3
false
```


**提示：**

* `1 <= s.length <= 10<sup>4</sup>`
* `s` 仅由括号 `'()[]{}'` 组成
