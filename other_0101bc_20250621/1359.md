# 问题描述

12.有一行电文,已按下面规律译成密码:
A→Z   a→z
B→Y   b→y
C→X   c→x
即第1个字母变成第26个字母,第i个字母变成第(26一i+1)个字母,非字母字符不变。要求编程序将密码译回原文,并输出密码和原文。

# 格式

## 输入

输入一个字符串数组，存储密码

## 输出

输出两行字符串 ：
第一行 ：密码，
第二行：破译密码后的原文

# 样例

输入

```input1
R droo erhrg Xsrmz mvcg dvvp.
```

输出

```output1
R droo erhrg Xsrmz mvcg dvvp.

I will visit China next week.
```
