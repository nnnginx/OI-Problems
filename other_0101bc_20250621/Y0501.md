<h2>说明</h2>

<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">输入一个数字</span>n<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">，表示身高，程序会输出一句英文，"</span>My height is&nbsp;ncm.<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">”，其中</span>n<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">代指输入的数字。例如：输入</span>180<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">，输出是“</span>My height is 180cm.<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">”。</span>
<h2>输入格式</h2>

<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">输入一个数&nbsp;</span>n<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">。</span>
<h2>输出格式</h2>

<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">输出"</span>My height is&nbsp;ncm.<span style="font-family:&quot;font-size:medium;background-color:#FCFCFC;">"</span>
<h2>样例</h2>
<pre><code class="language-input1">175</code></pre><pre><code class="language-output1">My height is 175cm.</code></pre>
