#### 题目描述

题目名称：不是3的倍数的数 题目描述：如果整数a能被整数b整除，那么a就是b的倍数。

例如：6能被3整除，所以6是3的倍数。

给定两个正整数m、n，统计从m到n之间不是3的倍数的数总共有多少。例如：m = 1，n = 15，

在1\~15之间，3的倍数有3，6，9，12，15，共有5个；剩下不是3的倍数的数总共有10个，所以输出10。

#### 输入

一行，包含两个正整数m、n（0 < m <= n < 1000）。相邻两个数用一个空格隔开。

#### 输出

一个整数，表示从m到n之间（包括m和n）不是3的倍数的数的个数。

#### 样例输入 **复制**

```
1 15
```

#### 样例输出 **复制**

```
10
```
