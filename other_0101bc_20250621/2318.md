## 问题描述

小杨有一个包含 n 个数字的序列 A，即 A = {a<sub>1</sub>, a<sub>2</sub>, ..., a<sub>n</sub>}，他想将其中大于 k 的数字都替换为序列的最大值，将其中小于 k 的数字都替换为序列的最小值，请你帮他计算出替换后的序列。

## 格式

### 输入

第一行包含两个正整数 n，k，含义如题面所示。
第二行包含 n 个数字，代表序列 A。

### 输出

输出 n 个整数，代表替换后的结果。

## 样例

```input1
5 0
-2 -1 0 1 2
```

```output1
-2 -2 0 2 2
```

```input2
4 3
1 4 2 5
```

```output2
1 5 1 5
```

```input3
6 5
3 6 4 7 5 8
```

```output3
3 8 3 8 5 8
```

## 提示

数据范围：1 ≤ n ≤ 10<sup>5</sup>，|a<sub>i</sub>| ≤ 10<sup>5</sup>。
