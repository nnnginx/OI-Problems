# 问题描述

给你一个满足下述两条属性的 `m x n` 整数矩阵：

* 每行中的整数从左到右按非严格递增顺序排列。
* 每行的第一个整数大于前一行的最后一个整数。

给你一个整数 `target` ，如果 `target` 在矩阵中，返回 `true` ；否则，返回 `false` 。

# 格式

## 输入

第一行两个整数m、n，表示m*n整数矩阵。
输入m行n列的矩阵。
输入一个整数 `target`。

## 输出

如果 `target` 在矩阵中，返回 `true` ；否则，返回 `false`

# 样例

```input1
3 4
1 3 5 7
10 11 16 20
23 30 34 60
3
```

```output1
true
```
