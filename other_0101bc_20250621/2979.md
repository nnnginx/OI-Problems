#### 题目描述

在一次考试中，知道了每个学生的学号及语文、数学和英语三科的成绩，要求按输入顺序输出学生的学号和总成绩。

#### 输入

输入：

第1行有1个整数，表示学生的人数n（1≤n≤100）。

其后有n行数据，每行包括4个正整数，分别表示学号、语文、数学和英语三科的成绩（不大于150），每两个数中间用一个空格分隔。

#### 输出

共n行，

每行包含两个整数，分别是学生的学号和总成绩，中间用单个空格分隔

#### 样例输入 **复制**

```
3
100001 90 80 70
100002 93 99 100
100003 89 91 69
```

#### 样例输出 **复制**

```
100001 240
100002 292
100003 249
```
