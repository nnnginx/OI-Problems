

# 问题描述

在一次寻宝的旅途中，一位探险家找到了一张古老的地图。地图上有一条指令，它要求探险家找到一个十位的宝藏代码，这个代码不仅包含从 0 到 9 的每一个数字恰好一次，且为 11 的倍数。最高位数字不为 0，以确保代码是一个有效的十位数。探险家知道可能存在多个这样的代码，于是他决定计算出所有合法代码中最大值与最小值之间的差值，以便破解最终的密码。

现在，他需要你的帮助来解决这个问题。请你计算出所有符合条件的十位数中，最大值与最小值的差。

# 格式

## 输入

无

## 输出

输出一个整数，表示所有符合条件的十位数中，最大值与最小值的差。


# 提示

**本题为填空题，只需要算出结果后，在代码中使用输出语句将结果输出即可。**
