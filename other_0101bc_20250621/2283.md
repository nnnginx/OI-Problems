# 问题描述

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

# 格式

## 输入

第一行一个整数n
第二行输入n个整数

## 输出

一个整数，表示能接多少雨水

# 样例1

![image](./2283/file/pZmZ4kyluEcrOV16e3xO0.png)

```input1
12
0 1 0 2 1 0 1 3 2 1 2 1
```

```output1
6
```
<pre>
<strong>解释：</strong>上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
</pre>

# 样例2

```input2
6
4 2 0 3 2 5
```

```output2
9
```
