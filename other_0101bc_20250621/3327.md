## 题目描述

小明用字母 A 对应数字 1，B 对应 2，以此类推，用 Z 对应 26。对于 27 及以上的数字，小明用两位或更长位的字符串来对应，例如 AA 对应 27，AB 对 应 28，AZ 对应 52，LQ 对应 329。

现在给你一个整数 n，请问 n 对应的字符串是什么？

## 输入格式

一个整数 n。

## 输出格式

一行字符串，表示数字 n 对应的字符串。

## 样例

输入1:

```input1
101
```

输出1:

```output1
CW
```

## 数据规模与约定

对于 30 的数据，1≤n≤1000；
对于 60 的数据，1≤n≤106；
对于 100 的数据，1≤n≤109。
