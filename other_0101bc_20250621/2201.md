# 问题描述

给定一个非负索引 `rowIndex`，返回「杨辉三角」的第 `rowIndex`*​ ​*行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。
![image](./2201/file/P1EwRO2Gxggh81o7un0mq.png)

# 格式

## 输入

输入一个正整数

## 输出

对应行数的杨辉三角格式

# 样例1

```input1
4
```

```output1
1 3 3 1
```

# 样例2

```input2
2
```

```output2
1 1
```


**提示:**

* `1 <= rowIndex <= 33`
