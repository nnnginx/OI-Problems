# 问题描述

给定一个非负整数 *`numRows`​，​*生成「杨辉三角」的前 *`numRows`* 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。
![image](./2200/file/6PIQEVQeOY4_Fxnq7UQZY.png)

# 格式

## 输入

输入一个正整数

## 输出

对应杨辉三角格式

# 样例1

```input1
5
```

```output1
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```

# 样例2

```input2
1
```

```output2
1
```

**提示:**

* `1 <= numRows <= 30`
