## 题目描述

一个素数，依次去掉它当前的最后一位数字后，剩余的数还是素数，直到剩余一位数时仍然是素数，那么这个素数就被称为完全系数。 例如:7331是素数，去掉最后一位数字1，733 也是素数，再去掉最后一位数字 3，73 也是系数，再去掉最后一位数字 3，7 也是素数，故 7331 是完全素数。 现给定两个整数 n 和 m，请计算 n 到m 之间(包括 n 和 m)有多少个完全素数?

## 输入

行包含两个整数 n 和 m ，整数之间 数据范围: 测试点1\~8:1<=n<=m<=10^4 测试点 9-10:1<=n<=m<=10^6

## 输出

一个整数，表示 n 到 m 之间（包括 n 和 m ）完全素数的个数。


```input1
20 30
```


```output1
2
```
