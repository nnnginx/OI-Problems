## 题目描述

输入两个不相等的四位正整数N（1000<=N<=9999）和M（1000<=M<=9999），其中N< M，中间以一个空格隔开，输出N与M之间（包含N和M）所有满足要求的正整数且正整数之间以一个英文逗号隔开。 要求每个正整数的各个位上的数都为偶数（注：0为偶数）

## 输入格式

在一行输入两个不相等的四位正整数N和M（N和M之间以一个空格隔开）

## 输出格式

输出N与M之间（包含N和M）所有满足要求的正整数且正数之间以一个英文逗号隔开

输入1:

```input1
4000 4008
```

输出1:

```output1
4000，4002，4004，4006，4008
```
