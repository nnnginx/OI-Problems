# 问题描述

实现字符串的压缩和解压缩功能。压缩规则如下：

* 对于连续重复的字符，将其替换为该字符和重复的次数。例如，字符串 `"aaabbbccd"` 压缩后变为 `"a3b3c2d1"`。
* 如果压缩后的字符串长度不小于原字符串长度，则输出原字符串。

同时，需要实现解压缩功能，将压缩后的字符串还原为原始字符串。

# 格式

## 输入

输入一串字符串

## 输出

第一行输出压缩

第二行输出解压

$注意$：始终都输出两行。

# 样例

```input1
aaabbbccd
```

```output1
a3b3c2d1
aaabbbccd
```

```input2
abcd
```

```output2
abcd
abcd
```

# 提示

### 数据范围

- 输入的字符串长度在 $100$ 以内。
