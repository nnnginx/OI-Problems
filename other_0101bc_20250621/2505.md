#### 题目描述

```
把一块m\*n米的土地分割成同样大的正方形，如果要求没有土地剩余，分割出的正方形土地最大边长是多少米？（最少不能少于1米\*1米）

如：一块6米 \* 4米的土地，能够分割的最大的正方形的边长为2米。（5.2.45）
```

![](https://oj.czos.cn:443/upload/image/20180623/20180623103608_24080.jpg)

#### 输入

两个整数m和n（m,n <= 1000000000000000000）

#### 输出

能够分割的最大正方形的边长

#### 样例输入              

```
6 4
```

#### 样例输出             

```
2
```
