## 题目描述

阿尔法星球有一个神奇的零件工厂， 这个工厂生产零件的过程很神奇。厂里共有 $n$ 个工人，分别编号为 $1 \sim n$ 号。部分工人之间存在传送带，可以双向传送零件。

零件的加工分为很多阶段，记作 $1$ 阶段、$2$ 阶段、 $3$ 阶段...

当第 $i$ 号工人准备加工的 $t~(t > 1)$ 阶段的零件时，**所有**与第 $i$ 号工人通过传送带**直接**连接的工人，都必须先加工 $t-1$ 阶段的零件，并传给第 $i$ 号工人（注意：第 $i$ 号工人自己**无需**进行 $t-1$ 阶段的加工）。

如果第 $i$ 号工人想加工 $1$ 阶段的零件时，**所有**与第 $i$ 号工人通过传送带**直接**连接的工人，都必须为第 $i$ 号工人提供一个原材料。

现有 $q$ 张工单，每张工单都指定了第 $x$ 号工人需要加工 $t$ 阶段的零件。

仔仔是这个工厂的第 $1$ 号工人。他想知道对于每张工单，是否需要他提供原材料。

## 输入格式

第一行为三个正整数 $n，m$ 与 $q$，即工人的个数、传送带的个数与工单的个数。

接下来 $m$ 行，每行为两个正整数 $u$ 与 $v$，即编号 $u$ 与编号 $v$ 的工人之间存在一条传送带。数据保证 $u \neq v$。

接下来 $q$ 行，每行两个正整数 $x$ 和 $t$，表示编号为 $x$ 的工人需要加工 $t$ 阶段的零件。

## 输出格式

共 $q$ 行，每行一个字符串 `Yes` 或者 `No`。如果对于任意一张工单，需要编号为 1 的仔仔提供原材料，输出 `Yes`；否则输出 `No`。

## 样例 #1

### 样例输入 #1

```
3 2 6
1 2
2 3
1 1
2 1
3 1
1 2
2 2
3 2
```

### 样例输出 #1

```
No
Yes
No
Yes
No
Yes
```

$1$ 号工人要加工 $1$ 阶段的零件，则 $2$ 号工人需要提供原材料。

$2$ 号工人要加工 $1$ 阶段的零件，则 $1$ 号与 $3$ 号工人需要提供原材料。

$2$ 号工人要加工 $2$ 阶段的零件，则 $1$ 号和 $3$ 号工人需要先加工 $1$ 阶段的零件，而 $2$ 号工人需要为两人提供原材料。

$3$ 号工人要加工 $2$ 阶段的零件，则 $2$ 号工人需要加工 $1$ 阶段的零件，而 $1$ 号和 $3$ 号工人需要提供原材料。

**数据规模与约定**
$1 \leq u, v, x \leq n$。
$1 \leq n, m, q \leq 10^5$，$1 \leq t \leq 10^9$。
