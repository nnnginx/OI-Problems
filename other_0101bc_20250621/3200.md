## 题目描述

一个长度为3的由大写英文字母组成的字符串T是一个由小写字母组成的字符串S的机场代码，T可以通过以下方法之一从S中得到：

1. 从S中取一个长度为3的子序列（不一定连续），并将其转换为大写字母形成T。
2. 从S中取一个长度为2的子序列（不一定连续），将其转换为大写字母，并在末尾添加"`X`"形成T。

## 输入格式

输入字符串S和T，判断T是否是S的机场代码。

## 输出格式

如果T是S的机场代码，输出"`Yes`"，否则输出"`No`"。

## 输入输出样例

### 输入样例1

```
narita
NRT
```

### 输出样例1

```
Yes
```

### 输入样例2

```
losangeles
LAX
```

### 输出样例2

```
Yes
```

### 输入样例3

```
snuke
RNG
```

### 输出样例3

```
No
```
