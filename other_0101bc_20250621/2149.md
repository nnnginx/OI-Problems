# 说明

给定一个不含重复数字的数组 $a$ ，返回其 *所有可能的全排列* 。请按照 **数组下标顺序** 输出答案。

# 输入

第一行一个整数 $n$；
第二行 $n$ 个整数 $a_i$。

# 输出

若干行，每行 $n$ 个整数。

# 样例

```input1
3
5 1 4
```

```output1
5 1 4
5 4 1
1 5 4
1 4 5
4 5 1
4 1 5
```

```input2
2
0 1
```

```output2
0 1
1 0
```

```input3
1
1
```

```output3
1
```

# 提示

#### 数据范围

* $1 \leq n \leq 6$
* $-10 \leq a_i \leq 10$
* $a$ 中的所有整数 **互不相同**
