# 说明

给你一个数组 $a$ 和一个值 $m$，你需要 **原地删除** 所有数值等于 $m$ 的元素，即不可使用额外的空间。

# 输入

第一行两个整数 $n, m$；
第二行 $n$ 个整数 $a_i$。

# 输出

第一行一个整数 $k$；
第二行为若干个整数，即数组 $a_i$ 删除所有等于 $m$ 的元素后的元素。

# 样例

```input1
4 3
3 2 2 3
```

```output1
2
2 2
```

```input2
8 2
0 1 2 2 3 0 4 2
```

```output2
5
0 1 3 0 4
```

# 提示

#### 数据范围

* $0 \leq n \leq 100$
* $0 \leq a_i \leq 50$
* $0 \leq m \leq 100$
