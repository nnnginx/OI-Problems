# 问题描述

定义结构体Student，包含姓名（字符串）、年龄（整数）、成绩（浮点数）。

* 输入n个学生的信息，按以下规则排序后输出：
  1. 按成绩降序排列；
  2. 成绩相同则按年龄升序排列；
  3. 年龄相同则按姓名字典序升序排列。

# 格式

## 输入

第一行一个整数n
接下来n行学会信息

## 输出

排序后的学生信息

# 样例

```input1
3
Alice 20 90.5
Bob 19 90.5
Charlie 22 85.0
```

```output1
Bob 19 90.5
Alice 20 90.5
Charlie 22 85.0
```
