## 问题背景

小杨有一个包含n个正整数的序列a。他认为一个序列是平衡的当且仅当存在一个整数i (1 ≤ i < n) 使得序列第1个到第i个数字的总和等于第i+1个到第n个数字的总和。

小杨想判断所给的序列是否是平衡的。

---

## 问题描述

给定一个正整数t，代表测试用例组数。

对于每组测试用例，第一行包含一个正整数n，代表序列的长度。

第二行包含n个正整数，代表序列a。

---

## 格式

### 输入

第一行包含一个正整数t，代表测试用例组数。

接下来是t组测试用例，每组测试用例包含两行。

- 第一行包含一个正整数n，代表序列的长度。
- 第二行包含n个正整数，代表序列a。

### 输出

对于每组测试用例，如果序列a是平衡的，输出Yes，否则输出No。

---

## 样例

### 输入数据 1

```
3  
3  
1 2 3 
4  
2 3 1 4  
5  
1 2 3 4 5
```

### 输出数据 1

```
Yes 
Yes 
No
```

---

## 提示

对于第一个测试用例，令i = 2，则有1 + 2 = 3，因此序列是平衡的。

对于第二个测试用例，令i = 2，则有2 + 3 = 1 + 4，因此序列是平衡的。

对于第三个测试用例，不存在满足要求的i。

| 子任务编号 | 数据量占比 | n | ai |
|--|--|--|--|
| 1 | 30% | n ≤ 100 | ai ≤ 10000 |
| 2 | 30% | n ≤ 100 | ai ≤ 10000 |
| 3 | 40% | n ≤ 10000 | ai ≤ 10000 |

对于全部数据，保证：
1 ≤ t ≤ 100，1 ≤ n ≤ 10000，ai ≤ 10000
