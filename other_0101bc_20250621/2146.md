# 说明

给定一个字符串 $s$ ，你需要反转字符串中每个单词的字符顺序，同时仍保留空格和单词的初始顺序。

# 输入

一行字符串 $s$。

# 输出

一行字符串。

# 样例

```input1
Let's take LeetCode contest
```

```output1
s'teL ekat edoCteeL tsetnoc
```

```input2
Mr Ding
```

```output2
rM gniD
```

# 提示

#### 数据范围

* $1 <= s.length <= 5 * 10^4$
* $s$ 包含可打印的 **ASCII** 字符。
* $s$ 不包含任何开头或结尾空格。
* $s$ 里 **至少** 有一个词。
* $s$ 中的所有单词都用一个空格隔开。
