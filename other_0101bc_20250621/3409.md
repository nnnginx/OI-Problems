## 题目描述

输入 $n$ 个不大于 $10^6$ 的正整数。要求全部储存在数组中，去除掉不是质数的数字，依次输出剩余的质数。

## 输入格式

第一行输入一个正整数 $n$，表示整数个数。

第二行输入 $n$ 个正整数 $a_i$，以空格隔开。

## 输出格式

输出一行，依次输出 $a_i$ 中剩余的质数，以空格隔开。

## 输入输出样例 #1

### 输入 #1

```input1
5
3 4 5 6 7
```

### 输出 #1

```output1
3 5 7
```

## 说明/提示

数据保证，$1\le n\le10^6$，$1 \leq a_i \leq 10^6$。
