# 问题描述

有一个 M 行 N 列的点阵，相邻两点可以相连。

一条纵向的连线花费一个单位，一条横 向的连线花费两个单位。

某些点之间已经有连线了，试问至少还需要花费多少个单位才能使 所有的点全部连通

# 格式

## 输入

第一行输入两个正整数 m 和 n。
以下若干行每行四个正整数 x1,y1,x2,y2，
表示第 x1 行第 y1 列的点 和 第 x2 行第 y2 列的 点已经有连线。输入保证|x1-x2|+|y1-y2|=1。

## 输出

输出使得连通所有点还需要的最小花费。

# 样例

```input1
2 2
1 1 2 1
```

```output1
3
```

# 提示

数据规模 ：

30%数据：n\*m<=1000
100%数据：m,n<=1000
