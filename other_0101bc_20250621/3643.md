- $1+1=?$ 显然是 $2$。

_以上纯属个人吐槽_

给定一个正整数 $n$，求将其分解成若干个素数之和的方案总数。

## 输入格式

一行一个正整数 $n$。

## 输出格式

一行一个整数表示方案总数。

## 输入输出样例 #1

### 输入 #1

```
7
```

### 输出 #1

```
3
```

## 输入输出样例 #2

### 输入 #2

```
20
```

### 输出 #2

```
26
```
