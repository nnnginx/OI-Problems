## 题目描述

题目描述:2020 年 2月，小蓝参加“蓝桥杯大赛青少年创意编程 C++组”选拔赛。在 一个这么“2”的时间里参赛，小蓝一时高兴，忍不住在键盘上敲出了一个会写“2”的 程序。

## 输入格式

一个整数 n。

## 输出格式

一个由“\*”组成的长、宽都是 n 的“2”字图形,具体请参见样例。

输入1:

```input1
5
```

输出1:

```output1
*****
   *
  *
 *
*****
```

输入2:

```input2
8
```

输出2:

```output2
********
      *
     *
    *
   *
  *
 *
********
```

## 数据规模与约定

对于 100 的数据，3\\l3b≤100。
