#### 题目描述

根据参数，画直角三角形。

#### 输入

输入一行，包括3个参数，第一个参数是正整数n，代表直角三角形底边的字符个数与高的行数（3 <= n <= 10）；第二个参数是一个字符（“\*”或者“#”），表示用来画图的符号；第三个参数是1或0（“1”代表实心，“0”代表空心）。每两个参数之间用一个空格隔开。

#### 输出

按给定的参数输出对应图形。

注意事项：

图案的第一行前面不能有多余的空行；

图案的最左侧从第一列开始输出，左边不能有多余的空格。

#### 样例输入 **复制**

```
5 * 0
```

#### 样例输出 **复制**

```
*
**
* *
*  *
*****
```
