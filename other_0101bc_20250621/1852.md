# 问题描述

**斐波那契数** （通常用 `F(n)` 表示）形成的序列称为 **斐波那契数列** 。该数列由 `0` 和 `1` 开始，后面的每一项数字都是前面两项数字的和。也就是：

```
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```

给定 `n` ，请计算 `F(n)` 。

# 格式

## 输入

一个整数 `n`。

## 输出

一个整数，即 `F(n)`。

# 样例

```input1
2
```

```output1
1
```

```input2
3
```

```output2
2
```

```input3
4
```

```output3
3
```

# 提示

### 样例说明

样例一：`F(2) = F(1) + F(0) = 1 + 0 = 1`
样例二：`F(3) = F(2) + F(1) = 1 + 1 = 2`
样例三：`F(4) = F(3) + F(2) = 2 + 1 = 3`

### 数据范围

* $0 \leq n \leq 30$
