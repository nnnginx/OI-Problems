## 题目背景

仔仔在玩《纸片马里奥RPG》这款游戏，他打算把这个游戏的支线任务按顺序全部做掉。

## 题目描述

主角马里奥的等级 $m$ 开始为 $0$ ，生命值上限始终是 $10$，并且在一开始是满的。
而支线任务总共有 $n$ 件，马里奥从第一件开始按顺序往下做，支线任务会消耗 $x$ 点生命值。
进行支线任务时要先计算付出的生命值，如果小于等于 $0$，则游戏结束，当前任务就算作失败，并且也无法完成接下来的任务。不过有些任务能回复一些生命值，也就是说 $x$ 有可能是负数，但生命不会回复到 $10$以上。
每个任务可以获得 $a$ 经验值，经验值如果大于等于当前升级所需的经验值则可以升级，并且超过的经验值可以用于下个等级的升级。

每个等级升级所需的经验如下：

马里奥的等级为 $m$ 。当经验值大于 $2^m$ 时即可消耗对应的经验值升级，等级变为 $m+1$ 。

下面举两个例子：

- 马里奥当前等级为 $2$，剩余经验值为 $2$，在一次任务后得到了 $15$ 点经验值，那么总经验值为 $17$，花费了 $2^2=4$ 点经验值升到了 $3$ 级，还剩下 $13$ 点经验值，于是继续花费 $2^3=8$ 点经验值升到了 $4$ 级，剩余 $5$ 点经验值，不足以支付当前的升级费用（ $2^4=16$）。

## 输入格式

第一行一个正整数 $n$，表示任务个数。

接下来 $n$ 行，每行一个实数 $x$ 和一个整数 $a$，分别为任务消耗的生命值和获得的经验值。

## 输出格式

一行，为等级和经验值。

## 样例 #1

### 样例输入 #1

```
2
5.5 15
4.5 24
```

### 样例输出 #1

```
4 0
```

## 提示

【数据范围】
对于 $100\%$ 的数据，$1\le n \le 20$，$1\le a \le 10^9$。
