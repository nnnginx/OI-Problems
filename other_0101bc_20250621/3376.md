# 题目描述

假设现在是y年m月d日h时而k小时后是y'年m'月d'日h'时，对于给定的y,m,d,h,k，小杨想请你帮他计算出对应的y',m',d',h'是多少。

# 格式

## 输入

输入包含五行，每行一个正整数，分别代表y,m,d,h,k。

## 输出

输出四个正整数，代表y',m',d',h'。

# 样例

```input1
2008 
2 
28 
23 
1
```

```output1
2008 2 29 0
```

# 提示

闰年判断规则

普通闰年：年份能被 4 整除，但不能被 100 整除。

世纪闰年：年份能被 400 整除。

满足以上任意一条规则的年份就是闰年，否则是平年。
