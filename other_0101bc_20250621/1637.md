# 问题描述

本例函数接收数组作为输入，由于函数只能返回一个结果，因此，可以给该函数添加3个指针类型参数，分别接收来自主调函数的最大值、最小值和平均值变量的地址，在函数内部就可以通过间址运算符去访问函数外部的最大值、最小值和平均值变量。
这样，函数本身可以不返回结果，而通过以上三个指针来直接操作主调函数中的三个结果变量。
数组为：{2,4,1,9,0,8,6,5,3,7}

# 格式

## 输入

无

## 输出

数组的最大值、最小值和平均值

# 样例

```input1

```

```output1
max=9,min=0,avg=4.50
```
