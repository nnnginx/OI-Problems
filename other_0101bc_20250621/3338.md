# 拼写单词

## 题目描述

四种水果的英文单词。分别是Apple、Banana、Orange、Pear。老师通过提示每个单词的首字母，让同学们将对应的单词拼写下来。

请编写程序：

当输入的大写字母为 "A" 时，则输出 "Apple" ；

当输出的大写字母为 "B" 时，则输出 "Banana" ；

当输出的大写字母为 "O" 时，则输出 "Orange" ；

当输出的大写字母为 "P" 时，则输出 "Pear" ；

## 输入格式

输入A、B、O、P中任意一个大写字母

## 输出格式

根据输入的大写字母输出对应的英文单词

## 样例 #1

### 样例输入 #1

```
A
```

### 样例输出 #1

```
Apple
```
