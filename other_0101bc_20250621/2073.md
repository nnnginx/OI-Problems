

# 问题描述

给定一个整数数组`prices`，其中第 *​ ​*`prices[i]` 表示第 `i` 天的股票价格 。​

设计一个算法计算出最大利润。在满足以下约束条件下，你可以尽可能地完成更多的交易（多次买卖一支股票）:

* 卖出股票后，你无法在第二天买入股票 (即冷冻期为 1 天)。

**​注意：​**你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

# 格式

## 输入

第一行一个整数 $n$，表示有多少天。
第二行 $n$ 个整数，表示每天的股票价格。

## 输出

一个整数，表示最大的利润。

# 样例

```input1
5
1 2 3 0 2
```

```output1
3
```

```input2
1
1
```

```output2
0
```

# 提示

#### 样例一说明

对应的交易状态为: [买入, 卖出, 冷冻期, 买入, 卖出]

#### 数据范围

* `1 <= n <= 5000`
* `0 <= prices[i] <= 1000`
