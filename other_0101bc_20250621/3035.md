# 题目描述

“对对碰”是一个以消除为目标的休闲小游戏。游戏中有几种不同造型的动物卡片，如下图所示。

![](https://cspoj.com/upload/cspoj.com/20240219//3101_1_20240219161345_34303.png)

一排动物卡片，只要相邻的两个卡片相同，就双双被消除。

一对动物卡片被消除后，两边的其他动物卡片（如果有的话）会占据它们留下的空位，使得原本相同但不相邻的两个动物卡片变为相邻，消除会连续进行。

如果连续3个以上动物卡片相同，且数量是奇数的，只能两两消除，剩下1个不能消除。

下图是例图消除后的情况。
![](https://cspoj.com/upload/cspoj.com/20240219//3101_2_20240219161345_70649.png)

用大写字母A\~F代表6种不同的动物卡片，输入一个由A\~F组成的字符串，代表消除前排成一排的所有动物卡片，输出消除后剩余的动物卡片数量

# 格式

## 输入

一个字符串，长度小于20，只包含大写A\~F字母，代表消除前的动物卡片。

## 输出

一个整数，为消除后剩余的动物卡片数量。如果所有动物卡片都被消除，输出0。

# 样例

```input1
ACBBCDBBBC
```

```output1
4
```
