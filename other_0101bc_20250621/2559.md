#### 题目描述

求一个数列的所有前缀最小值之和。
即：给出长度为n的数列a[i],求出对于所有1<=i<=n，min(a[1],a[2],...,a[i])的和。
由于读入较大，数列由随机种子生成。
其中a[1]=x，a[i]=(379\*a[i-1]+131)%997。

#### 输入

一行两个正整数n,x，分别表示数列的长度和随机种子。(n<=100000,x<997)

#### 输出

一行一个正整数表示该数列的前缀最小值之和。

#### 样例输入 

```
5 666
```

#### 样例输出 

```
1650
```

#### 提示

数列为{666,304,692,188,596}，前缀最小值为{666,304,304,188,188},和为1650。
