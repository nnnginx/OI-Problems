# 题目描述

小 A 有一个n行m列的矩阵A 。

小 A 认为一个2 * 2的矩阵D是好的，当且仅当![image](./3364/file/3AtUCikjTpaAz5I3MbQER.png) 。其中![image](./3364/file/TZrPayffJm7rt53k3JzZJ.png)

表示矩阵D的第i行第j列的元素。

小 A 想知道 中有多少个好的子矩阵。

# 格式

## 输入

第一行，两个正整数n,m。

接下来n行，每行 个整数![image](./3364/file/Q5QYB_fu-c8h63a8WQPBz.png)

## 输出

一行，一个整数，表示A中好的子矩阵的数量。

# 样例

```input1
3 4 

1 2 1 0 

2 4 2 1 

0 3 3 0
```

```output1
2
```
