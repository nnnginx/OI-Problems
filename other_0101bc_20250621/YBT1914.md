# 问题描述

程序设计对抗赛设有 N（0<N≤50 的整数）个价值互不相同的奖品，每个奖品的价值分
别为 S1，S2，S3……Sn（均为不超过 100 的正整数） 。现将它们分给甲乙两队，为了使得甲
乙两队得到相同价值的奖品，必须将这 N 个奖品分成总价值相等的两组。
编程要求：对给定 N 及N 个奖品的价值，求出将这 N 个奖品分成价值相等的两组，共有
多少种分法？
例如：N = 5，S1，S2，S3……Sn 分别为 1，3，5，8，9
则可分为{1，3，9}与{5，8}
仅有 1 种分法；
例如：N = 7，S1，S2，S3……Sn 分别为 1，2，3，4，5，6，7
则可分为：
{1,6,7}与{2,3,4,5}
{2,5,7}与{1,3,4,6}
{3,4,7}与{1,2,5,6}
{1,2,4,7}与{3,5,6}
有 4 种分法。

# 格式

## 输入

输入文件中包含 N 及S1，S2，S3……Sn。 （每两个相邻的数据之间有一个空格隔开）。

## 输出

输出文件包含一个整数，表示多少种分法的答案，数据若无解，则输出 0。

# 样例

```input1
7
1 2 3 4 5 6 7
```

```output1
4
```
