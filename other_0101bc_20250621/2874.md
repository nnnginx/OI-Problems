
# 问题描述

n个小朋友站成一列，从前到后编号分别为1、2、3、...、n，老师准备了一些水果糖，从编号为1的小朋友开始分发，老师规定每个小朋友一次只能领取一颗水果糖，如果这个小朋友已经领取到了想要的数量的水果糖，他就会离开队伍，否则，他会回到队尾继续排队。 给定每个小朋友想要的水果糖数量，请计算，老师发完第几颗水果糖后，编号为k的小朋友会离开队伍。

## 输入

第一行，一个整数n，表示小朋友的人数； 第二行包含n个整数a1，a2，a3，...，an，分别表示每个小朋友想要领取的水果糖数量； 第三行，一个整数k，表示要查询的小朋友的编号。

数据范围 测试点1\~10：1≤n≤10000，1≤ai≤100，1≤k≤n。

## 输出

一个整数，表示老师发完第几颗水果糖后，编号为k的小朋友会离开队伍。

## 样例

```input1
5
4 1 3 5 2
3
```

输出1:

```output1
11
```
