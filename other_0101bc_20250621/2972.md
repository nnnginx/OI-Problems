## 题目描述

在编程中，我们时常需要考虑时间复杂度，特别是对于循环的部分。 例如：如果代码中出现1层循环： for (i=1; i<=n; i++) op; 总共会执行n次op运算；

如果代码中出现2层循环： for (i=1; i<=n; i++) for (j=i+1;j<=n;j++) op; 总共会执行n\*(n-1)/2次op运算。

给定两个整数m，n，分别表示for循环的层数和每一层循环的终止值，且每层for 循环中循环变量的起始值是上一层循环变量的起始值+1（最外层循环变量的起始值是1），终止值都是n。请计算最后一共执行了多少次op运算。 注：每一层的for循环中无break等类似跳出、终止循环的操作。

## 输入

一行包含两个整数m、n，分别表示for循环的层数和每一层循环的终止值，整数之间以一个空格隔开。

数据范围 测试点1\~10：1≤m≤2000，1≤n≤2000。

## 输出

一个整数，表示执行的op运算次数对1007取模的结果。


```input1
2 3
```


```output1
3
```


```input2
4 6
```


```output2
15
```


```input3
3 7
```


```output3
35
```
