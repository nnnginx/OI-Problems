# 说明

给你一个正整数 $n$ 。如果 $n$ 是一个完全平方数，则输出 `YES` ，否则输出 `NO` 。

**完全平方数** 是一个可以写成某个整数的平方的整数。换句话说，它可以写成某个整数和自身的乘积。

不能使用任何内置的库函数，如  `sqrt` 。

# 输入

一行一个整数 $n$；

# 输出

`YES` 或者 `NO` 。

# 样例

```input1
16
```

```output1
YES
```

```input2
14
```

```output2
NO
```

# 提示

#### 数据范围

* $1 \leq n \leq 2^{31} - 1$
