# 说明

给你长度为 $n$ 的数组 $nums$，下标从 $0$ 开始）。请你构建一个 **同样长度** 的数组 $ans$，其中，对于每个$i$（$0 \le i < n$），都满足 $ans[i] = nums[nums[i]]$。输出构建好的数组 $ans$。

$nums$ 是一个由 $0$ 到 $n - 1$ 的不同整数组成的数组。

# 输入格式

第一行一个整数 $n$；
第二行 $n$ 个整数。

# 输出格式

一行 $n$ 个整数。

# 样例

```input1
6
0 2 1 5 3 4
```

```output1
0 1 2 4 5 3
```

```input2
6
5 0 1 2 3 4
```

```output2
4 5 0 1 2 3
```

# 提示

## 数据范围

* $1 <= n <= 1000$
* $0 <= nums[i] < n$
* $nums$ 中的元素 **互不相同**

**​进阶：​**你能在不使用额外空间的情况下解决此问题吗（即 `O(1)` 内存）？
