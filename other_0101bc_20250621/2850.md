#### 题目描述

柯南大学毕业后工作了一年，想分析一下这一年的收入状况。

请你帮助他设计一个程序，通过过去12个月中每月的收入，计算平均月收入。

#### 输入

输入一行，包含12个正整数，表示每个月的收入（单位：元），每个数都不大于20,000。两个数之间以空格分隔

#### 输出

输出一个正整数，为这12个月的平均月收入（单位：元；计算结果向下取整）。

#### 样例输入 **复制**

```
6000 6500 6000 6000 7000 6500 6500 6600 5800 5800 6500 12000
```

#### 样例输出 **复制**

```
6766
```
