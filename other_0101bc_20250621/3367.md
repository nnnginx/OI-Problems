# 题目描述

图书馆里有n本书，不幸的是，还混入了一只老鼠，老鼠每x小时能啃光一本书，假设老鼠在啃光一本书之前，不会啃另一本。请问y小时后图书馆里还剩下多少本完整的书。

# 格式

## 输入

三行，第一行一个正整数n，表示图书馆里书的数量；

第二行，一个正整数x，表示老鼠啃光一本书需要的时间；

第三行，一个正整数y，表示经过的总时间；

输入数据保证y小时后至少会剩下一本完整的书。

## 输出

一行，一个整数，表示y小时后图书馆里还剩下多少本完整的书。

# 样例

```input1
10
2
3
```

```output1
8
```

```input2
5
2
4
```

```output2
3
```
