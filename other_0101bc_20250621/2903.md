

## 题目描述

给定包含n个节点的二叉树，节点编号从1到n，且根节点编号为1，请输出二叉树交错式层序遍历的结果。 交错式层序遍历表示从左到右遍历第一层节点，再从右到左遍历第二层节点，再从左到右遍历第三层节点，以此类推。

## 输入

第一行包含一个整数n，表示二叉树的节点数量； 接下来n行，每行包含3个整数x，y，z，分别表示二叉树中每个节点编号，及其对应的左子节点和右子节点的编号（0表示对应子节点为空），整数之间以一个空格隔开。

数据范围 测试点1\~10：1≤n≤1000，1≤x,y,z≤n。

## 输出

一行包含n个整数，表示二叉树交错式层序遍历的结果，整数之间以一个空格隔开。

## 样例

```input1
6
1 2 3
2 4 0
3 6 5
4 0 0
5 0 0
6 0 0
```

```output1
1 3 2 4 6 5
```
