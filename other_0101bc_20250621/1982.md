给你一个下标从 **0** 开始的字符串数组 `words` 和一个字符 `x` 。

请你返回一个 **下标数组** ，表示下标在数组中对应的单词包含字符 `x` 。

## 输入

一个n表示要输入的字符串格式

对n个字符串进行输入

x匹配字符

## 输出

字符数组中匹配的字符串对应的下标

# 格式

```input1
2
0101 code
e
```

```output1
1
```

```input2
4
abc bcd aaaa cbc
a
```

```output2
0 2
```

**提示：**

* `1 <= words.length <= 50`
* `1 <= words[i].length <= 50`
* `x` 是一个小写英文字母。
* `words[i]` 只包含小写英文字母。
