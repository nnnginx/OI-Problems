## 题目描述

提示信息： 完全二叉树具备以下特点： 1.除去最后一层节点后就是满二叉树； 2.最下层的叶子节点一定集中于左部连续位置。 如下图所示： ![](https://cspoj.com/upload/oj.cspoj.com/20241007/bh_0LY3-8yBX1rW8--koK.png)

题目描述 给定一个完全二叉树的节点总数，请输出这个完全二叉树最后一层的节点个数。

## 输入

输入 一个整数n，表示完全二叉树的节点总数。

数据范围 测试点1\~10：1≤n≤10000。

## 输出

一个整数，表示输入的完全二叉树最后一层的节点个数。

## 样例1

```input1
7
```

```output1
4
```

## 样例2

```input2
32
```


```output2
1
```
