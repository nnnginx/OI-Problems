<h2>说明</h2>

&nbsp; 编写一个函数 print (n&#44;ch)，表示打印一行 n 个英文字母 ch，并换行。然后，在函数 main() 中输入 n 和 ch，调用函数 print() 打印一个字符三角形。
<h2>输入格式</h2>

一行一个整数 n 和一个英文字母 ch，之间用一个空格隔开，1≤n≤20。
<h2>输出格式</h2>

n 行，第 i 行有 i 个字母 ch。
<h2>样例</h2>
<pre><code class="language-input1">3 a</code></pre><pre><code class="language-output1">a
aa
aaa
</code></pre>
