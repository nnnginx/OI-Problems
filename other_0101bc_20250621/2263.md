# 说明

有 $n$ 个城市，按从 $0$ 到 $n-1$ 编号，城市间共有 $m$ 条双向道路。给出一个整数 $k$，表示距离阈值。

请找出路径距离在为 $k$ 以内可到达城市最少的城市。如果有多个这样的城市，则返回编号最大的城市。

注意，连接城市 $i$ 和 $j$ 的路径的距离等于沿该路径的所有边的权重之和。

# 输入格式

第一行三个整数 $n,m,k$，即城市数量、道路数量以及距离阈值；
接下来 $m$ 行，每行三个整数 $u,v,w$，即 城市 $u,v$ 之间有一条长度为 $w$ 的双向道路。

# 输出格式

一行一个整数，即答案城市的编号。

# 样例

```input1
4 4 4
0 1 3
1 2 1
1 3 4
2 3 1
```

```output1
3
```

```input2
5 6 2
0 1 2
0 4 8
1 2 3
1 4 2
2 3 1
3 4 1
```

## 数据范围

* $2 <= n <= 100$
* $1 <= m <= n * (n - 1) / 2$
* $0 <= u_i, v_i < n$
* $1 <= w_i, k <= 10^4$
* 所有 $(u_i, w_i)$ 都是不同的。
