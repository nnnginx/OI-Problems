# Description
两个球队的支持者要一起坐车去看球，他们已经排成了一列。我们要让他们分乘若干辆巴士，同一辆巴士上的人必须在队伍中是连续的。为了在车上不起冲突，希望两队的支持者人数尽量相等，差至多是 D。有一个例外，就是一辆车上的人全部都是一个球队的支持者。问要将这 N 个人全部送至球场，至少要几辆巴士。 

# Format

## Input
第一行是整数 N 和 D，1 ≤ N ≤ 2500，1 ≤ D ≤ N。

接下来的 N 行，按排队的顺序，描述每个人支持的球队，用 H 或 J 表示。 

## Output
至少要几辆巴士。

```input1
14 3
H
J
H
H
H

H
J
H
H
H

H
```

```output1
2
```
