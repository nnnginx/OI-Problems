#### **题目描述**

输入 $1$ 个整数 $N$，输出 $N$ 行的“*”组成的金字塔。第 $1$ 行 $1$ 个’*’，第 $2$ 行 $3$ 个’*’，第 $3$ 行$ 5$ 个’*’，…。例如：$N=4$ 时：

```
___* 

__*** 

_***** 

*******
```

(注：上面使用下划线’_’表示空格，以避免看不清造成误解)

#### **输入**

第一行$ 1 $个正整数：$N$，范围在$[1,20]$。

#### **输出**

$N $行的”*”号金字塔。

#### **样例输入**

```
3
```

#### **样例输出**

```
  *
 ***
*****
```
