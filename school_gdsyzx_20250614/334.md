#### **题目描述**

明明今天接到一个新任务：体育老师拿到一份成绩登记表，想请明明统计每个成绩等级（$A、B、C、D$）的人数各是多少。明明觉得很容易：对读入的分数进行判断统计，就解决问题了。但是当他拿到成绩表时，立刻就傻了。原来体育老师是用$A、B、C、D$来登记成绩的：$A \ D\  B\  A\  C\  B\  A……$ 那在程序里$A、B、C、D$该怎么表示呢？

#### **输入**

第一行输入一个整数$N$，它的范围是$[1..50]$； 第二行输入$N$个$A$\~$D$的字符。

#### **输出**

按$A$\~$D$的顺序每行输出它对应的个数。

#### **样例输入**

```
10 
A D B A C B A A B C
```

#### **样例输出**

```
A:4 
B:3 
C:2 
D:1
```
