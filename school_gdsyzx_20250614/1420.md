# Description
现在我们都知道信息学在广东实验中学里是金牌项目，但是，也许你不知道在很久之前我们其实是小透明。

学校现在有意向将信息技术科组分裂成信息学和电教组，这当然是不行的！我们是一个团体，并且分裂是一件很麻烦的事情，所有的设施必须对半平分。

首先，我们对所有的设施进行评估，如果两项设施的价值相同，我们认为它们是一样的设施。假设有 N（0 ＜ N ≤ 50）种设施（不同的种类，不同的价值）。

# Format

## Input
输入包含多个测试案例。

每个测试案例，第一行为数字 N（0 ＜ N ≤ 50），表示不同设施的种类数。

接下来 N 行，分别包含一个整数 V（0 ＜ V ≤ 50），表示设施的价值，以及整数 M（设施对应的数量，0 ＜ M ≤ 50），你可以假设所有的 V 都是不同的。

当 N 为负数时输入结束。

## Output
对于每组案例，输出一行，两个整数A 和 B，分别表示信息学组和电教组获得的价值。请尽量使得 A 和 B 相等，同时保证 A 不小于 B。

# Samples

```input1
2
10 1
20 1
3
10 1 
20 2
30 1
-1
```

```output1
20 10
40 40
```
