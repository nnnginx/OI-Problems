# Description
小明发明了一个数字对撞机，两个位数相同的整数可以进行碰撞。碰撞过程中，将两个整数的每一位进行比较，较小的那个数字会被撞得粉碎，较大的数字保留下来（如果两数相同，都会保留）。例如下面例子中：

![img](./1384/file/0001365数字对撞机.png)

两个整数 13570 和 21673 碰撞后，对应数位上较小的值已经消失，碰撞的结果为：第一个数字剩下 37，第二个数字剩下 2673。

现在小明想让你写一个程序来显示数字碰撞机的结果，输入两个整数，输出碰撞后的两个数字。（注意最终结果不能包含多余的前导 0）

# Format

## Input
第一行一个整数 x，表示输入的第一个整数。

第二行一个整数 y，表示输入的第二个整数。

## Output
输出包含两行，第一个表示 x 碰撞后的结果。

第二行表示 y 碰撞后的结果。

若 x 或者 y 所有数位上的数字都消失了，该行输出“BOOM”，注意都是大写的。

# Samples

```input1
13570
21673
```

```output1
37
2673
```
```input2
300
500
```

```output2
0
500
```
```input3
1234
5678
```

```output3
BOOM
5678
```

# Explanation

样例二说明：

300 第一位被撞碎了，剩下 00，因为不能包含前导 0，输出 0。500 没有任何一位被撞碎。

样例三说明：

1234 每一位都被撞碎了，输出“BOOM”，5678 没有任何一位被撞碎。

# Limitation
对于 50% 的数据，0 ≤ x，y ＜ $10^9$；

对于 80% 的数据，0 ≤ x，y ＜ $10^{100}$，即 x，y 的长度不超过 100。

对于 100% 的数据， 0 ≤ x，y ≤ $10^{1000}$，即 x，y 的长度不超过 1000。保证所有 x 和 y 的位数相同，且 x，y 本身没有多余的前导 0。
