# 燃烧木棍

### 【题目描述】

Tom是调皮的孩子，特别喜欢玩火，现在他手上有若干根长度分别为$1$和$\sqrt{2}$的木棍，还有一张不能燃烧的平板，他把平板划分成长度为1的单元格，并标上座标。然后按以下规则把平板上的木棍组成一个连通图：木棍的两端必须放在单元格的顶点上。即长度为1的木棍放在单元格的某一边上，长度为$\sqrt{2}$的木棍放在单元格的对角线上。

Tom的点火规则是，只能从木棍的两端点火，而不能从木棍的中间点火。如图，不能在A点点火，但在C点或B点点火都是充许的。点火后，火会沿着木棍向前燃烧（每根都有自己的燃烧速度），并能点燃与它相接的其它木棍。
![image](./1110/file/VsB-i_XxqQIcL5oS3cg3S.png) 

**任务：**写一程序计算从哪里开始点火，使得燃烧的总时间最短，输出最短时间。

### 【输入格式】

第一行为一个正整数$N$，表示组成图形的木棍数目，后面共有$N$行，每行$5$个数：$ X_1\  Y_1\  X_2\  Y_2\ T$,其中$（X_1, Y_1）$和$（X_2, Y_2）$分别表示木棍两端的坐标，$T$表示木棍燃烧时间，是指从木棍的某一端点火燃烧到别一端，燃完所需的时间。

### 【输出格式】

输出是一个保留$4$位小数的实数，表示所有木棍完全燃烧的最少时间。

### 【样例输入1】

```
1
0 0 1 1 1
```

### 【样例输出1】

```
1.0000
```

### 【样例解释1】

从任一端点火都行，燃烧时间都是1

### 【样例输入2】

```
5
0 0 0 1 1
1 0 0 1 10
0 0 1 0 1
0 0 1 1 1
2 2 1 1 1
```

### 【样例输出2】

```
3.2500
```

### 【样例解释2】

在$ (0,0$)位置点火

木棍 $1, 3$ 和$ 4$ 将被点燃，燃烧$0.5$分钟后，木棍$2$将被从中间点燃向两端燃烧，再过$0.5$分钟，木棍$1, 3, 4 $将被完全燃烧，木棍$5$ 将被点燃并在$1$分钟后燃烧完 (比木棍$2$早燃完)。

木棍$2$从中间向两端燃烧$0.5$分钟以后，变成两小段，每段的燃烧时间是$4.5$ 分钟。但因为此时两小段木棍的另一端也同时被点燃，燃烧速度变成原来的两倍，还需$2.25$ 分钟的燃烧时间， 所以总时间： $1 + 2.25 = 3.25$

### 【样例输入3】

```
3
1 1 1 2 10
1 2 2 2 10
1 1 2 2 50
```

### 【样例输出3】

```
35.0000
```

### 【样例解释3】

在$ (1,2)$位置点火， 木棍$(1\ 1, 1\ 2) $和$(1\  2, 2\  2)$将燃烧 $10$ 分钟。. 最后一根木棍在$10$分钟后从两端被点燃，燃烧时间为$25$分钟。

### 【数据范围】

$1≤n≤40$
