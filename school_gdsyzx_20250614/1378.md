# Description
扑克牌有 13 种代表不同点数的牌（不考虑花色），如下图所示。从左到右依次为“A”，“2”，“3”，“4”，...，“10”，“J”，“Q”，“K”。

![img](./1378/file/扑克牌游戏1.png)

小华正在玩一个扑克牌的游戏，在这个游戏中，每种点数的牌都有一个分数（不一定跟点数相同）。现在小华手上已经有 n 张扑克牌，它还可以挑选 m 张扑克牌，使得 n + m 张扑克牌的总分数最大。我们假定每种点数的扑克牌有无穷多张。

请编程计算小华在游戏中可以最多获得多少分？

# Format

## Input
共 3 行。

第 1 行 13 个整数，依次表示每种点数的牌所代表的分数。

第 2 行两个整数 n 和 m，表示小华已经有 n 张扑克牌，还可以挑选 m 张扑克牌。

第 3 行输入表示小华手上已经有的 n 张扑克牌的情况，输入的两张扑克牌信息之间没有空格分隔。

## Output
一行，一个整数，表示小华在游戏中可以获得的最大分数。

注意：小华选牌的的方案可能不唯一，但只要总分数最大即可，不需要输出选牌的方案。

# Samples

```input1
1 3 1 1 1 1 2 3 4 1 3 0 1
3 2
234
```

```output1
13
```
```input2
1 3 1 1 1 1 2 3 4 1 5 0 1
3 2
A3Q
```

```output2
12
```
```input3
1 3 1 1 1 1 2 3 4 1 3 0 5
3 2
A10K
```

```output3
17
```
# Explanation

样例一说明：

小华原来手上有 3 张牌，分别为“2”，“3”，“4”，对应的分数之和为 3 + 1 + 1 = 5，他可以再挑选 2 张扑克牌，都是点数为“9”的扑克牌，这 2 张牌的分数之和为 4 + 4 = 8，所以小华的总得分为13分。

样例二说明：

小华原来手上有 3 张牌，分别为“A”，“3”，“Q”，对应的分数之和为 1 + 1 + 0 = 2，他可以再挑选 2 张扑克牌，都是点数为“J”的扑克牌，这 2 张牌的分数之和为 5 + 5 = 10，所以小华的总得分为12分。

样例三说明：

小华原来手上有 3 张牌，分别为“A”，“10”，“K”，对应的分数之和为 1 + 1 + 5 = 7，他可以再挑选 2 张扑克牌，都是点数为“K”的扑克牌，这 2 张牌的分数之和为 5 + 5 = 10，所以小华的总得分为17分。

# Limitation
对于 50% 的数据，保证小华手上已经有的牌不会出现“A”、“10”、“J”、“Q”、“K”这 5 种代表不同点数的牌。

对于 80% 的数据，保证小华手上已经有的牌不会出现“10”这种点数的牌。

对于 100% 的数据，1 ≤ n ≤ 100，0 ≤ m ≤ 100，0 ≤ 每种点数的牌所代表的分数 ≤ 1000。
