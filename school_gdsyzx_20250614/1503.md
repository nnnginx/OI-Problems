# Description
众所周知，股票经纪人是非常讨厌谣言的。然而，你已经决定开发一种可以在股票经纪人中散布虚假消息的方法，使得你的老板可以在股票市场上占据优势。为了达到最大的效果，你必须以最快的方式传播谣言。

然而，不幸的是，股票经纪人只相信那些来自于他们认为是“可信来源”的消息。这意味着你再开始散布谣言时必须要考虑到他们的关系结构（毕竟职场上有竞争也有合作），一个特定的股票经纪人要花一定的时间把谣言传给他的每个同事。你的任务是编写一个程序，选择哪个股票经纪人作为你的谣言的起点，以及谣言在整个股票经纪人社区传播所需的时间。这个时间是根据最后一个人接受信息所需的时间来计算的。

# Format

## Input
第一行一个整数 a 代表有 a 个股票经纪人；

接下来 a 行，第 i 行代表 i 号经纪人认识的情况，其中第一个数字代表认识的人的数量 n，接下来两个数为认识的人的编号和向该人传递消息所需的时间（分钟为单位），以此类推。
经纪人的编号从 1 至 n，传递消息所需的时间范围是 1 至 10 分钟，经纪人的数量范围为 1 ≤ n ≤ 100。

## Output
一行，输出最快传播消息的起点股票经纪人的编号，以及最后一个人收到消息的时间（以整数分钟为单位）。当然不排除某些人是接收不到的，如果出现这样的情况，请输出“disjoint”。

注意：此题有可能A传输给B的时间与 B 传输给 A 的时间不一样。
