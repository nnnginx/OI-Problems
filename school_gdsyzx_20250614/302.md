#### **题目描述**

农夫$JOHN$为了保持奶牛拥有一个聪明的脑袋，所以让它们玩脑力开发玩具。其中一个较大型的玩具就是在牛棚中的电灯。农场中总共有$N（2<=N<=500)$个牛棚，每个牛棚上方都有一个编号为$1$到$N$的彩色灯泡。 傍晚时分，所有的灯泡都是关闭着的。奶牛们通过$N$个按钮来控制着灯泡的开与关。按下编号为$1$的按钮会使编号为I的灯泡的状态改变，比如从开到关，从关到开。 奶牛们阅读并执行一个由$M（1<=M<=2000)$个操作符和它的参数所组成的操作清单。每个操作符由一个整数表示（$0<=$操作符$<=1$）。

编号为$0$的操作符包括$2$个参数，$S_i$和$E_i(1<=S_i<=E_i<=N)$，要求奶牛依次按下了从第$S_i$号开关到第$E_i$号的开关。

编号为$1$的操作符仍然有$2$个参数，$S_i$和$E_i$。这条命令要求奶牛数出在第$S_i$号和第$E_i$号灯泡之中，有多少灯泡是亮着的。

请你帮助农夫$JOHN$确定奶牛是否按下了正确的按钮和数出正确的答案。

#### **输入**

第一行：$2$个由空格分开的整数$N$和$M$。
第二行到第$M+1$行：每一行有$3$个由空格分开的整数：操作符（$0$或$1$），$S_i，E_i$

#### **输出**

每次出现$1$号操作符就输出一次正确的当前灯光状态为开的灯泡数量。

#### **样例输入**

```plain
4 5 
0 1 2 
0 2 4 
1 2 3 
0 2 4 
1 1 4
```

#### **样例输出**

```plain
1 
2
```

#### **提示**

样例解释： 共有四个灯泡，五条指令。

$1\ \  2\ \  3\ \  4$ （灯泡编号） $O\ \  O\ \  O\ \  O$ （初始状态），$O$ 表示关，

$0\ \  1\ \  2 \ $->$\  O\ \  ，O$ 表示执行切换操作，切换编号为$1$和$2$的灯泡的状态，

$ 0\ \  2\ \  4\ $ -> $\ O\ $切换编号为$2，3，4$的灯泡的状态，

$1\ \  2\ \  3\ $ ->$\ 1\ ,\ 1$表示执行计数操作， 数出从编号$2$到编号$3$之间灯泡状态为开的灯泡数量。

$ 1\ \  1\ \  4\  $->$\  2$ 数出从编号$1$到编号$4$之间灯光状态为开的灯泡数量。
