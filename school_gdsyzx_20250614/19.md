题目描述

**原题来自：HNOI 2006**

给定 **n** 个字符串 $S_1,S_2,\cdots ,S_n$，要求找到一个最短的字符串 **T**，使得这 **n** 个字符串都是 **T** 的子串。

输入格式

第一行是一个正整数 **n**，表示给定的字符串的个数；

以下的 **n** 行，每行有一个全由大写字母组成的字符串。

输出格式

只有一行，为找到的最短的字符串 **T**。

在保证最短的前提下，如果有多个字符串都满足要求，那么必须输出按字典序排列的第一个。

样例

输入

```
2
ABCD
BCDABC
```

输出

```
ABCDABC
```

数据范围与提示

对于全部数据，$1\le n\le 12,1\le |S_i|\le 50$。
