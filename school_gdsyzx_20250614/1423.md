# Description
在金银岛上，人们使用的货币的值都是完全平方数，例如 1，4，9，......，289。支付十元钱有下列四种方法：

（1）十个一元的钱；

（2）一个四元的，六个一元的；

（3）两个四元的，两个一元的；

（4）一个九元的，一个一元的。

你的任务是对于给定的钱数（设其值少于 300），求出支付方法的总数。

# Format

## Input
输入共有 n + 1 行（n 未知），以数字 0 结束，每行为一个自然数 t（1 ≤ t ≤ 300）。

## Output
输出共有 n 行，每行表示对于给定的钱数 t，对应的支付方案总数。

# Samples

```input1
2
10
30
0
```

```output1
1
4
27
```
