#### **题目描述**

一组数，分别表示地平线的高度变化。高度值为整数，相邻高度用直线连接。找出并统计有多少个可能积水的低洼地？如图：地高变化为 $0 \ \ 1 \ \ 0\ \  2\ \  1\ \  2\ \  0\ \  0\ \  2 \ \ 0$

![121.jpg](http://ssoi.noip.space/upload/rumen/f81466b48c.jpg)

#### **输入**

两行，第一行$n$，表示有$n$个数。第二行连续$n$个数表示地平线高度变化的数据。 保证数据的首尾为$0$。$（3<= n <= 100000$，$0<=$ 高度$ <=1000 )$

#### **输出**

一个数。可能积水低洼地的数目。

#### **样例输入**

```
10 
0 1 0 2 1 2 0 0 2 0
```

#### **样例输出**

```
3
```
