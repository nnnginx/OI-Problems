#### **题目描述**

由于Tom现在只有一个孩子，所以根据中国$2016$年计划生育新政策，Tom要为他下一年出生的第二个小孩子准备个新名字，这次他想出了另一个办法：随便给出五个单词（长度大于等于$5$），第$1$个单词取第$1$个字母，第$2$个单词取第$2$个字母，……第$5$个单词取第$5$个字母，这$5$个字母组成的就是新名字。

#### **输入**

输入五行，每行一个字符串(长度不小于$5$)。

#### **输出**

输出一行，一个字符串。

#### **样例输入**

```
happy 
susan 
elizabeth 
hualg 
elleen
```

#### **样例输出**

```
huile
```
