#### 题目描述

$N$位同学站成一排，音乐老师要请其中的($N-K$)位同学出列，使得剩下的$K$位同学排成合唱队形。

合唱队形是指这样的一种队形：设K位同学从左到右依次编号为$1，2…，K$，他们的身高分别为T<sub> 1</sub>，T<sub>2</sub>，…，T<sub>K</sub>，  则他们的身高满足

T<sub>1</sub> < ...< T<sub>i</sub> > T<sub>i+1</sub> > …> T<sub>K</sub>(1< =i< =K)。

你的任务是，已知所有N位同学的身高，计算最少需要几位同学出列，可以使得剩下的同学排成合唱队形。

#### 输入

输入的第一行是一个整数$N(2  ≤ N ≤ 100)$，表示同学的总数。第一行有$n$个整数，用空格分隔，第$i$个整数$Ti$  ($130 ≤ Ti  ≤ 230$)是第i位同学的身高(厘米)。

#### 输出

输出包括一行，这一行只包含一个整数，就是最少需要几位同学出列。

#### 样例输入 

```plain
8
186 186 150 200 160 130 197 220
```

#### 样例输出

```plain
4
```
