# 区间增减区间求和

【题目描述】

给你$n$个整数，请问第$x$个到第$y$个的和是多少？

与上一题不同，在询问前后，这$n$个数中的某个区间的数可能会同时加上$z$。

【输入格式】

第一行：$n$和$m$，表示$n$个数，$m$个询问

第二行：$n$个整数

接下来$m$行，每行$3$或$4$个整数$o、x、y(、z)$：$o$表示求和还是更改数字，如果$o=1$，表示第$x个$数到第$y$个数都加上$z$；如果$o=2$，表示求第$x$个到第$y$个的和。

【输出格式】

对于每次询问和，输出区间的和。

【样例输入】

```
5 3
1 2 -3 6 5
2 2 4
1 2 5 1
2 1 3
```

【样例输出】

```
5
2
```

【数据范围】

$1<=x、y<=n；1<=o<=2$；$n$个数，每个数的绝对值不超过$1000$；$5<=n、m<=200000$

($10$个点数据范围：$5, 10, 100, 500, 1000, 5000, 10000, 50000, 100000, 150000, 200000$)
