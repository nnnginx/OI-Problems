# Description
一个整数，如果它的各个数字出现的次数都是偶数，那么这个整数称为“优质整数”。例如 22、4242、7720707727 都是“优质整数”。但 33033,、12345 都不是“优质整数”。

现在给出两个整数 X 和 Y，求闭区间 [X，Y] 内有多少个“优质整数”。

# Format

## Input
两个整数，X 和 Y。 0 ≤ X ≤ Y ≤ $10^{18}$-1。

## Output
一个整数。

# Samples

```input1
47 999
```

```output1
5
```

# Explanation
样例一说明：

55、66、77、88、99 这五个是满足题意的“优质整数”。

# Limitation
对于 60% 的数据，Y ≤ 100000。
