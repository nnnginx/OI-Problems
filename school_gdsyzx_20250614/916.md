# Description

一只奶牛自己独居，它每天需要吃 1 个水果和花费 X 元交暖气费。现在奶牛家里有 F 个水果和 D 元。奶牛最多能独居多少天？奶牛可以去超市买水果，超市每个水果 P 元。

# Format

## Input

多组测试数据。

第一行，一个整数 G，表示有 G 组测试数据。1 ≤ G ≤ 10。

每组测试数据格式：

一行，4 个整数 X，F，D，P。1 ≤ X，F，D，P ≤ 2000000000。

## Output

共 G 行，每行一个整数。
