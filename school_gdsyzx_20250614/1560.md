# Description
火柴盒长为 L，宽为 W（1 ≤ L，W ≤ 100），高度有一层楼这么高，给你 N（1 ≤ N ≤ 50）根火柴，知道每一根火柴的长度，问有多少根火柴可以平放进该火柴盒。（忽略火柴的厚度）

# Format

## Input
第一行输入 N，L 和 W。

接下来 N 行，每行输入一个 1 到 1000 范围内的一个整数，表示火柴的长度。

## Output
一行，输出能够装进火柴盒的火柴的数量。

# Samples

```input1
5 3 4
3
4
5
6
7
```

```output1
3
```
```input2
2 12 17
21
20
```

```output2
1
```
