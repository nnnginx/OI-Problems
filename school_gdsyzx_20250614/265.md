#### **题目描述**

某大会议室有$N$行$M$列的座位，开会时发现有些座位是空的，但每个人都只关注在他前方(靠前排的所有人)的缺席人数。现在想知道每个座位关注到多少座位是空的。 例如：$N=3,M=4$,下面格子中$1$表示有人，$0$表示空。

![30.jpg](http://ssoi.noip.space/upload/rumen/3bc7420996.jpg)

#### **输入**

第一行$2$个正整数：$N$和$M$，范围在$[1,100]$。 下面$N$行，每行$M$个整数：$0$或$1$。

#### **输出**

$N$行，每行$N$个整数。

#### **样例输入**

```
3 4 
1 0 1 1 
1 1 0 0 
0 1 1 1
```

#### **样例输出**

```
0 0 0 0 
1 1 1 1 
3 3 3 3
```
