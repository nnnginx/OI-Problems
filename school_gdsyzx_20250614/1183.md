# Description
桐桐的学校刚刚举行了期中考试，桐桐所在的七年级一共考了五个科目：语文、数学、英语、政治和历史。现在所有成绩都已经出来，班主任找到了桐桐，想让桐桐帮忙计算班里所有同学的五科总成绩，并按照总成绩由高到低的顺序排名，如果两个学生的总成绩相同，则学号小的排在前面。你能够编写程序帮助桐桐完成任务吗？

# Format

## Input
第 1 行：一个数 n（1 ≤ n < 100），表示桐桐所在班级的学生数；

第 2 行至第 n + 1 行：第 i 行表示学号为 i - 1 的学生的成绩（全部是整数，范围在 0 到 100 之间），每行有 5 个数，分别表示当前学生的五科成绩，相邻数之间用一个空格隔开。

## Output
只有 1 行，按总成绩由高到低的顺序输出对应学生的学号，相邻学号间用一个空格隔开。

# Samples

```input1
5
89 76 80 67 90
70 80 78 95 92
65 77 88 98 100
82 68 99 87 69
100 92 87 79 90
```

```output1
5 3 2 4 1 
```
