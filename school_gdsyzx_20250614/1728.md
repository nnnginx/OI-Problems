# Description
yk同学赫赫有名，为什么呢？因为他参加了若干竞赛，所以认识无数大朋友和小盆友。可是无数大朋友和小盆友也都认识他，为什么呢？因为yk同学实在是太牛了~ 我们甚至经常可以在路上听到同学之间的典型谈话：

小A：你认识yk吗？

小B：当然认识啦，就是那个XX竞赛班的大牛哇~

小A：他不光是XX竞赛班的神牛，还是XXX竞赛班的神牛呢~

小B：哦~~ 

然后小A小B一起无限膜拜中......

那么，yk同学到底神到什么地步呢？

世界上总共有 N 门竞赛，yk同学参加了全部，并且都能拿到保送！！运气好的时候呢，还能拿到一等奖！！怎么样，厉害吧~~

现在有 N 门竞赛，yk同学对于不同学科有不同的概率拿到一等奖。现在yk想知道，他有多少的概率至少拿到 M 个一等奖。

# Format

## Input
第一行，两个用空格隔开的正整数 N，M。

下面 N 行，每行一个 0 - 1 的实数，第 i + 1 行的实数表示第 i 门科目得到一等奖的概率。

## Output
一行，一个实数，表示至少得到 M 门一等奖的概率。实数保留六位小数。

# Samples

```input1
2 1
1
1
```

```output1
1.000000
```
```input2
3 1
0.046921
0.959868
0.008229
```

```output2
0.962066
```

# Limitation
对于 100% 的数据，M ≤ N ≤ 20。
