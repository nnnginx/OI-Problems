# Description
某种生物大小为整数，随机变异为不超过其大小的一半，当大小为 1，再次变异就只能消失了。求解变异图谱。

![img](./925/file/衰减.png)

# Format

## Input
一个数 n，为该生物第一代的大小。

## Output
前面若干行，每行为该生物的变异路径。

最后一行，一个数，为变异的可能情况数。

# Samples

```input1
10
```

```output1
10-1
10-2-1
10-3-1
10-4-1
10-4-2-1
10-5-1
10-5-2-1
7
```

# Limitation
对于 100% 的数据，N ≤ 200。
