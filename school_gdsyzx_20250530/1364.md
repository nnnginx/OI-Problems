# Description
给定 n 个整数，求里面出现次数最多的数，如果有多个重复出现的数，求出值最大的一个。

# Format

## Input
第一行输入一个整数 n（1 ≤ n ≤ 1000000），接下来一行输入 n 个int范围内的整数。

## Output
输出出现次数最多的数和出现的次数，中间用一个空格隔开，如果有多个重复出现的数，输出值最大的那个。

# Samples

```input1
5
1 1 2 3 4
```

```output1
1 2
```
```input2
10
9 10 27 4 9 10 3 1 2 6
```

```output2
10 2
```
