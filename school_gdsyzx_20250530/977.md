# Description
考虑自然数 1 到 N，在这 N 个数之前添加 “+” 或 “-”，然后将 N 数相加得到和 S（0 ＜ S ≤ 100000）。求可以获得和 S 最小的自然数 N。

# Format

## Input
输入文件仅包含 1 行，一个整数，表示和 S。

## Output
输出文件第 1 行一个正整数，表示获得和 S 的最小自然数 N。

接下来若干行，每行一个整数，表示添加了 “-” 的自然数（添加了 “+” 的自然数不输出），按从小到大的顺序输出。注意：添加 “+” 或 “-” 的方案可能不唯一。

# Samples

```input1
12
```

```output1
7
1
7
```

# Explanation
样例一说明：

因为 12 = -1 + 2 + 3 + 4 + 5 + 6 - 7，故获得和 12 的最小自然数是 7，在 1 到 7 的 7 个自然数中，在 1 和 7 之前添加了 “-”。

# Limitation
对于 10% 的数据，0 ＜ S ≤ 20；

对于 30% 的数据，0 ＜ S ≤ 150；

对于 60% 的数据，0 < S ≤ 50000；

对于 100% 的数据，0 < S ≤ 100000。
