# 问题 C: 有序表的最小和(优先队列)

**内存限制：128 MB**   **时间限制：1 S**

#### 题目描述

**给出两个长度为n的有序表A和B，在A和B中各任取一个元素，可以得到n2个和，求这些和中最小的n个。（不要去重）**

#### 输入

第一行包含一个整数 n（ *n*<=400000）; 第二行与第三行分别有n个整数，分别代表有序表A和B。整数之间由一个空格隔开，大小在长整型范围内，保证有序表的数据单调递增。

#### 输出

**输出共n行，每行一个整数，第i行为第i小的和。数据保证在长整型范围内。**

#### 样例输入

```
3
1 2 5
2 4 7
```

#### 样例输出

```
3
4
5
```

#### 提示

**《高级数据结构》例题3-3**
