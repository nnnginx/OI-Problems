# Description
桐桐想把一个自然数 N 分解成 K 个大于1的自然数相乘的形式，要求这 K 个数按从小到大排列，而且除了第 K 个数之外，前面（K - 1）个数是 N 分解出来的最小自然数。例如：N = 24，K = 2 时，输出为 24 = 2 × 12，而不是 24 = 4 × 6；如 N = 3，K = 1 时，3 = 3；N = 3，K = 2 时，输出则为“No answer!”。你能帮助她吗？

# Format

## Input
第 1 行：N（2 ≤ N ≤ $10^7$）；

第 2 行：K（1 ≤ K ≤ 100）。

## Output
输出样例格式的分解式。

# Samples

```input1
24
2
```

```output1
24=2*12
```
