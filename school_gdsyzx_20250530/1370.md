# Description
设有 n 种物品，每种物品有一个重量及一个价值。但每种物品的数量是无限的，同时有一个背包，最大载重量为 M，今从 n 种物品中选取若干件（同一种物品可以多次选取），使其重量的和小于等于 M，而价值的和为最大。

# Format

## Input
第一行，两个整数，M（背包容量，M ≤ 200）和 N（物品数量，N ≤ 30）；

第二至第 N + 1 行，每行二个整数 Wi，Ci，表示每个物品的重量和价值。

## Output
仅一行，一个数，表示最大总价值。

# Samples

```input1
10 4
2 1
3 3
4 5
7 9
```

```output1
max=12
```
