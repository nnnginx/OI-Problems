# Description
Farmer John面临一个很可怕的事实，因为防范失措导致他存储的所有稻草给澳大利亚蟑螂吃光了，他将面临没有稻草喂养奶牛的局面。在奶牛断粮之前，Farmer John拉着他的马车到Farmer Don的农场中买一些稻草给奶牛过冬。已知Farmer John的马车可以装的下 C（1 ≤ C ≤ 50000）立方的稻草。

Farmer Don有 H（1 ≤ H ≤ 5,000）捆体积不同的稻草可供购买，每一捆稻草有它自己的体积V_i（1 ≤ V_i ≤ C）。面对这些稻草Farmer John认真的计算如何充分利用马车的空间购买尽量多的稻草给他的奶牛过冬。

现在给定马车的最大容积 C 和每一捆稻草的体积 V_i，Farmer John如何在不超过马车最大容积的情况下买到最大体积的稻草？他不可以把一捆稻草分开来买。


# Format

## Input
第一行两个整数，分别为 C 和 H；

第二至第 H + 1 行，每一行一个整数，代表第 i 捆稻草的体积 V_i。

## Output
一个整数，为Farmer John能买到的稻草的体积。

# Samples

```input1
7 3
2
6
5
```

```output1
7
```
