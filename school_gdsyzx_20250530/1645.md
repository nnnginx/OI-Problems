# Description
任何一个整数 N 都能表示成另外两个整数 a 和 b 的平方差吗？如果能，那么这个数 N 就叫做 Couple number。你的工作就是判断一个数 N 是不是 Couple number。

# Format

## Input
仅一行，两个 long long 型范围内的整数 n1 和 n2，之间用 1 个空格隔开。

## Output
一行，输出在 n1 到 n2 范围（含 n1 和 n2）内有多少个 Couple number。

注意：n1 ＜ n2，n2 - n1 ≤ 10000000。

# Samples

```input1
1 10
```

```output1
7
```
