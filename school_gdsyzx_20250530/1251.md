# Description
奶牛在熊大妈的带领下排成了一条直队。

显然，不同的奶牛身高不一定相同......

现在，奶牛们想知道，如果找出一些连续的奶牛，要求最左边的奶牛 A 是最矮的，最右边的 B 是最高的，且 B 高于 A 奶牛。中间如果存在奶牛，则身高不能和 A，B 奶牛相同。问这样的奶牛最多会有多少头？

从左到右给出奶牛的身高，请告诉它们符合条件的最多的奶牛数（答案可能是 0，2，但不会是 1）。

# Format

## Input
第一行一个正整数 N，表示奶牛的头数。

接下来 N 行，每行一个正整数，从上到下表示从左到右奶牛的身高 hi。

## Output
一行一个整数，表示最多奶牛数。

# Samples

```input1
5
1
2
3
4
1
```

```output1
4
```

# Explanation
样例一说明：

取第 1 头到第 4 头奶牛，满足条件且为最多。

# Limitation
对于全部的数据，满足 2 ≤ N ≤ $10^5$，1 ≤ hi ＜ $2^{31}$。
