# Description
奶牛们最近在学习统计学理论，但那些机械烦琐的对数据的统计分析几乎把她们搞疯了。于是奶牛们找到了你，希望你能帮帮她们。

你拿到了一张写着 N（1 ≤ N ≤ 500）个数的表，表中第 i 个数字为 X_i（-5000 ≤ X_i ≤ 5000），你的任务是计算如下两个值：

所有数的平均数（所有数的和除以 N）

所有数的中位数（如果 N 是奇数，中位数为排序后的数列里位于正中的那个；如果 N 是偶数，中位数是排序后的数列位于正中的两个数的平均数）

答案保留 6 位小数。

# Format

## Input
第一行，一个正整数 N。

第二至第 N + 1 行，每一行分别为一个整数 X_i。

## Output
第一行，输出所有数的平均数。

第二行，输出所有数的中位数。

# Samples

```input1
5
12
4
6
8
2
```

```output1
6.400000
6.000000
```
