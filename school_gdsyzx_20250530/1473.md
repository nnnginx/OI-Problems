# Description
在一条数轴上有 N 个整数点，分别是 1 至 N。现在已经有 M 个警卫，第 i 个警卫的位置在 p[i]。每个警卫都能看守一段距离 K。第 i 个警卫能看守的范围从 p[i] - K 到 p[i] + K。现在的问题是：至少还需要增加多少个警卫，才能使得 1 至 N 所有的整点都能被警卫看守？注意：你可以把增加的警卫放到任意需要的地方。

# Format

## Input
第 1 行，一个整数 N。

第 2 行，一个整数 M。

第 3 行，一个整数 K。

接下来有 M 行，第 i 行是一个整数 p[i]。所有的警卫的位置都不会重叠。

## Output
一个整数。

# Samples

```input1
5
2
2
1
5
```

```output1
0
```
```input2
26
3
3
3
19
26
```

```output2
2
```

# Limitation
对于 100% 的数据，1 ≤ N ≤ 1000，1 ≤ M ≤ N，0 ≤ K ≤ N，1 ≤ p[i] ≤ N。
