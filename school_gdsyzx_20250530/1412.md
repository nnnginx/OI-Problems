
# Description

FJ饲养了 $n$ 种奶牛$(1≤n≤1000)$，编号 $1…n$，不同奶牛间友好程度不一样。如果$|a−b|≤4$，则品种 $a$ 和 $b$ 是相互友好的，否则，是相互不友好的。

​有一条路贯穿FJ的农场，在路的一边，有 $n$ 个领域，每个领域里有一种牛，所有品种的奶牛成一个序列排列。路的另一边也有 $n$ 个领域成一个序列，每个领域里也有一种牛。为了让奶牛们安全的过马路。FJ打算在马路上画斑马线。斑马线连接马路两边的两个领域。这两个领域里的奶牛品种必须是相互友好的才安全。每个领域只能连接一条斑马线。且所有的斑马线不能相互交叉。

​	请帮助FJ算出，最多能画出多少条斑马线。

# Format

## Input

第1行：$n$

第$2..n+1$行：每行 $1$ 个 $1$ 到 $n$ 之间的整数表示马路的一边的奶牛编号。

接下来 $n$ 行：每行 $1$ 个 $1$ 到 $n$ 之间的整数表示马路的另一边的奶牛编号。

## Output

​	最多能画出多少条友好斑马线.

# Samples

```input1
6
1
2
3
4
5
6
6
5
4
3
2
1
```

```output1
5
```

## Limitation

对于 100% 的数据，$1≤n≤1000$
