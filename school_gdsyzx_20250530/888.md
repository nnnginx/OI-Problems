# Description

学校体艺节前要维修运动场。有一条长度 N 米的跑道，跑道边上要用地砖铺垫，地砖有 2 种型号，1 种长度是 A 米，另一种长度是 B 米，但你只能采用一种型号地砖铺垫。

地砖有可能浪费一些零头，比如：N = 10，A = 8，A 型砖要 2 块，会浪费 6 米。

现在老师要求晨晨通过编计算机程序来计算最少会浪费多少米？

# Format

## Input

输入 N，A，B 的值。（各数均为大于 0 小于 10000 的整数）

## Output

一个整数，最少浪费的长度。

# Samples

```input1
10 3 7
```

```output1
2
```

# Explanation

样例一说明：

4 块 A 型砖浪费 2 米；2 块 B 型砖浪费 4 米。用 A 型地砖浪费最少。
