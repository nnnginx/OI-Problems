# 样例

```input1
5 3 2
2 2
2 3
```

```output1
2
```

```
至少产生 3 的利润，该帮派可以犯下罪 0 和罪 1 ，或仅犯下罪 1 。
 总的来说，有两种方案。
```

```input2
10 5 3
2 3 5
6 7 8
```

```output2
7
```

```
至少产生 5 的利润，只要他们犯其中一种罪就行，所以该帮派可以犯下任何罪行。
 有 7 种可能的计划：(0)，(1)，(2)，(0,1)，(0,2)，(1,2)，以及 (0,1,2) 。
```
