# 【题目描述】

芝麻同学的爸爸经营了一家手抓饼店，今天店里进购了一批纸袋用来装手抓饼，但是所有纸袋的尺寸并不是一致的，现在芝麻同学要帮爸爸整理这一批纸袋。整理过程如下，已知每个纸袋的长宽，当一个纸袋A的长度和宽度均大于另外一个纸袋B长度和宽度，那么就可以将纸袋B放入纸袋A里面，如同俄罗斯套娃一样。

现在请你帮忙芝麻同学计算一下 最多能有多少个 纸袋能组成一组“俄罗斯套娃”纸袋（即可以把一个纸袋放到另一个纸袋里面）。

注意：不允许旋转纸袋。

## 【输入格式】

从文件paper.in中读入数据。

第一行共一个整数 n，表示n个纸袋。

接下来n行，每一行有两个参数，分别表示纸袋的长度，宽度。

## 【输出格式】

输出到文件paper.out中。

共一行，一个整数，表示 最多能有多少个 纸袋能组成一组“俄罗斯套娃”纸袋

# 【样例1输入输出】

```input
4
5 4
6 4
6 7
2 3
```

```output
3
```

# 【样例1解释】

最多信封的个数为 3, 组合为: [2,3] => [5,4] => [6,7]。

# 【样例2输入输出】

```input
3
1 1
1 1
1 1
```

```output
1
```

# 【数据范围】

・ 1 <= n<= 10^5
