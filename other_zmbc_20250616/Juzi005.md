## B - n个奇数

#### 时空限制

1s / 128MB

#### 题目描述

读取一个整数 $x$，输出 $x$ 之后的 $n$ 个奇数，如果 $x$ 也是奇数，那么它也算作  $n$ 个奇数之一。

#### 输入格式

输入两个整数$x，n$，($0<x<10^9$，$0<n<10^6$)。

#### 输出格式

输出 $x$ 之后的 $n$ 个奇数，每个数末尾带一个空格。（如果 $x$ 也是奇数，那么 $x$ 也算作  $n$ 个奇数之一，并输出）。

#### 输入样例：

```input
3 3
```

#### 输出样例：

```
3 5 7
```

#### 输入样例2：

```input
2 3
```

#### 输出样例2：

```
3 5 7
```
