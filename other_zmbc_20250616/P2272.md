# 题目描述

两位正整数的简单算术运算（只考虑整数运算），算术运算为：

&nbsp;&nbsp;**+**&nbsp;，加法运算；

&nbsp;&nbsp;**\-**&nbsp;&nbsp;，减法运算；

&nbsp;&nbsp;**\***&nbsp;&nbsp;，乘法运算；

&nbsp;&nbsp;**/**&nbsp;&nbsp;，除法运算；

&nbsp;&nbsp;**%**&nbsp;，求余运算。

算术运算符的格式为（运算符前后可能有空格）：

&nbsp;&nbsp;**运算数&nbsp;&nbsp;运算符&nbsp;&nbsp;运算数**

请输出运算后的结果。

## 输入

一行字符串形式的算术表达式。

## 输出

整形运算的结果（结果不一定为2位数，可能多于2位或少于2位）。

# 样例

```input1
32+64
```

```output1
96
```
