# Description

为了更好的衡量每位学生总体的学习情况，橘子老师为每位学生设置了一个作业分数，规则如下：每位学生作业分数初始都是100分，对于每次作业，表现的好可以加5分，不交作业或表现不好则扣5分。现给出n位的学生的m次的作业情况，请你运用所学知识，帮橘子老师计算一下最后这n位学生的分数。

# Format

## Input

第一行，输入两个整数$n（1<=n<=100000）,m(0<=m<=10)$；

以下$m$行，每行$n$个整数 $a_1,a_2,..,a_n$分别代表序号为1,2,3...,n个学生的该次作业提交情况，作业表现好用1表示，不好用0表示，不交作业用-1表示。（a_i为{$-1，0，1$}之一）。

## Output

输出1行，按序号($1,2,3,...,n$)输出每位学生的最终分数并以空格隔开（末尾没有空格）。

# Samples

```input1
5 2
1 1 0 -1 1
1 0 1 0 -1
```

```output1
110 100 100 90 100
```

# Limitation

1000ms, 64MB
