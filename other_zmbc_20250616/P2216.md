# 题目描述

给定一个*n × n*的矩阵(3<=n<=100，元素的值都是非负整数)。通过(n-1)次实施下述的 **操作**，可把这个矩阵转换成一个*1 × 1*的矩阵。每次 **操作** 的过程如下：

* &nbsp;&nbsp;&nbsp;&nbsp;**1、首先对矩阵进行行归零：即对每一行上的所有元素，都在其原来值的基础上减去该行上的最小值，保证相减后的值仍然是非负整数，且这一行上至少有一个元素的值为0。**
* &nbsp;&nbsp;&nbsp;&nbsp;**2、接着对矩阵进行列归零：即对每一列上的所有元素，都在其原来值的基础上减去该列上的最小值，保证相减后的值仍然是非负整数，且这一列上至少有一个元素的值为0。**
* &nbsp;&nbsp;&nbsp;&nbsp;**3、然后对矩阵进行消减：即把*n × n*矩阵的第二行和第二列删除，使之转换为一个(*n-1*) *×* (*n-1*)的矩阵。**

下一次过程，对生成的(*n-1*) *×* (*n-1*)矩阵实施上述 **操作**。显然，经过(n-1)次上述 **操作**，*n × n*的矩阵会被转换为一个*1 × 1*的矩阵。

请求出每次 **操作** 前位于矩阵第二行第二列的元素的值。

## 输入

第一行是一个整数n。
接下来n行，每行有n个正整数，描述了整个矩阵。相邻两个整数间用单个空格分隔。

## 输出

输出为n行，每行上的整数为对应矩阵归零消减 **操作** 中，每次 **操作** 前位于矩阵第二行第二列的元素的值。

# 样例

```input1
3
1 2 3
2 3 4
3 4 5
```

```output1
3
0
0
```
