## D - 美观

#### 时空限制

1s / 128MB

#### 题目描述

所有的橘子采摘完了，橘子园里面只剩下一片绿油油的橘子树。

![](https://img0.baidu.com/it/u=3339156610,1039910624&fm=253&fmt=auto&app=120&f=JPEG?w=889&h=500)

假设橘子园中的所有橘子树排列成一排。橘子老师为了让橘子树们在明年丰收时能够**保持美观**，决定在特定的橘子树下增肥。橘子老师眼中的美观：**从左到右每棵树的高度依次单调递增，所谓单调递增指的是每一个数都比前一个数要大**。例如，假设有4棵橘子树，高度从左到右依次为 $1,2,3,4$，此时这排橘子树就是美观的。

橘子老师的肥料非常有效，第一年每施加 $1$ 袋肥料，第二年橘子树的高度就会加 $1$。

现给你 $N$ 棵橘子树从左到右的当前高度 $H_i$，对于每课树**至少**需要施加多少袋化肥？

#### 输入格式

第一行，一个正整数 $N$ （$0 < N < 10^5$），表示橘子树的数量。

随后一行，N个正整数 $H_i$ （$0 < H_i < 10^9$），分别表示 $N$ 棵橘子树从左到右的当前高度 $H_i$。

#### 输出格式

从左到右对于每一棵橘子树，输出至少需要施加的化肥袋数，以空格隔开。

#### 输入样例：

```input
4
1 2 1 3
```

#### 输出样例：

```
0 0 2 1
```

对输出样例进行说明：

​		橘子树的高度初始为：1, 2, 1, 3,

​				分别施加化肥数：0, 0, 2, 1,

则第二年的橘子高度变为：1, 2, 3, 4，此时是一排美观的橘子树。
