# 题目描述

对于一个字符串来说，定义一次循环位移操作为：奖字符串的第一个字符移动到末尾形成新的字符串。

给定两个字符串s1和s2，要求判定其中一个字符串是否是另一个字符串通过若干次循环位移后的新字符串的子串。

例如 CDAA 是由 AABCD 两次唯一后产生的新串 BCDAA 的子串，而 ABCD 与 ACBD 则不能通过多次移位来得到其中一个字符串是新串的子串。

## 输入

一行，包括两个字符串，中间由单个空格隔开。字符串只包含字母和数字，长度不超过30。

## 输出

如果一个字符串是另一字符串通过若干次循环位移产生的新串的子串，则输出true，否则输出false

# 样例

```input1
AABCD CDAA
```

```output1
true
```
