# 题目描述

给定一个大小为&nbsp;m&nbsp;\*&nbsp;n&nbsp;的矩阵A&nbsp;和另一个大小为&nbsp;r&nbsp;\*&nbsp;s&nbsp;的矩阵B（0 < r <= m，0 < s <= n），A、B所有元素值都是小于100的正整数。求A中一个大小为r&nbsp;\*&nbsp;s的子矩阵C，使得B和C的**对应元素差值的绝对值之和**最小，这时就称C位最匹配的矩阵。

如果有多个子矩阵同时满足条件，选择子矩阵左上角元素行号最小者，行号相同时，选择列号最小者。

## 输入

第一行是m和n，以一个空格分开。（1 <= m <= 100，1 <= n <= 100）

之后m行，每行有n个整数，表示A矩阵中的各行，数与数之间以一个空格分开。

第m+2行为r和s，以一个空格分开。

之后r行，每行有s个整数，表示B矩阵中的各行，数与数之间以一个空格分开。

## 输出

输出矩阵C，一共r行，每行s个整数，整数之间以一个空格分开。

# 样例

```input1
3 3
3 4 5
5 3 4
8 2 4
2 2
7 3
4 9
```

```output1
4 5
3 4
```
