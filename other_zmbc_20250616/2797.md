### 【问题描述】

小明现在又想挑战自己，输出一个**空心菱形**。这个菱形由上下两部分组成，上半部分是一个空心等腰三角形，下半部分是一个倒过来的空心等腰三角形，并且整个菱形有 N 行（N 为奇数）。除了最顶部、最底部以及最左最右的星号之外，中间是空格。

例如，当输入为 5 时，输出如下：

```
  *
 * *
*   *
 * *
  *
```

请你编写程序，实现这个空心菱形的输出。

### 【输入描述】

输入一个正奇数 N，表示菱形的行数。

1 ≤ N ≤ 50 且 N 为奇数

### 【输出描述】

输出一个高为 N 的空心菱形图案。每一行末尾不要有多余空格。

### 【输入样例】

```
5
```

### 【输出样例】

```
  *
 * *
*   *
 * *
  *
```


### 【数据范围】

- 输入的 N 是正奇数，1 ≤ N ≤ 50；

- 输入保证合法，无需校验。
