# 题目描述

甲流并不可怕，在中国，它的死亡率并不是很高。请根据截止2009年12月22日各省报告的甲流确诊数和死亡数，计算甲流在各省的死亡率。

**提示：死亡率 = 死亡数 ÷ 确诊数**

## 输入

输入仅一行，有两个整数，以空格分开，第一个为确诊数，第二个为死亡数。

约定确诊数大于等于1且小于1000000000，死亡数大于等于0且小于等于确诊数。

## 输出

输出仅一行，甲流死亡率，以百分数形式输出，精确到小数点后3位。

**提示：输出%可以使用printf("%%");**

# 样例

```input1
10433 60
```

```output1
0.575%
```
