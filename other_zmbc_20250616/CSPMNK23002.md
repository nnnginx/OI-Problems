# 【题目描述】

芝麻同学老家要新建造一个水库，其中水库的选址已经确定，其长度为n，在每个坐标为整数的位置存在一面石壁，且每面石壁的高度可能各不相同，当我们选取其中两面石壁作为水库的外墙时，其他石壁都会被铲除。但是工程队想要使得水库能存储的水的容量达到最大，于是请芝麻同学来帮忙，让芝麻同学选取两面石壁作为水库的外墙，使得水库能存储的水的容量达到最大（忽略水库的底部面积）。现在请你帮芝麻同学选取两面石壁作为水库外墙，使得当水库注满水后，水库中水的容量最大（注意：当注入水的高度超过最矮的石壁时，水会溢出）。

## 【输入格式】

从文件reservoir.in中读入数据

第一行共一个整数 n，表示水库的长度。水库的坐标为 0, 1, ……, n-1。

第二行共有 n 个整数，分别表示每个整数坐标上石壁的高度。

## 【输出格式】

输出到文件reservoir.out中

共一行，输出一个整数，水库能容纳的最大的水容量。

# 【样例1输入输出】

```input1
9
1 8 6 2 5 4 8 3 7
```

```output1
49
```

# 【样例1解释】

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，水库能够容纳水（表示为蓝色部分）的最大值为 49。

# 【样例2输入输出】

```input2
2
1 1
```

```output2
1
```

# 【数据范围】

・ 2 <= n <= 10^5

・ 0 <= 石壁高度 <= 10^4
