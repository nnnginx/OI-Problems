## C- 丰收

#### 时空限制

1s / 128MB

#### 题目描述

今年橘子园迎来了大丰收！橘子老师决定把所有的橘子采摘后送到果汁工厂加工成橘子汁。

![](https://5b0988e595225.cdn.sohucs.com/images/20191227/cb1d5f92b652471db2001cd24260bb09.png)

不过，为了保证橘子汁的口感，必须采用含糖量在 $30\% \sim 35\%$ 之间、且总重量不低于 $50g$ 的橘子，这样的橘子酸甜适中，美味可口。为此，橘子老师专门研发了一个能够检测出每个橘子糖分重量的感应芯片，并将其安装在了称重器上。只需将一个橘子放在称重器上，该称重器就可以同时显示橘子的重量 $X$ 和橘子中所含有糖分的重量 $Y$。

含糖量计算公式为：$含糖量 = Y ÷ X × 100\%$

假设今年橘子园一共有 $N$ 个橘子，分别给你这 $N$ 个橘子的重量 $X_i$ 和糖分重量 $Y_i$，请你帮忙计算有几个橘子能够满足要求？

为了显得更加专业一些，我们需要使用所学知识编写一个自动筛选橘子的程序，留下含糖量在 $30\% \sim 35\%$ 之间、且总重量不低于 $50g$ 的橘子，并输出这些橘子的数量。

#### 输入格式

第一行，一个正整数 $N$ （$0 < N < 10^5$），表示橘子总数。

随后 $N$ 行，每行两正整数 $X_i，Y_i$ （$0 < X_i ≤ Y_i < 500$），分别表示第 $i$ 个橘子的重量和糖分重量 (单位：$g$)。

#### 输出格式

输出一个整数，表示含糖量在 $30\% \sim 35\%$ 之间、且总重量不低于 $50g$ 的橘子数量。

#### 输入样例：

```input
3
50 17
100 33
10 3
```

#### 输出样例：

```
2
```
