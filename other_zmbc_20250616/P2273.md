# 题目描述

用手机发短信，一条短信资费为0.1元，但限定一条短信的内容在70个字以内（包括70个字）。如果你一次发送的短信超过了70个字，则会按照每70个字一条短信的限制把它分割成多条短信发送。

假设已经知道当月所发送的短信字数，试统计一下你当月短信的总资费。

## 输入

第1行有一个整数n （1 <= n <= 1000），表示当月发送短信的总次数。

第2~n+1行每行有一个整数（不为0，且不超过1000），表示每次短信的字数。

## 输出

第1行有一个小数，表示当月短信的总资费，单位为元，精确到小数点后1位。

# 样例

```input1
10
39
49
42
61
44
147
42
72
35
46
```

```output1
1.3
```
