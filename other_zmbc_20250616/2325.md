# 士兵站队

在一个划分成网格的操场上，n个士兵散乱地站在网格点上，由整数坐标（x，y）表示。
士兵们可以沿网格边上、下、左、右选一个方向移动一步，但在同一个时刻任一网格点上只能有一名士兵。
按照军官的命令，他们要整齐地列成一个水平队列，也就是排成一排，排成（x，y），（x+1，y）……（x+n-1，y）。
请问如何选择以上x和y的值，才能使士兵们以最少的总移动步数排成一排。
求出总移动步数。

## 输入

第一行是一个整数，代表士兵数n。
第2到n+1行，每行2个整数。
第i+1行的这两个整数分别表示第i个士兵的横纵坐标xi，yi。

## 输出

一个整数表示总移动步数。

```输入用例
5
1 2
2 2
1 3
3 -2
3 3
```

```输出用例
8
```
