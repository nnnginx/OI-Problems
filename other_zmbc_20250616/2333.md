# 看电影

一个学校有n个学生正计划去电影院看电影。对于某一个学生，他要么去电影院，要么不去。这取决于还有多少人会去。具体来说，每一个学生都说：“我想去看电影，前提是至少ai
个其他学生会去，不算我自己”。如果满足以下条件之一，则第i个学生会伤心。
1: 学生i去看电影了，但去看电影的总人数少于ai
2: 学生i没去看电影，但是去看电影的人数大于等于ai

问：选择x个学生去看电影，这样子所有学生都不会伤心。x有多少种可能的取值？

## 输入

第一行包含单个整数n（2≤n≤2000)——学生的人数。
第二行包含n整数a1，a2，…，an    ( 0 ≤ ai ≤ n−1 )

## 输出

一个整数，代表x有多少种可能的取值

```输入用例
2
1 1
```

```输出用例
2
```
