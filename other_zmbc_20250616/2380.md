# 样例

```input1
3 2
```

```output1
6
```

```
当n=3时，牌堆有1,2,3种类型的牌各四张。
手牌可能的方案为：[1,1],[2,2],[3,3],[1,2],[1,3],[2,3]。
总共有6种。所以返回6。
```

```input2
4 2
```

```output2
10
```

```
手牌可能的方案为：[1,1],[2,2],[3,3],[4,4],[1,2],[1,3],[1,4],[2,3],[2,4],[3,4]。
```
