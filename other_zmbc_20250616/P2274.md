# 题目描述

目前正是甲流盛行时期，为了更好地进行分流治疗，医院在挂号时要求对病人的提问和咳嗽情况进行检查，对于**体温超过37.5度（含37.5度）并且咳嗽的病人**初步判定为甲流病人（初筛）。现需要统计某天前来挂号就诊的病人中有多少人被初筛为甲流病人。

## 输入

第一行是某天来挂号就诊的病人数n。（1 <= n < 200）

其后有n行，每行都是病人的信息，包括三个信息：

&nbsp;&nbsp;**姓名**（字符串，不含空格，长度不大于8）

&nbsp;&nbsp;**体温**（浮点数）

&nbsp;&nbsp;**是否咳嗽**（整数，1表示咳嗽，0表示不咳嗽）。

每行之间以一个空格分开。

## 输出

按输入顺序一次输出所有被筛选为甲流病人的姓名，每个名字占一行。

之后再输出一行，表示为筛选为甲流的病人数量。

# 样例

```input1
5
Zhang 38.3 0
Li 37.5 1
Wang 37.1 1
Zhao 39.0 1
Liu 38.2 1
```

```output1
Li
Zhao
Liu
3
```
