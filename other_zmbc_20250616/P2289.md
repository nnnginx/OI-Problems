# 题目描述


有一种特殊的日历法，它的一天和我们现在用的日历法的一天是一样长的。它每天有10个小时，每个小时有100分钟，每分钟有100秒。10天算一周，10周算一个月，10个月算一年。

现在要你编写一个程序，将我们常用的日历法的日期转换成这种特殊的日历表示法。这种日历法的时、分、秒是从0开始计数的。日、月从1开始计数，年从0开始计数。描述为整数。

假设 0:0:0 1.1.2000 等同于特殊日历法的 0:0:0 1.1.0。

## 输入

第一行是一个正整数N，表明下面有N组输入。

每组输入有一行，格式如下：小时:分钟:秒 日.月.年。表示常规日期。日期总是合法的。2000 <= 年 <= 50000。

## 输出

对于每组输入，输出一行，格式如下：特殊小时:特殊分钟:特殊秒 特殊日.特殊月.特殊年。 是输入日期的特殊日历表示方法。

# 样例

```input1
7
0:0:0 1.1.2000
10:10:10 1.3.2001
0:12:13 1.3.2400
23:59:59 31.12.2001
0:0:1 20.7.7478
0:20:20 21.7.7478
15:54:44 2.10.20749
```

```output1
0:0:0 1.1.0
4:23:72 26.5.0
0:8:48 58.2.146
9:99:98 31.8.0
0:0:1 100.10.2000
0:14:12 1.1.2001
6:63:0 7.3.6848
```
