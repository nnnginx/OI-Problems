## 题目描述

给定 $n$ 个质数 $p_1,\,p_2,\,p_3,\,\cdots,\,p_n$，定义好数为所有质因数都是 $p_1,\,p_2,\,p_3,\,\cdots,\,p_n$ 之中的数。

给定 $k$，求第 $k$ 小的好数。

## 输入格式

第一行一个整数，表示 $n$。

第二行 $n $ 个整数，表示 $p_1,\,p_2,\,p_3,\,\cdots,\,p_n$。

第三行一个整数，代表 $k$。

## 输出格式

一行一个数，表示第 $k$ 小的好数。

## 样例

```input1
3
2 3 5
7
```

```output1
8
```

```input2
5
2 5 7 23 29
28386
```

```output2
148961113306250
```

## 提示/说明

### 样例 1 解释

满足条件的好数有：$1,\,2,\,3,\,4,\,5,\,6,\,8\,\cdots$，第 $7$ 个为 $8$。

需要注意的是，$1$ 也是好数。

### 数据范围

对于 $60\,\%$ 的数据，$1 \leq n \leq 8$。

对于 $100\,\%$ 的数据，$1 \leq n \leq 16$，$1 \leq p_i \leq 97$，$1 \leq k \leq 10^9$。
