$4202$ 年 $80$ 月 $41$ 日，ACOI 出题团的成员苦于其准备的比赛中缺少一道签到题。

于是，整个团中实力最差的选手 `Redshift_Shine` 想到了一道题。

## 题目描述

给出长度为 $n$ 的非负整数序列 $a'$，$a'$ 由以下的公式计算得出：

$$
a'_i=\bigoplus_{j=1}^n a_j[j\equiv0\pmod i]\\
$$

其中，$[\text{statement}]=1$ 当且仅当条件 $\text{statement}$ 为真，否则为 $0$。$\oplus$ 符号代表异或计算，即 `xor`。

请求出序列 $a$。

## 输入格式

考虑到输入文件过大，输入样例会给出随机种子，解题者的代码需按照给出的方式进行数据生成。

第一行两个整数 $n$ 和 $\text{seed}$，表示 $a'$ 序列长度以及随机种子。

本题采用 `splitmix64` 函数进行随机数生成，其内部实现如下。以下代码中 `ull` 表示 `unsigned long long`。

```c++
ull splitmix64()
{
    seed += 0x9e3779b97f4a7c15;
    seed = (seed ^ (seed >> 30)) * 0xbf58476d1ce4e5b9;
    seed = (seed ^ (seed >> 27)) * 0x94d049bb133111eb;
    return seed;
}
```

序列 $a'$ 的生成方法如下：

```c++
for (int i = 1; i <= n; i++)
{
    a[i] = splitmix64();
}
```

数据说明将会给出样例给出随机种子所对应的序列，可供检查生成正确性。

## 输出格式

考虑到输出文件同样过大，为了减少输出，请输出下列表达式的值。

$$
(\sum_{i=1}^n (a_i\times i))\bmod 2^{64}
$$
