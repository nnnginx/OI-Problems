## 题目描述

Nemophery 是一个可爱的女孩纸，她被扔进一个「梦的空间」。

在这个空间中，依次会有 $N$ 个事情发生。第 $i$ 个事情由一对整数 $(opt_i, x_i)$ 表示：

- 如果 $opt_i = 1$，她会找到一滴类型为 $x_i$ 的「梦之甘露」，她可以选择丢弃或保存在「梦之瓶」中。
- 如果 $opt_i = 2$, 她要实现一个类型为 $x_i$ 的「梦想」，实现它需要消耗一滴类型为 $x_i$ 的「梦之甘露」。

她需要实现所有梦想。

但是她用来保存「梦之甘露」的「梦之瓶」空间有限，所以她想要让「梦之瓶」中甘露最多的时候的甘露最少。

你想要看她卖萌，所以帮她解决这个问题吧。

## 输入格式

输入以以下格式从标准输入给出：

> $N$
> 
> $opt_1$ $x_1$
> 
> $opt_2$ $x_2$
> 
> $\vdots$
> 
> $opt_N$ $x_N$

## 输出格式

如果她无法实现所有「梦想」，就输出 `qwq`，否则输出「梦之瓶」中甘露最多的时候的甘露最少的滴数以及「梦之瓶」中甘露最多的时候的甘露最少的情况下依次输出每个甘露是否被丢弃，如果被丢弃就输出 `0`，否则输出 `1`，如果有多个答案，输出其中任意一个即可。

```input1
13
1 2
1 3
1 1
1 3
1 2
2 3
1 3
1 3
2 3
1 3
2 2
2 3
2 1
```

```output1
3
1 1 1 0 0 1 0 1
```

样例输出对应以下操作：

- 按顺序找到类型为 $2,3,1$ 的「梦之甘露」。保存这些「梦之甘露」。
- 按顺序找到类型为 $3,2$ 的「梦之甘露」。丢弃这些「梦之甘露」。
- 需要实现类型为 $3$ 的「梦想」。使用一滴类型为 $3$ 的「梦之甘露」来实现它。
- 找到一滴类型为 $3$ 的「梦之甘露」。保存它。
- 找到一滴类型为 $3$ 的「梦之甘露」。丢弃它。
- 需要实现类型为 $3$ 的「梦想」。使用一滴类型为 $3$ 的「梦之甘露」来实现它。
- 找到一滴类型为 $3$ 的「梦之甘露」。保存它。
- 需要实现类型为 $2$ 的「梦想」。使用一滴类型为 $2$ 的「梦之甘露」来实现它。
- 需要实现类型为 $3$ 的「梦想」。使用一滴类型为 $3$ 的「梦之甘露」来实现它。
- 需要实现类型为 $1$ 的「梦想」。使用一滴类型为 $1$ 的「梦之甘露」来实现它。

在所有操作中，「梦之瓶」中甘露最多的时候的甘露的滴数为 $3$ 滴，可以发现，没有比 $3$ 滴更少的方法

## 约束条件

- $1\leq N\leq2\times10^5$
- $1\leq opt _ i\leq2\ (1\leq i\leq N)$
- $1\leq x _ i\leq N\ (1\leq i\leq N)$
- 所有输入值都是整数。
