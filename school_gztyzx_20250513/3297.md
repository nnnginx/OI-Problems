## 题目背景

在新冠引来的裁员潮流后，城市里流行起裁村。如果一个村子不太和别的村子交往，就会被移出城市。

## 题目描述

一个有 $n$ 个点的无向联通图，共 $n-1$ 条边。

现在进行 $k$ 次操作，每次删除当前所有叶子节点。问 $k$ 次操作后剩余的点数。

**叶子节点的定义：** 度数小于 $2$ 的节点。

## 输入格式

第一行一个数 $T$，表示测试数据组数。

接下来每一个测试数据，第一行两个数 $n,k$。

第二到 $n$ 行，每行两个数 $u_i,v_i$，表示节点 $u_i$ 到节点 $v_i$ 有一条连边。

## 输出格式

对于每个测试数据，输出一行表示答案。

## 样例 #1

### 样例输入 #1

```
2
5 1
1 2
2 3
2 4
3 5

5 2
1 2
2 3
2 4
3 5
```

### 样例输出 #1

```
2
0
```

## 提示

**【样例解释】**

第一组样例中，节点 $1,4,5$ 将被删去，剩余 $2$ 个节点。

第二组样例中，全部节点将被删去。

**【数据范围】**

$1 \le T \le 10$

| 数据点 |        $n$         |        $k$         |             特殊性质             |
| :----: | :----------------: | :----------------: | :------------------------------: |
|  $1$   |  $1 \le n \le 10$  |  $1\le k \le 10$   |     保证出题人的马不是量产的     |
| $2-3$  | $1 \le n \le 10^3$ |  $1\le k \le 100$  |     保证验题人的马不是量产的     |
|  $4$   | $1 \le n \le 10^5$ |       $k=1$        |     保证做题人的马不是量产的     |
| $5-6$  | $1 \le n \le 10^5$ | $1 \le k \le 10^5$ |            图为一个链            |
| $7-10$ | $1 \le n \le 10^5$ | $1 \le k \le 10^5$ | 保证你不可能AK，AK的出题人请饮料 |
