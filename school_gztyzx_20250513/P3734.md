## 问题描述

桌上有 $n$ 上扑克牌，其中有 $m$ 张joker。有 $k$ 个玩家，每人取 $\frac{n}{k}$ 张牌（ $n$ 能被 $k$ 整除）。取得joker数量最多的人成为赢家，取得的分数为他的joker数减去其他玩家中joker最多的数量。如果有多个玩家都有最多的joker，那么他们都是赢家且分数为 $0$。请你求出赢家可能取得的最大分数。

## 输入格式

三个整数 $n,m,k$，分别表示扑克牌张数、joker数量、玩家的数量；

## 输出格式

一个整数，赢家可能取得的最大分数。

```input1
8 3 2
```

```output1
3
```

```input2
9 6 3
```

```output2
1
```

## 数据范围

对于 $30\%$ 的数据满足 $2 \leq n \leq 50, 0\leq m \leq n,2\leq k \leq n$；

对于 $100\%$ 的数据满足 $2\leq n \leq 10^6,0 \leq m \leq n, 2\leq k \leq n, n$ 能被 $k$ 整除。
