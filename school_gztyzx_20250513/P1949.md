## 问题描述

国庆期间，某超市举行购物优惠活动：所购物品不超过 $100$ 元时，按九折付款，如超过 $100$ 元，超过部分按六折收费。请你编一程序完成超市自动计费的工作。

## 输入格式

输入只有一个整数 $M$，表示购物的款数。

## 输出格式

输出打折后的款数（ 结果保留两位小数）。

```input1
23.31
```

```output1
20.98
```

```input2
120
```

```output2
102.00
```

## 数据范围

对于 $40\%$ 的数据，$2 ≤ n ≤ 40，1 ≤m≤ 40$。

对于 $90\%$ 的数据，$2 ≤ n ≤ 200，1 ≤m≤ 200$。

对于 $100\%$ 的数据，$2 ≤ n ≤ 1000，1 ≤m≤ 1000，1 ≤ p ≤m$。
