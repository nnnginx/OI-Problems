## 题目描述

小C今晚在公园玩得很晚才回家，路上都已经亮灯了。
路上的路灯有不同的颜色，万幸颜色种类只有26种，于是小C用小写字母表示路灯的每种颜色，不同位置路灯如果是相同颜色，则用同一字母表示，于是从公园到家里的路灯颜色可以组成一个字符串s。

小C走着走着，发现每走几步就会就会看到同一种颜色的路灯，于是，小C定义，如果字符串$s$的的每个长度不小于$k$的子串都出现了颜色$c$，则认为颜色c是$k-$重要的颜色。例如，对于颜色串$aba$，长度不小于$2$的子串有$ab,ba,aba$，他们都含有颜色$a$和颜色$b$，说明颜色$a$和颜色$b$都是$2-$重要的。

现在小C想找出最小的$k$，使得颜色串s中至少存在一种颜色是$k-$重要的，由于小C急着回家，所以他拜托你帮他解决问题。

由于小C家到公园距离很远，因此路灯数量很多，最好使用较快的输入方式，例如$scanf$。

样例解释：$a$是$2-$重要的，$b$是$4-$重要的，c是$4-$重要的，因此答案为$2$。

## 输入格式

共一行。

一个由小写字母组成的字符串$s$。

## 输出格式

一个整数，表示答案。

```input1
abacaba
```

```output1
2
```

```input2
zzzzz
```

```output2
1
```

## 数据范围

对于 $30\%$​​ 的数据，$1 \leq |s| \leq  300$​。

对于 $60\%$​​​ 的数据，$1 \leq |s| \leq  5000$​​。

对于 $80\%$​​​​ 的数据，$1 \leq |s| \leq  100000$​​​。

对于$100\%$​的数据，$1 \leq |s| \leq  5000000$​。​
