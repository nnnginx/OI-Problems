## 问题描述

众所周知，一场比赛需要一道签到题。

给定四个正整数 $n$，$l$，$r$，$k$ 以及一个只包含正整数的序列 $a$，从 $a$ 中选出若干个数值在 $[l\le r]$ 之间的数使得他们的和不超过 $k$ 且数量最多。

## 输入格式

第一行四个数表示 $n$，$l$，$r$，$k$。
第二行 $n$ 个数，表示序列 $a$。

## 输出格式

满足条件的最大数量

```input1
3 1 100 100
50 100 50
```

```output1
2
```

```input2
4 1000000000 1000000000 1000000000
1000000000 1000000000 1000000000 1000000000
```

```output2
1
```

## 样例解释

对于样例 1，选择 $50$，$50$。

## 数据范围

对于 $30\%$ 的数据 $1\le n\le100$，$1\le l,r,k,a_i\le10^9$

对于 $100\%$ 的数据，$1\le n\le10^6$，$1\le l,r,k,a_i\le10^9$
