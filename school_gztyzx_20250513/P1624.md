## 问题描述

FJ带他的 $N$ 头奶牛去参加比赛，比赛前 $N$ 头奶牛自觉的排好队，每头奶牛以它自己名字的首字母给出。但FJ对他们的顺序不满意，现在FJ想要把奶牛们重新排列，以达到他们的队列在字典顺序中最小。FJ的操作是这样的：他打算把奶牛从旧队列中一个一个拉出来，排到一个新队列去(刚开始新队列为空)，他每次只能把旧队列目前剩下的第一位奶牛或最后一位奶牛拉出来，排到新队列的最后一位去。

## 输入格式

第一行: 一个整数 $N , 1 \leq  N \leq 2,000$。

接下来有 $N$ 行，每行一个大写字母：`A`---`Z`。

这 $N$ 行字母就是题目描述的奶牛们们自觉排好队的旧队列。

## 输出格式

每行输出 $80$ 个字母，最后一行若不够 $80$ 个字母，则按实际个数输出。

```input1
6
A
C
D
B
C
B
```

```output1
ABCBCD
```

## 数据范围

说明：$6$ 头牛的旧队列：`ACDBCB`

样例输出：`ABCBCD`
