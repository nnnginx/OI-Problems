## 问题描述

期末考试即将来临，同学们正在摩拳擦掌准备制作自己的期末计划。

小仪在自己的期末计划中写上了一些数学表达式，准备计算表达式的结果。这些表达式中没有任何多余的括号，（多余的括号是指：删去一对匹配的括号后，表达式值不会发生改变，这样的一对括号就叫多余括号）。

然而，调皮的小东趁着小仪外出，偷偷在表达式上添加了一些多余的括号。但是小东很良心，他保证添加的括号都是满足数学规则的，不会出现不匹配的括号。

小仪回来后发现了表达式多了很多多余的括号，被小东气哭了！于是她请你帮他清除多余的括号！

为了简化问题，我们假设有以下规则：

1. 输入文件含有多个表达式，每个一行。
2. 表达式中所有变量都是单个大写字母。
3. 表达式中只有“+”“－”两种运算符。
   
  注意：你只能删除多余括号，而不允许在代数运算上进行处理。

## 输入格式

输入文件由若干个表达式组成。

输入的第一行为一个数$M（1≤M≤10）$，为表达式的数量。

接下来输入$M$行，每行有且只有一个语法正确的表达式。

可能存在多余空格，但每行（包括这些空格在内）长度不超过255个字符。

## 输出格式

对于每个表达式，输出去掉多余括号后的表达式。

注意必须按照案例在输入文件中的顺序输出相应答案。

每行输出一个表达式，并去除所有空格。

```input1
3
(A-B + C) - (A+(B - C)) - (C-(D- E) )
((A)-( (B)))
A-(B+C)
```

```output1
A-B+C-(A+B-C)-(C-(D-E))
A-B
A-(B+C)
```
