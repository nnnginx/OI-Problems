## 问题描述

现在有一台打字机，打字机只有 `a`-`z` 和 `L` (光标往左移动一格)，`R`（光标往右移动一格)这 $28$ 个按钮，现在给出按键顺序，问最后打出来的字符串是什么。注意：光标到了已经打出的字符串最左端的时候，即使按 `L` 光标也不会往左移动一格；最右端也是，即使按 `R` 光标也不会往右移动一格）

## 输入格式

一行只有 `a`-`z` 和 `L`，`R` 的字符串

## 输出格式

一行，为打出来的字符串。

```input1
abLcd
```

```output1
acdb
```

```input2
icpLLLLLacmRRRRRRRRRRRRc
```

```output2
acmicpc
```

## 数据范围

满足：

$30\%$ 的数据，字符串长度不超过 $1000$

$100\%$ 的数据，字符串长度不超过 $1000000$
