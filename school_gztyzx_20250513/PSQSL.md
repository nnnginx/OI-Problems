## 题目背景

今天，是一个奇特是日子，诞生了奇特的数列……

## 题目描述

求出奇特数列中奇特的数对的数量是奇数还是偶数。

定义奇特的数对：

1. 对于两数 $a_i$ 和 $a_j$，需满足 $i \ne j$。
2. 对于两数 $a_i$ 和 $a_j$，需满足 $(a_i+a_j) \ mod \ m \le x$

## 输入格式

共每个测试点共 $T$ 组数据。

对于每组数据，提供 $n,m,x,s,j,p$。

其中 $a_1=s,a_2=s+j+p,a_3=s+2j+p$ ……

**例：**

```
n=5
s=2,j=3,p=2
a[1]=2
a[2]=8
a[3]=11
a[4]=14
a[5]=17
```

## 输出格式

对于每组数据输出一行，如果是奇数，输出 `Oh Good Luck!`，如果是偶数，输出 `Oh No!`。

```input1
2
2 0 0 0 0 0
5 10 0 2 3 2
```

```output1
Oh No!
Oh No!
```

## 提示

$t\le10^5$

$n\le10^{100}$

$m,x,s,j,p\le10^{18}$
