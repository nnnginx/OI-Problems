## 问题描述

奶牛贝茜实在受不了Farmer John的虐待，于是她连夜制定了一个逃亡计划。

奶牛贝茜出逃啦！

当Farmer John听到了这个消息，十分气愤，于是他在贝茜出逃后的第 $t_i$ 秒开始追击。

可怜的贝茜只能在一条狭窄的小路上不断地跑。这条小路有一个奇怪的地方，那就是它上面布满了 $K$ 个格子，格子与格子之间没有空隙。当贝茜跑到第 $i$ 个格子时，由于各种原因，她总得花 $b_i$ 秒钟才能跑完这个格子。

而愤怒的Farmer John呢？他无论在哪一个格子，都能只用 $C$ 秒跑完。请问，贝茜能不能在不被Farmer John追上的情况下，跑完全程？如果能，输出 `Win`，如果不能，先输出 `Lose`，再换行输出贝茜是在第几个格子被Farmer John追上的。

现在贝茜正在向你求助，伟大而又聪明的你，一定能帮到她的，对吗？

## 输入格式

第一行输入两个整数 $K$ 和 $t_i$。

第二行，每行一个整数 $b_i$。

第三行输入一个整数 $C$。

## 输出格式

如果贝茜成功出逃，输出 `Win`。

否则输出 `Lose`，换行再输出在第几个格子被追上。

```input1
5 3
2 3 7 4 2
3
```

```output1
Lose
3
```

## 样例1解释

贝茜跑完第3个格子但还没跑到第四个格子所用的时间是12秒，而Farmer John也同为12秒，在第三个格子被追上。

```input2
3 1
7 6 8
9
```

```output2
Win
```

## 样例2解释

贝茜一直都比Farmer John跑得快，不会被追上。
