## 题目描述

求 $\large{\sum\limits_{i=1}^N ⌊\frac{N}{i}⌋}$ 的值

## 输入格式

一行一个整数 $N$

## 输出格式

一行一个整数，你的答案

```input1
3
```

```output1
5
```

```input2
10000000000
```

```output2
231802823220
```

## 提示

### 数据范围

- $ 1\ \leq\ N\ \leq\ 10^{12} $

### Sample Explanation 1

$ \left[\ \frac{3}{1}\ \right]+\left[\ \frac{3}{2}\ \right]+\left[\ \frac{3}{3}\ \right]=3+1+1=5 $ 。
