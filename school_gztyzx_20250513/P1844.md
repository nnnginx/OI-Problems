## 问题描述

梦之队是学校田径队的精英，现在要参加市里组织的自由组合跑比赛，比赛要求是比赛委员会临时给的时间段中，自由选择至少一员动动员参加，但是每个动动员的参赛时间都是一定的且不一定相同，而且每个学校的参加队员只能在指定的时间段里参加跑步，即整个学校的参赛时间不能超过这个时间段。

现在已知道了每个动动员在指定的时间里可以跑多少米了，如果你是教练，你将选择哪些运动员来参加比赛，使学校梦之队的总成绩(跑的距离多少米)为最大？

## 输入格式

输入的第一行有两个整数 $S（1 \leq S \leq 1000）$ 和 $N（1 \leq N \leq 100）$，用一个空格隔开，$S$ 代表学校参赛队总共跑步时间，$N$ 代表可选择的运动员数目。

接下来的 $2\sim N+1$行，每行包括两个在 $1$ 到 $100$ 之间（包括 $1$ 和 $100$）的整数，分别是一名运动员可跑时间和在这个时间段里的成绩。

## 输出格式

输出只有一行，含一个整数，表示在规定的时间内，梦之队可得到最高成绩。

```input1
100 5
77 92
22 22
29 87
50 46
99 90
```

```output1
133
```

## 数据范围

对于 $30\%$ 的数据，$M \leq 10$；

对于全部的数据，$M \leq 100$。
