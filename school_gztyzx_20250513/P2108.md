## 问题描述

昔日车神张弛考驾照失败了，于是他报名了登山运动。

他登山的路段是一段无限长的陡坡，其海拔高度从 $2$ 开始逐渐递增，登山的休息站点设立在海拔高度为 $2、3、5、7、11、13、17……$ 的位置（就是素数的位置），在两个站点之间登山，张弛需要控制好自己用到的登山力量，避免出现脱力。在两个站点之间登山需要使用的登山力量就是站点之间的海拔差，比如前 $7$ 个站点之间的海拔差就是 $1、2、2、4、2、4$。

现在，张弛位于海拔高度为 $n$ 的位置，请问他登山力量应该控制成多少？

举例说明，如果张弛位于海拔为 $15$ 的位置，他的登山力量应该控制为 $4$，因为 $15$ 的上一个站点时是 $13$，下一个站点是 $17$，$17-13=4$。

你能帮助张弛计算出登山力量顺利完成登山运动吗 ？

## 输入格式

一行，一个正整数 $N$，表示张弛所在位置的海拔。

## 输出格式

一行，一个整数 $K$，表示应该使用的登山力量。若 $N$ 所在位置本身是一个站点，输出 $0$。

```input1
10
```

```output1
4
```

```input2
11
```

```output2
0
```

## 数据范围

$20\%$  $2\leq N\leq 100$

$40\%$  $2\leq N\leq 1000$

$100\%$  $2\leq N\leq 1000000$
