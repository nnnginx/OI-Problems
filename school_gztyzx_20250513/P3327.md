## 问题描述

牛牛现在有 $n$ 个数分别是 $1,2,3,...,n$，牛牛特别喜欢数三元组，如果三个数 $a,b,c$ 满足 $b$ 是 $a$ 的倍数，$c$ 是 $b$ 的倍数，那么牛牛就觉得这三个数形成的三元组是有趣的。

现在给定 $n$ 的大小，牛牛想知道有多少个三元组 $(a, b, c)$，$a < b < c$是有趣的。

## 输入格式

一行一个整数 $n$ 表示牛牛现在有 $1,2,3,...,n$ 这些数。

## 输出格式

一个整数表示牛牛觉得有趣的三元组的个数。

```input1
10
```

```output1
9
```

## 数据范围

对于 $20\%$ 的数据，$1 \leq n \leq 500$

对于 $40\%$ 的数据，$1 \leq n \leq 1e5$

对于100%的数据，$1 \leq n \leq 1e6$
