## 问题描述

小明同学在使用oiclass时，没有按照网站管理要求备注实名，结果帐号被封禁😕

小明向管理员发送了申请解禁的邮件，没想到管理员要求他回答一下问题，如果回答出来才能给他解禁帐号。

题目是这样的：现在给出一个时间 $t$，单位是秒，要求按照电子表格式输出。

输出保证不会超过24小时。

电子表上的时间显示方法形如 xx:xx:xx。

## 输入格式

输入一行一个整数表示给定的时间秒数 $t$。（$t<86400$）

## 输出格式

一行，按电子表上的时间格式形如 xx:xx:xx。

```input1
1
```

```output1
00:00:01
```

```input2
3701
```

```output2
01:01:41
```
