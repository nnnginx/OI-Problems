## 问题描述

QWQ 有 $n$ 个数，第 $i$ 个数为 $a_i$。

在这 $n$ 数中选出 $3$ 个数有 $C^3_n$ 种选择方式，请你求出这 $C^3_n$ 种选择方式里每种选择方式中选到的最小数之和。

由于这个和可能很大，你只需要求出这个和 $\bmod 10^9 + 7$ 的值。

## 输入格式

第一行一个整数 $n$。

接下来 $n$ 个整数，第 $i$ 个整数为 $a_i$。

## 输出格式

一个整数，表示答案。

```input1
4
3 1 2 4
```

```output1
5
```

## 样例解释

有 $C^3_4 = 4$ 种选择方法。

$\min(3,1,2) = 1$

$\min(3,1,4) = 1$

$\min(3,2,4) = 2$

$\min(1,2,4) = 1$

答案为 $1 + 1 + 2 + 1 = 5$。

## 数据范围

对于 $50\%$ 的数据，$n \le 9$。

对于 $100\%$ 的数据，$3 \le n \le 10^5$，$0 \le w_i \le 10^9$。
