## 题目描述

因为阿潮惹小宇生气了，所以小宇对阿潮发起了一场冷战。但是，小宇告诉阿潮，只要他解开这道数学题，就和阿潮和好。阿潮立马欣然答应，但是他并不太会这道题，于是他把这道题给了你，让你这位聪明的同学帮忙解开这道题。

求下列方程的正整数解的组数模 $998244353$

$$
\dfrac{1}{n} + \dfrac{1}{m} = \dfrac{1}{\prod^{gcd(x,y)}_{i=1}i}
$$

## 输入格式

一行,两个整数 $x,y$ 。

## 输出格式

一行，一个整数 $ans$ 表示正整数解的组数模 $998244353$ 的值。

## 样例 #1

### 样例输入 #1

```
1 2
```

### 样例输出 #1

```
1
```

## 提示

- 对于 $30\%$ 的数据，$x,y\le 1000$ 。
- 对于 $100\%$ 的数据，$1 \le x,y\le 10^7$。
