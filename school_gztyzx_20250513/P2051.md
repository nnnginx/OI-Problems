## 题目描述

$3$ 个元素的集合 $\{5, 1, 2\}$ 的所有子集的和分别是 $0, 1, 2, 3, 5, 6, 7, 8$。发现最小的不能由该集合子集拼出的数字是 $4$。

现在给你一个 $n$ 个元素的集合，问你最小的不能由该集合子集拼出的数字是多少。

注意 32 位数字表示范围。

## 输入格式

第一行一个整数 $n$。

第二行 $n$ 个正整数 $a_i$，表示集合内的元素。

## 输出格式

一行，一个整数答案。

```input1
3
5 1 2
```

```output1
4
```

## Hint

对于 $30\%$ 的数据，满足 $n ≤ 15$。

对于 $60\%$ 的数据，满足 $n ≤ 1000$。

对于 $100\%$ 的数据，满足 $n ≤ 100000$, $1 ≤ a_i ≤ 10^9$。
