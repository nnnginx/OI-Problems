## 问题描述

有一天，一位小朋友在写数学作业。写完之后，他希望验证一下，便想到了用编程验算。他的数学作业是这样的：有 $n$ 个数和 $n-1$ 个数（符号），构成了一个表达式，他希望计算出这个表达式的值。符号用整数表示，`1` 表示加号，`2` 表示减号，`3` 表示乘号，保证没有除号。此外，本题不需要考虑先乘除后加减，一律从左到右计算。

## 输入格式

第一行是一个整数 $n$，第二行也是一个整数，表示第一个数，此后 $n-1$ 行每行两个整数，分别是第 $i-1$ 个数和此数与前一个数的符号。

## 输出格式

输出只有一行，表示此表达式的计算结果。

```input1
2
1
2 1
```

```output1
3
```

```input2
4
2
2 3
3 1
4 2
```

```output2
3
```

## 样例2解释

$2\times2=4$；

$4+3=7$；

$7-4=3$。

## 数据范围

$2≤n≤15$

保证输入数据所有数在 $[1,100]$ 中。

表达符号的数为 `1`、`2` 或 `3`。

## 提醒

注意数据范围，答案最大有多大。

不要提交 AC 记录来验证数据，见一个删一个。

验证数据规模正确性请用 `assert`。

本题不用高精度。
