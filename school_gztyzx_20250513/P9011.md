## 题目描述

学校的操场上有 $n$ 个位置，每一个位置 $i$ 上堆有 $a_i$ 个小方块。每一次你可以进行以下两个操作之一：

- 选择一个 $i$，把位置 $i$ 上的所有方块染色。
- 选择 $h,\,l,\,r$，如果 $l\sim r$ 的位置上**均有**不少于 $h$ 个方块，把这些位置上**从下往上**数的第 $h$ 个方块染上色。

求让所有方块都染上色需要的最小操作次数。

## 输入格式

第一行一个正整数 $n$，表示位置的个数。

第二行 $n$ 和整数，表示 $a_i$，即第 $i$ 个位置上的方块个数。

## 输出格式

一个整数，表示最小操作次数。

## 样例

```input1
5
2 2 1 2 1
```

```output1
3
```

```input2
10
894 1026 1714 1797 2189 2260 3077 3746 4568 5336
```

```output2
10
```

## 提示/说明

### 数据范围

对于 $40\,\%$ 的数据，$1 \leq n \leq 15$。

对于 $100\,\%$ 的数据，$1 \leq n \leq 8000$，$1 \leq a_i \leq 10^9$。

此外，有 $40\,\%$ 的数据满足对于 $i \neq j$，$a_i \neq a_j$，均匀分布在各档数据中。
