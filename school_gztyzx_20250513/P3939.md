## 问题描述

今天是圣诞节，程序龙送给你一个数组 $c_i(0\leq i \leq 9)$ ，长度为 $10$ ，以及 $n$ 和 $m$ 。

你需要构造两个十进制数正整数 $A,B$ ，其中 $A$ 有 $n$ 位，$B$ 有 $m$ 位（都允许有前导零），并且 $A,B$ 中数字 $i$ 的个数和不超过 $c_i$ 。求 $A\times B$ 的最小值。

## 输入格式

输入共两行：

第一行两个整数 $n,m$ ，表示 $A$ 和 $B$ 的位数。

第二行 $10$ 个整数 $c_i$，表示每个数字最大出现次数。

## 输出格式

输出共一行一个整数，表示 $A\times B$ 的最小值。

```input1
2 3
0 1 1 2 1 1 0 0 0 0
```

```output1
3042
```

## 样例解释

最优方案为 $A=13,B=234$。

## 数据范围

对于 $100\%$ 的数据，$1\leq n,m\leq 9,0\leq c_i\leq 10^9$ ，数据保证$\sum_{i=0}^9c_i\geq n+m$。
