## 题目描述

定义一对数是 “$0+1$” 的，仅当这两个数的数位的和相同。

现在给你 $n$ 个数 $a_i$，请求出数列内 $0+1$ 数对的数量。

## 输入格式

一个整数 $n$， 表示有 $n$ 个数。

接下来 $n$ 行，每行一个数 $a_i$。

## 输出格式

一行，一个数，表示 $0+1$ 数对的数量。

```input1
3
55
46
64
```

```output1
3
```

## 提示说明

对于 $30\%$ 的数据：$1 \le n \le 100$

对于 $100\%$ 的数据：$1 \le n \le 10^6$，$1 \le a_i \le 10^9$
