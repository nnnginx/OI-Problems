## 问题描述

小v来面试新一任cxl，面试官ckj出了道面试题。因为ckj喜欢香蕉，他喜欢的序列a是这样的：

$a[i](i<1)=0$

$a[i](i>=1)=max(a[i-a],a[i-a+1],a[i-a+2],...,a[i-1])+min(a[i-b],a[i-b+1],a[i-b+2],...,a[i-1])+i$

（所以这和香蕉有什么关系？）

现在要让小v求出ckj序列的前 $n$ 项，但他不会做，为了选上cxl，他只能找你帮忙了。

## 输入格式

输入 $n，a，b$，意义同题面。

## 输出格式

输出ckj序列的前 $n$ 项，用空格隔开。

```input1
5 3 2
```

```output1
1 3 7 14 26
```

## 样例解释

$a[1]=0+0+1=1$

$a[2]=1+0+2=3$

$a[3]=3+1+3=7$

$a[4]=7+3+4=14$

$a[5]=14+7+5=26$

## 数据范围

对于 $100\%$ 的数据 $n，a，b\leq 1000000$

保证答案不超过 long long
