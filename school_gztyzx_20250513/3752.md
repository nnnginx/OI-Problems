## 问题描述

小C给他自己和他的$n-1$个朋友准备了$n$份礼物盒，第$i$份礼物盒里面放了$a_i$个巧克力，但是小C还没有决定哪一份礼物盒留给自己。
为了让他的朋友们更开心，小C决定放弃给自己留礼物，将自己的礼物分到其他盒子里。

分配过程如下：

- 第$1$步：首先你可以选择任意的一个礼物盒。
- 第$2$步：将第$1$步中选择的礼物盒中所有的巧克力拿出来，重新分配到其它礼物盒中。

为了公平对待所有朋友，小C希望重新分配完之后所有$n-1$个礼物盒里的巧克力数量都一样。当然，由于选择的礼物盒是任意的，有可能会出现没办法使得剩余的$n-1$个礼物盒的巧克力数量都是相等的情况。

因此你可以事前在任意的礼物盒中添加若干个巧克力，使得无论你在第$1$步中选择了哪一个礼物盒，然后在第二步中把礼物盒的巧克力重新分配，肯定可以使得剩余的$n-1$个礼物盒的巧克力数量完全一致！

请你求出添加的若干个巧克力的最少数量。

## 输入格式

第一行输入一个正整数 $n$表示礼物盒的数量。

第二行输入 $n$ 个正整数，第$i$个正整数$a_i$表示第$i$个礼物盒中的巧克力数量。

## 输出格式

共一行，一个整数表示最少需要添加的巧克力。

```input1
3
3 2 2
```

```input2
4
2 2 3 2
```

```input3
3
0 3 0
```

```output1
1
```

```output2
0
```

```output3
3
```

## 数据范围

对于 $30\%$ 的数据，有$2\le n \le 10$，$0 \le a_i\le 10$ 。

对于 $60\%$ 的数据，有$2\le n \le 100$ ，$0 \le a_i\le 100$ 。

对于 $100\%$ 的数据，有$2\le n\le 10^6$，$0 \le a_i\leq 10^9$。
