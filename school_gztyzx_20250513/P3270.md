## 【问题描述】

你一定听说过骑士旅行问题（此后略去1万字）。<br />
让我们考虑另外一个骑士的问题。在这个问题上，一个骑士，在一个无穷大的棋盘上。他一开始位于点（fx，fy），目的地坐标为（tx，ty）。行走规则有m种，用ai，bi表示，那么如果骑士在(x&#44;y)点使用行走规则i，那么他将会到达点（x+ai，y+bi）。若一次行走规则算为走了一步，求到达终点的最少步数。<br />

## 【输入格式】

一个整数t（t &lt;= 20）&#44;表示测试数据组数。<br />
对于每组数据，包含四个整数fx&#44;fy&#44;tx&#44;ty。<br />
接下来一行一个整数m。表示有多少种移动方式。<br />
接下来m行，每行两个整数ai，bi，描述第i种移动方式。<br />

## 【输出格式】

对于每组数据，输出最小步数，若不能到达，则输出‘IMPOSSIBLE’。

```input1
2
0 0 6 6
5
1 2
2 1
2 2
1 3
3 1
0 0 5 5
2
1 2
2 1
```

```output1
3
IMPOSSIBLE
```

## 【数据范围】

$-5000 \leq fx，fy，tx，ty \leq 5000$，<br />
$0 < m \leq 10$;<br />
$-10 \leq mx,my \leq 10$; $|mx|+|my|>0$<br />
十个数据点的t值分别如下：<br />
$1，1，2，5，10，12，15，20，20，20$<br />
