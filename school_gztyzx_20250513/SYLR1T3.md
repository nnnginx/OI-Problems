## 题目描述

在操场里有 $n$ 个小朋友，每个小朋友都有一个TA最好的朋友 $friend_i$，还有一个快乐值 $happy_i$。

保证 $friend_i \ne i$。

现在，小朋友们需要分组做游戏，一组小朋友合法当且仅当满足以下条件：

- 对于**每一个**编号为 $j$ 的**在组里的**小朋友，满足编号为 $friend_j$ 的小朋友**在组里**。
- 对于**每一个**编号为 $j$ 的**在组里的**小朋友，**存在**一个编号为 $k ( j \ne k)$ 的小朋友，满足 $friend_k=j$ 且 $k$ **在组里**。

一个组的快乐度为这个组内的小朋友的快乐值之和。

### 提示

**不是所有小朋友都必须选到组里。**

**注意判断一个组是否可行是先选人分出一组，固定这个组的人，再检查条件。**

现在请你求所有可能的合法组别内的最大的快乐度。

## 形式化题意

集合 $S \subset \lbrace 1,2,3,\dots,n \rbrace$ 是好的当且仅当满足 $\forall i \in S,friend_i \in S$ 且 $\forall i \in S, \exists j \in S, j \ne i, friend_j = i$。

求 $\max\limits_{S \ is \ good}(\sum\limits_{i \in S}{happy_i})$ 的值。

## 输入格式

第一行一个整数 $n$。

第二行 $n$ 个整数，第 $i$ 个整数表示 $friend_i$。

第三行 $n$ 个整数，第 $i$ 个整数表示 $happy_i$。

## 输出格式

一行一个正整数，表示最大的答案。

```input1
3
2 3 1
1 10 100
```

```output1
111
```

```input2
6
2 3 1 5 6 4
7 2 8 3 2 7
```

```output2
29
```

## 样例解释

对于第一个样例，

$1,2,3$ 分成一组，所以这组的快乐度为 $1 + 10 + 100 = 111$。

对于第二个样例，

$1,2,3,4,5,6$ 分成一组，所以这组的快乐度为 $7 + 2 + 8  + 3 + 2 + 7 = 29$。

## 数据范围

| 数据点 | 范围 |
| --- | --- |
| 1~15 | $2 \leq n \leq 20$ |
| 16~30 | $2 \leq n \leq 5000$ |
| 31~50 | $2 \leq n \leq 5 \times 10^{6}$ |

每个测试点等分。

存在 $20\%$ 的数据，满足 $friend_i$ 是一个排列，均匀分布在数据点中。

对于所有数据，保证 $1 \leq friend_i \leq n , friend_i \ne i , 0 \leq happy_i \leq 10^{7}$
