## 题目描述

给出一个大整数 $A$ ， $A$ 的长度可能达到 $100$ 位，问是否能够将 $A$ 切成若干段（ $ \ge 2$ ），使得每段的数字之和相等。

## 输入格式

第一行一个整数 $n$ （ $2\le n\le 100$ ），对应整数的长度。

第二行一个包含 $n$ 个字符 $0-9$ 的字符串。

## 输出格式

如果能够划分，输出 $YES$ ，否则输出 $NO$ 。

```input1
5
73452
```

```output1
YES
```

### 样例解释

分为 $7,34,52$ ： $7=3+4=5+2$

## 数据范围

对于 $62\%$ 的数据， $2 \le n \le 8$ ；

对于 $74\%$ 的数据， $2 \le n \le 17$ ；

对于 $100\%$ 的数据， $2 \le n \le 100$ ；
