## 问题描述

可怜的小明还是小学生，程序龙就给他出了一道题。

这道题给出了一个$n$和$p$，让他求$(1!+2!+3!+…+n!) \pmod p$的值。

小明算了一个小时都没算完，他已经崩溃了。

于是他找到了你，想让你帮忙解决这道疯狂的题。

注：$n!$表示$n$的阶乘，即$1\times 2 \times \cdots \times n$

## 输入格式

输入文件只有一行，包含两个正整数$n$和$p$。

## 输出格式

输出文件只有一行，即为式子的值。

```input1
5 1000
```

```output1
153
```

```input2
7 10
```

```output2
3
```

## 数据范围

对于$20\%$的数据，$1≤n≤1000$。

对于$50\%$的数据，$1≤n≤10000000$。

对于$100\%$的数据，$1≤n≤10^{16}，1≤p≤10000000$。
