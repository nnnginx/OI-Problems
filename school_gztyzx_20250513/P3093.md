## 题目描述

众所周知，去找小新是一件困难的事情，因为你不得不爬很长的楼梯。小XL想去找小新作文辅导，但他不知道要爬多少级楼梯。于是，他想让你把楼梯画出来。

给定 $n$ 和 $m$ ，代表长为 $n$ ，高为 $m$ 的楼梯 $(n \geq m)$。你的任务是输出楼梯的阶数，并将楼梯以一个直角三角形的形式输出（楼梯以 `*` 代替）。
值得注意的是，如果 $n>m$ ，则在顶上输出一个平台。

对于样例输入，你还需要在楼梯上方的最右边输出一个 X 代表小新的位置。 （详情见样例）

由于小XL急着去找小新，没时间写代码，聪明的你可以帮帮他吗？

特别提示：输出是包含空格的哦！

## 输入格式

两个整数 $n、m$

## 输出格式

输出楼梯的阶数和楼梯的样子

```input1
3 3
```

```output1
3
  X
  *
 **
***
```

```input2
5 3
```

```output2
3
    X
  ***
 ****
*****
```

## 数据规模及约定

对于 $20\%$ 的数据，$n,m\leq 5$。
另外 $30\%$ 的数据，$n=m$。
对于 $100\%$ 的数据，$n,m\leq 10^2$ 且$ n\geq m$。
