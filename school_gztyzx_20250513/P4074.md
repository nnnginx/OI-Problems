## 问题描述

一个优秀的人工智能一定会 Stern-Brocot-Tree 吧。

定义翻转数为可以旋转 $\pi$ 后得到原数的非负整数，如 $1,69,8,010~(10)$ 。特别地，翻转数允许前导零存在，但输出时请不要输出前导零。

求与一个数相差最小的翻转数。

## 输入格式

第一行两个整数 $n$ 表示询问个数。

接下来 $n$ 行，每行一个整数 $a_i$ 表示每个询问。

## 输出格式

对每个询问，输出一行一个整数表示答案。如果有多个解，输出最小的那一个。

```input1
6
0
1
101
105
256
1000
```

```output1
0
1
101
101
181
1000
```

## 数据范围

对于 $30\%$ 的数据，满足 $a\leq1000$ 。

对于 $50\%$ 的数据，满足 $a\leq10^7$ 。

对于 $80\%$ 的数据，满足 $a\leq10^{300}$ 。

对于 $100\%$ 的数据，满足 $n\leq10,0\leq a\leq10^{3000}$ ，$a$ 无前导零。

$\pi$ 为 $180^\circ$ 。
