## 题目描述

给定一个数 $n$ 和一个数 $m$。

求 $n \bmod m$ 的值。

## 输入格式

第一行一行一个数 $m$。

第二行一行一个数 $n$。

## 输出格式

一行一个数，表示 $n \bmod m$ 的值。

```input1
114514
1919810
```

```output1
87586
```

```input2
234142341
34524234562624652353243254198459182942874361982746591823754913846759134875
```

```output2
28499456
```

## 数据范围

对于 $20 \%$ 的数据，$n \leq 10^{18}$。

对于所有数据，$1 \leq n \leq 10^{10^{6}}$，$2 \leq m \leq 10^{17}$。
