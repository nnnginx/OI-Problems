## 题目描述

给出长度为 $n$ 的序列 $a$ 和整数 $k$ 。

你可以将序列分成 $k$ 段，从左到右按顺序标号。

令 $f(i)$ 表示位置 $i$ 属于的段的编号。

求 $ \sum\limits_{i=1}^{n} (a_i \cdot f(i)) $ 的最大值。

## 输入格式

第一行两个整数， $n,k$ 。

第二行 $n$ 个整数，表示序列 $a$ 。

## 输出格式

输出最大值。

```input1
5 2
-1 -2 5 -4 8
```

```output1
15
```

```input2
7 6
-3 0 -1 -2 -2 -4 -1
```

```output2
-45
```

```input3
4 1
3 -1 6 0
```

```output3
8
```

对于 $100\%$ 的数据，满足 $|a_i|\le 10^9,1\le k\le n\le 10^6$ 。

对于 $50\%$ 的数据，满足 $n\le 5$ 。

对于 $90\%$ 的数据，满足 $n\le 5000$ 。

对于 $100\%$ 的数据，满足 $n\le 10^6$ 。
