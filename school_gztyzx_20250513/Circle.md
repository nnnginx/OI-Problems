
## 题目描述

你学习了一些超前的数学知识。

但是你遇到了一些麻烦，因此你需要借助计算机来帮你计算。

在平面直角坐标系 $xOy$ 中，给定四个点的坐标 $(x1,y1),(x2,y2),(x3,y3),(x4,y4)$ 。

若这四个点共圆，则输出 $1$ ，并且输出圆心的坐标 $(x,y)$ 以及这个圆的半径 $r$ 。

若这四个点不共圆，则输出 $2$ ，并且输出前三个坐标构成的圆的圆心坐标 $(x,y)$ 以及前三个点构成的圆的半径 $r$ 。

由于精度问题，本题采用SPJ，只要你输出的圆心坐标与正确答案的差距不超过 $0.02$ 且半径与正确答案的差距不超过 $0.1$ 即可。同时，在判断是否共圆时，若坐标差距不超过 $0.3$ 即可判断其共圆。

## 输入格式

四行，每行一组实数，分别为： $(x1,y1),(x2,y2),(x3,y3),(x4,y4)$ 。

## 输出格式

一行，一个整数，三个实数，含义如题所示，实数保留六位小数。

## 样例 #1

### 样例输入 #1

```
0 5.46
2 6
6 2
2 -2
```

### 样例输出 #1

```
1 2.005591 2.005591 3.994413
```

## 提示

### 样例解释

如图，四点共圆。但是样例输出的 $x$ 和 $y$ 坐标都与数学标准答案差了 $0.005591$ ，这是因为取 $A$ 点的 $y$ 坐标时出现了精度丢失的问题。而在判断是否共圆时，坐标差约为 $0.089$ ，因此判断其四点共圆。

### 数据范围

本题分为 $4$ 个 Subtask 评测。

| Subtask | 数据点个数 | 总分值 | 特殊性质 | 
| :----------: | :----------: | :----------: | :----------: |
| Subtask $1$ | $2$ | $15$ | 保证所有点的横纵坐标的绝对值均小于 $10$  | 
| Subtask $2$ | $3$ | $23$ | 保证所有点的横纵坐标的绝对值均小于 $500$ | 
| Subtask $3$ | $3$ | $26$ | 保证所有点的横纵坐标的绝对值均小于 $5000$ | 
| Subtask $4$ | $2$ | $36$ | 对于给出的四个点 $A,B,C,D$ ，保证 $AB \times CD + AD \times BC = AC \times BD$ |

对于 $100\%$ 的数据，保证所有点的横纵坐标的绝对值小于 $10000$ 。

注意，你需要通过一个 Subtask 内的所有测试点才能拿到该 Subtask 的分值。
