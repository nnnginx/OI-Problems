## 题目描述

输入$3$个整数$A, B, C$，如果其中两个整数相等且不等于另一个整数，则称这三个数$“poor”$。

如果这三个整数“poor”，输出```Yes```；否则输出```No```

## 输入格式

输入三个正整数 $ A $、 $ B $、 $ C $。

## 输出格式

输出```Yes```或者```No```


```input1
5 7 5
```

```output1
Yes
```

```input2
4 4 4
```

```output2
No
```

```input3
4 9 6
```

```output3
No
```

## 数据范围

- $ 1 \leq A,\ B,\ C \leq 9$
