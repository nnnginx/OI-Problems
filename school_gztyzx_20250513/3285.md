## 题目描述

放暑假啦！阿潮带着他的小宇出去玩。他们来到了一个游乐场旁边，发现了一个牌子，上面写着：

$$
“新店大挑战！答对一题即可免费游玩”
$$

阿潮高兴坏了，他为了在小宇面前展现自己的聪明才智，于是欣然接受了挑战。但当他看到题时，他懵了：

●设 $z=\neg\left[\left(\neg p \land q\right) \vee \left(q \oplus p \rightarrow q \right)\right] \leftrightarrow \left[\left(p \land \neg q \right)\land \left(q \oplus p \vee q\right) \vee p \right] \land \neg q$ ， $q^p = 1$ ，现在给你两个整数 $n$ 和 $k$ ，求下面式子的值:

$$
\frac{\sum_{i=1}^{\left(n+k\right)^2} \left(\prod_{j = 1}^{k}j \times z^n\right)}{\sum_{i=1}^{ 2nk +\left(\prod_{l=1}^{2}k\right)+n^2}1} \pmod {10^9 + 7}
$$

但是，阿潮忽略了牌子上的一行小字：“再给出一个九位数 $x$ ，如果它的前 $i$ 位都能被 $i$ 整除的话，那么就将式子乘 $2$ 再输出 。”你也知道，阿潮是班长。如果你帮助他做出这题的话，他会给你亿点点德育分哦！

## 输入格式

一行，三个整数 $n$ ， $k$ 和 $x$  。

## 输出格式

若只有一个答案，则输出一行，一个整数 $d$ ，表示结果。

若有多个答案，则输出一行多个整数 $d_i$ ，表示结果，答案从小到大排序。

## 样例 #1

### 样例输入 #1

```
1 1 123456789
```

### 样例输出 #1

```
1
```

## 提示

对于 $50\%$ 的数据， $n \leq 10^9$

对于 $100\%$ 的数据， $k \leq 10^{8}$ , $1 \le n \le 10^{1639}$

提示：命题逻辑符号运算优先级从大到小排序分别是

|  优先级排名| 符号 | 符号名称 |
| :----------: | :----------: |:----------: |
| 1 | $\neg,\oplus$ | 非、异或 |
| 2 | $\land$ | 逻辑与（合取） |
| 3 | $\vee$ | 逻辑或（析取） |
| 4 | $\rightarrow$ | 条件运算 |
| 5 | $\leftrightarrow$ | 双条件运算 |
