<h2>Description</h2>
&nbsp; &nbsp; 宁静者，为某校的某人，似乎从来就没有这个人似的……<br />
&nbsp; &nbsp; 但是，他依然在追求自己的梦想。<br />
&nbsp; &nbsp; 数学成绩出来，某T折戟沉沙，涉险及格。而宁静折取桂冠。然而为了E文测验，宁静带了个九连环回学校供她欣赏。<br />
&nbsp; &nbsp; 宁静：这个就是经典的九连环啦！九连环的每个环互相制约，只有第一环能够自由上下。要想下/上第n个环，就必须满足两个条件：<br />
&nbsp; &nbsp; 一、第n-1个环在架上；<br />
&nbsp; &nbsp; 二、第n-1个环前面的环全部不在架上。<br />
&nbsp; &nbsp; 这样一来的话如果我想解第一个环的话就是这样，两个环的话我先也可以直接下。三个环的时候就这样这样：<br />
&nbsp; &nbsp; 1、下1环<br />
&nbsp; &nbsp; 2、下3环<br />
&nbsp; &nbsp; 3、上1环<br />
&nbsp; &nbsp; 4、下2环<br />
&nbsp; &nbsp; 5、下1环<br />
&nbsp; &nbsp; 某T：那么，下9个环呢？<br />
&nbsp; &nbsp; 宁静双手赛百鸟，卡擦卡擦在10分钟之内全部下了。<br />
&nbsp; &nbsp; 某T：好玩好玩~，这个给我。<br />
&nbsp; &nbsp; 宁静：喂喂……<br />
&nbsp; &nbsp; 某T：这样吧，明天我问你当我要下N个环时需要多少步，如果你说出来的话一切好说，说不出来这个玩意归我了。<br />
&nbsp; &nbsp; 宁静：……（悲哀）<br />
&nbsp; &nbsp; 宁静无论如何也不会将时间浪费于和某T折腾，于是他想到了oier……<br />
<h2>Input Format</h2>
一个正整数N（1&lt;=N&lt;=800）,代表要拆卸的环数
<h2>Output Format</h2>
一个正整数M，代表所需的步数

20
</code></pre><pre><code class="language-output1">
699050
</code></pre>
