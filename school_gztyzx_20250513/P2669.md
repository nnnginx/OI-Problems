## 【问题描述】

在一片浓密的森林里生活着两种动物，牛和羊。久而久之，这两种动物形成了一种平衡：当野牛的数量和羊的数量一样多时，它们彼此互不侵犯，但只要牛的数量更多时，牛群就会用牛角恶狠狠地把羊群赶走；同样的，当羊群更多时，羊群也会用尖角把牛群赶走。</br>
有一天，科学家们发现了这个地方，他们把牛和羊分别用1和0来代表，0和1组成了一个字符串$S$，$S$表示一个区域内牛和羊的总数和分布情况，科学家可以按先后顺序带走字符串前或者后的牛和羊，请你根据这个字符串$S$统计，在科学家们的帮助下，能够共存的最多的牛和羊的区域，这个区域有多少头牛和羊？

## 【输入格式】

一个字符串，只包含$01$，长度不超过1000000。&nbsp;

## 【输出格式】

一行一个整数，最长的0与1的个数相等的子串的长度。

```input1
1011
```

```input2
0010100
```

```input3
111
```

```output1
2
```

```output2
4
```

```output3
0
```

## 【样例解释】

对于1011而言，它表示森林内牛和羊分布情况为：[牛&#44;羊&#44;牛&#44;牛]
那么最多只有[羊&#44;牛]这2只动物存在于某个区域内，所以结果为2

## 【数据范围】

对于10%的数据，字符串长度$≤10$；</br>
对于100%的数据，字符串长度$≤1000000$。
