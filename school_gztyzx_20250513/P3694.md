## 问题描述

在一个抛硬币游戏中，翻到正面的几率有多少？显然是$\frac{1}{2}$。

那么，连续两局都翻到正面的几率呢？由于首先要赢第一局有 $\frac{1}{2}$ 几率，而在其基础上再赢一局也只有 $\frac{1}{2}$ 几率，总的几率为 $\frac{1}{2}*\frac{1}{2}=\frac{1}{4}$。

以此类推，连续n局都翻到正面的几率为：$(\frac{1}{2})^n=1/(2^n)$。

HIM的生日会上大F云集，在酒足饭饱之余，他们想到了一个有趣的游戏：抛硬币。他们约定，谁先翻到 $n$ 次正面，谁就赢得比赛。

可是，HIM日理万机，他并无时间与大家苦战，于是，他决定用自己的超人智力速战速决――他要一下子连赢 $n$ 局！不过，HIM作为一个理智的大F，他也觉得这有很大难度，于是他决定要算出连赢 $n$ 局的概率。但他太忙了，他把任务给了您，并请您在一秒内给出答案。不过，HIM不喜欢漫长的小数，因此他请您给这个概率乘上 $m$，然后保留整数就好了。

当然，大F毕竟是大F，他们还有一个特殊规则：为纪念概率论的先驱之一帕斯卡（Pascal，1623―1662）――同时也是世界上第一台数字计算器的发明人，本题将以计算机特有的二进制来进行 $m$ 的输入，并且结果也需用二进制输出！（但 $n$ 的输入是十进制，因为别人并无这种需求……）

## 输入格式

输入共三行：

第一行一个十进制数 $n$；

第二行一个十进制数 $t$，表示二进制数 $m$ 的位数；

第三行一个 $t$ 位二进制数 $m$，各位之间以空格间隔。

$n，m$意义见题目描述。

## 输出格式

输出连赢 $n$ 局的概率乘上 $m$ 的结果，保留整数（无需以空格间隔）。

```input1
4
7
1 0 1 0 0 0 1
```

```output1
101
```

## 样例解释

$m$ 的十进制为 $81$，连赢 $n$ 局的概率为 $\frac{1}{2^4}=\frac{1}{16}$，结果为 $81*\frac{1}{16}=\frac{81}{16}=5.0625$，保留整数为 $5$，转为二进制即为 $101$。

## 数据约定

数据保证 $n≤t$。

$50\%$的数据：$n≤10，t≤20$；

$100\%$的数据：$n≤t≤10^4$。

## 提示

二进制，即是以$2$为基数的进制，逢$2$进一，因此在二进制数中只有$0$和$1$。
