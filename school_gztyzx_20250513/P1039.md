## 问题描述

给你$2$个分数，求他们的和$\frac{a}{b} + \frac{c}{d}$，并要求和为最简形式。

## 输入格式

一行包含四个正整数$a,b,c,d(0\lt a,b\lt 20,0\lt c,d\lt 10^9)$，表示两个分数 $\frac{a}{b}$ 和 $\frac{c}{d}$。

## 输出格式

输出两个整数 $e$ 和 $f$，表示 $\frac{a}{b} + \frac{c}{d}$ 的最简化结果是 $\frac{e}{f}$ ，每组输出占一行。

```input1
1 2 1 3
```

```output1
5 6
```

```input2
4 3 2 3
```

```output2
2 1
```
