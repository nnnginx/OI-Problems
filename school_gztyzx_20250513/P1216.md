## 问题描述

某大型企业有 $N(3\leq N\leq 100,000)$ 条生产线，每条生产线的产能为 $P_i（1\leq P_i\leq 10，000)$。由于该企业的霸主地位，其他竞争者生存艰难。为了继续生存，某竞争企业主打算只身进入该企业，以破坏这些生产线。为了节约时间，他只破坏连续的一部分生产线（从第 $i$ 条生产线破坏到第 $j$ 条生产线，$2\leq i\leq j\leq N-1$）。由于第 $1$ 条和最后一条生产线上装有监控，因此，他绝对不会破坏这两条生产线。此外，既然来了，他也不会什么都不做，因此他至少破坏一条生产线。现在他想知道，他该如何破坏才能使得剩下的生产线的平均产能最低（平均产能 = 产能总和 / 生产线数量）。


## 输入格式


输入数据共若干行。第一行包含一个正整数 $N$。接下来 $N$ 行，每行一个正整数 $P_i$，表示第 $i$ 条生产线的产能。

## 输出格式

输出数据仅一个实数，表示可能的最低平均产能，保留三位小数。

```input1
5
5
1
7
8
2
```
```output1
2.667
```


## 样例说明

破坏第三、四两条生产线后，总产能为 $5+1+2=8$，平均产能为 $8/3$。
