## 题目背景

**题目背景对解题无意义**

这天是全国青少年信息学奥林匹克（NOI）大赛的颁奖典礼，Lily 和 Chxulong 等铁一中的老师们在紧张等待。时间缓慢流逝，空气中弥漫着不安与期待。突然，JY123 提议大家玩个博弈论游戏来打发时间，并缓解紧张的气氛。Lily 和 Chxulong 等人对此提议很感兴趣，纷纷同意参与游戏，期待通过这个机会放松心情并展示各自的策略能力。

## 题目描述

现在是 Lily 和 Chxulong 的回合，刘老师给出了一个长为 $n$ 的字符串，这个字符串由 `a,b` 构成，形如 `abab..ab..`，他说：“林老师、陈老师，您们都是绝顶聪明的吧！每次你们都可以从字符串中去除一个字母，并且林老师，您是先手，希望最后剩下两个完全相同的字母，然而陈老师，您希望剩下不同的字母。现在开始游戏吧！”。

## 输入格式

输入一个 $n$ 表示字符串的长度。

## 输出格式

输出 `Lily Win!` 或 `Chxulong Win!` 分别表示是林老师胜利还是陈老师胜利。

## 样例 #1

### 样例输入 #1

```
2
```

### 样例输出 #1

```
Chxulong Win!
```

## 样例 #2

### 样例输入 #2

```
3
```

### 样例输出 #2

```
Lily Win!
```

## 提示

$2 \le n\le 10^9$

### 样例解释

样例一：

字符串长度为 $2$，字符串对应是 `ab`，不可能剩下两个相同的，陈老师胜利！

样例二：

字符串长度为 $3$，字符串对应是 `aba`，第一局林老师去掉 `b` 剩下两个是相同的，林老师胜利！
