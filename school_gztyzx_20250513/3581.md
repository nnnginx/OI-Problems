## 题目背景

小N最近学习了进制，于是想挑战一下进制加法。

## 题目描述

小$N$有 $n$ 个 $k$ 进制的整数，他想求出这些数的和对应的 $k$ 进制数，他来求助你。

## 输入格式

第一行，两个整数 $n$ 和 $k$ ，表示有 $n$ 个 $k$ 进制的数。

接下来 $n$ 行，每行一个 $k$ 进制的数 $v$ 。

## 输出格式

第一行，一个 $k$ 进制数 $sum$ ，表示答案。

## 样例 #1

### 样例输入 #1

```
3 2
1010
101
1
```

### 样例输出 #1

```
10000
```

## 样例 #2

### 样例输入 #2

```
2 16
AB
15
```

### 样例输出 #2

```
C0
```

## 提示

### 样例解释

### #1

$(1010)_2+(101)_2=(1111)_2$

$(1111)_2+(1)_2=(10000)_2$

### #2

$(AB)_{16}+(15)_{16}=(C0)_{16}$

### 数据范围

对于 $50\%$ 的数据，$1\le n\le 10^2$，$2\le k\le 10$ ，$|v| \le 5$ 。

对于 $100\%$ 的数据，$1\le n\le 10^5$，$2\le k\le 16$ ，$|v|\le 10$ 。
