## 题目描述

要开学了，康娜要交她的社会实践和研究性学习了。

社会实践总共要交三种材料，分别需要 $a,b,c$ 个。

可是康娜现在发现她每种材料分别有 $x,y,z$ 个，可能不够。

但是我们的康娜是龙呀~她会魔法的对不对？

她的魔法是这样的：随便挑选两份相同的材料，然后把它变成一份自己想要的材料。

但是问题是，康娜发现使用了魔法仍然可能无法凑齐足够的材料，因此她需要你帮她判断一下她能否顺利交作业呢？

## 输入格式

第一行一个整数 $T$ ，表示数据组数。

接下来 $T$ 行，每行 $6$ 个数，分别表示 $x,y,z,a,b,c$

## 输出格式

共 $T$ 行

对每组数据，如果能够达成输出 `YES`，反之输出 `NO`。

注意大小写，且最终输出均不含引号。

```input1
3
4 4 0 2 1 2
5 6 1 2 7 2
3 3 3 2 2 2
```

```output1
YES
NO
YES
```

## 数据范围

对于 $30\%$，$T≤100$

对于 $60\%$ 的数据，$1≤a,b,c,x,y,z≤2147483647$（C++标准下int），$T≤10000$

对于 $100\%$ 的数据，$1≤a,b,c,x,y,z≤10000000000000000$， $1≤T≤100000$
