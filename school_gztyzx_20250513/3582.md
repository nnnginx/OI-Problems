## 题目背景

小N是一个桌球爱好者，现在他正在练习。

## 题目描述

球台由四条边组成，每条边都可以反弹，当球碰到台边时，球会反弹，如图：

![](https://cdn.luogu.com.cn/upload/image_hosting/m7dxt8wk.png)

现在只看一条边，把球看作一个点，给出两个球的坐标（台边看作 $x$ 轴），问需要把第一个球打到哪个点（台边反弹点），才能**正好**打中第二个球。

小N如果可以提前计算出打击点，他比赛时获胜的几率也会更高。你能帮助他吗？

## 输入格式

四个正整数，表示两个球的坐标 $(y1,x1)$，$(y2,x2)$，按顺序输入。

## 输出格式

打击点的 $x$ 坐标，要求保留三位小数。

## 样例 #1

### 样例输入 #1

```
1 1 3 2
```

### 样例输出 #1

```
1.250
```

### 数据范围

对于 $30\%$ 的数据，$1\le y1,y2\le 10^3$，$0\le x1<x2\le 10^3$ 。

对于 $60\%$ 的数据，$1\le y1,y2\le 10^5$，$0\le x1<x2\le 10^5$ 。

对于 $100\%$ 的数据，$1\le y1,y2\le 10^9$，$0\le x1<x2\le 10^9$ 。

另外，其中有 $20\%$ 的数据满足 $y1=y2$ 。
