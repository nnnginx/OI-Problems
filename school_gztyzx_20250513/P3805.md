## 【问题描述】

给定正整数$n$。<br />
对于一个数$x$，一个块为一段由相同数字组成的连续区间。如$x=0003533$，有$1$个长度为$3$的块$000$，$1$个长度为$2$的块$33$，$2$个长度为$1$的块$3，5$。<br />
现求出在所有长度为$n$的整数（可含前导零）中，有多少个长度为$i(1 \leq i \leq n)$的块。如果$n=3$，那么这些整数为$000，001,002...998,999$。<br />

## 【输入格式】

一个整数 $n$。

## 【输出格式】

一行$n$个整数，第$i$个数$a_i$表示有$a_i$个长度为$i$的块。

```input1
1
```

```input2
2
```

```output1
10
```

```output2
180 10
```

## 【数据范围】

对于$30\%$的数据满足 $1\leq n \leq 10^3$；<br />
对于$100\%$的数据满足 $1 \leq n \leq 10^5$。<br />
