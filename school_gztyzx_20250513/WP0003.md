# Background

**坚持中国共产党领导的多党合作和政治协商制度的意义？（我国政党制度的优越性）**

①中国共产党领导的多党合作和政治协商制度，是发扬社会主义民主的重要形式。

②有利于反映民意，集中民智，促进科学民主决策。

③有利于维护社会稳定和谐。

④有利于推进祖国和平统一大业。

# Description

用 $n$ 个点组成 S-Tree，问高度大于等于 $h$ 的有多少个。

多测。

S-Tree：

1. 有 $n$ 个节点，编号由 $1$ 至 $n$。
2. 每个节点最多只有两个子节点。
3. 一个节点左儿子及其子树中所有节点的编号都小于这个节点的编号。
4. 一个节点右儿子及其子树中所有节点的编号都大于这个节点的编号。

# Format

## Input

一行一个数 $t$，表示数据测试数量。

接下来 $t$ 行，每行两个数 $n$ 和 $h$。

## Output

输出有 $t$ 行，每行一个数。

答案 $\bmod \ 10^9+7$。

# Samples

```input1
5
6 5
10 7
7 1
7 3
6 6
```

```output1
88
10880
429
429
32
```

# Limitation

对于 $20 \%$ 的数据，$h\leq n\leq 5$。

对于 $100 \%$ 的数据，$h\leq n\leq 500$，$t\leq 10^5$。
