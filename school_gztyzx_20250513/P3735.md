## 问题描述

有一个 $n\times m$ 的矩阵，开始矩阵上有些地方被覆盖的，有些地方是空的。`*` 表示被覆盖，`.` 表示没有被覆盖。

现在有无穷多个 $1\times 1$ 和 $1\times 2$ 的木块，其中每个 $1\times 1$ 的木块需要花费 $x$ 元。每个 $1\times 2$ 的木块需要花费 $y$ 元，并且只能横着放（不能旋转）。你需要用这些木块将所有空地覆盖，并且不能重复覆盖。请求出所需要的最小花费。

## 输入格式

第一行有四个整数 $n,m,x,y$，表示矩阵的大小和木块的价格。

接下来 $n$ 行，表示矩阵。

## 输出格式

一个整数，表示覆盖所有空地的最小花费。

```input1
3 3 3 7
..*
*..
.*.
```

```output1
18
```

## 数据范围

对于 $100\%$ 的数据，满足 $1 \leq n \leq 100,1 \leq m \leq 1000, 1 \leq x,y \leq 1000$。
