## 问题描述

一个优秀的仙人一定会种树吧。

很会种树真君（？）家的机关呈树状，你标记出了一条有起点和终点的简单路径，申鹤在这条路径上的其中一个节点（包括但不限于两端点），并且正要前往终点。

申鹤非常熟悉这棵树的形状，但她并不知道任何有关这条路径的信息（除了它是简单路径和她现在在路径上）。

这里白雾弥漫，她只能找到这条路径从当前节点往起点和终点方向的下一个节点（如果有的话。如果没有，她会知道这里是起点还是终点），而不能分辨出它是起点终点方向中的哪一个，也不知道它只是普通节点还是就是端点，更不知道再往后的节点的情况（对她来说该点子树中所有的点是端点的概率是一样的）。只有在到了路径两端端点后才能确认该点是起点还是终点。但是申鹤不会迷路，也即，她不会在错误到达起点后，忘记起点所在的方向，在去终点的路上走回头路。

现在，请求出在采取最优策略情况下，所有情况的期望路程和。

特别地，对于起点、终点、初始位置之一不同，即视为不同情况。任意两个不同情况出现概率完全相同。答案对 $998244\underline{\dot{\check{8}}}53$ 取模。

## 输入格式

第一行一个整数 $n$ 。

接下来 $n-1$ 行，每行 $2$ 个整数 $u_i,v_i,w_i$ ，表示树的一条边从 $u_i$ 到 $v_i$ ，长度为 $w_i$ 。

## 输出格式

一行一个整数，表示期望路程和。

```input1
2
1 2 1
```

```output1
2
```

```input2
3
1 2 1
2 3 1
```

```output2
12
```

```input3
4
1 2 1
1 3 1
1 4 1
```

```output3
30
```

```input4
4
1 2 1
2 3 1
3 4 1
```

```output4
38
```

### 样例解释

\#3 当申鹤在一号节点，起点为 3 号节点，终点为 4 号节点时，她能知道 3、4 对应的是两个端点方向。因为两点后面没有点了，所以她能确定这两点都是端点，但她也不知道哪个是终点。如果她以随机的策略选择一个点走下去，那么就有期望 $50\%$ 的概率走到起点， $50\%$ 的概率走到终点。走到起点的话，还要再走两步到达终点；走到终点就一步到位，因此这一种情况下平均期望步数是 $3\times0.5+1\times0.5=2$ 。

## 数据范围

对于 $30\%$ 的数据，满足 $n\leq500$ 。

对于 $50\%$ 的数据，满足 $n\leq5000$ 。

对于另外 $20\%$ 的数据，满足树是一条链或菊花图。

对于 $100\%$ 的数据，满足 $n\leq5\times10^5,1\leq u_i,v_i\leq n,0\leq w_i<998244353$ 。
