## 题目描述

“你知道你在干什么吗？你知道你应该干什么吗？你知道我知道你在干什么吗？”―― $chxulong$ 语录

美好的一天，从抓腐开始。

但是 $chxulong$ 的抓腐利器极域出了点问题，它连一个学生电脑上的被控端都连接不上！这让 $chxulong$ 为了难，于是他委托你帮他修一下极域的 $bug$。

现在极域的 $IP$ 换算出了故障，每个 $IP$ 地址都是由4个十进制数表示(两数之间点隔开)的，例如 $192.168.72.252$ 。但在传输时，往往把地址转换成二进制数传输，现已知每个十进制数对应8个二进制位，也就是说，传输一个IP地址需要32个二进制位。现在有 $n$ 个 $IP$ 地址，请你求出它们总共需要多少个二进制位。

## 输入格式

一个整数 $n$。

## 输出格式

一个整数，即 $n$ 个 $IP$ 地址共需要多少个二进制位。

```input1
9
```

```output1
288
```

## 数据规范

$1 \leqslant n \leqslant 10^6$
