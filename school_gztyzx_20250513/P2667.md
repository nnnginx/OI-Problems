## 【问题描述】

新初一的同学们将会在九月份正式入学！小潘老师有一份存在学生学号的纸质名单，现在作为新班主任的小潘老师需要给$n$个学生按学号的先后顺序排好！</br>
但是小潘老师的空闲时间很短很短（太夸张了吧）！所以她想把排学号的任务分成多次来做。因此小潘老师让你想尽办法把这个排序的任务分成多次来做。也就是要把学生的纸质名单进行分组（分组不打乱原来的顺序），使得小潘老师只需要把每组的学生的学号分别排序，就能把全班同学的纸质名单按学号先后顺序排好序。</br>
纸质名单开始时的次序为$a[i]$</br>
你能帮小潘老师最多把纸质名单分成多少个小组，保证名单上面的学号为$0...n-1$的一个排列。<br />

## 【输入格式】

第一行一个数$n$；<br />
第二行n个数表示$a[i]$，以空格隔开。
$n\leq 100000$<br />

## 【输出格式】

输出一个数，表示最多分出多少个小组。

```input1
5
4 3 2 1 0
```

```input2
3
2 1 0
```

```input3
8
2 1 0 3 7 5 4 6
```

```output1
1
```

```output2
1
```

```output3
3
```

## 【样例解释】

将卷子分成2组或者更多组，都无法得到所需的结果。
例如，分成 [4&#44; 3]&#44; [2&#44; 1&#44; 0] ，排序得到的结果是 [3&#44; 4&#44; 0&#44; 1&#44; 2]，这不是有序的数组。<br />

## 【数据范围】

对于20%的数据，$1≤n≤20$；<br />
对于53%的数据，$1≤n≤1000$；<br />
对于60%的数据，$1≤n≤2000$；<br />
对于100%的数据，$1≤n≤100000$。
