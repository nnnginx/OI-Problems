## 问题描述

大家都知道蒙氏数学教学法是面向幼儿园孩子的一种数学教学法，实际上就是使用大量的教具来帮助孩子们掌握数字的概念。这其实是很普通的教学方法，只不过披上了一个洋名字后就身价倍增了。

今天的课程中，老师使用了喜羊羊公仔作为教具，她先在自己左边放了几只喜羊羊，又在自己右边放在上几只，然后让孩子们数出左右两边各有多少只喜羊羊，并且要计算出两边一共有多少只喜羊羊。

小芷虽然才5岁，虽然没有进入过蒙氏数学班学习，但是她已经展现出了高人一等的数学才能，不仅喜羊羊的加法她会做，就连灰太狼的加法她也会做呢。而且，她还希望你帮她编写一个程序，解决老师出的喜羊羊问题。

## 输入格式

只有一行，共 $2$ 个数字 $n,m$，分别表示左右两边喜羊羊的个数。

## 输出格式

只有一个数据，即左右两边喜羊羊的数量和。

```input1
4 5
```

```output1

```

## 样例解释

左边有 $4$ 只喜羊羊，右边有 $5$ 只喜羊羊，一共有 $9$ 只喜羊羊。

## 数据范围

如果不满足输出格式得 $0$ 分；

对于 $100\%$ 的数据：$0\leq n,m\leq 30000$ 。
