## 题目描述

在星期一的早上 ，有$n$个学生随意凌乱地站在划分为网格的操场上。所有学生都站在网格的点上，网格点由整数坐标$(x,y)$表示。

学生们可以沿着网格向上、下、左、右移动，一步移动一个格子的距离，但在同一时刻任一网格点上只能有一名学生。按照老师的命令，学生们要整齐地列成一个水平队列准备进行升旗仪式，即排列后学生的坐标为$(x,y),(x+1,y),…,(x+n-1,y)$。

升旗仪式马上开始了，如何选择$x$ 和$y$的值才能使学生们以最少的总移动步数排成一列。

计算使所有学生排成一行需要的最少移动步数。

## 输入格式

第$1 $行是学生数$n，1≤n≤10000$。接下来$n $行是学生的初始位置，每行$2$ 个整数$x$和$y$，$-10000≤x，y≤10000$。

## 输出格式

第1 行中的数是学生排成一行需要的最少移动步数。

```input1
5
1 2
2 2
1 3
3 -2
3 3
```

```output1
8
```
