## 问题描述

小龙有一个括号序列，他定义一个合法的序列为：

1. 空串为合法序列
2. 如果 $S$ 为合法序列，那么 $(S)$ 也为合法序列
3. 如果 $S,K$ 为合法序列，那么 $SK$ (合并)也为合法序列

小龙现在有一个仅由括号组成的括号串（不一定合法），他现在想知道最长的合法括号串的长度为多少。

## 输入格式

输入一行一个字符串 $S$ ，表示小龙的字符串。

## 输出格式

输出仅一行，一个数，表示最长的合法字符串。

```input1
(()))(()(
```

```output1
4
```

样例一解释：可以找到的最长的括号序列为 $(())$ 。

```input2
))))((((
```

```output2
0
```

样例二解释：只有空串合法。

## 数据范围

对于 30% 的数据， $1 \le |S| \le 10^3$;

对于 100% 的数据， $1 \le |S| \le 10^7$
