## 问题描述

第三位面试官小 Y 的题目再一次难住了小 V……

小 Y 又给了你 $n$ 个数，$a_1, a_2, a_3...a_n$ 分别是 $1, 2, 3 ... n$，你要把这 $n$ 个数重新排序。

小 Y 定义一个排序的得分是 $|a_1 - a_2| + |a_2 - a_3| + |a_3 - a_4| + ... + |a_{n - 1} - a_n|$。

现在给你 $n$，请你告诉小 Y 最大的得分。<br />

## 输入格式

一个整数 $n$。

## 输出格式

一个整数，代表最大得分。

```input1
5
```

```output1
10
```

## 样例解释

把 $1,2,3,4,5$ 重新排序变成 $5,1,4,2,3$。

$|5 - 1| + |1 - 4| + |4 - 2| + |2 - 3|$

$= 4 + 3 + 2 + 1$

$= 10$

## 数据范围

对于 $100\%$ 的数据，$2 \le n \le 2 * 10^9$。
