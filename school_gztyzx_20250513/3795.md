## 题目背景

TYoi建队五周年来，已经形成了一个充满热情的团队，chxulong 是大boss，今天他要赐予TYoier热情的火花。

## 题目描述

chxulong 设定TYoier的火花值有 $x, y, z, w$ 四个非负整数的类型来衡量。但是为了维持队伍的平衡，现在他想要让 $x$ 和 $z$，$y$ 和 $w$ 保持一个相对稳定的比例。

现在他想要知道，是否可以找到一个正整数 $c$，使得 $x = z \times c$ 且 $y = w \times c$。

最近他忙着给TYoier准备别出心裁的礼物，找到这个正整数的任务就交给你吧！

## 输入格式

一行四个整数 $x, y, z, w$。

## 输出格式

一行一个整数。如果可以找到一个正整数 $c$ 满足题目的条件，则输出这个 $c$。否则输出 $-1$。

```input1
9 6 3 2
```

```output1
3
```

## 样例1解释

可以发现 $9 = 3 \times 3, 6 = 3 \times 2$，$3$ 可以成为样例 1 的 $c$。

```input2
7 2 3 5
```

```output2
-1
```

## 样例2解释

不能找到任何一个正整数 $c$ 满足题目条件。

```input3
0 6 0 2
```

```output3
3
```

## 样例3解释

需要注意 $x, y, z, w$ 有可能是 $0$。

## 数据规模与约定

对 $100\%$ 的数据，保证 $0 \leq x, y, z, w \leq 10^3$，且 $x, y, z, w$ 不同时为 $0$。
