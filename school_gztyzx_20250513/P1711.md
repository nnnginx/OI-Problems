## 问题描述

XXXX年突然有外星人造访，但大家语言不通，不过科学家们经过研究发现外星人用 $26$ 个英文字母组成的单词中最长不降子序列的长度来表述数字，且英文字母的排列顺序不同，现给出其排列顺序，再给出外星人说的每个数字（其实是每个英文单词，用空格隔开），翻译出外星人所说的数字（连续输出，最后加回车）。

(因为是最长不降子序列，所以数字中没有 $0$，也就是说外星人的数字是 $\geq$ 1的数字)。

例如：

我们正常的字母排列顺序是 `abcdefg...xyz`，代表 $a<b<c<…..<x<y<z$。

`abcd`，`efg`，`hhh`，`ihg` 四个字符串的最长不降子序列的长度分别为 $4,3,3,1$。

## 输入格式

第1行为 $26$ 个字母，表示外星人的字母顺序

第2行为若干个字符串，每个字符串之间用空格隔开



## 输出格式

输出答案

含义如题描述

```input1
abcdefghijklmnopqrstuvwxyz
abcd efg hhh ihg
```

```output1
4331
```

## 数据范围

$1\leq 第二行长度\leq 255$
