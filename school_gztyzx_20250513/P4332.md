## 题目描述

小林在$TY$机房的黑板上看到了一串长长的字符串$s$，字符串的长度为 $n$ ，小林想知道 $S$ 中包含多少个连续的 $ABC$。

## 输入格式

第一个行输入一个整数$n$，表示字符串的长度。

## 输出格式

输出字符串中包含的子串$abc$的个数。

```input1
10
ZABCDBABCQ
```

```output1
2
```

```input2
19
THREEONEFOURONEFIVE
```

```output2
0
```

```input3
33
ABCCABCBABCCABACBCBBABCBCBCBCABCB
```

```output3
5
```

## 数据范围

$ 3\ \leq\ n\ \leq\ 50 $；

$ S $ 中的字母都是大写字母。
