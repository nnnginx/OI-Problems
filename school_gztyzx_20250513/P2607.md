## 问题描述

最近小G新开了一家蛋糕店。开业第一天，一共来了 $n$ 位顾客。由于小G非常懒，他每次只会接待一位顾客。

每个顾客都想尽快的买到蛋糕，所以没有第一个买到蛋糕的顾客都会有一个愤怒值。最终排在第 $i$ 个位置的顾客x的愤怒值为 $i*a[x]$。小G想要排列队伍让所有顾客的愤怒值之和最小。求最小的愤怒值之和。

## 输入格式

第一行为一个整数 $n$，表示顾客数。

第二行输入 $n$ 个整数 $a[1]..a[n] $，含义见题面

## 输出格式

一行一个整数 $ans$，表示最小的愤怒值之和。

```input1
5
8 5 8 4 6
```

```output1
51
```

## 样例解释

$Ans=8*1+6*2+5*3+4*4=51$

## 数据范围

对于 $30\%$ 的数据，$1 ≤ n ≤ 10$。

对于 $60\%$ 的数据，$1 ≤ n ≤ 1000$。

对于 $100\%$ 的数据，$1 ≤ n ≤ 1000000$。
