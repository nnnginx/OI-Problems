<h2>Description</h2>
阿尔萨斯去到了北方大地，拿到了霜之哀伤，利用它强大的力量杀死了恐惧魔王，梅尔甘尼斯。同时，霜之哀伤也吞噬了阿尔萨斯的灵魂。然后他返回了洛丹伦，杀死了他的父皇。此时，他已听命于背后的始作俑者巫妖王了。<br />
<br />
在经历了屠城过后的阿尔萨斯王子现在内心充满了迷茫、困惑、无助以及软弱。现在的他已经拿起了拥有强大邪恶力量的利剑——霜之哀伤，放弃了代表圣光的光明使者之锤。他从一位拥抱圣光的圣骑士兼洛丹伦的王子变成了一个听命于巫妖王的堕落的死亡骑士。为了复活强大邪恶的人类法师克尔苏加德成为巫妖，他要拿到储存着他父皇灵魂的骨灰盒。现在，阻挡在他前面的，只剩下他的圣光导师——乌瑟尔•光明使者。为了获得骨灰盒，阿尔萨斯必须打败他。<br />
<br />
“For the Lich King!”阿尔萨斯毫不犹豫的冲了上去。<br />

<h2>Input Format</h2>
输入文件名为arthas.in。<br />
输入共包含两行。第一行包含两个正整数a1和b1，表示阿尔萨斯的血量和攻击伤害。<br />
第二行包含两个正整数a2和b2,表示乌瑟尔的血量和攻击伤害。<br />
<div>
	<br />
</div>
<h2>Output Format</h2>
输出文件名为arthas.out。<br />
输出只包含一行。如果阿尔萨斯能活着杀死乌瑟尔。就输出“yes”和杀死乌瑟尔需要的时间（每秒阿尔萨斯和乌瑟尔会互砍一刀），中间用空格隔开。反之，输出“gg”和阿尔萨斯死亡需要的时间。<br />
<div>
	<br />
</div>
<pre><code class="language-input1">
arthas1.in
3 3
3 1
arthas2.in
3 3
3 3
</code></pre><pre><code class="language-output1">
arthas1.out
yes 1
arthas2.out
gg 1
</code></pre>
<h2>Hint</h2>
【数据说明】<br />
对于70%的数据，1&lt;=a1,a2,b1,b2&lt;=10^10；<br />
对于100%的数据，1&lt;=a1,a2,b1,b2&lt;=10^99，1&lt;=砍的刀数&lt;=100,000。<br />
保证a1&gt;=b2，a2&gt;=b1。<br />
