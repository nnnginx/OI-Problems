## 题目描述

小C喜欢的游戏出了一个输出手，每次可以自由选择一个目标进行攻击，减少该目标体力A，同时可以减少其他敌人的体力B，当敌人体力小于等于$0$则会被击败。刚好有一个数量为$N$的敌人军团出现了，第i个敌人的体力为$h_i$，由于小$C$想要尽快击败敌人，所以他想知道在每次攻击都自由选择目标的情况下，最少需要多少次攻击即可击败敌人。由于小$C$想要打游戏，所以他拜托你帮他解决这个问题。

## 输入格式

输入第一行包含三个整数$N, A, B$。
接下来$N$行，第$i$行一个整数表示第$i$个敌人的血量。

## 输出格式

输出1个整数表示击败敌人需要的最少次数。

```input1
4 5 3
8
7
4
2
```

```output1
2
```

```input2
2 10 4
20
20
```

```output2
4
```

```input3
5 2 1
900000000
900000000
1000000000
1000000000
1000000000
```

```output3
800000000
```

## 样例解释

样例1：分别选择第1、2个敌人各攻击1次。
样例2：分别选择第1、2个敌人各攻击2次。

## 数据规模与约定

* $1\le N \le 10^5$
* $1\le B < A \le 10^9$
* $1\le h_i\le10^9$
