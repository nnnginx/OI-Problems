## 问题描述

本题中，我们把“序列$A$ 的第 $k$ 个数”记作 $A_k$。

给定一个长度为 $n$ 的整数序列 $a$。从中取出一些数，组成一个长度为 $m$ 的序列 $b$，其中 $m$ 是未给定的，即 $m$ 可以随意取值。序列 $b$ 满足：

- 存在一个长度为 $m$ 的整数序列 $c$，满足 $1≤c_1<c_2<c_3<⋯<c_m≤n$，且对于任意的整数 $i (1≤i≤m)，b_i=a_(c_i )$。

对于任意的整数$i (1≤i≤m)$，下面的两个条件中至少有一个成立：

- 对于任意的整数 $j (1≤j≤m)，b_i≥b_j$；
- 存在至少一个整数 $j (1≤j≤m)$，满足 $b_i<b_j$，且 $b_j-b_i+gcd (b_i,b_j )=lcm (b_i,b_j )$。

记满足以上条件的一个序列 $b$ 中的所有数的和为 $s$，即 $s=\sum_{i=1}^m b_i$。 你需要求出 $s$ 的最大值；如果它的最大值不存在，输出 $0$。

如果你不知道题目中一些符号的含义，你可以参考下面的定义。

- 对于整数$x$和$y$，$\gcd(x,y)$ 指的是 $x$ 和 $y$ 的最大公因数。当整数 $z$ 同时是 $x$ 和 $y$ 的因数时， $z$ 是 $x$ 和 $y$ 的公因数。$x$ 和 $y$ 的最大公因数就是 $x$ 和 $y$ 的最大的公因数。
- 对于整数 $x$ 和 $y$，$lcm(x,y)$ 指的是 $x$ 和 $y$ 的最小公倍数。当整数 $z$ 同时是 $x$ 和 $y$ 的倍数时，$z$ 是 $x$ 和 $y$ 的公倍数。$x$ 和 $y$ 的最小公倍数就是 $x$ 和 $y$ 的最小的正的公倍数。
- “$\sum$”是求和符号。对于一个序列 $A$， $\sum_{i=1}^n A_i =A_1+A_2+A_3+⋯+A_n$。

下面的信息可能对你解题有帮助。

- 一元二次方程是形如 $ax^2+bx+c=0$ 的方程，其中 $x$ 是未知量，$a≠0$。
- 当一个关于 $x$ 的一元二次方程写成 $(x-p)(x-q)=0$ 的形式时，式中的 $p、q$ 就是该方程的两个解。

## 输入格式

输入总共有两行。

输入的第一行是一个整数，$n$，即序列 $a$ 的长度。

输入的第二行包含 $n$ 个整数，分别为 $a_1,a_2,a_3,⋯,a_n$，表示序列 $a$。

## 输出格式

有且仅有一行，为一个整数，如果 $s$ 的最大值存在，则表示其最大值；如果它的最大值不存在，则为0。

```input1
5
1 4 3 2 5
```

```output1
7
```

## 样例1解释

序列 $b$ 可以为 $\{1,4,2\}$。 经过检验，该序列满足题目所述的条件，其中所有数的和为 $7$。可以证明，不存在另外一个满足题目条件的序列 $b$，使其所有数的和大于 $7$。

```input2
4
27 9 3 1
```

```output2
40
```

## 数据范围及约定

下表列出了 $n$ 在各个测试点中的取值范围以及一些特殊性质．

| 测试点编号 | $n≤$    | 特殊性质 |
| ------------ | -------- | ---------- |
| 1,2        |     $10$ | 无       |
| 3,4        |     $16$ | 无       |
| 5,6        |     $19$ | 无       |
| 7,8        |     $21$ | 无       |
| 9,10       |    $100$ | 性质B    |
| 11-13    |    $10^3$ | 性质B    |
| 14,15      | $4×10^4$ | 性质A    |
| 16,17      | $4×10^4$ | 性质B    |
| 18-20      | $4×10^4$ | 无       |
| 21-25      | $2×10^5$ | 无       |

表中的特殊性质：

- 性质A：存在一个整数 $x$，使得对于所有的整数 $i (1≤i≤n)$，$a_i=x$ 或 $a_i=2x$ 成立。
- 性质B：对于所有的整数$i (1≤i≤n)，a_i≤10^3$。

所有的测试数据满足：

- $1≤n≤2×10^5$。
- 对于所有的整数 $i (1≤i≤n)，1≤a_i≤10^7$。
- 序列a中的数对任意一个大于10^5的质数取模的结果随机。
- 题中出现的数字都是整数。
