## 问题描述

程序龙想到了一个新游戏：

他有 $n$ 堆石子，每堆石子有 $a_i$ 个石子，对于每一次操作，他可以将某堆的一个石子移动到另外一堆。游戏终止的条件是：存在一个 $x(x\gt 1)$，使得任意一堆石子满足：$a_i\%x==0（1\leq i\leq n）$，即 $a_i$ 能被 $x$ 整除。

程序龙认为这个游戏挺无聊的，所以他现在想早点结束这个游戏，请你告诉他游戏终止所需要的最少操作次数。

## 输入格式

第一行一个数，表示 $n(n\gt 0)$

第二行有 $n$ 个数，表示每一堆石子的个数 $(0\leq a_i\leq 100000)$

## 输出格式

一个数，表示最少的操作次数

```input1
5
1 2 3 4 5
```

```output1
2
```

## 样例解释

第一步，程序龙把第一堆的一个石子移到第二堆，此时为 `0 3 3 4 5`。

第二步，程序龙把第四堆的一个石子移到第五堆，此时为 `0 3 3 3 6`，此时如果 $x=3$ 时，满足游戏终止条件，故步数为 $2$。

## 数据范围

$30\%$ 的数据，$n\leq 20$

$100\%$ 的数据，$n\leq 100000$
