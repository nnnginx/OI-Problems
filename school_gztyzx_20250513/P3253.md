## 问题描述

C老师发明了JX编程语言，在这种编程语言里，只有 `+=` 这个运算符，可以将某个变量的值增加，现在输入 $a，b，n$，请问仅使用 $a，b$ 两个变量来进行 `+=` 运算，最少多少次能够严格超过 $n$ 的值？只能使用 $a+=b$，或者 $b+=a$，不能使用 $a+=a$ 或 $b+=b$。

## 输入格式

三个整数，$a，b，n$（三个数都不超过 $1e9$）

## 输出格式

一个整数，代表最少步数

```input1
1  2  3
```

```output1
2
```

```input2
5  2  14
```

```output2
3
```
