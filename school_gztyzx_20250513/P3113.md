## 问题描述

看着电脑上一道道难懂的题目，你趴在桌子上，想着既然这么无聊，还是睡一觉吧，于是渐渐进入了梦乡…

再抬头时，竟然回到了化学课堂上，虽然惊讶万分，可是先听听看吧。$∞$ 仍然正在白板上写着题目，定睛一看，居然是初中的离子反应。 惊讶于重返过去的神奇，回想着这一年发生的事情，你不禁感慨万分：“这一年，就这样过去了啊。”回味着昨晚熬夜看的电影的台词

“Time is infinity , while youth not.”

你还没缓过神来，便被 $∞$ 点名提问了：“现在有 $m$ 杯溶液，告知每杯中的所有 $n$ 个离子（用数字表示），请问各组离子分别是否满足离子共存？”因为你的化学太好了，所以开始时会给定你所有 $p$ 个不能共存的离子组($u,v$)且不重复，即对于每一个离子和它不共存的至多一个。

离子用一个整数 $r$ 表示，满足 $1\leq r\leq 30000$。

## 输入格式

第一行给出不共存离子组数 $p$，接下来 $p$ 行是一组离子（$u,v$）。

接下来 $1$ 行给出杯数 $m$，接下来 $m$ 行，每行第一个整数表示离子数 $n$，后面是 $n$ 个离子。

## 输出格式

回答共 $n$ 行，若能共存输出 $1$，不能输出 $0$。换行分隔每个答案。

```input1
2
1 2
3 4
3
3 1 5 6
3 1 2 3
3 3 999 4
```

```output1
1
0
0
```


## 样例解释

第一组离子有 $(1，2)$ 

第三组有 $(3，4)$

## 数据范围

对于前 $80\%$ 数据，$n\leq 1000$

对于前 $100\%$ 数据，$p\leq 200,m\leq 50,n\leq 30000$
