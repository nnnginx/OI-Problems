## 问题描述

众所周知，一有时间，爱新觉罗就会与周老师聚会

又到了新学期，已知这个学期有 $n$ 天，并且知道每天爱新觉罗和周老师的要上几节课

若他们当天上的课均小于 $k$ ，则他们便会出去聚会

“我昨晚失眠了，脑子有点缺氧，算不出来”，请你帮爱新觉罗算一算他们能聚会几天。

## 输入格式

第一行为两个整数 $n, k$

接下来 $n$ 行，每行两个数 $a_i$ ， $b_i$ ，分别表示第 $i$ 天爱新觉罗和周老师要上几节课

## 输出格式

一个整数 $x$ ，表示他们能聚会几天

```input1
5 4
1 2
9 0
5 7
3 5
3 3
```

```output1
2
```

## 数据范围

对于 $10\%$ 的数据 $n=1$

另有 $10\%$ 的数据 $k=0$

另有 $10\%$ 的数据 $a_i=0,b_i=0,1\leq k \leq 10^5$

对于 $50\%$ 的数据 $1\leq n \leq 100$ 且 $0\leq a_i,b_i \leq 10^5$

另有 $20\%$ 的数据 $a_i=0，10^8\leq b_i \leq 10^9$ (周老师哭了)

对于 $100\%$ 的数据 $1\leq n \leq 10^4$ 且 $0 \leq a_i,b_i,k \leq 10^{18}$
