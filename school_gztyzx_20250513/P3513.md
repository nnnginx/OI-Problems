## 问题描述

梯形的面积公式大家都知道吧！如下图：

![](/upload/image/20210714/20210714090737_90237.png)

那么面积：$s=\frac{(a+b)\times h}{2}$。

现已知梯形的上底 $a$，下底 $b$ 和高 $h$，求梯形的面积 $s$。

## 输入格式

输入三个正整数：$a、b、h$，之间用一个空格隔开。

## 输出格式

输出一个整数，梯形的面积$s$。

```input1
2 3 2
```

```output1
5
```

```input2
4 6 3
```

```output2
15
```

## 数据规范

$50\%$的数据，$a、b、h$均小于$100$；

$100\%$的数据，$a、b、h$均小于$1000$；

数据保证能够整除。
