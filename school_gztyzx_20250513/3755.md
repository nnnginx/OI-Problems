## 【问题描述】

小C在公园散步的时候，发现公园里有一条有$n$个花盆构成的花丛列，第$i$个花盆的花是$c_i$颜色的。小C想知道，按照花盆的顺序选择$4$个花盆，使得第$1$个第$3$个花盆颜色相同且第$2$个第$4$个颜色也相同的选法有多少种。

形式化地，给出 $n$ 个数 $c_1，c_2，\cdots，c_n$。求问有多少个四元组 $(i，j，k，l)$，使得这个四元组满足下列条件：

- $1 \leq i < j<k<l \leq n$；
- $c_i=c_k$ 并且 $c_j=c_l$。

## 【输入格式】

第一行输入一个数 $n$，表示有多少个花盆。
对于每组数据，第二行输入 $n$ 个数，表示 $c_1，c_2，\cdots，c_n$。

## 【输出格式】

一个整数，代表合法四元组的个数。

```input1
6
1 3 3 1 2 3
```

```input2
5
2 2 2 2 2
```

```output1
2
```

```output2
5
```

## 【数据范围】

对于 $50\%$ 的数据，有 $1\le n \le 10^2$。<br />
对于 $100\%$ 的数据，有 $1\le n \le 5000 ;1 \le a_i \le n$。<br />

## 样例解释

样例1：(1, 2, 4, 6)和(1, 3, 4, 6)
