## 问题描述

huhe老师在上课的时候有个习惯，它会让同学们坐成一排给学生拍照。然后同学们按照老师的坐成一排。

教室里总共有 $n$ 个同学。（$1\leq n\leq 100$） 每一个同学都有一个属于自己的号码牌。（$1\leq  p_i\leq  1000$）。

huhe老师每次上课时候都会给同学们拍照。但是对于每一组满足 $1\leq  i \leq  j \leq  n$ 的同学，huhe会从 $i$ 到 $j$ 之间的所有同学（包含 $i$ 和 $j$）拍一张照片。

后来huhe查看这些照片时候注意到，有些照片存在（平均号码牌）的同学。一位同学的号码牌，恰好是照片当中的同学们的号码的平均值。如果有这样的照片老师就会存储起来。

huhe的照片当中有几张存在平均的号码？

## 输入格式

输入的第一行包含 $N$。第二行包含 $N$ 个空格分隔的整数 $p_1…p_N$。

## 输出格式

输出存在平均的号码牌的照片数量。

```input1
4
1 1 2 3
```

```output1
6
```

## 样例解释

每张仅包含一位同学的的照片均会被计入答案（在这个样例中有 $4$ 张）。另外，在这个样例中 ($i,j$) 为 $(1,2)$ 和 $(2,4)$ 所对应的照片也存在平均的号码。

$1、1、2、3、11(平均值为1)、123（平均值为2）$
