## 问题描述

有 $n$ 条编号为 $1\cdots n$ 的奶牛天天参加茶话会，其中有 $m(1\leq m\leq 2000)$ 对奶牛已经在第一次茶话会以前见过认识。

在茶话会中，假如奶牛 $i$ 和奶牛 $j$ 都有共同认识的奶牛 $k$，那么经过奶牛 $k$ 介绍那么奶牛 $i$ 跟奶牛 $j$ 也就认识了。

在举办了足够多次茶话会直到全部奶牛都没有可能再认识新奶牛以后，现在有Q对询问，每一次询问包括两个整数 $X_j$ 和 $Y_j(1\leq  X_j\leq  N;1\leq  Y_j\leq N)$，代表奶牛 $X_j$ 和奶牛 $Y_j$ 是否认识。

例如：有 $5$ 条奶牛编号为 $1\cdots 5$，刚开始奶牛 $2$ 认识奶牛 $5$，奶牛 $2$ 认识奶牛 $3$，和奶牛 $4$ 认识奶牛 $5$，关系看下图（a）。

## 输入格式

第一行：三个空格分开的整数：$N,M,Q$

第二..$M+1$ 行：每一行两个整数代表本来认识的奶牛

第 $M+2..M+Q+1$ 行：每一行两个整数 $X_i$ 和 $Y_j$ （代表询问的奶牛 $X_i$ 和奶牛 $Y_i$ 是否认识）

## 输出格式

第 $1..Q$ 行：假如第 $i$ 次询问是认识的就输出 `Y` 否则输出 `N`

```input1
5 3 3
2 5
2 3
4 5
2 3
3 5
1 5
```

```output1
Y
Y
N
```
