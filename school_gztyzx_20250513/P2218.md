## 问题描述

在过去的8102年里，Chen.Jr去了外国的好多的国家，其中，令Chen.Jr最印象深刻的是埃及的金字塔。

金字塔是一个非常漂亮的锥形，而我们从正面看金字塔时，我们则可以看到一个三角形。

而说起三角形，Chen.Jr依稀记得，曾经在课上详细分析过如何输出三角形。

于是Chen.Jr想要检验一下大家的做题情况。

Chen.Jr想让你输入一个整数$n$，让你打出一个如下的三角形(下图为 $n=9$ 的情况)（$n\leq 1000$）：

```
        1
       121
      12321
     1234321
    123454321
   12345654321
  1234567654321
 123456787654321
12345678987654321
```




## 输入格式

一个整数 $n(n\leq 1000)$，表示金字塔的高度

## 输出格式

输出一个金字塔，格式见输出样例

```input1
5
```

```output1
    1
   121
  12321
 1234321
123454321
```

```input2
12
```

```output2
           1
          121
         12321
        1234321
       123454321
      12345654321
     1234567654321
    123456787654321
   12345678987654321
  1234567890987654321
 123456789010987654321
12345678901210987654321
```
