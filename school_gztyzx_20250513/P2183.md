## 问题描述

程序龙学习了余数 $r$ 的定义是：$a=k*b+r$，其中 $k$ 为整数，$0\leq r<b$。

在$c++$中，对于两个整数求余数，我们可以用 $\%$ 来运算，比如 $23\%12=11$。

但是，对于两个实数 $a$ 和 $b$ 求余数，就不能用 $\%$ 运算符了。

那应该怎样求实数 $a$ 除实数 $b$ 的余数呢？

程序龙只能请求你帮助了。

## 输入格式

输入一行，包括两个实数 $a$ 和 $b$。

## 输出格式

输出余数 $r$，保留两位小数。

```input1
12.56 12.06
```

```output1
0.50
```

## 数据范围

$0\leq a,b\leq 100$
