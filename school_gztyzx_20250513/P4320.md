## 题目描述

高桥君要将蓝球和红球按一定次序排成一行，现在此队列中没有球。

每次他将在队列末尾放上 $A$ 个蓝球，再放上 $B$ 个红球。此操作将循环 $10^{100}$ 次。

输入整数 $N,A,B$ ，输出前 $N$ 个球中的蓝球数目。

## 输入格式

输入三个整数 $ N $、 $ A $、 $ B $。

## 输出格式

输出前 $N$ 个球中的蓝球数目。

```input1
8 3 4
```

```output1
4
```

```input2
8 0 4
```

```output2
0
```

```input3
6 2 4
```

```output3
2
```

## 数据范围

- $ 1\ \leq\ N\ \leq\ 10^{18} $
- $ A,\ B\ \geq\ 0 $
- $ 0\ <\ A\ +\ B\ \leq\ 10^{18} $
