## 问题描述

牛棚面前有一大块草坪。

贝茜住的牛棚前有一大块草坪，这个草坪的中心是一个喷水器。

这个喷水器会一直喷水，且喷水的面积呈一个圆形。贝茜不想被水淋湿，但是越靠近喷水器的草越是鲜嫩。于是贝茜就想找到一个她和喷水器之间的距离，能让她既不被水淋到，又可以吃到尽可能鲜嫩的草。（π取3.14）

作为最聪明的奶牛，贝茜没一会就求出来了这个距离。你知道这个距离是多少吗？

## 输入格式

输入一个两位小数，表示被水淋到的面积。

## 输出格式

输出贝茜离喷水器的最短距离。(保留两位小数）

```input1
28.26
```

```output1
3.00
```

## 数据范围

$100\%$ 的数据，圆面积$≤1570$

## 提示

本题其实就是求圆的半径。

圆的面积公式：$面积=\pi *半径*半径$

科普：`sqrt(n)`，求的根号是 $n$。

需用头文件 `#include<cmath>`
