## 问题描述

兰姐姐是来自火星的女王。

由于兰姐姐实在太漂亮了，所以有许多仰慕她的人前来追求她。兰姐姐会给这个追求者一个考验：兰姐姐会打追求者许多巴掌，追求者有一个防御值 $x$，当兰姐姐的攻击力 $a$ 大于 $x$ 时他就会被打到火星去，一开始兰姐姐的攻击力为 $1$，每次追求者挡下攻击后，兰姐姐的攻击力就会翻倍，现在问：如果知道了追求者的防御值 $x$，求兰姐姐把他打到火星去时，兰姐姐的攻击力为几？

## 输入格式

只有一行，一个非负整数 $x$，追求者的防御值

## 输出格式

只有一行，兰姐姐把他打到火星去时，兰姐姐的攻击力

```input1
5
```

```output1
8
```

```input2
2
```

```output2
4
```

## 数据范围

数据保证 $1≤x≤2^{31}$
