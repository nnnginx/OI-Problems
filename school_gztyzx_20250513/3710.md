## 题目描述

小$C$又迷上了序列，现在他有一个长度为$n$的序列，他想知道满足$a_l​+\dots +a_r​=a_l\oplus \dots \oplus a_r​,l\le r$的区间$[l,r]$的数量。
注：$\oplus$表示异或

## 输入格式

输入第一行包含1个整数$n$。
第二行包含$n$个整数$a_i$。

## 输出格式

输出1个整数表示满足条件的区间数。

```input1
4
2 5 4 6
```

```output1
5
```

```input2
9
0 0 0 0 0 0 0 0 0
```

```output2
45
```

```input3
19
885 8 1 128 83 32 256 206 639 16 4 128 689 32 8 64 885 969 1
```

```output3
37
```

## 数据规模与约定

* $1\le n\le 200000$
* $0\le a_i​<2^{20}(即为1048576)$
