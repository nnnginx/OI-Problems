## 问题描述

给出一个 $N×N$ 的方格，每个方格中有一个非负整数。游戏的目标是从方格的左上角沿任何合法路径移动到右下角。任何一个方格内的数字规定了离开本方格沿着某个特定方向所必须移动的方格数。如果沿着某个方向移动越出了方格的边界，则这个方向的移动是禁止的。每一次移动只能是向下或向右。考虑如下图所示的 $4×4$ 方格，后面显示了从出发位置到目的位置的三条路径，其中隐去了与每条路径无关的数字。

![](/upload/image/20181129/20181129112933_20101.png)

## 输入格式

输入共有 $N+1$ 行；

第 $1$ 行是方格的行数 $N$，接下来是 $N$ 行，每行含有 $N$ 个 $0\sim 9$ 的数字，中间没有空格。

## 输出格式

输出共一行一个整数，输出从左上角到右下角的路径数。

```input1
4
2331
1213
1231
3110
```

```output1
3
```

```input2
4
3332
1213
1232
2120
```

```output2
0
```

```input3
5
11111
11111
11111
11111
11111
```

```output3
70
```

## 数据规模

对于 $100\%$ 的数据： $4≤N≤50；$
