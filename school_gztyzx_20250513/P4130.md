## 题目描述

在$TY$校园里，有一片绿茵茵的草地，是一个正方形的二维方阵，草地上每个格子都是空的。

$JY$老师准备组织一场特别的足球运动，在运动前$JY$将会逐一地将 $n（1≤n≤10^5）$位同学安排到草地的格子上站岗，他们的编号是$1...n$。

第$i$位同学将会被安排在格子$(x_i,y_i)$$（0≤x_i,y_i≤1000）$站着，保证每个同学的格子都是独一无二的，不会安排任意两个同学站在同一个格子上。

我们定义一位同学的感受是【不开心的】，当且仅当这个同学上下左右四个相邻的格子上恰好站着三个同学。比如一个同学是在格子$(2,3)$，那么如果他上下左三个格子$(1,3)、(3,3)、(2,2)$都有一个同学站在那里，而右边的格子$(2,4)$是空的，那么此时同学$(2,3)$便是【不开心的】。

为了让同学们都能开心地踢球，$JY$老师想要额外添加一些同学直到没有同学是【不开心的】（包括新添加的同学）。注意：额外添加的同学的 $x$ 和 $y$ 坐标取值范围是 $[-1000,1000]$。

现在，对于 $1…n$ 中的每个 $i$，$JY$老师想知道，在初始时已经安排了 $1…i$的同学站在草地对应的格子上时，为了使得没有同学是【不开心的】，需要额外加入的同学的最小数量。

## 输入格式

输入的第一行包含一个整数 $n$。

接下来输入 $n$ 行，每行包含两个空格分隔的整数，第$i$行表示安排的同学$i$所在的方格坐标 $(x，y)$。

## 输出格式

输出 $n$ 行，对于 $1…n$ 中的每个 $i$，输出一行，为$JY$需要额外加入的学生数量。

```input1
9
0 1
1 0
1 1
1 2
2 1
2 2
3 1
3 2
4 1
```

```output1
0
0
0
1
0
0
1
2
4
```

## 样例解释

当 $i=4$时，目前草地上已经安排了$4$位同学，其坐标分别是：$(0, 1),(1, 0),(1, 1),(1, 2)$，此时$(1,1)$是不开心的，因此$JY$ 需要在 $(2,1)$ 额外加入一位同学使得位于 $(1,1)$ 的同学不再【不开心的】，因此输出$1$。

当 $i=9$时，目前草地已经安排了$9$位同学，$JY$ 的最优方案是在 $(2，0)、(3，0)、(2,−1)$ 和 $(2,3)$ 额外加入总共$4$位同学。

## 数据范围

对于$100\%$的数据，$1≤n≤10^5$。
