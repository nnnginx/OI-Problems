## 【问题描述】

CWOI在第inf届NOI取得了前所未有的好成绩，高兴的X老师决定带大家出去吃饭。饭桌上，大家非常高兴。现在，到了要结账的时候，为表现自己的慷慨，大家都争相去付账单。收银台上有一个正方形的付款二维码，要想付款，必须要数出二维码上的黑色方块共构成几个联通块。由于巨佬ZJK AK了这次NOI，所以他想要买单。但他还要去AK IOI，没时间计算，于是他想让你帮他解决这个问题。<br />
现给出正方形二维码的边长$n$，黑色方块个数$m$，每个方块的左上角坐标$(x_1，y_1)$和右下角坐标$(x_2，y_2)$。请你求出这些黑色方块共构成几个联通块。<br />

## 【输入格式】

第一行两个整数$n,m$。<br />
接下来m行，每行两个坐标$(x_1 , y_1)$，$(x_2 , y_2)$，代表一个方块，左上角是$(x_1 , y_1)$，右下角是$(x_2,y_2)$。<br />

## 【输出格式】

输出一个整数$cnt$，代表连通块个数。

```input1
5 3
2 2 3 3
3 3 5 5
1 2 1 4
```

```input2
4 7
2 2 2 2
1 1 1 1
2 2 2 2
3 1 3 1
4 4 4 4
1 1 1 1
2 4 2 4
```

```output1
1
```

```output2
5
```

## 【样例解释】<br />

第一组样例，二维码用01矩阵表示为：<br />
0 1 1 1 0<br />
0 1 1 0 0<br />
0 1 1 1 1<br />
0 0 1 1 1<br />
0 0 1 1 1<br />
所以只有一个连通块。（当2个格子上下或者左右相邻，表示相联）<br />
第二组样例，二维码用01矩阵表示为：<br />
1 0 0 0<br />
0 1 0 1<br />
1 0 0 0<br />
0 0 0 1<br />
所以有五个连通块。（斜对角线，不算联通）<br />

## 【数据范围】<br />

不保证方块面积为0。保证$x1≤x2，y1≤y2$。<br />
共10个测试点。<br />
对于前4个测试点，保证$0≤n，m≤100$。<br />
对于前7个测试点，保证$0≤n，m≤800$。<br />
对于所有测试点，保证$0≤n，m≤2000$。<br />
