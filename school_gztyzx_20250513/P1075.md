## 问题描述

你玩过扫雷吗？这个游戏在某个被淘汰的操作系统里有。游戏目标是找出 $n*m$ 矩阵内的所有地雷。在本题中，你需要为每个单元格统计出它周围的地雷数。每个单元格最多和 $8$ 个单元格相邻。左图矩阵地雷用“`*`”表示，计算结果如右图。

```
..*...****          24*202****

***...****          ***313****

.**..*.*..          4**32*6*63

*.*..*.**.          *6*44*6**2

.***.*.*.*          4***4*4*5*

***.*...*.          ***6*543*2

.*.****...          3*6****543

..**..****          23**55****

.*...*...*          3*545*556*

**.***.**.          **3***3**2
```

## 输入格式

输入第一行有 $n，m(0\leq n，m\leq 100)$ ，分别表示这个矩阵的行数与列数。接下来的 $n$ 行每行包含 $m$ 个字符“`.`”表示安全，“`*`”表示地雷。

## 输出格式

输出包含一个 $n*m$ 矩阵。用数字表示该位置字符“`.`”周边的地雷数量，“`*`”还是表示地雷。

```input1
4 4 
*... 
.... 
.*.. 
....
```

```output1
*100
2210
1*10
1110
```
