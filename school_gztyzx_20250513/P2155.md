## 问题描述

国家统计局每年都对各城市的人的收入进行调查统计。针对不同人群之间收入差异度进行的统计。一般统计方法是：对不同人群收入进行求差，符合规定差值的数目统计出来。

## 输入格式

第一行包括 $2$ 个非负整数 $N$ (不同的人群数)和 $C$ (差值)，中间用空格隔开。

第二行有 $N$ 个整数(每个人群的收入)，中间用空格隔开。

## 输出格式

输出一行，表示人群差值为 $C$ 的数对的个数。

```input1
4 1
1 1 2 3
```

```output1
3
```

## 数据规模

对于 $90\%$ 的数据，$N\leq 2000$；

对于 $100\%$ 的数据，$N\leq  200000$。

所有输入数据都在 $10^{9}$ 范围内。
