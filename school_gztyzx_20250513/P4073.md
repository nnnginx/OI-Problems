## 问题描述

在网络各个领域中，有一个技术很重要：信息加密。

这是一种简单的加密，作用是将一个小写字符串 $c$ 加密。现在给你加密规则以及一个加密后的字符串，请你解密它并输出。

首先众所周知，一个字符 $x$ 可以转换为 `int` 类型，如`a`的对应值是 $97$。

这是加密规则：

- 先将字符串反转，如 `abc` 变为 `cba`
- 将字符串每个字符 $c_i$ 改为 $26-(c_i-96)+97$ 。 打个比方，`abc` 会变成 `zyx`，`d` 变为 `w` 。
- 在字符串后面加上一个任意小写字母 $ch$ 。
- 将此时字符串除了最后一个字符以外的所有字符 $c_i$ 改为 $(c_i+2^{ch})mod\;26+97$ 。在这里，字符将改为数字进行运算，最后再改为 `char` 类型。而 $2^{ch}$ 则表示将 $ch$ 个2相乘得到的结果。

## 输入格式

一个字符串

## 输出格式

一个字符串

```input1
xdrlsgisuhwyu
```

```output1
modqsceszthn
```

## 数据范围

$|S|$ 表示字符串$S$的长度。

对于 $20\%$ 的数据，$|S| \le 10$

对于 $40\%$ 的数据，$|S| \le 1000$

对于 $100\%$ 的数据，$|S| \le 1000000$
