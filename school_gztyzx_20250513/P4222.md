## 题目描述

慧音老师一直很担心琪露诺的数学，教了⑨年依然没有半点长进。无奈之下她只好让琪露诺加训，还花大价钱请了城管博丽灵梦全程监督。

这次加训的内容是：给定一个长为 $N$ 的整数序列$A_0,A_1⋯A_{N-1}$，初始时所有值均为 $0$。现要对该序列作 $K$ 次修改。每次修改给定两个参数 $c,r$，对于序列中每个数 $A_i$，其增量为
$max(0,r-|i-c|)$

$K$ 次修改之后，琪露诺需要统计有多少个数 $A_i$ 满足：

①$i-1\ge 0$ 且 $ i+1<N$

②$A_i>A_{i-1} $ 且 $ A_i>A_{i+1}$

之前提到琪露诺很喜欢炫富，而灵梦生性见钱眼开。所以趁慧音老师不在的时候，她拿⑨⑨⑨张百元大钞收买了灵梦，然后又一次把锅甩给了你。

## 输入数据

每个测试文件有多组数据。

第一行为一个正整数$T$，表示测试数据的组数；

之后每组测试数据，第一行是两个正整数$N,K$；

之后$K$行，每行两个非负整数$c,r$，表示该次修改的参数。

$0≤c<N,0<r≤N$

## 输出数据

每个测试数据输出一行，满足条件的数有多少个。

```input1
1
10 3
3 2
7 4
5 1
```

```output1
1
```

## 样例说明

第一次修改后的序列为：$0, 0, 1, 2, 1, 0, 0, 0, 0, 0$

第二次修改后：$0, 0, 1, 2, 2, 2, 3, 4, 3, 2$

第三次修改后：$0, 0, 1, 2, 2, 3, 3, 4, 3, 2$

## 数据范围

$T≤5,N≤10^5，K≤10^5$

本题共10个测试文件，部分测试点满足如下限制：

|测试点编号|$N,K$|$r$|$0\leq c-r$ 且 $c+r\lt N$|
|---|---|---|---|
|$1,2$|$\leq 1000$|无特殊限制||
|$3$|无特殊限制|$\leq 100$|满足|
|$4,5$|无特殊限制|$\leq 100$||
