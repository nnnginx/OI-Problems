<h2>Description</h2>
对于一个长度为n的正整数序列a_1&#44;a_2&#44;......&#44;a_n ，定义它的一个连续子段为A(l&#44;r)，即由原序列中的第l个元素到第r个元素所构成的子序列；对于一个子段A(l&#44;r)，定义它的平均值为(Max(l&#44;r)-Min(l&#44;r))/(r-l+k)，其中k是一个给定的非负整数。<br />
现在，对于给定的长度为n序列a及两个正整数L&#44;R，你需要求出在所有满足L&nbsp;≤ r − l + 1 ≤ R的子段 A(l&#44;r)中，平均值最大的子段的平均值是多少。<br />
<div>
	<br />
</div>
<h2>Input Format</h2>

	输入文件包含多组测试数据，输入的第一行为一个正整数T，表示数据组数。
<br />

	对于每组测试数据，输入的第一行为四个整数n&#44;k&#44;L&#44;R，第二行为n个正整数<span>a_1&#44;a_2&#44;......&#44;a_n</span>。
<br />
<div>
	<br />
</div>
<h2>Output Format</h2>
输出一个实数，为最大的平均值保留四位小数后的结果。
<pre><code class="language-input1">
1
4 1 2 3
3 3 4 4
</code></pre><pre><code class="language-output1">
0.5000
</code></pre>
<h2>Hint</h2>

	对于20%的数据，满足n ≤ 200。
<br />

	对于50%的数据，满足n ≤ 1000。
<br />

	对于另20%的数据，满足L ≤ 20。
<br />
对于100%的数据，满足1 ≤ n ≤ 50000，0 ≤ k ≤ n，1 ≤ a_i ≤ 2×10^9，T= 10，1 ≤ L ≤ R ≤ n。<br />
