## 题目描述

小C不知道从哪里复制来了一行有n个字的文本，这些文字排成一排，从左到右依次可以编号为第$1$到第$n$个字。

现在小C的光标在第$i$个文字上，由于小C想要使用第$l$到第$r$个字来断章取义，因此他急着把剩下的文字删除。

小C有4个键可以按：左移键、右移键、左删除键和右删除键。假设现在剩下的文字为从第$a$到第$b$个字（例如，如果一开始有5个字，删除第$1、2、5$个字，那么$a=3,b=4$）。

如果小C按了一下左移键，则光标会从$i$移动到$max(i-1,a)$

如果小C按了一下右移键，则光标会从$i$移动到$min(i+1,b)$

如果小C按了一下左删除键，则光标$i$往左的文字会被删除，即$[a,i−1]$这段文字会被删除，光标位置不变。

如果小C按了一下右删除键，则光标$i$往右的文字会被删除，即$[i+1,b]$这段文字会被删除，光标位置不变。

为了只留下初始编号从$l$到$r$的文字，小C至少需要按多少次按键？

## 输入格式

输入只有一行，包括四个整数$n,i,l,r$。

## 输出格式

一个整数，表示小C至少需要按多少次按键.

```input1
6 3 2 4
```

```output1
5
```

```input2
6 3 1 3
```

```output2
1
```

```input3
5 2 1 5
```

```output3
0
```

## 样例解释

样例1：小C按键为：左移键、左删除键、右移键、右移键、右删除键

样例2：右删除键

样例3：无需按键

## 数据范围

对于 $30\%$​​ 的数据，有 $1\le n\le 10$​​；
对于 $100\%$​​ 的数据，有 $1\le n\le 100$, $1\le pos\le n ，1\le l\le r\le n$
