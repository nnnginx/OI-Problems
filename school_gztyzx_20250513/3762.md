## 题目描述

周一早上需要进行升旗一起，小C的班级在操场上排成了一个队列，一共有n个人，从前往后身高为一个数字序列 $a_1,a_2,......,a_n$。

小C发现队伍没有完全按身高顺序从前往后排，但是由于升旗仪式马上就要开始了，小C只来得及从队伍中选出恰好一个人$i(1<=i<=n)$，然后把这个人$a_i$从队伍中拿出，然后再插入到队伍中任意位置。

现在小C想知道他在这样一次操作之后，能不能把队伍从前往后排成一个不降的序列。如原本队伍为$1,2,3,5,3，i = 5$，可以取出 $a_i=3$ ，然后在中间位置插入，可以变为 1,2,3,3,5 。

如果可以排好序，输出 YES，否则输出 NO。

# 输入格式

第一行 1 个整数n

第二行空格隔开的n个整数，代表队伍a序列。

# 输出格式

如果可以一次操作可以排好序，输出 YES，否则输出 NO。

```Input
5
1 2 4 5 3
```

```Output
YES
```

# 数据范围

对于30%的数据，满足$n<=1000$。

对于60%的数据，满足$n<=10^5$。

对于100%的数据，满足 $n<=10^6$,$1<=a_i<=10^6$。
