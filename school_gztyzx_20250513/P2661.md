## 问题描述

小Y在家做物理实验，他有 $n$ 台机器，每台机器有一个数值 $x_i$ ，表示这台机器可以发射编号 $0 − x_i$ 的粒子。任意两种粒子都可以互相融合产生新的粒子，A粒子和B粒子融合和，会产生编号为 $A \oplus B$ 的粒子。小Y的实验就是让这 $n$ 台机器每台发射一个粒子，然后让 $n$ 个粒子互相融合 $(x_1 \oplus x_2 \oplus x_3 \cdots )$，那么在足够多次试验后，能得到的最大编号的粒子是多少？

例如：现在有 $2$ 个整数 $x_1,x_2$ 分别是：$2，3$。可以让第一台机器发射编号 $2$ 的粒子，第二台机器发射编号 $1$ 的粒子，这样能够得到编号为 $3$ 的粒子。$3$ 也是所有试验中能够得到的最大编号的粒子。

$n≤1000,0≤x≤10^9$

## 输入格式

第一行一个正整数 $n$ 。

第二行 $n$ 个非负整数表示 $x[1...n]$ 。

## 输出格式

一行一个数表示答案。

```input1
3
2 2 2
```

```output1
3
```

```input2
5
3 7 5 2 6
```

```output2
7
```

```input3
7
2 1 4 7 8 9 1
```

```output3
15
```



## 数据范围

对于 $30\%$ 的数据，$0\leq x_i\leq 100$

对于 $50\%$ 的数据，$0\leq n\leq 100$

对于 $60\%$ 的数据，$0\leq x_i\leq 10000$

对于 $100\%$ 的数据，$0\leq n\leq 1000，0\leq x_i\leq 10^9$
