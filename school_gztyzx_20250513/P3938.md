## 问题描述

有一个 $n\times n$ 的棋盘，每个格子有一个路费，第 $i$ 行所有格子的路费为 $a_i$ 。有只蚂蚁在棋盘上旅行，从左上角到右下角，它想知道它经过的所有格子的最小路费和。

## 输入格式

输入共两行：

第一行两个整数 $n,m$ ，表示棋盘的行数和列数。

第二行 $n$ 个整数 $a_i$ ，表示第 $i$ 行的路费。

## 输出格式

输出共一行一个整数，表示最小路费和。

```input1
3 3
1 2 1
```

```output1
6
```

## 数据范围

对于 $30\%$ 的数据，$1\leq n,m\leq 2000$。

对于 $60\%$ 数据，$1\leq n\leq 10^4,1\leq m \leq 10^4$。

对于 $100\%$ 数据，$1\leq n\leq 10^6,1\leq m \leq 10^9,0\leq a_i \leq 10^9$。
