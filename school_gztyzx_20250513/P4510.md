## 题目描述

我们有一个网格，网格中有 $H$ 个水平方格行和 $W$ 个垂直方格列。对于某个格子$S[i][j]$其中$.$表示道路可以通过，$\#$表示墙壁不可以通过。

在方格 $(1, 1)$ 中有一个棋子。在一次移动中，它可以向右、向下或斜向右下方移动任意个位置到道路位置，而不会跳过墙壁位置。

棋子可以通过多少种方式从方格 $(1, 1)$ 移动到方格 $(H, W)$ ？求模数 $(10^9+7)$ 。

在这里，当且仅当存在 $i$ 使得棋子在走完 $i$ 步之后的位置在这两种走法中是不同的时候，两种走法才被认为是不同的。

## 输入格式

输入内容由标准输入法提供，格式如下：

$H$ ,$W$;

$S_{1,1}\ldots S_{1,W}$
$\vdots$
$S_{H,1}\ldots S_{H,W}$.

## 输出格式

输出棋子从方格 $(1, 1)$ 到方格 $(H, W)$ 的路径数，模数为 $(10^9+7)$ 。



```input1
3 3
...
.#.
...
```

```output1
10
```

有 $10$ 种路径方式，如下所示：

- $(1,1)\to (1,2)\to (1,3)\to (2,3)\to (3,3)$
- $(1,1)\to (1,2)\to (1,3)\to (3,3)$
- $(1,1)\to (1,2)\to (2,3)\to (3,3)$
- $(1,1)\to (1,3)\to (2,3)\to (3,3)$
- $(1,1)\to (1,3)\to (3,3)$
- $(1,1)\to (2,1)\to (3,1)\to (3,2)\to (3,3)$
- $(1,1)\to (2,1)\to (3,1)\to (3,3)$
- $(1,1)\to (2,1)\to (3,2)\to (3,3)$
- $(1,1)\to (3,1)\to (3,2)\to (3,3)$
- $(1,1)\to (3,1)\to (3,3)$ 

```input2
4 4
...#
....
..#.
....
```

```output2
84
```
从 $(1,1)$ 开始，棋子可以下到 $(1,2)$ 、 $(1,3)$ 、 $(2,1)$ 、 $(2,2)$ 、 $(3,1)$ 或 $(4,1)$ 。

通往 $(4,4)$ 的一条可能路径是 $(1,1)\to (3,1)\to (3,2)\to (4,3)\to (4,4)$ 。

```input3
8 10
..........
..........
..........
..........
..........
..........
..........
..........
```

```output3
13701937
```

## 数据范围

- $2 \leq H,W \leq 2000$
- $S_{ij}$ 是 `#` 或 `.`。
- $S_{1,1}$ 和 $S_{H,W}$ 是 `.`。
