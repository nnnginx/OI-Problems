## 问题描述

陈帅龙知道你们都十分喜欢玩游戏，这个游戏的规则是这样的：

一场比赛中，在规定时间内，进行任意多局游戏，还有可能一场也完成不了，也就是 $0:0$

任性的陈帅龙十分自傲，如果一场比赛中他获胜了，就让妈妈奖励他 $3$￥，如果一场比赛中他打平了，就让妈妈奖励他 $1$￥，如果一场比赛中他打输了，他就不让妈妈奖励他了

假如陈帅龙一共赢了 $S$ 局比赛，输 $T$ 局比赛，那么陈帅龙可能获得最多钱和最少钱的值分别是多少（一共比了 $N$ 场比赛，不算上其他队之间的比赛）？

现在陈帅龙想求一下这个问题，但是他不会求，于是想到帅气的你

## 输入格式

多组数据，每组数据一行：

一行三个整数 $S、T、N（0\leq S、T\leq 100000000，1\leq N\leq 100000000）$。

## 输出格式

对于每组数据输出一行，两个整数表示陈帅龙可能获得最多钱和最少钱的值分别是多少？

```input1
2 2 2
3 3 3
2 2 1
3 3 2
```

```output1
3 2
6 3
1 1
3 2
```

## 样例解释

在一场比赛中，有可能陈帅龙赢了很多局，输了很多局
