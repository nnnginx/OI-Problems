## 题目描述

我们的郭嘉大大经过一段时间发现了袁绍这个人干大事而惜身，见小利而忘义，又逢曹操在招兵买马，决定逃离袁绍去投曹操，而我们的曹操在第$M$天招募良材，我们的郭嘉大大既不能早去，也不能晚去，于是乎，他就趁着这一段时间到其他的城市游历一番，而每两个城市之间只能坐马车来往，由于我们的郭嘉大大很贪钱，他想用最少的费用，所以需要我们帮他求出这一个最小的费用。

## 输入数据

第一行包含两个数$n，m$， 表示有$n$个城市，和$m$天后曹操招纳良材。城市一就是郭嘉所在的城市，城市n就是曹操处。接下来$n * (n C 1)$行描述马车乘坐表。第2到第n行就是描述的城市$1$到$2… n$的马车乘坐表，第$n + 1$到第$2n-1$行描述的城市$2$到城市$1，3..n$的马车乘坐表… … 对每一行，首先有一个数T，表示城市I到城市$J$的马车以$T$为周期，接下来有$T$个数，表示每天的马车的价格，如果价格为$0$则表示没有马车可坐。<br />
（$n \leq 100，m \leq 200, T \leq 20， Price \leq 50000$）

## 输出数据

如果存在这样的路线使郭嘉第m天到达曹操处，则输出最少的费用，否则输出0！

```input1
3 5
2 130 150
3 75 0 80
2 110 100
4 60 70 60 50
3 0 135 140
2 70 80
```

```output1
355
```
