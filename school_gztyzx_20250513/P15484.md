## 题目背景

您十分喜欢做数据结构题，但是毒瘤出题人不会让您做喜欢的题，所以您需要做一道毒瘤**构造题**。

## 题目描述

给定一个正整数 $x$，你可以进行 $k$ 次操作（$k \leq 10^3$）。对于每一次操作，你可以选择一个 $x$ 的因子 $d$，然后让 $x$ 变成 $x-d$。要求在进行 $k$ 次操作后，使得 $x=1$。

另外还有一个限制，对于每一个 $d$ ，你至多只能使用 $8$ 次。

## 输入格式

多测。

第一行，一个正整数 $t$，表示数据组数。

接下来共 $t$ 行，每行一个正整数 $x$。

## 输出格式

输出共 $2 \times t$ 行。

对于每一个询问，输出两行。其中的第一行表示你需要的操作数 $k$ ，接下来一行共 $k$ 个数，第 $i$ 个数表示第 $i$ 次操作后的 $x$。

## 输入输出样例

```input1
3
3
6
17
```

```output1
2
2 1 
3
4 2 1 
5
16 8 4 2 1
```

## 说明/提示

对于 $30\%$ 的数据：$2 \leq x \leq 2\times 10^2$。

样例 ex_divisor2.in 满足此限制。

对于 $50\%$ 的数据：$2 \leq x \leq 2\times 10^5$，保证数据基本随机。

对于 $100\%$ 的数据：$1 \leq t \leq 10^3$，$2 \leq x \leq 10^9$。

样例 ex_divisor3.in 满足此限制。
