## 题目背景

【数据删除】是 OI 中非常重要的一个技巧。

## 题目描述

给出一个长度为 $n$ 的正整数序列 $a_1,a_2,\cdots,a_n$，我们可以对其进行如下操作若干次：

1. 找出一个满足 $|a_i-a_{i+1}|=1$ 的 $i(1 \le i < n)$。
2. 删掉 $a_{i+1}$，之后的数都往前移填补空缺（如果有的话），序列长度减小 $1$。（$n$ 随之变化）

请你通过操作尽可能减小序列的总和（$\sum\limits^n_{i=1}a_i$），求出最小的总和。

## 输入格式

第一行一个数 $n$，表示序列长度。

第二行 $n$ 个数，第 $i$ 个数为 $a_i$。

## 输出格式

输出一行表示答案。

## 样例 #1

```input1
5
7 8 6 9 8
```

```output1
16
```

## 样例 #2

```input2
3
4 3 4
```

```output2
4
```

## 提示

**【样例一解释】**

1. $i=2$，结果为 $[\underline{7},6,9,8]$
2. $i=2$，结果为 $[\underline{7},9,8]$
3. $i=3$，结果为 $[7,\underline{9}]$

最终和为 $7+9=16$，可以证明没有更小的和。

**【样例二解释】**

1. $i=3$，结果为 $[4,\underline{3}]$
2. $i=2$，结果为 $[\underline{4}]$

最终和为 $4$，可以证明没有更小的和。

**【数据范围】**

对于 $100\%$ 的数据：$1\leq n\leq 5\times 10^5,1\leq a_i \leq 10^9$。

|   数据点   |         $n$          |                  $a_i$                  |
| :--------: | :------------------: | :-------------------------------------: |
|   $1\sim2$    |   $\le 10$   |          $\le 10^3$           |
|   $3\sim4$    |  $\le 500$   |          $\le 10^6$           |
|    $5$     |  $ \le 3000$  | $\max^{n}_{i=1}a_i-\min^{n}_{i=1}a_i=1$ |
|    $6$     |  $\le 3000$  |          $\le 10^6$           |
| $7\sim10$ | $5 \times 10^5$ |  |

留空表示无额外限制。
