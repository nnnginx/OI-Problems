

## 题目描述

给定两个有英文字母组成的等长的字符串 $a$ 和 $b$，现在规定以下比较大小的规则：

- **忽略大小写字母之间的差异**。
- 按顺序从左到右比较。
- 若 $a$ 和 $b$ 在当前比较的位置上字母相同，则继续比较下一位。
- 如果当前位字母不同，则比较字母的大小，字母较小的则意味着字符串较小，反之亦然，此时停止枚举。
- 如果不存在不同的位置，那么认为 $a$ 和 $b$ 相等。

## 输入格式

输入共两行，每行一个字符串，分别是 $a$ 和 $b$。

## 输出格式

若 $a$ 和 $b$ 相等，输出 $0$，若 $a < b$ 则输出 $-1$，若 $a >b$ 则输出 $1$。

## 样例

```input1
Aaaa
aaAa
```

```output1
0
```

```input2
Abcd
Adcb
```

```output2
-1
```

```input3
GMHByTSrJTqlmesVcYFq
yMHBzgSrtThlhesVoYaq
```

```output3
-1
```

## 提示/说明

### 数据范围

对于 $100\,\%$ 的数据，$1 \leq |a|,\,|b| \leq 5 \times 10^4$。
