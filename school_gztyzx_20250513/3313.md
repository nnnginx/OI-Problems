## 问题描述

Lily非常关爱学生，担心学生的空余时间过多可能会迈向腐败，于是呕心沥血布置了一项又一项的作业。（听我说谢谢你！）这不，又精心挑选了若干个数学表达式准备布置作业。然而当Lliy转身去喝水时，K同学以迅雷不及掩耳之势在作业单中的一些表达式中加入了一些匹配的多余括号，但确保新的表达式语法正确而且它的值与原式的值相等。因为大家都知道Lliy最难以忍受多余的东西，她一定会花时间去复原。这就给大家留下了宝贵的休（fu）息（bai）时间。

然而，我们都忽略了，我们走过最多的路就是Llily的套路。当Llily发现后，微微一笑。最终我们的作业变成：编写一个程序删除文档中的多余括号。解决不了的操场十圈跑起。

还好K同学没有对原作业单大肆破坏，文档中存在以下规则：

1. 输入文件中有若干个表达式，一行一个。
2. 表达式中的所有变量用单个大写字母表示。
3. 表达式中只有“-”和“+”两种运算符。

程序只删除掉每个表达式的多余括号，而不可以在代数运算上进行处理。

## 输入格式

输入文件由若干个表达式组成。输入的第一行为一个数K（$1≤K≤10$），为表达式数量。以下 $K$ 行，每行有且只有一个语法正确的表达式，可能存在多余空格，每行长度不超过 $255$ 个字符（包括这些空格在内）。

## 输出格式

对于每个表达式，输出去掉多余括号后的结果。请注意，必须按照在输入文件中的顺序输出相应答案，且每行输出一个表达式并去除所有空格。

```input1
3
((A)-( (B)))
(A-B + C) - (A+(B - C)) - (C-(D- E) )
A-(B+C)
```

```output1
A-B
A-B+C-(A+B-C)-(C-(D-E))
A-(B+C)
```
