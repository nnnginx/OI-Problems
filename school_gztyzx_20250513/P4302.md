## 题目描述

高桥参加了 AtCoder 竞赛。比赛有 $N$ 个问题。高桥在比赛期间提交了 $M$ 个意见书。

第 $i$ 次提交是针对第 $p_i$ 个问题并收到判决 $S_i$（`AC` 或 `WA`）。

高桥正确答案的数量是他一次或多次获得 `AC` 的问题数量。

高桥对他一次或多次收到 `AC` 的问题的处罚数量是以下各项的总和：在该问题上首次获得 `AC` 之前收到的 `WA` 数量。

查找高桥正确答案和处罚的数字。

## 输入格式

第一行两个整数$ N $ ,$ M $表示有$N$个问题，$M$个意见书。

接下来$M$行，每行输入两个数据$p_i$和$S_i$，描述意见书的判决结果，即第$p_i$个问题的判决结果是$S_i$。


## 输出格式

输出正确答案的个数和处罚的个数，用空格隔开。

```input1
2 5
1 WA
1 AC
2 WA
2 AC
2 WA
```

```output1
2 2
```

```input2
100000 3
7777 AC
7777 AC
7777 AC
```

```output2
1 0
```

```input3
6 0
```

```output3
0 0
```

## 数据范围

- $ N $ , $ M $ , $ p_i $ 都是整数
- $ 1\ <\ =\ N\ <\ =\ 10^5 $
- $ 0\ <\ =\ M\ <\ =\ 10^5 $
- $ 1\ \leq\ p_i\ \leq\ N $
- $ S_i $ 只会是 `AC`或者 `WA`
