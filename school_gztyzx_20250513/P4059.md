## 问题描述

最近体测，小龙在练习蛙跳。

小龙的练习方式很特别。他会进行 $n$ 次练习，第 $i$ 次练习时用的跑道长 $a_i$ 单位距离。

每次练习的一开始，小龙的体力值为 $a_i$ ，每次跳跃可以跳的距离就是当前的体力值。也就是说，小龙一开始可以从起点开始直接跳到跑道的终点。每次小龙跳到了终点，他会回到起点重新跳，同时，他也会因为太累而导致体力值减 $1$ 。小龙会不停的跳直到体力值为 $0$ 。

为了记录自己练习的历程，小龙使用了特殊的跑道。跑道可以粗略的看成一条线，线上从起点往后一单位距离开始，每一单位距离都有一个计数器。每当小龙跳到其中一个计数器时，这个计数器就会加一。因为小龙很喜欢 $2$ 这个数字，所以每次练习结束时，他想知道所有计数器中有多少个数是 $2$ 的倍数。

每次练习互不影响。

## 输入格式

输入的第一行一个数 $n$ 。

接下来 $n$ 行一个数，第 $i$ 行的数表示 $a_i$ 的值。

## 输出格式

输出共 $n$ 行，每行一个数，表示结果。

```input1
3
3
5
10
```

```output1
2
3
7
```

## 样例1解释：

第一次训练，每次跳跃后跑道状态如下：

1. $[0,0,1]$
2. $[0,1,1]$
3. $[1,2,2]$

综上，答案为2。

第二次训练，每次跳跃后跑道状态如下：

5. $[0,0,0,0,1]$
6. $[0,0,0,1,1]$
7. $[0,0,1,1,1]$
8. $[0,1,1,2,1]$
9. $[1,2,2,3,2]$

综上，答案为3。

## 数据范围

对于 $30\%$ 的数据：$1 \le n \le 10^3$

对于 $100\%$ 的数据：$1 \le n \le 10^5$ ， $1 \le a_i \le 2^{64}-1$
