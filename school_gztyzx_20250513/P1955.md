## 问题描述

小张今年也$AK$了$CSPS-2023$，所以他今天想要学习下文化课。于是他在机房补习了化学的内容，课后他来到了实验室来做实验，实验室中有$n$个瓶子，这些瓶子全是“克莱因瓶”，可以理解为这些瓶子的容量是无限大的，在一开始时，所有的瓶子里面都有$1$升水。

小张觉得这$n$个瓶子数量太多了，妨碍他做实验，所以他想保留恰好$k$个瓶子。由于实验材料的特殊性，小张只能选择当前含水量相同的任意两个瓶子，然后把其中一个瓶子的水全部倒进另一个瓶子，然后空瓶子就可以丢弃了，但是要注意，不可以丢弃有水的瓶子。

然而，现实总是不尽人意的，有些时候小张并不能如愿保留$k$个瓶子，比如当$n=3,k=1$时，只能选择两个$1$升水的瓶子合并，丢弃一个瓶子，那么还剩下两个瓶子，容量分别是$1$和$2$，不能再合并了。

此时小张可以去找化学老师重新拿一些新的瓶子过来，和之前的瓶子继续合并，直到恰好剩下$k$个瓶子为止，新的瓶子一样会装着$1$升的水。

为了不给化学老师带来太大的困扰，小张想知道，最少需要找老师拿多少个新瓶子才能达成目的呢？

## 【输入格式】

第一行，输入两个正整数，$n$和$k$，表示瓶子的数量，和小张想要保留的瓶子数。

## 【输出格式】

一行一个非负整数，表示最少需要拿多少个新瓶子。

```input1
3 1
```

```input2
13 2
```

```input3
1000000 5
```

```output1
1
```

```output2
3
```

```output3
15808
```

## 数据范围

对于$100\%$的数据，$N,K(1 \leq N \leq 10^9，K \leq 1000)$。
