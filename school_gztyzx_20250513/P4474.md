## 题目描述

定义一个字符串是难以阅读的，当且仅当所有偶数位上是大写字母，奇数位上是小写字母。

给定一个字符串，判断是否是难以阅读的。如果是，则输出 `Yes`，否则输出 `No`。

下标从 $1$ 开始。


## 输入格式

一行输入一个字符串$ S $。


## 输出格式

如果是难以阅读的，则输出 `Yes`，否则输出 `No`。

```input1
dIfFiCuLt
```

```output1
Yes
```

```input2
eASY
```

```output2
No
```

```input3
a
```

```output3
Yes
```

## 数据范围

- $ S $的长度在$ 1 $ 以上 $ 1000 $ 以下。
