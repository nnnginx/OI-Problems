## 问题描述

经过了一个阶段的学习，同学们来到了$TYOI$秋季营的测试基地，为了缓解同学们的压力，程序龙决定跟同学们玩一个热身游戏：

在教室的地板上放着$n$个盒子，每个盒子都装着一些小球，对于第$i$个盒子装着$a_i$个小球。

你需要对这些盒子进行整理，过程如下：

- 第$1$步：首先你可以选择任意的一个盒子。
- 第$2$步：你可以将第$1$步中选择的盒子中所有的小球拿出来，重新分配到其它盒子中。

为了让盒子和里面装着的小球整齐划一，程序龙希望你在执行完上诉的两步操作后，可以使得剩余的$n-1$个盒子装着的小球数量是相等的。

当然，由于选择的盒子是任意的，有可能会出现没办法使得剩余的$n-1$个盒子的小球数量都是相等的情况。

因此你可以在任意的盒子中添加若干个小球，使得无论你在第$1$步中选择了哪一个盒子，然后在第二步中把盒子的小球重新分配，肯定可以使得剩余的$n-1$个盒子的小球数量完全一致！

请你求出添加的若干个小球的最少数量。


## 输入格式

第一行输入一个正整数 $n$表示盒子的数量。

第二行输入 $n$ 个正整数，第$i$个正整数$a_i$表示第$i$个箱子中的小球数量。

## 输出格式

共一行，一个整数表示最少需要添加的球数。

```input1
3
3 2 2
```

```input2
4
2 2 3 2
```

```input3
3
0 3 0
```

```output1
1
```

```output2
0
```

```output3
3
```

## 数据范围

对于 $30\%$ 的数据，有$2\le n \le 10$，$0 \le a_i\le 10$ 。

对于 $60\%$ 的数据，有$2\le n \le 100$ ，$0 \le a_i\le 100$ 。

对于 $100\%$ 的数据，有$2\le n\le 10^6$，$0 \le a_i\leq 10^9$。
