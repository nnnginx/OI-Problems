## 问题描述

听到最新的时尚趋势是有三个点的奶牛，Farmer John立即购买了一群三点牛。不幸的是，时尚潮流趋向的变化迅速，当前的时尚是最受欢迎的是只有一个点的奶牛！

Farmer John想通过让他的牛群的三个点合并为一个点从而使得他的牛群更时尚。奶牛的点由 $N×M$ 的字符矩阵表示， 如下所示：

```
................
..XXXX....XXX...
...XXXX....XX...
.XXXX......XXX..
........XXXXX...
..XXX....XXX....
```

在这里，每一个 $X$ 是指点的一个部分。如果两个 $X$ 是纵向或横向相邻（对角相邻的不计数），则属于同一个点。所以上面的数字正好有三点，FJ的所有奶牛都有三点。

FJ希望使用尽可能少的油漆而使三个点合并为一个点，在上面的例子中，他只能画四个额外的有 $X$ 标记的字符（新字符的下方都有 $*$ 标记，这样他们更容易看到）。

```
................
..XXXX....XXX...
...XXXX*...XX...
.XXXX..**..XXX..
...*....XXXXX...
..XXX....XXX....
```

请帮助FJ尽可能少的使用 $X$ 而使得三个点合并为一个大的点。

## 输入格式

第一行：$2$ 个空格分隔的整数，$N$ 和 $M$ ；

第 2.. N+1 行：每一行的长度用 $M$ 个字符表示，由一系列 $X$ 或 $.$ 组成，每一行代表一排牛身上的点；

## 输出格式

输出共一行一个整数，为获得一个单独的点而需添加到输入模式的 $X$ 的最小数目。

```input1
6 16
................
..XXXX....XXX...
...XXXX....XX...
.XXXX......XXX..
........XXXXX...
..XXX....XXX....
```

```output1
4
```

## 输入说明

该模式的输入显示3个不同的点组成的；

## 输出详细

增加四个’X’后使得三个点合并成为一个点。

## 数据规模

对于 $50\%$ 的数据： $1≤N,M≤20$；

对于 $100\%$ 的数据：$1≤N,M≤50$；
