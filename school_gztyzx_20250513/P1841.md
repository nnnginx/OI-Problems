## 问题描述

国际乒联现任主席沙拉拉自从上任以来就立志于推行一系列的改革，以推动乒乓球运动在全球的普及。其中 $11$ 分制就引起了很大的争议，有一部分球员因为无法适应新规则而只能选择退役。华华就是其中的一位，他退役之后走上了乒乓球研究工作，意图弄明白 $11$ 分制和 $21$ 分制对选手的不同影响。

在开展他的研究前，他首先需要对他多年比赛的统计数据进行一些分析，所以需要你的帮忙。

华华通过以下方式进行分析，首先把比赛每个球的胜负列成一张表，然后分别计算在 $11$ 分制和 $21$ 分制下，双方的比赛结果。

比如现在有这么一份记录，（其中W表示华华得一分，L表示表示华华对手得一分）

```
WWWWWWWWWWWWWWWWWWWWWWLW
```

在 $11$ 分制下，此时得比赛结果是华华第一局 $11$ 比 $0$ 获胜，第二局 $11$ 比 $0$ 获胜，正在进行第三局，当前比分 $1$ 比 $1$。而在 $21$ 分制下，此时得比赛结果是华华第一局 $21$ 比 $0$ 获胜，正在进行第 $2$ 局，比分 $2$ 比 $1$。如果一局比赛刚开始，则比分为 $0$ 比 $0$。

你的程序就是要对于一系列比赛信息的输入（WL形式），输出正确结果。

## 输入格式

每个输入文件包含若干行字符串（每行至多20个字母），字符串由大写得W、L和E组成，其中E表示比赛信息得结束，程序应忽略E之后得所有内容。

## 输出格式

输出由两部分组成，每部分若干行，每一行对应一局得比分（按比赛信息输入顺序）。其中第一部分是 $11$ 分制下的结果，第二部分是 $21$ 分制下的结果，两部分之间由一个空行分隔。

```input1
WWWWWWWWWWWWWWWWWWWW
WWLWE
```

```output1
11:0
11:0
1:1

21:0
2:1
```

## 提示

> **对于乒乓球知识的一些解释**。
> 乒乓球是一种双人对抗体育项目，每球得分制。11分制是指，选手双方谁先取得11分，就获得一局比赛的胜利；而21分制是指选手双方谁先取得21分，就获得一局比赛的胜利。当某人分数达到11分（11分制下）或21分（21分制下），但双方的得分差小于两分时（比如在21分制下21：20），则比赛继续进行，直到双方差距等于2分为止。请注意，本题中并不考虑一场比赛所包含的局数，也就是说一场比赛可能由任意数量的局组成。<br />
