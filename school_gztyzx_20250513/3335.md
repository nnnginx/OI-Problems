## 问题描述

设有一个整型数组 $A$，数组中存放的元素为 $0\sim n-1$之间的整数，且 $A[i]≠A[j]$（当 $i≠j$ 时）。

例如：$N=6$ 时，有：$A=(4,3,0,5,1,2)$

此时，数组A的编码定义如下：

$A[0]$ 的编码为 $0$；

$A[i]$ 的编码为：在 $A[0]，A[1]，……A[i-1]$ 中比 $A[i]$ 的值小的个数（$i=1，2……N-1$）

上面数组 $A$ 的编码为：$B=(0,0,0,3,1,2)$



**程序要求解决以下问题：**

①   给出数组 $A$ 后，求出其编码；

②   给出数组 $A$ 的编码 $B$ 后，求出 $A$ 中的原数据。

## 输入格式

第一行输入整数 $N$ 和 $k$，$k=0$表示接下来的是 $A$ 数组，$k=1$ 表示接下来的是 $B$ 数组

第二行输入 $N$ 个整数。


## 输出格式

当输入的是A=(...)，则输出其编码。

当输入的是B=(...)，则输出A中的原数据。

输出数据的格式和输入数据的格式是一样的。

```input1
6 0
4 3 0 5 1 2
```

```output1
0 0 0 3 1 2
```

```input2
6 1
0 0 0 3 1 2
```

```output2
4 3 0 5 1 2
```



## 数据范围

对于$100\%$的数据，$n\leq 10^4$.
