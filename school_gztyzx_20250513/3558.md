## 题目描述

在$TY$校园里有一个创智空间，里面可以体验一个飞船游戏。

飞船游戏是在一个二维的平面上飞行的，你可以驾驶一架飞船，从起点$(x_1,y_1)$，飞向终点$(x_2,y_2)$.

但是呢在飞行的过程中，每天都刮起大风，这个风的方向可以是上、下、左、右中一个方向的风。由于这个游戏已经被可恶的小哲破解了，因此你知道了后面每天风吹的方向，每天风的方向是有规律的，可以用一个字符串$s$​表示，包含$U,D,L,R$​四种字母，分别表示吹上、下、左、右中一个方向的风。

其中第一天的风向为$s_1$​，第二天吹的风向为$s_2$​，…第$n$​天吹的风向为$s_n$​，第$n+1$天吹的风向为$s_1$，第$n+2$天吹的风向为$s_2$…以此类推，形成一个循环。

- 假如吹的风往上, 船会从$(x,y)$移动到$(x,y+1)$;
- 假如吹的风往下,船会从$(x,y)$移动到$(x,y-1)$;
- 假如吹的风往左, 船会从$(x,y)$移动到$(x-1,y)$;
- 假如吹的风往右, 船会从$(x,y)$移动到$(x+1,y)$​;

与此同时你每天也可以从上下左右中选择一个方向，移动一个单位，当然也可以选择停下来，即每天的位移为风向和你自己移动的叠加。

例如某天风向向上，你选择往左移动了一个单位，那么你会从$(x,y)$移动到$(x-1,y+1)$​.又如某天风向向上，你选择往下移动一个单位，那么你会保留在原地。

为了快速抵达终点，请你计算出最少需要多少天可以到达终点。

## 输入格式

第一行两个整数表示$x_1,y_1$。

第二行两个整数表示$x_2,y_2$。

第三行一个整数$n$,表示$s$的长度。

第四行，字符串$s$.

## 输出格式

输出最少可以到达目的地的天数，如果到达不了输出$-1$.

### 输入 #1

```
0 0
4 6
3
UUU
```

### 输出 #1

```
5
```

### 输入 #2

```
0 3
0 0
3
UDD
```

### 输出 #2

```
3
```

### 输入 #3

```
0 0
0 1
1
L
```

### 输出 #3

```
-1
```

## 数据范围

对于 $50\%$​​​​ 的数据，$1 \le n \le 1000,0\le x_1,y_1,x_2,y_2\le1000$​​​​。

对于 $100\%$​​ 的数据，$1 \le n \le 10^5,0\le x_1,y_1,x_2,y_2\le 10^9$​​。
