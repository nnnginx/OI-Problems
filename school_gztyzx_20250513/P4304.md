## 题目你描述

输入 $a,b$，输出「由 $a$ 个 $b$ 组成的字符串」和「由 $b$ 个 $a$ 组成的字符串」当中，字典序更小的那一个。

## 输入格式

输入两个数字$a、b$.

## 输出格式

在按题目要求组成的$2$个字符串中，按词典顺序输出较小的字符串。（如果两个字符串相等，输出其中一个。）

```input1
4 3
```

```output1
3333
```

```input2
7 7
```

```output2
7777777
```

## 数据范围

- $ 1\ <\ =\ a\ <\ =\ 9 $
- $ 1\ <\ =\ b\ <\ =\ 9 $
- $ a,b $都是整数
