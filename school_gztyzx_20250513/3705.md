## 题目描述

小C和小D在玩游戏，小C给小D一个正整数n。小D可以从$2,3,\dots,n$选择一个数$k$不断进行如下操作：

* 如果$kOn$，那么令$n=n/k$，否则令$n=n-k$，不断操作直到$n$小于$k$

如果最后$n=1$，则小D胜利，现在小D想知道一共有多少种不同的$k$可以获得胜利。

## 输入格式

输入第一行包含一个整数$N$。

## 输出格式

输出1个整数表示答案。

```input1
6
```

```output1
3
```

```input2
3141
```

```output2
13
```

```input3
314159265358
```

```output3
9
```

## 样例解释

样例1：

* k=2时, 6->3->1
* k=5时, 6->1
* k=6时, 6->1
