## 问题描述

台风“山竹”从西太平洋呼啸而来，15级风力带来强降雨和海水倒灌，于是，oiclass就这样被水淹没了。

还好oiclass在某些重要的地方起一些围墙，用 `*` 号表示，而一个封闭的 `*` 号区域洪水是进不去的……

现在给出oiclass的围墙建设图，请计算出oiclass没被淹到的重要区域（由 `0` 表示）有多少。

现在，oiclass处于危难中，需要你的救援……

## 输入格式

第一行是两个数，$x$ 和 $y$（$x,y\leq 500$）

第二行及以下是一个由 `*` 和 `0` 组成的 $x*y$ 的图。

## 输出格式

输出没被水淹没的“0”的数量。

```input1
4 5
00000
00*00
0*0*0
00*00
```

```output1
1
```

```input2
5 5
*****
*0*0*
**0**
*0*0*
*****
```

```output2
5
```
