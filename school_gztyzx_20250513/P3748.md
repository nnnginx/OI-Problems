## 问题描述

“我们在这个有 $10^{90}$ 个基本粒子的宇宙中，它的物质是有限的，排列组合也是有限的。只要它持续演化 $10^{10^{122} }$， 年，总会回到以前经历过的某个状态。 尽管这个时间长得没办法想象，但总有一天，那时，你还是你，我还是我，宇宙还是这个宇宙，一切都回到现在。 同样，在很久以前，我们都曾出现过。”

我们生活在大地上，但我们的梦想超越天空。

那么，使用鞘翅飞到天空中去吧！

现在，CPP想要使用鞘翅飞上天空，他现在想要到达的目的地有 $n$ 个，每个地点距离CPP所处位置的距离为 $a_i$，由于CPP比较奇怪，他想要你编写一个程序，求出第 $s$ 小到第 $k$ 小的所有目的地个数 ，额外地，请分别告诉CPP这几个目的地的距离分别是多少，从小到大排列。

## 输入格式

共 $n+2$ 行，

第一行，一个整数 $n$，表示CPP有 $n$ 个目的地。

第二行，两个整数 $s$ 和 $k$。

接下来 $n$ 行，每行一个正整数 $a_i$，表示每个地点距离CPP所处位置的距离。

## 输出格式

共2行，

第一行，一个整数 $p$，表示第 $s$ 小到第 $k$ 小的所有目的地个数 。

第二行，共 $s-k+1$ 个整数，这几个目的地的距离分别是多少，从小到大排列。

```input1
3
1 2
3274
1273
923
```

```output1
2
923
1273
```

## 数据范围

对于 $20\%$ 的数据，$n \le 30$。

对于 $60\%$ 的数据，$n \le 2000$，$1 \le s \le k \le 500$。

对于 $100\%$ 的数据，$n \le 10^5$，$1 \le s \le k \le 10^7$，$a_i \le 10^9$
