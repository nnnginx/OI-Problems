## 问题描述

光阴似箭，日月如梭。转眼间，小学的生活就要结束了。CPP还没有具体地统计过他同学的学号。现在请你帮CPP统计一下吧！

现在告诉你CPP班级里同学的个数 $n$（包括CPP），要求你求出有多少人的学号是 $m$ 的倍数以及他们的学号。保证至少有一人是符合要求的。

## 输入格式

一行，两个整数 $n$ 和 $m$。

## 输出格式

共 $s$ 行，

第一行，一个整数 $k$，表示有 $k$ 个人的学号是 $m$ 的倍数。

接下来 $s-1$ 行，每行1个数，表示这个人的学号是符合条件的，从小到大排列。

```input1
300 60
```

```output1
5
60
120
180
240
300
```

## 数据范围

对于 $100\%$ 的数据，$m \le n \le 10^{15}$。
