## 题面描述

我们有上面写着 $1$ 的 $A$ 张牌，上面写着 $0$ 的 $B$ 张牌，上面写着 $-1$ 的 $C$ 张牌。

此时我们一次性从中取出 $K$ 张牌，求所取的牌的最大和。

## 输入格式

一行输入四个整数： $ A $， $ B $， $ C $ ，$ K $，含义如题。

## 输出格式

输出所取的牌的最大和。


```input1
2 1 1 3
```

```output1
2
```

```input2
1 2 3 4
```

```output2
0
```

```input3
2000000000 0 0 2000000000
```

```output3
2000000000
```

## 数据范围

对于$100\%$的数据，$ 0\ \leq\ A,\ B,\ C $

对于$100\%$的数据，$ 1\ \leq\ K\ \leq\ A\ +\ B\ +\ C\ \leq\ 2\ \times\ 10^9 $
