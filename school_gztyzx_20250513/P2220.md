## 问题描述

Chen.Jr最近得到了一些卡片，这些卡片上标有数字 $0$ 或 $5$。现在Chen.Jr可以选择其中一些卡片排成一列，使得排出的一列数字组成的数最大，且满足被 $90$ 整除这个条件。

同时这个数不能含有前导 $0$，即 $0$ 不能作为这串数的首位(即不能选取两个数 $0$ 和 $5$ 并构成数字 $05$，但是单独的一个 $0$ 可以作为首位)。

如果不能排出这样的数，输出“-1”(不用输出双引号)。

## 输入格式

第一行，卡片的个数 $n$。

第二行，分别给出了这 $n$ 个数（只能为数字 $5$ 或 $0$）。

## 输出格式

仅一行，如果可以排出，则输出这个数。否则输出“-1”(不用输出双引号)。

```input1
4
5 0 5 0
```

```output1
0
```

```input2
11
5 5 5 5 5 5 5 5 0 5 5
```

```output2
5555555550
```

## 数据规范

对于 $20\%$ 的数据，$n\leq 5$，且卡片上的数都是是 $5$；

对于 $40\%$ 的数据，$n\leq 10$；

对于 $100\%$ 的数据，$n\leq 1000$；
