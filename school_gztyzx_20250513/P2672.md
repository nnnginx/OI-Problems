## 问题描述

小$J$想吃小$M$的苹果，他们约定好，小$J$只有正确回答出小$M$提的问题，才能吃到苹果。小$M$的问题如下：

给定两个区间 $L$ 和 $R$，区间 $L$ 的范围是从 $A\sim B$，区间 $R$ 的范围是从 $C\sim D$（$A,B,C,D$ 均为整数），现在可任意次数使用 $L$ 区间内的整数，并将这些整数做加法，问通过上述操作，可以覆盖多少个区间 $R$ 中的不同整数。

你能帮帮小$J$想出问题的解法吗？

## 输入格式

第 $1$ 行：一个数 $T$，表示后面用作输入测试的数的数量。（$1 \leq T \leq 1000$)

第 $2 - T + 1$ 行：每行 $4$ 个数：$A,B,C,D$，中间用空格分隔。($1 \leq A < B \leq 10^{18}$; $1 \leq C < D \leq 10^{18}$)

## 输出格式

输出共 $T$ 行，每行 $1$个数，区间 $[C,D]$ 中可以由 $A\sim B$ 中的整数相加得到的不同整数的数量。

```input1
1
8 10 3 20
```

```output1
8
```

## 样例1解释

可凑出来的数字为 `8 9 10 16 17 18 19 20`

```input2
1
5 15 8 30
```

```output2
23
```

```input3
1
3 5 4 10
```





```output3
7
```



## 数据范围

对于 $20\%$ 的数据，$1 \leq A,B,C,D \leq 10000$

对于 $44\%$ 的数据，$1 \leq T \leq 10$

对于 $64\%$ 的数据，$1 \leq T \leq 100$

对于 $100\%$ 的数据，$1 \leq T \leq 1000$，$1 \leq A,B,C,D \leq 10^{18}$
