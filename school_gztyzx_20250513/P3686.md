## 问题描述

由于疫情，最近的演出活动被取消了，Fushimi Yuzuru感到无聊。Tenshouin Eichi建议弓弦去玩《偶像梦幻祭2》（我玩我自己）。

下载游戏需要装载很多很多个插件，第 $i$ 个插件占用的内存为 $2^i$ ，由于插件过多，伏见弓弦不可能下载所有插件。他只希望内存的利用率达到最高，即最后剩余的内存空间最小。请你帮他计算出利用率达到最高时所选择下载的插件编号。

## 输入格式

一个数 $n$，表示伏见弓弦的内存空间。

## 输出格式

输出共一行，由小到大输出利用率达到最高时选择下载的插件编号，两个数之间用空格隔开。

```input1
182
```

```output1
1 2 4 5 7
```

## 数据范围

$1 \leq n \leq 1e5$
