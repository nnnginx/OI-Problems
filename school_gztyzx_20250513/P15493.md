## 题目描述

定义一个数列 $f$，数列的第一项 $f_1$ 为 $1$，第二项 $f_2$ 为 $1-\frac{1}{4}$，此后对于第 $i$ 项，$f_i=1-\frac{1}{i^2}$。给定一个正整数 $n$ ，求 $\prod\limits_{i=1}^n f_i$。

## 输入格式

共一行，一个正整数 $n$。

## 输出格式

共一行，两个整数，分别表示答案的分子和分母，中间用“/”隔开。

## 输入输出样例

```input1
5
```

```output1
3/5
```

## 说明/提示

对于 $20\%$ 的数据，$2\leq n\leq10^5$。

样例 ex_number2.in/ex_number2.ans 满足此限制。

对于 $50\%$ 的数据，$2\leq n\leq10^{15}$。

样例 ex_number3.in/ex_number3.ans 满足此限制。

对于 $100\%$ 的数据，$2\leq n\leq10^{100000}$。

样例 ex_number4.in/ex_number4.ans 满足此限制。
