**【题目描述】**

ty的集训开始了，同学们在机房中努力训练。

在一次课间，波学长画出了编号为$1-N$的号码，并制定了一些游戏规则，这时候一个师弟看到了，觉得波学长在故弄玄虚，于是他喊出了那句名言。

因此"明学长"不得不鼓足勇气上前和波学长玩这个游戏。游戏规则是这样的：

波学长手中有$N$个号码，编号1~N(排列顺序不一定是按照编号顺序排列)。明学长需要将一段连续的号码进行翻转，也就是旋转180°，将连续号码的第一个与最后一个交换位置，第二个与倒数第二个交换位置...以此类推，比如一个序列$1 3 2 4 5$，明学长可以翻转$[1,3]$的连续号码。翻转之后可得新的序列：$2 3 145$。

但是波学长喜欢固定点，即号码的编号与它的位置相对应，编号为x的号码要刚好排在第x个位置，波学长希望出现的固定点越多越好，如果明学长能找到一段连续区间的号码进行翻转，且翻转之后的固定点最多，那么明学长取胜。

明学长可急坏了，因为实在是太难，于是他请同班同学的你帮助明学长，只能选择一段连续区间的号码进行颠倒，然后使得固定点的数目尽可能的多。

**【输入格式】**

第一行一个正整数$N$，为波学长手中的号码数。

第二行$N$个数，为手中每个号码的具体编号。

**【输出格式】**

输出两个数正整数$A，B$，表示将号码$A$到号码$B$之间的所有纸牌进行颠倒得到的固定点最多，如果有多种情况均满足条件，输出翻转纸牌最少的一种，如果翻转数目相同，输出$A$最小的情况。

**【样例1输入】**

```
4
3 2 1 4
```

**【样例1输出】**

```
3 1
```

**【样例1解释】**

将3 2 1旋转后得到1 2 3 ，此时固定的数目最多，有4个。

**【样例2输入】**

```
2
1 2
```

**【样例2输出】**

```
1 1
```

**【数据范围】**

对于30%的数据，$1≤N≤500$。

对于60%的数据，$1≤N≤5000$。

对于100%的数据，$1≤N≤500000$。
