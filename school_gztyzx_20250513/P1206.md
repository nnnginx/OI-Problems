## 问题描述

Farmer John一直努力让他的草地充满鲜美多汁的而又健康的牧草。可惜天不从人愿，他在植物大战人类中败下阵来。邪恶的乳草已经在他的农场的西北部份琢炝艘黄立足之地。

草地像往常一样，被分割成一个高度为 $Y(1\leq y \leq 100)$，宽度为 $X(1 \leq x \leq 100)$ 的直角网格。 $(1,1)$ 是左下角的格（也就是说坐标排布跟一般的 $X,Y$ 坐标相同）。乳草一开始琢炝烁 $(M_x,M_y)$。每个星期，乳草传播到已被乳草琢斓母褡铀拿姘朔降拿恳桓雒挥泻芏嗍头的格（包括垂直与水平相邻的和对角线上相邻的格）。$1$ 周之后，这些新琢斓母裼挚梢园讶椴荽播到更多的格里面了。

Bessie想要在草地被乳草完全琢熘前尽可能的享用所有的牧草。她很好奇到底乳草要多久才能琢煺个草地。如果乳草在 $0$ 时刻处于格 $(M_x,M_y)$，那么还在那个时刻它们可以完全琢烊肭终片草地呢（对给定的数据总是会发生）？

草地由一个图片表示。`.` 表示草，而 `*` 表示大石。比如这个 $X=4, Y=3$ 的例子。

```
....
..*.
.**.
```

如果乳草一开始在左下角（第 $1$ 排，第 $1$ 列），那么草地的地图将会以如下态势发展：

```
....  ....  MMM.  MMMM  MMMM 
..*.  MM*.  MM*.  MM*M  MM*M 
M**.  M**.  M**.  M**.  M**M 
  0     1     2     3     4
            星期数
```

乳草会在 $4$ 星期后琢煺片土地。

## 输入格式

第一行: 四个由空格隔开的整数: $X, Y, M_x, M_y$

第2到第Y+1行: 数据的第y+1行由 $X$ 个字符（`.` 表示草地，`*` 表示大石），描述草地的第(Y+2-y)行。

## 输出格式

第一行: 一个单独的整数表示最后一个不是大石块的格子被乳草琢斓男瞧谑。

```input1
4 3 1 1
....
..*.
.**.
```

```output1
4
```
